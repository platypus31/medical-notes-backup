# Upper GI Bleeding（上消化道出血）

> [!danger] 🚨 紅旗警訊（must-not-miss，先穩生命徵象再談內視鏡）
> **助記「血流動力學 → 靜脈曲張 → 抗凝」**
> 1. **血流動力學不穩定**（低血壓、心搏過速、[[Orthostatic Hypotension(姿勢性低血壓)]]、意識改變、Shock Index↑）→ 大失血，先復甦
> 2. **[[Esophageal varices(食道靜脈曲張)]] 出血**（肝硬化 / 酗酒背景、鮮血、又快又急）→ 死亡率高，需及早內視鏡 + 血管收縮劑 + 抗生素
> 3. **持續嘔鮮血 / 大量黑便 + 生命徵象惡化** → 活動性出血
> 4. **抗凝 / 抗血小板藥物 + 凝血異常** → 出血難止，先問藥物史
> 5. **主動脈 - 腸道廔管**（有腹主動脈瘤 / 人工血管病史 + 大量 GI 出血）→ 罕見但致命，「前哨少量出血」後可能大爆發
>
> ⚡ **床邊先量生命徵象 + 建立 2 條大孔 IV + 抽血驗血 + type & cross**，不穩定先輸液 / 輸血，別等內視鏡

## 🔀 鑑別診斷 DDx（值班從這裡連到疾病）
> 上消化道 = **Treitz 韌帶以上**（食道·胃·十二指腸）。先「靜脈曲張 vs 非靜脈曲張」分兩大類，決定用藥與預後。

| 疾病 | 支持特徵 | rule-out 線索 |
| --- | --- | --- |
| [[Peptic ulcer disease(消化性潰瘍)]]（最常見） | 上腹痛、H. pylori / [[NSAID]] / [[Steroid(類固醇)]] / 壓力史；[[Duodenal Ulcer(十二指腸潰瘍)]] 飯後 2–5h 痛、胃潰瘍進食即痛 | 內視鏡無潰瘍 |
| [[Esophageal varices(食道靜脈曲張)]]（門脈高壓） | 肝硬化 / 酗酒、鮮血又快又急、腹水 / 脾腫 / 蜘蛛痣 | 無肝病證據 + 內視鏡無曲張 |
| [[Mallory Weiss Syndrome(馬洛利魏斯症候群)]] | **劇烈嘔吐 / 反覆催吐後**吐鮮血、酗酒 / 減重催吐史 | 無先吐後血病史 |
| [[Corpus predominant gastritis(胃體炎)]] / 糜爛性胃炎 | [[NSAID]] / [[Aspirin]] / 酗酒 / 壓力（重症壓力性潰瘍） | 內視鏡黏膜正常 |
| [[Gastric cancer(胃癌)]] / 惡性 | 體重減輕、慢性、貧血、家族史、慢性隱血 | 無警訊 + 內視鏡陰性 |
| 藥物相關出血 | 抗凝 / 抗血小板、凝血異常 | 停藥 / 凝血正常 |
| 血管病變（[[Aneurysm(動脈瘤)]] 主動脈腸道廔管、Dieulafoy、血管發育不良） | 大量 / 反覆難解釋出血、主動脈瘤 / 人工血管史 | 影像 / 內視鏡排除 |

> [!warning] **黑便（melena, tarry stool）≈ 上消化道出血**（血在腸道被消化變黑）；但右側大腸慢速出血也可能黑便。**鮮血便（hematochezia）** 多為下消化道，但**大量快速的上消化道出血**也可能來不及變黑而排鮮血 → 別只憑顏色定上下，血流動力學不穩 + 鮮血便也要想 UGIB。

## ❓ 問診 / 身體檢查重點
- **出血型態**：嘔鮮血 / 咖啡渣樣（hematemesis）？黑便（melena）？鮮血便？量？
- **關鍵病史**：肝硬化 / 酗酒（曲張）、NSAID / aspirin / 類固醇（潰瘍 / 胃炎）、先劇烈嘔吐再吐血（Mallory-Weiss）、**抗凝 / 抗血小板藥**、體重↓（惡性）、腹主動脈瘤 / 人工血管
- **理學**：生命徵象 + [[Orthostatic Hypotension(姿勢性低血壓)]]、pale conjunctiva（貧血）、icteric sclera、慢性肝病徵（蜘蛛痣 spider angioma、palmar erythema、caput medusae、腹水、脾腫）、腹部（視→聽→叩→觸，先屈膝放鬆）、身上其他出血點
- **DRE 肛門指診**：確認黑便 / 血便性狀（徵得同意）

## 🩺 初步 workup（該開的檢查 / 影像）
> [!note] 黃金第一步：**穩定生命徵象（2 大孔 IV + 輸液 / 輸血）+ 抽 CBC / 凝血 / type & cross**，同時算 Glasgow-Blatchford 分流；內視鏡是診斷 + 治療但**先復甦後鏡檢**
- **CBC/DC**（Hb 初期可能正常，未血液稀釋前）、**凝血 PT/INR/aPTT**、**type & cross 備血**
- **電解質、BUN/Cr（BUN/Cr 比值↑ 支持上消化道）、肝功能（AST/ALT/Bilirubin/Alk-P）**
- **內視鏡（EGD）**：診斷 + 治療首選，**穩定後 24 小時內**；看到潰瘍可加做 H. pylori
- NG 灌洗：傳統參考（有血 → 支持 UGIB），**陰性不排除**（十二指腸出血、幽門關閉時抽不到血）→ 臨床價值有限，不必過度依賴
- CXR（穿孔看游離氣體）；反覆 / 找不到源 → CT 血管攝影 / 血管攝影

## ⚡ 值班即時處置（穩定 vs 不穩定分流）
```mermaid
flowchart TD
    A["疑上消化道出血"] --> B["生命徵象 + Shock Index<br/>2 大孔 IV + 抽血 + type&cross"]
    B --> C{"血流動力學?"}
    C -->|不穩定| D["積極輸液/輸血復甦<br/>矯正凝血 (依院內指引)<br/>會診 GI 緊急內視鏡"]
    C -->|穩定| E["算 Glasgow-Blatchford (GBS)"]
    E -->|GBS = 0-1 極低危| F["考慮門診 / 早期 EGD 追蹤"]
    E -->|GBS ≥ 2| G["住院 + 24h 內 EGD"]
    D & G --> H{"疑靜脈曲張?"}
    H -->|是 (肝硬化)| I["血管收縮劑 (依院內指引)<br/>+ 預防性抗生素<br/>+ 內視鏡結紮/硬化"]
    H -->|否 (潰瘍等)| J["PPI (依院內指引)<br/>+ 內視鏡止血 + H.pylori"]
```
- **復甦**：晶體液 → 輸血；**限制性輸血策略**（多數非大出血目標 Hb ~7 g/dL，心血管疾病者放寬，依院內指引）；矯正凝血 / 逆轉抗凝（依院內指引）
- **藥物**（品項 / 劑量一律**依院內指引**）：疑潰瘍 → PPI；**疑靜脈曲張 → 血管收縮劑（如 terlipressin / octreotide 類）+ 預防性抗生素**（降低感染 + 再出血 + 死亡）
- **內視鏡**：穩定後 24h 內；曲張 → 結紮 / 硬化療法；潰瘍 → 依 Forrest 分級止血
- ⚠️ 估血量參考：~60 kg 成人血量約 5 L；用生命徵象 + Shock Index + 尿量評估失血程度，別只看單次 Hb

## 📊 臨床評分 / 風險分層（scoring）★本卡核心
> UGIB 分流靠 **Glasgow-Blatchford（鏡前，決定住院 / 門診）** + **AIMS65（死亡風險）** + **Rockall（鏡後再出血 / 死亡）**。

### ① Glasgow-Blatchford Score（GBS，鏡前風險 · 逐項點數）
| 項目 | 數值 | 分數 |
| --- | --- | --- |
| **BUN**（血中尿素氮 mg/dL） | 18.2–22.3 | 2 |
| | 22.4–27.9 | 3 |
| | 28.0–69.9 | 4 |
| | ≥ 70 | 6 |
| **血紅素 Hb（男 g/dL）** | 12.0–12.9 | 1 |
| | 10.0–11.9 | 3 |
| | < 10.0 | 6 |
| **血紅素 Hb（女 g/dL）** | 10.0–11.9 | 1 |
| | < 10.0 | 6 |
| **收縮壓 SBP（mmHg）** | 100–109 | 1 |
| | 90–99 | 2 |
| | < 90 | 3 |
| **其他徵象（各自加）** | 心率 ≥ 100 | 1 |
| | 黑便 melena | 1 |
| | 暈厥 syncope | 2 |
| | 肝病史 | 2 |
| | 心衰竭史 | 2 |

| 總分 | 意義 | 處置分流 |
| --- | --- | --- |
| **0（或 ≤1）** | 極低危 | 再出血 / 需介入風險極低 → 可考慮**門診處理 + 早期門診內視鏡** |
| **≥ 2** | 需住院 | 住院 + 安排內視鏡（分數越高越急） |
| **高分（如 ≥7）** | 高危 | 需輸血 / 內視鏡介入機率高，積極處理 |

> [!tip] GBS 最大價值是**「揪出可安全回家的極低危病人」**（GBS 0）— 純鏡前臨床 + 抽血即可算，不需內視鏡結果。

### ② AIMS65（住院死亡風險，5 項各 1 分）
| 項目 | 1 分標準 |
| --- | --- |
| **A**lbumin | 白蛋白 < 3.0 g/dL |
| **I**NR | INR > 1.5 |
| **M**ental status | 意識改變 |
| **S**ystolic BP | 收縮壓 ≤ 90 mmHg |
| **65** | 年齡 ≥ 65 歲 |

> 分數越高，住院死亡率越高（0 分 ~0.3% → 5 分可達 ~30%+），簡單好記適合快速預後判斷。

### ③ Rockall Score（鏡後再出血 / 死亡）
- 含年齡、休克（HR/SBP）、共病、**內視鏡診斷 + 出血徵象（stigmata）** → 需內視鏡結果，屬**鏡後**分層；GBS / AIMS65 屬鏡前。
