# Vertigo（頭暈 / 眩暈）

> [!danger] 🚨 紅旗警訊（must-not-miss，眩暈最怕漏掉的是中樞中風）
> **助記「中樞找 5D + HINTS」**：周邊眩暈良性居多，但**別把後循環中風當耳石**
> 1. **後循環中風 / TIA**（腦幹 / 小腦）→ **5D**：Diplopia 複視、Dysarthria 構音障礙、Dysphagia 吞嚥困難、Dysmetria 辨距不良、Drop attack；加 Ataxia 步態站不住、單側肢體無力麻木
> 2. **中樞型眼震**（方向會變、垂直 / 純旋轉、凝視不抑制）
> 3. **HINTS 指向中樞**（見下 scoring）→ 比早期 MRI 更敏感抓急性小腦 / 腦幹中風
> 4. **無法站立 / 走路**（truncal ataxia 嚴重到坐不穩）→ 小腦病灶警訊
> 5. **突發劇烈頭痛 + 眩暈**（小腦出血）
>
> ⚡ **年長 + 血管危險因子 + 持續性眩暈 + 神經徵象** → 先當中樞處理排中風，不要急著貼「BPPV / 前庭神經炎」標籤

## 🔀 鑑別診斷 DDx（先分中樞 vs 周邊）
> 第一步先把「頭暈」定性：**旋轉性眩暈（vertigo）** vs 快昏（presyncope）vs 站不穩（disequilibrium）vs 頭昏沉。真正 vertigo 才走下表。

| 疾病 | 支持特徵 | rule-out 線索 |
| --- | --- | --- |
| **周邊（~80%，內耳）** | | |
| [[Benign Paroxysmal Positional Vertigo(良性陣發性位置性眩暈)]] BPPV | 耳石脫位、**變換頭位誘發、每次數秒**、Dix-Hallpike 陽性（後半規管最常見） | 持續性 / 靜止也暈 → 非 BPPV |
| [[Vestibular neuritis(前庭神經炎)]] | 感冒後突發、**嚴重且持續數天**、噁心嘔吐步態不穩、**無聽力損失** | 有聽損 → 想 labyrinthitis / Meniere；有神經徵 → 想中樞 |
| [[Meniere disease(美尼爾氏症)]] | 內淋巴水腫、**耳鳴 + 感音聽損 + 耳脹 + 眩暈數分至數小時**、反覆發作 | 無聽力 / 耳症狀 |
| 迷路炎 / 其他周邊 | [[Schwannoma(神經鞘瘤)]]（聽神經瘤）、aminoglycoside 毒性、[[Herpes zoster(帶狀疱疹)]]（Ramsay Hunt） | — |
| **中樞（~20%，腦幹 / 小腦）** | | |
| [[Stroke(中風)]]（後循環） | 血管危險因子、**持續性**、伴 5D / 肢體無力麻木 / 步態極不穩、**中樞型眼震**、HINTS 中樞 | HINTS 三徵皆周邊 + 無神經缺損 |
| [[Vestibular Migraine(前庭性偏頭痛)]] | 偏頭痛病史、反覆、畏光畏聲、可伴 / 不伴頭痛 | 屬臨床診斷，先排中樞血管 |

> [!warning] **AICA / PICA 小腦中風可以「只有眩暈 + 噁心」而神經學看似正常** → 這正是 HINTS 存在的理由。**急性持續性眩暈（AVS）+ 有血管危險因子** 時，HINTS 比發病 24–48h 內的 MRI-DWI 更敏感（早期 DWI 可能假陰性）。

## ❓ 問診 / 身體檢查重點
- **TiTrATE 框架**：**Ti**ming（發作 / 持續 / 反覆）+ **Tr**iggers（頭位變換？自發？）+ **A**ssociated symptoms + **T**argeted **E**xam
  - 陣發 + 頭位誘發、數秒 → BPPV
  - 持續數天 + 自發 → 前庭神經炎 vs 中樞（靠 HINTS 分）
  - 反覆 + 耳症狀 → Meniere；反覆 + 偏頭痛 → 前庭性偏頭痛
- **聽力症狀**：耳鳴 / 聽損 / 耳脹（Meniere / labyrinthitis / 聽神經瘤）— 有助偏周邊，但**不排除中樞**（AICA 中風可傷內耳血流）
- **神經系統回顧**：複視、構音 / 吞嚥、肢體無力麻木、走路
- **關鍵理學**：**眼震型態**（周邊：單向水平旋轉、凝視可抑制、疲勞；中樞：方向會變 / 垂直 / 純旋轉、凝視不抑制）、**Dix-Hallpike**（BPPV）、**HINTS**、指鼻 / 輪替 / 步態（小腦）、Romberg、完整腦神經

## 🩺 初步 workup（該開的檢查 / 影像）
> [!note] 黃金第一步：**床邊 HINTS 檢查**（限「急性持續性眩暈 + 眼震」情境）— 比早期 MRI 更能抓中樞；不是每個頭暈都要衝影像
- **HINTS 三合一**（見下 scoring）
- **Dix-Hallpike**：疑 BPPV（誘發旋轉性眼震 + 短暫眩暈 → 陽性，可接耳石復位）
- **神經學檢查 + 步態**、**血壓（含姿勢性）+ 血糖**（排 presyncope / 代謝）
- **影像**：HINTS 中樞 / 有神經徵 / 高風險 → **MRI（含 DWI）優於 CT**（CT 對後窩敏感度差，但排出血可先用）；心因性快昏 → ECG
- 依情境：聽力檢查（Meniere / 聽神經瘤）、電解質

## ⚡ 值班即時處置（穩定 vs 分流）
```mermaid
flowchart TD
    A["頭暈病人"] --> B{"是真的旋轉性眩暈?"}
    B -->|否 (快昏/站不穩/頭昏)| C["查血壓/血糖/心律/藥物<br/>走 presyncope / 代謝路徑"]
    B -->|是| D{"發作型態?"}
    D -->|頭位誘發 數秒| E["BPPV → Dix-Hallpike<br/>→ 耳石復位術 (Epley)"]
    D -->|持續數天 自發| F["做 HINTS + 神經學"]
    F -->|HINTS 中樞 / 有神經徵| G["當中風處理:<br/>MRI+DWI, 神經科/中風團隊<br/>依院內指引"]
    F -->|HINTS 周邊 + 無神經徵| H["前庭神經炎:<br/>症狀治療(止暈止吐)<br/>± 類固醇, 前庭復健"]
    D -->|反覆 + 耳鳴聽損| I["Meniere → 耳鼻喉<br/>低鹽/利尿; 反覆+偏頭痛→前庭性偏頭痛"]
```
- **BPPV**：耳石復位術（後半規管用 Epley），效果佳
- **前庭神經炎**：症狀治療（止暈止吐）、短期類固醇（依院內指引）、早期前庭復健；**前庭抑制劑不宜長用**（妨礙中樞代償）
- **疑中樞 / 中風**：走中風流程，依院內指引 + 神經科 / 中風團隊
- **Meniere**：耳鼻喉科、低鹽 / 利尿；**前庭性偏頭痛**：偏頭痛式處理

## 📊 臨床評分 / 風險分層（scoring）★本卡核心
> 眩暈的「分數」是床邊神經眼科檢查：**HINTS 分中樞 / 周邊**；疑 TIA 用 **ABCD2** 估短期中風風險。

### ① HINTS 檢查（急性持續性眩暈 · 逐項判「中樞 vs 周邊」）
> 適用：**急性前庭症候群（持續性眩暈 + 自發性眼震 + 噁心，且非陣發頭位型）**。三項中**任一指向中樞就當中樞**。助記中樞危險 = **INFARCT**。
| 項目 | 周邊（良性）表現 | **中樞（危險）表現** |
| --- | --- | --- |
| **HI** — Head Impulse（甩頭測試） | **異常**（甩向患側有 corrective saccade，前庭反射壞掉 = 周邊病灶） | **正常**（前庭反射完好卻在暈 → 反而危險） |
| **N** — Nystagmus（眼震方向） | 單向、水平、凝視可抑制 | **方向會變（凝視換邊變向）/ 垂直 / 純旋轉** |
| **TS** — Test of Skew（交替遮眼看垂直偏移） | 無 skew | **有垂直 skew deviation**（腦幹徵象） |

| 結果組合 | 判讀 | 處置 |
| --- | --- | --- |
| Head Impulse **異常** + 單向眼震 + **無 skew** | 周邊（前庭神經炎型） | 症狀治療 + 前庭復健 |
| Head Impulse **正常** / 變向眼震 / **有 skew**（任一） | **中樞（INFARCT）** | 當中風處理、MRI+DWI、神經科 |

> [!caution] HINTS **只用於「有持續性眼震的急性前庭症候群」**，不能拿去測 BPPV（陣發頭位型用 Dix-Hallpike）或已無眼震的病人。由熟練者執行才可靠；**「Head Impulse 正常」在會暈的人身上反而是壞消息**（違反直覺，別搞反）。加測聽力異常（HINTS+）可再提升敏感度。

### ② ABCD2 Score（疑後循環 TIA 的短期中風風險 · 逐項點數）
| 項目 | 分數 |
| --- | --- |
| **A**ge 年齡 ≥ 60 | 1 |
| **B**lood pressure SBP ≥ 140 或 DBP ≥ 90 | 1 |
| **C**linical 單側無力 | 2 |
| **C**linical 言語障礙但無無力 | 1 |
| **D**uration 症狀 ≥ 60 分 | 2 |
| **D**uration 10–59 分 | 1 |
| **D**iabetes 糖尿病 | 1 |

| 總分 | 2 天中風風險 | 處置 |
| --- | --- | --- |
| **0–3** | 低（~1%） | 門診評估可考慮 |
| **4–5** | 中（~4%） | 積極評估 / 住院考量 |
| **6–7** | 高（~8%） | 住院 + 緊急處置 |

> [!tip] 孤立性眩暈本身**不算** ABCD2 的局灶項，但眩暈若是後循環 TIA 表現，仍要用血管危險因子 + HINTS 一起判；ABCD2 是「已判 TIA 後」估風險的輔助工具，不能拿來排除中樞。

### 自我測驗（選擇題，答案摺疊）
**Q1.** 一位 65 歲男性患者，近幾天出現突然眩暈，伴有耳鳴和聽力下降。您首先應考慮哪種疾病？
- A. 前庭神經炎
- B. Meniere 病
- C. 良性陣發性位置性眩暈 (BPPV)
- D. 中樞中風

> [!success]- 答案
> **B** — 根據症狀，患者有耳鳴和聽力下降，最可能的診斷是 Meniere 病。前庭神經炎通常沒有聽力損失，而 BPPV 主要是頭位型眩暈，中樞中風則會出現其他局灶性症狀。

**Q2.** 一位 70 歲女性患者，近兩天出現單側無力和語言障礙，但無其他症狀。她的 ABCD2 分數是多少？
- A. 1
- B. 2
- C. 3
- D. 4

> [!success]- 答案
> **C** — 根據 ABCD2 計分標準，年齡 ≥ 60 得 1 分，單側無力得 2 分。因此總分為 3 分。

**Q3.** 一位患者出現急性眩暈和自發性眼震，但甩頭測試正常且無垂直偏移。您應如何處理？
- A. 立即轉介中風團隊
- B. 進行前庭神經炎的治療
- C. 設定門診跟蹤
- D. 嘗試耳石復位術

> [!success]- 答案
> **B** — 根據 HINTS 檢查結果，甩頭測試正常且無垂直偏移提示中樞病變可能性低，因此應考慮前庭神經炎。
