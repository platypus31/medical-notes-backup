# Cough（咳嗽）

> [!danger] 🚨 紅旗警訊（must-not-miss，慢性咳嗽別漏這幾個）
> **助記「咳血・減重・喘・燒・異物・心」**
> 1. **咳血 Hemoptysis** → 疑 [[Tuberculosis(結核病)]]、[[Lung Cancer(肺癌)]]、[[Bronchiectasis(支氣管擴張症)]]、[[Pulmonary Embolism(肺栓塞)]]
> 2. **體重減輕 + 慢性咳嗽 + 夜間盜汗** → **TB / 肺癌**，台灣值班永遠別忘記 TB
> 3. **急性咳嗽 + 呼吸窘迫/低血氧** → [[Pneumonia(肺炎)]]、[[Pulmonary Embolism(肺栓塞)]]、氣胸、心衰急性肺水腫
> 4. **發燒 + 咳膿痰** → 肺炎（走 CURB-65）
> 5. **突發咳嗽 + 嗆咳史**（尤其小孩/吞嚥差老人）→ **異物吸入**
> 6. **端坐呼吸/夜間陣咳 + 下肢水腫** → 心因性（心衰）
>
> ⚡ **咳嗽 > 3 週、有紅旗、或 ≥45 歲吸菸者 → 一定要一張胸片**

## 🔀 鑑別診斷 DDx（先按「病程長短」分類）
### 病程分類
- **急性 Acute（< 3 週）**：多為感染
- **亞急性 Subacute（3–8 週）**：常為感染後咳嗽
- **慢性 Chronic（> 8 週）**：三大常見 = 上呼吸道咳嗽症候群 + 氣喘 + 胃食道逆流

| 疾病 | 支持特徵 | rule-out 線索 |
| --- | --- | --- |
| [[Upper Respiratory Tract Infection(上呼吸道感染)]] | 急性、鼻炎/喉痛、自限、TOCC 相關 | 病程 >3 週、有紅旗 |
| [[Pneumonia(肺炎)]] | 發燒、膿痰、呼吸音囉音、CXR 浸潤 | CXR 乾淨、無發燒 |
| [[Upper airway cough syndrome(上呼吸道咳嗽症候群)]]（鼻涕倒流 UACS） | 喉嚨異物感、清喉、[[Allergic rhinitis(過敏性鼻炎)]]/[[Sinusitis(鼻竇炎)]]病史 | 無鼻部症狀、治療無效 |
| [[Asthma(氣喘)]] / 咳嗽變異型氣喘 | 夜間/運動誘發、陣發喘鳴、過敏史、支氣管擴張劑有效 | 肺功能/激發試驗陰性 |
| [[GastroEsophageal Reflux Disease(胃食道逆流)]] | 灼熱感、平躺/飯後加劇、餐後咳、制酸有效 | 無逆流症狀、PPI 無效 |
| Post-URI 感染後咳嗽 | URI 後咳嗽持續、自限、逐漸改善 | 咳嗽持續惡化 |
| 藥物 [[Angiotensin Converting Enzyme Inhibitor(ACEI)]] | 服 ACEI 後乾咳、停藥後改善 | 未使用 ACEI |
| [[Tuberculosis(結核病)]] | 慢性咳 >2 週、體重減輕、夜汗、咳血、接觸史 | 7 分篩檢低分 + 影像陰性 |
| [[Bronchiectasis(支氣管擴張症)]] | 慢性大量膿痰、反覆感染、咳血 | HRCT 無支氣管擴張 |
| [[Chronic Obstructive Pulmonary Disease(慢性阻塞性肺病)]] / 慢性支氣管炎 | 吸菸史、慢性咳痰、呼氣受限 | 無吸菸、肺功能正常 |
| [[Lung Cancer(肺癌)]] | 吸菸、體重減輕、咳血、咳嗽性質改變（僅佔慢咳 ~2% 但不可漏） | 影像陰性、低危險 |

> [!warning] 慢性咳嗽三巨頭（UACS + 氣喘 + GERD）常**多重並存**，一個治療無效不代表排除，可能需疊加處理。

## ❓ 問診 / 身體檢查重點
- **病程長短**（分類 acute/subacute/chronic）＋ 乾咳 vs 有痰、痰的量/顏色/血絲
- **TOCC**：**T**ravel 旅遊、**O**ccupation 職業、**C**ontact 接觸（TB/禽流感）、**C**luster 群聚
- **TB 系統回顧**：咳 >2 週、體重減輕、食慾差、夜汗、胸痛、接觸史
- **誘發/緩解**：夜間/運動（氣喘）、平躺/飯後（GERD）、鼻涕倒流清喉（UACS）
- **用藥**：**ACEI**（乾咳）
- **理學**：呼吸音（囉音/喘鳴/局部呼吸音↓）、鼻咽（鼻涕倒流）、頸淋巴、發燒、杵狀指、下肢水腫（心因）

## 🩺 初步 workup（該開的檢查 / 影像）
> [!note] 黃金第一步：**胸部 X 光**——慢性咳嗽（>3–8 週）、咳血、體重減輕、≥45 歲吸菸者的基本分流檢查，排 TB/肺癌/肺炎。
- **CXR**：浸潤（肺炎）、空洞/上肺病灶（TB）、腫塊（肺癌）
- **痰檢**：疑 TB → **痰塗片抗酸菌 AFB ×3 + 培養 + TB PCR**（NAAT）；膿痰 → 一般細菌培養
- **肺功能 / 支氣管激發試驗**：疑氣喘
- **鼻竇/耳鼻喉評估**：疑 UACS
- **試驗性治療**：慢性咳嗽常用「經驗性治療 UACS/氣喘/GERD」作為診斷
- 進階：胸部 CT / 支氣管鏡（影像可疑腫塊、咳血、反覆感染）

## ⚡ 值班即時處置（分流）
```mermaid
flowchart TD
    A["咳嗽"] --> B{"急性 (<3週)?"}
    B -->|是| C{"發燒/膿痰/呼吸窘迫?"}
    C -->|是| D["CXR → 疑肺炎<br/>CURB-65 分流"]
    C -->|否| E["多為 URI<br/>症狀治療 + TOCC"]
    B -->|否 (>3-8週)| F["查紅旗:咳血/減重/夜汗"]
    F -->|有紅旗| G["CXR + 痰 AFB×3<br/>疑 TB → 呼吸道隔離 + 通報"]
    F -->|無紅旗| H["停 ACEI (若用)<br/>依序試 UACS→氣喘→GERD"]
    H --> I{"治療反應?"}
    I -->|無效| J["胸部 CT / 支氣管鏡<br/>排肺癌/少見病因"]
```
- **急性感染**：肺炎依 **CURB-65** 決定門診/住院，抗生素依院內指引；URI 多支持性治療
- **疑 TB**：**呼吸道隔離（N95/負壓）+ 法定傳染病通報**，痰 AFB × 3 + 培養
- **慢性咳嗽**：先停 ACEI；依序經驗治療 UACS → 氣喘 → GERD（可疊加）
- **異物吸入**：內視鏡/支氣管鏡取出

## 📊 臨床評分 / 風險分層（scoring）★本卡核心

### ① 肺結核 7 分篩檢法（台灣，接觸者/疑似 TB；**≥ 5 分要小心，安排痰檢/胸片**）
| 項目 | 分數 |
| --- | --- |
| 咳嗽（有痰） | **2** |
| 咳嗽超過 2 週 | **2** |
| 體重減輕 | **1** |
| 沒有食慾 | **1** |
| 胸痛 | **1** |

> 加總 **≥ 5 分 → 高度警覺 TB**，安排痰塗片/培養 + 胸片（此為症狀篩檢輔助，非確診工具）。

### ② CURB-65（社區型肺炎嚴重度，各 1 分，總 0–5）
| 字母 | 項目 |
| --- | --- |
| **C** | Confusion 新出現意識混亂 |
| **U** | Urea（BUN）> 19 mg/dL（尿素 >7 mmol/L） |
| **R** | Respiratory rate ≥ 30/min |
| **B** | Blood pressure：收縮 <90 或 舒張 ≤60 mmHg |
| **65** | 年齡 ≥ 65 歲 |

| 總分 | 30 天死亡風險 | 分流建議 |
| --- | --- | --- |
| **0–1** | 低（~1.5%） | 門診治療 |
| **2** | 中（~9%） | 住院或密切觀察 |
| **3–5** | 高（~22%） | 住院，**4–5 考慮 ICU** |

> 院外簡化版 **CRB-65**（去掉 BUN）；≥1 分即建議就醫評估。

### ③ 咳嗽病程分類（決定 workup 深度）
| 病程 | 分類 | 常見病因 |
| --- | --- | --- |
| < 3 週 | 急性 | URI、肺炎、COPD AE |
| 3–8 週 | 亞急性 | 感染後咳嗽 |
| > 8 週 | 慢性 | UACS + 氣喘 + GERD（三巨頭）、TB、ACEI、肺癌 |
