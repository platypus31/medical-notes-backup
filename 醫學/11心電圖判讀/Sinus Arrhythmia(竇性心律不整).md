# Sinus Arrhythmia(竇性心律不整)

> [!abstract] 一句話：竇性節律但 PP 間期隨呼吸週期規律變動（吸氣時心率加快、吐氣時變慢），P 波形態正常、每個 P 後接一個 QRS，屬正常生理變異。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rhythm**：竇性，但 PP 間期不固定、隨呼吸週期規律變化（吸氣快、吐氣慢）
- **P 波**：正常直立（II 導向上），形態一致，每個 P 後接一個 QRS
- **PR**：固定且正常（P 波來源同為 SA node）
- **QRS**：窄、正常
- **Rate**：多在正常範圍，隨呼吸上下擺動

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
- PP 間期呈週期性長短變化，與呼吸同步（吸氣縮短、吐氣延長）
- 最大與最小 PP 間期差異通常 > 0.16 秒
- 常見於年輕人、運動員，屬良性正常變異，通常無需處置

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科
- 呼吸性竇性心律不整（respiratory type）為自主神經（迷走張力隨呼吸波動）之正常生理表現
- 非呼吸性 / 少數與心臟疾病、老化或 digoxin 相關

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 勿誤判為 [[醫學/11心電圖判讀/Sinus Arrest(竇性心律停止)]] 或竇房阻斷：本症每個週期仍有正常 P 波，只是間距隨呼吸變動，並無「漏拍」
- 與心房早期收縮（PAC）鑑別：本症 P 波形態一致、無提前異形 P 波
