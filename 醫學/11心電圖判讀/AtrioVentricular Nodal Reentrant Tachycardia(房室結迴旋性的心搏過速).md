# AtrioVentricular Nodal Reentrant Tachycardia(房室結迴旋性的心搏過速)

> [!abstract] 一句話：最常見的 PSVT，迴旋迴路侷限在房室結內（slow / fast pathway），規則、窄 QRS、常伴反向 P 波緊貼 QRS。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rate**：心室速率約 150–250/分
- **Rhythm**：規則（regular）
- **P 波**：反向 P 波（retrograde），常藏在 QRS 內或緊接其後；Typical（slow-fast）RP interval 短，可表現 pseudo R′ in V1 / pseudo S in II、III、aVF
- **QRS**：窄（迴路在心室之上）
- **ST/T**：可能出現 diffuse ST depression（rate-related）

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
- 可能會出現 diffuse ST depression，分成兩種：
- Typical AVNRT（Slow-Fast type）：走 Slow pathway，Short RP interval（左圖）（較常見）
- Atypical AVNRT（Fast-Slow type）：走 Fast pathway，Long RP interval（右圖）

> > 在 AVNRT 的情況，不管走 slow pathway 還是 fast pathway，都會形成窄的 QRS 波組（因為 loop 都在心室上），而且都會形成「反向 P 波」（由 loop 處往回活化心房造成，因為方向跟正常的心房去極化方向相反，所以 P 波波形會顛倒，故名。） 差別在於，走 slow pathway 者往下活化心室較慢，QRS 波組較慢形成，所以 RP interval（QRS 波組到反向 P 波的間距）較短；相反的，走 fast pathway 者往下活化心室較快，QRS 波組較快形成， 所以 RP interval 較長。

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科

- 屬於 [[醫學/11心電圖判讀/Paroxysmal SupraVentricular Tachycardia(陣發性上心室頻脈)]] 中最常見的一型（小的 loop 在 AV node 附近）。

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 與 [[醫學/11心電圖判讀/AtrioVentricular Reentrant Tachycardia(房室迴旋性心搏過速)]] 鑑別：AVNRT 迴路侷限房室結內、速率略慢；AVRT 經由額外路徑（accessory pathway）在房室間形成大迴路、速率略快。
- 與 [[醫學/11心電圖判讀/Sinus Tachycardia(竇性心搏過速)]] 鑑別：AVNRT 常無明顯正常竇性 P 波（反向 P 波多藏於 QRS）。
