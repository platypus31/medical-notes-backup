# Supraventricular tachycardias(室上性心搏過速)

> [!abstract] 一句話：起源於心房或房室結（希氏束以上）的快速心律總稱，特徵為窄 QRS（除非合併 aberrant conduction 或 pre-excitation）。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rate**：快速（依亞型而定）
- **Rhythm**：規則或不規則（依亞型，如 PSVT 規則、[[醫學/11心電圖判讀/Atrial Fibrillation(心房顫動)]] 不規則）
- **P 波**：型態依起源不同（竇性、異位、反向或 f/F 波）
- **QRS**：窄（narrow QRS），除非 aberrant conduction 或 pre-excitation
- **意義**：起源於希氏束以上（心房 / 房室結）的頻脈總稱

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 寬 QRS 的 SVT（aberrancy 或 antidromic pre-excitation）須與 [[Ventricular Tachycardia(心室頻脈)]] 鑑別。
- 依亞型鑑別：[[醫學/11心電圖判讀/Paroxysmal SupraVentricular Tachycardia(陣發性上心室頻脈)]]、[[醫學/11心電圖判讀/Atrial Fibrillation(心房顫動)]]、[[醫學/11心電圖判讀/Atrial Flutter(心房撲動)]]、[[醫學/11心電圖判讀/Paroxysmal Atrial Tachycardia(陣發性心房頻脈)]]。
