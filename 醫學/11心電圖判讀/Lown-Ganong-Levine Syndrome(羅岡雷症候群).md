# Lown-Ganong-Levine Syndrome(羅岡雷症候群)

> [!abstract] 一句話：一種預激症候群，旁路（James fiber）繞過房室結延遲但插入 His 束，故 PR 縮短、但 QRS 正常且**無 delta 波**（與 WPW 的關鍵區別）。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **PR interval**：縮短（< 0.12 秒）
- **QRS**：正常寬度、形態正常
- **Delta wave**：**無**（旁路繞過 AV node 但仍經正常 His-Purkinje 系統下傳，故無心室預激圖形）
- **臨床**：可伴發陣發性上心室頻脈（PSVT）

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
- 短 PR + 正常 QRS + 無 delta 波三者並存
- 由於缺乏 delta 波，LGL 常被視為描述性/爭議性診斷，需搭配臨床頻脈史

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 與 [[醫學/11心電圖判讀/Wolff-Parkinson-White syndrome(沃爾夫-巴金森-懷特氏症候群)]] 區分：WPW 有 delta 波、QRS 增寬；LGL 無 delta 波、QRS 正常
- 短 PR 也可見於正常變異、交界性節律（junctional rhythm）
