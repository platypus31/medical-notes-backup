# Af with RVR(快速心室反應)

> [!abstract] 一句話：[[醫學/11心電圖判讀/Atrial Fibrillation(心房顫動)]] 合併心室反應速率 >100/分（常 110–180），RR 仍完全不規則、無明顯 P 波，只是房室結放行的心房衝動變多。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rate**：心室速率 >100/分（RVR，常 110–180；急性可更快）
- **Rhythm**：完全不規則（Irregularly irregular）— RR 間距毫無規律
- **P 波**：無明顯 P 波，僅見混亂顫動波 f wave
- **PR**：無法測量（無 P 波）
- **QRS**：窄（除非合併 aberrancy / [[醫學/11心電圖判讀/Bundle Branch Block(束支傳導阻滯)]] / [[醫學/11心電圖判讀/Wolff-Parkinson-White syndrome(沃爾夫-巴金森-懷特氏症候群)]] 預激）
- **ST/T**：快速心室反應時可有需求性 ST 壓低（rate-related）

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科

- Rate control 首選 [[Beta Blocker]] 或 Non-DHP [[Calcium Channel Blocker(CCB)]]（Diltiazem、Verapamil），效果不佳可加 [[Digoxin]]
- **血行動力不穩**：[[Amiodarone]] ± [[Digoxin]]（血壓已差，不用 β-blocker / CCB），必要時同步電擊

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 極快速 RVR 時 RR 差異變小，易誤判為規則 → 仍要找不規則本質與 f wave
- 合併 [[醫學/11心電圖判讀/Wolff-Parkinson-White syndrome(沃爾夫-巴金森-懷特氏症候群)]] 預激時禁用 AV nodal blocker（CCB/β-blocker/Digoxin/Adenosine），可惡化成 [[醫學/11心電圖判讀/Ventricular fibrillation(心室纖維顫動)]]
