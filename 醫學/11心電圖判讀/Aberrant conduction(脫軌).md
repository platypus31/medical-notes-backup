# Aberrant conduction(脫軌)

> [!abstract] 一句話：心室上（supraventricular）衝動因心室內傳導異常而走成束支阻滯型態，造成「寬 QRS」的例外情形——不是真正的心室性心律不整。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **QRS**：寬（> 0.12s），但型態呈典型束支阻滯樣（RBBB / LBBB pattern）
- **Rhythm**：寬且**規則** → SVT with aberrant conduction；寬且**不規則** → Af with aberrant conduction
- **判讀重點**：先確認衝動來源在心室上（有前導 P 波 / 折返性 SVT 病史），QRS 寬是「傳導」問題非「起源」問題

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
> [!PDF|yellow] [[長庚銀蛋口袋書(印).pdf#page=141&selection=119,0,157,27&color=yellow|長庚銀蛋口袋書(印), p.141]]
> > - 寬的 QRS 波大多情況下是「心室性心律不整」所造成，只有一種情形例外： 「心室上心律不整 + 心室傳導異常（脫軌，Aberrant conduction）」
> > - 分類： 
> > 寬、規則 QRS：SVT with aberrant conduction 
> > 寬、不規則 QRS：Af with aberrant conduction

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科
- SVT with aberrancy、[[醫學/11心電圖判讀/Atrial Fibrillation(心房顫動)]] with aberrancy；機制多為前傳時某側束支仍在不反應期（如 Ashman 現象）

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- **最關鍵鑑別**：寬 QRS 心搏過速要先當 [[Ventricular Tachycardia(心室頻脈)]]（VT）處理，除非有明確證據支持 aberrancy——誤判成 SVT with aberrancy 而給錯藥可能致命
- 支持 aberrancy 的線索：典型 RBBB/LBBB 型態、有前導 P 波、既有束支阻滯病史；支持 VT 的線索：房室分離、fusion / capture beats、極度電軸偏移
