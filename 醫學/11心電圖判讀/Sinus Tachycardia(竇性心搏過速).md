# Sinus Tachycardia(竇性心搏過速)

> [!abstract] 一句話：竇性節律但速率 > 100 次/分，P 波正常直立、每個 P 後接一個 QRS、PR 固定，通常是生理或病理刺激的續發反應。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rate**：> 100 次/分（成人多在 100–160/分）
- **Rhythm**：規則的竇性節律
- **P 波**：正常直立（II 導向上），每個 P 後接一個 QRS；速率極快時 P 波可能埋入前一個 T 波
- **PR**：固定且正常
- **QRS**：窄、正常

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
- 規則、窄 QRS、P 波正常且每 P 接一 QRS，唯速率快（> 100/分）
- 通常為漸快漸慢（gradual onset/offset），非突發突止
- 幾乎都是續發性反應，需找背後原因而非只壓心率

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科
- 常見誘因：發燒、疼痛、焦慮、脫水/低血容、貧血、缺氧、甲狀腺亢進、[[Sepsis(敗血症)]]、藥物（擬交感、咖啡因）、運動
- 處置以矯正病因為主（treat the underlying cause）

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 與陣發性室上性心搏過速（PSVT / [[AVNRT(房室結迴旋性心搏過速)]]）鑑別：後者突發突止、常看不到正常 P 波；竇速為漸進且有正常直立 P 波
- 與 [[醫學/11心電圖判讀/Atrial Flutter(心房撲動)]] 2:1 傳導鑑別：flutter 心室率約 150/分且有鋸齒波，勿誤判為竇速
- 速率很快時 P 波藏於 T 波中，易漏判 P 波
