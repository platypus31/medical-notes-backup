# Ventricular Asystole(心電靜止)

> [!abstract] 一句話：心室完全無電氣活動，心電圖呈現一條近乎平坦的直線（flat line），無 QRS、無可電擊心律，屬心跳停止的一種。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rhythm**：無心室節律，呈平直基線（確認至少兩個導程以排除細顫或導線脫落）
- **QRS**：無 QRS 波
- **P 波**：可能完全消失，或偶見孤立 P 波（P-wave asystole）
- **Rate**：無有效心室速率

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
- 心電圖幾乎為一條直線，無可辨識的心室去極化
- 屬於 non-shockable rhythm（不可電擊），處置依 ACLS 流程以 CPR + Epinephrine 為主，並找尋並矯正可逆病因（H's & T's）

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科 / 急診

- 為心跳停止（cardiac arrest）的終末心律之一，需立即 CPR 與 ACLS 急救

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 出現「平直線」時務必排除：導線脫落、增益（gain）過低、或細小的 [[醫學/11心電圖判讀/Ventricular fibrillation(心室纖維顫動)]]（fine VF）誤判為 asystole——需切換導程與調高增益確認
- 與 [[醫學/11心電圖判讀/Pulseless Electrical Activity]] 鑑別：PEA 有電氣活動但無有效脈搏，asystole 則無電氣活動
