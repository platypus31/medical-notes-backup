# Paroxysmal Atrial Tachycardia(陣發性心房頻脈)

> [!abstract] 一句話：規則、速率 100–200/分的心房頻脈，由單一心房異位放電點或心房內迴旋迴路造成，EKG 上與竇性心搏過速難以區分。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rate**：每分鐘 100–200 次
- **Rhythm**：規則（regular）
- **P 波**：有 P 波（形態可能與竇性略異），可由單一異位放電點或心房迴旋迴路產生
- **QRS**：窄
- **鑑別重點**：EKG 上與 [[醫學/11心電圖判讀/Sinus Tachycardia(竇性心搏過速)]] 難以區分

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
> [!PDF|yellow] [[長庚銀蛋口袋書(印).pdf#page=135&selection=8,0,54,2&color=yellow|長庚銀蛋口袋書(印), p.135]]
> > - 規則
> > - 速率：每分鐘 100~200 次
> > - 可由單一心房放電點自發性增加起來，或心房中的迴旋迴路所造成
> > - 嚴格說起來，EKG 與 [[醫學/11心電圖判讀/Sinus Tachycardia(竇性心搏過速)]] 無法區分
> > - 最常見原因：[[Digoxin Intoxication(毛地黃中毒)]]（PAT、[[Premature Ventricular Contractions(室性早搏)]]）
> > - 頸動脈按摩術（Carotid Massage）：無明顯效果，可能稍微減緩速率
> >  ![[長庚銀蛋口袋書(印).pdf#page=135&rect=39,288,522,664&color=yellow|長庚銀蛋口袋書(印), p.135]]

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科

- 最常見原因為 [[Digoxin Intoxication(毛地黃中毒)]]（合併 PAT 與 [[Premature Ventricular Contractions(室性早搏)]]）。

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 與 [[醫學/11心電圖判讀/Sinus Tachycardia(竇性心搏過速)]] 難以區分（EKG 型態相近）。
- 頸動脈按摩術對 PAT 無明顯效果（僅可能稍微減緩速率），可作為鑑別線索。
