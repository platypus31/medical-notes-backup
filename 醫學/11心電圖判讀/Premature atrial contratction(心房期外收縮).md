# Premature atrial contratction(心房期外收縮)

> [!abstract] 一句話：源自心房異位灶的提早搏動，出現形態異常的提早 P 波（P'），其後通常接窄 QRS，並帶不完全代償間歇（noncompensatory pause）。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rhythm**：基本竇性節律中夾雜提早搏動，造成短暫不規則
- **P 波**：提早出現、形態與竇性 P 波不同的 P'（可能藏在前一個 T 波內）
- **PR**：P' 的 PR 可正常或延長
- **QRS**：多為窄 QRS；若異位灶太早、房室結或束支尚在不反應期 → 可脫軌傳導（aberrancy，寬 QRS）或未下傳（blocked PAC）
- **代償間歇**：不完全代償間歇（noncompensatory pause，因異位灶重設竇房結）

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科

- 常為良性，健康者亦可見；可與咖啡因、菸酒、壓力、電解質異常、甲亢等相關
- 頻發或有症狀時再評估誘因，多數不需特別治療

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- Blocked PAC（未下傳的 P'）造成的停頓易誤判為 [[醫學/11心電圖判讀/Sinus Arrest(竇性心律停止)]] 或竇房阻斷 → 找藏在 T 波裡的提早 P'
- PAC 合併脫軌傳導（[[醫學/11心電圖判讀/Aberrant conduction(脫軌)]]）呈寬 QRS，須與心室期外收縮鑑別（前者有提早 P'）
