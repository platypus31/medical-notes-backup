# Q waves

> [!abstract] 一句話：QRS 起始向下的偏折；病理性 Q 波（寬 ≥ 0.04 秒或深度 > 同導程 R 波 25%）常代表過去的心肌梗塞/壞死，但需與正常變異及其他成因鑑別。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **檢視每個導程 QRS 起始**：是否有 Q 波、寬度與深度多少
- **病理性 Q 波判準**：寬度 ≥ 0.04 秒（1 小格），或深度 > 同導程 R 波振幅的 25%
- **定位**：依出現的導程群推斷梗塞區域（如 II/III/aVF → 下壁；V1-V4 → 前壁）
- **合併判讀**：對照 ST-T 變化與臨床病史判斷急性 vs 陳舊

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
- 病理性 Q 波：夠寬（≥ 40 ms）或夠深（> 25% R 波）
- 成組出現於對應解剖區域的導程

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- **正常 septal q 波**：I、aVL、V5、V6 的小而窄 q 波為正常，勿誤判
- 肥厚性心肌病（HCM）可產生假性 Q 波
- [[醫學/11心電圖判讀/Wolff-Parkinson-White syndrome(沃爾夫-巴金森-懷特氏症候群)]] 的 delta 波可 mimic 病理性 Q 波
- 左束支傳導阻滯（[[Left Bundle Branch Block(左束支傳導阻滯)]]）改變 QRS 起始形態
