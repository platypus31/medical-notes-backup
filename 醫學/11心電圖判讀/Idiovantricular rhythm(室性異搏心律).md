# Idiovantricular rhythm(室性異搏心律)

> [!abstract] 一句話：上位節律點（竇房結、房室交界）全數失能時，由心室自行發出的最後防線逸搏心律，速率 20–40/分、QRS 寬、無相關 P 波。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rate**：20–40/分（純 idioventricular）；40–100/分為 **accelerated idioventricular rhythm, AIVR**
- **Rhythm**：規則或略不規則
- **P 波**：無相關 P 波，或呈房室分離（P 與 QRS 各走各的）
- **QRS**：**寬**（> 0.12s，起源於心室，走異常傳導路徑）
- **ST/T**：常呈與 QRS 主波相反方向的次發性改變

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
- 寬 QRS + 慢速率 + 無正常 P 波 → 心室當末端逸搏節律點（escape pacemaker）
- 屬保護性逸搏心律，**不可貿然抑制**（否則可能導致 asystole）
- AIVR 常見於急性心肌梗塞**再灌流**（reperfusion）時，通常良性、短暫、可自行緩解

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科
- 完全性房室傳導阻斷（complete AV block）、竇房結與交界節律點同時失能、急性 MI 再灌流（AIVR）、digoxin 中毒

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 與 [[Ventricular Tachycardia(心室頻脈)]] 鑑別：VT 速率 > 100；idioventricular / AIVR 速率慢，臨床處置完全不同
- 見到寬 QRS 慢心律時勿誤判為需要抑制的心律不整——逸搏是保命機制
