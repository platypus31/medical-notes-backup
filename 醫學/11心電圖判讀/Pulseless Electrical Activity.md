# Pulseless Electrical Activity

> [!abstract] 一句話：心電圖有可辨識的電氣活動（有節律、有 QRS），但病人卻摸不到脈搏、無有效心輸出的心跳停止狀態。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rhythm**：可呈現任何有組織的心律（規則或不規則）
- **QRS**：可有可辨識的 QRS 波組（窄或寬皆可能）
- **關鍵**：心電圖有電氣活動，但臨床上無可觸及的脈搏——診斷需結合臨床（非單看心電圖）
- **Rate**：心電圖有速率，但無有效機械性心輸出

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
- 屬於 non-shockable rhythm（不可電擊），處置依 ACLS 流程以 CPR + Epinephrine 為主
- 治療核心在於積極找尋並矯正可逆病因（H's & T's：Hypovolemia、Hypoxia、Hydrogen ion 酸中毒、Hypo-/Hyperkalemia、Hypothermia；Tension pneumothorax、Tamponade、Toxins、Thrombosis 冠狀動脈/肺）

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科 / 急診

- 為心跳停止（cardiac arrest）的一種表現，需立即 CPR 與 ACLS 急救並矯正可逆病因

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 與 [[醫學/11心電圖判讀/Ventricular Asystole(心電靜止)]] 鑑別：asystole 無電氣活動（平直線），PEA 有電氣活動但無脈搏
- 診斷不可只看螢幕心律——必須確認「有電氣活動 + 無脈搏」，需臨床觸診/監測配合
