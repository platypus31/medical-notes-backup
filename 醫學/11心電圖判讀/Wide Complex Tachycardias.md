# Wide Complex Tachycardias

> [!abstract] 一句話：QRS ≥ 0.12 秒且心率 > 100/分的頻脈；未經篩選族群中約 80% 為心室頻脈（VT），臨床上先當 VT 處理最安全。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rate**：> 100/分
- **QRS**：寬（≥ 0.12 秒）
- **判斷分歧**：VT vs SVT 併異常傳導（aberrancy）vs SVT 經旁路預激
- **支持 VT 的線索**：房室分離（AV dissociation）、fusion/capture beats、極度電軸偏移、V1-V6 concordance、QRS 極寬
- **臨床原則**：血行動力不穩 → 依 ACLS 電擊；不確定時當 VT 處理

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
- 房室分離、融合波（fusion beat）、奪獲波（capture beat）是 VT 的高度特異性指標
- 已知有束支阻斷病史 + 同型 QRS，較支持 SVT with aberrancy

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
