# Multifocal Atrial Tachycardia(多源性心房心搏過速)

> [!abstract] 一句話：≥3 種不同形態 P 波、PR 間期長短不一的不規則心房頻脈（速率常 100–200/分），多見於嚴重肺疾病人，又稱游移性心房節律點（Wandering atrial pacemaker）。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rate**：每分鐘 100–200 次（有時 <100，此時稱 wandering atrial pacemaker）
- **Rhythm**：不規則（心室心律不規則）
- **P 波**：至少 3 種不同形態的 P 波
- **PR**：PR 間期長度也各不相同
- **QRS**：窄
- 頸動脈按摩術無效（有別於部分 SVT）

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
> [!PDF|yellow] [[長庚銀蛋口袋書(印).pdf#page=134&selection=8,0,101,1&color=yellow|長庚銀蛋口袋書(印), p.134]]
> > - 不規則
> > - 速率：每分鐘 100~200 次（有時少於每分鐘 100 次）
> > - 又稱「游移不定的心房結律點（Wandering atrial pacemaker）」
> > - 常出現在嚴重肺疾如 [[Chronic Obstructive Pulmonary Disease(慢性阻塞性肺病)]] 的病人，也與[[Heart Failure(心臟衰竭)]]、[[Hypokalemia(低血鉀)]]、[[Hypomagnesemia(低血鎂)]]、Theophylline、慢性[[Renal Failure(腎衰竭)]]等有關
> > - 頸動脈按摩術（Carotid Massage）：無效
> > - 心電圖特徵：
> > P 波形狀明顯不同（至少三個）
> > PR 區間長度也不同 
> > 心室心律不規則
> > - 治療：先處理其他疾病，維持鉀、鎂正常，有症狀再給予 Rate control（[[Beta Blocker]] 或 Non-DHP [[Calcium Channel Blocker(CCB)]]）
> > - ![[長庚銀蛋口袋書(印).pdf#page=134&rect=52,192,516,565&color=yellow|長庚銀蛋口袋書(印), p.134]]

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科（常合併胸腔內科）

- 病因導向：先處理誘因（嚴重肺疾如 [[Chronic Obstructive Pulmonary Disease(慢性阻塞性肺病)]]、[[Heart Failure(心臟衰竭)]]、電解質異常），維持鉀、鎂正常
- 有症狀再給 Rate control（[[Beta Blocker]] 或 Non-DHP [[Calcium Channel Blocker(CCB)]]）

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 與 [[醫學/11心電圖判讀/Atrial Fibrillation(心房顫動)]] 皆為不規則窄 QRS 頻脈 → MAT 可見清楚且多形態的 P 波，Af 則無明顯 P 波（僅 f wave）
- 與 [[醫學/11心電圖判讀/Atrial Flutter(心房撲動)]] 鑑別：後者為規則鋸齒狀 flutter wave
