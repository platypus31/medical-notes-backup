# Af with SVR(慢速心室反應)

> [!abstract] 一句話：[[醫學/11心電圖判讀/Atrial Fibrillation(心房顫動)]] 合併心室反應速率偏慢（<60/分），RR 仍完全不規則、無明顯 P 波，反映房室結放行變少或傳導受抑制。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rate**：心室速率 <60/分（SVR）
- **Rhythm**：完全不規則（Irregularly irregular）— RR 間距毫無規律
- **P 波**：無明顯 P 波，僅見混亂顫動波 f wave
- **PR**：無法測量（無 P 波）
- **QRS**：窄（除非合併 aberrancy / [[醫學/11心電圖判讀/Bundle Branch Block(束支傳導阻滯)]]）
- 若心室反應極慢且規則 → 警覺完全性 [[醫學/11心電圖判讀/AtrioVentricular Block(房室傳導阻滯疾病)]]（regularized Af）合併交界或心室脫逸節律

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科

- 常見於房室結傳導本身受抑：藥物過量（[[Beta Blocker]]、Non-DHP [[Calcium Channel Blocker(CCB)]]、[[Digoxin]]）、房室結退化性疾病、迷走神經張力高
- 有症狀性心搏過緩 → 檢視並停用抑制房室結的藥物，必要時考慮節律器

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- Af 心室反應變規則且慢 = 「regularized Af」，是 [[Digoxin]] 中毒或完全性房室阻滯的警訊，不是好轉
- 需與 [[醫學/11心電圖判讀/Junctional Bradycardia(房室界性心律之心跳過緩)]] 鑑別（後者規則、無 f wave）
