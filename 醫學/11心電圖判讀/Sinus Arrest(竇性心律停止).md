# Sinus Arrest(竇性心律停止)

> [!abstract] 一句話：竇房結（SA node）暫時停止放電，造成一段沒有 P 波也沒有 QRS 的停頓（pause），且停頓長度不是基本 PP 間期的整數倍。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rhythm**：基本為竇性節律，但突然出現一段停頓（pause），停頓期間 P-QRS-T 全部消失
- **P 波**：停頓期間無 P 波（因為 SA node 未放電）
- **停頓特性**：停頓間距與基本 PP 間期無整數倍關係（用以與 [[醫學/11心電圖判讀/Sinus Exit Block(竇房結出路阻斷)]] 區分）
- **Escape**：停頓過久時可能出現 junctional / ventricular escape beat 保命
- **Rate**：整體平均速率下降，取決於停頓的頻率與長度

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
- 竇性節律中突發一段「P、QRS 皆消失」的長停頓
- 停頓長度 **不是** 基本 PP 間期的整數倍（非整數倍 → arrest；整數倍 → exit block）
- 長停頓可誘發頭暈、暈厥（Adams-Stokes attack）

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科
- 可為 [[Sick Sinus Syndrome(病竇症候群)]] 的表現之一
- 病因與 sinus node 功能失調類似：藥物、迷走神經張力過高、缺血、退化性纖維化

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 與 [[醫學/11心電圖判讀/Sinus Exit Block(竇房結出路阻斷)]] 鑑別：exit block 的停頓為基本 PP 間期的整數倍；arrest 為非整數倍
- 與 blocked PAC（未下傳的心房早期收縮）鑑別：後者停頓前埋有一個提前的 P 波
