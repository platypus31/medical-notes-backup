# Ventricular fibrillation(心室纖維顫動)

> [!abstract] 一句話：心室無序、混亂的顫動，心電圖無可辨識 QRS、呈不規則雜亂波形，無有效心輸出，屬死亡前危象必須立即電擊。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rhythm**：完全無序、混亂的顫動波，無規律
- **QRS**：無可辨識的 QRS 波組
- **P 波**：無
- **Rate**：無有效心室速率（無有效心輸出）
- 屬 shockable rhythm（可電擊心律），為心跳停止之一

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科 / 急診

- 為心跳停止（cardiac arrest）的致命心律，需立即 CPR + 去顫（非同步電擊）
- 常與心肌缺血、[[Ischemic Heart Disease(缺血性心臟病)]]、polymorphic VT 退化有關

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- fine VF（細顫）波形振幅低，易被誤判為 [[醫學/11心電圖判讀/Ventricular Asystole(心電靜止)]]——需切換導程並調高增益確認，兩者處置不同（VF 可電擊，asystole 不可電擊）
- 與 polymorphic [[醫學/11心電圖判讀/Ventricular Tachycardia(心室性心搏過速)]] / [[醫學/11心電圖判讀/Torsades de Pointes(多型性心室心搏過速)]] 鑑別：後者仍有可辨識的寬 QRS 週期，VF 則完全雜亂無 QRS
