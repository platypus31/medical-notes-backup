# Junctional Bradycardia(房室界性心律之心跳過緩)

> [!abstract] 一句話：竇房結失能時由房室交界（AV junction）接手當節律點，發出過慢（< 40/分）的逸搏心律，QRS 窄、P 波倒立或消失。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rate**：< 40/分（junctional escape 正常速率 40–60，低於此為 junctional bradycardia）
- **Rhythm**：規則（regular）
- **P 波**：無明顯 P 波，或在 II / III / aVF 呈**倒立 P 波**（逆行性 retrograde P）；P 可能位於 QRS 前、埋於 QRS 中或緊接 QRS 後
- **PR**：若倒立 P 出現在 QRS 前，PR 常縮短（< 0.12s）
- **QRS**：窄（起源於交界處，走正常傳導路徑）

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
- 竇房結衝動缺失或過慢 → 房室交界接手成為次級節律點（backup pacemaker）
- 逸搏機制：屬保護性心律，不應貿然抑制
- 與 sinus bradycardia 區別關鍵：**P 波型態（倒立/缺失 vs 正常竇性直立 P）**

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科
- 常見於竇房結功能不良（sick sinus syndrome）、迷走神經張力過高、下壁心肌梗塞、digoxin 中毒、β-blocker / CCB 過量

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 與 sinus bradycardia 混淆：後者有正常直立 P 波
- 與完全性房室傳導阻斷（complete AV block）的 junctional escape 鑑別：後者有獨立的 atrial activity（P 波與 QRS 無關）
