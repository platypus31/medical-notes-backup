# Sinus Exit Block(竇房結出路阻斷)

> [!abstract] 一句話：竇房結（SA node）正常放電，但衝動無法傳出至心房，造成漏掉一個 P-QRS 週期，且停頓長度為基本 PP 間期的整數倍。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rhythm**：竇性節律中出現「漏拍」，整段 P-QRS-T 消失
- **停頓特性**：停頓間距 **恰為** 基本 PP 間期的整數倍（2 倍、3 倍…），是與 [[醫學/11心電圖判讀/Sinus Arrest(竇性心律停止)]] 的關鍵區別
- **P 波**：停頓前後的 P 波形態正常、來源仍為 SA node
- **PR / QRS**：正常週期時 PR 固定、QRS 窄
- **Rate**：平均速率因漏拍而下降

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
- 竇性節律中規律或間歇性漏掉整個 P-QRS 週期
- 停頓長度為基本 PP 間期的整數倍（arrest 為非整數倍）
- 屬第二度型態時，停頓前 PP 間期可能呈漸短（Wenckebach 型 type I）或固定（type II）

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科
- 常為 [[Sick Sinus Syndrome(病竇症候群)]] 表現之一
- 病因與竇房結功能失調相似：迷走張力過高、藥物（β-blocker、CCB、digoxin）、缺血、退化性纖維化

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 與 [[醫學/11心電圖判讀/Sinus Arrest(竇性心律停止)]] 鑑別：exit block 停頓為 PP 整數倍，arrest 為非整數倍
- 與 [[醫學/11心電圖判讀/Sinus Arrhythmia(竇性心律不整)]] 鑑別：後者僅間距隨呼吸變動，並無「漏掉整個 P-QRS」
