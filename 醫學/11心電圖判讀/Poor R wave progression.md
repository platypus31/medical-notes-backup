# Poor R wave progression

> [!abstract] 一句話：胸前導程 V1→V6 的 R 波高度未依正常方式漸增（常定義為 V3 的 R 波 < 3 mm，或 R 波振幅未在 V3-V4 越過 S 波），是一個非特異性但需鑑別的 pattern。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **觀察胸前導程**：正常時 R 波由 V1 逐漸增高、S 波逐漸變淺，R/S 轉換點約在 V3-V4
- **PRWP 判準**：V3 的 R 波 < 3 mm，或 R 波遞增延遲/消失
- **合併判讀**：注意有無病理性 Q 波、ST-T 變化以協助鑑別

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
- R/S 轉換延後（transition zone 右移到 V5-V6）
- V1-V3 R 波振幅偏低、遞增不足

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 前壁陳舊性心肌梗塞（old anterior MI）
- 左心室肥厚（LVH）、左束支傳導阻滯（[[Left Bundle Branch Block(左束支傳導阻滯)]]）
- 導程位置放置錯誤（technical / lead misplacement）
- 心肌病變、慢性阻塞性肺病/肺氣腫（COPD、肺過度充氣使心臟轉位）
