# Junctional Tachycardia(房室交界處頻脈)

> [!abstract] 一句話：房室交界處自律性異常增高而搶當節律點，速率 > 100/分、QRS 窄、P 波倒立或消失。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rate**：> 100/分（60–100 為 accelerated junctional rhythm；> 100 為 junctional tachycardia）
- **Rhythm**：規則（regular）
- **P 波**：無明顯 P 波，或於 II / III / aVF 呈倒立 P（逆行）；可能出現房室分離（AV dissociation）
- **QRS**：窄（除非合併 aberrancy 或既有 BBB）

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
- 起源於房室交界的規則窄 QRS 心搏過速 + P 波異常（倒立/缺失）
- 自律性（automaticity）增強機制，非折返；故不易被電擊或迷走操作終止

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科
- 最典型病因：**digoxin 中毒**；其他包括心肌缺血 / 下壁 MI、心肌炎、心臟術後、theophylline / 兒茶酚胺過量

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 與 AVNRT / 其他 SVT 鑑別：JT 為自律性增強、常見房室分離；SVT 多為折返
- digoxin 中毒背景下出現規則窄 QRS 頻脈要高度警覺
