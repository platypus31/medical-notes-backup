# Junctional Premature Beat(房室交界處期外收縮)

> [!abstract] 一句話：房室交界處提早發出一個異位衝動，QRS 窄、無正常竇性 P 波（或倒立 P），提早出現後常伴非完全代償間歇。

## 🎯 判讀步驟（rate → rhythm → axis → P → PR → QRS → ST/T → QT）
- **Rhythm**：基本竇性節律中夾雜提早出現的異位搏動
- **P 波**：提早搏動**無正常直立 P 波**，或於 II / III / aVF 呈倒立 P（逆行）；P 可在 QRS 前、埋於 QRS 或緊接其後
- **PR**：若倒立 P 在 QRS 前，PR < 0.12s
- **QRS**：窄（與竇性搏動型態相同，除非合併 aberrancy）
- **代償間歇**：多為非完全代償間歇（noncompensatory pause）

## 🔑 關鍵特徵（要辨別的 pattern / 診斷準則）
- 提早（premature）+ QRS 窄 + P 波異常（倒立或缺失）→ 起源於房室交界
- 與 PAC（房性期外收縮）差別：PAC 有異常但**直立**的 P' 波在 QRS 前；JPB 為倒立或無 P

## 🧩 對應疾病（這個 pattern 指向什麼 → 連科別）
> 心臟內科
- 多為良性；可見於正常人、咖啡因 / 菸酒、電解質異常、心肌缺血、digoxin 中毒

## ⚠️ 陷阱 / 鑑別（易混淆 pattern）
- 與 [[Premature Atrial Contraction(心房期外收縮)]]（PAC）鑑別：看 P 波型態與方向
- 合併心室內傳導異常時 QRS 變寬，須與 [[Premature Ventricular Contraction(心室期外收縮)]]（PVC）區分
