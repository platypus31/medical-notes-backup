# Fluconazole（氟康唑 / Diflucan）

> [!abstract] 一句話：**三唑類抗黴菌**，抑制真菌 CYP450（14-α-demethylase）阻斷 ergosterol 合成 → 破壞細胞膜。**口服吸收極佳（生體可用率 >90%）、可穿血腦障壁 BBB、主要經腎排泄**。是念珠菌感染主力，但**是強 CYP 抑制劑、交互作用眾多、會延長 QT、有肝毒性**。對 **Candida krusei 天然抗藥、C. glabrata 劑量依賴性抗藥**、對 Aspergillus 無效。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **強 CYP2C9 / CYP3A4 抑制劑**：會拉高 **warfarin（出血）、phenytoin、statin（橫紋肌溶解）、sulfonylurea（低血糖）、tacrolimus/cyclosporine** 濃度 → 開藥前查病人藥單
> 2. **QT 延長**：與其他延長 QT 藥（ondansetron、amiodarone、macrolide、haloperidol）併用有 Torsades 風險 → 校正電解質、查基礎 EKG
> 3. **對哪些念珠菌無效要記**：**C. krusei 天然抗藥**、**C. glabrata 劑量依賴抗藥/常抗藥**；對 **Aspergillus 完全無效**（要用 voriconazole/echinocandin）
> 4. **腎功能不良需減量**：主要經腎排泄，CrCl ≤ 50 維持劑量減半；血液透析後補一劑
> 5. **肝毒性 + 孕期致畸**：監測肝功能；高劑量長期使用於孕早期有致畸風險（避免，尤其孕早期）

## 💊 適應症（治療哪些疾病）
- **念珠菌血症 / 侵襲性念珠菌感染 [[Candidemia(念珠菌血症)]]**（穩定、非 neutropenic、非 azole 抗藥菌株的降階或首選之一；重症/未知菌種先用 echinocandin）
- **口咽 / 食道念珠菌感染 [[Oropharyngeal Candidiasis(口咽念珠菌感染)]]**（鵝口瘡、食道炎）
- **外陰陰道念珠菌感染 [[Vulvovaginal Candidiasis(外陰陰道念珠菌感染)]]**（單劑口服）
- **隱球菌腦膜炎 [[Cryptococcal Meningitis(隱球菌腦膜炎)]]** 的**鞏固期與維持期**（急性期用 amphotericin B + flucytosine，之後 fluconazole 接手）
- **念珠菌泌尿道感染、腹膜炎；免疫低下病人的預防性用藥**

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 念珠菌血症 / 侵襲性念珠菌 | **負荷 800 mg（12 mg/kg）**，之後 **400 mg（6 mg/kg）QD** |
| 食道念珠菌感染 | 負荷 200–400 mg，之後 200–400 mg QD ×14–21 天 |
| 口咽念珠菌感染 | 200 mg 第 1 天，之後 100–200 mg QD |
| 外陰陰道念珠菌（單純） | **150 mg PO 單劑** |
| 隱球菌腦膜炎鞏固期 | 400–800 mg QD；維持期 200 mg QD |
- **負荷劑量觀念**：半衰期長（~30 小時），第一天給雙倍負荷劑量才能快速達治療濃度
- **腎功能調整**：CrCl > 50 不需調整；**CrCl ≤ 50 維持劑量減半**（負荷劑量照給）；血液透析病人每次透析後給一次完整劑量
- **口服 = 靜脈**：生體可用率 >90%，能口服就口服（IV 與 PO 劑量相同）
- **肝**：肝毒性者謹慎，一般不需固定減量但需監測

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：噁心、腹痛、腹瀉、頭痛、皮疹、暫時性肝酵素上升
- **嚴重**：
  - **肝毒性**（罕見急性肝衰竭，長療程/高劑量風險增）
  - **QT 延長 → Torsades de pointes**（尤其併用其他延長 QT 藥或低血鉀/鎂）
  - **嚴重皮膚反應**（SJS/TEN，免疫低下者風險較高）
  - 交互作用導致的間接毒性（warfarin 出血、statin 橫紋肌溶解、sulfonylurea 低血糖）
  - 高劑量脫髮（可逆）

## 🚫 禁忌症
- 對 fluconazole 或其他 azole 類過敏
- **併用會顯著延長 QT 的藥物**（如 pimozide、quinidine，部分情況 amiodarone）
- **併用 cisapride、terfenadine**（QT 風險）
- 孕期（尤其孕早期高劑量）相對禁忌 — 致畸風險

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：肝功能（LFT）、腎功能、電解質（K/Mg，尤其併 QT 藥）、EKG（高風險者）、對應併用藥（INR、phenytoin 濃度、血糖）
- **交互作用（重點，強 CYP2C9/3A4 抑制）**：
  - **Warfarin** → INR 上升出血 → 密切監測 INR
  - **Phenytoin** → 濃度上升毒性
  - **Statin（simvastatin/atorvastatin）** → 橫紋肌溶解風險
  - **Sulfonylurea（glipizide 等）** → 低血糖
  - **Tacrolimus / Cyclosporine** → 免疫抑制劑濃度上升、腎毒性
  - **延長 QT 藥物**（ondansetron、amiodarone、macrolide、抗精神病藥）→ Torsades
- **孕哺**：孕期避免（尤其孕早期高劑量致畸）；單劑 150 mg 治陰道炎風險較低但仍建議孕婦優先用局部藥；哺乳一般可用

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名長期服用 warfarin 的病人因念珠菌感染開始 fluconazole，最需要監測的是？
- A. 血糖
- B. INR（出血風險）
- C. 甲狀腺功能
- D. 血氨

> [!success]- 答案
> **B** — Fluconazole 抑制 CYP2C9，會顯著增強 warfarin 的抗凝作用，INR 上升導致出血，須密切監測 INR 並調整 warfarin 劑量。

**Q2.** 關於 fluconazole 的抗菌譜，下列何者正確？
- A. 對 Aspergillus 效果良好
- B. 對 Candida krusei 天然抗藥、對 Aspergillus 無效
- C. 對所有念珠菌都有效
- D. 無法穿越血腦障壁

> [!success]- 答案
> **B** — Fluconazole 對 C. krusei 天然抗藥、C. glabrata 常抗藥、對 Aspergillus 完全無效（需 voriconazole）；它能穿 BBB，用於隱球菌腦膜炎維持期。

**Q3.** 治療念珠菌血症時，fluconazole 為什麼第一天要給 800 mg 負荷劑量？
- A. 因為半衰期短需頻繁補充
- B. 半衰期長（~30h），負荷劑量可快速達到治療濃度
- C. 為了降低腎毒性
- D. 為了減少交互作用

> [!success]- 答案
> **B** — Fluconazole 半衰期約 30 小時，若只給維持劑量需數天才達穩態，故第一天給雙倍負荷劑量（800 mg）快速達治療濃度，之後 400 mg QD 維持。
