# Echinocandins（棘白菌素類總覽）

> [!abstract] 一句話：棘白菌素（Caspofungin / Micafungin / Anidulafungin）抑制真菌細胞壁的 **β-1,3-glucan 合成酶**，破壞細胞壁完整性 → 對念珠菌**殺菌**、對麴菌**抑菌**。**侵襲性念珠菌感染（含念珠菌血症）的第一線經驗用藥**，最大優點是**安全性高、藥物交互作用少、不需依腎功能調整**；最大限制是**幾乎不涵蓋 Cryptococcus（隱球菌）與部分黴菌**，且僅靜脈劑型。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **抗菌譜有洞**：**不涵蓋 Cryptococcus neoformans（隱球菌）**、不涵蓋 Fusarium、Mucorales（毛黴菌 / 接合菌）、Trichosporon → 這些感染別用棘白菌素
> 2. **對念珠菌血症是首選**，但**中樞神經、眼內炎、泌尿道念珠菌感染滲透差**（尿中濃度低、CNS/眼房水穿透差）→ 這些部位改 azole / amphotericin
> 3. **Caspofungin 需依肝功能調整**（中度肝損減量），Micafungin / Anidulafungin 一般不需依肝腎調整（**Anidulafungin 非酶降解、腎肝損皆不需調整**）
> 4. **Caspofungin 與 rifampin / cyclosporine 有交互作用**（rifampin 降 caspofungin 需增量；cyclosporine 使 caspofungin AUC↑ 且肝酵素可能上升）— 相較 azole 已少很多但非零
> 5. **輸注相關反應**：組織胺媒介的臉潮紅、皮疹、少數過敏；輸注速度過快尤明顯

## 💊 適應症（治療哪些疾病）
- **侵襲性念珠菌病 / 念珠菌血症 [[Candidemia(念珠菌血症)]]**（血流動力不穩、近期用過 azole、疑 azole 抗藥株如 C. glabrata/krusei 時首選）
- **食道念珠菌感染**（含 azole 抗藥或難治者）
- **侵襲性麴菌病的救援 / 合併治療**（Aspergillus，非首選單藥，voriconazole 才是首選）
- **中性球低下發燒的經驗性抗黴菌治療**（empirical antifungal therapy）

## 📏 劑量 / 用法（成人，僅靜脈）
| 藥 | 負荷 / 維持 |
| --- | --- |
| **Caspofungin** | 負荷 **70 mg IV** 第一天 → **50 mg IV QD**；中度肝損維持減至 35 mg |
| **Micafungin** | **100 mg IV QD**（念珠菌血症）；食道念珠菌 150 mg |
| **Anidulafungin** | 負荷 **200 mg IV** 第一天 → **100 mg IV QD** |
- **腎功能**：三者**皆不需依腎功能調整、不被血液透析清除**
- **肝功能**：僅 **Caspofungin** 中度肝損需減量；重度肝損資料不足
- 僅有靜脈劑型（無口服），輸注勿過快以降低組織胺反應

## ⚠️ 副作用（常見 / 嚴重）
- **整體安全性高、耐受良好**（相對 amphotericin B 與 azole）
- **常見**：發燒、輸注部位靜脈炎、頭痛、噁心、輸注相關組織胺反應（潮紅/皮疹）
- **肝**：肝酵素上升（尤 caspofungin + cyclosporine）
- **少見**：低血鉀、過敏反應、血球變化
- 無 azole 的 QT / CYP 大量交互作用，無 amphotericin 的嚴重腎毒性

## 🚫 禁忌症
- 對棘白菌素類過敏
- 針對**隱球菌腦膜炎、Fusarium、Mucorales** 等其涵蓋不到的病原體，不應作為治療藥（等同無效）
- 泌尿道念珠菌、念珠菌眼內炎、CNS 念珠菌感染因滲透差不建議單用

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：肝功能（尤 caspofungin）、感染反應與培養、輸注反應
- **交互作用**（明顯少於 azole）：
  - **Caspofungin + rifampin**（誘導）→ 需增量至 70 mg QD
  - **Caspofungin + cyclosporine** → caspofungin AUC↑、肝酵素可能上升
  - Micafungin 對經 CYP3A4 代謝藥物（如 sirolimus、nifedipine）有輕度影響
  - **Anidulafungin 幾乎無藥物交互作用**（非酶性化學降解）
- **孕哺**：資料有限，孕期僅在效益大於風險時使用

## 🔗 相關
- 疾病：[[Candidemia(念珠菌血症)]]、[[Invasive Aspergillosis(侵襲性麴菌病)]]
- 同領域 / 對照：[[Azoles(唑類抗黴菌總覽)]]（涵蓋隱球菌、口服可用、CYP 交互作用多）、[[Amphotericin B(兩性黴素B總覽)]]（廣譜但腎毒性大）、[[Fluconazole(氟康唑)]]
- 機轉概念：真菌細胞壁 β-1,3-glucan vs 細胞膜 ergosterol（azole/amphotericin 打細胞膜，棘白菌素打細胞壁）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名血流動力不穩的加護病房病人血培養長出念珠菌，近期曾用過 fluconazole。經驗性抗黴菌最合適的首選為何？
- A. 續用高劑量 fluconazole
- B. 棘白菌素（如 micafungin / caspofungin / anidulafungin）
- C. Nystatin 口服
- D. 口服 terbinafine

> [!success]- 答案
> **B** — 血流不穩、曾用 azole、可能為 azole 抗藥株（C. glabrata/krusei）的念珠菌血症，指引首選棘白菌素靜脈治療，殺菌且安全性高。

**Q2.** 下列何種感染「不應」使用棘白菌素作為治療？
- A. 念珠菌血症
- B. 食道念珠菌感染
- C. 隱球菌腦膜炎
- D. 侵襲性念珠菌病

> [!success]- 答案
> **C** — 棘白菌素不涵蓋 Cryptococcus，隱球菌腦膜炎需 amphotericin B + flucytosine 誘導、fluconazole 維持，用棘白菌素等同無效。

**Q3.** 關於棘白菌素的藥物動力學/安全性，何者正確？
- A. 需依腎功能大幅調整劑量
- B. 有強烈 CYP3A4 抑制與 QT 延長（同 azole）
- C. 整體安全性高、交互作用少，多不需依腎調整，caspofungin 需依肝功能調整
- D. 有嚴重腎毒性（同 amphotericin B）

> [!success]- 答案
> **C** — 棘白菌素安全性佳、交互作用遠少於 azole、無 amphotericin 的腎毒性，多不需依腎功能調整；僅 caspofungin 於中度肝損需減量。
