# Terbinafine + Griseofulvin（皮癬藥大類卡）

> [!abstract] 一句話：兩者都主打 **皮癬菌（dermatophyte，tinea/香港腳/灰指甲）**。**Terbinafine** 抑制 **squalene epoxidase** 阻斷麥角固醇合成（角鯊烯堆積殺菌），是現今灰指甲首選；**Griseofulvin** 干擾真菌**微管/有絲分裂**、沉積於角質，是老藥（誘導 CYP450、致畸胎、療程長）。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **Terbinafine → 肝毒性**：用藥前及療程中監測 **LFT**；有肝病者慎用/避免
> 2. **Terbinafine → 味覺喪失/障礙**（dysgeusia）可持續數週，是常考且會影響服藥順從性
> 3. **Griseofulvin → 致畸胎（孕婦禁）**：女性用藥期間及停藥後一段時間避孕（依院規/需查證）；男性也建議停藥後一段時間再受孕
> 4. **Griseofulvin → 誘導 CYP450**：**降低 Warfarin 抗凝效果、降低口服避孕藥效**（可能避孕失敗）
> 5. **Griseofulvin → disulfiram-like 反應**：與酒精併用可致潮紅、心悸、噁心
> 6. **Griseofulvin 需與高脂飲食併服**才能有效吸收；療程長（灰指甲需數月）

## 💊 適應症（治療哪些疾病）
- **皮癬菌感染（dermatophytosis）**：[[Tinea(皮癬菌感染)]]、足癬（香港腳）、體癬、股癬、頭癬（tinea capitis）
- **甲癬 / 灰指甲（onychomycosis）**：**Terbinafine 為口服首選**（療效與療程優於 Griseofulvin）
- **頭癬（tinea capitis）**：**Griseofulvin 仍是兒童頭癬的傳統選擇**（部分菌種），Terbinafine 亦有效
- 註：兩者對 **念珠菌 / 其他酵母菌** 效果有限，主要針對皮癬菌

## 📏 劑量 / 用法（成人）
| 藥 / 用途 | 劑量 |
| --- | --- |
| Terbinafine — 灰指甲（趾甲） | **250 mg PO QD ×12 週**（手指甲約 6 週）；依院規/需查證 |
| Terbinafine — 體癬/足癬 | 250 mg PO QD ×2–6 週或外用；依院規/需查證 |
| Griseofulvin — 皮癬 | microsize 500 mg/day（依劑型與體重調整），**與高脂餐併服**；療程數週至數月 |
| Griseofulvin — 灰指甲 | 療程可達數月（趾甲更久），現多被 Terbinafine 取代 |
- Griseofulvin 兒童頭癬劑量依體重（mg/kg）計算，依院規/需查證
- **Terbinafine 主要經肝代謝**；嚴重肝/腎功能不全避免或調整

## ⚠️ 副作用（常見 / 嚴重）
- **Terbinafine**
  - 常見：腸胃不適、頭痛、皮疹、**味覺喪失/障礙**（dysgeusia）
  - 嚴重：**肝毒性**（少數肝衰竭）、嚴重皮膚反應（SJS/TEN，罕見）、味覺長期喪失
- **Griseofulvin**
  - 常見：頭痛、噁心、光敏感、蕁麻疹
  - 嚴重：**致畸胎**、肝毒性、少見骨髓抑制；可誘發/加重紅斑性狼瘡

## 🚫 禁忌症
- **Terbinafine**：活動性/慢性肝病、對藥物過敏
- **Griseofulvin**：**孕婦（致畸胎）**、嚴重肝病、紫質症（porphyria）、對藥物過敏

## 🔍 監測 / 交互作用 / 孕哺
- **Terbinafine**
  - 監測：**LFT**（基線 + 療程中）
  - 交互作用：抑制 **CYP2D6** → 可能升高某些 β-blocker、TCA、SSRI 濃度
- **Griseofulvin**
  - **誘導 CYP450** → **↓ Warfarin 效果**（需追 INR、可能要加 Warfarin 劑量）、**↓ 口服避孕藥效**（建議加屏障避孕）
  - **酒精 → disulfiram-like 反應**
  - **高脂飲食 → ↑ 吸收**（務必衛教併餐服用）
- **孕哺**：Griseofulvin **孕婦禁**（致畸胎）；Terbinafine 孕哺資料有限，非必要延後治療（灰指甲非急症，可待產後）

## 🔗 相關
- 疾病：[[Tinea(皮癬菌感染)]]、灰指甲 / 甲癬（onychomycosis）
- 同類/對照：[[Azoles(唑類抗黴菌總覽)]]（Itraconazole 也可治灰指甲，走 CYP 抑制路線）、[[Nystatin(制黴菌素)]]（對象是念珠菌，非皮癬菌）
- 機轉家族：Terbinafine（squalene epoxidase）與 Azoles（14-α-demethylase）都打**麥角固醇合成**但作用點不同

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位 45 歲男性趾甲灰指甲，肝功能正常，欲口服治療。最合適的首選與必要監測是？
- A. Griseofulvin，監測 INR
- B. Terbinafine 250mg QD，療程中監測 LFT
- C. Nystatin 口服
- D. Fluconazole 單劑

> [!success]- 答案
> **B** — Terbinafine 是灰指甲口服首選（趾甲約 12 週），須基線與療程中監測肝功能；味覺喪失也要衛教。

**Q2.** 服用 Warfarin 的病人若加用 Griseofulvin，INR 會如何變化、為什麼？
- A. 上升，因抑制 CYP450
- B. 下降，因誘導 CYP450 加速 Warfarin 代謝
- C. 不變
- D. 上升，因蛋白結合競爭

> [!success]- 答案
> **B** — Griseofulvin 誘導 CYP450，加速 Warfarin 代謝使抗凝效果下降、INR 降低；需追蹤 INR 並可能調高 Warfarin 劑量。

**Q3.** 關於 Griseofulvin 的衛教，下列何者最正確？
- A. 空腹服用吸收最好
- B. 孕婦可安全使用
- C. 需與高脂餐併服、孕婦禁用、避免飲酒
- D. 可與口服避孕藥安心併用

> [!success]- 答案
> **C** — Griseofulvin 需高脂餐助吸收、致畸胎故孕婦禁用、與酒精有 disulfiram-like 反應，且會降低口服避孕藥效需加屏障避孕。
