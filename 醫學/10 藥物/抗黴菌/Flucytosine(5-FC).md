# Flucytosine（5-FC / 氟胞嘧啶）

> [!abstract] 一句話：真菌把 5-FC 用 **cytosine deaminase** 轉成 **5-FU**，再干擾 DNA/RNA 合成；人類細胞缺乏此酵素故有選擇性。臨床幾乎**只與 Amphotericin B 併用**治療隱球菌腦膜炎（協同），單用易速生抗藥性。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **絕不單獨使用** → 單用會快速產生抗藥性；經典組合是 **Amphotericin B + 5-FC** 治 [[Cryptococcal Meningitis(隱球菌腦膜炎)]] 誘導期
> 2. **骨髓抑制**（白血球/血小板低下）是最重要毒性 → 與血中濃度過高相關，**必須監測藥物血中濃度**（peak 建議控制，避免 > 100 µg/mL；依院規/需查證）
> 3. **腎功能不全需嚴格調降劑量** → 5-FC 主要經腎排除；併用 Amphotericin B 本身傷腎會使 5-FC 蓄積 → 惡性循環，需頻繁追蹤 Cr 與血中濃度
> 4. **肝毒性 / 肝酵素上升** → 用藥期間監測 LFT
> 5. 腸道菌叢可將 5-FC 代謝成 5-FU → 部分骨髓/腸胃毒性來源

## 💊 適應症（治療哪些疾病）
- **主戰場**：[[Cryptococcal Meningitis(隱球菌腦膜炎)]] 誘導期（與 Amphotericin B 併用，協同殺菌、加速 CSF 無菌化）
- 嚴重 [[Candidiasis(念珠菌感染)]]（特定情況，如念珠菌腦膜炎/心內膜炎，作為合併治療）
- 部分嚴重全身性黴菌感染的輔助合併用藥

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 一般成人（正常腎功能） | **100 mg/kg/day PO 分 4 次**（Q6H） |
| 隱球菌腦膜炎（合併 AmB） | 同上，療程依臨床反應（誘導期常 ≥2 週）；依院規/需查證 |
| 腎功能不全 | **依 CrCl 嚴格調降**（延長間隔或減量），並以血中濃度導引；依院規/需查證 |
- **血中濃度監測**：peak 目標區間避免過高（降低骨髓毒性），trough 需足夠（維持療效）；實際數值依院規/需查證
- 台灣供應以口服劑型為主（靜脈劑型不一定可得，依院規/需查證）

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：噁心、嘔吐、腹瀉、肝酵素輕度上升
- **嚴重**：
  - **骨髓抑制**：白血球低下、血小板低下、貧血（劑量/血中濃度依賴，可致命性感染或出血）
  - **肝毒性**：肝酵素上升、少數肝壞死
  - **腸炎 / 嚴重腹瀉**
  - 濃度過高時神經精神症狀

## 🚫 禁忌症
- 對 flucytosine 過敏
- **嚴重骨髓抑制 / 血球嚴重低下**者慎用或禁用
- 嚴重腎功能不全未能監測血中濃度時應極度謹慎
- 孕婦（動物具致畸胎性，屬高風險）——非必要不用

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：CBC（骨髓抑制）、LFT（肝毒性）、腎功能（Cr/CrCl）、**藥物血中濃度**（最關鍵，導引劑量）
- **交互作用**：
  - **Amphotericin B**：治療上「協同」，但 AmB 的腎毒性會使 5-FC 排除下降、蓄積 → 毒性增加，需更密切監測
  - 其他骨髓抑制藥（如化療、Ganciclovir）→ 骨髓毒性加乘
- **孕哺**：孕婦具致畸胎風險，避免使用；哺乳資料有限，一般不建議

## 🔗 相關
- 疾病：[[Cryptococcal Meningitis(隱球菌腦膜炎)]]、[[Candidiasis(念珠菌感染)]]
- 同類/併用：[[Amphotericin B(兩性黴素B總覽)]]（經典合併夥伴）、[[Azoles(唑類抗黴菌總覽)]]（維持期常接手 Fluconazole）
- 對照：[[Fluconazole(氟康唑)]]（隱球菌腦膜炎鞏固/維持期主力）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位 HIV 病人診斷隱球菌腦膜炎，主治開立 Amphotericin B。加上 Flucytosine 的主要理由是？
- A. 減少 Amphotericin B 的腎毒性
- B. 兩者協同、加速 CSF 無菌化並降低復發
- C. Flucytosine 可單獨治癒故加保險
- D. 預防細菌重複感染

> [!success]- 答案
> **B** — AmB + 5-FC 是隱球菌腦膜炎誘導期的標準組合，協同殺菌、加速 CSF 無菌化；5-FC 不可單用（易抗藥）。

**Q2.** 使用 Flucytosine 的病人最需要密切監測、以避免致命毒性的項目是？
- A. 血糖
- B. 全血球計數（骨髓抑制）與藥物血中濃度
- C. 甲狀腺功能
- D. 血中鈣離子

> [!success]- 答案
> **B** — 5-FC 的骨髓抑制與血中濃度過高相關，須監測 CBC 並以血中濃度導引劑量，腎不全時尤須調降。

**Q3.** 關於 Flucytosine 的選擇性毒性，下列何者正確？
- A. 人類細胞富含 cytosine deaminase 故毒性大
- B. 真菌具 cytosine deaminase 把 5-FC 轉成 5-FU，人類細胞缺乏此酵素
- C. 5-FC 直接毒殺所有分裂細胞無選擇性
- D. 選擇性來自真菌細胞壁厚

> [!success]- 答案
> **B** — 真菌內 cytosine deaminase 將 5-FC 轉為 5-FU，人類細胞缺乏此酵素，構成選擇性基礎；但腸道菌與蓄積仍可造成骨髓/腸胃毒性。
