# Nystatin（制黴菌素）

> [!abstract] 一句話：Nystatin 是**多烯類（polyene）抗黴菌藥**，與 amphotericin B 同機轉——結合真菌細胞膜 **ergosterol** 形成孔洞使細胞內容物外漏 → 對念珠菌殺菌。關鍵特性是**幾乎不被腸胃道吸收，僅作局部治療**（口腔鵝口瘡、皮膚黏膜念珠菌、陰道念珠菌），因此**全身毒性極低**，但也**無法治療侵襲性 / 全身性黴菌感染**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **僅供局部使用、口服不吸收**：口服懸液是用來「含漱作用於口咽黏膜」而非全身治療；**念珠菌血症、食道以下 / 侵襲性感染無效**，需改 fluconazole / 棘白菌素 / amphotericin B（靜脈）
> 2. **鵝口瘡的正確用法是「含漱後吞下」**（swish and swallow）：讓藥液充分接觸口腔黏膜停留數分鐘再吞，療效才夠；病人衛教不到位是失敗常見原因
> 3. **食道念珠菌 ≠ 口腔念珠菌**：吞嚥痛 / 胸骨後痛提示食道侵犯（免疫低下、HIV），nystatin 局部**不夠**，應全身性 fluconazole
> 4. **味道苦 / 順從性差**：兒童、老人、鼻胃管病人易吐或漏用；假牙者要一併浸泡消毒避免再感染
> 5. **反覆口腔念珠菌要找原因**：吸入型類固醇未漱口、免疫低下、糖尿病、抗生素、乾口症、HIV — 別只給 nystatin 而漏掉背後病因

## 💊 適應症（治療哪些疾病）
- **口腔念珠菌感染 / 鵝口瘡（oral thrush）[[Oral Candidiasis(口腔念珠菌感染)]]**（嬰兒、免疫低下、吸入型類固醇使用者、假牙者）
- **皮膚 / 皮膚皺褶念珠菌感染**（尿布疹合併念珠菌、間擦疹 intertrigo）
- **陰道念珠菌感染**（陰道錠 / 乳膏，非首選但可用）
- **腸道念珠菌（口服錠劑局部作用於腸腔）**、免疫低下者口咽念珠菌預防（部分情境）

## 📏 劑量 / 用法（成人 / 常見情境）
| 情境 | 用法 |
| --- | --- |
| 口腔鵝口瘡（成人） | **口服懸液 4–6 mL（400,000–600,000 U）含漱後吞下，每日 4 次**，症狀消失後續用 48 小時 |
| 嬰兒鵝口瘡 | 1–2 mL 塗抹於口腔黏膜，每日 4 次 |
| 皮膚念珠菌 | 乳膏 / 粉劑局部塗患處，每日 2–3 次 |
| 陰道念珠菌 | 陰道錠置入，每日 1 次連續數日 |
- **重點衛教**：含漱盡量停留、餐後使用、假牙浸泡、吸入型類固醇後漱口
- **無靜脈劑型**（多烯類全身治療用 amphotericin B，不是 nystatin）

## ⚠️ 副作用（常見 / 嚴重）
- **整體副作用極少**（因幾乎不吸收）
- **常見**：口感苦 / 噁心、口腔刺激、局部皮膚刺激
- **高劑量口服**：偶有噁心、嘔吐、腹瀉
- **少見**：局部過敏 / 接觸性皮膚炎
- **無全身性腎毒性 / 輸注反應**（與靜脈 amphotericin B 截然不同）

## 🚫 禁忌症
- 對 nystatin / 多烯類過敏
- **不可用於全身性 / 侵襲性黴菌感染**（無效，屬臨床誤用）
- 食道以下侵犯之念珠菌需全身治療，非 nystatin 適應症

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：臨床反應（口腔白斑消退、症狀改善）；反覆發作查免疫 / 血糖 / HIV / 用藥背景
- **交互作用**：因幾乎不吸收，**臨床顯著全身交互作用極少**
- **孕哺**：局部使用全身吸收極微，孕哺期一般視為安全（陰道 / 口腔局部）

## 🔗 相關
- 疾病：[[Oral Candidiasis(口腔念珠菌感染)]]、[[Esophageal Candidiasis(食道念珠菌感染)]]（需全身治療對照）
- 同機轉 / 對照：[[Amphotericin B(兩性黴素B總覽)]]（同為多烯類但靜脈全身用、廣譜、腎毒性大）、[[Fluconazole(氟康唑)]]（口腔/食道念珠菌全身性首選）、[[Azoles(唑類抗黴菌總覽)]]
- 機轉概念：多烯類結合 ergosterol 打細胞膜（vs 棘白菌素打細胞壁）

Nystatin 屬哪一類抗黴菌藥、機轉為何？::多烯類（polyene），結合真菌細胞膜 ergosterol 形成孔洞使內容物外漏，對念珠菌殺菌

Nystatin 與 amphotericin B 機轉相同，為何一個只能局部用？::Nystatin 幾乎不被腸胃道吸收，全身毒性太適合局部；全身治療用靜脈 amphotericin B

Nystatin 的主要適應症有哪些？::口腔鵝口瘡、皮膚黏膜念珠菌、陰道念珠菌等局部念珠菌感染

治療鵝口瘡時 nystatin 懸液的正確用法是什麼？::含漱使藥液充分接觸口腔黏膜停留數分鐘後再吞下（swish and swallow），每日4次

為什麼 nystatin 不能治療念珠菌血症或侵襲性感染？::不吸收無法達到全身治療濃度，只作用於接觸到的黏膜表面

食道念珠菌感染為何不能只用 nystatin？::食道侵犯屬全身性治療範疇，需 fluconazole 等全身性藥物，局部 nystatin 不足

Nystatin 的副作用為什麼很少？::幾乎不吸收，故無全身性腎毒性或輸注反應，主要僅口感苦與局部刺激

使用吸入型類固醇者反覆口腔念珠菌該如何預防？::每次吸藥後漱口，並檢視免疫、血糖等背景因素

假牙病人治療口腔念珠菌要額外做什麼？::假牙一併浸泡消毒，避免真菌殘留再感染

Nystatin 有靜脈劑型嗎？::沒有；需要全身多烯類治療時使用 amphotericin B

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名使用吸入型類固醇的氣喘病人出現口腔白色斑塊、可刮除。最適當的初始治療與衛教為何？
- A. 靜脈 amphotericin B
- B. Nystatin 懸液含漱後吞下，並衛教吸藥後漱口
- C. 口服 terbinafine
- D. 不需治療

> [!success]- 答案
> **B** — 典型口腔念珠菌（鵝口瘡），局部 nystatin 含漱吞下即可，同時衛教吸入型類固醇後漱口以防復發。輕症不需全身抗黴菌。

**Q2.** 一名 HIV 病人（CD4 低）主訴吞嚥疼痛、胸骨後灼痛，疑食道念珠菌。最適當治療？
- A. 只用 nystatin 含漱
- B. 全身性 fluconazole
- C. 局部皮膚乳膏
- D. 不需抗黴菌

> [!success]- 答案
> **B** — 吞嚥痛提示食道侵犯，屬全身性感染，需 fluconazole（口服/靜脈）等全身治療；nystatin 局部不吸收無法治療食道病灶。

**Q3.** 關於 nystatin 的敘述何者正確？
- A. 可靜脈注射治療全身黴菌血症
- B. 與 amphotericin B 機轉不同
- C. 幾乎不吸收、僅局部用、全身副作用少
- D. 是唑類（azole）抗黴菌藥

> [!success]- 答案
> **C** — Nystatin 與 amphotericin B 同為多烯類（結合 ergosterol），但幾乎不吸收，僅供局部治療，全身副作用極少，無靜脈劑型。
