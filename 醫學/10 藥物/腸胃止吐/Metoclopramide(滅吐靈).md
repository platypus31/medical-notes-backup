# Metoclopramide（滅吐靈 / Primperan）

> [!abstract] 一句話：**D2 受體拮抗劑**，中樞阻斷 CTZ 的 D2 止吐 + 周邊促進上消化道蠕動（加速胃排空、增加下食道括約肌張力）→ **止吐兼促動力（prokinetic）**。用於胃輕癱、噁心嘔吐與腸胃動力不足。因會穿越 BBB 阻斷紋狀體 D2，**EPS / 急性肌張力不全（尤其年輕人、大劑量）與長期遲發性運動障礙**是核心風險，故有**療程限制（一般 ≤ 12 週）**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **急性肌張力不全 / EPS**：**年輕人、女性、大劑量、快速 IV** 風險最高 → 眼球上吊、斜頸、喉痙攣；處置同 haloperidol：**benztropine 或 diphenhydramine IM/IV**
> 2. **遲發性運動障礙（tardive dyskinesia）**：長期使用可能不可逆 → **FDA 黑框警告，療程一般 ≤ 12 週**，老人尤慎
> 3. **腸阻塞 / 穿孔 / 出血 → 禁用**：促蠕動會加重機械性梗阻或穿孔；先排除 ileus/obstruction 再給
> 4. **嗜鉻細胞瘤（pheochromocytoma）→ 禁用**：可能誘發高血壓危象
> 5. **與 levodopa / dopamine agonist 相互拮抗**：D2 阻斷惡化帕金森；帕金森病人避免。亦可致 NMS、QT 延長、高泌乳素血症

## 💊 適應症（治療哪些疾病）
- **糖尿病性胃輕癱 [[Gastroparesis(胃輕癱)]]**（促進胃排空的首選 prokinetic）
- **噁心嘔吐 [[Nausea and Vomiting(噁心嘔吐)]]**（含化療輔助、術後、偏頭痛相關）
- **胃食道逆流輔助、腸胃動力不足**
- **偏頭痛急性期輔助（加速止痛藥吸收 + 止吐）**、放置小腸營養管前促進通過

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 噁心嘔吐 / 胃輕癱 | **10 mg PO/IV/IM，餐前及睡前 TID-QID** |
| IV 給藥 | 10 mg **緩慢 IV（至少 1–2 分鐘）** 以降低急性肌張力不全/躁動 |
| 療程限制 | **一般 ≤ 12 週**（遲發性運動障礙黑框警告） |
| 老人 / 衰弱 | 減量（如 5 mg）並縮短療程 |
| 每日上限 | 一般 ≤ 0.5 mg/kg/day（依院規/仿單） |
- **腎功能**：**CrCl < 40 mL/min 減量約 50%**（藥物與代謝物腎排泄，蓄積增 EPS）
- **肝功能**：肝損者謹慎
- **給藥途徑**：快速 IV 易致躁動與肌張力不全，務必緩推

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：嗜睡、疲倦、躁動不安、腹瀉、**靜坐不能（akathisia）**、高泌乳素血症（溢乳/月經失調）
- **嚴重**：
  - **急性肌張力不全 / EPS**（年輕人、大劑量、快速 IV）
  - **遲發性運動障礙 tardive dyskinesia**（長期，可能不可逆，黑框）
  - **抗精神病藥惡性症候群 NMS**
  - QT 延長、罕見高鐵血紅素血症（新生兒）
  - 憂鬱、罕見自殺意念

## 🚫 禁忌症
- 對 metoclopramide 過敏
- **腸胃道機械性阻塞、穿孔、出血**（促蠕動會加重）
- **嗜鉻細胞瘤 pheochromocytoma**（高血壓危象）
- **癲癇 / 抽搐病史**（降低閾值、增加發作頻率）
- **既往抗精神病藥/metoclopramide 引起的遲發性運動障礙**
- 帕金森病（相對禁忌）；併用其他 EPS 高風險藥物須警覺

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：EPS / 遲發性運動障礙（療程與劑量控制）、NMS 徵象、腎功能（減量依據）、憂鬱/情緒變化
- **交互作用**：
  - **levodopa / dopamine agonist**：藥效互相拮抗（D2 阻斷 vs 促進），帕金森惡化
  - **抗精神病藥（haloperidol/phenothiazine）**：EPS/NMS 風險疊加
  - **CNS 抑制劑（BZD/opioid/酒精）**：加重鎮靜
  - **改變胃排空**：影響其他口服藥吸收（如加速 acetaminophen 吸收、延緩 digoxin）
  - **MAOI**：謹慎；延長 QT 藥物累加
- **孕哺**：孕期相對常用且安全性資料較多（噁心嘔吐可考慮）；哺乳會分泌（可增泌乳但新生兒 EPS 顧慮），謹慎

## 🔗 相關
- 疾病：[[Gastroparesis(胃輕癱)]]、[[Nausea and Vomiting(噁心嘔吐)]]
- 同類 / 對照：[[Ondansetron(昂丹司瓊)]]（5-HT3 拮抗，不促動力、不 EPS）、[[Domperidone(多潘立酮)]]（周邊 D2 拮抗、較少穿 BBB 故 EPS 少但 QT 風險）、[[Haloperidol(氟哌啶醇)]]（同 D2 拮抗，同樣 EPS/NMS）
- 解毒 / 搭配：[[Benztropine(苯托品)]]、[[Diphenhydramine(苯海拉明)]]（治急性肌張力不全）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名 22 歲女性因噁心快速 IV push metoclopramide 後出現斜頸與眼球上吊。最恰當處置為？
- A. 再給一劑 metoclopramide
- B. 給 diphenhydramine 或 benztropine
- C. 給 ondansetron
- D. 觀察即可

> [!success]- 答案
> **B** — 這是急性肌張力不全（EPS），年輕女性、快速 IV 為高風險；應立即給抗膽鹼藥 diphenhydramine 或 benztropine，並避免再給 D2 拮抗劑。

**Q2.** 下列何者為 metoclopramide 的禁忌症？
- A. 糖尿病胃輕癱
- B. 化療噁心嘔吐
- C. 腸道機械性阻塞
- D. 術後噁心

> [!success]- 答案
> **C** — Metoclopramide 促進腸胃蠕動，機械性阻塞/穿孔/出血時會加重病情，屬禁忌；嗜鉻細胞瘤與癲癇病史亦為禁忌。

**Q3.** 關於 metoclopramide 的長期使用，何者正確？
- A. 可長期使用無限制
- B. 有遲發性運動障礙黑框警告，療程一般 ≤ 12 週
- C. 長期使用只會便秘
- D. 帕金森病人可安全長期使用

> [!success]- 答案
> **B** — Metoclopramide 有 FDA 黑框警告，長期使用可能導致不可逆遲發性運動障礙，療程一般限 ≤ 12 週；帕金森病人因 D2 阻斷惡化症狀應避免。
