# Other Antiemetics（其他止吐藥總覽）

> [!abstract] 一句話：**除了 metoclopramide 與 ondansetron，值班還常用一群「靠阻斷 D2 / H1 / 蕈毒鹼受體」的止吐藥**。吩噻嗪類（**prochlorperazine、promethazine**）阻斷中樞 D2；抗組織胺（**dimenhydrinate**）與抗膽鹼（**scopolamine**）主治**前庭性 / 動暈症噁心**；**domperidone** 是周邊 D2 阻斷（**不易過血腦障壁 → 錐體外副作用少**）。選藥看**噁心的機轉來源**（化療 / 前庭 / 胃輕癱 / 術後）。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **錐體外症狀（EPS）與 QT 延長**：吩噻嗪類（prochlorperazine、promethazine）與 metoclopramide 一樣會 D2 阻斷 → **急性肌張力不全、靜坐不能、遲發性運動障礙、NMS**，且**延長 QT**；年輕人、高劑量、快速靜脈給尤易。
> 2. **抗膽鹼副作用**：promethazine、dimenhydrinate、scopolamine → 口乾、視力模糊、尿滯留、便秘、**老人譫妄 / 認知惡化**；青光眼、前列腺肥大、腸阻塞慎用。
> 3. **Promethazine IV 外滲 / 動脈內注射 → 嚴重組織壞死（FDA 黑框）**：建議深部肌注或大量稀釋慢推、大靜脈給藥，避免小周邊靜脈。
> 4. **Domperidone 不過血腦障壁 → EPS 少，但周邊仍可 QT 延長 / 心律不整**（尤其高劑量、靜脈劑型已多國下市）；且會升泌乳素。
> 5. **動暈 / 前庭性噁心要「事前給」**：scopolamine 貼片、dimenhydrinate 應在**暈車 / 搭船前**預防性使用，症狀出現後效果差。

## 💊 各藥速覽

| 藥物 | 類別 / 機轉 | 最適場景 | 關鍵副作用 |
| --- | --- | --- | --- |
| **Prochlorperazine** | 吩噻嗪、中樞 D2 阻斷 | 一般噁心嘔吐、偏頭痛相關噁心 | EPS、QT、鎮靜、低血壓 |
| **Promethazine** | 吩噻嗪 + 強 H1/抗膽鹼 | 噁心、前庭性、鎮靜需求 | 強鎮靜、抗膽鹼、**IV 外滲組織壞死（黑框）** |
| **Domperidone** | 周邊 D2 阻斷（**不過 BBB**） | 胃輕癱、腸胃蠕動不足噁心 | EPS 少；QT / 心律不整、升泌乳素 |
| **Dimenhydrinate** | 抗組織胺 H1（+ 抗膽鹼） | **動暈症 / 前庭性眩暈噁心** | 嗜睡、抗膽鹼、老人譫妄 |
| **Scopolamine（東莨菪鹼）** | 抗蕈毒鹼（M 受體） | **動暈症預防（貼片）**、末期分泌物 | 抗膽鹼（口乾/視糊/尿滯留/譫妄） |

## 📏 劑量 / 用法（成人 · 依院規）
- **Prochlorperazine**：常見 5–10 mg PO/IM/IV（IV 慢推），或塞劑；⚠依院規。
- **Promethazine**：12.5–25 mg PO/IM/IV（**IV 需大量稀釋、慢推、大靜脈**；優先深部肌注）。
- **Domperidone**：10 mg PO 飯前，一天數次；**避免高劑量 / 長療程**（QT）。
- **Dimenhydrinate**：50 mg PO/IM/IV Q4–6H prn；動暈**出發前 30–60 分**先給。
- **Scopolamine 貼片**：1.5 mg 貼片貼耳後，動暈**出發前數小時**貼，效期約 72 h。
- ⚠️所有精確劑量、頻次、上限依院規與病人（尤其老人、腎肝、QT）調整。

## ⚠️ 副作用
- **D2 阻斷類（prochlorperazine、promethazine、domperidone）**：EPS（急性肌張力不全、靜坐不能、遲發運動障礙）、NMS、QT 延長、高泌乳素血症。
- **抗膽鹼 / 抗組織胺類（promethazine、dimenhydrinate、scopolamine）**：嗜睡、口乾、視力模糊、尿滯留、便秘、心搏過速、**老人譫妄 / 跌倒**。
- **Promethazine**：靜脈外滲 / 動脈內注射 → 嚴重組織壞死 / 壞疽（黑框）。
- 姿勢性低血壓（吩噻嗪類）。

## 🚫 禁忌症 / 慎用
- **QT 延長 / 電解質異常（低鉀低鎂）/ 併用其他延長 QT 藥**：D2 阻斷類慎用。
- **抗膽鹼類禁忌 / 慎用**：閉角性青光眼、前列腺肥大 / 尿滯留、腸阻塞、重症肌無力、老人譫妄風險。
- **Domperidone**：嚴重肝損、已知 QT 延長 / 心律不整、催乳素瘤。
- **兒童（尤其 < 2 歲）與 promethazine**：致命呼吸抑制（黑框，禁用於 2 歲以下）。
- 帕金森氏症：D2 阻斷類惡化症狀（此時 domperidone 因不過 BBB 相對可選）。

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：EPS / 錐體外徵象、QT（EKG，尤其併其他 QT 藥）、鎮靜程度、老人認知 / 跌倒、抗膽鹼徵象（尿滯留）。
- **交互作用**：與其他 QT 延長藥（ondansetron、macrolide、fluoroquinolone、抗精神病藥）相加；與其他抗膽鹼 / 中樞抑制劑（酒精、鴉片、benzodiazepine）相加鎮靜；CYP3A4 抑制劑升 domperidone 濃度增 QT 風險。
- **孕哺**：孕吐一般首選 doxylamine-pyridoxine、ondansetron/metoclopramide 視情況；吩噻嗪類部分可用需權衡；scopolamine、domperidone 孕哺資料有限。

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名 25 歲女性打完 prochlorperazine 後出現頸部歪扭、眼球上吊。最可能是？
- A. 過敏反應
- B. 急性肌張力不全（D2 阻斷所致錐體外症狀）
- C. 癲癇發作
- D. 低血糖

> [!success]- 答案
> **B** — 吩噻嗪類與 metoclopramide 一樣阻斷 D2，年輕人易出現急性肌張力不全（頸/眼肌痙攣），可用抗膽鹼藥（如 benztropine / diphenhydramine）緩解。

**Q2.** 準備搭船的乘客想預防暈船噁心，最恰當的用藥策略是？
- A. 出現噁心後再給 ondansetron
- B. 出發前先用 scopolamine 貼片或 dimenhydrinate 預防
- C. 用 domperidone 因為它不過 BBB
- D. 只要不吃東西就好

> [!success]- 答案
> **B** — 動暈症屬前庭性噁心，最有效的是抗膽鹼（scopolamine 貼片）或抗組織胺（dimenhydrinate），且須「事前」預防；症狀出現後效果差。5-HT3（ondansetron）對動暈症效果不佳。
