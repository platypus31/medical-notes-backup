# Aprepitant（NK1 拮抗劑）

> [!abstract] 一句話：**Neurokinin-1（NK1）受體拮抗劑**，阻斷 **substance P** 在中樞嘔吐中樞的作用 → 對**延遲性化療引起噁心嘔吐（delayed CINV）特別有效**；與 5-HT3 拮抗劑（如 [[Ondansetron(昂丹司瓊)]]）機轉互補，常「三合一」併用（NK1 拮抗劑 + 5-HT3 拮抗劑 + dexamethasone）。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **是「預防」用藥、不是「解救（rescue）」用藥** —— 用於高致吐化療前預防急性 + 延遲性噁心嘔吐，不是嘔吐當下才給。
> 2. **CYP3A4 交互作用是核心考點**：aprepitant 是 CYP3A4 的**中度抑制劑**（也有誘導作用）→ **提高 dexamethasone 血中濃度**（合併時 dexamethasone 常需**減量**）、影響經 CYP3A4 代謝的化療藥。
> 3. **降低 warfarin 效果** → 併用須**監測 INR**；**降低口服避孕藥效果** → 需採取額外 / 替代避孕。
> 4. 單用力道有限，須**與 5-HT3 拮抗劑 + 類固醇組合**才發揮完整止吐效果（尤其高致吐方案）。

## 💊 適應症（治療哪些疾病）
- **高度致吐性化療（HEC，如 cisplatin）的急性 + 延遲性噁心嘔吐預防** —— 尤其涵蓋 5-HT3 拮抗劑較弱的**延遲期（>24 小時）**。
- **中度致吐性化療（MEC）** 的止吐預防（部分方案）。
- **術後噁心嘔吐（PONV）預防**。
- 常與 [[Ondansetron(昂丹司瓊)]]（或其他 5-HT3 拮抗劑）+ dexamethasone **三合一** 併用。

## 📏 劑量 / 用法（成人）
| 用途 | 用法重點 |
| --- | --- |
| 化療止吐（口服 aprepitant） | 化療當天起始、續給數天涵蓋延遲期（典型 3 天療程，**確切 mg 依院規 / 需查證**） |
| 化療止吐（IV fosaprepitant） | 化療前單次 IV（fosaprepitant 為前驅藥，體內轉為 aprepitant） |
| 術後噁心嘔吐（PONV） | 麻醉誘導前口服單次（**依院規 / 需查證**） |
| 合併 dexamethasone | 因 CYP3A4 抑制，**dexamethasone 常需減量**（**依院規 / 需查證**） |
- 標準止吐組合：**NK1 拮抗劑 + 5-HT3 拮抗劑 + dexamethasone**（三合一）。
- 各方案確切劑量、天數與 dexamethasone 減量幅度 **依院規 / 需查證**。

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：疲倦、無力、腹瀉、打嗝、食慾不振、頭痛。
- **少見 / 嚴重**：罕見**過敏 / 過敏性反應**（含輸注反應，fosaprepitant IV）、肝功能指標上升。
- 多數屬輕度、可耐受。

## 🚫 禁忌症
- 對 aprepitant / fosaprepitant 過敏。
- **與 pimozide 併用禁忌**（CYP3A4 抑制使 pimozide 濃度上升 → 嚴重心律不整風險）；與其他高度依賴 CYP3A4 且治療指數窄的藥物併用須謹慎。
- 嚴重肝功能不全者慎用（**依院規 / 需查證**）。

## 🔍 監測 / 交互作用 / 孕哺
- **CYP3A4 雙重角色（核心）**：
  - **中度抑制** → 升高 **dexamethasone**、經 CYP3A4 代謝的化療藥（如某些長春花生物鹼 vinca alkaloids）濃度 → 需調整劑量 / 監測毒性。
  - 也具**誘導**作用 → **降低 warfarin 效果**（併用**監測 INR**，尤其療程後 1–2 週）、**降低口服避孕藥效果**（需額外避孕）。
  - 本身也受強效 CYP3A4 抑制 / 誘導劑影響。
- **監測**：INR（併 warfarin）、肝功能、止吐療效。
- **孕哺**：孕期使用資料有限，須權衡（**依院規 / 需查證**）。

## 🔗 相關
- 對照 / 併用：[[Ondansetron(昂丹司瓊)]]（5-HT3 拮抗劑，機轉互補，對急性期強；NK1 對延遲期強，兩者常併用）
- 併用第三方：dexamethasone（三合一止吐；因 CYP3A4 抑制常需減量）
- 同機轉：fosaprepitant（IV 前驅藥）、netupitant、rolapitant（其他 NK1 拮抗劑）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位接受 cisplatin（高度致吐化療）的病人，為預防噁心嘔吐，下列止吐組合最完整恰當的是？
- A. 只給 ondansetron
- B. NK1 拮抗劑（aprepitant）+ 5-HT3 拮抗劑 + dexamethasone
- C. 只給 dexamethasone
- D. 只給 metoclopramide

> [!success]- 答案
> **B** — 高度致吐化療的標準止吐是**三合一**：**aprepitant（NK1）+ 5-HT3 拮抗劑 + dexamethasone**，可同時涵蓋急性與延遲期。單用任一藥對 HEC 皆不足。

**Q2.** 關於 aprepitant 的藥物交互作用，下列何者正確？
- A. 對 CYP3A4 沒有影響
- B. 是 CYP3A4 中度抑制劑，會升高 dexamethasone 濃度、需其減量
- C. 會升高 warfarin 的抗凝效果
- D. 增強口服避孕藥效果

> [!success]- 答案
> **B** — Aprepitant 是 **CYP3A4 中度抑制劑**（兼具誘導），會**升高 dexamethasone**（合併需減量）；同時**降低** warfarin 效果（需監測 INR）與口服避孕藥效果（需額外避孕）。

**Q3.** Aprepitant 相較於 ondansetron，臨床上最突出的優勢是？
- A. 對急性期嘔吐最強、可單用
- B. 對延遲性 CINV 特別有效，與 5-HT3 拮抗劑機轉互補
- C. 可作為嘔吐發作當下的解救用藥
- D. 完全沒有藥物交互作用

> [!success]- 答案
> **B** — Aprepitant（NK1 拮抗劑）對**延遲性 CINV（>24 小時）** 特別有效，與 ondansetron（5-HT3，偏急性期）**機轉互補**，故常併用而非取代。
