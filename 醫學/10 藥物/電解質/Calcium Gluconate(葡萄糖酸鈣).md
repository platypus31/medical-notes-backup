# Calcium Gluconate（葡萄糖酸鈣）

> [!abstract] 一句話：**高血鉀急救的「心臟保護第一步」**——鈣穩定心肌細胞膜、對抗鉀對心肌的毒性，**但不降血鉀**（後續仍需 insulin/glucose、β-agonist、排鉀）。也用於症狀性低血鈣、MgSO4 中毒解毒、CCB/β-blocker 中毒。**外滲會組織壞死；digoxin 病人慎用（stone heart）**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **高血鉀給鈣是「穩膜」不是「降鉀」**：ECG 有高血鉀變化（寬 QRS、高尖 T）時先給 calcium gluconate 保護心臟，但**血鉀還在高點**，一定要接著給 insulin+glucose / β-agonist 移鉀 + 利尿/樹脂/透析排鉀，否則鈣效力消退（~30–60 min）後又危險
> 2. **Calcium Gluconate vs Calcium Chloride 元素鈣差 3 倍**：10% CaCl₂ 10mL ≈ 272 mg 元素鈣；10% Ca gluconate 10mL ≈ 90 mg 元素鈣（約 1/3）。**CaCl₂ 刺激性強、外滲壞死重，一般走中央靜脈/急救；周邊靜脈優先 gluconate**
> 3. **外滲（extravasation）→ 嚴重組織壞死/鈣化**，尤其 CaCl₂；確認 IV line 通暢、勿高濃度周邊快推
> 4. **服用 digoxin 的病人給鈣要極謹慎**（快速高鈣 + 洋地黃 → 「stone heart」心肌強直/致命心律不整），非緊急勿快速 IV push
> 5. **不可與含碳酸氫鈉/磷酸鹽同一條 line**（形成碳酸鈣/磷酸鈣沉澱）；與 [[Ceftriaxone(頭孢曲松)]] 在新生兒禁併（鈣鹽沉澱致死）
> 6. **快速 IV push 會心搏過緩、低血壓、心律不整**，非心停情境要 slow

## 💊 適應症（治療哪些疾病）
- **高血鉀症的心臟膜穩定 [[Hyperkalemia(高血鉀症)]]**（ECG 有變化時第一步，不降鉀）
- **症狀性低血鈣症 [[Hypocalcemia(低血鈣症)]]**（手足搐搦、Chvostek/Trousseau 徵、抽搐、QT 延長）
- **硫酸鎂（MgSO4）中毒解毒**（[[Magnesium Sulfate(硫酸鎂)]]）
- **鈣通道阻斷劑（CCB）/ β-blocker 中毒/過量**、氫氟酸（HF）灼傷
- 產科/輸大量含枸櫞酸血品後之低血鈣

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 / 用法 |
| --- | --- |
| 高血鉀（心臟保護） | **10% Calcium Gluconate 10 mL（1 g）IV 於 2–3 分鐘**，ECG 未改善可 5–10 分後重複；效力約 30–60 分鐘（依院規/仿單） |
| 症狀性低血鈣 | 1–2 g（10–20 mL 10%）IV 於 10–20 分鐘，接續輸注矯正（依血鈣/白蛋白校正） |
| MgSO4 中毒解毒 | **1 g（10 mL 10%）IV slow** |
| CCB/β-blocker 中毒 | 高劑量鈣，依毒物科/院規 |
| 心停高血鉀（ACLS） | 可用 CaCl₂ 10% 10 mL IV（急救走中央/大靜脈） |
- **腎功能不良**：高血鉀常見於腎衰竭，鈣用於穩膜可用，但需監測鈣、避免高鈣血症；根本仍需透析排鉀
- **肝功能**：一般不需依肝調整
- **元素鈣**：10% Ca gluconate 10mL ≈ 90 mg 元素鈣（約 4.6 mmol）；CaCl₂ 為其 ~3 倍

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：注射部位溫熱/刺激感、口中金屬味、輕度低血壓、快推時臉潮紅
- **嚴重**：
  - **外滲致組織壞死 / 鈣化**（CaCl₂ 尤重）
  - **快速 IV push → 心搏過緩、心律不整、心跳停止**
  - **digoxin 病人快速高鈣 → stone heart（心肌強直）致命心律不整**
  - 高鈣血症（過量）

## 🚫 禁忌症
- **高血鈣症（hypercalcemia）**
- **服用 digoxin 且非緊急者**（相對禁忌，需極謹慎緩給）
- 與含碳酸氫鈉/磷酸鹽同一 IV line（沉澱）
- 新生兒與含鈣製劑 + ceftriaxone 併用（鈣鹽沉澱禁忌）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：ECG（心律、給鈣後高血鉀 ECG 是否改善）、血鈣（校正白蛋白/離子鈣）、IV 部位有無外滲、血壓、心率
- **交互作用**：
  - **Digoxin**：鈣增強洋地黃毒性（stone heart）
  - **Ceftriaxone**：新生兒禁併，成人不同時併（鈣鹽沉澱）
  - **碳酸氫鈉/磷酸鹽**：同 line 沉澱
  - Thiazide 利尿劑減少尿鈣排除、鈣通道阻斷劑作用被拮抗
- **孕哺**：孕期哺乳期矯正低血鈣可用（生理需要）

## 🔗 相關
- 疾病：[[Hyperkalemia(高血鉀症)]]、[[Hypocalcemia(低血鈣症)]]
- 同類 / 對照：**Calcium Chloride（CaCl₂）**（元素鈣約 3 倍、刺激性強走中央靜脈）、[[Magnesium Sulfate(硫酸鎂)]]（其中毒由本藥解毒）、[[Potassium Chloride(氯化鉀KCl)]]
- 對照概念：高血鉀完整處置 = 鈣穩膜 → insulin/glucose + β-agonist 移鉀 → 利尿/樹脂/透析排鉀

### 自我測驗（選擇題，答案摺疊）

**Q1.** ESRD 病人 K 7.0，ECG 見寬 QRS。第一步最緊急處置？
- A. 立即血液透析
- B. Calcium gluconate 1g IV 穩定心肌膜
- C. 只給 insulin+glucose
- D. 口服 kayexalate 即可

> [!success]- 答案
> **B** — ECG 有變化的高血鉀，第一步先給 calcium gluconate 穩膜保護心臟；但它不降鉀，接著要 insulin/glucose+β-agonist 移鉀、並安排透析排鉀。

**Q2.** 關於 calcium gluconate 與 calcium chloride，何者正確？
- A. 兩者元素鈣含量相同
- B. Gluconate 元素鈣較多
- C. CaCl₂ 元素鈣約為 gluconate 的 3 倍、刺激性強走中央靜脈
- D. Gluconate 外滲比 CaCl₂ 更易壞死

> [!success]- 答案
> **C** — CaCl₂ 元素鈣約為 gluconate 的 3 倍，刺激性/外滲壞死更嚴重，一般走中央靜脈或急救；周邊靜脈優先用 gluconate。

**Q3.** 哪一類病人給鈣劑需特別警惕致命性心律不整？
- A. 高血壓病人
- B. 服用 digoxin（洋地黃）的病人
- C. 氣喘病人
- D. 糖尿病病人

> [!success]- 答案
> **B** — Digoxin 病人快速高鈣可致 stone heart（心肌強直），非緊急應緩給並謹慎評估。
