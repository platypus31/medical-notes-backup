# Potassium Chloride（氯化鉀 / KCl）

> [!abstract] 一句話：**補鉀第一線用藥**，K⁺ 是細胞內主要陽離子，維持靜息膜電位、心肌與神經肌肉興奮性。**IV 給太快/太濃會直接致死性心律不整（心跳停止）**，是全院最危險的常規藥之一；能口服就口服，IV 必須稀釋、限速、監測。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **KCl 絕對不可 IV push / bolus**：直接推注高濃度鉀會瞬間停心（asystole）。這是護理教育的頭號用藥安全事件，開醫囑務必寫明「稀釋後輸注、限速」
> 2. **周邊靜脈濃度上限 ≤10 mEq/100mL（即 ≤0.1 mEq/mL）**、輸速一般 **≤10 mEq/hr**；更高濃度或更快速度（如 20 mEq/hr）**必須走中央靜脈 + 心電圖監測 + 幫浦控速**（依院規/仿單）
> 3. **給鉀前先確認腎功能與尿量**：oliguria/AKI/ESRD 病人排鉀能力差，補鉀易反轉成致命高血鉀。無尿病人補鉀前要極度謹慎
> 4. **高血鉀比低血鉀更快致命**：補過頭 → 高尖 T 波 → QRS 增寬 → sine wave → VF/asystole。補鉀後要追蹤 K⁺ 與 ECG
> 5. **低血鉀常合併低血鎂**：低鎂會讓補鉀無效（腎持續漏鉀），**難治性低血鉀先補鎂**（[[Magnesium Sulfate(硫酸鎂)]]）
> 6. **周邊 IV 高濃度鉀灼痛/靜脈炎**：病人喊痛要理會，考慮降濃度、換中央、或口服

## 💊 適應症（治療哪些疾病）
- **低血鉀症 [[Hypokalemia(低血鉀症)]]**（K⁺ < 3.5 mEq/L）— 最主要適應症
- **利尿劑（loop/thiazide）引起的鉀流失**、腹瀉/嘔吐/NG 引流失鉀
- **糖尿病酮酸血症 DKA [[Diabetic Ketoacidosis(糖尿病酮酸血症)]]** 治療中（胰島素會把 K⁺ 打進細胞，需預防性補鉀，K<3.3 時應先補鉀再給胰島素）
- **周期性麻痹（低血鉀型）**、部分心律不整（低鉀誘發者）合併預防 digoxin 毒性

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 / 用法 |
| --- | --- |
| 口服補充（輕度低鉀 3.0–3.4） | **20–40 mEq PO QD~BID**（隨餐、配大量水，緩釋錠不可磨碎） |
| 口服積極補充 | 40–100 mEq/day 分次（依 K⁺ 值與腎功能，依院規/仿單） |
| IV 周邊靜脈 | 濃度 **≤10 mEq/100mL**，速度 **≤10 mEq/hr**（一般病房上限） |
| IV 中央靜脈（監測下） | 可 **10–20 mEq/hr**，需連續 ECG + 幫浦 + 中央導管（ICU/依院規） |
| 經驗法則 | 血清 K⁺ 每降 0.3 mEq/L 約缺 ~100 mEq 總體鉀（粗估，個體差異大） |
- **腎功能不良（eGFR ↓ / oliguria）**：減量、放慢、頻繁監測 K⁺，寧可慢補；無尿者原則上不補（除非確診缺鉀且嚴密監測）
- **肝功能**：一般不需依肝調整
- **給藥形式**：口服緩釋錠（K-Dur/Slow-K）刺激性較低但不可磨碎；液劑苦、刺激胃

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：口服胃腸刺激（噁心、腹痛、腹瀉）、周邊 IV 部位灼痛/靜脈炎
- **嚴重**：
  - **高血鉀症（醫源性補過量）→ 致命性心律不整（VF、asystole）**，最重要的嚴重副作用
  - 緩釋錠罕見造成上消化道潰瘍/穿孔
  - IV 快速/高濃度 → 心臟停止

## 🚫 禁忌症
- **高血鉀症**（現存 K⁺ 已高）
- **嚴重腎衰竭 / 無尿 / oliguria**（排鉀能力喪失）
- **未矯正的 Addison 病（腎上腺功能不全）**、嚴重脫水未矯正
- 使用保鉀利尿劑 / ACEI / ARB / spironolactone 且已高鉀者
- **絕不 IV push**（給藥途徑禁忌）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：血清 K⁺（補後回追）、腎功能/尿量、ECG（大量或快速補鉀時連續監測 T 波/QRS）、同時查並矯正 Mg²⁺
- **交互作用**：
  - **保鉀利尿劑（spironolactone/eplerenone/amiloride）、ACEI、ARB、NSAID、trimethoprim**：疊加高血鉀風險
  - **Digoxin**：低血鉀增強 digoxin 毒性；但補鉀過快又危險，需平衡
  - β-blocker、succinylcholine 影響鉀分布
- **孕哺**：孕期哺乳期補充生理需要量安全（維持正常血鉀）

## 🔗 相關
- 疾病：[[Hypokalemia(低血鉀症)]]、[[Diabetic Ketoacidosis(糖尿病酮酸血症)]]、[[Hyperkalemia(高血鉀症)]]
- 同類 / 對照：[[Magnesium Sulfate(硫酸鎂)]]（難治性低鉀先補鎂）、[[Calcium Gluconate(葡萄糖酸鈣)]]（高血鉀時穩定心肌）
- 對照概念：高血鉀處置（calcium gluconate 穩膜 + insulin/glucose + β-agonist 移鉀 + 利尿/透析排鉀）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 病房醫囑要幫低血鉀病人（K 3.0）補鉀，下列何者最安全？
- A. 20 mEq KCl 直接 IV push 快速矯正
- B. 周邊 IV：KCl 稀釋成 ≤10 mEq/100mL，速度 ≤10 mEq/hr
- C. 一次 100 mEq 快速滴注
- D. 不需查腎功能直接補

> [!success]- 答案
> **B** — KCl 絕不可 IV push；周邊靜脈標準是稀釋濃度 ≤10 mEq/100mL、速度 ≤10 mEq/hr。更快更濃需中央靜脈 + ECG。補前需確認腎功能與尿量。

**Q2.** 一名病人低血鉀，補了大量 KCl 血鉀仍拉不上來，最該檢查與矯正什麼？
- A. 血鈣
- B. 血鎂（低鎂會讓補鉀無效）
- C. 血鈉
- D. 血糖

> [!success]- 答案
> **B** — 低血鎂會使腎臟持續漏鉀、補鉀無效，難治性低血鉀應先補鎂（MgSO4）。

**Q3.** 關於補鉀的安全性，何者正確？
- A. 高血鉀不如低血鉀危險
- B. 無尿/嚴重腎衰竭病人可安心大量補鉀
- C. 補過量高血鉀可致 VF/asystole，補後應追蹤 K⁺ 與 ECG
- D. IV push 是最快最好的方式

> [!success]- 答案
> **C** — 醫源性高血鉀可致命（VF/asystole），比低血鉀進展更快。無尿/腎衰竭補鉀極危險，KCl 禁 IV push。
