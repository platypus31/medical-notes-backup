# 青光眼藥（Timolol / Latanoprost / Pilocarpine / Acetazolamide / Brimonidine / Dorzolamide）

> [!abstract] 一句話：**青光眼治療目標是降低眼內壓（IOP）**，兩大策略——① **減少房水（aqueous humor）生成**（β-blocker、碳酸酐酶抑制劑、α2 agonist）；② **增加房水排出**（前列腺素類似物走葡萄膜鞏膜路徑、縮瞳劑走小樑網路徑）。**第一線首選 Latanoprost（前列腺素類似物，每日一次、副作用少）**；急性隅角閉鎖是眼科急症要多藥齊下並急會診。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **急性隅角閉鎖性青光眼是眼科急症**：劇烈眼痛、頭痛、噁心嘔吐、視力模糊、瞳孔**中度放大且固定**、眼球堅硬如石 → 立即降眼壓（**Acetazolamide IV/口服 + Pilocarpine + 局部 Timolol** 等多藥齊下）並**儘速眼科會診雷射虹膜造口術**
> 2. **抗膽鹼藥物 / 散瞳劑（atropine、抗組織胺、部分抗憂鬱／抗精神病藥）會誘發或惡化隅角閉鎖** → 淺前房 / 窄隅角病人用藥前先想
> 3. **Timolol（β-blocker）全身吸收 → 心搏過緩、支氣管痙攣** → **氣喘 / COPD、心衰竭、心搏過緩／傳導阻滯病人禁用或慎用**（點眼後壓住鼻淚管可減少全身吸收）
> 4. **Brimonidine（α2 agonist）嬰幼兒禁用** → 中樞神經抑制、可致**呼吸暫停 / 昏睡**
> 5. **Acetazolamide 是磺胺類（sulfonamide）** → 磺胺過敏者交叉反應風險；並注意代謝性酸中毒、低血鉀、腎結石

## 💊 適應症（治療哪些疾病）
- **原發性開放隅角青光眼 [[Open-Angle Glaucoma(開放隅角青光眼)]]**：長期控制 IOP，**第一線 Latanoprost**，不足時加 β-blocker / CAI / α2 agonist
- **急性隅角閉鎖性青光眼 [[Acute Angle-Closure Glaucoma(急性隅角閉鎖性青光眼)]]**（急症）：Acetazolamide + Pilocarpine + 局部降壓多管齊下，急會診雷射虹膜造口
- **高眼壓症 / 各類次發性青光眼**的降壓輔助
- Acetazolamide 另有非眼科用途：高山症預防、部分癲癇、假性腦瘤（idiopathic intracranial hypertension）

## 📏 劑量 / 用法（成人 · 依院規 / 眼科醫囑）
| 藥物 | 分類 / 機轉 | 途徑 | 降壓機轉 |
| --- | --- | --- | --- |
| **Timolol** | β-blocker | 眼藥水（常 BID） | **減少**房水生成 |
| **Latanoprost** | 前列腺素類似物 | 眼藥水（**每日一次，睡前**） | **增加**葡萄膜鞏膜路徑排出（第一線） |
| **Pilocarpine** | 膽鹼激動劑 / 縮瞳劑 | 眼藥水 | 收縮睫狀肌、拉開小樑網 → **增加**排出 |
| **Acetazolamide** | 碳酸酐酶抑制劑（CAI） | **口服 / IV**（全身） | **減少**房水生成（急症全身給藥） |
| **Brimonidine** | α2 agonist | 眼藥水 | **減少**生成 + **增加**排出 |
| **Dorzolamide** | 局部碳酸酐酶抑制劑 | 眼藥水 | **減少**房水生成（常與 Timolol 複方 Cosopt） |
- 各藥點眼頻次 / Acetazolamide 精確 mg 劑量**依院規 / 眼科醫囑**（需查證）
- **多種眼藥水併用時每種間隔 ≥5 分鐘**，避免沖掉前一藥
- 點眼後壓迫鼻淚管（punctal occlusion）1–2 分鐘可減少全身吸收（尤其 Timolol / Brimonidine）

## ⚠️ 副作用（常見 / 嚴重）
- **Timolol**：**全身 β-blocker 效應——心搏過緩、支氣管痙攣（氣喘惡化）、低血壓、掩蓋低血糖症狀、疲倦**；局部眼刺激
- **Latanoprost**：**虹膜色素沉著加深（棕色，可能永久）、睫毛增長變濃、眼周皮膚色素沉著**、結膜充血；少見黃斑水腫
- **Pilocarpine**：**縮瞳 → 視力模糊（暗處更明顯）、近視偏移、眉／額頭痛（睫狀肌痙攣）**；全身膽鹼效應（流涎、盜汗、腹瀉，罕見）
- **Acetazolamide**：**代謝性酸中毒、低血鉀、手足麻感（paresthesia）、腎結石、味覺改變（碳酸飲料變味）、多尿**；磺胺過敏交叉反應、少見骨髓抑制
- **Brimonidine**：**口乾、疲倦嗜睡、眼過敏性結膜炎**；**嬰幼兒中樞抑制／呼吸暫停**
- **Dorzolamide**：局部刺激、苦味、點眼後燒灼感；全身副作用比口服 Acetazolamide 少

## 🚫 禁忌症
- **Timolol**：氣喘 / 嚴重 COPD、心搏過緩、二三度房室傳導阻滯、失代償心衰竭
- **Pilocarpine**：不適合用於**已發炎的葡萄膜炎相關青光眼**（縮瞳可致後粘連）；急性隅角閉鎖 IOP 極高（>40–50）時瞳孔括約肌可能缺血麻痺致 pilocarpine 一時無效，需先全身降壓
- **Acetazolamide**：磺胺過敏、嚴重肝腎功能不全、低血鉀 / 低血鈉、腎上腺功能不全
- **Brimonidine**：**嬰幼兒（尤 <2 歲）禁用**、正服用 MAO 抑制劑者
- 淺前房 / 窄隅角者避免抗膽鹼 / 散瞳劑（誘發閉鎖）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：IOP 反應、視野 / 視神經；Timolol → 心率 / 呼吸；Acetazolamide → 電解質（K⁺）、腎功能、酸鹼
- **交互作用**：Timolol + 全身 β-blocker / CCB → 心搏過緩加乘；Brimonidine + 中樞抑制劑 / MAOI；Acetazolamide + 其他排鉀藥 → 低血鉀加重（尤與 digoxin 併用心律風險）
- **孕哺**：多數青光眼藥孕期資料有限，需眼科與產科共同權衡；Brimonidine 哺乳嬰兒中樞抑制風險需留意
- **抗膽鹼 / 散瞳劑禁忌**：窄隅角病人使用前評估隅角，避免誘發急性閉鎖

## 🔗 相關
- 疾病：[[Open-Angle Glaucoma(開放隅角青光眼)]]、[[Acute Angle-Closure Glaucoma(急性隅角閉鎖性青光眼)]]、高眼壓症
- 同類 / 複方：Cosopt（Dorzolamide + Timolol 複方）；[[Acetazolamide(乙醯偶氮胺)]] 全身 CAI
- 機轉對照：減少生成（β-blocker / CAI / α2）vs 增加排出（前列腺素 / 縮瞳劑）

Latanoprost 增加房水排出的路徑是哪一條？::葡萄膜鞏膜路徑（uveoscleral outflow）

Latanoprost 的特殊局部副作用有哪些？::虹膜色素沉著加深（可能永久）、睫毛增長變濃、眼周色素沉著

Timolol 是哪類藥？最重要的全身副作用是什麼？::β-blocker（減少房水生成）；全身吸收可致心搏過緩、支氣管痙攣（氣喘 / COPD、心衰竭、心搏過緩者禁用）

Pilocarpine 的機轉與主要用途是什麼？::膽鹼激動劑 / 縮瞳劑，收縮睫狀肌拉開小樑網增加排出；用於急性隅角閉鎖性青光眼

Pilocarpine 常見的副作用是什麼？::縮瞳造成視力模糊（暗處更明顯）、近視偏移、眉／額頭痛

Acetazolamide 是哪類藥？急症如何給？::碳酸酐酶抑制劑（減少房水生成）；急性隅角閉鎖時全身給藥（口服 / IV）

Acetazolamide 的全身副作用有哪些？::代謝性酸中毒、低血鉀、手足麻感、腎結石、味覺改變；屬磺胺類，磺胺過敏者交叉反應

Brimonidine 是哪類藥？哪個族群絕對禁用？::α2 agonist（減少生成+增加排出）；嬰幼兒禁用（中樞神經抑制、可致呼吸暫停）

Dorzolamide 相對 Acetazolamide 的優點是什麼？常與誰做複方？::局部（眼藥水）碳酸酐酶抑制劑，全身副作用少；常與 Timolol 複方（Cosopt）

急性隅角閉鎖性青光眼的臨床表現有哪些？::劇烈眼痛、頭痛、噁心嘔吐、視力模糊、瞳孔中度放大且固定、眼球堅硬如石

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名氣喘病人被診斷開放隅角青光眼需要降眼壓。下列何者因全身副作用最應避免？
- A. Latanoprost
- B. Timolol
- C. Dorzolamide
- D. Brimonidine

> [!success]- 答案
> **B** — Timolol 是 β-blocker，全身吸收可誘發**支氣管痙攣**惡化氣喘、並致心搏過緩；氣喘 / COPD、心衰竭病人應避免。第一線可用 Latanoprost。

**Q2.** 急診來了一位急性隅角閉鎖性青光眼（劇烈眼痛、頭痛、噁心、視力模糊、瞳孔中度放大、眼球硬）。下列處置何者最不恰當？
- A. Acetazolamide 口服 / IV 全身降壓
- B. 局部 Pilocarpine 縮瞳
- C. 局部 Timolol
- D. 給 atropine 散瞳看清眼底

> [!success]- 答案
> **D** — **散瞳 / 抗膽鹼藥會惡化隅角閉鎖**，絕對禁忌。急症應多藥齊下降眼壓（Acetazolamide + Pilocarpine + 局部 Timolol）並急會診雷射虹膜造口術。

**Q3.** 關於 Latanoprost，下列敘述何者正確？
- A. 屬 β-blocker，會誘發氣喘
- B. 機轉是減少房水生成
- C. 增加葡萄膜鞏膜路徑排出，特殊副作用為虹膜色素加深與睫毛增長
- D. 嬰幼兒禁用因會呼吸暫停

> [!success]- 答案
> **C** — Latanoprost 是前列腺素類似物，經**葡萄膜鞏膜路徑增加房水排出**，每日一次為第一線；特殊副作用是**虹膜色素沉著加深（可能永久）與睫毛增長**。呼吸暫停禁忌的是 Brimonidine（α2 agonist）用於嬰幼兒。
