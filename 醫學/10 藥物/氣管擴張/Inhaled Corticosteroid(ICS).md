# Inhaled Corticosteroid（ICS / 吸入型類固醇）

> [!abstract] 一句話：吸入型類固醇壓制氣道**慢性發炎**（降低 airway inflammation 與 hyperresponsiveness），是**氣喘控制治療的基石（controller）**。代表藥 **Budesonide、Fluticasone、Beclomethasone**。**不是救急藥**——不會立刻擴張支氣管，需規律使用才見效。GINA 已強調**每一位氣喘病人都應有含 ICS 的治療**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **ICS 不能救急**：急性發作靠 SABA + 全身類固醇；ICS 是每天規律用的 controller
> 2. **口腔念珠菌（鵝口瘡）+ 聲音沙啞** 是最常見局部副作用 → **吸完務必漱口吐掉**、用 spacer
> 3. **突然停藥** → 氣喘失控、發作風險上升；要降階不要驟停
> 4. 高劑量長期使用仍有全身效應（骨密度↓、白內障/青光眼、兒童生長速度輕度影響），但**遠低於口服類固醇**

## 💊 適應症（治療哪些疾病）
- **[[Asthma(氣喘)]] 控制治療（controller，基石）**：GINA 各 step 的核心，含 ICS-formoterol「維持＋緩解」
- **[[COPD(慢性阻塞性肺病)]]**：**選擇性** add-on（反覆惡化、blood eosinophil 高者），常為 ICS/LABA 或三合一——**COPD 不單用 ICS**
- 過敏性鼻炎（鼻用型，另一劑型）

## 📏 劑量 / 用法（成人 · 吸入規律使用）
| 藥 | 低劑量（成人 · 每日概念） | 備註 |
| --- | --- | --- |
| **Budesonide** | 200–400 µg | budesonide/formoterol 可維持＋緩解 |
| **Fluticasone propionate** | 100–250 µg | 常見 fluticasone/salmeterol 複方 |
| **Beclomethasone** | 200–500 µg | 傳統 ICS |
| **Ciclesonide / Mometasone** | 依產品 | 局部副作用較少（ciclesonide 為前驅藥） |
- 劑量分低/中/高，依控制狀況**升階/降階**；穩定 3 個月可考慮降階
- 途徑：吸入；**吸完漱口**、建議搭配 spacer
- COPD 用 ICS 需權衡**肺炎風險**

## ⚠️ 副作用（常見 / 嚴重）
- **局部（常見）**：口咽念珠菌感染（鵝口瘡）、聲音沙啞（dysphonia）、咳嗽、口咽刺激
- **全身（高劑量長期）**：腎上腺抑制（少見）、骨密度下降、白內障 / 青光眼、皮膚變薄、兒童生長速度輕度減緩
- **COPD 特有**：**肺炎風險上升**（尤其高劑量 fluticasone）

## 🚫 禁忌症
- 對藥物成分過敏
- 呼吸道活動性感染（如未治療的肺結核、真菌感染）需謹慎
- 相對：COPD 反覆肺炎者重新評估 ICS 必要性

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：氣喘控制（症狀、發作、肺功能）、局部副作用（口腔檢查）、兒童生長、COPD 肺炎徵象
- **交互作用**：
  - **強效 CYP3A4 抑制劑**（如 ritonavir、ketoconazole）→ 增加 fluticasone/budesonide 全身暴露（Cushing 樣、腎上腺抑制風險）
- **孕哺**：**氣喘控制優先**，ICS（尤其 budesonide）在孕期使用安全性資料充分，勿自行停藥

## 🔗 相關
- 疾病：[[Asthma(氣喘)]]、[[COPD(慢性阻塞性肺病)]]
- 併用 / 對照：[[Long Acting Beta-Agonist(LABA)]]（氣喘必併）、[[Short Acting Beta-Agonist(SABA)]]（救急）、[[Long-Acting Muscarinic Antagonist(LAMA)]]（三合一）
- 對照：[[Steroid(類固醇)]]（全身性類固醇，急性發作用）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 氣喘病人抱怨用了新的吸入藥後「喉嚨白白的、聲音沙啞」。最可能是哪類藥、如何處理？
- A. SABA；停藥
- B. ICS；吸完漱口、用 spacer
- C. LAMA；多喝水
- D. LABA；換藥

> [!success]- 答案
> **B** — 鵝口瘡與聲音沙啞是 **ICS** 的典型局部副作用，處理是**吸完漱口吐掉**並使用 spacer，通常不需停藥。

**Q2.** 關於 ICS 的敘述，何者正確？
- A. 是氣喘急性發作的救急藥
- B. 是氣喘控制治療的基石，需規律使用
- C. COPD 應單獨長期高劑量使用
- D. 孕婦應立即停用

> [!success]- 答案
> **B** — ICS 是氣喘**控制基石**，需規律使用；非救急、COPD 不單用、孕婦不應自行停藥。

**Q3.** COPD 病人使用高劑量吸入 fluticasone，需特別監測下列哪項風險？
- A. 低血糖
- B. 肺炎
- C. 高血鉀
- D. 尿滯留

> [!success]- 答案
> **B** — COPD 使用 ICS（尤其高劑量 fluticasone）與**肺炎風險上升**相關，需依 eosinophil 與惡化史權衡。
