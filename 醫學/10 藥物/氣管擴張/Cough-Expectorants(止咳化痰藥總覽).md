# Cough Expectorants（止咳化痰藥總覽）

> [!abstract] 一句話：**咳嗽藥分兩大方向——「鎮咳（antitussive）」壓咳嗽反射 vs「化痰 / 祛痰（mucolytic / expectorant）」讓痰變稀好咳出**，兩者常搞混。**Dextromethorphan、codeine** 是中樞鎮咳（壓延腦咳嗽中樞）；**acetylcysteine、ambroxol** 是黏液溶解（打斷痰液雙硫鍵 / 稀化）；**guaifenesin** 是祛痰（增加呼吸道分泌使痰稀）。**有痰的濕咳原則上不強力鎮咳**（會讓痰積在肺裡）。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **兒童禁用 codeine（及 tramadol）**：FDA 禁用於 < 12 歲；12–18 歲肥胖 / 呼吸問題也禁。**超快代謝者（CYP2D6 ultra-rapid）**會把 codeine 快速轉成嗎啡 → 致命呼吸抑制；哺乳母親用 codeine 也可能經乳汁害嬰兒。
> 2. **Codeine = 鴉片類**：呼吸抑制、鎮靜、便秘、成癮 / 濫用（含止咳糖漿濫用）；與其他 CNS 抑制劑（酒精、benzodiazepine、其他鴉片）相加致命。
> 3. **氣喘 / 有痰濕咳慎用強力鎮咳**：壓掉咳嗽反射會使痰液滯留、加重阻塞與感染；濕咳應以化痰 + 治病因為主。
> 4. **Dextromethorphan + serotonergic 藥 → 血清素症候群**：與 SSRI/SNRI、MAOI、linezolid、tramadol 併用風險；**MAOI 為禁忌**。高劑量 DXM 也被濫用（解離 / 幻覺）。
> 5. **Acetylcysteine 用途雙面**：口服 / 吸入化痰（有硫臭味、吸入可能誘發支氣管痙攣，氣喘者慎）；**靜脈 NAC 是乙醯胺酚（acetaminophen）中毒解毒劑**（別和化痰劑用途混淆）。

## 💊 各藥速覽

| 藥物 | 類別 / 機轉 | 用途 | 關鍵副作用 / 注意 |
| --- | --- | --- | --- |
| **Dextromethorphan (DXM)** | 中樞鎮咳（非鴉片，σ / NMDA） | 乾咳鎮咳 | 血清素症候群（禁 MAOI）、高劑量濫用 |
| **Codeine** | 鴉片類中樞鎮咳 / 止痛 | 頑固乾咳（限成人） | **兒童禁用**、呼吸抑制、便秘、成癮、CYP2D6 超快代謝致命 |
| **Acetylcysteine (NAC)** | 黏液溶解（打斷黏蛋白雙硫鍵） | 濃痰化痰（口服/吸入）；IV 為 APAP 解毒 | 硫臭、吸入誘發支氣管痙攣（氣喘慎）、噁心 |
| **Ambroxol / Bromhexine** | 黏液調節 / 祛痰 | 濃痰、慢性支氣管炎 | 一般耐受佳、腸胃不適；罕見過敏 |
| **Guaifenesin** | 祛痰（增加呼吸道分泌稀化痰） | 濕咳有痰、多喝水配合 | 噁心、頭暈；效果溫和 |

## 📏 劑量 / 用法（成人 · 依院規）
- **Dextromethorphan**：10–20 mg PO Q4H 或 30 mg Q6–8H（依複方）；⚠不與 MAOI 併用。
- **Codeine（鎮咳）**：成人低劑量（如 10–20 mg PO Q4–6H prn）；**< 12 歲禁用**；限最短療程。
- **Acetylcysteine**：口服 / 吸入化痰依劑型；**吸入前後注意支氣管痙攣**（氣喘者可先用支氣管擴張劑）。
- **Ambroxol**：常見 30 mg PO TID（或緩釋劑型）。
- **Guaifenesin**：200–400 mg PO Q4H（上限依仿單），**多喝水**加強祛痰。
- ⚠️所有精確劑量、複方成分、上限依院規與病人（兒童、肝腎、呼吸狀態）調整。

## ⚠️ 副作用
- **Codeine**：呼吸抑制、鎮靜、便秘、噁心、依賴 / 成癮、瘙癢。
- **Dextromethorphan**：頭暈、噁心；血清素症候群（合併 serotonergic 藥）；濫用致解離 / 幻覺。
- **Acetylcysteine**：噁心嘔吐、硫臭味、吸入誘發支氣管痙攣、口炎；（IV 高劑量可類過敏反應）。
- **Ambroxol / Bromhexine**：腸胃不適、罕見皮膚過敏反應。
- **Guaifenesin**：噁心、嘔吐、頭暈（大致溫和）。

## 🚫 禁忌症 / 慎用
- **Codeine**：**< 12 歲兒童禁用**、12–18 歲呼吸風險 / 肥胖 / OSA 慎用、哺乳母親避免、呼吸抑制 / COPD 惡化 / 麻痺性腸阻塞、與其他 CNS 抑制劑併用。
- **Dextromethorphan**：**MAOI 併用禁忌（含停藥 14 天內）**；serotonergic 藥慎用。
- **有痰濕咳 / 氣喘急性**：避免強力中樞鎮咳（痰滯留）。
- **Acetylcysteine 吸入**：活動性支氣管痙攣 / 氣喘慎用。
- 嚴重肝病用 codeine 慎（代謝改變）。

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：咳嗽性質（乾 / 濕）、呼吸狀態與鎮靜（codeine）、氣喘者吸入 NAC 後喘鳴、痰液性狀改善。
- **交互作用**：
  - **Codeine**：其他鴉片 / benzodiazepine / 酒精 → 呼吸抑制相加；CYP2D6 抑制劑改變轉化。
  - **Dextromethorphan**：SSRI/SNRI、MAOI、linezolid、tramadol → 血清素症候群；CYP2D6 抑制劑升濃度。
  - **Acetylcysteine（化痰）**：一般少重大交互（避免與某些抗生素同時霧化）。
- **孕哺**：咳嗽藥孕哺多以非藥物 / 蜂蜜、guaifenesin 溫和選項為主；codeine 孕晚期 / 哺乳避免（新生兒呼吸抑制）；DXM 一般視為較低風險但仍權衡。

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名 6 歲兒童因感冒咳嗽就診，家長要求開止咳藥。下列何者絕對不該給？
- A. Guaifenesin 祛痰
- B. Codeine 鎮咳糖漿
- C. 多喝水、蜂蜜（>1歲）
- D. 症狀支持治療

> [!success]- 答案
> **B** — FDA 禁用 codeine（及 tramadol）於 <12 歲兒童，因 CYP2D6 超快代謝者會轉成過量嗎啡致命呼吸抑制。兒童咳嗽多以支持治療、多喝水、蜂蜜（>1歲）為主。

**Q2.** 一名服用 SSRI 的病人自行大量服用含 dextromethorphan 的止咳糖漿後出現躁動、發抖、高熱、反射亢進。最可能是？
- A. 過敏反應
- B. 血清素症候群
- C. 鴉片過量
- D. 低血糖

> [!success]- 答案
> **B** — Dextromethorphan 有 serotonergic 作用，與 SSRI/SNRI/MAOI/linezolid/tramadol 併用可致血清素症候群（躁動、發抖、高熱、反射亢進、肌陣攣）；MAOI 併用為明確禁忌。
