# Long-Acting Muscarinic Antagonist（LAMA / 長效抗膽鹼藥）

> [!abstract] 一句話：阻斷氣道 **M3 蕈毒鹼受體** → 抑制副交感介導的支氣管收縮與黏液分泌 → 支氣管擴張，作用**持續長（24h，多 QD）**。**COPD 維持治療的基石**（代表藥 **Tiotropium**）；氣喘為 add-on（step 4–5）。短效版是 SAMA（Ipratropium）。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **抗膽鹼副作用**：口乾（最常見）、**尿滯留**（前列腺肥大者小心）、**急性隅角閉鎖性青光眼惡化**（避免噴到眼睛）
> 2. LAMA 是**維持藥非救急**：急性發作靠 SABA
> 3. **不要 LAMA + SAMA 疊加**（tiotropium + ipratropium）→ 抗膽鹼作用重複、副作用增加
> 4. COPD 是 LAMA 主場；**氣喘不會單用 LAMA**，是 ICS/LABA 控制不佳時的 add-on

## 💊 適應症（治療哪些疾病）
- **[[COPD(慢性阻塞性肺病)]] 維持治療（基石）**：減少惡化、改善呼吸困難與運動耐受
- **[[Asthma(氣喘)]] add-on**：GINA step 4–5，ICS/LABA 控制不佳時加 tiotropium
- 三合一（ICS/LABA/LAMA）用於反覆惡化的 COPD

## 📏 劑量 / 用法（成人 · 吸入 QD 為主）
| 藥 | 裝置 / 用法 |
| --- | --- |
| **Tiotropium** | Handihaler 18 µg QD 或 Respimat 2.5 µg 2 噴 QD |
| **Umeclidinium** | 62.5 µg QD（Ellipta） |
| **Glycopyrronium** | QD（多為複方） |
| **Aclidinium** | BID |
- 途徑：吸入；**吸完漱口、避免噴到眼睛**（青光眼）
- 腎肝：嚴重腎功能不全者謹慎（部分經腎排除）

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：口乾（最常見）、咽喉刺激、便祕、咳嗽
- **嚴重 / 需注意**：
  - **尿滯留**（尤其 BPH 前列腺肥大男性）
  - **急性隅角閉鎖性青光眼**（藥物接觸眼睛）
  - 心搏過速、心律不整（少見）
  - 矛盾性支氣管痙攣（罕見）

## 🚫 禁忌症
- 對藥物或阿托品類（atropine）成分過敏
- **相對禁忌 / 謹慎**：隅角閉鎖性青光眼、症狀性前列腺肥大 / 膀胱頸阻塞、嚴重腎功能不全

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：症狀改善、惡化頻率、青光眼/排尿症狀
- **交互作用**：
  - **其他抗膽鹼藥**（抗組織胺、TCA、SAMA ipratropium）→ 抗膽鹼副作用加乘，避免疊加
- **孕哺**：資料有限；COPD 孕婦少見，需效益風險評估

## 🔗 相關
- 疾病：[[COPD(慢性阻塞性肺病)]]、[[Asthma(氣喘)]]
- 同類 / 對照：短效抗膽鹼 SAMA（Ipratropium，急性可用）
- 併用：[[Long Acting Beta-Agonist(LABA)]]、[[Inhaled Corticosteroid(ICS)]]（COPD 三合一）、[[Short Acting Beta-Agonist(SABA)]]（救急）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 72 歲男性 COPD 合併前列腺肥大，使用 tiotropium 後主訴解尿困難。最可能原因是？
- A. 藥物過敏
- B. 抗膽鹼作用導致尿滯留
- C. 低血鉀
- D. 感染

> [!success]- 答案
> **B** — LAMA 的**抗膽鹼作用**可導致尿滯留，前列腺肥大男性風險最高，需評估是否調整。

**Q2.** 下列何者是 COPD 維持治療的基石代表藥（LAMA）？
- A. Salbutamol
- B. Tiotropium
- C. Budesonide
- D. Montelukast

> [!success]- 答案
> **B** — **Tiotropium** 是長效抗膽鹼（LAMA），為 COPD 維持治療基石。

**Q3.** 關於 LAMA 使用的注意事項，何者**錯誤**？
- A. 可能誘發隅角閉鎖性青光眼
- B. 前列腺肥大者小心尿滯留
- C. 可作為氣喘急性發作的第一線救急藥
- D. 不應與 SAMA 疊加

> [!success]- 答案
> **C** — LAMA 是**維持藥非救急**，急性發作靠 SABA；其餘敘述皆正確。
