# Short Acting Beta-Agonist（SABA / 短效乙型受體促效劑）

> [!abstract] 一句話：刺激氣道平滑肌 **β2 受體** → cAMP↑ → 支氣管**快速**擴張（幾分鐘起效），是氣喘/COPD **急性發作的 rescue（救急）** 藥，代表藥 **Salbutamol（Albuterol／Ventolin）**。起效快、持續短（4–6h），適合「有喘才用」而非長期單獨維持。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **氣喘用 SABA 頻率↑（每週 >2 次、或一個月用超過 1 支 MDI）＝控制不良** → 該加 [[Inhaled Corticosteroid(ICS)]]，不是一直加 SABA
> 2. **只靠 SABA、無 ICS 的氣喘** → GINA 已不建議，與**氣喘死亡風險上升**相關
> 3. **過量 / nebulizer 連續使用**：心悸、心搏過速、**手抖、低血鉀（K⁺ 移入細胞）**、乳酸上升 → 監測 K⁺、心律
> 4. 缺氧的氣喘發作是**急症**：SABA + O2 + 全身性類固醇，反應差要想到 impending respiratory failure

## 💊 適應症（治療哪些疾病）
- **[[Asthma(氣喘)]] 急性發作 / rescue**：喘、胸悶、運動誘發性支氣管收縮（運動前 pre-treatment）
- **[[COPD(慢性阻塞性肺病)]] 急性惡化 / 症狀緩解**
- **高血鉀急救輔助**：β2 促效可把 K⁺ 暫時移入細胞內（配合 insulin+glucose 使用）

## 📏 劑量 / 用法（成人）
| 情境 | 用法 |
| --- | --- |
| 氣喘/COPD rescue（MDI） | Salbutamol 100 µg/puff，**2 puffs Q4–6H PRN**（急性可 Q20min×3 起始） |
| 急性發作（nebulizer） | Salbutamol **2.5–5 mg neb Q20min×3**，再依反應調整 |
| 運動誘發預防 | 運動前 15 分鐘 2 puffs |
| Terbutaline（SC/皮下） | 0.25 mg SC，嚴重發作可用（較少用） |
- 途徑以**吸入為主**（MDI 建議搭配 spacer 提升到達率）；口服/靜脈全身性副作用大，少用
- 腎肝：一般不需特別調整；重複大量使用時留意心律與 K⁺

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：心悸、心搏過速、手抖（tremor）、頭痛、焦躁不安
- **嚴重 / 過量**：
  - **低血鉀**（K⁺ 移入細胞，可誘發心律不整）
  - 心律不整、心肌缺血（冠心病患者）
  - **矛盾性支氣管痙攣**（罕見）
  - 高血糖、乳酸上升

## 🚫 禁忌症
- 對 salbutamol / 該製劑成分過敏
- **相對禁忌**：嚴重心律不整、未控制的甲亢、心肌缺血高風險者（謹慎、非絕對）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：症狀反應、心率、（大量使用時）K⁺、發作頻率（評估控制）
- **交互作用**：
  - **非選擇性 β-blocker**（如 propranolol）→ 拮抗支氣管擴張、可誘發支氣管痙攣
  - **利尿劑（thiazide/loop）+ SABA** → 低血鉀加乘
  - MAOI / TCA → 增強擬交感作用
- **孕哺**：氣喘控制優先，salbutamol 在孕期使用安全性資料充分，一般可用

## 🔗 相關
- 疾病：[[Asthma(氣喘)]]、[[COPD(慢性阻塞性肺病)]]
- 同類 / 對照：[[Long Acting Beta-Agonist(LABA)]]（維持非救急）、[[Beta Agonist(乙型受體促效劑)]]（母類）、[[Inhaled Corticosteroid(ICS)]]（氣喘的控制基石）
- 併用：[[Long-Acting Muscarinic Antagonist(LAMA)]]（COPD）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 25 歲氣喘病人回診表示「每週要用 salbutamol 3–4 次才不喘」。最適當的處置是？
- A. 增加 SABA 使用頻率
- B. 加上吸入型類固醇（ICS）
- C. 改口服 SABA
- D. 只需觀察

> [!success]- 答案
> **B** — 每週 SABA >2 次代表控制不良，應加上 **ICS** 做控制治療（GINA），單靠增加 SABA 無法改善根本發炎且增加風險。

**Q2.** 嚴重氣喘發作病人連續 nebulize salbutamol 後出現心悸與肌肉無力，抽血最可能看到？
- A. 高血鉀
- B. 低血鉀
- C. 高血鈣
- D. 低血鈉

> [!success]- 答案
> **B** — β2 促效會把 **K⁺ 移入細胞內**造成低血鉀，大量使用時需監測並注意心律。

**Q3.** 下列何者是 SABA 的代表藥、且屬「救急型」而非長期維持？
- A. Salmeterol
- B. Tiotropium
- C. Salbutamol
- D. Budesonide

> [!success]- 答案
> **C** — **Salbutamol** 是短效 β2-agonist（rescue）；Salmeterol 是 LABA、Tiotropium 是 LAMA、Budesonide 是 ICS。
