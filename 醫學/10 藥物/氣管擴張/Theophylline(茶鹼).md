# Theophylline（茶鹼）

> [!abstract] 一句話：甲基黃嘌呤（methylxanthine），機轉為**非選擇性磷酸二酯酶（PDE）抑制**（cAMP↑）＋**腺苷（adenosine）受體拮抗**→支氣管擴張、輕度抗發炎。**治療窗窄、毒性重、藥物交互作用多**，現多為氣喘/COPD 的**第二三線 add-on**。靜脈型為 **Aminophylline**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **治療窗窄**：血中濃度目標約 **5–15 µg/mL（10–20 舊標準）**，超過易毒；**要監測血中濃度（TDM）**
> 2. **毒性徵象**：噁心嘔吐、心悸、**心律不整**（甚至致命性）、**癲癇（seizure）**、低血鉀、低血鎂——嚴重中毒 seizure 可先於心律不整出現
> 3. **交互作用地雷**：**CYP1A2 抑制劑**（ciprofloxacin、erythromycin、cimetidine、fluvoxanole 類）→ 濃度↑ 中毒；**吸菸/CYP1A2 誘導**→ 濃度↓ 失效
> 4. 心衰、肝病、發燒、高齡 → 清除率↓ 需減量；**戒菸後濃度會上升**要注意

## 💊 適應症（治療哪些疾病）
- **[[Asthma(氣喘)]]**：控制不佳時的 add-on（第二三線）
- **[[COPD(慢性阻塞性肺病)]]**：symptomatic add-on
- 早產兒呼吸暫停（apnea of prematurity，caffeine 為首選，theophylline 為替代）

## 📏 劑量 / 用法（成人 · 需 TDM）
| 情境 | 用法 |
| --- | --- |
| 口服維持（緩釋） | 依體重與濃度調整，通常分次 BID；**目標血中濃度 5–15 µg/mL** |
| Aminophylline IV（急性） | loading + continuous infusion，須嚴密監測（現已少用於急性氣喘） |
| 劑量調整 | 心衰、肝硬化、發燒、高齡、合併 CYP1A2 抑制劑 → **減量**；吸菸者清除率高需較高劑量 |
- **一定要監測血中濃度**（治療窗窄）；抽血時機與劑型有關
- 肝功能是主要清除途徑，肝病者顯著減量

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：噁心、嘔吐、腹痛、頭痛、失眠、心悸、手抖、利尿
- **嚴重 / 中毒**：
  - **心律不整**（心搏過速、房性/室性心律不整）
  - **癲癇（seizure）** — 中毒重要致命表現
  - **低血鉀、低血鎂、高血糖、代謝性酸中毒**
  - 中毒處理：支持性、活性碳（多劑）、控制 seizure、嚴重時血液透析

## 🚫 禁忌症
- 對甲基黃嘌呤過敏
- 未控制的心律不整、活動性癲癇（相對禁忌 / 高風險）
- 嚴重肝功能不全（清除↓，需大幅減量）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：**血中濃度（TDM）**、心律、K⁺/Mg²⁺、中毒症狀
- **交互作用（多且重要）**：
  - **濃度↑（中毒）**：ciprofloxacin、erythromycin/clarithromycin、cimetidine、fluconazole、allopurinol、口服避孕藥、心衰/肝病/發燒
  - **濃度↓（失效）**：吸菸、rifampin、phenytoin、carbamazepine、（CYP1A2 誘導）
- **孕哺**：可通過胎盤與乳汁，需監測、謹慎使用

## 🔗 相關
- 疾病：[[Asthma(氣喘)]]、[[COPD(慢性阻塞性肺病)]]
- 同機轉：[[PDE inhibitor(磷酸二酯酶抑制劑)]]（roflumilast 為選擇性 PDE4）
- 對照 / 併用：[[Short Acting Beta-Agonist(SABA)]]、[[Inhaled Corticosteroid(ICS)]]、[[Long-Acting Muscarinic Antagonist(LAMA)]]

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位服用緩釋 theophylline 的氣喘病人因泌尿道感染開始服用 ciprofloxacin，數天後出現噁心、心悸、抽搐。最可能原因是？
- A. 感染惡化
- B. Ciprofloxacin 抑制 CYP1A2 使 theophylline 濃度上升中毒
- C. 低血糖
- D. 藥物過敏

> [!success]- 答案
> **B** — **Ciprofloxacin 為 CYP1A2 抑制劑**，會使 theophylline 濃度上升導致中毒（噁心、心律不整、癲癇），需驗濃度並停/減量。

**Q2.** 關於 theophylline 的敘述，何者正確？
- A. 治療窗很寬，不需監測濃度
- B. 治療窗窄，需監測血中濃度
- C. 無藥物交互作用
- D. 是氣喘急性發作第一線

> [!success]- 答案
> **B** — Theophylline **治療窗窄**、交互作用多，需**監測血中濃度**，多為 add-on 而非第一線。

**Q3.** Theophylline 嚴重中毒時，下列何者是重要且可能致命的表現？
- A. 便祕
- B. 癲癇與心律不整
- C. 高血鉀
- D. 尿滯留

> [!success]- 答案
> **B** — 嚴重中毒可出現**癲癇（seizure）與心律不整**，並伴低血鉀、低血鎂，屬急症。
