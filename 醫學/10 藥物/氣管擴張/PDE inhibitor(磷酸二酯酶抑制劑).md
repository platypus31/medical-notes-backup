# PDE inhibitor（磷酸二酯酶抑制劑）

> [!abstract] 一句話：抑制**磷酸二酯酶（PDE）**——分解 cAMP/cGMP 的酵素 → 細胞內 cAMP/cGMP 上升 → 平滑肌鬆弛、抗發炎、強心等（依 PDE 亞型而異）。呼吸科重點是 **PDE4 抑制劑 Roflumilast**（口服，用於嚴重 COPD 抗發炎）；Theophylline 屬**非選擇性** PDE 抑制。其他亞型：PDE3（cilostazol/milrinone 強心血管）、PDE5（sildenafil 肺高壓/勃起）。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **Roflumilast 不是支氣管擴張劑、不能救急**：是**抗發炎** add-on，用於**嚴重 COPD + 慢性支氣管炎 + 反覆惡化**
> 2. **Roflumilast 副作用**：**體重減輕、腹瀉、噁心、頭痛**、失眠，以及**神經精神症狀（焦慮、憂鬱、自殺意念）** → 需監測
> 3. **PDE 亞型別搞混**：PDE3（cilostazol、milrinone）、PDE4（roflumilast）、PDE5（sildenafil、tadalafil）用途完全不同
> 4. **PDE5 抑制劑 + 硝酸鹽（nitrate）= 禁忌**（嚴重低血壓）——常考的跨類地雷

## 💊 適應症（依 PDE 亞型）
| PDE 亞型 | 代表藥 | 用途 |
| --- | --- | --- |
| **PDE4** | **Roflumilast** | 嚴重 [[COPD(慢性阻塞性肺病)]]（慢性支氣管炎、反覆惡化）抗發炎 add-on |
| 非選擇性 | [[Theophylline(茶鹼)]] | 氣喘/COPD 支氣管擴張 |
| PDE3 | Cilostazol、Milrinone | 間歇性跛行；急性心衰強心 |
| PDE5 | Sildenafil、Tadalafil | 勃起功能障礙、肺動脈高壓 |
- 本卡以呼吸科 **Roflumilast（PDE4）** 為主，其餘亞型作對照

## 📏 劑量 / 用法（Roflumilast 為主）
| 藥 | 用法 |
| --- | --- |
| **Roflumilast** | 500 µg PO QD（口服，抗發炎維持） |
| Theophylline | 見 [[Theophylline(茶鹼)]]（需 TDM） |
- Roflumilast 為口服每日一次，**非救急**，需數週見效
- 嚴重肝功能不全者禁用（roflumilast 經肝代謝）

## ⚠️ 副作用（Roflumilast 為主）
- **常見**：**體重減輕**、腹瀉、噁心、食慾下降、頭痛、失眠、腹痛
- **嚴重 / 需注意**：**神經精神症狀**（焦慮、憂鬱、失眠、自殺意念）、明顯體重下降
- （PDE5 類：頭痛、潮紅、視覺異常、低血壓；PDE3 milrinone：心律不整/低血壓）

## 🚫 禁忌症
- **Roflumilast**：中重度肝功能不全（Child-Pugh B–C）
- 對藥物成分過敏
- 有憂鬱 / 自殺病史者使用 roflumilast 需審慎
- （跨類提醒）**PDE5 抑制劑禁與硝酸鹽併用**

## 🔍 監測 / 交互作用 / 孕哺
- **監測（roflumilast）**：體重、精神狀態、腸胃症狀、COPD 惡化頻率
- **交互作用**：
  - Roflumilast 經 **CYP3A4 / CYP1A2** 代謝 → 強效 CYP 誘導劑（rifampin）降效；抑制劑（erythromycin、ketoconazole、cimetidine）增暴露
  - **PDE5 抑制劑 + nitrate → 嚴重低血壓（禁忌）**
- **孕哺**：資料有限，孕哺謹慎

## 🔗 相關

### 自我測驗（選擇題，答案摺疊）

**Q1.** 嚴重 COPD 合併慢性支氣管炎、反覆惡化的病人，加用 roflumilast 的主要目的是？
- A. 立即支氣管擴張救急
- B. 口服抗發炎、減少惡化
- C. 強心
- D. 降血糖

> [!success]- 答案
> **B** — Roflumilast（PDE4 抑制劑）是**口服抗發炎 add-on**，用於嚴重 COPD 減少惡化，**非支氣管擴張、非救急**。

**Q2.** 服用 roflumilast 的病人回診時，下列哪組副作用最需主動詢問監測？
- A. 尿滯留、青光眼
- B. 體重減輕、腹瀉、情緒/精神症狀
- C. 高血鉀、心律不整
- D. 鵝口瘡、聲音沙啞

> [!success]- 答案
> **B** — Roflumilast 常見**體重減輕、腹瀉、噁心**，並可有**神經精神症狀**（焦慮、憂鬱、自殺意念），需監測。

**Q3.** 下列 PDE 抑制劑與用途配對，何者**錯誤**？
- A. PDE4 — Roflumilast — COPD 抗發炎
- B. PDE3 — Cilostazol — 間歇性跛行
- C. PDE5 — Sildenafil — 肺動脈高壓
- D. PDE5 — Sildenafil — 可安心與硝酸鹽併用

> [!success]- 答案
> **D** — **PDE5 抑制劑與硝酸鹽併用是禁忌**（嚴重低血壓）；其餘配對皆正確。
