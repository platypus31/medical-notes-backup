# Methylene Blue（亞甲藍 / Methylthioninium）

> [!abstract] 一句話：**變性血紅素血症（methemoglobinemia）的解毒劑**。作為電子載體被 NADPH-methemoglobin reductase 還原後，把 methemoglobin 的 **Fe³⁺ 還原回 Fe²⁺** 恢復攜氧能力。**G6PD 缺乏者禁用（無 NADPH → 無效且致溶血）**；與 serotonergic 藥併用可致血清素症候群。

> [!danger] 🚨 值班紅旗（給藥前先想）
> 1. **臨床線索**：血氧飽和 SpO₂ 卡在 ~85% 對 O₂ 無反應、**血呈巧克力棕色**、發紺與 PaO₂ 不成比例 → 測 **co-oximetry / methemoglobin level**；有症狀（通常 MetHb > 20–30%）或缺氧 / 心肺疾病者更低門檻即治療
> 2. **G6PD 缺乏 = 禁用**：缺 NADPH 使 methylene blue **既無效又誘發嚴重溶血**；懷疑 G6PD 缺乏（地中海 / 蠶豆症 / 特定族群）用前警覺，改考慮輸血 / 換血 / 高壓氧 / ascorbic acid
> 3. **本身高劑量會「反致」變性血紅素血症**：劑量過大（> 7 mg/kg 累積）或反覆給，methylene blue 反而氧化血紅素 → **勿超量**
> 4. **血清素症候群風險**：具 MAO 抑制活性，與 **SSRI / SNRI / MAOI / 其他 serotonergic 藥**併用可致血清素症候群 → 用前盤點用藥
> 5. **假性低血氧讀值**：靜脈注射後**脈搏血氧機讀值會暫時假性下降**（染料干擾），別誤判惡化 —— 以 co-oximetry 為準

## 💊 適應症（中毒情境）
- **後天性變性血紅素血症 [[Methemoglobinemia(變性血紅素血症)]]**（症狀性，通常 MetHb > 20–30%，或有缺氧 / 心肺疾病者更低即治）
  - 常見致因：**dapsone、benzocaine / lidocaine 等局麻、亞硝酸鹽 nitrates / nitrites、苯胺 aniline 染料、部分抗生素**
- （非解毒用途：血管麻痺性休克 vasoplegia 之升壓輔助、副甲狀腺 / 泌尿手術染色 —— 非本篇重點）

## 📏 劑量 / 用法（成人）
| 步驟 | 劑量 |
| --- | --- |
| 起始 | **1–2 mg/kg IV** 稀釋輸注 **5 分鐘以上**（勿快推） |
| 重複 | 症狀 / MetHb 未降可於 **30–60 分鐘後重複 1 mg/kg** |
| 累積上限 | **勿超過約 7 mg/kg**（過量反致 methemoglobinemia / 溶血） |
- **治療反應**：通常數十分鐘內發紺改善、MetHb 下降
- 無效 / 禁忌（G6PD）時：**換血、輸紅血球、高壓氧、ascorbic acid（維生素 C）** 為替代
- ⚠️ 確切上限與稀釋**依院內方案 / 毒物中心**

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：尿液 / 皮膚染藍綠色、噁心、頭痛、頭暈、口中金屬味、注射部位刺痛
- **假性 SpO₂ 下降**（脈搏血氧機染料干擾，暫時）
- **嚴重**：
  - **G6PD 缺乏者溶血**（可致命）
  - **血清素症候群**（與 serotonergic 藥併用）
  - **高劑量反致變性血紅素血症 / 溶血**
  - 罕見過敏、高血壓 / 胸痛

## 🚫 禁忌症
- **G6PD 缺乏症（絕對禁忌）** —— 無效且致嚴重溶血
- 已知對 methylene blue 過敏
- **正在使用 serotonergic 藥物（SSRI/SNRI/MAOI 等）**—— 相對禁忌 / 高度警覺血清素症候群
- 嚴重腎功能不良慎用（排泄下降）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：co-oximetry 的 methemoglobin level（**非**一般脈搏血氧）、發紺 / 缺氧改善、溶血徵象（Hb / LDH / haptoglobin / 尿）、血清素症候群徵象、G6PD 狀態
- **交互作用**：**serotonergic 藥（SSRI/SNRI/MAOI）→ 血清素症候群**；本身有 MAOI 活性
- **孕哺**：**孕期避免**（羊膜腔注射曾致胎兒腸閉鎖 / 溶血等）；救命權衡，諮詢毒物中心；哺乳暫停

## 🔗 相關
- 疾病：[[Methemoglobinemia(變性血紅素血症)]]
- 致病藥物：dapsone、benzocaine / lidocaine、nitrates / nitrites、苯胺染料
- 替代 / 輔助：ascorbic acid（維生素 C）、換血、高壓氧；[[G6PD Deficiency(G6PD缺乏症)]]（禁忌前提）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 病人 benzocaine 噴喉後發紺、SpO₂ 85% 給氧無改善、抽血呈巧克力棕色。最恰當解毒？
- A. Naloxone
- B. Methylene blue 1–2 mg/kg IV（先確認非 G6PD 缺乏）
- C. 碳酸氫鈉
- D. 活性碳

> [!success]- 答案
> **B** — 典型變性血紅素血症，methylene blue 為解毒劑；給前需留意 G6PD 缺乏（禁忌）與 serotonergic 用藥。

**Q2.** 下列何者是 methylene blue 的絕對禁忌？
- A. 高血壓
- B. G6PD 缺乏症
- C. 糖尿病
- D. 氣喘

> [!success]- 答案
> **B** — G6PD 缺乏者缺 NADPH，methylene blue 無效且誘發嚴重溶血，屬絕對禁忌，改用換血 / 輸血 / 高壓氧 / 維生素 C。

**Q3.** 關於 methylene blue，何者正確？
- A. 劑量越大越好
- B. 累積過量（>7 mg/kg）反而會造成變性血紅素血症 / 溶血
- C. 可與 SSRI 安心併用
- D. 用一般脈搏血氧判斷療效

> [!success]- 答案
> **B** — 過量會反致 methemoglobinemia / 溶血；且與 serotonergic 藥有血清素症候群風險，療效應以 co-oximetry 監測。
