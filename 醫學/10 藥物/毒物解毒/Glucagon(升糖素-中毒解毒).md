# Glucagon（升糖素 / Glucagon）— 中毒解毒用

> [!abstract] 一句話：**β-blocker 與部分鈣離子阻斷劑（CCB）中毒的解毒劑**。透過自身受體活化 adenylyl cyclase → **繞過被阻斷的 β 受體直接升高 cAMP**，增強心肌收縮力與心率。**β-blocker 過量首選正性肌力 / 變時解毒劑**。副作用主要是嘔吐與高血糖。

> [!danger] 🚨 值班紅旗（給藥前先想）
> 1. **β-blocker 過量的關鍵解毒劑**：低血壓 + 心搏過緩對輸液 / atropine 無反應時，**glucagon bolus 3–10 mg IV** 是首選之一（繞過 β 受體）
> 2. **嘔吐風險高 → 顧氣道**：大劑量 glucagon 常引發嘔吐，意識抑制者**先保護氣道**再給，避免吸入
> 3. **反應好就接輸注**：bolus 有效後改**持續輸注**維持（時速約等於起效 bolus 劑量 per hour）
> 4. **CCB 中毒是多管齊下**：glucagon 效果不如對 β-blocker 穩定；CCB 中毒還要 **IV 鈣、高劑量胰島素 + 葡萄糖（HIET / hyperinsulinemia-euglycemia）、升壓劑、脂肪乳（lipid emulsion）** 綜合治療
> 5. **備妥止吐與血糖監測**：高血糖為預期效應（尤其 CCB 中毒本就高血糖），監測即可；大劑量 glucagon 需多支（每支多為 1 mg），**院內存量可能不足**，早通知藥局備藥

## 💊 適應症（中毒情境）
- **β-blocker（乙型阻斷劑）過量 [[Beta-blocker Overdose(乙型阻斷劑中毒)]]**：症狀性心搏過緩 / 低血壓 —— 首選正性肌力解毒
- **鈣離子阻斷劑 CCB 過量 [[Calcium Channel Blocker Overdose(鈣阻斷劑中毒)]]**（verapamil / diltiazem）—— 輔助角色（合併 IV 鈣、HIET、升壓）
- （另有作為**低血糖急救**與**內視鏡 / 影像時鬆弛平滑肌**之非解毒用途）

## 📏 劑量 / 用法（成人）
| 步驟 | 劑量 |
| --- | --- |
| Bolus（負荷） | **3–10 mg IV** 慢推（部分方案 5–10 mg），無反應可重複 |
| 持續輸注 | 起效後 **時速 ≈ 起效 bolus 劑量 per hour**（例：5 mg bolus 起效 → 5 mg/h） |
| 輔助 | 併止吐劑（防吸入）、CCB 中毒加 IV 鈣 + HIET + 升壓 |
- ⚠️ 大劑量需多支（每支常 1 mg），確認稀釋溶劑；**確切上限與輸注依院內方案 / 毒物中心**

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：**噁心嘔吐（劑量相關，吸入風險）**、高血糖、低血鉀（大劑量 / 輸注時）
- **嚴重**：嚴重嘔吐致吸入性肺炎、罕見過敏；嗜鉻細胞瘤 / 胰島素瘤者慎用（誘發血壓 / 血糖劇變）
- 高血糖多為**預期效應**，監測補正即可

## 🚫 禁忌症
- 對 glucagon 過敏
- **嗜鉻細胞瘤（pheochromocytoma）**：可誘發高血壓危象 → 慎用 / 避免
- **胰島素瘤（insulinoma）**：可致反跳性低血糖
- 相對：意識抑制未保護氣道者，先顧氣道再給（防嘔吐吸入）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：心率 / 血壓 / 心電圖、血糖（高血糖）、血鉀（低血鉀）、意識與氣道（嘔吐）
- **交互作用**：與 β 受體無關的路徑升 cAMP，故對 β-blocker 有效；與 IV 鈣 / HIET / 升壓劑併用治 CCB
- **孕哺**：救命情境使用，資料有限，諮詢毒物中心

## 🔗 相關
- 疾病：[[Beta-blocker Overdose(乙型阻斷劑中毒)]]、[[Calcium Channel Blocker Overdose(鈣阻斷劑中毒)]]
- 併用療法：IV calcium、**高劑量胰島素-葡萄糖療法 HIET**、[[Lipid Emulsion(脂肪乳劑)]]、升壓劑（norepinephrine）
- 相關被解毒藥：Propranolol、Metoprolol、Verapamil、Diltiazem
