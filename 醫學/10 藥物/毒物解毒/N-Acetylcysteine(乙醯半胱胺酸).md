# N-Acetylcysteine（乙醯半胱胺酸 / NAC）

> [!abstract] 一句話：**Acetaminophen（乙醯胺酚 / Paracetamol）過量的解毒劑**，補充 glutathione 前驅物中和肝毒性代謝物 **NAPQI**。**攝入後 8–10 小時內給藥幾乎 100% 保肝**，即使晚到（>24h 或已猛暴性肝衰竭）仍應給。是否需要治療依 **Rumack-Matthew nomogram**（4 小時血中濃度）判定。

> [!danger] 🚨 值班紅旗（處置前先想）
> 1. **黃金時間窗**：攝入後 **8–10 小時內**啟動 NAC → 肝毒性幾乎可完全預防；**別為等 nomogram 而拖過 8h** —— 若攝入 ≥8h 或濃度無法及時取得，**先給 NAC 再說**
> 2. **Nomogram 只適用「單次急性、已知時間、4h 以後」**：慢性 / 分次 / 不明時間中毒不能套 nomogram，改看肝功能 + acetaminophen level + 臨床，門檻放寬
> 3. **類過敏反應（anaphylactoid）**：IV NAC 常在第一段快速輸注時出現（潮紅、蕁麻疹、支氣管痙攣、低血壓）—— **多為輸速相關**，暫停 / 減速 + 抗組織胺後多可續用，非真過敏
> 4. **別漏掉共服藥**：合併劑（如含 acetaminophen 的複方感冒 / 止痛藥）、緩釋劑型；併服抗膽鹼 / 鴉片延遲胃排空 → 濃度高峰延後
> 5. 治療終點看**肝功能與 acetaminophen 已測不到**，不是「跑完固定時數就停」—— 猛暴性肝衰竭要延長並轉肝臟移植中心

## 💊 適應症（中毒情境）
- **Acetaminophen 急性過量 [[Acetaminophen Overdose(乙醯胺酚中毒)]]**：依 nomogram 在治療線以上，或攝入量 > 150 mg/kg（或 ≥ 7.5–10 g）
- 攝入時間不明 / 慢性過量 + 肝酵素上升或濃度可測
- 猛暴性肝衰竭（含非乙醯胺酚原因）—— 部分中心用於改善預後
- （另有作為化痰祛痰劑之呼吸道用途，非本篇解毒重點）

## 📏 劑量 / 用法（成人）
**IV 三段式 21 小時療程（最常用）**
| 段別 | 劑量 | 時間 |
| --- | --- | --- |
| Loading（第 1 段） | **150 mg/kg** 稀釋於 200 mL | 輸注 **60 分鐘**（過去 15min 已改為 1h 降低類過敏） |
| 第 2 段 | **50 mg/kg** 稀釋於 500 mL | 輸注 **4 小時** |
| 第 3 段 | **100 mg/kg** 稀釋於 1000 mL | 輸注 **16 小時** |
- **總量 300 mg/kg / 21 小時**；療程結束前複查 acetaminophen level + AST/ALT + INR，異常則**延長第 3 段**直到改善
- **口服（PO）替代方案**：140 mg/kg loading，之後 70 mg/kg Q4H × 17 次（72 小時），味道差易嘔吐
- ⚠️ **精確 mg/kg 換算與稀釋濃度依院內方案 / 毒物中心**，體重 > 100kg 常設劑量上限，**務必依院規核對**

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：噁心嘔吐（PO 尤甚）、輸注部位不適
- **類過敏反應（anaphylactoid，IV）**：潮紅、蕁麻疹、搔癢、血管性水腫、支氣管痙攣、低血壓 —— 多與第一段快速輸注相關，減速可續
- **嚴重**：罕見真過敏 / 嚴重支氣管痙攣（氣喘者較易）、極罕見低血鈉（大量稀釋液，兒童注意輸液量）

## 🚫 禁忌症
- 對 acetylcysteine 曾發生**真性嚴重過敏**（類過敏反應非絕對禁忌，可減速續用）
- 相對注意：嚴重氣喘活動期（支氣管痙攣風險）、需限水者（IV 稀釋量大）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：acetaminophen level（4h 起）、AST/ALT、INR/PT、bilirubin、腎功能、血糖、酸鹼（猛暴肝衰竭看 King's College criteria 評估移植）；輸注時監測過敏徵象與血壓
- **交互作用**：少臨床顯著藥物交互；活性碳若同時使用不顯著影響 PO NAC 療效（可分開給）
- **孕哺**：**孕婦 acetaminophen 過量務必照常給 NAC**（可過胎盤保護胎兒肝、母體救命優先）；哺乳可用

### 自我測驗（選擇題，答案摺疊）

**Q1.** 病人 6 小時前吞下大量 acetaminophen，血中濃度尚未回報。最正確的做法？
- A. 等 nomogram 結果再決定
- B. 抽 4h 後 level，但若可能延誤到 8h 應先啟動 NAC
- C. 只給活性碳觀察
- D. 直接排肝移植

> [!success]- 答案
> **B** — 8–10h 內給 NAC 幾乎完全保肝；不可為等濃度而拖過黃金窗，必要時先給 NAC。

**Q2.** IV NAC 輸注第一段時病人出現潮紅、蕁麻疹、輕度支氣管痙攣。處置？
- A. 判定真過敏，永久停用 NAC
- B. 這是輸速相關類過敏反應，暫停/減速 + 抗組織胺後多可續用
- C. 立即氣管插管
- D. 改口服活性碳取代

> [!success]- 答案
> **B** — anaphylactoid 反應多與快速輸注有關，減速加抗組織胺後通常可續用；acetaminophen 中毒不治肝毒性風險更高。

**Q3.** 關於 Rumack-Matthew nomogram，何者正確？
- A. 可用於任何時間任何劑型的中毒
- B. 只適用於已知時間的「單次急性」攝入，且從 4 小時後的濃度判讀
- C. 只看攝入劑量不看濃度
- D. 濃度在治療線以下仍一定要跑完療程

> [!success]- 答案
> **B** — Nomogram 僅適用單次急性、時間已知、4h 後的濃度；慢性 / 分次 / 不明時間須改依肝功能與臨床。
