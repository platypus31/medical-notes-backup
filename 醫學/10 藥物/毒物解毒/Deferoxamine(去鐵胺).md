# Deferoxamine（去鐵胺 / Desferrioxamine）

> [!abstract] 一句話：**急性鐵中毒的靜脈螯合解毒劑**，結合血中游離鐵形成 **ferrioxamine** 經腎排出（尿呈特徵性**「vin rosé 玫瑰紅 / 橘紅色」**）。用於嚴重鐵中毒（休克、代謝性酸中毒、高鐵濃度）。**輸速過快致低血壓**；長期 / 長時間輸注有 **ARDS 與 Yersinia 感染**風險。

> [!danger] 🚨 值班紅旗（給藥前先想）
> 1. **給藥適應症看嚴重度不只看濃度**：休克 / 反覆嘔吐 / 代謝性酸中毒 / 意識改變 / 顯著症狀，或 **serum iron > 500 µg/dL**（部分 >350 有症狀即治）→ 螯合；輕微無症狀 + 濃度低者觀察即可
> 2. **輸速過快 → 低血壓**：起始 **15 mg/kg/h IV**，出現低血壓要**減速**；不是越快越好
> 3. **尿變玫瑰紅（vin rosé）= 螯合到鐵**的傳統指標（不夠敏感 / 特異，臨床改善才是主要終點）
> 4. **停藥終點**：臨床好轉、酸中毒緩解、尿色轉正常、鐵濃度下降；**長時間輸注（>24h）增加 ARDS 與 Yersinia enterocolitica 敗血症風險** → 儘量縮短、達標即停
> 5. **腸道去污要靠全腸道灌洗**：鐵**不被活性碳吸附**（AC 無效）；大量攝入 / X 光見鐵錠 → **whole-bowel irrigation**

## 💊 適應症（中毒情境）
- **急性鐵中毒 [[Iron Poisoning(鐵中毒)]]**，符合下列任一：
  - 血行動力學不穩（休克、低血壓）、代謝性酸中毒、意識改變
  - 反覆嘔吐 / 腸胃道出血等顯著症狀
  - **serum iron > 500 µg/dL**（或 >350 µg/dL 且有症狀）
  - 腹部 X 光可見大量鐵錠 / 大量攝入
- （慢性鐵過載 / 輸血依賴地中海貧血亦用於除鐵，屬慢性用途，劑量 / 途徑不同）

## 📏 劑量 / 用法（成人）— 急性鐵中毒
| 步驟 | 劑量 |
| --- | --- |
| 起始輸注 | **15 mg/kg/h IV**（多數方案上限概念 ~6–8 g/24h，重症個別評估） |
| 低血壓時 | **減慢輸速**，補液支持 |
| 療程 | 通常 **≤ 24 小時**（長時間增 ARDS / Yersinia 風險），依臨床 / 尿色 / 鐵濃度 / 酸中毒滴定停藥 |
- **停藥終點**：臨床穩定、酸中毒緩解、尿色轉正常
- ⚠️ 上限與長時程限制**依院內方案 / 毒物中心**；孕婦鐵中毒仍應治療（母體致命優先）

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：**輸注相關低血壓（速度相關）**、潮紅、蕁麻疹、尿變玫瑰紅 / 橘紅、注射部位反應
- **嚴重**：
  - **急性呼吸窘迫症候群（ARDS）**（長時間輸注 > 24h）
  - **Yersinia enterocolitica 敗血症 / 感染**（鐵螯合物促其生長）
  - 過敏 / 過敏性反應（快速輸注）
  - 長期用：視 / 聽神經毒性、生長遲緩（兒童，慢性用途）
  - 腎功能不良者 ferrioxamine 蓄積

## 🚫 禁忌症
- 對 deferoxamine 過敏
- 嚴重腎功能不全 / 無尿（排泄依賴腎，需評估透析清除 ferrioxamine）
- 相對：輕微無症狀鐵暴露不需螯合

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：血壓 / 輸速（低血壓）、尿色（vin rosé）、serum iron 趨勢、動脈血氣 / 酸鹼、腎功能與尿量、呼吸（ARDS）、感染徵象（Yersinia）；腹部 X 光看殘留鐵錠
- **交互作用**：合併 **維生素 C** 於慢性除鐵可增動員（急性中毒非常規）；避免與其他腎毒性 / 低血壓藥疊加
- **孕哺**：**孕婦急性鐵中毒務必治療**（母體致命毒性優先於藥物風險）；哺乳資料有限，諮詢毒物中心

## 🔗 相關
- 疾病：[[Iron Poisoning(鐵中毒)]]、慢性鐵過載（地中海貧血除鐵）
- 去污：**全腸道灌洗 whole-bowel irrigation**（鐵不被活性碳吸附）
- 對照：[[Activated Charcoal(活性碳)]]（對鐵**無效**）、口服鐵螯合劑 deferasirox / deferiprone（慢性用途）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 幼童誤食大量鐵劑，休克、代謝性酸中毒、serum iron 700 µg/dL。最恰當解毒？
- A. 活性碳
- B. Deferoxamine 15 mg/kg/h IV（監測低血壓）+ 全腸道灌洗
- C. Naloxone
- D. Methylene blue

> [!success]- 答案
> **B** — 嚴重鐵中毒（休克 / 酸中毒 / iron >500）需 deferoxamine 螯合；鐵不被活性碳吸附，去污靠全腸道灌洗。

**Q2.** 使用 deferoxamine 時最需限制療程時間的原因是？
- A. 藥物太貴
- B. 長時間輸注（>24h）增加 ARDS 與 Yersinia 敗血症風險
- C. 會造成低血糖
- D. 會使尿變黑

> [!success]- 答案
> **B** — 超過 24 小時輸注與 ARDS 及 Yersinia enterocolitica 感染相關，應達標即停、儘量縮短。

**Q3.** 關於急性鐵中毒的腸胃道去污，何者正確？
- A. 活性碳效果最好
- B. 鐵不被活性碳吸附，大量攝入應以全腸道灌洗處理
- C. 一律洗胃即可
- D. 給碳酸氫鈉鹼化

> [!success]- 答案
> **B** — 鐵為金屬離子，活性碳無效；X 光見鐵錠 / 大量攝入時用 whole-bowel irrigation，解毒靠 deferoxamine。
