# Activated Charcoal（活性碳 / Activated Charcoal）

> [!abstract] 一句話：**多孔性吸附劑**，在腸胃道吸附尚未吸收的毒物阻斷其進入血液。**攝入後 1 小時內給藥效益最大**（越早越好）。關鍵安全問題：**意識不清 / 氣道未保護者禁用（吸入致命）**，且對**金屬（鐵/鋰）、醇類、腐蝕劑、碳氫化合物無效**。

> [!danger] 🚨 值班紅旗（給藥前先想）
> 1. **氣道未保護 = 絕對禁忌**：意識不清、抽搐、無嘔吐反射、預期會惡化者，給 AC 會**吸入 → 化學性肺炎 / ARDS 致命**；要給須先插管保護氣道
> 2. **時間敏感**：**1 小時內**效益最大，越晚越差；緩釋 / 抗膽鹼 / 大團塊藥物延遲胃排空者，晚給仍可能有益（個案評估）
> 3. **無效或不該給的毒物 —— 記憶法「PHAILS」**：**P**esticides(部分)/**H**ydrocarbons(碳氫)、**A**lcohols/**A**cids-alkali(腐蝕劑)、**I**ron(鐵)、**L**ithium(鋰)、**S**olvents/**S**alts(電解質)；金屬與離子小分子吸附差
> 4. **腐蝕劑（酸鹼）禁用**：無吸附效果又妨礙內視鏡評估、增加嘔吐穿孔風險
> 5. **腸阻塞 / 腸蠕動消失 / 腸穿孔**：禁用；**別配 sorbitol 反覆給**於腸阻塞者

## 💊 適應症（中毒情境）
- **可吸附毒物之急性口服過量 [[Acute Poisoning(急性中毒)]]**，尤其**攝入 < 1 小時**且氣道安全者
- 大多數藥物過量（TCA、抗癲癇、乙醯胺酚、水楊酸、benzodiazepine 等有機分子）
- **多劑活性碳（MDAC）**：用於特定藥物（carbamazepine、theophylline、phenobarbital、dapsone、quinine）促進「腸肝 / 腸道透析」清除

## 📏 劑量 / 用法（成人）
| 情境 | 劑量 |
| --- | --- |
| 單劑（一般過量） | **成人 25–100 g PO/NG**（常用 1 g/kg）以水調成漿 |
| 多劑活性碳 MDAC | 首劑後 **每 2–4 小時 12.5–25 g**（特定藥物） |
- **炭 : 毒物比 ≈ 10:1** 效果佳（故需足量）
- 給予途徑：清醒能配合則 PO；經 NG 管前**確認氣道保護 / 管路位置**
- ⚠️ 不常規加瀉劑（sorbitol）；反覆多劑禁配瀉劑（電解質紊亂 / 脫水）

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：噁心嘔吐、便秘、黑便、牙齒 / 口腔染黑
- **嚴重**：
  - **吸入 → 化學性 / 阻塞性肺炎、ARDS**（氣道未保護時最致命）
  - 腸阻塞、腸道石（charcoal bezoar）—— 尤其 MDAC + 腸蠕動差
  - 反覆給 + 瀉劑 → 脫水 / 電解質紊亂
  - 罕見角膜擦傷（嘔吐噴濺）

## 🚫 禁忌症
- **氣道未保護的意識障礙 / 抽搐 / 無嘔吐反射**（未插管前）
- **腐蝕劑（強酸強鹼）攝入**
- 腸阻塞、腸穿孔、腸蠕動消失、消化道出血未穩定
- 攝入物為**無吸附效益者**（鐵、鋰、醇類、碳氫化合物、電解質）— 給了無益反增風險

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：意識與氣道、腸鳴音 / 腹脹（MDAC）、嘔吐、血氧（防吸入）
- **交互作用**：會**吸附同時口服的其他解毒劑 / 藥物**（如口服 NAC 可分開給）；降低後續口服藥吸收
- **孕哺**：不經吸收全身，孕哺可用（以氣道安全為前提）

## 🔗 相關
- 情境：[[Acute Poisoning(急性中毒)]]、[[Acetaminophen Overdose(乙醯胺酚中毒)]]、TCA / 水楊酸 / 抗癲癇過量
- 對照 / 搭配：[[N-Acetylcysteine(乙醯半胱胺酸)]]、[[Sodium Bicarbonate(碳酸氫鈉-解毒)]]（水楊酸鹼化）、[[Deferoxamine(去鐵胺)]]（鐵中毒 —— AC 無效）
- 概念：全腸道灌洗 whole-bowel irrigation（鐵 / 鋰 / 緩釋 / 藏毒者）

### 自我測驗（選擇題，答案摺疊）

**Q1.** GCS 8 分、無嘔吐反射的過量病人尚未插管，家屬要求給活性碳。最正確？
- A. 立即經口灌入活性碳
- B. 氣道未保護為禁忌，需先插管保護氣道才可經 NG 給予
- C. 加瀉劑一起灌
- D. 給雙倍量加快吸附

> [!success]- 答案
> **B** — 意識不清 / 無嘔吐反射者給活性碳會吸入導致致命化學性肺炎；必須先插管保護氣道。

**Q2.** 下列何種中毒給活性碳「無效」？
- A. Amitriptyline（TCA）過量
- B. Carbamazepine 過量
- C. 急性鐵劑過量
- D. Acetaminophen 過量

> [!success]- 答案
> **C** — 鐵屬金屬離子，活性碳吸附差無效；鐵中毒去污靠全腸道灌洗，解毒靠 deferoxamine。

**Q3.** 關於活性碳給藥時機，何者最正確？
- A. 攝入後 24 小時內都一樣有效
- B. 攝入後 1 小時內效益最大，越晚越差
- C. 只能在攝入後 6 小時後給
- D. 時間與效益無關

> [!success]- 答案
> **B** — 1 小時內給效益最大；緩釋 / 抗膽鹼類延遲胃排空者晚給仍可能有益，須個案評估並確保氣道安全。
