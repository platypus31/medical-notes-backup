# Naloxone（納洛酮 / Narcan）

> [!abstract] 一句話：**純競爭性鴉片受體（μ）拮抗劑**，秒級逆轉鴉片過量造成的**呼吸抑制與昏迷**。半衰期短（~30–90min）**遠短於多數 opioid**，故救回後常會「再度昏睡」需重複給藥或持續輸注。慢性鴉片使用者要**小劑量滴定**避免誘發急性戒斷。

> [!danger] 🚨 值班紅旗（給藥前先想）
> 1. **目標是恢復呼吸，不是完全清醒**：慢性成癮者從**0.04–0.4 mg IV 小劑量滴定**，過猛完全逆轉會誘發急性戒斷（躁動、嘔吐吸入、交感風暴、甚至肺水腫）
> 2. **半衰期短於幾乎所有 opioid**：尤其 methadone（半衰期 15–60h）、緩釋劑、fentanyl patch —— 救回後 20–90 分鐘可能**再度呼吸抑制**，必須留觀 ≥ 4–6h（長效者更久）並準備**持續輸注**（時速約給予有效甦醒總劑量的 2/3 per hour）
> 3. **合成鴉片（fentanyl、carfentanil）可能需要大劑量**（累加數 mg 甚至更多）才逆轉；給 2–3 次無反應且高度懷疑時可再加大
> 4. **混合中毒別被誤導**：Naloxone 無效不排除鴉片，但要想到共服 benzodiazepine / 酒精 / TCA 等
> 5. **不是鎮靜劑逆轉劑**：對 benzo（用 flumazenil，且慎用）、酒精、GHB 無效

## 💊 適應症（中毒 / 急症情境）
- **鴉片類過量 [[Opioid Overdose(鴉片過量)]]**：呼吸 < 12/min、針狀瞳孔、意識抑制三聯徵 → 首選逆轉
- 不明原因昏迷 + 呼吸抑制之**經驗性診斷 / 治療**（配合 blood glucose、thiamine 一併考量）
- 新生兒鴉片暴露呼吸抑制（**現行不建議常規用於產婦成癮兒**，恐誘發新生兒戒斷 —— 依 NRP 以通氣為主）
- 院前 / 公衛：鼻噴劑（IN 4 mg）供旁人施打

## 📏 劑量 / 用法（成人）
| 情境 | 劑量 |
| --- | --- |
| 慢性成癮者 / 保留自主呼吸的滴定 | **0.04–0.4 mg IV**，每 2–3 分鐘視呼吸加倍向上滴定 |
| 呼吸停止 / 院前無 IV | **0.4–2 mg IV/IM/IN**，可每 2–3 分鐘重複 |
| 鼻噴劑（社區 / 家用） | **4 mg IN 單噴**，可另一鼻孔重複 |
| 合成鴉片高度懷疑無反應 | 逐步加大，累加至數 mg 仍可再評估 |
| 持續輸注（長效鴉片再昏睡） | 起始時速 ≈ **甦醒有效總劑量的 2/3 per hour**，依呼吸滴定 |
- **給藥後留觀**：短效鴉片 ≥ 4–6h；methadone / 緩釋 / patch 需更久（常 ≥ 24h、收住院監測）
- 途徑：IV 最快；無 IV 時 IM / IN / 骨內皆可

## ⚠️ 副作用（常見 / 嚴重）
- **急性戒斷症候群**（成癮者過度逆轉）：躁動、大汗、流淚流涕、噁心嘔吐（吸入風險）、腹痛、豎毛、心搏過速、高血壓
- **嚴重**：
  - **急性肺水腫**（尤其快速大劑量完全逆轉）
  - 交感神經風暴 → 心律不整、少數心肌缺血
  - 嘔吐 + 意識起伏 → 吸入性肺炎
  - 逆轉止痛作用 → 原本疼痛 / 術後病人劇痛

## 🚫 禁忌症
- 對 naloxone 過敏（極罕見）
- **相對**：慢性鴉片依賴者，需權衡「戒斷風險 vs 呼吸抑制致命」—— 以維持通氣為底線，不追求完全清醒

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：呼吸速率與血氧為首要終點、意識、心律、有無再昏睡（reload / 輸注）；備妥 BVM 通氣與氣道
- **交互作用**：逆轉所有 μ 促效劑（含 buprenorphine 部分促效 —— 逆轉較困難常需更大劑量）
- **孕哺**：孕婦鴉片過量仍應救命使用（母體缺氧對胎兒更危險），但可能誘發母胎急性戒斷 / 早產 —— 小劑量滴定，胎兒監測

## 🔗 相關
- 疾病 / 情境：[[Opioid Overdose(鴉片過量)]]、不明原因昏迷
- 對照：[[Flumazenil(氟馬西尼)]]（benzo 拮抗，慎用）、[[Naltrexone(納曲酮)]]（長效口服拮抗，用於戒治非急救）
- 相關被逆轉藥：Morphine、Fentanyl、Methadone、Oxycodone、Heroin

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名長期海洛因使用者過量，呼吸 6/min、針狀瞳孔、可觸及脈搏。最恰當的初始處置？
- A. 立即 IV push naloxone 2 mg 完全逆轉
- B. 先保持氣道與 BVM 通氣，naloxone 0.04–0.4 mg IV 小劑量滴定至呼吸恢復
- C. 只給活性碳
- D. 觀察等其自行代謝

> [!success]- 答案
> **B** — 目標是恢復呼吸而非完全清醒；成癮者過猛逆轉會誘發急性戒斷（嘔吐吸入、交感風暴、肺水腫）。先通氣再小劑量滴定。

**Q2.** 病人 methadone 過量經 naloxone 逆轉後清醒，留觀多久最安全？
- A. 30 分鐘可離院
- B. 1 小時
- C. 因 methadone 半衰期長（15–60h），需 ≥24h 監測並準備持續輸注
- D. 不需留觀

> [!success]- 答案
> **C** — Naloxone 半衰期遠短於 methadone，逆轉後極可能再度呼吸抑制，需長時間監測、常需持續輸注並收住院。

**Q3.** 關於 naloxone 對合成鴉片（fentanyl / carfentanil）過量，何者正確？
- A. 完全無效
- B. 可能需要遠大於一般的累加劑量才逆轉
- C. 只能用鼻噴劑
- D. 會加重呼吸抑制

> [!success]- 答案
> **B** — 高效價合成鴉片常需累加至數 mg 甚至更高劑量才逆轉，給 2–3 次無反應且高度懷疑時應繼續加大並維持通氣。
