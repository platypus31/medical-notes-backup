# Atropine + Pralidoxime（有機磷解毒組合 / Atropine + 2-PAM）

> [!abstract] 一句話：**有機磷農藥 / 神經毒劑中毒的雙藥解毒**。**Atropine** 拮抗蕈毒鹼（muscarinic）症狀（SLUDGE / 支氣管分泌），需**大劑量倍增滴定**至「肺部乾燥」為終點；**Pralidoxime（2-PAM）**重活化被磷酸化的乙醯膽鹼酯酶（AChE），改善**菸鹼（nicotinic）症狀（肌無力 / 呼吸肌麻痺）**，需在 AChE「老化 aging」前給。

> [!danger] 🚨 值班紅旗（處置前先想）
> 1. **Atropine 終點是「支氣管分泌乾燥、肺部清、氧合改善」，不是瞳孔或心率**；有機磷中毒可能需要**極大量 atropine（累加數十至上百 mg）**，**倍增滴定**（每 5 分鐘加倍）別因怕「阿托品化」而給太少 → 病人常死於呼吸道分泌與支氣管痙攣
> 2. **Pralidoxime 要早給、趁 AChE「老化」前**（有機磷酸化的 AChE 隨時間變不可逆 aging）；晚給效益差但仍可考慮 —— **必須與 atropine 併用**（單用 2-PAM 不夠，且氧合前先給 atropine）
> 3. **氨基甲酸酯（carbamate）中毒**：AChE 抑制**可自行水解回復**，通常**只需 atropine**，**pralidoxime 非必要**（部分意見認為對嚴重 carbaryl 可能有害 / 無益）—— 但混合農藥或不確定成分時仍可用
> 4. **自我防護**：處理有機磷 / 神經毒劑污染病人須**去污 + PPE**，避免醫護二次暴露
> 5. **呼吸衰竭是主要死因**：菸鹼型肌無力 + 分泌物 + 中樞抑制 → 早期插管與呼吸支持；**避免 succinylcholine**（被 AChE 抑制延長麻痺）

## 💊 適應症（中毒情境）
- **有機磷農藥中毒 [[Organophosphate Poisoning(有機磷中毒)]]**（parathion、malathion、dichlorvos 等）
- **神經性毒劑暴露**（sarin、VX 等，比照有機磷）
- **氨基甲酸酯（carbamate）中毒 [[Carbamate Poisoning(氨基甲酸酯中毒)]]** —— 以 atropine 為主
- 典型徵候：**SLUDGE**（Salivation 流涎 / Lacrimation 流淚 / Urination 尿失禁 / Defecation 腹瀉 / GI 腸絞痛 / Emesis 嘔吐）+ 支氣管漏（bronchorrhea）、瞳孔縮小、肌束顫動、呼吸衰竭

## 📏 劑量 / 用法（成人）
**Atropine（抗蕈毒鹼）**
| 步驟 | 劑量 |
| --- | --- |
| 起始 | **2–5 mg IV**（重症從高端） |
| 滴定 | **每 5 分鐘加倍**上一次劑量，直到**支氣管分泌乾燥、肺乾、氧合改善** |
| 維持 | 達終點後改**持續輸注**（如每小時給予累積阿托品化總量的 10–20%），依分泌物滴定 |

**Pralidoxime（2-PAM，重活化 AChE）**
| 步驟 | 劑量 |
| --- | --- |
| Loading | **1–2 g IV** 稀釋於 100 mL，輸注 **15–30 分鐘**（勿快推，快推致低血壓 / 心跳停） |
| 維持 | **持續輸注 ~8 mg/kg/h（約 500 mg/h）** 或 1 g Q4–6H，依症狀 / 病程 |
- ⚠️ Atropine 需求量可極大且個別差異大；**確切滴定終點與輸注公式依院內方案 / 毒物中心**
- Pralidoxime **必須在 atropine 之後 / 併用**，且輸注勿過快

## ⚠️ 副作用（常見 / 嚴重）
- **Atropine 過量（抗膽鹼）**：口乾、視力模糊、尿滯留、腸蠕動停、高體溫、譫妄、心搏過速 —— 但有機磷中毒需求量本就高，以「肺乾」為準
- **Pralidoxime**：快速輸注 → **低血壓、心搏過速 / 過緩、喉痙攣、肌僵硬、暫時視力模糊**、頭痛、噁心
- **嚴重**：呼吸衰竭本身（疾病）、心律不整、中間症候群（intermediate syndrome，數天後近端肌 / 呼吸肌無力）

## 🚫 禁忌症
- 對各藥過敏（罕見）
- **無真正絕對禁忌於救命情境**；atropine 相對注意心肌缺血 / 心搏過速，但有機磷中毒救命優先
- Pralidoxime 腎功能不良需減量（腎排泄）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：肺部聽診 / 分泌物（atropine 終點）、氧合與呼吸、心律、瞳孔、肌力（菸鹼症狀 / 中間症候群）、血漿或紅血球 AChE 活性（若可測）
- **交互作用**：**避免 succinylcholine**（AChE 抑制延長麻痺）；避免其他抗膽鹼 / 減弱腸蠕動藥疊加造成中毒難判讀
- **孕哺**：救命優先使用，胎兒監測；諮詢毒物中心

## 🔗 相關
- 疾病：[[Organophosphate Poisoning(有機磷中毒)]]、[[Carbamate Poisoning(氨基甲酸酯中毒)]]、神經性毒劑暴露
- 概念：SLUDGE 症候群、AChE aging、中間症候群 intermediate syndrome
- 對照：Atropine 亦用於心搏過緩（劑量與情境不同）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 農民噴藥後大量流涎、瞳孔縮小、肺部濕囉音滿佈、呼吸費力。Atropine 給藥的滴定終點是？
- A. 瞳孔放大
- B. 心率 >100
- C. 支氣管分泌乾燥、肺乾、氧合改善
- D. 病人完全清醒

> [!success]- 答案
> **C** — 有機磷中毒 atropine 以「肺乾 / 分泌物消失 / 氧合改善」為終點，需倍增滴定，常需極大劑量，不看瞳孔或心率。

**Q2.** 關於 pralidoxime（2-PAM），何者正確？
- A. 可單用取代 atropine
- B. 可快速 IV push 加快起效
- C. 重活化 AChE，需早給（AChE 老化前）並與 atropine 併用，輸注勿過快
- D. 主要改善蕈毒鹼症狀

> [!success]- 答案
> **C** — 2-PAM 重活化 AChE 改善菸鹼型肌無力，須在老化前早給、與 atropine 併用；快速輸注會致低血壓 / 心跳停。

**Q3.** 純氨基甲酸酯（carbamate）中毒的解毒重點是？
- A. 只用 pralidoxime
- B. 以 atropine 為主，pralidoxime 通常非必要（AChE 抑制可自行回復）
- C. 大量活性碳即可
- D. 使用 fomepizole

> [!success]- 答案
> **B** — carbamate 對 AChE 的抑制可自行水解回復，通常只需 atropine 抗蕈毒鹼支持，pralidoxime 非必要。
