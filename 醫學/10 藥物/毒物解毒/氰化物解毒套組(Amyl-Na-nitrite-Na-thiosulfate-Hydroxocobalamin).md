# 氰化物解毒套組（Amyl / Na nitrite / Na thiosulfate / Hydroxocobalamin）

> [!abstract] 一句話：**氰離子（CN⁻）抑制粒線體細胞色素 c 氧化酶（cytochrome c oxidase）阻斷有氧代謝 → 細胞無法用氧（histotoxic hypoxia）、乳酸中毒、儘管動脈血氧正常**。解毒兩路線：① 傳統套組「誘導高鐵血紅素抓 CN⁻ + 硫供體排出」；② 現代 **Hydroxocobalamin 直接結合 CN⁻**，安全、適合火場煙霧合併 CO 中毒。

> [!danger] 🚨 值班紅旗（給藥前先想）
> 1. **合併一氧化碳（CO）中毒（火場煙霧吸入最常見）時，禁用亞硝酸鹽類（Amyl / Sodium nitrite）** → 亞硝酸鹽誘導 methemoglobinemia，會**再降低已被 CO 佔據的攜氧能力**，兩者疊加可致命 → 此時**首選 Hydroxocobalamin**
> 2. **氰化物中毒的線索**：密閉火場煙霧吸入、不明原因**嚴重乳酸中毒（lactate 常 ≥ 8–10）**、心血管崩潰、動脈血氧看似正常但靜脈血氧異常偏高（組織用不到氧）
> 3. **不要等實驗室確認**：高度懷疑（火場 + 昏迷 + 高乳酸 + 低血壓）就經驗性給藥，氰化物致死極快
> 4. **Sodium thiosulfate 可與 Hydroxocobalamin 併用**加強清除；但 Hydroxocobalamin 與 thiosulfate **不要同一條 IV line 同時輸**（會反應），分開給
> 5. **亞硝酸鹽會造成低血壓**（血管擴張）→ 給 Sodium nitrite 時監測血壓

## 💊 適應症（中毒 / 急症情境）
- **急性氰化物中毒 [[Cyanide Poisoning(氰化物中毒)]]**：工業（電鍍、冶金）、火場燃燒含氮聚合物（羊毛、聚氨酯）釋放 HCN、苦杏仁苷、Nitroprusside 長期輸注副產物
- **火場煙霧吸入合併疑似氰化物 ± CO 中毒** → **Hydroxocobalamin 首選**（可安全用於未確認 CO 狀態者）
- **Nitroprusside 長時間高劑量輸注**引發之氰化物累積 → thiosulfate 預防／治療

## 📏 劑量 / 用法（成人 · 依院規 / 毒物中心指示）
| 藥物 | 途徑 / 角色 | 機轉 |
| --- | --- | --- |
| **Amyl nitrite** | **吸入**（安瓿捏破吸），IV 建立前的過渡 | 快速誘導部分 methemoglobinemia |
| **Sodium nitrite** | **IV** | 誘導 methemoglobinemia：Fe²⁺→Fe³⁺（高鐵血紅素）與 CN⁻ 結合成 cyanmethemoglobin，把 CN⁻ 從細胞色素氧化酶拉走 |
| **Sodium thiosulfate** | **IV** | 提供硫供體，經 **rhodanese 酶**將 CN⁻ 轉成毒性較低的 **thiocyanate** 經腎排出 |
| **Hydroxocobalamin (Cyanokit)** | **IV** | 直接與 CN⁻ 結合成無毒 **cyanocobalamin（維生素 B12）** 經尿排出，不影響攜氧 |
- 傳統套組順序：**Amyl nitrite 吸入（過渡）→ Sodium nitrite IV → Sodium thiosulfate IV**
- 各藥精確 mg / 劑量與可重複次數**依院規 / Cyanokit 仿單 / 毒物中心指示**（需查證，勿憑記憶）
- **CO 合併或狀態不明** → 跳過亞硝酸鹽，直接 **Hydroxocobalamin（± thiosulfate）**

## ⚠️ 副作用（常見 / 嚴重）
- **Amyl / Sodium nitrite**：
  - **低血壓**（血管擴張）、反射性心搏過速、頭痛、潮紅
  - **過量 methemoglobinemia**（本身也會缺氧）—— 需監測，嚴重時考慮 methylene blue（但氰化物情境下 methylene blue 使用需謹慎）
- **Sodium thiosulfate**：一般安全，快速輸注可噁心、潮紅、低血壓；長期產生的 thiocyanate 於腎衰竭者累積
- **Hydroxocobalamin**：**皮膚、黏膜、尿液暫時性紅色（洋紅色）**（無害但會干擾以顏色為基礎的檢驗，如血氧計 / 部分實驗室比色）、高血壓、少見過敏

## 🚫 禁忌症
- **亞硝酸鹽類（Amyl / Sodium nitrite）**：**合併 CO 中毒 / 攜氧能力已受損者**（火場煙霧）為相對禁忌 → 改用 Hydroxocobalamin
- 嚴重低血壓時給亞硝酸鹽須極謹慎
- Hydroxocobalamin 對其成分過敏（罕見）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：血壓（亞硝酸鹽低血壓）、乳酸趨勢（治療反應指標）、血氧與 methemoglobin 濃度（給亞硝酸鹽後）、意識與心律
- **檢驗干擾**：Hydroxocobalamin 的紅色會**干擾比色法檢驗**（co-oximetry、部分血液生化、尿液試紙）—— 抽血採檢盡量在給藥前
- **交互作用**：Hydroxocobalamin 與 Sodium thiosulfate、其他藥物**不共用同一 IV line**（會反應／沉澱）
- **孕哺**：氰化物致命，孕婦仍應救命解毒；Hydroxocobalamin 相對安全較受青睞

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名消防員從密閉火場救出，昏迷、低血壓、lactate 12、SpO2 顯示正常但無法確認 CO 狀態。最恰當的解毒劑是？
- A. Sodium nitrite IV
- B. Amyl nitrite 吸入
- C. Hydroxocobalamin
- D. 先誘導 methemoglobinemia 再說

> [!success]- 答案
> **C** — 火場煙霧常合併 CO 中毒，**亞硝酸鹽誘導的 methemoglobinemia 會再降低攜氧能力**與 CO 疊加致命；此情境首選 **Hydroxocobalamin**（可安全用於未確認 CO 狀態者）。

**Q2.** 關於 Sodium thiosulfate 在氰化物中毒的角色，何者正確？
- A. 誘導 methemoglobinemia 抓住 CN⁻
- B. 提供硫供體，經 rhodanese 酶把 CN⁻ 轉成 thiocyanate 排出
- C. 直接與 CN⁻ 結合成維生素 B12
- D. 逆轉一氧化碳結合

> [!success]- 答案
> **B** — Sodium thiosulfate 提供硫供體，透過 **rhodanese 酶**將 CN⁻ 轉成毒性較低的 **thiocyanate** 經腎排出；誘導 methemoglobinemia 是亞硝酸鹽的角色，直接結合成 B12 是 Hydroxocobalamin。

**Q3.** 氰化物中毒為什麼病人動脈血氧（SaO2）常正常卻嚴重缺氧？
- A. 血紅素被破壞
- B. 氰離子抑制細胞色素氧化酶，細胞無法利用氧（histotoxic hypoxia）
- C. 肺泡通氣不足
- D. 貧血

> [!success]- 答案
> **B** — 氰離子抑制粒線體 **細胞色素 c 氧化酶**，屬 histotoxic（組織中毒性）缺氧：氧仍在血中運送（SaO2 正常）但組織用不到，導致嚴重乳酸中毒。
