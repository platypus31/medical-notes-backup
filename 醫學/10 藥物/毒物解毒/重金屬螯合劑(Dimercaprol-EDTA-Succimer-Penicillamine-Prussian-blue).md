# 重金屬螯合劑（Dimercaprol / EDTA / Succimer / Penicillamine / Prussian Blue）

> [!abstract] 一句話：**螯合劑用多齒配位基「抓住」游離金屬離子形成穩定水溶性複合物 → 經腎（或腸道）排出**，降低組織毒性。核心觀念是**不同金屬配不同螯合劑**：鉛 → BAL＋EDTA / Succimer；砷汞 → BAL、DMSA；銅（Wilson）→ Penicillamine；銫（Cs-137）／鉈（Thallium）→ Prussian Blue。**選錯藥可能無效甚至更毒**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **鉛腦病（lead encephalopathy）順序不能顛倒**：**先給 Dimercaprol（BAL），數小時後再加 CaNa2EDTA**。若單獨先給 EDTA，會把周邊組織的鉛動員入血、**再分布進腦**惡化腦病
> 2. **EDTA 只能用「Calcium disodium EDTA（CaNa2EDTA）」**，**絕對不可誤植 disodium EDTA（Na2EDTA / edetate disodium）** → 會螯合血鈣造成**致命性低血鈣**與心律不整（經典致死給藥錯誤）
> 3. **Dimercaprol（BAL）禁用於鐵、鎘（cadmium）中毒** → 形成更毒的複合物；**G6PD 缺乏者**可致溶血
> 4. **Penicillamine 會消耗維生素 B6（吡哆醇）** → 需併補 B6；且可誘發**紅斑性狼瘡樣症候群**、骨髓抑制
> 5. **螯合本身有腎毒性**（尤其 EDTA） → 治療前後監測腎功能與尿量；確保水分充足促排出

## 💊 適應症（依中毒金屬選藥）
- **鉛中毒 [[Lead Poisoning(鉛中毒)]]**：
  - 重度／腦病 → **Dimercaprol（BAL）先給 → 再加 CaNa2EDTA**
  - 兒童輕中度、可口服 → **Succimer（DMSA）**（方便、不需住院靜脈給）
- **砷（arsenic）／汞（mercury）／金（gold）急性中毒**：**Dimercaprol（BAL）**；亞急性可用 Succimer
- **銅中毒／威爾森氏症 [[Wilson Disease(威爾森氏症)]]**：**Penicillamine**（第一線螯合，促尿銅排出）
- **銫（Cesium-137，放射性落塵）／鉈（Thallium）中毒**：**Prussian Blue（亞鐵氰化鐵）**
- Penicillamine 也可作**鉛／汞中毒的口服二線**選項

## 📏 劑量 / 用法（成人 · 依院規 / 毒物中心指示）
| 藥物 | 途徑 / 特性 | 主要用途 |
| --- | --- | --- |
| **Dimercaprol (BAL)** | **深部肌肉注射**（油性、含花生油賦形劑），劑量與間隔依中毒程度／院規 | 砷、汞、鉛腦病（配 EDTA） |
| **CaNa2EDTA** | IV 緩慢輸注／IM，劑量依院規 | 鉛中毒主力（腦病須先給 BAL） |
| **Succimer (DMSA)** | **口服** | 兒童輕中度鉛、砷汞亞急性 |
| **Penicillamine** | **口服**（空腹），**須併吡哆醇 B6** | Wilson disease、鉛/汞二線 |
| **Prussian Blue** | **口服**（不吸收，腸道作用） | 銫-137、鉈中毒 |
- 詳細 mg/kg 劑量與療程**依院規 / 榮總毒藥物防治諮詢中心指示**，勿憑記憶下 loading（需查證）
- 螯合前後**確保水分與尿量**，監測腎功能與電解質

## ⚠️ 副作用（常見 / 嚴重）
- **Dimercaprol (BAL)**：注射處疼痛（油性肌注）、高血壓／心搏過速、發燒、噁心、頭痛；**G6PD 缺乏者溶血**
- **CaNa2EDTA**：**腎毒性（急性腎小管壞死）**、低血鋅（長期）、注射處反應；**誤用 Na2EDTA → 致命低血鈣**
- **Succimer (DMSA)**：腸胃不適、皮疹、**可逆性肝功能（transaminase）上升**、金屬味、少見嗜中性球低下
- **Penicillamine**：**紅斑性狼瘡樣症候群**、蛋白尿／腎病症候群、**骨髓抑制**（白血球／血小板低下）、味覺改變、皮疹；長期消耗 B6
- **Prussian Blue**：便秘（可加纖維／輕瀉）、糞便藍色染色、低血鉀（大量長期）

## 🚫 禁忌症
- **Dimercaprol**：**鐵、鎘中毒禁用**（形成更毒複合物）；花生油過敏（賦形劑）；嚴重肝功能不全慎用；G6PD 缺乏慎用
- **CaNa2EDTA**：無尿／嚴重腎衰竭；**切勿使用 disodium EDTA**
- **Penicillamine**：penicillin 過敏史者慎用（交叉過敏疑慮）、既有骨髓抑制、嚴重腎功能不全
- **Prussian Blue**：無絕對禁忌，胃腸道阻塞時慎用

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：血中／尿中金屬濃度、腎功能（BUN/Cr）、尿量、CBC（Penicillamine 骨髓抑制）、肝功能（DMSA）、電解質（尤其鈣、鉀、鋅）
- **鉛腦病給藥順序**：**BAL 先，EDTA 後** —— 是本主題最高頻考點與臨床安全點
- **交互作用**：Penicillamine 與鐵劑／制酸劑併服會**降低吸收**（需隔開）；螯合劑會同時排出必需微量元素（鋅、鐵、銅）需留意補充
- **孕哺**：螯合治療孕婦需權衡（金屬毒性 vs 藥物風險），依毒物中心與產科共同評估；Penicillamine 孕期可致胎兒結締組織異常，非必要避免

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名幼童鉛中毒併發腦病（抽搐、意識改變）。下列螯合治療順序何者正確？
- A. 先給 CaNa2EDTA，再給 Dimercaprol
- B. 先給 Dimercaprol（BAL），再加 CaNa2EDTA
- C. 只給口服 Succimer
- D. 先給 Penicillamine

> [!success]- 答案
> **B** — 鉛腦病須**先給 BAL 再加 EDTA**；若單獨先給 EDTA 會使鉛再分布入腦惡化腦病。Succimer 用於輕中度、無腦病者。

**Q2.** 關於用 EDTA 治療鉛中毒，哪一點是最重要的用藥安全點？
- A. 必須空腹口服
- B. 必須使用 disodium EDTA
- C. 必須使用 Calcium disodium EDTA，誤用 disodium EDTA 會致命性低血鈣
- D. 可與 Dimercaprol 同一針筒混合

> [!success]- 答案
> **C** — 只能用 **CaNa2EDTA**；誤植 **disodium EDTA** 會螯合血鈣造成致命低血鈣與心律不整，是經典致死給藥錯誤。

**Q3.** 一名放射事故暴露到銫-137（Cesium-137）的病人，最專一的解毒劑是？
- A. Dimercaprol
- B. Penicillamine
- C. Prussian Blue（亞鐵氰化鐵）
- D. CaNa2EDTA

> [!success]- 答案
> **C** — **Prussian Blue** 在腸道與銫／鉈離子交換、阻斷腸肝循環再吸收，是銫-137 與鉈中毒的專一解毒劑，口服給予。
