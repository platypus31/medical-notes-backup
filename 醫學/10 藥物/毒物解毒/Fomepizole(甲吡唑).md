# Fomepizole（甲吡唑 / 4-Methylpyrazole）

> [!abstract] 一句話：**alcohol dehydrogenase（ADH）競爭性抑制劑**，阻斷 **甲醇 → 甲酸**、**乙二醇 → 乙醇酸 / 草酸** 的毒性代謝，是甲醇 / 乙二醇中毒的**首選解毒劑**（優於 ethanol，因劑量穩定、無醉態、不需頻繁監測血中濃度）。**趁毒醇尚未代謝成毒酸前及早給**。

> [!danger] 🚨 值班紅旗（處置前先想）
> 1. **搶在代謝前給**：fomepizole 阻斷的是「母醇 → 毒酸」，一旦已生成大量甲酸 / 草酸（高陰離子間隙代謝性酸中毒、視力受損、腎損傷）解毒效益下降 → **懷疑就早給**
> 2. **適應症門檻**：疑似 / 確診甲醇或乙二醇攝入，且**血中濃度 > 20 mg/dL**，或有代謝性酸中毒 + 高滲透壓間隙 + 相符病史 —— 別等濃度回報才給
> 3. **不能取代血液透析**：嚴重酸中毒、末器官損傷（視力 / 腎）、高濃度者仍需 **HD 清除毒醇與毒酸**；透析中 fomepizole 需**增量 / 縮短給藥間隔**
> 4. **合併治療**：甲醇 → 補 **folinic acid / folate**（促甲酸代謝）；乙二醇 → 補 **thiamine + pyridoxine**（導向無毒代謝）
> 5. **異丙醇（isopropanol）中毒不需 fomepizole**（其代謝物 acetone 無毒酸、不造成陰離子間隙酸中毒）

## 💊 適應症（中毒情境）
- **甲醇中毒 [[Methanol Poisoning(甲醇中毒)]]**（工業酒精 / 假酒 / 擋風玻璃清潔劑）—— 視神經毒性、失明
- **乙二醇中毒 [[Ethylene Glycol Poisoning(乙二醇中毒)]]**（防凍劑）—— 草酸鈣結晶致急性腎衰竭
- 高滲透壓間隙 + 高陰離子間隙代謝性酸中毒 + 相符病史之經驗性治療

## 📏 劑量 / 用法（成人）
| 步驟 | 劑量 |
| --- | --- |
| Loading（負荷劑量） | **15 mg/kg IV** 稀釋輸注 ≥ 30 分鐘 |
| 維持（前 48h） | **10 mg/kg IV Q12H × 4 劑** |
| 維持（48h 後） | **15 mg/kg IV Q12H**（因自我誘導代謝，48h 後需增量） |
| 透析中調整 | 給藥**間隔縮短 / 加量**（HD 會清除 fomepizole，依方案） |
- **停藥終點**：毒醇濃度 < 20 mg/dL 且酸中毒 / 症狀緩解
- ⚠️ 精確稀釋與透析期調整**依院內方案 / 毒物中心**

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：頭痛、噁心、頭暈、注射部位疼痛 / 靜脈炎、金屬味
- **嚴重**：少見；偶見肝酵素短暫上升、過敏 / 嗜酸性球增多、輕度心搏或血壓變化
- 整體**耐受性遠優於 ethanol 療法**（無醉態、無低血糖 / 呼吸抑制風險）

## 🚫 禁忌症
- 對 fomepizole 或 pyrazole 類過敏
- **異丙醇中毒不適用**（非禁忌但無效益）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：血中甲醇 / 乙二醇濃度、陰離子間隙、滲透壓間隙、動脈血氣（酸中毒）、腎功能、視力（甲醇）、電解質 / 鈣（乙二醇草酸鈣）
- **交互作用**：與 **ethanol 併用會延長彼此代謝**（不建議同時用兩種 ADH 抑制劑）；透析加速清除 fomepizole
- **孕哺**：資料有限；權衡母體致命毒性應救命使用，諮詢毒物中心

## 🔗 相關
- 疾病：[[Methanol Poisoning(甲醇中毒)]]、[[Ethylene Glycol Poisoning(乙二醇中毒)]]
- 對照 / 替代：**Ethanol**（ADH 抑制替代方案，需頻繁監測血中酒精、副作用多）、**血液透析 HD**（清除毒醇 / 毒酸）
- 輔助：folinic acid（甲醇）、thiamine + pyridoxine（乙二醇）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 喝下假酒後高陰離子間隙代謝性酸中毒 + 高滲透壓間隙 + 視力模糊，最恰當的解毒劑？
- A. 活性碳
- B. Fomepizole（並準備血液透析）
- C. Naloxone
- D. 碳酸氫鈉單獨即可

> [!success]- 答案
> **B** — 典型甲醇中毒，首選 fomepizole 阻斷 ADH，並依酸中毒 / 濃度 / 視力損傷準備血液透析，另補 folinic acid。

**Q2.** 關於 fomepizole 48 小時後劑量，何者正確？
- A. 停藥
- B. 減半
- C. 因自我誘導代謝需增量為 15 mg/kg Q12H
- D. 改口服

> [!success]- 答案
> **C** — Fomepizole 會誘導自身代謝，48h 後維持劑量需上調為 15 mg/kg Q12H。

**Q3.** 下列何者「不需」fomepizole？
- A. 乙二醇（防凍劑）中毒
- B. 甲醇（假酒）中毒
- C. 異丙醇（isopropanol）中毒
- D. 高濃度乙二醇合併腎損傷

> [!success]- 答案
> **C** — 異丙醇代謝為 acetone，不生成毒酸、不造成陰離子間隙酸中毒，不需 fomepizole（支持性治療 ± 透析即可）。
