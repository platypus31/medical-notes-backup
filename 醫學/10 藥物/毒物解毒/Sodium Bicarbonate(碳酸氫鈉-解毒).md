# Sodium Bicarbonate（碳酸氫鈉 / NaHCO₃）— 中毒解毒用

> [!abstract] 一句話：解毒兩大用途 —— ① **三環抗憂鬱劑（TCA）中毒**：血清鹼化 + 鈉負荷**逆轉 Na 通道阻斷造成的 QRS 增寬 / 低血壓 / 心律不整**（QRS > 100 ms 是給藥指標）；② **水楊酸（aspirin）中毒**：**鹼化血 / 尿促水楊酸排出、減少進入腦部**。目標依血 / 尿 pH 滴定。

> [!danger] 🚨 值班紅旗（給藥前先想）
> 1. **TCA 中毒看 QRS**：**QRS > 100 ms** 或右軸偏移（aVR 終末 R 波）→ 心臟毒性風險高 → 給 **1–2 mEq/kg IV bolus**，可重複至 QRS 變窄；目標血 pH **7.45–7.55**
> 2. **水楊酸中毒別漏鹼化尿**：目標**尿 pH 7.5–8**促離子捕捉排泄；**避免讓病人插管後過度低通氣**（TCA / 水楊酸插管後若 minute ventilation 不足 → 酸中毒惡化 → 水楊酸更易入腦、TCA 心毒性加劇，可致死）
> 3. **緊盯低血鉀**：鹼化把 K⁺ 打進細胞 → **低血鉀妨礙尿液鹼化**（水楊酸）並致心律不整 → **一定要補鉀**才能有效鹼化
> 4. **透析門檻**：嚴重水楊酸中毒（濃度極高、意識改變、肺水腫、腎衰竭、酸鹼難矯正）→ **血液透析**，碳酸氫鈉只是橋接
> 5. **不是萬用鹼**：注意鈉 / 容積負荷（心衰 / 肺水腫）、低血鈣（游離鈣下降）、代謝性鹼中毒過頭

## 💊 適應症（中毒情境）
- **三環抗憂鬱劑（TCA）中毒 [[TCA Overdose(三環抗憂鬱劑中毒)]]**：QRS 增寬（> 100 ms）、室性心律不整、低血壓
- **水楊酸 / 阿斯匹靈中毒 [[Salicylate Poisoning(水楊酸中毒)]]**：鹼化血清（減入腦）+ 鹼化尿液（促排泄）
- 其他 **Na 通道阻斷性心臟毒物**（部分抗心律不整 Ic、cocaine 相關 QRS 增寬、diphenhydramine 大量等）
- 嚴重代謝性酸中毒之部分情境（非本篇重點）

## 📏 劑量 / 用法（成人）
| 情境 | 劑量 |
| --- | --- |
| **TCA 心臟毒性** | **1–2 mEq/kg IV bolus**，QRS 仍寬可重複；之後可持續輸注維持血 pH 7.45–7.55 |
| **水楊酸鹼化輸注** | 常配 **150 mEq NaHCO₃ 加入 1 L D5W** 輸注，滴定**尿 pH 7.5–8**、血 pH ≤ 7.55 |
| 補鉀 | 輸注中**加鉀維持正常血鉀**（低鉀無法有效鹼化尿） |
- ⚠️ 濃度換算（8.4% 溶液 1 mL = 1 mEq）、輸注配方**依院內方案**；插管者務必維持足夠分鐘通氣避免酸中毒惡化

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：低血鉀、代謝性鹼中毒、鈉 / 容積負荷、注射外滲組織傷害（高滲）
- **嚴重**：
  - **容積過載 → 肺水腫**（水楊酸中毒本就易肺水腫）
  - 低血鈣（游離鈣下降）→ 手足搐搦 / 心律不整
  - 過度鹼化 → 血 pH 太高
  - 與含鈣輸液沉澱、與 catecholamine 不相容（分開管路）

## 🚫 禁忌症
- 代謝性 / 呼吸性鹼中毒已存在、嚴重低血鉀未矯正（先補鉀）
- 容積過載 / 肺水腫 / 嚴重心衰慎用
- 低血鈣未矯正、對鈉負荷不耐者

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：心電圖 / QRS 寬度（TCA）、動脈血氣與血 / 尿 pH、血鉀 / 血鈣 / 血鈉、水楊酸濃度趨勢、意識、呼吸（插管者確認足夠通氣）、輸出入 / 肺部（容積）
- **交互作用**：與含鈣輸液 / catecholamine 不相容（分開）；改變尿 pH 影響其他弱酸 / 弱鹼藥排泄
- **孕哺**：救命情境使用；孕婦水楊酸中毒對母胎皆危險，積極處置並諮詢毒物中心

### 自我測驗（選擇題，答案摺疊）

**Q1.** amitriptyline（TCA）過量，心電圖 QRS 130 ms、低血壓。首要藥物處置？
- A. 給 calcium gluconate
- B. Sodium bicarbonate 1–2 mEq/kg IV bolus，目標縮窄 QRS、血 pH 7.45–7.55
- C. 給 digoxin
- D. 只給活性碳觀察

> [!success]- 答案
> **B** — QRS > 100 ms 反映 Na 通道阻斷心臟毒性，碳酸氫鈉（血清鹼化 + 鈉負荷）為首選，bolus 後視 QRS 重複 / 輸注。

**Q2.** 水楊酸中毒鹼化尿液效果不佳，最常見的可矯正原因？
- A. 血鈉太高
- B. 低血鉀（未補鉀無法有效鹼化尿液）
- C. 血糖太低
- D. 體溫太低

> [!success]- 答案
> **B** — 低血鉀時腎小管保 K⁺ 排 H⁺，尿液無法鹼化；必須補鉀才能有效鹼化尿液促水楊酸排泄。

**Q3.** 關於水楊酸 / TCA 中毒病人插管，何者最正確？
- A. 可深度鎮靜並允許低通氣
- B. 必須維持足夠分鐘通氣，避免酸中毒惡化（水楊酸入腦 / TCA 心毒性加劇）
- C. 插管後即不需再驗血氣
- D. 插管後停用碳酸氫鈉

> [!success]- 答案
> **B** — 插管後若通氣不足使 pH 下降，水楊酸更易入腦、TCA 心毒性加劇，可致死；須維持過度通氣 / 足夠分鐘通氣並續鹼化。
