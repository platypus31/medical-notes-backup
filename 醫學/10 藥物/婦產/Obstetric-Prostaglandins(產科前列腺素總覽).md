# Obstetric Prostaglandins（產科前列腺素總覽）

> [!abstract] 一句話：**前列腺素**促進**子宮頸軟化成熟**與**子宮平滑肌收縮**，是產科引產、產後出血、流產的核心用藥。三大主角：**Misoprostol（PGE1，最便宜多用途）、Dinoprostone（PGE2，子宮頸成熟）、Carboprost（PGF2α，頑固產後出血）**。共同大忌是**子宮過度收縮**；Carboprost 另有**氣喘禁忌**（支氣管收縮）。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **子宮過度收縮（tachysystole）→ 胎兒窘迫、子宮破裂**：所有引產前列腺素通病；**前胎剖腹產 / 子宮有疤者引產禁用 Misoprostol/PGE2**（破裂風險高）
> 2. **Carboprost（PGF2α）→ 支氣管收縮 → 氣喘病人禁用**：頑固產後出血選藥前先問氣喘史；也會發燒、腹瀉、噁心、高血壓
> 3. **Misoprostol 劑量隨適應症天差地遠**：足月引產是**微克級（25 μg）**，早孕流產是**數百微克**——足月用錯高劑量 → 子宮破裂災難；牢記「越足月、劑量越小」
> 4. **給前列腺素引產後再接 Oxytocin 要間隔**（避免收縮劑疊加過度刺激），Dinoprostone 陰道錠出現過度收縮時要能**取出**
> 5. **Dinoprostone（PGE2）→ 低血壓、發燒**；青光眼/氣喘/心血管不穩慎用；用於引產需胎心監測

## 💊 適應症（治療哪些疾病）
| 藥（前列腺素類型） | 主要用途 |
| --- | --- |
| **Misoprostol（PGE1）** | 子宮頸成熟 / 引產、**藥物流產（配 mifepristone）**、**產後出血預防與治療（尤其資源有限、可舌下/直腸）**、不完全流產處理、胃潰瘍預防（原始適應症） |
| **Dinoprostone（PGE2）** | **子宮頸成熟 / 足月引產**（陰道栓劑/凝膠） |
| **Carboprost（PGF2α, Hemabate）** | **子宮收縮乏力所致頑固產後出血**（oxytocin 無效時的二線） |

- 相關疾病：[[Induction of Labor(引產)]]、[[Postpartum Hemorrhage(產後出血)]]、[[Abortion-Miscarriage(流產)]]

## 📏 劑量 / 用法（成人）
| 藥 / 用途 | 劑量（⚠️依院規） |
| --- | --- |
| Misoprostol 足月引產（子宮頸成熟） | **25 μg 陰道 q3–6h**（微克級！），或口服低劑量 |
| Misoprostol 藥物流產 | 配 mifepristone，**數百微克**（如 800 μg 陰道/舌下），劑量遠高於足月引產 |
| Misoprostol 產後出血 | **800–1000 μg 直腸/舌下**（oxytocin 缺乏時替代） |
| Dinoprostone 子宮頸成熟 | 陰道栓劑 **10 mg 緩釋** 或凝膠，過度收縮可取出 |
| Carboprost 產後出血 | **250 μg IM/子宮肌層 q15–90min，總量上限 2 mg（8 劑）** |
- **原則**：越接近足月、子宮越敏感 → 劑量越低、間隔越謹慎；有子宮疤痕避免用於引產

## ⚠️ 副作用（常見 / 嚴重）
- **共通**：**子宮過度收縮**、腹痛、噁心嘔吐、腹瀉、發燒/寒顫
- **Misoprostol**：腹瀉、發燒寒顫（產後出血劑量常見）、子宮過度刺激
- **Dinoprostone**：發燒、低血壓、噁心嘔吐、子宮過度刺激
- **Carboprost（PGF2α）**：**支氣管收縮（氣喘惡化）**、發燒、腹瀉、噁心、**高血壓**、臉潮紅
- **嚴重共通**：子宮破裂（尤其有疤子宮或高劑量足月）、胎兒窘迫

## 🚫 禁忌症
- **子宮過度收縮 / 胎兒窘迫已存在者**
- **前胎剖腹產或古典切口子宮疤痕引產** → 禁用 Misoprostol / PGE2（子宮破裂）
- **Carboprost：氣喘**（支氣管收縮）；活動性心肺肝腎疾病慎用
- Dinoprostone：對前列腺素過敏、青光眼/氣喘/心血管不穩慎用
- 不宜陰道生產之產科情況（前置胎盤、胎位不正等）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：引產用時**連續胎心監測 + 宮縮頻率**；Carboprost 用時監測呼吸（氣喘）、血壓、體溫
- **交互作用**：
  - **與 Oxytocin 序貫使用要間隔**（避免收縮劑加成過度刺激）
  - Misoprostol 產後出血配合其他子宮收縮劑（oxytocin、麥角）階梯使用
- **孕哺**：Misoprostol/Dinoprostone 用於引產或流產（依妊娠狀態）；Carboprost 用於產後；Misoprostol 也可短暫進入乳汁但一般單劑影響小

### 自我測驗（選擇題，答案摺疊）

**Q1.** 產後大出血 oxytocin 反應不佳，欲加 Carboprost。詢問病史時最關鍵要排除什麼？
- A. 糖尿病
- B. 氣喘
- C. 甲狀腺疾病
- D. 高血脂

> [!success]- 答案
> **B** — Carboprost（PGF2α）會造成支氣管收縮，氣喘病人禁用；此時改用其他子宮收縮劑（如 misoprostol）或處理其他出血原因。

**Q2.** 關於 Misoprostol 用於足月引產的劑量，下列何者正確？
- A. 與藥物流產劑量相同
- B. 使用微克級低劑量（如 25 μg），因足月子宮敏感
- C. 越足月劑量越大
- D. 不受孕週影響

> [!success]- 答案
> **B** — 足月引產用 25 μg 微克級低劑量；早孕流產才用數百微克。原則是越足月子宮越敏感、劑量越小，用錯高劑量會子宮破裂。

**Q3.** 一名前胎剖腹產的產婦需引產。關於前列腺素使用，下列何者最正確？
- A. 可放心使用高劑量 misoprostol
- B. 有疤子宮引產用 misoprostol/PGE2 有子宮破裂風險，應避免
- C. Carboprost 是首選引產藥
- D. 前列腺素對有疤子宮更安全

> [!success]- 答案
> **B** — 前胎剖腹產（有疤子宮）用前列腺素引產會顯著增加子宮破裂風險，應避免；引產方式需個別謹慎評估。
