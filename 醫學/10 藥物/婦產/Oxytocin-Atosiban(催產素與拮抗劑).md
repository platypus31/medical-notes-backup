# Oxytocin & Atosiban（催產素與拮抗劑）

> [!abstract] 一句話：**Oxytocin** 是下視丘合成、腦垂體後葉分泌的九胜肽荷爾蒙，作用於子宮平滑肌 oxytocin receptor（Gq → IP3 → Ca²⁺ 上升）→ **引產、增強產程、產後止血**（收縮子宮壓迫血管）；**Atosiban** 是其競爭性受體拮抗劑，**抑制子宮收縮用於安胎（tocolysis）**。兩者一推一拉，值班處理產房催生與早產安胎必懂。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **Oxytocin 過度刺激 → 子宮過度收縮（tachysystole / hyperstimulation）→ 胎兒窘迫（胎心減速）、子宮破裂（尤其前胎剖腹產、多產婦）**：一定要連續胎心監測（CTG），出現 tachysystole 立刻減速/停藥，必要時給 tocolytic（terbutaline）搶救
> 2. **Oxytocin 抗利尿作用（結構近似 ADH）→ 大劑量長時間 + 大量低張輸液 → 水中毒、低血鈉、抽搐、昏迷**：避免用大量無電解質葡萄糖水稀釋、監測入出量與血鈉
> 3. **Oxytocin 快速大量靜脈推注 → 低血壓、反射性心搏過速、心律不整**：產後止血要**稀釋慢滴**，不可 undiluted IV push（尤其剖腹產麻醉中）
> 4. **禁用於明顯胎位不正 / 頭骨盆不對稱 / 前置胎盤 / 臍帶脫垂 / 不宜陰道生產者**：硬催會災難性子宮破裂
> 5. **Atosiban 相對安全但別當萬靈丹**：僅延長妊娠數天（爭取安胎類固醇促胎肺成熟 + 轉院時間），不改善新生兒長期預後；孕週與適應症要符合（一般 24–33⁺⁶ 週）

## 💊 適應症（治療哪些疾病）
**Oxytocin（催產素）**
- **引產 [[Induction of Labor(引產)]]**（過期妊娠、破水未產、妊娠併發症需終止）
- **產程增強 augmentation**（子宮收縮乏力 uterine inertia）
- **產後出血 [[Postpartum Hemorrhage(產後出血)]] 預防與治療**（第三產程主動處置首選子宮收縮劑）
- **流產後 / 引產後子宮收縮不良**

**Atosiban（催產素拮抗劑）**
- **早產安胎 [[Preterm Labor(早產)]]**（歐洲常用一線 tocolytic，副作用少於 β-agonist）

## 📏 劑量 / 用法（成人）
| 藥 / 用途 | 劑量 |
| --- | --- |
| Oxytocin 引產 | **稀釋後 IV 滴注**，起始低劑量（如 1–2 mU/min）每 15–30 min 漸增，滴定到有效宮縮，⚠️依院規 protocol |
| Oxytocin 產後止血（陰道產） | **10 units IM**，或 **10–40 units 加入 500–1000 mL 輸液慢滴** |
| Oxytocin 剖腹產後 | 稀釋慢滴 / 慢推，**禁 undiluted bolus**（低血壓風險） |
| Atosiban 安胎 | 三階段：**6.75 mg IV bolus（1 min）→ 高劑量負荷滴注 → 維持滴注**，療程最多 48 小時，⚠️依院規 |
- **Oxytocin 半衰期短（~1–6 分鐘）**，故用連續滴注控制、停藥後迅速消退
- 引產時稀釋建議用含電解質溶液（降低水中毒風險），避免大量純葡萄糖水

## ⚠️ 副作用（常見 / 嚴重）
**Oxytocin**
- **常見**：子宮收縮痛、噁心嘔吐
- **嚴重**：
  - **子宮過度收縮 / 破裂**、胎兒窘迫（胎心減速、缺氧）
  - **水中毒、低血鈉**（抗利尿效應）→ 頭痛、抽搐、昏迷
  - **快速 IV → 低血壓、心搏過速、心律不整**
  - 產後子宮弛緩反彈（停藥後）

**Atosiban**
- **常見**：噁心、頭痛、頭暈、注射部位反應、心搏過速（遠少於 β-agonist）
- 相對副作用少、母體心肺負擔小（比 ritodrine/terbutaline 安全）

## 🚫 禁忌症
**Oxytocin**：頭骨盆不對稱、胎位不正不宜陰道產、前置胎盤、臍帶脫垂、子宮破裂高風險（多次剖腹產/古典切口）、胎兒窘迫需立即剖腹者、對藥過敏
**Atosiban**：孕週 < 24 或 > 33⁺⁶ 週、破水 > 30 週、胎兒窘迫/死胎、子癇前症或子癇需終止妊娠、產前出血需立即分娩、前置胎盤、胎盤早期剝離、對藥過敏

## 🔍 監測 / 交互作用 / 孕哺
- **Oxytocin 監測**：**連續胎心監測（CTG）+ 宮縮頻率/強度**（防 tachysystole）、血壓心律、長療程監測血鈉與入出量、產後失血量
- **交互作用**：
  - **前列腺素類（misoprostol/dinoprostone）** 併用增強子宮收縮 → 過度刺激風險（引產序貫使用要間隔）
  - **升壓劑 + 環丙烷麻醉** 併 oxytocin 可能嚴重高血壓
  - Atosiban 與其他 tocolytic 併用資料有限
- **孕哺**：Oxytocin 本就用於孕產期（產時）；Atosiban 專用於孕期安胎；哺乳期 oxytocin 促排乳

### 記憶卡（Spaced Repetition 相容 · `Q::A`）

Oxytocin 的作用機轉為何？::作用於子宮平滑肌 oxytocin receptor（Gq → IP3 → 細胞內 Ca²⁺ 上升）促進子宮收縮；也促乳腺肌上皮收縮排乳

Oxytocin 三大臨床用途？::①引產 ②產程增強（宮縮乏力）③產後出血預防與治療（第三產程首選子宮收縮劑）

Oxytocin 過度使用最危險的併發症是什麼？::子宮過度收縮（tachysystole）→ 胎兒窘迫（胎心減速缺氧）與子宮破裂，需連續 CTG 監測、出現即減量停藥

為什麼 Oxytocin 大劑量長時間 + 大量低張輸液會水中毒？::Oxytocin 結構近似 ADH（抗利尿激素），有抗利尿作用 → 水滯留、低血鈉 → 頭痛/抽搐/昏迷

產後止血給 Oxytocin 為何不能 undiluted IV push？::快速大量靜脈推注會造成低血壓、反射性心搏過速與心律不整，應稀釋慢滴（尤其剖腹麻醉中）

Oxytocin 的半衰期特性有何臨床意義？::半衰期極短（約 1–6 分鐘），故用連續滴注滴定控制，停藥後藥效迅速消退，利於調節

Atosiban 是什麼藥、用途為何？::催產素受體競爭性拮抗劑，用於早產安胎（tocolysis），抑制子宮收縮

Atosiban 相較 β-agonist（ritodrine/terbutaline）安胎的優點？::母體心肺副作用少很多（較少心搏過速、肺水腫、高血糖），耐受性佳

安胎（tocolysis）的真正治療目標是什麼？::多只延長妊娠數天，爭取施打促胎肺成熟類固醇（betamethasone）與轉院時間，不改善新生兒長期預後

Oxytocin 的絕對禁忌有哪些代表情況？::頭骨盆不對稱、胎位不正不宜陰道產、前置胎盤、臍帶脫垂、子宮破裂高風險（多次剖腹產）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 引產中的產婦出現宮縮過於頻繁（tachysystole）併胎心明顯減速，下列處置何者最適當？
- A. 加大 oxytocin 劑量加速生產
- B. 立即減量或停止 oxytocin 滴注，改善胎兒供氧，必要時給 tocolytic
- C. 給更多低張葡萄糖水
- D. 直接 IV push 更多 oxytocin

> [!success]- 答案
> **B** — 過度刺激造成胎兒窘迫時應立刻減量/停 oxytocin，改左側臥、給氧、補液，必要時 terbutaline 抑制宮縮搶救；加量或推注只會惡化。

**Q2.** 關於 Atosiban，下列敘述何者正確？
- A. 是子宮收縮劑，用於引產
- B. 是催產素受體拮抗劑，用於早產安胎，母體副作用較 β-agonist 少
- C. 可長期使用改善新生兒長期預後
- D. 適用於足月引產

> [!success]- 答案
> **B** — Atosiban 拮抗 oxytocin 受體、抑制宮縮安胎；相較 ritodrine/terbutaline 心肺副作用少，但僅短期延長妊娠爭取類固醇與轉院時機。

**Q3.** 一名產婦剖腹產後需給 oxytocin 止血，麻醉科最擔心的給藥方式相關風險是？
- A. 過敏
- B. 快速 undiluted IV bolus 造成低血壓與心律不整
- C. 口服吸收差
- D. 過期失效

> [!success]- 答案
> **B** — 剖腹麻醉中快速大量靜脈推注 oxytocin 會引發顯著低血壓、反射性心搏過速與心律不整，應稀釋後慢滴。
