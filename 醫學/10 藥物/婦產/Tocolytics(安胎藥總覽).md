# Tocolytics（安胎藥總覽）

> [!abstract] 一句話：**安胎藥**藉不同機轉抑制子宮平滑肌收縮，**目的是延長妊娠 ~48 小時**以爭取施打**促胎肺成熟類固醇（betamethasone）+ 硫酸鎂神經保護 + 轉院**，並不改善新生兒長期預後。五大類：**β2-agonist（Ritodrine/Terbutaline）、鈣通道阻斷劑（Nifedipine）、硫酸鎂（MgSO4）、前列腺素合成酶抑制劑（Indomethacin）、催產素拮抗劑（Atosiban）**，各有代表性毒性。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **β-agonist（Ritodrine/Terbutaline）→ 母體肺水腫（最致命）、心悸/心搏過速、心律不整、高血糖、低血鉀**：合併輸液過量或多胎、感染時肺水腫風險大增；限制輸液、監測心肺、避免與類固醇不當疊加
> 2. **MgSO4 中毒 → 深部肌腱反射消失（第一個消失）→ 呼吸抑制 → 心跳停止**：定期查膝反射、呼吸速率、尿量；中毒解毒劑 = **靜脈 calcium gluconate**
> 3. **Indomethacin（吲哚美辛）→ 胎兒動脈導管（ductus arteriosus）早閉 + 羊水過少（腎血流下降）**：**孕週 ≥ 32 週避免用**、療程 ≤ 48–72 小時，需監測導管與羊水
> 4. **Nifedipine 與 MgSO4 併用 → 加成低血壓、神經肌肉阻斷加重**：避免同時大量併用；Nifedipine 本身致母體低血壓、頭痛、臉潮紅
> 5. **安胎有明確禁忌時別硬安**：絨毛膜羊膜炎、胎兒窘迫/死胎、重度子癇前症/子癇、大出血、胎盤早剝——這些情況該終止妊娠而非安胎

## 💊 適應症（治療哪些疾病）
- **早產陣痛安胎 [[Preterm Labor(早產)]]**（一般 24–34 週、有規則宮縮併子宮頸變化）
- 短期延遲分娩以完成 **產前類固醇促胎肺成熟** 與 **母體轉送至有 NICU 醫院**
- MgSO4 另有**胎兒神經保護（< 32 週）**與**子癇前症/子癇抽搐預防**用途（見 [[Magnesium Sulfate(硫酸鎂)]]）

## 📏 劑量 / 用法（成人）— 五大類對照
| 類別 | 代表藥 | 機轉 | 標誌性毒性 |
| --- | --- | --- | --- |
| β2-agonist | **Ritodrine / Terbutaline** | β2 → cAMP 上升 → 子宮鬆弛 | **母體肺水腫**、心悸、心搏過速、高血糖、低血鉀 |
| 鈣通道阻斷劑 | **Nifedipine** | 阻 L 型 Ca²⁺ 通道 → 平滑肌鬆弛 | 母體低血壓、頭痛、臉潮紅；與 MgSO4 加成 |
| 硫酸鎂 | **MgSO4** | 拮抗鈣、穩定膜 | **反射消失→呼吸抑制→心停**（中毒） |
| 前列腺素合成抑制 | **Indomethacin** | 抑制 COX → 前列腺素下降 | **胎兒動脈導管早閉**、羊水過少（≥32 週禁） |
| 催產素拮抗劑 | **Atosiban** | 拮抗 oxytocin 受體 | 副作用少（見 [[Oxytocin-Atosiban(催產素與拮抗劑)]]） |
- 各藥劑量⚠️依院規 protocol，療程多以 **48 小時**為主（爭取類固醇窗口）
- **首選傾向**：母體心臟疾病避開 β-agonist；孕週 < 32 週偏好 Nifedipine 或 Atosiban；Indomethacin 限早孕週短療程

## ⚠️ 副作用（常見 / 嚴重）
- **β-agonist**：心悸、顫抖、噁心、**高血糖（糖尿病孕婦需注意）、低血鉀**；嚴重 **肺水腫、心律不整、心肌缺血**
- **Nifedipine**：頭痛、臉潮紅、心搏過速、**低血壓**、水腫
- **MgSO4**：臉潮紅、噁心、嗜睡、**反射減弱**；中毒 → **呼吸抑制、心停**
- **Indomethacin**：母體胃腸不適、血小板功能抑制；**胎兒動脈導管早閉、羊水過少、新生兒壞死性腸炎/腦室內出血風險（長療程）**

## 🚫 禁忌症（通用安胎禁忌 → 該終止妊娠）
- 絨毛膜羊膜炎、子宮內感染
- 胎兒窘迫、致命畸形、死胎
- 重度子癇前症 / 子癇
- 產前大出血、胎盤早期剝離、前置胎盤大量出血
- 個別禁忌：β-agonist（母體心臟病、控制不良糖尿病、甲亢）；Indomethacin（孕週 ≥ 32 週、寡羊水、母體出血傾向/腎損）；MgSO4（重症肌無力）

## 🔍 監測 / 交互作用 / 孕哺
- **β-agonist**：心電圖/心率、肺部聽診（肺水腫）、血糖血鉀、嚴格限制輸液量
- **MgSO4**：**膝反射、呼吸速率（≥12/min）、尿量（≥30 mL/hr）、血鎂濃度**；床邊備 **calcium gluconate** 解毒
- **Nifedipine**：血壓（低血壓）；**避免與 MgSO4 大量併用**（加成低血壓/神經肌肉阻斷）
- **Indomethacin**：胎兒超音波監測**動脈導管**與**羊水量**，≥32 週不用
- **孕哺**：全為孕期使用藥；產後排除各自禁忌後多可哺乳

### 記憶卡（Spaced Repetition 相容 · `Q::A`）

安胎藥（tocolytics）的真正治療目標是什麼？::延長妊娠約 48 小時，爭取施打促胎肺成熟類固醇（betamethasone）+ MgSO4 神經保護 + 轉院至有 NICU 的醫院；不改善長期預後

安胎藥五大類機轉各是什麼？::β2-agonist（cAMP 上升）、Nifedipine（阻 L 型鈣通道）、MgSO4（拮抗鈣穩定膜）、Indomethacin（抑 COX 降前列腺素）、Atosiban（拮抗 oxytocin 受體）

β-agonist（Ritodrine/Terbutaline）安胎最致命的母體副作用是什麼？::母體肺水腫；其他還有心悸、心搏過速、心律不整、高血糖、低血鉀

MgSO4 中毒的臨床順序與解毒劑？::深部肌腱反射消失（最早）→ 呼吸抑制 → 心跳停止；解毒劑為靜脈 calcium gluconate

Indomethacin 安胎為何孕週 ≥ 32 週要避免？::抑制前列腺素會造成胎兒動脈導管（ductus arteriosus）早閉，並致腎血流下降造成羊水過少

Nifedipine 與 MgSO4 併用要小心什麼？::兩者加成造成母體低血壓與神經肌肉阻斷加重，避免同時大量併用

哪些情況是安胎的禁忌（該終止妊娠而非安胎）？::絨毛膜羊膜炎、胎兒窘迫/死胎、重度子癇前症/子癇、產前大出血、胎盤早期剝離

MgSO4 安胎/神經保護時床邊要監測哪四項？::膝反射（有無消失）、呼吸速率（≥12/min）、尿量（≥30 mL/hr）、血鎂濃度

β-agonist 對糖尿病孕婦特別要注意什麼代謝副作用？::高血糖（β2 促糖解/糖質新生）與低血鉀，可能惡化血糖控制

孕週 < 32 週安胎較偏好哪些藥、避開哪類？::偏好 Nifedipine 或 Atosiban（副作用少）；母體心臟病避開 β-agonist；Indomethacin 僅限早孕週短療程

### 自我測驗（選擇題，答案摺疊）

**Q1.** 使用 Ritodrine 安胎的產婦出現呼吸急促、雙肺囉音、血氧下降。最可能的併發症是？
- A. 肺栓塞
- B. β-agonist 引起的母體肺水腫
- C. 氣胸
- D. 心肌梗塞

> [!success]- 答案
> **B** — β-agonist 安胎最需警惕母體肺水腫，尤其合併過量輸液、多胎、感染時；處置為停藥、限液、利尿、給氧。

**Q2.** 一名孕 34 週產婦安胎，超音波發現胎兒動脈導管收縮、羊水過少。最可能是哪個安胎藥造成？
- A. Nifedipine
- B. Terbutaline
- C. Indomethacin
- D. Atosiban

> [!success]- 答案
> **C** — Indomethacin 抑制前列腺素 → 胎兒動脈導管早閉 + 腎血流下降致羊水過少，故孕週 ≥ 32 週避免使用。

**Q3.** MgSO4 安胎中護理發現產婦膝反射消失、呼吸 10 次/分。首要處置為何？
- A. 加快 MgSO4 滴注
- B. 停 MgSO4 並靜脈給 calcium gluconate
- C. 給 naloxone
- D. 觀察即可

> [!success]- 答案
> **B** — 反射消失 + 呼吸抑制是鎂中毒徵象，應立即停藥並以 calcium gluconate 解毒，同時支持呼吸。
