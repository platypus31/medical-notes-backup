# Ergot Alkaloids（麥角生物鹼 — 產後出血）

> [!abstract] 一句話：**Ergonovine（麥角新鹼）與 Methylergonovine（甲基麥角新鹼）**是麥角生物鹼類子宮收縮劑，作用於子宮平滑肌 **α-腎上腺素能與 serotonin（5-HT2）受體 → 強力、持續、非節律性的子宮收縮**，用於**產後出血（oxytocin 之後的二線子宮收縮劑）**。因會**全身性血管收縮**，**高血壓與子癇前症病人禁用**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **高血壓 / 子癇前症 / 子癇 → 禁用**：麥角生物鹼收縮全身血管會使血壓飆高，可誘發高血壓危象、腦中風、抽搐
> 2. **冠狀動脈痙攣 → 心肌缺血/梗塞**：冠心病、缺血性心臟病病人禁用；也會造成周邊血管痙攣
> 3. **絕不可用於「生產前 / 胎兒未娩出」催生**：其強力持續（非節律）收縮會造成胎兒窘迫與子宮破裂，僅用於**胎兒與胎盤娩出後**止血
> 4. **靜脈給藥要極謹慎（一般肌肉注射）**：IV push 可致嚴重高血壓、噁心嘔吐、腦血管意外；緊急才稀釋慢推
> 5. **與強效 CYP3A4 抑制劑併用（如某些巨環內酯 erythromycin、蛋白酶抑制劑、azole）→ 麥角中毒（ergotism：嚴重血管痙攣、肢端缺血）**

## 💊 適應症（治療哪些疾病）
- **產後出血 [[Postpartum Hemorrhage(產後出血)]]（子宮收縮乏力）**：oxytocin 之後的**第二線子宮收縮劑**
- **產後 / 流產後子宮復舊不良、子宮弛緩出血**
- （胎盤娩出後的第三產程積極處置替代方案之一）

## 📏 劑量 / 用法（成人）
| 用途 | 劑量（⚠️依院規） |
| --- | --- |
| 產後出血（子宮收縮乏力） | **Methylergonovine 0.2 mg IM**，必要時 q2–4h 重複 |
| 頑固出血後續 | 口服 0.2 mg tid–qid 數天維持子宮張力 |
| 靜脈途徑 | **一般避免**；危及生命才極緩慢稀釋 IV 並嚴密監測血壓 |
- **給藥前必量血壓**：血壓偏高或子癇前症者不給
- 使用時機：**務必在胎兒＋胎盤娩出之後**

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：噁心、嘔吐、腹部絞痛、頭痛、血壓上升
- **嚴重**：
  - **高血壓危象、腦出血/中風**（尤其原本高血壓/子癇前症）
  - **冠狀動脈痙攣 → 心肌缺血/梗塞、心律不整**
  - 周邊血管痙攣、肢端缺血（**ergotism 麥角中毒**）
  - 罕見過敏、癲癇

## 🚫 禁忌症
- **高血壓、子癇前症、子癇**（絕對禁忌）
- **缺血性心臟病 / 冠狀動脈疾病**、周邊血管疾病
- **胎兒尚未娩出（生產前催生）**
- 敗血症、嚴重肝腎功能不全
- 與強效 CYP3A4 抑制劑併用（麥角中毒風險）
- 對麥角生物鹼過敏

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：**血壓（給藥前後）**、子宮張力與出血量、胸痛/末梢循環徵象
- **交互作用**：
  - **CYP3A4 抑制劑**（erythromycin/clarithromycin、蛋白酶抑制劑、azole 抗黴菌）→ 麥角血中濃度升高、麥角中毒
  - **血管收縮劑 / 擬交感神經藥**：加成高血壓
  - **triptans（偏頭痛藥）**：加成血管收縮
- **孕哺**：僅用於**產後**（孕期禁用，會誘發收縮）；哺乳期短期使用一般可接受，但可能抑制泌乳、少量進乳汁，建議謹慎與短療程

### 自我測驗（選擇題，答案摺疊）

**Q1.** 產後出血產婦，病史有子癇前症、血壓 165/105。下列子宮收縮劑何者應避免？
- A. Oxytocin
- B. Methylergonovine（麥角生物鹼）
- C. Misoprostol
- D. 子宮按摩

> [!success]- 答案
> **B** — 麥角生物鹼收縮全身血管，子癇前症/高血壓病人禁用（高血壓危象、腦中風風險）；此情境宜用 oxytocin、misoprostol 或處理其他出血原因。

**Q2.** 關於麥角生物鹼的使用時機，下列何者正確？
- A. 可用於產前引產加速產程
- B. 只能在胎兒與胎盤娩出後用於止血
- C. 用於安胎
- D. 首選靜脈快速推注

> [!success]- 答案
> **B** — 麥角生物鹼產生強力持續非節律收縮，用於胎兒與胎盤娩出後的產後止血；產前使用會致胎兒窘迫與子宮破裂。

**Q3.** 使用麥角生物鹼止血的產婦同時在服用 clarithromycin，最需擔心什麼？
- A. 藥效不足
- B. 麥角中毒（CYP3A4 抑制致血中濃度升高、血管痙攣肢端缺血）
- C. 低血壓
- D. 泌乳過多

> [!success]- 答案
> **B** — clarithromycin 抑制 CYP3A4 使麥角血中濃度升高，增加 ergotism（嚴重血管痙攣、肢端缺血）風險，應避免併用。
