# GLP-1 agonist（GLP-1 受體促效劑總覽 / incretin mimetic）

> [!abstract] 一句話：模擬腸泌素 **GLP-1**，**葡萄糖依賴性地促胰島素、抑升糖素、延緩胃排空、抑制食慾中樞** → 降血糖 + 顯著減重。因葡萄糖依賴故**單用低血糖風險低**，並有**心血管保護**（MACE↓）。代表藥 **Semaglutide（胰妥讚/Ozempic）/ Liraglutide（胰妥善/Victoza）/ Dulaglutide（易週糖/Trulicity）**，多為皮下針劑（semaglutide 有口服劑 Rybelsus）。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **急性胰臟炎風險**：出現持續性上腹痛放射至背、噁心嘔吐 → 立即停藥並查 lipase/amylase；有胰臟炎病史者慎用
> 2. **甲狀腺髓質癌（MTC）/ 多發性內分泌腫瘤第二型（MEN2）家族史或病史 → 禁用**（動物出現甲狀腺 C 細胞腫瘤，黑框警語）
> 3. **腸胃道副作用主導**：噁心、嘔吐、腹瀉、便秘極常見（劑量緩慢向上滴定可減輕）；嚴重嘔吐脫水可致 AKI
> 4. **顯著體重下降**：治療效益但衰弱/低體重/厭食老人需留意肌少症與營養不良
> 5. **延緩胃排空 → 麻醉吸入性肺炎風險**：手術/內視鏡/麻醉前需依最新指引暫停（可能殘留胃內容物）；併用口服藥吸收也受影響

## 💊 適應症（治療哪些疾病）
- **第二型糖尿病 [[Diabetes Mellitus(糖尿病)]]**（尤其合併肥胖或心血管高風險者優先）
- **肥胖 / 體重管理**（liraglutide Saxenda、semaglutide Wegovy 高劑量減重適應症）
- **動脈粥狀硬化性心血管疾病次級預防**（部分藥物證實降 MACE）
- 附帶：血壓小幅下降、脂肪肝改善

## 📏 劑量 / 用法（成人）
| 藥物 | 給法 / 頻率 | 備註 |
| --- | --- | --- |
| **Exenatide（穩爾糖 Byetta）** | SC BID 餐前；長效 Bydureon 每週一次 | 短效 |
| **Liraglutide（胰妥善 Victoza / Saxenda）** | SC QD，自 0.6 mg 每週滴定 | 減重版 Saxenda 至 3.0 mg |
| **Dulaglutide（易週糖 Trulicity）** | SC 每週一次 | 免滴定固定筆 |
| **Semaglutide（胰妥讚 Ozempic / Wegovy / 口服 Rybelsus）** | SC 每週一次；口服 QD 空腹 | 減重版 Wegovy 高劑量 |
- **緩慢向上滴定**是減少噁心嘔吐的關鍵，不要一次到目標劑量
- **腎功能**：多數不需大幅調整（非腎排泄為主），但脫水致 AKI 時留意；exenatide 重度腎不全避免
- 皮下注射部位輪替（腹部/大腿/上臂）

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：噁心、嘔吐、腹瀉、便秘、食慾下降、注射部位反應（**腸胃道為主，通常隨時間耐受**）
- **嚴重**：
  - **急性胰臟炎**
  - **甲狀腺髓質癌（動物訊號，人類黑框警語）**
  - 嚴重嘔吐腹瀉致脫水 / AKI
  - 膽結石 / 膽囊炎（快速減重相關）
  - 合併 SU/胰島素時低血糖

## 🚫 禁忌症
- **甲狀腺髓質癌（MTC）個人或家族病史、MEN2 → 絕對禁用**
- 對該藥或成分過敏
- 急性胰臟炎病史者慎用/避免
- 第一型糖尿病、糖尿病酮酸中毒（非適應症）
- 嚴重胃輕癱（延緩胃排空加重）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：血糖/HbA1c、體重、腸胃道耐受、胰臟炎症狀、腎功能（脫水時）、甲狀腺結節/頸部腫塊
- **交互作用**：
  - **延緩胃排空** → 影響口服藥吸收速率（如口服抗生素、避孕藥時機）
  - **Sulfonylurea / 胰島素**：合併時低血糖 → 常需減量
  - 麻醉/鎮靜內視鏡前需暫停（吸入性肺炎風險）
- **孕哺**：**孕期不建議使用**（減重與代謝改變不利胎兒，備孕前停藥，semaglutide 建議停藥後間隔一段時間再懷孕）；哺乳不建議

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名第二型糖尿病合併肥胖及冠心病病人，除 metformin 外欲加藥兼顧減重與心血管保護。最適合？
- A. Sulfonylurea
- B. GLP-1 受體促效劑（如 semaglutide）
- C. TZD
- D. 基礎胰島素

> [!success]- 答案
> **B** — GLP-1 促效劑兼具顯著減重與心血管（MACE）保護，適合肥胖 + ASCVD 的第二型糖尿病；SU 與胰島素會增重，TZD 亦增重且心衰禁忌。

**Q2.** 開立 GLP-1 促效劑前，下列哪項病史必須排除？
- A. 高血壓
- B. 甲狀腺髓質癌或 MEN2 家族史
- C. 氣喘
- D. 痛風

> [!success]- 答案
> **B** — 甲狀腺髓質癌/MEN2 為黑框禁忌（動物出現甲狀腺 C 細胞腫瘤）；此外亦須留意胰臟炎病史。
