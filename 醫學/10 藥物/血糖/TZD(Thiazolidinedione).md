# TZD（Thiazolidinedione / glitazone）

> [!abstract] 一句話：活化細胞核 **PPAR-γ** 受體 → 增加組織對胰島素敏感度（↑ GLUT-4、脂肪重分布）。**單用不會低血糖**，但**水鈉滯留 → 水腫 / 心衰惡化**是最重要的值班雷；另有骨折、膀胱癌（Pioglitazone）疑慮，藥效要數週至數月才顯現。

> [!danger] 🚨 值班紅旗
> 1. **水鈉滯留 → 水腫、體重增加、誘發/惡化心衰**：**NYHA III–IV 心衰禁用**；與 insulin 併用時水腫/心衰風險更高
> 2. 病人服 TZD 後**下肢水腫 + 喘 + 體重快速上升** → 想到 TZD 誘發的液體滯留 / 心衰，不是單純腎心問題
> 3. **骨折風險增加**（尤其停經後女性，遠端肢體骨折）
> 4. **Pioglitazone 與膀胱癌**風險訊號 → 有膀胱癌病史/血尿未查明者避用；Rosiglitazone 曾有心梗爭議

## 💊 適應症（治療哪些疾病）
- **第 2 型糖尿病**（[[Diabetes Mellitus(糖尿病)]]）：以**胰島素阻抗**為主者
- 常作為 metformin 無法達標時的合併用藥
- 對 NASH（非酒精性脂肪肝炎）有部分證據（Pioglitazone）

## 📏 劑量 / 用法（成人）
| 藥名 | 給法 |
| --- | --- |
| Pioglitazone（Actos） | 15–45 mg PO QD |
| Rosiglitazone（Avandia） | 4–8 mg PO QD（心血管爭議，臨床少用） |
- 不需依進食調整；**藥效慢（weeks–months）**才完全顯現
- 降 HbA1c 約 **1.0%**（一等）
- **肝代謝** → 活動性肝病 / ALT > 2.5×ULN 禁用；用藥期間監 LFT

## ⚠️ 副作用（常見 / 嚴重）
- **水腫、體重增加**（水鈉滯留）
- **心衰惡化 / 誘發**（最重要）
- **骨折**（停經後女性↑）
- 改善 lipid profile（Pioglitazone ↑ HDL、↓ TG）
- **嚴重 / 爭議**：Pioglitazone → 膀胱癌訊號；Rosiglitazone → 心梗爭議；肝毒性（罕見）

## 🚫 禁忌症
- **NYHA III–IV 心衰**（或任何有症狀心衰）
- 活動性肝病、ALT > 2.5× 正常上限
- 膀胱癌病史 / 不明原因血尿（Pioglitazone）
- 孕婦、哺乳
- 第 1 型糖尿病、DKA

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：體重 / 水腫 / 心衰徵象、肝功能（LFT）、血糖 HbA1c、（停經後女性）骨密度
- **交互作用**：
  - **與 insulin 併用 → 水腫/心衰風險加乘**
  - CYP2C8 影響（gemfibrozil ↑ pioglitazone 濃度、rifampin ↓）
- **孕哺**：不建議，改 insulin

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位糖尿病合併 NYHA III 心衰的病人，血糖控制不佳，下列何藥最應避免？
- A. Metformin（eGFR 正常）
- B. Pioglitazone
- C. Linagliptin
- D. Insulin

> [!success]- 答案
> **B** — TZD 造成水鈉滯留會惡化心衰，NYHA III–IV 為禁忌，應避免 pioglitazone。

**Q2.** 服用 pioglitazone 一個月後出現雙下肢水腫、體重增加 4 kg、活動性呼吸困難。最可能原因是？
- A. 低血糖
- B. TZD 引起的液體滯留/心衰惡化
- C. 乳酸中毒
- D. 藥物過敏

> [!success]- 答案
> **B** — TZD 的水鈉滯留可致水腫、體重快速上升與心衰惡化，出現此三聯徵應停藥並評估心臟。

**Q3.** 關於 TZD 的敘述何者正確？
- A. 單用常低血糖
- B. 屬胰島素增敏劑，單用不低血糖，主要雷是水腫/心衰與骨折
- C. 藥效數小時內出現
- D. 心衰病人首選

> [!success]- 答案
> **B** — TZD 為 PPAR-γ 促效增敏劑，單用不低血糖，主要副作用為水腫/心衰、骨折，藥效需數週至數月。
