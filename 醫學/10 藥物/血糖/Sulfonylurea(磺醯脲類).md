# Sulfonylurea（磺醯脲類 / SU）

> [!abstract] 一句話：抑制胰臟 β 細胞的 **ATP-sensitive K⁺ channel** → 去極化 → 促進胰島素分泌（**不依賴血糖**）。降血糖強效（一等），但**最重要的值班雷是低血糖**，尤其長效型（Glibenclamide）、老人與肝腎功能不佳者，且低血糖可**遷延復發**需住院觀察。

> [!danger] 🚨 值班紅旗
> 1. **低血糖（hypoglycemia）**是最重要副作用：因促泌作用**不依賴血糖濃度**，禁食 / 進食少 / 腎功能差時特別危險
> 2. **SU 引起的低血糖會遷延、復發**：半衰期長（尤其 Glibenclamide 有 active metabolite），一次 D50 拉起來後可能再度掉下去 → **需持續 glucose 輸注 + 住院監測數十小時**，不能給一次就放走
> 3. **老人 / 腎功能不佳優先避開長效 SU**（Glibenclamide 風險最高）；改用短效（Glipizide/Gliclazide）或其他機轉
> 4. 低血糖時**避免用純升糖素（glucagon）當唯一解**（會再刺激胰島素）→ 以葡萄糖為主

## 💊 適應症（治療哪些疾病）
- **第 2 型糖尿病**（[[Diabetes Mellitus(糖尿病)]]）：Metformin 之外的合併或替代用藥
- 適合 **β 細胞仍有分泌功能**、非肥胖、以飯前血糖偏高為主者
- 費用低，健保廣泛使用

## 📏 劑量 / 用法（成人，第二代常用）
| 藥名 | 特性 / 給法 |
| --- | --- |
| Glimepiride（Amaryl） | 最長效，QD；1–4 mg（max 8 mg） |
| Glibenclamide（Glyburide, Euglucon） | 長效、**低血糖風險最高**（active metabolite），老人/腎差避免 |
| Gliclazide（Diamicron） | 中長效 QD–BID，低血糖風險相對低 |
| Glipizide（Minidiab） | 中短效 QD–BID，腎差者較安全 |
- **餐前 30 分鐘**服用（配合進食避免低血糖）
- 降 HbA1c 約 **1.0–1.5%**（強效一等），主要降飯前血糖
- **肝代謝、腎排泄** → 肝腎差者用短效、減量

## ⚠️ 副作用（常見 / 嚴重）
- **低血糖（最重要，可致命/遷延）**
- **體重增加**
- 少見：皮疹、腸胃不適、膽汁鬱積型肝損、**低血鈉（SIADH，尤 Chlorpropamide 舊藥）**
- 磺胺（sulfa）過敏者需留意交叉過敏可能

## 🚫 禁忌症
- 第 1 型糖尿病、DKA / HHS（需 insulin）
- 嚴重肝腎功能不全（尤其長效型）
- 已知磺醯脲/磺胺嚴重過敏
- 孕婦、哺乳（一般改 insulin）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：血糖 / HbA1c、低血糖症狀、體重、肝腎功能
- **交互作用**：
  - **加強低血糖**：酒精、其他降糖藥、β-blocker（並遮蔽警覺）、水楊酸、sulfonamide、fluconazole（抑 CYP2C9）、warfarin
  - **削弱降糖**：類固醇、thiazide、利尿劑
- **孕哺**：不建議，改 insulin

## 🔗 相關
- 疾病：[[Diabetes Mellitus(糖尿病)]]、[[Hypoglycemia(低血糖)]]
- 同機轉（促泌）：[[Glinide(格列奈類)]]（作用位點相同但短效）
- 同類對照：[[Biguanide(雙胍類)]]、[[DPP-4 inhibitor(DPP-4抑制劑)]]、[[TZD(Thiazolidinedione)]]、[[α-glucosidase inhibitor(α-葡萄糖苷酶抑制劑)]]
- 對照：[[Insulin(胰島素)]]

### 自我測驗（選擇題，答案摺疊）

**Q1.** 80 歲糖尿病患服用 glibenclamide，因胃口差沒吃早餐後意識不清，血糖 35。給了 1 amp D50 後清醒，接下來最正確的處置是？
- A. 血糖正常即可返家
- B. 住院持續 glucose 輸注監測，因 SU 低血糖會遷延復發
- C. 立即再加一顆 glibenclamide
- D. 給 glucagon 即可

> [!success]- 答案
> **B** — Glibenclamide 為長效且有 active metabolite，SU 低血糖會**遷延復發**，一次 D50 拉起後可能再掉，需住院持續葡萄糖輸注與數十小時監測，不可給一次就放走。

**Q2.** 下列口服降血糖藥中，哪一類「單獨使用就可能造成低血糖」？
- A. Metformin
- B. Sulfonylurea
- C. DPP-4 inhibitor
- D. α-glucosidase inhibitor

> [!success]- 答案
> **B** — Sulfonylurea 屬胰島素促泌劑，作用不依賴血糖，單用即可低血糖；Metformin / DPP-4 / α-glucosidase / TZD 單用一般不會低血糖。

**Q3.** 為 78 歲、eGFR 35 的糖尿病患選擇 SU，較合適的選擇是？
- A. Glibenclamide
- B. Glipizide 或 Gliclazide（短效、腎差較安全）
- C. 劑量加倍的 Glimepiride
- D. 任何一種皆可

> [!success]- 答案
> **B** — 老人與腎功能不佳者應避開長效、有 active metabolite 的 glibenclamide，改用短效的 glipizide 或 gliclazide 以降低遷延性低血糖風險。
