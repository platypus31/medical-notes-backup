# Rifampin（利福平 / Rifampicin）

> [!abstract] 一句話：抑制細菌 **DNA-dependent RNA polymerase → 阻斷 mRNA 轉錄**，是抗結核 **RIPE 四藥（R）** 的骨幹殺菌藥。三大臨床標記：**紅橙色體液**、**超強 CYP450 誘導**、**肝毒性**。

> [!danger] 🚨 值班紅旗
> 1. **紅橙色體液**（尿、汗、淚、唾液）→ 會**永久染色隱形眼鏡**；無害但**一定要事先衛教**，否則病人嚇到自行停藥
> 2. **最強效的 CYP450 誘導劑之一** → 加速代謝、**降低大量藥物濃度**：口服避孕藥（**避孕失敗**！需改屏障法）、warfarin、DOAC、HIV 蛋白酶抑制劑、azole 抗黴菌、類固醇、methadone、statin、某些降血糖/免疫抑制劑
> 3. **肝毒性**（與 [[Isoniazid(異菸鹼醯胺)]] 併用加重）→ 監測 LFT，黃疸/噁心/腹痛要警覺
> 4. **間歇給藥可致 flu-like syndrome**（發燒、肌痛、寒顫）、少見血小板低下 / 溶血 / 急性腎損傷
> 5. RIPE 中的搭配：R(ifampin) + I(soniazid) + P(yrazinamide) + E(thambutol)

## 💊 適應症（治療哪些疾病）
- **結核病（TB）**：RIPE 一線骨幹（活動性 TB + 潛伏 TB 之一）
- **腦膜炎雙球菌（Neisseria meningitidis）接觸後預防性投藥**
- **嚴重 / 人工植入物相關葡萄球菌感染**（與其他抗生素併用，尤其生物膜 / prosthetic joint、心內膜炎）
- **痲瘋（leprosy）**（multidrug therapy 之一）
- 部分 MRSA 深部感染輔助

## 📏 劑量 / 用法（成人）
| 情境 | 劑量 |
| --- | --- |
| **活動性 TB** | **10 mg/kg/day（常 600 mg）PO QD，空腹**（飯前 1h 或飯後 2h） |
| 腦膜炎球菌接觸預防 | 600 mg PO Q12h ×2 天 |
| 潛伏 TB（4R 方案） | 600 mg PO QD ×4 個月 |
- **空腹服用吸收最佳**
- 肝功能不全需調整 / 密切監測；腎功能不全一般不需大幅調整（肝代謝為主）

## ⚠️ 副作用（常見 / 嚴重）
- **紅橙色體液 / 分泌物**（尿汗淚，染隱形眼鏡）——無害但需衛教
- **肝毒性**（轉胺酶升高、膽汁鬱積性黃疸；與 INH 併用加重）
- **腸胃道**：噁心、嘔吐、腹痛、厭食
- **間歇給藥相關免疫反應**：flu-like syndrome、**血小板減少**、溶血性貧血、急性間質性腎炎 / 腎衰
- 皮疹、搔癢

## 🚫 禁忌症
- 對 rifamycin 類過敏
- 嚴重活動性肝病 / 顯著肝功能異常
- **併用某些 HIV 蛋白酶抑制劑**（濃度被大幅降低，需改 rifabutin）
- 相對：與治療窗窄且被 CYP 代謝的關鍵藥併用（需重新評估劑量或換藥）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：基線與追蹤 **LFT**、CBC（血小板）、腎功能、TB 治療反應
- **交互作用（超強 CYP450 / P-gp 誘導 → 降低下列濃度）**：
  - **口服避孕藥 → 避孕失效**（改屏障避孕）
  - warfarin / DOAC（抗凝失效）、類固醇、azole、HIV PI、methadone、statin、某些降糖與免疫抑制劑、digoxin
- **孕哺**：TB 治療利大於弊，孕期可用（一線）；哺乳可用
- 記憶：**"Rifampin ramps up（誘導）→ 別人濃度掉下來"**

## 🔗 相關
- 同方案（RIPE）：[[Isoniazid(異菸鹼醯胺)]]、[[Pyrazinamide(吡嗪醯胺)]]、[[Ethambutol(乙胺丁醇)]]
- 疾病：[[Tuberculosis(結核病)]]、[[Latent TB(潛伏結核)]]、[[Meningococcal Meningitis(腦膜炎球菌腦膜炎)]]
- 同類：[[Rifabutin(利福布汀)]]（CYP 誘導較弱，HIV 病人替代）
- 對照：Isoniazid（周邊神經病變需補 B6）vs Rifampin（紅色體液、CYP 誘導）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 開始 RIPE 治療的女性病人，用藥衛教中「與避孕」相關最重要的一句是？
- A. 口服避孕藥可照常使用
- B. Rifampin 會使口服避孕失效，需改用屏障避孕
- C. 停經前不用擔心
- D. 增加避孕藥劑量即可

> [!success]- 答案
> **B** — Rifampin 是**超強 CYP450 誘導劑**，加速口服避孕藥代謝造成**避孕失敗**，必須衛教改用**屏障避孕**（保險套）等非荷爾蒙方式。

**Q2.** 病人服用 rifampin 後尿液變橘紅色，很緊張。正確的處置是？
- A. 立即停藥並驗尿
- B. 這是 rifampin 已知且無害的體液變色，事先衛教即可繼續用藥
- C. 代表急性腎衰竭
- D. 代表溶血

> [!success]- 答案
> **B** — Rifampin 造成**紅橙色體液（尿汗淚）是無害的預期反應**，只需事前衛教（並提醒會染隱形眼鏡），不需停藥。

**Q3.** 一位吃 warfarin 且 INR 穩定的病人開始服用 rifampin，兩週後 INR 明顯下降。最可能原因？
- A. Rifampin 升高 warfarin 濃度
- B. Rifampin 誘導 CYP450 加速 warfarin 代謝，使抗凝效果下降
- C. 病人自行停 warfarin
- D. 與 rifampin 無關

> [!success]- 答案
> **B** — Rifampin **誘導 CYP450（2C9 等）加速 warfarin 代謝**，使血中濃度與 INR 下降、抗凝失效，需增加 warfarin 劑量並密切監測 INR（停 rifampin 後又要反向調整）。
