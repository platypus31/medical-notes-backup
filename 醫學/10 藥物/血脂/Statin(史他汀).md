# Statin（史他汀類）

> [!abstract] 一句話：抑制膽固醇合成限速酶 **HMG-CoA reductase** → 肝內膽固醇↓ → 上調肝 **LDL receptor** → 清除血中 LDL。是降 LDL 最強、**心血管事件二級/一級預防的基石**（益處不只降脂，還有 pleiotropic/斑塊穩定效應）。值班兩大警覺：**橫紋肌溶解**與**肝酵素上升**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **Myopathy / 橫紋肌溶解**：肌肉痛 + **CK 大幅上升** ± 深色尿/急性腎損；風險因子＝高劑量、腎功能差、甲狀腺低下、併用 **fibrate（尤其 gemfibrozil）** 或 **CYP3A4 抑制劑**
> 2. **肝酵素上升**：baseline 驗 LFT，症狀性/明顯上升（>3× ULN）才停；無症狀輕度上升不需停
> 3. **孕婦禁用**（膽固醇為胎兒發育所需）→ 準備懷孕/懷孕停藥
> 4. **CYP3A4 交互作用**：simvastatin/lovastatin/atorvastatin 經 CYP3A4，與 clarithromycin、azole、amiodarone、葡萄柚汁併用 → statin 濃度↑肌病；**rosuvastatin/pravastatin/pitavastatin 較少 CYP 依賴**

## 💊 適應症（治療哪些疾病）
- **動脈粥狀硬化心血管疾病（ASCVD）一級/二級預防**：[[Myocardial Infarction(心肌梗塞)]]、[[Ischemic Stroke(缺血性中風)]]、[[Acute Coronary Syndrome(急性冠心症)]] 後
- **高膽固醇血症 / 家族性高膽固醇血症（FH）**
- **糖尿病病人的 ASCVD 風險降低**
- 依 2018 ACC/AHA 四大 statin benefit group（ASCVD / LDL≥190 / 糖尿病 40–75歲 / 10年風險≥7.5%）

## 📏 劑量 / 用法（依強度分類）
| 強度 | 代表藥 / 劑量 | LDL 降幅 |
| --- | --- | --- |
| High-intensity | **Atorvastatin 40–80mg**、**Rosuvastatin 20–40mg** | ≥50% |
| Moderate-intensity | Atorvastatin 10–20、Rosuvastatin 5–10、Simvastatin 20–40、Pravastatin 40–80 | 30–49% |
| Low-intensity | Simvastatin 10、Pravastatin 10–20 | <30% |
- **服用時機**：短效（simvastatin）**睡前服**（膽固醇合成夜間旺盛）；長效（atorvastatin/rosuvastatin）任何時間皆可
- **腎**：rosuvastatin 嚴重腎不全減量；一般不需大調整
- **肝**：活動性肝病禁用

## ⚠️ 副作用（常見 / 嚴重）
- **肌肉症狀**：肌痛（最常見）、myositis、**rhabdomyolysis（罕見但嚴重）**
- **肝酵素上升**（transaminase）
- **新發糖尿病**（風險小幅增加，益處仍大於風險）
- 少見：認知抱怨、周邊神經病變
- 腸胃不適、頭痛

## 🚫 禁忌症
- **懷孕 / 哺乳**（禁用）
- 活動性肝病、原因不明持續 transaminase 上升
- 對 statin 過敏、既往 statin 相關嚴重肌病
- （相對）與強 CYP3A4 抑制劑併用時劑量須調整

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：baseline **LFT**（不需常規追蹤，除非有症狀）、有肌肉症狀時驗 **CK**、治療後追 lipid panel 看達標
- **交互作用**：
  - **Fibrate（gemfibrozil ＞ fenofibrate）** → ↑肌病風險（gemfibrozil 抑制 statin glucuronidation，最危險組合）
  - **CYP3A4 抑制劑**（clarithromycin、azole、amiodarone、diltiazem、葡萄柚汁）→ ↑simvastatin/atorvastatin 濃度
  - **[[Warfarin(可邁丁)]]** → statin 可增強 warfarin 藥效（INR↑）
  - Cyclosporine、[[Ticagrelor(替格瑞洛)]] → ↑statin 濃度
- **孕哺**：禁用
- **搭配**：LDL 未達標可加 [[Ezetimibe(依澤替米貝)]] 或 PCSK9 抑制劑

## 🔗 相關
- 疾病：[[Myocardial Infarction(心肌梗塞)]]、[[Ischemic Stroke(缺血性中風)]]、[[Acute Coronary Syndrome(急性冠心症)]]
- 同類 / 加成：[[Ezetimibe(依澤替米貝)]]（NPC1L1，加成降 LDL）、PCSK9i
- 交互：[[Fibric acid(纖維酸類)]]（併用增肌病）、[[Warfarin(可邁丁)]]（INR↑）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位服 simvastatin 的病人因肺炎開始服 clarithromycin，數日後出現嚴重肌肉痛、CK 顯著上升、尿色變深。最可能機轉是？
- A. 肺炎併發肌炎
- B. Clarithromycin 抑制 CYP3A4，升高 simvastatin 濃度導致橫紋肌溶解
- C. 抗生素過敏
- D. 與 statin 無關

> [!success]- 答案
> **B** — clarithromycin 是強 **CYP3A4 抑制劑**，會升高 simvastatin 濃度→**橫紋肌溶解**。應停 statin、驗 CK/腎功能、水化，並避免此類組合。

**Q2.** 下列何者為 high-intensity statin 治療（LDL 降幅 ≥50%）？
- A. Simvastatin 20mg
- B. Pravastatin 40mg
- C. Atorvastatin 40–80mg
- D. Lovastatin 20mg

> [!success]- 答案
> **C** — **Atorvastatin 40–80mg 與 Rosuvastatin 20–40mg** 屬 high-intensity。其餘為 moderate/low intensity。

**Q3.** 一位計畫懷孕的高膽固醇女性，關於 statin 的建議何者正確？
- A. 繼續服用不影響胎兒
- B. 懷孕/準備懷孕應停用 statin
- C. 改用高劑量即可
- D. 只在第一孕期停用

> [!success]- 答案
> **B** — statin **孕期禁用**，計畫懷孕即應停藥（膽固醇為胎兒發育所需）。孕期血脂管理以飲食/生活型態為主。
