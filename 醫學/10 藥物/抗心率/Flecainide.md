# Flecainide（Class Ic 抗心律不整）

> [!abstract] 一句話：Class **Ic** — **最強的 Na⁺ 通道阻斷**（大幅減慢傳導、明顯拉長 QRS），對再極化（QT）影響小。用於**無結構性心臟病者**的心房顫動整流/維持（「pill-in-the-pocket」）與 SVT。**結構性心臟病 / 缺血性心臟病絕對禁用**（CAST 試驗證實增死亡率）——這是 Ic 類最重要的鐵律。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **結構性心臟病 / 冠心病 / 心衰竭 / 心肌梗塞後 → 絕對禁用**（CAST：Flecainide/Encainide 增死亡率）——用前務必確認**心臟結構正常**
> 2. **心房撲動「1:1 傳導」危機**：Flecainide 減慢心房率，可能讓房室 1:1 下傳致心室率暴衝 → **常需併 AV 結阻斷劑（β-blocker / non-DHP CCB）**
> 3. **致心律不整（proarrhythmia）**：拉長 QRS、誘發單型性 VT
> 4. **Brugada 症候群顯露 / 惡化**（Na 阻斷）——有猝死家族史者慎用
> 5. **腎/肝功能不全累積**

## 💊 適應症（治療哪些疾病）
- **陣發性心房顫動 / 撲動（無結構性心臟病）**：整流 + 維持竇律；**pill-in-the-pocket**（單次口服自我整流）→ [[醫學/11心電圖判讀/Atrial Fibrillation(心房顫動)]]
- **陣發性室上性心搏過速 PSVT / AVNRT / AVRT**
- **WPW 相關心律不整**（減慢旁路）
- 症狀性室性早期收縮（無結構性心臟病）

## 📏 劑量 / 用法（成人）
| 情境 | 劑量 |
| --- | --- |
| 陣發性 AF 維持 | 50–150 mg PO Q12H |
| Pill-in-the-pocket（單次整流）| 200–300 mg PO ×1（**須先在監測下驗證安全，且併 AV 結阻斷劑**）|
- **必配 AV 結阻斷劑**於 AF/撲動整流，避免 1:1 下傳
- **腎功能**：腎衰減量；血中治療濃度 0.2–1.0 mcg/mL

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：頭暈、視力模糊（視覺閃爍/blurred vision）、頭痛、噁心
- **嚴重**：
  - **致心律不整（proarrhythmia）**：VT、心房撲動 1:1 傳導、心搏停止
  - **結構性心臟病者猝死風險 ↑**
  - 負性肌力 → 心衰竭惡化
  - 傳導阻滯（QRS 過寬）

## 🚫 禁忌症
- **結構性心臟病、冠心病 / 缺血、心衰竭、心肌梗塞後**（絕對）
- 二/三度 AV block / 束支阻滯（無節律器）
- 心因性休克
- Brugada 症候群（慎用/避免）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：ECG（QRS 寬度，運動試驗評估運動誘發 proarrhythmia）、確認左心室功能與冠脈狀態
- **交互作用**：CYP2D6 抑制劑（fluoxetine/quinidine）↑ 濃度；與其他負性肌力/傳導藥加乘
- **孕哺**：可用於胎兒心律不整（經母體），孕婦本身用需評估

### 自我測驗（選擇題，答案摺疊）

**Q1.** 45 歲、心臟超音波正常、無冠心病的陣發性 AF 病人，想要能自我整流的方案。合理選擇？
- A. Flecainide pill-in-the-pocket，併 AV 結阻斷劑
- B. Flecainide 單獨大量頓服
- C. 因有冠心病禁用
- D. Digoxin 頓服

> [!success]- 答案
> **A** — 無結構性心臟病的陣發性 AF 可用 Flecainide「pill-in-the-pocket」，但須先在監測下驗證安全並**併用 AV 結阻斷劑**防心房撲動 1:1 下傳。

**Q2.** 心肌梗塞病史（EF 35%）的病人頻發室性早期收縮，實習醫師想開 Flecainide。最大問題？
- A. 太貴
- B. 結構性/缺血性心臟病用 Class Ic 會增加死亡率（CAST），絕對禁用
- C. 會低血鉀
- D. 沒問題

> [!success]- 答案
> **B** — CAST 試驗顯示 Class Ic 於心肌梗塞後/結構性心臟病**增加死亡率**，屬絕對禁忌；此類病人應改用 β-blocker、amiodarone 等。

**Q3.** 病人服 Flecainide 控制 AF 後，某次發作 ECG 為規則、心室率 ~250、寬 QRS。最可能機轉？
- A. TdP
- B. 心房撲動經房室 1:1 傳導（Flecainide 減慢心房率所致）
- C. 竇性心搏過速
- D. 低血鉀

> [!success]- 答案
> **B** — Flecainide 減慢心房撲動率後，房室結可能以 **1:1** 下傳造成極快心室率合併寬 QRS（Na 阻斷）；故整流時須併用 AV 結阻斷劑預防。
