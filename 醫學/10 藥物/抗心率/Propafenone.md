# Propafenone（Class Ic 抗心律不整）

> [!abstract] 一句話：Class **Ic**（同 Flecainide）強力阻 Na⁺ 通道、大幅拉長 QRS，對 QT 影響小；**額外帶輕度 β-阻斷作用**。用途與 Flecainide 幾乎相同——**無結構性心臟病者**的 AF 整流/維持與 pill-in-the-pocket。**結構性/缺血性心臟病同樣絕對禁用**（CAST 鐵律）。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **結構性心臟病 / 冠心病 / 心衰竭 / post-MI → 絕對禁用**（Class Ic 增死亡率）
> 2. **心房撲動 1:1 傳導**：整流 AF/撲動須併 **AV 結阻斷劑**（本身弱 β-阻斷不足以完全保護）
> 3. **弱 β-阻斷 → 支氣管痙攣**：**氣喘 / 嚴重 COPD 慎用/避免**（與 Flecainide 的差異點）
> 4. **心搏過緩 / 傳導阻滯**（合弱 β 效應更明顯）
> 5. **CYP2D6 代謝**：慢代謝者或併 CYP2D6 抑制劑 → 濃度上升、β-阻斷效應增強

## 💊 適應症（治療哪些疾病）
- **陣發性心房顫動 / 撲動（無結構性心臟病）**：整流 + 維持竇律；**pill-in-the-pocket** → [[醫學/11心電圖判讀/Atrial Fibrillation(心房顫動)]]
- **陣發性室上性心搏過速 PSVT / AVNRT / AVRT**
- **WPW 相關心律不整**
- 症狀性室性早期收縮（無結構性心臟病）

## 📏 劑量 / 用法（成人）
| 情境 | 劑量 |
| --- | --- |
| 速效（IR）維持 | 150–300 mg PO Q8H |
| 緩釋（SR）| 225–425 mg PO Q12H |
| Pill-in-the-pocket | 450–600 mg PO ×1（先監測下驗證安全 + 併 AV 結阻斷劑）|
- **腎/肝**：肝功能不全減量（肝代謝，CYP2D6）
- 起始一般由低劑量、心電圖監測下加量

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：**金屬味 / 味覺異常**（特徵性）、頭暈、噁心、便祕、視力模糊
- **嚴重**：
  - **致心律不整（proarrhythmia）**：VT、撲動 1:1 傳導
  - **支氣管痙攣**（弱 β-阻斷，氣喘者）
  - **心搏過緩 / 傳導阻滯**
  - 心衰竭惡化（負性肌力）

## 🚫 禁忌症
- **結構性心臟病、冠心病 / 缺血、心衰竭、post-MI**（絕對）
- **氣喘 / 支氣管痙攣性疾病**（弱 β-阻斷）
- 嚴重心搏過緩、二/三度 AV block、束支阻滯（無節律器）
- 心因性休克、嚴重低血壓、電解質不平衡

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：ECG（QRS、HR）、確認心臟結構正常、氣喘病史
- **交互作用**：
  - **CYP2D6 抑制劑（fluoxetine/paroxetine/quinidine）** ↑ 濃度
  - **↑ Digoxin、↑ Warfarin 效應**（需調整/監測 INR）
  - 與 β-blocker 加乘心搏過緩
- **孕哺**：資料有限，效益大於風險時使用

### 自我測驗（選擇題，答案摺疊）

**Q1.** 氣喘病史、心臟結構正常的陣發性 AF 病人需維持竇律。在 Flecainide 與 Propafenone 之間，哪個更需謹慎、為什麼？
- A. Flecainide，因它有 β-阻斷
- B. Propafenone，因其弱 β-阻斷可能誘發支氣管痙攣
- C. 兩者對氣喘皆安全
- D. 都禁用於 AF

> [!success]- 答案
> **B** — Propafenone 帶弱 β-阻斷作用，氣喘/支氣管痙攣性疾病者更易誘發支氣管痙攣；此情境相對偏好無 β-阻斷的 Flecainide（仍需無結構性心臟病）。

**Q2.** 病人服 Propafenone 期間抱怨「嘴巴一直有金屬味」。最恰當的判斷？
- A. 這是嚴重中毒須立即停藥急救
- B. 味覺異常是 Propafenone 常見且特徵性的副作用
- C. 代表 QT 延長
- D. 與藥無關

> [!success]- 答案
> **B** — 金屬味/味覺異常是 Propafenone 常見且具特徵性的副作用，通常非危險徵象；仍應評估心律不整控制與其他毒性。

**Q3.** 使用 Propafenone 維持竇律的 AF 病人同時服 warfarin，INR 突然升高。最可能原因？
- A. Propafenone 增強 warfarin 效應
- B. Propafenone 降低 warfarin 效應
- C. 與 Propafenone 無關
- D. Propafenone 是維生素 K

> [!success]- 答案
> **A** — Propafenone 會**增強 warfarin 的抗凝效應**使 INR 上升（亦升 digoxin 濃度），併用時須加密監測 INR 並調整劑量。
