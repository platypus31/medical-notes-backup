# Dofetilide（Class III 抗心律不整）

> [!abstract] 一句話：Class **III** — **純選擇性阻斷 IKr（快速延遲整流 K⁺ 電流）** → 延長動作電位 / 有效不反應期、延長 QT，但**不影響傳導（QRS）也無負性肌力**。用於**心房顫動/撲動的整流與維持竇律**，**可用於結構性心臟病 / 心衰竭**（不同於 Class Ic）。代價是 **TdP 風險高，美國規定必須住院起始 + 依腎功能/QT 嚴格劑量化**。台灣少見（未上市），但觀念重要。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **強制住院起始 3 天**：連續 ECG 監測，每劑後 2–3 小時測 QTc；**QTc 延長 >15% 或 >500 ms 要減量/停藥**
> 2. **劑量嚴格依 CrCl + QTc 計算**；**CrCl <20 mL/min 禁用**
> 3. **TdP（torsades）是主風險**——低血鉀、低血鎂會放大，起始前先矯正電解質
> 4. **大量禁忌併用藥**：cimetidine、verapamil、ketoconazole、trimethoprim、hydrochlorothiazide、prochlorperazine、megestrol 等（阻斷腎小管分泌 → ↑ 濃度 → TdP）
> 5. 其他 QT 延長藥一律避免併用

## 💊 適應症（治療哪些疾病）
- **心房顫動 / 心房撲動**：藥物整流 + 維持竇性節律 → [[醫學/11心電圖判讀/Atrial Fibrillation(心房顫動)]]、[[醫學/11心電圖判讀/Atrial Flutter(心房撲動)]]
- **可用於結構性心臟病 / 心衰竭病人**（維持竇律的少數安全選項之一，與 amiodarone 並列）→ [[Heart Failure(心臟衰竭)]]

## 📏 劑量 / 用法（成人）
| CrCl（mL/min） | 起始劑量 |
| --- | --- |
| >60 | 500 mcg PO BID |
| 40–60 | 250 mcg PO BID |
| 20–40 | 125 mcg PO BID |
| <20 | **禁用** |
- **住院起始**：首劑後 2–3 h 測 QTc，若 QTc 增 >15% 或 >500 ms → 減半；再超標 → 停藥
- **腎排除為主**，故以 CrCl 定劑量

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：頭痛、頭暈、胸痛、噁心
- **嚴重**：
  - **Torsades de pointes（TdP）/ QT 延長**（劑量相關，最重要）
  - 其他室性心律不整
  - 傳導系統影響小、不惡化心衰竭（這是它的優勢）

## 🚫 禁忌症
- 基線 **QTc >440 ms（或 >500 ms 有傳導異常）**
- **CrCl <20 mL/min**
- 併用會升高其濃度的禁忌藥（見紅旗）
- 未矯正的低血鉀 / 低血鎂
- 先天性長 QT 症候群 / TdP 病史

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：**連續 ECG（QTc）住院起始**、腎功能（CrCl）、電解質（K⁺/Mg²⁺）
- **交互作用**：抑制腎小管陽離子分泌的藥（cimetidine/trimethoprim/ketoconazole/verapamil/HCTZ 等）→ ↑ Dofetilide → TdP；所有 QT 延長藥避免
- **孕哺**：資料有限，一般避免

## 🔗 相關

### 自我測驗（選擇題，答案摺疊）

**Q1.** EF 30% 的心衰竭合併陣發性 AF 病人要維持竇律。下列抗心律不整何者「可用於結構性心臟病」？
- A. Flecainide
- B. Propafenone
- C. Dofetilide
- D. Disopyramide

> [!success]- 答案
> **C** — Class Ic（Flecainide/Propafenone）與強負性肌力的 Disopyramide 在結構性心臟病/心衰竭禁用；**Dofetilide（Class III）** 無負性肌力、可用於此族群（與 amiodarone 並列）。

**Q2.** 開始 Dofetilide 為何規定住院並密集測 QTc？
- A. 監測血壓
- B. 因劑量相關的 TdP 風險，需依 QTc 反應調整或停藥
- C. 監測血糖
- D. 觀察 QRS 增寬

> [!success]- 答案
> **B** — Dofetilide 延長 QT，**TdP 為劑量相關的主要致命風險**，須住院連續 ECG，首劑後 QTc 增 >15% 或 >500 ms 即減量/停藥。

**Q3.** 服用 Dofetilide 的病人因泌尿道感染被開 trimethoprim。最主要顧慮？
- A. 過敏
- B. Trimethoprim 抑制腎小管分泌使 Dofetilide 濃度上升 → TdP
- C. 降低 Dofetilide 效果
- D. 沒有交互作用

> [!success]- 答案
> **B** — Trimethoprim（連同 cimetidine、verapamil、ketoconazole、HCTZ 等）會**抑制 Dofetilide 的腎小管分泌**，升高血中濃度誘發 TdP，屬**禁忌併用**。
