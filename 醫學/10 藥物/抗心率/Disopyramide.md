# Disopyramide（Class Ia 抗心律不整）

> [!abstract] 一句話：Class **Ia**（同族 Procainamide、Quinidine）阻 Na⁺ + K⁺ 通道 → 減慢傳導、延長 QT。特色是**最強的負性肌力 + 顯著抗膽鹼作用**——這讓它在多數心律不整被冷落，卻在**肥厚性阻塞型心肌病（HOCM）** 靠負性肌力減輕左室流出道阻塞而有一席之地。台灣臨床少用。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **強負性肌力 → 誘發/惡化心衰竭**（Class Ia 中最強）；**HFrEF 禁用**
> 2. **QT 延長 → torsades de pointes（TdP）**
> 3. **顯著抗膽鹼副作用**：尿滯留（BPH 男性）、青光眼惡化、口乾、便秘、視力模糊
> 4. **腎功能不全累積**（腎排除為主）→ 減量
> 5. 不用於結構性心臟病 / 缺血後 VT 長期治療（Class I 增死亡率）

## 💊 適應症（治療哪些疾病）
- **肥厚性阻塞型心肌病（HOCM）**：利用負性肌力減輕 LVOT 阻塞、改善症狀（β-blocker/verapamil 效果不足時加用）→ [[Hypertrophic Cardiomyopathy(肥厚型心肌病)]]
- 維持竇性節律（心房顫動/撲動整流後，次要選項）→ [[醫學/11心電圖判讀/Atrial Fibrillation(心房顫動)]]
- 症狀性室上/室性早期收縮（少用）

## 📏 劑量 / 用法（成人）
| 情境 | 劑量 |
| --- | --- |
| 一般口服（速效）| 100–200 mg PO Q6H |
| 緩釋（CR）| 200–300 mg PO Q12H |
| 最大 | 一般 ≤400–800 mg/day（依腎功能） |
- **腎功能**：**腎衰減量並延長間隔**（腎排除為主）
- **監測 level**：治療濃度約 2–5 mcg/mL

## ⚠️ 副作用（常見 / 嚴重）
- **常見（抗膽鹼）**：口乾、便秘、視力模糊、**尿滯留**、青光眼加重
- **嚴重**：
  - **心衰竭惡化**（負性肌力）
  - **TdP / QT 延長**
  - 傳導阻滯、低血壓
  - 低血糖（罕見但需知）

## 🚫 禁忌症
- **收縮性心衰竭 / 心因性休克**（負性肌力）
- QT 延長 / TdP 病史
- 二/三度 AV block（無節律器）
- **閉角性青光眼、尿滯留 / 嚴重 BPH、重症肌無力**（抗膽鹼）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：ECG（QRS、QT）、心衰竭症狀、抗膽鹼副作用（排尿、眼壓）、血糖
- **交互作用**：其他 QT 延長藥加乘 TdP；其他抗膽鹼藥加乘；CYP3A4 抑制劑 ↑ 濃度
- **孕哺**：資料有限，僅效益大於風險時使用

## 🔗 相關

### 自我測驗（選擇題，答案摺疊）

**Q1.** HOCM 病人在 β-blocker 下仍有明顯左室流出道阻塞與症狀，可加用何藥利用其負性肌力？
- A. Amlodipine
- B. Disopyramide
- C. Digoxin
- D. Hydralazine

> [!success]- 答案
> **B** — Disopyramide 的強負性肌力可減輕 HOCM 的 LVOT gradient 改善症狀，是此情境的特殊選擇；反之正性肌力/血管擴張藥（digoxin/hydralazine）會惡化阻塞。

**Q2.** 70 歲 BPH 男性使用 Disopyramide 後解不出尿、腹脹。最可能機轉？
- A. 負性肌力
- B. 抗膽鹼作用造成尿滯留
- C. QT 延長
- D. 低血糖

> [!success]- 答案
> **B** — Disopyramide 有顯著**抗膽鹼**作用，可致尿滯留（BPH 男性尤甚）、青光眼惡化、口乾便秘，這類病人應避免使用。

**Q3.** 下列何者是 Disopyramide 的絕對禁忌？
- A. 高血壓
- B. 收縮性心衰竭
- C. 心房早期收縮
- D. 竇性心搏過速

> [!success]- 答案
> **B** — Disopyramide 為 Class Ia 中負性肌力最強者，**收縮性心衰竭/心因性休克禁用**，會惡化心輸出。
