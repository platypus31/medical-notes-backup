# β-lactamase inhibitors（β內醯胺酶抑制劑總覽 / Clavulanate · Sulbactam · Tazobactam · Avibactam）

> [!abstract] 一句話：**β內醯胺酶抑制劑（BLI）**本身**幾乎沒有抗菌力**，作用是**抑制細菌分泌的 β-lactamase（水解 β-lactam 的酵素）** → 「保護」搭配的 β-lactam 不被分解、**恢復並拓寬其抗菌譜**（尤其對產酶菌與部分厭氧菌）。經典複方：Amoxicillin-Clavulanate（Augmentin）、[[Piperacillin-Tazobactam(哌拉西林-他唑巴坦)]]、Ampicillin-Sulbactam（Unasyn）、Ceftazidime-Avibactam（打 CRE）。**觀念題必考：BLI 是「盾」不是「劍」，抗菌譜由搭配的 β-lactam + 被抑制酵素種類共同決定。**

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **BLI 單獨無治療意義**：抗菌力來自搭配的 β-lactam，看複方要問「哪個 β-lactam + 哪種酶被擋」
> 2. **不是所有 β-lactamase 都能擋**：傳統 BLI（clavulanate/sulbactam/tazobactam）主擋 class A（含多數 ESBL），**對 class B 金屬酶（MBL, e.g. NDM）與多數 carbapenemase 無效**；**Avibactam** 是新型（diazabicyclooctane）能擋 class A/C/部分 D（含 KPC、OXA-48），但**仍擋不了 MBL**
> 3. **Clavulanate 腸胃副作用明顯**：Augmentin 腹瀉率高（clavulanate 促腸蠕動），高劑量比例配方注意
> 4. **Amoxicillin-clavulanate 肝毒性**：膽汁鬱積性肝炎（clavulanate 相關，可延遲數週出現）
> 5. **Sulbactam 本身對 Acinetobacter 有內在抗菌活性**（少數 BLI 例外），是 MDR Acinetobacter 治療的一環

## 💊 適應症（依搭配複方而定）
- **Amoxicillin-Clavulanate（Augmentin）**：中耳炎、鼻竇炎、社區肺炎、咬傷、皮膚軟組織、泌尿道（口服主力）
- **Ampicillin-Sulbactam（Unasyn）**：腹腔內感染、婦科、吸入性肺炎、糖尿病足；sulbactam 另用於 MDR **Acinetobacter**
- **Piperacillin-Tazobactam（Tazocin）**：院內肺炎、腹腔內感染、發燒性嗜中性球低下（涵蓋綠膿桿菌）→ 見 [[Piperacillin-Tazobactam(哌拉西林-他唑巴坦)]]
- **Ceftazidime-Avibactam / Ceftolozane-Tazobactam / Meropenem-Vaborbactam**：多重抗藥 G(-)、CRE（KPC/OXA-48）保留藥
- 通用價值：**恢復對產 ESBL 及產酶菌的活性、涵蓋厭氧菌**

## 📏 劑量 / 用法（成人）— 依複方
| 複方 | 常見用法 ⚠️依院規/需查證 |
| --- | --- |
| Amoxicillin-Clavulanate | 875/125 mg PO Q12H 或 500/125 mg Q8H |
| Ampicillin-Sulbactam | 1.5–3 g IV Q6H |
| Piperacillin-Tazobactam | 4.5 g IV Q6–8H（延長輸注）→ 見專卡 |
| Ceftazidime-Avibactam | 2.5 g IV Q8H（依腎功能調整） |
- **腎功能**：多數複方（含搭配的 β-lactam）**需依腎功能調整**；BLI 部分亦經腎排
- BLI 比例是固定複方，不可自行拆解調整
- 抗菌譜/劑量以「搭配的 β-lactam」為主軸判讀

## ⚠️ 副作用（常見 / 嚴重，主要來自搭配 β-lactam + BLI 本身）
- **常見**：腹瀉（clavulanate 尤明顯）、噁心、皮疹、注射反應
- **嚴重**：
  - **Amoxicillin-clavulanate 膽汁鬱積性肝炎**（可延遲出現）
  - β-lactam 類過敏反應（含 anaphylaxis）、C. difficile 腸炎
  - 血球低下、腎功能異常
  - Piperacillin-tazobactam + vancomycin 併用可能增急性腎損傷（訊號，仍有爭議）

## 🚫 禁忌症
- 對 penicillin / β-lactam 或該複方任一成分過敏
- Amoxicillin-clavulanate 曾致膽汁鬱積性黃疸/肝功能異常者禁再用
- 依搭配 β-lactam 之各自禁忌

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：肝功能（尤 Augmentin 長療程）、腎功能、CBC、腹瀉/腸炎
- **交互作用**：
  - **Probenecid** → 延緩 β-lactam 腎排泄、升高血中濃度
  - 與 aminoglycoside 同注射器混合會失活（分開給）
  - 各複方依搭配 β-lactam 之交互作用
- **孕哺**：多數 penicillin 類複方孕期可用（B 級概念）；avibactam 等新藥資料有限

## 🔗 相關
- 同族複方 / 搭配 β-lactam：[[Penicillins(青黴素類)]]、[[Piperacillin-Tazobactam(哌拉西林-他唑巴坦)]]、[[Cephalosporins(頭孢菌素類分代總覽)]]、[[Carbapenems(碳青黴烯類)]]
- 抗藥對照：[[Colistin-PolymyxinB(黏菌素)]]（MBL/CRE 最後防線，avibactam 擋不了 MBL 時）
- 概念關聯：ESBL、carbapenemase（KPC/NDM/OXA-48）、抗藥機轉

### 自我測驗（選擇題，答案摺疊）

**Q1.** 關於 β內醯胺酶抑制劑，下列敘述何者正確？
- A. 本身有強大抗菌力可單獨使用
- B. 主要抑制細菌 β-lactamase 以保護搭配的 β-lactam
- C. 能抑制所有種類的 β-lactamase 包含金屬酶
- D. 只能與 aminoglycoside 併用

> [!question]- 答案
> **B** — BLI 本身幾無抗菌力，作用是抑制 β-lactamase 保護搭配的 β-lactam。傳統 BLI 擋不了金屬酶（MBL/NDM），連 avibactam 也擋不了 MBL。

**Q2.** 一株 CRE 檢出產 NDM 金屬酶（MBL），下列何者「無法」有效治療？
- A. Colistin（多黏菌素）
- B. Ceftazidime-avibactam
- C. 含 aztreonam 的組合策略
- D. 依藥敏選擇的最後防線藥物

> [!question]- 答案
> **B** — Avibactam 可擋 KPC/OXA-48，但擋不了 class B 金屬酶（MBL/NDM），故 ceftazidime-avibactam 對產 NDM 菌無效，需靠 colistin 或含 aztreonam 策略等。
