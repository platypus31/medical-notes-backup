# Ceftriaxone（頭孢曲松 / Rocephin）

> [!abstract] 一句話：**第三代頭孢菌素**，抑制細菌 transpeptidase（PBP）阻斷細胞壁合成 → 殺菌。廣效涵蓋 G(-) > G(+)，**半衰期長（~8h）可 QD 給藥**、**可穿 BBB 治腦膜炎**、**50% 經膽汁排泄故腎功能不良多不需調劑量**。值班最常開的空床經驗性抗生素之一。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **新生兒 + 同時輸注含鈣輸液（如 Ringer's lactate）→ 禁用**：會形成 ceftriaxone-calcium 沉澱物栓塞肺腎，可致死。新生兒高膽紅素血症也禁用（競爭白蛋白結合位加重黃疸）
> 2. **不涵蓋 Pseudomonas**：懷疑綠膿桿菌（免疫低下、院內感染、燒傷）要改 ceftazidime / cefepime / pip-tazo
> 3. **不涵蓋 MRSA、腸球菌 Enterococcus、非典型菌**（Mycoplasma/Legionella 要另加 macrolide）
> 4. **Penicillin 嚴重過敏（anaphylaxis）**：cephalosporin 交叉過敏率雖低（~1-2%）但仍需警覺
> 5. **膽泥/假性膽結石（biliary pseudolithiasis）**：兒童與長療程尤其，多在停藥後可逆

## 💊 適應症（治療哪些疾病）
- **社區型肺炎 [[Community-Acquired Pneumonia(社區型肺炎)]]**（常與 azithromycin 併用涵蓋非典型菌）
- **腦膜炎 [[Meningitis(腦膜炎)]]**（可穿 BBB，成人細菌性腦膜炎經驗治療基石，常 + vancomycin ± ampicillin）
- **腎盂腎炎 / 複雜性泌尿道感染 [[Pyelonephritis(腎盂腎炎)]]**
- **淋病 [[Gonorrhea(淋病)]]**（現行首選，單劑 IM，常 + doxycycline 涵蓋披衣菌）
- **自發性細菌性腹膜炎 SBP、膽道感染、傷寒、萊姆病神經侵犯、感染性心內膜炎（部分）**

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 一般感染（肺炎/泌尿道/軟組織） | **1–2 g IV/IM QD** |
| 細菌性腦膜炎 | **2 g IV Q12H**（腦膜炎要高劑量分兩次） |
| 淋病（無併發） | **500 mg IM 單劑**（體重 ≥150kg 用 1g）+ doxycycline |
| 感染性心內膜炎 | 2 g IV QD（依菌種與 guideline） |
- **腎功能**：主要不需依腎調整（雙重排泄，膽汁 40–50% + 腎）；但嚴重腎+肝同時衰竭時 max ≤ 2 g/day 並監測
- **肝**：一般不需調；重度肝腎併損才減量
- 溶劑：與含鈣輸液**不可同一條 IV line 併行**（新生兒禁；成人不同時段/沖洗後可）

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：腹瀉、注射部位疼痛、皮疹、暫時性肝酵素上升
- **嚴重**：
  - **Clostridioides difficile 相關腹瀉 / 偽膜性結腸炎**（廣效抗生素通病）
  - **膽泥 / 假性膽結石**（可致右上腹痛，常可逆）
  - **新生兒 ceftriaxone-鈣沉澱**（肺/腎栓塞致死）
  - **膽紅素置換 → 新生兒核黃疸 kernicterus**
  - 過敏反應（含罕見 anaphylaxis、Stevens-Johnson）
  - 罕見溶血性貧血、血球低下

## 🚫 禁忌症
- 對 ceftriaxone 或其他 cephalosporin 過敏
- **新生兒（≤28 天）需要或預期使用含鈣輸液者 → 絕對禁用**
- **新生兒高膽紅素血症、早產兒**
- Penicillin anaphylaxis 病史需審慎評估

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：腎肝功能、感染反應（發燒/CRP/培養）；長療程注意膽道症狀與 CBC
- **交互作用**：
  - **含鈣 IV 輸液（Ringer's lactate、TPN 含鈣）**：新生兒禁併，成人不同時併
  - **Warfarin**：可能增強抗凝（腸道菌叢改變影響 Vit K）→ 監測 INR
  - 與其他腎毒性藥（aminoglycoside）併用注意
- **孕哺**：孕期一般可用（B 級概念，臨床常用）；哺乳可用

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名 3 天大新生兒因疑似敗血症需抗生素，同時正輸注 Ringer's lactate（含鈣）。下列何者最正確？
- A. 可直接同一條 line 給 ceftriaxone
- B. Ceftriaxone 與含鈣輸液在新生兒為禁忌，應改藥或分開
- C. 只要調慢速度即可
- D. 加大 ceftriaxone 劑量增加療效

> [!success]- 答案
> **B** — 新生兒 ceftriaxone + 含鈣輸液會形成致命的鈣鹽沉澱（肺腎栓塞），為絕對禁忌，應選用其他 cephalosporin（如 cefotaxime）或完全分開。

**Q2.** 65 歲男性院內獲得性肺炎，懷疑綠膿桿菌感染。單用 ceftriaxone 的主要問題是？
- A. 劑量不足
- B. Ceftriaxone 不涵蓋 Pseudomonas
- C. 無法口服
- D. 半衰期太短

> [!success]- 答案
> **B** — Ceftriaxone（第三代，除 ceftazidime 外）對綠膿桿菌效果不足，懷疑 Pseudomonas 應改 ceftazidime / cefepime / piperacillin-tazobactam。

**Q3.** 成人細菌性腦膜炎經驗性治療，關於 ceftriaxone 何者正確？
- A. 1g QD 即足夠
- B. 不能用因無法穿越 BBB
- C. 2g Q12H，且常合併 vancomycin
- D. 只需口服

> [!success]- 答案
> **C** — Ceftriaxone 可穿 BBB，腦膜炎需高劑量 2g Q12H；因肺炎鏈球菌抗藥性，經驗治療常加 vancomycin（必要時再加 ampicillin 涵蓋 Listeria）。
