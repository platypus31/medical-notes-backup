# Fluoroquinolones（氟喹諾酮類 / Levofloxacin、Ciprofloxacin、Moxifloxacin）

> [!abstract] 一句話：**廣效殺菌型抗生素**，抑制 DNA gyrase / topoisomerase IV 阻斷 DNA 複製。**口服生體可用率極佳（近 IV）**，組織穿透好，涵蓋 G(-)（含 Pseudomonas，尤其 cipro/levo）與非典型菌；但**黑框警語一堆（肌腱斷裂、主動脈剝離、QT 延長、周邊神經病變、CNS 副作用）**，FDA 已建議**單純感染（鼻竇炎/支氣管炎/單純 UTI）避免當首選**。三者定位：Cipro 偏 G(-)/綠膿、Levo 呼吸道+泌尿廣用、Moxi 呼吸道+厭氧（但不入尿、不治 UTI）。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **肌腱炎 / 肌腱斷裂（黑框）**：阿基里斯腱最常見，老年、類固醇併用、器官移植者風險高；出現肌腱痛立即停藥
> 2. **主動脈瘤 / 主動脈剝離風險上升（黑框級警示）**：有主動脈瘤病史、馬凡氏症、老年高血壓者避免使用
> 3. **QT 延長 → Torsades**：Moxifloxacin 最明顯；避免與其他 QT 延長藥（macrolide、抗精神病、antiarrhythmic）併用、低血鉀低血鎂先矯正
> 4. **周邊神經病變（黑框）**：可能不可逆，出現感覺異常/麻木立即停藥
> 5. **與制酸劑 / 鈣鎂鐵鋅 / 乳製品螯合 → 吸收大減**：需間隔（口服 FQ 前 2h 或後 6h 再給二價/三價陽離子）
> 6. **C. difficile 高風險**：FQ 是偽膜性結腸炎的高風險族群之一
> 7. **血糖異常**：可致低血糖（尤其糖尿病用口服降血糖藥者）或高血糖
> 8. **Moxifloxacin 不用於 UTI**：尿中濃度不足

## 💊 適應症（治療哪些疾病）
- **社區型肺炎 [[Community-Acquired Pneumonia(社區型肺炎)]]**（呼吸道 FQ = levofloxacin / moxifloxacin，單藥可涵蓋肺炎鏈球菌 + 非典型菌）
- **複雜性泌尿道感染 / 腎盂腎炎 [[Pyelonephritis(腎盂腎炎)]]**（cipro / levo；單純膀胱炎現不建議首選）
- **急性攝護腺炎、副睪炎**（FQ 組織穿透佳）
- **腹腔內感染 [[Intra-abdominal Infection(腹腔內感染)]]**（cipro/levo + metronidazole 涵蓋厭氧）
- **綠膿桿菌感染 [[Pseudomonas Infection(綠膿桿菌感染)]]**（ciprofloxacin 為口服可用的抗綠膿選項）
- **旅行者腹瀉 / 傷寒 / 桿菌性痢疾**（cipro）
- **骨髓炎、慢性感染需長口服療程者**

## 📏 劑量 / 用法（成人）
| 藥物 / 用途 | 劑量 |
| --- | --- |
| **Levofloxacin** 肺炎 / 複雜 UTI | **750 mg PO/IV QD**（或 500 mg QD，依適應症） |
| **Ciprofloxacin** 一般 | **400 mg IV Q12H** 或 **500–750 mg PO Q12H** |
| **Ciprofloxacin** 複雜/綠膿 | 400 mg IV Q8H（重症） |
| **Moxifloxacin** 呼吸道/腹腔 | **400 mg PO/IV QD**（不需依腎調整；不用於 UTI） |
- **腎功能**：Levofloxacin、Ciprofloxacin **需依 CrCl 調整**（延長給藥間隔或減量，依仿單）；**Moxifloxacin 不需腎調整**（主要肝代謝）
- **肝**：Moxifloxacin 重度肝損慎用；其餘一般不需大幅調整
- **給藥時機**：與含二價/三價陽離子（制酸劑、鐵鈣鎂鋅、sucralfate、乳品）間隔（詳細間隔時間依仿單）

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：噁心、腹瀉、頭痛、頭暈、失眠、光敏感、肝酵素上升
- **嚴重**：
  - **肌腱炎 / 肌腱斷裂**（黑框，阿基里斯腱）
  - **周邊神經病變**（黑框，可能不可逆）
  - **CNS 效應**（黑框：癲癇、混亂、幻覺、譫妄、焦慮；可誘發或惡化重症肌無力 → 重症肌無力者禁用）
  - **QT 延長 → Torsades de pointes**（moxi 最強）
  - **主動脈瘤 / 剝離風險上升**
  - **C. difficile 相關腹瀉 / 偽膜性結腸炎**
  - **血糖異常**（低血糖可致命）
  - 罕見肝衰竭、Stevens-Johnson、光毒性

## 🚫 禁忌症
- 對氟喹諾酮類過敏
- **重症肌無力（myasthenia gravis）**病史（可誘發致命肌無力惡化）
- 已知 QT 延長 / 併用其他 QT 延長藥（moxifloxacin 尤須避免）
- 肌腱疾病 / 先前 FQ 引起肌腱病變史
- 兒童與孕婦一般避免（軟骨毒性顧慮，除非利大於弊、無替代）— 詳細依仿單/院規
- 主動脈瘤 / 剝離高風險者相對禁忌

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：QTc（尤其 moxi 或併 QT 藥）、血糖（糖尿病者）、肌腱症狀、腎功能、感染反應
- **交互作用**：
  - **二價/三價陽離子（Al、Mg、Ca、Fe、Zn）、sucralfate、乳品** → 螯合減少吸收（分開服用）
  - **Warfarin** → 增強抗凝，監測 INR
  - **QT 延長藥**（class III 抗心律不整、macrolide、抗精神病、ondansetron）→ 疊加 QT 風險
  - **Theophylline / caffeine**（尤其 cipro 抑制 CYP1A2）→ 濃度上升、毒性
  - **口服降血糖藥 / 胰島素** → 血糖波動
  - NSAID 併用可能增加 CNS 刺激/癲癇風險
- **孕哺**：孕期一般避免（軟骨/關節毒性動物數據），哺乳需權衡（依仿單/院規，需查證）

氟喹諾酮類的作用機轉是什麼？::抑制細菌 DNA gyrase（topoisomerase II）與 topoisomerase IV，阻斷 DNA 複製，殺菌型、濃度依賴

三個代表性氟喹諾酮的臨床定位差異？::Ciprofloxacin 偏 G(-)/抗綠膿；Levofloxacin 呼吸道+泌尿廣用；Moxifloxacin 呼吸道+厭氧但不入尿、不治 UTI

氟喹諾酮類最著名的肌肉骨骼黑框副作用是什麼？::肌腱炎與肌腱斷裂（阿基里斯腱最常見），老年、類固醇併用、器官移植風險高

哪一個氟喹諾酮 QT 延長最明顯、且不需腎調整？::Moxifloxacin（QT 延長最強，主要肝代謝故不需依腎調整，但也因此不用於 UTI）

為什麼口服氟喹諾酮不能跟制酸劑或鐵鈣一起吃？::二價/三價陽離子（Al、Mg、Ca、Fe、Zn）會與 FQ 螯合大幅降低吸收，需分開間隔服用

氟喹諾酮在哪種神經肌肉疾病是禁忌？::重症肌無力（myasthenia gravis）—可誘發致命的肌無力惡化

除了肌腱，氟喹諾酮還有哪兩個可能不可逆的神經黑框？::周邊神經病變、中樞神經副作用（癲癇/混亂/幻覺/譫妄）

為什麼 FDA 建議單純鼻竇炎/支氣管炎/單純 UTI 不要首選氟喹諾酮？::因其嚴重副作用（肌腱、神經、主動脈、QT）風險超過單純感染的效益，應保留給無替代選項者

氟喹諾酮對糖尿病病人要注意什麼？::可致血糖異常，尤其與口服降血糖藥併用時可能發生嚴重（可致命）低血糖

Ciprofloxacin 會抑制哪個 CYP 酵素、影響哪個常見藥？::抑制 CYP1A2 → 使 theophylline / caffeine 濃度上升，增加毒性風險

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名 72 歲男性因複雜性泌尿道感染口服 levofloxacin 後第 5 天出現右腳跟劇痛、無法踮腳。最可能的診斷與處置？
- A. 痛風發作，加 NSAID 繼續 FQ
- B. 阿基里斯腱炎/斷裂，立即停藥並評估
- C. 深部靜脈栓塞，加抗凝
- D. 正常藥物反應，繼續觀察

> [!success]- 答案
> **B** — 氟喹諾酮類引起肌腱炎/肌腱斷裂為黑框警語，阿基里斯腱最常見，老年為高風險。出現肌腱疼痛應立即停藥、避免患肢負重並評估。

**Q2.** 下列何者最不適合用 moxifloxacin 治療？
- A. 社區型肺炎
- B. 急性單純膀胱炎（UTI）
- C. 腹腔內感染（併 metronidazole）
- D. 慢性支氣管炎急性惡化

> [!success]- 答案
> **B** — Moxifloxacin 在尿中濃度不足，不用於泌尿道感染；且它 QT 延長最明顯、不需腎調整（肝代謝）。

**Q3.** 病人正服用制酸劑（含鋁鎂）及口服 ciprofloxacin。關於給藥何者正確？
- A. 兩者可同時服用不影響
- B. 制酸劑會增強 cipro 吸收
- C. 二價/三價陽離子會螯合 cipro 大幅降低吸收，需分開服用
- D. 應停用 cipro 改注射唯一方法

> [!success]- 答案
> **C** — 鋁、鎂、鈣、鐵、鋅等陽離子與氟喹諾酮螯合，顯著降低口服吸收，必須錯開服藥時間（依仿單間隔）。
