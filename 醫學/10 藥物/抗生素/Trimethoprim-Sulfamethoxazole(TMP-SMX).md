# Trimethoprim-Sulfamethoxazole（TMP-SMX / Baktar、Septra）

> [!abstract] 一句話：**複方磺胺**，SMX 抑制 dihydropteroate synthase、TMP 抑制 dihydrofolate reductase，**序列阻斷細菌葉酸合成**（人類靠外源葉酸故選擇性毒殺細菌）。**PJP（肺囊蟲肺炎）預防與治療的首選**，也用於 UTI、MRSA 軟組織、Nocardia、Stenotrophomonas。值班三大陷阱：**高血鉀**、**血中 creatinine 假性上升（TMP 抑制腎小管分泌，非真腎損）**、**磺胺過敏 → 皮疹甚至 SJS/TEN**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **高血鉀（hyperkalemia）**：TMP 有類 amiloride 作用阻斷遠端腎小管 Na 通道 → 保鉀；高劑量、腎功能差、併用 ACEI/ARB/保鉀利尿劑者尤危險
> 2. **血 creatinine 假性上升**：TMP 抑制 creatinine 的腎小管分泌 → Cr 上升但 GFR 未真的下降（並非真腎損傷，需區辨）
> 3. **磺胺過敏 → 嚴重皮膚反應**：皮疹常見，警覺 Stevens-Johnson / TEN / DRESS
> 4. **與 Warfarin 增強抗凝**（抑制 CYP2C9 + 移換蛋白結合）→ INR 大幅上升、出血
> 5. **孕晚期 / 新生兒避免**：磺胺置換膽紅素 → 核黃疸 kernicterus；孕早期葉酸拮抗致畸顧慮
> 6. **G6PD 缺乏 → 溶血**

## 💊 適應症（治療哪些疾病）
- **肺囊蟲肺炎 PJP [[Pneumocystis Pneumonia(肺囊蟲肺炎)]]**（治療與預防首選，免疫低下/HIV/移植/長期類固醇）
- **泌尿道感染 [[Urinary Tract Infection(泌尿道感染)]]**（單純膀胱炎經驗選項，依當地抗藥性）
- **社區型 MRSA 皮膚軟組織感染 [[MRSA Skin and Soft Tissue Infection(MRSA皮膚軟組織感染)]]**（膿瘍引流後口服選項）
- **Nocardia 感染、Stenotrophomonas maltophilia 感染**（首選）
- **弓漿蟲 Toxoplasmosis 預防**（HIV，替代方案）
- **旅行者腹瀉、某些 Shigella / 傷寒（依感受性）**
- **急性中耳炎 / 鼻竇炎（替代）**

## 📏 劑量 / 用法（成人）
> 劑量常以 **TMP 成分（trimethoprim）mg** 計算。單顆 SS（single-strength）= TMP 80/SMX 400 mg；DS（double-strength）= TMP 160/SMX 800 mg。

| 用途 | 劑量 |
| --- | --- |
| 單純 UTI | **1 顆 DS PO BID × 3–5 天**（依適應症） |
| MRSA 軟組織 | 1–2 顆 DS PO BID |
| PJP **治療** | **TMP 15–20 mg/kg/day** 分 3–4 次（IV/PO）× 21 天（重症加類固醇） |
| PJP **預防** | **1 顆 DS PO QD**（或 SS QD / DS 每週三次，依方案） |
- **腎功能**：**需依 CrCl 調整**（CrCl 降低時減量或延長間隔；嚴重腎衰竭/透析多不建議或大幅調整，依仿單）
- **肝**：重度肝損慎用
- **PJP 高劑量療程**：務必監測血鉀、血球、腎功能、皮疹

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：噁心、皮疹、光敏感、頭痛、食慾差、血 Cr 輕度上升（假性）
- **嚴重**：
  - **高血鉀**（尤其高劑量 + 腎功能差 + ACEI/ARB）
  - **嚴重皮膚反應 SJS / TEN / DRESS**
  - **骨髓抑制**（葉酸拮抗 → 血球減少、巨球性貧血；長期可補 leucovorin/葉酸，依情況）
  - **溶血性貧血**（G6PD 缺乏）
  - **肝毒性、間質性腎炎、無菌性腦膜炎**（罕見）
  - 低血糖（罕見，尤其腎功能差者）

## 🚫 禁忌症
- 對磺胺類（sulfonamide）或 trimethoprim 過敏
- **巨球性貧血因葉酸缺乏**
- **懷孕晚期 / 哺乳新生兒 / 早產兒**（核黃疸風險）；孕早期葉酸拮抗致畸顧慮 → 一般避免
- 嚴重肝腎功能不全
- G6PD 缺乏（溶血風險）— 相對禁忌
- 併用 dofetilide（QT 相關禁忌，依仿單）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：血鉀、腎功能（區辨假性 Cr 上升 vs 真腎損）、CBC（長療程/高劑量）、皮疹
- **交互作用**：
  - **ACEI / ARB / 保鉀利尿劑 / spironolactone** → 加重高血鉀
  - **Warfarin** → 增強抗凝（抑制 CYP2C9 + 蛋白置換），INR 上升
  - **Sulfonylurea（口服降血糖）** → 低血糖
  - **Phenytoin** → 濃度上升、毒性
  - **Methotrexate** → 葉酸拮抗疊加 → 骨髓抑制加重
  - **Dofetilide** → QT 相關風險
- **孕哺**：孕早期（致畸）與孕晚期（核黃疸）均一般避免；哺乳期新生兒/早產/黃疸嬰兒禁（依仿單/院規）

## 🔗 相關

### 自我測驗（選擇題，答案摺疊）

**Q1.** HIV 病人 CD4 極低，需預防肺囊蟲肺炎（PJP），首選藥物為？
- A. Azithromycin
- B. Trimethoprim-sulfamethoxazole
- C. Fluconazole
- D. Acyclovir

> [!success]- 答案
> **B** — TMP-SMX 是 PJP 治療與預防的首選；預防常用 1 顆 DS PO QD。

**Q2.** 一名腎功能中度不全、正服用 ACEI 的病人開始服用高劑量 TMP-SMX，3 天後 K⁺ 6.2 mmol/L、Cr 由 1.2 上升到 1.6。下列判讀何者最正確？
- A. 一定是急性腎損傷，立即洗腎
- B. 需警覺高血鉀（TMP 保鉀 + ACEI 疊加）；Cr 上升部分可能是 TMP 抑制分泌的假性上升
- C. 藥物完全無關
- D. 高血鉀與此藥無關

> [!success]- 答案
> **B** — TMP 阻斷遠端 Na 通道保鉀，與 ACEI 疊加易致高血鉀（需處理）；同時 TMP 抑制 creatinine 腎小管分泌會造成部分假性 Cr 上升，須與真腎損區辨。

**Q3.** 關於 TMP-SMX 的嚴重副作用，下列何者「不」典型？
- A. Stevens-Johnson syndrome
- B. 高血鉀
- C. 骨髓抑制/血球減少
- D. QT 延長引起的肌腱斷裂

> [!success]- 答案
> **D** — 肌腱斷裂是氟喹諾酮的特徵，非 TMP-SMX。TMP-SMX 的嚴重副作用包括 SJS/TEN、高血鉀、骨髓抑制、溶血（G6PD）等。
