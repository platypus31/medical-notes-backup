# Metronidazole（甲硝唑 / Flagyl）

> [!abstract] 一句話：**厭氧菌與原蟲殺手**，在低氧環境被還原成毒性中間產物破壞 DNA。**橫膈以下厭氧菌**（Bacteroides fragilis 等）與 C. difficile、原蟲（滴蟲、阿米巴、梨形鞭毛蟲）的主力；口服吸收極佳（近 100%）、組織/CNS 穿透好。值班兩大記憶點：**喝酒會 disulfiram-like reaction（禁酒）**、**與 warfarin 併用會增強抗凝**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **禁酒（disulfiram-like reaction）**：治療期間及**停藥後至少 3 天**內飲酒 → 潮紅、噁心嘔吐、心悸、頭痛；含酒精藥水/漱口水也要注意
> 2. **只涵蓋厭氧菌與原蟲，不涵蓋好氧菌**：腹腔/骨盆感染需搭配涵蓋好氧 G(-) 的藥（如 ceftriaxone / cipro / pip-tazo）
> 3. **與 Warfarin 併用 → 增強抗凝**（抑制其代謝）→ INR 上升、出血風險，需監測減量
> 4. **周邊神經病變 / CNS 毒性**：長療程或高劑量可致感覺異常、罕見腦病變、癲癇、無菌性腦膜炎、小腦症狀
> 5. **金屬味口感 + 尿液變深色**（無害但先告知病人）

## 💊 適應症（治療哪些疾病）
- **C. difficile 感染 [[Clostridioides difficile Infection(困難梭菌感染)]]**（輕中度替代選項；現行 guideline 首選多為 vancomycin/fidaxomicin，metronidazole 用於無法取得時或輕症）
- **腹腔內感染 [[Intra-abdominal Infection(腹腔內感染)]]**（涵蓋厭氧，需 + 好氧 G(-) 涵蓋藥）
- **骨盆腔發炎 PID / 婦科厭氧感染**
- **細菌性陰道炎 [[Bacterial Vaginosis(細菌性陰道炎)]]** 與 **滴蟲感染 [[Trichomoniasis(滴蟲陰道炎)]]**
- **阿米巴痢疾 / 阿米巴肝膿瘍**（+ 腔內殺蟲劑如 paromomycin 清除囊體）
- **梨形鞭毛蟲 Giardiasis**
- **牙源性/口腔厭氧感染、腦膿瘍、糖尿病足厭氧成分**
- **幽門螺旋桿菌 H. pylori 除菌**（多藥組合之一）

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 腹腔/厭氧感染 | **500 mg IV/PO Q8H**（重症 IV） |
| C. difficile（輕中度替代） | **500 mg PO TID × 10–14 天** |
| 細菌性陰道炎 | 500 mg PO BID × 7 天（或陰道凝膠，依仿單） |
| 滴蟲陰道炎 | 2 g PO 單劑 或 500 mg BID × 7 天（性伴侶同治） |
| 阿米巴肝膿瘍/腸道 | 500–750 mg PO TID × 7–10 天 + 腔內殺蟲劑 |
| 梨形鞭毛蟲 | 250 mg PO TID × 5–7 天（依仿單） |
- **腎功能**：一般不需依腎調整；**嚴重腎衰竭（ESRD/透析）**代謝物可能蓄積，依院規考慮調整（需查證）
- **肝**：**重度肝功能不全需減量**（如減至半量，依仿單）— 肝代謝為主

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：金屬味、噁心、食慾差、頭痛、尿液深色、口乾、腹瀉
- **嚴重**：
  - **Disuliram-like reaction**（併酒精）
  - **周邊神經病變**（長療程/高劑量，多可逆但可能持續）
  - **中樞神經毒性**：腦病變、癲癇、小腦功能障礙、無菌性腦膜炎（罕見）
  - 罕見胰臟炎、可逆性血球低下（白血球減少）
  - 長期高劑量的致癌性顧慮（動物數據，臨床短療程不需過度擔憂）

## 🚫 禁忌症
- 對 metronidazole / 硝基咪唑類過敏
- **同時或近期（治療期間及停藥 3 天內）飲酒或使用含酒精製劑**
- **懷孕第一孕期相對避免**（尤其高劑量單次 2g 方案；需權衡，依仿單/院規）
- 併用 disulfiram（近 2 週內）→ 急性精神症狀/混亂風險

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：長療程注意神經症狀（感覺異常）、CBC；肝功能不全者調劑量
- **交互作用**：
  - **酒精 / 含酒精製劑** → disulfiram-like reaction
  - **Warfarin** → 增強抗凝、INR 上升（監測、可能需減 warfarin）
  - **Disulfiram** → 精神混亂 / 急性精神病
  - **Lithium** → 鋰濃度上升、毒性
  - **CYP 誘導劑（phenytoin、phenobarbital、rifampin）** → 降低 metronidazole 濃度
  - 抑制某些藥代謝（如 5-FU 毒性增加）
- **孕哺**：第一孕期一般避免（尤其高劑量單劑）；哺乳期若單劑 2g 建議暫停哺乳一段時間（依仿單/院規，需查證）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 病人服用 metronidazole 治療細菌性陰道炎，當晚喝了兩杯啤酒後出現臉部潮紅、劇烈噁心嘔吐、心悸。最可能原因？
- A. 藥物過敏 anaphylaxis
- B. Disulfiram-like reaction（酒精與 metronidazole 交互）
- C. 急性酒精中毒
- D. 低血糖

> [!success]- 答案
> **B** — Metronidazole 抑制乙醛去氫酶類似反應，與酒精併用產生 disulfiram-like reaction。應衛教治療期間及停藥後至少 3 天禁酒（含酒精藥水）。

**Q2.** 關於 metronidazole 的抗菌譜，下列何者正確？
- A. 廣泛涵蓋好氧 G(-) 桿菌含綠膿
- B. 主要涵蓋厭氧菌與原蟲，不涵蓋好氧菌
- C. 是 MRSA 的首選
- D. 對非典型菌（Mycoplasma）效果佳

> [!success]- 答案
> **B** — Metronidazole 針對厭氧菌與原蟲，對好氧菌無效；腹腔/骨盆感染須搭配涵蓋好氧 G(-) 的抗生素。

**Q3.** 一名長期服用 warfarin 的病人因牙源性厭氧感染加開 metronidazole，最需監測什麼？
- A. 血糖
- B. QTc
- C. INR（抗凝過強、出血風險）
- D. 血鉀

> [!success]- 答案
> **C** — Metronidazole 抑制 warfarin 代謝增強抗凝，INR 會上升、出血風險增加，需密切監測 INR 並考慮調整 warfarin 劑量。
