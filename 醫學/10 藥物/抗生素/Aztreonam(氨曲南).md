# Aztreonam（氨曲南 / Azactam）

> [!abstract] 一句話：**唯一上市的單環 β-lactam（monobactam）**，與 PBP3 結合阻斷細胞壁合成 → 殺菌。**只涵蓋革蘭氏陰性需氧菌（含 Pseudomonas）**，對 G(+) 與厭氧菌完全無效。最大臨床價值：**與 penicillin / cephalosporin 幾乎無交叉過敏**（側鏈結構不同），是嚴重 β-lactam 過敏病人涵蓋 G(-) 的救援選項。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **只殺 G(-) 需氧菌 → 絕不能單用做經驗性治療**：完全不涵蓋 G(+)（葡萄球菌/鏈球菌/腸球菌）與厭氧菌，懷疑混合感染必須併其他藥（如 vancomycin ± metronidazole）
> 2. **不涵蓋非典型菌**（Mycoplasma / Legionella / Chlamydia）
> 3. **與 ceftazidime 側鏈相同** → 對 ceftazidime 曾 anaphylaxis 者可能交叉過敏，是唯一需警覺的例外（但對 penicillin 過敏仍安全）
> 4. **腎功能不良需減量**（主要經腎排泄），漏調會累積
> 5. **抗藥性可誘導**：對產 ESBL / AmpC 的腸桿菌可能無效，重症前先看抗藥性型態

## 💊 適應症（治療哪些疾病）
- **G(-) 桿菌感染而病人對 penicillin / cephalosporin 嚴重過敏**（最核心價值）
- **複雜性泌尿道感染 / 腎盂腎炎 [[Pyelonephritis(腎盂腎炎)]]**（G(-) 為主）
- **院內肺炎 / 呼吸道 G(-) 感染**（含 Pseudomonas，常與抗 G(+) 藥併用）
- **腹腔感染**（須併 metronidazole 涵蓋厭氧菌）
- **敗血症 G(-) 部分**（併涵蓋 G(+) 的藥物）
- 吸入劑型（Cayston）用於**囊性纖維化 Pseudomonas 慢性感染**

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 一般 G(-) 感染 | **1–2 g IV Q8–12H** |
| 嚴重 / Pseudomonas 感染 | **2 g IV Q6–8H** |
| 泌尿道感染（輕） | 500 mg–1 g IV/IM Q8–12H |
| 吸入劑（Cayston, CF） | 75 mg 吸入 TID |
- **腎功能調整**：CrCl 10–30 → 負荷後劑量減半；CrCl <10 → 減至 1/4；血液透析後補充劑量
- **肝**：一般不需調整
- IV 給藥，重症採 Q6–8H；半衰期約 1.7–2 小時（腎功能正常）

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：注射部位反應 / 靜脈炎、皮疹、腹瀉、噁心、暫時性肝酵素上升
- **嚴重**：
  - 過敏反應（罕見 anaphylaxis；整體交叉過敏率極低）
  - **Clostridioides difficile 相關腹瀉**（廣效抗生素通病，相對較少）
  - 罕見血球低下、Stevens-Johnson syndrome
  - 極少數癲癇（高劑量 + 腎衰累積）

## 🚫 禁忌症
- 對 aztreonam 本身過敏
- **對 ceftazidime 曾發生嚴重過敏者慎用**（共用側鏈）
- （注意：penicillin / 其他 cephalosporin 過敏**不是**禁忌，反而是使用時機）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：腎功能（調劑量）、肝酵素、感染反應、長療程注意 CBC 與 C. diff 症狀
- **交互作用**：
  - 與其他腎毒性藥（aminoglycoside）併用注意腎功能
  - 相對交互作用少（不經 CYP450）
- **孕哺**：孕期一般視為可用（臨床常用）；哺乳少量分泌，一般認為可用

## 🔗 相關
- 疾病：[[Pyelonephritis(腎盂腎炎)]]、院內肺炎、腹腔感染
- 同族機轉：[[Penicillins(青黴素類)]]、[[Cephalosporins(頭孢菌素類分代總覽)]]、[[Carbapenems(碳青黴烯類)]]（皆 β-lactam）
- 常搭配補洞：[[Vancomycin(萬古黴素)]]（補 G(+)）、[[Metronidazole(甲硝唑)]]（補厭氧菌）
- 涵蓋 Pseudomonas 對照：[[Piperacillin-Tazobactam(哌拉西林-他唑巴坦)]]、Ceftazidime、Cefepime

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名對 penicillin 有 anaphylaxis 病史的病人發生 G(-) 桿菌泌尿道敗血症，最適合涵蓋 G(-) 的 β-lactam 為何？
- A. Piperacillin-tazobactam
- B. Ceftriaxone
- C. Aztreonam
- D. Meropenem

> [!success]- 答案
> **C** — Aztreonam（monobactam）與 penicillin/cephalosporin 幾乎無交叉過敏，是嚴重 β-lactam 過敏者涵蓋 G(-)（含 Pseudomonas）的救援首選。

**Q2.** 關於 aztreonam 的抗菌譜，下列何者正確？
- A. 廣泛涵蓋 G(+) 與 G(-)
- B. 只涵蓋革蘭氏陰性需氧菌，對 G(+) 及厭氧菌無效
- C. 主要用於厭氧菌感染
- D. 對 MRSA 有效

> [!success]- 答案
> **B** — Aztreonam 的抗菌譜極窄，只涵蓋 G(-) 需氧菌（含 Pseudomonas），因此經驗性治療常需併用抗 G(+)（如 vancomycin）與抗厭氧（如 metronidazole）藥物。

**Q3.** 使用 aztreonam 時，唯一需要注意可能交叉過敏的藥物是？
- A. Amoxicillin
- B. Ceftriaxone
- C. Ceftazidime
- D. Meropenem

> [!success]- 答案
> **C** — Aztreonam 與 ceftazidime 共用相同側鏈，對 ceftazidime 曾嚴重過敏者可能交叉過敏；對 penicillin 及多數 cephalosporin 則無交叉反應。
