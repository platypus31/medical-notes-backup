# Colistin / Polymyxin B（黏菌素 / 多黏菌素）

> [!abstract] 一句話：**多黏菌素類**（陽離子環脂肽），結合 G(-) 外膜 **LPS 的 lipid A**、置換鈣鎂橋、破壞外膜與內膜通透性 → 細胞內容物外漏、殺菌。**專打多重抗藥 G(-)**：CRE、多重抗藥 Pseudomonas、Acinetobacter（**MDR/XDR G(-) 的最後防線**）。因**腎毒性與神經毒性大**曾一度退場，現因抗藥危機重返。**Colistin 以前驅物 CMS 給藥**，劑量學複雜易失誤。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **腎毒性（劑量限制毒性）**：急性腎損傷發生率高，需監測腎功能與尿量；與其他腎毒性藥（aminoglycoside、vancomycin、顯影劑）併用風險劇增
> 2. **神經毒性**：感覺異常（口周/肢端麻）、頭暈、**神經肌肉阻斷 → 呼吸抑制/停止**（與神經肌肉阻斷劑、麻醉、重症肌無力併用尤危險）
> 3. **Colistin 需給「負荷劑量」且劑量以 CMS/colistin base activity 換算複雜**：不同單位（IU vs mg CBA）換算錯誤是致命用藥錯誤來源 → 開藥務必核對院內換算表
> 4. **單獨用於肺炎滲透差**：靜脈給藥肺濃度低，重症 G(-) 肺炎常需**併吸入劑或合併其他抗生素**
> 5. **只涵蓋 G(-)**：對 G(+)、厭氧菌、Proteus/Serratia/Burkholderia 等天然抗藥菌無效

## 💊 適應症
- **多重抗藥 G(-) 感染最後防線**：CRE（抗碳青黴烯腸桿菌）、MDR/XDR Pseudomonas aeruginosa、Acinetobacter baumannii
- **院內/呼吸器相關肺炎（HAP/VAP）**（多合併治療 ± 吸入劑）[[Hospital-Acquired Pneumonia(院內肺炎)]]
- 複雜性泌尿道感染、菌血症（MDR G(-)）
- 吸入劑用於囊狀纖維化慢性 Pseudomonas 感染
- 腸道去污（口服不吸收，選擇性去污）

## 📏 劑量 / 用法（成人）⚠️劑量換算複雜、依院規/需查證
| 藥物 | 給藥 |
| --- | --- |
| Colistin（以 CMS 給） | **負荷劑量 ~9 million IU（≈300 mg CBA）**，之後維持約 4.5 MIU IV Q12H ⚠️依腎功能與院內換算表 |
| Polymyxin B | 負荷 2–2.5 mg/kg，維持 1.25–1.5 mg/kg IV Q12H（**Polymyxin B 不需依腎調整**，此點與 colistin 不同） |
| 吸入型 colistin | 依產品/院規 |
- **腎功能**：**Colistin（CMS）需依腎調整**；**Polymyxin B 不需依腎調整**（重要區別）
- **一定給負荷劑量**才能快速達治療濃度
- 換算單位（IU / mg CBA / mg polymyxin B）務必核對，避免致命劑量錯誤

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：腎功能惡化、口周/肢端感覺異常、頭暈、注射反應
- **嚴重**：
  - **急性腎損傷**（劑量限制毒性，多可逆但可嚴重）
  - **神經肌肉阻斷 → 呼吸抑制/停止**（可致命）
  - 神經毒性（意識改變、抽搐罕見）
  - 過敏反應；吸入劑可致支氣管痙攣

## 🚫 禁忌症
- 對多黏菌素過敏
- **重症肌無力**（神經肌肉阻斷惡化）— 相對/絕對禁忌
- 併用神經肌肉阻斷劑時高度警覺

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：**每日腎功能與尿量**、神經症狀（麻木/無力/呼吸）、電解質；重症併呼吸監測
- **交互作用**：
  - **其他腎毒性藥**（aminoglycoside、vancomycin、amphotericin B、顯影劑）→ 疊加腎毒性
  - **神經肌肉阻斷劑、麻醉劑**→ 呼吸抑制加乘
  - 與其他抗生素常「合併治療」以增效並延緩抗藥
- **孕哺**：孕期資料有限，僅在無替代且利大於弊時使用；哺乳慎用

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名 CRE 菌血症病人使用 colistin，第 3 天血中肌酐上升、尿量減少，最可能原因？
- A. Colistin 對 CRE 無效
- B. Colistin 的劑量限制毒性——腎毒性
- C. 劑量不足
- D. 與腎無關的過敏反應

> [!question]- 答案
> **B** — 腎毒性（急性腎損傷）是多黏菌素的劑量限制毒性，需監測腎功能與尿量，並避免併用其他腎毒性藥物。

**Q2.** 關於 colistin 與 polymyxin B 的比較，何者正確？
- A. 兩者皆需依腎功能調整劑量
- B. Colistin 以 CMS 前驅物給藥且需依腎調整，polymyxin B 不需依腎調整
- C. 兩者皆涵蓋 G(+) 菌
- D. 兩者皆無神經毒性

> [!question]- 答案
> **B** — Colistin 以 CMS 前驅物給藥且需依腎功能調整；polymyxin B 不需依腎調整。兩者皆只涵蓋 G(-)、皆有腎與神經毒性。
