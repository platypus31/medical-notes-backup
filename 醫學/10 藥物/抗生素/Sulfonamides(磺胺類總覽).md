# Sulfonamides（磺胺類總覽 / Sulfamethoxazole · Sulfadiazine）

> [!abstract] 一句話：**磺胺類**為 **PABA 結構類似物**，競爭性抑制 **dihydropteroate synthase**、阻斷細菌自行合成葉酸（folate）→ 抑制 DNA/RNA 合成（抑菌型）。因人類直接攝取葉酸不受影響，故具**選擇性毒性**。常與 **trimethoprim** 合併成 [[Trimethoprim-Sulfamethoxazole(TMP-SMX)]] 阻斷葉酸路徑兩步驟達殺菌與協同。**磺胺是嚴重皮膚藥物過敏（SJS/TEN）的經典元兇之一**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **Stevens-Johnson 症候群 / 毒性表皮壞死溶解（SJS/TEN）**：磺胺是最經典致命皮膚過敏藥物之一，出現黏膜糜爛/水疱/皮膚剝離立即停藥
> 2. **G6PD 缺乏 → 溶血性貧血**：氧化壓力誘發，開藥前留意 G6PD 病史
> 3. **新生兒 / 孕婦足月禁用 → 核黃疸（kernicterus）**：磺胺置換白蛋白上的膽紅素，游離膽紅素沉積新生兒腦部
> 4. **結晶尿 / 腎結石 / 急性腎損傷**：磺胺代謝物在酸性尿易析出結晶 → 衛教多喝水、維持尿量
> 5. **多種藥物交互作用**：增強 warfarin、sulfonylurea（低血糖）、methotrexate 毒性（置換蛋白結合 + 抑代謝）

## 💊 適應症
（磺胺單方現多被 TMP-SMX 取代，臨床以複方為主）
- **泌尿道感染**（歷史一線，現多 TMP-SMX）[[Urinary Tract Infection(泌尿道感染)]]
- **Pneumocystis jirovecii 肺炎（PCP）治療與預防**（TMP-SMX）
- **弓漿蟲腦炎**（sulfadiazine + pyrimethamine）
- **諾卡氏菌（Nocardia）感染**
- 局部外用（磺胺嘧啶銀 silver sulfadiazine 用於燒傷）
- 部分呼吸道、皮膚軟組織感染（依敏感性）

## 📏 劑量 / 用法（成人）
| 情境 | 說明 |
| --- | --- |
| 複方 TMP-SMX | 見 [[Trimethoprim-Sulfamethoxazole(TMP-SMX)]] 專卡（依適應症變動大） |
| Sulfadiazine（弓漿蟲） | 高劑量分次 + pyrimethamine + leucovorin ⚠️依院規/需查證 |
| Silver sulfadiazine | 外用塗抹燒傷創面 |
- **腎功能**：CrCl 下降需減量（磺胺代謝物腎排、易結晶）；重度腎損慎用
- **衛教多喝水維持尿量**，鹼化尿液可減少結晶（依情境）
- 磺胺單方現少單用，多為複方或特定適應症

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：噁心、皮疹、光敏感、頭痛、味覺改變
- **嚴重**：
  - **SJS / TEN / DRESS**（嚴重皮膚藥物過敏）
  - **G6PD 缺乏溶血性貧血**
  - **結晶尿 / 腎結石 / 急性腎損傷**
  - **新生兒核黃疸（kernicterus）**
  - 骨髓抑制（葉酸相關：貧血/白血球/血小板減少，尤缺葉酸者）
  - 肝毒性、高血鉀（TMP 成分）、無菌性腦膜炎（罕見）

## 🚫 禁忌症
- 對磺胺類過敏（含嚴重皮膚反應史）
- **孕婦足月（近分娩）、哺乳期未滿月/早產/黃疸嬰兒、新生兒（< 2 個月）**
- **G6PD 缺乏**（相對禁忌）
- 嚴重肝腎功能不全
- 巨球性貧血（葉酸缺乏相關）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：皮膚黏膜（警覺 SJS）、CBC（長療程/高劑量）、腎功能與尿量、電解質（K⁺）
- **交互作用**：
  - **Warfarin** → 增強抗凝（置換蛋白結合 + 抑代謝）→ 監測 INR
  - **Sulfonylurea（磺醯脲類降血糖藥）** → 低血糖
  - **Methotrexate** → 毒性增加（置換蛋白 + 疊加抑葉酸）
  - **Phenytoin** → 濃度上升
  - 尿液鹼化劑可減少結晶尿
- **孕哺**：孕早中期需權衡（抗葉酸致畸慮）、**足月禁用**（核黃疸）；哺乳新生兒/G6PD 嬰兒避免

## 🔗 相關
- 疾病：[[Urinary Tract Infection(泌尿道感染)]]、[[Pneumocystis Pneumonia(肺囊蟲肺炎)]]、[[Toxoplasmosis(弓漿蟲症)]]、[[Stevens-Johnson Syndrome(史蒂芬強生症候群)]]
- 同類 / 複方：[[Trimethoprim-Sulfamethoxazole(TMP-SMX)]]（磺胺 + TMP 阻斷葉酸兩步驟，臨床主力）
- 機轉對照：抑葉酸合成（vs β-lactam 抑細胞壁、macrolide 抑蛋白質合成）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 使用磺胺類藥物數天後出現廣泛皮膚水疱、口腔黏膜糜爛與皮膚剝離，最正確處置？
- A. 減量繼續使用
- B. 立即停藥，高度懷疑 SJS/TEN
- C. 加抗組織胺續用原藥
- D. 增加劑量

> [!question]- 答案
> **B** — 磺胺是 SJS/TEN 的經典元兇，出現黏膜糜爛/水疱/皮膚剝離應立即停藥、支持治療並轉燒傷/皮膚科，切勿再暴露同類藥物。

**Q2.** 關於磺胺類在特殊族群的禁忌，何者正確？
- A. 足月孕婦與新生兒可安全使用
- B. G6PD 缺乏者可致溶血、新生兒可致核黃疸，皆應避免
- C. 只有肝功能不良才需注意
- D. 對葉酸代謝無影響

> [!question]- 答案
> **B** — 磺胺在 G6PD 缺乏者致溶血、在新生兒/足月孕婦置換膽紅素致核黃疸，皆屬禁忌/避免；且會抑制葉酸代謝、致巨球性貧血。
