# Clindamycin（克林達黴素 / Dalacin C）

> [!abstract] 一句話：**林可醯胺類**，結合細菌 **50S 核糖體**（與 macrolide 相同 23S 位點）阻斷肽鏈延長 → 抑制蛋白質合成（抑菌型）。強項是**厭氧菌**與 **G(+)（含部分 MRSA/GAS）**，並能**抑制外毒素合成**（用於毒素介導疾病如壞死性筋膜炎、鏈球菌毒性休克）。**橫膈以上厭氧菌**涵蓋佳。最惡名昭彰的副作用是 **C. difficile 偽膜性腸炎**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **C. difficile 偽膜性結腸炎的經典元兇**：是最容易引起 CDI 的抗生素之一，出現用藥後嚴重水瀉/血便/發燒 → 停藥 + 驗 C. diff toxin，勿給止瀉劑
> 2. **不涵蓋大多數 G(-) 桿菌與腸道厭氧菌抗藥株**：橫膈以下（腹腔）厭氧菌（如 B. fragilis）抗藥漸增，重症腹腔感染別單靠它
> 3. **誘導型 MLSb 抗藥（inducible resistance）**：對 erythromycin 抗藥但 clindamycin 敏感的葡萄球菌/鏈球菌，需做 **D-test**，D-test 陽性代表治療中可能誘導抗藥而失敗
> 4. **加在壞死性筋膜炎/毒性休克要「合併」而非單用**：clindamycin 抑毒素但殺菌力弱，需 + β-lactam（如 penicillin）
> 5. **不穿血腦障壁**：不可用於中樞神經感染/腦膜炎

## 💊 適應症
- **厭氧菌感染（橫膈以上）**：吸入性肺炎、肺膿瘍、口腔/齒源性感染、扁桃體周膿瘍
- **皮膚軟組織感染**：蜂窩性組織炎、膿瘍（含社區型 MRSA 選項）
- **壞死性筋膜炎 / 鏈球菌毒性休克症候群**（+ penicillin，抑制外毒素）[[Necrotizing Fasciitis(壞死性筋膜炎)]]
- **骨髓炎、關節感染**（骨穿透性佳）
- **弓漿蟲腦炎、瘧疾、細菌性陰道炎（陰道劑）、痤瘡（外用）**
- Penicillin 過敏者做**牙科/手術預防性投藥**替代

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 一般感染 PO | **300–450 mg PO Q6–8H** |
| 中重度感染 IV | **600–900 mg IV Q8H** |
| 壞死性筋膜炎 / 毒性休克 | 900 mg IV Q8H（+ penicillin） |
| 吸入性肺炎/肺膿瘍 | 600 mg IV Q8H |
- **腎功能**：**不需依腎調整**（主要經肝代謝）
- **肝功能**：嚴重肝損慎用/監測
- IV 需稀釋慢輸（快速推注可致低血壓、心跳停止風險）

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：腹瀉、噁心、皮疹、金屬味、口服食道刺激
- **嚴重**：
  - **C. difficile 相關腹瀉 / 偽膜性結腸炎**（可致中毒性巨結腸、致命）
  - 過敏反應（含罕見 SJS/TEN、DRESS）
  - 肝酵素上升、罕見肝毒性
  - 快速 IV 注射 → 低血壓、心律不整
  - 血球低下（罕見嗜中性球/血小板減少）

## 🚫 禁忌症
- 對 clindamycin 或 lincomycin 過敏
- 曾因 clindamycin 發生偽膜性結腸炎者慎用
- 中樞神經感染（不穿 BBB，非適應症）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：腹瀉/腸炎症狀（最重要）；長療程監肝功能與 CBC
- **交互作用**：
  - **神經肌肉阻斷劑**：clindamycin 有神經肌肉阻斷作用 → 加乘（麻醉/重症肌無力注意）
  - **Erythromycin**：拮抗（爭同一 50S 位點），勿併用
  - CYP3A4 誘導劑（rifampin）可能降低血中濃度
- **孕哺**：孕期一般可用（臨床常用）；乳汁分泌少量，哺乳大致可用但注意嬰兒腸道症狀

### 自我測驗（選擇題，答案摺疊）

**Q1.** 使用 clindamycin 一週後出現嚴重水瀉、發燒與腹痛，最正確的處置是？
- A. 給予 loperamide 止瀉
- B. 停 clindamycin、送驗 C. difficile toxin
- C. 增加 clindamycin 劑量
- D. 給予益生菌即可

> [!question]- 答案
> **B** — 高度懷疑 C. difficile 偽膜性結腸炎，應停藥並送驗毒素，勿用止瀉劑（易致毒性巨結腸），確診後以口服 vancomycin 或 fidaxomicin 治療。

**Q2.** 關於 clindamycin 在壞死性筋膜炎的角色，何者正確？
- A. 可單獨使用，殺菌力極強
- B. 主要價值在抑制外毒素，須與 penicillin 合併
- C. 能穿越血腦障壁治療併發腦膜炎
- D. 只涵蓋 G(-) 桿菌

> [!question]- 答案
> **B** — Clindamycin 抑制核糖體使外毒素合成停止（Eagle effect 相關），但殺菌力弱，須與 β-lactam（penicillin）合併，並不涵蓋 G(-) 也不穿 BBB。
