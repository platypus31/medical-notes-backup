# Daptomycin（達托黴素 / Cubicin）

> [!abstract] 一句話：**環脂肽抗生素**，鈣依賴地插入 G(+) 細菌**細胞膜**、形成孔洞造成鉀離子外流、膜去極化 → **快速殺菌（濃度依賴 concentration-dependent）**。專打抗藥 G(+)：**MRSA、VRE**。**最關鍵鐵律：不可用於肺炎**——會被肺泡表面活性物質（surfactant）結合失活。主力戰場是 **MRSA 菌血症 / 右側心內膜炎**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **絕不用於肺炎**：daptomycin 被肺泡 surfactant 失活，肺部感染無效（即使菌血症源自肺炎也要換藥治肺）
> 2. **橫紋肌溶解 / CPK 上升**：每週監測 **CPK**，出現肌痛/無力 + CPK 明顯升高（>5–10×ULN 或有症狀）要停藥
> 3. **與 statin 併用增肌毒性**：療程中考慮暫停 statin，並密切監測 CPK
> 4. **嗜酸性肺炎（eosinophilic pneumonia）**：用藥數週後新發呼吸困難/發燒/嗜酸球增加 + 浸潤 → 罕見但需停藥
> 5. **腎功能不良需延長給藥間隔**：CrCl < 30 或透析者改 Q48H，劑量不當易累積毒性

## 💊 適應症
- **MRSA 菌血症 / 右側感染性心內膜炎**（金黃色葡萄球菌，首選之一）[[Infective Endocarditis(感染性心內膜炎)]]
- **複雜性皮膚軟組織感染**（G(+)，含 MRSA）
- **VRE 感染 / 菌血症**（常用選項）
- 骨髓炎、關節/人工裝置感染（G(+)，多合併或依感染科建議）
- vancomycin 失敗、MIC 偏高或不耐者的替代

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 複雜性皮膚軟組織感染 | **4 mg/kg IV Q24H** |
| 菌血症 / 右側心內膜炎 | **6 mg/kg IV Q24H**（重症/深部感染常用 8–10 mg/kg，依感染科） |
- **腎功能**：CrCl < 30（含血透/CAPD）→ **改 Q48H**（血透日於透析後給）
- **給藥**：IV 每日一次（濃度依賴殺菌），快速輸注或注射皆可
- 療程中**每週監測 CPK**（合併 statin 或腎不良者更頻繁）

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：注射部位反應、噁心、腹瀉、頭痛、失眠、CPK 輕度上升
- **嚴重**：
  - **橫紋肌溶解 / 肌肉毒性**（CPK 上升、肌痛/無力）
  - **急性嗜酸性肺炎**（罕見但可致命，需停藥）
  - 周邊神經病變（罕見）
  - 過敏反應、C. difficile 腸炎
  - 假性延長 PT/INR（實驗室干擾，非真凝血異常）

## 🚫 禁忌症
- 對 daptomycin 過敏
- **肺炎（治療目的）** — 藥理上無效，屬使用禁忌情境
- 與可能致肌毒性藥物併用時需審慎（相對禁忌）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：**每週 CPK**（有肌症狀更頻繁）、腎功能、呼吸症狀（警覺嗜酸性肺炎）
- **交互作用**：
  - **HMG-CoA reductase 抑制劑（statin）**：疊加肌毒性 → 療程中考慮暫停 statin
  - 其他潛在肌毒性藥（fibrate）注意
  - 干擾某些凝血試劑造成 **PT/INR 假性延長**（抽血時機避開峰值、換試劑）
- **孕哺**：孕期資料有限（B 級概念），利大於弊時使用；哺乳資料有限

### 自我測驗（選擇題，答案摺疊）

**Q1.** MRSA 肺炎併菌血症的病人，關於 daptomycin 何者正確？
- A. Daptomycin 是肺炎首選
- B. Daptomycin 可治菌血症但對肺炎無效（被 surfactant 失活）
- C. 只要加大劑量就能治肺炎
- D. Daptomycin 對 MRSA 無效

> [!question]- 答案
> **B** — Daptomycin 被肺泡 surfactant 失活，肺部感染無效。若有肺炎需改用能治肺的抗 MRSA 藥（如 vancomycin、linezolid）。

**Q2.** 使用 daptomycin 的病人出現雙腿肌痛與 CPK 明顯上升，最合理處置？
- A. 繼續原劑量觀察
- B. 評估肌毒性、停用併服的 statin、視情況停 daptomycin
- C. 增加 daptomycin 劑量
- D. 這與 daptomycin 無關

> [!question]- 答案
> **B** — Daptomycin 可致肌肉毒性/橫紋肌溶解，與 statin 疊加。應監測並暫停 statin，若 CPK 大幅升高或有症狀應停 daptomycin。
