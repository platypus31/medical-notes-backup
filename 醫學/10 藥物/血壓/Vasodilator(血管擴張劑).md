# Vasodilator（血管擴張劑）

> [!abstract] 一句話：直接鬆弛血管平滑肌降低血管阻力。**依作用血管床分：動脈型**（降 afterload：Hydralazine、Minoxidil、Fenoldopam）、**靜脈型**（降 preload：硝酸鹽 NTG/ISDN/ISMN）、**動脈+靜脈混合**（Nitroprusside）。多為**第二/三線或急症用藥**，因反射性心搏過速與鈉水滯留常需併 β-blocker + 利尿劑。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **反射性心搏過速 + 鈉水滯留** → 單用會抵銷降壓、誘發心絞痛；動脈型常需併 β-blocker + 利尿劑
> 2. **Nitroprusside → 氰化物 / 硫氰酸中毒**（長時間、大劑量、腎/肝功能差）→ 代謝性酸中毒、意識改變；避光、限時、監測
> 3. **硝酸鹽 + PDE5 抑制劑（sildenafil 等）= 絕對禁忌** → 致命性低血壓
> 4. **硝酸鹽耐受性（tolerance）** → 需每日留 10–12 小時 nitrate-free interval
> 5. **Hydralazine → 藥物性紅斑性狼瘡**（劑量 >200 mg/day、慢乙醯化者）
> 6. **右心室梗塞 / 前負荷依賴者用硝酸鹽 → 嚴重低血壓**

## 💊 適應症（治療哪些疾病）
- **高血壓急症 / urgency**：Nitroprusside、Fenoldopam、Nicardipine（→ [[Hypertensive emergency(高血壓急症)]]）
- **心絞痛 / ACS**：硝酸鹽（NTG 舌下/IV 減 preload、緩解胸痛）→ [[Acute Coronary Syndrome(急性冠心症)]]
- **急性 / 慢性心衰竭**：Hydralazine + 硝酸鹽（尤其**非裔、無法用 ACEI/ARB 者**降死亡率）；急性肺水腫用 NTG
- **頑固性高血壓**：Minoxidil（最強口服降壓，用於他藥無效者）
- **孕期高血壓 / 子癲前症**：Hydralazine（IV，孕期可用）
- 其他：Minoxidil 外用治雄性禿；ISMN/ISDN 慢性心絞痛預防

## 📏 劑量 / 用法（成人）
| 藥 | 血管床 / 機轉 | 常用劑量 | 重點 |
| --- | --- | --- | --- |
| Hydralazine | 動脈，機轉未全明 | 10–50 mg PO QID；IV 10–20mg（急症/孕期）| SLE、反射心搏過速，常配 β+利尿 |
| Minoxidil | 動脈，開 K⁺ 通道 | 2.5–10 mg/day | **最強口服降壓**；多毛、水腫，須配 β+loop |
| Nitroprusside | 動脈+靜脈，釋 NO | IV 0.3–10 mcg/kg/min 滴定 | **高血壓急症**；避光、限時、防氰中毒 |
| Fenoldopam | 動脈，D1 agonist | IV 0.1–0.3 mcg/kg/min | 高血壓急症、可保腎血流 |
| NTG（硝化甘油） | 靜脈>動脈，釋 NO | 舌下 0.4mg / IV 滴定 | ACS 胸痛、急性肺水腫 |
| ISDN / ISMN | 靜脈，釋 NO | ISDN TID~QID / ISMN QD~HS | 慢性心絞痛預防，留無藥期 |
- **腎肝**：Nitroprusside 腎/肝差時氰化物累積風險高，限用

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：頭痛（硝酸鹽典型）、臉潮紅、姿勢性低血壓、反射心搏過速、下肢水腫、Minoxidil 多毛
- **嚴重**：
  - **氰化物 / 硫氰酸中毒**（Nitroprusside）
  - **藥物性 SLE**（Hydralazine，可逆，停藥改善）
  - **嚴重低血壓**（+ PDE5i、右室梗塞）
  - Minoxidil：心包積液、明顯鈉水滯留

## 🚫 禁忌症
- **硝酸鹽**：併用 PDE5 抑制劑（sildenafil/tadalafil）、右室梗塞、嚴重主動脈瓣狹窄、肥厚性阻塞型心肌病
- **Nitroprusside**：嚴重肝腎不全（相對）、代償性高血壓
- **Hydralazine**：既有 SLE、冠心病單用（誘發心絞痛）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：血壓（急症連續動脈壓）、HR、Nitroprusside 監測酸鹼/乳酸/硫氰酸濃度
- **交互作用**：
  - **硝酸鹽 + PDE5i** → 致命低血壓（絕對禁忌）
  - 動脈型 + 交感興奮 → 反射心搏過速（配 β-blocker）
- **孕哺**：**Hydralazine 為孕期高血壓/子癲前症可用**；Nitroprusside 孕期避免（胎兒氰化物）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 胸痛 ACS 病人到急診，護理師要給舌下 NTG。給之前最該問的一句話是？
- A. 有沒有氣喘
- B. 24–48 小時內有沒有服用威而鋼類（PDE5 抑制劑）
- C. 有沒有痛風
- D. 有沒有便秘

> [!success]- 答案
> **B** — 硝酸鹽與 PDE5 抑制劑併用會造成**致命性低血壓**（絕對禁忌），給 NTG 前務必確認近期未服 sildenafil/tadalafil 等；同時留意右室梗塞者亦禁。

**Q2.** 高血壓急症用 Nitroprusside IV 滴注 24 小時後，病人出現代謝性酸中毒、意識改變、乳酸升高。最可能？
- A. 敗血症
- B. 氰化物中毒
- C. 低血糖
- D. 藥物性 SLE

> [!success]- 答案
> **B** — Nitroprusside 代謝產生氰化物，長時間/大劑量或肝腎功能差時累積 → **氰化物中毒**（乳酸性酸中毒、意識改變），須停藥、給解毒劑（hydroxocobalamin/sodium thiosulfate）。

**Q3.** 難治型高血壓加用 Minoxidil 後血壓下降但心跳變快、雙腳水腫。最正確的合併處置是？
- A. 停 Minoxidil
- B. 併用 β-blocker（抑反射心搏過速）+ loop 利尿劑（處理鈉水滯留）
- C. 加另一個動脈擴張劑
- D. 加 PDE5 抑制劑

> [!success]- 答案
> **B** — 強力動脈擴張劑（Minoxidil）會引起反射性心搏過速與明顯鈉水滯留，標準做法是**併用 β-blocker + loop 利尿劑**以抵銷此兩效應，而非停藥。
