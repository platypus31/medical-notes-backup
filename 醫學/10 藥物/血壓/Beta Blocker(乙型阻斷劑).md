# Beta Blocker（乙型阻斷劑 / -olol）

> [!abstract] 一句話：阻斷 **β-腎上腺素受體** → 心率 ↓、心收縮力 ↓、心輸出 ↓、腎素分泌 ↓ → 降壓、抗心絞痛、控制心律、降 HFrEF 死亡率。字尾 **-olol**。分三型：**選擇性 β1**（心臟為主，較不誘發氣喘）、**非選擇性 β1+β2**（Propranolol）、**α1+β**（Carvedilol、Labetalol）。降壓效果較弱，但在 **CAD/心衰竭/心律** 有不可取代角色。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **急性失代償心衰竭（濕、喘、不穩）禁止起始/加量** → 負性肌力會壓垮心輸出；心衰竭 β-blocker 只在**穩定期慢慢加**
> 2. **氣喘 / 嚴重 COPD** → β2 阻斷誘發支氣管痙攣（非選擇性尤甚，選擇性 β1 相對安全但非零風險）
> 3. **心搏過緩 / 二三度房室傳導阻滯 / SSS** → 禁用（會更慢、停搏）
> 4. **遮蔽低血糖警覺**（心悸、顫抖被壓，冒汗仍在）→ 糖尿病易發生無感低血糖
> 5. **不可驟停** → 上調的 β 受體反彈 → 反彈性高血壓、心搏過速、心絞痛甚至心梗；需逐步減量
> 6. **不與 non-DHP CCB（Verapamil/Diltiazem）併用** → 加乘的心搏過緩 / 房室阻滯 / 心衰

## 💊 適應症（治療哪些疾病）
- **高血壓** [[Hypertension(高血壓)]]（現非單純高血壓一線，但合併下列時優先）
- **冠心病 / 心絞痛** [[Chronic Stable Angina(慢性穩定型心絞痛)]]、[[Acute Coronary Syndrome(急性冠心症)]]（降 HR 減心肌耗氧）
- **心肌梗塞後二級預防** [[Acute Myocardial infarction(急性心肌梗塞)]]（降死亡率）
- **收縮性心衰竭 HFrEF**：**僅 Bisoprolol、Metoprolol succinate、Carvedilol 三者有實證降死亡率**（記！）
- **心律控制**：[[醫學/11心電圖判讀/Atrial Fibrillation(心房顫動)]] rate control、SVT
- **其他（Propranolol 多用途）**：[[Hyperthyroidism(甲狀腺機能亢進)]] 交感症狀、[[Portal Hypertension(門脈高壓)]] 靜脈曲張出血預防、原發性顫抖、偏頭痛預防、表現焦慮

## 📏 劑量 / 用法（成人）
| 藥（商品名） | 選擇性 | 常用起始→維持 | 定位 |
| --- | --- | --- | --- |
| Bisoprolol（Concor） | β1 選擇 | 1.25 mg QD →10 mg QD | **心衰竭三藥之一**、乾淨 |
| Metoprolol（succinate ER） | β1 選擇 | 25 mg QD →200 mg QD | **心衰竭用 succinate（緩釋）** |
| Atenolol | β1 選擇 | 25–50 mg QD →100 mg QD | 老藥、腎排除 |
| Propranolol（Inderal） | 非選擇 β1β2 | 10 mg TID | 甲亢/門脈高壓/顫抖/偏頭痛 |
| Carvedilol（Dilatrend） | α1+β | 3.125 mg BID →25 mg BID | **心衰竭三藥之一** |
| Labetalol | α1+β | 100 mg BID（口服）/ IV | 降壓溫和、**孕期可用**、高血壓急症針劑 |
- **心衰竭原則**：務必「**start low, go slow**」，穩定期每 2 週倍增；急性失代償不加
- **腎肝**：Atenolol 腎排除需減量；Propranolol/Carvedilol/Labetalol 肝代謝

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：疲倦、心搏過緩、四肢冰冷 / 跛行（β2 → 周邊血管收縮）、性功能障礙、頭暈
- **嚴重**：
  - **心搏過緩 / 房室傳導阻滯**
  - **急性心衰竭惡化**（不當使用時）
  - **支氣管痙攣**（氣喘/COPD）
  - **遮蔽低血糖**
  - **憂鬱、睡眠障礙、惡夢**（脂溶性如 Propranolol 進 CNS）
  - **驟停反彈**（反彈性高血壓/心絞痛）

## 🚫 禁忌症
- 急性失代償性心衰竭（不穩定期）
- 嚴重心搏過緩、二/三度 AV block、SSS（無節律器保護）
- 氣喘急性發作 / 嚴重反應性呼吸道疾病（非選擇性）
- 未控制的低血壓、心因性休克
- 與 Verapamil/Diltiazem 併用（相對禁忌）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：HR、血壓、心衰竭症狀（體重/水腫/呼吸）、糖尿病人低血糖警覺
- **交互作用**：
  - **Non-DHP CCB（Verapamil/Diltiazem）** → 心搏過緩 / AV block / 心衰（重要！）
  - **胰島素 / 磺醯尿素** → 遮蔽低血糖症狀
  - **Clonidine** → 若同時停 Clonidine，β-blocker 未停會惡化反彈性高血壓
  - **TCA（三環抗憂鬱劑）** → 可能 tremor（Atenolol 脂溶性低較少 CNS 副作用；tremor 反可用 Propranolol 治）
  - Thiazide 併用 → ↑ 新發糖尿病風險
- **孕哺**：**Labetalol 為孕期高血壓首選之一**；Atenolol 孕期避免（胎兒生長遲滯）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 62 歲 HFrEF（EF 30%）病人門診穩定，欲加 β-blocker 降死亡率。下列最適當？
- A. Propranolol 逐步加量
- B. Atenolol 100mg QD
- C. Carvedilol 3.125mg BID 起始，穩定則每 2 週倍增
- D. 急性喘時立刻加最大劑量

> [!success]- 答案
> **C** — 心衰竭具實證的 β-blocker 為 Bisoprolol / Metoprolol succinate / Carvedilol，須在**穩定期低劑量起始、緩慢加量**；急性失代償期禁止起始或加量。

**Q2.** 氣喘病史的高血壓病人需 β-blocker 控制心絞痛。若非用不可，相對較安全的選擇是？
- A. Propranolol
- B. 選擇性 β1（如 Bisoprolol）並嚴密監測
- C. Carvedilol
- D. Labetalol

> [!success]- 答案
> **B** — 非選擇性（Propranolol）與含 β 的 α+β（Carvedilol/Labetalol）較易誘發支氣管痙攣；若氣喘病人必須用，選**選擇性 β1** 並密切觀察呼吸症狀。

**Q3.** 糖尿病病人服 β-blocker 後低血糖時「沒有平常的心悸手抖，直接意識模糊」。原因？
- A. β-blocker 使血糖更低
- B. β-blocker 遮蔽交感介導的低血糖警覺（心悸/顫抖），冒汗仍在
- C. 一定是胰島素過量
- D. 與 β-blocker 無關

> [!success]- 答案
> **B** — β-blocker 會**遮蔽心悸、顫抖**等交感警覺症狀（冒汗由膽鹼介導較不受影響），造成無感低血糖，是糖尿病人用藥的重要交互風險。
