# Acetazolamide（乙醯唑胺 / Diamox）

> [!abstract] 一句話：**碳酸酐酶抑制劑**，抑制近曲小管（與眼睫狀體、脈絡叢）的碳酸酐酶→**HCO₃⁻（碳酸氫根）隨尿排出**，產生弱利尿並使尿液鹼化、身體傾向**代謝性酸中毒**。臨床少當利尿劑，主要用於**青光眼降眼壓、預防與治療高山症、代謝性鹼中毒的矯正、部分癲癇與特發性顱內高壓**。屬 **sulfa 類**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **代謝性酸中毒（正常陰離子間隙、高血氯型）**：排 HCO₃⁻ 的直接結果；慢性阻塞性肺病或已酸中毒者慎用，可加重
> 2. **低血鉀（hypokalemia）**：HCO₃⁻ 到遠端帶走 K⁺ → 低血鉀，尤其併用其他利尿劑或原本低鉀時
> 3. **腎結石（尤鈣磷酸鹽結石）**：尿液鹼化 + 枸櫞酸尿減少 → 促結石；有結石病史者慎用
> 4. **Sulfa 過敏**：acetazolamide 為磺胺衍生物，磺胺嚴重過敏者慎用/避免（含 Stevens-Johnson 風險）
> 5. **利尿效果會自限（self-limiting）**：數天後 HCO₃⁻ 耗竭→利尿與降眼壓效果減弱（耐受），不適合長期單用利尿

## 💊 適應症（治療哪些疾病）
- **青光眼 [[Glaucoma(青光眼)]]**（降眼內壓：減少睫狀體房水生成；急性閉角青光眼與開角型皆可輔助）
- **高山症（急性高山病 / 預防）[[Acute Mountain Sickness(高山症)]]**（誘發代謝性酸中毒刺激呼吸→改善通氣與適應；登山前一天起預防性使用）
- **代謝性鹼中毒 [[Metabolic Alkalosis(代謝性鹼中毒)]]**（尤其利尿劑引起、心衰合併鹼中毒時；排 HCO₃⁻ 矯正）
- **特發性顱內高壓（IIH / 假性腦瘤）[[Idiopathic Intracranial Hypertension(特發性顱內高壓)]]**（減少脈絡叢腦脊液生成）
- **輔助性癲癇治療、部分週期性麻痺**
- **（矯正呼吸性酸中毒代償的過度鹼中毒，脫呼吸器輔助）**

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 高山症預防 | **125–250 mg PO Q12H**，登高前 1 天開始，持續至最高點後 2–3 天 |
| 急性高山病治療 | 250 mg PO Q12H（連同下降海拔） |
| 青光眼 | 250 mg PO QID，或緩釋 500 mg BID；急性可 500 mg IV/PO 起始 |
| 代謝性鹼中毒 | 250–500 mg IV/PO（單劑或短期）⚠️依血氣與院規調整 |
| 特發性顱內高壓 | 起始 250–500 mg BID，可依耐受漸增 ⚠️依神經科指引 |
- **腎功能**：eGFR 下降時作用減弱且代謝性酸中毒風險增；**嚴重腎損（CrCl < 10）避免使用**
- **監測血鉀與酸鹼狀態**，長期使用注意結石
- ⚠️各適應症劑量與療程依院規 / 眼科 / 神經科指引覆核

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：肢端與口周**感覺異常（paresthesia，麻刺感）**、味覺改變（碳酸飲料變味）、多尿、疲倦、食慾不振
- **嚴重**：
  - **代謝性酸中毒**（正常陰離子間隙）
  - **低血鉀**、電解質紊亂
  - **腎結石**
  - 磺胺類過敏反應（皮疹、罕見 **Stevens-Johnson / TEN**、血液惡病質如再生不良性貧血——罕見但嚴重）
  - 少見肝損、代謝性腦病（肝硬化者因 NH₃ 排泄受影響）

## 🚫 禁忌症
- 對 acetazolamide 或磺胺（sulfa）過敏
- **嚴重腎功能不全、嚴重肝功能不全 / 肝硬化**（誘發肝性腦病）
- **已存在的代謝性酸中毒、嚴重低血鉀 / 低血鈉**
- 腎上腺功能不全（Addison's）
- 慢性閉角型青光眼長期單用（掩蓋惡化）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：電解質（**K⁺、Cl⁻、HCO₃⁻**）、酸鹼（血氣/靜脈 CO₂）、腎功能；長期使用注意結石與 CBC
- **交互作用**：
  - **Aspirin / 高劑量水楊酸**：可增水楊酸毒性（代謝性酸中毒加重中樞毒性）
  - **其他利尿劑**：低血鉀加成
  - **尿液鹼化**：改變弱酸/弱鹼藥物排泄（如影響 amphetamine、quinidine、methotrexate 清除）
  - **Topiramate**：兩者皆致代謝性酸中毒與結石，加成
- **孕哺**：孕期資料有限，權衡利弊使用；哺乳少量分泌，一般可用但個別評估

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名登山客服用 acetazolamide 預防高山症，其改善高原適應的主要機轉是？
- A. 直接擴張肺血管
- B. 誘發代謝性酸中毒刺激呼吸中樞增加通氣
- C. 增加血紅素攜氧
- D. 收縮腦血管降 ICP

> [!success]- 答案
> **B** — Acetazolamide 排 HCO₃⁻ 造成**代謝性酸中毒**，刺激呼吸驅動增加通氣量、加速對高海拔低氧的適應。

**Q2.** 使用 acetazolamide 後最典型的血氣與電解質變化組合是？
- A. 代謝性鹼中毒 + 高血鉀
- B. 正常陰離子間隙代謝性酸中毒 + 低血鉀
- C. 呼吸性酸中毒 + 高血鈉
- D. 陰離子間隙增高的代謝性酸中毒

> [!success]- 答案
> **B** — 排 HCO₃⁻ →**正常陰離子間隙（高血氯型）代謝性酸中毒**，並帶走 K⁺ 導致**低血鉀**。

**Q3.** 下列何者是 acetazolamide 的禁忌或需高度警覺情境？
- A. 磺胺（sulfa）嚴重過敏史
- B. 需要預防高山症
- C. 利尿劑引起的代謝性鹼中毒
- D. 開角型青光眼

> [!success]- 答案
> **A** — Acetazolamide 為**磺胺衍生物**，磺胺嚴重過敏者應避免（含 Stevens-Johnson 風險）；B/C/D 均為其合理適應症。
