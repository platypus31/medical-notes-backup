# Vaptans（Tolvaptan / Conivaptan · 血管加壓素拮抗劑）

> [!abstract] 一句話：阻斷腎集尿管上的 **Vasopressin V2 受體**（ADH 的水通道作用）→ **排水不排鈉（aquaresis）** → 提升血鈉；與傳統利尿劑不同，它不改變電解質的排出比例，是治療「稀釋性低血鈉」的機轉性用藥。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **矯正低血鈉太快 → 滲透性脫髓鞘症候群（osmotic demyelination syndrome / 中央橋腦髓鞘溶解 CPM）**，可致不可逆神經損傷。用藥初期須**住院密切監測血鈉**。
> 2. 血鈉上升速度須有上限（建議前 24–48 小時不宜過快，常見教科書上限約 **<8–10 mEq/L / 24hr**，**院內確切 protocol 需查證**）；用藥期間**暫停不限水限制**、頻繁追蹤血鈉。
> 3. **Tolvaptan 有肝毒性黑框警語** → 用於多囊腎時限制連續使用天數並監測肝功能，避免用於已有肝病者。
> 4. **Conivaptan 僅 IV、且為非選擇性（V1a + V2）**，V1a 阻斷會擴張血管 → **低血容積性低血鈉者須謹慎**（可能加重低血壓）。

## 💊 適應症（治療哪些疾病）
- **[[SIADH(抗利尿激素不當分泌症候群)]] 所致的低血鈉** —— 機轉正對症（過多 ADH → 集尿管過度回收水 → 稀釋性低血鈉）。
- **心衰竭合併低血鈉**（等 / 高血容積性低血鈉）—— 排水提鈉、緩解容積過載的稀釋性低鈉。
- **Tolvaptan 另一大適應症：多囊腎病（ADPKD, autosomal dominant polycystic kidney disease）** —— 減緩囊腫生長與腎功能下降（此為口服 tolvaptan 的重要專屬用途）。
- **不適用**：低血容積性（脫水型）低血鈉不是首選；高血鈉、無尿者禁用。

## 📏 劑量 / 用法（成人）
| 藥物 | 途徑 | 用途 / 用法重點 |
| --- | --- | --- |
| **Tolvaptan** | 口服 | 等 / 高血容積性低血鈉、ADPKD；**起始須住院監測血鈉**，起始低劑量再視反應調整（**確切 mg 依院規 / 需查證**） |
| **Conivaptan** | 僅 IV | 住院病人等 / 高血容積性低血鈉；先 loading 再持續輸注（**劑量依院規 / 需查證**） |
- **共通原則**：起始 / 調量期間**必須在醫院內**，頻繁追蹤血鈉；血鈉上升過快時**減量 / 停藥並可補充自由水（如 D5W）** 減緩上升。
- 用藥期間**解除限水**（否則血鈉可能升太快）。
- 確切起始劑量、輸注速率與監測頻率 **依院規 / 需查證**。

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：口渴、多尿、口乾、頻尿、脫水傾向、姿勢性低血壓、高血鈉（矯正過度時）。
- **嚴重**：
  - **滲透性脫髓鞘症候群（ODS / CPM）** —— 矯正太快所致，最致命。
  - **Tolvaptan 肝毒性**（黑框警語，尤其 ADPKD 長期使用）—— 監測 ALT/AST/膽紅素。
  - **Conivaptan（非選擇性）** 因 V1a 阻斷 → 血管擴張、低血壓，低血容者更明顯。
  - 脫水 / 高血鈉（若飲水不足或監測不當）。

## 🚫 禁忌症
- **需要立即升高血鈉的急症**（如嚴重症狀性低血鈉、癲癇 / 昏迷）不宜單靠 vaptan 慢調 —— 這類須用高張鹽水在密切監測下處理。
- **無法感知或反應口渴、無法適當飲水者**（易矯正過快 / 脫水）。
- **低血容積性（脫水型）低血鈉**。
- 無尿（anuria）。
- Tolvaptan：明顯肝功能異常 / 肝病者（黑框）。
- 與強效 CYP3A4 抑制劑併用（濃度大增，見下）。

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：**血鈉**（起始期每數小時 ~ 每日，避免上升過快）、容積狀態、血壓、**肝功能（tolvaptan）**、腎功能。
- **交互作用**：
  - **CYP3A4 受質**：強效 CYP3A4 抑制劑（如 ketoconazole、clarithromycin、葡萄柚汁）**大幅升高** vaptan 濃度 → 避免併用；CYP3A4 誘導劑則降低療效。
  - 與其他升鈉措施（高張鹽水、限水）同時進行時，血鈉可能疊加上升過快 —— 須整合監測。
- **孕哺**：一般不建議孕期使用（**依院規 / 需查證**）。

## 🔗 相關
- 疾病：[[SIADH(抗利尿激素不當分泌症候群)]]、低血鈉（hyponatremia）、多囊腎病 ADPKD
- 對照 / 機轉：傳統利尿劑（[[Loop Diuretics(環利尿劑總覽)]]、[[Thiazide Diuretics(噻嗪類利尿劑總覽)]]）—— 那些「排水又排鈉」；vaptan「只排水不排鈉」
- 相反狀況：Desmopressin（DDAVP，V2 致效劑，用於尿崩症 / DI）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位 SIADH 導致慢性低血鈉（Na 118 mEq/L）的病人開始使用 tolvaptan。用藥初期最重要的監測 / 安全考量是？
- A. 每日量體重即可，血鈉不必頻繁追
- B. 密切追蹤血鈉，避免上升過快導致滲透性脫髓鞘
- C. 嚴格限水以加速升鈉
- D. 立即合併高張鹽水快速拉高血鈉

> [!success]- 答案
> **B** — Vaptan 起始須**住院密切監測血鈉**，血鈉上升過快會造成**滲透性脫髓鞘症候群（ODS/CPM）**。用藥期間反而要**解除限水**、控制升鈉速度（常見上限約 <8–10 mEq/L/24hr，院規需查證）。

**Q2.** 關於 vaptan 的機轉，下列敘述何者正確？
- A. 抑制 Na-K-2Cl 共同轉運子，排水又排鈉
- B. 拮抗集尿管 V2 受體，排水不排鈉（aquaresis）
- C. 抑制碳酸酐酶，排 HCO3
- D. 拮抗醛固酮受體，保鉀排鈉

> [!success]- 答案
> **B** — Vaptan 拮抗集尿管的 **V2 受體**（ADH 水通道），造成**只排自由水、不排鈉**的 aquaresis，因此能提升稀釋性低血鈉的血鈉。A 是 loop、C 是 acetazolamide、D 是保鉀利尿劑。

**Q3.** 下列哪一項是 tolvaptan 的重要黑框 / 專屬考量？
- A. 造成高血鉀
- B. 肝毒性（尤其用於多囊腎長期治療）
- C. 造成代謝性鹼中毒
- D. 導致嚴重低血糖

> [!success]- 答案
> **B** — Tolvaptan 具**肝毒性黑框警語**，用於 **ADPKD（多囊腎）** 長期治療時須限制連續天數並監測肝功能。
