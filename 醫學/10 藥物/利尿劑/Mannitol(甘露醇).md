# Mannitol（甘露醇）

> [!abstract] 一句話：**滲透性利尿劑**，靜脈給予的六碳糖醇不被腎小管再吸收，把水「拉」在管腔內帶走→利尿；同時提高血漿滲透壓，把腦組織與眼球的水分拉回血管內→**降顱內壓 (ICP)、降眼內壓 (IOP)**。值班用於**腦水腫/顱內高壓、青光眼危象**；給藥前務必確認**尿量（有無無尿）與心肺負荷**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **必須用 in-line filter（≥5 μm 過濾器）給藥**：mannitol 濃度高（20%）**低溫易析出結晶**，直接輸入可致結晶栓塞 → 給藥前檢查有無結晶，有結晶需**溫水回溶**後再用，並全程使用 filter
> 2. **心衰 / 肺水腫惡化**：mannitol 先擴張血漿容積（把組織水拉進血管）→ 心臟前負荷驟增，**心衰或少尿病人可能誘發/惡化肺水腫**
> 3. **反彈性顱內壓升高（rebound ICP）**：長期/重複給藥時 mannitol 可漏入腦組織反把水拉進腦內 → 反彈升 ICP；血腦障壁受損處尤甚
> 4. **無尿 / 腎功能衰竭且對試驗劑量無反應者禁用**：藥排不出去→血漿高滲、體液過載、加重心肺負擔
> 5. **監測滲透壓 gap 與 serum osmolality**：一般維持 **serum osm < 320 mOsm/kg**、滲透壓 gap 過大提示體內蓄積 → 停藥；血鈉、腎功能一併追

## 💊 適應症（治療哪些疾病）
- **腦水腫 / 顱內高壓 [[Increased Intracranial Pressure(顱內壓升高)]]**（創傷性腦損傷、腦出血、腦瘤引起的 ICP 升高；急性疝脫前的橋接治療）
- **急性青光眼 / 高眼壓 [[Glaucoma(青光眼)]]**（急性閉角型青光眼降 IOP）
- **神經外科術中 / 術前降腦容積**
- **（歷史用途）預防橫紋肌溶解或造影劑相關的腎小管損傷 — 現證據有限，非首選**

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 降顱內壓（ICP） | **0.25–1 g/kg IV**，15–30 分鐘內輸注；可每 6–8 小時重複 |
| 急性青光眼降 IOP | **1.5–2 g/kg IV**（20% 溶液），30–60 分鐘輸注 ⚠️依眼科指引 |
| 腦疝急救 | 通常 1 g/kg 快速輸注（依神外指示） |
- **濃度**：常用 20%（也有 15%、25%）；**必經 in-line filter**
- **給藥前試驗劑量（少尿/腎功能不明時）**：小劑量（如 0.2 g/kg over 3–5 min）觀察是否產生尿量，有反應才續用
- **監測目標**：serum osmolality < 320 mOsm/kg、osmolar gap、尿量、CVP/心肺狀態
- ⚠️各適應症具體劑量、輸注速率與重複頻率依院規 / 神外 / 眼科指引，臨床前請覆核

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：多尿、口渴、頭痛、噁心、輸注部位反應
- **嚴重**：
  - **急性肺水腫 / 心衰惡化**（初期血漿容積擴張）
  - **反彈性顱內壓升高**（重複給藥）
  - **電解質紊亂**：初期稀釋性低血鈉、後期利尿脫水致高血鈉；低血鉀
  - **急性腎損傷（"osmotic nephrosis"）**：高劑量蓄積傷腎小管，尤其原本腎功能差者
  - 體液過載、高滲狀態、脫水（利尿過度）

## 🚫 禁忌症
- **無尿 / 已知嚴重腎衰竭且對試驗劑量無反應**
- **嚴重心衰 / 肺水腫（進行中）**
- **活動性顱內出血（除神外開顱減壓術中外）**
- 嚴重脫水、進行性腎損傷併蓄積
- 已知對 mannitol 過敏

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：serum osmolality（目標 < 320）、滲透壓 gap、電解質（Na/K）、腎功能、尿量、心肺負荷（聽診/SpO₂/CVP）、神經學狀態與 ICP（若有監測）
- **交互作用**：
  - **腎毒性藥物（aminoglycoside、cyclosporine）**：腎損風險加成
  - **其他利尿劑**：利尿與電解質流失加成
  - 併用可能影響 lithium 排泄
- **孕哺**：資料有限，僅在明確適應症（如威脅生命的顱內高壓）評估利弊後使用

## 🔗 相關
- 疾病：[[Increased Intracranial Pressure(顱內壓升高)]]、[[Glaucoma(青光眼)]]
- 同類 / 對照：[[Loop Diuretics(環利尿劑總覽)]]（降 ICP 的另一選擇，或與 mannitol 併用）、[[Acetazolamide(乙醯唑胺)]]（青光眼降 IOP 另一機轉）；高張鹽水（3% saline）為降 ICP 的替代/併用滲透劑
- 機轉概念：滲透壓、血腦障壁

### 自我測驗（選擇題，答案摺疊）

**Q1.** 護理師準備輸注 20% mannitol 時發現袋內有結晶，最適當的處置是？
- A. 直接輸注，結晶會在血中溶解
- B. 溫水回溶並使用 in-line filter 後再給
- C. 換成 loop diuretic
- D. 加壓快速推注打散結晶

> [!success]- 答案
> **B** — Mannitol 低溫易析出結晶，須**溫水回溶**並**全程使用 in-line filter**，避免結晶栓塞。

**Q2.** 一名嚴重心衰竭且少尿的病人因腦水腫考慮用 mannitol，主要顧慮是？
- A. 沒有顧慮，可放心使用
- B. 初期血漿容積擴張可能誘發肺水腫，且藥排不出致體液過載
- C. 會造成低血糖
- D. 需搭配抗生素

> [!success]- 答案
> **B** — Mannitol 初期擴張血漿容積增加前負荷，心衰/少尿病人易惡化肺水腫且藥物蓄積，屬相對/絕對禁忌，需極謹慎並先試驗劑量。

**Q3.** 關於監測 mannitol 治療，下列何者最正確？
- A. 只需監測血糖
- B. 維持 serum osmolality < 320 mOsm/kg，並注意滲透壓 gap
- C. serum osm 越高越好
- D. 不需監測腎功能

> [!success]- 答案
> **B** — 應維持 **serum osm < 320**，滲透壓 gap 過大提示藥物蓄積需停藥，並持續監測電解質與腎功能。
