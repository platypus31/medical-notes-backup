# Lugol's solution（盧戈氏碘液）

> [!abstract] 一句話：**無機碘劑**（含 5% iodine + 10% KI）。高劑量碘透過 **Wolff-Chaikoff effect** 短暫抑制甲狀腺素合成與釋放，並**減少甲狀腺血流** → 用於 **thyroid storm** 與 **甲狀腺術前準備**。關鍵時序：**必須在抗甲狀腺藥（PTU/Methimazole）之後才給**，且只能短期用（會逃逸）。

> [!danger] 🚨 值班紅旗
> 1. **給藥順序**：**先抗甲狀腺藥（阻斷合成）→ 至少 1 小時後再給碘**。若碘先給，會被拿去合成更多甲狀腺素（Jod-Basedow）而惡化甲亢/風暴
> 2. **只能短期使用**：長期碘會發生 **escape（逃逸）**，甲狀腺重新合成 → 反而甲亢惡化；一般用 7–10 天（術前）
> 3. **禁用於單純結節性甲狀腺腫 / 碘誘發甲亢**、對碘過敏者
> 4. 不可用於**放射碘治療前後短期內**（碘會稀釋、干擾 I-131 攝取）

## 💊 適應症（治療哪些疾病）
- **甲狀腺風暴 thyroid storm**（[[Thyroid storm(甲狀腺風暴)]]）— 抗甲狀腺藥後給
- **甲狀腺切除術前準備**（[[Hyperthyroidism(甲狀腺亢進)]]/Graves'）：減少甲狀腺血流與腺體充血、降低術中出血
- 核災 / 放射碘暴露的甲狀腺保護（KI 阻斷放射碘攝取，此屬公共衛生用途）

## 📏 劑量 / 用法（成人）
| 情境 | 劑量 |
| --- | --- |
| Thyroid storm | 通常 **5–10 滴 Q6–8H PO**（於 PTU/MMI 後 ≥ 1 小時給） |
| 術前準備 | 3–5 滴 TID × 約 10 天 |
- 稀釋於水/果汁飲用（減少胃刺激、金屬味）
- 濃度：**5% iodine + 10% potassium iodide**（每滴約含碘 6–8 mg）

## ⚠️ 副作用（常見 / 嚴重）
- **碘中毒（iodism）**：金屬味、口腔灼熱、流涎、鼻炎、皮疹
- 腸胃不適
- **甲狀腺逃逸**（長期用 → 甲亢反彈）
- 過敏（罕見但可嚴重，含血管性水腫）
- 唾液腺腫脹

## 🚫 禁忌症
- 碘過敏
- 單純非毒性結節腫、可能碘誘發甲亢者
- 放射碘治療計畫前後（干擾攝取）
- 孕婦長期使用（胎兒甲狀腺腫 / 甲低）
- 抗甲狀腺藥尚未給之前（單獨先給碘）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：甲狀腺功能、症狀（避免逃逸反彈）、碘中毒徵象
- **交互作用**：
  - **與抗甲狀腺藥時序**：PTU/MMI 先、碘後
  - 與 **lithium** 併用 → 加成甲狀腺抑制（甲低風險）
  - 保鉀利尿劑 / ACEI（KI 含鉀）→ 注意高血鉀
- **孕哺**：短期可，長期禁（胎兒甲狀腺腫/甲低）；哺乳大量碘會入乳影響嬰兒甲狀腺

### 自我測驗（選擇題，答案摺疊）

**Q1.** Thyroid storm 病人已給 PTU，接下來給 Lugol's 碘液的時機是？
- A. 與 PTU 同時
- B. PTU 之前
- C. PTU 給藥後至少 1 小時
- D. 不需要給碘

> [!success]- 答案
> **C** — 須先用 PTU 阻斷合成，至少 1 小時後再給碘，避免碘被用於合成更多甲狀腺素而惡化風暴。

**Q2.** 關於 Lugol's solution 的使用，何者正確？
- A. 可長期單獨使用治療甲亢
- B. 只能短期使用，長期會逃逸致甲亢反彈
- C. 碘過敏者可用
- D. 是放射碘治療前的首選前處置

> [!success]- 答案
> **B** — 碘劑長期使用會發生 escape 逃逸現象，甲狀腺重新合成致甲亢反彈，故僅短期（如術前 7–10 天或 storm）使用。

**Q3.** Lugol's solution 抑制甲狀腺素合成與釋放的效應稱為？
- A. Jod-Basedow effect
- B. Wolff-Chaikoff effect
- C. Thyroid escape
- D. Sick euthyroid

> [!success]- 答案
> **B** — 高劑量碘短暫抑制甲狀腺素合成與釋放為 Wolff-Chaikoff effect；相對地，碘誘發甲亢為 Jod-Basedow。
