# Levothyroxine / Liothyronine（左旋甲狀腺素 T4 / T3）

> [!abstract] 一句話：外源性甲狀腺素補充。**Levothyroxine（T4）為甲狀腺低下的標準治療**——半衰期長（~7 天）、體內周邊轉換為活性 T3、劑量穩定；**Liothyronine（T3）**作用快、半衰期短，主要用於**黏液水腫昏迷急救**等需快速起效或特殊情形。治療目標以 TSH 正常化為準。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **空腹服用、與多種藥物間隔**：需**早餐前 30–60 分鐘空腹**、與**鈣片、鐵劑、制酸劑（含鋁鎂）、PPI、膽固醇結合樹脂**間隔至少 4 小時，否則吸收大減 → TSH 控制不良
> 2. **老人、缺血性心臟病患者「從低劑量起始、緩慢增量」**：一次給足量可誘發心絞痛、心律不整、心肌梗塞
> 3. **黏液水腫昏迷（myxedema coma）是內科急症**：意識障礙 + 低體溫 + 低血鈉 → 需**靜脈給高劑量 T4（±T3）**並同時給壓力劑量類固醇（先排除/覆蓋腎上腺功能不全），不能只口服
> 4. **未診斷的腎上腺功能不全先補甲狀腺素會誘發腎上腺危象**：合併垂體/腎上腺問題時，**先給類固醇再給甲狀腺素**
> 5. **過量 = 醫源性甲亢**：心悸、心房顫動、體重減輕、骨質疏鬆（尤其停經婦女與老人）；孕婦需求增加約 25–30%，需提早增量並密集追蹤

## 💊 適應症（治療哪些疾病）
- **原發性甲狀腺功能低下 [[Hypothyroidism(甲狀腺功能低下)]]**（Levothyroxine 為首選終身補充）
- **中樞性（垂體/下視丘）甲狀腺低下**（此時以游離 T4 而非 TSH 為調整依據）
- **黏液水腫昏迷 [[Myxedema Coma(黏液水腫昏迷)]]**（急救用 IV，常加 T3）
- 甲狀腺癌術後 TSH 抑制治療、先天性甲狀腺低下
- Liothyronine（T3）：特殊快速需求、部分核醫檢查前準備

## 📏 劑量 / 用法（成人）
| 情境 | Levothyroxine（T4） |
| --- | --- |
| 年輕健康成人完全替代 | 約 **1.6 µg/kg/day PO QD**（空腹） |
| 老人 / 缺血性心臟病 | **起始 12.5–25 µg/day**，每 4–6 週依 TSH 緩慢增量 |
| 次臨床甲低起始 | 低劑量 25–50 µg/day |
| 孕婦 | 確診後需求↑約 25–30%，提早增量 |
| 黏液水腫昏迷 | **IV loading 200–400 µg 後 50–100 µg/day**（±Liothyronine T3 5–20 µg IV）+ 壓力劑量 hydrocortisone |
- **服法**：早晨空腹、飯前 30–60 分鐘，配白開水；固定時間、固定廠牌（生體可用率差異）
- 調整週期：**每 4–6 週複查 TSH**（T4 半衰期長，太早驗無意義）

## ⚠️ 副作用（常見 / 嚴重）
- 治療劑量正確時通常無副作用（本質為生理補充）
- **過量（醫源性甲亢）**：心悸、心搏過速、心房顫動、失眠、焦慮、體重下降、腹瀉、怕熱、月經異常
- **長期過量**：**骨質疏鬆**（停經婦女、老人）、心律不整
- 心臟病患者增量過快：心絞痛、心肌梗塞

## 🚫 禁忌症
- 對藥物成分過敏
- **未經治療的腎上腺功能不全 → 須先補類固醇再給甲狀腺素**（否則腎上腺危象）
- 急性心肌梗塞、未控制的甲狀腺毒症
- 老人/心臟病患禁「一次高劑量起始」

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：原發性甲低看 **TSH**（每 4–6 週）；中樞性看**游離 T4**；心臟症狀
- **交互作用（吸收 / 代謝）**：
  - **降低吸收**：鈣、鐵、制酸劑（鋁/鎂）、PPI、sucralfate、膽固醇結合樹脂（cholestyramine）、豆類/高纖 → **間隔 ≥4 小時**
  - **增加需求 / 加速代謝**：雌激素（增 TBG）、rifampin、phenytoin、carbamazepine
  - 加強 warfarin 抗凝作用（甲狀腺素↑代謝凝血因子）→ 監測 INR
- **孕哺**：**孕期必須維持治療且需求增加**（胎兒神經發育依賴母體 T4），提早增量密切追蹤 TSH；哺乳安全

### 自我測驗（選擇題，答案摺疊）

**Q1.** 78 歲男性有穩定型心絞痛，新診斷甲狀腺低下（TSH 明顯升高）。起始 levothyroxine 最適當方式？
- A. 直接 1.6 µg/kg 完全替代量
- B. 12.5–25 µg/day 起始，每 4–6 週依 TSH 緩慢增量
- C. 靜脈注射
- D. 併用 T3 快速達標

> [!success]- 答案
> **B** — 老人與缺血性心臟病患者一次給足量可誘發心絞痛/心肌梗塞/心律不整，需低劑量起始緩慢增量。

**Q2.** 病人抱怨吃了甲狀腺素還是 TSH 一直偏高，追問發現他與鈣片、綜合維他命（含鐵）同時服用。最可能原因？
- A. 劑量不足需加倍
- B. 鈣、鐵干擾 levothyroxine 吸收，需間隔 ≥4 小時
- C. 藥物失效
- D. 甲狀腺癌

> [!success]- 答案
> **B** — 鈣、鐵、制酸劑、PPI 等會顯著降低 levothyroxine 吸收，應空腹服用並與這些藥間隔至少 4 小時。
