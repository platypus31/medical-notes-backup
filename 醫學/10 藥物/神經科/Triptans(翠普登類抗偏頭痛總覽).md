# Triptans（翠普登類 / 5-HT1B/1D 促效劑）

> [!abstract] 一句話：**促效 5-HT1B/1D 受體 → 顱內血管收縮 + 抑制三叉神經血管系統釋放 CGRP 等血管活性神經肽 → 中止急性偏頭痛發作**。屬**急性中止藥（abortive），非預防藥**；代表藥 sumatriptan，其餘尚有 rizatriptan / zolmitriptan / eletriptan / naratriptan / almotriptan / frovatriptan。因會收縮血管，**心腦血管疾病為禁忌**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **心腦血管禁忌**：缺血性心臟病 / 冠心病、心肌梗塞病史、未控制高血壓、缺血性腦血管病（中風/TIA）、周邊血管病 → **禁用**（血管收縮可誘發心肌梗塞、心律不整、中風）
> 2. **與麥角類（ergotamine / DHE）間隔至少 24 小時**：兩者皆強力血管收縮，併用加重缺血
> 3. **與 MAOI 併用禁忌**：sumatriptan / rizatriptan / zolmitriptan 部分經 MAO-A 代謝 → 血中濃度升高；且增加血清素症候群風險（MAOI 停藥後亦需間隔）
> 4. **血清素症候群**：與 SSRI / SNRI 併用時警覺（躁動、發熱、反射亢進、肌陣攣、自律神經不穩）
> 5. **偏頭痛亞型限制**：半身麻痺型（hemiplegic）、基底動脈型（basilar）偏頭痛，及帶延長先兆者傳統上慎用 / 避免
> 6. **藥物過量性頭痛（MOH）**：每月使用天數過多（一般 triptan ≥10 天/月）反致慢性化頭痛，需限制使用頻率

## 💊 適應症
- **急性偏頭痛發作之中止治療**（有或無先兆），愈早服用（頭痛初起）效果愈好
- **叢發性頭痛（cluster headache）之急性發作**：sumatriptan 皮下注射 / zolmitriptan 鼻噴為選項
- ⚠️ **非預防用藥**：頻繁發作需另用預防藥（β-blocker、topiramate、CGRP 單株抗體等）

## 📏 劑量 / 用法（成人）
| 藥物 / 劑型 | 起始劑量 | 可否重複 / 24h 上限 |
| --- | --- | --- |
| **Sumatriptan 口服** | **50–100 mg PO** | 若有部分反應，**2 小時後可重複**；24h 上限 **200 mg** |
| **Sumatriptan 皮下注射（SC）** | **6 mg SC**（用於嚴重 / 併嘔吐無法口服 / 叢發性頭痛） | 至少間隔 1 小時可再一次；24h 上限 12 mg |
| Sumatriptan 鼻噴 | 5–20 mg | 2 小時後可重複；上限依製劑 |
| Rizatriptan 口服 | 5–10 mg | 2 小時後可重複；24h 上限 30 mg（併用 propranolol 減量） |
| Zolmitriptan 口服 / 鼻噴 | 2.5–5 mg | 2 小時後可重複；24h 上限 10 mg |
| Eletriptan 口服 | 20–40 mg | 2 小時後可重複；24h 上限 80 mg（CYP3A4 交互強） |

- **重點**：若首劑「完全無效」，同次發作再加一劑通常也無效（不建議重複）；「部分有效」才考慮 2 小時後重複 ⚠️依院規/需查證
- 口服吸收慢 / 併嚴重噁心嘔吐 → 優先**皮下注射或鼻噴**劑型
- 每月使用天數需限制（避免 MOH），頻繁發作應轉預防治療

## ⚠️ 副作用
- **常見**：胸悶 / 胸壓迫感（「triptan sensations」，多非心因，但需與心絞痛鑑別）、頸/喉/下顎緊繃感、刺痛感覺異常、潮紅、頭暈、嗜睡、噁心
- **注射部位**：SC sumatriptan 注射處疼痛/紅
- **嚴重 / 需警覺**：
  - **心肌缺血 / 心肌梗塞 / 冠狀動脈痙攣、嚴重心律不整**（罕見但為主要安全顧慮）
  - **腦血管事件**（中風、蛛網膜下腔出血罕見報告）
  - **血清素症候群**（併 SSRI/SNRI/MAOI）
  - 周邊血管收縮 → 缺血、雷諾現象加重、罕見腸缺血

## 🚫 禁忌症
- 缺血性心臟病 / 冠心病、心肌梗塞病史、Prinzmetal（變異型）心絞痛 / 冠狀動脈痙攣
- 未控制之高血壓
- 缺血性腦血管病（中風 / TIA）、周邊血管疾病、缺血性腸病
- 半身麻痺型 / 基底動脈型偏頭痛（傳統禁忌 / 慎用）
- **24 小時內使用過麥角類或其他 triptan**
- 併用 MAOI（或停用 MAOI 未滿 2 週，適用經 MAO 代謝之 triptan）
- 嚴重肝功能不全（部分藥物）
- 對該 triptan 過敏

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：首次使用注意胸痛/心血管症狀；血壓；使用頻率（防 MOH）；併用血清素能藥物時警覺血清素症候群徵象
- **交互作用**：
  - **麥角胺 / DHE**：24 小時內禁併（雙重血管收縮）
  - **MAOI**：禁併 sumatriptan / rizatriptan / zolmitriptan（經 MAO-A 代謝）
  - **SSRI / SNRI**：血清素症候群風險（臨床可審慎併用但需衛教徵象）
  - **CYP3A4 抑制劑（如 ketoconazole、部分蛋白酶抑制劑）**：顯著提高 **eletriptan** 濃度 → 避免
  - **Propranolol**：升高 rizatriptan 濃度 → rizatriptan 減量
- **孕哺**：sumatriptan 為孕期相對較多資料者，權衡使用；多數 triptan 少量分泌至乳汁，哺乳評估（sumatriptan 常被視為可接受）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名 58 歲男性有陳舊性心肌梗塞病史，因偏頭痛欲用 sumatriptan。最正確的處置是？
- A. 直接給 100 mg PO
- B. Triptans 在缺血性心臟病為禁忌，應改用其他急性治療（如 NSAID、gepant）
- C. 減半劑量即可安全使用
- D. 改成皮下注射就沒有心血管風險

> [!success]- 答案
> **B** — Triptans 會收縮血管，缺血性心臟病/冠心病、心肌梗塞病史為禁忌，任何劑型與減量都不改變此禁忌；應選非血管收縮的急性治療（NSAID、metoclopramide、gepant/ditan 等）。

**Q2.** 病人 6 小時前已使用過含麥角胺的偏頭痛藥，現頭痛未緩解。下列何者正確？
- A. 立刻加一劑 triptan 效果加乘
- B. 麥角類與 triptan 須間隔至少 24 小時，現在不可給 triptan
- C. 只要不是同一種藥就可併用
- D. 併用可降低血清素症候群風險

> [!success]- 答案
> **B** — 麥角類與 triptan 皆為血管收縮劑，24 小時內併用會加重血管收縮與缺血風險，屬禁忌，須間隔至少 24 小時。

**Q3.** 關於 triptans 的臨床使用，何者正確？
- A. 是偏頭痛的預防用藥，需每天服用
- B. 首劑完全無效時，同次發作再補一劑通常仍有效
- C. 屬急性中止藥，愈早服用愈有效，且需限制月使用天數以免藥物過量性頭痛
- D. 沒有任何心血管禁忌

> [!success]- 答案
> **C** — Triptans 是急性中止藥（非預防），頭痛初起愈早使用愈有效；使用過於頻繁會導致藥物過量性頭痛（MOH），需限制月使用天數；首劑「完全無效」時同次再補通常無效；且有明確心腦血管禁忌。
