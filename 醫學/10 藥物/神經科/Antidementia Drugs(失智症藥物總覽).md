# Antidementia Drugs（失智症藥物總覽）

> [!abstract] 一句話：阿茲海默症失智核心是**膽鹼性神經退化**，治療分兩大類：**膽鹼酯酶抑制劑（AChEI：Donepezil / Rivastigmine / Galantamine）** 增加突觸乙醯膽鹼（**輕至中度**用），**NMDA 受體拮抗劑（Memantine）** 減少麩胺酸興奮毒性（**中至重度**用，可與 AChEI 併用）。這些藥**只延緩症狀進展、不逆轉病程**。值班重點：AChEI 的**心搏過緩、暈厥、GI（噁心腹瀉）與消化道潰瘍出血**風險。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **AChEI → 迷走神經興奮：心搏過緩、心臟傳導阻滯、暈厥、跌倒**。有病竇症候群、傳導阻滯或正服 β-blocker/digoxin 的老人要特別小心，暈厥病人先查 ECG
> 2. **AChEI 促胃酸分泌 → 消化性潰瘍 / 上消化道出血**，尤其有潰瘍史或併用 NSAID/抗凝
> 3. **Rivastigmine 貼片過量常見於「忘了撕舊貼片又貼新的」→ 膽鹼性危象（噁心嘔吐、盜汗、心搏過緩、肌無力）**；換貼片時務必撕除舊片
> 4. **手術/麻醉前**：AChEI 會延長 succinylcholine 這類去極化肌鬆劑作用，麻醉前應告知
> 5. **Memantine 需依腎功能調量**（腎排泄）；勿把失智藥當「越多越好」，AChEI + Memantine 併用僅用於中重度，且不會加更多 AChEI

## 💊 適應症（治療哪些疾病）
- **阿茲海默症失智 [[Alzheimer Disease(阿茲海默症)]]**（AChEI 輕中度、Memantine 中重度、中重度可併用）
- **路易體失智 / 帕金森失智 [[Dementia with Lewy Bodies(路易體失智症)]]**（Rivastigmine 對認知與精神症狀證據較佳）
- **血管性失智**（部分使用，證據較弱）
- Galantamine/Donepezil 亦用於部分混合型失智（依評估）

## 📏 劑量 / 用法（成人）
| 藥物（類別） | 常用劑量 | 分期 / 重點 |
| --- | --- | --- |
| **Donepezil**（AChEI，可逆） | 5 mg QHS 起，4 週後可加至 10 mg（重度可到 23 mg） | **輕–中–重度皆可**；每日一次睡前，肝代謝 |
| **Rivastigmine**（AChE + BuChE 抑制） | 口服 1.5 mg BID 漸增；**貼片 4.6 → 9.5 → 13.3 mg/24h** | 輕中度；**貼片減少 GI 副作用**、換片撕舊；對 Lewy body / PD 失智佳 |
| **Galantamine**（AChEI + 菸鹼受體調節） | 8 mg/day 起漸增至 16–24 mg/day | 輕中度；肝腎不良需減量 |
| **Memantine**（NMDA 拮抗） | 5 mg QD 起，每週加 5 mg 至 **10 mg BID（或 20 mg 緩釋 QD）** | **中–重度**；可與 AChEI 併用；**腎功能不良減量** |

- ⚠️依院規：健保給付對 MMSE/CDR 分數與療效追蹤有規範，開立與續方依院內失智共照/神經精神科規定
- 滴定原則：**慢慢加**以減少 GI 與心臟副作用

## ⚠️ 副作用（常見 / 嚴重）
- **AChEI 共通（膽鹼性）**：**噁心、嘔吐、腹瀉、食慾不振、體重減輕**、流涎、多汗、腹絞痛、**心搏過緩、暈厥**、失眠/多夢（Donepezil 睡前服可能夢魘）、肌肉痙攣、尿失禁
- **嚴重**：**竇性心搏過緩 / 房室傳導阻滯致暈厥跌倒**、**消化道潰瘍出血**、罕見癲癇、體重顯著下降、Rivastigmine 貼片過量致膽鹼性危象
- **Memantine**：頭暈、頭痛、意識混亂、便秘、幻覺、高血壓；整體耐受性較 AChEI 好、無膽鹼性 GI 問題

## 🚫 禁忌症
- **AChEI**：對藥物過敏；相對禁忌 — 顯著心搏過緩 / 病竇症候群 / 未裝節律器之傳導阻滯、活動性消化性潰瘍、嚴重氣喘或 COPD（膽鹼性支氣管收縮）、癲癇未控
- **Memantine**：對藥物過敏；嚴重腎衰竭需大幅減量
- 共通：末期失智已無治療效益時，依家庭會議與照護目標評估停藥

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：心率 / ECG（AChEI 尤其暈厥時）、體重與營養、認知量表（MMSE/CDR）追蹤療效、消化道症狀、Memantine 用腎功能
- **交互作用**：
  - **AChEI ↔ β-blocker / digoxin / 非二氫吡啶類鈣阻斷劑 → 加重心搏過緩、傳導阻滯**
  - **AChEI ↔ 抗膽鹼藥（oxybutynin、抗組織胺、TCA）→ 藥效互相抵消**（老人常見不合理併用）
  - **AChEI ↔ succinylcholine → 延長肌鬆作用**（麻醉前告知）
  - **AChEI ↔ NSAID → 增加消化道潰瘍出血風險**
  - Memantine ↔ 其他 NMDA 拮抗（amantadine、ketamine、dextromethorphan）避免併用
- **孕哺**：失智藥物幾乎用於年長族群，孕哺非適用情境；資料極少（⚠️依院規）

## 🔗 相關

阿茲海默症失智藥物分哪兩大機轉類別？::膽鹼酯酶抑制劑（AChEI，增乙醯膽鹼）與 NMDA 受體拮抗劑（Memantine，減麩胺酸興奮毒性）

AChEI 適用於哪個失智分期，Memantine 又用於哪期？::AChEI 用於輕–中度（Donepezil 到重度亦可）；Memantine 用於中–重度，可與 AChEI 併用

AChEI 最需注意的心臟副作用是什麼？::迷走神經興奮造成心搏過緩、房室傳導阻滯、暈厥與跌倒

為什麼 AChEI 會增加消化道潰瘍出血風險？::增加乙醯膽鹼 → 促進胃酸分泌，尤其有潰瘍史或併用 NSAID 者

Rivastigmine 貼片的優點與換片時的關鍵注意事項？::貼片減少胃腸道副作用；換片時務必撕除舊片，否則兩片並存易致膽鹼性危象（過量）

Memantine 的作用機轉與需依器官功能調整的原因？::NMDA 受體拮抗劑，減少麩胺酸興奮毒性；經腎排泄，腎功能不良須減量

失智老人同時服用 AChEI 與抗膽鹼藥（如 oxybutynin）有何問題？::兩者作用相反互相抵消，是常見不合理多重用藥，應檢視移除抗膽鹼藥

哪種失智特別偏好使用 Rivastigmine？::路易體失智 / 帕金森病失智（對認知與精神症狀證據較佳）

抗失智藥能治癒阿茲海默症嗎？::不能，只能延緩症狀進展、不逆轉病程

### 自我測驗（選擇題，答案摺疊）

**Q1.** 78 歲阿茲海默症病人服 Donepezil，近日反覆暈厥。最該優先安排的檢查是？
- A. 腦部 MRI
- B. 心電圖（ECG）評估心搏過緩 / 傳導阻滯
- C. 甲狀腺功能
- D. 腰椎穿刺

> [!success]- 答案
> **B** — AChEI 迷走興奮可致竇性心搏過緩或房室阻滯而暈厥，暈厥病人應先查 ECG、心率，必要時減量或停藥。

**Q2.** 關於失智藥物分期使用，下列何者正確？
- A. Memantine 為輕度失智首選
- B. AChEI 只能用於重度失智
- C. Memantine 用於中重度，可與 AChEI 併用
- D. 兩種 AChEI 應同時併用以增效

> [!success]- 答案
> **C** — Memantine（NMDA 拮抗）用於中重度並可與 AChEI 併用；不同時併用兩種 AChEI。

**Q3.** 使用 Rivastigmine 貼片的照護者最常導致過量的錯誤是什麼？
- A. 貼在手臂
- B. 沒撕除舊貼片就貼新片，造成兩片並存
- C. 白天貼晚上撕
- D. 貼片剪半使用

> [!success]- 答案
> **B** — 未撕舊片即貼新片會造成劑量疊加，導致膽鹼性危象（噁心嘔吐、盜汗、心搏過緩、肌無力）。
