# Antiparkinson Drugs（帕金森藥物總覽）

> [!abstract] 一句話：帕金森病核心是**黑質-紋狀體多巴胺缺乏**，治療策略都是「補多巴胺 / 模仿多巴胺 / 延緩多巴胺分解」。**Levodopa-Carbidopa 是療效金標準**（但長期出現 on-off 與異動症 dyskinesia），年輕病人常先用**DA agonist**（延後 levodopa 併發症，但衝動控制障礙 + 嗜睡風險）；**MAO-B / COMT** 為佐劑延長 levodopa 作用；**Amantadine** 抗異動症。值班重點：**Levodopa 絕不可突然停藥**（致命 parkinsonism-hyperpyrexia）。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **Levodopa 突然停藥或大幅減量 → parkinsonism-hyperpyrexia syndrome（類 NMS：高燒、僵硬、意識改變、CK 上升）可致死**。住院禁食 NPO 病人的帕金森藥常被漏開 → 一定要保留原本抗帕金森藥（可經 NG / 貼片 rotigotine 替代）
> 2. **DA agonist（Pramipexole / Ropinirole）→ 衝動控制障礙（病態賭博、購物、性慾亢進、暴食）+ 突發性睡眠發作（開車途中睡著）**。開藥要主動衛教並詢問，家屬常先察覺
> 3. **Levodopa 不可與非選擇性 MAOI 併用**（高血壓危象）；**Selegiline/Rasagiline 為選擇性 MAO-B**，但仍避免與 SSRI/SNRI/pethidine/tramadol 併用（serotonin 症候群）
> 4. **抗多巴胺藥（metoclopramide、haloperidol、prochlorperazine 傳統止吐/抗精神病藥）會惡化帕金森 + 誘發藥物性 parkinsonism** → 帕金森病人止吐改用 domperidone 或 ondansetron
> 5. **wearing-off / on-off 現象**：長期 levodopa 後藥效波動，突然「off」時病人可能完全僵住（freezing）跌倒

## 💊 適應症（治療哪些疾病）
- **帕金森病 [[Parkinson Disease(帕金森病)]]**（原發性，多巴胺缺乏，主軸適應症）
- **不寧腿症候群 [[Restless Legs Syndrome(不寧腿症候群)]]**（DA agonist Pramipexole/Ropinirole 低劑量為首選）
- 藥物性 parkinsonism 的區分與部分處理（首要是停掉致病藥）
- Amantadine 另用於抗病毒（A 型流感，現少用）與 levodopa 誘發的異動症

## 📏 劑量 / 用法（成人）
| 藥物（類別） | 常用劑量 | 重點 |
| --- | --- | --- |
| **Levodopa-Carbidopa**（多巴胺前驅物 + 去羧酶抑制劑） | 起始 25/100 mg TID，依反應漸增 | Carbidopa 阻止周邊代謝、減少噁心；**空腹或餐前 30–60 min 服**（高蛋白餐降吸收） |
| **Pramipexole**（DA agonist，非麥角） | 0.125 mg TID 起漸增 | 腎排泄，**腎功能不良需減量**；衝動控制障礙 |
| **Ropinirole**（DA agonist，非麥角） | 0.25 mg TID 起漸增 | 肝代謝（CYP1A2）；嗜睡、衝動控制障礙 |
| **Selegiline / Rasagiline**（MAO-B 抑制劑） | Selegiline 5 mg BID；Rasagiline 0.5–1 mg QD | 佐劑延長 levodopa；Rasagiline QD 較方便 |
| **Entacapone**（COMT 抑制劑） | 200 mg 隨每劑 levodopa | **只在有 levodopa 時有效**；尿液橘紅色、腹瀉；Tolcapone 有肝毒性少用 |
| **Amantadine**（NMDA 拮抗 + 促多巴胺釋放） | 100 mg BID | 抗 levodopa 異動症；**腎排泄需依腎調量**；livedo reticularis、下肢水腫、幻覺 |

- ⚠️依院規：各藥滴定速度、住院轉換（口服↔rotigotine 貼片）劑量換算依主治與神經科調整
- **NPO / 手術 / 禁食病人**：務必與神經科討論改 rotigotine 透皮貼片或 NG 給藥，勿直接停

## ⚠️ 副作用（常見 / 嚴重）
- **Levodopa**：噁心嘔吐（周邊多巴胺）、姿勢性低血壓、幻覺/精神症狀（尤高齡）、長期**異動症 dyskinesia**、on-off/wearing-off 波動、尿液/汗液變深色
- **DA agonist**：**衝動控制障礙、突發性睡眠、幻覺、下肢水腫、姿勢性低血壓**；麥角類（bromocriptine、cabergoline）另有心臟瓣膜/肺纖維化 → 現多用非麥角類
- **MAO-B**：失眠（Selegiline 代謝成安非他命樣物質，避免睡前服）、姿勢性低血壓、serotonin 症候群風險
- **COMT（Entacapone）**：腹瀉、尿橘紅、加重 levodopa 異動症；**Tolcapone → 猛爆性肝衰竭**（需監測肝功能，少用）
- **Amantadine**：livedo reticularis、腳踝水腫、視幻覺、突停也可能加重帕金森

## 🚫 禁忌症
- **Levodopa**：窄角性青光眼、惡性黑色素瘤（理論上可能刺激）、與非選擇性 MAOI 併用
- **DA agonist**：對衝動控制障礙病史、嚴重精神症狀者審慎
- **MAO-B**：與 pethidine（哌替啶）、tramadol、SSRI/SNRI、其他 MAOI 併用（serotonin 症候群 / 高血壓危象）
- **Amantadine**：嚴重腎衰竭（未減量）、癲癇史審慎

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：運動症狀波動（on-off 日誌）、姿勢性血壓、精神狀態/幻覺、衝動控制行為（主動問家屬）、Amantadine/Pramipexole 用腎功能、Tolcapone 用肝功能
- **交互作用**：
  - **抗多巴胺藥（metoclopramide、haloperidol、傳統抗精神病藥、prochlorperazine）↔ 拮抗療效並誘發藥物性 parkinsonism**
  - **高蛋白飲食 ↔ 與 levodopa 競爭腸道與 BBB 胺基酸轉運 → 降藥效**
  - **鐵劑 ↔ 螯合 levodopa 降吸收**（分開服用）
  - **MAO-B ↔ serotonergic 藥（serotonin 症候群）、擬交感藥（高血壓）**
- **孕哺**：資料有限，多避免；需與神經科/產科個別評估（⚠️依院規）

### 記憶卡（Spaced Repetition 相容 · `Q::A`）

帕金森病的核心神經化學缺陷是什麼？::黑質-紋狀體路徑的多巴胺神經退化（多巴胺缺乏）

Carbidopa 加在 Levodopa 裡的作用是什麼？::抑制周邊 dopa 去羧酶，減少 levodopa 在周邊被代謝 → 增加進入腦的量、減少噁心與姿勢性低血壓

為什麼 Levodopa 絕對不可突然停藥？::會誘發 parkinsonism-hyperpyrexia syndrome（類 NMS：高燒、肌僵直、意識改變、CK 上升），可致命

Pramipexole / Ropinirole 這類 DA agonist 最需衛教的兩大副作用？::衝動控制障礙（賭博/購物/性慾/暴食）與突發性睡眠發作

Levodopa 長期使用會出現哪些運動併發症？::異動症（dyskinesia）與藥效波動 on-off / wearing-off

Selegiline / Rasagiline 屬哪類、主要用途？::選擇性 MAO-B 抑制劑，作為 levodopa 佐劑延長作用；避免與 SSRI/pethidine/tramadol 併用（serotonin 症候群）

Entacapone（COMT 抑制劑）使用有何前提與典型副作用？::只在同時有 levodopa 時才有效；尿液呈橘紅色、腹瀉、加重異動症

Amantadine 在帕金森治療的特殊角色與需注意的排泄？::可減輕 levodopa 誘發的異動症；經腎排泄，腎功能不良需減量，副作用 livedo reticularis、下肢水腫、幻覺

帕金森病人需要止吐時，為何不能用 metoclopramide？::它是中樞抗多巴胺藥，會拮抗療效並惡化帕金森 → 改用 domperidone 或 ondansetron

高蛋白飲食為何會影響 Levodopa 療效？::大量胺基酸與 levodopa 競爭腸道與血腦障壁的轉運系統 → 降低藥效（建議餐前服用）

住院 NPO 的帕金森病人抗帕金森藥該怎麼辦？::不可停，改 NG 給藥或 rotigotine 透皮貼片，與神經科討論劑量換算

### 自我測驗（選擇題，答案摺疊）

**Q1.** 72 歲帕金森病人因手術禁食（NPO）48 小時，護理師詢問是否暫停口服抗帕金森藥。最正確處置為何？
- A. 直接停藥，術後恢復飲食再給
- B. 減半劑量口服
- C. 不可驟停，改 rotigotine 貼片或 NG 給藥並與神經科討論
- D. 只給安眠藥即可

> [!success]- 答案
> **C** — Levodopa 驟停可致 parkinsonism-hyperpyrexia（類 NMS），NPO 期間須以貼片或 NG 維持給藥，不能單純暫停。

**Q2.** 一名服用 Pramipexole 的帕金森病人，家屬抱怨他近月開始沉迷網路賭博與大量購物。最可能原因？
- A. 失智進展
- B. DA agonist 引起的衝動控制障礙
- C. 劑量不足
- D. 憂鬱症

> [!success]- 答案
> **B** — DA agonist（Pramipexole/Ropinirole）典型的衝動控制障礙，需減量或換藥並衛教家屬。

**Q3.** 關於 Entacapone，下列敘述何者正確？
- A. 單獨使用即可治療帕金森
- B. 屬 MAO-B 抑制劑
- C. 為 COMT 抑制劑，須與 levodopa 併用才有效，會使尿液變橘紅
- D. 主要副作用為肝衰竭

> [!success]- 答案
> **C** — Entacapone 是 COMT 抑制劑，延緩 levodopa 周邊分解，須併用 levodopa；尿橘紅、腹瀉常見。猛爆性肝衰竭是 Tolcapone 的問題。
