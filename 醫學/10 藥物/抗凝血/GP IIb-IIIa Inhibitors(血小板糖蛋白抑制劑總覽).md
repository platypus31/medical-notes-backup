# GP IIb/IIIa Inhibitors（血小板糖蛋白抑制劑總覽）

> [!abstract] 一句話：**阻斷血小板表面 GP IIb/IIIa 受體 → 阻止 fibrinogen 交聯血小板 → 抑制血小板聚集的「最終共同路徑」**，是作用最強的靜脈抗血小板藥。主要用於**高危險急性冠心症（ACS）與經皮冠狀動脈介入（PCI）**時的輔助抗血小板。三個藥：**Abciximab（單株抗體）、Tirofiban、Eptifibatide（小分子）**。最大代價是**出血與血小板低下**。

> [!danger] 🚨 值班紅旗（用藥前先想）
> 1. **出血風險高**：與肝素、抗血小板併用，穿刺點/GI/顱內出血風險顯著，PCI 後密切監測
> 2. **血小板急性重度低下（thrombocytopenia）**：可在給藥後**數小時內驟降**（尤其 abciximab），需**給藥後 2–4 小時及 24 小時查 CBC**；<50,000 需停藥
> 3. **腎功能調量**：**Tirofiban、Eptifibatide 經腎排泄，CrCl 低要減量**；**Eptifibatide 在嚴重腎衰竭（CrCl <30 / 透析）禁用**
> 4. **靜脈專用、僅限住院/導管室使用**，非口服長期藥
> 5. **近期出血 / 大手術 / 顱內病灶 / 嚴重高血壓禁用**；與溶栓劑併用大出血風險極高

## 💊 適應症（治療哪些疾病）
- **高危險非 ST 上升急性冠心症 [[NSTE-ACS(非ST上升急性冠心症)]]**（尤其擬做 PCI、troponin 陽性、血栓負荷高者）
- **經皮冠狀動脈介入 PCI [[Percutaneous Coronary Intervention(經皮冠狀動脈介入)]]** 術中輔助抗血小板（減少缺血併發症）
- **PCI 中大量血栓 / no-reflow 情境的補救（bailout）**
- （較少）ST 上升心肌梗塞 primary PCI 佐劑

## 📏 劑量 / 用法（成人）
| 藥物 | 特性 | 用法（概要） |
| --- | --- | --- |
| **Abciximab** | 單株抗體 Fab、與受體結合久、半衰期作用長（血小板功能恢復需 24–48h） | bolus + 12 小時輸注，⚠️依院規 |
| **Tirofiban** | 小分子非胜肽、可逆、半衰期短（~2h） | 依體重 bolus + 輸注，**腎功能不良減半**，⚠️依院規 |
| **Eptifibatide** | 小分子環狀胜肽、可逆、半衰期短 | 雙 bolus + 輸注，**CrCl <50 減量、CrCl <30 或透析禁用**，⚠️依院規 |
- **共同**：常與 **肝素/bivalirudin + 口服雙抗（aspirin + P2Y12）** 併用；藥效停止後血小板功能：小分子快（數小時）、abciximab 慢（24–48h）
- 精確劑量依體重、腎功能、院內導管室 protocol，**務必查證**

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：出血（穿刺部位、瘀青）、低血壓
- **嚴重**：
  - **大出血**（GI、後腹膜、顱內）
  - **急性重度血小板低下**（abciximab 較明顯，可 <20,000，需緊急停藥）
  - 過敏反應（abciximab 為嵌合抗體，可有過敏；再次使用可產生抗體）
  - 罕見血栓性血小板低下

## 🚫 禁忌症
- **活動性內出血 / 近期（30 天）大出血**
- **近期顱內出血 / 中風、顱內腫瘤 / 血管畸形 / 動脈瘤**
- **近期大手術或外傷、嚴重未控制高血壓**
- **血小板低下（<100,000）**、出血體質
- **Eptifibatide：嚴重腎衰竭（CrCl <30 / 透析）禁用**
- 對藥物成分過敏

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：**CBC（血小板）給藥後 2–4h 及每日**、Hb/Hct、出血徵象、腎功能、aPTT/ACT（併用肝素時）
- **交互作用**：與**肝素、口服抗血小板（[[Aspirin(阿斯匹靈)]]、[[Clopidogrel(保栓通)]]、[[Ticagrelor(替格瑞洛)]]）、溶栓劑** → 出血風險相加
- **孕哺**：資料有限，孕期僅在明確效益大於風險時使用

### 自我測驗（選擇題，答案摺疊）

**Q1.** 病人做 PCI 後 3 小時，血小板從 250,000 驟降到 18,000，無其他明顯原因。最可能與哪個藥有關？
- A. Aspirin
- B. Abciximab（GP IIb/IIIa 抑制劑）
- C. Metoprolol
- D. Atorvastatin

> [!success]- 答案
> **B** — GP IIb/IIIa 抑制劑（尤其 abciximab）可致**急性重度血小板低下**，數小時內驟降。需立即停藥並監測 CBC；<50,000 停藥。

**Q2.** 使用 Eptifibatide 時，哪個情況為禁忌？
- A. Troponin 陽性 ACS
- B. 準備做 PCI
- C. 嚴重腎衰竭（CrCl <30 / 透析）
- D. 合併使用 aspirin

> [!success]- 答案
> **C** — Eptifibatide 經腎排泄，**CrCl <30 或透析禁用**；CrCl <50 需減量。Tirofiban 腎功能不良也需減量。

**Q3.** 關於 GP IIb/IIIa 抑制劑何者正確？
- A. 為口服長期抗血小板藥
- B. 抑制血小板聚集的最終共同路徑，靜脈給藥
- C. 無出血風險
- D. 不需監測血小板

> [!success]- 答案
> **B** — GP IIb/IIIa 抑制劑作用於血小板聚集**最終共同路徑**、為靜脈藥、僅住院/導管室使用；出血與血小板低下風險高，須密切監測 CBC。
