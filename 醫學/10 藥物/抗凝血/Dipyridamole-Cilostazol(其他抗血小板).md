# Dipyridamole / Cilostazol（其他抗血小板）

> [!abstract] 一句話：**兩個非 aspirin / 非 P2Y12 的抗血小板藥**。**Dipyridamole**：抑制血小板腺苷再攝取 + 抑制 PDE → 升高 cAMP/cGMP 抗聚集 + 血管擴張，臨床多以**緩釋 dipyridamole + aspirin 複方（Aggrenox）做二級中風預防**。**Cilostazol**：**PDE3 抑制劑**，抗血小板 + 血管擴張，用於**間歇性跛行（周邊動脈疾病）改善行走距離**，**心衰竭禁用**（PDE3 抑制劑增加死亡）。

> [!danger] 🚨 值班紅旗（用藥前先想）
> 1. **Cilostazol 於任何嚴重程度心衰竭 → 絕對禁用**（PDE3 抑制劑類如 milrinone 已知增加心衰病人死亡率）
> 2. **Dipyridamole 血管擴張作用**：可致**頭痛、臉潮紅、低血壓**；**冠狀動脈竊血（coronary steal）** → 靜脈 dipyridamole 用於心肌灌注壓力測試（也因此不穩定心絞痛慎用）
> 3. **Cilostazol 常見頭痛、心悸、腹瀉**，並可能**心搏過速**——冠心病病人注意
> 4. **兩者皆增加出血風險**（與其他抗血小板/抗凝併用）
> 5. **Cilostazol 經 CYP3A4/2C19 代謝**：與 CYP3A4 強抑制劑（ketoconazole、diltiazem、erythromycin、葡萄柚汁）併用需減量

## 💊 適應症（治療哪些疾病）
**Dipyridamole**
- **缺血性中風 / TIA 二級預防 [[Ischemic Stroke(缺血性中風)]]**（緩釋 dipyridamole + aspirin 複方 Aggrenox）
- 人工心臟瓣膜輔助抗血栓（+ warfarin，較少用）
- 靜脈型：**心肌灌注壓力測試**（藥物性負荷，非治療）

**Cilostazol**
- **間歇性跛行 / 周邊動脈疾病 [[Peripheral Arterial Disease(周邊動脈疾病)]]**（改善無痛行走距離）
- （部分）預防支架再狹窄、腦部小血管病變輔助（依指引）

## 📏 劑量 / 用法（成人）
| 藥物 | 劑量 |
| --- | --- |
| **Dipyridamole ER + Aspirin（Aggrenox）** | dipyridamole ER 200mg + aspirin 25mg，**一天兩次 BID** |
| **Cilostazol** | **100 mg PO BID**（飯前 30 分或飯後 2 小時，空腹吸收好）；與 CYP3A4/2C19 抑制劑併用減至 50mg BID |
- **Cilostazol 起效慢**：改善跛行需**持續數週（2–12 週）**才明顯
- **腎/肝**：嚴重腎（CrCl <25）或中重度肝損 cilostazol 慎用

## ⚠️ 副作用（常見 / 嚴重）
**Dipyridamole**
- 常見：**頭痛**（血管擴張，最常見）、頭暈、臉潮紅、腸胃不適、低血壓
- 嚴重：冠狀動脈竊血致心絞痛惡化（尤其靜脈型負荷測試）、出血

**Cilostazol**
- 常見：**頭痛、心悸 / 心搏過速、腹瀉、頭暈、周邊水腫**
- 嚴重：**心衰竭惡化 / 增加死亡（PDE3 效應）**、心律不整、出血

## 🚫 禁忌症
**Dipyridamole**：對成分過敏；不穩定心絞痛 / 近期心肌梗塞慎用（血管擴張/竊血）
**Cilostazol**：
- **任何嚴重程度的鬱血性心衰竭（CHF）→ 絕對禁用**
- 活動性出血、出血體質
- 嚴重腎（CrCl <25）/ 中重度肝功能不全
- 對成分過敏

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：出血徵象、（cilostazol）心率/心悸/心衰徵象、行走距離療效、血壓
- **交互作用**：
  - **Cilostazol + CYP3A4/2C19 抑制劑**（ketoconazole、erythromycin、diltiazem、omeprazole、葡萄柚汁）→ 濃度↑需減量
  - 兩者與其他抗血小板/抗凝 → 出血相加
  - **Dipyridamole 減弱 adenosine 代謝** → 併用 adenosine（陣發性心搏過速治療）作用增強需注意
- **孕哺**：資料有限，避免非必要使用

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名間歇性跛行合併 EF 30% 心衰竭的病人，下列哪個藥為禁忌？
- A. Aspirin
- B. Cilostazol
- C. Clopidogrel
- D. Statin

> [!success]- 答案
> **B** — **Cilostazol 於任何嚴重程度心衰竭為絕對禁忌**（PDE3 抑制劑增加心衰病人死亡率）。此病人 EF 30% 心衰，不可用 cilostazol。

**Q2.** 使用緩釋 dipyridamole + aspirin 複方最常見的副作用是？
- A. 便秘
- B. 頭痛
- C. 高血糖
- D. 聽力喪失

> [!success]- 答案
> **B** — Dipyridamole 血管擴張作用使**頭痛**成為最常見副作用，常導致病人停藥，可漸進加量減緩。

**Q3.** 關於 cilostazol 的用途與特性，何者正確？
- A. 用於急性心肌梗塞立即再灌流
- B. PDE3 抑制劑，用於間歇性跛行，心衰竭禁用
- C. 是口服抗凝血劑
- D. 需監測 INR

> [!success]- 答案
> **B** — Cilostazol 是 **PDE3 抑制劑**，抗血小板 + 血管擴張，用於**間歇性跛行**改善行走距離，**心衰竭禁用**。它不是抗凝血劑、不需 INR。
