# Desmopressin（去氨加壓素 / DDAVP — 止血用途）

> [!abstract] 一句話：**合成血管加壓素（vasopressin）類似物**，只保留抗利尿作用（V2 受體）、去除升壓作用。**止血用途**：刺激血管內皮釋放**儲存的 von Willebrand factor（vWF）與 factor VIII** → 提升止血。用於**輕型血友病 A、type 1 von Willebrand disease、尿毒症血小板功能異常**的出血/術前預防。也用於**中樞性尿崩症、夜間遺尿**（本卡聚焦止血）。

> [!danger] 🚨 值班紅旗（用藥前先想）
> 1. **低血鈉 / 水中毒是最重要併發症**：抗利尿作用 → **水滯留、稀釋性低血鈉、可致抽搐**；用藥期間**限水、監測血鈉**，兒童與老人尤危險
> 2. **重複給藥會「快速耐受（tachyphylaxis）」**：vWF/VIII 儲存被排空，**約 2–4 次後效果遞減**，不能長期反覆依賴
> 3. **心血管慎用**：早期升壓/血管作用 + 水滯留 → **不穩定冠心病、心衰竭慎用**；老年心血管病人注意
> 4. **不適用型別**：**嚴重（type 3）von Willebrand disease、type 2B vWD（可致血小板低下惡化）、嚴重血友病 A** 無效或禁用 → 需用凝血因子製劑
> 5. **試驗劑量**：血友病/vWD 擇期手術前建議先做**治療前反應測試**，確認個別有效才依賴

## 💊 適應症（止血相關）
- **輕型血友病 A [[Hemophilia A(血友病A)]]**（factor VIII 輕度不足，DDAVP 可暫時提升）
- **Type 1 von Willebrand disease [[von Willebrand Disease(類血友病)]]**（最反應良好的型別）
- **尿毒症血小板功能異常 [[Uremic Bleeding(尿毒出血)]]**（腎衰竭病人出血或術前）
- **手術 / 拔牙 / 侵入處置前止血預防**（上述族群）
- （非止血）中樞性尿崩症、兒童夜間遺尿——不在本卡重點

## 📏 劑量 / 用法（成人，止血）
| 途徑 | 劑量 |
| --- | --- |
| **靜脈 IV** | **0.3 mcg/kg** 稀釋於生理食鹽水，緩慢輸注 20–30 分鐘（止血標準） |
| **皮下 SC** | 0.3 mcg/kg |
| **鼻噴劑（高濃度止血劑型）** | 依體重，⚠️與尿崩症/遺尿劑型不同，勿混用 |
- **起效**：IV 給後約 **30–60 分鐘** vWF/VIII 達高峰，效果持續數小時
- **限制頻率**：因 tachyphylaxis，通常**每 12–24 小時一次、連續數次後失效**
- **給藥時併限水、監測血鈉**（尤其重複給或兒童）

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：臉潮紅、頭痛、**輕度血壓變化 / 心搏過速**、鼻塞（鼻噴）
- **嚴重**：
  - **低血鈉 / 水中毒 → 抽搐、意識改變**（最重要）
  - 血栓事件（罕見，尤其心血管高風險者）
  - 快速輸注致低血壓
  - 過敏反應（罕見）

## 🚫 禁忌症
- **低血鈉 / 有低血鈉風險（心衰、SIADH、過度飲水）**
- **Type 2B von Willebrand disease**（可加重血小板低下）、**type 3 vWD / 嚴重血友病 A 無效**
- 不穩定冠心病、嚴重心衰竭（慎用/避免）
- CrCl 嚴重下降（抗利尿劑型考量）
- 對成分過敏

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：**血鈉、體液平衡 / 尿量、血壓**、出血止血反應；重複給藥更需密切監測血鈉
- **交互作用**：與**促進 SIADH / 低血鈉的藥（SSRI、carbamazepine、chlorpropamide、NSAID）**併用 → 低血鈉風險↑
- **孕哺**：孕期可用於必要止血（經驗上安全）；哺乳一般可用

## 🔗 相關
- 止血搭配：[[Tranexamic Acid(傳明酸)]]（抗纖溶，常與 DDAVP 合併用於黏膜/牙科出血）、factor 製劑、cryoprecipitate
- 對照概念：真正嚴重凝血因子缺乏需**因子取代療法**（DDAVP 僅輕型有效）
- 相關逆轉：[[Anticoagulant Reversal Agents(抗凝逆轉劑總覽)]]

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名輕型血友病 A 病人擬拔牙，欲暫時提升凝血因子。最適合的藥物是？
- A. Desmopressin (DDAVP)
- B. Warfarin
- C. Heparin
- D. Aspirin

> [!success]- 答案
> **A** — DDAVP 刺激釋放儲存的 vWF/factor VIII，**輕型血友病 A** 可暫時提升 factor VIII 供小手術/拔牙止血（常合併 tranexamic acid）。嚴重型則需 factor 取代。

**Q2.** 使用 DDAVP 止血最需監測與預防哪個併發症？
- A. 高血鉀
- B. 低血鈉 / 水中毒
- C. 高血糖
- D. 肝衰竭

> [!success]- 答案
> **B** — DDAVP 有抗利尿作用致水滯留，最重要併發症是**低血鈉 / 水中毒**（可致抽搐）。用藥期間須限水並監測血鈉，兒童與老人風險最高。

**Q3.** 關於 DDAVP 重複給藥，何者正確？
- A. 可無限次重複維持效果
- B. 會快速耐受（tachyphylaxis），數次後失效
- C. 越給效果越強
- D. 不影響血鈉

> [!success]- 答案
> **B** — DDAVP 依賴內皮儲存的 vWF/VIII，重複給藥會**快速耐受**（約 2–4 次後遞減），不能長期依賴；且重複給更易低血鈉。
