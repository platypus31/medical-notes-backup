# Heparin（未分餾肝素 / UFH）

> [!abstract] 一句話：結合並活化 **antithrombin III (ATIII)**，加速中和 **thrombin (IIa)** 與 **Factor Xa**（1:1 抑制兩者）→ 立即抗凝血。半衰期短、可 IV titrate、可被 **protamine 完全逆轉**，是需要「快開快關」情境（術中、腎衰、高出血風險）的首選抗凝血劑。監測 **aPTT**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **HIT（Heparin-induced thrombocytopenia）**：Type II 為免疫性（抗 PF4-heparin 抗體），典型在用藥 **第 4–14 天**血小板下降 >50%，弔詭的是造成**血栓（不是出血）**。懷疑 HIT → **立即停所有 heparin（含 flush、LMWH）**，改用非肝素抗凝（argatroban / fondaparinux），**絕不輸血小板**
> 2. **監測 aPTT**：治療劑量目標 aPTT 1.5–2.5 倍（或依 anti-Xa protocol），依 nomogram 調整 infusion
> 3. **出血**：活動性出血、近期手術/中風、嚴重血小板低下 → 禁用；逆轉用 **protamine sulfate**（1mg 中和 ~100 U heparin，依剩餘時間遞減）
> 4. **腎衰竭病人優於 LMWH**：UFH 主要由網狀內皮系統/肝代謝，**不依賴腎排除**，CrCl 很低時比 LMWH 安全可控

## 💊 適應症（治療哪些疾病）
- **靜脈血栓栓塞（VTE）**：[[Deep Vein Thrombosis(深部靜脈栓塞)]]、[[Pulmonary Embolism(肺栓塞)]] 之急性抗凝
- **急性冠心症**：[[Acute Coronary Syndrome(急性冠心症)]] / [[Myocardial Infarction(心肌梗塞)]]（NSTEMI/STEMI 抗凝，常配 PCI）
- **心房顫動**：[[醫學/11心電圖判讀/Atrial Fibrillation(心房顫動)]] 住院期間 bridging / 快速抗凝
- **體外循環 / 洗腎 / 導管沖洗**：CPB、CRRT、透析迴路抗凝
- **孕婦抗凝**（不過胎盤，孕期安全，取代 warfarin）

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| VTE 治療（weight-based） | Bolus **80 U/kg** IV → **18 U/kg/hr** infusion，依 aPTT nomogram 調 |
| ACS 抗凝 | Bolus 60 U/kg（max 4000U）→ 12 U/kg/hr（依院內 protocol） |
| 預防（DVT prophylaxis） | **5000 U SC Q8–12H**（低劑量，不需監測 aPTT） |
| 逆轉 | Protamine sulfate 1mg / 100U heparin（依時間遞減，max 50mg/次） |
- **監測**：治療劑量測 **aPTT**（目標 1.5–2.5×）或 **anti-Xa**（0.3–0.7）；預防劑量不需監測
- **腎功能**：不需依腎調整（優於 LMWH）；**半衰期短 ~1–1.5h**，停藥後迅速消退
- 給法：治療用 **continuous IV**；預防用 SC

## ⚠️ 副作用（常見 / 嚴重）
- **出血**（最常見嚴重）：GI、顱內、注射部位
- **HIT**（最需警覺）：Type I（非免疫、輕微、早期、可續用）vs **Type II（免疫、4–14 天、血栓、必須停藥）**
- **骨質疏鬆**（長期使用 >1 個月）
- **高血鉀**（抑制醛固酮合成）
- 注射部位瘀青、輕微 transaminase 上升

## 🚫 禁忌症
- 活動性大出血、出血性疾病（嚴重血小板低下、血友病）
- **HIT 病史**（絕對禁用，含 LMWH）
- 近期中樞神經/眼科手術、出血性中風
- 嚴重未控制高血壓
- 對 heparin 過敏

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：aPTT（治療劑量 q6h 調整至穩定）、**血小板數（baseline + 定期監測 HIT）**、Hb/Hct、K⁺
- **逆轉劑**：**Protamine sulfate**（可完全中和 UFH；對 LMWH 僅部分逆轉）
- **交互作用**：與其他抗凝/抗血小板（[[Aspirin(阿斯匹靈)]]、[[Clopidogrel(保栓通)]]、[[Warfarin(可邁丁)]]、NSAID）出血加乘；warfarin 轉換時需 overlap
- **孕哺**：**孕期首選抗凝**（分子大不過胎盤），哺乳安全
- **HIT 檢驗**：4T score 評估 → PF4 antibody ELISA → serotonin release assay 確診

## 🔗 相關

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位接受 UFH 治療第 7 天的病人，血小板由 250K 降到 90K，同時新發現右下肢 DVT。最適當的處置是？
- A. 輸血小板並繼續 heparin
- B. 停所有 heparin，改用 argatroban，送 PF4 抗體
- C. 加大 heparin 劑量治療新血栓
- D. 改用 LMWH

> [!success]- 答案
> **B** — 血小板 >50% 下降 + 用藥 4–14 天 + 新發血栓，高度懷疑 **HIT Type II**。必須**立即停所有 heparin（含 LMWH，兩者有交叉反應）**，改非肝素抗凝（argatroban），送 4T score/PF4 抗體。**絕不輸血小板**（會助長血栓）。

**Q2.** 關於 UFH 與 LMWH 的比較，下列何者正確？
- A. LMWH 需常規監測 aPTT
- B. UFH 主要靠腎排除，腎衰時首選
- C. UFH 半衰期短、可被 protamine 完全逆轉，適合高出血風險情境
- D. protamine 對 LMWH 可完全逆轉

> [!success]- 答案
> **C** — UFH 半衰期短、IV 可 titrate、protamine **完全**逆轉，適合術前/腎衰/高出血風險。LMWH 不需常規監測、靠**腎**排除（腎衰慎用），protamine 對其僅**部分**逆轉。

**Q3.** 一位需要緊急手術但正在洗腎（CrCl 極低）的病人需要抗凝，下列何者最合適？
- A. Enoxaparin 1mg/kg Q12H
- B. Warfarin loading
- C. UFH continuous infusion
- D. Fondaparinux

> [!success]- 答案
> **C** — **UFH** 不依賴腎排除、半衰期短、可即時逆轉，是腎衰竭 + 需要隨時可關掉抗凝（術前）情境的首選。LMWH/fondaparinux 靠腎排除會蓄積，warfarin 起效慢且不可急關。
