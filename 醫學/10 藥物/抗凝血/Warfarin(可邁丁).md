# Warfarin（可邁丁 / 華法林）

> [!abstract] 一句話：抑制 **VKORC1（維生素 K epoxide reductase）**，減少維生素 K 依賴凝血因子 **II、VII、IX、X** 及抗凝蛋白 **protein C、S** 的合成 → 抗凝血。**口服、起效慢（數天）、需監測 INR（目標 2–3）**，是長期口服抗凝的經典藥；食物/藥物交互作用極多，治療窗窄。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **INR 是命根**：目標多為 **2–3**（機械瓣膜 2.5–3.5）；INR 過高→出血、過低→血栓。定期回診調藥
> 2. **Warfarin-induced skin necrosis**：起始初期 **protein C（半衰期短）先降** → 短暫**高凝血狀態** → 皮膚壞死；故高血栓風險者起始須 **heparin/LMWH bridging**，勿單獨 warfarin loading
> 3. **食物-藥物交互作用海量**：綠葉蔬菜（維生素 K）↓藥效、抗生素/amiodarone/azole/[[Statin(史他汀)]] 等 CYP2C9 抑制劑 ↑藥效；飲食與併用藥一改就要回驗 INR
> 4. **孕婦絕對禁用**（過胎盤、致畸 fetal warfarin syndrome、胎兒出血）→ 孕期改 heparin/LMWH
> 5. **大出血逆轉**：**維生素 K + PCC（凝血酶原複合體）**（急救首選，比 FFP 快），非急性可只調藥/口服 vit K

## 💊 適應症（治療哪些疾病）
- **心房顫動抗凝**：[[醫學/11心電圖判讀/Atrial Fibrillation(心房顫動)]] 中風預防（CHA₂DS₂-VASc）
- **靜脈血栓栓塞**：[[Deep Vein Thrombosis(深部靜脈栓塞)]]、[[Pulmonary Embolism(肺栓塞)]] 長期抗凝
- **機械性人工心臟瓣膜**（DOAC 禁用，**warfarin 為唯一選擇**，目標 INR 2.5–3.5）
- **抗磷脂質症候群**（APS，DOAC 效果不佳，用 warfarin）
- 部分心內血栓、擴張型心肌病

## 📏 劑量 / 用法（成人）
| 項目 | 內容 |
| --- | --- |
| 起始 | 通常 **5 mg PO QD**（老年/出血風險高者 2.5mg），依 INR 調 |
| 維持 | 依 INR 個別化（常見 2–7.5mg/day） |
| 目標 INR | 一般 **2–3**；機械瓣膜 **2.5–3.5** |
| 起效 | **慢（36–72h 才見效、5–7 天達穩定）**，因既有凝血因子需耗盡 |
- **腎/肝**：嚴重肝病使凝血因子合成↓、代謝↓，作用增強需謹慎；腎不需大調整
- **bridging**：急性血栓 / 高風險者起始以 heparin/LMWH 重疊至 **INR 連續 2 天 ≥2**

## ⚠️ 副作用（常見 / 嚴重）
- **出血**（最常見嚴重）：GI、顱內、泌尿道
- **Warfarin-induced skin necrosis**（起始期，protein C 缺乏者風險高）
- **Purple toe syndrome**（膽固醇栓塞，罕見）
- 致畸性（孕期）
- 掉髮、皮疹（少見）

## 🚫 禁忌症
- 活動性大出血、出血傾向、近期大手術/出血性中風
- **懷孕**（致畸，絕對禁忌）
- 嚴重肝病、無法配合 INR 監測者
- 跌倒高風險 + 認知障礙（相對禁忌，出血後果嚴重）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：**INR/PT**（起始密集、穩定後可 4 週一次）；CBC、肝功能
- **逆轉**：
  - 輕度 INR 高、無出血 → 停藥 ± 口服 vit K
  - 大出血 → **維生素 K（IV）+ PCC（4-factor）**（首選，快）或 FFP
- **交互作用（極多，代謝經 CYP2C9）**：
  - **↑藥效（增出血）**：amiodarone、metronidazole、fluconazole/azole、TMP-SMX、[[Statin(史他汀)]]、cimetidine、NSAID、抗生素（改變腸道菌叢/vit K）
  - **↓藥效（增血栓）**：rifampin、barbiturate、phenobarbital、carbamazepine、大量維生素 K 食物（深綠蔬菜）
  - **基因**：VKORC1 / CYP2C9 多型性影響劑量需求
- **孕哺**：孕期禁用（改 heparin/LMWH）；**哺乳可**（不進母乳）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位剛開始 warfarin 治療 DVT 的病人，第 3 天出現大腿皮膚黑色壞死斑塊。最可能原因是？
- A. 藥物過敏
- B. Protein C 先降造成的 warfarin-induced skin necrosis
- C. INR 太低
- D. 蜂窩性組織炎

> [!success]- 答案
> **B** — 起始期 **protein C（半衰期短）先下降** → 短暫高凝血 → 微血栓皮膚壞死。預防之道是**起始重疊 heparin/LMWH bridging**，勿單獨 warfarin loading。

**Q2.** 一位長期服 warfarin 且 INR 穩定的病人因肺炎開始服用 metronidazole，一週後回診。最需注意的是？
- A. INR 可能升高、出血風險增
- B. INR 可能降低、血栓風險增
- C. 無交互作用
- D. 需停 warfarin 改 aspirin

> [!success]- 答案
> **A** — metronidazole 抑制 CYP2C9 → **warfarin 藥效增強、INR 升高、出血風險增**。加任何新藥都應回驗 INR 並視情況減量。

**Q3.** 一位機械二尖瓣置換病人詢問可否改用 apixaban（DOAC）以免驗 INR。最正確的回答是？
- A. 可以，DOAC 都能取代 warfarin
- B. 不行，機械瓣膜只能用 warfarin（DOAC 禁用）
- C. 可以，但要加倍劑量
- D. 改用 aspirin 即可

> [!success]- 答案
> **B** — **機械性人工瓣膜是 DOAC 的禁忌**（RE-ALIGN 試驗顯示 dabigatran 血栓/出血都增加），**必須用 warfarin**（目標 INR 2.5–3.5）。
