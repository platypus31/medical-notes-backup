# Aspirin（阿斯匹靈 / ASA）

> [!abstract] 一句話：**不可逆**乙醯化 COX-1 → 抑制 thromboxane A2（TXA2）→ 抗血小板凝集；低劑量（75–100mg）走「心血管保護」，高劑量走「止痛/退燒/抗發炎」。因為血小板無細胞核不能再合成 COX，作用**持續整個血小板壽命 7–10 天**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **活動性出血 / 近期 GI bleeding / 出血性中風** → 禁用
> 2. **兒童 + 病毒感染（流感、水痘）** → **Reye syndrome**（急性腦病 + 肝衰竭），絕對禁用退燒用途
> 3. **Aspirin 過敏 / AERD**（Aspirin-exacerbated respiratory disease：氣喘＋鼻息肉＋ASA 敏感）→ 可誘發嚴重支氣管痙攣
> 4. **手術/侵入性處置前** → 是否停藥要權衡「出血 vs 血栓」，心血管高風險者常「不停」（尤其近期放支架）

## 💊 適應症（治療哪些疾病）
- **心血管二級預防（主戰場）**：[[Acute Coronary Syndrome(急性冠心症)]]、[[Myocardial Infarction(心肌梗塞)]]、PCI 後、[[Ischemic Stroke(缺血性中風)]] / TIA
- **急性 ACS loading**：懷疑心梗立即 **160–325mg 嚼服**（嚼碎加速吸收）
- **雙抗血小板治療 DAPT**：與 [[Clopidogrel(保栓通)]] / [[Ticagrelor(替格瑞洛)]] 合用（放支架後）
- **其他**：川崎病（高劑量→低劑量）、抗磷脂質症候群、子癲前症高危孕婦預防（低劑量）
- **高劑量（現已少用）**：止痛、退燒、抗發炎

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| ACS 急性 loading | 160–325 mg **嚼服** ×1 |
| 心血管二級預防維持 | **75–100 mg PO QD**（台灣多用 100mg） |
| 止痛 / 退燒 | 325–650 mg PO Q4–6H（max 4g/day） |
| 川崎病急性期 | 80–100 mg/kg/day 分次（退燒後降 3–5 mg/kg/day 抗血小板） |
- **腎功能**：嚴重腎衰慎用（NSAID 效應傷腎）；**肝**：嚴重肝病避免
- 低劑量抗血小板**不需**依腎調整，但整體出血風險要評估

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：消化不良、GI 刺激、輕微出血傾向（瘀青、牙齦出血）
- **嚴重**：
  - **GI bleeding / 消化性潰瘍**（最常見的嚴重併發，長期使用可併 PPI 保護）
  - **顱內出血**（尤其合併抗凝血劑 / 未控制高血壓）
  - **Reye syndrome**（兒童病毒感染）
  - **過敏 / AERD 誘發氣喘**
  - **耳鳴、聽力障礙**（水楊酸中毒早期徵象）
  - **水楊酸中毒**（過量）：呼吸性鹼中毒 + 代謝性酸中毒混合、高溫、意識改變

## 🚫 禁忌症
- 活動性出血 / 出血性疾病（血友病、嚴重血小板低下）
- Aspirin 或 NSAID 過敏、AERD
- 活動性消化性潰瘍
- 兒童病毒感染（Reye）
- 嚴重肝腎功能不全
- 孕婦**第三孕期**（高劑量→胎兒動脈導管早閉、產程出血）；低劑量子癲前症預防為例外

## 🔍 監測 / 交互作用 / 孕哺
- **交互作用**：
  - **NSAID（如 ibuprofen）**：競爭 COX-1 結合位 → **削弱 aspirin 的心血管保護**（服用順序有講究，ASA 先服）
  - **抗凝血劑（[[Warfarin(可邁丁)]] / DOAC）+ 其他抗血小板**：出血加乘
  - **ACEI / ARB / 利尿劑**：降低降壓效果、增腎損風險
  - **Methotrexate**：↑ MTX 毒性
  - 酒精 → ↑ GI 出血
- **孕哺**：第三孕期禁（如上）；哺乳低劑量一般可
- **G6PD 缺乏**：高劑量可誘發溶血，慎用

## 🔗 相關
- 疾病：[[Acute Coronary Syndrome(急性冠心症)]]、[[Ischemic Stroke(缺血性中風)]]
- 同類 / 併用：[[Clopidogrel(保栓通)]]、[[Ticagrelor(替格瑞洛)]]、[[NSAID(非類固醇消炎藥)]]
- 對照：[[Warfarin(可邁丁)]]（抗凝血 ≠ 抗血小板，機轉不同）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 65 歲男性因胸痛到急診，心電圖顯示 ST 上升，懷疑 STEMI。下列 Aspirin 給法何者最適當？
- A. 100mg 吞服
- B. 160–325mg 嚼服
- C. 等心導管室確認後再給
- D. 先給 Clopidogrel，Aspirin 可不用

> [!success]- 答案
> **B** — ACS 急性期應立即給 160–325mg **嚼服**，嚼碎加速吸收以盡快抗血小板；不需等導管室，且 Aspirin 是 ACS 的基石而非可省略。

**Q2.** 4 歲兒童因流感高燒，家長想用家中 Aspirin 退燒。最主要的顧慮是？
- A. 傷腎
- B. Reye syndrome
- C. 過敏
- D. 耳鳴

> [!success]- 答案
> **B** — 兒童在病毒感染（流感、水痘）期間使用 Aspirin 與 **Reye syndrome**（急性腦病＋肝衰竭）相關，應改用 acetaminophen。

**Q3.** 長期服用低劑量 Aspirin 做心血管二級預防的病人，因關節痛自行加服 ibuprofen。此舉的主要問題是？
- A. 兩者都是 NSAID 會中毒
- B. Ibuprofen 競爭 COX-1，削弱 Aspirin 的心血管保護
- C. 會導致低血糖
- D. 完全沒有交互作用

> [!success]- 答案
> **B** — Ibuprofen 與 Aspirin 競爭血小板 COX-1 結合位，若順序不當會**削弱 Aspirin 的抗血小板/心血管保護**效果；建議先服 Aspirin，或選用不干擾的止痛選項。
