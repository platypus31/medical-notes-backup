# Anticoagulant Reversal Agents（抗凝逆轉劑總覽）

> [!abstract] 一句話：**各種抗凝血劑各有其專屬逆轉方式**——記住「**誰對誰**」是值班遇大出血時的救命關鍵。Protamine↔肝素、Vitamin K↔warfarin、Idarucizumab↔dabigatran、Andexanet alfa↔Xa 抑制劑、4-factor PCC↔warfarin/Xa（起效快）。急性大出血或緊急手術時**時間就是生命**，選對逆轉劑 + 停用原藥 + 支持性輸血。

> [!danger] 🚨 值班紅旗（用藥前先想）
> 1. **對應關係要背熟**：肝素→**Protamine**；warfarin→**Vitamin K + PCC/FFP**；dabigatran→**Idarucizumab**；Xa 抑制劑(-xaban)→**Andexanet alfa 或 PCC**
> 2. **Vitamin K 起效慢**：口服/IV 需 **6–24 小時**才完全逆轉 INR，**急性大出血不能只靠它**，須併用 **4-factor PCC（分鐘級）**或 FFP
> 3. **Protamine 本身可致命**：過快輸注→**低血壓、心搏過緩、過敏性休克**；魚類過敏 / 曾用魚精蛋白胰島素(NPH) / 輸精管結紮者過敏風險高；**只能部分逆轉 LMWH**
> 4. **PCC 優於 FFP**（急救）：**4-factor PCC 濃縮、體積小、起效快、不需解凍配血**；FFP 量大、需時間、有輸血反應與容積負荷風險
> 5. **逆轉後血栓反彈風險**：逆轉抗凝等於恢復原本的高凝狀態（AF、機械瓣膜、VTE），出血穩定後需評估**何時重啟抗凝**

## 💊 適應症（逆轉哪種藥）
| 逆轉劑 | 對應抗凝劑 | 機轉 |
| --- | --- | --- |
| **Protamine sulfate** | **肝素 [[Heparin(未分餾肝素UFH)]]**（完全）；**LMWH [[Enoxaparin(依諾肝素)]]**（部分~60%） | 帶正電與帶負電肝素結合形成無活性複合體 |
| **Vitamin K（phytonadione）** | **Warfarin [[Warfarin(可邁丁)]]** | 補充維生素 K，恢復 II/VII/IX/X 合成（起效慢） |
| **4-factor PCC**（含 II/VII/IX/X + protein C/S） | **Warfarin**（首選急救）、**Xa 抑制劑**（替代） | 直接補充凝血因子，起效快 |
| **Idarucizumab（泛肝寧）** | **Dabigatran** | 單株抗體片段，高親和力結合 dabigatran 中和 |
| **Andexanet alfa** | **Rivaroxaban / Apixaban / Edoxaban（Xa 抑制劑）** | 重組去活化 Xa 誘餌，競爭結合藥物 |
| **FFP（新鮮冷凍血漿）** | 廣泛凝血因子缺乏 / 無 PCC 時 | 補充全部凝血因子（體積大、慢） |

## 📏 劑量 / 用法（成人）
| 藥物 | 用法 |
| --- | --- |
| **Protamine** | 逆轉 UFH：**1 mg 中和約 100 units 肝素**（依最近一次肝素劑量與時間計算），**慢速 IV（≥10 分鐘）**，單次上限 ~50mg |
| **Vitamin K** | 大出血：**10 mg 慢速 IV**（配合 PCC）；INR 高無出血：口服 1–10mg 依 INR；⚠️IV 過快有過敏風險 |
| **4-factor PCC** | 依體重與 INR（如 25–50 units/kg），⚠️依院規/仿單 |
| **Idarucizumab** | **5 g IV**（2 × 2.5g bolus），單次固定劑量 |
| **Andexanet alfa** | 依最後一次 Xa 抑制劑劑量與時間分高/低劑量方案（bolus + 2hr 輸注），⚠️依仿單 |
- **急性 warfarin 大出血標準組合**：**停 warfarin + Vitamin K 10mg IV + 4-factor PCC**（PCC 立即補因子，Vit K 維持後續合成）

## ⚠️ 副作用（常見 / 嚴重）
- **Protamine**：**低血壓、心搏過緩、過敏 / 過敏性休克**（快速輸注時），肺動脈高壓（罕見嚴重反應）
- **Vitamin K**：IV 快速給藥可致**類過敏反應**（建議稀釋慢給）
- **PCC / FFP**：**血栓栓塞風險**（補太多凝血因子）、FFP 有輸血反應/TRALI/容積負荷
- **Idarucizumab / Andexanet**：血栓事件、輸注反應；Andexanet 有黑框血栓警語

## 🚫 禁忌症
- **Protamine**：已知魚精蛋白嚴重過敏（相對禁；魚過敏、NPH 胰島素、輸精管結紮病史者慎用並備急救）
- **PCC**：已知肝素誘發血小板減少症（HIT，因含微量肝素）、DIC 活動期慎用
- 各逆轉劑對其成分過敏者禁用

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：Protamine 給時嚴密監測血壓/心律；warfarin 逆轉追 **INR**（PCC 後 30 分、之後 6–12h，Vit K 效果延後）；肝素逆轉追 aPTT；出血徵象與 Hb
- **注意**：Protamine 過量本身有**抗凝作用**（矛盾性出血）；逆轉後評估血栓反彈與抗凝重啟時機
- **孕哺**：Vitamin K 孕期可用；其他救命情境個別權衡

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名長期服用 warfarin 病人發生顱內出血，INR 5.2。最適當的立即處置？
- A. 只給口服 Vitamin K 觀察
- B. 停 warfarin + Vitamin K 10mg IV + 4-factor PCC
- C. 給 Protamine
- D. 給 Idarucizumab

> [!success]- 答案
> **B** — warfarin 大出血需立即逆轉：停藥 + Vit K 10mg IV（維持後續因子合成）+ 4-factor PCC（分鐘級補因子）。單靠 Vit K 起效太慢；Protamine 逆轉肝素、Idarucizumab 逆轉 dabigatran。

**Q2.** 服用 dabigatran 的病人需緊急手術，最適合的逆轉劑？
- A. Protamine
- B. Vitamin K
- C. Idarucizumab
- D. Andexanet alfa

> [!success]- 答案
> **C** — Dabigatran（直接凝血酶抑制劑）的專一逆轉劑是 **Idarucizumab（5g IV）**。Andexanet alfa 是逆轉 Xa 抑制劑（-xaban）用的。

**Q3.** 使用 Protamine 逆轉肝素時，下列何者正確？
- A. 快速大量 IV 推注最有效
- B. 慢速 IV 給藥，注意低血壓與過敏
- C. 對 LMWH 可完全逆轉
- D. 魚過敏者可安全使用

> [!success]- 答案
> **B** — Protamine 須**慢速 IV（≥10 分鐘）**，快速輸注會致低血壓、心搏過緩、過敏性休克；對 LMWH 僅部分逆轉；魚過敏 / NPH 胰島素 / 輸精管結紮者過敏風險高需慎用。
