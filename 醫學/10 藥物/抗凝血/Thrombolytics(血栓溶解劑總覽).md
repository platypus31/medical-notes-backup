# Thrombolytics（血栓溶解劑總覽）

> [!abstract] 一句話：**把 plasminogen 轉成 plasmin → 分解 fibrin 溶解已形成的血栓**。與抗凝血劑（防新血栓）、抗血小板（防聚集）不同，血栓溶解劑是「**主動打通**」既有栓塞。三大救命適應症：**急性 STEMI（無法及時 PCI 時）、急性缺血性中風（時間窗內）、大塊高危險肺栓塞**。最大代價是**出血、尤其顱內出血**，時間窗與禁忌清單是命根。

> [!danger] 🚨 值班紅旗（用藥前先想）
> 1. **顱內出血（ICH）是最致命併發症**：中風給藥前**必須先 CT 排除出血性中風**；任何時候給藥後神經惡化 / 劇烈頭痛 / 血壓飆 → 立即停藥、查 CT
> 2. **時間窗嚴格**：急性缺血性中風 **≤4.5 小時**（部分擴大到 4.5–9h 或影像 mismatch 需專科決定）；STEMI 症狀發作 **≤12 小時**且無法在 120 分鐘內 PCI 才考慮
> 3. **絕對禁忌清單**（記住）：**近期顱內出血 / 已知顱內血管病灶（AVM、動脈瘤、腫瘤）/ 近期缺血性中風（≤3 個月）/ 近期顱內或脊椎手術 / 活動性內出血 / 主動脈剝離懷疑 / 近期嚴重頭部外傷（≤3 個月）/ 出血體質**
> 4. **劑量依「適應症」不同**：中風劑量 ≠ 心肌梗塞劑量 ≠ 肺栓塞劑量，**絕不可套用**（例：中風 alteplase 0.9 mg/kg 上限 90mg；PE 為 100mg 固定劑量）
> 5. **給藥後 24 小時內避免抗凝 / 抗血小板 / 侵入性處置**（中風尤其）；血壓需嚴格控制（中風 <185/110 才給、給後維持 <180/105）

## 💊 適應症（治療哪些疾病）
- **急性 ST 段上升心肌梗塞 [[STEMI(急性心肌梗塞)]]**：無法在 120 分鐘內做 primary PCI 時的再灌流選擇（Tenecteplase 最常用，可單次 bolus）
- **急性缺血性腦中風 [[Acute Ischemic Stroke(急性缺血性中風)]]**：Alteplase / Tenecteplase，時間窗 ≤4.5h
- **大塊 / 高危險肺栓塞 [[Pulmonary Embolism(肺栓塞)]]**：合併血流動力學不穩（低血壓/休克）時的救命治療（Alteplase 100mg / 2hr）
- **其他**：導管內溶栓（周邊動脈栓塞、深部靜脈栓塞的 catheter-directed）、堵塞的中央靜脈導管（Alteplase 低劑量通管）

## 📏 劑量 / 用法（成人，依適應症）
| 藥物 | 適應症 | 劑量 |
| --- | --- | --- |
| **Alteplase (tPA)** | 急性缺血性中風 | **0.9 mg/kg（上限 90mg）**，10% bolus 1 分鐘、其餘 90% 於 60 分鐘輸注 |
| **Alteplase (tPA)** | 大塊肺栓塞 | **100 mg IV / 2 小時** |
| **Alteplase (tPA)** | STEMI | 加速給法（accelerated / front-loaded），依體重，⚠️需查證院規 |
| **Tenecteplase (TNK)** | STEMI | **單次 IV bolus 依體重**（30–50 mg，<60kg 30mg 起每 +10kg +5mg），5 秒推注 |
| **Tenecteplase** | 中風 | 0.25 mg/kg 單次 bolus（上限 25mg，越來越常用）⚠️依院規 |
| **Streptokinase** | STEMI（舊藥、開發中國家仍用） | 1.5 MU IV / 30–60 分鐘 |
| **Urokinase** | 導管溶栓 / PE（少用） | 依適應症，⚠️需查證 |
- **Streptokinase 特性**：來自 β 溶血性鏈球菌 → **具抗原性**，會產生抗體，**一生只能用一次**（重複給無效且過敏），可致低血壓
- 給藥前 baseline：CBC、PT/aPTT、血型交叉配對、血壓、（中風）NIHSS + CT

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：出血（穿刺點、牙齦、皮下瘀青）、低血壓（尤其 streptokinase）
- **嚴重**：
  - **顱內出血（ICH）** — 最致命，中風後約 2–6%
  - **大出血**（GI、後腹膜、泌尿道）
  - **再灌流心律不整**（STEMI 溶栓後）
  - **過敏 / 過敏性休克**（streptokinase 抗原性）
  - **血管性水腫**（alteplase 中風，尤其併用 ACEI 者，注意呼吸道）
  - 膽固醇栓塞、少見溶栓後全身性栓塞

## 🚫 禁忌症
**絕對禁忌**（尤其中風）：
- 任何時間曾**顱內出血**
- 已知**顱內血管畸形 / 動脈瘤 / 腫瘤**
- **近 3 個月缺血性中風、嚴重頭部外傷、顱內或脊椎手術**
- **活動性內出血 / 出血體質 / 血小板 <100,000**
- 懷疑**主動脈剝離**、感染性心內膜炎
- 血壓無法控制（>185/110 中風；嚴重高血壓 STEMI）
**相對禁忌**：近期大手術 / 內出血（2–4 週）、懷孕、近期心肺復甦、口服抗凝且 INR 高、糖尿病出血性視網膜病變

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：神經學變化（中風 NIHSS）、生命徵象/血壓（頻繁）、出血徵象、Hb/Hct、給藥後 24h CT（中風常規追蹤）
- **交互作用**：同時使用**抗凝血劑 [[Heparin(未分餾肝素UFH)]] / [[Warfarin(可邁丁)]] / NOAC**、抗血小板 [[Aspirin(阿斯匹靈)]] → 出血風險大增；中風後 24h 內原則避免
- **孕哺**：懷孕為相對禁忌（出血風險）；救命情境個別權衡

## 🔗 相關
- 疾病：[[STEMI(急性心肌梗塞)]]、[[Acute Ischemic Stroke(急性缺血性中風)]]、[[Pulmonary Embolism(肺栓塞)]]
- 對照 / 併用：[[Heparin(未分餾肝素UFH)]]（溶栓後常接續抗凝）、[[Aspirin(阿斯匹靈)]]、[[Tranexamic Acid(傳明酸)]]（相反機轉：抗纖溶止血）
- 逆轉：出血時 → [[Anticoagulant Reversal Agents(抗凝逆轉劑總覽)]]、cryoprecipitate（補 fibrinogen）、[[Tranexamic Acid(傳明酸)]]

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名 70 歲男性突發右側偏癱，發作 3 小時，血壓 170/95，NIHSS 12。下一步最正確？
- A. 立即給 alteplase 0.9 mg/kg
- B. 先做腦部 CT 排除出血再決定溶栓
- C. 直接給 heparin
- D. 觀察等症狀自行改善

> [!success]- 答案
> **B** — 缺血性中風溶栓前**必須先 CT 排除出血性中風**。若 CT 無出血、無其他禁忌、時間窗 ≤4.5h、血壓可控（<185/110），才給 alteplase 0.9 mg/kg。

**Q2.** 關於 Streptokinase 何者正確？
- A. 可重複多次使用
- B. 無抗原性
- C. 具抗原性，一生原則只用一次，且易致低血壓
- D. 是首選中風溶栓藥

> [!success]- 答案
> **C** — Streptokinase 來自鏈球菌具抗原性，重複使用會產生抗體而無效並可過敏，並常引起低血壓；現多被 alteplase/tenecteplase 取代。

**Q3.** 大塊（高危險）肺栓塞合併休克，血栓溶解劑的正確給法？
- A. Alteplase 0.9 mg/kg
- B. Alteplase 100 mg IV / 2 小時
- C. Tenecteplase 依中風劑量
- D. 禁用任何溶栓

> [!success]- 答案
> **B** — 血流動力學不穩的大塊肺栓塞為溶栓救命適應症，alteplase 標準為 **100 mg IV / 2 小時**（勿套用中風的體重劑量）。
