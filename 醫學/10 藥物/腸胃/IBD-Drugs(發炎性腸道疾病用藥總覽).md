# IBD-Drugs（發炎性腸道疾病用藥總覽）

> [!abstract] 一句話：IBD（潰瘍性結腸炎 UC + 克隆氏症 CD）採**階梯式（step-up）或降階式（top-down）**免疫調節治療 — 由輕到重依序為 **5-ASA（美沙拉嗪，UC 首選）→ 類固醇（誘導緩解、不可長期）→ 免疫調節劑（Azathioprine / 6-MP，維持緩解、需查 TPMT）→ 生物製劑（Infliximab 等 anti-TNF，中重度、用前篩結核）**。核心是「誘導緩解（induction）」後轉「維持緩解（maintenance）」，類固醇只誘導不維持。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **Azathioprine / 6-MP 用前必查 TPMT（thiopurine methyltransferase）活性或基因型**：TPMT 缺乏者代謝不掉 → **致命性骨髓抑制（嚴重白血球低下、agranulocytosis）**。用後也須定期追 CBC + 肝功能
> 2. **Infliximab / 所有 anti-TNF 用前必篩潛伏結核（IGRA / TST + 胸片）與 B 型肝炎**：TNF 抑制會使**潛伏結核再活化**（可致播散性/粟粒性 TB）、**B 肝再活化猛爆性肝炎**
> 3. **類固醇只用來「誘導緩解」，絕不可長期維持**：長期 → 骨鬆、感染、腎上腺抑制、血糖升高；停藥須逐步 taper
> 4. **中毒性巨結腸（toxic megacolon）**：急性重度大腸炎併腹脹、發燒、心搏過速 → 禁用止瀉/抗膽鹼藥（會惡化），是外科急症
> 5. **生物製劑 = 免疫抑制**：活動性感染、敗血症時暫停；接種活性疫苗禁忌；長期有淋巴瘤等惡性風險（尤其合併 thiopurine 的年輕男性 → 肝脾 T 細胞淋巴瘤）

## 💊 適應症（治療哪些疾病）
- **潰瘍性結腸炎 UC [[Ulcerative Colitis(潰瘍性結腸炎)]]**：5-ASA 為輕中度首選（誘導 + 維持）
- **克隆氏症 CD [[Crohn Disease(克隆氏症)]]**：5-ASA 效果有限，中重度多需免疫調節劑 / 生物製劑
- **維持緩解**：Azathioprine / 6-MP、生物製劑
- **廔管型克隆氏症、腸外表現（關節炎、壞疽性膿皮症）**：anti-TNF 生物製劑

## 📏 劑量 / 用法（成人 · ⚠️依院規/專科醫師調整）
| 藥物 | 角色 | 劑量（參考，需查證） |
| --- | --- | --- |
| **Mesalamine（5-ASA）** | UC 誘導+維持 | 口服 2–4.8 g/day 分次；遠端病灶可加**灌腸/栓劑** |
| **Sulfasalazine** | 5-ASA 前驅（結腸菌分解出 5-ASA + sulfapyridine） | 2–4 g/day 分次；副作用多來自 sulfapyridine |
| **Prednisolone（類固醇）** | 誘導緩解 | 40–60 mg/day 起，症狀改善後逐步 taper；**不維持** |
| **Budesonide** | 局部作用類固醇（迴腸/右結腸 CD、首渡效應高 → 全身副作用少） | 9 mg/day |
| **Azathioprine** | 維持緩解 | 2–2.5 mg/kg/day（用前查 TPMT） |
| **6-Mercaptopurine（6-MP）** | 維持緩解 | 1–1.5 mg/kg/day |
| **Infliximab（anti-TNF）** | 中重度誘導+維持 | IV 5 mg/kg 第 0、2、6 週誘導 → 每 8 週維持 |
- Sulfasalazine 使用者建議**補充葉酸**（抑制葉酸吸收）
- 其他生物製劑類別：**Adalimumab（皮下 anti-TNF）**、**Vedolizumab（抗 α4β7 integrin，腸道選擇性）**、**Ustekinumab（抗 IL-12/23）**

## ⚠️ 副作用（常見 / 嚴重）
- **5-ASA**：頭痛、噁心、腹瀉；罕見**間質性腎炎**（須定期追腎功能）、胰臟炎
- **Sulfasalazine**：噁心、頭痛、**可逆性男性不孕（精子減少）**、皮疹、溶血（G6PD 缺乏）、抑制葉酸吸收
- **Azathioprine / 6-MP**：**骨髓抑制**（TPMT 缺乏者致命）、肝毒性、胰臟炎、感染、長期淋巴瘤/皮膚癌風險
- **類固醇**：血糖上升、骨鬆、感染、Cushing 樣、腎上腺抑制
- **anti-TNF（Infliximab）**：輸注反應、**結核 / B 肝再活化**、嚴重感染、脫髓鞘、狼瘡樣症候群、淋巴瘤

## 🚫 禁忌症
- **Azathioprine/6-MP**：TPMT 完全缺乏、活動性嚴重感染、懷孕相對禁（6-MP 具致畸性，Azathioprine 個別評估）
- **anti-TNF**：活動性結核、未治療潛伏 TB、活動性 B 肝、嚴重感染/敗血症、中重度心衰（NYHA III–IV）、去髓鞘疾病
- **Sulfasalazine**：sulfa 過敏、腸阻塞、G6PD 缺乏須謹慎
- **一般**：中毒性巨結腸禁用止瀉/抗膽鹼藥

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：
  - Azathioprine/6-MP → 用前 **TPMT**；用後 **CBC（前 8 週每 1–2 週）+ 肝功能**
  - anti-TNF → 用前 **IGRA/TST + 胸片 + HBsAg/anti-HBc**；用中警覺感染
  - 5-ASA → 定期腎功能（間質性腎炎）
- **交互作用**：
  - **Allopurinol + Azathioprine/6-MP → 致命**：allopurinol 抑制 xanthine oxidase，使 thiopurine 代謝物暴增 → 骨髓抑制。必須併用時 thiopurine 減量至 1/4 且密切監測
  - Sulfasalazine ↓葉酸吸收、可能增強 warfarin/sulfonylurea
- **孕哺**：5-ASA、Azathioprine、anti-TNF（尤其懷孕前中期）多可續用維持緩解（IBD 活動本身對胎兒更危險）；Methotrexate **絕對禁孕**（另藥，此處提醒）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名中重度克隆氏症病人準備開始 Infliximab 治療，開藥前最必要的檢查是？
- A. 心臟超音波
- B. 潛伏結核篩檢與 B 型肝炎血清學
- C. 骨密度
- D. 甲狀腺功能

> [!question]- 答案
> **B** — 所有 anti-TNF 生物製劑用前必須篩潛伏結核（IGRA/TST + 胸片）與 B 肝，因 TNF 抑制會使潛伏結核與 B 肝再活化（可致播散性 TB、猛爆性肝炎）。

**Q2.** 一名 UC 病人正服用 Azathioprine 維持緩解，因痛風開立 allopurinol 後出現嚴重白血球低下。最可能原因是？
- A. Allopurinol 直接抑制骨髓
- B. Allopurinol 抑制 xanthine oxidase 使 thiopurine 代謝物累積導致骨髓抑制
- C. 兩藥皆為腎毒性
- D. UC 惡化造成

> [!question]- 答案
> **B** — Allopurinol 抑制 xanthine oxidase（thiopurine 代謝關鍵酵素），使 6-MP/azathioprine 活性代謝物暴增 → 骨髓抑制。併用時 thiopurine 須減量至 1/4 並密切監測 CBC。
