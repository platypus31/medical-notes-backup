# Laxatives（瀉劑 / 便秘用藥總覽）

> [!abstract] 一句話：瀉劑依機轉分五類——**滲透性**（Lactulose、PEG、Mg citrate、Glycerin：拉水入腸腔）、**刺激性**（Bisacodyl、Senna：刺激腸神經叢促蠕動）、**膨脹性**（psyllium 纖維：吸水成團）、**軟便劑**（Docusate：界面活性軟化糞便）、**潤滑劑**（礦物油）。值班選擇看急緩與病人腎肝：**PEG 安全泛用；Lactulose 兼治肝性腦病（降血氨）；Mg 類避用於腎損；Bisacodyl/Senna 快但長期依賴**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **腸阻塞 / 疑似機械性梗阻 / 急腹症 → 所有瀉劑禁用**（尤其刺激性、大量滲透性），先影像排除；毒性巨結腸、腸穿孔風險
> 2. **Lactulose 用於肝性腦病 [[Hepatic Encephalopathy(肝性腦病)]]**：機轉是酸化腸腔促 NH₃→NH₄⁺ 排出降血氨，劑量調到**每天 2–3 次軟便**為目標（非愈瀉愈好，過度致脫水低血鉀反加重腦病）
> 3. **腎功能不良避用含鎂瀉劑**（Mg citrate、氧化鎂）→ 高血鎂（嗜睡、低血壓、心搏過緩、呼吸抑制）；含磷酸鈉灌腸劑亦致高磷/急性磷酸鹽腎病
> 4. **老年/長期臥床避長期刺激性瀉劑**：可致電解質流失（低血鉀）、瀉劑結腸（cathartic colon）、依賴
> 5. **Docusate 實證效果有限**：單用軟便劑對真正便秘常不足，別把它當唯一治療

## 💊 適應症（治療哪些疾病）
- **便秘 [[Constipation(便秘)]]**（急性/慢性、鴉片類引起、臥床相關）
- **肝性腦病 [[Hepatic Encephalopathy(肝性腦病)]]**（Lactulose 降血氨為首選）
- **腸道準備**（大腸鏡/手術前，PEG 大容量或 Mg citrate、⚠️依院規）
- **糞便嵌塞（fecal impaction）解除**（灌腸 + 口服）
- **鴉片類引起便秘（OIC）**（併用刺激性/滲透性，或專用 PAMORA 如 methylnaltrexone）

## 📏 劑量 / 用法（成人）
| 類別 | 代表藥 | 常用劑量 | 起效 |
| --- | --- | --- | --- |
| 滲透性(糖) | Lactulose | 15–30 mL PO QD–BID（腦病調至 2–3 次軟便） | 1–2 天 |
| 滲透性(PEG) | Polyethylene glycol 3350 | 17 g 溶水 QD | 1–3 天 |
| 滲透性(鹽) | Magnesium citrate/氧化鎂 | 依產品；腎損避用 | 0.5–3 小時（高劑量） |
| 滲透性(直腸) | Glycerin 甘油球 | 1 顆 PR | 15–60 分鐘 |
| 刺激性 | Bisacodyl | 5–10 mg PO 睡前 / 10 mg PR | 口服 6–12h、栓劑 15–60 分 |
| 刺激性 | Senna 番瀉葉 | 依產品睡前 | 6–12 小時 |
| 膨脹性 | Psyllium 纖維 | 依產品 + 足量水分 | 12–72 小時 |
| 軟便劑 | Docusate | 100 mg PO BID | 1–3 天（效果有限） |

- **腎功能**：避用含鎂/磷酸鈉製劑；PEG、lactulose、bisacodyl 相對安全
- **膨脹性纖維必須搭配足夠水分**，否則反致糞石阻塞；腸阻塞禁用
- 起效需求急（如腸準備、糞便嵌塞）用刺激性/直腸給藥，慢性維持用 PEG/滲透性

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：腹脹、腸絞痛、脹氣（Lactulose 尤其）、腹瀉、噁心
- **嚴重 / 特殊**：
  - **含鎂瀉劑 → 高血鎂**（腎損者嗜睡、低血壓、心搏過緩、呼吸抑制）
  - **磷酸鈉灌腸 → 高磷血症 / 急性磷酸鹽腎病 / 低血鈣**（老年、腎損高危）
  - **過度瀉下 → 脫水、低血鉀、低血鈉**（肝性腦病病人尤其危險）
  - **長期刺激性瀉劑 → 瀉劑結腸、電解質流失、依賴**
  - 膨脹性纖維水分不足 → 食道/腸道阻塞
  - 礦物油 → 吸入性脂質肺炎（臥床/吞嚥困難者避用）

## 🚫 禁忌症
- **腸阻塞、疑似機械性梗阻、腸穿孔、急腹症、毒性巨結腸**
- 未診斷的急性腹痛
- 含鎂/磷酸鈉製劑：嚴重腎功能不良禁用
- 礦物油：吞嚥困難、臥床、幼兒（吸入風險）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：長期使用查電解質（K、Mg、Na）、腎功能；肝性腦病病人監測意識與排便次數
- **交互作用**：
  - 瀉劑加速腸道通過 → 可能降低其他口服藥吸收
  - 膨脹性纖維、礦物油可與某些藥物螯合/影響吸收
  - Lactulose：與其他降血氨措施併用；抗生素（neomycin/rifaximin）為肝性腦病輔助
- **孕哺**：便秘常見於孕期，**PEG、膨脹性纖維、lactulose 相對安全為首選**；刺激性瀉劑短期審慎、避免長期；哺乳多數可用

### 記憶卡（Spaced Repetition 相容 · `Q::A`）

瀉劑依機轉分哪五大類，各舉一例？::滲透性(Lactulose/PEG/Mg citrate)、刺激性(Bisacodyl/Senna)、膨脹性(psyllium纖維)、軟便劑(Docusate)、潤滑劑(礦物油)

Lactulose 治療肝性腦病的機轉為何？::酸化腸腔使 NH₃ 轉為不易吸收的 NH₄⁺ 隨糞排出而降血氨，劑量調到每天 2–3 次軟便為目標

肝性腦病用 Lactulose 為何「不是愈瀉愈好」？::過度瀉下致脫水、低血鉀低血鈉，反而加重腦病

腎功能不良病人為何要避免含鎂瀉劑？::鎂經腎排除，腎損易累積致高血鎂（嗜睡、低血壓、心搏過緩、呼吸抑制）

所有瀉劑的絕對禁忌是什麼？::腸阻塞/機械性梗阻/腸穿孔/急腹症（風險：毒性巨結腸、穿孔），用前需排除

刺激性瀉劑（Bisacodyl/Senna）的機轉與起效時間？::刺激腸肌神經叢促進蠕動、減少水分吸收；口服約 6–12 小時、栓劑 15–60 分鐘

慢性便秘長期維持首選哪類瀉劑、為什麼？::PEG（polyethylene glycol）；安全、耐受佳、不易依賴，滲透性拉水軟化糞便

膨脹性纖維使用要注意什麼？::必須搭配足量水分，否則反致食道/腸道阻塞；腸阻塞禁用

Docusate（軟便劑）的臨床限制是什麼？::實證效果有限，單用對真正便秘常不足，不應作為唯一治療

孕期便秘首選哪些瀉劑？::PEG、膨脹性纖維、Lactulose 相對安全；刺激性瀉劑短期審慎避免長期

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名肝硬化併肝性腦病病人，最適合的瀉劑與治療目標是？
- A. Bisacodyl，目標大量水瀉
- B. Lactulose，目標每天 2–3 次軟便
- C. Magnesium citrate，快速排空
- D. 礦物油潤滑

> [!success]- 答案
> **B** — Lactulose 酸化腸腔降血氨為肝性腦病首選，目標調至每天 2–3 次軟便；過度瀉下致脫水低血鉀反加重腦病。

**Q2.** 慢性腎病第 4 期病人便秘，下列何者「應避免」？
- A. PEG 3350
- B. Lactulose
- C. Magnesium citrate
- D. Bisacodyl

> [!success]- 答案
> **C** — 含鎂瀉劑（Mg citrate、氧化鎂）在腎功能不良會累積致高血鎂；PEG、lactulose、bisacodyl 相對安全。

**Q3.** 一名腹脹、無排氣排便、腹部 X 光疑腸阻塞的病人便秘，最正確的處置？
- A. 立即給 senna 通便
- B. 給大量 PEG 灌腸
- C. 禁用瀉劑，先評估與影像確認排除梗阻
- D. 給 bisacodyl 栓劑

> [!success]- 答案
> **C** — 疑似腸阻塞為瀉劑禁忌，貿然給藥有毒性巨結腸/穿孔風險，應先影像確認並外科評估。
