# Ursodeoxycholic acid（熊去氧膽酸 / UDCA）

> [!abstract] 一句話：一種**親水性膽酸**，① 取代肝內累積的**疏水性毒性膽酸**（降低對肝細胞的傷害）② **降低膽固醇在膽汁中的飽和度**（促進膽固醇性膽結石溶解）③ 具細胞保護與輕度免疫調節作用 —— 是**原發性膽汁性膽管炎（PBC）的第一線用藥**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **膽道完全阻塞（complete biliary obstruction）禁用** —— 沒有膽汁流通的路徑，反而可能加重。
> 2. **溶石療法只對「小型、純膽固醇、且膽囊功能仍在」的結石有效** —— 鈣化石 / 色素石無效；療程數月至數年且停藥後易復發，臨床上多已被腹腔鏡膽囊切除取代，僅保留給不適開刀者。
> 3. **PBC 治療反應不佳（生化指標未達標）時** → 須考慮加上二線藥物（如 obeticholic acid）；不要只是無限期單用 UDCA 而不評估反應。
> 4. 整體耐受性佳，最常見副作用是**腹瀉**；不是止痛藥、不能緩解急性膽絞痛 / 膽囊炎。

## 💊 適應症（治療哪些疾病）
- **[[Primary Biliary Cholangitis(原發性膽汁性膽管炎)]]（PBC）第一線** —— 改善膽汁鬱積相關生化指標（ALP / 膽紅素）、延緩病程，是標準起手藥。
- **膽固醇性膽結石的溶石療法** —— 僅對**小型、純膽固醇、非鈣化**結石且膽囊功能正常者；用於**不適 / 拒絕手術者**。
- **妊娠期肝內膽汁淤積症（ICP, intrahepatic cholestasis of pregnancy）** —— 緩解搔癢、改善膽汁酸與生化（產科常用）。
- 其他膽汁鬱積性肝病的輔助（**依院規 / 需查證**）。

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| PBC | 依體重給藥，常見約 **13–15 mg/kg/day** 分次（**確切依院規 / 需查證**） |
| 膽固醇性膽石溶石 | 依體重給藥，療程數月至數年，定期影像追蹤（**依院規 / 需查證**） |
| 妊娠肝內膽汁淤積（ICP） | 依產科 protocol 給藥（**依院規 / 需查證**） |
- 通常分次服用、可隨餐；溶石療程需長期並定期超音波評估結石變化。
- 各適應症確切每日劑量與療程 **依院規 / 需查證**。

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：**腹瀉**（最常見）、稀便、輕微腸胃不適 —— 整體**耐受性佳**。
- 少見：噁心、腹痛、皮疹。
- **嚴重**：罕見；膽道完全阻塞時使用可能加重膽汁鬱積 —— 屬禁忌而非單純副作用。

## 🚫 禁忌症
- **膽道完全阻塞（complete biliary obstruction）**
- 對膽酸類 / 本藥過敏
- 鈣化結石、色素性結石、膽囊功能喪失（溶石無效，不建議）
- 急性膽囊炎 / 膽管炎、頻繁膽絞痛需手術者（UDCA 不能取代處置）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：PBC 追蹤 **ALP、膽紅素、肝功能** 評估反應（未達標考慮二線）；溶石療程定期**腹部超音波**看結石。
- **交互作用**：
  - **膽酸結合劑（cholestyramine、colestipol）** 與**含鋁制酸劑** → 結合 UDCA、降低吸收，應**間隔數小時**服用。
  - 可能影響 cyclosporine 等藥物吸收（**依院規 / 需查證**）。
- **孕哺**：常用於**妊娠期肝內膽汁淤積（ICP）**，產科視情況使用（**依院規 / 需查證**）。

## 🔗 相關
- 疾病：[[Primary Biliary Cholangitis(原發性膽汁性膽管炎)]]、膽固醇性膽結石、妊娠肝內膽汁淤積（ICP）
- 二線 / 延伸：obeticholic acid（PBC UDCA 反應不佳時的二線，可提及）
- 對照：膽酸結合劑（cholestyramine，會降低 UDCA 吸收，須錯開服用）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位中年女性因慢性疲倦、搔癢與 ALP 上升被診斷為原發性膽汁性膽管炎（PBC）。下列何者為第一線用藥？
- A. Prednisolone
- B. Ursodeoxycholic acid
- C. Cholestyramine
- D. Rifampin

> [!success]- 答案
> **B** — **Ursodeoxycholic acid（UDCA）** 是 PBC 的第一線用藥，可改善膽汁鬱積生化指標與延緩病程。若反應不佳再考慮二線 obeticholic acid。Cholestyramine 只緩解搔癢、不改變病程。

**Q2.** 關於 UDCA 溶解膽結石的療法，下列敘述何者正確？
- A. 對所有膽結石都有效
- B. 只對小型、純膽固醇、非鈣化結石且膽囊功能正常者有效
- C. 可快速溶解鈣化石
- D. 是急性膽囊炎的首選治療

> [!success]- 答案
> **B** — UDCA 溶石**只對小型、純膽固醇、非鈣化、膽囊功能正常**的結石有效，療程長且停藥易復發；鈣化 / 色素石無效，急性膽囊炎需手術 / 處置而非 UDCA。

**Q3.** 下列何者是 UDCA 的禁忌症？
- A. 妊娠期肝內膽汁淤積
- B. 原發性膽汁性膽管炎
- C. 膽道完全阻塞
- D. 小型膽固醇結石

> [!success]- 答案
> **C** — **膽道完全阻塞**時膽汁無流通路徑，使用 UDCA 屬禁忌。A、B、D 反而是其適應症範疇。
