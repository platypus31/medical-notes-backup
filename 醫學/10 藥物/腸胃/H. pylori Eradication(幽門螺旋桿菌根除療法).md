# H. pylori Eradication（幽門螺旋桿菌根除療法）

> [!abstract] 一句話：H. pylori 根除是**多藥組合療程**，核心是 **PPI 強力抑酸（提高抗生素在胃內穩定性與濃度）＋ 兩種以上抗生素 ± 鉍鹽**。因 **clarithromycin 抗藥性上升**，現行趨勢是**含鉍四合一（bismuth quadruple）**或**併合療法**優於傳統三合一，療程延長至 **14 天**提高根除率。適應症：消化性潰瘍、MALT 淋巴瘤、ITP、長期 NSAID/抗血小板前、胃癌家族史等。**療效關鍵：足療程、足抑酸、依當地抗藥性選方、事後驗證除菌**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **Clarithromycin 抗藥性決定成敗**：當地 clarithromycin 抗藥性 >15% 或病人近期用過 macrolide → **避免含 clarithromycin 三合一，改含鉍四合一**
> 2. **青黴素過敏 → 不能用 amoxicillin**：改含鉍四合一（metronidazole + tetracycline）或含 clarithromycin + metronidazole 方案
> 3. **足療程足抑酸是根除率關鍵**：療程建議 **14 天**、PPI 用足（部分用高劑量/雙倍或 vonoprazan 更強抑酸），順從性差是失敗主因
> 4. **抗生素個別紅旗**：clarithromycin 與 metronidazole 皆延長 **QT**；metronidazole **禁酒（disulfiram 反應）**；tetracycline 孕婦/幼兒禁（牙齒骨骼）；含鉍黑便黑舌
> 5. **除菌後需驗證療效**：停藥後（PPI 停 ≥2 週、抗生素停 ≥4 週）以 **尿素呼氣試驗或糞便抗原**確認除菌成功，勿只憑症狀

## 💊 適應症（哪些人該驗與治）
- **消化性潰瘍 [[Peptic Ulcer Disease(消化性潰瘍)]]**（胃/十二指腸潰瘍，含出血過）
- **胃 MALT 淋巴瘤**（早期常隨除菌緩解）
- **早期胃癌切除後、胃癌一等親家族史**
- **長期 NSAID / 低劑量阿斯匹靈使用前**（尤其有潰瘍史）
- **不明原因缺鐵性貧血、免疫性血小板低下（ITP）**
- **功能性消化不良（test-and-treat 策略）**、確診感染者

## 📏 療法方案（成人，療程建議 14 天）
| 方案 | 組成 | 適用情境 |
| --- | --- | --- |
| **含鉍四合一（首選趨勢）** | PPI + Bismuth + Metronidazole + Tetracycline | 高 clarithromycin 抗藥性、青黴素過敏、曾失敗 |
| **傳統三合一** | PPI + Amoxicillin + Clarithromycin | 僅限低 clarithromycin 抗藥性地區、未用過 macrolide |
| **併合療法(concomitant)** | PPI + Amoxicillin + Clarithromycin + Metronidazole | 非鉍四合一替代 |
| **序列療法(sequential)** | 前5天 PPI+Amoxicillin，後5天 PPI+Clarithromycin+Metronidazole | 現較少用 |
| **含鉍+levofloxacin 或 vonoprazan 方案** | 救援/新式強抑酸 | 二線/救援、依抗藥性 |

- **PPI 一般 BID、足劑量**；vonoprazan（P-CAB）抑酸更強、對含 clarithromycin 方案根除率佳（部分指引採用）
- **常用抗生素劑量（參考，⚠️依院規/仿單）**：Amoxicillin 1 g BID、Clarithromycin 500 mg BID、Metronidazole 500 mg TID–QID、Tetracycline 500 mg QID、Bismuth 依產品 QID
- **療程 14 天**根除率優於 7/10 天（尤其抗藥性上升區）
- **二線 / 救援**：避開已用過的抗生素；可考慮含 levofloxacin 方案或做**抗藥性/培養**導向治療

## ⚠️ 副作用（療程相關）
- **常見**：味覺改變（金屬味，clarithromycin/metronidazole）、噁心、腹瀉、腹脹、黑便黑舌（鉍）
- **嚴重 / 特殊**：
  - **Metronidazole**：**禁酒（disulfiram 反應：潮紅、嘔吐、心悸）**、周邊神經病變（長期）、QT
  - **Clarithromycin**：QT 延長、強 **CYP3A4 抑制**（大量交互作用）、味覺障礙
  - **Amoxicillin**：過敏/皮疹（青黴素過敏禁用）、腹瀉
  - **Tetracycline**：光敏感、孕婦/8 歲以下禁（牙齒染色骨骼）、食道潰瘍（需足水站立服）
  - **抗生素相關腹瀉 / C. difficile**
  - 順從性差 → 除菌失敗 + 誘導抗藥性

## 🚫 禁忌症 / 注意
- 對方案內任一藥物過敏（青黴素過敏 → 不用 amoxicillin）
- **孕婦**：tetracycline、部分方案禁；孕期非急迫多延後治療
- 嚴重肝腎功能不良需調整抗生素劑量
- QT 延長 / 併多種 QT 藥物者評估
- Metronidazole 治療期間及停藥後數日禁酒

## 🔍 監測 / 交互作用 / 追蹤
- **根除驗證（必做）**：療程結束後 **PPI 停 ≥2 週、抗生素停 ≥4 週**，以**尿素呼氣試驗（UBT）或糞便抗原**確認；血清抗體不能用於判斷除菌成功
- **交互作用**：
  - **Clarithromycin（強 CYP3A4 抑制）**：升高 statin（橫紋肌溶解）、warfarin、colchicine、部分 CCB、benzodiazepine 等 → 審視合併用藥
  - **Metronidazole**：增強 warfarin、禁酒（disulfiram）
  - **Bismuth/Tetracycline 與制酸劑、鈣鐵、乳製品螯合** → 影響吸收
  - PPI 本身交互作用（見 PPI 條目，clopidogrel）
- **失敗處理**：確認順從性 → 換方案（避開用過抗生素）→ 必要時抗藥性導向

### 自我測驗（選擇題，答案摺疊）

**Q1.** 在 clarithromycin 抗藥性高（>15%）的地區，H. pylori 一線治療首選是？
- A. PPI + Amoxicillin + Clarithromycin 三合一
- B. 含鉍四合一（PPI + Bismuth + Metronidazole + Tetracycline）
- C. 單用 PPI
- D. 單用 amoxicillin

> [!success]- 答案
> **B** — clarithromycin 抗藥性高時傳統三合一根除率下降，應選含鉍四合一（或不含 clarithromycin 的方案），療程建議 14 天。

**Q2.** 病人完成 H. pylori 療程後，正確的除菌驗證方式與時機是？
- A. 立即抽血驗抗體
- B. 療程結束隔天做呼氣試驗
- C. PPI 停 ≥2 週、抗生素停 ≥4 週後做尿素呼氣試驗或糞便抗原
- D. 只憑症狀改善判定

> [!success]- 答案
> **C** — 需在 PPI 停 ≥2 週、抗生素停 ≥4 週後以尿素呼氣試驗或糞便抗原檢測；抗體無法判斷除菌成功。

**Q3.** 關於 H. pylori 療程中的抗生素注意事項，何者正確？
- A. Metronidazole 可與酒精併用
- B. Clarithromycin 為強 CYP3A4 抑制劑，需審視合併用藥
- C. Tetracycline 對孕婦安全
- D. Amoxicillin 可用於青黴素過敏者

> [!success]- 答案
> **B** — Clarithromycin 強抑制 CYP3A4（升高 statin、warfarin、colchicine 等）；metronidazole 禁酒、tetracycline 孕婦/幼兒禁、青黴素過敏者不可用 amoxicillin。
