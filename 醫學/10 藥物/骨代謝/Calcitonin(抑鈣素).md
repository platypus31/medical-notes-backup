# Calcitonin（抑鈣素 / 降鈣素）

> [!abstract] 一句話：**Calcitonin** 是甲狀腺 C 細胞（濾泡旁細胞）分泌的荷爾蒙，作用與副甲狀腺素（PTH）相反：**抑制蝕骨細胞（osteoclast）→ 降低骨吸收、降血鈣**，並促腎排鈣。臨床用 **鮭魚抑鈣素（效價高）** 於**急性高血鈣（快速起效但短暫）、骨質疏鬆（次選）、Paget 病、急性椎體壓迫骨折止痛**。特色：**起效快但快速去敏（tachyphylaxis）**、鼻噴劑會鼻出血。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **快速起效但快速去敏（tachyphylaxis）**：治療高血鈣時**約 48 小時後降鈣效果減弱**，只能當「爭取時間」的橋接（bridge），須同時給水化 + 雙磷酸鹽等持久藥
> 2. **只適合暫時控制高血鈣，非長期單靠**：高血鈣主力仍是生理食鹽水水化 + IV bisphosphonate；calcitonin 用於**快速但暫時**降鈣
> 3. **鼻噴劑（骨鬆用）→ 鼻黏膜刺激、鼻出血、鼻炎**：使用前教導正確鼻噴技巧、兩鼻孔交替
> 4. **長期使用與惡性腫瘤風險訊號**：長期鼻噴 calcitonin 曾有癌症風險疑慮，故骨鬆已非首選、限短期/次選；優先選雙磷酸鹽/denosumab
> 5. **過敏（鮭魚蛋白來源）**：可能過敏反應，高風險者可先做皮膚測試；也常見注射後噁心、臉潮紅

## 💊 適應症（治療哪些疾病）
- **急性高血鈣 [[Hypercalcemia(高血鈣)]]**（快速起效橋接，配水化 + 雙磷酸鹽）
- **骨質疏鬆症 [[Osteoporosis(骨質疏鬆)]]**（次選，尤其**急性椎體壓迫骨折疼痛**有止痛作用）
- **Paget 骨病（變形性骨炎）**

## 📏 劑量 / 用法（成人）
| 用途 | 劑量（⚠️依院規） |
| --- | --- |
| 急性高血鈣 | **鮭魚抑鈣素 4 IU/kg SC/IM q12h**，可增至 8 IU/kg q6–12h（效果約 48h 後衰減） |
| 骨質疏鬆 / 椎體骨折止痛 | **鼻噴劑 200 IU 每日一次**（兩鼻孔交替）或 SC |
| Paget 病 | SC/IM，依反應調整 |
- **起效快（數小時內降鈣）** 但**維持短**，高血鈣時務必同步啟動持久性治療
- 鼻噴劑使用前後注意鼻黏膜狀況

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：噁心、嘔吐、臉潮紅（注射尤其）、注射部位反應
- **鼻噴劑**：**鼻炎、鼻黏膜刺激、鼻出血、鼻乾/結痂**
- **嚴重 / 需注意**：
  - **快速耐藥性（tachyphylaxis）**——降鈣效果約 48h 後減弱
  - 過敏反應（鮭魚蛋白）
  - 低血鈣（過度反應）
  - 長期使用的惡性腫瘤風險訊號（限制骨鬆長期使用）

## 🚫 禁忌症
- 對 calcitonin 或鮭魚蛋白過敏
- 低血鈣
- （相對）長期骨鬆治療優先其他藥物，避免不必要長期暴露

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：**血鈣（高血鈣治療時密切追蹤）**、鼻噴劑者定期鼻黏膜檢查；過敏徵象
- **交互作用**：
  - 與**其他降鈣藥（雙磷酸鹽、水化）併用是刻意的協同**（快 + 持久）
  - 與鋰鹽等交互作用少見
- **孕哺**：孕期資料有限，一般僅在明確需要時使用；calcitonin 為大分子胜肽，哺乳影響一般較小但仍謹慎

## 🔗 相關
- 疾病：[[Hypercalcemia(高血鈣)]]、[[Osteoporosis(骨質疏鬆)]]、[[Paget's Disease(佩吉特骨病)]]
- 對照 / 協同：[[Bisphosphonates(雙磷酸鹽總覽)]]（持久降鈣，與 calcitonin 快速起效互補）、[[Calcium-VitaminD(鈣與維生素D總覽)]]
- 生理對手：副甲狀腺素 PTH（作用相反，升血鈣）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 惡性腫瘤高血鈣（血鈣 14 mg/dL）病人，關於 calcitonin 的使用敘述何者正確？
- A. 可長期單獨使用維持正常血鈣
- B. 起效快但約 48 小時去敏，應搭配水化與雙磷酸鹽
- C. 起效慢需數天
- D. 會升高血鈣

> [!success]- 答案
> **B** — Calcitonin 快速降鈣但很快發生 tachyphylaxis，僅作橋接爭取時間，須同時給生理食鹽水水化與持久的 IV 雙磷酸鹽。

**Q2.** 骨質疏鬆使用 calcitonin 鼻噴劑最常見的局部副作用是？
- A. 味覺喪失
- B. 鼻炎與鼻出血
- C. 牙齦增生
- D. 耳鳴

> [!success]- 答案
> **B** — 鼻噴劑 calcitonin 常見鼻黏膜刺激、鼻炎、鼻出血，使用時兩鼻孔交替並注意鼻黏膜狀況。

**Q3.** 下列何者最能說明 calcitonin 與副甲狀腺素（PTH）的關係？
- A. 兩者都升高血鈣
- B. Calcitonin 降血鈣，PTH 升血鈣，作用相反
- C. 兩者都降血鈣
- D. Calcitonin 由副甲狀腺分泌

> [!success]- 答案
> **B** — Calcitonin（甲狀腺 C 細胞）抑制骨吸收降血鈣，與 PTH（升血鈣）作用相反，是鈣恆定的一對拮抗荷爾蒙。
