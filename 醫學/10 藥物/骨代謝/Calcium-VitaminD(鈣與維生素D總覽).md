# Calcium & Vitamin D（鈣與維生素 D 總覽）

> [!abstract] 一句話：**鈣補充劑（Calcium carbonate / Calcium citrate）** 提供元素鈣，**維生素 D（Cholecalciferol D3 / Ergocalciferol D2 → 經肝腎活化為 Calcitriol）** 促進腸道鈣吸收與骨骼礦化。兩者是**骨質疏鬆治療的基礎（所有抗骨鬆藥的地基）、低血鈣與維生素 D 缺乏的補充、慢性腎病礦物質骨病變**用藥。過量最怕**高血鈣、腎結石**；**CKD 病人腎不能活化維生素 D，須用活性型 calcitriol**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **高血鈣（hypercalcaemia）**：補過頭或與其他升鈣狀況（腫瘤、副甲亢、maalox milk-alkali）疊加 → 噁心、便祕、多尿、意識改變、心律不整；補鈣前先確認基礎血鈣
> 2. **腎結石**：長期高劑量鈣（尤其與維生素 D 併）增加尿鈣 → 結石；有結石病史者謹慎、分次小劑量、多喝水
> 3. **CKD 用「活性型」calcitriol（骨化三醇），不是普通 D3**：腎衰竭時腎臟 1α-hydroxylase 失能無法把 25-OH-D 活化成 1,25-(OH)₂-D，補普通 D3 無效 → 需 calcitriol；但 calcitriol 更易致高血鈣需嚴密監測
> 4. **Calcium carbonate 需胃酸溶解 → 隨餐服；Calcium citrate 不需胃酸 → 可空腹、適合制酸劑/PPI/胃切除者**：選錯型別吸收差
> 5. **鈣干擾多種藥物吸收**：與**雙磷酸鹽、levothyroxine、四環黴素/氟喹諾酮抗生素、鐵劑**同服會螯合降低吸收 → 必須錯開時間

## 💊 適應症（治療哪些疾病）
- **骨質疏鬆症 [[Osteoporosis(骨質疏鬆)]] 基礎治療**（與雙磷酸鹽/denosumab 併用的地基）
- **低血鈣 [[Hypocalcemia(低血鈣)]] / 維生素 D 缺乏**補充
- **慢性腎病礦物質骨病變（CKD-MBD）**：活性型 calcitriol、碳酸鈣（兼磷結合劑）
- 副甲狀腺功能低下、佝僂病/骨軟化症、孕哺與老年營養補充

## 📏 劑量 / 用法（成人）
| 藥 | 特性 / 用法 |
| --- | --- |
| **Calcium carbonate（碳酸鈣）** | 元素鈣含量高（40%）、便宜；**需胃酸 → 隨餐服**；分次每次 ≤500–600 mg 元素鈣吸收較佳；兼可做磷結合劑 |
| **Calcium citrate（檸檬酸鈣）** | 元素鈣 21%；**不需胃酸 → 可空腹、適合 PPI/制酸劑/胃切除/胃酸不足者**、較不易結石 |
| **Cholecalciferol（D3）/ Ergocalciferol（D2）** | 一般維生素 D 補充（缺乏治療），需經**肝（25-OH）→ 腎（1,25-(OH)₂ 活化）** |
| **Calcitriol（1,25-(OH)₂-D，活性型）** | **已活化**，用於 **CKD / 副甲低下 / 腎臟無法活化者**；作用強、半衰期短、**易高血鈣需嚴密監測** |
- 成人常見：元素鈣總量約 1000–1200 mg/day（含飲食）、維生素 D 依缺乏程度補充
- **分次小劑量**吸收較佳並降低結石風險

## ⚠️ 副作用（常見 / 嚴重）
- **鈣補充**：便祕、脹氣、噯氣（碳酸鈣尤其）
- **嚴重 / 過量**：
  - **高血鈣**（噁心、便祕、多尿、脫水、意識改變、心律不整）
  - **腎結石 / 腎鈣化**（長期高鈣尿）
  - **Milk-alkali syndrome**（大量碳酸鈣 + 鹼 → 高血鈣、代謝性鹼中毒、腎損）
  - 維生素 D 過量 → 高血鈣、高尿鈣
- Calcitriol 因作用強**最易致高血鈣**

## 🚫 禁忌症
- **高血鈣、高尿鈣、維生素 D 中毒**
- 嚴重腎結石病史（相對）
- 對成分過敏
- （併用時）已有其他升鈣因素（惡性高血鈣、原發性副甲亢）未控制

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：**血鈣、血磷、腎功能、維生素 D 濃度（25-OH-D）**；CKD 用 calcitriol 者更需密切追蹤血鈣防高血鈣
- **交互作用（吸收干擾——臨床超常考）**：
  - **鈣 + 雙磷酸鹽 / levothyroxine / 四環黴素 / 氟喹諾酮 / 鐵劑** → 螯合降低吸收，需**間隔數小時**
  - **Thiazide 利尿劑** + 鈣/維生素 D → 減少尿鈣排出、增加高血鈣風險
  - **PPI / 制酸劑** → 降低碳酸鈣吸收（改用檸檬酸鈣）
  - **類固醇** → 減少腸鈣吸收（長期類固醇者需補鈣+D）
- **孕哺**：鈣與維生素 D 為孕哺常規補充，依需求量給予

### 記憶卡（Spaced Repetition 相容 · `Q::A`）

維生素 D 的活化路徑為何？::皮膚/飲食來的 D3/D2 → 肝臟 25-羥化成 25-OH-D → 腎臟 1α-羥化成 1,25-(OH)₂-D（calcitriol，活性型），促腸道鈣吸收

為什麼 CKD 病人要用 calcitriol 而非普通 D3？::腎衰竭時腎臟 1α-hydroxylase 失能無法把 25-OH-D 活化成活性型，補普通 D3 無效，需直接給已活化的 calcitriol

Calcium carbonate 與 Calcium citrate 的服用時機差異？::碳酸鈣需胃酸溶解 → 隨餐服；檸檬酸鈣不需胃酸 → 可空腹，適合 PPI/制酸劑/胃切除/胃酸不足者

鈣補充過量的兩大危害是什麼？::高血鈣（噁心/便祕/多尿/意識改變/心律不整）與腎結石

哪些藥物與鈣同服會因螯合降低吸收、須錯開？::雙磷酸鹽、levothyroxine、四環黴素、氟喹諾酮抗生素、鐵劑

Milk-alkali syndrome 是什麼？::大量攝取鈣（碳酸鈣）+ 可吸收鹼 → 高血鈣、代謝性鹼中毒、急性腎損傷三聯

為什麼鈣與維生素 D 是所有抗骨鬆藥的「地基」？::雙磷酸鹽/denosumab 等抑制骨吸收會降血鈣，若無足夠鈣與維生素 D 會低血鈣且療效打折，故須先補足

Thiazide 利尿劑與鈣/維生素 D 併用要注意什麼？::Thiazide 減少尿鈣排出，會增加高血鈣風險

Calcitriol 相較普通維生素 D 最需警惕的副作用？::作用強、半衰期短，最容易造成高血鈣，需密切監測血鈣

碳酸鈣單次吸收較佳的元素鈣量約為多少？::每次 ≤ 500–600 mg 元素鈣，分次服用吸收較佳並降低結石風險

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名長期服用 PPI 的骨鬆病人需補鈣，哪種鈣鹽最適合、為什麼？
- A. 碳酸鈣，因便宜
- B. 檸檬酸鈣，因不需胃酸即可吸收
- C. 兩者皆需隨餐
- D. 都不能用

> [!success]- 答案
> **B** — PPI 降低胃酸，碳酸鈣需胃酸溶解吸收會變差；檸檬酸鈣不需胃酸，可空腹服用，適合 PPI/制酸劑/胃切除者。

**Q2.** 末期腎病（CKD 5）病人維生素 D 缺乏，補充普通 cholecalciferol（D3）效果不佳，主因是？
- A. 腸道吸收障礙
- B. 腎臟 1α-hydroxylase 失能無法把 25-OH-D 活化成活性型
- C. 肝臟無法 25 羥化
- D. D3 劑量太高

> [!success]- 答案
> **B** — CKD 腎臟無法完成 1α 羥化活化，需直接給活性型 calcitriol；但 calcitriol 易致高血鈣，須監測血鈣血磷。

**Q3.** 開立鈣片給正在服用 Alendronate（雙磷酸鹽）與 levothyroxine 的病人，最重要的衛教是？
- A. 三者一起吞下
- B. 鈣與雙磷酸鹽、levothyroxine 需間隔數小時服用避免螯合降低吸收
- C. 鈣可增強雙磷酸鹽吸收
- D. 不需注意

> [!success]- 答案
> **B** — 鈣會螯合雙磷酸鹽與 levothyroxine 大幅降低其吸收，必須錯開服藥時間（間隔數小時）。
