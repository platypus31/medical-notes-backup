# 標靶藥物（Targeted Therapy — 概論）

> [!abstract] 一句話：不像傳統化療「無差別殺快速分裂細胞」，標靶藥物針對癌細胞特定的**分子標的**（如驅動突變的激酶、表面抗原、血管新生訊號），選擇性阻斷腫瘤生長路徑，正常細胞影響相對較小。兩大類：**小分子抑制劑（-nib，如 tyrosine kinase inhibitor TKI）** 與 **單株抗體（-mab）**。命名有規律：**-nib = 小分子激酶抑制劑（口服）**，**-mab = 單株抗體（注射）**。

> [!danger] 🚨 值班紅旗
> 1. **輸注反應（infusion reaction）** — 單株抗體（-mab）首次輸注常見發燒、寒顫、蕁麻疹、低血壓，嚴重可過敏性休克 → 需前置給藥（抗組織胺 + acetaminophen ± 類固醇）與慢速輸注
> 2. **B 型肝炎再活化** — **Rituximab（anti-CD20）** 等會清除 B 細胞/免疫抑制的標靶藥，用前必篩 HBsAg / anti-HBc，帶原者需預防性抗病毒
> 3. **腫瘤溶解症候群（TLS）** — 高腫瘤負荷的血液腫瘤用 rituximab 等要防 TLS
> 4. **各藥獨特毒性** — trastuzumab 心臟毒性（用前查 LVEF）、bevacizumab 出血/高血壓/傷口不癒/腸穿孔、TKI 有各自的皮疹/腹瀉/QT/肝毒性

## 💊 適應症與代表藥（概念性對照）
| 藥物（代表） | 類別 | 標的 / 機轉 | 主要適應症 |
| --- | --- | --- | --- |
| **Imatinib（基利克）** | 小分子 TKI | 抑制 **BCR-ABL** tyrosine kinase | **慢性骨髓性白血病（CML）**、GIST（c-KIT） |
| **Gefitinib（Iressa）/ Erlotinib** | 小分子 TKI | 抑制 **EGFR** tyrosine kinase | EGFR 突變 **非小細胞肺癌** |
| **Rituximab（Rituxan / Mabthera）** | 單株抗體 | anti-**CD20**，清除 B 細胞 | B 細胞淋巴瘤、CLL、類風濕、部分自體免疫 |
| **Trastuzumab（Herceptin）** | 單株抗體 | anti-**HER2** | HER2 陽性乳癌、胃癌 |
| **Bevacizumab（Avastin）** | 單株抗體 | anti-**VEGF**，抑制血管新生 | 大腸癌、肺癌、腎癌等 |
| **Bortezomib（Velcade）** | 小分子 | **蛋白酶體（proteasome）抑制** | 多發性骨髓瘤 |

> 命名規則：**-nib** 結尾多為口服小分子激酶抑制劑（TKI）；**-mab** 結尾為單株抗體（monoclonal antibody，注射）。

## 📏 劑量 / 用法
- **依各藥 protocol 與適應症**（口服 TKI 多為固定每日劑量；單株抗體多按體重/BSA 靜脈輸注，如 rituximab 375 mg/m²）
- 標靶藥常與傳統化療組合（如 **R-CHOP** = Rituximab + 傳統化療治療淋巴瘤）
- 此為概論卡，個別藥物實際劑量請依 protocol / 各藥仿單

## ⚠️ 副作用（依類別）
- **單株抗體（-mab）共通**：輸注反應、感染（免疫抑制）、B 肝再活化（rituximab）
- **TKI（-nib）共通**：皮疹、腹瀉、肝功能異常、QT 延長、疲倦、水腫（imatinib）
- **各藥獨特**：
  - Trastuzumab → **心臟毒性 / 心衰竭**（用前後追 LVEF）
  - Bevacizumab → **高血壓、出血、蛋白尿、傷口不癒、腸穿孔、血栓**
  - Imatinib → 水腫（尤其眶周）、肌肉痙攣、血球低下
  - Bortezomib → 周邊神經病變、血小板低下

## 🚫 禁忌症
- 對特定藥物過敏
- Trastuzumab：明顯心衰竭 / 低 LVEF
- Bevacizumab：近期大手術/活動性出血/未癒合傷口
- Rituximab：活動性嚴重感染；帶原 B 肝未預防

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：依藥而異 — LVEF（trastuzumab）、血壓/尿蛋白（bevacizumab）、CBC/肝功能（TKI）、輸注反應、HBV 狀態（rituximab 前）
- **交互作用**：多數 TKI 經 **CYP3A4** 代謝 → 與 azole、rifampin、葡萄柚汁交互作用
- **孕哺**：多數標靶藥致畸胎 / 資料有限，孕期避免，治療期避孕

## 🔗 相關
- 對照：傳統細胞毒性化療（無差別殺快速分裂細胞）
- 併用：[[Steroid(類固醇)]]（化療組合、輸注前置給藥、止吐）
- 免疫治療延伸：免疫檢查點抑制劑（-mab 如 pembrolizumab，機轉不同，屬免疫治療）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位 B 細胞淋巴瘤病人即將接受含 rituximab 的化療，治療前最重要的篩檢是？
- A. 心臟超音波
- B. B 型肝炎血清標記（HBsAg / anti-HBc）
- C. 甲狀腺功能
- D. 骨密度

> [!success]- 答案
> **B** — Rituximab（anti-CD20）清除 B 細胞、免疫抑制，可致 **B 肝再活化**（可致命猛爆性肝炎）。治療前必篩 HBV，帶原者需預防性抗病毒。

**Q2.** 使用 trastuzumab（anti-HER2）治療乳癌，需特別監測下列何者？
- A. 左心室射出分率（LVEF），因心臟毒性
- B. 血糖
- C. 骨密度
- D. 甲狀腺

> [!success]- 答案
> **A** — Trastuzumab 的重要毒性是**心臟毒性/心衰竭**，用藥前後需追蹤 **LVEF**。

**Q3.** 藥名以「-nib」結尾（如 imatinib、gefitinib）通常屬於哪一類標靶藥？
- A. 單株抗體
- B. 小分子酪胺酸激酶抑制劑（TKI）
- C. 化療烷化劑
- D. 荷爾蒙藥

> [!success]- 答案
> **B** — **-nib** 結尾為**小分子激酶抑制劑（TKI）**，多為口服；**-mab** 才是單株抗體。
