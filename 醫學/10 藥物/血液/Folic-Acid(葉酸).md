# Folic Acid（葉酸 / Vitamin B9）

> [!abstract] 一句話：**維生素 B9**，作為一碳單位轉移的輔酶，是 **DNA/嘌呤/胸腺嘧啶合成**與紅血球成熟必需。缺乏 → **巨球性（大球性）貧血**。用於葉酸缺乏、**孕前與孕期預防胎兒神經管缺損**、溶血/懷孕等高需求狀態、以及 methotrexate 等抗葉酸藥的副作用預防。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **未排除 B12 缺乏前單補葉酸 = 陷阱**：葉酸可**矯正巨球性貧血的血液學表現，卻無法阻止 B12 缺乏的神經病變（亞急性合併退化）持續惡化**，甚至掩蓋診斷 → **貧血是巨球性時務必同時查 B12，缺 B12 者先補/併補 B12**
> 2. **孕前補充要「提早」**：神經管在懷孕極早期閉合，須**受孕前 1 個月即開始**才有預防神經管缺損效果；曾生育神經管缺損兒者需高劑量
> 3. Methotrexate 相關：一般類風濕/乾癬用 **folic acid** 減副作用；**MTX 過量或化療解毒須用 folinic acid（leucovorin，亞葉酸）**——兩者不可混淆，folic acid 無法在 DHFR 被抑制時繞過
> 4. 大劑量葉酸長期使用可能**掩蓋 B12 缺乏**的血液指標，讓神經損害無聲進展
> 5. 抗癲癇藥（phenytoin 等）與葉酸互相干擾濃度，需留意癲癇控制

## 💊 適應症（治療哪些疾病）
- **葉酸缺乏性巨球性貧血 [[Megaloblastic Anemia(巨球性貧血)]]**
- **孕前 / 孕期預防神經管缺損 [[Neural Tube Defect(神經管缺損)]]**（及一般孕期補充）
- 高需求狀態：慢性溶血性貧血（如地中海貧血、鐮刀型）、透析、酒精性、營養不良
- **Methotrexate 治療期間**減少口腔炎/腸胃/血液副作用（風濕/皮膚科用途）

## 📏 劑量 / 用法（成人）
| 情境 | 劑量 |
| --- | --- |
| 葉酸缺乏治療 | **1–5 mg PO QD**，缺乏矯正後視需要維持 |
| 一般孕期預防 | **0.4–0.8 mg/day**，受孕前 1 個月起至孕期 |
| 曾生育神經管缺損兒 | **4–5 mg/day**（高風險高劑量） |
| 慢性溶血 | 1 mg/day |
| MTX 副作用預防（風濕） | 每週 folic acid（依方案，非與 MTX 同日） |
- 口服吸收良好；嚴重吸收不良可注射
- **MTX 過量解毒用 folinic acid（leucovorin）非本藥**

## ⚠️ 副作用（常見 / 嚴重）
- 一般**耐受良好**，水溶性維生素過量多由尿排出
- 罕見：過敏反應、腸胃不適、大劑量偶見睡眠/情緒改變
- **主要「危害」在於掩蓋 B12 缺乏**導致神經病變被延誤（非葉酸本身毒性）

## 🚫 禁忌症
- 對葉酸製劑過敏
- **未經評估的巨球性貧血不可貿然只補葉酸**（需先確認/排除 B12 缺乏）——屬臨床使用禁忌情境而非藥物本身禁忌

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：CBC / MCV、網狀紅血球反應、血清葉酸與 **B12（必查）**；治療反應
- **交互作用**：
  - **Methotrexate / trimethoprim / pyrimethamine（抗葉酸藥）**：作用相互拮抗，抗葉酸藥毒性需用 folinic acid 解救
  - **Phenytoin、其他抗癲癇藥**：互相降低血中濃度，影響癲癇控制
  - Sulfasalazine、酒精降低葉酸吸收
- **孕哺**：**孕期建議且重要**（預防神經管缺損）；哺乳安全

## 🔗 相關
- 疾病：[[Megaloblastic Anemia(巨球性貧血)]]、[[Neural Tube Defect(神經管缺損)]]、慢性溶血性貧血
- 對照 / 必查搭配：[[Vitamin-B12(維生素B12-Cobalamin)]]（巨球性貧血必同時評估，先給 B12 再給葉酸）
- 相關：Folinic acid（leucovorin，MTX 解毒）、[[Iron Supplements(鐵劑總覽)]]（小球性貧血對照）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名巨球性貧血病人，醫師只開葉酸未查 B12。三個月後貧血改善但出現進行性下肢麻木、步態不穩。最可能原因？
- A. 葉酸過量中毒
- B. 未察覺的 B12 缺乏神經病變被葉酸掩蓋而持續惡化
- C. 缺鐵
- D. 藥物過敏

> [!success]- 答案
> **B** — 葉酸矯正血液學指標卻無法阻止 B12 缺乏引起的亞急性合併退化，反而掩蓋診斷；故巨球性貧血務必同時查 B12。

**Q2.** 關於孕期葉酸補充，何者正確？
- A. 懷孕確認後才開始即可
- B. 應於受孕前 1 個月即開始以預防神經管缺損
- C. 對神經管缺損無預防效果
- D. 只有素食者需要

> [!success]- 答案
> **B** — 神經管在懷孕極早期閉合，須受孕前 1 個月開始補充才有預防效果；曾生育神經管缺損兒者用高劑量 4–5 mg。
