# Vitamin B12（維生素 B12 / Cobalamin）

> [!abstract] 一句話：**鈷胺素（cobalamin）**，DNA 合成與神經髓鞘維持所需。缺乏 → **巨球性貧血 + 神經病變（亞急性合併退化 SCD）**。吸收需**胃壁內因子（intrinsic factor）**，故**惡性貧血、胃切除、迴腸疾病、素食、長期 PPI/metformin** 皆致缺乏。**神經症狀可能在貧血出現前就發生，且延誤可不可逆**——治療常需**肌肉注射補充**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **必須先給 B12 再給葉酸**：巨球性貧血若同時缺兩者，先補葉酸而未補 B12 會讓**神經病變（SCD）持續惡化**；單看血液反應好轉會誤判
> 2. **神經損害可不可逆**：亞急性合併退化（後索/側索）——本體感覺喪失、步態不穩、麻木、認知改變；**延誤治療數月以上恢復不全**，懷疑就驗、就治療
> 3. **惡性貧血/胃切除/迴腸病口服吸收不足**：需**肌肉注射**（缺內因子時口服吸收有限）；補鉀留意——嚴重貧血治療初期紅血球大量生成可致**低血鉀**
> 4. **治療起效初期網狀紅血球爆增**、鐵與鉀被大量利用 → 監測 K⁺、必要時補鐵
> 5. **長期 metformin、PPI/H2 blocker 使用者篩檢 B12**（吸收受影響），神經精神症狀不典型易漏診

## 💊 適應症（治療哪些疾病）
- **維生素 B12 缺乏性巨球性貧血 [[Megaloblastic Anemia(巨球性貧血)]]**
- **惡性貧血 [[Pernicious Anemia(惡性貧血)]]**（自體免疫破壞胃壁細胞/內因子）
- **亞急性合併退化 SCD / B12 缺乏神經病變**
- 吸收不良（胃切除、迴腸切除/克隆氏症、廣泛裂頭絛蟲）、嚴格素食、長期 metformin/PPI
- （Hydroxocobalamin 高劑量另用於**氰化物中毒**解毒）

## 📏 劑量 / 用法（成人）
| 情境 | 劑量 |
| --- | --- |
| B12 缺乏 / 惡性貧血起始 | **Cyanocobalamin 1000 µg IM，每日或隔日數次 × 1–2 週**，續每週，再每月維持 |
| 惡性貧血維持 | **1000 µg IM 每月一次**（終身） |
| 高劑量口服替代 | 部分非內因子缺乏者可 1000–2000 µg PO QD（吸收約 1% 被動擴散） |
| 氰化物中毒 | Hydroxocobalamin 高劑量 IV（不同適應症） |
- **惡性貧血/嚴重吸收不良首選 IM**；輕度飲食缺乏可口服高劑量
- 治療初期監測 **K⁺**（低血鉀）與網狀紅血球反應

## ⚠️ 副作用（常見 / 嚴重）
- 一般**耐受良好**（生理補充）
- 常見：注射部位疼痛、輕度腹瀉、搔癢
- **治療初期**：低血鉀（紅血球大量生成消耗鉀）、需鐵時缺鐵浮現
- 罕見：過敏反應（cyanocobalamin 少見）、痤瘡樣皮疹
- 高劑量本身毒性極低（水溶性）

## 🚫 禁忌症
- 對 cobalamin / 鈷或成分過敏
- Leber 視神經病變者慎用 cyanocobalamin（傳統顧慮）
- 未確認診斷即長期治療應避免（但懷疑缺乏時寧可先治療）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：CBC/MCV、網狀紅血球（治療 1 週後反應）、**血鉀**、血清 B12、必要時 methylmalonic acid / homocysteine（早期敏感指標）、神經症狀恢復
- **交互作用 / 誘因**：
  - **Metformin、PPI、H2 blocker**：長期降低 B12 吸收
  - 葉酸大劑量掩蓋 B12 缺乏血液表現
  - 氯黴素可能干擾造血反應
- **孕哺**：孕哺期需求存在、可安全補充（尤其素食母親）

## 🔗 相關
- 疾病：[[Megaloblastic Anemia(巨球性貧血)]]、[[Pernicious Anemia(惡性貧血)]]、亞急性合併退化（SCD）
- 必查搭配：[[Folic-Acid(葉酸)]]（巨球性貧血須同時評估，**先給 B12 再給葉酸**）
- 對照：[[Iron Supplements(鐵劑總覽)]]（小球性貧血）；誘因藥物 [[Biguanide(雙胍類)]]（metformin 長期降 B12）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 惡性貧血病人，關於 B12 補充何者最正確？
- A. 只需口服綜合維他命即可
- B. 因缺內因子吸收差，需 IM 補充並終身每月維持
- C. 補葉酸就能取代 B12
- D. 一次注射即可痊癒不需維持

> [!success]- 答案
> **B** — 惡性貧血缺內因子致口服吸收有限，需肌肉注射 1000 µg 起始並終身每月維持；葉酸無法取代 B12（否則神經病變惡化）。

**Q2.** 開始 B12 治療的嚴重巨球性貧血病人，第 3 天出現肌肉無力、心律不整。最需監測與矯正的是？
- A. 血鈉
- B. 血鉀（低血鉀）
- C. 血鈣
- D. 血糖

> [!success]- 答案
> **B** — 治療初期紅血球大量生成大量攝取鉀，可致低血鉀誘發心律不整與無力，需監測並補鉀。
