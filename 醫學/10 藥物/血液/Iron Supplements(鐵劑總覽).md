# Iron Supplements（鐵劑總覽）

> [!abstract] 一句話：補充**缺鐵性貧血 IDA** 的鐵元素以合成血紅素。**口服首選**（ferrous sulfate/gluconate/fumarate，便宜、有效、需空腹吸收但 GI 副作用常見）；**靜脈鐵**（iron sucrose、ferric carboxymaltose、iron dextran）用於口服不耐、吸收不良、CKD、需快速補充者，**dextran 型有過敏/過敏性休克風險需 test dose**。補鐵是「補庫存」，血紅素反應需數週、鐵庫回填需數月。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **靜脈鐵過敏反應 / 過敏性休克**：iron dextran 風險最高（給 test dose 25mg 觀察 ≥30–60 分鐘、備急救），新型 iron sucrose / ferric carboxymaltose 較安全但仍可能發生 → 給藥時床邊備 epinephrine，觀察生命徵象
> 2. **口服鐵 GI 副作用 + 黑便**：便秘、噁心、腹痛極常見致順從性差；**黑便屬正常染色**但要與消化道出血（黑色柏油便 melena）鑑別，別誤判
> 3. **與制酸劑 / 鈣 / 四環素 / 甲狀腺素 / 喹諾酮間隔給藥**：制酸劑、H2 blocker、PPI 升高胃 pH 減少鐵吸收；鈣鹽、tetracycline、levothyroxine、fluoroquinolone 會與鐵螯合互相干擾 → 錯開至少 2 小時
> 4. **兒童鐵過量致命**：誤食鐵錠是兒童中毒常見致死原因，**解毒劑 deferoxamine（去鐵胺）**；鐵劑務必兒童無法取得處存放
> 5. **補鐵前先確認診斷是缺鐵**：對非缺鐵性貧血（如地中海型貧血、慢性病貧血）盲目補鐵無效甚至造成鐵過載

## 💊 適應症（治療哪些疾病）
- **缺鐵性貧血 [[Iron Deficiency Anemia(缺鐵性貧血)]]**（最主要適應症：月經過多、慢性消化道失血、營養不足、吸收不良）
- **孕期缺鐵 / 預防性補鐵**（孕期鐵需求增加）
- **CKD / 洗腎病人缺鐵**（常搭配 [[Erythropoietin(紅血球生成素總覽)]]，靜脈鐵優先）
- **手術前貧血優化、化療相關缺鐵**
- **胃切除 / 減重手術 / 吸收不良症候群**（多需靜脈鐵）

## 📏 劑量 / 用法（成人）
| 劑型 | 元素鐵含量 / 劑量 |
| --- | --- |
| Ferrous sulfate 325 mg 錠 | 約含 **65 mg 元素鐵**；一般 **1 錠 QD–TID**（成人治療目標約 100–200 mg 元素鐵/日） |
| Ferrous gluconate 300 mg | 約含 35 mg 元素鐵 |
| Ferrous fumarate 200 mg | 約含 65 mg 元素鐵 |
| 口服吸收要點 | **空腹吸收最佳**（餐前 1h 或餐後 2h）；不耐可隨餐但吸收降；搭配維生素 C 助吸收 |
| Iron sucrose IV | 常見 **100–200 mg/次**，數次補足總缺量；⚠️依院規與計算公式 |
| Ferric carboxymaltose IV | 可單次大劑量（如 750–1000 mg），需依體重/Hb ⚠️依院規 |
| Iron dextran IV/IM | 可算總劑量一次補足，但**須先 test dose 25 mg** 觀察；⚠️依院規 |
- **總鐵缺量估算**：常用公式（如 Ganzoni）依體重、目標 Hb 與現值計算，實際劑量**⚠️依院規/藥師計算**，勿憑記憶給
- **口服療程**：Hb 正常後仍需續補 **3–6 個月**回填鐵庫（ferritin 目標）
- 隔日給藥（alternate-day）新證據顯示可能提升吸收率、減少副作用（依院規）

## ⚠️ 副作用（常見 / 嚴重）
- **口服**：便秘、噁心、上腹痛、腹瀉、金屬味、**黑便（正常染色）**、牙齒染色（液劑）
- **靜脈**：
  - **過敏 / 過敏性休克**（dextran 最高）、輸注反應（低血壓、潮紅、肌肉關節痛 "Fishbane reaction"）
  - 注射部位刺激、靜脈炎；肌肉注射可致皮膚永久染色
  - **低磷血症**（ferric carboxymaltose 較明顯，長期/重複使用注意）
- **過量**：急性鐵中毒（GI 出血、代謝性酸中毒、肝壞死、休克）→ deferoxamine 解毒

## 🚫 禁忌症
- **非缺鐵性貧血、鐵過載狀態**（血色素沉著症 hemochromatosis、反覆輸血鐵超載）
- 對特定鐵製劑或其成分過敏（尤其 dextran 過敏史）
- 活動性感染時靜脈鐵相對禁忌（鐵可能助長某些病原體，依臨床權衡）
- 肝硬化 / 嚴重肝病使用靜脈鐵需謹慎

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：CBC（Hb 反應約 2–4 週內網狀紅血球先升）、**ferritin + transferrin saturation（TSAT）** 評估鐵庫；靜脈鐵給藥中監測生命徵象與過敏
- **交互作用**：
  - **制酸劑 / PPI / H2 blocker**（升胃 pH ↓吸收）→ 錯開
  - **鈣鹽、tetracycline、fluoroquinolone、levothyroxine、bisphosphonate**（螯合互擾）→ 間隔 ≥2h
  - 維生素 C 促進吸收；茶、咖啡、麩皮、蛋、乳製品降低吸收
- **孕哺**：孕期常規使用（孕期鐵需求高）；哺乳安全

## 🔗 相關
- 疾病：[[Iron Deficiency Anemia(缺鐵性貧血)]]
- 同分區 / 對照：[[Folic Acid(葉酸)]]、[[Vitamin B12(維生素B12)]]（三者為貧血鑑別核心：小球性→鐵；巨球性→B12/葉酸）、[[Erythropoietin(紅血球生成素總覽)]]（CKD 貧血常合併補鐵）
- 解毒對照：deferoxamine（去鐵胺，鐵中毒/鐵過載螯合劑）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名 35 歲女性因月經過多致缺鐵性貧血，口服 ferrous sulfate 後主訴便秘、上腹痛且出現黑便。最適當的處置是？
- A. 立即安排上消化道內視鏡查出血
- B. 說明黑便為鐵劑正常染色，可調整為隨餐服用或隔日給以改善耐受
- C. 立刻停藥改為靜脈鐵
- D. 加倍劑量加速補鐵

> [!success]- 答案
> **B** — 黑便是口服鐵的正常染色（需與 melena 鑑別，但此處為預期副作用）；GI 不耐可隨餐服用或隔日給藥改善順從性，不需第一時間停藥或內視鏡。

**Q2.** 關於靜脈 iron dextran，下列何者最正確？
- A. 完全無過敏風險，可直接全量輸注
- B. 給藥前應先給 test dose 並觀察過敏反應、床邊備急救
- C. 可肌肉注射且不會造成染色
- D. 是所有病人的首選補鐵方式

> [!success]- 答案
> **B** — iron dextran 過敏/過敏性休克風險最高，須先 test dose（25mg）觀察並備急救藥；肌注會造成皮膚永久染色；靜脈鐵僅用於口服不耐/吸收不良/CKD 等。

**Q3.** 一名 2 歲幼童誤食大量成人鐵錠後嘔吐、嗜睡、代謝性酸中毒。最關鍵的特異性解毒劑是？
- A. N-acetylcysteine
- B. Naloxone
- C. Deferoxamine
- D. Vitamin K

> [!success]- 答案
> **C** — 急性鐵中毒的特異性螯合解毒劑為 deferoxamine（去鐵胺）。NAC 用於乙醯胺酚、naloxone 用於鴉片、Vit K 用於 warfarin。
