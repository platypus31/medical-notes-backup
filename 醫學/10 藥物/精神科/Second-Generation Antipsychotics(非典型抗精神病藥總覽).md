# Second-Generation Antipsychotics（非典型抗精神病藥總覽 / SGA）

> [!abstract] 一句話：**第二代（atypical）抗精神病藥**同時拮抗 **D2 + 5-HT2A 受體**，相較第一代（typical）**錐體外症狀 EPS 與高泌乳素較少**，但代價是**代謝症候群（體重↑、血糖↑、血脂↑）明顯**。值班兩大關鍵：**譫妄躁動 → 低劑量 quetiapine**（EPS 最少，帕金森/Lewy 可用）；**Clozapine → 需 ANC 監測防 agranulocytosis + 心肌炎 + 癲癇**，是療效最強但最危險的一支。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **Clozapine 三大致命性**：① **顆粒球缺乏症 agranulocytosis**（需定期 ANC 監測，ANC < 500 立即停藥）② **心肌炎 / 心肌病變**（用藥前數週胸痛/呼吸困難/CK-MB/troponin↑/嗜酸性球↑ 要警覺）③ **癲癇閾值下降**（高劑量劑量依賴）。另有**便祕→腸阻塞/腸缺血**可致死，勿輕忽
> 2. **代謝症候群**：olanzapine、clozapine 增重與代謝異常最嚴重 → 監測體重、腰圍、空腹血糖、血脂；可誘發**高血糖高滲透壓（HHS）/ DKA**
> 3. **QT 延長**：所有 SGA 均可，ziprasidone、部分劑量 quetiapine 較顯著 → 查 QTc + K⁺/Mg²⁺，避免與其他延長 QT 藥疊加
> 4. **NMS 惡性症候群**：SGA 仍會發生（雖較 typical 少）→ 高燒、僵硬、意識改變、CK 飆 → 停藥、支持、dantrolene/bromocriptine
> 5. **失智症老人**：所有抗精神病藥有**黑框警告增加死亡率**（心血管/感染/中風）→ 非首選，最低劑量最短療程
> 6. **譫妄合併帕金森 / Lewy body**：**首選 quetiapine 或 clozapine**（D2 阻斷最弱，較不惡化錐體外症狀）；勿用 risperidone/olanzapine 高劑量

## 💊 適應症（治療哪些疾病）
- **思覺失調症 [[Schizophrenia(思覺失調症)]]**（急性期與維持，尤其對負性症狀優於 typical）
- **雙相情緒障礙躁期 [[Bipolar Disorder(雙相情緒障礙)]]**（quetiapine、olanzapine、aripiprazole、risperidone 皆核可）
- **憂鬱症輔助治療 [[Major Depressive Disorder(重鬱症)]]**（aripiprazole、quetiapine XR 作為 augmentation）
- **譫妄躁動 [[Delirium(譫妄)]]**（低劑量 quetiapine，尤其帕金森/Lewy body 病人）
- **頑固型思覺失調 / 高自殺風險**（**Clozapine 為唯一有實證的頑固型首選**，但因副作用列為第三線）

## 📏 劑量 / 用法（成人，⚠️依院規/仿單滴定）
| 藥物 | 特點 | 常見起始 / 範圍（成人） |
| --- | --- | --- |
| **Quetiapine（思樂康）** | 鎮靜強、EPS/泌乳素最少、帕金森可用 | 譫妄 **12.5–25 mg PO qhs prn**；思覺失調 150–750 mg/day 分次 |
| **Risperidone（理思必妥）** | 高效價、**高泌乳素**、劑量↑ 則 EPS↑ | 1–6 mg/day（老人 0.25–0.5 mg 起）；長效針劑 consta |
| **Olanzapine（金普薩）** | **代謝副作用最重**、鎮靜、可 IM 快速鎮靜 | 5–20 mg/day；急性躁動 IM 5–10 mg（**勿與 IV BZD 併用致心肺抑制**） |
| **Aripiprazole（安立復）** | **D2 部分促效**、代謝/泌乳素/鎮靜最輕、可能致躁動/靜坐不能 | 10–30 mg/day |
| **Clozapine（可致律）** | **療效最強、副作用最危險**、第三線 | 12.5 mg 起極緩滴定，需 ANC 監測；⚠️依專科方案 |
- **老人 / 譫妄**：一律大幅減量、緩慢滴定
- **腎/肝**：多經肝代謝，重度肝損減量；risperidone 主要腎排泄需依腎調整
- **給藥途徑**：多為 PO；olanzapine、ziprasidone 有 IM 針劑用於急性躁動

## ⚠️ 副作用（共性 + 個別差異）
- **共性**：
  - **代謝症候群**（體重↑、血糖↑、血脂↑）— SGA 標誌性副作用
  - QT 延長、鎮靜、姿勢性低血壓、抗膽鹼作用（口乾/便祕/尿滯留）
  - EPS 與高泌乳素**較 typical 少但非零**（劑量依賴）
  - NMS、遲發性運動障礙（TD）— 較 typical 少但仍可能
- **個別差異**：
  - **Clozapine**：agranulocytosis、心肌炎/心肌病、癲癇、致命性便祕/腸阻塞、流涎、鎮靜
  - **Olanzapine / Clozapine**：代謝副作用最重（增重、糖脂異常、可誘發 DKA/HHS）
  - **Risperidone**：**高泌乳素**（溢乳、月經失調、性功能障礙）、劑量高則 EPS
  - **Aripiprazole**：靜坐不能（akathisia）、失眠、噁心（部分促效特性）
  - **Quetiapine**：鎮靜、姿勢性低血壓；EPS/泌乳素最少

## 🚫 禁忌症
- 對該藥過敏
- 已知 QT 延長、未矯正低血鉀/低血鎂、近期心肌梗塞、失代償心衰（相對）
- **Clozapine**：既往 clozapine 誘發之 agranulocytosis、未受控癲癇、麻痺性腸阻塞、嚴重心/骨髓疾病
- 昏迷 / 嚴重中樞抑制
- 失智相關精神症狀（黑框警告，非絕對禁忌但須權衡）

## 🔍 監測 / 交互作用 / 孕哺
- **共通監測**：**體重/BMI/腰圍、空腹血糖、HbA1c、血脂**、血壓、QTc、EPS（AIMS）
- **Clozapine 特別監測**：**ANC / CBC（起始每週，依方案）**、心肌炎徵象（troponin/CK-MB/CRP/EKG）、血中濃度、便祕評估
- **交互作用**：
  - CYP1A2（clozapine、olanzapine）：**吸菸誘導 → 戒菸/住院禁菸時血中濃度驟升**，fluvoxamine 為強抑制劑
  - CYP3A4/2D6：多種 SGA 受影響
  - 其他延長 QT 藥、CNS 抑制劑（BZD/opioid/酒精）累加
  - **olanzapine IM + 腸道外 BZD → 嚴重心肺抑制/低血壓，避免併用**
- **孕哺**：權衡使用；未治療精神病本身也有風險，需精神科共同決策

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名 82 歲 Lewy body 失智合併夜間譫妄躁動，非藥物介入無效需短期藥物。最適合的選擇為？
- A. Haloperidol 5 mg IM
- B. 低劑量 quetiapine
- C. 高劑量 risperidone
- D. Olanzapine 20 mg

> [!success]- 答案
> **B** — 帕金森/Lewy body 病人對 D2 阻斷極敏感，應選 D2 阻斷最弱、EPS 最少的低劑量 quetiapine；haloperidol 與高劑量 risperidone/olanzapine 會惡化運動症狀並增敏感反應。

**Q2.** 服用 clozapine 兩週的病人回診主訴發燒、胸悶、喘。除感染外，最需優先排除的致命併發症是？
- A. 遲發性運動障礙
- B. 心肌炎
- C. 高泌乳素血症
- D. 姿勢性低血壓

> [!success]- 答案
> **B** — Clozapine 起始數週內的心肌炎/心肌病變是致命併發症，出現胸痛、呼吸困難、發燒應查 troponin/CK-MB/CRP/嗜酸性球/EKG；同時 clozapine 也需監測 ANC 排除 agranulocytosis。

**Q3.** 關於 SGA 的共通代謝副作用監測，何者正確？
- A. 只需監測血壓
- B. 應定期監測體重/BMI、空腹血糖、HbA1c、血脂
- C. SGA 不會影響血糖
- D. Aripiprazole 代謝副作用最嚴重

> [!success]- 答案
> **B** — SGA 標誌性副作用為代謝症候群，須定期監測體重/BMI/腰圍、空腹血糖/HbA1c、血脂；代謝負擔最重者為 olanzapine 與 clozapine，aripiprazole 反而最輕。
