# EPS & NMS Management（錐體外症候群與惡性症候群處置）

> [!abstract] 一句話：抗精神病藥（尤其**典型 / 高效價**如 haloperidol）阻斷紋狀體 **D2 受體** → 引發**錐體外症候群（EPS）四型**（急性肌張力不全 → 靜坐不能 → 藥物性帕金森 → 遲發性運動異常）與最致命的**惡性症候群（NMS）**。**處置核心＝「認出是哪一型 → 用對解藥」**：dystonia 給抗膽鹼、akathisia 給 propranolol、parkinsonism 減量 + 抗膽鹼、TD 停 / 換藥（勿再加抗膽鹼）、NMS 立即停藥 + 支持 + dantrolene / bromocriptine。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **NMS 是致命急症**（死亡率 ~10%）：高燒 + 鉛管樣僵直 + 意識改變 + 自主神經不穩 + **CK 大幅升高** → **立即停所有抗精神病藥**、ICU 支持、降溫水化
> 2. **急性肌張力不全給抗膽鹼（benztropine / diphenhydramine），不是給 BZD 當第一線**；眼動危象（oculogyric crisis）、喉頭肌痙攣可危及呼吸道
> 3. **靜坐不能（akathisia）易被誤判為精神病惡化而「加抗精神病藥」→ 越加越糟**！正解是**減量 + propranolol**
> 4. **遲發性運動異常（TD）常不可逆，且抗膽鹼會「惡化」TD**——不要對 TD 再加 benztropine（與 dystonia 處置相反）
> 5. **NMS vs 血清素症候群鑑別**：NMS 是**鉛管樣僵直、反射↓、數天緩起**；血清素症候群是**陣攣 clonus、反射亢進、數小時急起 + 腹瀉**，處置與致病藥完全不同

## 💊 適應症（本篇處置對象）
- 使用**抗精神病藥（typical > atypical）**、止吐劑（**metoclopramide、prochlorperazine**）、部分藥物後出現的錐體外副作用
- 涵蓋：急性肌張力不全、靜坐不能、藥物性帕金森、遲發性運動異常、惡性症候群
- 高風險藥：[[Haloperidol(氟哌啶醇)]]、[[Chlorpromazine(氯丙嗪)]]、高效價典型抗精神病藥、[[Metoclopramide(甲氧氯普胺)]]

## 📏 劑量 / 用法（成人）— 各型處置對照
| 型 / 症候 | 發作時序 | 臨床特徵 | 處置 & 劑量 |
| --- | --- | --- | --- |
| **急性肌張力不全 Acute dystonia** | 數小時～數天 | 眼動危象、斜頸、牙關緊閉、喉痙攣 | **Benztropine 1–2mg IM/IV** 或 **Diphenhydramine 25–50mg IM/IV**，可重複；喉痙攣危及呼吸道立即給 |
| **靜坐不能 Akathisia** | 數天～數週 | 坐立難安、來回踱步、主觀焦躁 | **減抗精神病藥劑量** + **Propranolol 10–20mg PO BID–TID**；或加 BZD（lorazepam / clonazepam），抗膽鹼效果較差 |
| **藥物性帕金森 Drug-induced parkinsonism** | 數週～數月 | 僵直、靜止性顫抖、動作遲緩、面具臉 | **減量 / 換非典型（如 quetiapine）** + **Benztropine 1–2mg BID** 或 amantadine |
| **遲發性運動異常 Tardive dyskinesia (TD)** | 數月～數年（常不可逆） | 口舌 / 顏面不自主動作、扮鬼臉 | **停 / 減藥或換 clozapine**；**VMAT2 抑制劑（valbenazine / deutetrabenazine）**；**避免抗膽鹼（會惡化）** |
| **惡性症候群 NMS** | 數天緩起 | 高燒、鉛管樣僵直、意識改變、自主神經不穩、**CK↑↑、白血球↑** | **立即停藥** + 支持（降溫、輸液、電解質）+ **Dantrolene 1–2.5mg/kg IV**（解僵直）+ **Bromocriptine 2.5mg PO Q8H**（DA 促效） |
- **抗膽鹼藥禁忌**：老人、閉角型青光眼、尿滯留、腸阻塞、認知障礙者慎用（抗膽鹼副作用）
- 劑量以院規 / guideline 為準 ⚠️依院規 / 需查證

## ⚠️ 副作用（處置藥本身）
- **抗膽鹼（benztropine / diphenhydramine）**：口乾、視力模糊、便祕、尿滯留、譫妄（尤其老人）、心搏過速
- **Propranolol**：心搏過緩、低血壓、支氣管痙攣（氣喘慎用）、掩蓋低血糖
- **Dantrolene**：肝毒性、肌肉無力
- **Bromocriptine**：噁心、姿勢性低血壓、精神症狀惡化（DA 促效可能加重原精神病）

## 🚫 禁忌症
- **抗膽鹼藥**：閉角型青光眼、尿路阻塞 / 攝護腺肥大尿滯留、麻痺性腸阻塞、重症肌無力；**用於 TD 會惡化**
- **Propranolol**：嚴重氣喘 / COPD 支氣管痙攣、心因性休克、嚴重心搏過緩 / 房室傳導阻滯
- **Dantrolene**：活動性肝病
- **Bromocriptine**：未控制高血壓、對麥角類過敏

## 🔍 監測 / 交互作用 / 鑑別
- **監測**：NMS → 體溫、CK、腎功能（橫紋肌溶解 → 急性腎損傷）、電解質、生命徵象、意識；EPS → 症狀變化與復發
- **鑑別診斷**：
  - **NMS ↔ 血清素症候群**：後者有**反射亢進 + 陣攣 clonus + 腹瀉 + 數小時急起**，致病藥為血清素藥（SSRI/SNRI/tramadol/linezolid）
  - **NMS ↔ 惡性高熱**：後者由吸入性麻醉劑 / succinylcholine 誘發，處置同樣用 dantrolene
  - **NMS ↔ 中央性抗膽鹼症候群 / 熱衰竭 / 中樞感染**
- **關鍵原則**：**dystonia 給抗膽鹼、TD 忌抗膽鹼**——同為運動障礙但處置相反，是最常見錯誤

### 記憶卡（Spaced Repetition 相容 · `Q::A`）

EPS 四型依「發作時序由快到慢」排列為何？::急性肌張力不全（數小時–數天）→ 靜坐不能（數天–數週）→ 藥物性帕金森（數週–數月）→ 遲發性運動異常（數月–數年）

急性肌張力不全（acute dystonia）的第一線處置？::抗膽鹼藥——benztropine 1–2mg IM/IV 或 diphenhydramine 25–50mg IM/IV

靜坐不能（akathisia）的正確處置？為何不能加抗精神病藥？::減抗精神病藥劑量 + propranolol（β-blocker），或加 BZD；加抗精神病藥會被誤當精神病惡化而越加越糟

遲發性運動異常（TD）的處置原則、為何禁抗膽鹼？::停 / 減藥或換 clozapine、用 VMAT2 抑制劑（valbenazine）；抗膽鹼會「惡化」TD 故禁用

NMS 的五大臨床特徵？::高燒、鉛管樣僵直、意識改變、自主神經不穩、CK 大幅升高（常合併白血球升高）

NMS 的處置三步驟？::立即停所有抗精神病藥 + 支持治療（降溫 / 輸液 / 電解質）+ dantrolene（解僵直）± bromocriptine（DA 促效）

NMS 與血清素症候群如何鑑別？::NMS＝鉛管樣僵直、反射↓、數天緩起；血清素症候群＝陣攣 clonus、反射亢進、腹瀉、數小時急起

藥物性帕金森的處置？::減量 / 換非典型抗精神病藥 + benztropine 或 amantadine

哪一類止吐劑也會引起 EPS？::多巴胺拮抗類止吐劑，如 metoclopramide、prochlorperazine

「dystonia 與 TD 都是運動障礙，處置卻相反」指的是什麼？::dystonia 要「給」抗膽鹼；TD 則「禁」抗膽鹼（會惡化），是常見錯誤

NMS 為何要監測 CK 與腎功能？::肌肉持續僵直造成橫紋肌溶解，CK 大幅升高並可致急性腎損傷

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名思覺失調病人打完 haloperidol IM 後 2 小時出現雙眼上吊（眼動危象）、頸部扭轉。最適當的處置是？
- A. 立即停藥並觀察
- B. 給 benztropine 或 diphenhydramine
- C. 給 propranolol
- D. 給 dantrolene

> [!success]- 答案
> **B** — 這是急性肌張力不全（acute dystonia），第一線給抗膽鹼藥（benztropine IM/IV）或 diphenhydramine，症狀通常數分鐘內緩解；喉痙攣危及呼吸道時更需立即給藥。

**Q2.** 一名服用抗精神病藥數日的病人主訴「坐不住、一直想走動、非常煩躁」，客觀見不停踱步。最正確的處置？
- A. 增加抗精神病藥劑量（懷疑精神病惡化）
- B. 給 benztropine
- C. 減抗精神病藥劑量並加 propranolol
- D. 立即給 dantrolene

> [!success]- 答案
> **C** — 這是靜坐不能（akathisia）。常被誤判為精神病惡化而加藥（A）導致惡化；正解是減量並加 propranolol（β-blocker），必要時加 BZD。

**Q3.** 一名長期服 haloperidol 病人出現 40°C 高燒、全身鉛管樣僵直、意識混亂、血壓不穩，CK 8000。下列處置何者「錯誤」？
- A. 立即停用抗精神病藥
- B. 積極降溫與輸液
- C. 給 dantrolene 或 bromocriptine
- D. 增加抗精神病藥以控制躁動

> [!success]- 答案
> **D** — 此為惡性症候群（NMS），是致命急症。抗精神病藥正是致病元凶，必須「立即停用」而非增加；處置為支持治療 + dantrolene（解僵直）± bromocriptine（DA 促效）。
