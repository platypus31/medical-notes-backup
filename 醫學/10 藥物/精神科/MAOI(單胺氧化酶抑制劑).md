# MAOI（單胺氧化酶抑制劑總覽）

> [!abstract] 一句話：MAOI 抑制單胺氧化酶（MAO-A / MAO-B）阻止正腎上腺素 / 血清素 / 多巴胺被分解 → 神經傳導物累積 → 抗憂鬱；代價是兩大致命交互作用——**tyramine 高血壓危象**（飲食）與**血清素症候群**（併用其他血清素藥），現今多為難治型憂鬱的三、四線用藥。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **Tyramine 高血壓危象** → 非選擇性 / 不可逆 MAOI（Tranylcypromine、Phenelzine）併用**高酪胺（tyramine）食物**（陳年起司、醃製 / 煙燻肉品、紅酒、發酵食物、醬油）→ 大量正腎上腺素釋放 → 劇烈頭痛、高血壓危象、中風風險
> 2. **血清素症候群（serotonin syndrome）** → 與 SSRI / SNRI / TCA / meperidine（Pethidine）/ tramadol / dextromethorphan / linezolid 併用禁忌 → 高熱、肌陣攣、反射亢進、意識改變，可致命
> 3. **必須有洗滌期（washout）** → 換藥停 MAOI 或停 SSRI 改 MAOI，一般間隔 **約 2 週**；**Fluoxetine 因半衰期長需約 5 週**
> 4. **與擬交感神經藥（含市售感冒藥 pseudoephedrine / phenylephrine）併用** → 也會誘發高血壓危象
> 5. Moclobemide（RIMA）相對安全，但仍非零風險；懷疑高血壓危象時**IV 短效降壓**（如 phentolamine / nitroprusside，依院規）

## 💊 適應症（治療哪些疾病）
- **難治型 / 非典型憂鬱症**：[[Major Depressive Disorder(重度憂鬱症)]]（三、四線，其他抗憂鬱無效時）
- **非典型憂鬱、伴焦慮 / 恐慌** 的部分病人
- **Selegiline（低劑量）**：[[Parkinson Disease(帕金森氏症)]] → 見 [[Antiparkinson Drugs(帕金森藥物總覽)]]（MAO-B 選擇性抑制）
- 現今臨床少用，多被 SSRI / SNRI 取代

## 📏 各藥重點（成人）
| 藥物 | 類型 | 特點 | tyramine 風險 |
| --- | --- | --- | --- |
| **Tranylcypromine** | 非選擇性、**不可逆** MAOI | 難治型憂鬱；起效相對快，有輕微興奮作用 | **高**（需嚴格低酪胺飲食） |
| **Phenelzine** | 非選擇性、不可逆 MAOI | 非典型憂鬱 / 焦慮 | **高**（需嚴格低酪胺飲食） |
| **Moclobemide** | **可逆性 MAO-A 選擇性（RIMA）** | 飲食 / 藥物交互作用風險**較低**（tyramine 可競爭性置換藥物），比傳統 MAOI 安全 | 低（仍建議適度節制） |
| **Selegiline** | 低劑量選擇性 **MAO-B** 抑制劑 | 帕金森氏症；**高劑量 / 貼片**會失去 MAO-B 選擇性 → 恢復 tyramine 風險 | 低劑量低、高劑量 / 貼片高 |

- 起始劑量、滴定、洗滌期天數**依院規 / 精神科指引 / 需查證**

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：姿勢性低血壓（矛盾但常見）、失眠 / 鎮靜、體重增加、性功能障礙、口乾、水腫
- **嚴重**：
  - **Tyramine 高血壓危象**（見紅旗）
  - **血清素症候群**（見紅旗）
  - 肝毒性（Phenelzine 較多）
  - Tranylcypromine 過量 → 交感興奮、抽搐
- **Selegiline 高劑量**：失去 MAO-B 選擇性後同樣有上述風險

## 🚫 禁忌症
- 併用 **SSRI / SNRI / TCA / 其他 MAOI**（未過洗滌期）
- 併用 **meperidine（Pethidine）/ tramadol / dextromethorphan / linezolid / 擬交感藥 / 三環**
- 未能配合低酪胺飲食者（不可逆 MAOI）
- 嗜鉻細胞瘤、未控制高血壓、心血管疾病
- 併用其他升壓 / 擬交感感冒成藥
- 對該藥過敏

## 🔍 監測 / 交互作用 / 孕哺
- **飲食衛教（最重要）**：不可逆 MAOI 需嚴格避免高酪胺食物（陳年起司、煙燻 / 醃製肉、紅酒、生啤、發酵豆製品、蠶豆）
- **洗滌期**：MAOI ↔ 其他抗憂鬱互換需間隔約 **2 週**（Fluoxetine 約 **5 週**）
- **血壓監測**：留意突發劇烈頭痛（高血壓危象前兆）
- **交互作用**：見禁忌；含 tyramine 食物、擬交感藥、血清素藥、部分止痛藥皆危險
- **孕哺**：一般避免，資料有限 / 需查證
- **Moclobemide（RIMA）**：交互作用較輕，但仍避免與血清素藥、pethidine 併用

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位服用 Tranylcypromine 的病人晚餐吃了大量陳年起司與紅酒，隨後劇烈頭痛、血壓飆到 210/120。最可能的機轉是？
- A. 血清素症候群
- B. tyramine 引發正腎上腺素大量釋放
- C. 過敏反應
- D. 低血糖

> [!success]- 答案
> **B** — 不可逆 MAOI 使 tyramine 無法被腸道 / 肝臟 MAO 分解，大量進入循環促使正腎上腺素釋放，造成高血壓危象；需以短效降壓藥處置。

**Q2.** 病人原本服用 Fluoxetine，醫師欲改用 MAOI。關於洗滌期，正確的是？
- A. 可立即換藥
- B. 停 Fluoxetine 約 1 天即可
- C. 因 Fluoxetine 半衰期長，需約 5 週洗滌期
- D. MAOI 與 SSRI 可安全併用

> [!success]- 答案
> **C** — MAOI 與 SSRI 併用會引發血清素症候群；一般洗滌期約 2 週，但 Fluoxetine（及活性代謝物）半衰期長，需約 **5 週**才能安全改用 MAOI。

**Q3.** 下列哪一種 MAOI 因「可逆且選擇性抑制 MAO-A」而飲食限制相對寬鬆？
- A. Phenelzine
- B. Tranylcypromine
- C. Moclobemide
- D. 高劑量 Selegiline

> [!success]- 答案
> **C** — Moclobemide 是 RIMA（可逆性 MAO-A 抑制劑），tyramine 可競爭性把藥物從酵素置換下來，故 tyramine 危象風險較低、飲食限制較寬鬆。
