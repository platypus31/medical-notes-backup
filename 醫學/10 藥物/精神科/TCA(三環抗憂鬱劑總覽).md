# TCA（三環抗憂鬱劑總覽 / Imipramine · Amitriptyline · Nortriptyline）

> [!abstract] 一句話：**抑制 NE 與 5-HT 再回收**提升單胺濃度而抗憂鬱；但因**同時阻斷 muscarinic（抗膽鹼）、H1（鎮靜）、α1（姿勢性低血壓）受體**副作用多。最大值班威脅是**過量致死**——治療窗窄，過量以**心臟毒性（QRS 增寬）、抽搐、昏迷**表現，處置核心為 **NaHCO₃**。現多為二線，因 SSRI/SNRI 更安全。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **過量致死「3C」**：**Cardiotoxicity**（阻斷 Na⁺ 通道 → QRS 增寬、寬 QRS 心律不整、低血壓）、**Convulsion**（抽搐）、**Coma**（昏迷）。**QRS > 100 ms 即給 NaHCO₃**，> 160 ms 高度心室心律不整風險。這是 TCA 最危險處——**一週劑量即可致命**，開給有自殺風險病人須極謹慎、少量給藥
> 2. **抗膽鹼副作用（抗蕈毒鹼）**：口乾、便秘、尿滯留、視力模糊、**譫妄/意識混亂**（老年人尤其）、青光眼惡化。老人、BPH（攝護腺肥大）、閉角性青光眼要當心
> 3. **姿勢性低血壓**（α1 阻斷）：老人跌倒骨折風險，起身要慢
> 4. **鎮靜 + 體重增加**（H1 阻斷），Amitriptyline 最明顯
> 5. **心臟病、近期心肌梗塞、傳導阻滯**病人避免；用前建議查 **EKG**（尤其老年或有心臟病史）
> 6. **禁與 MAOI 併用**（血清素症候群、高血壓危象），中間須 washout（通常 ≥ 14 天）

## 💊 適應症
- **重度憂鬱症 [[SSRI(選擇性血清素回收抑制劑總覽)]] 之二線**（現多以 SSRI/SNRI 為首選，TCA 用於難治型）
- **神經病理性疼痛 / 糖尿病神經痛 / 帶狀疱疹後神經痛**（Amitriptyline、Nortriptyline 低劑量常用）
- **偏頭痛預防**（Amitriptyline）
- **失眠**（Amitriptyline 低劑量，取其鎮靜）
- **兒童遺尿症 nocturnal enuresis**（Imipramine——經典適應症）
- **強迫症 OCD**（Clomipramine，血清素選擇性最強的 TCA）

## 📏 劑量 / 用法（成人）
| 藥物 | 特性 | 起始 / 常用劑量 |
| --- | --- | --- |
| **Imipramine** | 平衡型（NE≈5-HT）；兒童遺尿經典 | 憂鬱 起始 25–75 mg/day，漸增至 100–200 mg/day；遺尿 兒童睡前 10–25 mg |
| **Amitriptyline** | 鎮靜最強、抗膽鹼最強；疼痛/偏頭痛/失眠常用 | 疼痛/預防 起始 **10–25 mg 睡前**，漸增；憂鬱可至 75–150 mg/day |
| **Nortriptyline** | Amitriptyline 活性代謝物，**副作用（鎮靜/姿勢性低血壓/抗膽鹼）較輕**，老人較耐受，有**治療濃度區間** | 起始 25 mg，常用 50–100 mg/day（可測血中濃度 50–150 ng/mL） |
- **老年人減量、緩慢滴定**；睡前給藥利用鎮靜助眠並減日間副作用
- ⚠️依院規/需查證：疼痛與精神科目標劑量差異大，實際依適應症與耐受度個別化

## ⚠️ 副作用（常見 / 嚴重）
- **常見（受體阻斷）**：
  - 抗膽鹼：口乾、便秘、尿滯留、視力模糊、認知模糊
  - 抗組織胺（H1）：鎮靜、嗜睡、體重增加
  - 抗 α1：姿勢性低血壓、頭暈
- **嚴重**：
  - **心臟毒性**：QRS 增寬、QT 延長、傳導阻滯、寬 QRS 心律不整（過量時致命）
  - **癲癇發作**（降低發作閾值）
  - **譫妄 / 抗膽鹼性精神狀態改變**（老人）
  - 抗憂鬱藥共通：**年輕人自殺意念增加**（黑框警語）、轉躁（雙相未診斷者）
  - 血清素症候群（併其他血清素藥）

## 🚫 禁忌症
- **與 MAOI 併用或未滿 washout 期**（血清素症候群/高血壓危象）
- **近期急性心肌梗塞、嚴重心臟傳導障礙**（bundle branch block、QT 延長）
- 閉角性青光眼（未控制）
- 已知過敏
- **急性過量高致死**——嚴重自殺風險者相對禁忌 / 須限量給藥

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：用前與滴定期 **EKG**（QRS/QT，老人或心臟病史）；情緒與自殺意念；Nortriptyline 可測**血中濃度**；老人查認知與姿勢性血壓
- **過量處置（值班關鍵）**：
  - ABC、心電監測、抽 EKG
  - **QRS > 100 ms → 靜脈 NaHCO₃**（鹼化血液 + 提供 Na⁺，逆轉 Na⁺ 通道阻斷、縮窄 QRS、穩定心肌）
  - 低血壓：輸液 + 升壓劑；抽搐：benzodiazepine
  - **禁用 class IA/IC 抗心律不整藥**（如 procainamide，會加重 Na⁺ 阻斷）
  - 活性碳（早期）；**不建議催吐**
- **交互作用**：MAOI（禁）、其他血清素藥（SSRI/SNRI/tramadol/linezolid → 血清素症候群）、QT 延長藥、CYP2D6 抑制劑（fluoxetine/paroxetine 升 TCA 濃度）、抗膽鹼藥加成、酒精加重鎮靜
- **孕哺**：非首選；效益風險評估後可用，新生兒可能有戒斷/抗膽鹼症狀

TCA 過量時為何禁用 class IA/IC 抗心律不整藥（如 procainamide）？::它們同樣阻斷 Na⁺ 通道，會加重心臟傳導障礙與心律不整

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名 22 歲女性服用大量 amitriptyline 自殺，到院意識不清、血壓低、EKG 顯示 QRS 140 ms。最優先的處置是？
- A. 給予 procainamide 控制心律
- B. 靜脈輸注 NaHCO₃
- C. 立即血液透析
- D. 給予 flumazenil

> [!success]- 答案
> **B** — TCA 過量心臟毒性核心是 Na⁺ 通道阻斷致 QRS 增寬。QRS > 100 ms 即應給 NaHCO₃（鹼化 + 供 Na⁺ 逆轉阻斷）。class IA（procainamide）會加重阻斷（A 錯）；TCA 蛋白結合高、分布體積大，透析無效（C 錯）；flumazenil 用於 BZD 過量且可能誘發抽搐（D 錯）。

**Q2.** 78 歲男性用 amitriptyline 治療失眠後出現尿不出來、便秘、視力模糊與意識混亂。最可能的機轉是？
- A. α1 受體阻斷
- B. H1 受體阻斷
- C. Muscarinic（抗膽鹼）受體阻斷
- D. 血清素再回收抑制

> [!success]- 答案
> **C** — 口乾、便秘、尿滯留、視力模糊、譫妄皆為抗膽鹼（抗蕈毒鹼）症狀，老年人尤其明顯，故老人多改用副作用較輕的 nortriptyline 或改藥。
