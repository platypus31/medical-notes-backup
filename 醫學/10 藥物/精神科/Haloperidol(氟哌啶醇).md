# Haloperidol（氟哌啶醇 / Haldol）

> [!abstract] 一句話：**高效價第一代（typical）抗精神病藥**，強力阻斷中腦邊緣系統 **D2 受體**。值班處理**急性躁動（agitation）與譫妄（delirium）最常用**的針劑，鎮靜與抗膽鹼作用相對「乾淨」（低姿勢性低血壓、低抗膽鹼），代價是**錐體外症狀 EPS 最多、QT 延長、NMS** 風險。IM/IV/PO 皆可，老人與譫妄需大幅減量。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **QT 延長 / Torsades**：給藥前查基礎 QTc 與 K⁺/Mg²⁺；**IV 給藥 QT 風險最高**（FDA 警告 IV 為非仿單用法，需連續心電監測）。避免與其他延長 QT 藥（ondansetron、azithromycin、methadone）疊加
> 2. **急性肌張力不全（acute dystonia）**：眼球上吊（oculogyric crisis）、斜頸、喉痙攣 → **立即給 anticholinergic：benztropine 1–2 mg IM/IV 或 diphenhydramine 25–50 mg IM/IV**，喉痙攣是急症
> 3. **抗精神病藥惡性症候群 NMS**：高燒、肌肉僵硬（rigidity）、意識改變、自律神經不穩、CK 飆高 → 立即停藥、支持治療、dantrolene / bromocriptine
> 4. **失智症老人行為問題**：所有抗精神病藥有 **黑框警告增加死亡率**（心血管/感染），非首選、用最低劑量最短療程
> 5. **Parkinson's disease / Lewy body 失智**：D2 阻斷會惡化錐體外症狀與引發嚴重敏感反應 → **改用 quetiapine 或 clozapine**；譫妄合併帕金森勿用 haloperidol

## 💊 適應症（治療哪些疾病）
- **急性躁動 / 攻擊行為 [[Acute Agitation(急性躁動)]]**（急診、加護、精神科值班第一線針劑）
- **譫妄 [[Delirium(譫妄)]]**（低劑量短期控制躁動型譫妄，非治本；先找誘因）
- **思覺失調症 [[Schizophrenia(思覺失調症)]]**（急性期與維持，含長效針劑 haloperidol decanoate）
- **妥瑞症 / 抽動症 [[Tourette Syndrome(妥瑞症)]]**（頑固抽動）
- **頑固性噁心嘔吐、化療止吐**（次選，作用於 CTZ 的 D2）、術後躁動

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 急性躁動（成人） | **2–5 mg IM/IV**，可 Q4–6H 重複；輕中度或 PO 可 0.5–2 mg |
| 譫妄（住院/加護） | **0.5–1 mg PO/IM Q4–6H prn**（老人從 0.25–0.5 mg 起）；監測 QT |
| 老人 / 衰弱 | 起始 **0.25–0.5 mg**，緩慢滴定；避免高劑量 |
| 思覺失調維持 | PO 2–20 mg/day 分次（依反應與耐受滴定） |
| 長效針劑（decanoate） | 每 4 週 IM，劑量依院規/仿單換算，非值班急用 |
| 每日上限 | 一般 ≤ 20 mg/day；高劑量顯著增 EPS/QT（依院規/仿單） |
- **腎功能**：主要經肝代謝，腎功能不良一般不需大幅調整；透析不太移除
- **肝功能**：肝代謝（CYP3A4/2D6），重度肝損宜減量、謹慎滴定
- **給藥途徑**：IM 吸收快；**IV 為非仿單用法需心電監測**；PO 生體可用率約 60–70%

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：
  - **錐體外症狀 EPS**（高效價 typical 最突出）：急性肌張力不全、類帕金森（僵硬/顫抖/面具臉）、靜坐不能（akathisia，坐立難安易誤判為躁動加重）
  - 鎮靜（比低效價 chlorpromazine 輕）、高泌乳素血症（溢乳、月經失調、性功能障礙）
- **嚴重**：
  - **QT 延長 → Torsades de pointes / 猝死**（IV 尤甚）
  - **抗精神病藥惡性症候群 NMS**（罕見但致命）
  - **遲發性運動障礙 tardive dyskinesia**（長期使用，常不可逆）
  - 癲癇閾值下降、罕見血球低下（agranulocytosis）
  - 失智老人死亡率增加（黑框）

## 🚫 禁忌症
- 對 haloperidol 或 butyrophenone 類過敏
- **昏迷 / 嚴重中樞抑制**（酒精、巴比妥、鴉片急性中毒）
- **Parkinson's disease、Lewy body 失智**（相對禁忌，惡化 EPS）
- **已知 QT 延長、未矯正低血鉀/低血鎂、近期心肌梗塞、失代償心衰**

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：**心電圖 QTc + 電解質（K⁺/Mg²⁺）**、EPS（AIMS 量表）、生命徵象、NMS 徵象（體溫/僵硬/CK）；譫妄病人監測意識與鎮靜深度
- **交互作用**：
  - **其他延長 QT 藥物**（ondansetron、methadone、azithromycin、amiodarone、citalopram）→ 累加 QT 風險
  - **CNS 抑制劑**（benzodiazepine、opioid、酒精）→ 加重鎮靜與呼吸抑制
  - **CYP3A4/2D6 抑制劑**（ketoconazole、paroxetine）→ 升高 haloperidol 濃度
  - **levodopa / dopamine agonist**：藥效互相拮抗（D2 阻斷 vs 促進）
- **孕哺**：孕期權衡使用（第一代資料相對多；第三孕期暴露新生兒可能有短暫 EPS/戒斷）；哺乳少量分泌，謹慎

### 自我測驗（選擇題，答案摺疊）

**Q1.** 加護病房一名 78 歲譫妄躁動老人給予 haloperidol 後出現頸部後仰、眼球持續上吊。最恰當的立即處置為？
- A. 再加一劑 haloperidol
- B. 給 benztropine 或 diphenhydramine
- C. 給 naloxone
- D. 立即插管

> [!success]- 答案
> **B** — 這是急性肌張力不全（acute dystonia，含 oculogyric crisis），屬 EPS，應立即給抗膽鹼藥 benztropine 或 diphenhydramine 緩解；不可再加 haloperidol。

**Q2.** 關於 haloperidol 與 QT 延長，何者正確？
- A. 只有口服會延長 QT
- B. IV 給藥風險最高，需監測心電圖與電解質
- C. 與 ondansetron 併用完全安全
- D. 低血鉀不影響風險

> [!success]- 答案
> **B** — IV haloperidol QT/Torsades 風險最高（FDA 警告），需連續心電監測；並應矯正低血鉀/低血鎂、避免與其他延長 QT 藥（如 ondansetron）疊加。

**Q3.** 一名 Parkinson's disease 病人住院期間出現譫妄躁動，下列藥物最不建議？
- A. Quetiapine
- B. Clozapine
- C. Haloperidol
- D. 非藥物環境介入

> [!success]- 答案
> **C** — Haloperidol 強力 D2 阻斷會顯著惡化帕金森運動症狀，帕金森/Lewy body 病人應避免；此族群若需抗精神病藥首選 quetiapine 或 clozapine。
