# Chlorpromazine（氯丙嗪 / Wintermin 冬眠靈）

> [!abstract] 一句話：**低效價第一代（typical）抗精神病藥**，除阻斷 D2 外還強力阻斷 **α₁（低血壓）、H1（鎮靜）、M（抗膽鹼）** 受體。特色是**鎮靜作用最強、EPS 比 haloperidol 少**，但**自律神經副作用多**（姿勢性低血壓、抗膽鹼、光敏感）。臨床上是**頑固性呃逆（intractable hiccups）**的經典用藥。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **姿勢性低血壓（orthostatic hypotension）**：α₁ 阻斷所致，IM/IV 給藥後尤明顯 → 給藥後平躺、監測血壓，老人跌倒風險高。低血壓時**用純 α agonist（如 phenylephrine/norepinephrine），勿用 epinephrine**（β 效應被 α 阻斷放大反而更低血壓 → "epinephrine reversal"）
> 2. **QT 延長 / Torsades**：低效價 phenothiazine 亦延長 QT，查 QTc 與 K⁺/Mg²⁺，避免疊加其他 QT 藥
> 3. **降低癲癇閾值**：癲癇病史者慎用，高劑量易誘發抽搐
> 4. **抗膽鹼負荷**：口乾、便秘、尿瀦留、視力模糊、加重青光眼與譫妄；老人 anticholinergic burden 需警覺
> 5. **NMS 與 EPS**：仍可發生抗精神病藥惡性症候群；EPS 雖較 haloperidol 少但非零，遲發性運動障礙長期仍會出現

## 💊 適應症（治療哪些疾病）
- **思覺失調症 [[Schizophrenia(思覺失調症)]]**（第一代經典藥，鎮靜強適合躁動明顯者）
- **躁動 / 急性精神病激躁 [[Acute Agitation(急性躁動)]]**（鎮靜效果強）
- **頑固性呃逆 [[Intractable Hiccups(頑固性呃逆)]]**（經典適應症）
- **頑固性噁心嘔吐**（作用於 CTZ D2）、術前鎮靜
- **急性間歇性紫質症（porphyria）相關躁動、破傷風肌肉緊張輔助**（歷史/特殊用途）

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 精神病 / 躁動（PO） | **25–50 mg PO TID-QID 起**，依反應滴定，重症可加量（依院規/仿單） |
| 急性激躁（IM） | **25–50 mg IM**，可視需要重複；給藥後監測血壓 |
| 頑固性呃逆 | **25–50 mg PO/IM TID-QID**；無效且需靜脈者依院規以極慢速稀釋 IV + 補液監測血壓 |
| 噁心嘔吐 | 10–25 mg PO Q4–6H prn（依院規/仿單） |
| 老人 / 衰弱 | 從低劑量起（如 10–25 mg），密切監測姿勢性低血壓 |
- **腎功能**：主要經肝代謝，腎損一般不需大幅調整
- **肝功能**：肝代謝為主，重度肝損（含膽汁鬱積型肝炎風險）宜減量、謹慎
- **給藥途徑**：IV 需極慢速並嚴密監測血壓（低血壓風險高）；IM 注射部位刺激；避免皮膚接觸藥液（接觸性皮膚炎）

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：
  - **鎮靜 / 嗜睡**（H1 阻斷，最突出）
  - **姿勢性低血壓、心跳加快**（α₁ 阻斷）
  - **抗膽鹼**：口乾、便秘、尿瀦留、視力模糊
  - **光敏感（photosensitivity）**、皮膚色素沉著（長期灰藍色）
- **嚴重**：
  - **QT 延長 → Torsades**
  - **膽汁鬱積型肝炎（cholestatic jaundice）**（phenothiazine 特徵性）
  - **抗精神病藥惡性症候群 NMS**、遲發性運動障礙
  - **降低癲癇閾值 → 抽搐**
  - 罕見血球低下（agranulocytosis）、角膜/水晶體沉積
  - 高泌乳素血症

## 🚫 禁忌症
- 對 chlorpromazine 或 phenothiazine 類過敏
- **昏迷 / 嚴重中樞抑制**（酒精、巴比妥、鴉片急性中毒）
- **嚴重心血管疾病 / 明顯低血壓**
- **已知 QT 延長、未矯正低血鉀/低血鎂**
- 骨髓抑制、嚴重肝功能不全（相對）；狹角性青光眼、尿瀦留（抗膽鹼相對禁忌）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：血壓（含姿勢性）、心電圖 QTc + 電解質、肝功能、EPS/NMS 徵象、CBC；長期監測遲發性運動障礙與眼部沉積
- **交互作用**：
  - **降血壓藥、其他 α 阻斷劑**：加重低血壓
  - **Epinephrine**：α 被阻斷 → "epinephrine reversal" 反而更低血壓（低血壓救援用 phenylephrine/norepinephrine）
  - **其他延長 QT 藥、CNS 抑制劑（BZD/opioid/酒精）**：累加效應
  - **抗膽鹼藥**：抗膽鹼負荷疊加；**降低癲癇閾值藥物**併用增抽搐風險
- **孕哺**：孕期權衡使用（第三孕期暴露新生兒可能 EPS/戒斷、鎮靜）；哺乳會分泌，謹慎

### 自我測驗（選擇題，答案摺疊）

**Q1.** 使用 chlorpromazine 後嚴重低血壓，下列升壓藥何者最不恰當？
- A. Phenylephrine
- B. Norepinephrine
- C. Epinephrine
- D. 輸液擴容

> [!success]- 答案
> **C** — Chlorpromazine 阻斷 α 受體，epinephrine 的 β₂ 血管擴張作用失去拮抗會使血壓更低（epinephrine reversal）；應選純 α 效應的 phenylephrine 或 norepinephrine。

**Q2.** 關於 chlorpromazine 與 haloperidol 的比較，何者正確？
- A. Chlorpromazine 的 EPS 比 haloperidol 多
- B. Chlorpromazine 鎮靜與姿勢性低血壓較明顯，EPS 較少
- C. 兩者都不延長 QT
- D. Haloperidol 抗膽鹼作用較強

> [!success]- 答案
> **B** — 低效價的 chlorpromazine 鎮靜（H1）、姿勢性低血壓（α₁）、抗膽鹼（M）作用強而 EPS 較少；高效價 haloperidol 則 EPS 多、自律神經副作用少。兩者皆可延長 QT。

**Q3.** 一名末期病人頑固性呃逆連續數日，下列何者為經典有效藥物？
- A. Ondansetron
- B. Chlorpromazine
- C. Loperamide
- D. Omeprazole

> [!success]- 答案
> **B** — Chlorpromazine 是頑固性呃逆（intractable hiccups）的經典用藥；給藥時注意姿勢性低血壓與鎮靜。
