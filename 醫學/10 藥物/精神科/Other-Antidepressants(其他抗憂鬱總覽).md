# Other Antidepressants（其他抗憂鬱藥總覽）

> [!abstract] 一句話：本篇整合 4 支「非 SSRI/SNRI」的抗憂鬱與抗焦慮藥，**各藥機轉完全不同**：**Mirtazapine**＝α2 拮抗 + H1 拮抗（助眠、增食慾）；**Trazodone**＝5-HT2 拮抗 + 弱 SRI（低劑量助眠）；**Bupropion**＝NDRI（NE/DA 回收抑制，不致性功能障礙、不增重、**降癲癇閾值**）；**Buspirone**＝5-HT1A 部分促效（廣泛性焦慮、起效慢、不成癮）。值班挑藥要「對症選副作用」。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **Bupropion 降低癲癇閾值 → 癲癇病史、暴食症 / 神經性厭食症、酒精 / BZD 戒斷中「絕對禁用」**（電解質不穩 + 戒斷本身易癲癇，加乘）
> 2. **Trazodone → 陰莖異常勃起（priapism）是泌尿科急症**！勃起 > 4 小時需緊急處理，事前務必衛教男性病人立即就醫
> 3. **Mirtazapine 罕見但致命的顆粒球缺乏（agranulocytosis）**：用藥中出現發燒 / 喉嚨痛 / 感染徵象要查 CBC
> 4. **Mirtazapine「越低劑量越鎮靜」**：15mg 的抗組織胺鎮靜效應反而比 30–45mg 強（高劑量正腎上腺素效應抵銷部分鎮靜），失眠病人常用低劑量
> 5. **Buspirone 起效慢（約 2 週）不能當急性焦慮 PRN 用**；且與 MAOI 併用有高血壓危象風險、與強 CYP3A4 抑制劑（葡萄柚汁、ketoconazole）併用血中濃度飆升

## 💊 適應症
| 藥 | 主要定位 |
| --- | --- |
| **Mirtazapine** | 憂鬱症合併**失眠 / 食慾差 / 體重減輕 / 惡病質（cachexia）**；老人 / 癌症 / 化療噁心憂鬱者尤佳 |
| **Trazodone** | 低劑量（25–100mg）當**失眠助眠劑**（非管制、不成癮）；高劑量才作抗憂鬱 |
| **Bupropion** | 憂鬱症（**不想要性功能障礙 / 不想增重**者首選）、**戒菸輔助**（商品名 Zyban）、季節性情感障礙、成人 ADHD 輔助 |
| **Buspirone** | **廣泛性焦慮症 GAD**（長期、非急性）；SSRI 性功能障礙或性慾低下的**輔助加藥** |

## 📏 劑量 / 用法（成人）
| 藥 | 起始 / 常用劑量 | 備註 |
| --- | --- | --- |
| **Mirtazapine** | 15mg PO QHS 起，可增至 **15–45mg QHS** | 睡前給；低劑量更鎮靜 |
| **Trazodone**（助眠） | **25–100mg PO QHS** | 抗憂鬱需 150–300mg/day 但少單用 |
| **Bupropion**（SR） | 150mg QD → 可增至 **150mg BID**（SR）；XL 150–300mg QD | **單次≤200mg、日總量≤450mg**（超量增癲癇風險）⚠️依劑型 |
| **Buspirone** | 7.5mg BID 起，每 2–3 天調，常用 **15–30mg/day 分次** | 需規律服用非 PRN |
- **腎 / 肝功能不良**：Bupropion、Buspirone 於肝腎損傷需減量 ⚠️依院規 / 需查證
- **Bupropion 劑型很重要**：IR / SR / XL 各有單次上限，換算錯易誘發癲癇

## ⚠️ 副作用
- **Mirtazapine**：**嗜睡、體重增加、食慾大增**、口乾、頭暈；罕見 → **顆粒球缺乏**、肝酵素上升
- **Trazodone**：**鎮靜、姿勢性低血壓**（老人跌倒風險）、頭暈、口乾；嚴重 → **priapism 陰莖異常勃起**、QT 延長
- **Bupropion**：**失眠、躁動、口乾、頭痛、震顫、血壓上升**、食慾降低；嚴重 → **癲癇發作**（劑量相依）
- **Buspirone**：頭暈、頭痛、噁心、神經質；整體副作用少、**不鎮靜、不成癮、無戒斷**

## 🚫 禁忌症
- **共通**：與 MAOI 併用或 14 天內（血清素症候群 / 高血壓危象）
- **Bupropion**：**癲癇病史、暴食症 / 神經性厭食症、酒精或 BZD 急性戒斷**、正在停 MAOI
- **Trazodone**：近期心肌梗塞、已知 priapism 傾向（鐮狀細胞、多發性骨髓瘤、陰莖解剖異常）審慎
- **Mirtazapine**：對成分過敏；曾有血球低下病史者謹慎
- **Buspirone**：嚴重肝腎損傷、與 MAOI 併用

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：
  - Mirtazapine → 體重、CBC（發燒 / 感染時）、鎮靜程度
  - Trazodone → 血壓（姿勢性）、心電圖 QT、男性 priapism 衛教
  - Bupropion → 癲癇徵象、血壓、失眠 / 躁動
  - Buspirone → 焦慮改善（需 2 週評估）
- **交互作用**：
  - **全部忌 MAOI**；與其他血清素藥（SSRI/SNRI/tramadol/linezolid）併用注意**血清素症候群**
  - **Bupropion 抑制 CYP2D6** → 提高 TCA、部分 β-blocker、tamoxifen（降效）等濃度
  - **Buspirone 為 CYP3A4 受質** → 葡萄柚汁、ketoconazole、erythromycin 使其濃度上升；rifampin 使其失效
  - Trazodone + 其他降壓 / 中樞抑制劑 → 加重低血壓 / 鎮靜
- **孕哺**：Bupropion / Mirtazapine 孕期部分可用需權衡；Trazodone、Buspirone 資料較少，孕哺一律個案評估 ⚠️需查證

一位失眠 + 食慾差 + 體重下降的憂鬱病人，最適合哪支？::Mirtazapine（助眠、增進食慾與體重）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名 30 歲女性因憂鬱合併神經性暴食症求診，下列何種抗憂鬱藥「應避免」？
- A. Fluoxetine
- B. Mirtazapine
- C. Bupropion
- D. Sertraline

> [!success]- 答案
> **C** — Bupropion 會降低癲癇閾值，暴食症 / 神經性厭食症病人（常合併電解質不穩）使用會顯著增加癲癇風險，屬禁忌。此類病人 SSRI（如 fluoxetine）反而是首選。

**Q2.** 一名男性服用某抗憂鬱藥後出現持續數小時的疼痛性勃起，最可能的藥物是？
- A. Buspirone
- B. Bupropion
- C. Trazodone
- D. Mirtazapine

> [!success]- 答案
> **C** — Trazodone 可致 priapism（陰莖異常勃起），為泌尿科急症，需衛教病人立即就醫，避免海綿體缺血壞死。

**Q3.** 關於 Buspirone，下列敘述何者正確？
- A. 屬 BZD 類，會成癮
- B. 可作為急性恐慌發作的 PRN 用藥
- C. 為 5-HT1A 部分促效劑，起效慢但不鎮靜不成癮
- D. 與葡萄柚汁併用會使其濃度下降

> [!success]- 答案
> **C** — Buspirone 是 5-HT1A 部分促效劑，用於 GAD，需規律服用約 2 週起效，不鎮靜、不成癮、無戒斷；為 CYP3A4 受質，葡萄柚汁會「升高」而非降低其濃度。
