# Norepinephrine（正腎上腺素 / Levophed）

> [!abstract] 一句話：**敗血性休克第一線升壓劑**。強效 α1（血管收縮 → 升高 SVR、拉高 MAP）+ 中度 β1（正性肌力/心率），α 效應為主。因 α1 為主，升壓同時反射性心搏可能減緩，較不易誘發心律不整（相對 dopamine/adrenaline）。**必須以 infusion pump 持續泵注、首選中央靜脈導管（central line）**，外滲會致組織壞死。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **一律用 infusion pump 持續泵注，絕不 IV push / bolus**：半衰期極短（~1–2 分鐘），需連續滴定。
> 2. **首選 central line 給藥**；緊急無中央路徑時可短暫用大口徑近端周邊 IV，但需嚴密監測外滲並儘速換 central。
> 3. **外滲（extravasation）= 急症**：α1 強力血管收縮致皮膚缺血壞死。處置：立即停該 line、抽回殘液、局部浸潤注射 **phentolamine**（α 拮抗劑，常 5–10 mg 稀釋於 NS，劑量依院規需查證）逆轉血管收縮。
> 4. **先補足容積再升壓**：低血容量休克單靠升壓劑掩蓋灌流不足；敗血症建議先/同時積極輸液復甦。
> 5. **監測遠端灌流**：肢端／腸系膜／腎灌流可能因過度血管收縮受損；持續看尿量、乳酸、肢端溫度。

## 💊 適應症（治療哪些疾病）
- **敗血性休克 [[Septic Shock(敗血性休克)]]**（Surviving Sepsis：**第一線首選升壓劑**，目標 MAP ≥ 65 mmHg）
- **各類分佈性 / 血管張力低下休克**（部分神經性休克情境）
- **嚴重低血壓 [[Hypotension(低血壓)]]**（容積復甦後仍不足時）
- **輔助**：心因性休克合併嚴重低血壓時，常與正性肌力藥（如 dobutamine）併用（依血流動力學型態，需查證）

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 敗血性休克起始 | **0.05–0.1 mcg/kg/min**（或固定劑量起始 ~5 mcg/min）持續泵注 |
| 滴定範圍 | 依 MAP 反應每數分鐘上調，常見 **0.05–0.5 mcg/kg/min**；難治性可更高（上限依院規，需查證） |
| 目標 | MAP ≥ 65 mmHg（敗血症），依個別病人調整 |

- **給藥途徑**：中央靜脈導管 + infusion pump 連續泵注；緊急短暫可近端大周邊 IV（嚴密防外滲）。
- **稀釋濃度**：常配 **4 mg 或 8 mg 加入 250 mL D5W / NS**（具體濃度依院規/藥局標準配方，需查證）。建議用 **含葡萄糖溶液（D5W）**稀釋（酸性環境較穩定）。
- **腎/肝功能**：一般不需依腎肝調整劑量（依血流動力學反應滴定）。
- **停藥**：逐步減量（tapering）避免反彈性低血壓，不可驟停。
- **劑量單位陷阱**：可用 mcg/min（固定）或 mcg/kg/min（體重校正）兩種表達，交班時務必講清楚哪一種避免十倍誤差。

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：反射性心搏減緩、頭痛、焦慮、四肢末梢冰冷／蒼白（血管收縮）。
- **嚴重**：
  - **外滲致組織壞死**（皮膚脫落、潰瘍）
  - **周邊 / 腸系膜 / 指端缺血**（過度血管收縮，嚴重可致肢端壞疽）
  - **心律不整**（相對其他兒茶酚胺較少，但仍可能）
  - **反射性心搏過緩**（α1 升壓 → baroreceptor 反射）
  - 器官灌流不足（腎／腸）若劑量過高

## 🚫 禁忌症
- 對本品成分過敏。
- **未矯正的低血容量**（相對禁忌 — 應先/同時容積復甦，不可單以升壓劑掩蓋）。
- 周邊血管栓塞 / 嚴重末梢缺血病人審慎（可能加重）。
- （傳統警語）與環丙烷/鹵化麻醉劑併用增加心律不整風險（現代少見，需查證）。

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：連續動脈血壓（建議 arterial line）、心率/心電圖、尿量、乳酸、末梢灌流／注射部位（外滲）、MAP 目標達成。
- **交互作用**：
  - **MAO 抑制劑 / 三環抗憂鬱劑（TCA）**：加強升壓效應 → 嚴重高血壓危象，需大幅減量並嚴密監測。
  - **非選擇性 β 阻斷劑**：α 效應無對抗 → 可能加重反射性高血壓/心搏過緩。
  - 其他兒茶酚胺／升壓劑併用時注意疊加。
- **孕哺**：可能減少子宮胎盤血流；孕期僅在利益大於風險時使用（分級需查證）。

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名敗血性休克病人已完成積極輸液但 MAP 仍 55 mmHg。下列何者是首選升壓劑？
- A. Dopamine
- B. Norepinephrine
- C. Phenylephrine
- D. Dobutamine

> [!success]- 答案
> **B** — Surviving Sepsis 建議 norepinephrine 為敗血性休克第一線升壓劑，目標 MAP ≥ 65 mmHg。Dopamine 心律不整風險較高，dobutamine 是正性肌力藥非首選升壓劑。

**Q2.** 使用 norepinephrine 經周邊 IV 給藥發生外滲，最適當的緊急處置是？
- A. 熱敷加速吸收
- B. 局部注射 phentolamine 並停止該輸注管路
- C. 加快輸注速度
- D. 靜脈注射 calcium

> [!success]- 答案
> **B** — 立即停該 line、抽回殘液，並局部浸潤注射 phentolamine（α 拮抗劑）逆轉強力血管收縮，防止組織缺血壞死。

**Q3.** 關於 norepinephrine 的給藥方式，何者正確？
- A. 可 IV push 快速升壓
- B. 半衰期長，可 QD 給藥
- C. 需 infusion pump 連續泵注、首選中央靜脈導管
- D. 口服吸收良好

> [!success]- 答案
> **C** — Norepinephrine 半衰期極短（約 1–2 分鐘），必須以 infusion pump 連續泵注並滴定，且首選中央靜脈導管給藥以避免外滲壞死。
