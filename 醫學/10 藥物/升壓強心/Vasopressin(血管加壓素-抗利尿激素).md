# Vasopressin（血管加壓素 / 抗利尿激素 ADH）

> [!abstract] 一句話：**Vasopressin 就是內源性抗利尿激素（ADH）**，作用於血管平滑肌 **V1 受體 → 強力收縮血管升壓**（不經腎上腺素受體，故心律影響小），作用於腎集尿管 **V2 受體 → 抗利尿保水**。臨床三大用途：**① 敗血性休克第二線升壓（加在 norepinephrine 之上）② 食道靜脈瘤出血（內臟血管收縮止血）③ 中樞性尿崩症（補充 ADH）**。

> [!danger] 🚨 值班紅旗（用藥前先想）
> 1. **末梢 / 內臟缺血**：強力血管收縮可致**指趾缺血壞死、腸繫膜缺血、心肌缺血**；周邊靜脈外滲會組織壞死 → **建議中央靜脈給藥**，密切看四肢與腹部。
> 2. **敗血性休克用固定低劑量、不 titrate**：常規 0.03 U/min 固定加在 norepinephrine 上以減少兒茶酚胺用量；**不像 norepinephrine 依血壓上下調**，過量升壓致缺血。
> 3. **低血鈉（V2 抗利尿）**：留水稀釋 → 稀釋性低血鈉，尤其非休克長時間使用或尿崩治療；監測鈉。
> 4. **心肌缺血 / 冠心病、心衰慎用**：後負荷↑ + 冠脈可能收縮 → 心肌氧供需失衡。
> 5. **食道靜脈瘤出血現多用 terlipressin / octreotide**：純 vasopressin 全身缺血副作用大，若用常合併硝酸甘油減少心臟缺血；現代多首選 terlipressin（較選擇性、副作用少）或體抑素類似物。

## 💊 適應症
- **敗血性休克**：第二線升壓，加在 norepinephrine 之上（減少兒茶酚胺劑量、拉抬 MAP）
- **食道 / 胃靜脈瘤出血**：收縮內臟血管降門脈壓止血（現多用 terlipressin / octreotide 替代）
- **中樞性（腦下垂體性）尿崩症 Diabetes Insipidus**：補充缺乏的 ADH（門診多用 desmopressin/DDAVP）
- **心跳停止（歷史）**：曾為 ACLS 選項，2020 指引已移除（不優於 epinephrine）
- 其他：術中血管麻痺（vasoplegia）、對兒茶酚胺反應差的分布性休克

## 📏 劑量 / 用法（成人 · 依院規）
| 用途 | 劑量 |
| --- | --- |
| **敗血性休克** | **0.03 U/min IV 固定輸注**（加在 norepinephrine 之上；一般不 titrate，部分可到 0.04 U/min） |
| 食道靜脈瘤出血 | 0.2–0.4 U/min IV 輸注（可加至 0.8），常合併 nitroglycerin；⚠️現多改 terlipressin |
| 中樞性尿崩（急性） | 依尿量/鈉滴定，或改用 desmopressin |
- **給藥途徑**：升壓用建議**中央靜脈**（外滲致壞死）。
- **不依血壓大幅 titrate**（敗血性休克固定劑量）；停藥宜漸減觀察血壓。
- ⚠️精確劑量與品牌濃度依院規/加護病房 protocol。

## ⚠️ 副作用
- **缺血**：指趾/肢體缺血壞死、腸繫膜缺血、心肌缺血/心絞痛、腦缺血
- **心血管**：血壓過高、心搏過緩（反射性）、心律不整、心輸出量下降（後負荷↑）
- **低血鈉**（V2 抗利尿留水）、水中毒
- 皮膚蒼白、腹絞痛、噁心（內臟收縮）
- 外滲組織壞死

## 🚫 禁忌症 / 慎用
- 對 vasopressin 過敏
- **慎用**：冠狀動脈疾病 / 心肌缺血、周邊血管疾病、腸繫膜缺血病史、嚴重心衰、慢性腎病水分滯留、低血鈉
- 妊娠（催產可能顧慮，救命情境權衡）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：MAP / 血壓、心律、四肢末梢灌流與皮膚色、腹部症狀（腸缺血）、尿量、血鈉、乳酸、心肌缺血跡象（EKG/troponin）。
- **交互作用**：與其他升壓劑合用加重缺血；與 ganglionic blocker 可增強升壓；某些藥（carbamazepine、TCA）增強抗利尿；lithium、demeclocycline 減弱抗利尿。
- **孕哺**：孕期救命情境使用需權衡（可能影響子宮）；哺乳資料有限。

### 自我測驗（選擇題，答案摺疊）

**Q1.** 敗血性休克病人已用 norepinephrine 但 MAP 仍不足，加上 vasopressin 的典型用法是？
- A. 依血壓每分鐘上下調整劑量
- B. 固定低劑量 0.03 U/min 加在 norepinephrine 上以拉 MAP、減兒茶酚胺
- C. 大劑量 bolus 快速推注
- D. 周邊靜脈長期輸注即可

> [!success]- 答案
> **B** — 敗血性休克 vasopressin 為第二線，常規固定 0.03 U/min 加在 norepinephrine 之上，一般不大幅 titrate；且升壓應中央靜脈給藥（D 錯，外滲壞死）。

**Q2.** 使用 vasopressin 升壓時，下列哪組是最需監測的併發症？
- A. 高血鉀與高血糖
- B. 末梢/腸繫膜/心肌缺血與低血鈉
- C. 過敏性休克與發燒
- D. 高血鈣與甲亢

> [!success]- 答案
> **B** — V1 強力血管收縮易致指趾、腸繫膜、心肌缺血；V2 抗利尿留水致低血鈉，皆須密切監測。
