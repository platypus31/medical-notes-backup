# Milrinone（米力農）

> [!abstract] 一句話：**Milrinone 抑制心肌與血管的 PDE3 → cAMP 上升**，在心肌 → **增強收縮力（正性肌力）**，在血管 → **擴張動靜脈（降前後負荷）**，故稱 **inodilator（強心 + 血管擴張）**。**作用不經 β 受體**，對長期 β-blocker 治療或 β 受體下調的心衰病人仍有效。臨床用於**急性失代償心衰、右心衰 / 肺高壓、心臟術後低心輸出**。**經腎排出、半衰期較長（約 2.3 h，腎衰更長）**是值班要記的重點。

> [!danger] 🚨 值班紅旗（用藥前先想）
> 1. **低血壓**：血管擴張作用常致低血壓，尤其容積不足者；**低血壓/低血容量時慎用或先補液**，可能需併升壓劑（norepinephrine）維持灌流。
> 2. **心律不整**：cAMP 上升致心室（及心房）心律不整增加，長期口服曾因增加死亡率而不用於慢性治療（PROMISE trial）。
> 3. **腎排出、半衰期長**：**腎功能不良會蓄積**（半衰期延長），需依 CrCl 減量；停藥後作用退得慢（不像多巴酚丁胺可快速停）。
> 4. **不宜與 dobutamine 混淆的差別**：milrinone 不經 β 受體（β-blocker 病人仍有效），但**低血壓與心律不整、蓄積風險**比 dobutamine 更需警覺。
> 5. **loading dose 常致血壓驟降**：許多單位在血流動力不穩時**省略 bolus、直接維持輸注**以避免低血壓。

## 💊 適應症
- **急性失代償性心衰**（尤其需要正性肌力 + 降後負荷者）
- **右心衰竭 / 肺動脈高壓**（血管擴張降肺血管阻力，對右心特別有利）
- **心臟手術後低心輸出症候群**（尤其接受 β-blocker、β 受體下調者）
- 心臟移植前橋接、末期心衰居家/安寧靜脈強心（選擇性）

## 📏 劑量 / 用法（成人 · 依院規 / 腎功能）
- **Loading（選用）**：50 mcg/kg IV over 10 分鐘 — ⚠️血流動力不穩者常**省略**以免低血壓。
- **維持輸注**：0.375–0.75 mcg/kg/min IV，依血壓 / 心輸出 / 反應調整。
- **腎功能不良需減量**（milrinone 主要經腎排出，CrCl 下降時延長半衰期、降低維持速率）。
- ⚠️精確 loading/維持與腎調整依院規 / 加護病房 protocol。

## ⚠️ 副作用
- **低血壓**（血管擴張，最常見須留意）
- **心律不整**：心室性早搏 / 室速、心房顫動 / 心搏過速
- **心肌缺血 / 心絞痛**（冠心病者）
- 頭痛、低血鉀、血小板減少（較少見）
- 長期口服增加死亡率（故僅短期靜脈用）

## 🚫 禁忌症 / 慎用
- 對 milrinone 過敏
- **嚴重低血壓 / 低血容量未矯正**
- 嚴重阻塞性瓣膜疾病（如重度主動脈瓣狹窄 / 肥厚性阻塞型心肌病）— 增強收縮/降後負荷可惡化血流動力
- **腎功能不良**需減量並密切監測（蓄積）
- 急性心肌梗塞後慎用（心律不整/缺血）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：連續血壓 / MAP、心電圖（心律不整）、心輸出 / 心臟指數（有 CO 監測時）、腎功能、電解質（鉀）、容積狀態、末梢灌流 / 乳酸。
- **交互作用**：與其他血管擴張劑 / 降壓藥加重低血壓；與其他正性肌力 / 促心律不整藥合用增加心律不整；常與升壓劑（norepinephrine）併用以維持血壓。
- **孕哺**：孕期資料有限，救命情境權衡使用；哺乳資料不足。

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名長期服用 carvedilol（β-blocker）的失代償心衰病人需要正性肌力支持。相較 dobutamine，選 milrinone 的主要理由是？
- A. Milrinone 沒有低血壓風險
- B. Milrinone 作用不經 β 受體，β-blocker 病人仍有效
- C. Milrinone 半衰期短好停藥
- D. Milrinone 不會引起心律不整

> [!success]- 答案
> **B** — Milrinone 抑 PDE3 作用在 β 受體下游，β-blocker 或 β 受體下調病人仍有效；但它仍有低血壓、心律不整風險，且半衰期長經腎排出（A/C/D 皆錯）。

**Q2.** 使用 milrinone 時，下列何者最正確？
- A. 腎衰竭病人不需調整劑量
- B. 低血壓/低血容量是理想適應症
- C. 腎功能不良會蓄積、半衰期延長，需減量並可能併用升壓劑維持血壓
- D. 可作為慢性心衰長期口服用藥

> [!success]- 答案
> **C** — Milrinone 主要經腎排出，腎衰蓄積需減量；其血管擴張常致低血壓（B 錯），故常併升壓劑；長期口服增加死亡率不用於慢性治療（D 錯）。
