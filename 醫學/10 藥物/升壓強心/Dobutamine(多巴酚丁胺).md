# Dobutamine（多巴酚丁胺 / Dobutrex）

> [!abstract] 一句話：**β1 為主的正性肌力藥（inotrope）**，主要增強心肌收縮力、提高心輸出量（CO），用於**心因性休克、失代償性心衰**等「幫浦無力」狀態。β2 輕度血管擴張 → 常使 SVR/血壓略降，故**低血壓病人常需併用升壓劑（如 norepinephrine）**。半衰期極短（~2 分鐘），須 infusion pump 連續泵注滴定。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **不是升壓劑**：dobutamine 增加的是「心收縮力／CO」，β2 擴血管可能讓血壓更低。**收縮壓過低（如 SBP < 90）者不宜單用**，需先/併用升壓劑（norepinephrine）撐住 MAP
> 2. **增加心肌耗氧 + 致心律不整**：可誘發或惡化心搏過速、心房顫動、心室性心律不整；有心肌缺血/急性冠心症者慎用（可能擴大梗塞）
> 3. **一律 infusion pump 連續泵注、絕不 IV push**：半衰期 ~2 分鐘需連續滴定
> 4. **低血容量要先矯正**：容積不足時給 inotrope 效果差且加重心搏過速
> 5. **長期/持續使用會產生耐受性（β 受體 down-regulation）**；且觀察性資料提示正性肌力藥長期使用可能增加死亡率，僅用於必要的血流動力學支持

## 💊 適應症（治療哪些疾病）
- **心因性休克 [[Cardiogenic Shock(心因性休克)]]**（低 CO、灌流不足；常在 MAP 由升壓劑撐住的前提下加 dobutamine 提升 CO）
- **急性失代償性心衰竭 [[Acute Decompensated Heart Failure(急性失代償性心衰)]]**（低輸出、末梢冰冷／灌流差的「cold」表型短期支持）
- **心臟壓力測試（dobutamine stress echo）**——藥物負荷評估冠心病（非急救用途）
- **輔助**：敗血症合併心肌抑制、CO 偏低時，於升壓劑之上加 inotrope（依血流動力學型態，需查證）

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 起始 | **2–5 mcg/kg/min** 持續泵注 |
| 常用滴定範圍 | **2–20 mcg/kg/min**，依 CO／灌流／血壓與心律反應調整 |
| 上限 | 一般不超過 **20 mcg/kg/min**（更高心律不整/耗氧風險劇增，依院規） |
- **給藥途徑**：中央或大口徑周邊 IV + infusion pump 連續泵注（外滲刺激性相對 norepinephrine 小，但仍建議妥善靜脈）。
- **稀釋濃度**：常配於 D5W / NS（具體濃度依院規/藥局標準配方，需查證）。
- **腎/肝功能**：一般不需依腎肝調整，依血流動力學反應滴定。
- **停藥**：逐步減量避免反彈性低輸出，不宜驟停。
- **劑量單位**：以 mcg/kg/min（體重校正）表達，交班務必講清楚。

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：心搏過速、心悸、血壓變動（可能升亦可能降）、頭痛、噁心、震顫。
- **嚴重**：
  - **心律不整**（心房顫動快速心室反應、室性早搏、室性心搏過速）
  - **心肌缺血惡化 / 胸痛**（增加耗氧，冠心病者可擴大梗塞）
  - **明顯低血壓**（β2 血管擴張，尤其低血容量者）
  - 高劑量長期使用之耐受性與潛在死亡率風險
  - 罕見嗜酸性球性心肌炎、過敏（含亞硫酸鹽敏感者）

## 🚫 禁忌症
- 對 dobutamine 或亞硫酸鹽（sulfite）過敏。
- **肥厚型阻塞性心肌病（HOCM / IHSS）** — 增強收縮力加重左室流出道阻塞。
- 未矯正的嚴重低血容量（相對禁忌，應先容積復甦）。
- 嚴重未控制的快速性心律不整（相對禁忌，恐惡化）。

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：連續心電圖/心率、血壓（建議 arterial line）、CO/灌流指標（尿量、乳酸、末梢溫度）、有創血流動力學（如有 Swan-Ganz）。
- **交互作用**：
  - **非選擇性 β 阻斷劑**：拮抗 dobutamine 的 β 效應 → 療效下降（α 效應相對凸顯可能升壓）。
  - **其他兒茶酚胺／升壓劑**：疊加心律不整與血流動力學效應。
  - MAO 抑制劑：可能加強效應（需監測）。
- **孕哺**：資料有限，孕期僅在利益大於風險時使用（分級需查證）。

## 🔗 相關
- 疾病：[[Cardiogenic Shock(心因性休克)]]、[[Acute Decompensated Heart Failure(急性失代償性心衰)]]
- 同類 / 對照：[[Milrinone(米力農)]]（PDE3 抑制 inodilator，不依賴 β 受體、可用於已用 β-blocker 者）、[[Norepinephrine(正腎上腺素)]]（升壓劑，低血壓時併用撐 MAP）、[[Dopamine(多巴胺)]]（劑量依賴、心律不整較多）、[[Epinephrine(腎上腺素)]]
- 對照概念：inotrope（強心）vs vasopressor（升壓）的區別；心因性休克常「norepinephrine 撐壓 + dobutamine 提 CO」

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名急性心衰病人 CO 低、四肢冰冷、SBP 85 mmHg，欲加 dobutamine 提升心輸出。下列處置何者最正確？
- A. 直接單用 dobutamine 快速滴高即可
- B. 因 SBP 偏低，宜同時用升壓劑（如 norepinephrine）撐住 MAP，再用 dobutamine 提 CO
- C. Dobutamine 可 IV push 加速起效
- D. 併用可矯正低血容量無需輸液

> [!success]- 答案
> **B** — Dobutamine β2 擴血管可能讓血壓更低；SBP 偏低時應併用升壓劑維持 MAP，dobutamine 專責提升 CO。半衰期短需連續泵注，不可 push；仍應評估容積狀態。

**Q2.** 使用 dobutamine 時最需監測與警覺的併發症是？
- A. 低血糖
- B. 心搏過速與心律不整、心肌缺血惡化
- C. 高血鉀
- D. 便秘

> [!success]- 答案
> **B** — Dobutamine 增加心肌耗氧，易誘發心搏過速、房顫快速心室反應與室性心律不整，冠心病者可惡化缺血，須連續 ECG 監測。

**Q3.** 下列何者是 dobutamine 的禁忌／應避免情況？
- A. 心因性休克
- B. 急性失代償性心衰
- C. 肥厚型阻塞性心肌病（HOCM）
- D. 低心輸出狀態

> [!success]- 答案
> **C** — HOCM/IHSS 病人使用 dobutamine 會增強收縮力、加重左室流出道阻塞，屬禁忌。其餘為其常見適應情境。
