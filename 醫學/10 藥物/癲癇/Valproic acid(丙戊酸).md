# Valproic acid（丙戊酸 / Depakine）

> [!abstract] 一句話：**最廣效的傳統抗癲癇藥**，多機轉（增強 GABA、阻斷 Na channel、抑制 T 型 Ca channel），對局部、全身強直陣攣、**absence、肌陣攣**都有效，也用於**雙相情感障礙與偏頭痛預防**。值班四大警訊：**① 肝毒性 + 高血氨腦病**（可在正常肝功能下發生）、**② 血小板低下**、**③ 胰臟炎**、**④ 強致畸（神經管缺損，育齡女性/孕婦禁用）**。治療濃度窗 **50–100 µg/mL**。與 **carbapenem 併用會使濃度暴跌**，是重要交互作用陷阱。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **高血氨腦病（hyperammonemic encephalopathy）**：意識改變/嗜睡/意識混亂時**即使肝功能與 valproate 濃度正常也要驗血氨**，處置為停藥 ± **L-carnitine**
> 2. **肝毒性**：可致命性猛爆性肝衰竭，**<2 歲、多重抗癲癇藥、粒線體疾病（POLG 突變）** 風險最高——**POLG 相關疾病為絕對禁忌**
> 3. **育齡女性/孕婦禁用（除非無替代）**：**神經管缺損（脊柱裂）**、顱面/心臟缺陷、致認知與自閉風險上升；開藥前確認未懷孕並落實避孕
> 4. **血小板低下 + 出血傾向**：手術/侵入性處置前查 CBC 與凝血；也可致 vWF 功能異常
> 5. **急性胰臟炎**：新發腹痛/嘔吐要驗 amylase/lipase
> 6. **與 carbapenem（meropenem/imipenem/ertapenem）併用 → valproate 濃度暴跌可能致癲癇復發**，應避免併用

## 💊 適應症（治療哪些疾病）
- **廣效抗癲癇——局部、全身強直陣攣 [[Generalized Tonic-Clonic Seizure(全身強直陣攣發作)]]、失神發作 [[Absence Seizure(失神發作)]]、肌陣攣 [[Myoclonic Seizure(肌陣攣發作)]]**（全面型癲癇首選之一）
- **癲癇重積狀態 [[Status Epilepticus(癲癇重積)]]**（IV，benzodiazepine 後第二線）
- **雙相情感障礙 [[Bipolar Disorder(雙相情感障礙)]]**（急性躁期與維持）
- **偏頭痛預防 [[Migraine(偏頭痛)]]**

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 癲癇起始 | **10–15 mg/kg/day** 分次，每週增 5–10 mg/kg 至反應/濃度 |
| 一般維持 | **常用達 1000–3000 mg/day**（依濃度與反應） |
| 癲癇重積 loading（IV） | **20–40 mg/kg IV**（依院規/仿單） |
| 目標濃度 | **50–100 µg/mL**（部分躁症/難治可至更高，需查證） |
- **腎功能**：腎排泄非主要途徑，一般不需大幅依腎調整；但透析與低白蛋白會影響游離型判讀
- **肝**：主要肝代謝，**肝功能不良為禁忌/慎用**，用前後追肝功能
- 提醒：高度蛋白結合，低白蛋白病人游離型上升，判讀總濃度要留意

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：**體重增加、噁心/腸胃不適、震顫、掉髮、鎮靜、水腫**
- **嚴重**：
  - **肝毒性**（可猛爆性，<2 歲/多重用藥/POLG 突變高危）
  - **高血氨腦病**（可在肝功能正常時發生）
  - **急性胰臟炎**（可出血壞死致命）
  - **血小板低下、凝血異常、出血傾向**
  - **致畸**（神經管缺損等）
  - 罕見 DRESS、多囊卵巢/高雄性素、骨質流失

## 🚫 禁忌症
- **肝功能明顯異常 / 肝病史**
- **粒線體疾病（POLG 基因突變相關，如 Alpers-Huttenlocher）**——絕對禁忌
- **尿素循環障礙（urea cycle disorders）**
- **懷孕（用於癲癇/偏頭痛時；除非其他藥皆無效且落實避孕）**
- 已知胰臟炎病史者慎用；對 valproate 過敏

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：**血中濃度 50–100 µg/mL**、**肝功能（起始頭 6 個月密集）**、**CBC/血小板/凝血**、意識改變時**驗血氨**、amylase/lipase（疑胰臟炎）
- **交互作用**：
  - **Carbapenem（meropenem/imipenem/ertapenem）→ valproate 濃度暴跌**，避免併用
  - **抑制多種代謝酵素（酵素抑制劑）+ 高蛋白結合置換** → 升高 **lamotrigine（皮疹/SJS 風險大增，需減量緩加）、phenobarbital、游離 phenytoin、warfarin** 等
  - 與 aspirin 併用置換蛋白結合 → 游離 valproate 上升
- **孕哺**：**抗癲癇藥中致畸性最高**——神經管缺損、顱面/心臟缺陷、認知障礙與自閉風險；育齡女性應避免，必要時補**高劑量葉酸**並落實避孕。哺乳分泌量少，一般認為相對可（**個案依專科評估**）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 服用 valproate 的病人出現嗜睡、意識混亂，肝功能與 valproate 濃度皆正常。下一步最關鍵檢查？
- A. 加量 valproate
- B. 驗血氨（懷疑高血氨腦病）
- C. 腦部 MRI 後再說
- D. 只需觀察

> [!success]- 答案
> **B** — Valproate 可致高血氨腦病，且可在肝功能與藥物濃度正常下發生；應驗血氨，處置為停藥並考慮 L-carnitine。

**Q2.** 26 歲女性癲癇病人計畫懷孕，目前用 valproate 控制良好。最適當處置？
- A. 維持 valproate 不變
- B. 因強致畸（神經管缺損）應與神經科討論換藥（如 levetiracetam/lamotrigine）並補葉酸
- C. 只需減半劑量
- D. 懷孕後再停藥即可

> [!success]- 答案
> **B** — Valproate 致畸性最高（神經管缺損、認知/自閉風險），育齡計畫懷孕者應在孕前換成致畸較低的藥並補高劑量葉酸，不可等懷孕後才處理。

**Q3.** 加護病房病人同時使用 valproate 控制癲癇，因感染需加抗生素。下列何者會使 valproate 濃度大幅下降？
- A. Vancomycin
- B. Meropenem（carbapenem）
- C. Ceftriaxone
- D. Azithromycin

> [!success]- 答案
> **B** — Carbapenem（meropenem/imipenem/ertapenem）會使 valproate 血中濃度暴跌，可能導致癲癇復發，應避免併用或改其他抗癲癇策略。
