# Gabapentin / Pregabalin（加巴噴丁 / 普瑞巴林）

> [!abstract] 一句話：**結合電壓門控鈣通道 α2δ 次單元 → 減少突觸前興奮性神經傳導物（glutamate / substance P / CGRP）釋放**，用於神經病理性疼痛與部分性癲癇輔助。**名稱雖含「GABA」但並不直接作用於 GABA 受體**（命名誤導）。**幾乎全經腎排泄、不經肝代謝、不誘導 CYP**，故腎功能不良需依 CrCl 大幅減量。Pregabalin 效價高、生體可用率高且吸收線性，現已列管制藥。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **腎功能不良必須依 CrCl 大幅減量**：兩者幾乎 100% 經腎以原形排出，不減量會蓄積 → 過度鎮靜、意識混亂、肌陣攣。洗腎病人透析後需補充劑量
> 2. **鎮靜 / 頭暈 / 嗜睡 / 運動失調 → 老人跌倒風險**：起始要低劑量慢加（start low, go slow），住院老人開了要留意夜間跌倒
> 3. **與鴉片類（opioids）併用 → 呼吸抑制、死亡風險上升**：FDA 已警示；兩者皆有中樞抑制，尤其 pregabalin。合併 benzodiazepine 亦然
> 4. **濫用 / 成癮潛能**：Pregabalin 在台灣及多國已列管制藥（第 4 級），gabapentin 亦有濫用報告，尤其有物質使用障礙病史者
> 5. **突然停藥可能誘發癲癇或戒斷症狀**（焦慮、失眠、盜汗、噁心）：即使只用於止痛也應逐步減量停藥
> 6. **周邊水腫 / 體重增加**：易誤判為心衰惡化；也可能加重既有水腫

## 💊 適應症
- **神經病理性疼痛**：糖尿病周邊神經病變痛、帶狀疱疹後神經痛（postherpetic neuralgia）
- **纖維肌痛症（fibromyalgia）**：pregabalin 有正式適應症
- **部分性（局部起始）癲癇之輔助治療**（add-on，非單藥首選）
- **廣泛性焦慮症 GAD**：pregabalin 在歐洲等部分地區為適應症
- **下肢不寧症候群（Restless Legs Syndrome）**：gabapentin enacarbil 有適應症
- （臨床亦常 off-label 用於慢性咳嗽、術後止痛輔助等）

## 📏 劑量 / 用法（成人）
### Gabapentin（腎功能正常）
| 用途 | 劑量 |
| --- | --- |
| 神經病理性疼痛起始 | 300 mg 睡前第 1 天 → 300 mg BID 第 2 天 → 300 mg TID 第 3 天，之後依反應調 |
| 常用有效範圍 | **900–3600 mg/day 分 3 次** |
| 帶狀疱疹後神經痛 | 逐步加至 1800 mg/day 分 3 次 |

### Pregabalin（腎功能正常）
| 用途 | 劑量 |
| --- | --- |
| 神經病理性疼痛 / 纖維肌痛起始 | **75 mg BID（或 50 mg TID）** |
| 常用範圍 | **150–600 mg/day 分 2–3 次**（1 週內可加至 300 mg/day） |

### 腎功能調整（關鍵，依 CrCl）⚠️依院規/需查證
| CrCl (mL/min) | Gabapentin | Pregabalin |
| --- | --- | --- |
| ≥60 | 900–3600 mg/day 分 3 | 150–600 mg/day 分 2–3 |
| 30–59 | 400–1400 mg/day 分 2 | 75–300 mg/day 分 2–3 |
| 15–29 | 200–700 mg/day QD | 25–150 mg/day 分 1–2 |
| <15 | 100–300 mg/day QD | 25–75 mg/day QD |
| 血液透析 | 洗腎後補充單劑 | 洗腎後補充單劑 |

- **重點對照**：Pregabalin 生體可用率 ≥90% 且吸收呈線性、效價約為 gabapentin 的數倍（150 mg pregabalin ≈ 900–1200 mg gabapentin 概念）；gabapentin 吸收具飽和性（劑量↑吸收比例↓）故高劑量需分次

## ⚠️ 副作用
- **常見**：頭暈、嗜睡、鎮靜、運動失調、周邊水腫、體重增加、視力模糊、口乾、便秘
- **嚴重 / 需警覺**：
  - **與 opioid / CNS 抑制劑併用之呼吸抑制**（可致命）
  - 過度鎮靜 / 意識混亂（尤其腎功能不良蓄積）、肌陣攣
  - 濫用、依賴、戒斷（突停）
  - 罕見：血管性水腫、DRESS（藥物疹伴嗜酸性球增多及全身症狀）、橫紋肌溶解、情緒/行為改變（少數自殺意念警訊）

## 🚫 禁忌症
- 對 gabapentin / pregabalin 或其成分過敏
- （相對）嚴重呼吸功能不全 + 併用鴉片類 / 鎮靜劑者需極謹慎
- 半乳糖不耐 / 特定賦形劑不耐（依製劑）
- 無絕對「器官功能」硬性禁忌，但腎衰竭必須減量而非禁用

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：腎功能（CrCl / Cr）以定劑量、鎮靜程度與跌倒風險、水腫與體重、情緒行為變化、疼痛與癲癇控制反應；長期使用評估濫用/依賴
- **交互作用**：
  - **Opioids / benzodiazepines / 酒精 / 其他 CNS 抑制劑**：加重鎮靜與呼吸抑制（最重要）
  - **含鋁 / 鎂制酸劑**：降低 gabapentin 吸收 → 兩者間隔至少 2 小時
  - 不經 CYP450 代謝、不誘導/抑制酵素 → **少見藥物代謝性交互作用**（相對「乾淨」，這也是其優點）
- **孕哺**：孕期資料有限（可能增加畸形風險，權衡使用）；會分泌至乳汁，哺乳需評估

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名 CrCl 20 mL/min 的糖尿病神經痛病人，欲使用 gabapentin。下列處置何者最正確？
- A. 直接給常規 900 mg TID
- B. 依腎功能大幅減量（如 200–700 mg/day，QD 給予）
- C. Gabapentin 經肝代謝，腎功能不影響劑量
- D. 腎功能不良為絕對禁忌，完全不能用

> [!success]- 答案
> **B** — Gabapentin 幾乎全經腎原形排泄，CrCl 15–29 應大幅減量（約 200–700 mg/day，QD），並非禁用，也非不需調整。不減量會蓄積導致過度鎮靜與肌陣攣。

**Q2.** 關於 gabapentinoids 的作用機轉，何者正確？
- A. 直接活化 GABA-A 受體如同 benzodiazepine
- B. 抑制 GABA 再回收
- C. 結合電壓門控鈣通道 α2δ 次單元、減少興奮性神經傳導物釋放
- D. 阻斷鈉通道如同 phenytoin

> [!success]- 答案
> **C** — 儘管名稱含 GABA，實際機轉是與鈣通道 α2δ 次單元結合，減少 glutamate、substance P、CGRP 等興奮性神經肽的釋放；並不直接作用於 GABA 受體，也非鈉通道阻斷劑。

**Q3.** 一名慢性疼痛併鴉片類止痛的老年病人，加上 pregabalin。最需要警覺的風險是？
- A. 高血壓危象
- B. 呼吸抑制與過度鎮靜（含跌倒風險）
- C. 肝衰竭
- D. 心律不整（QT 延長）

> [!success]- 答案
> **B** — Gabapentinoids 與 opioids 併用會加重中樞及呼吸抑制，FDA 已警示可致死；老年人更有嗜睡、運動失調與跌倒風險。應低劑量起始、密切監測。
