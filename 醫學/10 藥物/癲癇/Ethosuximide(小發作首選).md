# Ethosuximide（乙琥胺 / 小發作首選）

> [!abstract] 一句話：**抑制丘腦（thalamus）神經元的 T-type 鈣離子通道**，切斷失神性癲癇（absence seizure）背後的 3-Hz 棘慢波振盪 → 是**失神性小發作的第一線首選**；此機轉為本藥獨有，與 Na 通道類抗癲癇藥完全不同。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **只對「失神性發作（absence / 小發作）」有效**，對強直陣攣（GTC / 大發作）、局部發作**無效** —— 這是最常考的區辨點。
> 2. **若病人同時有全身強直陣攣發作** → 單用 Ethosuximide 不能保護大發作，須改用或併用 [[Valproic acid(丙戊酸)]]（同時涵蓋 absence + GTC）。
> 3. 罕見但致命：**骨髓抑制（血球低下）**、**Stevens-Johnson syndrome**、**藥物引起狼瘡樣症候群** —— 出現發燒、皮疹、喉嚨痛、瘀青要警覺，考慮驗 CBC。
> 4. 常見腸胃不適與嗜睡，通常隨時間耐受；突然停藥可能誘發失神重積（absence status）。

## 💊 適應症（治療哪些疾病）
- **[[Absence Seizure(失神性癲癇)]]（小發作）第一線首選** —— 兒童典型 absence（3-Hz spike-wave）療效與 valproate 相當、副作用更少。
- **單純 absence 型患者**優先選它（避開 valproate 的肝毒/致畸/體重增加）。
- **合併 GTC 時的替代思路**：改用 [[Valproic acid(丙戊酸)]]（廣效）或 valproate + ethosuximide 併用。

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 起始（成人 / >6 歲兒童） | 500 mg/day PO，之後每 4–7 天視情況加量 |
| 維持 | 依療效與血中濃度調整，常見 750–1500 mg/day 分次 |
| 兒童（3–6 歲） | 起始劑量較低，依體重與院規調整 |
| 治療血中濃度參考 | 約 40–100 µg/mL（**依院規 / 需查證**） |
- 半衰期長，可 QD–BID 分次；腸胃不適明顯者可隨餐、分次服用降低刺激。
- 具體加量速度與目標濃度**依院規 / 需查證**。

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：
  - **腸胃不適**（噁心、嘔吐、腹痛、食慾不振）—— 最常見，分次或隨餐可減輕
  - 嗜睡、頭痛、頭暈、疲倦、打嗝
  - 行為改變（少數兒童易怒、注意力變化）
- **嚴重（罕見但要命）**：
  - **骨髓抑制**：白血球低下 / 顆粒球缺乏 / 再生不良性貧血 —— 監測 CBC
  - **Stevens-Johnson syndrome / 嚴重皮疹**
  - **藥物引起狼瘡樣症候群（drug-induced lupus）**
  - 精神症狀（少數）

## 🚫 禁忌症
- 對 succinimide 類（ethosuximide 屬 succinimide）過敏
- 嚴重肝、腎功能不全者慎用（**依院規 / 需查證**）
- 已知血液惡病質 / 骨髓抑制病史者慎用

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：出現發燒、喉嚨痛、口腔潰瘍、異常瘀青出血、皮疹時 → 驗 **CBC**（警覺骨髓抑制 / SJS）；長期使用可定期追蹤血球。
- **交互作用**：
  - **Valproic acid** 可能**升高** ethosuximide 血中濃度（代謝抑制），併用時留意毒性。
  - 酵素誘導型抗癲癇藥（carbamazepine、phenytoin、phenobarbital）可**降低**其濃度。
- **孕哺**：抗癲癇藥孕期用藥須權衡發作控制 vs 致畸風險，個別評估（**依院規 / 需查證**）。

## 🔗 相關
- 疾病：[[Absence Seizure(失神性癲癇)]]
- 同類 / 對照：[[Valproic acid(丙戊酸)]]（廣效，涵蓋 absence + GTC；本藥只涵蓋 absence）
- 機轉對照：Na 通道類抗癲癇藥（[[Phenytoin-Fosphenytoin(苯妥英)]]、[[Carbamazepine-Oxcarbazepine(卡馬西平總覽)]]）—— 機轉與適應症皆不同

### 自我測驗（選擇題，答案摺疊）

**Q1.** 8 歲女童上課時常突然「放空」數秒、眼神呆滯後恢復，腦波顯示 3-Hz 棘慢波。若無合併其他發作型態，下列何者為第一線首選用藥？
- A. Phenytoin
- B. Ethosuximide
- C. Carbamazepine
- D. Gabapentin

> [!success]- 答案
> **B** — 典型兒童失神性癲癇（absence，3-Hz spike-wave）的第一線首選是 **Ethosuximide**，透過抑制丘腦 T-type 鈣通道切斷振盪。Phenytoin / Carbamazepine 對 absence 無效甚至可能惡化。

**Q2.** 承上題，若該病人除了失神發作外，另有一次全身強直陣攣大發作，用藥策略最恰當的是？
- A. 續用 Ethosuximide 即可，劑量加倍
- B. 改為單用 Ethosuximide + 觀察
- C. 改用（或併用）Valproic acid
- D. 改用 Gabapentin

> [!success]- 答案
> **C** — Ethosuximide **只涵蓋失神發作、對 GTC 無效**，合併大發作時須改用或併用 **Valproic acid**（廣效，同時涵蓋 absence 與 GTC）。

**Q3.** 服用 Ethosuximide 的病人回診主訴發燒、喉嚨痛、身上多處不明瘀青。最需要優先排除的嚴重不良反應是？
- A. 高血鉀
- B. 骨髓抑制（血球低下）
- C. 低血糖
- D. 橫紋肌溶解

> [!success]- 答案
> **B** — 發燒、喉嚨痛、異常瘀青提示可能的**骨髓抑制**（顆粒球缺乏 / 血球低下），應驗 CBC；Ethosuximide 也可能引起 SJS 與狼瘡樣症候群。
