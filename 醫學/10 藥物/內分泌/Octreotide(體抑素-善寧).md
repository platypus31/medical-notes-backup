# Octreotide（體抑素 / 善寧 / Sandostatin）

> [!abstract] 一句話：長效**體抑素（somatostatin）類似物**，模擬體抑素**廣泛抑制作用**——抑制生長激素、升糖素、胰島素、胃腸胜肽、內臟血流。臨床用於**食道靜脈瘤/靜脈曲張出血（降內臟血流與門脈壓）**、**肢端肥大症（降 GH/IGF-1）**、**類癌症候群與 VIPoma 等神經內分泌腫瘤（止瀉/控症狀）**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **膽結石 / 膽泥**：長期使用抑制膽囊收縮 → 膽石、膽泥、膽囊炎（最常見長期併發），追蹤膽道症狀
> 2. **血糖雙向失衡**：同時抑制胰島素與升糖素 → **可高血糖也可低血糖**，糖尿病人需密切監測、調整降糖藥
> 3. **心搏過緩 / 傳導異常**：可致竇性心搏過緩、QT 延長，心臟病患留意；併其他減慢心率藥更需注意
> 4. **靜脈曲張出血情境**：octreotide 為**輔助止血**（降門脈壓），**不能取代內視鏡結紮/硬化治療**與抗生素預防，須整體處置
> 5. 突然停用類癌/VIPoma 治療可能症狀反彈；注射相關腸胃不適（腹痛、脹氣、脂肪瀉）常見

## 💊 適應症（治療哪些疾病）
- **食道/胃靜脈曲張出血 [[Esophageal Variceal Bleeding(食道靜脈瘤出血)]]**（急性期降門脈壓輔助止血）
- **肢端肥大症 [[Acromegaly(肢端肥大症)]]**（手術/放療後或無法手術者降 GH/IGF-1）
- **神經內分泌腫瘤**：**類癌症候群（carcinoid syndrome）**的潮紅腹瀉、**VIPoma**、**升糖素瘤**等控制症狀
- 其他：頑固性腹瀉、腸胰廔管（減少分泌）、傾食症候群（dumping）

## 📏 劑量 / 用法（成人）
| 情境 | 劑量 |
| --- | --- |
| **急性靜脈曲張出血** | **IV bolus 50 µg 後連續輸注 50 µg/hr**，續 2–5 天 |
| 類癌 / VIPoma 症狀 | SC 100–600 µg/day 分 2–4 次 |
| 肢端肥大症 | SC 起始 50 µg TID，依 GH/IGF-1 調整；或長效 LAR IM 每 4 週 |
| 長效劑型（Sandostatin LAR） | IM 每 4 週一次 |
- **給法**：短效可 SC 或 IV；長效 LAR 為深部臀肌 IM 每月一次
- **腎功能不全**可能需調整輸注/劑量

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：腹痛、脹氣、噁心、脂肪瀉、注射部位疼痛、血糖波動
- **嚴重**：
  - **膽結石 / 膽泥 / 膽囊炎**（長期使用）
  - **高血糖或低血糖**
  - **心搏過緩、QT 延長、傳導阻滯**
  - 甲狀腺功能低下（長期）、維生素 B12 吸收下降
  - 胰臟炎（少見）

## 🚫 禁忌症
- 對 octreotide 或成分過敏
- 相對禁忌 / 慎用：既有膽石症、明顯心搏過緩/傳導異常、控制不良的糖尿病（需嚴密監測）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：血糖、膽道超音波（長期）、心率/心電圖、甲狀腺功能、肢端肥大者 GH/IGF-1、類癌者症狀與尿 5-HIAA
- **交互作用**：
  - **降低 cyclosporine 血中濃度**（可致排斥）
  - 影響需胰島素/口服降糖藥劑量
  - 併用減慢心率藥（β-blocker、CCB）加重心搏過緩
  - 減少 bromocriptine 等藥吸收
- **孕哺**：孕期資料有限，僅在效益大於風險時使用；哺乳謹慎

## 🔗 相關

### 自我測驗（選擇題，答案摺疊）

**Q1.** 肝硬化病人急性食道靜脈曲張出血，關於 octreotide 何者正確？
- A. 可單獨取代內視鏡治療
- B. 作用是降低門脈壓輔助止血，須合併內視鏡與抗生素
- C. 用來升高血壓
- D. 禁用於肝病

> [!success]- 答案
> **B** — Octreotide 降內臟血流與門脈壓，屬輔助止血；標準處置仍須內視鏡結紮/硬化 + 預防性抗生素。

**Q2.** 一名長期使用 octreotide 控制類癌症候群的病人出現右上腹痛與發燒。最可能併發症？
- A. 心肌梗塞
- B. 膽石症 / 膽囊炎
- C. 腎結石
- D. 消化性潰瘍

> [!success]- 答案
> **B** — Octreotide 抑制膽囊收縮，長期使用易致膽泥、膽結石與膽囊炎，是最常見的長期併發症，需膽道超音波追蹤。
