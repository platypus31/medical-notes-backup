# Amiodarone（胺碘酮）

> [!abstract] 一句話：主要阻斷 **K⁺ channel（Class III）延長動作電位與 QT**，同時兼具鈉/鈣通道與 β 阻斷特性 → **廣效抗心律不整**。ACLS 中用於**電擊無效的 VF / 無脈性 VT**，也用於穩定型 VT 與 AF。長期使用有**大量器官毒性**（肺、甲狀腺、肝、眼、皮膚）。

> [!danger] 🚨 值班紅旗
> 1. **VF / 無脈性 VT（電擊+EPI 後仍不轉）**：**300 mg IV/IO push**，無效再 **150 mg**
> 2. **穩定型 VT / 心律不整**：**150 mg IV over 10 min**，接著 **1 mg/min ×6h → 0.5 mg/min ×18h**（24h max ~2.2g）
> 3. 快速輸注會**低血壓 + 心搏過緩**（含賦形劑 polysorbate 80）→ 非心跳停止時要慢推
> 4. **含碘 → 甲狀腺**（可高可低甲）、**肺纖維化**是最致命長期毒性；QT 延長要小心 torsades（但實際致 TdP 機率相對低）
> 5. **顯著升高 warfarin 與 digoxin 濃度** → 兩者需減量（digoxin 減半、warfarin 減 30–50%）並監測

## 💊 適應症（治療哪些疾病）
- **心跳停止**：電擊無效的 [[Ventricular Fibrillation(心室顫動)]] / 無脈性 VT（ACLS）
- **穩定型室性心搏過速（VT）**
- **心房顫動 / 撲動**：rate/rhythm control（尤其心衰或結構性心臟病時較安全）
- **維持竇性節律**（長期口服）

## 📏 劑量 / 用法（成人）
| 情境 | 劑量 |
| --- | --- |
| **VF / 無脈性 VT** | **300 mg IV/IO push**，第二劑 **150 mg** |
| **穩定 VT / 心律不整** | **150 mg IV over 10 min** → 1 mg/min ×6h → 0.5 mg/min ×18h |
| AF 藥物整流 | 150mg over 10min → maintenance drip（同上）或口服 loading |
| 口服 loading（門診） | 常 200mg TID/BID 數週後降維持 200mg QD |
- **24h 總量上限約 2.2 g**
- 賦形劑刺激血管 → 高濃度 / 長時間走中央靜脈；用玻璃或非 PVC/DEHP 管路較佳

## ⚠️ 副作用（急性 / 長期）
- **急性（IV）**：低血壓、心搏過緩、QT 延長、注射部位靜脈炎
- **長期（器官毒性，多且嚴重）**：
  - **肺**：間質性肺炎 / **肺纖維化**（最致命）
  - **甲狀腺**：含碘 → **高甲或低甲**（AIT / AIH）
  - **肝**：轉胺酶升高、肝炎
  - **眼**：角膜微沉積（多無症狀）、視神經病變
  - **皮膚**：光敏感、**藍灰色皮膚變色**
  - 神經：周邊神經病變、震顫、共濟失調

## 🚫 禁忌症
- 嚴重竇房結病變 / 二三度 AV block（無節律器）、明顯心搏過緩
- 心因性休克
- 碘過敏、嚴重甲狀腺疾病（相對）
- 已知 QT 延長 / TdP 病史（相對，需權衡）
- 孕婦（可致胎兒甲狀腺異常）——除非救命

## 🔍 監測 / 交互作用 / 孕哺
- **監測（長期）**：TFT、LFT、CXR（±肺功能 PFT）、眼科、心電圖 QT；急性期看血壓/心率
- **交互作用**：
  - **↑ Warfarin**（出血風險）、**↑ Digoxin**（毒性）→ 減量監測
  - 與其他 QT 延長藥（fluoroquinolone、azole、抗精神病藥）→ 疊加 QT
  - CYP 抑制（3A4/2C9/P-gp）→ 多種藥物濃度上升
- **孕哺**：避免（胎兒/新生兒甲狀腺與神經影響），救命例外

### 自我測驗（選擇題，答案摺疊）

**Q1.** VF 心跳停止已電擊 3 次並給過 epinephrine，仍持續 VF。下一個抗心律不整藥與劑量？
- A. Amiodarone 150mg over 10min
- B. Amiodarone 300mg IV/IO push
- C. Adenosine 6mg
- D. Atropine 1mg

> [!success]- 答案
> **B** — 電擊無效的 VF/無脈性 VT，ACLS 給 **Amiodarone 300 mg IV/IO push**（第二劑 150mg）。150mg over 10min 是穩定型 VT 的給法，不是心跳停止。

**Q2.** 一位長期口服 amiodarone 的病人出現漸進性乾咳、呼吸困難、CXR 兩側浸潤。最需警覺的毒性是？
- A. 甲狀腺風暴
- B. 肺纖維化 / 間質性肺炎
- C. 角膜沉積
- D. 藍灰皮膚

> [!success]- 答案
> **B** — Amiodarone 最致命的長期毒性是**肺毒性（間質性肺炎 / 肺纖維化）**，表現為乾咳、呼吸困難、兩側浸潤，需停藥並評估。

**Q3.** 病人開始 amiodarone 後 INR 明顯上升、且原本穩定的 digoxin 出現噁心與心律不整。最可能原因？
- A. 巧合
- B. Amiodarone 升高 warfarin 與 digoxin 濃度
- C. 甲狀腺功能亢進
- D. 腎衰竭

> [!success]- 答案
> **B** — Amiodarone 抑制代謝/P-gp，**同時升高 warfarin（INR↑出血）與 digoxin（中毒）**濃度。加用時 warfarin 減 30–50%、digoxin 約減半並密切監測。
