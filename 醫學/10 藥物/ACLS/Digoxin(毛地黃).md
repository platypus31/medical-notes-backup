# Digoxin（毛地黃 / 地高辛）

> [!abstract] 一句話：抑制心肌 **Na⁺/K⁺-ATPase → 細胞內 Na⁺↑ → 經 Na/Ca 交換使 Ca²⁺↑ → 正性肌力（↑收縮）**；同時**增加迷走張力 → 減慢 AV 傳導與心率**。用於**心衰竭症狀控制**與**AF 心室率控制**，但**治療窗極窄、中毒常見**，非急救首選。

> [!danger] 🚨 值班紅旗
> 1. **治療窗極窄**（AF 目標 dig level 0.5–2.0 ng/mL；心衰竭 **0.5–0.9** ng/mL 較安全）——懷疑中毒**先驗 dig level ＋ K⁺**
> 2. **低鉀、低鎂、高鈣 → 加重毒性**（利尿劑併用最常見的中毒誘因）；補鉀常是處理中毒的關鍵
> 3. **中毒表現**：噁心嘔吐、厭食、**黃綠視 / 視覺光暈**、意識混亂 ＋ **各種心律不整**（特徵：PVC、雙向性 VT、AF 併完全 AV block、加速交界節律）
> 4. **腎排除**：腎功能惡化 / 脫水 → 濃度飆升；老人、腎衰要減量
> 5. **解毒劑：Digoxin immune Fab（DigiFab）**——用於危及生命的心律不整、高血鉀（>5）、大量攝入
> 6. **與 amiodarone / verapamil / quinidine 併用 → digoxin 濃度上升**（常需減半）

## 💊 適應症（治療哪些疾病）
- **心房顫動 / 撲動的心室率控制**（尤其合併心衰、久坐病人；運動時控制較差）
- **收縮性心衰竭（HFrEF）症狀控制**（減住院，不降死亡率）
- 註：AF rate control 首選多為 β-blocker / CCB，digoxin 為輔助或特定族群

## 📏 劑量 / 用法（成人）
| 情境 | 劑量 |
| --- | --- |
| Loading（急需，IV/PO） | 0.25 mg Q6h（總 1–1.5 mg / 24h），依反應 |
| 維持 | **0.125–0.25 mg PO QD**（老人/腎功能差 0.0625–0.125mg） |
| 目標濃度 | AF 0.5–2.0 ng/mL；**心衰竭 0.5–0.9 ng/mL** |
- **腎功能不全 / 老人 / 低體重 → 減量**（腎排除為主）
- 抽 dig level 應在給藥後 **≥6–8 小時**（分布完成）

## ⚠️ 副作用 / 中毒（Digitalis toxicity）
- **腸胃**：噁心、嘔吐、厭食（常最早）
- **視覺**：黃綠視、光暈、視力模糊（典型）
- **神經**：疲倦、意識混亂、頭痛
- **心律不整（最危險）**：PVC、**雙向性室性心搏過速（bidirectional VT，特徵性）**、AF 併完全 AV block、加速交界性節律、各式傳導阻滯
- **毛地黃效應（治療濃度的正常心電圖變化，非中毒）**：**不對稱「杓狀 scooped」ST 段下降、T 波變平 / 反轉**——與缺血或左室再極化異常不易區分（銀蛋口袋書 p.104）

## 🚫 禁忌症
- VF、預激症候群（WPW）併 AF（可能加速旁路）
- 二度 / 三度 AV block（無節律器）、明顯心搏過緩
- 肥厚型阻塞性心肌病（HOCM，正性肌力惡化阻塞）
- 已有毛地黃中毒

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：dig level、K⁺ / Mg²⁺ / Ca²⁺、腎功能、心電圖、中毒症狀
- **交互作用**：
  - **↑ digoxin 濃度**：amiodarone、verapamil、quinidine、propafenone、clarithromycin（常需減半）
  - **利尿劑（thiazide/loop）致低鉀低鎂 → 加重毒性**
  - β-blocker / CCB：AV 傳導抑制加乘
- **孕哺**：相對安全（需監測濃度）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 服用 digoxin + furosemide 的老人出現噁心、黃綠視、心電圖見雙向性 VT。最關鍵的初始檢查與處置是？
- A. 只驗 dig level
- B. 驗 dig level 與 K⁺，糾正低鉀、必要時 DigiFab
- C. 立即給更多 digoxin
- D. 給 atropine

> [!success]- 答案
> **B** — 典型 digoxin 中毒（利尿劑致低鉀誘發）。應**同時驗 dig level 與 K⁺/Mg²⁺**，**糾正低鉀低鎂**，危及生命的心律不整/高血鉀用 **DigiFab** 解毒。

**Q2.** 關於 digoxin 的「毛地黃效應」，下列何者正確？
- A. 一定代表中毒需停藥
- B. 是治療濃度下的正常心電圖變化（杓狀 ST 下降、T 波變平）
- C. 只出現在過量時
- D. 與心肌缺血完全無法區分所以要立即冠脈攝影

> [!success]- 答案
> **B** — 「毛地黃效應」是**治療濃度即出現的正常變化**（不對稱杓狀 ST 下降、T 波變平/反轉），不等於中毒；但外觀與缺血/束支阻斷相似，需臨床鑑別。

**Q3.** 病人原本 digoxin 濃度穩定，加用 amiodarone 後兩週出現噁心與心律不整。最可能原因？
- A. Amiodarone 降低 digoxin
- B. Amiodarone 升高 digoxin 濃度導致中毒
- C. 甲狀腺亢進
- D. 兩藥無交互作用

> [!success]- 答案
> **B** — **Amiodarone 顯著升高 digoxin 濃度**（抑制 P-gp/清除），常需在加用時把 digoxin **減半**並監測，否則易中毒。
