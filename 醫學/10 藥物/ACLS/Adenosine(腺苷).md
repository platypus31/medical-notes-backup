# Adenosine（腺苷）

> [!abstract] 一句話：活化 AV node 的 **A1 腺苷受體** → 短暫**完全阻斷 AV node 傳導** → 中斷折返性室上心搏過速（reentrant SVT）。半衰期 **<10 秒**，所以必須**近端大靜脈超快速 IV push + 立刻沖生理食鹽水**，否則到心臟前就代謝掉沒效。

> [!danger] 🚨 值班紅旗
> 1. **給法就是成敗**：**6mg 極速 IV push（1–3 秒）＋ 立即用 20mL NS 快沖 ＋ 抬高手臂**；針要打**越靠近心臟越好**（前肘窩或中央靜脈）
> 2. **會造成數秒 asystole / 心臟短暫停跳**——病人會感覺胸悶、快死掉、臉潮紅，這是**預期反應**，先跟病人與團隊說明、備好監視器與電擊器
> 3. **氣喘 / COPD 慎用**：可誘發**支氣管痙攣**
> 4. **無效或惡化的心律**：對 AF/atrial flutter/VT **無終止效果**（但可短暫揭露 flutter 波幫助診斷）；**WPW 併 AF/寬 QRS 不規則** → 可能加速旁路傳導致 VF，避免
> 5. 只用在**血流動力學穩定**的規則窄 QRS SVT；不穩定 → 直接電擊（同步整流）

## 💊 適應症（治療哪些疾病）
- **陣發性室上心搏過速（PSVT / reentrant SVT）**：第一線藥物治療（迷走刺激無效後）
- **窄 QRS 規則心搏過速的診斷輔助**：短暫阻斷 AV node 揭露底層心房活動（如 flutter 波）
- 少數穩定、規則、單型的寬 QRS 心搏過速（懷疑 SVT with aberrancy）——需謹慎

## 📏 劑量 / 用法（成人）
| 步驟 | 劑量 | 給法 |
| --- | --- | --- |
| 第 1 劑 | **6 mg** | 極速 IV push（1–3 秒）＋ 立即 20mL NS 快沖 ＋ 抬手臂 |
| 第 2 劑（1–2 分後無效） | **12 mg** | 同法 |
| 第 3 劑（部分舊指引） | 12 mg | 同法（AHA 2020 主要為 6→12） |
- **中央靜脈 / 心臟移植病人 / 併用 dipyridamole 或 carbamazepine** → **減量至 3mg 起始**（作用被放大）
- **併用 theophylline / caffeine（甲基黃嘌呤）** → 拮抗作用，可能需**加量**
- 小兒：0.1 mg/kg（max 6mg）→ 0.2 mg/kg（max 12mg）

## ⚠️ 副作用（短暫，多在數秒內消退）
- **臉部潮紅（最常見）**、胸悶 / 胸痛、呼吸困難、瀕死感
- **短暫 asystole / 竇性停搏 / 心搏過緩**（預期內）
- 支氣管痙攣（氣喘）
- 少見：短暫 AF、頭痛
- 因半衰期極短，副作用通常**數秒到十幾秒自行消失**

## 🚫 禁忌症
- 第二度（Mobitz II）/ 第三度 AV block、病竇症候群（**無節律器保護時**）
- 已知對 adenosine 過敏
- 氣喘 / 支氣管痙攣性疾病（相對禁忌）
- **WPW 併心房顫動 / 不規則寬 QRS**（可能加速旁路 → VF）
- 血流動力學不穩定的心搏過速（應直接電擊，不是給藥）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：全程連續心電圖（記錄轉復瞬間）、備妥電擊器與急救設備
- **交互作用**：
  - **Dipyridamole**（阻斷 adenosine 再回收）、**Carbamazepine**（加強 AV 阻斷）→ **減量至 3mg**
  - **Theophylline / caffeine**（腺苷受體拮抗）→ 效果減弱，需加量
  - 心臟移植（去神經 → 超敏感）→ 減量
- **孕哺**：SVT 孕婦可用（相對安全，屬急救選項）

## 🔗 相關
- 疾病：[[Supraventricular Tachycardia(室上心搏過速)]]、[[WPW Syndrome(預激症候群)]]、[[醫學/11心電圖判讀/Atrial Flutter(心房撲動)]]
- 同類 / 對照：[[Verapamil(維拉帕米)]]、[[Diltiazem(地爾硫䓬)]]（CCB，用於持續 rate control）、[[Amiodarone(胺碘酮)]]
- 手法：迷走刺激（Valsalva / 頸動脈竇按摩）先於藥物

### 自我測驗（選擇題，答案摺疊）

**Q1.** 30 歲女性心悸，心電圖為規則窄 QRS 心搏過速 180/min，血壓穩定，迷走刺激無效。下一步最適當？
- A. Adenosine 6mg 緩慢 IV drip
- B. Adenosine 6mg 極速 IV push ＋ 立即 20mL NS 快沖
- C. Amiodarone 300mg
- D. 立即非同步電擊

> [!success]- 答案
> **B** — 穩定規則窄 QRS SVT、迷走無效 → **Adenosine 6mg 極速 IV push＋立即快沖**。緩慢 drip 會因半衰期 <10 秒而失效；穩定的病人不需電擊。

**Q2.** 準備給 Adenosine 前，最重要的病人與團隊準備是？
- A. 告知會有數秒心臟停跳/瀕死感並備妥監視器與電擊器
- B. 先給鎮靜劑
- C. 禁食 8 小時
- D. 先驗甲狀腺功能

> [!success]- 答案
> **A** — Adenosine 會造成**短暫 asystole 與強烈瀕死/胸悶感**，屬預期反應。事前告知病人、全程連續心電圖、備妥急救設備最重要。

**Q3.** 下列哪種心律給 Adenosine 可能有害？
- A. 規則窄 QRS 的 reentrant SVT
- B. WPW 併心房顫動（不規則寬 QRS）
- C. 迷走刺激無效的 PSVT
- D. 診斷不明的規則窄 QRS 心搏過速

> [!success]- 答案
> **B** — **WPW 併 AF** 時阻斷 AV node 可能使衝動全走旁路、加速心室反應，惡化為 **VF**。此類不規則寬 QRS 心搏過速應避免 adenosine。
