# Atropine（阿托品）

> [!abstract] 一句話：阻斷 **muscarinic（M）膽鹼受體** → 拮抗迷走神經 → **加快心率、加速 AV 傳導**（ACLS 症狀性心搏過緩第一線），並可對抗**有機磷 / 膽鹼酯酶抑制劑中毒**的毒蕈鹼症狀。

> [!danger] 🚨 值班紅旗
> 1. **症狀性心搏過緩：1 mg IV，Q3–5min，max 3 mg**（AHA 2020 起始劑量從 0.5mg 提高到 **1mg**）
> 2. **劑量太低（<0.5mg）反而可能造成反常性心搏過緩**（中樞性副交感興奮）→ 不要給小劑量
> 3. **對 Mobitz II 型二度 / 三度 AV block（寬 QRS）常無效**——阻斷點在 AV node 以下，atropine 加速的是 node → 應直接準備**經皮節律器 / 升壓 infusion（dopamine/epinephrine）**
> 4. **心臟移植（去神經心臟）atropine 無效**（沒有迷走張力可阻斷）
> 5. 有機磷中毒需**大劑量、反覆倍增**（2–4mg 起，每 3–5 分倍增）直到**呼吸道分泌物乾掉**（atropinization 終點是肺部乾淨，不是瞳孔或心率）

## 💊 適應症（治療哪些疾病）
- **症狀性心搏過緩 / 高迷走張力**（低血壓、意識改變、胸痛、心衰）
- **有機磷 / carbamate（膽鹼酯酶抑制劑）中毒**：對抗毒蕈鹼症狀（SLUDGE）
- **麻醉前 / 術中減少分泌物、對抗迷走反射**
- 眼科散瞳、緩解膽道/腸道痙攣（少用）

## 📏 劑量 / 用法（成人）
| 情境 | 劑量 | 給法 |
| --- | --- | --- |
| **症狀性心搏過緩** | **1 mg IV** | Q3–5min，**max 3 mg** |
| **有機磷中毒** | 2–4 mg IV 起，**每 3–5 分倍增** | titrate 到分泌物乾（可達數十~上百 mg） |
| 麻醉前抑制分泌 | 0.4–0.6 mg IV/IM | 術前 |
| 小兒心搏過緩 | 0.02 mg/kg（min 0.1mg，max 0.5mg/劑） | IV/IO |
- **最低單劑不要 <0.5mg**（反常性心搏過緩風險）；小兒 min 0.1mg 也是此理

## ⚠️ 副作用（抗膽鹼症候群「紅、乾、熱、盲、瘋、脹」）
- **口乾、視力模糊 / 散瞳、畏光**
- **心搏過速**、心悸
- **尿滯留、腸蠕動減少（便秘/腸阻）**
- **無汗 → 皮膚乾熱、發燒（尤其兒童/老人危險）**
- **中樞**：躁動、譫妄、幻覺（「瘋如帽匠」）
- 記憶口訣：**"Hot as a hare, dry as a bone, red as a beet, blind as a bat, mad as a hatter"**

## 🚫 禁忌症
- 閉角性青光眼（散瞳誘發急性發作）
- 前列腺肥大 / 尿滯留、腸阻塞 / 麻痺性腸阻、重症肌無力
- 心搏過速性心律不整、甲亢（相對）
- 註：**急救（心搏過緩、有機磷中毒）情境下無絕對禁忌**，救命優先

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：心率、心律、意識、體溫、尿量、有機磷中毒看**肺部分泌物**（治療終點）
- **交互作用**：與其他抗膽鹼藥（抗組織胺、TCA、抗精神病藥）加乘抗膽鹼毒性
- **孕哺**：需要時可用（急救）

## 🔗 相關
- 疾病：[[Bradycardia(心搏過緩)]]、[[AV Block(房室傳導阻滯)]]、[[Organophosphate Poisoning(有機磷中毒)]]
- 同類 / 對照：[[Pralidoxime(2-PAM)]]（有機磷中毒併用，重活化膽鹼酯酶）、[[Epinephrine(腎上腺素)]] / [[Dopamine(多巴胺)]]（心搏過緩 atropine 無效時）
- 節律處置：經皮 / 經靜脈節律器（pacing）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 70 歲男性心跳 38/min、血壓 80/50、意識改變，心電圖為竇性心搏過緩。第一線藥物與劑量？
- A. Atropine 0.5mg IV
- B. Atropine 1mg IV，Q3–5min，max 3mg
- C. Adenosine 6mg
- D. Amiodarone 150mg

> [!success]- 答案
> **B** — 症狀性心搏過緩第一線是 **Atropine 1mg IV，Q3–5min，最多 3mg**（AHA 2020 起始已從 0.5mg 提高到 1mg）。低劑量可能反常性減慢心率。

**Q2.** 農夫噴灑農藥後大量流涎、流淚、瞳孔縮小、支氣管分泌物多、肌肉顫動。Atropine 的給藥終點是？
- A. 心率 >100
- B. 瞳孔散大
- C. 呼吸道 / 肺部分泌物乾掉
- D. 血壓正常

> [!success]- 答案
> **C** — 有機磷中毒 atropinization 的終點是**呼吸道分泌物乾掉（肺部聽診乾淨）**，不是瞳孔或心率。常需大劑量反覆倍增，並合併 pralidoxime。

**Q3.** 病人為三度 AV block、寬 QRS 逸搏、心跳 30，給了 atropine 3mg 完全無反應。下一步最適當？
- A. 再給 atropine 3mg
- B. 經皮節律器 / dopamine 或 epinephrine infusion
- C. Adenosine
- D. 觀察即可

> [!success]- 答案
> **B** — 三度 AV block 阻斷點在 AV node 以下，**atropine 通常無效**。應立即**經皮節律器**或啟動 **dopamine/epinephrine infusion**，並準備經靜脈節律器。
