# NTG（硝化甘油 / Nitroglycerin）

> [!abstract] 一句話：在血管平滑肌釋放 **NO → ↑cGMP → 血管鬆弛**（以**靜脈擴張為主 → ↓preload**，高劑量也擴動脈 → ↓afterload）→ 減少心肌耗氧、緩解心絞痛，並用於急性肺水腫與高血壓危症。

> [!danger] 🚨 值班紅旗（給之前一定要問/看）
> 1. **下壁 MI / 右心室（RV）梗塞** → 這類病人**前負荷依賴**，NTG 大降 preload 會造成**嚴重低血壓/休克** → **禁用或極謹慎**（給前看 II/III/aVF、必要時右側導程 V4R）
> 2. **24 小時內用過 PDE5 抑制劑**（sildenafil/威而鋼 24h、tadalafil 48h、vardenafil）→ **致命性低血壓，絕對禁用**
> 3. **低血壓 SBP <90** 或比基礎值降 ≥30 → 不給
> 4. **嚴重主動脈瓣狹窄 / 肥厚型阻塞性心肌病（HOCM）** → 前負荷依賴，NTG 可致暈厥/猝死
> 5. **顱內壓升高** → 加重
> 6. 反覆給要監測血壓、備好靜脈輸液（低血壓時抬腿 + 補液）

## 💊 適應症（治療哪些疾病）
- **心絞痛 / 急性冠心症（ACS）胸痛緩解**（SL 反覆給，或 IV drip）
- **急性心衰竭 / 急性肺水腫**（↓preload/afterload，降肺充血）
- **高血壓急症 / 危症**（IV titrate）
- 冠狀動脈痙攣（變異型心絞痛）

## 📏 劑量 / 用法（成人）
| 劑型 / 情境 | 劑量 |
| --- | --- |
| **舌下錠 / 噴劑（急性胸痛）** | **0.3–0.4 mg SL，每 5 分一次，最多 3 次**；仍痛 → 疑 ACS 送醫 |
| **IV drip（ACS/肺水腫/高血壓）** | **5–10 mcg/min 起，每 3–5 分 ↑5–10，titrate**（可到 100–200 mcg/min） |
| 外用貼片 / 軟膏 | 慢性心絞痛預防（需 nitrate-free interval） |
- **監測血壓**：SBP 維持 >90（或不低於基礎 −25%）
- **避免耐受性**：長期使用需每日**留一段 nitrate-free interval（10–12h）**

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：**搏動性頭痛（最常見）**、臉部潮紅、姿勢性低血壓、頭暈、反射性心搏過速
- **嚴重**：
  - **嚴重低血壓 / 休克**（尤其 RV 梗塞、併 PDE5 抑制劑、低血容量）
  - 反射性心搏過速加重心肌缺血
  - 高鐵血紅素血症（methemoglobinemia，罕見、大劑量）
- **耐受性（tolerance）**：連續使用效果遞減

## 🚫 禁忌症
- **24–48h 內使用 PDE5 抑制劑**（sildenafil/tadalafil/vardenafil）
- **RV 梗塞 / 下壁 MI（前負荷依賴）**
- 收縮壓 <90 或明顯低血壓、低血容量
- 嚴重主動脈瓣狹窄、HOCM
- 顱內壓升高、嚴重貧血
- 對硝酸鹽過敏

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：血壓（連續，尤其 IV）、心率、胸痛緩解程度
- **交互作用**：
  - **PDE5 抑制劑（sildenafil 等）→ 致命低血壓**（絕對禁忌）
  - 其他降壓藥、酒精 → 疊加低血壓
  - 併用 heparin 時 NTG 可能降低 heparin 效果（需監測）
- **孕哺**：需要時可用（如子癲前症/急性肺水腫），權衡低血壓

### 自我測驗（選擇題，答案摺疊）

**Q1.** 60 歲男性胸痛，心電圖 II/III/aVF ST 上升（下壁 STEMI），血壓 100/70。給硝化甘油前最重要的評估是？
- A. 直接給 SL NTG
- B. 評估是否合併右心室梗塞（右側導程 V4R），RV 梗塞則避免 NTG
- C. 先給 PDE5 抑制劑
- D. 先驗甲狀腺

> [!success]- 答案
> **B** — 下壁 MI 常合併**右心室梗塞**，此類病人**前負荷依賴**，NTG 會造成嚴重低血壓。應先評估（右側導程 V4R），RV 梗塞應避免/極謹慎給 NTG，並以補液維持前負荷。

**Q2.** 急診胸痛病人準備給 SL NTG，問診時最必須確認的用藥史是？
- A. 過去 24–48h 是否服用 sildenafil / tadalafil 等 PDE5 抑制劑
- B. 是否吃過止痛藥
- C. 是否喝咖啡
- D. 疫苗接種史

> [!success]- 答案
> **A** — **PDE5 抑制劑（威而鋼類）24–48h 內併用 NTG 會致命性低血壓**，是絕對禁忌，必須主動詢問。

**Q3.** 病人使用 SL NTG 後最常出現的不適是？
- A. 高血壓
- B. 搏動性頭痛
- C. 便秘
- D. 高血糖

> [!success]- 答案
> **B** — NTG 因血管擴張最常見**搏動性頭痛**，以及潮紅、姿勢性低血壓、反射性心搏過速。
