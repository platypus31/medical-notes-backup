# Thiopental（硫噴妥鈉 / Pentothal）

> [!abstract] 一句話：**超短效巴比妥類靜脈麻醉誘導劑**，增強 **GABA-A 受體**（延長 Cl⁻ channel 開放時間）抑制中樞 → 快速失去意識。**高脂溶性**使其 IV 後 ~30–45 秒起效、蘇醒快（**靠再分布 redistribution 而非代謝**），單次劑量作用僅數分鐘。經典用於麻醉誘導與**降腦壓/腦保護（降低腦代謝率 CMRO₂）**，現多被 propofol 取代，但仍見於神經麻醉與難治癲癇重積。

> [!danger] 🚨 值班紅旗（給藥前先想）
> 1. **急性間歇性卟啉症（porphyria）→ 絕對禁用**：誘導 δ-ALA synthase，誘發致命卟啉發作（barbiturate 是經典禁忌藥）
> 2. **劑量依賴性呼吸抑制與呼吸暫停（apnea）**：誘導後需備好面罩/插管與正壓通氣，切勿無氣道準備下推藥
> 3. **心肌抑制 + 靜脈擴張 → 低血壓**：低血容量、心衰、休克病人易嚴重掉血壓，需減量慢推
> 4. **喉痙攣 / 支氣管痙攣（laryngospasm）**：巴比妥無支氣管擴張作用，淺麻醉刺激氣道時易誘發，**氣喘病人相對慎用**
> 5. **動脈內誤注 / 外滲 → 劇痛、血管痙攣、組織壞死**：強鹼性（pH ~10.5），務必確認在靜脈內；勿與酸性藥（如 rocuronium、部分抗生素）同一 line 混合會沉澱

## 💊 適應症（用於哪些情境）
- **全身麻醉誘導 [[全身麻醉(General Anesthesia)]]**（快速失去意識，經典誘導劑）
- **顱內壓升高 / 腦保護**：降低 CMRO₂ 與腦血流，用於神經外科、外傷性腦損傷（barbiturate coma）
- **難治性癲癇重積 [[Status Epilepticus(癲癇重積)]]**（refractory status，作為麻醉深度抑制）
- **快速誘導插管（RSI）** 的誘導藥選項之一（現多用 propofol / etomidate / ketamine）
- 曾用於「真話血清」歷史用途（現已不作臨床用）

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 麻醉誘導 | **3–5 mg/kg IV 慢推**（老年/血流動力不穩者減量） |
| 兒童誘導 | 5–6 mg/kg IV（嬰兒 6–8 mg/kg，依院規/麻醉科決定） |
| 顱內壓危象（急救降腦壓） | 1.5–3 mg/kg IV bolus（後續依 EEG burst-suppression 滴定） ⚠️依院規 |
| 癲癇重積（refractory） | 誘導後持續輸注滴定至 burst-suppression ⚠️依麻醉/神經科決定 |
- **起效**：IV 約 30–45 秒；**作用時間**：單劑僅 5–10 分鐘（靠再分布終止）
- **給法**：常配成 2.5% 溶液，緩慢 IV；**強鹼性勿外滲、勿動脈內注射**
- **反覆給藥會蓄積**（脂肪飽和後代謝慢、半衰期長達數小時至十幾小時），恢復延遲

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：呼吸抑制、低血壓、注射部位痛、打噴嚏/咳嗽、蘇醒後嗜睡
- **嚴重**：
  - **呼吸暫停 apnea**（誘導後常見，需支持通氣）
  - **喉痙攣 / 支氣管痙攣**
  - **嚴重低血壓 / 心肌抑制**（低血容、心衰病人）
  - **動脈內注射 → 動脈痙攣、遠端壞疽**；外滲致組織壞死
  - **過敏 / 類過敏反應（anaphylactoid）**
  - **卟啉症急性發作**（可致命）

## 🚫 禁忌症
- **急性間歇性卟啉症及相關卟啉症 → 絕對禁用**
- 對巴比妥類過敏
- **無適當氣道 / 復甦設備時禁用**（無法處理呼吸抑制）
- 相對禁忌：嚴重心血管不穩（休克、縮窄性心包炎、嚴重心衰）、氣道阻塞/氣喘急性發作、嚴重低血容

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：呼吸/SpO₂、血壓、心電圖、麻醉深度（腦保護用途以 EEG burst-suppression 為目標）
- **交互作用**：
  - **加成其他中樞抑制劑**（opioid、benzodiazepine、alcohol）→ 呼吸循環抑制加重
  - **肝酵素誘導劑（CYP 誘導）**：長期使用可加速其他藥物代謝（如 warfarin、口服避孕藥、theophylline）
  - **與酸性藥物混合沉澱**（rocuronium、vecuronium、部分抗生素）→ 不同 line
- **孕哺**：可穿胎盤（產科麻醉曾用）；一般麻醉誘導短時使用，依麻醉科評估；卟啉症孕婦禁用

## 🔗 相關
- 同類 / 對照：[[Propofol(異丙酚)]]（現代首選誘導劑）、[[Etomidate(依託咪酯)]]（血流動力穩定）、[[Ketamine(氯胺酮)]]（維持血壓/支氣管擴張）、[[Midazolam(咪達唑侖)]]
- 機轉同族：其他巴比妥（[[Phenobarbital(苯巴比妥)]]）— 皆作用於 GABA-A
- 疾病 / 情境：[[Status Epilepticus(癲癇重積)]]、顱內壓升高處置

### 自我測驗（選擇題，答案摺疊）

**Q1.** 關於 thiopental 單次誘導劑量後「作用時間短」的主要機轉，下列何者正確？
- A. 快速被肝臟代謝
- B. 快速經腎排除
- C. 由腦部再分布到肌肉與脂肪組織
- D. 快速與血漿蛋白結合失活

> [!success]- 答案
> **C** — 高脂溶性讓藥物先衝進腦部（快起效），單劑作用終止靠「再分布」離開腦部到周邊組織，而非代謝或排除；因此蘇醒快但反覆給藥會蓄積。

**Q2.** 一名擬行急診手術的病人，病史提及急性間歇性卟啉症。下列誘導藥何者應避免？
- A. Propofol
- B. Etomidate
- C. Thiopental
- D. Ketamine

> [!success]- 答案
> **C** — 巴比妥類（thiopental）會誘導 δ-ALA synthase 觸發卟啉症急性發作，為絕對禁忌；可改用 propofol 等。

**Q3.** 使用 thiopental 誘導時最需立即準備的支持措施是？
- A. 降血糖藥物
- B. 氣道管理與正壓通氣（處理呼吸抑制/apnea）
- C. 抗凝血劑
- D. 利尿劑

> [!success]- 答案
> **B** — Thiopental 造成劑量依賴性呼吸抑制與呼吸暫停，誘導前必須備妥面罩/插管與正壓通氣設備，並注意低血壓。
