# Ketamine（氯胺酮 / Ketalar）

> [!abstract] 一句話：**NMDA 受體拮抗劑**，產生「解離麻醉」（catatonia、鎮痛、遺忘，眼睛可張開但意識解離）。**同時具鎮痛作用、維持呼吸驅動與氣道反射、擬交感升血壓**，是**低血壓/休克病人 RSI 誘導**與**氣喘（支氣管擴張）**的優選；紅旗是 emergence delirium、分泌物增多與升顱內壓爭議。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **Emergence delirium（甦醒期譫妄/幻覺）**：成人尤其明顯，可先/併用 benzodiazepine（midazolam）減緩，甦醒環境宜安靜昏暗
> 2. **分泌物增多**：唾液/支氣管分泌增加，可能需 antisialagogue（atropine/glycopyrrolate），警覺喉痙攣（尤其兒童）
> 3. **擬交感升血壓/心率**：對缺血性心臟病、未控制高血壓、主動脈剝離、嚴重心衰要小心（心肌耗氧增加）；但在**兒茶酚胺耗竭的重症休克**反而可能出現心肌抑制致低血壓
> 4. **升顱內壓爭議**：傳統認為升 ICP 應避開頭部外傷；**現代證據顯示在維持血壓/通氣下影響有限**，創傷休克 RSI 仍可用（⚠️ 依情境與院規權衡）
> 5. **升眼內壓**：青光眼/開放性眼球傷相對注意

## 💊 適應症（治療哪些疾病 / 情境）
- **快速誘導插管 RSI [[Rapid Sequence Intubation(快速誘導插管)]]**（尤其低血壓/休克病人的誘導劑）
- **氣喘急性發作 [[Asthma(氣喘)]]** / 支氣管痙攣（有支氣管擴張作用）
- **程序性鎮靜 [[Procedural Sedation(程序性鎮靜)]]**（兒童尤常用，維持氣道反射）
- **急性/慢性難治性疼痛、opioid-sparing 鎮痛**（次麻醉劑量）
- **難治性憂鬱症**（esketamine，另一適應症）

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| RSI / 全麻誘導 | **1–2 mg/kg IV**（休克病人常減至 ~0.5–1 mg/kg） |
| 肌肉注射（無 IV 時） | **4–5 mg/kg IM** |
| 程序性鎮靜 | 0.5–1 mg/kg IV，視需追加 |
| 次麻醉鎮痛（analgesic dose） | **0.1–0.3 mg/kg IV**（或低速輸注） |
| 分泌物處理 | 併 glycopyrrolate/atropine（⚠️依院規） |
- ⚠️ 精確劑量依血流動力狀態與院規滴定；休克/兒茶酚胺耗竭者減量

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：升血壓/心率、唾液與支氣管分泌增多、複視/眼球震顫、噁心嘔吐、甦醒期幻覺
- **嚴重**：
  - **Emergence delirium / 精神反應**（激動、幻覺）
  - **喉痙攣**（尤其兒童、分泌物多時）
  - 缺血性心臟病者心肌耗氧增加/缺血
  - 休克末期心肌抑制性低血壓
  - 長期濫用：膀胱炎（ketamine cystitis）、認知/記憶損害

## 🚫 禁忌症
- 對 ketamine 已知過敏
- **需嚴格避免血壓/心率上升者**：未控制的嚴重高血壓、主動脈剝離、近期心肌梗塞/嚴重冠心病（相對禁忌）
- 開放性眼球傷/青光眼（升眼內壓，相對）
- 嚴重精神疾病活動期（相對，emergence 反應）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：血壓、心率、SpO₂、氣道分泌與喉痙攣徵象、甦醒期精神狀態
- **交互作用**：
  - 與 benzodiazepine 併用減少 emergence delirium
  - 擬交感效應與其他升壓/兒茶酚胺藥協同
  - 與 theophylline 併用可能降低抽搐閾值
- **孕哺**：非首選但特定情境可用（⚠️依院規）

## 🔗 相關
- 同類誘導劑對照：[[Etomidate(依託咪酯)]]（血流動力穩定但無鎮痛）、[[Propofol(異丙酚)]]（降血壓）
- 情境：[[Rapid Sequence Intubation(快速誘導插管)]]、[[Asthma(氣喘)]]、[[Procedural Sedation(程序性鎮靜)]]
- 常搭配：[[Midazolam(咪達唑侖)]]（減 emergence）、glycopyrrolate（減分泌）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名多重外傷、血壓 80/40 的休克病人需 RSI 誘導。下列誘導劑何者兼具「血流動力相對穩定+鎮痛」？
- A. Propofol
- B. Ketamine
- C. 高劑量 midazolam
- D. 標準 propofol + 無鎮痛

> [!success]- 答案
> **B** — Ketamine 擬交感可維持/提升血壓且有鎮痛作用，是低血壓休克 RSI 的常用誘導劑（etomidate 亦血流動力穩定但無鎮痛）；休克末期仍須注意減量與可能的心肌抑制。

**Q2.** 關於 ketamine 甦醒期的敘述，何者正確？
- A. 幾乎不會有精神反應
- B. 常見 emergence delirium，可併 benzodiazepine 減緩
- C. 一定要用 atropine 才會清醒
- D. 只有兒童會發生

> [!success]- 答案
> **B** — Emergence delirium（幻覺/激動）在成人尤其常見，併用 midazolam 並提供安靜環境可減緩。

**Q3.** 一名急性重度氣喘無法插管前需鎮靜，選 ketamine 的主要理由是？
- A. 降血壓
- B. 支氣管擴張且維持呼吸驅動
- C. 抑制所有分泌物
- D. 完全無精神副作用

> [!success]- 答案
> **B** — Ketamine 具支氣管擴張作用並維持呼吸驅動與氣道反射，適合氣喘，但要注意分泌物增多與喉痙攣風險。
