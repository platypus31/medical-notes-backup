# Etomidate（依託咪酯 / Amidate）

> [!abstract] 一句話：**GABA-A 受體增強的咪唑類靜脈誘導劑**，最大特色是**血流動力學穩定（對心輸出量與血壓影響極小）**，是**心血管不穩/低血壓病人 RSI 誘導**的優選。無鎮痛作用；主要紅旗是**單次劑量即可抑制腎上腺皮質類固醇合成**與注射時肌陣攣。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **腎上腺皮質抑制**：抑制 11β-hydroxylase，**即使單次誘導劑量也會短暫抑制皮質醇合成**（可持續數小時至 24h）；**敗血症/腎上腺功能不全病人須警覺**，連續輸注用於鎮靜已因增加死亡率而禁止
> 2. **無鎮痛作用**：僅催眠，插管刺激的疼痛與交感反應需另加 opioid
> 3. **肌陣攣（myoclonus）**：注射時常見不自主肌抽動，可能干擾判讀（易誤為抽搐），可先給 opioid/benzodiazepine 減緩；並非真正癲癇
> 4. **注射痛與靜脈刺激**（丙二醇載體）、**術後噁心嘔吐（PONV）較高**
> 5. **不抑制呼吸驅動較 propofol 輕，但仍可能短暫呼吸抑制**；仍需備氣道
> 6. **無鎮痛+腎上腺抑制** → 不適合長期鎮靜，只作單次誘導

## 💊 適應症（治療哪些疾病 / 情境）
- **快速誘導插管 RSI [[Rapid Sequence Intubation(快速誘導插管)]]**（尤其血流動力不穩/低血壓/心臟病人）
- **全身麻醉誘導 [[General Anesthesia(全身麻醉)]]**（心血管風險高者）
- **短時程程序鎮靜**（如心臟整流，血流動力穩定情境）

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| **RSI / 全麻誘導** | **0.2–0.3 mg/kg IV**（常用約 0.3 mg/kg） |
| 老人/衰弱/嚴重心臟病 | 減量（下修至 ~0.15–0.2 mg/kg，⚠️依院規） |
| 肌陣攣預防 | 誘導前給少量 opioid（如 fentanyl）或 benzodiazepine（⚠️依院規） |
- **不建議連續輸注鎮靜**（腎上腺抑制增加死亡率）
- **無鎮痛** → RSI 常搭配 opioid（fentanyl）與肌鬆劑
- ⚠️ 精確劑量依病人血流動力與院規

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：肌陣攣、注射部位疼痛、術後噁心嘔吐、短暫呼吸抑制
- **嚴重**：
  - **腎上腺皮質功能抑制**（單劑短暫；連續輸注致死率上升，禁用於長期鎮靜）
  - 過敏反應（罕見）
  - 誘導時短暫呼吸抑制/呼吸暫停

## 🚫 禁忌症
- 對 etomidate 已知過敏
- **不用於長時間連續鎮靜輸注**（腎上腺抑制）
- 已知腎上腺皮質功能不全者需謹慎（相對）
- 敗血性休克是否使用單劑仍有爭議 → 依臨床權衡（⚠️依院規）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：血壓、心率、SpO₂/呼吸、肌陣攣情形；高危病人可考慮監測/補充類固醇
- **交互作用**：
  - 與 opioid/benzodiazepine 併用可減肌陣攣、加深鎮靜
  - 與其他 CNS 抑制劑協同呼吸抑制
- **孕哺**：資料有限，特定情境使用（⚠️依院規）

## 🔗 相關
- 同類誘導劑對照：[[Propofol(異丙酚)]]（降血壓）、[[Ketamine(氯胺酮)]]（升壓且有鎮痛）
- 情境：[[Rapid Sequence Intubation(快速誘導插管)]]、[[General Anesthesia(全身麻醉)]]
- 常搭配：opioid [[Fentanyl(吩坦尼)]]（補鎮痛、減肌陣攣）、肌鬆劑

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名嚴重主動脈狹窄、血壓偏低的病人需麻醉誘導，最能維持血流動力穩定的誘導劑是？
- A. Propofol 標準劑量
- B. Etomidate
- C. 大劑量 midazolam
- D. 高濃度 sevoflurane 面罩

> [!success]- 答案
> **B** — Etomidate 對血壓與心輸出量影響最小，是心血管高風險/低血壓病人誘導的優選（惟無鎮痛、有腎上腺抑制與肌陣攣）。

**Q2.** 關於 etomidate 的腎上腺效應，何者正確？
- A. 只有連續輸注才有影響，單劑完全安全
- B. 單次誘導劑量也會短暫抑制皮質醇合成，連續輸注更增死亡率
- C. 會刺激腎上腺分泌
- D. 與腎上腺無關

> [!success]- 答案
> **B** — 即使單劑也短暫抑制腎上腺皮質醇合成；連續輸注因增加死亡率已被禁止用於長期鎮靜。

**Q3.** Etomidate 誘導時病人出現肢體不自主抽動，最恰當的理解是？
- A. 病人癲癇發作需抗癲癇
- B. 常見的肌陣攣，非真正癲癇，可用 opioid/benzodiazepine 減緩
- C. 過敏反應
- D. 需立即停止所有麻醉

> [!success]- 答案
> **B** — 肌陣攣是 etomidate 常見現象，非真正癲癇，誘導前給 opioid 或 benzodiazepine 可減少。
