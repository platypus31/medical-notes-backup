# 驅蟲藥（Anthelmintics 總覽）

> [!abstract] 一句話：驅蟲藥按寄生蟲門類分工——苯并咪唑（Albendazole / Mebendazole）打線蟲、Praziquantel 打吸蟲與絛蟲、Ivermectin 打糞線蟲與絲蟲、Pyrantel 是 OTC 蟯蟲蛔蟲藥；共通機轉都是「癱瘓或餓死蟲體」。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **Ivermectin 治糞線蟲前，先想 hyperinfection（高感染量症候群）** → 免疫抑制 / 長期類固醇者，糞線蟲可自體感染大爆發（甚至播散性感染 + 革蘭氏陰性菌血症），治療與類固醇使用要謹慎評估
> 2. **囊蟲症（neurocysticercosis）用 Albendazole 殺蟲會引發發炎反應** → 蟲體崩解引起腦部發炎 / 癲癇惡化，常需**併用類固醇**（±抗癲癇藥）
> 3. **蟯蟲會傳染全家** → 建議全家一起治療 + 2 週後重複一劑（打斷再感染循環）
> 4. **苯并咪唑類孕婦（尤其第一孕期）避免** → 動物實驗有致畸性，孕期驅蟲需權衡 / 依院規
> 5. **問旅遊 / 疫區 / 生食史** → 判斷蟲種與選藥（線蟲 vs 吸蟲 vs 絛蟲差很多）

## 💊 適應症（治療哪些疾病）
- **線蟲感染**：蛔蟲、鉤蟲、鞭蟲、蟯蟲 → Albendazole / Mebendazole / Pyrantel
- **組織線蟲**：[[Neurocysticercosis(神經囊蟲症)]]、包蟲症（echinococcosis）→ Albendazole（長療程）
- **吸蟲感染**：血吸蟲病（schistosomiasis）、肝吸蟲 → Praziquantel
- **絛蟲感染**：豬肉 / 牛肉絛蟲 → Praziquantel
- **糞線蟲症（strongyloidiasis）、盤尾絲蟲病（河盲症 onchocerciasis）** → Ivermectin

## 📏 各藥重點（成人）
| 藥物 | 打誰 | 機轉 | 備註 |
| --- | --- | --- | --- |
| **Albendazole / Mebendazole** | 廣效線蟲（蛔 / 鉤 / 鞭 / 蟯）；Albendazole 加打囊蟲症 / 包蟲症 | 抑制蟲體 **microtubule 聚合**（阻斷 β-tubulin），阻斷葡萄糖攝取使蟲「餓死」 | Albendazole 空腹吸收差、與脂餐同服吸收佳；囊蟲症 / 包蟲症需**長療程**且可能加類固醇 |
| **Praziquantel** | 吸蟲（血吸蟲）、絛蟲 | 增加蟲體細胞膜對**鈣離子**通透性 → 鈣內流 → 蟲體痙攣麻痺、外皮受損 | 單日 / 短療程；神經囊蟲症使用需小心發炎反應 |
| **Ivermectin** | 糞線蟲、盤尾絲蟲（河盲症）、疥瘡 | 活化蟲體 **glutamate / GABA-gated 氯離子通道** → 過極化 → 麻痺癱瘓 | 治糞線蟲前排除 hyperinfection 風險；哺乳動物血腦障壁擋住藥物故對人相對安全 |
| **Pyrantel** | 蟯蟲、蛔蟲 | **去極化神經肌肉阻斷劑** → 蟲體痙攣性癱瘓後隨蠕動排出 | 常見 **OTC** 藥；與 piperazine 作用相反不可併用 |

- 劑量、療程天數、重複給藥時機**依院規 / 寄生蟲指引 / 需查證**（不同蟲種差異大）

## ⚠️ 副作用（常見 / 嚴重）
- **苯并咪唑（Albendazole / Mebendazole）**：腸胃不適、頭痛；長療程 → **肝功能異常、骨髓抑制（白血球低下）**，需監測 CBC / LFT；囊蟲症治療期腦部發炎
- **Praziquantel**：頭暈、頭痛、腹痛、噁心（部分是蟲體崩解反應）；神經囊蟲症時可誘發癲癇 / 顱內壓升高
- **Ivermectin**：一般耐受良好；盤尾絲蟲治療 → **Mazzotti 反應**（微絲蟲死亡引起搔癢 / 發燒 / 淋巴結腫）；糞線蟲 hyperinfection 相關重症
- **Pyrantel**：輕微腸胃不適、頭暈，多屬輕微（不太吸收）

## 🚫 禁忌症
- 對該藥過敏
- **苯并咪唑類**：孕婦（尤其第一孕期）避免；嚴重肝功能不全慎用
- **Ivermectin**：合併羅阿絲蟲（Loa loa）高微絲蟲血症者→治療可致嚴重腦病，需先評估
- **Pyrantel**：不與 piperazine 併用（作用拮抗）
- 眼部 / 中樞受侵犯的囊蟲症：驅蟲前需影像評估，避免驟然殺蟲引發嚴重發炎

## 🔍 監測 / 交互作用 / 孕哺
- **長療程 Albendazole**（囊蟲症 / 包蟲症）：定期 **CBC + LFT**，警覺骨髓抑制與肝毒性
- **囊蟲症殺蟲反應**：併用**類固醇**降低腦部發炎，必要時抗癲癇藥
- **Ivermectin**：治糞線蟲前評估免疫狀態與 hyperinfection 風險；盤尾絲蟲注意 Mazzotti 反應
- **孕哺**：苯并咪唑孕婦（尤其早期）避免；Praziquantel / Ivermectin 孕哺使用依疫區公衛指引 / 需查證
- **交互作用**：Albendazole 與 Cimetidine / Dexamethasone / Praziquantel 併用可升高血中濃度（治囊蟲時反而有利）；Pyrantel 與 piperazine 拮抗

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位長期使用高劑量類固醇的病人被診斷出糞線蟲症。開立 Ivermectin 前，最需要警覺的是？
- A. 視網膜病變
- B. hyperinfection / 播散性感染
- C. 低血糖
- D. QT 延長

> [!success]- 答案
> **B** — 免疫抑制（尤其長期類固醇）者糞線蟲可自體感染大爆發成 hyperinfection / 播散性感染，甚至合併革蘭氏陰性菌血症，治療與類固醇使用需謹慎評估。

**Q2.** 診斷為神經囊蟲症的病人準備用 Albendazole 驅蟲。為降低治療併發症，通常會加上？
- A. 抗凝血劑
- B. 類固醇（±抗癲癇藥）
- C. 利尿劑
- D. 抗組織胺

> [!success]- 答案
> **B** — 殺蟲後蟲體崩解會引起腦部發炎，可能惡化癲癇或升高顱內壓，故常**併用類固醇**（必要時加抗癲癇藥）。

**Q3.** 關於治療血吸蟲病（吸蟲感染），下列何者為首選藥物？
- A. Mebendazole
- B. Pyrantel
- C. Praziquantel
- D. Ivermectin

> [!success]- 答案
> **C** — Praziquantel 是吸蟲（血吸蟲）與絛蟲的首選，機轉為增加蟲體膜鈣通透性造成痙攣麻痺。
