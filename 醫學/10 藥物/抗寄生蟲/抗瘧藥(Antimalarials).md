# 抗瘧藥（Antimalarials 總覽）

> [!abstract] 一句話：抗瘧藥依「攻擊瘧原蟲哪個生命週期」分工——血液期用 Chloroquine / Artemisinin / Quinine / Mefloquine 殺，肝內休眠體（hypnozoite）只有 **Primaquine** 能根除；重症瘧疾（腦型瘧）是內科急症，首選 **Artesunate**，而 Primaquine 開藥前**必查 G6PD**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **重症瘧疾 / 腦型瘧疾（cerebral malaria）是急症** → 意識改變、抽搐、嚴重貧血、酸中毒、寄生蟲血症高 → 首選 **IV Artesunate**，不要口服拖延
> 2. **Primaquine 開藥前必驗 G6PD** → G6PD 缺乏者會爆發**溶血性貧血**；孕婦禁用（胎兒 G6PD 狀態未知）
> 3. **旅遊史問清楚地區與抗藥型態** → P. falciparum 對 Chloroquine 抗藥性普遍，去抗藥區要換藥；問「去哪、有無預防投藥、返國多久」
> 4. **多種抗瘧藥（Chloroquine / Mefloquine / Quinine）都會 QT 延長** → 併用其他 QT 延長藥物或電解質異常時小心心律不整
> 5. **Mefloquine 有神經精神黑框警語** → 有精神病史 / 癲癇者避免

## 💊 適應症（治療哪些疾病）
- **瘧疾治療與預防**：[[Malaria(瘧疾)]]（P. falciparum / vivax / ovale / malariae / knowlesi）
- **根除治療（radical cure）**：P. vivax / P. ovale 的肝內休眠體 → Primaquine
- 部分藥物跨界用途：Chloroquine 也用於腸外阿米巴、部分自體免疫（如 [[Systemic Lupus Erythematosus(全身性紅斑性狼瘡)]] 用 Hydroxychloroquine，屬同家族）

## 📏 各藥重點（成人）
| 藥物 | 作用期 / 定位 | 機轉 | 關鍵副作用 |
| --- | --- | --- | --- |
| **Chloroquine** | 血液期殺蟲；非抗藥區首選 | 抑制血基質聚合（heme polymerase），使有毒 heme 累積毒殺瘧原蟲 | **視網膜病變**（長期高劑量，需定期眼科檢查）、QT 延長、搔癢 |
| **Primaquine** | **唯一清除肝內休眠體**（hypnozoite），P. vivax/ovale 根除 | 破壞瘧原蟲粒線體 | **溶血（G6PD 缺乏）**、高鐵血紅素血症 |
| **Artemisinin 類（Artesunate）** | **重症瘧 / 多重抗藥瘧首選** | 內過氧橋活化產生**自由基**破壞瘧原蟲蛋白 | 少見神經毒性、血液學異常（遲發性溶血 post-artesunate） |
| **Mefloquine** | 預防用藥選項、抗藥區治療 | 干擾血基質解毒（類似 quinoline） | **神經精神副作用**（噩夢、焦慮、精神病，黑框警語）、QT 延長 |
| **Quinine（奎寧）** | 重症瘧治療備選 | 抑制血基質聚合 | **cinchonism**（耳鳴、頭暈、視力模糊、頭痛）、**低血糖**（刺激胰島素）、QT 延長 |

- **ACT（artemisinin-based combination therapy）**：Artemisinin 類**必須**與另一種長效抗瘧藥合併（如 artemether-lumefantrine），單用易誘發抗藥性
- 劑量細節、療程天數、預防投藥起訖時間**依院規 / CDC 旅遊醫學指引 / 需查證**（隨地區抗藥型態調整）

## ⚠️ 副作用（常見 / 嚴重）
- **Chloroquine**：搔癢（深膚色族群常見）、腸胃不適；長期高劑量 → **不可逆視網膜病變**；過量 → 心臟毒性（QT、傳導阻滯，過量致命）
- **Primaquine**：**溶血性貧血（G6PD 缺乏最危險）**、高鐵血紅素血症、腸胃不適
- **Artesunate / Artemisinin**：一般耐受良好；**遲發性溶血**（治療後 1–3 週，需追蹤 Hb）、罕見神經毒性
- **Mefloquine**：**神經精神症狀**（生動噩夢、焦慮、憂鬱、精神病、癲癇）、頭暈、QT 延長
- **Quinine**：**cinchonism**（耳鳴 / 頭暈 / 視力模糊）、**低血糖**、QT 延長、罕見血栓性血小板低下 / 溶血尿毒（blackwater fever）

## 🚫 禁忌症
- **Primaquine**：**G6PD 缺乏、孕婦、哺乳（嬰兒 G6PD 未知）**
- **Mefloquine**：精神病 / 憂鬱 / 焦慮 / 癲癇病史、心臟傳導異常
- **Chloroquine**：已知視網膜 / 黃斑部病變、QT 延長高風險
- **Quinine**：G6PD 缺乏（可誘發溶血）、視神經炎、重症肌無力、QT 延長高風險
- 共通：對該藥過敏

## 🔍 監測 / 交互作用 / 孕哺
- **G6PD 篩檢**：Primaquine（及高劑量 Quinine / Chloroquine）用前必驗
- **眼科追蹤**：長期 Chloroquine / Hydroxychloroquine → 定期視網膜檢查
- **心電圖 / 電解質**：QT 延長藥物（Chloroquine / Quinine / Mefloquine）併用其他 QT 藥物、低血鉀 / 低血鎂時監測
- **血糖**：Quinine 治療重症瘧時監測**低血糖**（尤其孕婦）
- **孕哺**：Chloroquine 一般可用於孕婦預防；Artesunate 可用於重症孕婦瘧疾（利大於弊）；**Primaquine / 高劑量 Quinine 需謹慎，Primaquine 孕婦禁用**；細節依旅遊醫學指引 / 需查證
- **交互作用**：多藥 QT 加乘；Mefloquine 併其他抗瘧 quinoline 增神經毒性；細節依院規 / 需查證

### 自我測驗（選擇題，答案摺疊）

**Q1.** 28 歲女性自非洲返國後高燒、意識混亂，血液抹片 P. falciparum 寄生蟲血症 8%。最適當的初始治療是？
- A. 口服 Chloroquine
- B. 靜脈 Artesunate
- C. 口服 Primaquine
- D. 口服 Mefloquine

> [!success]- 答案
> **B** — 這是重症 / 腦型瘧疾（高寄生蟲血症＋意識改變），屬急症，首選 **IV Artesunate**；且 P. falciparum 對 Chloroquine 抗藥性普遍，口服藥不足以應付重症。

**Q2.** 一位 P. vivax 瘧疾病人急性期已用 Chloroquine 控制血液期，醫師欲加開 Primaquine 做根除治療。下一步最重要的是？
- A. 直接開藥
- B. 先驗 G6PD
- C. 先做眼科檢查
- D. 先驗血糖

> [!success]- 答案
> **B** — Primaquine 在 G6PD 缺乏者會導致溶血性貧血，開藥前**必須檢查 G6PD**；孕婦則禁用。

**Q3.** 一位有憂鬱症病史、需前往抗藥性瘧疾流行區的旅客，下列預防投藥何者最應避免？
- A. Mefloquine
- B. Atovaquone-proguanil
- C. Doxycycline
- D. 以上皆可

> [!success]- 答案
> **A** — Mefloquine 有神經精神副作用黑框警語（可誘發或惡化憂鬱、焦慮、精神病），有精神病史者應避免，改用其他預防方案。
