# 抗原蟲藥（Antiprotozoal 總覽）

> [!abstract] 一句話：本卡聚焦 Metronidazole 之外的抗原蟲藥——**Tinidazole**（長效版硝基咪唑，可單劑）、**Pentamidine**（PJP 與昏睡病備選，會低血糖 / 低血壓）、**Paromomycin**（口服不吸收胺基糖苷，打腸內阿米巴）；Metronidazole 有獨立卡。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **Pentamidine 會致命性低血糖** → 也可能之後轉高血糖 / 胰島素依賴糖尿病；用藥期間監測血糖
> 2. **Pentamidine IV 快速輸注 → 嚴重低血壓** → 一定要**慢速輸注**並監測血壓；也有 QT 延長、腎毒性、胰臟炎
> 3. **Tinidazole / Metronidazole 服藥期間及停藥後禁酒** → disulfiram-like reaction（噁心、嘔吐、潮紅、心悸）；Tinidazole 停藥後**至少 72 小時**內避酒
> 4. **PJP 首選是 TMP-SMX**，Pentamidine 是過敏 / 禁忌時的替代 → 不要一線就上 Pentamidine
> 5. **Paromomycin 口服幾乎不吸收** → 適合腸腔內原蟲（無症狀阿米巴帶原者），全身性阿米巴（肝膿瘍）不能只靠它

## 💊 適應症（治療哪些疾病）
- **Tinidazole**：陰道滴蟲病（trichomoniasis）、[[Giardiasis(梨形鞭毛蟲病)]]、[[Amebiasis(阿米巴病)]]、細菌性陰道炎
- **Pentamidine**：[[Pneumocystis Pneumonia(肺囊蟲肺炎 PJP)]]（TMP-SMX 過敏 / 禁忌時，吸入或 IV）、非洲昏睡病（早期）、利什曼原蟲病
- **Paromomycin**：腸內阿米巴病（無症狀囊體帶原者的 luminal agent）、[[Cryptosporidiosis(隱孢子蟲病)]]、部分利什曼原蟲病

## 📏 各藥重點（成人）
| 藥物 | 定位 | 機轉 | 關鍵風險 |
| --- | --- | --- | --- |
| **Tinidazole** | 同 Metronidazole 家族，半衰期更長 → **可單一劑量** | nitroimidazole 於厭氧環境還原成毒性中間物破壞 DNA | disulfiram 反應（禁酒）、金屬味、周邊神經病變（長期） |
| **Pentamidine** | PJP / 昏睡病 / 利什曼**備選** | 干擾原蟲 DNA / RNA / 蛋白合成（機轉多元） | **低血糖 → 高血糖**、**低血壓（快速輸注）**、QT 延長、腎毒性、胰臟炎、白血球 / 血小板低下 |
| **Paromomycin** | 口服**不吸收**胺基糖苷，腸腔內原蟲 | 抑制原蟲核糖體蛋白質合成（30S） | 全身副作用少（不吸收）；主要腸胃不適；腸黏膜受損時可能吸收增腎 / 耳毒性 |

- 給藥途徑（吸入 vs IV vs 口服）、劑量、療程**依院規 / 感染科指引 / 需查證**

## ⚠️ 副作用（常見 / 嚴重）
- **Tinidazole**：噁心、金屬味、頭痛、頭暈；**禁酒（disulfiram-like）**；長期 / 高劑量 → 周邊神經病變、罕見中樞神經毒性
- **Pentamidine**：
  - **低血糖（可致命）**，之後可能高血糖 / 糖尿病
  - **低血壓**（IV 快速輸注）、**QT 延長 / 心律不整**
  - **腎毒性**、**胰臟炎**、白血球 / 血小板低下
  - 吸入劑型：支氣管痙攣、咳嗽（可先給支氣管擴張劑）
- **Paromomycin**：腹瀉、腹痛、噁心（口服局部作用為主）；腸黏膜嚴重受損時全身吸收增加耳 / 腎毒性風險

## 🚫 禁忌症
- 對該藥過敏
- **Tinidazole**：懷孕第一孕期避免（同 nitroimidazole 家族考量）、正在飲酒 / 用 disulfiram
- **Pentamidine**：需謹慎於低血糖高風險、QT 延長、腎功能不全、胰臟炎病史者（相對禁忌，權衡使用）
- **Paromomycin**：腸阻塞；已知胺基糖苷過敏

## 🔍 監測 / 交互作用 / 孕哺
- **Pentamidine**：用藥全程監測**血糖（低→高）**、**血壓**（IV 慢輸）、**心電圖 QT**、腎功能、血糖 / 血球、脂肪酶（胰臟炎）
- **Tinidazole**：避免酒精及含酒精製劑（含停藥後 72 小時）；與 Warfarin 併用可增強抗凝（同 metronidazole 家族）→ 監測 INR
- **Paromomycin**：口服全身吸收少，交互作用少；腸壁受損時留意耳 / 腎毒性
- **孕哺**：nitroimidazole 類第一孕期避免；Pentamidine / Paromomycin 孕哺依感染科評估 / 需查證
- **QT 加乘**：Pentamidine 與其他 QT 延長藥物併用小心

## 🔗 相關

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位 AIDS 病人罹患 PJP，對 TMP-SMX 嚴重過敏，改用 Pentamidine IV。輸注過程中最需要即時監測的兩項是？
- A. 視力與聽力
- B. 血糖與血壓
- C. 體溫與呼吸
- D. 尿量與體重

> [!success]- 答案
> **B** — Pentamidine 可致命性低血糖，且快速輸注會造成嚴重低血壓，兩者都需即時監測（另需 QT、腎功能等）。

**Q2.** 一位陰道滴蟲病病人以單劑 Tinidazole 治療。衛教時最重要的一項是？
- A. 多喝水
- B. 服藥期間及停藥 72 小時內禁酒
- C. 避免日曬
- D. 每天量血壓

> [!success]- 答案
> **B** — Tinidazole 屬 nitroimidazole，與酒精併用會產生 disulfiram-like reaction（噁心、嘔吐、潮紅、心悸），需避酒至停藥後至少 72 小時。

**Q3.** 針對無症狀的腸內阿米巴囊體帶原者，最適合的 luminal（腸腔）用藥是？
- A. Pentamidine
- B. Tinidazole
- C. Paromomycin
- D. Chloroquine

> [!success]- 答案
> **C** — Paromomycin 口服不吸收，可在腸腔達高濃度清除囊體，是清除無症狀帶原者常用的 luminal agent。
