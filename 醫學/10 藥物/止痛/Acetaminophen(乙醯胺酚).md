# Acetaminophen（乙醯胺酚 / 普拿疼）

> [!abstract] 一句話：中樞性**止痛、退燒**藥（機轉未完全明，主要作用於中樞，**幾乎無周邊抗發炎、不傷胃、不抑制血小板**），是**WHO 止痛階梯第一階**與退燒首選。安全治療劑量下很安全，但**過量→肝毒性（肝壞死）**是毒理學重點，解毒劑為 **NAC（N-acetylcysteine）**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **每日上限 4 g/day（成人）**；慢性酗酒 / 肝病 / 營養不良 / 消瘦者上限降至 **2–3 g/day**
> 2. **過量 = 肝衰竭風險**：早期（<24h）症狀輕微易被忽略 → 用 **Rumack-Matthew nomogram**（以攝入後 4h 血中濃度）判斷是否給 NAC
> 3. **NAC 越早給越有效**（<8–10h 效果最佳），懷疑就給、別等到 GOT/GPT 飆高
> 4. **隱藏來源**：許多綜合感冒藥、複方止痛藥含 acetaminophen → 病人不自覺累加超量

## 💊 適應症（治療哪些疾病）
- **輕中度疼痛**：頭痛、肌肉痛、關節痛、術後疼痛（常與 opioid 合併省 opioid 劑量）
- **退燒（antipyretic）**：各種發燒（兒童、孕婦皆為首選）
- 適合**不能用 NSAID**者（消化性潰瘍、腎功能不全、抗凝血、AERD、孕婦）

## 📏 劑量 / 用法（成人）
| 情境 | 劑量 |
| --- | --- |
| 一般止痛退燒 | 500–1000 mg PO Q4–6H PRN |
| 每日上限（健康成人） | **≤ 4 g/day** |
| 高風險（酗酒/肝病/消瘦） | **≤ 2–3 g/day** |
| 兒童 | 10–15 mg/kg/dose Q4–6H（單日勿超上限） |
- **肝**是主要考量（非腎）；嚴重肝病減量或避免
- 靜脈型（IV acetaminophen）用於無法口服者，注意總劑量換算

## ⚠️ 副作用（常見 / 嚴重）
- **治療劑量**：一般耐受良好，副作用少（這是它的優點）
- **嚴重（過量）**：
  - **肝毒性 / 猛爆性肝衰竭**（NAPQI 中間產物耗竭 glutathione → 肝細胞壞死）
  - 過量四階段：①<24h 輕微/無症狀 ②24–72h 肝指數上升、右上腹痛 ③72–96h 肝衰竭高峰 ④>96h 恢復或死亡
  - 罕見：嚴重皮膚反應（SJS/TEN，極罕見）

## 🚫 禁忌症
- 對 acetaminophen 過敏
- 嚴重活動性肝病 / 肝衰竭
- 相對：慢性酗酒、營養不良（降低上限、謹慎）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：總每日劑量（含複方隱藏來源）、肝功能（高風險或過量時）、過量時 acetaminophen 血中濃度 + nomogram
- **交互作用**：
  - **酒精 / CYP2E1 誘導劑（慢性）** → ↑ NAPQI 生成、↑ 肝毒性
  - **Warfarin**：長期高劑量可能輕度↑ INR
- **孕哺**：**孕期止痛退燒首選**（相對最安全）；哺乳可用

## 🔗 相關
- 對照：[[NSAID(非類固醇消炎藥)]]（有抗發炎但傷胃/腎；acetaminophen 無抗發炎但不傷胃）
- 階梯：WHO 止痛階梯第一階；重度可加 [[Morphine(嗎啡)]] / [[Opiate(鴉片類)]]
- 解毒：NAC（N-acetylcysteine）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位病人服用大量普拿疼企圖自傷，就診時（攝入後 6 小時）僅輕微噁心、肝指數正常。最適當處置是？
- A. 因肝指數正常可放心離院
- B. 以攝入後 4h 濃度對 Rumack-Matthew nomogram 判斷，需要就及早給 NAC
- C. 只給制酸劑
- D. 等肝指數上升再處理

> [!success]- 答案
> **B** — 早期肝指數常正常，需用 **nomogram** 判斷，達治療線就**及早給 NAC**（<8–10h 最有效），不可因症狀輕就放心。

**Q2.** 關於 acetaminophen 的敘述，何者正確？
- A. 有強力抗發炎作用
- B. 會抑制血小板增加出血
- C. 不傷胃、不抑制血小板，適合不能用 NSAID 者
- D. 過量主要傷腎

> [!success]- 答案
> **C** — Acetaminophen **不傷胃、不抑制血小板**，適合潰瘍/抗凝血/腎不全者；無明顯抗發炎，過量主要傷**肝**。

**Q3.** 慢性酗酒的病人使用 acetaminophen 時，建議每日上限與原因為？
- A. 維持 4 g/day，無影響
- B. 降至 2–3 g/day，因 CYP2E1 誘導增加肝毒性
- C. 可增加到 6 g/day
- D. 完全禁用退燒

> [!success]- 答案
> **B** — 慢性酗酒 CYP2E1 誘導使 NAPQI 增加、肝毒性上升，上限應**降至 2–3 g/day**。
