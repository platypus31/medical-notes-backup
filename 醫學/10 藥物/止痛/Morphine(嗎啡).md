# Morphine（嗎啡）

> [!abstract] 一句話：**μ-鴉片受體（mu）促效劑**的原型藥，強效止痛，用於**中重度急慢性疼痛、癌痛、急性肺水腫、心肌梗塞疼痛**。核心風險是**呼吸抑制**（致死原因）；解毒劑 **Naloxone**。**腎衰竭者慎用**（活性代謝物 M6G 累積致呼吸抑制延長）。屬管制藥。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **呼吸抑制**是致死主因 → 監測呼吸次數、血氧、意識；過量三聯徵 **針狀瞳孔（pinpoint pupil）+ 呼吸抑制 + 意識下降**
> 2. **解毒劑 Naloxone**（0.04–0.4 mg IV 起、可重複）；naloxone 半衰期短，長效 opioid 可能需重複給/持續輸注
> 3. **腎衰竭**：**活性代謝物 morphine-6-glucuronide（M6G）累積** → 呼吸抑制延長 → 腎不全者換 fentanyl / hydromorphone 或大幅減量
> 4. **低血壓**（釋放組織胺、降前負荷）、**便祕（幾乎必發、不會耐受）**、噁心嘔吐
> 5. 給藥 titrate 到止痛，勿一次大劑量；合併 benzodiazepine/酒精 → 呼吸抑制加乘

## 💊 適應症（治療哪些疾病）
- **中重度疼痛**：術後、外傷、[[Myocardial Infarction(心肌梗塞)]] 疼痛
- **癌症疼痛**（WHO 止痛階梯第三階，強效 opioid）
- **急性肺水腫 / 心衰**（降前負荷、緩解呼吸困難與焦躁——現代較保守使用）
- 安寧緩和的呼吸困難（dyspnea）與疼痛

## 📏 劑量 / 用法（成人 · titrate）
| 情境 | 用法 |
| --- | --- |
| 急性中重度痛（IV） | 2–4 mg IV 慢推，每 5–15 分鐘依反應追加（titrate） |
| 口服（慢性/癌痛） | 速效 + 長效（緩釋）組合，依需求換算 |
| PCA（病人自控） | 依醫囑設定 bolus/lockout |
- **腎功能不全**：減量或改用 fentanyl/hydromorphone（避 M6G 累積）
- **老人、肝病、呼吸儲備差**：減量、密切監測
- 換算不同 opioid 時用 equianalgesic dose，並打折避免過量

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：**便祕（不會耐受，需常規軟便劑）**、噁心嘔吐、鎮靜、搔癢（組織胺）、尿滯留、縮瞳
- **嚴重**：
  - **呼吸抑制**（致死）
  - 低血壓（組織胺釋放、交感抑制）
  - 意識障礙、譫妄
  - **依賴 / 成癮 / 耐受性**；突然停用→戒斷症狀
  - 過量：pinpoint pupil + 呼吸抑制 + 昏迷

## 🚫 禁忌症
- 對嗎啡 / opioid 過敏
- **急性呼吸抑制、嚴重氣喘發作、呼吸衰竭風險**
- 麻痺性腸阻塞（paralytic ileus）
- 相對禁忌：嚴重腎功能不全（改藥）、頭部外傷/顱內壓升高（掩蓋神經評估、CO₂ 上升）、未控制低血壓

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：呼吸次數、SpO₂、意識、血壓、疼痛分數、腸道功能（便祕）
- **交互作用**：
  - **中樞抑制劑（benzodiazepine、酒精、其他 opioid、gabapentinoid）→ 呼吸抑制加乘**（重要！）
  - MAOI（尤其 pethidine 類）→ serotonin syndrome / 高熱
- **孕哺**：孕期長期使用 → 新生兒戒斷症候群（NAS）、生產前呼吸抑制；哺乳少量分泌需謹慎

### 自我測驗（選擇題，答案摺疊）

**Q1.** 使用 opioid 後意識不清、呼吸 6 次/分、瞳孔如針尖。最優先處置是？
- A. 給鎮靜劑
- B. 給 naloxone 並支持呼吸
- C. 觀察即可
- D. 給更多止痛藥

> [!success]- 答案
> **B** — 典型 opioid 過量（pinpoint pupil + 呼吸抑制 + 意識下降），優先**給 naloxone** 並維持呼吸道/通氣；naloxone 半衰期短需注意重複給。

**Q2.** 慢性腎衰竭病人需強效止痛，為何 morphine 需特別小心？
- A. 會傷胃
- B. 活性代謝物 M6G 累積導致呼吸抑制延長
- C. 會低血糖
- D. 沒有影響

> [!success]- 答案
> **B** — Morphine 的活性代謝物 **M6G 經腎排除**，腎衰竭時累積造成呼吸抑制延長，宜改 fentanyl/hydromorphone 或減量。

**Q3.** 關於 morphine 的副作用，下列何者**不會產生耐受性**、需常規預防？
- A. 噁心
- B. 鎮靜
- C. 便祕
- D. 搔癢

> [!success]- 答案
> **C** — **便祕**幾乎必發且**不會產生耐受**，使用 opioid 應常規併用軟便劑。
