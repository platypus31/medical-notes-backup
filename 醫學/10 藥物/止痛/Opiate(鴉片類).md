# Opiate（鴉片類 / 嗎啡類）

> [!abstract] 一句話：作用於**鴉片受體（μ、κ、δ）**的一大類止痛藥（WHO 止痛階梯第二三階）。共通效果：止痛、鎮靜、**呼吸抑制、便祕、縮瞳、依賴成癮**。依效力/特性選藥（弱→強、腎肝安全性、特殊風險）。共同解毒劑 **Naloxone**。原型藥見 [[Morphine(嗎啡)]]。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **呼吸抑制**是所有 opioid 的共通致死風險；合併 benzodiazepine/酒精/gabapentinoid 加乘
> 2. **各藥特殊地雷**：**Pethidine(Meperidine)** 代謝物 normeperidine 致 **癲癇**（腎不全累積）、與 **MAOI → serotonin syndrome**；**Codeine/Tramadol** 經 **CYP2D6** 代謝，超快代謝者→嗎啡過量（兒童/哺乳婦禁）；**Tramadol** 降癲癇閾值 + serotonin syndrome
> 3. **腎衰竭**：morphine（M6G）、pethidine（normeperidine）累積 → 選 **fentanyl / hydromorphone**
> 4. **便祕不會耐受**、需常規軟便劑；**縮瞳（pinpoint）** 是中毒線索
> 5. 管制藥，注意成癮/濫用/轉移（尤其長期 oxycodone）

## 💊 適應症（治療哪些疾病）
- **中重度疼痛**（術後、外傷、癌痛、心肌梗塞）
- **止咳**（codeine，低劑量）
- **止瀉**（loperamide 為周邊作用 opioid）
- **麻醉/術中鎮痛**（fentanyl、remifentanil）
- **戒癮替代治療**（methadone、buprenorphine）

## 📏 常見 opioid 比較（概念）
| 藥 | 強度/角色 | 特別注意 |
| --- | --- | --- |
| **Codeine** | 弱（前驅藥，經 CYP2D6→morphine） | 超快代謝者過量；兒童/哺乳婦禁 |
| **Tramadol** | 弱-中（+ 抑制 serotonin/NE 回收） | 降癲癇閾值、serotonin syndrome |
| **Morphine** | 強（原型） | M6G 腎累積（見子卡） |
| **Oxycodone** | 強（口服） | 濫用風險高 |
| **Hydromorphone** | 強 | 腎不全相對可（較 morphine 安全） |
| **Fentanyl** | 強（脂溶、快、貼片/IV） | 腎不全首選之一；貼片起效慢勿急性 titrate |
| **Pethidine(Meperidine)** | 中 | normeperidine 致癲癇、MAOI 禁、現少用 |
| **Methadone** | 強（長效） | QT 延長、半衰期長難換算 |
| **Buprenorphine** | 部分促效 | 天花板效應、戒癮用 |
- 換藥用 equianalgesic dose 並打折（避免不完全交叉耐受致過量）

## ⚠️ 副作用（共通 / 嚴重）
- **共通**：便祕（不耐受）、噁心嘔吐、鎮靜、縮瞳、尿滯留、搔癢、低血壓
- **嚴重**：**呼吸抑制（致死）**、依賴/成癮/耐受、戒斷症候群、譫妄
- **特殊**：pethidine→癲癇；tramadol/pethidine→serotonin syndrome；methadone→QT 延長

## 🚫 禁忌症
- 對特定 opioid 過敏
- 急性呼吸抑制、嚴重氣喘發作、呼吸衰竭風險、麻痺性腸阻塞
- **Codeine**：12 歲以下兒童、扁桃腺/腺樣體術後兒童、哺乳婦（CYP2D6 超快代謝致嬰兒嗎啡過量）
- **Pethidine + MAOI**：禁（serotonin syndrome / 高熱）
- 相對：腎衰竭（選對藥）、頭部外傷/顱內壓升高

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：呼吸、SpO₂、意識、疼痛分數、便祕、（methadone）QT/ECG
- **交互作用**：
  - **中樞抑制劑（BZD、酒精、gabapentinoid）→ 呼吸抑制加乘**
  - **serotonergic 藥（SSRI/SNRI/MAOI）+ tramadol/pethidine → serotonin syndrome**
  - **CYP2D6**（codeine/tramadol/oxycodone）、**CYP3A4**（fentanyl/methadone）交互作用
- **孕哺**：長期使用→新生兒戒斷症候群（NAS）；codeine 哺乳禁

### 自我測驗（選擇題，答案摺疊）

**Q1.** 腎功能不全的癌痛病人使用某 opioid 後反覆抽搐。最可能是哪種藥的代謝物所致？
- A. Fentanyl
- B. Pethidine(Meperidine) 的 normeperidine
- C. Hydromorphone
- D. Buprenorphine

> [!success]- 答案
> **B** — **Pethidine** 的代謝物 **normeperidine** 在腎不全累積會**誘發癲癇**，故腎不全與需長期止痛者避用。

**Q2.** 哺乳中的母親服用下列哪種止痛藥，可能因 CYP2D6 超快代謝使嬰兒發生嗎啡中毒？
- A. Acetaminophen
- B. Ibuprofen
- C. Codeine
- D. Naloxone

> [!success]- 答案
> **C** — **Codeine** 經 CYP2D6 轉為 morphine，超快代謝母親可致嬰兒**嗎啡過量**，哺乳婦與兒童禁用。

**Q3.** 一位服用 SSRI 的病人加用 tramadol 後出現躁動、發抖、高熱、反射亢進。最可能診斷是？
- A. 低血糖
- B. Serotonin syndrome
- C. 敗血症
- D. 甲狀腺風暴

> [!success]- 答案
> **B** — Tramadol 具 serotonergic 作用，與 SSRI 併用可致 **serotonin syndrome**（躁動、震顫、高熱、反射亢進、陣攣）。
