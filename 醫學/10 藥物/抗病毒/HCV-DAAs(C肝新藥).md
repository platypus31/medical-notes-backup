# HCV DAAs（C 肝新藥大類卡）

> [!abstract] 一句話：**DAA（Direct-Acting Antivirals）** 直接打 HCV 生活史的關鍵蛋白，8–12 週口服療程、**治癒率 SVR > 95%**，把 C 肝從「長期干擾素注射」變成「口服短療程可治癒」。依 **genotype** 選方案；Ribavirin 已退居特定合併角色（致畸胎、溶血）。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **Amiodarone + Sofosbuvir → 嚴重心搏過緩 / 心律不整**（FDA 警語）→ 儘量避免併用，必要時需嚴密心律監測
> 2. **Ribavirin 致畸胎（Category X）**：**男女雙方**在用藥期間及**停藥後 6 個月**都須嚴格避孕
> 3. **Ribavirin → 溶血性貧血**（劑量依賴）→ 監測 Hb，貧血者慎用
> 4. **HBV 再活化風險**：DAA 治療 HCV 期間，若合併 HBV 感染可能誘發 **B 肝再活化**（甚至猛爆性肝炎）→ 治療前須篩 HBsAg/anti-HBc，必要時預防性抗 HBV
> 5. **藥物交互作用多**：Ledipasvir/Sofosbuvir 受 **制酸劑（PPI/H2）、amiodarone、某些抗癲癇/抗結核藥（CYP/P-gp 誘導劑如 rifampin）** 影響，開藥前務必查交互作用

## 💊 適應症（治療哪些疾病）
- **慢性 [[Hepatitis C(C型肝炎)]] 感染**（各 genotype，依方案涵蓋範圍）
- 目標：達成 **SVR（sustained virologic response，持續病毒學反應）**＝功能性治癒，降低肝硬化 / [[Hepatocellular Carcinoma(肝細胞癌)]] 風險
- 涵蓋代償性肝硬化、部分失代償肝硬化（方案選擇不同）、肝移植前後（依院規/需查證）

## 📏 劑量 / 用法（成人）
| 藥物 / 複方 | 機轉分類 | 用法重點 |
| --- | --- | --- |
| **Sofosbuvir** | **NS5B polymerase 抑制劑**（核苷類） | 常為骨幹，多與 NS5A 抑制劑複方 |
| **Ledipasvir** | **NS5A 抑制劑** | 與 Sofosbuvir 複方＝**Harvoni（夏奉寧）** |
| **Daclatasvir** | **NS5A 抑制劑** | 常與 Sofosbuvir 併用 |
| **Ribavirin** | 廣效核苷類似物（輔助） | 僅特定 genotype/肝硬化/難治情況加用 |
| 療程 | — | 一般 **8–12 週口服**（依 genotype、肝硬化、治療史調整；依院規/需查證） |
- **依 genotype（1–6 型）選方案**：不同 DAA 組合對各 genotype 涵蓋度不同；近年泛基因型（pangenotypic）方案（如 sofosbuvir/velpatasvir、glecaprevir/pibrentasvir）簡化選擇
- **Ledipasvir 吸收依賴胃酸** → 併用制酸劑需調整服藥時間/劑量（依院規/需查證）

## ⚠️ 副作用（常見 / 嚴重）
- **DAA 本身（Sofosbuvir/Ledipasvir/Daclatasvir）**：一般耐受良好；常見疲倦、頭痛、噁心、失眠
- **Ribavirin（毒性主要來源）**：
  - **溶血性貧血**（劑量依賴，需監測 Hb）
  - **致畸胎（Category X）**
  - 咳嗽、皮疹、疲倦、高尿酸
- **嚴重**：與 amiodarone 併用之嚴重心搏過緩；合併 HBV 者之 **HBV 再活化 / 肝衰竭**

## 🚫 禁忌症
- **Ribavirin**：孕婦及其伴侶未避孕、嚴重貧血、嚴重腎功能不全（需大幅調整）
- **強效 CYP/P-gp 誘導劑併用**（如 rifampin、carbamazepine、St. John's Wort）會降低 DAA 濃度 → 避免
- 對成分過敏
- Sofosbuvir 在**嚴重腎功能不全（eGFR 低）**使用需謹慎評估（依院規/需查證）

## 🔍 監測 / 交互作用 / 孕哺
- **治療前**：HCV genotype/病毒量、肝纖維化/肝硬化評估、**HBsAg / anti-HBc（防 HBV 再活化）**、腎功能、用藥史（查交互作用）
- **治療中/後**：療效追 HCV RNA，療程結束後 **12 週測 SVR12**；Ribavirin 者追 Hb
- **交互作用**：
  - **Amiodarone**（嚴重心搏過緩，避免）
  - **制酸劑 PPI/H2**（影響 Ledipasvir 吸收）
  - **CYP/P-gp 誘導劑**（rifampin、抗癲癇藥、St. John's Wort → 降 DAA 濃度）
  - 部分 statin（如 rosuvastatin）濃度升高需注意
- **孕哺**：含 Ribavirin 方案孕婦絕對禁（致畸胎，男女停藥後 6 個月避孕）；純 DAA 之孕期資料有限，非必要延後治療

## 🔗 相關
- 疾病：[[Hepatitis C(C型肝炎)]]、[[Hepatocellular Carcinoma(肝細胞癌)]]

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位慢性 C 肝病人開始 Sofosbuvir/Ledipasvir，同時因心房顫動長期服用 Amiodarone。最需警覺的風險是？
- A. 溶血性貧血
- B. 嚴重心搏過緩 / 心律不整
- C. 致畸胎
- D. 甲狀腺風暴

> [!success]- 答案
> **B** — Sofosbuvir 與 Amiodarone 併用有嚴重心搏過緩的 FDA 警語，應儘量避免併用，必要時嚴密心律監測。

**Q2.** 關於 Ribavirin，下列敘述何者正確？
- A. 是 NS5A 抑制劑
- B. 無致畸胎性可安心用於孕婦
- C. 主要毒性是溶血性貧血且為 Category X 致畸胎，男女停藥後仍須避孕 6 個月
- D. 現在是所有 C 肝方案的必需骨幹

> [!success]- 答案
> **C** — Ribavirin 主要造成劑量依賴的溶血性貧血、且屬 Category X 致畸胎；現多退居特定合併角色而非必需骨幹。

**Q3.** 準備替一位 C 肝病人開 DAA，下列治療前檢查何者「不可省略」以避免嚴重併發？
- A. 心臟超音波
- B. HBsAg / anti-HBc 篩檢（防 HBV 再活化）
- C. 骨密度
- D. 甲狀腺功能

> [!success]- 答案
> **B** — DAA 治療 HCV 可能誘發合併 HBV 感染者的 B 肝再活化，治療前必須篩 HBV，必要時給預防性抗 HBV。
