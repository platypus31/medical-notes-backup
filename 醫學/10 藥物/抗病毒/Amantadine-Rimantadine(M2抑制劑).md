# Amantadine + Rimantadine（M2 抑制劑）

> [!abstract] 一句話：阻斷**流感 A 病毒 M2 離子通道** → 抑制病毒**脫殼（uncoating）**。但因**抗藥性普遍**，現今流感治療**幾乎已棄用**（改用神經胺酸酶抑制劑如 Oseltamivir）。**Amantadine** 另有神經科用途——**帕金森氏症**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **抗藥性普遍 → 流感治療幾乎棄用**（這是最常考的重點）：目前流感 A 對 M2 抑制劑高度抗藥，且對流感 B **本來就無效**（B 型無 M2 標的）
> 2. **中樞神經副作用**：幻覺、意識混亂、失眠、頭暈 → **老人與腎功能不全者尤其嚴重**（藥物蓄積）
> 3. **腎功能不全需減量**：Amantadine 主要**經腎排除未代謝**，腎不全會蓄積 → CNS 毒性/譫妄（Rimantadine 經肝代謝，腎影響較小）
> 4. **突然停用 Amantadine（帕金森病人）** → 可能誘發類 NMS（惡性症候群）或帕金森症狀急性惡化，勿驟停
> 5. **抗膽鹼副作用 + livedo reticularis（網狀青斑）** 是 Amantadine 特色表現

## 💊 適應症（治療哪些疾病）
- **（歷史用途）流感 A 預防/治療**：因抗藥性現已**不建議常規使用**；現行首選為神經胺酸酶抑制劑 [[Oseltamivir(克流感)]]
- **Amantadine — 帕金森氏症**：增加多巴胺釋放/減少再回收 + NMDA 拮抗 → 改善僵硬/動作遲緩，尤其**改善左旋多巴引起的異動症（levodopa-induced dyskinesia）**
- **Amantadine — 藥物誘發之錐體外症狀（EPS）** 輔助治療（部分情況）
- 相關參照：[[Antiparkinson Drugs(帕金森藥物總覽)]]

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 帕金森（Amantadine） | **100 mg PO BID**（常見起始，依反應調整）；依院規/需查證 |
| 流感（歷史） | 已不建議常規使用（抗藥性）；若特殊情境依院規/需查證 |
| 腎功能不全 | **依 CrCl 減量/延長間隔**（Amantadine 經腎排除，蓄積易致 CNS 毒性）；依院規/需查證 |
- **Rimantadine** 主要經**肝代謝**，腎不全時劑量調整需求較 Amantadine 小，CNS 副作用亦較少
- 老人劑量宜保守，密切觀察譫妄/幻覺

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：噁心、頭暈、失眠、緊張、口乾（抗膽鹼）
- **中樞神經**（Amantadine 較明顯）：**幻覺、意識混亂、譫妄**（老人/腎不全高風險）
- **Livedo reticularis（網狀青斑）**：Amantadine 長期使用之特徵性皮膚表現，常伴下肢水腫
- **抗膽鹼性副作用**：口乾、便秘、尿滯留、視力模糊
- **嚴重/罕見**：驟停引發類 NMS、心律不整、癲癇閾值下降

## 🚫 禁忌症
- 對藥物過敏
- 嚴重腎功能不全未調整劑量（Amantadine 蓄積）
- 未控制之精神病/嚴重幻覺病史者慎用（CNS 副作用惡化）
- 癲癇病史、嚴重心衰/心律不整者慎用
- 孕哺：資料有限，非必要避免（依院規/需查證）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：腎功能（導引劑量）、精神狀態（幻覺/譫妄，尤其老人）、下肢皮膚/水腫（livedo reticularis）
- **交互作用**：
  - 其他**抗膽鹼藥** → 抗膽鹼作用加乘（譫妄、尿滯留）
  - **CNS 興奮劑 / 抗帕金森藥** → 加乘 CNS 刺激或幻覺
  - 酒精 → CNS 副作用增強
  - 利尿劑（如 HCTZ/triamterene）可能↓ Amantadine 腎排除 → 蓄積
- **孕哺**：資料有限，非必要避免；哺乳不建議

## 🔗 相關

### 自我測驗（選擇題，答案摺疊）

**Q1.** 關於 M2 抑制劑（Amantadine/Rimantadine）用於流感治療，下列何者正確？
- A. 是目前流感 A 與 B 的首選
- B. 因抗藥性普遍，現已幾乎棄用於流感治療
- C. 對流感 B 特別有效
- D. 無任何中樞神經副作用

> [!success]- 答案
> **B** — M2 抑制劑因流感 A 抗藥性普遍已幾乎棄用、對流感 B 本就無效；現行首選為神經胺酸酶抑制劑（Oseltamivir）。

**Q2.** 一位 78 歲腎功能不全的病人服用 Amantadine 後出現幻覺與意識混亂。最可能的機轉是？
- A. 藥物經腎排除，腎不全導致蓄積產生 CNS 毒性
- B. 過敏反應
- C. 低血糖
- D. 藥物完全無關

> [!success]- 答案
> **A** — Amantadine 主要經腎排除未代謝，腎不全與老人易蓄積造成幻覺/譫妄等 CNS 毒性，需依 CrCl 減量。

**Q3.** 下列哪一項是 Amantadine 特徵性的長期使用表現？
- A. 牙齦增生
- B. Livedo reticularis（網狀青斑）
- C. 齒齦出血
- D. 手掌紅斑

> [!success]- 答案
> **B** — Livedo reticularis（網狀青斑）合併下肢水腫是 Amantadine 長期使用的特徵性皮膚副作用。
