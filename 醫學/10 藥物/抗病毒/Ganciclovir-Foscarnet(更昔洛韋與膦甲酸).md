# Ganciclovir & Foscarnet（更昔洛韋與膦甲酸）

> [!abstract] 一句話：兩者都是**巨細胞病毒（CMV）**感染（尤其移植後、AIDS）的主力用藥。**Ganciclovir** 是 aciclovir 的近親核苷類似物，需病毒 **UL97 kinase** 首次磷酸化 → 抑制 CMV DNA polymerase，主要毒性是**骨髓抑制**；**Foscarnet** 直接結合 DNA polymerase 的焦磷酸位（**不需病毒 kinase 活化**），是 ganciclovir 抗藥或無法耐受骨髓抑制時的替代，主要毒性是**腎毒性 + 電解質全面紊亂**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **Ganciclovir = 骨髓抑制之王**：中性球低下、血小板低下、貧血。用藥期間每週追 CBC；ANC < 500 或血小板 < 25,000 常需停藥或減量。與其他骨髓抑制藥（zidovudine、化療、mycophenolate）疊加更危險
> 2. **Foscarnet = 腎毒性 + 電解質災難**：低鈣、低鎂、低鉀、低（或高）磷都會發生。**低游離鈣可致抽搐、QT 延長、心律不整、感覺異常**。給藥前後必須**大量水化（saline pre-hydration）**並每次追腎功能與電解質
> 3. **Foscarnet 誘發癲癇/抽搐**：常繼發於嚴重低鈣血症；離子鈣（ionized Ca）比總鈣更能反映真實狀態
> 4. **Ganciclovir 具致畸 / 致癌 / 抑制精子生成**：孕婦禁用（動物致畸），育齡男女需避孕；細胞毒藥物調配要防護
> 5. **口服選纈化前驅藥**：Valganciclovir（口服前驅藥，生體可用率遠高於口服 ganciclovir）用於維持治療；Foscarnet 只有靜脈劑型，且**需中央靜脈或大量稀釋**（周邊靜脈刺激、生殖器潰瘍）

## 💊 適應症（治療哪些疾病）
- **CMV 視網膜炎 [[CMV Retinitis(巨細胞病毒視網膜炎)]]**（AIDS CD4 < 50 典型；移植後）— 眼科急症，會失明
- **移植後 CMV 疾病 / 預防**（實體器官移植、造血幹細胞移植 [[Bone Marrow Transplant(造血幹細胞移植)]]）：CMV 肺炎、腸炎、肝炎、病毒血症
- **Ganciclovir/Valganciclovir**：CMV 預防（prophylaxis）與搶先治療（pre-emptive）
- **Foscarnet**：① Ganciclovir 抗藥性 CMV（UL97 突變株）② 無法耐受骨髓抑制者 ③ **Aciclovir 抗藥性 HSV / VZV**（因不需病毒 kinase 活化，可對付 TK-deficient 株）

## 📏 劑量 / 用法（成人）
| 藥 / 用途 | 劑量 |
| --- | --- |
| Ganciclovir 誘導期（CMV 疾病） | **5 mg/kg IV Q12H × 14–21 天** |
| Ganciclovir 維持期 | 5 mg/kg IV QD |
| Valganciclovir 誘導 | 900 mg PO BID（隨餐） |
| Valganciclovir 維持 / 預防 | 900 mg PO QD |
| Foscarnet 誘導期 | **60 mg/kg IV Q8H 或 90 mg/kg Q12H × 14–21 天** |
| Foscarnet 維持期 | 90–120 mg/kg IV QD |
- **腎功能**：兩者皆主要經腎排除，**依 CrCl 嚴格調整劑量**（⚠️依院規/藥典 nomogram 查證）；Foscarnet 腎損時尤需頻繁重算
- **Foscarnet 給藥前後水化**：每劑前後 give NS 500–1000 mL（除非容量禁忌）以降腎毒性
- Ganciclovir 為細胞毒藥物，需依化療藥調配與棄置規範

## ⚠️ 副作用（常見 / 嚴重）
- **Ganciclovir**：
  - **骨髓抑制**（嗜中性球↓、血小板↓、貧血）— 最重要、劑量限制毒性
  - 發燒、皮疹、腹瀉、肝酵素上升
  - 中樞神經（意識混亂、抽搐、罕見）；長期致癌 / 生殖毒性
- **Foscarnet**：
  - **腎毒性（急性腎損傷 / ATN）** — 劑量限制毒性
  - **電解質紊亂**：低鈣、低鎂、低鉀、低/高磷 → 抽搐、心律不整、QT 延長、感覺異常
  - **生殖器潰瘍**（藥物由尿排出刺激）
  - 噁心、頭痛、注射部位靜脈炎

## 🚫 禁忌症
- **Ganciclovir**：對 ganciclovir / aciclovir 過敏；**孕婦（致畸）**；嚴重骨髓抑制（ANC < 500 或 Plt < 25,000）為相對禁忌
- **Foscarnet**：對 foscarnet 過敏；嚴重腎功能不全（CrCl 過低 nomogram 外）；與其他明顯腎毒 / 低鈣藥物併用需極謹慎

## 🔍 監測 / 交互作用 / 孕哺
- **Ganciclovir 監測**：**每週 CBC + differential**（追 ANC / Plt）、腎功能、肝功能
- **Foscarnet 監測**：**每次給藥前追腎功能 + 電解質（Ca/Mg/K/PO4）**、離子鈣、心電圖（若電解質異常）
- **交互作用**：
  - Ganciclovir + **zidovudine / mycophenolate / 化療 / TMP-SMX** → 骨髓抑制疊加
  - Ganciclovir + **probenecid / imipenem**（imipenem 併用增加抽搐風險）
  - Foscarnet + **amphotericin B / aminoglycoside / cidofovir / IV pentamidine** → 腎毒性 & 低鈣疊加
  - Foscarnet + **延長 QT 藥物** → 心律不整
- **孕哺**：Ganciclovir 孕期禁用（致畸），育齡雙方避孕；哺乳不建議。Foscarnet 孕期僅重症權衡使用

Ganciclovir 的機轉與活化步驟為何？::核苷類似物，需病毒 UL97 kinase 首次磷酸化，再抑制 CMV DNA polymerase 終止 DNA 合成

Valganciclovir 相對於口服 ganciclovir 的優勢？::是 ganciclovir 的纈胺酸前驅藥，口服生體可用率大幅提高，用於維持與預防

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名腎移植後病人使用 ganciclovir 治療 CMV 病毒血症，第 10 天 ANC 掉到 400、血小板 22,000。最適當處置為何？
- A. 增加劑量加強療效
- B. 這是預期的骨髓抑制，考慮停藥/減量並追 CBC，必要時給 G-CSF
- C. 改用高劑量口服 aciclovir
- D. 加上 zidovudine

> [!success]- 答案
> **B** — 骨髓抑制是 ganciclovir 的劑量限制毒性，ANC < 500 / Plt < 25,000 應停藥或減量並密切追蹤，必要時 G-CSF 支持。加 zidovudine（A/D）會雪上加霜，aciclovir 對 CMV 效力不足。

**Q2.** 病人使用 foscarnet 治療抗藥性 CMV 視網膜炎，出現口周麻木與手足抽搐。最可能的原因？
- A. 高血鈣
- B. Foscarnet 誘發的低游離鈣（±低鎂）
- C. 骨髓抑制
- D. 藥物過敏

> [!success]- 答案
> **B** — Foscarnet 會螯合二價陽離子造成低鈣（尤其游離鈣）與低鎂，臨床表現為感覺異常、抽搐，嚴重可致心律不整。應查離子鈣/鎂並矯正。

**Q3.** 關於 foscarnet 與 ganciclovir 的比較，何者正確？
- A. 兩者都需病毒 kinase 活化
- B. Foscarnet 主要毒性是骨髓抑制
- C. Foscarnet 不需病毒 kinase 活化，可用於 ganciclovir 抗藥株，但腎毒性與電解質紊亂明顯
- D. Ganciclovir 只有靜脈劑型

> [!success]- 答案
> **C** — Foscarnet 直接作用於 DNA polymerase 焦磷酸位、不需 kinase 活化，是抗藥株替代藥，代價是腎毒性與電解質紊亂。Ganciclovir 需 UL97 活化、有口服前驅藥 valganciclovir、主要毒性為骨髓抑制。
