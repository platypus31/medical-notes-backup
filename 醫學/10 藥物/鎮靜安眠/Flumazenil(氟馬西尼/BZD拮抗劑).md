# Flumazenil（氟馬西尼 / BZD 拮抗劑）

> [!abstract] 一句話：**GABA-A 受體上 BZD 結合位點的競爭性拮抗劑**，快速逆轉 BZD 的鎮靜 / 呼吸抑制。**用途窄且風險高**：主要用於**術後鎮靜逆轉**與**單純 BZD 過量**；但**半衰期短（約 1 小時）比多數 BZD 短 → 可能再鎮靜（resedation）需重複給或輸注**；且**在長期用 BZD 者或混合藥物過量（尤其 TCA）會誘發癲癇**，臨床使用需極度謹慎。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **長期服用 BZD 者給 flumazenil → 誘發急性戒斷 / 癲癇發作**！這類病人 GABA 系統依賴 BZD 維持，突然拮抗會去除抑制 → 癲癇
> 2. **不明藥物 / 混合過量、尤其疑似 TCA（三環抗憂鬱）共食 → 禁用**：BZD 本身有抗癲癇保護作用，flumazenil 去除此保護，讓 TCA 的致癲癇 / 心律不整效應爆發，可致命
> 3. **半衰期短（~1h）< 多數 BZD** → 病人可能「醒過來後又睡回去（resedation）」與再度呼吸抑制，需**持續監測、重複給藥或連續輸注**，不可給一劑就離開
> 4. **不可作為常規「叫醒工具」**：不明意識不清病人不應例行給 flumazenil，弊大於利（誘發癲癇 / 心律不整風險）
> 5. **依賴 BZD 控制的病況禁用**：癲癇正用 BZD 控制、顱內壓升高、酒精 / BZD 戒斷中者禁

## 💊 適應症
- **苯二氮平（BZD）過量逆轉**（限單純 BZD 過量、無合併致癲癇藥、非長期依賴者）
- **手術 / 檢查後鎮靜（procedural sedation）逆轉**（如胃鏡、麻醉後用 midazolam 鎮靜的甦醒）
- 疑似 BZD 引起的意識障礙之**診斷性 / 治療性試驗**（審慎個案）

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 鎮靜逆轉（術後 / 檢查） | **0.2mg IV over 15 秒**；未醒 45 秒後再 0.2mg，每分鐘可追加，**總量常 ≤ 1mg** |
| 疑似 BZD 過量 | **0.2mg IV over 15–30 秒**；每分鐘追加 0.2mg（可漸增至 0.3–0.5mg），**總量 1–3mg** |
| 再鎮靜（resedation） | 症狀復發時重複給藥；反覆者可考慮**持續輸注 0.1–0.4mg/hr** |
- **給藥要慢**：快速大量推注增加癲癇 / 焦躁風險
- **給藥後持續監測至少 1–2 小時**（因半衰期短，防再鎮靜與再度呼吸抑制）
- 肝功能不良者清除變慢 ⚠️依院規 / 需查證

## ⚠️ 副作用
- **常見**：噁心嘔吐、頭暈、焦躁不安、臉紅、注射部位疼痛、出汗
- **嚴重**：
  - **癲癇發作**（尤其長期 BZD 依賴、混合 TCA 過量者）
  - **再鎮靜 / 再度呼吸抑制**（半衰期短於 BZD）
  - **急性 BZD 戒斷症候群**（焦躁、心悸、高血壓、癲癇）
  - 心律不整（合併 TCA 時）

## 🚫 禁忌症
- **長期使用 BZD 且已產生依賴者**（誘發戒斷 / 癲癇）
- **疑似或已知合併三環抗憂鬱（TCA）或其他致癲癇藥過量**
- **正以 BZD 控制潛在致命病況者**：癲癇持續狀態、顱內壓升高
- 對 flumazenil 或 BZD 過敏
- 混合藥物過量、藥物不明的昏迷病人（相對禁忌，弊大於利）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：意識狀態、呼吸（防再鎮靜）、心律（合併 TCA 時）、癲癇徵象；給藥後留觀至少 1–2 小時
- **交互作用**：
  - **TCA / bupropion / theophylline 等降癲癇閾值藥**：合併過量時 flumazenil 去除 BZD 保護 → 癲癇 / 心律不整爆發
  - 對非 BZD 的中樞抑制劑（酒精、鴉片類、巴比妥）**無逆轉作用**——鴉片過量要用 naloxone，不是 flumazenil
- **孕哺**：資料有限，僅在明確效益大於風險時使用 ⚠️需查證

## 🔗 相關
- 逆轉對象：[[Benzodiazepines(苯二氮平類)]]、[[Lorazepam(勞拉西泮)]]、[[Z-drugs(唑吡坦類)]]（Z-drug 過量部分也可反應）
- 對照解毒劑：Naloxone（鴉片類拮抗）、N-acetylcysteine（乙醯胺酚）
- 機轉關聯：GABA-A 受體與鎮靜安眠藥

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名長期服用 alprazolam 的病人因過量昏睡送急診，住院醫師想給 flumazenil 叫醒。最正確的判斷是？
- A. 立即給 flumazenil，安全有效
- B. 長期 BZD 依賴者給 flumazenil 可能誘發癲癇 / 戒斷，應避免
- C. 加大 flumazenil 劑量效果更好
- D. Flumazenil 可同時逆轉酒精

> [!success]- 答案
> **B** — 長期使用 BZD 已產生依賴者，flumazenil 拮抗會誘發急性戒斷與癲癇發作，屬禁忌；此類病人以支持性照護（呼吸道 / 監測）為主。

**Q2.** 關於 flumazenil 的半衰期，下列敘述何者正確？
- A. 半衰期很長，給一劑可維持一整天
- B. 半衰期約 1 小時，短於多數 BZD，可能再鎮靜
- C. 半衰期與所逆轉的 BZD 完全相同
- D. 半衰期長到不需要監測

> [!success]- 答案
> **B** — Flumazenil 半衰期約 1 小時，比多數 BZD 短，病人可能在藥效退去後再次鎮靜與呼吸抑制，故需持續監測、必要時重複給藥或連續輸注。

**Q3.** 一名意識不清病人送醫，疑似混合藥物過量、無法排除 TCA。關於 flumazenil 使用何者最正確？
- A. 常規給 flumazenil 當叫醒工具
- B. 疑似 TCA 共食時 flumazenil 禁用，因去除 BZD 抗癲癇保護可誘發癲癇
- C. Flumazenil 可安全逆轉所有鎮靜劑
- D. 混合過量時 flumazenil 是首選解毒劑

> [!success]- 答案
> **B** — 疑似 TCA 或不明混合過量時，flumazenil 去除 BZD 的抗癲癇保護作用，可誘發癲癇與心律不整，屬禁忌；不應把 flumazenil 當常規「叫醒工具」。
