# Lorazepam（勞拉西泮 / Ativan）

> [!abstract] 一句話：**中效 BZD**，值班最好用的 benzo。**經 glucuronidation 代謝、無活性代謝物** → 肝損、老人、腎損相對安全、蓄積風險低；為**癲癇重積狀態 IV 第一線**，也用於酒精戒斷、急性焦慮/激躁、化療止吐輔助。**可 IV / IM / PO**，起效與作用時間介於 diazepam 與 midazolam 之間。高濃度連續輸注須警覺**溶劑丙二醇（propylene glycol）毒性**。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **呼吸抑制加乘**：與 opioid、酒精、其他 CNS 抑制劑併用會協同壓制呼吸；IV 推注過快尤其危險 → 給藥時備 flumazenil 與監視器。
> 2. **高劑量連續 IV 輸注 → 丙二醇（propylene glycol）毒性**：注射劑溶劑含丙二醇，長時間高劑量（如癲癇重積、ICU 鎮靜）可致**代謝性酸中毒（陰離子間隙升高）、急性腎損傷、高滲透壓間隙**→ 監測 osmolar gap / lactate / 腎功能。
> 3. **老人 / 譫妄**：仍屬 Beers Criteria 藥物，增加跌倒與譫妄；ICU 譫妄病人應盡量避免 BZD 鎮靜（優選 dexmedetomidine/propofol）。
> 4. **戒斷**：長期使用突然停藥可致癲癇/譫妄，需 taper。
> 5. **IV 推注速度**：一般不超過 2 mg/min，過快增加呼吸抑制與低血壓。

## 💊 適應症（治療哪些疾病）
- **癲癇重積狀態 [[Status Epilepticus(癲癇重積狀態)]]**（**IV 第一線首選**，作用時間比 diazepam 長、腦內再分布較慢，抗癲癇維持較久）
- **酒精戒斷 [[Alcohol Withdrawal(酒精戒斷)]]**（無活性代謝物，肝功能不良的酒癮病人優選）
- **急性焦慮 / 恐慌 / 激躁 [[Anxiety Disorder(焦慮症)]]**（急診躁動常用 IM lorazepam ± haloperidol）
- **失眠 [[Insomnia(失眠)]]**（短期）
- **化療引起的預期性噁心嘔吐（anticipatory nausea）輔助**
- **手術前鎮靜、程序前抗焦慮**

## 📏 劑量 / 用法（成人）
| 用途 | 劑量 |
| --- | --- |
| 癲癇重積 | **4 mg IV**（緩推 ≤2 mg/min），10–15 分鐘無效可重複一次（總量約 8 mg） |
| 急性焦慮 / 激躁 | 0.5–2 mg PO/IM/IV，視需要 Q4–6H |
| 酒精戒斷 | 依 CIWA-Ar 症狀導向給藥，常 2–4 mg PO/IV |
| 失眠 | 0.5–2 mg PO 睡前 |
| 手術前鎮靜 | 0.05 mg/kg IM（最大約 4 mg），術前 2 小時 |

- **給藥途徑**：PO / **IM（可靠吸收，優於 diazepam IM）** / IV
- **腎功能**：主要肝代謝，一般不需大幅調整；但**嚴重腎損時丙二醇溶劑與代謝物 lorazepam-glucuronide 可蓄積** → 高劑量連續輸注要小心
- **肝功能**：走 glucuronidation 不需 CYP，**肝損相對安全**（優於 diazepam）；嚴重肝衰竭仍宜減量
- **老人**：起始劑量減半（如 0.5 mg），避免長期

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：嗜睡、頭暈、鎮靜、步態不穩、順向失憶、精神運動遲緩
- **嚴重**：
  - **呼吸抑制**（IV 過快、併用 opioid/酒精）
  - **依賴、耐受、戒斷**（癲癇、譫妄、反彈焦慮）
  - **丙二醇毒性**（高劑量連續輸注：代謝性酸中毒、AKI、高滲透壓間隙）
  - **反常反應**（激動、去抑制，尤其兒童/老人）
  - 跌倒、低血壓（IV）

## 🚫 禁忌症
- 對 lorazepam / 其他 BZD 過敏
- **重度呼吸功能不全**、急性呼吸衰竭、未治療之睡眠呼吸中止症
- **重症肌無力（Myasthenia Gravis）**
- 急性閉角性青光眼
- 嚴重肝功能不全（相對禁忌，需減量慎用）
- 注射劑對聚乙二醇/丙二醇/苯甲醇過敏者

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：鎮靜/呼吸（IV 必備監視器 + 備 flumazenil）、**高劑量連續輸注監測 osmolar gap / 陰離子間隙 / lactate / 腎功能（丙二醇毒性）**、老人跌倒與譫妄、長期使用依賴徵象
- **半衰期比較（值班速記）**：
  - **Lorazepam ~10–20 h**（中效、無活性代謝物）
  - Diazepam 母藥 ~20–50 h，但**活性代謝物 desmethyldiazepam 可達 >100 h**（老人/肝損易蓄積）
  - Midazolam ~2–6 h（短效）
- **交互作用**：
  - **Opioids / 酒精 / 其他 CNS 抑制劑**：協同呼吸抑制、鎮靜（FDA 黑框：與 opioid 併用）
  - 因**不經 CYP 代謝**，與 CYP 抑制/誘導劑交互作用**少**（相對 midazolam/diazepam 的優點）
  - Probenecid、valproate 可能抑制 glucuronidation → 升高 lorazepam 濃度
- **孕哺**：孕期避免常規使用；臨產前使用可致新生兒 floppy baby syndrome（低張、呼吸抑制）與戒斷；哺乳會分泌至乳汁，短期小劑量偶用一般可接受，需觀察嬰兒鎮靜/餵食

### 記憶卡（Spaced Repetition 相容 · `Q::A`）

Lorazepam 代謝上有什麼特別、臨床意義為何？::走 glucuronidation、**無活性代謝物**、不經 CYP → 肝損、老人、腎損相對安全，蓄積與藥物交互作用少

Lorazepam 半衰期大約多少、屬短中長效哪一種？::約 10–20 小時，屬**中效** BZD

為什麼酒精戒斷的肝硬化病人優選 lorazepam？::無活性代謝物、不依賴 CYP，肝功能不良時不易蓄積，較 diazepam/chlordiazepoxide 安全

急診躁動病人 IM 給藥常用什麼組合？::IM lorazepam ± haloperidol（lorazepam IM 吸收可靠）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一名癲癇重積病人在 ICU 接受高劑量 lorazepam 連續輸注數日，出現陰離子間隙代謝性酸中毒與急性腎損傷、高滲透壓間隙。最可能原因？
- A. Lorazepam 本身腎毒性
- B. 注射劑溶劑丙二醇（propylene glycol）蓄積毒性
- C. 過敏反應
- D. 低血糖

> [!success]- 答案
> **B** — Lorazepam 注射劑溶劑含丙二醇，高劑量長時間輸注可蓄積，造成陰離子間隙代謝性酸中毒、AKI 與高滲透壓間隙；監測 osmolar gap / lactate / 腎功能，必要時改藥。

**Q2.** 關於 lorazepam 相較 diazepam 用於癲癇重積，何者正確？
- A. Diazepam 起效較慢
- B. Lorazepam 腦內再分布較慢，抗癲癇作用維持較久
- C. 兩者作用時間完全相同
- D. Lorazepam 不能 IV 給

> [!success]- 答案
> **B** — Diazepam 脂溶性高、起效快，但很快從腦部再分布使抗癲癇作用短；lorazepam 腦內停留較久、抗癲癇維持較長，故為 IV 第一線首選。

**Q3.** 一名 80 歲肝硬化病人需短期鎮靜抗焦慮，最適合的 BZD 是？
- A. Diazepam
- B. Chlordiazepoxide
- C. Lorazepam
- D. 巴比妥類

> [!success]- 答案
> **C** — Lorazepam 走 glucuronidation、無活性代謝物、不依賴 CYP，肝損與老人相對安全，蓄積風險低（起始劑量仍應減半）。
