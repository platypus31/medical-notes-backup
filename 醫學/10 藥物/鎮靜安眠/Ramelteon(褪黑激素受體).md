# Ramelteon（褪黑激素受體促效劑）

> [!abstract] 一句話：Ramelteon 是選擇性 melatonin 受體（**MT1 / MT2**）促效劑，模擬褪黑激素「調節生理時鐘、促進入睡」，**非苯二氮平類、無成癮性、非管制藥物**，主打**入睡困難型失眠**，特別適合有物質濫用史或老年人（無跌倒 / 呼吸抑制風險）。

> [!danger] 🚨 值班紅旗（開藥前先想）
> 1. **與 Fluvoxamine 併用為禁忌** → Fluvoxamine 是強效 CYP1A2 抑制劑，會大幅升高 Ramelteon 血中濃度（可達數倍到數十倍）
> 2. **經 CYP1A2 代謝，肝功能不全需謹慎** → 中重度肝損傷小心，嚴重肝損傷避免
> 3. **定位是「入睡困難」不是「睡眠維持」** → 對半夜易醒 / 早醒型失眠效果有限，別誤用
> 4. **睡前吃、吃完就上床、避免與高脂餐同服**（延緩吸收）→ 也避免併用酒精（加乘鎮靜）
> 5. **罕見內分泌影響** → 泌乳素上升、睪固酮下降等，長期使用出現月經異常 / 溢乳 / 性慾改變時留意

## 💊 適應症（治療哪些疾病）
- **入睡困難型失眠（sleep-onset insomnia）**：[[Insomnia(失眠)]]（以縮短入睡潛伏期為主）
- **適合族群**：有**物質濫用 / 成癮病史**者、**老年人**（相對 BZD / Z-drugs 無跌倒、無呼吸抑制、無成癮）
- 非緊急鎮靜用藥，不用於急性焦躁 / 譫妄的快速鎮靜

## 📏 劑量 / 用法（成人）
| 項目 | 內容 |
| --- | --- |
| 常用劑量 | 睡前約 8 mg（就寢前 30 分鐘內服用，依院規 / 需查證） |
| 服用時機 | 睡前、服後即上床；**避免與高脂餐同服**（降低 / 延緩吸收） |
| 療程 | 可較長期使用（無成癮性），但仍應定期評估睡眠衛生 |
| 老年人 | 一般不需特別減量，但整體謹慎 |
- 肝功能不全：中重度謹慎、嚴重避免；腎功能不全影響較小（依院規 / 需查證）

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：頭暈、嗜睡、疲倦、頭痛、噁心（整體耐受良好，副作用輕微）
- **較少 / 需留意**：
  - **內分泌影響**：泌乳素上升（可能溢乳 / 月經異常）、睪固酮下降
  - 隔日殘餘嗜睡（較 BZD 輕）
  - 罕見過敏 / 血管性水腫、複雜睡眠行為（夢遊等，各類安眠藥的類效應）
- **相對優勢**：**無成癮 / 無依賴 / 無戒斷、無明顯呼吸抑制、跌倒風險低**（對比 BZD / Z-drugs）

## 🚫 禁忌症
- **併用 Fluvoxamine**（強效 CYP1A2 抑制劑）
- 嚴重肝功能不全
- 曾對 Ramelteon 發生血管性水腫 / 嚴重過敏
- 睡眠呼吸中止未評估者仍需謹慎（雖呼吸抑制風險低）

## 🔍 監測 / 交互作用 / 孕哺
- **交互作用**：
  - **Fluvoxamine（禁忌）** 與其他 **CYP1A2 抑制劑**（如 ciprofloxacin）↑ 濃度
  - **CYP1A2 誘導劑**（如吸菸 / rifampin）↓ 濃度、降低療效
  - **酒精 / 其他中樞抑制劑** → 鎮靜加乘
- **內分泌**：長期使用出現溢乳 / 月經異常 / 不孕困擾時，考慮驗泌乳素 / 睪固酮
- **孕哺**：資料有限，孕哺應謹慎 / 依院規 / 需查證
- **睡眠衛生**：藥物之外強調規律作息、限制咖啡因 / 螢幕光

## 🔗 相關
- 疾病：[[Insomnia(失眠)]]
- 同分區 / 對照：[[Benzodiazepines(苯二氮平類)]]、[[Z-drugs(唑吡坦類)]]（有成癮 / 呼吸抑制 / 跌倒風險，對照 Ramelteon 的安全性）、[[Lorazepam(勞拉西泮)]]
- 概念：褪黑激素 MT1/MT2、CYP1A2 代謝、非管制安眠藥

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位有酒癮病史、抱怨入睡困難的中年男性，最適合的安眠藥選擇是？
- A. Lorazepam
- B. Zolpidem
- C. Ramelteon
- D. Diazepam

> [!success]- 答案
> **C** — Ramelteon 非管制、無成癮性且無呼吸抑制風險，特別適合有物質濫用 / 成癮病史者的入睡困難；BZD 與 Z-drugs 都有成癮與濫用疑慮。

**Q2.** 一位服用 Fluvoxamine（治療強迫症）的病人失眠，醫師考慮加安眠藥。下列何者為禁忌？
- A. Ramelteon
- B. Melatonin
- C. 睡眠衛生教育
- D. 認知行為治療

> [!success]- 答案
> **A** — Fluvoxamine 是強效 CYP1A2 抑制劑，會大幅升高 Ramelteon 血中濃度，兩者**併用為禁忌**。

**Q3.** 關於 Ramelteon 的敘述，何者正確？
- A. 屬第四級管制藥，有成癮性
- B. 主要用於半夜易醒的睡眠維持障礙
- C. 促效 MT1 / MT2 受體，無呼吸抑制與跌倒風險，適合老年人
- D. 應與高脂餐一起服用以增加吸收

> [!success]- 答案
> **C** — Ramelteon 是 MT1 / MT2 促效劑，非管制、無成癮、無呼吸抑制與低跌倒風險，適合老年人；主打入睡困難，且應避免與高脂餐同服。
