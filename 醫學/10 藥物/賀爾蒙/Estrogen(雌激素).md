# Estrogen（雌激素 / Estradiol）

> [!abstract] 一句話：作用於雌激素受體，用於**更年期症狀（HRT）、性腺功能低下、避孕（與 progestin 併用）**。值班/門診核心紅旗是**血栓（VTE）風險、子宮內膜癌（無對抗雌激素）、乳癌與心血管風險**——用藥前必問吸菸、血栓史、乳癌史。

> [!danger] 🚨 值班紅旗
> 1. **靜脈血栓栓塞（VTE / DVT / PE）+ 中風**風險升高 → 有血栓史/易栓症、**吸菸 > 35 歲**者，雌激素（尤含 estrogen 避孕藥）風險大增
> 2. **有子宮者用雌激素必須加 progestin**：單用（unopposed）雌激素 → **子宮內膜增生 / 子宮內膜癌**；子宮已切除者可單用
> 3. **乳癌**：雌激素/HRT 與乳癌風險 → 已知/疑乳癌禁用；不明陰道出血須先查明
> 4. **急性用藥前評估**：任何雌激素製劑前問血栓史、乳癌史、吸菸、肝病、未查明陰道出血

## 💊 適應症（治療哪些疾病）
- **更年期症狀**（熱潮紅、陰道萎縮）荷爾蒙補充療法（HRT）
- **女性性腺功能低下 / 卵巢早衰 / Turner 症候群**荷爾蒙補充
- **避孕**（與 progestin 併用的複方口服避孕藥）
- 骨質疏鬆預防（次要選項）、部分月經失調

## 📏 劑量 / 用法（成人）
| 情境 | 用法 |
| --- | --- |
| 更年期 HRT | 最低有效劑量、最短時間；**有子宮者併 progestin** |
| 劑型 | 口服 estradiol / conjugated estrogen、經皮貼片/凝膠、陰道局部（萎縮） |
| 避孕 | 複方口服避孕藥（ethinyl estradiol + progestin） |
- **經皮劑型 VTE 風險較口服低**（不經肝首渡）→ 血栓風險族群優先
- 陰道局部低劑量用於泌尿生殖症狀，全身吸收少

## ⚠️ 副作用（常見 / 嚴重）
- **常見**：噁心、乳房脹痛、水腫、頭痛、不規則出血、情緒變化
- **嚴重**：
  - **VTE / DVT / PE / 缺血性中風**
  - **子宮內膜癌**（unopposed，有子宮者）
  - **乳癌**風險增加（長期 HRT）
  - 膽結石 / 膽汁鬱積、高三酸甘油酯（口服）
  - 高血壓、肝腺瘤（罕見）

## 🚫 禁忌症
- **VTE / 肺栓塞病史、已知易栓症**
- 已知 / 疑似**乳癌、雌激素依賴性腫瘤**
- **不明原因陰道出血**（未查明）
- 活動性肝病
- 冠心病 / 中風病史
- **吸菸且 ≥ 35 歲**（含雌激素避孕藥）
- 懷孕

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：血壓、乳房檢查/乳房攝影、不正常出血（有子宮者評估內膜）、血脂、血栓症狀
- **交互作用**：
  - **酵素誘導劑**（rifampin、部分抗癲癇藥、St. John's wort）↓ 雌激素/避孕效果
  - 與 warfarin、甲狀腺素、皮質類固醇代謝互相影響
  - 吸菸協同增加心血管/血栓風險
- **孕哺**：懷孕禁用；哺乳期含雌激素避孕藥可能減少乳汁

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位仍有子宮的更年期女性要用 HRT，為降低子宮內膜癌風險應？
- A. 單用雌激素
- B. 雌激素併用 progestin
- C. 只用高劑量雌激素
- D. 不需任何處置

> [!success]- 答案
> **B** — 有子宮者須雌激素併 progestin，避免 unopposed estrogen 造成子宮內膜增生/癌；子宮切除者才可單用雌激素。

**Q2.** 下列何者最應避免使用含雌激素的複方口服避孕藥？
- A. 24 歲不吸菸女性
- B. 38 歲每天吸菸女性
- C. 20 歲運動員
- D. 30 歲偶爾飲酒女性

> [!success]- 答案
> **B** — 吸菸且 ≥ 35 歲是含雌激素避孕藥的重要禁忌，心血管與血栓風險大幅上升。

**Q3.** 更年期女性有 DVT 病史但需處理嚴重熱潮紅，關於雌激素的敘述何者正確？
- A. 全身雌激素完全安全
- B. VTE 病史是雌激素禁忌，需考慮非荷爾蒙療法或個別權衡
- C. 加大劑量即可
- D. 只能口服

> [!success]- 答案
> **B** — VTE/肺栓塞病史是雌激素禁忌，應優先考慮非荷爾蒙療法；若必要須嚴格個別化評估（經皮風險較低）並與病人充分討論。
