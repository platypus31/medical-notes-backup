# Androgen（雄性素 / Testosterone）

> [!abstract] 一句話：作用於雄性素受體，補充男性性腺功能低下（hypogonadism）；主要用於**經生化確認的睪固酮低下併症狀**。值班/門診重點在**紅血球增多（多血症）、前列腺監測、心血管風險、與濫用（運動員/健身）造成的性腺軸抑制與不孕**。

> [!danger] 🚨 值班紅旗
> 1. **紅血球增多症（erythrocytosis / polycythemia）**：睪固酮刺激紅血球生成 → Hct 過高增血栓風險 → **Hct > 54% 應暫停、放血**
> 2. **前列腺**：不可用於**已知或懷疑前列腺癌/男性乳癌**；用藥前後監測 PSA + 肛診
> 3. **外源性睪固酮抑制 HPG 軸 → 精子生成下降、不孕、睪丸萎縮**（想生育者不可用 TRT，改 hCG/其他）
> 4. **經皮凝膠會轉移給接觸者**（伴侶/孩童）→ 塗抹部位須遮蓋、洗手

## 💊 適應症（治療哪些疾病）
- **男性性腺功能低下**（原發或次發 hypogonadism，[[Hypogonadism(性腺功能低下)]]）— 生化確認 + 症狀
- 青春期延遲（特定情況）
- 部分遺傳/內分泌疾病的荷爾蒙補充
- （**非適應**、需警惕濫用）健美增肌、抗老化——不建議

## 📏 劑量 / 用法（成人男性補充）
| 劑型 | 用法 |
| --- | --- |
| 肌肉注射 testosterone enanthate/cypionate | 每 2–4 週 IM（濃度波動大） |
| Testosterone undecanoate 長效 IM | 每 10–14 週 |
| 經皮凝膠 | 每日塗抹（注意轉移他人） |
| 皮膚貼片 | 每日 |
- 目標：把睪固酮拉回**正常中段**，依症狀與濃度調整
- **老年、心血管高風險者謹慎**，逐步評估

## ⚠️ 副作用（常見 / 嚴重）
- **紅血球增多症**（血栓風險）
- 痤瘡、油脂分泌增加、體毛增加
- **睪丸萎縮、精子↓、不孕**（HPG 軸抑制）
- 男性女乳症（芳香酶轉為雌激素）
- 水鈉滯留、血壓上升；睡眠呼吸中止惡化
- 情緒/攻擊性改變
- 肝毒性（**口服 17-α 烷基化製劑**，現少用）；血脂惡化（HDL↓）
- 前列腺良性增生症狀惡化

## 🚫 禁忌症
- **前列腺癌、男性乳癌**（已知/懷疑）
- 未評估的前列腺結節 / PSA 升高
- 未控制的重度心衰
- 紅血球增多症（Hct 過高）
- 想生育者（改用促性腺激素）
- 孕婦（致胎兒男性化）

## 🔍 監測 / 交互作用 / 孕哺
- **監測**：睪固酮濃度、**Hct/Hb**（多血症）、**PSA + 肛診**（前列腺）、血脂、肝功能（口服製劑）、症狀
- **交互作用**：
  - 增強 warfarin 抗凝
  - 影響胰島素/降糖藥需求（改善胰島素敏感度）
  - 與皮質類固醇 → 加重水腫
- **孕哺**：孕婦禁用（胎兒男性化）

### 自我測驗（選擇題，答案摺疊）

**Q1.** 一位接受睪固酮補充的男性回診 Hct 56%。最適當的處置是？
- A. 增加劑量
- B. 暫停治療並考慮放血（erythrocytosis）
- C. 不予理會
- D. 加抗血小板藥即可

> [!success]- 答案
> **B** — 睪固酮刺激紅血球生成，Hct > 54% 增加血栓風險，應暫停治療、必要時放血，並重新評估劑量。

**Q2.** 下列何者是雄性素治療的禁忌？
- A. 疲倦
- B. 前列腺癌
- C. 骨質疏鬆
- D. 貧血

> [!success]- 答案
> **B** — 前列腺癌（及男性乳癌）為雄性素治療禁忌，因可能刺激腫瘤生長。

**Q3.** 一位 32 歲男性想改善精蟲數而自行使用睪固酮，結果精子更少。原因是？
- A. 劑量不足
- B. 外源睪固酮抑制 HPG 軸，反而降低精子生成
- C. 睪固酮與精子無關
- D. 需併用雌激素

> [!success]- 答案
> **B** — 外源性睪固酮負回饋抑制下視丘-腦垂體-性腺軸，降低 FSH/LH 與內生睪固酮，使精子生成下降；想生育應改用 hCG/促性腺激素。
