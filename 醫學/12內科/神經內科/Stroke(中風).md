# Stroke（中風）

> [!abstract] 一句話：**急性局部腦血流障礙造成的神經功能缺損**，分**缺血性（Ischemic，~80%）**與**出血性（Hemorrhagic，~20%）**兩大類——值班第一件事永遠是**「Time is Brain」**：立刻定 last-known-well 時間、驗血糖、急做 non-contrast Brain CT 分辨梗塞 vs 出血，因為兩者處置**完全相反**（缺血要打通、出血要止血降壓）。缺血性有 **tPA 4.5 小時**與**取栓 24 小時**兩個可救命的時間窗，錯過就回不來。

> [!danger] 🚨 值班紅旗
> 1. **驗血糖優先於一切**：低血糖（<60）可完全模仿中風（stroke mimic），且是可立即逆轉的死因——**任何疑似中風先扎血糖**，再談影像。
> 2. **問到 last-known-well（最後正常時間），不是「發現症狀時間」**：這決定 tPA/取栓資格。醒來就有症狀（wake-up stroke）以入睡時間算，但可用 MRI mismatch / perfusion 爭取治療。
> 3. **急做 non-contrast Brain CT 分 ICH vs infarction**：兩者處置相反。看到出血 → **停所有抗栓/抗凝、緊急降壓、查並逆轉抗凝血劑**；看到（或 CT 陰性但臨床像）梗塞 → 走 stroke code 評估溶栓/取栓。
> 4. **缺血性溶栓前血壓必須 <185/110**，否則不能打 tPA；打完 24 小時內維持 <180/105。降壓用 labetalol / nicardipine。
> 5. **懷疑大血管阻塞（LVO：NIHSS 高、凝視偏移、失語+偏癱）→ 立刻啟動取栓路徑**，做 CTA 找 occlusion，時間窗可延伸到 **24 小時（with perfusion mismatch, DAWN/DEFUSE-3）**。
> 6. **出血性（ICH）**：SBP 150–220 → 目標降到 **140（維持 130–150）**；急查 INR/抗凝史，**warfarin 用 PCC + vitamin K、DOAC 用專一逆轉劑**逆轉；神經惡化/後顱窩出血/腦室出血 → 緊急照會神經外科。
> 7. **後循環中風易漏診**：眩暈、複視、共濟失調、意識改變被當「內耳/腸胃炎」，basilar occlusion 死亡率極高——後循環症狀不要輕放。

## 📖 定義 / 病生理（這是什麼）
- **定義**：急性起病、可對應血管領域的**局部神經功能缺損**，因腦血流中斷造成腦組織缺血或出血性損傷。與 [[Transient Ischemic Attack(暫時性腦缺血)]] 的差別在於 **TIA 無梗塞證據且短暫可逆**，中風已有梗塞/出血且常留下缺損。

| 大類 | 佔比 | 機轉 | 影像（NCCT） | 核心處置方向 |
| --- | --- | --- | --- | --- |
| **缺血性 [[Ischemic Stroke(缺血性中風)]]** | ~80% | 血管阻塞→組織缺血壞死 | 早期常正常（早期徵象：hyperdense MCA sign、灰白質界線模糊、島帶消失） | **再灌流**：tPA / 取栓 + 抗栓 |
| **出血性 [[Hemorrhagic Stroke(出血性中風)]]** | ~20% | 血管破裂→血腫壓迫 | **立即高密度**（白色）血腫 | **止血、降壓、逆轉抗凝、外科減壓** |

- **缺血性中風分型（TOAST）**：
  | 型 | 說明 | 代表危險因子 |
  | --- | --- | --- |
  | **大血管動脈粥狀硬化** | 頸動脈/顱內狹窄、動脈到動脈栓塞 | 高血壓、高血脂、抽菸、糖尿病 |
  | **心源性栓塞** | **心房顫動**最常見，心瓣膜、心內血栓、近期 MI | [[Atrial Fibrillation(心房顫動)]] |
  | **小血管（腔隙性 lacunar）** | 穿通枝閉塞，典型腔隙症候群（純運動/純感覺） | 高血壓、糖尿病 |
  | **其他/不明** | 血管剝離、高凝、血管炎、cryptogenic | 年輕中風要追 |
- **缺血核心 vs 半暗帶（penumbra）**：核心壞死不可逆，周邊半暗帶是**血流低但仍存活、可搶救的組織**——再灌流治療的目標就是救半暗帶，這就是「Time is Brain」的生理基礎（每分鐘約損失 190 萬個神經元）。
- **出血性病生理**：高血壓性 ICH 好發於**基底核、丘腦、橋腦、小腦**（穿通枝 Charcot-Bouchard 微動脈瘤破裂）；腦葉出血在老年要想 **cerebral amyloid angiopathy (CAA)**。血腫擴大（hematoma expansion）與 IVH、水腦是惡化主因。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Focal Weakness(局部無力)]]、[[Aphasia(失語症)]]、[[Facial Palsy(顏面神經麻痺)]]、[[Dysarthria(構音障礙)]]、[[Hemianopia(偏盲)]]
- **突發（sudden onset）**、可對應血管領域、單側為主的神經缺損：
  - **前循環（頸動脈系 / MCA、ACA）**：對側偏癱與偏身感覺缺失、失語（優勢半球，多為左）、忽略（非優勢半球）、凝視偏向病灶側、同側單眼黑矇。
  - **後循環（椎基底動脈系）**：眩暈、複視、構音/吞嚥障礙、共濟失調、交叉性徵象（同側顏面+對側肢體）、視野缺損、意識改變（basilar 大範圍時）。
  - **腔隙症候群**：純運動偏癱、純感覺、共濟失調性輕癱、構音障礙-笨手。
- **辛辛那提院前中風量表（FAST 快篩）**：**微笑殭屍會說話**——臉（Face 歪斜）、手（Arm 抬手一邊掉）、說話（Speech 含糊/答非所問），任一異常即高度懷疑，記錄 Time。
- **提示出血性**（非絕對）：劇烈頭痛（「這輩子最痛」想到 SAH）、嘔吐、意識快速下降、血壓極高、發病即 GCS 差；但**臨床無法可靠區分缺血 vs 出血，一律靠 CT**。
- **stroke mimic 鑑別**：低血糖、癲癇後 Todd's paralysis、偏頭痛先兆、Bell's palsy、暈厥、轉化症、感染/代謝性腦病。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Blood Glucose(血糖)]]、[[ECG(心電圖)]]、[[NIHSS(美國國衛院中風量表)]]　影像 [[Brain CT(腦部電腦斷層)]]、[[CT Angiography(CTA血管攝影)]]、[[Brain MRI(腦部磁振造影)]]、[[Carotid Doppler(頸動脈超音波)]]
- 辛辛那提簡易判斷：微笑殭屍會說話（微笑,抬手,說話）
- 做 Brain CT 看是 ICH 或是 infarction
- **急測（床邊，數分鐘內）**：**血糖**（排除低血糖 mimic）、生命徵象與血壓、SpO₂、**NIHSS 分數**、last-known-well 時間、抗凝血藥史。
- **影像（stroke code 核心）**：
  - **Non-contrast Brain CT**：第一線，**主要目的是排除出血**與大範圍早期梗塞（ASPECTS 評分）；超急性梗塞 CT 常正常。
  - **CTA（頭頸）**：找**大血管阻塞（LVO）**決定取栓；也看頸動脈/顱內狹窄、血管剝離。
  - **CT perfusion / MRI DWI-PWI mismatch**：定義可搶救半暗帶，是**延伸時間窗（6–24h）取栓**與 wake-up stroke 溶栓的依據（DAWN/DEFUSE-3、WAKE-UP）。
  - **MRI DWI**：對超急性小梗塞、後循環、lacunar 最敏感。
- **血管評估**：頸動脈超音波 / CTA / MRA 找症狀性狹窄（次級預防決策）。
- **心臟評估**：**12 導程 EKG**（找 AF、心肌缺血）＋住院心電監測；懷疑陣發性 AF 用 **Holter / 延長監測**；心臟超音波找心源栓塞、PFO、心內血栓。
- **抽血**：血糖/HbA1c、CBC（血小板）、**PT/INR、aPTT**（溶栓前與逆轉決策）、腎功能（顯影劑）、血脂、電解質；年輕/隱源性中風追高凝、血管炎、心臟評估。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
![[截圖 2024-12-03 上午10.20.43.png]]
> [!PDF|yellow] [[長庚銀蛋口袋書(印).pdf#page=106&selection=147,0,159,1&color=yellow|長庚銀蛋口袋書(印), p.106]]
> > 心電圖變化： 廣泛的 T 波反轉、顯著的 U 波
- **神經性心電圖變化**（如上：廣泛 T 波倒置、U 波、QT 延長，尤其 **SAH / 出血性**）是**腦源性（neurogenic）**心肌影響，不一定代表原發心臟病，但別漏掉真正的心肌缺血。
- **CT 陰性不能排除超急性梗塞**：發病 <6 小時的缺血 CT 常正常，臨床像中風就走流程，別被「CT 沒事」誤導。
- **時間就是大腦**：door-to-needle 目標 **<60 分鐘**，先做 CT、驗血糖、量血壓，能同步就同步，不要序列式浪費時間。
- **降壓的方向缺血 vs 出血相反**：缺血性**不溶栓時容許到 <220/120（permissive HTN 保住半暗帶）**，只在溶栓/取栓才嚴格降到 <185/110；出血性要積極降到 140。搞反會害人。
- **後循環易被當周邊眩暈**：用 **HINTS**（Head Impulse-Nystagmus-Test of Skew）區分中樞 vs 周邊，中樞特徵（direction-changing nystagmus、skew deviation、正常 head impulse）要當中風。
- **wake-up / 時間不明中風不是自動出局**：MRI DWI-FLAIR mismatch 或 perfusion mismatch 仍可能有溶栓/取栓資格。
- **溶栓後 24 小時內**：不給抗血小板/抗凝、嚴密監測神經學與血壓，惡化立刻回 CT 排除**出血轉化（hemorrhagic transformation）**。

## 💊 治療（Acute / Chronic）
> 用藥 [[Alteplase(tPA血栓溶解劑)]]、[[Tenecteplase(TNK)]]、[[Aspirin(阿斯匹靈)]]、[[Clopidogrel(氯吡格雷)]]、[[Statin(他汀類)]]、[[Labetalol(拉貝洛爾)]]、[[Nicardipine(尼卡地平)]]
> ⚠️ 所有劑量／時間窗依當地 stroke protocol 與最新指引，用藥前查藥典

### 缺血性 — 急性（Acute Ischemic Stroke）
- **靜脈溶栓（IV thrombolysis）**：
  - **時間窗**：發病 **4.5 小時內**（last-known-well 起算）。藥物 **alteplase (tPA) 0.9 mg/kg（最大 90 mg，10% bolus，其餘 60 分鐘輸注）**；**tenecteplase (TNK) 0.25 mg/kg 單次 bolus** 已被指引接受、尤其取栓前橋接更方便。
  - **溶栓前血壓必須 <185/110**（超過先用 labetalol / nicardipine 降），溶栓後 **24 小時維持 <180/105**。
  - **主要禁忌（部分）**：CT 已見出血、近期顱內手術/出血/嚴重外傷、活動性內出血、血小板 <100k、INR >1.7 或用中 DOAC、近期大手術、BP 頑固 >185/110、血糖過低（先矯正）。⚠️ **完整禁忌清單依 protocol 逐條核對**。
- **機械取栓（endovascular thrombectomy, EVT）**：
  - 適應：**前循環大血管阻塞（LVO：ICA、MCA-M1/近端 M2）**。
  - **時間窗**：**6 小時內**直接做；**6–24 小時**在符合 **DAWN / DEFUSE-3**（臨床-影像或 perfusion mismatch）條件下仍建議取栓。取栓與溶栓**不互斥**，符合就都給（溶栓不延誤取栓）。
- **血壓管理**：**不做再灌流治療者採 permissive HTN，只在 >220/120 才降**（緩降約 15%）；溶栓/取栓者依上述嚴格閾值。
- **一般照護**：血糖控制（避高血糖與低血糖）、體溫、吞嚥評估（NPO 直到過關防吸入）、DVT 預防、早期復健。
- **抗血小板**：**非溶栓者 24–48 小時內給 aspirin**；溶栓者延到 **24 小時後回 CT 無出血再給**。

### 缺血性 — 慢性 / 次級預防（Secondary Prevention）
- **抗栓（依病因）**：
  - **非心源性**：單一抗血小板（aspirin 50–325mg、或 clopidogrel 75mg、或 aspirin+dipyridamole）。
  - **輕微中風 / 高風險 TIA**：**短期雙抗（DAPT）aspirin + clopidogrel，早期（12–24h 內、至少 7 天內）啟動，用 21–90 天後轉單一**（CHANCE / POINT）——長期雙抗增加出血，療程要短。
  - **心源性（AF）→ 抗凝血**（DOAC 優於 warfarin，非瓣膜性），非只抗血小板。
- **降血脂**：**高強度 statin**，積極降 LDL（目標 <70 mg/dL），必要時加 ezetimibe / PCSK9i。
- **血壓**：長期目標一般 <130/80。
- **血糖、戒菸、減重、運動**等危險因子控制。
- **症狀性頸動脈狹窄 ≥50–70%**：**頸動脈內膜切除（CEA）或支架**，黃金期約**兩週內**評估。

### 出血性 — 急性（Intracerebral Hemorrhage, ICH）
- **停用並逆轉抗栓/抗凝**：warfarin → **PCC（首選）+ vitamin K**；DOAC → 專一逆轉劑（dabigatran 用 idarucizumab；Xa 抑制劑用 andexanet alfa 或 PCC）；抗血小板相關不常規輸血小板（除非手術）。
- **降血壓**：SBP 150–220 → **目標 140，維持 130–150（1 小時內達標）**；**避免降到 <130（可能有害）**。用 nicardipine / labetalol 靜脈滴定。
- **顱內壓 / 手術**：神經惡化、腦室內出血合併水腦（考慮 EVD）、**小腦出血 >3cm 或壓迫腦幹/腦室**→ 緊急神經外科減壓；抬高床頭、必要時高滲透治療。
- **一般**：血糖/體溫控制、抗癲癇僅在臨床發作時給、DVT 預防用間歇氣壓（早期避藥物性）。

## ⚠️ 併發症
- **缺血性**：**出血轉化（hemorrhagic transformation，尤其溶栓後）**、惡性腦水腫（大範圍 MCA 梗塞 → 可能需去骨瓣減壓）、癲癇、吸入性肺炎、出血性溶栓併發症。
- **出血性**：**血腫擴大**、腦室內出血與**急性水腦**、顱內壓升高與腦疝、再出血、癲癇。
- **共通**：吞嚥障礙 → 吸入性肺炎、深部靜脈栓塞 / 肺栓塞、壓瘡、尿路感染、憂鬱、跌倒、後遺神經功能障礙與失能。

## 📊 嚴重度評分 / 風險分層（scoring）
> 用哪個 score 判斷 severity / 預後 / treatment 強度

| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **NIHSS** | 缺血性中風嚴重度、監測惡化、預後、溶栓/取栓決策 | 0–42 分。粗分：**0 無、1–4 輕、5–15 中、16–20 中重、21–42 重**；分數越高預後越差、越可能 LVO |
| **辛辛那提 / FAST** | 院前/床邊快篩 | 臉歪／抬手掉／說話異常任一 → 疑中風，記錄時間 |
| **ASPECTS** | 早期缺血變化範圍（NCCT，MCA 領域） | 10 分制，扣分越多梗塞越廣；**低分（如 <6）**再灌流獲益差、出血風險高 |
| **ICH score** | 出血性中風 30 天死亡率 | GCS、血腫量（≥30mL）、IVH、幕下、年齡≥80；分數越高死亡率越高 |
| **CHA₂DS₂-VASc**（合併 AF） | 決定 AF 是否抗凝 | 分數越高栓塞風險越高（中風本身即 ≥2 分）→ 抗凝 |
| **mRS** | 出院/追蹤功能預後 | 0 無症狀 → 6 死亡 |
