# Parkinson's disease（帕金森氏症）

> [!abstract] 一句話：**黑質緻密部（SNpc）多巴胺神經元退化**造成的動作徐緩性運動障礙，核心是 **bradykinesia ＋（靜止性顫抖 resting tremor／rigidity／postural instability）**，診斷靠**臨床＋對 levodopa 有反應**；值班最重要的一句話是「別急著停病人的抗巴金森藥，突然停 levodopa/dopamine agonist 可能引發致命的 **parkinsonism-hyperpyrexia syndrome**」。

> [!danger] 🚨 值班紅旗
> 1. **絕對不要突然停掉 levodopa／dopamine agonist**（NPO、手術、住院忘記開）→ 可引發類似 [[Neuroleptic Malignant Syndrome(惡性症候群)]] 的 **parkinsonism-hyperpyrexia syndrome / akinetic crisis**（高燒、僵直、意識改變、CK↑、橫紋肌溶解）；NPO 時要想辦法給（NG 管磨粉、rotigotine 貼片替代），劑量與轉換**⚠️需查證，依藥典/神經科**。
> 2. **PD 病人突然變差先找誘因（trigger）而非加藥**：感染（尤其 [[Urinary Tract Infection(泌尿道感染)]]、肺炎）、脫水、電解質、新加的**多巴胺拮抗劑**（haloperidol、metoclopramide、prochlorperazine）都會急性惡化——這些藥 PD 病人**禁用/慎用**。
> 3. **止吐與精神躁動別亂開**：需止吐選 **domperidone / ondansetron**（不過 BBB 較少）；PD 精神病／躁動選 **quetiapine 或 clozapine**，避免 typical antipsychotic 與 risperidone/olanzapine（惡化動作）。
> 4. **跌倒與吞嚥**：postural instability ＋ 姿勢性低血壓 → 高跌倒風險（顱內出血、髖骨折）；吞嚥障礙 → 吸入性肺炎，是 PD 主要死因之一。
> 5. **急性 parkinsonism ＋ 發燒／意識改變**要鑑別 NMS、serotonin syndrome、腦炎，別直接歸因「PD 惡化」。

## 📖 定義 / 病生理（這是什麼）
- **定義**：一種進行性神經退化疾病，病理為**黑質緻密部（substantia nigra pars compacta, SNpc）多巴胺神經元喪失** ＋ 神經元內 **α-synuclein 聚集（Lewy body / Lewy neurite）**，屬 **synucleinopathy**。
- **病生理**：SNpc → 紋狀體（striatum）的 nigrostriatal dopamine 路徑退化 → 基底核 direct/indirect pathway 失衡 → 運動輸出被過度抑制 → **動作變少變慢**。症狀通常在**多巴胺神經元喪失 60–80%** 後才出現。
- **Braak 分期假說**：α-synuclein 病理由下（延腦背側運動核、嗅球）往上（中腦→皮質）擴散，解釋**非運動前驅症狀**（便秘、嗅覺喪失、RBD 快速動眼期睡眠行為異常）早於運動症狀數年出現。
- **和「parkinsonism（巴金森症候群）」的區別**：Parkinsonism 是一組症狀（bradykinesia + rigidity/tremor），成因很多；**PD 是其中最常見的原因**。次要鑑別見下方相關卡。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Tremor(顫抖)]]、[[Gait Disturbance(步態異常)]]
- **四大運動核心（TRAP）**：
  | 徵象 | 特徵 |
  | --- | --- |
  | **T**remor | **靜止性顫抖**（4–6 Hz，pill-rolling 搓丸樣），動作時減輕，**單側起病**、不對稱 |
  | **R**igidity | **鉛管樣（lead-pipe）** 或 **齒輪樣（cogwheel）** 僵直 |
  | **A**kinesia/Bradykinesia | 動作變慢變小，**必要核心**；小字症（micrographia）、面具臉、聲音變小、眨眼少 |
  | **P**ostural instability | 姿勢反射喪失、後拉試驗（pull test）陽性、**易跌倒**（通常較晚出現） |
- **其他運動徵象**：慌張步態（festinating gait）、起步凍結（freezing of gait）、前傾駝背姿勢、擺臂減少。
- **非運動症狀（常被忽略但影響生活品質更大）**：便秘、嗅覺喪失（hyposmia）、**RBD**、憂鬱／焦慮、認知退化／失智（PDD）、幻覺、**姿勢性低血壓**與自律神經失調、流涎、睡眠障礙、疼痛。
- **不對稱性**是重要臨床線索：對稱起病要懷疑非典型或藥物性。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Levodopa challenge(左旋多巴測試)]]　影像 [[Brain MRI(腦部磁振造影)]]、[[DAT scan(多巴胺轉運體掃描)]]
- **臨床診斷**（MDS Clinical Diagnostic Criteria, 2015）：核心是 **bradykinesia ＋（rest tremor 或 rigidity）**，加上支持條件（明確 levodopa 反應）、排除條件與紅旗（red flags）。
- **對 levodopa 反應**：明顯且持續的改善是強力支持；反應差要重新想診斷（非典型 parkinsonism）。
- **影像**：
  - **MRI 腦部**：主要用來**排除**次發原因（血管性、常壓性水腦、Wilson、structural lesion），典型 PD 的 MRI 常正常。
  - **DAT-SPECT（¹²³I-ioflupane）**：區分「真的 presynaptic dopaminergic 缺損（PD/非典型）」與「essential tremor／drug-induced／psychogenic（DAT 正常）」，**不能區分 PD 與非典型 parkinsonism**。
- **實驗室**：年輕發病（< 50 歲）查 **Wilson's disease**（ceruloplasmin、24h 尿銅、K-F ring）；依情況查 B12、甲狀腺。
- **診斷是臨床導向**：沒有單一確診檢查，追蹤病程與治療反應同樣重要。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **不對稱、靜止性顫抖、對 levodopa 有反應** = 支持 idiopathic PD。
- **紅旗提示「非 PD」**（Parkinson-plus / 次發）：早期跌倒與失智、垂直凝視麻痺（→ [[Progressive Supranuclear Palsy(進行性上眼神經核麻痺)]] PSP）、早期嚴重自律神經衰竭＋小腦徵象（→ [[Multiple System Atrophy(多系統萎縮)]] MSA）、對稱起病、快速進展、對 levodopa 無反應。
- **藥物性 parkinsonism** 是最常被忽略的可逆原因：**metoclopramide、prochlorperazine、haloperidol、risperidone、valproate**；停藥可改善（但可能延遲數月）。
- **Essential tremor vs PD**：ET 是**動作性/姿勢性**顫抖、對稱、常有家族史、喝酒改善；PD 是**靜止性**、不對稱。
- **wearing-off、on-off、dyskinesia** 是長期 levodopa 治療的動作波動，屬進階照護議題。
- **PD 病人住院一定核對居家抗巴金森藥並按時給**，漏一餐都可能明顯惡化。

## 💊 治療（Acute / Chronic）
> 用藥 [[Levodopa(左旋多巴)]]、[[Dopamine agonist(多巴胺受體促效劑)]]
> ⚠️ 以下為原則，實際起始劑量／滴定／轉換**依神經科與藥典**，勿憑記憶下硬數字
### 症狀治療（沒有 disease-modifying 藥物，全為症狀控制）
- **Levodopa/carbidopa**：最有效的症狀藥，**老年、失能明顯者首選**；長期用會有動作波動與 dyskinesia，但不應為此延遲需要的治療。
- **Dopamine agonist**（pramipexole、ropinirole、rotigotine 貼片）：年輕、症狀較輕者可先用以延後 levodopa；注意**衝動控制障礙（賭博、購物、性慾）、嗜睡、幻覺、水腫**。
- **MAO-B inhibitor**（selegiline、rasagiline）：輕症單用或作為輔助。
- **COMT inhibitor**（entacapone）、**amantadine**（對 dyskinesia 有幫助）、抗膽鹼藥（trihexyphenidyl，年輕、顫抖為主，老人慎用怕認知副作用）。
- 具體藥物與劑量**⚠️需查證，依藥典/指引**。
### 非運動症狀
- 姿勢性低血壓（非藥物＋midodrine/fludrocortisone）、便秘、憂鬱、RBD（clonazepam/melatonin）、失智（rivastigmine）、精神病（quetiapine/clozapine/pimavanserin）。
### 進階治療
- **Deep Brain Stimulation（DBS，STN/GPi）**：適用於藥物反應好但有嚴重動作波動/dyskinesia 者；**levodopa-carbidopa intestinal gel**、apomorphine pump。
### 非藥物（同等重要）
- 運動復健、物理/職能/語言治療、跌倒預防、吞嚥評估。

## ⚠️ 併發症
- **吸入性肺炎**（吞嚥障礙）— 主要死因之一
- **跌倒 → 髖骨折／[[Subdural Hemorrhage(硬腦膜下出血)]]**
- **失智（Parkinson's disease dementia, PDD）** 與精神病、幻覺
- **姿勢性低血壓** → 昏厥、跌倒
- **Parkinsonism-hyperpyrexia syndrome / akinetic crisis**（突然停藥）
- 藥物副作用：dyskinesia、衝動控制障礙、幻覺
- 便秘、腸阻塞、營養不良、憂鬱、褥瘡（晚期臥床）

## 📊 嚴重度評分 / 風險分層（scoring）
> PD 用**分期**與**功能量表**，非單一 acute score
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **Hoehn & Yahr staging** | 疾病分期 | 1=單側；2=雙側無平衡障礙；3=雙側＋姿勢不穩但可自理；4=需協助但可站/走；5=輪椅/臥床 |
| **MDS-UPDRS** | 綜合嚴重度追蹤 | 四部分（非運動日常/運動日常/運動檢查/運動併發症），分數越高越重，追蹤病程與治療反應 |
| **Levodopa response** | 診斷與手術評估 | 反應明顯支持 idiopathic PD；DBS 前評估 on/off 差異 |
| **MoCA / MMSE** | 認知合併症 | 篩檢 PDD |
