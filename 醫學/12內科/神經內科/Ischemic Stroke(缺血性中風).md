# Ischemic Stroke（缺血性中風）

> [!abstract] 一句話：**腦血管阻塞→局部腦組織缺血壞死**造成的急性局灶神經缺損；「**Time is brain**」——每分鐘死約 190 萬顆神經元，核心是**先 CT 排除出血**，符合時間窗與條件者**靜脈溶栓（IV tPA/tenecteplase）＋大血管阻塞取栓（thrombectomy）**，同時找病因防再發（TOAST 分型）。

> [!danger] 🚨 值班紅旗
> 1. **急性局灶神經缺損 → 立刻啟動 stroke code、記錄「最後正常時間 LKW」、盡速 non-contrast CT** 排除出血（決定能否溶栓）；**時間就是腦，別排隊**
> 2. **符合條件者 IV 溶栓時間窗**：**tPA ≤ 4.5 小時（部分 ≤ 3h）**、大血管阻塞取栓可到 **6h（部分依 perfusion 影像至 24h，DAWN/DEFUSE-3）**——**先確認 LKW，別等到過窗**
> 3. **溶栓禁忌快速盤點**：近期出血／手術、顱內出血史、血壓 > 185/110 控制不下、INR/凝血異常、血糖過低過高（先驗 glucose，低血糖會偽裝中風！）
> 4. **後循環中風易被漏**：突發眩暈＋複視＋吞嚥困難＋共濟失調＋意識改變 → 想腦幹／小腦，**小腦大梗塞會腫脹壓迫腦幹＝致命，需神外減壓**
> 5. **溶栓後突然頭痛、意識下降、血壓飆、神經惡化** → **症狀性顱內出血**，停藥、急 CT、備血、神外會診

## 📖 定義 / 病生理（這是什麼）
- **定義**：腦血流中斷導致腦組織缺血，**症狀 > 24h 或影像有梗塞**（< 24h 完全緩解且無梗塞＝[[Transient Ischemic Attack(暫時性腦缺血)]]）。占所有中風約 **85%**（其餘為 [[Hemorrhagic Stroke(出血性中風)]]）。
- **缺血核心 vs penumbra**：核心（core）已不可逆壞死；**缺血半影區（penumbra）**血流低但尚存活，是**再灌流治療搶救的目標**。
- **TOAST 病因分型（防再發策略核心）**：
  | 分型 | 機轉 | 典型 |
  | --- | --- | --- |
  | **大動脈粥狀硬化** | 頸動脈／顱內狹窄 → 血栓/動脈-動脈栓塞 | 分水嶺／皮質梗塞，有頸動脈斑塊 |
  | **心源性栓塞** | AF、心瓣膜、LV 血栓、卵圓孔 | 多發、皮質、突發最重 |
  | **小血管（lacunar）** | 穿通支脂質玻璃樣變性 | 純運動／純感覺、內囊／橋腦 |
  | **其他確定病因** | 剝離、血管炎、高凝 | 年輕、無傳統危險因子 |
  | **原因不明（cryptogenic/ESUS）** | 查無 | 需長程心律監測找隱匿 AF |
- **危險因子**：高血壓（最重要）、AF、糖尿病、高血脂、吸菸、頸動脈狹窄、既往中風/TIA、年齡。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Hemiparesis(偏癱)]]、[[Aphasia(失語症)]]、[[Vertigo(眩暈)]]
- **突發**局灶缺損，**依血管分布定位**：
  | 血管 | 典型症候 |
  | --- | --- |
  | **MCA（中大腦動脈）** | 對側臉/手 > 腳無力、感覺喪失、凝視偏向病灶側、優勢半球 → 失語、非優勢 → 忽略 |
  | **ACA** | 對側腳 > 手無力、意志缺失（abulia） |
  | **PCA** | 對側同名偏盲、視覺失認 |
  | **後循環（VB）** | 眩暈、複視、吞嚥困難、共濟失調、交叉性症候（Wallenberg）、意識改變 |
  | **Lacunar** | 純運動偏癱、純感覺、運動失調輕癱、構音障礙-笨手 |
- **BE-FAST 篩檢**：Balance、Eyes、Face、Arm、Speech、Time。
- **注意偽裝**：低血糖、癲癇後 Todd's paralysis、偏頭痛先兆、Bell's palsy、周邊眩暈——**先驗血糖**。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Blood glucose(血糖)]]、[[CBC(全血球計數)]]、[[Coagulation profile(凝血功能)]]、[[ECG(心電圖)]]　影像 [[CT(電腦斷層)]]、[[MRI(磁振造影)]]、[[CT Angiography(電腦斷層血管攝影)]]
- **急診第一步**：**Non-contrast CT 腦**——**排除出血**（決定能否溶栓），早期缺血徵象（hyperdense MCA sign、灰白質界線模糊、insular ribbon）；ASPECTS 評分。
- **立即抽血/檢查**：**指尖血糖（先做，排除低血糖偽中風）**、CBC、凝血（PT/INR/aPTT）、電解質、腎功能、troponin、**12 導程 ECG（找 AF）**。
- **血管影像**：**CTA（頭頸）**找大血管阻塞（LVO）→ 決定取栓；CT perfusion 或 MRI DWI/PWI 評估 core/penumbra（延伸窗取栓）。
- **MRI DWI**：對超急性缺血最敏感（分鐘級即亮），後循環／小病灶更佳。
- **病因檢查（住院後）**：頸動脈超音波、心臟超音波、**長程心律監測**（找隱匿 AF）、血脂/糖化血色素、年輕人加高凝/血管炎。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **先驗血糖**——低血糖與高血糖都可偽裝或加重中風，且是溶栓前必查。
- **CT 早期常「看起來正常」**——不能因 CT 乾淨就排除中風；DWI 才敏感，但**CT 乾淨＋在時間窗＝可溶栓**（CT 的目的是排出血不是確診梗塞）。
- **後循環中風最常被漏**（單純眩暈被當內耳）——**HINTS 檢查**（中樞型：direction-changing nystagmus、skew deviation、正常 head impulse）可辨中樞 vs 周邊。
- **溶栓前血壓門檻 < 185/110**；不溶栓者血壓**允許偏高（permissive hypertension）**避免降太快減灌流。
- **Wake-up stroke / 不明發作時間**：可用 MRI DWI-FLAIR mismatch 或 perfusion 選溶栓／取栓對象。
- **小腦梗塞會腫**——48–72h 水腫高峰壓迫第四腦室／腦幹，**需監測、必要時去顱減壓**。
- 給溶栓後 24h 內**不給抗血小板/抗凝**、嚴密監測神經與血壓。

## 💊 治療（Acute / Chronic）
> 用藥 [[Alteplase(tPA)]]、[[Aspirin(阿斯匹靈)]]、[[Clopidogrel(保栓通)]]、[[Statin(史他汀)]]、[[Anticoagulant(抗凝血劑)]]
### 超急性再灌流（救 penumbra）
- **IV 溶栓（alteplase / tenecteplase）**：症狀 **≤ 4.5 小時**（部分 ≤ 3h，依年齡/病史）內、CT 無出血、無禁忌 → 給予；**劑量與適應/禁忌依 AHA/ASA 2019 acute stroke guideline 與 stroke team protocol，需查證**（alteplase 常規 0.9 mg/kg、最大 90 mg，10% bolus 其餘 1h 輸注——⚠️務必依 protocol 核對體重與禁忌）。
- **機械取栓（mechanical thrombectomy）**：**大血管阻塞（LVO，前循環近端）**，一般 **≤ 6h**；符合 perfusion mismatch（DAWN/DEFUSE-3）可延至 **16–24h**。取栓與溶栓不互斥（bridging）。
- **血壓管理**：溶栓者維持 **< 185/110**（給藥前）與 < 180/105（給藥後 24h）；不溶栓者容許較高。
### 急性期一般處置
- 未溶栓且無禁忌 → **抗血小板（aspirin）48h 內起始**；輕型中風/高危 TIA 可短期 **DAPT（aspirin + clopidogrel）**（依 CHANCE/POINT，⚠️劑量與療程查證）。
- 血糖、體溫、氧合、吞嚥評估（防吸入性肺炎）、DVT 預防、早期復健。
### 次級預防（防再發，依 TOAST）
- **心源性（AF）** → **抗凝**（DOAC 或 warfarin，依 CHA₂DS₂-VASc；出血風險與起始時機依 guideline，⚠️查證）。
- **大動脈粥狀硬化** → 抗血小板 + **高強度 statin** + 控制危險因子；頸動脈狹窄達標 → **CEA / 支架**。
- **Lacunar** → 抗血小板 + 血壓/血糖控制。
- 全面：戒菸、控糖控壓控脂、生活型態。

## ⚠️ 併發症
- **腦水腫／惡性 MCA 梗塞**（大面積，48–72h，可需去顱減壓）
- **出血性轉化**（自發或溶栓後症狀性 ICH）
- 癲癇、吸入性肺炎、DVT/PE、壓瘡、憂鬱、失能
- 小腦梗塞 → 腦幹壓迫、阻塞性 [[Hydrocephalus(水腦症)]]
- 中風後復發（未防再發者）

## 📊 嚴重度評分 / 風險分層（scoring）
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **NIHSS（0–42）** | 中風嚴重度、追蹤、治療決策 | 0 無、1–4 輕、5–15 中、16–20 中重、21–42 重；分數越高越重、溶栓/取栓決策參考 |
| **ABCD²（0–7）** | TIA 後短期中風風險 | Age≥60(1)、BP≥140/90(1)、Clinical（單側無力2/言語障礙1）、Duration（≥60min2/10–59min1）、Diabetes(1)；≥4 高風險，收治 |
| **ASPECTS（0–10）** | CT 早期缺血範圍 | ≤6 大範圍梗塞、取栓/溶栓效益/出血風險評估 |
| **CHA₂DS₂-VASc** | AF 病人抗凝決策 | 分數越高中風風險越高 → 需抗凝 |
| **mRS（0–6）** | 中風後功能結局 | 0 無症狀 … 5 重殘、6 死亡 |
