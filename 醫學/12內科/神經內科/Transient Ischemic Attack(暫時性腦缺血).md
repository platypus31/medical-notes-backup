# Transient Ischemic Attack（暫時性腦缺血）

> [!abstract] 一句話：**局部腦/視網膜缺血造成的短暫神經功能缺損，且無急性梗塞證據（組織型定義）**，症狀多在 1 小時內、無殘留；它是**即將中風的警報**——**發作後 48 小時到 90 天是中風高峰**，值班要當「還沒發生的中風」積極評估與啟動次級預防，別因「已經好了」而鬆懈。

> [!danger] 🚨 值班紅旗
> 1. **TIA = 急性中風前哨**：發作後**48 小時內中風風險最高**，要**急做影像＋盡快啟動抗血小板與危險因子控制**，不是門診慢慢排。**高風險（ABCD² 高分、頸動脈狹窄、AF、crescendo TIA、DWI 有病灶）需住院/急觀察**。
> 2. **症狀還在 = 當急性中風處理，走 stroke code / 評估再灌流時間窗**，不要因為「可能會自己好」而錯過血栓溶解/取栓時機。
> 3. **一定要驗血糖**：低血糖可完全模仿局部神經缺損（stroke mimic），先排除再說。
> 4. **找心房顫動**：EKG／延長心律監測；AF 相關 TIA 的次級預防是**抗凝血**（非只抗血小板），漏診 AF = 漏掉最有效預防。
> 5. **頸動脈症狀性狹窄（同側 amaurosis fugax / 半身症狀）** 需盡快血管影像；**症狀性狹窄 ≥50–70% 的頸動脈內膜切除/支架有時效性**（黃金期兩週內獲益最大）。

## 📖 定義 / 病生理（這是什麼）
- **定義（組織型，2009 AHA/ASA）**：**腦、脊髓或視網膜局部缺血造成的短暫神經功能障礙，且無急性梗塞的組織學/影像證據**。舊的「時間型（<24 小時）」定義已被取代——重點是**有無梗塞（DWI）**而非時間長短。
- **病生理**：短暫的局部血流不足（未達不可逆梗塞）→ 症狀隨血流恢復而消退。機轉同缺血性中風：
  | 機轉 | 說明 |
  | --- | --- |
  | **大血管動脈粥狀硬化** | 頸動脈/顱內狹窄、動脈到動脈栓塞 |
  | **心源性栓塞** | **心房顫動**、心瓣膜、心內血栓、卵圓孔未閉 |
  | **小血管（腔隙）** | 高血壓/糖尿病的穿通枝病變 |
  | 其他 | 血管剝離、血液高凝、血管炎 |
- **危險因子**：高血壓、糖尿病、高血脂、抽菸、心房顫動、頸動脈狹窄、年齡、既往中風/TIA。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Focal Weakness(局部無力)]]、[[Aphasia(失語症)]]、[[Amaurosis Fugax(一過性黑矇)]]
- **突發、局部、可對應血管領域**的神經缺損，**通常數分鐘至 1 小時內完全恢復**：
  - **前循環（頸動脈系）**：對側肢體無力/麻木、失語（優勢半球）、**同側單眼一過性黑矇（amaurosis fugax）**、忽略。
  - **後循環（椎基底動脈系）**：眩暈＋複視/構音障礙/吞嚥困難/共濟失調、雙側或交叉症狀、視野缺損。
- **crescendo TIA**：短時間內反覆發作、越來越頻繁 → 極高中風風險。
- **注意 stroke mimic**：低血糖、癲癇（Todd's）、偏頭痛先兆、暈厥、周邊前庭病變、心因性。
- **孤立性眩暈/頭暈很少單獨是 TIA**，但合併其他後循環徵象時要當真（用 HINTS 等區分中樞 vs 周邊）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[ECG(心電圖)]]、[[Blood Glucose(血糖)]]、[[Holter Monitor(24小時心電圖)]]　影像 [[Brain MRI(腦部磁振造影)]]、[[Carotid Doppler(頸動脈超音波)]]
- **急測血糖**（排除低血糖 mimic）＋ 生命徵象、詳細神經學檢查。
- **腦影像**：**MRI 含 DWI 為首選**（有無急性梗塞決定是 TIA 還是 minor stroke，並排除出血/佔位）；無法及時做 MRI 時先 CT 排除出血。
- **血管影像**：**頸動脈超音波／CTA／MRA**找症狀性狹窄與顱內血管病變。
- **心臟評估**：**12 導程 EKG**＋監測；懷疑陣發性 AF 用 **Holter／延長心律監測（可達數週）**；必要時心臟超音波（找心源栓塞、PFO）。
- **抽血**：血糖/HbA1c、血脂、CBC、凝血、腎功能；依情況查高凝狀態（年輕、無傳統危險因子）。
- **快速分流**：高風險 TIA 應住院或急速 TIA clinic，24 小時內完成關鍵評估。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **「已經好了」是陷阱**：TIA 的價值在於**它預告即將來的中風**，處置急迫性等同急性事件。
- **ABCD² 用來分層短期中風風險**，但**不能單靠它排除高風險病人**（頸動脈狹窄、AF、DWI 陽性、crescendo 都是獨立高風險，需影像＋心律資訊）。
- **DWI 有病灶其實已是梗塞（minor stroke）**，預後與處置比純 TIA 更積極。
- **找到 AF → 抗凝血**（不是只給 aspirin）；找到症狀性頸動脈狹窄 → 血管介入時效性。
- **高風險 TIA/minor stroke 的早期雙抗（DAPT）短期使用**（見治療）可降低早期復發——但療程要短、依 protocol，長期雙抗增加出血。
- **一律排除低血糖與其他 mimic**，別把每個短暫症狀都當 TIA。

## 💊 治療（Acute / Chronic）
> 用藥 [[Aspirin(阿斯匹靈)]]、[[Clopidogrel(氯吡格雷)]]、[[Statin(他汀類)]]
> ⚠️ 抗栓方案／療程依當地 stroke protocol 與指引，劑量查藥典
### 急性 / 次級預防啟動（越快越好）
- **抗血小板**：確認非出血後**盡早給 aspirin**；**高風險 TIA（ABCD² ≥4）或 minor stroke** 依指引可用**短期雙抗（aspirin + clopidogrel，約 21–90 天後轉單一）**降低早期復發（POINT / CHANCE 證據）——**療程與劑量⚠️需查證，依 protocol**。
- **心源性（AF）→ 抗凝血**（DOAC 或 warfarin，依情況），非只抗血小板。
- **他汀（高強度）**：積極降 LDL。
- **危險因子控制**：血壓（急性期避免過度降壓）、血糖、戒菸。
### 血管介入
- **症狀性頸動脈狹窄 ≥50–70%**：**頸動脈內膜切除術（CEA）或支架**，**盡早（黃金期約兩週內）**評估執行。
### 症狀還在時
- 依急性缺血性中風流程評估**再灌流治療（血栓溶解/取栓）時間窗**——不要因「可能是 TIA」而觀望。

## ⚠️ 併發症
- **後續缺血性中風**（最重要，尤其 48 小時–90 天內）
- 反覆 TIA、crescendo TIA
- 抗栓治療相關出血（尤其雙抗/抗凝）
- 血管介入相關（CEA/支架的中風、心肌梗塞、再狹窄）
- 未矯正危險因子 → 心血管事件（MI）並存風險

## 📊 嚴重度評分 / 風險分層（scoring）
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **ABCD² score** | TIA 後短期（2/7/90 天）中風風險分層 | **A**ge ≥60(1)、**B**P ≥140/90(1)、**C**linical（單側無力 2／言語障礙無無力 1）、**D**uration（≥60min 2／10–59min 1）、**D**iabetes(1)；總 0–7。**0–3 低、4–5 中、6–7 高**風險（高分傾向住院/急評估） |
| **DWI 陽性** | 組織學分層 | 有梗塞 = minor stroke，復發風險更高 |
| **頸動脈狹窄程度** | 血管介入決策 | 症狀性 ≥50–70% → 考慮 CEA/支架 |
| **CHA₂DS₂-VASc**（若合併 AF） | AF 抗凝決策 | 分數越高中風風險越高，指引抗凝 |
