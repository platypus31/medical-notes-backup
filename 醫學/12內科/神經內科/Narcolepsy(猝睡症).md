# Narcolepsy（猝睡症）

> [!abstract] 一句話：**下視丘 orexin（hypocretin）神經元喪失**造成的慢性睡眠─清醒調節障礙，核心是**無法抗拒的日間嗜睡（EDS）**，第一型伴**猝倒（cataplexy，情緒誘發的短暫肌張力喪失）**；診斷靠**多次睡眠潛伏期測試（MSLT）見 SOREMPs**，值班意義多在鑑別「日間過度嗜睡」與避免誤診/誤用鎮靜藥。

> [!danger] 🚨 值班紅旗（猝睡症本身少急症，重點在別誤判 EDS 的危險成因）
> 1. **新發、突然的日間嗜睡＋神經缺損／頭痛／內分泌異常** → 排除**下視丘/第三腦室病灶**（腫瘤、發炎）造成的次發性 narcolepsy，影像
> 2. **猝倒（cataplexy）易被誤當癲癇或昏厥** → 猝倒**意識清醒、情緒（大笑）誘發、無後遺**，別當 seizure 亂給抗癲癇/誤插管
> 3. **嚴重日間嗜睡的鑑別**：睡眠呼吸中止（[[Sleep Apnea Syndrome(呼吸停止症候群)]]）、睡眠剝奪、藥物、憂鬱、甲狀腺低下——先排這些常見可逆因
> 4. **興奮劑治療的病人**：注意心血管副作用、藥物交互與濫用；**sodium oxybate（GHB）過量會呼吸抑制**，勿與鎮靜/酒精併用
> 5. **駕駛與工作安全**：EDS 病人猝睡可致意外，衛教與職業評估

## 📖 定義 / 病生理（這是什麼）
- **定義**：慢性神經疾患，REM 睡眠調控失常，特徵為 EDS 與 REM 現象闖入清醒（猝倒、睡眠麻痺、入睡幻覺）。
- **病生理**：**下視丘外側 orexin（hypocretin）神經元選擇性喪失**（第一型 CSF orexin 極低）；被認為是**自體免疫**（與 **HLA-DQB1*06:02** 強相關、鏈球菌/流感（含 2009 H1N1 疫苗 Pandemrix）觸發報告）。
- **分型**：
  | 型別 | 猝倒 | CSF orexin | 特點 |
  | --- | --- | --- | --- |
  | **Type 1** | **有** | 低（≤110 pg/mL） | orexin 缺失明確 |
  | **Type 2** | 無 | 正常/未測 | 診斷靠 MSLT |
  | **次發性** | 依病灶 | 依病灶 | 下視丘病變、Prader-Willi、髓鞘疾病 |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Excessive daytime sleepiness(日間過度嗜睡)]]、[[Cataplexy(猝倒)]]
- **四聯徵（narcoleptic tetrad，不一定全有）**：
  | 症狀 | 描述 |
  | --- | --- |
  | **日間過度嗜睡（EDS）** | 核心且必備；無法抗拒的睡意、突發小睡、睡後短暫清爽 |
  | **猝倒（cataplexy）** | Type 1 特有；**情緒（尤大笑）誘發**的雙側肌張力突然喪失（膝軟、下顎垂、倒地），**意識清醒**、數秒～分鐘、可逆 |
  | **睡眠麻痺** | 入睡/醒來時暫時無法動彈（REM 肌張力抑制闖入） |
  | **入睡/醒前幻覺** | 生動、常帶恐懼的視聽幻覺 |
- **其他**：夜間睡眠破碎、自動行為、肥胖傾向、兒童可能表現學習/行為問題。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Polysomnography(多項睡眠生理檢查)]]、[[MSLT(多次睡眠潛伏期測試)]]、[[CSF orexin(腦脊髓液食慾素)]]　影像 [[MRI(磁振造影)]]（疑次發性時）
- **前置**：先排除睡眠剝奪、輪班、藥物、其他睡眠疾患（做**睡眠日誌/actigraphy**≥1–2 週、前一晚 PSG 排除 OSA/確保足夠睡眠）。
- **夜間 PSG（前一晚）**：排除其他睡眠障礙，並確認足夠睡眠。
- **MSLT（隔日，5 次小睡）**：**平均睡眠潛伏期 ≤ 8 分鐘 ＋ ≥ 2 次入睡期 REM（SOREMPs）**（前一晚 PSG 的 SOREMP 可算 1 次）——診斷關鍵。
- **CSF orexin/hypocretin-1 ≤ 110 pg/mL**：Type 1 高度支持（尤其猝倒不典型時）。
- **HLA-DQB1*06:02**：支持但非診斷（一般人也常帶）。
- **影像**：疑次發性（下視丘病灶）時做 MRI。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **猝倒＝意識清醒＋情緒誘發＋肌張力喪失**，與**癲癇（意識改變、腦電異常）**、**昏厥（低灌流、意識喪失）**不同——別誤診。
- **MSLT 前必須確保足夠睡眠且停用會影響 REM 的藥（抗憂鬱、興奮劑）足夠時間**，否則偽陽/偽陰。
- **EDS 最常見原因不是猝睡症**——先排睡眠不足、OSA、輪班、憂鬱、藥物。
- **Type 1 有明確生物標記（低 orexin）**，Type 2 純臨床＋MSLT，較難、需追蹤。
- **抗憂鬱藥突然停用可能誘發「status cataplecticus」**（猝倒惡化）。
- **sodium oxybate 是管制藥（GHB）**，呼吸抑制風險，禁與酒精/鎮靜併用。

## 💊 治療（Acute / Chronic）
> 用藥 [[Modafinil(莫達非尼)]]、[[Methylphenidate(派醋甲酯)]]、[[Sodium oxybate(羥基丁酸鈉)]]、[[Pitolisant(匹托利生)]]、[[Venlafaxine(文拉法辛)]]
### 非藥物（基礎）
- **規律排定小睡（scheduled naps）**、充足夜眠、規律作息、避酒精、駕駛/職業安全衛教。
### 日間嗜睡（EDS）
- **促醒劑**：**modafinil / armodafinil**（第一線）、**pitolisant（H3 逆效劑）**、**solriamfetol**；效果不足用 **methylphenidate / amphetamine 類**（⚠️劑量依藥典，心血管監測、成癮性）。
### 猝倒 / REM 現象
- **sodium oxybate（GHB）**：對猝倒與 EDS 皆有效，但管制、呼吸抑制風險（⚠️嚴依 protocol）。
- **抗猝倒**：**venlafaxine / SSRI/SNRI / clomipramine**（抑制 REM），**pitolisant** 亦有效。
### 次發性
- 針對下視丘病灶病因治療。

## ⚠️ 併發症
- 意外傷害（駕駛、操作機械時猝睡/猝倒）
- 學業/職業表現受損、社交與心理影響（憂鬱常見）
- 肥胖、代謝問題
- 藥物副作用（興奮劑心血管/成癮、oxybate 呼吸抑制）
- 猝倒惡化（抗憂鬱藥驟停）

## 📊 嚴重度評分 / 風險分層（scoring）
> 無「致命度」score；用嗜睡量表與客觀睡眠測試
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **Epworth Sleepiness Scale（ESS，0–24）** | 主觀日間嗜睡 | > 10 過度嗜睡；追蹤治療反應 |
| **MSLT** | 客觀診斷 | 平均潛伏期 ≤ 8 min ＋ ≥ 2 SOREMPs |
| **CSF orexin** | Type 1 標記 | ≤ 110 pg/mL 支持 Type 1 |
| **ICSD-3 準則** | 診斷分型 | Type 1（含猝倒/低 orexin）vs Type 2 |
