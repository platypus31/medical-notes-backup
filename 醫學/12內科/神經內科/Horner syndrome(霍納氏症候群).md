# Horner syndrome（霍納氏症候群）

> [!abstract] 一句話：**同側交感神經眼路徑（三級神經元）中斷**造成的三聯徵——**縮瞳（miosis）＋眼瞼下垂（ptosis，輕度）＋無汗（anhidrosis）**；本身不致命，但**它是「路徑上某處被病灶壓迫／破壞」的訊號**，值班關鍵是找出病灶（尤其**頸動脈剝離、Pancoast tumor、腦幹中風**），別只當作眼科小問題。

> [!danger] 🚨 值班紅旗
> 1. **急性 Horner ＋ 同側頸痛／臉痛／頭痛** → **內頸動脈剝離（carotid dissection）**直到排除為止，是年輕人中風的重要成因，**趕快 CTA/MRA 頸部血管**，別拖
> 2. **Horner ＋ 眩暈／複視／吞嚥困難／同側顏面痛、對側肢體溫痛覺喪失** → **延髓外側症候群（Wallenberg，[[Ischemic Stroke(缺血性中風)]]）**，走急性中風流程
> 3. **Horner ＋ 手臂痛／手內在肌無力／頸肩腫塊／吸菸史** → **肺尖 Pancoast tumor** 侵犯下臂神經叢與交感鏈，安排胸部影像
> 4. **兒童 Horner ＋ 腹部腫塊／異色瞳（heterochromia）** → **神經母細胞瘤（neuroblastoma）**，查尿液 VMA/HVA、影像
> 5. **瞳孔不等（anisocoria）在暗處更明顯** → 是「小瞳孔異常」＝交感受損（Horner），**別誤判成對側的大瞳孔為異常**（那會誤導成動眼神經／Adie 方向）

## 📖 定義 / 病生理（這是什麼）
- **定義**：支配眼與臉的**交感神經路徑（oculosympathetic pathway）**任一段受損，導致同側 **miosis + ptosis + anhidrosis**（±眼球內陷 enophthalmos 的錯覺）。
- **三級神經元路徑（定位核心）**：
  | 級別 | 走向 | 常見病因（定位意義） |
  | --- | --- | --- |
  | **一級（中樞）** | 下視丘 → 腦幹 → 睫脊中樞（C8–T2 Budge 中樞） | 腦幹中風（**Wallenberg**）、脊髓病灶、腫瘤、去髓鞘 |
  | **二級（節前 preganglionic）** | 脊髓 → 經肺尖 → 上頸交感節 | **Pancoast tumor**、頸／縱膈腫瘤、鎖骨下手術、胸廓外傷 |
  | **三級（節後 postganglionic）** | 上頸節 → 沿**內頸動脈**入海綿竇 → 眼 | **頸動脈剝離**、海綿竇病灶、cluster headache、頸內動脈瘤 |
- **無汗的分布幫定位**：一級／二級（節前）→ 同側**半臉／半身**無汗（交感汗腺纖維在頸節前分出）；三級（節後）→ 汗腺纖維多沿外頸動脈走，**臉部無汗很輕或無**——臉會不會流汗是節前 vs 節後的線索。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Ptosis(眼瞼下垂)]]、[[Anisocoria(瞳孔不等)]]
- **三聯徵**：
  - **Miosis（縮瞳）**：同側瞳孔小，**暗處瞳孔放大延遲（dilation lag）**、瞳孔不等在暗處更明顯。
  - **Ptosis（輕度上眼瞼下垂）**：因 Müller 肌（交感支配）無力，**下垂程度比動眼神經麻痺輕**；下眼瞼可能輕微上抬（**upside-down ptosis / reverse ptosis**）→ 造成眼球內陷的錯覺。
  - **Anhidrosis（無汗）**：同側臉／半身，依病灶級別分布不同（見上）。
- **伴隨徵象定位**（值班重點）：頸痛（剝離）、腦幹徵象（複視、吞嚥困難、共濟失調、對側溫痛覺喪失＝Wallenberg）、臂叢徵象（Pancoast）、異色瞳（先天或兒童期發病）。
- **先天型 Horner**：常伴**虹膜異色（heterochromia iridis）**——患側虹膜色較淺（黑色素細胞交感依賴）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Pharmacologic pupil testing(藥物瞳孔測試)]]　影像 [[MRI(磁振造影)]]、[[CT Angiography(電腦斷層血管攝影)]]
- **臨床確認**：先在**明處與暗處**量瞳孔——Horner 的 anisocoria **暗處放大（因患側縮瞳肌失能，健側正常放大）**，且有 dilation lag。
- **藥理測試（確認 + 定位，非急診必需但典型）**：
  - **Apraclonidine 0.5–1%（首選確認）**：Horner 患側因去神經超敏 → **瞳孔反而放大（anisocoria 反轉）**；正常眼無反應。（⚠️新生兒／嬰兒慎用，有中樞抑制風險）
  - 傳統 **Cocaine 4–10% test**：正常眼放大、Horner 眼**不放大**（阻斷 NE 再回收，需完整節後末梢釋放 NE）。
  - **Hydroxyamphetamine 1%（定位節前 vs 節後）**：節後（三級）病灶 → **不放大**；節前／中樞 → 放大。（須與 cocaine 隔 ≥ 24–48h）
- **影像（真正決定處置）**：依定位選擇——
  - 疑**節後／頸痛** → **頸部＋腦部 MRI/MRA 或 CTA**（找頸動脈剝離、海綿竇）。
  - 疑**節前** → 胸部 CT（肺尖 Pancoast）＋頸部影像。
  - 疑**中樞** → **腦部 MRI**（腦幹／脊髓）。
  - **急性 Horner 一律當「須找病灶」處理**；病因不明的急性痛性 Horner → 血管影像不能省。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **異常的是「小瞳孔」那側（交感）**——別把對側正常大瞳孔當異常；判斷方法就是「暗處誰放不大」。
- **Horner 的 ptosis 是輕度且瞳孔小**；動眼神經（CN III）麻痺是**重度 ptosis ＋瞳孔大（散大）＋眼球位置異常**——方向相反，別混。
- **急性痛性 Horner ＝頸動脈剝離**，是可治療、可致命（中風）的診斷，**最不能漏**。
- **無汗分布** = 節前 vs 節後的床邊線索（半臉／半身 vs 幾乎無）。
- **Pancoast**：Horner 可能是唯一線索，配上臂痛＋吸菸史要照胸。
- **兒童新發 Horner**：務必排 neuroblastoma（腹部腫塊、尿 VMA/HVA）。
- Apraclonidine 測試有**假陰性（急性期去神經超敏未建立，需數天～2週）**——太早驗可能不顯。

## 💊 治療（Acute / Chronic）
> 用藥 [[Antiplatelet(抗血小板藥物)]]、[[Anticoagulant(抗凝血劑)]]
- **Horner 本身沒有專屬治療——治療針對「找到的病因」**：
  - **頸動脈剝離** → 抗栓治療（抗血小板或抗凝，**依剝離範圍、是否有缺血、出血風險決定，依神經內科／中風團隊 protocol**，⚠️劑量與藥物選擇需查證）；部分需血管介入。
  - **腦幹中風（Wallenberg）** → 走急性缺血性中風流程（見 [[Ischemic Stroke(缺血性中風)]]），吞嚥評估防吸入。
  - **Pancoast tumor / 腫瘤** → 腫瘤科／胸腔外科處置。
  - **Neuroblastoma** → 兒童腫瘤團隊。
- **對稱美觀**：ptosis 明顯者慢性期可考慮 Müller 肌手術（少用）。

## ⚠️ 併發症
- 本身併發症少；**風險來自未診斷的病因**：頸動脈剝離 → 缺血性中風；Pancoast → 腫瘤進展、臂叢功能喪失；腦幹病灶 → 吞嚥障礙／吸入性肺炎。
- 兒童：neuroblastoma 延遲診斷。

## 📊 嚴重度評分 / 風險分層（scoring）
> Horner 本身無專用 score；「嚴重度」在於背後病因
| 工具 | 用途 | 意義 |
| --- | --- | --- |
| **無專用 score** | — | Horner 是徵候不是疾病；風險依病因分層 |
| 定位（三級神經元） | 病因方向 | 節後＋痛 → 剝離；節前 → Pancoast；中樞 → 腦幹 |
| 病因對應之 score | 依病因 | 若為中風走 [[NIHSS]]；剝離看是否缺血 |
