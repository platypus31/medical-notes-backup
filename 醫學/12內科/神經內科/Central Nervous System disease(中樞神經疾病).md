# Central Nervous System disease（中樞神經疾病）

> [!abstract] 一句話：這是**腦、脊髓、腦膜與腦血管一切病變的總稱（傘狀概念，非單一診斷）**，值班時的用途是**框架**——遇到「意識改變／局灶神經缺損／頭痛／抽搐」時，用「血管、感染、腫瘤、脫髓鞘、代謝/中毒、外傷、退化、癲癇（VITAMINS-D 式）」的軸線快速定位；臨床提醒：**先分辨『可救的急症（中風、出血、腦膜炎、ICP 升高、癲癇重積）』，因為時間就是腦組織**。

> [!danger] 🚨 值班紅旗
> 1. **突發局灶神經缺損（臉歪、單側無力、口齒不清）** → 當急性中風處理，**立刻 CT 分辨缺血 vs 出血**，走 stroke code（[[CerebroVascular Accident(腦血管意外傷害)]]）；缺血有 tPA/取栓時間窗，分秒必爭。
> 2. **發燒 + 頭痛 + 頸僵硬 + 意識改變** → 細菌性腦膜炎/腦炎是急症，**別等影像/LP 才給抗生素**（依 protocol 先經驗性抗生素 ± acyclovir），延誤致死或永久失能。
> 3. **ICP 升高徵象**（劇烈頭痛、噴射狀嘔吐、意識下降、Cushing triad：高血壓＋心搏過緩＋不規則呼吸、單側瞳孔散大）→ 腦疝前兆，神經外科急會診、頭抬高、控 CO₂/滲透治療。
> 4. **癲癇重積（seizure > 5 min 或反覆未恢復）** → 立即 benzodiazepine + 抗癲癇，找病因（[[Seizure(癲癇)]]）。
> 5. **意識改變先驗可逆代謝因**（血糖、鈉、鈣、氨、缺氧、藥物/中毒、[[Wernicke encephalopathy(魏尼凱氏腦病變)]]）——別急著全歸「腦本身壞了」。

## 📖 定義 / 病生理（這是什麼）
- **定義**：涵蓋中樞神經系統（大腦、小腦、腦幹、脊髓、腦膜、腦室與腦血管）之各類疾病的**總稱型分類**，臨床上作為鑑別診斷（DDx）的頂層節點。
- **病因分類框架（值班快速掃描）**：
  | 類別 | 代表疾病（連結） |
  | --- | --- |
  | **血管性 Vascular** | [[Ischemic Stroke(缺血性中風)]]、[[Hemorrhagic Stroke(出血性中風)]]、[[Transient Ischemic Attack(暫時性腦缺血)]]、[[CerebroVascular Accident(腦血管意外傷害)]] |
  | **感染/發炎 Infection/Inflammation** | 腦膜炎、腦炎（含自體免疫 anti-NMDA-R）、腦膿瘍 |
  | **腫瘤 Neoplasm** | 原發/轉移腦瘤、[[Schwannoma(神經鞘瘤)]] |
  | **脫髓鞘 Demyelinating** | 多發性硬化症、[[Central Pontine Myelinolysis(中央橋腦髓鞘溶解症)]] |
  | **退化 Degenerative** | [[Dementia(失智症)]]、[[Parkinson's disease(帕金森氏症)]]、[[Motor Neuron Disease(運動神經元疾病)]] |
  | **代謝/中毒 Metabolic/Toxic** | 肝性/尿毒性腦病變、[[Wernicke encephalopathy(魏尼凱氏腦病變)]]、[[Alcohol withdrawal(酒精戒斷)]] |
  | **癲癇/發作性 Paroxysmal** | [[Seizure(癲癇)]]、[[Migraines(偏頭痛)]] |
  | **結構/腦脊液 Structural/CSF** | [[Hydrocephalus(水腦症)]]、[[Pseudotumor Cerebri(假性腦瘤)]] |
  | **意識/行為狀態** | [[Coma(昏迷)]]、[[Delirium(譫妄)]]、[[Akinetic mutism(不動不語症)]]、[[Catatonia(緊張症)]]、[[Lock in syndrome(閉鎖症候群)]] |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Altered mental status(意識改變)]]、[[Headache(頭痛)]]、[[Focal neurological deficit(局灶神經缺損)]]
- **意識層面**：清醒度下降（嗜睡→木僵→昏迷）或內容異常（譫妄、混亂）。
- **局灶徵象**：偏癱、偏側感覺異常、失語、視野缺損、顱神經麻痺、共濟失調。
- **激發性**：癲癇發作、頭痛、噁心嘔吐（ICP）、頸僵硬（腦膜刺激）。
- **定位思維**：由徵象組合推病灶（皮質 vs 皮質下 vs 腦幹 vs 小腦 vs 脊髓）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 血糖/電解質/氨/血氣、CBC/CRP、血培養、腰椎穿刺(CSF)　影像 [[Brain CT(腦部電腦斷層)]]、[[Brain MRI(腦部磁振造影)]]、CTA/MRA
- **急診三步**：ABC + 血糖 → 神經學定位 → **影像（先 CT 分辨出血/中線位移/水腦，再視情況 MRI）**。
- **腰椎穿刺**：疑感染（腦膜炎/腦炎）、SAH（CT 陰性但高懷疑）、發炎/自體免疫；**LP 前若有局灶徵象或 ICP 疑慮先影像排腫塊/腦疝風險**。
- **EEG**：疑癲癇/NCSE、不明意識改變。
- **實驗室**：找可逆代謝/中毒因、感染指標、自體免疫與腦炎抗體。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- 本卡是**鑑別診斷的頂層 hub**，實戰請往下鑽到專屬疾病卡。
- **時間敏感的急症優先**：中風、出血、腦膜炎/腦炎、ICP 升高、癲癇重積——先排這些「可救且會惡化得很快」的。
- **意識改變 ≠ 腦結構壞掉**：先掃可逆代謝/中毒/低血糖。
- **LP 前的影像判斷**（局灶徵象/ICP 疑慮）避免腦疝。
- 定位（localization）永遠先於病因清單，能省大量無效檢查。

## 💊 治療（Acute / Chronic）
> 用藥 依病因（見各專屬疾病卡）
- **原則**：先穩定生命徵象與 ABC → 針對**具體診斷**治療（本卡不開單一藥物，導向各病專卡）。
- **時間敏感急症**：缺血中風 tPA/取栓、出血控壓與逆轉抗凝、細菌性腦膜炎經驗性抗生素、癲癇重積 benzodiazepine、ICP 升高滲透治療與外科減壓——**細節與劑量見各病專卡並依 protocol/藥典**（⚠️需查證）。

## ⚠️ 併發症
- 腦疝、續發性腦損傷、癲癇
- 吸入性肺炎、深部靜脈栓塞、壓瘡（臥床）
- 長期失能、認知障礙
- 各病因專有併發症（見各專卡）

## 📊 嚴重度評分 / 風險分層（scoring）
> 依「意識」與「病因」選對應量表——本傘狀分類無單一 score
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **Glasgow Coma Scale（GCS）** | 意識分層（通用） | 3–15，≤ 8 常需保護呼吸道 |
| **NIHSS** | 中風嚴重度 | 見 [[CerebroVascular Accident(腦血管意外傷害)]] |
| **ICH Score / Hunt-Hess / Fisher** | 出血/SAH 預後 | 見 [[Hemorrhagic Stroke(出血性中風)]] |
| 病因特異量表 | 各病分層 | 見對應疾病卡 |

> 中樞神經疾病為總稱，實際嚴重度分層落在各專屬疾病卡。
