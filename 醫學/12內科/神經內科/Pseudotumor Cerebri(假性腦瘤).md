# Pseudotumor Cerebri（假性腦瘤）

> [!abstract] 一句話：**沒有腫瘤或水腦的顱內壓升高（idiopathic intracranial hypertension, IIH）**，典型是**育齡肥胖女性**的**頭痛＋視乳突水腫（papilledema）＋短暫視力模糊**，診斷需先**影像排除佔位／靜脈竇栓塞**再靠**腰椎穿刺量到高壓、CSF 成分正常**；值班最不能漏的是**進行性視力喪失（可致永久失明）**。

> [!danger] 🚨 值班紅旗
> 1. **視力急速惡化／視野缺損** = 視神經受威脅的**眼科急症** → 緊急眼科＋神經科會診，可能需**緊急降壓／手術（視神經鞘開窗、分流）**，別排隊等門診。
> 2. **診斷 IIH 前一定先影像排除次發原因**：佔位病灶、**腦靜脈竇栓塞（CVST）**、水腦；尤其**非典型病人（男性、瘦、老人、小孩）**更要積極找次發原因，別直接貼 IIH 標籤。
> 3. **突發最劇烈頭痛（thunderclap）＋ 頸僵** 不是 IIH，要想 [[Subarachnoid Hemorrhage(蜘蛛膜下腔出血)]]／CVST，走那條路徑。
> 4. **第六對腦神經麻痺（複視）** 是顱內高壓的 false localizing sign，看到別誤以為有結構性病灶，但仍需影像。
> 5. **腰椎穿刺前一定要先影像**排除佔位與 midline shift（避免腦疝）；IIH 的 LP 兼具**診斷（量壓）與治療（放液暫時緩解）**。

## 📖 定義 / 病生理（這是什麼）
- **定義**：**顱內壓升高但無佔位病灶、無水腦、CSF 成分正常、無其他可解釋原因**的症候群。慣稱 pseudotumor cerebri，現多稱 **idiopathic intracranial hypertension (IIH)**。
- **病生理（尚未完全清楚）**：可能與 **CSF 吸收障礙（蛛網膜顆粒）**、**腦靜脈竇壓力升高／橫竇狹窄**、肥胖造成腹內/胸內壓升高影響靜脈回流、以及維生素 A／類視色素代謝有關。
- **好發族群**：**育齡期、肥胖女性**（典型）。
- **誘發因素／次發（secondary）原因（要主動問藥物史）**：**維生素 A 過量、tetracycline／doxycycline／minocycline、類固醇（尤其減量時）、生長激素、口服避孕藥（關聯較弱）、鋰、amiodarone、視黃酸（isotretinoin）**、腎衰、SLE、貧血。這些有明確原因者嚴格說是「pseudotumor cerebri syndrome」而非「idiopathic」。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Headache(頭痛)]]、[[Visual Disturbance(視力障礙)]]
- **頭痛**：最常見（>90%），多為每日、瀰漫、姿勢相關（躺下/valsalva 加重），可像偏頭痛。
- **短暫視力模糊（transient visual obscurations, TVOs）**：數秒的單/雙眼視力灰暗，常隨姿勢改變或 valsalva。
- **搏動性耳鳴（pulsatile tinnitus）**：頗具特異性的線索。
- **複視（diplopia）**：**第六對腦神經（CN VI / abducens）麻痺** — false localizing sign。
- **視乳突水腫（papilledema）**：關鍵徵象，眼底可見；長期未治 → 視神經萎縮 → **永久視野缺損／失明**。
- **視野缺損**：早期為**生理盲點擴大、周邊視野縮小**，中心視力通常較晚受影響。
- 神經學檢查**其他項目通常正常**（意識清楚、無局灶無力）——「清醒、聰明、只有頭痛與 papilledema」是典型畫面。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Lumbar Puncture(腰椎穿刺)]]、[[Fundoscopy(眼底鏡檢查)]]、[[Visual Field Test(視野檢查)]]　影像 [[Brain MRI(腦部磁振造影)]]、[[MR Venography(磁振靜脈造影)]]
- **改良 Dandy criteria（IIH 診斷要件）**：
  1. 顱內高壓的**症狀/徵象**（頭痛、papilledema）；
  2. **神經學檢查正常**（可有 CN VI 麻痺）；
  3. **神經影像正常**（無佔位、無水腦、無異常腦膜強化），且**排除靜脈竇栓塞**（需 **MRV**）；
  4. **腰椎穿刺開放壓升高**（成人一般 **> 25 cmH₂O**，兒童門檻不同）且 **CSF 成分正常**；
  5. **無其他可解釋的病因**。
- **影像**：先做 **MRI 腦部 ＋ MRV**（排除 CVST、佔位）。支持 IIH 的間接影像徵象：**空蝶鞍（empty sella）、視神經鞘擴張與後鞏膜扁平、橫竇狹窄**。
- **眼科**：**視野檢查（perimetry）與 OCT** 監測視神經，是決定治療強度與追蹤的核心。
- **注意族群**：非典型（男性、非肥胖、老人、兒童）務必更嚴格排除次發原因。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **視力保衛戰才是重點，頭痛只是症狀**：治療目標第一是**保住視力**，第二才是止頭痛。
- **一定要做 MRV**：橫竇栓塞（CVST）可完全模仿 IIH，漏掉會延誤抗凝治療。
- **CN VI 麻痺是 false localizing sign**，不代表有腦幹/後顱窩病灶。
- **LP 兼具診斷與治療**：放 CSF 可暫時緩解，但不是長期治療手段。
- **可逆藥物因**（tetracycline 家族、維生素 A、視黃酸、類固醇減量）要主動問、主動停。
- **體重下降是最有效的長期治療**之一（減 5–10% 體重常顯著改善）。

## 💊 治療（Acute / Chronic）
> 用藥 [[Acetazolamide(乙醯偶氮胺)]]
> ⚠️ 劑量依藥典/眼神經科，勿憑記憶下硬數字
### 一般
- **減重**（最重要的 disease-modifying 措施）＋ 停用致病藥物。
### 藥物
- **Acetazolamide（碳酸酐酶抑制劑）**：第一線，降低 CSF 生成；副作用感覺異常、味覺改變、腎結石、代謝性酸中毒。劑量**⚠️需查證，依藥典滴定**。
- **Topiramate**：兼有減重與降 CSF 效果，可作替代。
- **利尿劑 furosemide**（輔助）；頭痛另予對症治療。
### 手術／介入（視力受威脅或藥物失敗）
- **視神經鞘開窗術（optic nerve sheath fenestration）**：以保護視力為主。
- **CSF 分流術（VP/LP shunt）**：頭痛與視力兼顧。
- **靜脈竇支架（venous sinus stenting）**：針對橫竇狹窄／壓力梯度顯著者（選擇性）。
- **緊急**：暴發性 IIH 伴急速視力惡化 → 連續 LP／腰椎引流過渡到手術。

## ⚠️ 併發症
- **永久視力喪失／失明**（最嚴重，源於長期 papilledema → 視神經萎縮）
- 視野缺損、中心視力受損
- 慢性頭痛、生活品質下降
- 復發（尤其體重回升）
- 手術相關併發症（分流阻塞/感染、支架相關）

## 📊 嚴重度評分 / 風險分層（scoring）
> 無單一評分；以**視功能監測**分層決定治療強度
| 評分 / 指標 | 用途 | 意義 |
| --- | --- | --- |
| **改良 Dandy criteria** | 診斷 | 5 要件全符合方可診斷 IIH |
| **開放壓（opening pressure）** | 診斷/嚴重度 | 成人 > 25 cmH₂O 支持；極高壓＋視力惡化 = 需積極治療 |
| **Frisén papilledema grade** | 視乳突水腫分級 | 0–5 級，越高越嚴重 |
| **視野（perimetry）＋ OCT RNFL** | 監測與治療決策 | 進行性視野缺損 → 升級治療/手術 |
| 病程分型 | 治療緊迫性 | **暴發性（fulminant）**：數天內急速視力惡化，需緊急手術 |
