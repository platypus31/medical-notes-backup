# Coma（昏迷）

> [!abstract] 一句話：**意識（清醒 arousal ＋ 覺知 awareness）持續喪失、無法被喚醒、對外界刺激無有意義反應的狀態**，本質是**雙側大腦半球廣泛失能**或**上腦幹網狀活化系統（ARAS）受損**。值班遇到「叫不醒」永遠是**先救命再找病因**——**ABC → 驗血糖 → 給 O₂ → 必要時 DONT 經驗性給藥**，同時用 **AEIOU-TIPS** 系統性鑑別，並警覺**腦疝（herniation）**這個會要命的可逆急症。

> [!danger] 🚨 值班紅旗
> 1. **先救命再找病因**：昏迷病人**第一步是 ABC 不是診斷**——確保 Airway（GCS ≤8 考慮插管保護呼吸道）、Breathing（給 O₂、監測 SpO₂/血氣）、Circulation（IV、監測血壓/心律），穩住生命徵象再談鑑別。
> 2. **馬上驗血糖（fingerstick）**：低血糖是**最常見、最快可逆、漏掉會腦死**的昏迷原因，任何昏迷病人第一時間 POCT 血糖，低就補糖（不用等中央實驗室）。
> 3. **DONT 經驗性給藥**（血糖未知或懷疑時）：**D**extrose（50% glucose 50mL IV，**若懷疑酒癮/營養不良先或同時給 Thiamine 再給糖**，避免誘發 Wernicke）、**O**₂、**N**aloxone（0.4–2mg IV，opioid overdose ＋呼吸抑制/針尖瞳孔）、**T**hiamine（100mg IV）。
> 4. **警覺腦疝三兆**：**單側瞳孔放大固定（blown pupil）＋ 對側肢體無力 ＋ 意識急速惡化**；**Cushing triad（高血壓＋心跳慢＋呼吸不規則）** 是顱內壓危象。→ 立即抬高床頭 30°、過度換氣暫時降 CO₂、高張治療（mannitol／高張鹽水）、**急 neurosurgery**。
> 5. **不對稱神經學徵象 = 結構性昏迷警報**：瞳孔不等、眼位偏斜、單側肢體反應差、posturing 不對稱 → 傾向**結構性病灶（出血/梗塞/腫瘤/疝）**，**急做 non-contrast 頭部 CT**，別當單純代謝性拖延。
> 6. **別忘可逆的 mimic**：非驚厥性癲癇重積（NCSE，需 EEG）、CO 中毒、體溫過低/過高、腦膜炎/腦炎（發燒＋頸僵硬要打抗生素＋考慮 LP）、鎮靜藥殘留。

## 📖 定義 / 病生理（這是什麼）
- Caused by focal lesions in brainstem (reticular activating system), thalamus, or diffuse dysfunction of both cerebral hemispheres.
- **意識的兩大要素**：**Arousal（清醒／覺醒）** 由**腦幹上段的網狀活化系統（ARAS）＋視丘**維持；**Awareness（覺知／內容）** 由**雙側大腦皮質**產生。昏迷 = 兩者皆持續喪失、無法喚醒。
- **產生昏迷的三大機轉**（記「要嘛壞掉開關、要嘛兩邊燈都滅、要嘛毒素/代謝把整體關掉」）：
  | 機轉 | 解剖 | 典型病因 |
  | --- | --- | --- |
  | **上腦幹 ARAS 受損** | 中腦/橋腦被蓋、視丘 | 腦幹出血/梗塞、基底動脈栓塞、腦幹腫瘤 |
  | **雙側大腦半球廣泛受損** | 兩側皮質/白質 | 廣泛缺氧缺血、雙側大梗塞、瀰漫性腦水腫、腦炎 |
  | **佔位效應／腦疝壓迫腦幹** | 天幕裂孔/枕骨大孔疝 | 大量出血、大梗塞、腫瘤、膿瘍 → 壓迫 ARAS |
  | **代謝／中毒／全身性** | 瀰漫（生化層次） | 低血糖、缺氧、電解質、肝腎衰竭、藥物中毒、感染 |
- **結構性 vs 代謝性昏迷（值班關鍵二分法）**：
  | 特徵 | 結構性（structural） | 代謝性（metabolic） |
  | --- | --- | --- |
  | 神經學徵象 | **不對稱／局部**（focal） | **對稱、無局部徵象** |
  | 瞳孔 | 常不等大、光反射喪失 | **多為小瞳孔但光反射保留**（opioid 為針尖、抗膽鹼為散大） |
  | 腦幹反射 | 可早期喪失／不對稱 | **通常保留**（毒物/代謝性的標誌） |
  | 起病 | 常突發、進展快 | 常漸進、可波動 |
  | 異常動作 | 局部抽搐、posturing | asterixis、tremor、myoclonus、全身抽搐 |
  | 處置導向 | 急影像 → 常需外科 | 矯正生化／解毒，多為內科 |
  - **腦幹反射（角膜、瞳孔光反射、oculocephalic/oculovestibular、gag）保留** → 強烈支持代謝/中毒性；**不對稱或喪失** → 結構性病灶定位。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Altered Mental Status(意識改變)]]、[[Seizure(癲癇)]]
- **意識障礙分級（連續光譜，越後越深）**：
  - **Alert（清醒）** → **Lethargy／Somnolence（嗜睡，可喚醒、喚醒即回應但易再睡）** → **Obtundation（遲鈍，需較強刺激才醒、反應遲緩）** → **Stupor（木僵，僅強烈/疼痛刺激才有短暫喚醒）** → **Coma（昏迷，任何刺激皆無法喚醒、無有意義反應）**。
  - 臨床上少用模糊字眼，**直接用 GCS ＋描述對刺激的實際反應**（如「對疼痛僅有屈曲」）較無歧義。
- **昏迷病人的身體檢查重點（定位用）**：
  - **呼吸型態**：Cheyne–Stokes（雙側大腦/間腦）、central neurogenic hyperventilation（中腦/橋腦上段）、apneustic/ataxic（橋腦下段/延腦，瀕危）。
  - **瞳孔**：針尖（opioid／橋腦出血）、單側放大固定（第三腦神經受壓／鉤回疝）、中位固定（中腦）、雙側散大固定（缺氧腦病／嚴重）；**光反射保留與否是結構 vs 代謝的關鍵**。
  - **眼球運動**：oculocephalic（doll's eye，昏迷但無頸傷才做）、oculovestibular（冷水灌耳 caloric）測腦幹完整性；凝視偏斜（往病灶側 = 大腦病灶）。
  - **運動反應／posturing**：**去皮質（decorticate，屈曲）** 病灶較高（大腦/間腦）；**去大腦（decerebrate，伸直）** 病灶較低（中腦/上橋腦）、預後較差；無反應最差。
  - **腦膜刺激徵**（頸僵硬、Kernig/Brudzinski）→ 腦膜炎/SAH。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Blood Glucose(血糖)]]、[[Arterial Blood Gas(動脈血氣)]]、[[Ammonia(血氨)]]、[[Electrolytes(電解質)]]、[[EEG(腦波)]]　影像 [[Brain CT(腦部電腦斷層)]]、[[Brain MRI(腦部磁振造影)]]
- GCS 下降![[Pocket Medicine The Massachusetts General Hospital Handbook of Internal Medicine, 8th Edition.pdf#page=563&rect=77,291,536,439|Pocket Medicine The Massachusetts General Hospital Handbook of Internal Medicine, 8th Edition, p.563]]
- **AEIOU-TIPS 鑑別診斷助記**（昏迷病因系統性掃描，別漏可逆病因）：
  | 字母 | 病因 | 值班對應處置 |
  | --- | --- | --- |
  | **A** | **Alcohol** 酒精中毒／戒斷、**A**cidosis 酸中毒 | 酒精濃度、血氣、先給 Thiamine 再給糖 |
  | **E** | **E**pilepsy 癲癇（含 NCSE 發作後狀態）、**E**lectrolytes 電解質（Na/Ca/Mg）、**E**ndocrine 內分泌 | EEG、電解質、甲狀腺/腎上腺 |
  | **I** | **I**nsulin（**低血糖／DKA/HHS**） | **POCT 血糖第一優先** |
  | **O** | **O**piates 鴉片類、**O**verdose 其他藥物中毒、**O**₂ 缺氧/CO 中毒 | naloxone、毒物篩、O₂、carboxyHb |
  | **U** | **U**remia 尿毒、肝性腦病 | BUN/Cr、氨、肝功能 |
  | **T** | **T**rauma 外傷/顱內出血、**T**emperature 體溫過高/過低 | 頭部 CT、核心體溫 |
  | **I** | **I**nfection 感染（腦膜炎/腦炎/敗血症） | 血液培養、發燒/頸僵硬 → 抗生素 ± LP |
  | **P** | **P**sychiatric 精神（排除診斷）、**P**oisoning 中毒 | 神經學一致性、毒物 |
  | **S** | **S**troke 中風（尤其**腦幹/基底動脈**、雙側視丘）、**S**hock、**S**AH、**S**eizure | 急影像、血壓、CTA |
- **檢查順序（值班 workup）**：① **POCT 血糖 ＋ 生命徵象 ＋ SpO₂**（床邊立即）→ ② **抽血**：CBC、電解質（Na/Ca/Mg/PO₄）、BUN/Cr、肝功能、**血氨**、血糖、**動脈血氣**（酸鹼/CO₂/O₂/carboxyHb）、TSH/cortisol、乳酸、血中酒精/毒物、必要時凝血/氨水楊酸/paracetamol → ③ **12 導程 EKG**（心律/QTc/MI）→ ④ **non-contrast 頭部 CT**（凡不是明確低血糖/藥物且未清醒者，尤其有局部徵象或外傷）→ ⑤ **懷疑感染且無禁忌 → LP**（先影像排除佔位再 LP，抗生素勿因等 LP 而延誤）→ ⑥ **懷疑 NCSE 或病因不明 → EEG** → ⑦ 進一步 **MRI/MRA/CTA**（後循環中風、腦幹、剝離）。
- **GCS（Glasgow Coma Scale，3–15）**：**昏迷慣以 GCS ≤8 為插管閾值**；記錄要拆分 E/V/M（如 E2V3M4=9）——**M（運動）最具預後價值**。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **血糖永遠第一**：任何昏迷先 fingerstick，別讓可逆的低血糖變成不可逆腦傷。
- **Thiamine 要在 Dextrose 之前或同時給**（酒癮/營養不良/孕吐/減重術後）——單獨衝糖可誘發 Wernicke 腦病。
- **腦幹反射保留 ＋ 對稱 ＋ 無局部徵象 → 想代謝/中毒**；**任何不對稱（瞳孔、眼位、肢體、posturing）→ 想結構性 → 急 CT**。
- **針尖瞳孔**別只想 opioid：**橋腦出血**也是針尖瞳孔，兩者用其他徵象/影像區分。
- **昏迷 ＋ 眼位/瞳孔正常但四肢不動** → 小心 **locked-in syndrome**（橋腹側病灶，意識清醒卻無法動，只剩垂直眼動/眨眼）——別誤判為昏迷（見相關）。
- **NCSE 是「無抽搐的癲癇昏迷」**：不明原因昏迷、眼球細微抽動、瞳孔波動 → 打 EEG，別漏。
- **GCS 在插管/鎮靜/失語/眼腫時失真**，記得標註「T」（無法評 V）並輔以 FOUR score 或腦幹反射描述。
- **低體溫下瞳孔固定、反射消失不等於腦死**——**判定腦死前必須先回溫、排除鎮靜藥/代謝干擾**。
- **雙側視丘/中腦頂蓋（top of the basilar）梗塞**可只表現昏迷而無明顯肢體偏癱，易被當代謝性——後循環症狀要積極影像。

## 💊 治療（Acute / Chronic）
> 用藥 [[Dextrose(葡萄糖)]]、[[Naloxone(納洛酮)]]、[[Thiamine(維生素B1)]]、[[Mannitol(甘露醇)]]
> ⚠️ 劑量依院內 protocol 與病人狀況查證；經驗性給藥前盡量取血備檢
### 第一步：穩定生命徵象（ABC，先於診斷）
- **Airway**：保護呼吸道，**GCS ≤8 或無法保護 airway → 考慮氣管插管**；頸椎未清除前 in-line 固定。
- **Breathing**：給 O₂ 維持 SpO₂，監測血氣、避免高/低 CO₂（懷疑高顱壓時勿讓 CO₂ 過高）。
- **Circulation**：建立 IV、監測血壓/心律，處理休克/心律不整；抽血同時送檢。
### 第二步：經驗性「DONT」給藥（血糖未知或病因不明時）
- **D**extrose：低血糖 → 50% glucose 50mL IV（成人）；**懷疑酒癮/營養不良先或同時給 Thiamine**。
- **O**xygen：矯正低血氧；懷疑 CO 中毒給高濃度/高壓氧。
- **N**aloxone：opioid overdose（呼吸抑制＋針尖瞳孔）0.4–2mg IV，半衰期短**可能需重複或滴注**。
- **T**hiamine：100mg IV，防 Wernicke。
- （**Flumazenil 一般不常規給** benzodiazepine 逆轉——恐誘發癲癇/戒斷，慎用。）
### 第三步：針對病因治療（依 AEIOU-TIPS 結果）
- 感染 → 抗生素/抗病毒（腦膜炎/腦炎勿延誤，疑 HSV 給 acyclovir）；電解質/內分泌 → 矯正；尿毒 → 透析；肝性腦病 → lactulose；中毒 → 解毒劑；**缺血性中風/基底動脈栓塞 → 依時間窗評估再灌流**；癲癇/NCSE → 抗癲癇藥。
### 腦疝／高顱壓危象處置
- 抬高床頭 30°、頭正中、鎮靜止痛；**短暫過度換氣**（暫時性降 CO₂ 救急，非長期）；**高張治療 mannitol 或高張鹽水**；控制發燒與癲癇；**急照 CT ＋ neurosurgery 會診**（外科減壓/引流可救命）。

## ⚠️ 併發症
- **腦疝（herniation）→ 腦幹壓迫 → 呼吸循環衰竭、死亡**（最急，可逆時間窗短）
- **缺氧性腦病**（airway 失守/延遲插管 → 二次腦傷）
- **吸入性肺炎**（嘔吐＋呼吸道保護喪失）
- 深部靜脈栓塞/肺栓塞、壓瘡、關節攣縮、感染（長期臥床）
- 誤診／延遲診斷可逆病因（低血糖、NCSE、中毒）→ **本可逆卻變不可逆腦傷**
- 進展為**植物人狀態／最小意識狀態、腦死**


## 📊 嚴重度評分 / 風險分層（scoring）
> 用哪個 score 判斷 severity / 預後 / treatment 強度
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **GCS（Glasgow Coma Scale）** | 意識障礙嚴重度／插管決策／追蹤趨勢 | **E**ye(1–4)＋**V**erbal(1–5)＋**M**otor(1–6)，總 **3–15**；**≤8 = 重度（考慮插管保護 airway）**、9–12 中度、13–15 輕度；記錄拆分 E/V/M，**M 最具預後價值**；插管記「V=1T」 |
| **FOUR score** | 昏迷/插管病人評估（含腦幹反射與呼吸，補 GCS 無法測 V 的缺口） | 4 項（Eye／Motor／Brainstem／Respiration）各 0–4，總 **0–16**；含瞳孔/角膜反射與呼吸型態，**插管病人比 GCS 更完整**，可辨識 locked-in |
| **瞳孔＋腦幹反射** | 結構 vs 代謝定位、預後 | 光反射喪失/不對稱 → 結構性；缺氧後雙側瞳孔固定 → 預後差 |
| **GCS-P（GCS + Pupil）** | 重症/創傷腦傷預後 | GCS 減去無反應瞳孔數（0/1/2），**1–15**，低分預後越差 |
