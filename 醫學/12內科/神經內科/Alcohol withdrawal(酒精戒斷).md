# Alcohol withdrawal（酒精戒斷）

> [!abstract] 一句話：長期飲酒者**突然減量/停酒後 GABA 抑制驟撤、麩胺酸興奮過亢**，沿時間軸出現顫抖→戒斷癲癇→酒精性幻覺→震顫性譫妄（DT）；核心處置是 **benzodiazepine（症狀導向、以 CIWA-Ar 給藥）＋ 先給 thiamine 再給糖**，臨床提醒：**DT 是會致死的內科急症（未治療死亡率可達雙位數），值班遇到躁動戒斷病人先想「上一次喝酒是什麼時候」**。

> [!danger] 🚨 值班紅旗
> 1. **Thiamine 一定要在葡萄糖之前給**：先打糖會誘發／惡化 [[Wernicke encephalopathy(魏尼凱氏腦病變)]]（眼動異常＋步態失調＋意識混亂三聯）；酒癮病人來就先 thiamine IV/IM。
> 2. **震顫性譫妄（DT）＝ 急症**：發生在停酒後約 **48–96 小時**，表現高燒、大汗、心搏過速、高血壓、嚴重躁動＋鮮明幻覺，**未治療死亡率高**，需重症監護與積極 benzodiazepine 滴定。
> 3. **戒斷癲癇（rum fits）**：多在停酒後 **6–48 小時**、常為全身性強直陣攣、多為單次或短叢發；但**別把所有抽搐都歸給酒精**——首發、局灶性、頭傷、反覆發作要查 CT 排除 [[Hemorrhagic Stroke(出血性中風)]]／硬膜下血腫，並排除低血糖、低血鈉、感染。
> 4. **難治型戒斷（benzo 反應差）** → 想 **kindling 現象**（反覆戒斷史使發作更兇）、共病感染、電解質崩壞；可能需 phenobarbital 或加輔助藥，並升級照護。
> 5. **躁動的酒癮病人不是只有戒斷**：務必同時排除頭部外傷、感染/敗血症、肝性腦病變、低血糖、電解質異常（低血鎂/低血鉀/低血磷）——這些會被「戒斷」標籤掩蓋。

## 📖 定義 / 病生理（這是什麼）
- **定義**：對酒精生理依賴者於停酒或明顯減量後，因中樞神經興奮-抑制失衡而出現的一組臨床症候群，嚴重度從輕微顫抖到致命 DT。
- **病生理**：慢性酒精 → **增強 GABA-A 抑制、抑制 NMDA（麩胺酸）興奮**，腦部代償性下調 GABA 受體、上調 NMDA 受體。**驟停 → 抑制驟撤 + 興奮過亢** → 交感風暴與神經興奮（顫抖、抽搐、譫妄）。
- **Kindling 現象**：每次戒斷都使神經敏化，**反覆戒斷者下一次更容易癲癇與 DT**。
- **時間軸（記憶關鍵）**：
  | 時間（末次飲酒後） | 表現 |
  | --- | --- |
  | **6–12 小時** | 輕度：顫抖、焦慮、噁心、失眠、心悸、出汗、高血壓 |
  | **12–24 小時** | **酒精性幻覺（alcoholic hallucinosis）**：意識清楚下的幻視/幻聽（與 DT 不同，感官清醒、生命徵象較穩） |
  | **12–48 小時** | **戒斷癲癇**（全身強直陣攣，多短暫） |
  | **48–96 小時** | **震顫性譫妄（DT）**：意識混亂＋自律神經風暴＋幻覺，死亡率高 |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Seizure(癲癇)]]、[[Delirium(譫妄)]]、[[Tremor(顫抖)]]
- **自律神經亢進**：心搏過速、高血壓、發燒、大汗、瞳孔擴大。
- **神經精神**：焦慮、躁動、失眠、顫抖（手部粗大震顫）、幻覺（幻視「看到蟲/小動物」典型）、定向力障礙。
- **戒斷癲癇**：全身性強直陣攣，多發生於早期、常單次。
- **DT**：以上全面惡化 + 深度譫妄與意識波動，屬 [[Delirium(譫妄)]] 的一種病因。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 電解質(Na/K/Mg/PO4)、血糖、肝功能、氨、血中酒精濃度、CBC　影像 [[Brain CT(腦部電腦斷層)]]（新發癲癇/頭傷/局灶徵象時）
- **臨床診斷**：明確飲酒史 + 停酒時間軸 + 典型症狀，並用 CIWA-Ar 量化。
- **必查實驗室**：血糖（排低血糖）、電解質（低鎂/低鉀/低磷常見且會誘發抽搐/心律不整）、肝功能與氨（肝性腦病變鑑別）、CBC、血中酒精濃度（仍高卻已戒斷提示高耐受）。
- **影像/EEG**：新發或非典型癲癇、局灶神經學缺損、頭部外傷、意識與抽搐不符病程時 → CT 排除出血/硬膜下血腫；反覆抽搐可考慮 EEG。
- **DDx 一定要排除**：低血糖、[[Hepatic encephalopathy(肝性腦病變)]]、感染/敗血症、中樞感染、頭部外傷、其他中毒/戒斷、[[Wernicke encephalopathy(魏尼凱氏腦病變)]]。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **Thiamine 先於 glucose**——最常考也最常在忙亂中做反的一步。
- **酒精性幻覺 vs DT**：幻覺者**感官清醒、生命徵象相對穩**；DT 有**意識混亂 + 自律神經風暴**，兩者別混。
- **戒斷癲癇通常自限、單次**——若是叢發、局灶或持續，回頭找別的病因。
- **症狀導向（symptom-triggered）給藥優於固定劑量**：以 CIWA-Ar 驅動 benzodiazepine，總量更少、住院更短（前提是能可靠評分且非重症）。
- 別忘補 **鎂、鉀、磷、葉酸**；低鎂會讓抽搐與心律不整難控。
- 有 kindling 史者門檻放低、及早積極治療。

## 💊 治療（Acute / Chronic）
> 用藥 [[Benzodiazepine(苯二氮平類)]]（Diazepam/Lorazepam/Chlordiazepoxide）、[[Thiamine(維生素B1)]]、[[Phenobarbital(苯巴比妥)]]
- **Benzodiazepine（第一線基石）**：重建 GABA 抑制、抗癲癇、抗自律神經亢進。
  - **症狀導向給藥**：依 **CIWA-Ar** 分數觸發劑量（一般 ≥ 8–10 起始治療），滴定至鎮靜且生命徵象穩定。
  - 藥物選擇：長效（diazepam、chlordiazepoxide）自我漸退平順；**肝功能差/老年/嚴重肝病選短效無活性代謝物的 lorazepam**（經葡萄糖醛酸化，不經 CYP 氧化）。
  - **具體單次與累積劑量依 protocol/CIWA 反應與藥典滴定**（⚠️需查證，勿憑記憶下固定 mg）。
- **Thiamine**：**在任何含糖輸液之前**給予，預防/治療 Wernicke；劑量與途徑依院內 protocol（⚠️需查證）。
- **輔助**：補鎂/鉀/磷/葉酸、水分電解質矯正、環境安靜、監測生命徵象。
- **難治型/大量需求**：benzodiazepine 反應不佳 → 加/改 **phenobarbital**，或加輔助（如 dexmedetomidine 控自律神經、propofol 於插管重症）；此類屬重症處置，依專科與 protocol（⚠️需查證）。
- **DT**：ICU 監護、積極 benzodiazepine 滴定、支持治療。
- **長期（防復發）**：戒酒衛教與轉介、naltrexone/acamprosate/disulfiram 等戒酒藥物（門診處理，非急性戒斷用藥）。

## ⚠️ 併發症
- **震顫性譫妄（DT）** 與其自律神經風暴 → 心律不整、循環衰竭、死亡
- **戒斷癲癇**、癲癇重積
- **Wernicke-Korsakoff 症候群**（thiamine 缺乏）
- 吸入性肺炎、橫紋肌溶解、電解質崩壞
- 外傷（跌倒/抽搐）、共病肝病惡化、脫水與腎損傷

## 📊 嚴重度評分 / 風險分層（scoring）
> 用 **CIWA-Ar** 量化戒斷嚴重度並驅動給藥；用 PAWSS 預測風險
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **CIWA-Ar**（Clinical Institute Withdrawal Assessment for Alcohol, revised；10 項：噁心嘔吐、顫抖、盜汗、焦慮、躁動、觸/聽/視幻覺、頭痛、定向力） | 量化嚴重度 + 症狀導向給藥 | **< 8–10 輕度**（多可支持治療）；**≥ 8–10** 起始/加 benzodiazepine；**≥ 15–20** 中重度、密切監測 |
| **PAWSS**（Prediction of Alcohol Withdrawal Severity Scale） | 入院早期預測是否會發展成複雜/重度戒斷 | 分數高 → 提前積極預防性治療 |
| Glasgow Modified Alcohol Withdrawal Scale（GMAWS） | 部分院所替代 CIWA | 較精簡 |
