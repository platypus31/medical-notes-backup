# Hydrocephalus（水腦症）

> [!abstract] 一句話：**腦脊髓液（CSF）在腦室系統內過度堆積**，可因阻塞（obstructive/non-communicating）、吸收障礙（communicating）或生成過多；核心是分辨**急性顱內壓升高（會腦疝、致命）vs 慢性 NPH（步態＋認知＋尿失禁三聯徵，可逆但常被誤診失智）**——值班最怕漏掉「shunt 病人急性惡化＝shunt malfunction」。

> [!danger] 🚨 值班紅旗
> 1. **急性水腦 ＋ 頭痛／嘔吐／意識下降／瞳孔變化** → 急性顱內壓升高，可快速**腦疝致命**，緊急影像＋神經外科會診（可能需**緊急腦室外引流 EVD**）
> 2. **已裝 VP/VA shunt 的病人突然頭痛、嗜睡、嘔吐、意識改變** → **shunt malfunction（阻塞或感染）直到排除**，照 shunt series ＋ CT 比對舊片，神外會診，別歸因「一般腸胃炎」
> 3. **發燒 ＋ shunt ＋ 意識改變／腹痛** → **shunt infection / 腦室炎**，抽 CSF（經 shunt reservoir，神外執行）＋血培養
> 4. **CT 見腦室擴大 ＋ 快速惡化的昏迷** → 可能急性阻塞（腫瘤、出血、IVH 阻塞導水管），時間就是腦
> 5. **後顱窩病灶／小腦出血 ＋ 第四腦室受壓** → 阻塞性水腦 + 腦幹壓迫雙重致命，緊急處置

## 📖 定義 / 病生理（這是什麼）
- **定義**：CSF 於腦室系統累積導致**腦室擴大**，多伴顱內壓升高（NPH 例外）。
- **CSF 生理**：脈絡叢生成（~500 mL/day，總量 ~150 mL），經腦室 → 第四腦室出口 → 蜘蛛膜下腔 → **蜘蛛膜顆粒吸收**。任一環節失衡即可積水。
- **分類（決定處置）**：
  | 類型 | 機轉 | 常見病因 |
  | --- | --- | --- |
  | **阻塞性（non-communicating）** | 腦室系統**內部**阻塞（CSF 流不出腦室） | 導水管狹窄、後顱窩腫瘤、松果體區腫瘤、IVH、Chiari |
  | **交通性（communicating）** | 腦室外吸收／流動障礙（腦室間通暢） | SAH 後、腦膜炎後、腦膜癌病、**NPH** |
  | **ex vacuo（假性）** | 腦萎縮致腦室代償性擴大，**非真正積水**、壓力正常 | 失智、退化——**不是真 hydrocephalus，別做分流** |
- **NPH（Normal Pressure Hydrocephalus）**：腦室擴大但腰椎穿刺開放壓「正常上限」，慢性交通性，特徵三聯徵，是**可逆性失智**的重要鑑別。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Headache(頭痛)]]、[[Altered mental status(意識改變)]]、[[Gait disturbance(步態障礙)]]
- **急性／顱內壓升高**：頭痛（晨起、平躺加重）、噁心嘔吐、**視乳頭水腫（papilledema）**、複視（CN VI 假性定位）、意識下降、**Cushing 反應**（高血壓 + 心搏過緩 + 呼吸不規則，晚期腦疝徵象）。
- **兒童（囟門未閉）**：**頭圍快速增大**、囟門膨隆、頭皮靜脈怒張、**落日眼（sunset eyes，向上凝視麻痺）**、餵食困難、嗜睡。
- **NPH 經典三聯徵（Hakim triad，「Wet, Wobbly, Wacky」）**：
  | 症狀 | 描述 | 順序 |
  | --- | --- | --- |
  | **步態障礙（wobbly）** | 磁吸步態（magnetic gait）、寬基底、起步困難 | 最早、最可逆 |
  | **認知障礙（wacky）** | 皮質下失智、動作變慢、注意力差 | 中 |
  | **尿失禁（wet）** | 急迫性 → 失控 | 最晚 |

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Lumbar puncture(腰椎穿刺)]]（NPH tap test；急性阻塞禁做）　影像 [[CT(電腦斷層)]]、[[MRI(磁振造影)]]
- **影像（第一線）**：
  - **CT**：急診首選，看腦室大小、**顳角擴大**、periventricular 低密度（transependymal CSF flow）、找阻塞病灶（腫瘤、出血）。
  - **MRI**：分辨阻塞位置（導水管狹窄）、後顱窩病灶；NPH 特徵 **DESH（disproportionately enlarged subarachnoid space hydrocephalus）**、callosal angle 變窄、**Evans index > 0.3**（腦室額角最大寬 / 顱內最大寬）。
- **急性阻塞性水腦**：**腰椎穿刺禁忌**（會誘發腦疝）——先影像、先神外。
- **NPH 評估**：**大量放液測試（large-volume LP tap test，放 30–50 mL CSF）**後看步態／認知是否改善，可預測分流反應；或連續腰椎引流試驗。
- **開放壓**：阻塞／急性者高；NPH「正常」（多在正常上限，故名）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **ex vacuo（腦萎縮）不是真水腦**——腦室擴大但溝回也擴大、壓力正常，**分流無效**，別搞混。
- **急性阻塞性水腦禁做 LP**（腦疝風險）；NPH 才靠 tap test。
- **shunt 病人任何新神經惡化 → 先想 shunt malfunction / infection**，比對舊 CT 看腦室有無再擴大。
- **NPH 三聯徵中「步態」最早出現、最可能因分流改善**；等到失智才處理常已不可逆。
- **兒童頭圍曲線快速上升**是水腦重要線索（囟門未閉可代償一段時間）。
- 假性定位徵象：**CN VI 麻痺（複視）**可因顱內壓升高，非真正病灶定位。

## 💊 治療（Acute / Chronic）
> 用藥 [[Acetazolamide(乙醯偶氮胺)]]（暫時性，非根治）
- **急性顱內壓升高 / 阻塞**（救命）：
  - **緊急 CSF 引流**：**腦室外引流（EVD / external ventricular drain）**或緊急分流——**神經外科執行**。
  - 顱內壓處理原則（床頭抬高、避免低血壓低氧、必要時滲透治療）——依神外／神經重症 protocol（⚠️藥物與劑量需查證）。
- **根治性分流手術**：
  - **VP shunt（腦室腹腔分流）**最常見；VA（心房）、LP（腰腹）視情況。
  - **內視鏡第三腦室造廔術（ETV）**：適用於**阻塞性**水腦（如導水管狹窄），避開植入異物。
- **NPH**：對 tap test 反應好者 → **VP shunt（可調壓閥）**，步態改善率最高。
- **暫時性減少 CSF 生成**：**acetazolamide** 可短期輔助（⚠️劑量需查證，非根治，不用於急性阻塞代替引流）。

## ⚠️ 併發症
- **未治療急性水腦** → 腦疝、死亡。
- **shunt 併發症**：阻塞（最常）、感染／腦室炎、過度引流（**subdural hematoma、slit ventricle syndrome**）、機械故障、腹部假性囊腫。
- NPH 延遲治療 → 認知不可逆。
- 兒童：發展遲緩、視力損害。

## 📊 嚴重度評分 / 風險分層（scoring）
> 無單一通用 score；依臨床＋影像＋病因分層
| 工具 | 用途 | 意義 |
| --- | --- | --- |
| **Evans index（> 0.3）** | 影像判腦室擴大 | 額角最大寬 / 顱內最大寬 |
| **NPH tap test 反應** | 預測分流成效 | 放液後步態／認知改善 → 分流反應佳 |
| **GCS / 瞳孔** | 急性惡化分層 | 意識下降＋瞳孔變化 → 腦疝，緊急 |
| **DESH（MRI）** | NPH 影像特徵 | 支持 NPH 分流候選 |
