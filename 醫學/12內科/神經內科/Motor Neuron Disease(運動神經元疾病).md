# Motor Neuron Disease（運動神經元疾病）

> [!abstract] 一句話：**上下運動神經元同時進行性退化**的疾病（最常型＝ALS），造成無力、肌肉萎縮、束顫與痙攣，但**感覺、眼動、括約肌、認知（多數）相對保留**；無根治，值班最致命的是**呼吸肌衰竭與吞嚥障礙導致的呼吸危象／吸入性肺炎**。

> [!danger] 🚨 值班紅旗
> 1. **進行性呼吸困難、端坐呼吸、晨起頭痛、FVC/NIF 下降** → **神經肌肉性呼吸衰竭**，是 MND 主要死因；監測 FVC/NIF，惡化考慮 NIV，**別等到 CO₂ 昏迷**
> 2. **球麻痺（bulbar）：構音不清、嗆咳、流涎、吞嚥困難** → 高吸入性肺炎風險，禁經口進食評估、抬高床頭、吞嚥團隊
> 3. **急性呼吸窘迫的 MND 病人** → 先確認**病人事先的維生意願（advance directive）/ 是否插管**，這是需與病人家屬討論的關鍵決策
> 4. **痰液清除困難＋咳嗽無力** → 黏液栓塞可致急性窒息，抽痰/咳嗽輔助
> 5. 用藥謹慎：對呼吸儲備差的病人慎用鎮靜/嗎啡類壓抑呼吸驅動（安寧情境例外，依 protocol）

## 📖 定義 / 病生理（這是什麼）
- **定義**：一組選擇性侵犯運動神經元的退化性疾病。**ALS（Amyotrophic Lateral Sclerosis）**是最常見型，同時侵犯**上運動神經元（UMN，皮質脊髓束）＋下運動神經元（LMN，前角細胞/腦幹運動核）**。
- **病生理**：多因（麩胺酸興奮毒性、氧化壓力、蛋白錯誤折疊 TDP-43、粒線體功能異常、神經發炎）；~10% 家族性（**SOD1、C9orf72**（也連結額顳葉失智）、TDP-43/FUS 等）。
- **關鍵「保留」**：**感覺系統、眼外肌（眼動）、膀胱/腸括約肌**通常到晚期才受影響——這是與其他病鑑別的線索。
- **臨床亞型（依受累比例）**：
  | 亞型 | UMN | LMN | 特徵 |
  | --- | --- | --- | --- |
  | **ALS（典型）** | ✓ | ✓ | 混合 UMN+LMN，最常 |
  | **Progressive Bulbar Palsy** | ± | ✓ | 以球麻痺起病，預後差 |
  | **Primary Lateral Sclerosis (PLS)** | ✓ | — | 純 UMN、進展慢 |
  | **Progressive Muscular Atrophy (PMA)** | — | ✓ | 純 LMN |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Muscle weakness(肌肉無力)]]、[[Dysphagia(吞嚥困難)]]、[[Fasciculation(肌束顫動)]]
- **混合 UMN + LMN 徵象並存於同一區域是特徵**：
  | | UMN 徵象 | LMN 徵象 |
  | --- | --- | --- |
  | 肌張力 | 痙攣（spasticity） | 鬆弛 |
  | 反射 | 亢進、Babinski (+) | 減弱/消失 |
  | 肌肉 | — | **萎縮、束顫（fasciculation）** |
  | 球部 | 痙攣性構音、下顎反射亢進、假性延髓情緒 | 舌萎縮/束顫、流涎、嗆咳 |
- **起病**：常局部（手部無力/笨拙、垂足、或球麻痺）→ 逐漸擴散至鄰近區域。
- **保留**：感覺正常、眼動正常、括約肌控制多保留、多數認知正常（**約 15% 合併額顳葉失智 FTD**）。
- **晚期**：呼吸肌無力（呼吸困難、端坐呼吸）、體重減輕、溝通困難。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[EMG(肌電圖)]]、[[Nerve conduction study(神經傳導檢查)]]、[[Pulmonary function test(肺功能)]]（FVC）　影像 [[MRI(磁振造影)]]（排除脊髓/腦幹壓迫）
- **臨床＋電生理診斷（排除法）**：MND 無單一確診檢查，靠**進行性 UMN+LMN 徵象擴散＋排除其他病**。
- **EMG/NCS**：關鍵——顯示**廣泛的去神經與再神經支配**（fibrillation、positive sharp waves、fasciculation、大運動單位電位），**多節段（腦幹/頸/胸/腰薦）**受累，感覺傳導正常。
- **MRI 頸椎/腦**：**排除可治療的模仿病**——頸椎病變壓迫、脊髓空洞、腦幹病灶、多節神經根病。
- **血液**：排除甲狀腺/副甲狀腺、鉛中毒、HIV、抗 GM1（multifocal motor neuropathy 可治療）、CK 輕度↑。
- **診斷準則**：**El Escorial / Awaji criteria**（依 UMN+LMN 受累區域數分 possible/probable/definite）。
- **呼吸評估（追蹤與處置核心）**：**FVC（用力肺活量）、NIF（吸氣負壓）、SNIP、夜間血氧/CO₂**。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **UMN + LMN 徵象同時出現、感覺正常、無疼痛** → MND 的診斷指紋。
- **一定要排除可治療的模仿病**：頸椎脊髓病（有感覺症狀/膀胱）、multifocal motor neuropathy with conduction block（可治療！查抗 GM1）、重症肌無力（疲勞性、眼睛）、甲狀腺/副甲狀腺、鉛。
- **眼動保留、括約肌保留**——若早期就有眼肌麻痺或大小便失禁，要懷疑「不是 ALS」。
- **呼吸功能是預後與處置關鍵**——追 FVC/NIF，別只看 SpO₂（神經肌肉衰竭早期 SpO₂ 常還正常，先 CO₂ 上升）。
- **球麻痺型吞嚥困難**：積極營養（PEG 造廔時機要趁 FVC 還好時）與防吸入。
- **假性延髓情緒（不自主哭笑）**常見，可治療。

## 💊 治療（Acute / Chronic）
> 用藥 [[Riluzole(利魯唑)]]、[[Edaravone(依達拉奉)]]
### 疾病修飾（延緩，非治癒）
- **Riluzole**（抗麩胺酸興奮毒性）——延長存活數個月，需監測肝功能（⚠️劑量依藥典查證）。
- **Edaravone**（抗氧化，靜脈/口服）——部分病人減緩功能下降。
- 新藥（如 tofersen 用於 SOD1 型）依基因型與最新核准。
### 症狀與支持（照護核心，多團隊）
- **呼吸**：追 FVC/NIF；**非侵襲性通氣（NIV/BiPAP）**改善存活與生活品質；咳嗽輔助/抽痰清痰；末期呼吸器/氣切決策依病人意願。
- **營養/吞嚥**：吞嚥評估、增稠飲食，適時 **PEG 胃造廔**（趁呼吸功能尚可時放置）。
- **症狀**：痙攣（baclofen）、流涎（抗膽鹼/肉毒）、假性延髓情緒（dextromethorphan-quinidine）、疼痛、抽筋、憂鬱——⚠️劑量查證。
- **溝通輔助**、復健、**安寧與預立醫療照護諮商（ACP）**——早期即討論維生意願。

## ⚠️ 併發症
- **呼吸衰竭**（主要死因）、**吸入性肺炎**
- 營養不良、脫水
- 溝通喪失、失能、褥瘡、DVT
- 假性延髓情緒、憂鬱、焦慮
- 合併額顳葉失智（部分）

## 📊 嚴重度評分 / 風險分層（scoring）
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **ALSFRS-R（0–48）** | 功能追蹤（球部/精細/粗大/呼吸 12 項） | 分數越低失能越重；下降速率＝預後指標 |
| **FVC（% predicted）** | 呼吸儲備 | 進行性下降 → NIV/PEG 時機、預後 |
| **El Escorial / Awaji** | 診斷確定度 | possible / probable / definite（UMN+LMN 區域數） |
| **NIF / SNIP** | 吸氣肌力 | 惡化 → 呼吸支持 |
