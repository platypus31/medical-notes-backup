# Peripheral Neuropathy（周邊神經病變）

> [!abstract] 一句話：**周邊神經（運動／感覺／自律）受損**的一大類疾病，最常見型態是**糖尿病造成的遠端對稱性感覺運動多發性神經病變（distal symmetric polyneuropathy）**；診斷靠**病史＋神經學檢查＋神經傳導檢查（NCS/EMG）**分辨 axonal vs demyelinating，值班最不能漏的是**快速上行無力的 [[Guillain-Barré Syndrome(格林巴利症候群)]]**（會呼吸衰竭）。

> [!danger] 🚨 值班紅旗
> 1. **急性／亞急性上行性對稱無力 ＋ 深部肌腱反射消失** → 懷疑 **GBS**，馬上驗 **呼吸功能（FVC、NIF/MIP）**、監測自律神經不穩（心律、血壓大起大落）；FVC 下降或延髓無力要 ICU、備插管，別等到喘才處理。
> 2. **鞍區麻木（saddle anesthesia）＋ 大小便失禁 ＋ 雙下肢無力** → **[[Cauda Equina Syndrome(馬尾症候群)]]**，是外科急症，緊急 MRI ＋ 神經外科會診。
> 3. **糖尿病足感覺喪失** → 無痛性潰瘍、感染、Charcot 足；每次值班經過都要看腳，感染可快速變 [[Necrotizing Fasciitis(壞死性筋膜炎)]] / 敗血症。
> 4. **B12 缺乏＋亞急性合併變性（SCD）**：麻木＋本體感覺喪失＋反射變化，**先驗 B12 再補葉酸**（單補葉酸會惡化神經病變）。
> 5. **不對稱、多條神經、快速進展（mononeuritis multiplex）** → 想血管炎、感染（HIV、hansen）、糖尿病，可能需要類固醇/免疫治療，別當單純周邊神經病拖延。

## 📖 定義 / 病生理（這是什麼）
- **定義**：周邊神經系統（脊神經根、神經叢、周邊神經）功能或結構受損，可影響**感覺、運動、自律**纖維。
- **病生理分型（決定檢查與預後）**：
  - **軸突性（axonal）**：神經細胞軸突退化，最常見（糖尿病、毒素、代謝、營養）；長度依賴 → **遠端起病、襪套-手套分布**。
  - **脫髓鞘性（demyelinating）**：髓鞘受損（GBS、CIDP、CMT1、某些遺傳性）；NCS 傳導速度明顯變慢、傳導阻斷。
- **依分布分類**：
  | 型態 | 分布 | 常見成因 |
  | --- | --- | --- |
  | 遠端對稱多發（DSP） | 襪套-手套，下肢先 | 糖尿病、酒精、B12、尿毒、化療、amyloid |
  | 單神經病變（mononeuropathy） | 單一神經 | 壓迫（[[Carpal Tunnel Syndrome(腕隧道症候群)]]）、外傷 |
  | 多發性單神經病變（mononeuritis multiplex） | 多條、不對稱 | 血管炎、糖尿病、感染、癌 |
  | 多發性神經根神經病變 | 近端＋遠端 | GBS、CIDP |
- **常見病因記憶（DANG THERAPIST）**：Diabetes、Alcohol、Nutritional（B12/B1/B6）、GBS、Toxins/drugs、Hereditary、Environmental、Rheumatologic（血管炎）、Amyloid、Paraneoplastic/Paraprotein、Infection（HIV、hansen、lyme）、Systemic（尿毒、甲狀腺低下）、Tumor。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Numbness(麻木)]]、[[Weakness(無力)]]
- **感覺**：麻木、刺痛、灼熱、電擊感、異感（paresthesia）、痛覺過敏（allodynia）；本體感覺喪失 → 步態不穩、Romberg 陽性。
- **運動**：無力、肌肉萎縮、抽筋、垂足（foot drop）。
- **自律**：姿勢性低血壓、便秘/腹瀉、排尿障礙、勃起功能障礙、發汗異常、靜止心搏過速（糖尿病常見）。
- **反射**：深部肌腱反射**減弱或消失**（尤其踝反射），是重要客觀徵象。
- **分布線索**：長度依賴（襪套-手套、下肢先於上肢）支持代謝/毒素性；近端＋遠端、快速對稱無力支持 GBS。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Nerve Conduction Study(神經傳導檢查)]]、[[Electromyography(肌電圖)]]　影像 [[Spine MRI(脊椎磁振造影)]]
- **神經學檢查**：確認分布（對稱/不對稱、遠端/近端）、纖維類型（感覺/運動/自律）、反射。
- **NCS/EMG**：核心工具，分辨 **axonal（波幅↓）vs demyelinating（傳導速度↓、傳導阻斷）**、確認範圍與嚴重度。
- **血液常規篩查**（DSP 找可逆病因）：**空腹血糖/HbA1c 或 OGTT、B12（±MMA/homocysteine）、TSH、腎功能、肝功能、SPEP/免疫固定電泳（找 paraprotein）**、ESR/ANA（血管炎線索）。
- **進階**：懷疑 GBS/CIDP 做 **腰椎穿刺**（GBS 典型 **albuminocytologic dissociation**：蛋白↑、細胞正常）；不明原因或不對稱可做神經切片、基因（遺傳性）。
- **影像**：懷疑神經根病變/馬尾/壓迫做 **spine MRI**。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **反射消失 ＋ 上行無力 ＋ CSF 蛋白細胞分離** = GBS，優先於一切 workup 保護呼吸道。
- **B12 缺乏**：麻木＋本體感覺／振動覺喪失＋可能反射亢進（合併脊髓側索）＝ SCD；**驗 B12，別只補葉酸**。
- **糖尿病病人任何足部傷口都要看**，無痛性潰瘍是感覺喪失的警訊。
- **不對稱 = 想結構性/血管炎/壓迫**，對稱長度依賴 = 想代謝/毒素。
- **藥物性**：化療（vincristine、platinum、taxane）、isoniazid（→ 補 B6）、metronidazole、amiodarone、nitrofurantoin。
- **神經病理性疼痛的藥不是普通止痛藥**：NSAID/opioid 效果差，要用 gabapentinoid / TCA / SNRI。

## 💊 治療（Acute / Chronic）
> 用藥 [[Gabapentin(加巴噴丁)]]、[[Pregabalin(普瑞巴林)]]
> ⚠️ 劑量依藥典/指引滴定，勿憑記憶下硬數字
### 治本（治療根本病因，最重要）
- 糖尿病 → **血糖控制**；酒精 → 戒酒＋補 thiamine；B12 缺乏 → 補 B12；尿毒 → 透析；甲狀腺低下 → 補甲狀腺素；停用致病藥物；血管炎/CIDP/GBS → 免疫治療。
- **GBS**：**IVIG 或 血漿置換（plasma exchange）**（兩者效力相當，不合併使用），加上呼吸與自律神經監測支持；類固醇單用無效。
- **CIDP**：類固醇、IVIG、血漿置換。
### 治標（神經病理性疼痛）
- 第一線：**gabapentin/pregabalin**、**TCA（amitriptyline/nortriptyline）**、**SNRI（duloxetine）**；局部 lidocaine/capsaicin 貼片。劑量**⚠️需查證，依藥典滴定**。
### 支持
- 足部照護與衛教、跌倒預防、復健/物理治療、輔具（垂足用踝足支架 AFO）。

## ⚠️ 併發症
- **呼吸衰竭**（GBS 延髓/呼吸肌無力）
- **自律神經不穩**（GBS：心律不整、血壓劇烈波動、猝死）
- **糖尿病足潰瘍 → 感染／截肢、Charcot 關節病變**
- 跌倒與骨折（本體感覺喪失、垂足）
- 神經病理性慢性疼痛、憂鬱、睡眠障礙
- 姿勢性低血壓昏厥（自律神經病變）

## 📊 嚴重度評分 / 風險分層（scoring）
> 依情境選量表；DSP 無單一 acute score
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **Erasmus GBS Respiratory Insufficiency Score (EGRIS)** | GBS 預測需插管 | 依起病到入院天數、面部/延髓無力、MRC sum score；分數高 → 高機率需機械通氣 |
| **MRC sum score** | 肌力量化（GBS 追蹤） | 6 組肌群雙側 0–5，0–60 分，越低越無力 |
| **Toronto Clinical Neuropathy Score / MNSI** | 糖尿病周邊神經病變篩檢/嚴重度 | 分數越高神經病變越重 |
| **CSF 蛋白細胞分離** | GBS 支持診斷 | 蛋白↑、白血球正常（albuminocytologic dissociation） |
