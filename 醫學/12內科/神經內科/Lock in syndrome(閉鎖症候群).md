# Lock in syndrome（閉鎖症候群）

> [!abstract] 一句話：**橋腦腹側（ventral pons）病灶**（最常為基底動脈栓塞）造成四肢癱瘓＋無法言語吞嚥，但**意識清醒、認知完整**，僅保留**垂直眼球運動與眨眼**——是「清醒卻被鎖在身體裡」；值班關鍵是**別把清醒病人誤判成昏迷／植物人**，要主動測試垂直眼動溝通。

> [!danger] 🚨 值班紅旗
> 1. **看似「昏迷／無反應」的病人，務必測試「向上看／眨眼」指令** → 若能垂直眼動應答＝意識清醒的 locked-in，**絕非昏迷**，處置與預後完全不同
> 2. **急性四肢癱＋構音不能＋意識清醒 → 疑基底動脈栓塞**，這是**可取栓的急性中風**，緊急 CTA/MRI＋中風團隊，**時間就是腦幹**
> 3. **與昏迷/植物人/腦死鑑別攸關倫理與撤除維生決策** → 誤判成無意識可能造成災難性錯誤，需神經學細查與影像
> 4. **橋腦中央髓鞘溶解（CPM）** 可致 locked-in——**問低血鈉快速矯正史**（[[Central Pontine Myelinolysis(中央橋腦髓鞘溶解症)]]）
> 5. 病人**清醒且能感受疼痛與焦慮** → 溝通、鎮痛、心理支持、家屬衛教不可忽略（不是「沒感覺」）

## 📖 定義 / 病生理（這是什麼）
- **定義**：**運動去傳出（de-efferented）狀態**——皮質脊髓束與皮質延髓束（在橋腦腹側）中斷，四肢與延髓支配肌（臉、舌、咽、水平眼動）全癱，但**上位中腦被蓋（tegmentum）與網狀活化系統完好 → 意識清醒**。
- **保留功能**：**垂直眼球運動與眨眼**（由中腦控制，位於病灶上方），是唯一溝通管道。
- **病因**：
  | 類別 | 病因 |
  | --- | --- |
  | **血管（最常）** | **基底動脈血栓/栓塞**致橋腦腹側梗塞 |
  | 出血 | 橋腦出血 |
  | 脫髓鞘 | **中央橋腦髓鞘溶解（CPM，滲透壓性）**、MS |
  | 外傷/腫瘤 | 橋腦挫傷、腦幹腫瘤 |
  | 感染 | 橋腦膿瘍 |
- **分類（Bauer）**：Classic（全癱＋僅垂直眼動/眨眼）、Incomplete（保留部分自主動作）、Total（連眼動也無，最難診斷、易誤為昏迷）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Quadriplegia(四肢癱瘓)]]、[[Altered mental status(意識改變)]]
- **四肢癱瘓（quadriplegia）**、**anarthria（無法發聲構音）**、無法吞嚥。
- **臉部與延髓肌癱**：無表情、無法水平轉眼。
- **保留**：**垂直眼動（向上/向下看）、眨眼**、聽覺、**意識與認知完整**（可依指令用眼動回答 yes/no）。
- **呼吸**：橋腦廣泛病灶可影響呼吸型態，可能需呼吸器。
- **感覺**多保留（病人能感到疼痛與不適）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Neurological examination(神經學檢查)]]（垂直眼動指令）、[[EEG(腦電圖)]]（顯示清醒節律，鑑別昏迷）　影像 [[MRI(磁振造影)]]、[[CT Angiography(電腦斷層血管攝影)]]
- **床邊關鍵**：對「無反應」病人下**「請往上看」「請眨眼」**指令——能執行＝意識清醒＝locked-in，**這一步最不能省**。
- **MRI 腦（尤其 DWI）**：橋腦腹側梗塞/病灶為診斷確認；**MRA/CTA** 找**基底動脈阻塞**（決定能否取栓）。
- **EEG**：多顯示**大致正常的清醒背景節律**（有反應性）——與真正昏迷（瀰漫慢波）、腦死不同，有助鑑別。
- **鑑別診斷（關鍵）**：昏迷（意識喪失）、植物人/UWS（有睡醒週期但無覺察）、微意識狀態、**腦死**、嚴重 [[Guillain- Barre Syndrome(格林─巴利症候群)]]/重症肌無力危象（周邊性全癱但病灶不在腦幹）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **最大陷阱：把清醒的 locked-in 當成昏迷或植物人**——一律測垂直眼動與眨眼指令。
- **意識完整**：病人聽得到、想得到、會焦慮會痛——溝通與心理支持是照護核心。
- **與 GBS/MG 危象區別**：後兩者是**周邊神經肌肉**全癱、瞳孔/眼動可能保留、腦幹影像正常；locked-in 病灶在橋腦。
- **基底動脈栓塞是可治療的急症**——急性 locked-in 想到取栓。
- **溝通系統**：眼動編碼（往上=是）、字母板、腦機介面可重建溝通。
- 預後兩極：血管性大範圍者差；部分（尤其非血管、年輕）可部分恢復，長期存活者生活品質未必如外界想像的差（研究顯示不少病人自評尚可）。

## 💊 治療（Acute / Chronic）
> 用藥 依病因（無專屬藥物）
- **針對病因**：
  - **基底動脈栓塞** → 急性缺血性中風流程（IV 溶栓 / **機械取栓**，依時間窗與 protocol，見 [[Ischemic Stroke(缺血性中風)]]，⚠️藥物劑量查證）。
  - **橋腦出血** → 神經重症/神外處置。
  - **CPM** → 支持治療、避免再度快速電解質波動（見 [[Central Pontine Myelinolysis(中央橋腦髓鞘溶解症)]]）。
- **支持照護（長期核心）**：呼吸道與呼吸器管理、營養（鼻胃管/胃造廔）、防壓瘡與攣縮、DVT/肺炎預防、積極復健。
- **溝通重建**：眼動溝通系統、輔助溝通科技、腦機介面。
- **心理／倫理**：確認病人保有意識與決策能力，尊重其溝通與意願；家屬衛教「他聽得到」。

## ⚠️ 併發症
- 呼吸衰竭（需呼吸器）、吸入性肺炎
- 壓瘡、關節攣縮、DVT/PE
- 泌尿道感染、營養不良
- 憂鬱、焦慮（意識清醒面對重殘）
- 長期臥床相關併發症

| 工具 | 用途 | 意義 |
| --- | --- | --- |
| **無專用 score** | — | 依「意識」與「病灶」鑑別為主 |

| **GCS（受限）** | 意識評估但易誤判 | 運動/語言項會低估——locked-in 意識其實清醒，GCS 不適合單用 |
