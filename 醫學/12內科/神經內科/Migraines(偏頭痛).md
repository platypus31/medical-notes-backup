# Migraines（偏頭痛）

> [!abstract] 一句話：**反覆發作、中重度、單側搏動性頭痛，伴噁心／畏光畏聲，活動加重**，可有先兆（aura）；是**原發性頭痛**的臨床診斷，值班第一要務是**用紅旗徵象排除次發性致命頭痛（SAH、腦膜炎、腫瘤、靜脈竇栓塞）**，再談止痛與預防。

> [!danger] 🚨 值班紅旗（SNNOOP10——先排除次發性頭痛）
> 1. **「這輩子最痛」的霹靂頭痛（thunderclap，數秒到頂）** → **蜘蛛膜下腔出血 SAH** 直到排除；CT（±LP 找 xanthochromia），別當偏頭痛
> 2. **頭痛＋發燒＋頸僵／意識改變** → **腦膜炎／腦炎**，走感染流程
> 3. **新發＞50歲、癌症/免疫低下、姿勢改變加重、清晨痛醒、進行性加重** → 次發性（腫瘤、顱內壓）警訊，影像
> 4. **頭痛＋局灶神經缺損（非典型 aura）／視乳頭水腫／新發癲癇** → 結構性病灶或**腦靜脈竇栓塞（CVST）**（尤其產後、口服避孕、高凝）
> 5. **首次發作的「偏頭痛先兆」在中老年、或每次都同一側 aura** → 需與 TIA/中風鑑別（[[Transient Ischemic Attack(暫時性腦缺血)]]）

## 📖 定義 / 病生理（這是什麼）
- **定義**：原發性頭痛疾患，反覆發作，特徵性單側搏動痛＋自律/感覺敏感（畏光畏聲、噁心）。分**有先兆（with aura, ~25%）**與**無先兆**。
- **病生理（現代觀）**：**三叉神經血管系統（trigeminovascular system）活化** → 釋放 **CGRP、substance P** → 神經性發炎與血管擴張、痛覺敏感化（central sensitization → allodynia）；先兆源自**皮質擴散性抑制（cortical spreading depression, CSD）**。非單純「血管收縮-擴張」舊說。
- **常見誘因**：睡眠不足/過多、壓力（及壓力解除後）、月經（雌激素下降）、飢餓、脫水、酒精（紅酒）、特定食物、強光/氣味、天氣變化。
- **分類**：episodic vs **chronic migraine（頭痛 ≥ 15 天/月 ≥ 3 個月，其中 ≥ 8 天偏頭痛特徵）**；特殊型：偏癱性偏頭痛、前庭性偏頭痛（[[Vestibular Migraine(前庭性偏頭痛)]]）、腦幹先兆偏頭痛。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Headache(頭痛)]]、[[Photophobia(畏光)]]、[[Nausea(噁心)]]
- **四階段**（不一定全有）：前驅期（prodrome，數小時～1天，情緒/食慾/打呵欠變化）→ **先兆（aura，5–60 分鐘，可逆）**→ 頭痛期 → 後期（postdrome，疲憊/宿醉感）。
- **頭痛特徵（POUND 助記）**：**P**ulsatile 搏動、**O**ne-day（4–72h）、**U**nilateral 單側、**N**ausea 噁心、**D**isabling 活動加重致失能。
- **先兆（最常視覺）**：閃光暗點（scintillating scotoma）、鋸齒狀光譜（fortification）、逐漸擴散（**進展性**，與中風的「突發」不同）；也可感覺（麻木由手往臉爬）、言語。
- **前庭性偏頭痛**：反覆眩暈±頭痛/畏光，是眩暈常見中樞原因。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 病史為主（ICHD-3 診斷準則）　影像 [[CT(電腦斷層)]]／[[MRI(磁振造影)]]（僅在紅旗時）
- **臨床診斷（ICHD-3）**：
  - **無先兆偏頭痛**：≥ 5 次發作、頭痛 4–72h、具 ≥ 2 項（單側/搏動/中重度/活動加重）、且 ≥ 1 項（噁心嘔吐 或 畏光＋畏聲），無法用他病更好解釋。
  - **有先兆偏頭痛**：≥ 2 次發作、≥ 1 種完全可逆先兆（視覺/感覺/言語等），先兆漸進 ≥ 5 分鐘、每症狀 5–60 分鐘。
- **影像非常規**：**典型病史、神經學正常、無紅旗 → 不需影像**；有 SNNOOP10 紅旗才做 CT/MRI（±CTV/MRV 排 CVST、LP 排 SAH/感染）。
- **鑑別**：緊張型頭痛、叢集性頭痛（單側眼周、自律症狀、坐立難安）、次發性頭痛（見紅旗）、TIA（老年新發 aura）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **先兆是「漸進、可逆、通常先於或伴隨頭痛」**；**突發**的局灶缺損要當中風/TIA，不是 aura。
- **首次或型態改變的頭痛、有紅旗 → 先排次發性**，別急著貼「偏頭痛」標籤。
- **止痛藥過度使用頭痛（MOH）**：每月用急性止痛 ≥ 10–15 天 → 頭痛反而變多變慢性；問藥物使用頻率。
- **偏頭痛＋aura ＋吸菸＋口服避孕（含雌激素）→ 缺血性中風風險上升**，避免合併雌激素避孕。
- **triptans 禁忌**：缺血性心臟病、未控高血壓、偏癱性/腦幹先兆偏頭痛——這些不能用 triptan。
- **越早給急性藥越有效**（中央敏感化/allodynia 出現後效果差）。

## 💊 治療（Acute / Chronic）
> 用藥 [[NSAIDs(非類固醇消炎藥)]]、[[Triptans(翠普登)]]、[[Antiemetics(止吐劑)]]、[[Topiramate(妥泰)]]、[[Propranolol(普潘奈)]]、[[CGRP inhibitor(CGRP 抑制劑)]]
### 急性期（越早越好）
- **輕中度**：**NSAIDs / aspirin / acetaminophen**（±咖啡因複方）。
- **中重度或 NSAID 無效**：**triptans（如 sumatriptan）**——選擇性 5-HT1B/1D，注意心血管禁忌；**gepants（CGRP 受體拮抗，如 rimegepant）**、**ditans（lasmiditan）**為新選擇。
- **止吐/輔助**：metoclopramide、prochlorperazine（兼止吐與止痛），值班常用靜脈。
- **狀態性偏頭痛（status migrainosus）** → 補水、止吐劑、NSAID/triptan，難治可考慮短期類固醇/其他——**劑量依 protocol/藥典，⚠️查證**。
- **避免**：長期/頻繁使用鴉片與 butalbital（依賴、MOH）。
### 預防（發作頻繁/失能/急性藥用太多時）
- 口服：**β-blocker（propranolol）、topiramate、valproate、amitriptyline、candesartan**、flunarizine（依共病選，⚠️劑量查證，孕婦避 valproate/topiramate）。
- **CGRP 單株抗體（erenumab/fremanezumab/galcanezumab）**、gepants 用於預防。
- **Botulinum toxin A** 用於**慢性偏頭痛**（PREEMPT protocol）。
- 非藥物：規律睡眠/運動、誘因管理、放鬆、生物回饋。

## ⚠️ 併發症
- **止痛藥過度使用頭痛（MOH）**、慢性化（episodic → chronic）
- **偏頭痛性梗塞（migrainous infarction）**——罕見，aura > 60 分鐘且影像有梗塞
- 持續性先兆、偏頭痛觸發癲癇（migralepsy，罕見）
- **偏頭痛（尤有 aura）與缺血性中風風險上升**
- 生活品質、工作能力、共病憂鬱焦慮

## 📊 嚴重度評分 / 風險分層（scoring）
> 偏頭痛無「嚴重度救命 score」；用失能量表評估治療需求
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **MIDAS** | 過去 3 個月失能天數 | 分數越高失能越重 → 考慮預防治療 |
| **HIT-6** | 頭痛對生活衝擊 | ≥ 60 嚴重衝擊 |
| **ICHD-3 準則** | 診斷（非嚴重度） | 見診斷段 |
| **SNNOOP10 紅旗** | 篩次發性頭痛 | 任一陽性 → 影像/進一步檢查 |
