# Hemorrhagic Stroke（出血性中風）

> [!abstract] 一句話：**腦血管破裂出血造成的中風**，兩大類——**腦實質內出血（ICH，高血壓最常見）** 與**蛛網膜下腔出血（SAH，動脈瘤破裂為主，典型「這輩子最痛的頭痛」）**；核心處置是 **快速控血壓、逆轉抗凝、監測顱內壓（ICP）、神外急會診**；臨床提醒：**CT 一看到出血，tPA/抗血栓全部禁忌——與缺血中風處置完全相反，這一步錯就致命**。

> [!danger] 🚨 值班紅旗
> 1. **CT 有出血 = 絕對禁用 tPA/抗血小板/抗凝**，方向與缺血中風相反；立即改走出血 protocol。
> 2. **積極控血壓（但別過度）**：急性 ICH 一般目標**收縮壓降至 ~140 mmHg**（依 protocol，如 nicardipine/labetalol 靜脈滴定，⚠️劑量查證）；降太猛也有害，要平順可控。
> 3. **抗凝/抗血小板相關出血要立即逆轉**：warfarin 用 **PCC（四因子）＋ vitamin K**（PCC 優於 FFP）；DOAC 用專一逆轉劑（dabigatran → idarucizumab；Xa 抑制劑 → andexanet/PCC）；查 INR/凝血/血小板（劑量與品項依 protocol/藥典，⚠️需查證）。
> 4. **ICP 升高 / 意識快速下降 / 瞳孔不等大 / Cushing triad（高血壓＋心搏過緩＋不規則呼吸）** → 腦疝急症：頭抬高 30°、鎮靜鎮痛、短期過度換氣、滲透治療（mannitol/高張鹽水）、**神外急會診**評估手術/腦室引流。
> 5. **SAH「雷擊性頭痛（thunderclap headache，數秒達最痛）」** → CT 陰性但高懷疑仍要 **LP 找黃變（xanthochromia）**；確診後防**再出血（早期最致命）** 與**血管痙攣（第 3–14 天，用 nimodipine）**，並找動脈瘤處理（夾/栓）。

## 📖 定義 / 病生理（這是什麼）
- **定義**：因腦血管破裂導致血液進入腦組織或腦膜腔的中風，約占所有中風 **~13%**，但**死亡率與失能率高於缺血性**。
- **兩大類**：
  | 類型 | 位置 | 主因 | 典型 |
  | --- | --- | --- | --- |
  | **腦實質內出血 ICH** | 腦實質（基底核、視丘、橋腦、小腦、腦葉） | **慢性高血壓**（深部小血管 Charcot-Bouchard 微動脈瘤破裂）；老年腦葉出血想**類澱粉血管病(CAA)**；也見凝血異常、AVM、腫瘤、擬交感藥物 | 局灶缺損 + 頭痛/嘔吐/意識下降 + 血壓極高 |
  | **蛛網膜下腔出血 SAH** | 蛛網膜下腔 | **囊狀動脈瘤破裂**（最常見），也見 AVM、外傷 | **雷擊性頭痛**、頸僵硬、畏光、短暫意識喪失 |
- **病生理**：出血 → 直接組織破壞 + 血腫佔位效應 → 局部缺血、水腫、**ICP 升高** → 中線移位、腦疝；**血腫擴大（hematoma expansion，前數小時）** 與腦室內出血/水腦是惡化關鍵。SAH 另有再出血與遲發性腦缺血（血管痙攣）。
- **危險因子**：**高血壓（最重要）**、抗凝/抗血小板治療、高齡、CAA（腦葉出血）、AVM/動脈瘤、酗酒、擬交感物質（古柯鹼/安非他命）、凝血異常。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Headache(頭痛)]]、[[Altered mental status(意識改變)]]、[[Hemiparesis(偏癱)]]
- **ICH**：突發局灶神經缺損（依部位）＋ **頭痛、噁心嘔吐、意識下降、血壓顯著升高**；進展常較缺血中風快（血腫擴大）。
  - 部位線索：基底核/視丘（對側偏癱偏身感覺異常、視丘出血眼球下內偏）、橋腦（昏迷＋針尖瞳孔＋四肢癱）、小腦（眩暈、嘔吐、共濟失調、可急速壓迫腦幹—**神外急症**）。
- **SAH**：**雷擊性頭痛（瞬間達最痛、「這輩子最嚴重」）**、頸部僵硬、畏光、噁心嘔吐、短暫或持續意識喪失、可有動眼神經麻痺（後交通動脈瘤）。
- ICP 升高徵象：意識惡化、瞳孔變化、Cushing triad、噴射狀嘔吐、視乳突水腫。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 CBC、**凝血(PT/INR/aPTT)**、血小板、電解質、血糖、腎功能、（依線索）毒物　影像 [[Brain CT(腦部電腦斷層)]]（首選，高密度）、CTA（找動脈瘤/AVM/spot sign）、MRI/MRA、腰椎穿刺(SAH 且 CT 陰性)
- **非顯影 CT（首選）**：急性血液呈高密度，立即分辨 ICH vs SAH vs 缺血；評估血腫大小、部位、腦室內出血、中線移位、水腦。
- **CTA / CT venography**：找**動脈瘤、AVM、血管畸形**；ICH 的 **spot sign** 預測血腫擴大。
- **SAH 且 CT 陰性但高懷疑** → **腰椎穿刺找黃變（xanthochromia）**（發病 6–12 小時後最可靠）。
- **實驗室**：凝血功能與血小板（決定逆轉策略）、血糖、腎功能、毒物（擬交感）。
- **量化**：ICH 用 **ICH Score**；SAH 用 **Hunt-Hess（臨床）與 modified Fisher（CT 血量→血管痙攣風險）**。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **看到出血 = tPA/抗血栓全禁**，處置與缺血中風完全相反。
- **降壓要積極但平順**（ICH 目標約 SBP 140）；SAH 動脈瘤未處理前也控壓防再出血。
- **抗凝相關出血立即逆轉**：warfarin → PCC + vit K（PCC > FFP）；DOAC → 專一逆轉劑。
- **老年、腦葉、反覆出血 → 想類澱粉血管病(CAA)**；深部（基底核/視丘）→ 高血壓性。
- **小腦出血是神外急症**（可快速壓迫腦幹/阻塞第四腦室致水腦），別當單純眩暈。
- **SAH 兩大殺手**：早期**再出血**（處理動脈瘤前最致命）＋ 第 3–14 天**血管痙攣/遲發缺血**（用 **nimodipine** 改善結局）。
- **雷擊性頭痛**永遠先想 SAH，CT 陰性別掉以輕心（LP）。

## 💊 治療（Acute / Chronic）
> 用藥 [[Nicardipine(尼卡地平)]]／[[Labetalol(拉貝洛爾)]]（控壓）、[[Prothrombin complex concentrate(凝血酶原複合物PCC)]]、[[Mannitol(甘露醇)]]／高張鹽水、[[Nimodipine(尼莫地平)]]（SAH）
- **ICH 急性處置**：
  - **控血壓**：目標 SBP ~140 mmHg（依 INTERACT/ATACH 概念與 protocol），用靜脈可滴定藥（nicardipine/labetalol），**平順避免驟降**（劑量⚠️查證）。
  - **逆轉抗凝/矯正凝血**：warfarin → PCC + vitamin K；DOAC → idarucizumab（dabigatran）/andexanet 或 PCC（Xa）；血小板低下依情況輸注（抗血小板者常規輸血小板無益，除非需手術，⚠️依指引）。
  - **ICP/腦水腫**：頭抬高 30°、鎮痛鎮靜、必要時滲透治療（mannitol/高張鹽水）、控制發燒血糖、癲癇時抗癲癇（不常規預防性給）。
  - **手術**：**小腦出血 > 3 cm 或壓迫腦幹/水腦 → 急手術減壓**；幕上大血腫、腦室內出血合併水腦（腦室外引流 EVD）依神外評估。
- **SAH 處置**：
  - **防再出血**：控壓、止痛、安靜、**盡早處理動脈瘤（血管內栓塞 coiling 或手術夾閉 clipping）**。
  - **防血管痙攣/遲發缺血**：**nimodipine 口服**（改善神經結局，⚠️劑量查證）、維持正常血容量。
  - **併發**：水腦（EVD/分流）、低鈉（腦耗鹽 vs SIADH）、癲癇、心肺併發。
- **一般照護**：VTE 預防（急性期先機械性，出血穩定後再評估藥物）、吞嚥評估、血糖/發燒管理、復健。
- **次級預防（chronic）**：**嚴格控血壓**（最關鍵）；CAA 者抗栓藥高風險需個別權衡；戒酒/戒毒；找並處理 AVM/動脈瘤。

## ⚠️ 併發症
- **血腫擴大、腦室內出血、急性水腦、腦疝**（早期最致命）
- **SAH：再出血、血管痙攣/遲發性腦缺血、低鈉血症、神經源性肺水腫、心肌損傷**
- 癲癇、ICP 升高、腦積水
- 吸入性肺炎、深部靜脈栓塞/肺栓塞、壓瘡
- 長期失能、認知障礙、憂鬱

## 📊 嚴重度評分 / 風險分層（scoring）
> ICH 用 **ICH Score**；SAH 用 **Hunt-Hess** 與 **modified Fisher**
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **ICH Score（0–6）**（GCS、血腫體積 ≥30 mL、腦室內出血、幕下來源、年齡 ≥80） | ICH 30 天死亡率預測 | 分數越高死亡率越高（0 分近乎存活，5–6 分死亡率極高） |
| **Hunt-Hess（I–V）** | SAH 臨床分級 | I 輕微頭痛 → V 深昏迷去大腦僵直；越高預後越差 |
| **modified Fisher（1–4）** | SAH CT 血量 → **血管痙攣風險** | 血量越多、有 IVH → 血管痙攣/遲發缺血風險越高 |
| **WFNS（I–V）** | SAH（依 GCS ± 局灶徵象） | 另一常用 SAH 分級 |
| **GCS / NIHSS** | 意識與缺損程度 | 通用，追蹤惡化 |
