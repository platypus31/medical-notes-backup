# Optic Neuritis（視神經炎）

> [!abstract] 一句話：**視神經發炎/去髓鞘**造成的**急性～亞急性單眼視力下降＋眼球轉動疼痛＋色覺變差**，伴**相對傳入性瞳孔缺損（RAPD/Marcus Gunn）**；典型型與**多發性硬化症（MS）**高度相關，值班關鍵是**與缺血性視神經病變、NMOSD/MOG、感染／壓迫性病灶鑑別**——因為治療與預後大不同。

> [!danger] 🚨 值班紅旗（辨「非典型 ON」→ 別當單純 MS 型放走）
> 1. **雙眼同時、極嚴重（近乎失明）、快速惡化、對類固醇反應差** → 疑 **NMOSD（抗 AQP4）或 MOG 抗體**，需血清抗體＋積極治療（可能需血漿置換），漏了會永久失明
> 2. **視乳頭蒼白/嚴重水腫＋出血、無痛、老年＋血管危險因子** → 疑**缺血性視神經病變（AION）**，若合併頭痛/下顎跛行/ESR↑ → **巨細胞動脈炎（GCA）＝眼科急症，立即高劑量類固醇＋切片**，延誤致對側失明
> 3. **視力持續惡化 > 2 週、無自發改善、伴全身症狀** → 疑感染（梅毒/結核/病毒）、肉芽腫（sarcoid）、壓迫性（腫瘤）——非典型 ON，需影像＋抽血
> 4. **RAPD 是傳入性視路病變的客觀徵象**——單眼視力下降務必測 swinging flashlight，別漏
> 5. **合併腦幹/脊髓症狀** → 中樞去髓鞘（MS/NMOSD），完整神經學＋MRI

## 📖 定義 / 病生理（這是什麼）
- **定義**：視神經的發炎性去髓鞘，導致傳入視覺障礙。**典型（typical）ON** 多為特發性或 MS 相關；**非典型（atypical）ON** 涵蓋 NMOSD、MOG、感染、肉芽腫、自體免疫。
- **解剖分型**：
  | 型 | 位置 | 眼底 |
  | --- | --- | --- |
  | **球後視神經炎（retrobulbar）** | 視神經球後段（MS 最常） | **眼底正常**——「病人看不到，醫師也看不到」 |
  | **乳頭炎（papillitis）** | 視神經頭 | 視乳頭腫脹（兒童/感染較多） |
- **病生理**：自體免疫 T 細胞介導去髓鞘 → 傳導阻滯；MS 型與中樞髓鞘抗原相關；NMOSD 為**抗 AQP4（星狀膠質細胞水通道）**、MOG-AD 為**抗 MOG（髓鞘）**。
- **流行**：好發**年輕女性（20–40 歲）**，典型 ON 是 MS 常見的首發表現。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Visual loss(視力喪失)]]、[[Eye pain(眼痛)]]、[[RAPD(相對傳入性瞳孔缺損)]]
- **典型三主徵**：
  | 特徵 | 描述 |
  | --- | --- |
  | **急性單眼視力下降** | 數小時～數天惡化，程度輕重不一 |
  | **眼球轉動疼痛** | >90% 有，眼動時加劇（視神經鞘牽扯） |
  | **色覺障礙（dyschromatopsia）** | 紅色去飽和最敏感，常比視力更早/更明顯 |
- **客觀徵象**：**RAPD（相對傳入性瞳孔缺損，Marcus Gunn pupil）**——患眼在 swinging flashlight 時「反常放大」，是傳入病變客觀證據。
- **視野缺損**：常中心暗點（central scotoma），也可各式。
- **Uhthoff 現象**：體溫上升（運動/熱水）時視力暫時變差。
- 眼底：球後型正常、乳頭炎型腫脹。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Visual acuity(視力)]]、[[Color vision(色覺)]]、[[Visual field(視野)]]、[[OCT(光學同調斷層)]]、[[VEP(視覺誘發電位)]]、[[AQP4 antibody(水通道蛋白抗體)]]、[[MOG antibody(髓鞘寡突膠質細胞醣蛋白抗體)]]　影像 [[MRI(磁振造影)]]（腦＋眼眶，含 gadolinium）
- **臨床診斷**：典型病史（年輕、痛性單眼視力下降、色覺差、RAPD）＋眼科檢查。
- **MRI 腦＋眼眶（含顯影）**：看視神經強化；**腦白質去髓鞘病灶預測 MS 風險**（ONTT 研究：基線腦 MRI 病灶越多，日後 MS 機率越高）；長節段視神經/視交叉/脊髓病灶 → 疑 NMOSD/MOG。
- **抗體（非典型時必查）**：**血清抗 AQP4（NMOSD）、抗 MOG（MOG-AD）**。
- **輔助**：OCT（RNFL 變薄，追蹤）、VEP（P100 潛時延長）、視野。
- **非典型/雙眼/嚴重/不恢復** → 加做感染（梅毒 RPR/TPHA、結核、病毒）、發炎（sarcoid ACE、胸片）、自體免疫、LP（寡株帶）。
- **鑑別**：**AION（缺血，無痛、老年、血管危因、GCA）**、壓迫性視神經病變（腫瘤，進行性無痛）、中毒/營養性（雙眼、無痛）、視乳頭水腫（顱內壓高，雙眼）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **「病人看不到，醫師（眼底）看不到」＝球後視神經炎**（MS 型眼底常正常）。
- **RAPD 是關鍵客觀徵象**——單眼視力問題一定做 swinging flashlight。
- **痛性（眼動痛）＋年輕女性＝典型 ON；無痛＋老年＋血管危因＝想 AION（並問 GCA 症狀）**——治療方向相反（GCA 要馬上類固醇）。
- **雙眼、嚴重、不恢復、對類固醇差 → 非典型，查 AQP4/MOG**，NMOSD 漏診會累積失明與脊髓炎。
- **口服 prednisone 單用（低劑量）是禁忌**——ONTT 顯示單用口服類固醇**增加復發率**且不改善最終視力；急性期用**IV 高劑量甲基潑尼松龍**。
- **視力多數自發恢復（典型 ON）**，類固醇加速恢復但不改變最終視力——治療目的與病人期待要說清楚。

## 💊 治療（Acute / Chronic）
> 用藥 [[Methylprednisolone(甲基潑尼松龍)]]、[[Plasmapheresis(血漿置換)]]、[[Disease-modifying therapy(疾病調節治療)]]
### 急性期（典型 ON）
- **IV 高劑量 methylprednisolone**（ONTT 方案，如 1 g/day × 3–5 天，後續口服漸減——⚠️確切劑量/療程依 protocol/藥典查證），可**加速視力恢復**但不改變最終視力。
- **避免單用低劑量口服 prednisone**（增復發，ONTT）。
### 非典型 / 嚴重（NMOSD、MOG、對類固醇反應差）
- 加做/縮短時間到 **血漿置換（PLEX）**；確診 NMOSD → 長期免疫治療（rituximab、eculizumab、satralizumab、inebilizumab 等，⚠️依專科）。
- 感染/肉芽腫性 → 對因治療。
### 長期 / 預防
- **典型 ON ＋MRI 有去髓鞘病灶＝MS 高風險** → 神經科評估**疾病調節治療（DMT）**以延緩 MS。
- NMOSD/MOG → 各自的長期免疫維持治療、避免用某些 MS 藥（可能惡化 NMOSD）。

## ⚠️ 併發症
- 永久視力/色覺/對比敏感度殘損（尤其 NMOSD、反覆發作）
- 復發、對側眼受累
- 進展為 **多發性硬化症（MS）**（典型 ON）
- NMOSD 相關橫貫性脊髓炎、失明
- 類固醇副作用

## 📊 嚴重度評分 / 風險分層（scoring）
> 無單一嚴重度 score；用視功能指標＋MS 風險分層
| 工具 | 用途 | 意義 |
| --- | --- | --- |
| **無專用 score** | — | 依視力/視野/色覺/RAPD 與抗體分型 |
| **基線腦 MRI 病灶數（ONTT）** | 預測 MS 風險 | 病灶越多 → 10 年進展 MS 機率越高 |
| **視力 / 色覺 / OCT RNFL** | 嚴重度與追蹤 | 惡化程度與恢復 |
| **AQP4 / MOG 抗體** | 分型 | 陽性 → 非 MS 型，改變治療 |
