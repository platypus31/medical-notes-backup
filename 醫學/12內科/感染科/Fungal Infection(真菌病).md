# Fungal Infection（真菌病）

> [!abstract] 一句話：從**淺表念珠菌／皮癬菌**到**侵襲性黴菌感染（IFI）**的一大類病——值班最關鍵一句：**免疫低下（嗜中性白血球低下、移植、長期廣效抗生素/類固醇、TPN、中央靜脈導管）病人發燒抗生素治不好 → 想到黴菌，血培養抓 Candida、送 galactomannan/β-D-glucan、疑 candidemia 先拔/換 CVC**。

> [!danger] 🚨 值班紅旗
> 1. **Candidemia（念珠菌血症）**：血培養長 Candida **不是污染，是敗血症**。→ 立即 echinocandin（如 `Caspofungin` loading 70 mg → 50 mg QD）、**拔除或更換中央靜脈導管**、**照會眼科排除 endophthalmitis**、每日追血培養至陰性後再算 14 天療程。
> 2. **持續性 febrile neutropenia（廣效抗生素 4–7 天仍發燒）**→ 加**經驗性抗黴菌**（echinocandin 或 liposomal amphotericin B），並找 IFI 病灶（胸腹 CT、鼻竇）。
> 3. **Mucormycosis（毛黴菌病）＝外科急症**：糖尿病酮酸中毒／血液腫瘤病人，**鼻竇-眼眶-腦（rhino-orbito-cerebral）黑色壞死焦痂、眼眶蜂窩炎、顏面痛** → 立刻 **`Liposomal Amphotericin B` 高劑量 + 緊急清創手術 + 矯正 DKA/停鐵螯合劑**，延誤即致命。
> 4. **Cryptococcal meningitis（隱球菌腦膜炎）**：HIV/免疫低下 + 亞急性頭痛發燒 → LP 測**開口壓（常明顯升高）**、CSF 隱球菌抗原/墨汁染色；治療核心之一是**反覆 LP 降顱內壓**（顱內壓是主要死因）。
> 5. **中性球低下病人避免 fluconazole 當經驗用藥**（打不到 mold 如 Aspergillus / mucor），要用有 mold 涵蓋的 echinocandin 或 amphotericin。

## 📖 定義 / 病生理（這是什麼）
- **常見致病黴菌與情境**：
  | 病原 | 典型宿主/情境 | 首選藥 |
  | --- | --- | --- |
  | **Candida spp.**（念珠菌） | 導管、TPN、廣效抗生素、術後、免疫低下 | Echinocandin（重症）/ Fluconazole（穩定、感受性佳） |
  | **Aspergillus**（麴菌） | 嗜中性球低下、移植、慢性肺病/空洞 | **Voriconazole** |
  | **Cryptococcus**（隱球菌） | **HIV/AIDS**、類固醇、器官移植 | Ampho B + Flucytosine → Fluconazole |
  | **Mucorales（毛黴菌）** | **DKA/糖尿病**、血液腫瘤、去鐵胺 | 高劑量 Liposomal Ampho B + 手術 |
  | **Pneumocystis jirovecii (PCP)** | HIV CD4<200、長期類固醇/免疫抑制 | 高劑量 **TMP-SMX** + 類固醇（若缺氧） |
  | **Histoplasma / 地方性黴菌** | 旅遊史、鳥/蝙蝠糞、洞穴 | Itraconazole / Ampho B |
- **抗黴菌藥四大類機轉**：
  - **Polyene（Amphotericin B）**：結合 ergosterol 打洞→廣效、殺菌，但腎毒性/輸注反應；liposomal 劑型毒性較低。
  - **Azole（Fluconazole/Voriconazole/Itraconazole/Posaconazole/Isavuconazole）**：抑制 ergosterol 合成（CYP450，**大量交互作用**）；voriconazole 是 Aspergillus 首選、isavuconazole/posaconazole 涵蓋 mucor。
  - **Echinocandin（Caspofungin/Micafungin/Anidulafungin）**：抑制 β-1,3-glucan 合成→對 Candida 佳、對 mold 有限、**不涵蓋 Cryptococcus**；安全性高。
  - **Flucytosine（5-FC）**：搭配 ampho B 用於隱球菌腦膜炎，注意骨髓抑制。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Fever(發燒)]]、[[Neutropenic fever(嗜中性白血球低下症)]]
- **Candidemia**：不明原因持續發燒、敗血症；併發**眼內炎（視力模糊/飛蚊）、皮膚結節、肝脾念珠菌病**。
- **侵襲性肺麴菌病**：發燒、咳嗽、咳血、胸痛；影像 halo sign / air-crescent sign。
- **隱球菌腦膜炎**：亞急性頭痛、發燒、噁心、意識/視力改變、顱內壓升高徵象。
- **Mucormycosis**：鼻竇痛、顏面/眼眶腫、黑色壞死焦痂、快速進展至腦。
- **PCP**：亞急性漸進性乾咳、運動性喘、**顯著低血氧（走路 desat）**、胸部 X 光雙側間質浸潤。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Blood culture(血液培養)]]、[[Beta-D-glucan]]、[[Galactomannan]]　影像 [[Chest CT(胸部電腦斷層)]]
- **血液/組織培養**：Candida 血培養是金標準之一（但敏感度僅約 50%）。
- **非培養標記**：
  - **β-D-glucan**：廣泛黴菌（Candida、Aspergillus、PCP）陽性，**但 mucor、Cryptococcus 陰性**。
  - **Galactomannan（GM）**：Aspergillus，血清/BAL。
  - **Cryptococcal antigen（CrAg）**：血清/CSF，高敏感高特異。
- **影像**：胸部 CT（麴菌 halo/air-crescent）、鼻竇/腦 MRI（mucor）。
- **組織病理 + 特殊染色（GMS、PAS）**：mucor 需看到寬闊、無隔、直角分支菌絲確診並區別 Aspergillus。
- **PCP**：誘導痰/BAL 做染色或 PCR；LDH 常升高。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **血培養長 Candida 不能當污染** → 一律當 candidemia 治療、拔導管、驗眼底。
- **β-D-glucan 陰性不能排除 mucor 與 Cryptococcus**（兩者不含 β-glucan）——經典考點。
- **Fluconazole 打不到 mold**（Aspergillus、mucor）與部分 Candida（C. krusei 天然抗、C. glabrata 常抗）。
- **Voriconazole = Aspergillus 首選**；需監測血中濃度、視覺障礙、光敏感、肝功能、CYP 交互作用。
- **Mucor 靠手術清創救命**，單靠藥物常失敗；矯正 DKA、停去鐵胺是關鍵。
- **PCP 若 PaO₂<70 或 A-a gradient>35 → 加類固醇**（降低呼吸衰竭與死亡）。

## 💊 治療（Acute / Chronic）
> 用藥 [[Amphotericin B]]、[[Fluconazole]]、[[Voriconazole]]、[[Caspofungin]]、[[Flucytosine]]、[[TMP-SMX]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### Candidemia / 侵襲性念珠菌
- **首選（尤其重症/中性球低下）**：echinocandin，如 `Caspofungin 70 mg loading → 50 mg IV QD` 或 `Micafungin 100 mg IV QD`。
- 穩定、無 azole 暴露、感受性佳 → 可 step-down `Fluconazole 800 mg loading → 400 mg QD`。
- **拔/換 CVC + 眼底檢查 + 追血培養轉陰後 14 天**。
### 侵襲性麴菌病
- **`Voriconazole`（首選）**，替代 isavuconazole、liposomal ampho B。
### 隱球菌腦膜炎
- 誘導：`Liposomal Amphotericin B + Flucytosine（5-FC）` ≥2 週 → 鞏固 `Fluconazole 400–800 mg` → 維持 `Fluconazole 200 mg`；**反覆 LP 降顱內壓**。
### Mucormycosis
- **`Liposomal Amphotericin B` 高劑量（5–10 mg/kg/day）+ 緊急外科清創 + 矯正潛在病因**；可 step-down/salvage 用 isavuconazole/posaconazole。
### PCP
- `TMP-SMX` 高劑量（15–20 mg/kg/day of TMP，分次）× 21 天；**低血氧加 prednisone**；之後依免疫狀態預防投藥。

## ⚠️ 併發症
- Candidemia：**眼內炎（可致失明）**、感染性心內膜炎、肝脾念珠菌病、敗血性休克
- 麴菌：咳血/大出血、播散、腦膿瘍
- 隱球菌：顱內壓升高致失明/死亡、腦積水、IRIS
- Mucor：眼眶/顱內侵犯、海綿竇血栓、致死
- 抗黴菌藥毒性：ampho B 腎毒性/低血鉀鎂/輸注反應、azole 肝毒性與 QT 延長/交互作用

## 📊 嚴重度評分 / 風險分層（scoring）
> 以宿主風險 + 診斷確定度分層（EORTC/MSG 架構）
| 分層 | 定義 | 處置傾向 |
| --- | --- | --- |
| **Proven** | 組織/無菌部位培養或病理證實 | 標靶治療 |
| **Probable** | 宿主因子 + 臨床特徵 + 微生物標記（GM/glucan/培養） | 依證強力治療 |
| **Possible** | 宿主因子 + 臨床特徵，缺微生物證據 | 經驗治療/持續找病灶 |
| **宿主高風險** | 中性球低下、移植、長期類固醇、AIDS、DKA | 提早經驗抗黴菌 |
