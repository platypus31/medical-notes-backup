# S. haematobium（埃及血吸蟲 / Urogenital schistosomiasis）

> [!abstract] 一句話：非洲/中東淡水接觸感染的吸蟲，成蟲居膀胱靜脈叢，蟲卵沉積膀胱壁致**終末血尿與膀胱纖維化**，長期與**膀胱鱗狀細胞癌**相關——can't-miss：非洲旅居史 + 無痛終末血尿。

> [!danger] 🚨 值班紅旗
> 1. **撒哈拉以南非洲 / 中東淡水接觸史（游泳、涉水）+ 無痛終末血尿** → 想到 *S. haematobium*。
> 2. **急性 Katayama 症候群**：感染後數週的發燒、蕁麻疹、嗜酸性球增高——是免疫反應而非慢性期。
> 3. 慢性膀胱刺激、反覆泌尿道感染 + 流行區長住 → 警覺**膀胱鱗狀細胞癌**風險。
> 4. **神經血吸蟲病（脊髓型）**：橫斷性脊髓炎 / 馬尾症候群，是罕見但可致殘的急症，需及早類固醇 + praziquantel。

## 📖 定義 / 病生理
- **病原體**：*Schistosoma haematobium*（血吸蟲，一種吸蟲 trematode / fluke）。
- **傳播途徑**：接觸含**尾動幼蟲（cercariae）**的淡水，cercariae **直接鑽入皮膚**；中間宿主為 *Bulinus* 屬淡水螺。
- **地理**：撒哈拉以南非洲、尼羅河流域、中東（埃及、阿拉伯半島）；亦有科西嘉島（Corsica）本土爆發。
- **病生理**：cercariae 鑽皮 → 血流 → 成熟為成蟲，雌雄合抱寄居**膀胱與骨盆靜脈叢**。雌蟲產卵，蟲卵（具**終末棘 terminal spine**）穿過膀胱壁排入尿液；滯留的蟲卵引發**肉芽腫性發炎與纖維化** → 膀胱壁增厚、鈣化、輸尿管阻塞。慢性刺激與發炎為膀胱鱗狀細胞癌的致癌機轉。

## 🩺 臨床表現
> 連症狀卡 [[血尿]]、[[發燒]]
- **游泳者癢疹（cercarial dermatitis）**：鑽皮處短暫搔癢丘疹。
- **急性血吸蟲病 / Katayama fever**（感染後 2–8 週）：發燒、寒顫、肌痛、蕁麻疹、咳嗽、肝脾腫、**顯著嗜酸性球增高**（初次感染 naïve 宿主較明顯）。
- **慢性泌尿生殖型**：
  - **終末血尿（terminal hematuria）**：排尿末段血尿，經典表現；排尿困難、頻尿。
  - 膀胱纖維化 / 鈣化、輸尿管狹窄 → **腎積水、腎功能受損**。
  - **女性生殖器血吸蟲病（FGS）**：陰道/子宮頸砂樣斑塊，增加 HIV 感染風險與不孕。
- **異位型**：神經血吸蟲病（脊髓炎最常見於 *S. haematobium*）。

## 🔬 診斷
> 檢查 [[尿液鏡檢]]　血清學　膀胱鏡
- **尿液鏡檢查蟲卵**：見**具終末棘（terminal spine）的蟲卵**為確診標準；建議**午間（10:00–14:00）尿液**、可濾膜濃縮，排卵有日中高峰。
- **尿液常規**：血尿、蛋白尿；嗜酸性球增高。
- **血清學抗體**：對旅遊者 / 低負荷感染或 Katayama 期（蟲卵尚未排出）有幫助，但無法區分過去/現在。
- **抗原檢測**：尿液 **CCA / CAA** point-of-care 檢測。
- **影像 / 內視鏡**：超音波看膀胱壁增厚 / 鈣化、腎積水；膀胱鏡見砂樣斑塊、切片（慢性者評估惡性變）。
- **鑑別 *S. mansoni* / *S. japonicum***（腸道與肝脾型，蟲卵具**側棘 lateral spine**，糞便檢查）。

## ⭐ 特徵 / 注意（pearls & pitfalls）
- **終末棘蟲卵 + 尿液**＝ *S. haematobium*；**側棘蟲卵 + 糞便**＝ *S. mansoni*。
- 感染早期（Katayama）蟲卵尚未排出，**尿檢可偽陰**，此時靠血清學；必要時**數週後重驗**。
- 慢性 *S. haematobium* 是**膀胱鱗狀細胞癌（squamous cell carcinoma）**的重要危險因子，與吸菸致膀胱移行細胞癌不同。
- **praziquantel 對成蟲有效、對未成熟蟲效果差**；Katayama 期或近期暴露者常需**間隔數週重複治療**。

## 💊 治療（Acute / Chronic）
> 用藥 [[Praziquantel]]
- **Praziquantel**：泌尿生殖與所有血吸蟲種的首選抗蟲藥，**依指引劑量**（WHO 常用單日方案，流行區 MDA 亦用之）。
  - 因對未成熟蟲效果不佳，**急性 / 近期暴露者需於數週後重複一次**。
- **Katayama fever / 神經血吸蟲病**：常需**類固醇**控制免疫反應，再給（或延後給）praziquantel。
- **併發症處置**：阻塞性尿路病變、腎積水的泌尿外科處置；膀胱惡性變依腫瘤治療。

## ⚠️ 併發症
- 膀胱纖維化 / 鈣化、輸尿管狹窄、腎積水、慢性腎病、末期腎衰竭。
- **膀胱鱗狀細胞癌**。
- 反覆或續發泌尿道細菌感染（含 Salmonella 慢性帶菌）。
- 女性生殖器血吸蟲病（不孕、增加 HIV 易感性）、神經血吸蟲病（截癱）。

## 📊 嚴重度評分 / 風險分層
- 無單一標準評分；以**尿液蟲卵計數（eggs/10 mL，反映感染強度）**、超音波膀胱/上泌尿道病變分級評估負荷與器官損傷。
- 臨床分期：cercarial dermatitis → 急性 Katayama → 慢性泌尿生殖型 → 併發症期（阻塞 / 惡性變）。
