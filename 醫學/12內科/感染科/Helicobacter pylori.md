# Helicobacter pylori（幽門螺旋桿菌）

> [!abstract] 一句話：螺旋狀、微需氧、產尿素酶（urease）的革蘭氏陰性菌，定殖胃黏膜；為**消化性潰瘍、胃 MALT 淋巴瘤、胃腺癌**的主因，WHO 列第一級致癌物，可根除治療。

> [!danger] 🚨 值班紅旗
> 1. 消化性潰瘍出血（吐血 / 黑便 / 血紅素↓ / 心搏過速）→ 先穩定生命徵象、輸液 / 輸血、PPI、內視鏡止血，**根除療法待急性期後**。
> 2. **警訊症狀（alarm features）**：體重減輕、吞嚥困難、貧血、反覆嘔吐、>50 歲新發消化不良、腸胃道出血、腹部腫塊 → **先安排內視鏡**排除胃癌，不可只當單純胃炎。
> 3. 胃 MALT 淋巴瘤早期單靠**根除 H. pylori 即可能消退** → 別漏掉這個可治病因。
> 4. NSAID / aspirin 使用者合併 H. pylori 潰瘍出血風險加乘。

## 📖 定義 / 病生理（這是什麼）
- **病原體**：Helicobacter pylori，螺旋狀 / 微需氧革蘭氏陰性桿菌，具鞭毛（穿透黏液層）。
- **關鍵毒力**：**Urease** 分解尿素產氨中和胃酸（利於存活、也是診斷基礎）；**CagA、VacA** 毒素破壞上皮、促發炎與癌化。
- **傳染途徑**：口口 / 糞口，多於幼年感染，家庭內傳播；全球盛行率高。
- **病生理**：定殖胃竇 → 慢性活動性胃炎 → 依模式演變為十二指腸潰瘍（胃竇為主、胃酸↑）或胃體萎縮性胃炎 → 腸化生 → 胃腺癌。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[上腹痛]]、[[消化不良]]、[[腸胃道出血]]
- 多數**無症狀**帶菌。
- **消化性潰瘍**：上腹痛（十二指腸潰瘍典型空腹 / 夜間痛、進食緩解；胃潰瘍進食後痛）、消化不良、脹氣。
- **併發症表現**：黑便 / 吐血（出血）、突發劇烈腹痛 + 腹膜炎（穿孔）、嘔吐（幽門阻塞）。
- MALT 淋巴瘤 / 胃癌：常無特異症狀，或出現警訊症狀。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Urea breath test 尿素呼氣試驗]]、[[糞便抗原]]、[[內視鏡切片]]
- **非侵入性**（無警訊症狀者首選 test-and-treat）：
  - **尿素呼氣試驗（UBT, ¹³C）** — 高敏感 / 特異，追蹤根除首選。
  - **糞便抗原檢測（stool antigen）** — 敏感度高，可診斷 + 追蹤。
  - 血清抗體：只證明曾感染，**不能用於判斷根除**。
- **侵入性（內視鏡）**：切片行 **rapid urease test（CLO test）**、組織學、細菌培養 + 抗藥性檢測。
- ⚠️ **檢測前停藥**：UBT / 糞抗 / CLO 須停 **PPI ≥ 2 週、抗生素 / bismuth ≥ 4 週**，否則偽陰性。

## ⭐ 特徵 / 注意（pearls & pitfalls）
- 記憶：**Urease 是靈魂** — 致病（中和胃酸）也是診斷（UBT / CLO test）核心。
- 消化性潰瘍雙主因：**H. pylori + NSAID**；找到潰瘍要驗 HP。
- **根除後務必驗證清除**（UBT 或糞抗，停藥後 ≥4 週）：胃潰瘍 / MALT / 出血潰瘍 / 胃癌術後皆須確認。
- 胃 MALT 淋巴瘤（低惡性度、局限）根除 HP 後多可退化。
- 治療失敗多因 **clarithromycin 抗藥性**上升 → 各地建議依抗藥性選方案。

## 💊 治療（Acute / Chronic）
> 用藥 [[PPI]]、[[Clarithromycin]]、[[Amoxicillin]]、[[Bismuth]]、[[Metronidazole]]
根除療法（依當地 clarithromycin 抗藥性選擇，療程多 14 天）：
- **含 bismuth 四合一療法**（高抗藥性地區首選）：**PPI + Bismuth + Metronidazole + Tetracycline**。
- **併存四合一（concomitant）**：PPI + Amoxicillin + Clarithromycin + Metronidazole（無 bismuth）。
- **傳統三合一**（僅限 clarithromycin 抗藥性 <15% 地區）：**PPI + Clarithromycin + Amoxicillin**。
- 盤尼西林過敏：以 Metronidazole 取代 amoxicillin。
- **第一線失敗**：改含 **levofloxacin** 方案或 bismuth 四合一；理想上依培養 / 分子抗藥性檢測選藥。
- 具體劑量依當地指引（Maastricht VI / 台灣共識），不確定時**依當地指引**。

## ⚠️ 併發症
- 消化性潰瘍出血、穿孔、幽門狹窄阻塞。
- **胃腺癌**（WHO 第一類致癌物）、**胃 MALT 淋巴瘤**。
- 萎縮性胃炎、腸化生（癌前病變）；缺鐵性貧血、維生素 B12 缺乏、免疫性血小板缺乏（ITP，部分根除後改善）。

## 📊 嚴重度評分 / 風險分層
> 潰瘍出血以 **Glasgow-Blatchford / Rockall score** 分層再出血與死亡風險、決定住院與內視鏡時機；胃癌風險以萎縮 / 腸化程度（OLGA / OLGIM 分期）分層追蹤。
