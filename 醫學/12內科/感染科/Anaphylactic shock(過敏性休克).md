# Anaphylactic shock（過敏性休克）

> [!abstract] 一句話：**IgE 介導（第一型過敏）的全身性急性反應**，肥大細胞/嗜鹼性球大量去顆粒 → 血管擴張 + 通透↑ + 支氣管痙攣 → **分布性休克 + 呼吸道阻塞**。**第一線且唯一救命藥是肌肉注射 Epinephrine**，任何延遲都增加死亡。

> [!danger] 🚨 值班紅旗
> 1. **第一線立刻打 Epinephrine IM（大腿前外側）— 成人 0.3–0.5 mg（1 mg/mL＝1:1000），每 5–15 分鐘可重複**。不要先給抗組織胺/類固醇拖延，也不要因血壓還可以就不打
> 2. **1:1000（IM）與 1:10000（IV 急救）不可搞混**——IM 用 1 mg/mL；未經稀釋 IV bolus 1:1000 會致命性心律不整/高血壓
> 3. **快速氣道評估**：喉頭水腫/喘鳴 → 及早準備插管（水腫惡化後插管極困難）
> 4. **雙相反應（biphasic）**：初步緩解後 1–72 小時可再發 → 需留觀（重症/需多次腎上腺素者觀察更久）

## 📖 定義 / 病生理（這是什麼）
- **Anaphylaxis**：急性、可致命的全身性過敏反應；**anaphylactic shock** 指伴隨循環衰竭（低血壓/休克）者。
- **機轉**：
  - **IgE 介導（免疫型）**：食物（堅果/海鮮/蛋奶）、藥物（**β-lactam 抗生素**、NMBA、NSAID）、蟲螫、乳膠 → 再次接觸 → 交聯肥大細胞表面 IgE → 去顆粒
  - **非 IgE / 類過敏（anaphylactoid）**：顯影劑、萬古黴素（[[Red man syndrome(紅人症候群)]] 為輸注速率相關，非真過敏）、opioids 直接去顆粒
- **介質**：**histamine、tryptase、白三烯、PAF、prostaglandin** → 血管擴張、微血管滲漏（血漿外滲致相對血容不足）、支氣管痙攣、黏膜水腫。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Urticaria(蕁麻疹)]]、[[Dyspnea(呼吸困難)]]、[[Hypotension(低血壓)]]
- **皮膚黏膜（~80–90%）**：蕁麻疹、搔癢、潮紅、血管性水腫（嘴唇/舌/眼周）——**但可無皮膚表現**（勿因無疹排除）
- **呼吸道**：喉頭緊/聲音沙啞/喘鳴（喉水腫）、支氣管痙攣（喘鳴、呼吸困難、低血氧）
- **循環**：低血壓、心搏過速、頭暈、暈厥、休克
- **腸胃**：絞痛、嘔吐、腹瀉
- 常在**暴露後數分鐘至 1–2 小時內**快速進展

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Serum tryptase(血清類胰蛋白酶)]]
- **臨床診斷（不要等檢驗才治療）**。符合下列任一即高度可能（WAO/NIAID 準則精神）：
  1. **急性發作（分鐘~數小時）皮膚/黏膜症狀 + 呼吸障礙 或 低血壓/器官灌流不足**
  2. 暴露於**可能過敏原後**，快速出現 ≥2 系統症狀（皮膚、呼吸、循環、腸胃）
  3. 暴露於**已知過敏原後**單純低血壓
- **血清 tryptase**：發作後 **15 分–3 小時**抽（支持診斷，尤蟲螫/藥物；正常不排除，食物性常不升）；與 baseline 比較更有意義。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **Epinephrine 沒有絕對禁忌**——過敏性休克時延遲/不給腎上腺素是主要可預防死因；抗組織胺與類固醇是**輔助不是救命藥**。
- **無皮膚症狀不能排除**（約 10–20% 無疹）。
- **正在服 β-blocker**者對腎上腺素反應差、易頑固性低血壓 → 可加 **Glucagon**（⚠️ 劑量查證）。
- **雙相反應**：緩解後仍需留觀（一般至少數小時，重症/多劑更久）。
- 平躺抬腳（休克），孕婦左側躺；勿突然直立（可誘發致命性 empty ventricle）。
- 區分 [[Red man syndrome(紅人症候群)]]（vancomycin 輸注速率過快，減速+抗組織胺即可，非真過敏）。

## 💊 治療（Acute / Chronic）
> 用藥 [[Epinephrine(腎上腺素)]]、[[抗生素]]（找出致敏抗生素並停用/標記過敏）
### 🔴 急性（救命順序）
1. **Epinephrine IM 立即給**：**成人 0.3–0.5 mg（0.01 mg/kg，單次上限 0.5 mg）**，用 **1 mg/mL（1:1000）** 注射於**大腿前外側（vastus lateralis）**；**每 5–15 分鐘可重複**。（兒童 0.01 mg/kg，單次上限 0.3 mg）
2. **移除過敏原**（停正在輸注的藥物/顯影劑等）
3. **ABC + 高流量氧氣**，喉頭水腫及早準備氣道
4. **平躺抬下肢**（維持前負荷）
5. **大量 IV 晶體液**（休克時快速輸注，成人常 1–2 L bolus，依反應追加⚠️劑量依體重/情況）
6. **頑固性低血壓 / 反覆發作** → 移至監護環境，**IV epinephrine 輸注**（稀釋、幫浦、心電監測；**IV bolus 用 1:10000 且僅在心跳停止/瀕死並嚴密監測下**）
7. **β-blocker 使用者頑固性低血壓 → Glucagon**（⚠️ 劑量查證）
8. **輔助藥（非第一線、不可取代腎上腺素）**：H1 抗組織胺（緩解皮膚癢/疹）、H2 抗組織胺、支氣管痙攣加吸入 β2-agonist（salbutamol）、類固醇（可能減少雙相反應，證據有限）
### 慢性 / 出院
- **開立 Epinephrine auto-injector（EpiPen）+ 衛教使用時機**、**避免過敏原**、記錄並標註藥物過敏（病歷/手圈）、轉介過敏免疫科評估、必要時減敏。

## ⚠️ 併發症
- **呼吸道完全阻塞**（喉水腫）→ 窒息、死亡
- **難治性休克**、缺氧性腦傷、心律不整/心肌缺血（可為反應本身或腎上腺素）
- **雙相 / 遷延性反應**、吸入性肺炎

## 📊 嚴重度評分 / 風險分層（scoring）
> 過敏性休克以「器官受累範圍 + 呼吸/循環受損」分級，重點是及早給藥非計分
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **臨床分級（Ring & Messmer）** | 反應嚴重度 | I 皮膚 / II 多系統中度 / III 危及生命（喉水腫、支氣管痙攣、休克）/ IV 心肺停止 |
| **是否低血壓/缺氧** | 是否 shock | 有循環/呼吸衰竭 → anaphylactic shock，最積極 |
| **是否需 >1 劑腎上腺素** | 雙相風險/留觀時間 | 需多劑或重症 → 延長觀察 |
