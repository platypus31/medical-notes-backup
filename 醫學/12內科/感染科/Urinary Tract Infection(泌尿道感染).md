# Urinary Tract Infection（泌尿道感染）

> [!abstract] 一句話：泌尿道的細菌感染，從**下泌尿道（膀胱炎 cystitis）**到**上泌尿道（腎盂腎炎 pyelonephritis）**到**尿路敗血症**成連續光譜。**E. coli 最常見**。臨床關鍵：分辨 **單純 vs 複雜**、**上 vs 下**，並判斷是否為**無症狀菌尿（多數不治療）**。

> [!danger] 🚨 值班紅旗
> 1. **發燒 + 腰痛（CVA tenderness）/寒顫/噁心嘔吐** → 腎盂腎炎，不是單純膀胱炎，要抽血培養 + 積極抗生素
> 2. **UTI + 敗血症徵象（qSOFA↑、低血壓、lactate↑）** = urosepsis → 走敗血症 bundle
> 3. **阻塞性尿路感染（結石/腫瘤 + 感染 + 發燒）= 泌尿科急症**，需**緊急引流減壓**（輸尿管支架/腎造廔），光給抗生素會惡化成休克
> 4. **無症狀菌尿只在孕婦、泌尿侵入性手術前才治療**——其他（含長者、留置導管）驗到細菌別亂投抗生素

## 📖 定義 / 病生理（這是什麼）
- **分類**：
  - **下泌尿道**：膀胱炎（cystitis）、尿道炎（urethritis）
  - **上泌尿道**：腎盂腎炎（pyelonephritis）
  - **單純（uncomplicated）**：健康非孕婦、結構功能正常
  - **複雜（complicated）**：男性、孕婦、留置導管、結石/阻塞、糖尿病/免疫低下、腎功能異常、近期器械操作、院內/抗藥菌
- **致病菌**：**E. coli（~80% 社區單純型）**、其他腸桿菌（Klebsiella、Proteus——與結石有關）、Enterococcus、Staph saprophyticus（年輕女性）；院內/複雜型 → Pseudomonas、抗藥 GNR、Candida。
- **病生理**：多為**上行性**（尿道→膀胱→輸尿管→腎），少數血行性（如 S. aureus 腎膿瘍）。危險因子：女性、性行為、憋尿、阻塞/逆流、導管、雌激素低下、糖尿病。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dysuria(排尿疼痛)]]、[[Fever(發燒)]]、[[Flank pain(腰痛)]]
- **膀胱炎**：頻尿、急尿、**排尿灼痛（dysuria）**、恥骨上不適、血尿；**通常無發燒**
- **腎盂腎炎**：發燒、寒顫、**腰痛 / CVA 敲痛**、噁心嘔吐 ± 下泌尿道症狀
- **長者/導管者**：可能僅意識改變、虛弱、跌倒（非典型）——但別把菌尿當唯一解釋而漏其他病因

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Urinalysis(尿液分析)]]、[[Urine culture(尿液培養)]]　影像 [[Renal ultrasound(腎臟超音波)]]、[[CT KUB]]
- **尿液常規**：**膿尿（pyuria，WBC↑）**、**亞硝酸鹽（nitrite，腸桿菌）**、白血球酯酶（leukocyte esterase）陽性、可見細菌/血尿。
- **尿液培養**：確診 + 感受性；複雜型/腎盂腎炎/治療失敗必送。單純膀胱炎典型病史可經驗治療不一定送培養。
- **血液培養**：腎盂腎炎/疑 urosepsis。
- **影像**：一般膀胱炎不需；**腎盂腎炎治療 48–72h 無改善、疑阻塞/結石/膿瘍、反覆感染** → 腎超音波 / CT（找阻塞、氣腫性腎盂腎炎、腎周膿瘍）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **無症狀菌尿（ASB）**：僅**孕婦**與**泌尿侵入性手術前**要篩+治；老人/導管/糖尿病驗到菌尿無症狀**不治療**（濫用助長抗藥）。
- **留置導管者**：膿尿幾乎必有，**單靠膿尿不能診斷 CAUTI**，要有全身症狀且排除其他病因；診斷 CAUTI 建議先**換/移除導管**再採檢。
- **Proteus（尿液鹼性、struvite 結石）**、反覆同菌 → 想結石。
- **氣腫性腎盂腎炎**（糖尿病、影像見氣體）= 重症急症，常需引流甚至腎切除。
- 男性 UTI 幾乎都算複雜型，要想攝護腺炎/結構異常。

## 💊 治療（Acute / Chronic）
> 用藥 [[Quinolone]]、[[Cephalosporin]]、[[抗生素]]
### 急性 — 單純膀胱炎（女性，經驗性，依當地抗藥）
- **Nitrofurantoin** 100 mg BID ×5 天｜**TMP-SMX** DS BID ×3 天（當地 E. coli 抗藥 <20% 才用）｜**Fosfomycin** 3 g 單劑
- ⚠️ Nitrofurantoin **不用於腎盂腎炎**（腎實質濃度不足）、eGFR 過低者避用；具體劑量依腎功能查證
### 急性 — 腎盂腎炎
- **門診可耐受**：口服 fluoroquinolone（如 ciprofloxacin）×7 天，或先一劑長效 IV（ceftriaxone）後接口服
- **住院/嘔吐/重症**：IV **ceftriaxone 1–2 g q24h**、或抗綠膿 β-lactam（cefepime / piperacillin-tazobactam）、重症/抗藥風險用 carbapenem；退燒穩定後降階口服，總療程 7–14 天（依藥物/嚴重度）
### 複雜型 / urosepsis
- 廣效涵蓋抗藥 GNR ± 綠膿，**解除阻塞（引流）是關鍵**，走敗血症流程。
### 慢性 / 預防
- 反覆單純 UTI：行為（多喝水、性行為後排尿）、停經婦女陰道雌激素、必要時預防性抗生素或性行為後單劑；評估結構因素。

## ⚠️ 併發症
- **腎盂腎炎 → urosepsis / 敗血性休克**、腎/腎周膿瘍、**氣腫性腎盂腎炎**
- **阻塞 + 感染 → 膿腎（pyonephrosis）**、腎功能損害
- 孕婦 ASB 未治 → 腎盂腎炎、早產/低體重
- 慢性/反覆 → 腎瘢痕（尤兒童逆流）

## 📊 嚴重度評分 / 風險分層（scoring）
> UTI 分層靠「單純 vs 複雜、上 vs 下、是否 urosepsis」
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **單純 vs 複雜** | 選藥+療程 | 複雜型需培養、較廣效、較長療程、找結構因素 |
| **qSOFA / SOFA** | 是否 urosepsis | ≥2 → 走敗血症 bundle |
| **是否阻塞** | 是否需引流 | 阻塞+感染 = 急症，抗生素+緊急減壓 |
