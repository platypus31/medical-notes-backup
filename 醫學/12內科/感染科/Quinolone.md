# Quinolone

> [!abstract] 一句話：抑制細菌 **DNA gyrase / topoisomerase IV** 的廣效殺菌抗生素，**口服吸收極佳（bioavailability≈IV）**；值班要記的是**一堆黑框警語**——肌腱斷裂、QTc 延長、主動脈瘤/剝離、周邊神經病變、CNS 效應，以及**與二價/三價陽離子螯合會失效**。

> [!danger] 🚨 值班紅旗
> 1. **QTc 延長 → Torsades**：用前/合併其他延長 QT 藥物者查 **baseline EKG**，避開低血鉀/低血鎂。
> 2. **主動脈瘤 / 主動脈剝離風險上升**：**有主動脈瘤、Marfan、老年、高血壓者避免**（黑框警語，見 [[Aortic dissection(主動脈剝離)]]）。
> 3. **肌腱炎/肌腱斷裂（尤其阿基里斯腱）**：老年、類固醇併用、移植病人風險高；出現腱痛立即停藥。
> 4. **與 Ca²⁺/Mg²⁺/Al³⁺/Fe²⁺/制酸劑/牛奶螯合 → 吸收銳減**：分開 **服前 1h 或服後 2h**（口服失效是常見治療失敗原因）。
> 5. **孕婦 / <16 歲避免**（軟骨毒性）；併發 **CNS（癲癇、譫妄）**、**周邊神經病變（可不可逆）**、**血糖異常**、**加重重症肌無力**。

## 📖 定義 / 病生理（機轉 / 分類）
- **機轉**：抑制 **DNA gyrase（topoisomerase II，GNB 主要靶）** 與 **topoisomerase IV（GPC 主要靶）** → 阻斷 DNA 複製 → **殺菌（concentration-dependent，療效看 AUC/MIC、Cmax/MIC）**。
- **藥動特性**：口服生體可用率高、組織穿透佳（前列腺、骨、肺、腎）。
- **世代 / 抗菌譜（詳見下方治療區銀蛋口袋書表格）**：

| 世代 | 代表藥 | 抗菌譜重點 |
| --- | --- | --- |
| 第二代 | **Ciprofloxacin** | GNB 強（含 **Pseudomonas**）、非典型；GPC 弱；泌尿/腸道/前列腺 |
| 第三代 | **Levofloxacin** | GNB + **加強 GPC（肺炎鏈球菌）**；"呼吸道 FQ" — CAP、UTI |
| 第四代 | **Moxifloxacin** | 加**厭氧菌**；**不涵蓋 PsA、泌尿道濃度低不治 UTI**；肝代謝不需調腎 |

## 🩺 抗菌譜 / 臨床適應症（概覽）
> 詳細各藥 spectrum / dosage 見下方 💊 治療區銀蛋口袋書表格與 PDF 摘錄
- **Ciprofloxacin**：[[Urinary Tract Infection(泌尿道感染)]]、前列腺炎、Pseudomonas、[[Infectious Diarrhea(感染性腹瀉)]]（Salmonella/Shigella）。
- **Levofloxacin**：[[Community-Acquired Pneumonia(社區型肺炎)]]（含非典型）、UTI、複雜性感染。
- **Moxifloxacin**：吸入性/厭氧相關[[Pneumonia(肺炎)]]、[[Intra-Abdominal Infections(腹腔內感染)]]、軟組織——但**不能治 UTI、不 cover PsA**。
- 其他：分枝桿菌（含 MDR-TB 二線）、骨關節感染、旅行者腹瀉。

## 🔬 使用前評估 / 監測
> 用藥前查：baseline EKG（QTc）、電解質（K⁺/Mg²⁺）、藥物交互（陽離子/warfarin/theophylline/降糖藥）
- **心臟**：baseline EKG，避免與其他 QT 延長藥/低鉀鎂併用。
- **交互作用**：金屬陽離子螯合（分開服用）；抑制代謝酵素 → 升高 warfarin（INR↑）、theophylline（毒性）。
- **族群禁忌/慎用**：孕婦、幼兒、主動脈瘤、肌腱病史、重症肌無力、癲癇、糖尿病（血糖波動）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **口服 = IV 吸收**：能口服就別浪費靜脈線（除非吸收受影響）。
- **陽離子螯合**是口服失效常見原因（制酸劑、鈣鎂鐵補充、牛奶、灌食配方）。
- **Moxifloxacin 不治 UTI**（泌尿道濃度低）、**不 cover Pseudomonas**——選錯藥常見陷阱。
- 一連串**黑框警語**（肌腱、主動脈、周邊神經、CNS、重症肌無力、血糖）→ 非必要不首選，保留給有適應症者。
- 抑制 warfarin 代謝 → INR 飆升出血。

## ⚠️ 副作用 / 併發症（多為黑框警語）
- **心臟**：QTc 延長 → Torsades；**主動脈瘤 / 主動脈剝離**風險上升。
- **肌腱**：肌腱炎、**肌腱斷裂（阿基里斯腱）**，老年/類固醇/移植者高風險。
- **神經**：**周邊神經病變（可不可逆）**、CNS（癲癇、譫妄、頭痛）、**加重重症肌無力**。
- **代謝**：血糖異常（低血糖/高血糖，gatifloxacin 尤甚已下市）。
- **其他**：光敏感、CDAD、骨髓抑制、軟骨毒性（幼兒/孕婦）、視網膜剝離爭議。
- **交互**：金屬陽離子螯合（失效）、升高 warfarin/theophylline。

## 📊 風險分層（用藥安全評估）
> 用前先過一遍禁忌與交互，決定可否用/需否監測
| 判斷點 | 用途 | 意義 |
| --- | --- | --- |
| QTc / 電解質 / 併用藥 | Torsades 風險 | baseline EKG、避低鉀鎂與其他 QT 藥 |
| 主動脈瘤 / Marfan / 老年高血壓 | 剝離風險 | 有風險 → 避用 |
| 肌腱風險（老年/類固醇/移植） | 肌腱斷裂 | 出現腱痛立即停 |
| 陽離子併服 / 交互（warfarin） | 療效與出血 | 分開服用；監測 INR |
