# Penicillin

> [!abstract] 一句話：**β-lactam** 抗生素的元老，靠抑制**細胞壁合成（PBP / transpeptidase）**殺菌；值班最要記的是**過敏（含 anaphylaxis）**、**依腎功能調劑量（Oxacillin 例外走肝）**，以及各次類的抗菌譜差異。

> [!danger] 🚨 值班紅旗
> 1. **給藥後數分鐘–1 小時內蕁麻疹/喉頭水腫/低血壓/喘鳴 → Anaphylaxis**：立即停藥、**IM Adrenaline 0.3–0.5 mg（1:1000）大腿外側**、氧氣/輸液/呼吸道，別只給抗組織胺。
> 2. **問清「青黴素過敏」是哪種**：真正 IgE 型 vs 只是腸胃不適/家族史；標籤化過敏會被迫用更廣效/更貴抗生素。與 cephalosporin 交叉過敏率低（尤其側鏈不同者）。
> 3. **Neurosyphilis / IE 用高劑量 Penicillin G（2–4 MU IV Q4H）**——別給錯成長效 IM benzathine（後者穿不進 CNS）。
> 4. **腎功能不全要減量**（癲癇/腦病變風險，尤其高劑量 PCN-G）；**Oxacillin/Nafcillin 走肝代謝不需調腎**。
> 5. **Tazocin（Piperacillin-tazobactam）鈉含量高**（~2.79 mEq Na/g），心衰/低血鈉/體液過載病人注意。

## 📖 定義 / 病生理（機轉 / 分類）
- **機轉**：β-lactam ring 結合 **penicillin-binding proteins (PBP / transpeptidase)** → 阻斷細胞壁 peptidoglycan 交聯 → 對**分裂中細菌殺菌（time-dependent，療效看 T>MIC）**。
- **抗藥機轉**：① 細菌產生 **β-lactamase / penicillinase** 水解環（→ 加 β-lactamase inhibitor 或用 penicillinase-resistant PCN）② PBP 改變（如 MRSA 的 PBP2a、肺炎鏈球菌）③ 通透性/外排。
- **分類（依抗菌譜演進，詳見下方治療區銀蛋口袋書表格）**：

| 次類 | 代表藥 | 抗菌譜重點 |
| --- | --- | --- |
| 天然 PCN | Penicillin G/V、Benzathine | Strep、螺旋體（梅毒）、厭氧（口腔）、Listeria |
| Penicillinase-resistant | Oxacillin、Cloxacillin、Dicloxacillin | **MSSA 首選** |
| Aminopenicillin | Ampicillin、Amoxicillin (±Sulbactam/Clavulanate) | 加 GNB、Listeria、Enterococcus、H. influenzae |
| Antipseudomonal | Piperacillin-tazobactam | 加 **Pseudomonas** + 厭氧 + Amp-C GNB |

## 🩺 抗菌譜 / 臨床適應症（概覽）
> 詳細各藥 spectrum / dosage 見下方 💊 治療區銀蛋口袋書表格與 PDF 摘錄
- 天然 PCN：[[Syphilis(梅毒)]]（首選、含 neurosyphilis）、Group A Strep（[[Cellulitis(蜂窩性組織炎)]]、咽炎、風濕熱預防）、[[Infective Endocarditis(感染性心內膜炎)]]（敏感 strep）、Actinomycosis。
- 抗葡萄球菌 PCN：**MSSA** 菌血症/軟組織/骨關節。
- Aminopenicillin：Listeria [[Meningitis(腦膜炎)]]、Enterococcus、上呼吸道感染、H. pylori（amoxicillin 組合）、Unasyn 用於咬傷/腹腔/A. baumannii。
- Antipseudomonal：[[Hospital-Acquired Pneumonia(院內型肺炎)]]、腹腔內感染、發燒性嗜中性球低下（FN，合併 aminoglycoside）。

## 🔬 使用前評估 / 監測
> 用藥前查：過敏史 [[Drug allergy(藥物過敏)]]、腎功能（CrCl）
- **過敏史釐清**（IgE 型 vs 非免疫型），必要時 penicillin skin test / 減敏（梅毒孕婦首選仍是 PCN）。
- **腎功能**（多數需依 CrCl 調劑量；Oxacillin/Nafcillin 例外）。
- 長療程/高劑量：監測 CBC（骨髓抑制/嗜中性球低下）、肝功能（Oxacillin）、電解質（Tazocin 鈉負荷）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **Ampicillin/Amoxicillin + EBV 傳染性單核球增多症 → maculopapular rash**（不代表真過敏，是經典考點）。
- **Oxacillin 是唯一主要肝代謝、可致肝炎的 penicillin**，不需依腎調整。
- **不需依腎功能調整的抗生素**（肝代謝）：Oxacillin、Ceftriaxone、Moxifloxacin、Clindamycin、Tigecycline、Linezolid（見下方紅字 PDF）。
- Piperacillin-tazobactam 抗 PsA **遜於 Cefepime**；重症/高 MIC 用**延長輸注**。
- β-lactam 是 **time-dependent** 殺菌 → 靠增加給藥頻率/延長輸注提高 T>MIC，而非單次大劑量。

## ⚠️ 副作用 / 併發症
- **過敏**：從皮疹到 **anaphylaxis**；少數 SJS/TEN、間質性腎炎、血清病樣反應、drug fever。
- **血液**：骨髓抑制、嗜中性球低下（長療程/高劑量）、Coombs 陽性溶血。
- **腸胃**：腹瀉、**CDAD（困難梭菌）**。
- **神經**：高劑量 + 腎功能不全 → 癲癇/腦病變。
- **肝**：Oxacillin 肝炎。
- **電解質**：Tazocin 鈉負荷（低血鈉/體液過載者注意）。

## 📊 風險分層（過敏 / 劑量調整）
> 用藥前先分「過敏型別」與「腎/肝代謝」定安全用法
| 判斷點 | 用途 | 意義 |
| --- | --- | --- |
| 過敏型別（IgE 速發 vs 非免疫型/家族史） | 是否可用/需否減敏 | 真 IgE 型 anaphylaxis → 避用；含糊標籤 → 釐清避免濫升階抗生素 |
| 代謝途徑（腎 vs 肝） | 劑量調整 | 多數依 CrCl 調整；**Oxacillin/Nafcillin 走肝不需調腎** |
| 鈉負荷（Tazocin） | 體液/電解質風險 | 心衰/低血鈉/體液過載慎用 |
