# Fever of Unknown Origin（不明熱）

> [!abstract] 一句話：**反覆發燒 >38.3°C、持續 >3 週、經適當初步檢查仍找不到原因**。四大類病因：**感染、惡性腫瘤、結締組織/自體發炎、其他雜項**。核心不是「趕快投抗生素」，而是**系統性地找線索**（history/PE/影像/切片）。

> [!danger] 🚨 值班紅旗
> 1. **FUO 不等於「先給廣效抗生素」**——盲目投藥會遮蔽診斷、拖延病因；除非血流動力不穩/嗜中性球低下才經驗性治療
> 2. **嗜中性球低下（neutropenic）FUO 是例外** → 需**立即經驗性抗綠膿抗生素**（發燒性嗜中性球低下走另一套流程）
> 3. **反覆問 history + 反覆理學檢查**：旅遊、動物接觸、藥物、職業、性行為、家族、新雜音/淋巴結/皮疹——線索常在重複檢查時才浮現
> 4. 別漏**藥物熱**（drug fever）、**深部膿瘍**、**感染性心內膜炎**、**顳動脈炎（老年）**、**淋巴瘤**

## 📖 定義 / 病生理（這是什麼）
- **經典 Petersdorf-Beeson 定義**：① 體溫 **>38.3°C（101°F）多次**、② 持續 **>3 週**、③ 住院 1 週（現代改為 **3 天住院 或 3 次門診 或 1 週智慧型門診檢查**）仍未確診。
- **四大分類（現代 Durack-Street）**：
  - **classic**（一般族群）
  - **nosocomial**（住院 ≥48h 發生、入院時無感染徵象）— 想導管、C. difficile、藥物熱、血栓、鼻竇炎
  - **neutropenic / immunodeficient**（嗜中性球低下）— 想伺機性感染、黴菌
  - **HIV-associated**
- **四大病因群（classic FUO）**：
  1. **感染**（膿瘍、結核、心內膜炎、骨髓炎、EBV/CMV、地方性感染）
  2. **惡性腫瘤**（淋巴瘤、白血病、腎細胞癌、肝轉移）
  3. **結締組織/自體發炎**（成人 Still、巨細胞動脈炎/PMR、SLE、血管炎）
  4. **其他雜項**（藥物熱、DVT/PE、亞急性甲狀腺炎、family periodic fever、factitious）
- 隨檢查進步，**「undiagnosed」比例上升**（很多良性自限），且**老年人以結締組織病（尤 GCA）**比例高。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Fever(發燒)]]、[[Weight loss(體重減輕)]]、[[Lymphadenopathy(淋巴結腫大)]]
- 發燒型態（持續/間歇/回歸熱——參考價值有限但偶有提示）
- **系統性線索**：體重減輕、盜汗、皮疹、關節痛、淋巴結腫、肝脾腫大、新雜音、眼部、顳動脈壓痛
- **反覆理學檢查**找新出現徵象（如新雜音→IE、皮膚栓塞病灶、關節炎）

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Blood culture(血液培養)]]、[[CBC(全血球計數)]]、[[ESR/CRP]]　影像 [[CT(電腦斷層)]]、[[PET-CT]]
- **策略：從線索（PDC, potentially diagnostic clues）出發，而非亂槍打鳥。**
- **初階必做**：CBC/DC + 抹片、生化肝腎、ESR/CRP、尿檢+培養、**血液培養 ≥3 套（未用抗生素時）**、胸腹影像、HIV、結核檢查（IGRA/培養）、自體抗體（ANA、RF）、LDH、蛋白電泳、CMV/EBV、鐵蛋白（成人 Still 極高）。
- **進階**：胸腹骨盆 CT 找膿瘍/淋巴結/腫瘤、心臟超音波（IE）、**18F-FDG PET-CT（找隱匿發炎/腫瘤/血管炎的高產出工具）**、切片（淋巴結/骨髓/顳動脈/肝）。
- **停用非必要藥物**測試藥物熱；停抗生素以利培養。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **PET-CT 對 FUO 診斷產出高**，尤其揪出大血管炎、隱匿膿瘍、淋巴瘤。
- **老年新發頭痛 + 視覺症狀 + ESR 很高 → 巨細胞動脈炎（GCA）**，疑似即先給類固醇再切片（防失明）。
- **鐵蛋白極高（>1000, 甚至數千）+ 關節痛/咽痛/rash/白血球高 → 成人 Still 病**（[[Adult onset Still’s disease(成人型史迪爾症候群)]]）。
- **藥物熱**易漏：可無皮疹、可有相對心搏過緩（脈溫分離）；停藥後 48–72h 退燒是線索。
- **不要因焦慮而經驗性抗生素/類固醇**（除非重症/嗜中性球低下/疑 GCA）——會遮蔽診斷。
- factitious fever（人為）少見但要想（醫護背景、體溫與脈搏/外觀不符）。

## 💊 治療（Acute / Chronic）
> 用藥 [[抗生素]]（僅特定情況）
- **原則：找到病因再針對性治療**，多數 classic FUO 不需盲目經驗性抗生素。
- **例外經驗性治療**：① 血流動力不穩/敗血症徵象；② **嗜中性球低下發燒**（立即抗綠膿抗生素）；③ 疑 **GCA**（先類固醇防失明）；④ 疑培養陰性心內膜炎/結核之特定情境（依證據）。
- **診斷未明且穩定** → 密切追蹤觀察，避免亂投藥（很多良性自限，預後佳）。

## ⚠️ 併發症
- **延誤診斷**導致可治病因（膿瘍、IE、GCA、淋巴瘤、結核）進展
- 盲目經驗治療的副作用 + 遮蔽診斷
- 依最終病因而定（腫瘤/血管炎/感染各有其併發）

## 📊 嚴重度評分 / 風險分層（scoring）
> FUO 無單一 score；靠「分類 + 線索導向」與宿主免疫狀態分層
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **FUO 分類** | 決定路徑 | classic / nosocomial / neutropenic / HIV，路徑與病因群不同 |
| **PDC（診斷線索）導向** | 檢查排序 | 有線索先追線索，無線索走系統性 tier |
| **宿主免疫狀態** | 是否經驗治療 | neutropenic → 立即經驗抗生素；immunocompetent classic → 先找病因 |
