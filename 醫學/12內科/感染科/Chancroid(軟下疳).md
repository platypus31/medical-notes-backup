# Chancroid（軟下疳）

> [!abstract] 一句話：Haemophilus ducreyi 引起的**疼痛性生殖器潰瘍 + 疼痛化膿性腹股溝淋巴腫（bubo）**——與梅毒無痛硬下疳、HSV 群聚淺潰瘍構成「生殖器潰瘍三大鑑別」；台灣罕見但為 STI 潰瘍 DDx 必考。

> [!danger] 🚨 值班紅旗
> 1. **疼痛性潰瘍 + 疼痛化膿性淋巴腫 = 軟下疳典型**；「**痛且軟**」對比梅毒的「**不痛且硬**」。
> 2. **生殖器潰瘍 = 一律先排除梅毒與 HSV**（暗視野/血清 + HSV PCR），**且務必驗 HIV**（潰瘍增加 HIV 傳播）。
> 3. **可與梅毒/HSV 共感染** → 單一診斷不能安心，需完整潰瘍 STI panel。
> 4. **化膿的 bubo 需針抽引流**（避免破裂形成竇道），不建議切開引流。
> 5. HIV+ 或未包皮環切者**治療反應較差**，需追蹤與可能延長療程。

## 📖 定義 / 病生理（這是什麼）
- **病原體**：**Haemophilus ducreyi**（革蘭氏陰性、需 X 因子、成鏈狀「school of fish / railroad track」排列的桿菌，培養困難）。
- **機轉**：經皮膚黏膜微小破損進入，局部產生毒素造成組織壞死性潰瘍；引流至腹股溝淋巴結形成化膿性淋巴腫。
- **流行病學**：好發於**非洲、加勒比海、東南亞**部分地區；已開發國家與台灣**罕見**（診斷須高度懷疑 + 排除他因）。潛伏期 **3–10 天**。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Genital ulcer(生殖器潰瘍)]]、[[Lymphadenopathy(淋巴結腫大)]]
- **潰瘍**：起於柔軟丘疹/膿疱 → **疼痛、邊緣不規則、底部有灰黃色壞死性膿性滲出、質地軟、易出血**；常**多發**（自體接種）。
- **腹股溝淋巴腫（bubo）**：約半數，**單側、疼痛、化膿**，可破裂流膿形成竇道。
- 男性好發於**包皮/冠狀溝/陰莖體**；女性好發於**陰唇/陰道口/會陰**（女性症狀可較不明顯）。
- 一般**無全身症狀**。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Dark field microscopy(暗視野顯微鏡)]]、[[HSV PCR(單純疱疹PCR)]]、[[NAAT(核酸增幅檢測)]]
- **臨床診斷（CDC probable 標準，需全部符合）**：
  1. 一或多個疼痛性生殖器潰瘍；
  2. 潰瘍表現與（有無）局部淋巴腫**與典型軟下疳一致**；
  3. **暗視野 / 梅毒血清（潰瘍出現 ≥ 7 天後）排除梅毒**；
  4. **潰瘍液 HSV PCR / 培養陰性**。
- **確診**：H. ducreyi 培養（特殊培養基，敏感度 < 80%，多數實驗室無法做）；或 NAAT（研究用，非常規）。
- **必驗 HIV + 梅毒血清**（同時、並於 3 個月後複驗）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **生殖器潰瘍三大鑑別（高頻考點）**：

| 特徵 | 軟下疳 Chancroid | 梅毒硬下疳 Syphilis (chancre) | 生殖器 HSV |
|------|-----------------|------------------------------|-----------|
| 疼痛 | **痛** | **不痛** | **痛** |
| 潰瘍質地/邊緣 | 軟、邊緣不規則、壞死 | **硬（indurated）**、邊緣整齊乾淨 | 群聚小水泡 → 淺潰瘍 |
| 數目 | 常多發 | 常單一 | 群聚多發 |
| 淋巴結 | **疼痛化膿性（bubo，會破）** | **無痛、橡皮硬、不化膿** | 疼痛、雙側柔軟 |
| 病原 | H. ducreyi | Treponema pallidum | HSV-1/2 |
- **LGV**（chlamydia L 型）：潰瘍小/無痛而**淋巴腫明顯**，DDx 需納入。
- **Behçet、固定型藥疹、癌性潰瘍**為非感染性 DDx。
- **潰瘍會增加 HIV 傳播 3 倍以上** → 一律驗 HIV 並衛教。

## 💊 治療（Acute / Chronic）
> 用藥 [[Azithromycin(阿奇黴素)]]、[[Ceftriaxone(頭孢曲松)]]、[[Ciprofloxacin(環丙沙星)]]
依 **CDC STI Treatment Guidelines 2021**（任一）：
- **Azithromycin 1 g PO 單次**，或
- **Ceftriaxone 250 mg IM 單次**，或
- **Ciprofloxacin 500 mg PO BID × 3 天**，或
- **Erythromycin base 500 mg PO TID × 7 天**。
- **治療反應**：潰瘍通常 **3 天內症狀改善、7 天內客觀好轉**；大潰瘍可能需 > 2 週癒合。
- **bubo（化膿淋巴腫）**：**針抽（needle aspiration）** 引流，避免切開（切開易延遲癒合/形成竇道）。
- **HIV+ / 未包皮環切者**：反應較慢，需密切追蹤，可能需延長療程。
- **性伴侶**：症狀前 10 天內有性接觸者，無論有無症狀皆需**檢查與經驗性治療**。
- **禁性行為至潰瘍完全癒合**。

## ⚠️ 併發症
- **bubo 破裂 → 竇道 / 慢性引流性潰瘍**。
- 大範圍組織壞死、繼發細菌感染。
- **包皮嵌頓（phimosis/paraphimosis）**。
- **增加 HIV 感染與傳播風險**。

## 📊 嚴重度評分 / 風險分層（scoring）
- 無專屬 scoring。風險分層邏輯：
  - **有無化膿性 bubo** → 決定是否需針抽引流。
  - **HIV 狀態 / 是否包皮環切** → 預測治療反應與追蹤密度。
  - **潰瘍大小/範圍** → 影響癒合時間與是否延長療程。
