# Gray baby syndrome（灰嬰症候群）

> [!abstract] 一句話：**新生兒／早產兒使用 Chloramphenicol（氯黴素）時，因肝臟 glucuronidation（UGT）與腎臟排泄尚未成熟 → 藥物累積 → 粒線體毒性 → 循環衰竭**的致命藥物不良反應。值班關鍵一句：**新生兒用氯黴素前務必減量並監測血中濃度；出現灰藍發紺 + 低血壓 → 立刻停藥**。

> [!danger] 🚨 值班紅旗
> 1. **新生兒（尤其 <1 個月、早產兒）使用氯黴素 24–48 小時內**出現**灰藍色皮膚（cyanosis）、腹脹嘔吐、餵食差、低體溫、低血壓、呼吸急促、代謝性酸中毒** → 想到 gray baby syndrome，**立即停用氯黴素**。
> 2. **機轉是藥物累積**：新生兒 UGT 不成熟無法把氯黴素 glucuronide 化 + 腎排泄差 → 血中濃度飆高 → 抑制粒線體蛋白合成/氧化磷酸化 → 心血管虛脫。
> 3. **處置**：停藥 + 支持性照護（呼吸循環支持）；嚴重高濃度可考慮**換血（exchange transfusion）或血液灌流（charcoal hemoperfusion）**加速清除。
> 4. **預防勝於治療**：新生兒應盡量避免氯黴素；必須用時**大幅減量、監測血中濃度（維持治療範圍）、監測 CBC**。

## 📖 定義 / 病生理（這是什麼）
- **本質**：氯黴素在新生兒體內**累積性中毒**，不是感染本身惡化。
- **兩大代謝缺陷（新生兒生理不成熟）**：
  | 缺陷 | 後果 |
  | --- | --- |
  | **肝 UGT（glucuronosyltransferase）活性低** | 無法將氯黴素接合成無毒 glucuronide |
  | **腎小球濾過/排泄尚未成熟** | 未接合的活性藥物排不掉、持續累積 |
- **毒性機轉**：高濃度氯黴素抑制**粒線體核糖體（70S）蛋白質合成**與氧化磷酸化 → 細胞能量衰竭 → 心肌與血管功能障礙 → 休克。
- **與骨髓抑制的區別**：氯黴素另有 dose-dependent 可逆骨髓抑制與罕見 idiosyncratic aplastic anemia，機轉不同於 gray baby syndrome（後者是累積性粒線體毒性）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Cyanosis(發紺)]]、[[Hypotension(低血壓)]]
- 用藥後**數小時至數天**（常 24–48 小時）漸進出現：
  - **灰藍色皮膚（ashen-gray cyanosis）**
  - 腹脹、嘔吐、拒食
  - **低體溫、低血壓、循環衰竭**
  - 呼吸不規則/急促、發紺
  - 肌張力低下、嗜睡 → 進展至休克死亡

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Chloramphenicol level(氯黴素血中濃度)]]、[[Arterial blood gas(動脈血氣體分析)]]
- **臨床診斷**：新生兒 + 氯黴素用藥史 + 典型症狀。
- **氯黴素血中濃度升高**（超出治療範圍）支持診斷。
- **動脈血氣**：代謝性酸中毒（lactate 升高，反映粒線體毒性/組織缺氧）。
- 需排除敗血症、先天性心臟病、先天代謝異常等其他新生兒休克原因。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **經典國考點**：新生兒缺乏 glucuronidation 能力 → 氯黴素累積 → gray baby syndrome（同理新生兒對多種需 glucuronide 化的藥物敏感）。
- 灰嬰症候群是**累積性劑量毒性**（可預防），與**idiosyncratic aplastic anemia**（罕見、不可預測、與劑量無關）是兩回事。
- 不只新生兒——**高劑量下的較大孩童/成人**理論上也可能出現類似循環毒性。
- 現代已少用氯黴素（有較安全替代藥），但在特殊感染/資源受限地區仍可能遇到。

## 💊 治療（Acute / Chronic）
> 用藥 [[Chloramphenicol(氯黴素)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 急性（已發生）
- **立即停用氯黴素**。
- **支持性治療**：呼吸循環支持、矯正酸中毒、維持體溫與血壓。
- **嚴重高濃度**：可考慮**換血（exchange transfusion）**或**活性碳血液灌流（charcoal hemoperfusion）**加速藥物清除。
### 預防（重點）
- 新生兒**盡量避免**氯黴素，優先選擇較安全的替代抗生素。
- 必須使用時：**依日齡/體重大幅減量**、**監測血中濃度**維持治療窗、監測 CBC。

## ⚠️ 併發症
- 循環衰竭 / 休克 / 死亡（未及時停藥）
- 代謝性酸中毒、多重器官灌流不足
- （氯黴素其他毒性）劑量相關骨髓抑制、罕見再生不良性貧血、視神經炎（長期）

## 📊 嚴重度評分 / 風險分層（scoring）
> 無正式評分；以症狀嚴重度＋血中濃度決定介入強度
| 分層 | 特徵 | 處置 |
| --- | --- | --- |
| 早期/輕度 | 餵食差、輕度腹脹、濃度偏高 | 停藥、密切監測、支持 |
| 進展/重度 | 灰藍發紺、低血壓、酸中毒 | 加護支持、換血/血液灌流 |
