# Sexually transmitted diseases（性傳播疾病）

> [!abstract] 一句話：一組經性行為傳播的感染，臨床上以**症候群（生殖器潰瘍 / 尿道-子宮頸炎 / 分泌物 / 骨盆腔炎）**分類經驗治療；值班重點——**淋病經驗性一律「涵蓋披衣菌」雙覆蓋，且所有 STD 都要驗 HIV／梅毒、追蹤性伴侶。**

> [!danger] 🚨 值班紅旗
> 1. **骨盆腔炎（PID）門檻要低**：年輕女性下腹痛 + 子宮頸搖痛（cervical motion tenderness）→ 直接經驗治療，延誤 → 不孕、子宮外孕、Fitz-Hugh-Curtis。
> 2. **播散性淋菌感染（DGI）**：發燒 + 多關節痛／腱鞘炎 + 膿疱疹三聯 → 需住院 IV ceftriaxone，別當單純關節炎。
> 3. **淋病治療一律「ceftriaxone + 涵蓋披衣菌」**（共感染率高、抗藥性上升）；單用 azithromycin 治淋病已不建議。
> 4. **神經梅毒 / 眼梅毒 / 耳梅毒**任何期別皆可發生 → 視力改變、聽損、腦膜炎症狀要做腰椎穿刺，治療改 **IV penicillin G**。
> 5. 所有新診斷 STD → **同時篩 HIV + 梅毒 + B/C 肝**；HIV 有**空窗期**，高風險暴露考慮 4th-gen Ag/Ab 併 RNA 及重驗。急性 HIV 表現如單核球增多症樣。

## 📖 定義 / 病生理（這是什麼）
- 依**臨床症候群**分類最實用（病原常重疊，經驗治療照症候群走）：

| 症候群 | 主要病原 |
| --- | --- |
| **生殖器潰瘍** | 梅毒（*T. pallidum*，無痛硬下疳）、HSV（多發疼痛水泡／潰瘍）、軟性下疳（*H. ducreyi*，痛）、LGV（*C. trachomatis* L1-3） |
| **尿道炎 / 子宮頸炎** | 淋病（*N. gonorrhoeae*）、披衣菌（*C. trachomatis* D-K）、*Mycoplasma genitalium*、滴蟲 |
| **陰道炎 / 分泌物** | 滴蟲（*Trichomonas*）、細菌性陰道炎（BV，非典型 STD）、念珠菌 |
| **骨盆腔炎（PID）** | 淋病 + 披衣菌 + 厭氧菌混合上行感染 |
| **疣 / 贅生物** | HPV（尖形濕疣 / 子宮頸癌）、傳染性軟疣 |
| **全身性** | HIV、梅毒（多期）、B/C 型肝炎 |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Genital ulcer(生殖器潰瘍)]]、[[Urethral discharge(尿道分泌物)]]
- **男性尿道炎**：排尿灼痛、尿道分泌物（淋病膿性、披衣菌黏液樣、常無症狀）。
- **女性子宮頸炎／PID**：分泌物異常、性交後出血、下腹痛、發燒、子宮頸搖痛、附件壓痛。
- **潰瘍**：梅毒硬下疳（無痛、硬、乾淨）vs HSV（多發、痛、水泡）vs 軟性下疳（痛、軟、化膿）。
- **梅毒二期**：全身斑丘疹（**含手掌腳掌**）、扁平濕疣（condyloma lata）、黏膜斑、全身淋巴結腫。
- **DGI**：遊走性多關節痛、腱鞘炎、膿疱／出血性皮疹。
- **急性 HIV**：發燒、咽痛、皮疹、淋巴結腫（似 EBV mononucleosis）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[NAAT(核酸增幅檢驗)]]、[[VDRL-RPR(梅毒血清)]]　影像 [[Pelvic ultrasound(骨盆超音波)]]
- **淋病 / 披衣菌**：**NAAT（核酸增幅）**為首選（尿液、子宮頸／尿道、咽喉、直腸依暴露部位取樣）；淋病加做**培養 + 藥敏**（抗藥性監測）。
- **梅毒**：非特異（**RPR / VDRL**，追療效與活動性）+ 特異（**TPHA / TPPA / FTA-ABS**）**雙軌**判讀；懷疑神經梅毒做 **CSF-VDRL**。
- **HSV**：病灶 PCR（首選）／培養；血清型別分型輔助。
- **HIV**：第 4 代 Ag/Ab 篩檢 + 確認；急性期加 HIV RNA。
- **滴蟲 / BV**：濕片、pH、whiff test、NAAT。
- PID：臨床診斷為主 + 超音波排除卵巢輸卵管膿瘍（TOA）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **手掌腳掌皮疹** → 想二期梅毒（與 RMSF、腸病毒鑑別）。
- **淋病與披衣菌共感染率高** → 治淋病永遠雙覆蓋。
- 梅毒治療後 **Jarisch-Herxheimer 反應**（數小時內發燒、寒顫、肌痛，非過敏，支持治療即可）。
- **RPR 假陽性**：懷孕、自體免疫（抗磷脂質）、其他感染 → 需特異性試驗確認。
- **prozone 效應**：二期梅毒抗體過高導致 RPR 假陰，稀釋後轉陽。
- 反覆 STD、直腸／咽部感染 → 主動提供 HIV PrEP 衛教與伴侶追蹤。

## 💊 治療（Acute / Chronic）
> 用藥 [[Ceftriaxone(頭孢曲松)]]、[[Doxycycline(去氧羥四環素)]]、[[Benzathine penicillin(苄星青黴素)]]
> ⚠️ 劑量依最新 CDC/台灣指引，用藥前查藥典
- **淋病（無併發）**：**Ceftriaxone 500 mg IM 單劑**（≥150 kg 用 1 g）+ 若未排除披衣菌加 **doxycycline 100 mg BID × 7 天**。
- **披衣菌**：**Doxycycline 100 mg BID × 7 天**（優於 azithromycin，尤其直腸）；孕婦用 azithromycin 1 g 單劑。
- **梅毒**：早期（一／二／早期潛伏）**Benzathine penicillin G 240 萬單位 IM 單劑**；晚期潛伏／三期 → **每週一次 × 3 週**；**神經／眼／耳梅毒 → aqueous penicillin G IV 10–14 天**。青黴素過敏孕婦需**減敏**。
- **HSV**：acyclovir / valacyclovir（初發、復發或抑制療法）。
- **PID（門診）**：ceftriaxone 500 mg IM + doxycycline 100 mg BID × 14 天 + metronidazole；**住院**：IV cefotetan/cefoxitin + doxycycline。
- **滴蟲**：metronidazole（女性 500 mg BID × 7 天）；**伴侶同治**。
- **DGI**：住院 ceftriaxone 1 g IV/IM q24h。

## ⚠️ 併發症
- **PID → 不孕、子宮外孕、慢性骨盆痛、TOA、Fitz-Hugh-Curtis 症候群（肝周圍炎）**
- **梅毒 → 神經梅毒、心血管梅毒（[[Syphilitic aortitis(梅毒動脈炎)]]）、樹膠腫、先天性梅毒**
- **淋病 → DGI、關節炎、（新生兒）眼炎**
- **HPV → 子宮頸／肛門／口咽癌**
- HIV → AIDS 及機會性感染

## 📊 嚴重度評分 / 風險分層（scoring）
> STD 無單一 score；以症候群與併發風險分層
| 分層 | 用途 | 意義 |
| --- | --- | --- |
| PID 住院 indication | 決定門診 vs 住院 | 懷孕、TOA、無法口服／服從性差、經口治療 48–72h 無反應、外科急症無法排除 |
| 梅毒期別 | 決定療程與 LP | 神經／眼／耳症狀、HIV 併梅毒、RPR 高 → CSF 檢查 |
