# Q fever（Q 熱 / 查詢熱，非流感）

> [!warning] 命名更正：本檔名「Q型流感」為誤植。**Q fever = Q 熱（Query fever / 查詢熱）**，是由**立克次體類細胞內細菌 *Coxiella burnetii*** 引起的**人畜共通感染**，**與流行性感冒（influenza）無關**，非病毒性流感。

> [!abstract] 一句話：接觸牛羊胎盤/羊水/氣溶膠感染的 *Coxiella burnetii*；急性以非典型肺炎與肝炎、慢性以**培養陰性心內膜炎**為主——can't-miss：牛羊接觸史 + 培養陰性心內膜炎（尤其有瓣膜病 / 人工瓣膜者）。

> [!danger] 🚨 值班紅旗
> 1. **牛/羊/山羊接觸、農牧/獸醫/屠宰職業 + 不明發燒 + 非典型肺炎或肝炎** → 想到急性 Q 熱。
> 2. **培養陰性心內膜炎（culture-negative endocarditis）**：既有瓣膜病 / 人工瓣膜 + 反覆發燒，Q 熱是重要且易漏的病因——標準血培養**不會長**。
> 3. **孕婦**感染可致流產、早產、胎死——屬高風險族群，需特別處置。
> 4. 慢性 Q 熱可潛伏數月至數年後才發病（心內膜炎、血管感染 / 感染性動脈瘤）。

## 📖 定義 / 病生理
- **病原體**：***Coxiella burnetii***——**專性細胞內革蘭氏陰性菌**（立克次體目 Coxiellaceae），可形成**芽孢樣（spore-like）耐久型**，環境中極穩定、抗乾燥、**極低感染劑量（單顆菌即可）**，屬**潛在生物恐怖劑（Category B）**。
- **傳播途徑**：
  - **氣溶膠吸入**（主要）：受污染的牛/羊/山羊**胎盤、羊水、分泌物、糞尿**乾燥後隨風擴散。
  - 接觸產畜、未消毒乳製品（次要）；人傳人罕見。
- **抗原相變異（phase variation）**：**Phase II** 抗原為急性感染主要免疫目標；**Phase I** 抗體高效價提示**慢性感染**——為血清學分期關鍵。
- **病生理**：吸入後於巨噬細胞內增殖 → 急性全身感染；宿主有瓣膜病 / 免疫低下 / 血管病變 / 懷孕時易轉為慢性持續感染。

## 🩺 臨床表現
> 連症狀卡 [[發燒]]、[[非典型肺炎]]、[[肝炎]]
- **急性 Q 熱**（潛伏約 2–3 週，約半數無症狀）：
  - 突發高燒、劇烈頭痛、肌痛、畏光——似流感樣（此為「Q型流感」誤名來源，但病原截然不同）。
  - **非典型肺炎（atypical pneumonia）**、**肉芽腫性肝炎（granulomatous hepatitis，"doughnut" 環狀肉芽腫）**。
- **慢性 Q 熱**（<5% 急性者進展，數月至數年後）：
  - **心內膜炎（最常見且致命）**：培養陰性、常見於既有瓣膜病 / 人工瓣膜。
  - 血管感染（感染性動脈瘤 / 人工血管感染）、慢性骨髓炎、慢性肝炎。
- **孕婦**：流產、早產、低出生體重、胎死。

## 🔬 診斷
> 檢查 血清學（IFA）　PCR　超音波心圖
- **血清學（IFA 間接免疫螢光）為主**（*C. burnetii* **不做常規培養** — 需 BSL-3）：
  - **急性**：**Phase II IgG/IgM** 上升（配對血清 4 倍上升或單次高效價）。
  - **慢性**：**Phase I IgG 高效價（典型 ≥1:800）**是心內膜炎的重要指標。
- **PCR**：早期（血清尚未轉陽）或組織 / 瓣膜檢體偵測 *C. burnetii* DNA。
- **急性期線索**：肝功能異常、輕度血小板下降。
- **心內膜炎評估**：經食道超音波心圖；Q 熱已納入**改良 Duke criteria**（單次 Phase I IgG ≥1:800 或血清/PCR 陽性為主要條件）。

## ⭐ 特徵 / 注意（pearls & pitfalls）
- **「培養陰性心內膜炎」務必想到 Q 熱與 Bartonella、Brucella**；標準血培養陰性不代表無菌。
- **Phase I vs Phase II 抗體是分辨慢性 vs 急性的關鍵**：Phase II 主導＝急性；Phase I 高效價＝慢性。
- 診斷主要靠**血清學/PCR**，不是培養（*C. burnetii* 危險且難培養）。
- 慢性 Q 熱療程極長（心內膜炎常需 ≥18 個月），且需追蹤 Phase I 抗體效價下降判斷療效。
- 急性感染後高風險族群（瓣膜病、孕婦、免疫低下）應追蹤血清學以早期偵測慢性化。

## 💊 治療（Acute / Chronic）
> 用藥 [[Doxycycline]]、[[Hydroxychloroquine]]
- **急性 Q 熱**：**Doxycycline** 為首選，**依指引療程**（一般約 2 週）；有症狀者及早給藥效果最佳。
- **慢性 Q 熱（心內膜炎 / 血管感染）**：**Doxycycline + Hydroxychloroquine 長期合併**（hydroxychloroquine 鹼化吞噬溶酶體、增強 doxycycline 殺菌力），療程常 **≥18 個月**，並依 **Phase I IgG 效價下降**監測療效。
- **孕婦**：doxycycline 相對禁忌，依指引常用 **trimethoprim-sulfamethoxazole**（依產科感染科建議）。
- 心內膜炎合併瓣膜破壞時需心臟外科評估換瓣。

## ⚠️ 併發症
- 慢性 Q 熱心內膜炎（未治療致命）、瓣膜破壞、心衰竭。
- 感染性動脈瘤 / 人工血管感染破裂。
- 慢性疲勞後 Q 熱症候群（post-Q fever fatigue syndrome）。
- 孕婦不良妊娠結局。

## 📊 嚴重度評分 / 風險分層
- 分期依血清學：**急性（Phase II 主導）vs 慢性（Phase I IgG 高效價）**。
- 心內膜炎診斷用**改良 Duke criteria**（Q 熱有專屬主要條件）。
- 慢性化高風險族群：既有瓣膜病 / 人工瓣膜、血管移植 / 動脈瘤、免疫抑制、孕婦。
