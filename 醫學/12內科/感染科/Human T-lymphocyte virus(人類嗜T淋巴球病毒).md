# Human T-lymphotropic Virus type 1（人類嗜T淋巴球病毒 HTLV-1）

> [!abstract] 一句話：反轉錄病毒（retrovirus），感染 CD4 T 細胞終身潛伏；**大多終生無症狀，少數 → 成人 T 細胞白血病 / 淋巴瘤（ATLL）或脊髓病變（HAM/TSP）**；日本西南、加勒比海、非洲地方性流行。

> [!danger] 🚨 值班紅旗
> 1. **ATLL 急性型**：高鈣血症（hypercalcemia）+ 溶骨病灶 + 白血球暴增 + 花瓣核細胞（flower cell）+ 皮膚 / 肝脾腫大 → 進展快、預後極差，須查鈣、緊急處理高血鈣。
> 2. ATLL 病人細胞免疫缺損 → **伺機性感染（尤 Strongyloides 糞小桿線蟲過度感染、PCP、CMV）**，發燒時廣查。
> 3. **HAM/TSP**：緩慢進行的痙攣性下肢麻痺 + 膀胱功能障礙 → 別誤當單純退化 / MS。
> 4. 輸血 / 器官移植 / 母乳 傳染源篩檢（地方性流行區）。

## 📖 定義 / 病生理（這是什麼）
- **病原體**：Retroviridae，Deltaretrovirus，單股 RNA 反轉錄病毒；HTLV-1 為主要致病型（HTLV-2 致病性弱）。
- **傳染途徑**：**母乳（母嬰垂直，最主要）**、性接觸、血液 / 輸血、共用針具；感染後終身帶原。
- **病生理**：病毒整合入 CD4 T 細胞基因組，**Tax、HBZ 病毒蛋白**驅動 T 細胞增殖與基因不穩定 → 數十年潛伏後少數轉變為單株惡性增生（ATLL）；或引發自體免疫 / 發炎性脊髓損傷（HAM/TSP）。
- 終生罹 ATLL 風險約 2-5%、HAM/TSP <5%。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[淋巴結腫大]]、[[下肢無力]]、[[高鈣血症]]
- **多數：無症狀帶原者（asymptomatic carrier）**。
- **ATLL（成人 T 細胞白血病 / 淋巴瘤）**四亞型：
  - 急性型 / 淋巴瘤型：全身淋巴結腫大、肝脾腫、皮膚浸潤、高鈣、溶骨、白血病相。
  - 慢性型 / 悶燒型（smoldering）：皮膚病灶為主、病程較緩。
- **HAM/TSP（HTLV-1-associated myelopathy / tropical spastic paraparesis）**：對稱性**痙攣性下肢輕癱**、步態障礙、膀胱 / 腸道 / 性功能障礙、下背痛，緩慢進展。
- 其他：HTLV-1 相關葡萄膜炎、感染性皮膚炎、多發性肌炎。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[HTLV-1 抗體 ELISA/Western blot]]、[[周邊血液抹片]]、[[血鈣]]
- **感染診斷**：血清 anti-HTLV-1 抗體（ELISA 篩檢 → Western blot 確認）；PCR 測 proviral DNA。
- **ATLL**：周邊血 / 組織可見特徵性**「花瓣細胞（flower cell）」多葉核淋巴球**；免疫表現 CD4+ CD25+；證實**單株 HTLV-1 整合**；查血鈣、LDH、骨骼影像。
- **HAM/TSP**：血清 + CSF 皆有 HTLV-1 抗體、CSF 淋巴球增多；MRI 可見脊髓萎縮 / 白質病變；排除其他脊髓病因。

## ⭐ 特徵 / 注意（pearls & pitfalls）
- 記憶：**一個病毒兩大病 — ATLL（血液惡性）+ HAM/TSP（神經）**；同一病毒既致癌又致自體免疫。
- **Flower cell + 高鈣血症** = ATLL 經典組合。
- ATLL 病人啟動化療 / 類固醇前務必**篩檢並預防性驅蟲 Strongyloides**，避免致命的過度感染症候群。
- 地方性流行區（日本九州 / 沖繩、加勒比海、南美、西非、澳洲原住民）病史很重要。
- 母乳為主要垂直傳染途徑 → 帶原母親於流行區建議避免長期哺餵。

## 💊 治療（Acute / Chronic）
> 用藥 [[Zidovudine]]、[[Interferon-alpha]]
- **無症狀帶原**：不治療，衛教避免傳播（不捐血 / 器官、安全性行為、母乳建議）。
- **ATLL**：
  - 急性 / 淋巴瘤型：多藥化療（如 CHOP 類）；**Zidovudine (AZT) + Interferon-α** 對白血病型有效；異體造血幹細胞移植為潛在治癒選項；抗 CCR4 單株抗體 mogamulizumab（日本核准）。
  - 處理高鈣血症（水化、bisphosphonate）。
- **HAM/TSP**：無根治，**類固醇**緩解急性惡化、症狀治療（痙攣、膀胱），復健。
- 無疫苗；預防靠阻斷傳播與血品篩檢。

## ⚠️ 併發症
- ATLL 高鈣危象、伺機性感染（Strongyloides hyperinfection、PCP）、化療抗藥、預後差（急性型中位存活數月）。
- HAM/TSP 進行性下肢癱瘓、輪椅依賴、泌尿併發症。

## 📊 嚴重度評分 / 風險分層
> ATLL 依 **Shimoyama 分類**分急性 / 淋巴瘤 / 慢性 / 悶燒四型 → 急性與淋巴瘤型預後最差、需積極治療；慢性 / 悶燒型可觀察或較緩處置。
