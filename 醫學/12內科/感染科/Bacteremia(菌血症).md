# Bacteremia（菌血症）

> [!abstract] 一句話：**血液中存在活菌**（血液培養陽性）。本身是「檢驗發現」，臨床意義取決於**菌種 + 宿主 + 是否有持續感染源**；**S. aureus / GNR 幾乎都有意義**，CoNS 常是污染。

> [!danger] 🚨 值班紅旗
> 1. **Staphylococcus aureus 菌血症（SAB）= 從不當污染看待** → 一律找感染源、做心臟超音波（排 IE）、48–72h **重抽血培養確認清除**、ID 會診
> 2. **持續菌血症（follow-up 培養仍陽性）** → 想 endocarditis、感染性血栓、未引流膿瘍、感染的人工物（導管/人工瓣/關節）
> 3. **有中心靜脈導管 + 菌血症** → 考慮 CRBSI，多數需**拔除導管**（尤 S. aureus / Candida / 綠膿桿菌）
> 4. **抽血培養一定在給抗生素前**、**≥2 套不同部位**；單套陽性 CoNS 多為污染，臨床判讀勿過度治療也勿漏真感染

## 📖 定義 / 病生理（這是什麼）
- **菌血症**：血中有活菌，經血液培養證實。
- **時間型態**：
  - **短暫性（transient）**：刷牙、拔牙、內視鏡、膿瘍操作後短暫入血，多自清（見 [[Transient Bacteremia(短暫性菌血症)]]）
  - **間歇性（intermittent）**：膿瘍週期性入血
  - **持續性（continuous）**：血管內感染（IE、感染性血栓、感染的人工物）—— **持續菌血症的特徵**
- **入血途徑**：黏膜/皮膚屏障破壞、局部感染擴散、血管內裝置、腸道移轉。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Fever(發燒)]]、[[Chills(寒顫)]]
- 發燒、寒顫（rigors，尤其菌血症陣發時）、心搏過速
- 可完全無症狀（污染或短暫性），也可進展為 [[Sepsis(敗血症)]]、[[Septic shock(敗血性休克)]]
- 找**轉移性感染灶**：新雜音（IE）、關節腫痛、脊椎壓痛（[[Vertebral Osteomyelitis(脊柱骨髓炎)]]）、皮膚栓塞病灶

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Blood culture(血液培養)]]、[[CBC(全血球計數)]]
- **血液培養 ≥2 套（好氧+厭氧），不同穿刺部位、給抗生素前**；適當採血量最影響陽性率。
- **真感染 vs 污染判讀**：
  - 幾乎都真：S. aureus、GNR（如 E. coli/Klebsiella/Pseudomonas）、Candida、肺炎鏈球菌
  - 多為污染：CoNS（除非有人工物）、Corynebacterium、Bacillus、Cutibacterium
  - 佐證真感染：多套陽性、同菌種、time-to-positivity 短、臨床相符
- **SAB / candidemia** → 追加：心臟超音波（TTE→必要時 TEE）、找轉移灶影像、**follow-up 血培養**。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **CoNS 單套陽性**多為污染，但**有人工瓣膜/導管/關節**時要當真。
- **SAB 的四要件（照顧品質指標）**：ID 會診、追蹤血培養確認清除、心臟超音波、找並處理源頭 → 顯著降低死亡率。
- **Candida / S. aureus / 綠膿桿菌**菌血症合併中心導管 → 傾向拔管。
- **腸道菌（如 Streptococcus gallolyticus / bovis、Clostridium septicum）菌血症** → 找大腸腫瘤（安排大腸鏡）。

## 💊 治療（Acute / Chronic）
> 用藥 [[抗生素]]、[[Anti-MRSA]]
- **經驗性抗生素**：依疑似來源 + 宿主 + 當地抗藥性，涵蓋最可能菌種（重症/MRSA 風險加 vancomycin；GNR 風險加抗綠膿 β-lactam）。
- **降階**：培養+感受性出來後窄化。
- **源頭控制**：拔感染導管、引流膿瘍、清創、處理人工物。
- **療程**：依菌種與有無併發。**SAB**：無併發（source 已控、無 IE/轉移灶、追蹤培養陰、去除導管）至少 **14 天 IV**；有併發（IE、轉移灶、金屬植入）**4–6 週以上**。GNR 菌血症無併發者常 7–14 天（部分可較短），視臨床。

## ⚠️ 併發症
- **感染性心內膜炎**（[[Infective Endocarditis(感染性心內膜炎)]]）、轉移性膿瘍、脊椎骨髓炎、化膿性關節炎
- 進展為 [[Sepsis(敗血症)]] / [[Septic shock(敗血性休克)]]、DIC
- 人工物感染、感染性血栓（septic thrombophlebitis）

## 📊 嚴重度評分 / 風險分層（scoring）
> 菌血症本身無單一 score；以「有無器官障礙」與「菌種/來源」分層
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **qSOFA / SOFA** | 是否進展為 sepsis | ≥2 → 高風險，比照敗血症處理 |
| **Pitt bacteremia score** | 菌血症預後 | 綜合體溫/血壓/意識/呼吸/心跳，分數越高死亡率越高 |
| **污染 vs 真感染判讀** | 是否治療 | 依菌種、套數、time-to-positivity、臨床 |
