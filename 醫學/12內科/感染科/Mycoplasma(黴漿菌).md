# Mycoplasma（黴漿菌 / Mycoplasma pneumoniae）

> [!abstract] 一句話：**無細胞壁**的最小自營細菌，*M. pneumoniae* 是**社區型非典型肺炎（atypical / walking pneumonia）**最常見病原，好發於學齡兒童與年輕成人；特徵是「**症狀輕但影像重、乾咳久、胸片浸潤明顯**」，且因無細胞壁 → **β-lactam 完全無效，首選 macrolide / doxycycline / fluoroquinolone**。

> [!danger] 🚨 值班紅旗
> 1. **β-lactam（penicillin/cephalosporin）對黴漿菌完全無效**——非典型肺炎經驗性治療要涵蓋非典型菌（macrolide 或 doxycycline / respiratory FQ）。
> 2. **肺外併發症**：Stevens-Johnson 樣黏膜炎（**MIRM, Mycoplasma-induced rash and mucositis**）、腦炎、橫斷性脊髓炎、自體免疫溶血（**冷凝集素 cold agglutinin**）、心肌炎——年輕肺炎病人出現神經 / 皮膚 / 溶血症狀要想到。
> 3. **免疫溶血性貧血**：冷凝集素引起，寒冷加重、末梢發紺、血紅素驟降。
> 4. 大環內酯抗藥性（macrolide-resistant *M. pneumoniae*）在部分地區（尤東亞）上升 → 治療反應不佳時改 doxycycline / fluoroquinolone。

## 📖 定義 / 病生理（這是什麼）
- **病原體**：*Mycoplasma pneumoniae*，屬 Mollicutes；**無細胞壁**（故革蘭氏染色不顯色、對抑制細胞壁合成的抗生素天然抗藥），是能自我複製的最小細菌。
- **傳播途徑**：**飛沫傳播**，人與人密切接觸（家庭、學校、軍營等群聚），潛伏期約 1–3 週；全年可見、每 3–7 年有社區流行高峰。
- **好發族群**：**學齡兒童與年輕成人**（<40 歲）最常見，但各年齡皆可。
- **病生理**：黏附於呼吸道上皮（P1 adhesin）→ 產生過氧化物損傷纖毛上皮 → 持續性乾咳；並以**分子模擬（molecular mimicry）** 誘發交叉反應性免疫，造成多種**免疫介導的肺外表現**（冷凝集素、神經、皮膚黏膜）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Cough(咳嗽)]]、[[Fever(發燒)]]、[[Sore throat(喉嚨痛)]]
- **呼吸道**：漸進性發燒、頭痛、倦怠、咽痛，續以**頑固性乾咳（後期可少量痰）**，可持續數週。
- **「Walking pneumonia」**：全身症狀相對輕微，病人常仍能活動，但**胸片浸潤程度不成比例地明顯**（症狀-影像不符）。
- **理學**：肺部聽診常僅輕微囉音，與影像不成比例；可有咽紅、頸淋巴結。
- **上呼吸道 / 其他**：咽炎、支氣管炎、耳膜炎（bullous myringitis，經典但不常見）、氣喘惡化。
- **肺外表現**：
  - 皮膚黏膜：紅斑 / 多形性紅斑、**MIRM（黏膜為主的類 SJS）**。
  - 血液：**冷凝集素引起自體免疫溶血性貧血**。
  - 神經：腦炎、無菌性腦膜炎、橫斷性脊髓炎、Guillain-Barré、小腦失調。
  - 心臟：心肌炎、心包炎；關節痛。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Chest X-ray(胸部X光)]]、[[PCR(核酸檢測)]]、[[Cold agglutinin(冷凝集素)]]
- **臨床診斷為主**（非典型肺炎表現 + 年齡 + 群聚）；微生物確診用於重症 / 不典型。
- **胸部 X 光**：常見**單側下葉間質 / 網狀浸潤或斑片狀實質化**，程度常大於理學所見；可少量肋膜積液。
- **NAAT / PCR（呼吸道檢體）**：靈敏且快速，為現行首選確診工具。
- **血清學**：急性 + 恢復期 IgM/IgG 配對（4 倍上升），IgM 於急性期可陽性但兒童 / 再感染判讀受限。
- **冷凝集素（cold agglutinins）**：非特異但支持診斷，高效價與溶血相關。
- **培養**：極慢、不常規使用。
- **DDx**：其他非典型肺炎（*Chlamydophila pneumoniae*、*Legionella*、病毒）、典型細菌性肺炎（肺炎鏈球菌）、百日咳、結核。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **記憶點**：無細胞壁 → **β-lactam 無效**；症狀輕、影像重 → walking pneumonia；肺外要想到**冷凝集素溶血、神經、MIRM**。
- **冷凝集素**是經典考點：IgM 型，寒冷誘發末梢凝集 / 溶血。
- **bullous myringitis**（大疱性鼓膜炎）雖不常見，是古典教科書關聯徵象。
- **macrolide 抗藥性上升**（尤東亞）：治療 48–72 小時無改善時考慮換 doxycycline / 呼吸道 fluoroquinolone。
- 兒童使用 doxycycline 短療程一般可接受（現行觀點對牙齒著色風險已放寬），但 fluoroquinolone 兒童慎用——依指引與年齡選藥。
- 其他致病 Mycoplasma：***M. genitalium***（尿道炎 / 子宮頸炎，性傳染）、***Ureaplasma***——與 *M. pneumoniae* 不同臨床範疇。

## 💊 治療（Acute / Chronic）
> 用藥 [[Azithromycin(阿奇黴素)]]、[[Doxycycline(去氧羥四環素)]]、[[Levofloxacin(左氧氟沙星)]]
（⚠️確切劑量 / 療程依現行指引與年齡 / 抗藥性）：
- **首選（門診 / 輕中度）**：
  - **Macrolide**：**Azithromycin**（如成人 500 mg day 1 後 250 mg × 4 天）或 clarithromycin。
  - **替代 / macrolide 抗藥區**：**Doxycycline 100 mg PO BID**。
- **成人重症 / 需住院 / 抗藥**：**呼吸道 fluoroquinolone**（**Levofloxacin 750 mg qd** 或 moxifloxacin）。
- **無效的藥**：所有 β-lactam（penicillin、cephalosporin）、vancomycin——皆作用於細胞壁，對無壁的黴漿菌無效。
- **支持治療**：退燒、止咳、補水；重症溶血 / 神經併發症依情況加免疫調節（類固醇 / IVIG，證據多為個案）。
- **療程**：一般 5–10 天，依藥物與嚴重度。

## ⚠️ 併發症
- 呼吸：重症肺炎 / ARDS（少見）、肋膜積液、氣喘惡化、閉塞性細支氣管炎。
- 血液：冷凝集素性溶血性貧血。
- 神經：腦炎、橫斷性脊髓炎、GBS、小腦炎（可留後遺）。
- 皮膚黏膜：MIRM / 多形性紅斑 / SJS 樣病變。
- 心臟：心肌炎、心包炎；罕見腎炎、關節炎。
