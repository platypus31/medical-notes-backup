# Tuberculosis（結核病）

> [!abstract] 一句話：**Mycobacterium tuberculosis** 經飛沫核（droplet nuclei）空氣傳播的慢性肉芽腫感染，最常侵犯**肺尖**；值班關鍵——**慢性咳嗽 >2–3 週 + 體重減輕 / 夜汗 / 咳血 → 疑 TB 就先「呼吸道隔離（負壓）」再驗痰**，並記得它是第三類法定傳染病、要通報。

> [!danger] 🚨 值班紅旗
> 1. **疑似活動性肺結核 → 立即負壓隔離 + N95**，別讓病人在急診／病房到處走（保護醫護與其他病人）。
> 2. **粟粒性 TB（miliary）／結核性腦膜炎（TBM）是急症**：意識改變、頸僵硬、腦神經麻痺 → 立即 LP，**治療不要等培養**（TBM 延誤 = 高死亡/失能）。
> 3. **大咳血（massive hemoptysis）**：舊 TB 空洞 + Rasmussen aneurysm 或 aspergilloma → 氣道保護、患側朝下、IR 支氣管動脈栓塞。
> 4. **HIV 共感染**：表現不典型（下肺葉、無空洞、肺外多）→ CD4 低者門檻要更低；注意抗病毒與抗結核**藥物交互作用（rifampin）**與 **IRIS**。
> 5. **開始治療前查基線肝功能、視力、尿酸、聽力**（四藥都有毒性）；INH 併用 **B6（pyridoxine）**預防周邊神經病變。

## 📖 定義 / 病生理（這是什麼）
- **病原**：*Mycobacterium tuberculosis*（抗酸菌，緩慢生長、需氧、細胞內）。
- **傳播**：吸入含菌**飛沫核**（空氣傳播）；巨噬細胞吞噬後 → **細胞媒介（Th1 / IFN-γ / TNF）肉芽腫（caseating granuloma）**局限或潛伏。
- **潛伏結核（LTBI）vs 活動性**：多數感染後成 LTBI（IGRA/TST ＋、無症狀、不具傳染力），約 5–10% 終生活化（免疫低下大增）。
- **再活化好發肺尖**（含氧高、有利需氧菌）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Chronic cough(慢性咳嗽)]]、[[Hemoptysis(咳血)]]、[[Night sweats(夜間盜汗)]]
- **肺結核**：慢性咳嗽（>2–3 週）、咳痰／咳血、發燒、**夜汗、體重減輕、食慾不振**（消耗性 B symptoms）。
- **肺外結核（約 15–20%，HIV 更高）**：淋巴結（scrofula）、胸膜炎／積液、TB 腦膜炎、Pott's disease（脊椎，見 [[Vertebral Osteomyelitis(脊柱骨髓炎)]]）、泌尿生殖、腹膜、粟粒性播散。

| | 發病 | 表現 | CXR |
| ---------------------- | ------- | ------------------------- | ------------------------------------ |
| Primary TB | 通常在幼年 | 在中下肺葉形成granuloma | Ghon complex：Ghon focus + hilar LAPs |
| Progressive primary TB | AIDS或老人 | 從中下肺葉lesion血行擴散，嚴重的甚至肺外結核 | Miliary TB |
| Reactivation TB | 免疫變差時 | 鈣化內殘餘的菌跑出，尤其在上肺葉 | 鈣化邊緣模糊，出現新的浸潤或cavitation |

| 分類 | |
| ------ | ------------ |
| 疑似結核個案 | 咳嗽超過三週，有症狀懷疑 |
| 結核病個案 | 細菌學證實或由醫師診斷 |
| 確診結核個案 | 培養陽性 |

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Sputum AFB smear(痰抗酸菌抹片)]]、[[TB culture(結核菌培養)]]、[[IGRA(丙型干擾素釋放試驗)]]　影像 [[Chest X-ray(胸部X光)]]、[[Chest CT(胸部電腦斷層)]]
- **痰抗酸菌抹片 ×3（AFB smear）**：敏感度低但快、評估**傳染力**；抹片陽性 = 高傳染性。
- **痰培養**：**金標準**（含藥敏），但需 2–8 週。
- **NAA（核酸增幅，如 Xpert MTB/RIF）**：數小時、可區分 TB/NTM、同時測 **rifampin 抗藥性**，臨床高度懷疑者優先。
- **影像**：上肺葉浸潤／空洞、粟粒狀、鈣化肉芽腫；CT 更敏感。
- **LTBI 篩檢**：**IGRA（不受 BCG 干擾，優於 TST）**或 TST（≥10 mm，免疫低下 ≥5 mm）；僅代表曾感染，**不能確診活動性**。
- 肺外：組織切片（caseating granuloma）+ 培養、CSF（TBM：淋巴球↑、蛋白↑↑、糖↓、ADA↑）、pleural ADA。

| 方法 | |
| -------------------------------- | ------------------------------------------- |
| Acid fast stain | （1~2天）5000~10000隻/ml才會陽性 |
| Culture | （2~8週）10~100隻/ml即可培養出來 |
| Nucleic acid amplification (NAA) | （數小時）1~10隻/ml有反應，可以分NTM/TB，臨牀高度懷疑的病人可以考慮NAA |

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **抹片陰性不排除 TB**（需 5000–10000 隻/ml 才陽性）；臨床高度懷疑仍要培養 + NAA + 治療。
- **上肺葉空洞 + 慢性消耗症狀** = 經典再活化 TB。
- **HIV / 免疫低下表現不典型**（下肺葉、無空洞、肺外、抹片陰性）→ 門檻要低。
- **NTM（非結核分枝桿菌）**與 TB 抹片都抗酸菌陽性 → NAA 或培養鑑別（治療完全不同）。
- **抗結核藥前先查基線**：肝功能（INH/RIF/PZA 肝毒）、視力與色覺（EMB）、尿酸（PZA）、聽力（SM）。
- 開始有效治療約 2 週且臨床改善、連續痰抹片轉陰 → 傳染力大降。

## 💊 治療（Acute / Chronic）
> 用藥 [[Isoniazid(INH)]]、[[Rifampin(RMP)]]、[[Ethambutol(EMB)]]、[[Pyrazinamide(PZA)]]
> ⚠️ 劑量／療程依疾管署與最新指引，用藥前查藥典
- **標準初治（drug-susceptible）**：**強化期 HRZE × 2 個月 → 持續期 HR × 4 個月**（共 6 個月）。
- **INH 併 pyridoxine（B6）**預防周邊神經病變。
- **DOTS（直接觀察治療）**：確保服藥順從性、防抗藥。
- **TB 腦膜炎 / 骨結核**：療程延長（TBM 常 9–12 個月）+ **輔助類固醇（TBM、心包炎降低死亡/後遺）**。
- **潛伏結核（LTBI）**：INH 9 個月、或 INH+RIF、或 rifapentine+INH（3HP）短療程。
- **MDR-TB**：對 INH+RIF 皆抗藥，需多重二線藥（如 bedaquiline、linezolid、fluoroquinolone），療程長、需專科。

| 治療結果 | |
| ----------------------- | --------------------------------------------- |
| 治癒(cured) | 治療過程至少一次痰陰性 + 最後一個月治療時痰陰性 |
| 完治(treatment completed) | 完成治療但無法歸類治癒或失敗 |
| 失敗(failure) | (1)治療四個月培養陽性、五個月AFS陽性<br>(2)治療前細菌性陰性，治療兩個月後陽性 |

| 全名 | 副作用 | 特點 |
| -------------------- | ------------------------------------------------------ | --------- |
| Isoniazid(INH, H) | [[Peripheral Neuropathy(周邊神經病變)]](pyridoxine即Vit.B6預防) | 肝毒性 |
| Rifampin(RMP, R) | 橘色尿，類似感冒症狀，[[Hemolytic Anemia(溶血性貧血)]] | 肝毒性 |
| Ethambutol(EMB, E) | [[Optic Neuritis(視神經炎)]](小孩禁用) | 腎代謝，唯一抑菌性 |
| Pyrazinamide(PZA, Z) | [[Hyperuricemia(高尿酸)]] | 肝毒性 |
| Streptomycin(SM) | 耳毒性，腎毒性，孕婦不可 | 腎代謝 |
- HERZ2個月+ HER4個月
- 多重抗藥性(MDRTB)，指對Isoniazid及Rifampin有抗藥性。兩者合用要注意肝功能
- BCG可預防TB meningitis & miliary TB

> [!note] 補充藥物要點
> - **Rifampin 是強力 CYP450 誘導劑**：降低多種藥物濃度（口服避孕藥、warfarin、抗排斥藥、部分 HIV 抗病毒藥）→ 用藥前查交互作用。
> - **EMB 用藥前後追視力／色覺**（視神經炎劑量／時間相關，多可逆）。
> - **PZA** 用於強化期 2 個月；痛風活動期慎用。
> - 治療中規律驗肝功能（有症狀或基線異常者密集）。

## ⚠️ 併發症
- **咳血 / 大咳血**（空洞 Rasmussen aneurysm、續發 aspergilloma）
- 肺纖維化、支氣管擴張、毀損肺、慢性呼吸衰竭
- **粟粒性播散、TB 腦膜炎、Pott's disease、心包炎（縮窄性）**
- 藥物毒性：肝炎、視神經炎、周邊神經病變、痛風、耳/腎毒性
- HIV 共感染之 **IRIS**、抗藥性（MDR/XDR-TB）

## 📊 嚴重度評分 / 風險分層（scoring）
> TB 無單一嚴重度 score；用個案分類 + 肺外/播散/抗藥分層
| 分層 | 用途 | 意義 |
| --- | --- | --- |
| 抹片陽性/陰性 | 傳染力 | 抹片陽性 = 高傳染，需嚴格隔離 |
| 藥敏（DST / Xpert RIF） | 決定方案 | RIF 抗藥 → 疑 MDR，轉專科二線治療 |
| 肺外 / 粟粒 / TBM | 預後 | 播散與中樞侵犯 → 高死亡，療程延長 + 類固醇 |
| 宿主 | 風險 | HIV、糖尿病、免疫抑制、營養不良、洗腎 → 高活化與重症風險 |
