# Dermatophyte（皮癬菌）

> [!abstract] 一句話：一群**嗜角質（keratinophilic）黴菌**（Trichophyton、Microsporum、Epidermophyton），只侵犯角質層 / 毛髮 / 指甲等含角蛋白組織，造成各部位「癬（tinea）」；診斷靠 **KOH 顯微鏡看到菌絲（hyphae）**，多數體表可局部藥治，頭癬與甲癬需口服。

> [!danger] 🚨 值班紅旗
> 1. **免疫低下 / 糖尿病病人的足癬（tinea pedis）是蜂窩性組織炎的入侵門戶**——趾縫浸潤糜爛可讓細菌從破損處進入，處理下肢蜂窩織炎時務必檢查並治療足癬。
> 2. **看到「癬」形病灶前先問有沒有擦過類固醇**：外用類固醇造成 **tinea incognito**（病灶不典型、範圍擴大、鱗屑消失），易誤診誤治。
> 3. 頭癬（tinea capitis）合併紅腫化膿團塊 → **kerion**，是強烈發炎反應，需口服抗黴菌 ± 短期類固醇，延誤可致永久禿髮瘢痕。

## 📖 定義 / 病生理
- **皮癬菌 = 三屬黴菌**：**Trichophyton（毛癬菌）、Microsporum（小孢子菌）、Epidermophyton（表皮癬菌）**。
- **嗜角質**：分泌角蛋白酶，只侵犯角質層、毛髮、指甲，一般不侵入活組織 / 深層。
- 依來源分：**親人型（anthropophilic）**（人傳人、慢性、發炎輕）、**親動物型（zoophilic）**（動物傳人、發炎劇烈）、**親土型（geophilic）**。

## 🩺 臨床表現（各部位 tinea）
> 連症狀卡 [[皮疹]]、[[搔癢]]
- 典型病灶：**環狀（annular）、邊緣隆起脫屑、中央清亮（central clearing）** 的紅斑，搔癢。

| 部位 | 名稱 | 特徵 |
| --- | --- | --- |
| 頭皮 / 毛髮 | **Tinea capitis（頭癬）** | 兒童常見，落髮、脫屑、斷髮；重者 kerion；Microsporum 可 Wood's 燈綠螢光 |
| 軀幹 / 四肢 | **Tinea corporis（體癬）** | 環狀病灶、邊緣隆起、中央清亮（ringworm） |
| 腹股溝 | **Tinea cruris（股癬，jock itch）** | 大腿內側 / 腹股溝，多見男性，陰囊常倖免（與念珠菌不同） |
| 足 / 趾縫 | **Tinea pedis（足癬，香港腳）** | 趾縫浸潤脫皮、水疱型或莫卡辛型；蜂窩織炎門戶 |
| 指 / 趾甲 | **Tinea unguium / Onychomycosis（甲癬，灰指甲）** | 甲板增厚、變色、易碎、甲下角化 |
| 手 | Tinea manuum（手癬） | 常單手（one hand-two feet 綜合徵） |
| 臉 | Tinea faciei | 臉部環狀病灶，易被誤為濕疹 |

## 🔬 診斷（含診斷標準）
> 檢查 [[KOH]]、[[黴菌培養]]、[[Wood's lamp]]
- **KOH（氫氧化鉀）製片鏡檢**：刮取鱗屑加 KOH 溶解角質後看到**分節菌絲（septate branching hyphae）** → 確診首選、快速。
- **黴菌培養（Sabouraud dextrose agar）**：鑑定菌種，較慢（數週），治療頑固或需確認菌屬時用。
- **Wood's lamp（伍氏燈）**：部分 Microsporum 頭癬呈**綠色螢光**（Trichophyton tonsurans 不螢光），輔助但敏感度有限。
- 難治或不典型可做皮膚切片 PAS 染色。

## ⭐ 特徵 / 注意（pearls & pitfalls）
- **中央清亮 + 邊緣活躍脫屑** 是與濕疹、環狀肉芽腫的鑑別重點；刮片取**邊緣**陽性率最高。
- **Tinea incognito**：擦過外用類固醇後病灶變不典型（發炎減、範圍大、鱗屑少），最易誤診。
- 股癬「**陰囊倖免**」有助與念珠菌（candida，常侵陰囊、有衛星病灶）鑑別。
- **KOH 前勿先給口服抗黴菌**：會降低鏡檢與培養陽性率。

## 💊 治療（Acute / Chronic）
> 用藥 [[Terbinafine]]、[[Itraconazole]]、[[Griseofulvin]]、[[外用azole]]
- **體表侷限（corporis / cruris / pedis）→ 局部外用**：
  - 外用 **azole**（clotrimazole、ketoconazole、miconazole）或 **terbinafine 乳膏**，通常 2–4 週（症狀消退後續用 1–2 週）。
- **需口服的情況（局部藥難達 / 面積廣 / 免疫低下）**：
  - **Tinea capitis（頭癬）→ 必須口服**（外用無法穿透毛囊）：**terbinafine**（Trichophyton 佳）或 **griseofulvin**（Microsporum 佳，傳統首選）、itraconazole。
  - **Tinea unguium / 甲癬 → 口服為主**：**terbinafine**（趾甲療程約 12 週、指甲約 6 週為主流）或 itraconazole pulse。
  - 廣泛體癬 / 難治：口服 terbinafine 或 itraconazole。
- **注意肝毒性 / 藥物交互作用**（azole 抑制 CYP3A4；口服治療前後監測肝功能）。

## ⚠️ 併發症
- 續發細菌感染、蜂窩性組織炎（尤其足癬 + 糖尿病 / 淋巴水腫）。
- Kerion（頭癬強烈發炎）→ 瘢痕性禿髮。
- Id reaction（dermatophytid，遠端無菌性過敏疹）。
- 甲癬慢性化、復發率高。
