# Purulent Arthritis / Septic Arthritis（化膿性 / 敗血性關節炎）

> [!abstract] 一句話：細菌侵入關節腔造成的化膿性感染，是**內科兼骨科急症**，關節軟骨破壞以「小時」計，延遲引流與抗生素會導致永久失能，未治療死亡率可達 10-15%。

> [!danger] 🚨 值班紅旗
> 1. **急性單一關節紅腫熱痛 + 無法承重 / 主動被動活動皆劇痛** → 先當敗血性關節炎處理，不排除不放心。
> 2. **穿刺（arthrocentesis）是不能拖的第一步**：抗生素前先抽關節液送 Gram stain / 培養 / WBC 分類 / 結晶（除非會嚴重延誤治療）。
> 3. **關節液 WBC > 50,000/µL（多核球 > 90%）高度懷疑化膿性**；但無單一 cut-off 能完全排除，臨床高度懷疑即使 WBC 較低仍要治療。
> 4. **有結晶（痛風/假性痛風）不能排除同時合併感染** — 兩者可並存。
> 5. **人工關節（prosthetic joint）紅腫痛 → 照會骨科**，處置與自然關節不同（多需移除/更換）。
> 6. 年輕性活躍者 + 多關節痛/腱鞘炎/皮疹 → 想 **淋球菌（DGI）**。

## 📖 定義 / 病生理
細菌經以下途徑進入關節腔：
- **血行播散（最常見）**：菌血症時細菌沉積於滑膜血流豐富處。
- **直接接種**：穿刺、關節注射、手術、外傷。
- **鄰近擴散**：骨髓炎、蜂窩性組織炎波及。

滑膜無基底膜、血流豐富易被菌定殖；細菌 + 宿主發炎反應釋放蛋白酶與 cytokine，**48 小時內即可造成軟骨不可逆破壞**，故時效性極高。

**常見致病菌**
| 族群 | 主要菌 |
| --- | --- |
| 全年齡最常見 | **Staphylococcus aureus（含 MRSA）** |
| 次之 | Streptococcus（pyogenes、pneumoniae、agalactiae）|
| 年輕性活躍者 | **Neisseria gonorrhoeae（DGI，散播性淋病）** |
| 免疫低下 / IVDU / 老年 | Gram-negative bacilli（E. coli、Pseudomonas）|
| 人工關節早期 | S. aureus；晚期 coagulase-negative Staph（S. epidermidis）|
| 被貓狗咬 | Pasteurella multocida |

**危險因子**：既有關節疾病（RA、OA）、人工關節、糖尿病、免疫抑制、IVDU、菌血症、高齡、皮膚感染、關節注射史。

## 🩺 臨床表現
- **典型**：急性（數小時至數天）**單一大關節**（膝最常見 > 髖 > 肩 > 腕 / 踝）紅、腫、熱、痛，關節積液，主動與被動活動皆受限且劇痛。
- **全身**：發燒、寒顫（約半數有發燒，無發燒不能排除）。
- **淋球菌性（DGI）特殊表現**：常為**多關節游走性關節痛 + 腱鞘炎（tenosynovitis）+ 膿疱性皮疹**三聯徵，或化膿性單關節炎。
> 連症狀卡 [[Joint pain(關節痛)]]、[[Fever(發燒)]]

## 🔬 診斷
**關節穿刺（arthrocentesis）為診斷核心** — 抗生素前完成：
| 檢查 | 意義 |
| --- | --- |
| WBC 計數 + 分類 | > 50,000/µL 且 PMN > 90% 高度懷疑化膿性；> 100,000 更強；亦有較低值仍為感染者 |
| Gram stain | 陽性率約 30-50%（G+ cocci 較高、淋球菌低）|
| 培養（含厭氧）| 診斷金標準；懷疑淋球菌另送 Thayer-Martin / NAAT |
| 結晶檢查 | 排除痛風（負性雙折射針狀）/ 假性痛風（正性菱形），但不排除合併感染 |

**輔助**：血液培養（陽性率約 50%，尤其血行播散）、CBC、ESR / CRP（敏感但非特異）、懷疑淋球菌時採泌尿生殖道 / 咽 / 直腸檢體 NAAT。
**影像**：X-ray 早期多正常（用於基準與排除骨髓炎/骨折）；超音波定位積液引導抽吸；MRI 評估骨髓炎 / 深部關節（髖、SI joint）。
> 檢查 [[Arthrocentesis(關節穿刺)]]　影像 [[MRI]]

## ⭐ 特徵 / 注意（pearls & pitfalls）
- **「結晶陽性」≠「排除感染」** — 痛風合併化膿性關節炎並不罕見，兩者要各自判讀。
- 關節液 WBC **無絕對安全門檻**：臨床高度懷疑即使 WBC < 50,000 也要治療 / 追蹤。
- 免疫低下、人工關節、部分感染者**可無發燒或發炎指標偏低**，勿以此排除。
- **髖與肩關節**位置深、外觀腫脹不明顯，易延誤 — 靠影像導引穿刺。
- Gram stain 陰性**不能排除**感染，等培養並依經驗性治療。
- 反覆抽吸引流的量與頻率是療效指標；引流不足是治療失敗常見原因。

## 💊 治療（Acute / Chronic）
**兩大支柱：① 及時徹底引流　② 經驗性後續針對性抗生素**（劑量依腎功能與指引調整，不確定寫「依指引」）
- **引流**：反覆針抽、關節鏡沖洗清創，或開放手術引流；**人工關節、髖關節、引流不佳者傾向手術**。
- **經驗性抗生素**（先抽培養再給）：
  - 涵蓋 MRSA：**Vancomycin**（依 AUC/trough 調劑）。
  - 依 Gram stain / 危險因子加抗 Gram-negative（如第三代 cephalosporin ceftriaxone / ceftazidime、或抗綠膿桿菌 cefepime / piperacillin-tazobactam）。
  - 懷疑淋球菌：**Ceftriaxone**（依現行指引劑量）+ 針對可能合併之 Chlamydia 加 doxycycline / azithromycin。
- **降階（de-escalation）**：培養與藥敏出來後轉窄效藥。
- **療程**：一般 non-gonococcal 約 **3-4 週**（部分靜脈後轉口服）；淋球菌性療程較短（約 7-14 天）。人工關節感染療程更長且常需移除。
> 用藥 [[Vancomycin(萬古黴素)]]、[[Ceftriaxone]]

## ⚠️ 併發症
- 軟骨破壞 → **永久關節破壞、僵直、失能**。
- 骨髓炎（osteomyelitis）擴散。
- 敗血症 / 敗血性休克。
- 生長板受損（兒童）→ 肢體發育異常。
- 人工關節感染 → 假體鬆脫、需移除翻修。

## 📊 嚴重度評分 / 風險分層
> 敗血性關節炎無單一公認的 severity score；預後與時效及宿主相關。
| 工具 | 用途 | 重點 |
| --- | --- | --- |
| Newman criteria | 診斷分層（培養陽性 / Gram 陽性 / 化膿性關節液 + 危險因子）| 支持診斷 |
| Kocher criteria（兒童髖）| 兒童化膿性 vs 暫時性滑膜炎鑑別 | 發燒 > 38.5°C / 無法承重 / ESR > 40 / WBC > 12,000，符合越多機率越高 |
| qSOFA / SOFA | 合併敗血症時之全身風險 | 見 [[SOFA score]] |

**預後不良因子**：高齡、既有關節病 / RA、人工關節、延遲引流、Gram-negative 或 MRSA 感染、多關節受侵、免疫低下。
