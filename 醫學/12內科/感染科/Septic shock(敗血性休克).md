# Septic shock（敗血性休克）

> [!abstract] 一句話：敗血症的**最嚴重亞型** — 儘管**充分輸液**仍需**升壓劑維持 MAP ≥65** 且 **lactate >2 mmol/L**。本質是感染引起的**分布性休克 + 細胞代謝障礙**，院內死亡率 **>40%**。

> [!danger] 🚨 值班紅旗
> 1. **輸液後仍低血壓需升壓劑 + lactate >2** = septic shock，死亡率極高 → 儘早上 **norepinephrine**（不要一路灌水等血壓回來）
> 2. **norepinephrine 第一線升壓劑**，目標 **MAP ≥65 mmHg**；血壓危急可**周邊靜脈短暫給藥**同時準備中心靜脈導管，不要為等 CVC 而延誤
> 3. **持續升壓劑需求 → 加 hydrocortisone 200 mg/day**（IV，分次或連續輸注）
> 4. 別忘**源頭控制**：膿瘍要引流、感染導管要拔、壞死組織要清創——沒處理源頭，升壓劑再多也壓不住

## 📖 定義 / 病生理（這是什麼）
- **Sepsis-3 臨床定義**：敗血症患者，**充分輸液復甦後**仍需**升壓劑維持 MAP ≥65 mmHg** 且血清 **lactate >2 mmol/L**（>18 mg/dL）。此組合可辨識出死亡率 >40% 的高危族群。
- **病生理**：發炎介質（NO 過量等）→ 全身血管擴張 + 血管對兒茶酚胺反應性下降（vasoplegia）+ 微血管滲漏 → **相對與絕對血容不足** + 心肌抑制 + 微循環障礙 + 粒線體氧利用障礙（細胞層次休克，SvO2 可正常甚至偏高卻仍缺氧）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Hypotension(低血壓)]]、[[Altered mental status(意識改變)]]
- **暖休克**（早期）：四肢溫暖、脈搏洪大、脈壓寬（血管擴張）
- **冷休克**（晚期/心肌抑制）：四肢冰冷、皮膚花斑、微血管填充延遲
- 低血壓、少尿、意識改變、代謝性酸中毒（陰離子間隙↑、lactate↑）

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Lactate(乳酸)]]、[[Blood culture(血液培養)]]、[[ABG(動脈血氣)]]
- 同 [[Sepsis(敗血症)]] 的全套 + **強調 lactate 追蹤**與**血流動力評估**（動態指標：passive leg raise、pulse pressure variation、床邊超音波評估容積反應性與心功能）。
- 中心靜脈血氧飽和度（ScvO2）、CVP 為輔助（不再作單一目標，2021 SSC 已淡化 EGDT 固定數值）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **休克 ≠ 一定血壓很低才算**：升壓劑需求 + 高 lactate 就是 shock，別等 SBP 掉到很低才警覺。
- **過度輸液有害**：前期 30 mL/kg 後改用動態指標指引，避免肺水腫/組織水腫惡化預後。
- **norepinephrine 優於 dopamine**（dopamine 心律不整風險高、死亡率不利）。
- **類固醇**適應症是「持續升壓劑需求」，非所有 septic shock 常規給；不影響 source control 與抗生素優先。
- 難治性休克要重新想：**源頭沒控制？抗生素沒涵蓋？腎上腺不足？合併其他休克（心因/阻塞）？**

## 💊 治療（Acute / Chronic）
> 用藥 [[Norepinephrine(正腎上腺素)]]、[[Vasopressin(血管加壓素)]]、[[Hydrocortisone(氫化可體松)]]、[[抗生素]]
### 急性（在 Sepsis Hour-1 Bundle 基礎上）
- **輸液**：晶體液 30 mL/kg 起始，之後動態評估反應性再補
- **升壓劑階梯**：
  1. **Norepinephrine** 第一線，滴定至 MAP ≥65
  2. 劑量偏高仍不足 → 加 **Vasopressin**（固定劑量約 0.03 U/min，作為 catecholamine-sparing）
  3. 仍不足 → 加 **Epinephrine**；心輸出不足考慮 **dobutamine**（強心）
- **類固醇**：持續 norepinephrine ≥0.25 μg/kg/min 達數小時仍需要 → **Hydrocortisone 200 mg/day IV**
- **抗生素 + 源頭控制**同 Sepsis，休克者更強調 1 小時內給藥。
- 目標導向：MAP ≥65、lactate clearance、尿量恢復、末梢灌流改善。
### 慢性 / 恢復期
- 存活者 PICS、腎功能追蹤（部分需長期透析）、復健、再入院風險管理。

## ⚠️ 併發症
- **難治性休克 / MODS**、急性腎損傷、ARDS、**DIC**、腸/肢體缺血壞死
- 升壓劑相關：末梢/腸系膜缺血、心律不整
- 高死亡率（>40%），存活者長期失能

## 📊 嚴重度評分 / 風險分層（scoring）
> lactate 與升壓劑需求是床邊最實用的嚴重度指標
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **SOFA** | 器官障礙嚴重度 | 心血管項含 MAP/升壓劑劑量；分數越高死亡率越高 |
| **Lactate + clearance** | 灌流與預後 | >2 = shock 條件之一；未下降預後差 |
| **APACHE II** | ICU 預後 | 整體生理紊亂量化 |
