# Subacute Sclerosing Panencephalitis（亞急性硬化性全腦炎 SSPE）

> [!abstract] 一句話：**麻疹（measles）感染後數年**由變異麻疹病毒持續潛伏引起的**遲發性、進行性、致命性腦炎**；以認知退化 + 肌抽躍為特徵，MMR 疫苗可預防，預後極差。

> [!danger] 🚨 值班紅旗
> 1. 兒童 / 青少年**進行性認知退化 + 週期性肌抽躍（myoclonus）** + 曾罹麻疹（尤 2 歲前感染）→ 高度懷疑 SSPE。
> 2. 疾病**幾乎必然致死**（多在數月至數年內），無有效根治，及早診斷重在支持與家庭準備。
> 3. 未接種疫苗 / 疫苗接種率下降地區病例回升 → 麻疹疫情後數年的公衛警訊。
> 4. 早期易被誤診為學習障礙 / 行為問題 / 精神疾病 → 出現肌抽躍時務必查 EEG + 麻疹抗體。

## 📖 定義 / 病生理（這是什麼）
- **病原體**：**變異型麻疹病毒（defective measles virus，M 蛋白缺陷）** 於中樞神經系統持續潛伏感染。
- **潛伏期**：初次麻疹感染後平均 **7-10 年**（可 1 至數十年）遲發；**2 歲前感染麻疹風險最高**。
- **病生理**：缺陷病毒無法組裝完整病毒顆粒 → 在神經元 / 膠質細胞內慢性複製 → 慢性發炎、神經退化、脫髓鞘、膠質增生（sclerosing）→ 廣泛灰白質受損（panencephalitis）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[認知退化]]、[[肌抽躍]]、[[癲癇]]
分期（Jabbour 分期）：
- **第一期**：人格改變、行為問題、學業退步、認知下降（易被忽略）。
- **第二期**：**週期性肌抽躍（myoclonic jerks）**、抽搐、運動失調、視力障礙（視網膜炎 / 皮質性失明）。
- **第三期**：僵直、去皮質 / 去大腦姿勢、自主神經不穩、意識惡化。
- **第四期**：植物狀態、緘默、最終死亡。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[EEG]]、[[CSF 麻疹抗體]]、[[MRI]]
- **EEG**：特徵性 **Radermecker complex — 週期性同步高波幅慢波複合波**（與肌抽躍同步）。
- **CSF**：**麻疹病毒抗體（IgG）高效價**、寡株帶（oligoclonal band）、IgG 指數上升；細胞數多正常。
- **血清 / CSF 麻疹抗體比值上升**（鞘內合成）為關鍵。
- **MRI**：早期可正常，後期見皮質 / 皮質下白質 T2 高訊號、腦萎縮。
- 診斷需綜合：典型臨床 + EEG + 麻疹抗體證據。

## ⭐ 特徵 / 注意（pearls & pitfalls）
- 記憶：**麻疹病史 + 數年後 + 肌抽躍 + EEG 週期複合波 + CSF 麻疹抗體高** = SSPE。
- 與急性麻疹腦炎（ADEM，感染後數週）、進行性麻疹腦炎（免疫低下者）區分：SSPE 是**免疫正常者、潛伏多年後**發病。
- 疫苗株病毒**不會**引起 SSPE；反而普遍接種可大幅降低發生率（SSPE 個案幾乎皆為自然感染麻疹者）。
- 病程可波動甚至短暫穩定，但整體不可逆進行。

## 💊 治療（Acute / Chronic）
> 用藥 [[Isoprinosine]]、[[Interferon-alpha]]
- **無根治療法**；治療目標為延緩進展與症狀緩解。
- **免疫調節嘗試**（證據有限、非治癒）：口服 **Isoprinosine（inosine pranobex）** ± 鞘內 / 靜脈 **Interferon-α**，可能延緩部分個案病程。
- **症狀治療**：抗癲癇 / 抗肌抽躍藥（如 valproate、clonazepam）、痙攣處理、營養支持、感染防治、安寧照護。
- **預防（唯一有效手段）**：**MMR 疫苗**普及接種，杜絕自然麻疹感染即可預防 SSPE。

## ⚠️ 併發症
- 進行性神經退化至植物狀態、失明、頑固癲癇 / 肌抽躍。
- 吞嚥困難吸入性肺炎、營養不良、褥瘡、感染 → 常見死因。
- 疾病幾乎全數致死。

## 📊 嚴重度評分 / 風險分層
> 依 **Jabbour 臨床分期（I-IV）** 追蹤病程進展；分期越高預後越差，第四期為終末植物 / 死亡階段。
