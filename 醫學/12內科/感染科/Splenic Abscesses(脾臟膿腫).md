# Splenic Abscesses（脾臟膿腫）

> [!abstract] 一句話：脾臟內化膿性病灶，**最常來自遠處感染的血行播散——首要找源頭是「感染性心內膜炎」**；經典三聯 **發燒 + 左上腹痛 + 脾腫**齊全者少，值班要有意識去 image，否則延誤致敗血症／破裂而死亡率高。

> [!danger] 🚨 值班紅旗
> 1. **持續／反覆菌血症找不到源頭 + 左上腹痛 / 左側肋膈角壓痛 → 想脾膿腫，安排 CT**。
> 2. **同時查「有沒有感染性心內膜炎」**（最常見血行來源）——做血培養 ×3、心臟超音波（TEE）。
> 3. **破裂 → 腹膜炎 + 出血性休克**，是外科急症；血流動力不穩要立刻外科／IR。
> 4. 治療核心 = **抗生素 + 引流（IR 經皮或脾切除）**，單靠抗生素多不夠，尤其多發或大膿腔。
> 5. 免疫低下（糖尿病、HIV、化療、IVDU）、鐮刀型貧血、外傷後脾梗塞是高危背景。

## 📖 定義 / 病生理（這是什麼）
- **定義**：脾實質內局限性化膿感染，可單發或多發（micro-abscess）。
- **致病途徑**：
  - **血行播散（最常見）**：感染性心內膜炎、菌血症（其他部位感染灑到脾）。
  - **脾梗塞繼發感染**：鐮刀型貧血、栓塞、血管炎。
  - **鄰近蔓延 / 外傷後血腫感染**、免疫低下之伺機／黴菌感染。
- **常見菌**：
  - **細菌**：Streptococcus、Staphylococcus、腸道 G(-)（*E. coli*、Klebsiella）、厭氧菌；台灣／亞洲注意 **Klebsiella pneumoniae**（尤其糖尿病、可合併肝膿腫、內眼炎轉移症）。
  - **黴菌**：Candida（免疫低下、化療後 → 多發微膿腫，肝脾同時）。
  - 特殊：分枝桿菌、Bartonella、Melioidosis（*Burkholderia pseudomallei*，東南亞／台灣南部）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Fever(發燒)]]、[[Left upper quadrant pain(左上腹痛)]]
- **經典三聯**：發燒 + 左上腹痛 + 脾腫（三者齊全 <三分之一）。
- 左側肋膈角疼痛、放射到左肩（膈肌刺激）、左側胸膜積液、左下肺呼吸音減弱。
- 全身性：寒顫、盜汗、體重減輕（亞急性者）。
- 敗血症徵象；破裂則急性腹膜炎 + 休克。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Blood culture(血液培養)]]　影像 [[CT abdomen(腹部電腦斷層)]]、[[Echocardiography(心臟超音波)]]
- **腹部 CT（含顯影，首選）**：低密度病灶 ± 氣泡 ± 邊緣強化；辨單／多發、大小、破裂。
- **腹部超音波**：床邊、可導引引流。
- **血培養 ×3**（找菌 + 評估心內膜炎）；引流液送**培養（含厭氧、黴菌、抗酸）**。
- **心臟超音波（TEE 優於 TTE）**：找 vegetation（血行來源）。
- 檢驗：白血球↑、CRP/PCT↑、貧血；糖尿病者查血糖／HbA1c。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **脾膿腫 = 找心內膜炎的信號**，反之亦然——兩者互相提示。
- **多發微膿腫（肝脾同時）+ 免疫低下 → 想念珠菌（hepatosplenic candidiasis）**，常在 neutropenia 恢復期發燒。
- 亞洲糖尿病人 **Klebsiella 侵襲症候群**：肝／脾膿腫 + 內眼炎 + 腦膜炎，需積極眼底檢查。
- 東南亞旅遊 / 農務接觸 → 考慮 **melioidosis**。
- 脾切除後病人未來需 **encapsulated 菌預防接種**（肺炎鏈球菌、Hib、腦膜炎雙球菌）與衛教（OPSI 風險）。

## 💊 治療（Acute / Chronic）
> 用藥 [[Piperacillin-tazobactam(哌拉西林-他唑巴坦)]]、[[Vancomycin(萬古黴素)]]
> ⚠️ 劑量依最新指引與培養結果，用藥前查藥典
### 抗生素（經驗性，後依培養降階）
- 廣效涵蓋 G(+)/G(-)/厭氧：**piperacillin-tazobactam** 或 **carbapenem**；疑 MRSA 加 **vancomycin**；疑 Candida 加 **echinocandin / fluconazole**。
### 引流 / 手術（關鍵，抗生素單獨常不夠）
- **經皮導管引流（IR）**：單發、局限、內容物可引流者首選。
- **脾切除（splenectomy）**：多發、破裂、引流失敗、大膿腔、血流動力不穩者。
- 破裂 / 休克 → 緊急外科。
### 慢性 / 後續
- 依培養完成足量療程（通常數週）；治療原發源（心內膜炎、控糖）；脾切除後預防接種。

## ⚠️ 併發症
- **破裂 → 腹膜炎、出血性休克**（最致命）
- 敗血症 / 感染性休克、多器官衰竭
- 膈下膿瘍、左側膿胸／肺炎、胰／胃瘻管
- 脾切除後 **OPSI（overwhelming post-splenectomy infection）** 終生風險

## 📊 嚴重度評分 / 風險分層（scoring）
> 無專屬 score；用敗血症 + 病灶特徵分層
| 評分 / 因子 | 用途 | 意義 |
| --- | --- | --- |
| **qSOFA / SOFA** | 敗血症分層 | ≥2 → 高風險，積極引流 + ICU |
| 病灶特徵 | 決定引流 vs 手術 | 多發／破裂／大膿腔／引流失敗 → 傾向脾切除 |
| 宿主 | 預後 | 免疫低下、延遲診斷、Candida、破裂 → 死亡率上升 |
