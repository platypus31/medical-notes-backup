# Human Papillomavirus（人類乳突病毒 HPV）

> [!abstract] 一句話：非套膜 dsDNA 病毒，性接觸傳染最常見；**低危型（6/11）→ 生殖器疣（菜花），高危型（16/18）→ 子宮頸癌 / 口咽癌 / 肛門癌**，疫苗可有效預防。

> [!danger] 🚨 值班紅旗
> 1. 生殖器疣不是「良性小事」— 女性須轉介 **子宮頸抹片 / HPV DNA 篩檢**，避免漏掉共存的高危型病變（CIN）。
> 2. 免疫低下（HIV、器官移植、懷孕）者疣體易巨大化、頑固難治，且高危型進展快 → 積極追蹤。
> 3. **口咽鱗狀細胞癌（HPV-16）** 年輕、不抽菸族群無痛性頸部腫塊 → 別當單純淋巴結炎。
> 4. 兒童生殖器疣須警覺 **性侵害 / 母嬰垂直傳染** 可能。

## 📖 定義 / 病生理（這是什麼）
- **病原體**：Papillomaviridae，非套膜（non-enveloped）環狀雙股 DNA 病毒；已知 200+ 基因型，依致癌風險分低危 / 高危。
- **傳染途徑**：性接觸（皮膚黏膜）為主；亦可垂直傳染（產道 → 嬰兒喉乳突瘤 recurrent respiratory papillomatosis）、間接接觸。
- **致病機轉**：病毒進入基底層角質細胞 → 高危型 **E6 蛋白降解 p53、E7 蛋白抑制 Rb** → 細胞週期失控 → 癌化。多數感染 1-2 年內被免疫系統清除，持續感染才致病。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[生殖器潰瘍/腫塊]]、[[頸部腫塊]]
- **低危型（6/11）**：Condyloma acuminata 尖形濕疣（菜花）— 外陰、陰莖、肛周、會陰之柔軟疣狀突起；一般疣（尋常疣、足底疣、扁平疣）。
- **高危型（16/18/31/33/45…）**：多數無症狀，長期持續感染 → 子宮頸上皮內贅瘤（CIN）、子宮頸癌、肛門癌、陰道 / 外陰 / 陰莖癌、**口咽癌**。
- **RRP 喉乳突瘤**：兒童聲音沙啞、呼吸道阻塞。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Pap smear 子宮頸抹片]]、[[HPV DNA test]]
- **臨床視診**：典型菜花即可診斷；醋酸白化試驗（acetowhitening）輔助。
- **子宮頸篩檢**：Pap smear 細胞學 + **HPV DNA / mRNA 分型**（高危型偵測）；異常 → 陰道鏡 + 切片。
- **組織病理**：koilocyte（挖空細胞）是 HPV 感染特徵。
- 口咽癌：切片 + **p16 免疫染色**（HPV 相關 marker）。

## ⭐ 特徵 / 注意（pearls & pitfalls）
- **6/11 → 疣、16/18 → 癌**（記憶：低號碼低危、16/18 兩大高危）。16/18 約佔子宮頸癌 70%。
- Koilocyte + p16 陽性 = HPV 相關。
- 疫苗為**預防**性（無治療效果），已感染仍建議打（可防其他型別）；打疫苗不取代篩檢。
- HPV 相關口咽癌**預後優於**抽菸 / 喝酒相關者。

## 💊 治療（Acute / Chronic）
> 用藥 [[Imiquimod]]、[[Podophyllotoxin]]
- **生殖器疣**：
  - 病人自用：Imiquimod 5% cream（免疫調節劑）、Podophyllotoxin 0.5%、Sinecatechins 綠茶萃取。
  - 醫師處置：冷凍治療（cryotherapy 液態氮）、三氯醋酸（TCA）、電燒 / 雷射 / 手術切除。
  - 疣會復發，無根除病毒之藥。
- **癌前病變 / 癌**：依 CIN 分級行 LEEP / 冷凍 / 錐狀切除；侵襲癌走腫瘤科治療。
- **預防（最關鍵）**：**9 價疫苗 Gardasil-9**（涵蓋 6/11/16/18/31/33/45/52/58），建議 9-14 歲兩劑、15 歲以上或免疫低下三劑；男女皆宜。

## ⚠️ 併發症
- 子宮頸癌、肛門癌、口咽癌、陰莖 / 外陰 / 陰道癌。
- 兒童 RRP 呼吸道阻塞。
- 免疫低下者巨大 Buschke-Löwenstein 腫瘤。

## 📊 嚴重度評分 / 風險分層
> 子宮頸病變依 **Bethesda 系統**（ASC-US / LSIL / HSIL）+ CIN 1-3 分級決定追蹤 / 治療強度。
