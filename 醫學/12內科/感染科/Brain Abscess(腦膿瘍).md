# Brain Abscess（腦膿瘍）

> [!abstract] 一句話：腦實質內的**局部化膿性感染**，經歷腦炎期 → 包膜化形成膿腔。典型影像為**環狀強化病灶（ring-enhancing lesion）**。核心：**找感染源頭 + 抗生素 + 神經外科引流**，是內外科合作的急症。

> [!danger] 🚨 值班紅旗
> 1. **頭痛 + 發燒 + 局部神經缺損**（經典三聯，齊全者 <50%）+ 顱內壓升高 → 疑腦膿瘍，先做**顯影 CT/MRI**
> 2. **懷疑腦膿瘍/占位病灶時，腰椎穿刺相對禁忌**（疝脫風險）——先影像
> 3. **顱內壓升高徵象**（意識下降、瞳孔變化、Cushing 反應）→ 神經外科急會診，處理 ICP
> 4. **抗生素前盡量取膿檢體培養**（立體定位抽吸），但重症/顱高壓不可為取檢延誤——先經驗性抗生素

## 📖 定義 / 病生理（這是什麼）
- **腦實質局部化膿感染**，四期：早期腦炎（1–3 天）→ 晚期腦炎 → 早期包膜 → 晚期包膜（形成膿腔壁）。
- **感染途徑決定病原**：
  - **鄰近擴散（contiguous）**：中耳炎/乳突炎、鼻竇炎、齒源感染 → 常單一膿瘍，鏈球菌 + 厭氧菌
  - **血行播散（hematogenous）**：肺膿瘍/膿胸、心內膜炎、發紺性心臟病/肺動靜脈瘻（右到左分流繞過肺過濾）→ **多發膿瘍**、灰白質交界
  - **穿刺傷/術後**：S. aureus、GNR
  - **免疫低下**：Toxoplasma（HIV）、Nocardia、真菌（Aspergillus/Mucor）、結核
- **常見菌**：鏈球菌（含 Strep. anginosus/milleri 群）、厭氧菌（Bacteroides/Prevotella）、S. aureus、腸道 GNR，常**多菌種**。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Headache(頭痛)]]、[[Seizure(癲癇發作)]]、[[Focal neurological deficit(局部神經缺損)]]
- **頭痛（最常見）**、發燒（僅約半數）、局部神經缺損、癲癇
- 顱內壓升高：噁心嘔吐、意識改變、視乳頭水腫
- 依部位表現不同（額葉人格改變、小腦運動失調等）
- 亞急性病程（數天~數週），常缺全身性敗血症表現

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Blood culture(血液培養)]]、[[Aspiration culture(抽吸培養)]]　影像 [[Brain MRI(腦部磁振造影)]]、[[Brain CT(腦部電腦斷層)]]
- **顯影 MRI（首選）/ CT**：**環狀強化病灶 + 周圍水腫**；**DWI 擴散受限（膿腔 restricted diffusion）**幫助與腫瘤/壞死區分。
- **立體定位抽吸（stereotactic aspiration）**：取膿送 **Gram stain + 需氧/厭氧/黴菌/分枝桿菌培養**（診斷 + 治療性減壓）。
- **血液培養**、感染源評估（心臟超音波、鼻竇/齒/耳影像、胸部影像）。
- ⚠️ **占位病灶疑腦膿瘍時 LP 相對禁忌**（疝脫），且 CSF 常無診斷性。
- 免疫低下：Toxoplasma 血清學、影像特徵、必要時切片。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **只有不到一半有發燒**，別因不燒就排除；**頭痛 + 局部徵象 + 亞急性病程**要想。
- **DWI restricted diffusion** 有助膿瘍 vs 腫瘤壞死（膿瘍受限、腫瘤多不受限）。
- **多發膿瘍 → 想血行來源**（找心內膜炎、右到左分流、肺源）。
- **HIV + 多發環狀強化 → Toxoplasma vs 原發性中樞淋巴瘤**鑑別。
- **膿瘍 >2.5 cm 傾向手術引流**；小、深部、多發或早期腦炎期可先藥物 + 密切追蹤影像。

## 💊 治療（Acute / Chronic）
> 用藥 [[Cephalosporin]]、[[Other anti]]（metronidazole）、[[Anti-MRSA]]
### 急性經驗性（依來源，通常組合涵蓋鏈球菌+厭氧+GNR）
- **鄰近擴散/來源不明**：**第三代頭孢（ceftriaxone 2 g IV q12h 或 cefotaxime）+ Metronidazole**（涵蓋厭氧）
- **術後/穿刺傷/疑 S. aureus/MRSA**：**加 Vancomycin**
- **免疫低下**：依疑似病原加抗 Toxoplasma（pyrimethamine + sulfadiazine）、抗黴菌、抗 Nocardia（TMP-SMX）等
- **療程長**：常 **6–8 週 IV**（依引流與反應），影像追蹤縮小
- **輔助**：顱內壓升高/明顯水腫 → 類固醇（dexamethasone，短期，權衡減少抗生素滲透）+ 抗癲癇（有癲癇者）
### 外科
- **立體定位抽吸或開顱引流**：診斷 + 減壓，**>2.5 cm、占位效應明顯、鄰近腦室、對藥物無反應**者。
- **處理源頭**（鼻竇/中耳/齒源清創、心內膜炎處理）。

## ⚠️ 併發症
- **腦膿瘍破入腦室（ventriculitis）= 預後極差**、顱內壓升高疝脫
- 癲癇（急性 + 遠期後遺）、局部神經永久缺損、腦積水
- 復發、鄰近靜脈竇栓塞
