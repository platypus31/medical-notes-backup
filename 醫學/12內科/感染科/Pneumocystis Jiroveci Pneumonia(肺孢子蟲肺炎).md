# Pneumocystis Jiroveci Pneumonia（肺孢子蟲肺炎）

> [!abstract] 一句話：由真菌 **Pneumocystis jirovecii** 在**細胞免疫低下者（尤其 HIV CD4<200、長期類固醇/免疫抑制）**造成的機會性肺炎。值班特徵：**亞急性乾咳喘 + 運動時去飽和 + LDH↑ + CXR 雙側間質浸潤而聽診偏乾淨** → 想 PCP，先驗 HIV。

> [!danger] 🚨 值班紅旗
> 1. **免疫低下 + 進行性喘 + 運動/活動時 SpO₂ 掉（exertional desaturation）** = PCP 直到排除；**A-a gradient 增大**是特徵。
> 2. **治療首選 TMP-SMX**，臨床高度懷疑就先給、別等確診（誘導痰/BAL 之後再驗，病原染色可維持陽性數天）。
> 3. **加類固醇的門檻（HIV 病人，重症）**：**PaO₂ <70 mmHg（室氣）或 A-a gradient ≥35** → **prednisone 40 mg BID×5d → 40 mg QD×5d → 20 mg QD×11d**，且**要在抗生素前/同時給**（延後給無效）。
> 4. 新診斷 PCP = **AIDS-defining**，務必驗 HIV 並評估其他機會感染。
> 5. 通常**無肋膜積液**；若有明顯積液/實變要想別的診斷或合併感染。

## 📖 定義 / 病生理（這是什麼）
- **病原**：*Pneumocystis jirovecii*（過去稱 *P. carinii*；現歸類為**真菌**，但對一般抗真菌藥無效，治療靠 TMP-SMX）。
- **宿主**：**細胞免疫缺損**——HIV **CD4 <200**、長期全身性類固醇（≥20 mg prednisone ≥3 週）、器官/骨髓移植、化療、生物製劑、原發免疫缺陷。
- **病生理**：肺泡內增生 → 泡沫狀滲出物填塞肺泡 → 氣體交換障礙、瀰漫性肺損傷 → 低氧與 A-a gradient 增大。
- **兩種宿主表型差異**：

| 特徵 | HIV 相關 PCP | 非 HIV（類固醇/移植）PCP |
| --- | --- | --- |
| 起病 | 亞急性（數週） | **較急、較猛** |
| 病原負荷 | 高（易驗到） | 低（較難驗到） |
| 預後 | 相對較好 | **死亡率較高** |
| 類固醇輔助 | 依 PaO₂/A-a 門檻 | 證據較弱、個別評估 |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Cough(咳嗽)]]、[[Hypoxemia(低血氧)]]
- **三聯**：進行性**呼吸困難 + 乾咳 + 發燒**（數天到數週）。
- **運動時去飽和**很典型；理學聽診常**相對乾淨**（與影像嚴重度不成比例）。
- 嚴重者靜態低氧、呼吸衰竭。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[LDH]]、[[Beta-D-glucan]]、[[HIV Ag/Ab combo assay]]　影像 [[Chest X-ray(胸部X光)]]、[[HRCT Chest(高解析胸部電腦斷層)]]
- **影像**：CXR **雙側肺門周圍/間質性浸潤**（可正常）；**HRCT 瀰漫性毛玻璃樣變化（GGO）**（敏感，正常幾乎可排除）。
- **實驗室**：**LDH 升高**（敏感非特異）、**血清 β-D-glucan 升高**（輔助）、動脈血氣算 **A-a gradient**。
- **確診（找病原）**：**誘導痰**或 **支氣管肺泡灌洗（BAL）**做染色（GMS、DFA/免疫螢光）或 **PCR**（敏感，但陽性也可能是定植，需搭配臨床）。
- 新診斷者**驗 HIV + CD4**。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **聽診乾淨但 CXR/CT 一片瀰漫 + 活動去飽和** = 經典 PCP 情境。
- **LDH↑ 敏感但不特異**；HRCT 陰性幾乎可排除。
- 常見於**未診斷的 HIV** 以此為初現 → 一定驗 HIV。
- **類固醇要在抗生素前/同時給**（重症才給），事後補給無效。
- 曾接受**吸入 pentamidine 預防**者可能出現非典型上肺/氣胸。

## 💊 治療（Acute / Chronic）
> 用藥 [[Trimethoprim-Sulfamethoxazole(TMP-SMX)]]、[[Prednisone]]、[[Primaquine]]、[[Clindamycin]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### Acute（21 天）
- **首選：TMP-SMX**，TMP 15–20 mg/kg/day 分 3–4 次（PO/IV）× **21 天**；監測腎功能、鉀、血球、皮疹。
- **輔助類固醇（重症門檻，見紅旗）**：PaO₂ <70 或 A-a ≥35 → prednisone 遞減 21 天療程。
- **磺胺過敏/失敗替代**：
  - Primaquine（先查 **G6PD**）+ Clindamycin
  - IV Pentamidine（腎毒性、低血糖、QT）
  - Atovaquone（輕–中度）、TMP + dapsone（先查 G6PD）
### Chronic（次級預防）/ 初級預防
- **HIV CD4 <200（或口咽念珠菌/AIDS）**：**TMP-SMX 1 顆 DS QD 或 SS QD 預防**，直到 ART 使 **CD4 >200 持續 ≥3 個月**。
- 替代預防：dapsone（查 G6PD）、atovaquone、吸入 pentamidine。

## ⚠️ 併發症
- 急性呼吸窘迫、呼吸衰竭需插管
- **氣胸**（尤其壞死/囊狀病灶、吸入 pentamidine 者）
- 治療相關：TMP-SMX 骨髓抑制/高鉀/腎損/皮疹（可致 SJS）；pentamidine 低血糖/腎毒
- 播散性 pneumocystosis（罕見）

## 📊 嚴重度評分 / 風險分層（scoring）
> 以氧合分層決定是否加類固醇與住院層級
| 指標 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **PaO₂（室氣）** | 是否加輔助類固醇 | **<70 mmHg → 加類固醇** |
| **A-a gradient** | 重症判定 | **≥35 → 重症、加類固醇** |
| SpO₂ / 需氧程度 | 住院/ICU 分流 | 靜態低氧/呼吸衰竭 → ICU |
