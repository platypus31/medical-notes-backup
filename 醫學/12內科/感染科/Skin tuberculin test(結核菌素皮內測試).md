# Skin tuberculin test（結核菌素皮內測試 / TST / Mantoux）

> [!abstract] 一句話：皮內注射 **PPD（純化蛋白衍生物）** 引發**第四型遲發性過敏反應**，48–72 小時判讀**硬結（induration）橫徑**，用來偵測**潛伏結核感染（LTBI）**。它測的是「免疫記憶」不是「現在有沒有活動性結核」，卡介苗（BCG）接種會造成偽陽性。

> [!danger] 🚨 值班重點
> 1. **判讀量的是「硬結（induration）」不是紅斑（erythema）**，用觸診沿手臂橫軸量 mm；只有紅無硬結記 0 mm。
> 2. **48–72 小時內判讀**；超過 72 小時未讀且陰性者需重新測試。
> 3. **TST 陽性只代表曾被致敏（LTBI 或既往感染 / BCG），不能診斷活動性結核**；有症狀者需胸部 X 光 + 痰檢排除活動性 TB。
> 4. **陰性不能排除活動性結核**：粟粒性 / 重症 TB、免疫低下、剛感染（window period 8–10 週）可 anergy 偽陰性。

## 📖 定義 / 病生理（這是什麼）
- **原理**：皮內注射 **5 TU PPD-S（0.1 mL）** 於前臂掌側 → 若曾對結核分枝桿菌抗原致敏，記憶 T 細胞聚集 → **第四型（delayed-type / cell-mediated）過敏反應** → 局部硬結。
- **注射法（Mantoux）**：斜面朝上皮內注射，形成 6–10 mm 白色隆起（wheal）才算成功。
- **與 IGRA 的差別**：IGRA（QuantiFERON、T-SPOT）測血中 T 細胞對 **ESAT-6 / CFP-10** 反應，**不受 BCG 與多數環境分枝桿菌影響**，適合曾打 BCG 者。

## 🩺 臨床表現（判讀）
> 連 [[Tuberculosis(結核病)]]、[[Latent TB infection(潛伏結核感染)]]
判讀依「風險分層」採不同硬結切點（美國 CDC/ATS 標準）：

| 硬結 ≥ | 判為陽性的族群 |
| --- | --- |
| **≥5 mm** | HIV 感染者、近期接觸活動性 TB、胸部 X 光有舊 TB 纖維化病灶、器官移植 / 免疫抑制（含 ≥15 mg/day prednisone、anti-TNF 使用者） |
| **≥10 mm** | 近 5 年由高盛行區移入、IVDU、高危機構住民/員工（醫護、監獄、遊民收容）、實驗室人員、糖尿病/矽肺/慢性腎病/某些血液惡性等高風險共病、<4 歲兒童 |
| **≥15 mm** | 無任何危險因子的一般人 |

## 🔬 診斷用途 / 相關檢查
> 檢查 [[IGRA(丙型干擾素釋放試驗)]]、[[Chest X-ray]]、[[痰抗酸菌染色]]
- **Two-step testing（兩階段測試）**：用於將定期反覆檢測者（如醫護新進）。第一次陰性者 1–3 週後再測一次，抓出因久未受刺激而反應減弱、第二次才顯現的 **booster 效應**，避免日後誤判為新感染（轉陽）。
- TST 或 IGRA 陽性 → 續做胸部 X 光排除活動性 TB；有症狀或影像異常再送痰抗酸菌塗片 / 培養 / NAAT。

## ⭐ 特徵 / 注意（pearls & pitfalls）
- **BCG 造成偽陽性**：台灣普遍接種 BCG，TST 陽性判讀受干擾，此時 **IGRA 較特異**。
- **Booster 現象 vs 真轉陽（conversion）**：兩階段可區分。真轉陽定義為兩年內硬結增加 **≥10 mm**。
- **Anergy（偽陰性）原因**：HIV/免疫低下、粟粒性或重症 TB、近期病毒感染或活疫苗（如 MMR，應間隔）、營養不良、極端年齡、注射技術不良（打太深變皮下）。
- 測的是**過去的致敏**，不能區分 LTBI、既往已治癒感染、或 BCG。
- 局部強反應可能起水泡 / 潰瘍，但不代表活動性 TB。

## 💊 處置（陽性後）
> 用藥 [[Isoniazid(INH)]]、[[Rifampin]]
- **確認為 LTBI（陽性 + 排除活動性 TB）** → 評估 **LTBI 治療**以防日後發病：常用如 **INH 單方 6–9 個月**、**INH + Rifapentine 每週一次共 12 週（3HP）**、或 **Rifampin 4 個月（4R）**（實際方案 / 劑量依現行指引與肝功能評估）。
- 用 INH 前後注意肝毒性、補充 **維生素 B6（pyridoxine）** 預防周邊神經病變。
- **活動性結核**須完整抗結核多重藥物治療（見 [[Tuberculosis(結核病)]]），不能只給 LTBI 單方（會誘導抗藥）。

## ⚠️ 侷限 / 併發
- 偽陽性（BCG、非結核分枝桿菌）、偽陰性（anergy）。
- 需回診判讀（依從性問題）、注射與判讀者間變異。
- 極少數強烈局部反應、罕見全身過敏。
