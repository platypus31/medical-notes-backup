# Toxic Epidermal Necrolysis（毒性表皮壞死症）

> [!abstract] 一句話：藥物誘發的**嚴重皮膚黏膜壞死（SJS/TEN spectrum）**，全層表皮壞死剝離、Nikolsky 徵象陽性；**依剝離體表面積分：SJS <10%、overlap 10–30%、TEN >30%**。值班第一動作——**立刻停掉可疑致病藥（時間就是命）**，是死亡率極高（TEN 可達 25–35%）的皮膚科急症。

> [!danger] 🚨 值班紅旗
> 1. **立即停掉所有可疑致病藥**——「越早停致病藥，死亡率越低」是唯一被證實改善預後的措施。
> 2. **黏膜侵犯（口腔、眼、生殖器）+ 疼痛性紅斑 + 皮膚一撕就掉（Nikolsky ＋）→ 當 TEN 處理**，不要當一般藥疹。
> 3. **像大面積燒傷**：體液流失、電解質失衡、體溫調節失衡、續發感染／敗血症（主要死因）→ 轉燒燙傷中心 / ICU。
> 4. **眼科即刻會診**（延誤 → 角膜潰瘍、瞼球黏連、失明）。
> 5. **前驅期像感冒／喉嚨痛 + 剛加的新藥（1–4 週內）→ 提高警覺**，尤其 allopurinol、抗癲癇（carbamazepine、lamotrigine、phenytoin）、磺胺、NSAID、nevirapine。台灣患者用 carbamazepine/allopurinol 前應查 **HLA-B*15:02 / HLA-B*58:01**。

## 📖 定義 / 病生理（這是什麼）
- **本質**：藥物引發的 **T 細胞／granulysin 媒介的角質細胞大量凋亡**，導致表皮全層壞死與真皮-表皮分離；屬 **type IV 過敏反應**。與多形性紅斑（EM，多為感染 HSV 誘發）不同。
- **SJS–TEN 是同一光譜，依剝離體表面積（BSA）分**：

| 分類 | 表皮剝離 BSA |
| --- | --- |
| **Stevens-Johnson syndrome (SJS)** | < 10% |
| **SJS/TEN overlap** | 10–30% |
| **Toxic epidermal necrolysis (TEN)** | > 30% |

- **常見致病藥**（發病前 **1–4 週**新加）：**allopurinol**、抗癲癇（carbamazepine、lamotrigine、phenytoin、phenobarbital）、**磺胺類抗生素**、NSAID（oxicam 類）、nevirapine、抗生素（β-lactam、quinolone）。
- **感染誘因**：*Mycoplasma pneumoniae*（尤其兒童黏膜為主之 MIRM）、部分病毒。
- **基因**：**HLA-B*15:02**（carbamazepine，漢族／台灣高）、**HLA-B*58:01**（allopurinol）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Skin rash(皮疹)]]、[[Mucosal lesion(黏膜病灶)]]
- **前驅期**（1–3 天）：發燒、喉嚨痛、倦怠、眼刺激（似上呼吸道感染）。
- **皮膚**：疼痛性紅斑／紫紅斑 → **不典型靶形病灶** → 水泡、大片表皮剝離（濕潤真皮），**Nikolsky 徵象（＋）**、Asboe-Hansen（水泡側推）＋。
- **黏膜侵犯（>90%，≥2 處）**：出血性唇炎／口腔糜爛、結膜炎、尿道／生殖器糜爛。
- 全身：高燒、脫水、低血壓；嚴重可累及呼吸道、消化道黏膜。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Skin biopsy(皮膚切片)]]　影像 [[Chest X-ray(胸部X光)]]
- **臨床診斷為主** + 詳細**用藥時序**（用 ALDEN 演算法評估致病藥因果）。
- **皮膚切片**：全層表皮壞死、真皮-表皮分離、稀疏發炎浸潤（排除 SSSS：後者裂在角質層下 granular layer，靠 frozen section 快速鑑別）。
- 檢驗：CBC（血球減少）、電解質、腎功能、血糖、白蛋白、發炎指標、血/傷口培養（監測敗血症）。
- 眼科、泌尿、呼吸評估黏膜受累。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **最重要區別 SSSS（葡萄球菌燙傷樣皮膚症候群）**：SSSS 多見兒童、**無黏膜侵犯**、裂在表淺 granular layer；TEN 有黏膜 + 全層壞死。
- **Nikolsky 陽性 + 黏膜受累 + 新藥史** = TEN 直到排除。
- **類固醇仍有爭議**、**不要用預防性全身抗生素**（僅感染時針對性用）。
- **一律標記藥物過敏**，病人與家屬終生避用該藥及交叉反應藥（如芳香族抗癲癇彼此交叉）。
- 恢復後長期後遺：眼（乾眼、瞼球黏連、失明）、皮膚色素、指甲、黏膜狹窄。

## 💊 治療（Acute / Chronic）
> 用藥 [[Cyclosporine(環孢素)]]、[[IVIG(靜脈免疫球蛋白)]]
> ⚠️ 免疫調節藥證據仍有爭議，用藥前查最新指引
### 核心（支持療法，如燒傷）
1. **立即停用致病藥**（改善存活最確定的措施）。
2. **轉燒燙傷中心 / ICU**：體液電解質復甦、體溫維持、營養、傷口照護（非黏著性敷料）、止痛。
3. **感染監測**（傷口/血培養），**不預防性用抗生素**，有感染再針對性投藥。
4. **多科會診**：眼科（每日眼護理、防黏連）、泌尿／婦科、呼吸道。
### 免疫調節（證據分歧，依中心經驗）
- **Cyclosporine 3–5 mg/kg/day**：多數近期資料支持可能降死亡率，較常被採用。
- **IVIG**、**etanercept（anti-TNF）**：部分研究支持。
- **全身類固醇**：爭議，早期短療程或有幫助但感染風險。
### 慢性 / 追蹤
- 過敏藥物永久註記；眼科長期追蹤；心理支持。

## ⚠️ 併發症
- **敗血症 / 感染性休克（主要死因）**、多器官衰竭
- 大量體液電解質流失、低血容休克、AKI
- 呼吸道黏膜脫落 → 呼吸衰竭
- **眼部**：角膜潰瘍、瞼球黏連、失明（長期最常致殘）
- 皮膚／黏膜狹窄、色素異常、指甲缺損

## 📊 嚴重度評分 / 風險分層（scoring）
> **SCORTEN**（入院 24 小時內評估死亡率）
| SCORTEN 項目（各 1 分） |
| --- |
| 年齡 ≥ 40 歲 |
| 合併惡性腫瘤 |
| 心率 ≥ 120 /min |
| 初始表皮剝離 BSA > 10% |
| 血中尿素氮 BUN > 28 mg/dL（>10 mmol/L urea） |
| 血糖 > 252 mg/dL（>14 mmol/L） |
| 血中碳酸氫根 HCO₃⁻ < 20 mEq/L |

| 分數 | 預估死亡率 |
| --- | --- |
| 0–1 | ~3% |
| 2 | ~12% |
| 3 | ~35% |
| 4 | ~58% |
| ≥5 | ~90% |
