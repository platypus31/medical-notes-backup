# Hospital-Acquired Pneumonia（院內型肺炎）

> [!abstract] 一句話：**入院 ≥48 小時後**新發生的肺炎（HAP）；**插管 ≥48 小時後**發生者為呼吸器相關肺炎（VAP）。相較社區型，**多重抗藥菌（MDR）+ Pseudomonas + MRSA** 風險高 → 經驗性抗生素要更廣。

> [!danger] 🚨 值班紅旗
> 1. **HAP/VAP 經驗性抗生素要涵蓋 Pseudomonas**；有 MRSA 風險再加抗 MRSA（vancomycin/linezolid）
> 2. **抗生素前先送呼吸道檢體（痰/氣管抽吸/BAL）+ 血培**，但重症不可為取檢延誤第一劑
> 3. **敗血症/呼吸衰竭** → 走敗血症 bundle + 及早評估呼吸支持
> 4. **48–72 小時看培養降階（de-escalate）**，避免長期廣效養出抗藥菌；療程多為 **7 天**（依反應）

## 📖 定義 / 病生理（這是什麼）
- **HAP**：入院 **≥48 h** 後新發、入院時非潛伏中的肺炎。
- **VAP**：氣管插管 **≥48 h** 後發生的肺炎。
- （HCAP「醫療照護相關肺炎」概念已於 2016 IDSA/ATS 指引**廢除**。）
- **病生理**：口咽定殖菌**微吸入（microaspiration）**、生物膜（氣管內管）、宿主防禦下降 → 下呼吸道感染。
- **致病菌**：**Pseudomonas aeruginosa、腸道 GNR（Klebsiella/Enterobacter/E. coli/Acinetobacter）、S. aureus（含 MRSA）**；多重抗藥（ESBL、CRE、MDR-Pseudomonas）風險依當地生態與病人。
- **MDR 危險因子**：近 90 天內靜脈抗生素、住院 ≥5 天、當地高抗藥率、VAP 前 ARDS/透析/敗血性休克等。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Fever(發燒)]]、[[Cough(咳嗽)]]、[[Dyspnea(呼吸困難)]]
- 新發/惡化的發燒、膿痰、咳嗽、呼吸困難、低血氧
- **新的肺浸潤** + 白血球升高/降低
- VAP：氧合惡化、呼吸器分泌物增加、發燒；插管者症狀不易主訴，靠客觀指標

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Sputum culture(痰液培養)]]、[[Blood culture(血液培養)]]　影像 [[Chest X-ray(胸部X光)]]
- **臨床 + 影像**：**新出現/進展的肺浸潤** + 至少兩項（發燒、白血球異常、膿痰/氧合惡化）。
- **微生物**：**呼吸道檢體（痰、氣管內抽吸、BAL）Gram stain + 培養**（盡量抗生素前）、血液培養；VAP 可半定量/定量培養。
- **鑑別**：肺水腫、肺栓塞/肺梗塞、ARDS、肺出血、肺不張、藥物肺——避免把非感染浸潤都當肺炎投藥。
- procalcitonin 輔助療程決策（不單獨作起始）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **經驗性選藥依「當地抗生素圖譜（antibiogram）+ 病人 MDR 風險」**，不是死背——先看你單位的 GNR/MRSA 抗藥率。
- **不是每個 HAP 都要雙重抗綠膿 + 抗 MRSA**：低 MDR 風險、單位抗藥率低者可單一抗綠膿藥、不常規加 MRSA 涵蓋（依 2016 IDSA/ATS）。
- **48–72h 培養回來一定要降階**（de-escalation）——這是抗生素 stewardship 核心。
- **Daptomycin 不能治肺炎**（被肺泡界面活性劑滅活）——MRSA 肺炎用 vancomycin 或 linezolid。
- 療程通常 **7 天**（含 non-fermenting GNR 如 Pseudomonas，只要臨床改善），不需一律拉長。

## 💊 治療（Acute / Chronic）
> 用藥 [[抗生素]]、[[Anti-MRSA]]、[[Carbapenem]]、[[Cephalosporin]]
### 急性經驗性（依 MDR 風險 + 當地抗藥率調整）
- **抗綠膿 β-lactam（擇一）**：**Piperacillin-tazobactam**、**Cefepime**、**Ceftazidime**、**Meropenem/Imipenem**
- **加抗 MRSA（有 MRSA 風險或單位 MRSA 率高）**：**Vancomycin**（依 level）或 **Linezolid**
- **雙重抗綠膿（高抗藥風險/重症/當地抗藥率高）**：上述 β-lactam **+ 抗綠膿 fluoroquinolone（ciprofloxacin/levofloxacin）或 aminoglycoside**（避兩個同 β-lactam）
- ⚠️ 具體劑量依體重/腎功能/當地指引查證（尤 vancomycin、aminoglycoside 需監測濃度）
### 降階 + 療程
- **48–72h 依培養與臨床降階**，窄化為針對性單藥。
- **療程約 7 天**（臨床改善者，含 Pseudomonas；未改善/膿瘍/免疫低下另議）。
### 預防（VAP bundle）
- 床頭抬高 30–45°、每日評估拔管與鎮靜假期、口腔照護（chlorhexidine 證據有爭議）、避免不必要再插管、DVT/PUD 預防。

## ⚠️ 併發症
- **呼吸衰竭 / ARDS**、敗血症/敗血性休克
- 肺膿瘍、膿胸、續發菌血症、轉移性感染
- 抗藥菌選殖、C. difficile 感染、住院延長/死亡率上升

## 📊 嚴重度評分 / 風險分層（scoring）
> HAP/VAP 主要靠「MDR 風險 + 器官障礙」分層（CURB-65 是社區型工具，院內型參考性有限）
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **MDR 危險因子** | 選藥廣度 | 近期 IV 抗生素、住院≥5天、當地高抗藥 → 更廣/雙重涵蓋 |
| **qSOFA / SOFA** | 是否敗血症 | ≥2 → 走敗血症流程 |
| **CPIS（Clinical Pulmonary Infection Score）** | VAP 輔助 | 綜合體溫/白血球/分泌物/氧合/影像/培養，輔助診斷與停藥（非單獨決策） |
