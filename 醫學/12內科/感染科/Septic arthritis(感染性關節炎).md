# Septic arthritis（感染性關節炎）

> [!abstract] 一句話：**細菌侵入關節腔**造成的化膿性關節炎，是**骨科/內科急症**——**數天內即可破壞軟骨致永久殘障**。**急性單一關節紅腫熱痛 = 抽關節液排除為止**。**S. aureus 最常見**；年輕性活躍者想淋病雙球菌。

> [!danger] 🚨 值班紅旗
> 1. **急性單關節紅腫熱痛 + 動則劇痛 = 先抽關節液（arthrocentesis）**，不要只當痛風就給消炎藥打發
> 2. **關節液 WBC >50,000（常 >100,000）、PMN >75%** 高度支持化膿性；送 Gram stain + 培養 + 結晶（可與痛風並存）
> 3. **化膿性關節 = 需要引流（外科清創/關節鏡或反覆抽吸）+ IV 抗生素**，光靠抗生素不夠
> 4. **人工關節 + 紅腫痛/鬆脫** → 人工關節感染（PJI），需骨科處理，抗生素前盡量取檢體

## 📖 定義 / 病生理（這是什麼）
- **細菌經血行（最常見）、鄰近感染擴散、或直接接種（外傷/注射/手術）**進入關節腔 → 化膿、發炎酵素+免疫反應**快速破壞軟骨**。
- **致病菌**：
  - **Staphylococcus aureus（最常見，含 MRSA）**
  - **Streptococcus**（次之）
  - **Neisseria gonorrhoeae（淋病雙球菌）**：年輕、性活躍者，常**多關節/游走性 + 腱鞘炎 + 皮膚膿疱疹**（[[Disseminated gonococcal infection(瀰漫性淋病雙球菌感染)]]）
  - GNR（老人、免疫低下、靜脈藥癮）；人工關節早期 S. aureus / 晚期 CoNS
- **危險因子**：既有關節病（RA、OA）、人工關節、糖尿病、免疫低下、靜脈藥癮、皮膚感染、關節注射。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Joint pain(關節痛)]]、[[Fever(發燒)]]
- **急性單關節**（膝最常見）**紅、腫、熱、痛 + 活動範圍受限、動則劇痛**
- 發燒（未必都有）、關節積液
- **淋病性**：可**多關節/游走性、腱鞘炎（tenosynovitis）、皮膚膿疱**三聯

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Arthrocentesis(關節穿刺)]]、[[Blood culture(血液培養)]]　影像 [[Joint X-ray(關節X光)]]、[[MRI(磁振造影)]]
- **關節穿刺（arthrocentesis）為關鍵** — 抽關節液送：
  - **細胞數 + DC**：**WBC >50,000/µL（常 >100,000）、PMN >75%** 高度懷疑（無單一絕對閾值，越高越像）
  - **Gram stain + 培養**（陽性率有限，Gram 約 50%）
  - **結晶檢查**（排除/合併痛風、CPPD——**可與感染並存**，有結晶不代表沒感染）
- **血液培養**（常陽性，尤 S. aureus）、CBC、CRP/ESR、乳酸。
- **淋病疑似**：泌尿生殖道/咽/直腸 NAAT + 培養。
- **影像**：X 光初期常正常（追蹤看軟骨/骨破壞）；MRI 評估關節周圍/骨髓炎；超音波導引抽吸。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **痛風與化膿性關節炎可並存**——關節液有尿酸結晶**不能**排除感染，臨床像感染仍要培養+治療。
- **正常的關節液 WBC 不排除**（早期/免疫低下/部分治療過），臨床高度懷疑就治療。
- **人工關節** 感染表現可較不典型（低度發炎、鬆脫、竇道），需骨科 + 多次深部檢體。
- **延遲引流/治療 = 不可逆軟骨破壞**——這是「時間就是關節」的急症。
- 淋病性關節炎對抗生素反應好，非淋病性（尤 S. aureus）常需外科引流。

## 💊 治療（Acute / Chronic）
> 用藥 [[Anti-MRSA]]、[[Cephalosporin]]、[[抗生素]]
### 急性經驗性（抽完關節液/血培後即給，依 Gram stain 與宿主）
- **涵蓋 MRSA**：**Vancomycin**（15–20 mg/kg IV q8–12h，依 level 調整⚠️依體重腎功能）
- **加涵蓋 GNR**（老人/免疫低下/藥癮）：第三/四代頭孢（**ceftriaxone** 或 **cefepime**）或抗綠膿 β-lactam
- **疑淋病性**：**Ceftriaxone**（依現行 STI 指引劑量⚠️查證）+ 治療合併披衣菌
- **依 Gram stain 調整**：G(+)cocci → 續抗葡萄球菌；G(−)bacilli → 抗綠膿涵蓋
- **依培養降階**，療程常 **2–4 週（IV 轉口服）**，人工關節/骨髓炎更長。
### 引流（不可少）
- **關節引流**：外科清創 / 關節鏡沖洗 / 反覆針抽——化膿性關節炎必須引流膿液。
- 人工關節感染：骨科手術（清創保留假體 vs 兩階段置換）+ 長療程抗生素。

## ⚠️ 併發症
- **不可逆軟骨破壞、關節僵直/退化、功能喪失**
- **骨髓炎**（[[Vertebral Osteomyelitis(脊柱骨髓炎)]] 等鄰接骨）、關節脫位、生長板受損（兒童）
- **菌血症 → 敗血症/敗血性休克**、轉移性感染
- 人工關節失敗、慢性竇道

## 📊 嚴重度評分 / 風險分層（scoring）
> 靠關節液分析 + 宿主 + 是否人工關節分層
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **關節液 WBC / PMN%** | 感染可能性 | >50,000 且 PMN>75% 高度懷疑；越高越像（非絕對閾值） |
| **Gram stain / 培養** | 病原+選藥 | 陽性指引治療；陰性不排除 |
| **Kocher criteria（兒童髖）** | 兒童鑑別 | 發燒/不負重/ESR↑/WBC↑ 越多項越可能化膿性 |
