# Human Immunodeficiency Virus（人類免疫缺乏病毒）

> [!abstract] 一句話：一種侵犯 **CD4+ T 細胞**的 retrovirus，未治療會逐步免疫崩解走向 AIDS（機會性感染/腫瘤）。值班關鍵：**看到不尋常的機會性感染就要想 HIV**，且**CD4 越低、要防的病原越多**（用 CD4 當地圖）。

> [!danger] 🚨 值班紅旗
> 1. **急性反轉錄病毒症候群（ARS）**：新近高風險暴露 + **發燒、咽炎、淋巴結腫、皮疹、無菌性腦膜炎樣** → 此時**抗體可能仍陰性**，要驗 **HIV RNA / p24 抗原（4th gen combo）**，別只信抗體。
> 2. **CD4 <200 + 亞急性呼吸喘 + 運動時去飽和 + LDH↑ + CXR 雙側間質浸潤 → PCP**，先經驗性 **TMP-SMX**；**PaO₂ <70 或 A-a gradient ≥35 → 加類固醇**（見 [[Pneumocystis Jiroveci Pneumonia(肺孢子蟲肺炎)]]）。
> 3. **CD4 <100 + 頭痛/發燒 → Cryptococcal meningitis**（驗血清/CSF CrAg，測開口壓，高壓要反覆腰椎放液）；**局部神經缺損 + 環狀強化病灶 → Toxoplasma vs 淋巴瘤**。
> 4. **暴露後預防（PEP）黃金 72 小時**、職業針扎立即啟動三合一 ART；**PrEP** 給高風險未感染者。
> 5. **IRIS**：開始 ART 後既有感染反而惡化（尤其結核/隱球菌），別誤判 ART 失敗；治療潛在感染後再/續 ART，時機需權衡。

## 📖 定義 / 病生理（這是什麼）
- **病毒學**：HIV-1（全球主流）/HIV-2；ssRNA retrovirus，gp120 結合 **CD4 + 共受體 CCR5/CXCR4** 進入 T 細胞 → reverse transcriptase 逆轉錄 → integrase 嵌入宿主基因體 → 建立**潛伏儲存池（無法根治的原因）**。
- **自然病程**：急性感染（病毒血症高峰、ARS）→ 臨床潛伏（CD4 緩降）→ **AIDS（CD4 <200 或出現 AIDS-defining illness）**。
- **傳播**：性接觸、血液/針具共用、母嬰垂直（產程/哺乳）。
- **CD4 對照易感染病原（值班地圖）**：

| CD4 (cells/µL) | 需警覺的機會性感染/腫瘤 |
| --- | --- |
| **<500** | 帶狀疱疹、口腔念珠菌、肺結核、Kaposi sarcoma、念珠菌陰道炎 |
| **<200** | **PCP**、其他 |
| **<100** | **Toxoplasma 腦炎、Cryptococcus 腦膜炎**、食道念珠菌 |
| **<50** | **CMV 視網膜炎/腸炎、MAC（播散性）**、CNS 淋巴瘤 |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Fever(發燒)]]、[[Lymphadenopathy(淋巴結腫大)]]、[[Weight loss(體重減輕)]]
- **急性（ARS）**：單核球增多症樣——發燒、咽炎、全身淋巴結腫、斑丘疹、肌痛、口腔潰瘍，可有無菌性腦膜炎。
- **潛伏期**：多無症狀，可持續型全身淋巴結腫。
- **AIDS 期**：反覆/不尋常感染、體重減輕、慢性腹瀉、鵝口瘡、口腔毛狀白斑（EBV）、盜汗；依 CD4 出現對應機會性疾病。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[HIV Ag/Ab combo assay]]、[[CD4 count]]、[[HIV Viral Load]]
- **篩檢/確診（4th generation）**：**HIV-1/2 抗體 + p24 抗原 combo**（縮短空窗期）→ 陽性再做抗體分型確認 → **HIV RNA** 定量。
- **急性感染**：抗體可陰性，**靠 HIV RNA / p24**（RNA 通常暴露後 1–2 週可測到）。
- **分期與監測**：**CD4 count**（免疫狀態/機會感染風險）+ **病毒量（治療反應）**；治療前查抗藥性基因型、HLA-B*5701（若考慮 abacavir）。
- **合併篩檢**：結核（IGRA/胸片）、B/C 肝、梅毒、其他 STD、隱球菌抗原（CD4 低者）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **不明原因、不尋常或反覆的感染 → 驗 HIV**（PCP、食道念珠菌、Kaposi 等都是 AIDS-defining）。
- 急性期 window：**抗體陰性不能排除**，需 RNA/p24。
- **開始 ART 前**先評估並治療潛在活動性感染（尤其 TB、隱球菌）以降低 IRIS 與顱內壓災難。
- U=U：**病毒量測不到 = 不具性傳染力（Undetectable = Untransmittable）**。
- 現代 ART 目標是**確診當天/儘早開始（rapid start）**，不再等 CD4 門檻。

## 💊 治療（Acute / Chronic）
> 用藥 [[Antiretroviral Therapy(抗反轉錄病毒治療)]]、[[Tenofovir]]、[[Dolutegravir]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### Chronic（ART，終生）
- **首選骨幹（多數初治）**：**INSTI-based**，如 **Bictegravir/Tenofovir alafenamide/Emtricitabine（BIC/TAF/FTC，單錠）** 或 **Dolutegravir + (TAF/TDF)/FTC 或 3TC**。
- 原則：**多藥合併壓制病毒量至測不到**、恢復/維持 CD4；規律服藥避免抗藥性。
- **機會性感染預防（依 CD4）**：CD4 <200 → **TMP-SMX 預防 PCP**（同時預防 Toxoplasma，CD4<100 更重要）；CD4 <50 → 依情況考慮 MAC/CMV 監測。
### Acute / 特殊情境
- **PEP（暴露後）**：72h 內啟動三合一 ART（如 TDF/FTC + DTG）× 28 天。
- **PrEP（暴露前）**：TDF/FTC 或 TAF/FTC 或長效 cabotegravir 注射，給高風險未感染者。
- 各機會性感染有各自療法（PCP→TMP-SMX±類固醇；隱球菌→amphotericin B + flucytosine → fluconazole；Toxoplasma→pyrimethamine + sulfadiazine）。

## ⚠️ 併發症
- 機會性感染（PCP、TB、Cryptococcus、Toxoplasma、CMV、MAC、食道念珠菌）
- 惡性腫瘤：Kaposi sarcoma、非何杰金氏淋巴瘤/CNS 淋巴瘤、子宮頸/肛門癌（HPV）
- HIV 相關腎病變（HIVAN）、消耗症候群、HIV 相關神經認知障礙（HAND）
- **IRIS**（開始 ART 後免疫重建發炎反應）
- 長期：心血管風險、代謝/骨質、藥物毒性

## 📊 嚴重度評分 / 風險分層（scoring）
> CD4 分期定機會感染風險與預防；病毒量定治療反應
| 指標 | 用途 | 意義 |
| --- | --- | --- |
| **CD4 count** | 免疫分期/OI 預防 | <200 = AIDS 門檻、啟動 PCP 預防；<100/<50 分層 |
| **HIV viral load** | 治療效果/傳染力 | 目標測不到（U=U）；反彈 → 順從性/抗藥性 |
| **AIDS-defining illness** | 臨床分期 | 出現即定義 AIDS（不論 CD4） |
