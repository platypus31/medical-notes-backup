# Coxsackievirus（克沙奇病毒）

> [!abstract] 一句話：Enterovirus 屬 RNA 病毒，糞口 / 飛沫傳染；**A 群 → 手足口病 / 疱疹性咽峽炎，B 群 → 心肌炎 / 心包炎 / Bornholm disease**；台灣須警覺 EV71 重症腦幹腦炎。

> [!danger] 🚨 值班紅旗
> 1. **心肌炎（B 群）**：新發胸痛 + 心衰竭 + troponin↑ + 心電圖變化 + 心臟超音波 EF 下降 → 可猝死，需監測心律 / 心衰。
> 2. **腸病毒重症前驅徵象（EV71）**：嗜睡、持續嘔吐、肌抽躍（myoclonic jerk）、心跳呼吸急促 → 恐進展腦幹腦炎 / 神經性肺水腫，兒童致命。
> 3. 疱疹性咽峽炎高燒 + 咽峽水疱 → 進食困難脫水，注意小兒水分。
> 4. 新生兒 B 群感染 → 猛爆性心肌炎 / 肝炎 / 敗血症樣症候群，死亡率高。

## 📖 定義 / 病生理（這是什麼）
- **病原體**：Picornaviridae，Enterovirus 屬，單股正義（+）RNA、非套膜；分 Coxsackie A（~23 型）、Coxsackie B（6 型）。
- **傳染途徑**：糞口為主，亦飛沫 / 接觸口鼻分泌物、水疱液；夏秋季流行。
- **病生理**：病毒經腸道上皮增殖 → 病毒血症 → 依親和性侵犯皮膚黏膜（A 群）、心肌 / 腦膜 / 肌肉（B 群）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[發燒]]、[[皮疹]]、[[胸痛]]
- **Coxsackie A 群**：
  - **手足口病（HFMD）**：口腔潰瘍 + 手掌足底 + 臀部丘疹水疱，發燒。
  - **疱疹性咽峽炎（Herpangina）**：高燒 + 軟顎 / 咽峽後部灰白水疱→潰瘍，咽痛拒食。
- **Coxsackie B 群**：
  - **心肌炎 / 心包炎**：胸痛、心悸、心衰、心包摩擦音。
  - **Bornholm disease（流行性胸肌痛 / devil's grip）**：突發劇烈胸腹肌肉痛、隨呼吸加劇。
- **共通**：無菌性腦膜炎、非特異性發熱疹、結膜炎。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Troponin]]、[[心電圖]]、[[RT-PCR]]
- **臨床診斷**：HFMD / herpangina 多依典型表現。
- **病原確認**：咽拭子 / 糞便 / CSF / 水疱液 **RT-PCR 或病毒培養**（enterovirus）。
- **心肌炎評估**：troponin、CK-MB、ECG（ST-T 變化 / 心律不整）、心臟超音波（EF、心包積液）、必要時心臟 MRI。
- 無菌性腦膜炎：CSF 淋巴球增多、糖正常、PCR 陽性。

## ⭐ 特徵 / 注意（pearls & pitfalls）
- 記憶：**A 群 → 皮膚 / 黏膜（Above，口手足），B 群 → 內臟（Body，心 / 肌肉）**。
- Herpangina 病灶在**後咽 / 軟顎**（vs. HFMD 前口腔 + 手足），需與 HSV 齦口炎（前齒齦）鑑別。
- **EV71**（嚴格說是 Enterovirus A71，非 Coxsackie，但同屬並常一起討論）在台灣造成手足口病重症、腦幹腦炎；Coxsackie A6 / A16 亦常見流行株。
- 目前無特效抗病毒藥，支持性療法為主。

## 💊 治療（Acute / Chronic）
> 用藥 [[支持性療法]]
- **支持性治療**：退燒、止痛、補水；口腔痛可局部麻醉漱口。
- **心肌炎**：住院監測、心衰藥物治療（利尿、ACEi/ARB、β-blocker 依情況）、嚴重者暫時性機械循環支持；避免劇烈運動。
- **無特異抗病毒藥**（IVIG、pleconaril 於重症 / 新生兒個案考慮，非常規）。
- **EV71 重症預防**：台灣有 EV71 疫苗可預防 EV71 型手足口病重症（不涵蓋 Coxsackie）。

## ⚠️ 併發症
- 猛爆性心肌炎 → 心因性休克 / 猝死、擴張性心肌病。
- EV71 腦幹腦炎 → 神經性肺水腫、死亡 / 神經後遺症。
- 新生兒敗血症樣症候群、無菌性腦膜炎、脫水。

## 📊 嚴重度評分 / 風險分層
> 腸病毒重症分期（台灣疾管署）：第1期一般手足口 → 第2期中樞神經侵犯（肌抽躍 / 嗜睡）→ 第3期心肺衰竭；出現第2期徵象即需住院密切監測。
