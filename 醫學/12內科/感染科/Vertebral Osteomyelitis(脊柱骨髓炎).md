# Vertebral Osteomyelitis（脊柱骨髓炎）

> [!abstract] 一句話：椎體／椎間盤的感染，**最常經血行播散、金黃色葡萄球菌居首**；值班關鍵——**新發／惡化背痛 + 發燒 + 發炎指標高 → 想脊椎感染，做 MRI + 血培養**，並警戒「硬膜外膿瘍壓迫脊髓」這條**神經外科急症**時間窗。

> [!danger] 🚨 值班紅旗
> 1. **背痛 + 發燒 + 新發神經學缺損（下肢無力、大小便失禁、鞍區麻木）→ 硬膜外膿瘍壓迫脊髓，急做 MRI + 神外／脊椎外科會診**，減壓有時間窗（延誤 → 永久癱瘓）。
> 2. **夜間痛 / 休息不緩解的背痛 + 發炎指標（ESR/CRP）高 + 危險宿主（IVDU、洗腎、糖尿病、近期菌血症／術後）→ 別當單純機械性下背痛。**
> 3. **血流動力穩定、無神經缺損者，抗生素前盡量取得微生物學（血培養＋切片）**，不要盲目經驗投藥掩蓋病原（除非敗血症）。
> 4. **S. aureus 菌血症找不到源頭 → 想脊椎（與感染性心內膜炎並列要排除的轉移灶）。**
> 5. 慢性、亞急性、來自流行區或有肺 TB 病史 → 想 **結核性脊椎炎（Pott's disease）**，治療完全不同。

## 📖 定義 / 病生理（這是什麼）
- **定義**：椎體骨髓炎，常合併**椎間盤炎（discitis）與椎旁／硬膜外膿瘍**，合稱 spondylodiscitis。
- **感染途徑**：
  1. **血行播散（最常見）**：遠處感染灶（皮膚、泌尿、心內膜炎、靜脈導管、IVDU）經動脈到椎體終板。
  2. **鄰近蔓延 / 直接接種**：脊椎手術、硬脊膜外注射、外傷。
- **好發部位**：**腰椎最多** > 胸椎 > 頸椎（結核好發胸腰椎）。
- **常見病原**：
  - **金黃色葡萄球菌（S. aureus，居首，含 MRSA）**
  - 腸道 G(-)（*E. coli*、Klebsiella，泌尿源）、Streptococcus、CoNS（術後／植入物）
  - **IVDU**：S. aureus、**Pseudomonas**、Candida
  - **慢性 / 亞急性**：結核（Pott's）、Brucella（流行區）、黴菌
- **危險因子**：糖尿病、洗腎、免疫低下、IVDU、近期菌血症／血管導管、脊椎手術、高齡。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Back pain(背痛)]]、[[Fever(發燒)]]
- **背／頸痛**：漸進、持續、**夜間與休息不緩解**（發炎性），局部叩痛。
- **發燒**約半數（亞急性者可無明顯發燒 → 易延誤）。
- **神經學缺損**（進展警訊）：神經根痛、下肢無力／麻木、反射改變、括約肌功能障礙 → 硬膜外膿瘍壓迫。
- 全身：倦怠、體重減輕、盜汗（尤其 TB／慢性）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Blood culture(血液培養)]]、[[ESR-CRP(發炎指標)]]　影像 [[MRI spine(脊椎磁振造影)]]
- **MRI（含 gadolinium，首選）**：最敏感／特異，早期即可見椎體終板／椎間盤 T2 高訊號、骨髓水腫、椎旁／**硬膜外膿瘍**與脊髓壓迫。
- **血培養 ×2–3**：陽性率約 50–70%，陽性且與臨床相符**可免切片直接治療**。
- **CT 導引切片 / 培養**（血培養陰性時）：送需氧／厭氧／黴菌／**抗酸菌（TB）+ 組織病理**。
- **發炎指標 ESR / CRP**：幾乎都升高，敏感度高，也用於**追蹤療效**。
- 加驗：HIV、TB（IGRA）、Brucella 血清（流行區暴露史）。
- X 光早期常正常（2–8 週後才見終板破壞、椎間隙變窄）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **抗生素前取微生物學**（血培養＋必要時切片），盲目經驗治療會讓後續培養陰性、難以標靶。
- **ESR/CRP 正常幾乎可排除**（陰性預測值高）——但影像仍為診斷核心。
- **S. aureus 菌血症 → 主動想脊椎轉移灶**；反之脊椎感染要找源頭（含心內膜炎）。
- **結核性（Pott's）特徵**：胸腰椎、椎間盤相對保留、大椎旁冷膿瘍、後凸畸形（gibbus）、病程慢。
- **鑑別退化性 Modic change**：影像可像，靠臨床（發燒、發炎指標、菌血症）與 MRI 型態區分。

## 💊 治療（Acute / Chronic）
> 用藥 [[Vancomycin(萬古黴素)]]、[[Ceftriaxone(頭孢曲松)]]
> ⚠️ 劑量／療程依 IDSA 指引與培養結果，用藥前查藥典
### 抗生素
- **取得培養後**針對性靜脈抗生素；經驗性（敗血症或無法等）涵蓋 **MRSA（vancomycin）+ G(-)（ceftriaxone 或 cefepime）**。
- **療程一般 6 週靜脈／高生體可用度口服**（IDSA）；植入物、膿瘍未引流、S. aureus 者依情況延長。
- **結核性**：抗結核療程延長（常 9–12 個月）。
### 手術（神經外科 / 脊椎外科）
- **急診減壓 indication**：進展性神經缺損、脊髓壓迫、大硬膜外膿瘍、脊椎不穩／破壞、內科治療失敗、需取檢體。
- 引流膿瘍 + 必要時固定。
### 追蹤
- **ESR/CRP 連續追蹤**評估反應（趨勢比單一值重要）；MRI 影像改善落後臨床，不必過早重照。

## ⚠️ 併發症
- **硬膜外膿瘍 → 脊髓壓迫 → 永久性癱瘓 / 大小便失禁**（最嚴重）
- 敗血症、感染性心內膜炎（同源）
- 椎體破壞、脊椎不穩、後凸畸形、慢性疼痛
- 復發（療程不足、植入物、S. aureus）

## 📊 嚴重度評分 / 風險分層（scoring）
> 無單一 score；以神經狀態 + 病原 + 結構穩定分層
| 分層 | 用途 | 意義 |
| --- | --- | --- |
| 神經學缺損 | 手術急迫性 | 進展性缺損 / 脊髓壓迫 → 急診減壓 |
| 病原 | 療程 / 難度 | MRSA、TB、黴菌、植入物相關 → 較長療程 / 較差預後 |
| 結構穩定 | 是否需固定 | 椎體破壞、不穩 → 手術固定 |
| ESR/CRP 趨勢 | 療效監測 | 持續不降 → 治療失敗 / 引流不足 |
