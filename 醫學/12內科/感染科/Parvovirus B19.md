# Parvovirus B19

> [!abstract] 一句話：唯一致人類疾病的 parvovirus，親和紅血球前驅細胞（P antigen 為受體）；**兒童 → 巴掌臉第五病，成人 → 對稱性關節炎，溶血病人 → aplastic crisis，孕婦 → 胎兒水腫**。

> [!danger] 🚨 值班紅旗
> 1. **Aplastic crisis（再生不良危象）**：慢性溶血（鐮狀細胞、遺傳性球形紅血球、地中海貧血）病人突發嚴重貧血 + 網狀紅血球（reticulocyte）幾近 0 → 需輸血，可致命。
> 2. **孕婦感染**：病毒穿胎盤 → 胎兒重度貧血 → **胎兒水腫（hydrops fetalis）** / 流產，需轉產科胎兒監測（超音波 MCA 血流）。
> 3. **免疫低下（HIV、化療、移植）**：無法清除病毒 → 慢性純紅血球再生不良（pure red cell aplasia）→ 慢性貧血。
> 4. 出疹期兒童對溶血 / 免疫低下 / 孕婦具傳染力（但典型出疹的一般兒童此時已不太傳染）。

## 📖 定義 / 病生理（這是什麼）
- **病原體**：Parvoviridae，Erythroparvovirus 屬，非套膜、單股 DNA（最小的 DNA 病毒之一）。
- **受體 / 親和性**：**P antigen（globoside）** — 存在紅血球前驅細胞、胎兒心肌、內皮；病毒直接感染並溶解紅血球前驅細胞 → 暫時停止紅血球生成。
- **傳染途徑**：飛沫 / 呼吸道分泌物為主，亦垂直傳染、輸血 / 血品。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[皮疹]]、[[關節痛]]、[[貧血]]
- **兒童：Erythema infectiosum（第五病）** — 前驅微燒後出現**雙頰鮮紅「巴掌臉（slapped cheek）」**，續發軀幹四肢**網狀（lacy / reticular）紅疹**，可反覆數週。
- **成人（尤女性）**：對稱性多發**關節炎 / 關節痛**（手、腕、膝），可似類風濕，多自限。
- **溶血病人**：transient aplastic crisis — 急性貧血、蒼白、疲倦、心搏過速。
- **孕婦 → 胎兒**：hydrops fetalis、胎死腹中。
- **免疫低下**：慢性貧血、pure red cell aplasia。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Parvovirus B19 IgM/IgG]]、[[Reticulocyte count]]、[[PCR]]
- **免疫正常者**：血清學 **anti-B19 IgM（近期感染）/ IgG（既往免疫）**。
- **免疫低下 / aplastic crisis / 胎兒**：血清學可能陰性（無法製造抗體）→ **B19 DNA PCR**（血、骨髓、羊水）。
- **CBC**：貧血、reticulocytopenia；骨髓可見**巨大 pronormoblast（giant proerythroblast，"lantern cell"）**。
- 胎兒：超音波胎兒水腫、MCA-PSV 上升（貧血指標）。

## ⭐ 特徵 / 注意（pearls & pitfalls）
- 記憶口訣：**「臉紅（slapped cheek）、關節痛（成人）、血不生（aplastic crisis）、胎水腫（孕婦）」** 四大情境。
- 出疹 / 關節症狀屬**免疫複合體**反應 → 出疹時病毒血症多已消退、傳染力低；aplastic crisis 反而在病毒血症高峰、傳染力強。
- 骨髓 giant proerythroblast 是特徵。
- 與德國麻疹（rubella）鑑別：B19 有網狀疹 + slapped cheek。

## 💊 治療（Acute / Chronic）
> 用藥 [[IVIG]]、[[輸血]]
- **免疫正常**：支持性（NSAID 緩解關節痛），自限。
- **Aplastic crisis**：**輸血**支持至骨髓恢復（通常 1-2 週）；病人隔離避免傳給其他溶血 / 免疫低下 / 孕婦。
- **免疫低下慢性感染**：**IVIG**（含中和抗體）清除病毒；盡量恢復免疫（如調整化療 / ART）。
- **孕婦胎兒水腫**：轉母胎醫學，嚴重貧血行**胎兒宮內輸血（intrauterine transfusion）**。
- 無特異抗病毒藥、無疫苗。

## ⚠️ 併發症
- 致命性 aplastic crisis、慢性 pure red cell aplasia。
- 胎兒水腫、流產、死產。
- 少見：心肌炎、腦炎、血球減少（血小板 / 白血球）。

## 📊 嚴重度評分 / 風險分層
> 依宿主分層決定風險：免疫正常自限、溶血病人 → aplastic crisis 需輸血、免疫低下 → 慢性貧血、孕婦（尤其 <20 週）→ 胎兒水腫風險最高。
