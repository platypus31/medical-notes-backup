# Other Antibiotics（其他抗生素總覽）

> [!abstract] 一句話：Penicillin / Cephalosporin / Carbapenem / Quinolone / Aminoglycoside / Anti-MRSA 大類以外，**值班會遇到的其他重要抗生素**速查 — 記「抗菌譜 + 標誌性毒性」比記全劑量更實用。

> [!danger] 🚨 值班紅旗（毒性一句話記）
> 1. **Clindamycin / 廣效抗生素 → C. difficile 偽膜性結腸炎**（新發水瀉查 CDI）。
> 2. **Metronidazole + 酒精 → disulfiram-like reaction**（潮紅、噁心、心悸）；療程中與後 3 天禁酒。
> 3. **Daptomycin 不能治肺炎**（被 surfactant 抑制）；會使 **CK 上升 / 橫紋肌溶解**，停 statin 並驗 CK。
> 4. **Linezolid → 骨髓抑制**（長療程驗 CBC）+ **serotonin syndrome**（勿與 SSRI/MAOI 併用）。
> 5. **Aminoglycoside / Colistin → 腎毒性**；aminoglycoside 另有**耳毒性**（不可逆）。
> 6. **TMP-SMX → 高血鉀 / 腎功能惡化 / G6PD 溶血 / SJS**。
> 7. **Chloramphenicol → 灰嬰症候群、再生不良性貧血**（現極少用）。

## 📊 分類速查表
| 分類 | 代表藥 | 抗菌譜重點 | 關鍵毒性 / 注意 |
| --- | --- | --- | --- |
| **Macrolides** | azithromycin、clarithromycin、erythromycin | 非典型（Mycoplasma、Chlamydia、Legionella）、社區肺炎、百日咳、部分 G+ | **QT 延長**；clarithromycin/erythromycin 為 **CYP3A4 抑制劑**（交互作用多）；腸胃不適 |
| **Tetracyclines** | doxycycline、minocycline、tigecycline | 非典型、立克次體、萊姆病、CA-MRSA 皮膚軟組織、tigecycline 廣效含 MDR | **孕婦 / < 8 歲禁**（牙齒染色 / 骨影響）、光敏感、食道潰瘍（坐直服藥）；tigecycline 血中濃度低，**不用於菌血症 / UTI** |
| **Aminoglycosides** | gentamicin、amikacin、tobramycin | 好氧 Gram-negative（含 Pseudomonas）、與 β-lactam 協同治 G+（腸球菌 / 綠色鏈球菌 IE） | **耳毒性（不可逆）+ 腎毒性**；需監測 **trough**（每日單次給藥策略）；厭氧無效、膿瘍酸性環境效果差 |
| **Lincosamide** | clindamycin | 厭氧（橫膈以上）、G+ 含 CA-MRSA、**抑制毒素**（壞死性筋膜炎、TSS 加用） | **C. difficile 結腸炎風險高**；D-test 檢測誘導型抗藥 |
| **Nitroimidazole** | metronidazole | **厭氧（橫膈以下）**、原蟲（Trichomonas、Giardia、Entamoeba）、**C. difficile**、幽門螺旋桿菌 | **disulfiram-like reaction（禁酒）**、周邊神經病變（長療程）、金屬味 |
| **Sulfonamide 合劑** | TMP-SMX（cotrimoxazole） | **PCP（Pneumocystis）治療 / 預防**、UTI、CA-MRSA、Nocardia、Stenotrophomonas | **高血鉀、肌酸酐上升、G6PD 溶血、SJS/TEN、骨髓抑制**；與 warfarin/ACEI 交互作用 |
| **Nitrofuran** | nitrofurantoin | **單純下泌尿道感染（膀胱炎）** | **腎功能差（CrCl < 30-45）禁用**（尿中濃度不足 + 毒性）；不治腎盂腎炎 / 全身感染；肺纖維化（長期）、G6PD 溶血 |
| **Phosphonic** | fosfomycin | **單純 UTI**（含部分 ESBL / VRE 膀胱炎） | **單次 3 g PO** 便利；不用於上泌尿道 / 全身感染 |
| **Oxazolidinone** | linezolid、tedizolid | **VRE、MRSA**、G+（含肺炎、皮膚軟組織），口服吸收佳 | **骨髓抑制（血小板 / 貧血，> 2 週）、serotonin syndrome（MAOI 效應）、周邊 / 視神經病變** |
| **Cyclic lipopeptide** | daptomycin | **MRSA / VRE 菌血症、IE、皮膚軟組織** | **不能治肺炎（被 surfactant 去活化）**；**CK 上升 / 橫紋肌溶解**（每週驗 CK、停 statin）、嗜酸性肺炎 |
| **Polymyxin** | colistin（polymyxin E）、polymyxin B | **多重抗藥 Gram-negative 最後手段**（CRE、MDR Pseudomonas / Acinetobacter） | **腎毒性、神經毒性**；劑量計算複雜（依指引） |
| **Amphenicol** | chloramphenicol | 廣效（現極少用，特定情境 / 資源受限地） | **灰嬰症候群、劑量依賴性骨髓抑制、特異體質再生不良性貧血** |
| **Glycylcycline** | tigecycline | 廣效含 MRSA/VRE/ESBL/部分 CRE、腹腔 / 皮膚軟組織 | 血 / 尿濃度低 → **不用於菌血症 / UTI**；噁心嘔吐明顯、重症死亡率警訊 |

## ⭐ 值班速記（Pearls）
- **厭氧覆蓋分區**：橫膈**以上**厭氧（口咽 / 肺膿瘍）→ clindamycin；橫膈**以下**厭氧（腹腔 / 骨盆）→ metronidazole。
- **非典型肺炎**（Mycoplasma / Chlamydia / Legionella）→ macrolide 或 doxycycline 或 respiratory fluoroquinolone。
- **抗 MRSA 口服選擇**：doxycycline、TMP-SMX、clindamycin、linezolid（皮膚軟組織）；**菌血症 / IE** → vancomycin 或 daptomycin。
- **PCP 首選**永遠先想 **TMP-SMX**（重症合併類固醇）。
- **腎功能差三禁**：nitrofurantoin（CrCl < 30-45）、避免 aminoglycoside 累積、colistin 需調劑監測。
- **QT 延長組合**：macrolide + fluoroquinolone + 抗精神病藥要留意。
- **新發水瀉 + 近期抗生素** → 驗 C. difficile；治療用 **口服 vancomycin 或 fidaxomicin**（現行首選），metronidazole 退居次要。
