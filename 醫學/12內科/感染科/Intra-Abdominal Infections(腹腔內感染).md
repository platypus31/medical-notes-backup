# Intra-Abdominal Infections（腹腔內感染）

> [!abstract] 一句話：腹膜腔/內臟的感染，涵蓋**腹膜炎（primary/SBP、secondary、tertiary）**與**腹腔內膿瘍**。多為**腸道菌 + 厭氧菌的多菌種感染**。核心鐵律：**source control（源頭控制）+ 抗生素**——沒引流/沒修補穿孔，抗生素壓不住。

> [!danger] 🚨 值班紅旗
> 1. **急性腹痛 + 腹膜炎徵象（rebound、rigidity、板狀腹）+ 敗血症** → 外科急會診，**source control 是治療核心**
> 2. **肝硬化腹水 + 發燒/腹痛/意識改變 → 診斷性腹水穿刺**，**PMN ≥250/µL = SBP**，立刻抗生素（別等培養）
> 3. **游離氣體（free air）= 空腔臟器穿孔** → 外科急症
> 4. **抗生素前抽血培養 + 術中/引流檢體培養**；重症走敗血症 bundle，不因等檢延誤第一劑

## 📖 定義 / 病生理（這是什麼）
- **腹膜炎分類**：
  - **Primary / SBP（自發性細菌性腹膜炎）**：**無腹腔內穿孔**，多見**肝硬化腹水**，經血行/腸道移轉；**多為單一菌**（E. coli、Klebsiella、肺炎鏈球菌）
  - **Secondary（續發性）**：**空腔臟器穿孔/壞死/吻合口漏**（闌尾炎、憩室炎、消化性潰瘍穿孔、腸缺血、術後漏）→ **多菌種**（腸道 GNR + 厭氧 Bacteroides + 腸球菌）
  - **Tertiary**：續發性治療後持續/復發性腹膜炎（重症、抗藥菌/念珠菌，預後差）
- **腹腔內膿瘍**：局部化膿聚集（憩室、闌尾、術後、肝/脾/腎周）。
- **complicated（cIAI）**：感染超出單一臟器、進入腹膜腔或形成膿瘍。
- **社區型 vs 院內型**：院內型/術後/近期抗生素者抗藥菌（ESBL、Pseudomonas、腸球菌、Candida）風險高。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Abdominal pain(腹痛)]]、[[Fever(發燒)]]
- 腹痛、發燒、噁心嘔吐、腸蠕動異常
- **腹膜炎徵象**：反彈痛（rebound）、肌僵直（guarding/rigidity）、板狀腹、腸音減弱
- **SBP**：肝硬化病人**發燒、腹痛、腹水增加、肝腦病變/意識改變、腎功能惡化**——**症狀可很輕微**，門檻要低
- 進展為 [[Sepsis(敗血症)]] / [[Septic shock(敗血性休克)]]

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Paracentesis(腹水穿刺)]]、[[Blood culture(血液培養)]]　影像 [[Abdominal CT(腹部電腦斷層)]]
- **secondary/膿瘍**：**腹部 CT（含顯影，首選）**找穿孔、free air、膿瘍、來源；立位胸/腹 X 光看 free air。
- **SBP**：**診斷性腹水穿刺**送 cell count（**PMN ≥250/µL 即診斷**）、培養（床邊接種血培瓶提高陽性率）、生化（SAAG、蛋白、glucose、LDH 助分辨續發性）。
- **血液培養**、CBC、生化、乳酸、凝血；術中/引流檢體培養（含厭氧、必要時黴菌）。
- **鑑別 SBP vs 續發性腹膜炎**：續發性線索——PMN 極高、多菌、蛋白高、glucose 低、LDH 高、治療無反應 → 需影像找穿孔。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **source control 是預後決定因子**：穿孔要修補、膿瘍要引流（經皮/手術）、壞死要清除；**光靠抗生素治不好續發性腹膜炎**。
- **SBP 診斷靠 PMN ≥250，不需等培養陽性**（培養常陰性）；同時**白蛋白輸注**（尤 Cr/BUN/bilirubin 升高者）可降肝腎症候群與死亡。
- **多菌培養 + 蛋白高/glucose 低/LDH 高 → 想續發性（穿孔）**，不是單純 SBP，要找外科來源。
- **腸球菌與 Candida** 涵蓋在**院內型/術後/免疫低下/重症**才常規考慮，社區型輕中度多不需。
- SBP 病人後續**預防性抗生素**（曾 SBP、低蛋白腹水+特定條件）可降復發。

## 💊 治療（Acute / Chronic）
> 用藥 [[抗生素]]、[[Cephalosporin]]、[[Carbapenem]]、[[Other anti]]（metronidazole）
### SBP（自發性細菌性腹膜炎）
- **Ceftriaxone 2 g IV q24h**（或 cefotaxime）×5 天左右
- **併白蛋白輸注**（降肝腎症候群/死亡，尤腎功能惡化者）
- 復發預防：長期口服 fluoroquinolone/TMP-SMX（依適應症）
### 續發性 cIAI 經驗性（依社區 vs 院內、嚴重度）
- **社區型輕中度**：**Ceftriaxone + Metronidazole**，或 ertapenem，或 moxifloxacin
- **社區型重度 / 院內 / 高抗藥風險**：**Piperacillin-tazobactam**、或**抗綠膿頭孢（cefepime/ceftazidime）+ Metronidazole**、或 **carbapenem（meropenem/imipenem）**；院內/術後加考慮涵蓋**腸球菌 ± Candida**（antifungal 如 echinocandin）
- ⚠️ 具體劑量依體重/腎功能/當地指引查證
### Source control（核心，不可少）
- **穿孔修補、膿瘍引流（影像導引經皮 或 手術）、壞死組織清創、移除感染異物**；引流+抗生素後降階、療程多 **4–7 天（source 已控者）**。

## ⚠️ 併發症
- **敗血症 / 敗血性休克**、MODS
- **殘餘/復發膿瘍**、腸皮瘻、吻合口漏、腹腔室症候群
- SBP → **肝腎症候群、肝腦病變**、高復發與高死亡
- tertiary peritonitis（抗藥菌/念珠菌，重症）

## 📊 嚴重度評分 / 風險分層（scoring）
> 以「社區 vs 院內 + 嚴重度 + source 可否控制」決定抗生素廣度與手術時機
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **社區 vs 院內 / MDR 風險** | 選藥廣度 | 院內/術後/近期抗生素 → 更廣 ± 腸球菌/Candida |
| **腹水 PMN（SBP）** | 診斷 | **≥250/µL = SBP**，立即抗生素 |
| **qSOFA / SOFA / APACHE II** | 嚴重度預後 | 越高越重、越需積極 source control + ICU |
