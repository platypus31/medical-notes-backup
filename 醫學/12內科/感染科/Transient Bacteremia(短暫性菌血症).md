# Transient Bacteremia（短暫性菌血症）

> [!abstract] 一句話：日常活動或黏膜操作（刷牙、咀嚼、洗牙、內視鏡）造成的**短暫、低量、自限性**細菌入血，數分鐘內被網狀內皮系統清除，多數無症狀；臨床意義主要在於「感染性心內膜炎（IE）預防」的判斷。

> [!danger] 🚨 值班紅旗
> 1. **持續 / 反覆陽性血培養** → 不是短暫性菌血症，要找**血管內感染源**（IE、感染性血栓、血管內導管、人工植入物）。
> 2. **S. aureus 菌血症從來不當「污染 / 短暫」看待** → 一律當真、找感染源、追蹤血培養、評估 IE。
> 3. **單瓶陽性 coagulase-negative Staph / 類白喉桿菌**多為污染，但在人工瓣膜 / 植入物病人要提高警覺。
> 4. IE 預防**只給高風險心臟病患** + 高風險操作，不是人人術前吃抗生素。

## 📖 定義 / 病生理
**菌血症（bacteremia）三型別對比：**
| 型別 | 特徵 | 代表情境 |
| --- | --- | --- |
| **Transient（短暫性）** | 低量、數分鐘至 15-30 分鐘內清除、常無症狀 | 刷牙、咀嚼、洗牙、拔牙、內視鏡、導尿、正常黏膜操作 |
| **Intermittent（間歇性）** | 週期性入血，隨感染灶起伏 | 未引流膿瘍、蜂窩組織炎、肺炎、腹腔感染 |
| **Continuous / Persistent（持續性）** | 血中持續有菌 | **血管內感染：IE、感染性血栓、感染性動脈瘤、血管內導管感染** |

**機轉**：黏膜（尤其牙齦溝、腸道、泌尿道）有大量常在菌，破壞黏膜屏障時細菌短暫入血；健康宿主的網狀內皮系統（肝脾巨噬細胞、Kupffer cell）與補體迅速清除，故短暫且無臨床後果。**在有異常心臟結構者，這股短暫菌血症可能定殖於受損 / 人工瓣膜 → IE。**

## 🩺 臨床表現
- **多數完全無症狀**，屬生理現象，不需處置。
- 偶有一過性低度發燒 / 寒顫（尤其操作後短時間內）。
- **若出現持續發燒、栓塞現象、新雜音、Osler nodes / Janeway lesions → 已非「短暫」，往 IE 方向查。**
> 連症狀卡 [[Fever(發燒)]]

## 🔬 診斷
- **臨床情境判斷為主**：黏膜操作後短暫發燒、單套血培養偶陽而後續轉陰、病人無敗血徵象。
- **關鍵鑑別工具＝連續多套血培養**：
  - 短暫性 → 隨後血培養轉陰。
  - 持續性（IE）→ **多套血培養持續同一菌種陽性**（IE 主要 Duke criteria 之一）。
- 抽血培養規範：**不同部位、不同時間抽 2-3 套**以區分真菌血症與污染。
- 懷疑血管內來源 → 心臟超音波（TTE/TEE）、影像找感染灶。
> 檢查 [[Blood culture(血液培養)]]　影像 [[Echocardiography(心臟超音波)]]

## ⭐ 特徵 / 注意（pearls & pitfalls）
- **「短暫性菌血症」是回溯性判斷**：只有在血培養轉陰、病人穩定後才安心貼標籤；當下要先排除持續性感染。
- **牙科操作的菌血症風險其實與日常刷牙相近** → 這也是 2007 年後 AHA 大幅限縮預防性抗生素適應症的理由（日常暴露遠多於偶發牙科操作，普遍預防效益低、抗藥性 / 過敏風險高）。
- S. aureus、Candida、腸球菌陽性血培養**不可視為短暫 / 污染**。
- 免疫低下、無脾者清除能力差，短暫菌血症也可能演變嚴重。

## 💊 治療（Acute / Chronic）
- **短暫性菌血症本身不需抗生素治療**（自限、無感染灶）。
- **重點在於預防特定高風險族群的 IE**（IE prophylaxis，見下）。
- 若查出為持續性 / 有感染灶 → 依感染源（IE、膿瘍、導管）走該疾病之完整療程，非本卡範疇。

### 🦷 感染性心內膜炎預防（AHA 2007 / 2021 更新，僅高風險者）
**只有下列高風險心臟情況才建議預防：**
- 人工心臟瓣膜（含經導管植入 TAVR）或曾用人工材料修補瓣膜
- **曾患感染性心內膜炎**
- 特定先天性心臟病：未修補的發紺型 CHD、修補後 6 個月內、或修補處仍有殘餘缺損
- 心臟移植後**併發瓣膜病變**者

**且僅在下列操作前給予：**
- 涉及牙齦組織 / 牙根尖區 / 口腔黏膜穿孔的**牙科操作**
- 呼吸道黏膜切開 / 切片等侵入操作
- （感染皮膚 / 肌肉骨骼組織操作視情況）
- **不再常規建議**單純泌尿道 / 腸胃道操作前預防。

**用藥（術前 30-60 分鐘單次，劑量依指引 / 體重）：**
| 情境 | 藥物 |
| --- | --- |
| 口服首選 | **Amoxicillin 2 g PO 單次**（成人）|
| 無法口服 | Ampicillin IV/IM（或 cefazolin / ceftriaxone）|
| 青黴素過敏（非嚴重）| Cephalexin |
| 青黴素嚴重過敏 | Azithromycin / clarithromycin，或 doxycycline（依現行指引）|
> 用藥 [[Amoxicillin(安莫西林)]]

## ⚠️ 併發症
- 對絕大多數健康人：**無併發症**。
- 高風險心臟病患：定殖受損 / 人工瓣膜 → **感染性心內膜炎**。
- 有人工植入物（人工關節、血管移植物、心律調節器導線）者：遠端植入物感染（機率低但後果重）。

## 📊 嚴重度評分 / 風險分層
> 短暫性菌血症本身不做 severity 評分；風險分層在於「宿主端 IE 風險」。
| 分層 | 說明 |
| --- | --- |
| IE 高風險心臟情況 | 人工瓣膜 / 曾患 IE / 特定 CHD / 移植後瓣膜病變 → 需預防 |
| 操作風險 | 牙齦 / 根尖 / 黏膜穿孔牙科操作、呼吸道侵入操作屬高風險 |
| 菌種風險 | S. aureus / 腸球菌 / Candida 陽性 → 一律當真並找源 |
