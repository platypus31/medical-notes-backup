# Sepsis（敗血症）

> [!abstract] 一句話：**感染** + **失控的宿主反應** → **危及生命的器官功能障礙**（Sepsis-3 定義：疑似/確診感染 + **SOFA 分數急升 ≥2**）。核心是「感染引起的 organ dysfunction」，不是單純菌血症或發燒。

> [!danger] 🚨 值班紅旗
> 1. **qSOFA ≥2**（RR ≥22、SBP ≤100、意識改變任二）→ 高死亡風險，立刻升級評估
> 2. **Hour-1 Bundle 一小時內做完**：抽 lactate + 抽**血液培養（給抗生素前）** + 廣效抗生素 + 低血壓/lactate≥4 給晶體液 30 mL/kg + 需要時上升壓劑
> 3. **抗生素不要等培養結果**——經驗性廣效抗生素**每延遲 1 小時死亡率上升**（尤其已休克者）
> 4. **lactate ≥2 要重測**追蹤 clearance；持續高 lactate = 組織灌流未改善

## 📖 定義 / 病生理（這是什麼）
- **Sepsis-3 定義（2016）**：對感染產生**失調的宿主反應**（dysregulated host response）所致的**危及生命的器官功能障礙**。以 **SOFA 分數急性上升 ≥2 分**操作型定義器官障礙（假設 baseline SOFA=0）。
- 廢除舊「SIRS + 感染」及「severe sepsis」用語（SIRS 太敏感不特異）。
- **病生理**：病原相關分子（PAMPs）+ 損傷相關分子（DAMPs）→ 過度發炎 + 免疫麻痺並存 → 內皮受損、微血管滲漏、微血栓（DIC）、血管張力喪失 → **分布性休克** + 組織缺氧 + 粒線體功能障礙。
- **常見感染源**：肺（最常見）、腹腔、泌尿道、皮膚軟組織、導管相關、CNS。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Fever(發燒)]]、[[Altered mental status(意識改變)]]
- 發燒或**低體溫**（低體溫預後更差）、心搏過速、呼吸過速、寒顫
- **器官障礙徵象**：低血壓、少尿、意識改變/躁動、皮膚花斑（mottling）、微血管填充延遲、末梢冰冷
- 檢驗：**lactate 升高**、WBC 過高或過低、band 增加、血小板下降、凝血異常、Cr/膽紅素上升

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Lactate(乳酸)]]、[[Blood culture(血液培養)]]、[[CBC(全血球計數)]]　影像 [[Chest X-ray(胸部X光)]]
- **臨床診斷**：疑似感染 + 器官功能障礙證據（SOFA ↑≥2）。
- **必開**：血液培養 **≥2 套不同部位**（給抗生素前）、lactate、CBC/DC、生化（Cr、bilirubin、電解質）、凝血（PT/aPTT、D-dimer、fibrinogen）、CRP/procalcitonin、尿液常規+培養。
- **找感染源**：依症狀開 CXR、腹部影像、尿檢、痰培養、必要時 LP、影像找膿瘍。
- **procalcitonin**：輔助判斷細菌感染 + 指引抗生素停用（不可單獨作起始決策）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **qSOFA 是床邊快篩不是診斷工具**——2021 SSC 已不建議單用 qSOFA 作篩檢（敏感度不足），建議 SIRS/NEWS/MEWS 等搭配臨床。
- **老年/免疫低下可無發燒**，甚至以低體溫、意識混亂、跌倒表現 → 別漏。
- lactate 高 ≠ 一定是缺氧（也可 β-agonist、肝衰、痙攣）；但敗血症情境高 lactate 一律積極處理。
- 「培養前抽血」很重要，但**不可為了抽培養而延誤抗生素超過該給的時窗**（尤其休克）。

## 💊 治療（Acute / Chronic）
> 用藥 [[抗生素]]、[[Norepinephrine(正腎上腺素)]]
### 急性（Surviving Sepsis Campaign 2021 — Hour-1 Bundle）
1. **測 lactate**（≥2 需重測追蹤 clearance）
2. **給抗生素前抽血液培養**
3. **廣效經驗性抗生素**：疑似敗血症/敗血性休克應**盡早（休克者 1 小時內）**給藥；依感染源、當地抗藥性、宿主風險選（涵蓋 MRSA/Pseudomonas 視風險）
4. **晶體液**：低血壓或 lactate ≥4 → **30 mL/kg 晶體液**於前 3 小時內（之後改動態指標評估反應，避免過度輸液）
5. **升壓劑**：輸液後仍低血壓 → **norepinephrine 為第一線**，目標 **MAP ≥65 mmHg**
- **源頭控制（source control）**：能引流/移除的感染源（膿瘍引流、移除感染導管、清創）**盡快處理**，是預後關鍵。
- **降階（de-escalation）**：培養與感受性出來後依結果窄化抗生素；procalcitonin 輔助決定停藥。
### 慢性 / 恢復期
- **PICS（Post-Intensive Care Syndrome）**、sepsis 後長期認知/身體功能下降、再入院風險高 → 復健與追蹤。

## ⚠️ 併發症
- **敗血性休克**（[[Septic shock(敗血性休克)]]）、**MODS**（多重器官衰竭）
- **ARDS**、急性腎損傷（AKI）需透析、**DIC**、腸道缺血、肝功能異常
- ICU-acquired weakness、續發院內感染、PICS

## 📊 嚴重度評分 / 風險分層（scoring）
> qSOFA 快篩床邊；SOFA 定義器官障礙 + 追蹤嚴重度
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **SOFA** | 定義器官障礙 + 嚴重度 | 6 系統（呼吸/凝血/肝/心血管/CNS/腎）各 0–4，急升 ≥2 = sepsis 器官障礙；分數越高死亡率越高 |
| **qSOFA** | 床邊快篩（非診斷） | RR≥22 / SBP≤100 / GCS<15 各 1 分，≥2 高風險 |
| **APACHE II** | ICU 預後量化 | 綜合生理+年齡+慢性病，分數越高預後越差 |
