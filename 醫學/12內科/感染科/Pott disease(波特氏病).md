# Pott disease（波特氏病）

> [!abstract] 一句話：**脊椎的肺外結核（tuberculous spondylitis）**，最常見的骨骼結核，好發**胸腰椎**、破壞椎體與椎間盤前緣、形成**冷膿瘍（cold abscess）**。值班關鍵：**慢性背痛 + 結核風險 + 神經學缺損 → 想脊椎結核，別漏了脊髓壓迫這個外科急症**。

> [!danger] 🚨 值班紅旗
> 1. **背痛 + 下肢無力/麻木/大小便失禁 → 脊髓壓迫（Pott's paraplegia）**：**神經外科/骨科急照會 + 急做 MRI**，延誤會永久癱瘓。
> 2. **胸腰椎慢性背痛數週–數月 + 夜間痛/發燒/體重減輕 + 結核暴露/免疫低下/移民史** → 影像找骨破壞與椎旁膿瘍。
> 3. **冷膿瘍**可沿腰大肌下行到鼠蹊/大腿而**局部無紅熱**（跟化膿性不同），易被忽略。
> 4. 開始有效抗結核前務必**取檢體（切片/膿液）送 AFB/培養/PCR + 抗藥性**，並依規定**通報結核**。

## 📖 定義 / 病生理（這是什麼）
- **病原**：*Mycobacterium tuberculosis*，經**血行播散**到富含血流的椎體。
- **好發部位**：**下胸椎–腰椎**（T/L junction 最常見）。
- **病生理**：感染始於椎體前下緣 → 破壞椎體與**椎間盤**（相對後期才受累，與化膿性椎間盤炎不同）→ 相鄰椎體塌陷 → **後凸畸形（gibbus / kyphosis）**；乾酪樣壞死聚成**冷膿瘍**，可沿韌帶/腰大肌流注。
- **危險因子**：結核流行區、HIV/免疫低下、營養不良、糖尿病、既往肺結核。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Back pain(背痛)]]、[[Neurological deficit(神經學缺損)]]
- **慢性局部背痛**（數週–數月），可有夜間痛、活動受限。
- 全身：**低度發燒、盜汗、體重減輕、倦怠**（結核體質症狀）。
- 進展：**後凸畸形（gibbus deformity）**、椎旁/腰大肌冷膿瘍、脊髓/神經根壓迫（下肢無力、感覺異常、括約肌失禁）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[AFB smear/culture(抗酸菌染色培養)]]、[[TB-PCR]]、[[ESR/CRP]]　影像 [[MRI Spine(脊椎核磁共振)]]
- **MRI（首選）**：評估骨髓水腫、椎旁/硬膜外膿瘍、脊髓壓迫程度；典型**相對保留椎間盤、椎旁大膿瘍、多節/跳躍病灶**。
- **X 光**：晚期見椎體破壞、椎間隙變窄、後凸；早期常正常。
- **確診（取檢體）**：CT 導引切片或膿液做 **AFB 染色、分枝桿菌培養、TB-PCR（如 Xpert MTB/RIF）+ 藥敏**；病理見乾酪樣肉芽腫。
- 輔助：ESR/CRP 升高、IGRA/PPD（支持但不確診）、找其他部位結核（胸片）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- 與**化膿性椎體骨髓炎**鑑別：結核**病程慢、椎間盤相對保留、多節/跳躍、大椎旁冷膿瘍、後凸畸形**；化膿性起病快、單節、椎間盤早受累。
- **冷膿瘍無典型紅熱痛**，可下流到腰大肌/鼠蹊表現為腫塊。
- **一定要取組織/膿液做培養與藥敏**（抗藥結核影響療程與用藥）。
- 診斷需**高度懷疑**——胸腰椎慢性背痛 + 結核風險就要想。

## 💊 治療（Acute / Chronic）
> 用藥 [[Isoniazid]]、[[Rifampin]]、[[Ethambutol]]、[[Pyrazinamide]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 抗結核（主軸，療程長）
- **標準四合一（HRZE）**：Isoniazid + Rifampin + Pyrazinamide + Ethambutol **強化期 2 個月** → **維持期（HR）**，骨關節結核**總療程常延長至 9–12 個月**（依反應/指引）。
- 加 **維生素 B6（pyridoxine）**防 INH 週邊神經病變；監測肝功能。
- 疑抗藥結核依藥敏調整（MDR-TB 另循方案）。
### 手術 / 外科介入
- **適應症**：進行性神經缺損/脊髓壓迫、脊椎不穩或嚴重後凸、大膿瘍需引流、對藥物反應不佳。→ 減壓 + 清創 + 穩定融合。
- 神經缺損者及早外科可改善預後。

## ⚠️ 併發症
- **脊髓壓迫 → 截癱（Pott's paraplegia）**
- 後凸畸形（gibbus）、脊椎不穩/塌陷
- 冷膿瘍流注、瘻管
- 慢性疼痛、抗結核藥物毒性（肝毒、視神經、周邊神經）

## 📊 嚴重度評分 / 風險分層（scoring）
> 以神經功能與脊椎穩定度決定是否手術，無單一通用 score
| 分層依據 | 用途 | 意義 |
| --- | --- | --- |
| 神經功能（有無缺損/進展） | 決定急手術 | 進行性缺損/脊髓壓迫 → 急照會、可能手術 |
| 脊椎穩定 / 後凸角度 | 手術與否 | 不穩/嚴重後凸 → 融合固定 |
| 膿瘍大小/流注 | 引流 | 大膿瘍/壓迫 → 引流 |
