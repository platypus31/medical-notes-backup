# Encephalitis（腦炎）

> [!abstract] 一句話：**腦實質（parenchyma）發炎**，以**意識改變/人格行為改變 + 發燒 + 局部神經徵象/癲癇**表現（vs 腦膜炎主症是腦膜刺激徵）。**HSV-1 是最重要的可治療病因** → 懷疑就先給 acyclovir，別等 PCR。

> [!danger] 🚨 值班紅旗
> 1. **意識改變 + 發燒 + 局部神經徵象/癲癇/人格改變** → 疑腦炎，**立刻經驗性 Acyclovir 10 mg/kg IV q8h**，不要等 MRI/PCR
> 2. **HSV 腦炎延遲治療 = 高死亡與嚴重後遺症**；早期 acyclovir 顯著改善預後——寧可先給再依 PCR 停
> 3. Acyclovir **需充分水化 + 依腎功能調量**（結晶性腎病風險），慢速輸注（≥1 小時）
> 4. 常與腦膜炎重疊（meningoencephalitis）→ 同時涵蓋細菌性腦膜炎經驗抗生素直到排除

## 📖 定義 / 病生理（這是什麼）
- **腦炎 = 腦實質發炎 + 神經功能障礙**；與腦膜炎重疊時稱 **meningoencephalitis**。
- **感染性病因**：
  - **病毒（最常見）**：**HSV-1（最重要可治療）**、VZV、腸病毒、日本腦炎（台灣流行、夏季蚊媒）、EBV/CMV（免疫低下）、狂犬病、麻疹（SSPE）、arbovirus
  - 其他：細菌（Listeria 菱腦炎）、結核、真菌、寄生蟲
- **自體免疫/副腫瘤腦炎**：抗-NMDA receptor 腦炎（年輕女性 ± 畸胎瘤）、LGI1 等 → 感染排除後要想。
- **病生理**：病毒直接侵犯神經元 + 免疫介導發炎 → 腦水腫、神經元壞死；**HSV 好侵犯顳葉/邊緣系統**（temporal lobe）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Altered mental status(意識改變)]]、[[Seizure(癲癇發作)]]、[[Fever(發燒)]]
- **意識改變 / 認知行為人格改變**（核心，vs 腦膜炎的頸僵為主）
- 發燒、頭痛、**癲癇**、局部神經缺損（失語、偏癱、視野缺損）
- **HSV 腦炎**：顳葉症狀（記憶障礙、嗅幻覺、失語、行為異常、複雜局部癲癇）
- 自體免疫腦炎：亞急性精神症狀、運動障礙、自律神經不穩、癲癇

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Lumbar puncture(腰椎穿刺)]]、[[EEG(腦電圖)]]　影像 [[Brain MRI(腦部磁振造影)]]
- **CSF**：淋巴球增多、蛋白略↑、糖多正常（病毒型）；**送 HSV PCR（金標準）**、VZV/腸病毒 PCR；紅血球可見（HSV 出血性壞死）。
- **MRI（優於 CT）**：**HSV → 顳葉內側/額葉眼眶面 T2/FLAIR 高訊號**（常單/雙側不對稱）。
- **EEG**：瀰漫性慢波；HSV 可見**顳葉週期性側化癲癇樣放電（PLEDs）**。
- **自體免疫腦炎**：血清/CSF 自體抗體（anti-NMDAR 等）、找腫瘤（畸胎瘤）。
- ⚠️ **HSV PCR 早期（發病 <72h）可能偽陰**，臨床高度懷疑時陰性也要續治 + 重驗。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **懷疑就給 acyclovir，不等 PCR**——治療的機會成本遠低於漏掉 HSV 的災難。
- **HSV PCR 早期偽陰**：陰性不排除，48–72h 後重驗；臨床強烈懷疑續治。
- 腦膜炎 vs 腦炎：**腦膜炎主症頸僵、意識相對清**；**腦炎主症意識/行為/局部神經**（實務常重疊）。
- **年輕人亞急性精神+運動障礙+癲癇、感染檢查陰** → 想 **anti-NMDAR 腦炎**（找畸胎瘤，免疫治療有效）。
- 台灣夏季 + 蚊媒 + 鄉村 → 想**日本腦炎**（支持療法為主、有疫苗）。

## 💊 治療（Acute / Chronic）
> 用藥 [[Acyclovir(艾賽可威)]]
### 急性
- **經驗性 Acyclovir 10 mg/kg IV q8h**（依理想體重 + 腎功能調整；充分水化、慢輸注）——**懷疑立刻給**，HSV 確診療程 **14–21 天 IV**。
- 若同時未排除細菌性腦膜炎 → **併經驗性抗生素 ± dexamethasone**（見 [[Meningitis(腦膜炎)]]）。
- **支持療法**：癲癇控制、顱內壓管理、呼吸道保護、體液電解質。
- 日本腦炎/多數 arbovirus/腸病毒：**無特效抗病毒，支持療法**。
### 自體免疫腦炎
- 免疫治療（類固醇、IVIG、血漿置換，二線 rituximab/cyclophosphamide）+ 切除相關腫瘤。

## ⚠️ 併發症
- 頑固性癲癇/癲癇重積、腦水腫、顱內壓升高、腦疝脫
- 永久神經/認知/行為後遺症（HSV 存活者常見記憶與人格障礙）
- 死亡（未治療 HSV 腦炎死亡率極高）

## 📊 嚴重度評分 / 風險分層（scoring）
> 腦炎無單一通用 score；以意識、影像範圍、病因決定預後
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **GCS / 意識程度** | 重症與預後 | 昏迷、癲癇重積預後差 |
| **病因** | 可治療性 | HSV 早治可改善；狂犬病幾乎致命；自體免疫可逆 |
| **MRI 侵犯範圍** | 預後 | 廣泛雙側/深部侵犯預後較差 |
