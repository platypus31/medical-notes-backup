# Epstein-Barr Virus（愛潑斯坦-巴爾病毒）

> [!abstract] 一句話：EBV（HHV-4）是嗜 B 淋巴球的皰疹病毒，青少年/年輕成人原發感染引起**傳染性單核球增多症（IM，"kissing disease"）**：發燒 + 咽炎 + 頸淋巴結腫 + 脾腫大 + 非典型淋巴球。**避免打 ampicillin（誘發皮疹）+ 避免劇烈運動（脾破裂）**是值班重點。

> [!danger] 🚨 值班紅旗
> 1. **脾腫大 → 避免接觸性/劇烈運動 3–4 週**（自發性脾破裂罕見但致命，突發左上腹痛/休克要想到）
> 2. **給 amoxicillin/ampicillin 出現全身斑丘疹** ≠ 真過敏，是 IM + aminopenicillin 特有反應
> 3. **嚴重扁桃腺腫致呼吸道阻塞** → 可能需類固醇 + 耳鼻喉評估
> 4. EBV 相關**淋巴增生/血球吞噬症候群（HLH）**、移植後 PTLD、鼻咽癌/淋巴瘤關聯

## 📖 定義 / 病生理（這是什麼）
- **病原體**：EBV（human herpesvirus 4，dsDNA），感染 **B 淋巴球（CD21 受體）**與咽上皮，終身潛伏。
- **傳播**：**唾液**（親吻、共食器），潛伏期 4–6 週。
- **病生理**：B 細胞感染 → 活化的 **CD8 細胞毒性 T 細胞反應**（即血中「非典型淋巴球」，是宿主反應而非受感染細胞）→ 淋巴組織增生、脾腫大。
- **關聯疾病**：Burkitt/Hodgkin 淋巴瘤、鼻咽癌、移植後淋巴增生疾病（PTLD）、口腔毛狀白斑（HIV）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Fever(發燒)]]、[[Sore throat(喉嚨痛)]]、[[Lymphadenopathy(淋巴結腫大)]]
- **典型三聯**：**發燒 + 滲出性咽炎/扁桃腺炎 + 後頸/全身淋巴結腫**
- **脾腫大**（約半數）、肝腫大/輕度肝炎、疲倦（可持續數週）
- 眼瞼水腫、上顎瘀點、皮疹（尤其用 aminopenicillin 後）
- 兒童常無症狀或輕微

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Serology(血清學)]]、[[CBC]]
- **CBC + DC**：淋巴球增多，**>10% 非典型淋巴球（atypical lymphocytes）**
- **異嗜性抗體（Monospot / heterophile）**：快速篩，但**兒童/早期偽陰**
- **EBV 特異血清學**：**VCA-IgM（+）= 急性**、VCA-IgG（+）、**EBNA-IgG 延遲數月才陽性**（EBNA 陽性 = 過去感染）
- 肝功能常輕度異常；懷疑併發做腹部超音波（脾）

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **給 ampicillin/amoxicillin → 廣泛斑丘疹**（IM 典型，非真藥物過敏，記錄清楚）
- **VCA-IgM 陽 + EBNA 陰 = 急性原發**；**EBNA 陽 = 過去感染**（早期 EBNA 陰是重點）
- **脾破裂**：避免劇烈/接觸運動至少 3–4 週
- Monospot 早期/兒童可偽陰 → 臨床高度懷疑用特異血清學
- **鑑別**：CMV、急性 HIV、弓漿蟲、鏈球菌咽炎（可共存）

## 💊 治療（Acute / Chronic）
> 用藥 [[Corticosteroid(類固醇)]]
- **支持療法為主**：休息、退燒止痛、補水；**多自限 2–4 週**
- **無常規抗病毒**（acyclovir 對病程無明顯助益）
- **類固醇**：僅限**呼吸道阻塞（嚴重扁桃腺腫）、嚴重血小板低下/溶血、HLH** 等併發症，不常規使用
- **避免 aminopenicillin**、避免劇烈運動（脾）
- 併發症（HLH/PTLD/淋巴瘤）依血液腫瘤/感染科處理

## ⚠️ 併發症
- **脾破裂**（罕見致命）、上呼吸道阻塞
- 自體免疫溶血性貧血、血小板低下、嗜中性球低下
- 肝炎、腦炎/腦膜炎、Guillain-Barré、心肌炎
- **HLH（血球吞噬）**、慢性活動性 EBV、PTLD、EBV 相關惡性腫瘤

## 📊 嚴重度評分 / 風險分層（scoring）
> IM 無正式評分，重點是辨識併發症
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **臨床/血清學** | 診斷分期 | VCA-IgM+/EBNA− → 急性；EBNA+ → 過去 |
| **併發症警訊** | 決定介入 | 呼吸道阻塞/脾破裂/HLH → 積極處理 |
