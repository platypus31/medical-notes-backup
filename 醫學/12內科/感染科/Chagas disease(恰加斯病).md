# Chagas disease（恰加斯病 / American trypanosomiasis）

> [!abstract] 一句話：中南美洲 kissing bug 傳播的 *Trypanosoma cruzi* 原蟲感染；急性期常無症狀（偶見 Romaña sign），數十年後慢性期以擴張性心肌病與消化道巨器官（megaesophagus/megacolon）致命——can't-miss：來自中南美洲的不明原因擴張性心肌病 + 傳導阻滯。

> [!danger] 🚨 值班紅旗
> 1. **來自中南美洲**、有不明原因**擴張性心肌病 + 右束支傳導阻滯（RBBB）/ 左前分支阻滯**或猝死家族史 → 想到慢性 Chagas 心臟病。
> 2. **輸血 / 器官移植受贈者**或**新生兒**出現急性感染徵象 → 先天性與輸血傳播是非流行區的主要途徑。
> 3. **免疫抑制（HIV、移植後）宿主再活化（reactivation）**可致急性心肌炎或腦膜腦炎，病情兇猛。
> 4. 吞嚥困難 + 嚴重便秘 / 巨結腸腸扭轉（volvulus）→ 消化道型併發症，可急性腸阻塞。

## 📖 定義 / 病生理
- **病原體**：*Trypanosoma cruzi*（錐蟲，一種鞭毛原蟲 protozoa），細胞內增殖形成 amastigote。
- **傳播途徑**：
  - **媒介**：三角錐鼻蟲（triatomine bug / kissing bug）叮咬後排出含錐蟲的**糞便**，蟲糞經抓破的傷口或黏膜進入（非叮咬本身注入）。
  - **非媒介**：輸血、器官移植、**母嬰垂直傳播（先天性 Chagas）**、經口攝入（受汙染果汁/甘蔗汁，可致爆發）。
- **地理**：中南美洲地方性流行（墨西哥至阿根廷）。
- **病生理**：急性期原蟲血症與組織寄生 → 免疫反應清除多數 → 進入無症狀 indeterminate 期 → 部分（約 20–30%）數十年後因慢性發炎、神經節破壞（自律神經去神經化）與微血管病變 → 心肌與消化道器官擴張。

## 🩺 臨床表現
> 連症狀卡 [[心臟衰竭]]、[[Dysphagia(吞嚥困難)]]
- **急性期**（感染後數週，多數無症狀或輕微）：發燒、淋巴結腫、肝脾腫大。
  - **Romaña sign**：叮咬處在眼周時出現**單側眼周無痛性水腫**（結膜為入侵門戶的徵象）。
  - **Chagoma**：叮咬處皮膚硬結。
  - 重症急性心肌炎 / 腦膜腦炎少見（嬰兒、免疫低下者風險高）。
- **Indeterminate（無症狀潛伏）期**：血清陽性但無症狀、心電圖與影像正常，可持續數十年。
- **慢性期**（20–30% 進展）：
  - **心臟型**：擴張性心肌病（dilated cardiomyopathy）、心衰竭、**傳導異常（RBBB + 左前分支阻滯典型）**、心室心律不整、心尖動脈瘤、血栓栓塞、猝死。
  - **消化道型**：**megaesophagus 巨食道**（吞嚥困難、逆流，似 achalasia）、**megacolon 巨結腸**（慢性便秘、腸扭轉）。

## 🔬 診斷
> 檢查 [[血液抹片]]　血清學抗體　[[心電圖 EKG]]
- **急性期**：直接偵測血中錐蟲——**周邊血抹片（Giemsa）**見錐蟲、buffy coat 濃縮、PCR。
- **慢性期**：**血清學抗體**為主（原蟲血症極低），需**兩種不同原理的檢測同時陽性**才確診（IFA、ELISA、免疫層析）。
- **先天性**：新生兒微量血離心法查錐蟲；抗體因母源移行需 9 個月後再驗。
- **器官評估**：心電圖、心臟超音波、Holter；消化道型做食道/大腸攝影。

## ⭐ 特徵 / 注意（pearls & pitfalls）
- **Romaña sign** 是急性期經典但不常見的體徵，出現於眼結膜為入侵處時。
- 慢性 Chagas 心臟病的 EKG **RBBB + 左前分支半阻滯**組合是強烈提示。
- 巨食道臨床上與 **achalasia** 難分，病史（中南美洲居住）是關鍵鑑別線索。
- 台灣為非流行區，但**新住民、移工、輸血 / 移植**情境仍須警覺。
- 治療慢性期心臟病的抗寄生蟲效益有限，**愈早（急性 / indeterminate 期）治療效果愈好**。

## 💊 治療（Acute / Chronic）
> 用藥 [[Benznidazole]]、[[Nifurtimox]]
- **抗寄生蟲藥**：**benznidazole（首選）**或 **nifurtimox**，療程約 60（benznidazole）/ 60–90（nifurtimox）天，**依現行指引調整**。
  - 適應症：急性期、先天性、再活化、兒童與年輕成人的慢性 indeterminate 期。
  - 已進展的慢性心臟病抗寄生蟲效益有限。
- **支持治療**：心衰竭與心律不整標準處置（利尿劑、beta-blocker、抗心律不整、必要時 ICD / 起搏器）、抗凝（心尖動脈瘤 / 血栓風險）；消化道型行擴張術或手術（食道肌切開、乙狀結腸切除）。

## ⚠️ 併發症
- 擴張性心肌病、頑固性心衰竭、致命性心室心律不整、猝死。
- 心尖動脈瘤與心內血栓 → 系統性栓塞、腦中風。
- 巨食道（吸入性肺炎、營養不良）、巨結腸（腸扭轉、腸阻塞、穿孔）。
- 免疫抑制宿主再活化：急性心肌炎、腦膜腦炎。

## 📊 嚴重度評分 / 風險分層
- **Chagas 心肌病 Rassi risk score**：整合 NYHA class III/IV、心臟擴大、Holter 非持續性心室頻脈、心電圖低電壓、男性等因子，分低/中/高危預測 10 年死亡率。
- 臨床分期：急性 → indeterminate（血清陽性、器官正常）→ 慢性心臟型 / 消化道型。
