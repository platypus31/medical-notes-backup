# Syphilitic aortitis（梅毒動脈炎）

> [!abstract] 一句話：**三期（tertiary）梅毒**的心血管表現——*T. pallidum* 侵犯**升主動脈 vasa vasorum** 造成中膜壞死，典型導致**升主動脈瘤 + 主動脈瓣逆流**；感染後 **10–30 年**才發病，值班少見但破裂／夾層可致命，且是「可治療的動脈瘤病因」，別漏掉梅毒血清。

> [!danger] 🚨 值班紅旗
> 1. **升主動脈瘤 / 新發主動脈瓣逆流 + 梅毒危險因子 → 驗 RPR + TPPA**，這是「可用抗生素治療原因」的動脈瘤，別只當退化性。
> 2. **胸主動脈瘤破裂 / 夾層** → 撕裂痛、休克，外科急症（處置同 [[Aortic dissection(主動脈剝離)]]）。
> 3. **冠狀動脈開口狹窄（coronary ostial stenosis）**：可致心絞痛／猝死，運動時胸痛要想到。
> 4. 診斷心血管梅毒者**務必排除神經梅毒**（做 LP + CSF-VDRL），治療方案不同。
> 5. 治療用 **penicillin G**；青黴素過敏須**減敏**（doxycycline 資料不足，心血管／神經梅毒不建議替代）。

## 📖 定義 / 病生理（這是什麼）
- **病原 / 期別**：*Treponema pallidum* 未治療梅毒的**三期**表現（與神經梅毒、樹膠腫並列），潛伏期長達 **10–30 年**。
- **機轉**：螺旋體侵犯**主動脈 vasa vasorum → 閉塞性動脈內膜炎（obliterative endarteritis）→ 中膜彈性纖維與平滑肌壞死（medial necrosis）→ 主動脈壁擴張**。
- **好發部位**：**升主動脈（ascending aorta）與主動脈弓**（相對於退化性動脈瘤好發於降主／腹主，且梅毒**很少造成剝離**、少侵犯腹主動脈——與粥狀硬化的鑑別點）。
- **病理三結果**：
  1. **升主動脈瘤**（囊狀為主）
  2. **主動脈瓣環擴大 → 主動脈瓣逆流（AR）**
  3. **冠狀動脈開口狹窄**（vasa 內膜炎延伸至冠脈口）

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Chest pain(胸痛)]]、[[Dyspnea(呼吸困難)]]
- 多**長期無症狀**，影像意外發現升主動脈瘤 / 主動脈鈣化。
- **主動脈瓣逆流**：漸進性心衰、水衝脈、寬脈壓、舒張期雜音。
- **冠脈口狹窄**：勞力性心絞痛、心肌缺血、猝死。
- **動脈瘤壓迫**：聲音沙啞（喉返神經）、吞嚥困難、咳嗽、上腔靜脈壓迫、胸壁搏動性腫塊。
- 破裂 / 夾層：撕裂痛、休克。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[VDRL-RPR(梅毒血清)]]、[[TPPA(梅毒螺旋體血清)]]　影像 [[CTA(電腦斷層血管攝影)]]、[[Echocardiography(心臟超音波)]]
- **梅毒血清雙軌**：非特異（RPR/VDRL，三期可能低效價或陰性）+ 特異（**TPPA/TPHA/FTA-ABS，三期仍陽性**，是關鍵）。
- **CXR**：升主動脈擴大、**線狀鈣化（linear "eggshell" calcification of ascending aorta）**是經典提示。
- **CTA / MRA**：定位動脈瘤、範圍、冠脈口。
- **心臟超音波**：評估 AR 嚴重度、主動脈根部。
- **腰椎穿刺（CSF-VDRL、細胞、蛋白）**：排除同時存在的神經梅毒。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **「升主動脈鈣化 + 動脈瘤 + AR」三合一 → 想梅毒**（退化性／粥狀硬化好發降主與腹主，且較少單純升主動脈鈣化）。
- **梅毒動脈炎極少造成主動脈剝離**（中膜纖維化反而黏著）——與 Marfan／高血壓夾層鑑別。
- 三期 RPR 可陰性（burnt-out）→ **靠特異性 treponemal test（TPPA）確診**。
- 現代抗生素普及使其罕見，但 HIV 共感染、未治療潛伏梅毒族群仍會遇到。
- 治療只能**阻止進展、不能逆轉已成形的動脈瘤**——動脈瘤仍依大小手術。

## 💊 治療（Acute / Chronic）
> 用藥 [[Benzathine penicillin(苄星青黴素)]]、[[Aqueous penicillin G(水溶性青黴素G)]]
> ⚠️ 劑量依最新 CDC/台灣指引，用藥前查藥典
### 抗生素（殺螺旋體、阻止進展）
- **無神經侵犯之晚期梅毒**：**Benzathine penicillin G 240 萬單位 IM，每週 1 次 × 3 週（共 720 萬）**。
- **合併神經梅毒**：**Aqueous crystalline penicillin G 300–400 萬單位 IV q4h（或持續輸注）× 10–14 天**。
- 青黴素過敏 → **減敏後仍用 penicillin**（心血管／神經梅毒不建議 doxycycline 替代）。
- 注意 **Jarisch-Herxheimer 反應**。
### 手術（治療已成形的併發症）
- 升主動脈瘤達手術門檻（一般 ≥5.5 cm，或快速增大／症狀）→ **主動脈置換 ± 瓣膜手術**。
- 顯著 AR → 主動脈瓣置換／修補。
- 冠脈口狹窄 → 血運重建。

## ⚠️ 併發症
- **升主動脈瘤破裂**（最致命）、少數夾層
- **主動脈瓣逆流 → 左心衰竭**
- **冠狀動脈開口狹窄 → 心肌缺血 / 猝死**
- 動脈瘤壓迫（喉返神經、食道、氣管、SVC）

## 📊 嚴重度評分 / 風險分層（scoring）
> 無專屬 score；依動脈瘤大小 + 併發症決定手術
| 因子 | 用途 | 意義 |
| --- | --- | --- |
| 升主動脈瘤直徑 | 手術決策 | ≥5.5 cm、快速增大、症狀性 → 手術 |
| AR 嚴重度 | 瓣膜手術 | 重度 AR + 症狀／LV 擴大 → 換瓣 |
| 神經梅毒共存 | 決定療程 | CSF 陽性 → 改 IV penicillin 10–14 天 |
