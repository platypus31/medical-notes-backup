## - 筆記

| 特徵           | [[Asthma(氣喘)]]                                                                                                                                                                                                                                                                                                        | [[Chronic Obstructive Pulmonary Disease(慢性阻塞性肺病)]]                                                                                                                            |
| ------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **定義**       | 氣道慢性發炎，導致氣道高反應性及可逆性氣流阻塞。                                                                                                                                                                                                                                                                                  | 持續性氣流阻塞，通常是因暴露於有害顆粒或氣體所引起的氣道及/或肺泡異常。                                                                                                                             |
| **病因/病程**  | 過敏原、病毒感染、運動、冷空氣等→氣管慢性發炎。**主要為過敏體質相關。**<br><br>**可逆性氣流受限**（Reversible），症狀波動性大。                                                                                                                                                                                                                          | 長期吸菸、空氣污染、職業暴露等有害氣體→破壞氣道及肺泡（肺氣腫、慢性支氣管炎）。<br><br>**不可逆性氣流受限**，治療無法完全恢復，**呈進行性惡化**。                                                                                              |
| **發病年齡**     | 通常在**兒童期或青少年期**發病，但任何年齡皆可能。                                                                                                                                                                                                                                                                                    | 通常在**中年以後（>40-50歲）**發病，多有長期吸菸史。                                                                                                                                       |
| **臨床表現**     | **間歇性發作**，症狀多變：喘鳴（wheezing）、呼吸困難、胸悶、咳嗽。常在夜間、清晨或接觸誘發因子後加重。                                                                                                                                                                                                                                   | **持續性、進行性**的呼吸困難、慢性咳嗽、咳痰。症狀通常逐漸加重，活動後呼吸困難更明顯。                                                                                                                     |
| **免疫反應**     | CD4+ T cell中**Th2**引發mast cell, eosinophil（嗜伊紅性球）等發炎細胞活化，釋放介質（如IL-4, IL-5, IL-13）。<br><br>IgE介導的過敏反應常見。                                                                                                                                                                                                 | CD4+ T cell中**Th1**引發CD8+ T cell及macrophage（巨噬細胞）、neutrophil（嗜中性球）等免疫細胞。NF-kB & AP-1為活化發炎介質IL-8, TNF-α, IL-1β的重要轉錄因子。                                                      |
| **病理變化**     | 氣道上皮脫離(shedding)，Goblet cell（杯狀細胞）增加、黏膜分泌增加、平滑肌增生肥大、基底膜增厚、嗜伊紅性球浸潤。                                                                                                                                                                                                                             | 小氣道狹窄、閉塞、纖維化。肺實質破壞（肺氣腫：肺泡壁破壞，氣腔擴大）。氣道上皮扁平化、鱗狀化(squamous metaplasia)，嗜中性球浸潤（造成濃痰）。                                                                          |
| **肺功能**       | [[Obstructive lung disease(阻塞性肺疾病)]]：FEV1/FVC<70% 或低於正常預測值下限(LLN)。<br><br>Total Lung Capacity (TLC) 上升，Residual Volume (RV) 上升（因空氣滯留）。<br><br>**支氣管擴張劑反應 (Bronchodilator response)**：FEV1增加200cc 且 大於12%(成人看baseline，兒童跟predicted比) → **陽性**。<br><br>[[Methacholine provocation test(支氣管激發劑試驗)]]：FEV1減少20% (PC20指 FEV1減少20%所需要的Methacholine的濃度，<4mg/mL表示陽性) → **陽性** (氣道高反應性)。 | [[Obstructive lung disease(阻塞性肺疾病)]]：**支氣管擴張劑後 (post-bronchodilator) FEV1/FVC < 70%** → **不可逆性氣流受限**。<br><br>TLC 上升，RV 上升。<br><br>**支氣管擴張劑反應**：FEV1增加通常**不明顯**或不達診斷標準。<br><br>**氣道高反應性不常見**，Methacholine test通常為陰性或需更高濃度。 |
| **GOLD分級**     | 不適用。                                                                                                                                                                                                                                                                                                            | 依據支氣管擴張劑後FEV1百分比：<br>GOLD class 1 (輕度)：FEV1 ≥ 80% predicted<br>GOLD class 2 (中度)：FEV1 50~79% predicted<br>GOLD class 3 (重度)：FEV1 30~49% predicted<br>GOLD class 4 (極重度)：FEV1 < 30% predicted |
| **影像學 (胸部X光)** | 通常正常，發作時可見肺部過度充氣。                                                                                                                                                                                                                                                                                    | 肺部過度充氣、橫膈膜扁平、心臟下垂。重度肺氣腫可見肺泡破壞、肺大泡。                                                                                                                                   |
| **類固醇反應**     | **反應良好**，吸入性類固醇是主要控制藥物。                                                                                                                                                                                                                                                                                    | **反應有限**，主要用於急性惡化或嚴重病例的症狀控制。                                                                                                                                     |
| **預後**         | 若能妥善診斷與治療，症狀可得到良好控制，生活品質佳。                                                                                                                                                                                                                                                                                  | 疾病呈進行性惡化，雖可減緩惡化速度，但無法治癒。是全球重要的死因之一。                                                                                                                               |

---
- 補充

**總結鑑別要點：**

*   **可逆性 vs 不可逆性：** 氣喘是可逆的氣流阻塞，COPD是不可逆的。
*   **發作模式：** 氣喘是間歇性、波動性發作，COPD是持續性、進行性惡化。
*   **病因：** 氣喘多與過敏體質相關，COPD多與長期吸菸或有害暴露相關。
*   **發炎細胞：** 氣喘以嗜伊紅性球為主，COPD以嗜中性球、巨噬細胞為主。
*   **類固醇反應：** 氣喘對類固醇反應良好，COPD反應有限。

這個表格應該能提供一個非常全面且易於理解的氣喘與COPD比較。
