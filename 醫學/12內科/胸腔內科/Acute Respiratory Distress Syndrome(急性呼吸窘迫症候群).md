# Acute Respiratory Distress Syndrome（急性呼吸窘迫症候群）

> [!abstract] 一句話：**瀰漫性肺泡損傷**造成的**非心因性**急性低血氧性呼吸衰竭，特徵是雙側浸潤 + 頑固低血氧。值班最關鍵一點：**救命靠 lung-protective ventilation（低潮氣量 6 mL/kg PBW、Pplat < 30）**，不是靠藥物；找到並治療**underlying cause（多為敗血症/肺炎）**才是根本。

> [!danger] 🚨 值班紅旗
> 1. **急性發作（1 週內）雙側浸潤 + P/F ratio 低 + 排除心衰** → 就是 ARDS，別再等；先想「敗血症/肺炎源頭在哪」
> 2. **潮氣量 6 mL/kg 預測體重（PBW，非實際體重）、Pplat < 30 cmH₂O** — 大潮氣量 = 呼吸器誘發肺損傷（VILI），是可避免的醫源性死亡
> 3. **重度 ARDS（P/F < 100）→ 早期俯臥（prone）≥ 16 hr/day**（PROSEVA 證實降死亡率）
> 4. **保守輸液**（達成血行穩定後 net even/negative），過度輸液惡化肺水腫
> 5. 別誤當單純心衰狂打利尿劑就好 — ARDS 是發炎性微血管滲漏，PAWP ≤ 18、BNP 不高

## 📖 定義 / 病生理（這是什麼）
- ARDS是許多造成全身發炎疾病的肺部表現，肺本身有傷害（肺炎，肺部挫傷，胃液或氣體的吸入性傷害），肺外問題（敗血症，感染，燒燙傷，胰臟炎，開完刀等）
- **機轉**：直接/間接損傷 → 肺泡-微血管屏障破壞、通透性↑ → 富含蛋白質的肺水腫 + hyaline membrane 形成 → surfactant 失能、肺泡塌陷 → shunt、順應性下降、頑固低血氧。

|                    | 期別    | 天數    | 敘述                              |
| ------------------ | ----- | ----- | ------------------------------- |
| 滲出期(Exudate)       | 肺水腫   | 1~3天  | 發炎細胞浸潤，微血管通透性增加、肺泡滲出液           |
|                    | 透明膜   | 3~7天  | Eosinophilic的高蛋白滲出液沉着-透明膜       |
| 增生期(Proliferative) | 纖維增生期 | 7~10天 | 纖維母細胞、第二型肺泡細胞增生，膠原蛋白增加，肺間質發炎纖維化 |

- **常見病因**：**敗血症（最常見）**、肺炎、吸入（誤吸胃酸/溺水）、嚴重外傷/肺挫傷、急性胰臟炎、大量輸血（TRALI）、燒燙傷、藥物過量。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Hypoxemia(低血氧)]]
- 誘因後 6–72 小時內急性進行性呼吸困難、頻呼吸、發紺。
- 對氧氣反應差（refractory hypoxemia，因 shunt）。
- 雙側瀰漫性 crackles；常伴敗血症/休克的全身表現。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Arterial blood gas(動脈血液氣體分析)]]、[[BNP]]　影像 [[Chest X-ray(胸部X光)]]、[[Echocardiography(心臟超音波)]]
- 急性，PaO2/FiO2<200，CXR浸潤，無[[Congestive Heart Failure(鬱血性心臟病)]](PAWP<18mmHg)
- **柏林定義（Berlin 2012，現行標準）**：需滿足 4 項
  1. **時序**：已知誘因後 **1 週內** 急性發作。
  2. **影像**：雙側浸潤，非全由積水/塌陷/結節解釋。
  3. **來源**：呼吸衰竭**非**心衰或體液過量單獨解釋（必要時用心臟超音波客觀排除）。
  4. **氧合（PEEP ≥ 5）**：
     | 嚴重度 | PaO₂/FiO₂ |
     | --- | --- |
     | 輕度 Mild | 200 – 300 |
     | 中度 Moderate | 100 – 200 |
     | 重度 Severe | ≤ 100 |
- 註：2023 全球新定義納入 **SpO₂/FiO₂ ≤ 315**、HFNC ≥ 30 L/min、及超音波影像，讓資源有限處也能診斷。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **PBW（預測體重）用身高算**，不是量體重——胖病人給 actual weight 的潮氣量 = 過度膨脹肺。
- P/F ratio 要在 **PEEP ≥ 5** 下量才算數。
- 早期 CXR 可能落後於臨床（「白肺」出現前病人已很喘）。
- ARDS ≠ 診斷終點：**一定要找 underlying cause**（血/痰培養、影像找感染源）。
- 類固醇有角色但非全部：中重度早期 dexamethasone 可考慮（DEXA-ARDS），COVID-ARDS 更明確。

## 💊 治療（Acute / Chronic）
> 用藥 [[Dexamethasone(地塞米松)]]、[[Cisatracurium]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 急性（核心是呼吸器策略 + 治病因）
- 低TV低Pplat，維持PEEP，允許[[Hypercapnia(高碳酸血症)]]
- 類固醇
- **Lung-protective ventilation（ARDSNet，降死亡率的基石）**：
  - **潮氣量 6 mL/kg PBW**（可 4–8），**Plateau pressure < 30 cmH₂O**。
  - **允許性高碳酸血症**（permissive hypercapnia），維持 pH > 7.20。
  - PEEP 依 FiO₂ 用 PEEP/FiO₂ table 滴定；目標 SpO₂ 88–95%。
  - Driving pressure（Pplat − PEEP）**< 15** 與存活相關。
- **重度（P/F < 150）**：**俯臥 ≥ 16 hr/day**（PROSEVA 降死亡率）；早期 **神經肌肉阻斷（cisatracurium 48 hr）** 於重度可考慮（ACURASYS，惟 ROSE 未複製，選擇性使用）。
- **保守輸液策略**（FACTT：血行穩定後偏 negative balance，縮短呼吸器天數）。
- **難治性低血氧**：recruitment maneuver（謹慎，避免血行不穩）、吸入 NO/prostacyclin（暫時橋接）、**VV-ECMO**（P/F < 80 或無法維持保護性通氣，EOLIA/CESAR）。
- **類固醇**：中重度早期 dexamethasone 20 mg/day ×5 → 10 mg/day ×5（DEXA-ARDS）；COVID-19 相關用 dexamethasone 6 mg/day。
- **治病因**：敗血症早期抗生素、感染源控制。
### 慢性 / 復原期
- 脫離呼吸器、肺復健；存活者常見肌肉萎縮、認知與 PTSD（post-ICU syndrome）。

## ⚠️ 併發症
- 呼吸器誘發肺損傷（VILI）、氣胸/氣壓傷、呼吸器相關肺炎（VAP）。
- 多重器官衰竭（常與敗血症並行）。
- 肺纖維化（增生期後）、長期肺功能下降、ICU 後神經肌肉與認知後遺症。

## 📊 嚴重度評分 / 風險分層（scoring）
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **Berlin P/F ratio** | ARDS 嚴重度分級 | Mild 200–300 / Moderate 100–200 / Severe ≤100（越低死亡率越高） |
| **Driving pressure** | 呼吸器風險 | Pplat − PEEP > 15 與死亡率上升相關 |
| **Murray Lung Injury Score** | 舊式肺損傷量化 / ECMO 評估 | 分項越高越嚴重 |
| **SOFA** | 合併器官衰竭 | 綜合器官衰竭評估預後 |
