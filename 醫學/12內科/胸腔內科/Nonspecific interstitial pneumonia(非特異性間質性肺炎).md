# Nonspecific interstitial pneumonia（非特異性間質性肺炎）

> [!abstract] 一句話：一種**時序均一（temporally uniform）**的間質性肺炎，以**磨玻璃樣（GGO）+ 相對 subpleural sparing** 為特徵，常與**結締組織病（尤其硬皮症、抗合成酶症候群）**相關，**對類固醇/免疫抑制反應好、預後遠優於 IPF/UIP**。

## 📖 定義 / 病生理（這是什麼）
- **本質**：肺間質瀰漫、**均一**的發炎（cellular type）或纖維化（fibrotic type），病灶「同一時間點」發生（對比 UIP 的 temporal heterogeneity，新舊病灶並存 + fibroblastic foci）。
- **分兩型**：
  | 型別 | 組織 | 影像 | 反應/預後 |
  | --- | --- | --- | --- |
  | **Cellular NSIP** | 間質慢性發炎為主、少纖維化 | GGO 為主 | 類固醇反應極好、預後佳 |
  | **Fibrotic NSIP** | 均一纖維化、**無 honeycombing、無 fibroblastic foci** | GGO + 網狀 + traction bronchiectasis | 反應較差但仍優於 UIP |
- **常見關聯**：CTD-ILD（**硬皮症 SSc、皮肌炎/多發性肌炎、抗合成酶症候群 anti-Jo-1**）、藥物、慢性 HP；也有**特發性 NSIP**。
- 屬 restrictive 生理：FVC/TLC/DLCO↓（見 [[Interstitial Lung Disease(間質性肺疾病)]]）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Chronic cough(慢性咳嗽)]]
- **亞急性～慢性**勞動性喘 + 乾咳（病程通常比 IPF 短、較年輕、女性偏多）。
- 雙下肺 velcro crackles；**杵狀指較 IPF 少見**。
- **找 CTD 線索**：Raynaud、機械工手（mechanic's hands）、Gottron papules、關節炎、近端肌無力 → 抗合成酶/皮肌炎相關 NSIP。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Autoantibody panel(自體抗體)]]、[[Pulmonary function test(肺功能檢查)]]　影像 [[HRCT(高解析度電腦斷層)]]
- **HRCT**：**雙側對稱 GGO ± 網狀 + traction bronchiectasis，subpleural sparing（相對保留胸膜下區）是相對特異的線索**；**無/極少 honeycombing**（有 honeycombing 傾向 UIP）。
- **血清學**：ANA、anti-Scl-70、anti-Jo-1（及其他 anti-ARS）、anti-MDA5、肌酶 → 篩 CTD。
- **PFT + DLCO**、6MWT。
- **確診常需 MDD**；影像不典型或需與 UIP 區分時做**外科肺切片**（NSIP 是排除性病理診斷）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **NSIP 常是 CTD 的肺部先發表現** — 診斷 NSIP 一定要全面找 CTD，日後可能才浮現關節炎/肌炎。
- **subpleural sparing + GGO 為主、無 honeycombing → NSIP > UIP**；反之 honeycombing + basal/subpleural → UIP/IPF。
- 相較 IPF：**較年輕、女性、類固醇有效、預後好** — 分清楚才不會誤把可治的當 IPF 放棄。
- 影像可與 COP、慢性 HP、藥物性肺炎重疊，需臨床整合。

## 💊 治療（Acute / Chronic）
> 用藥 [[Prednisolone(潑尼松龍)]]、[[Mycophenolate(黴酚酸酯)]]、[[Azathioprine(硫唑嘌呤)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
- **一線：類固醇** — prednisolone 0.5–1 mg/kg/day（常見 ≤60 mg/day 起）數週後緩慢減量；cellular NSIP 反應尤佳。
- **合併/類固醇減量劑（steroid-sparing）**：Mycophenolate mofetil、Azathioprine；重症/CTD 相關可用 **Rituximab、Cyclophosphamide**。
- **Fibrotic NSIP 進行性纖維化（PF-ILD）**：可考慮 **Nintedanib**（INBUILD 涵蓋非 IPF 進行性纖維化）。
- 支持性：氧療、肺復健、疫苗、戒菸、治療原發 CTD。

## ⚠️ 併發症
- 進展為 fibrotic NSIP → 慢性呼吸衰竭
- 肺高壓 / 肺心症（[[Cor pulmonale(肺心症)]]）
- 急性惡化（較 IPF 少見但可發生）
- 免疫抑制治療的感染風險

## 📊 嚴重度評分 / 風險分層（scoring）
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **FVC / DLCO 追蹤** | 惡化評估 | 年降 FVC ≥10% / DLCO ≥15% = 進展 |
| **Cellular vs Fibrotic** | 預後 | Cellular 預後佳；Fibrotic 較差但仍優於 UIP |
