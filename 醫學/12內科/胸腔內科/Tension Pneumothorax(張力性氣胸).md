# Tension Pneumothorax（張力性氣胸）

> [!abstract] 一句話：胸膜腔氣體因**單向活瓣**只進不出→胸內壓持續↑壓迫縱膈與大血管→**靜脈回流塌陷、阻塞性休克**。這是**臨床診斷、不等 CXR** 的即刻致命急症，懷疑就先針刺減壓。

> [!danger] 🚨 值班紅旗
> 1. **喘 + 低血壓 + 患側呼吸音消失 + 氣管往健側偏移 + 頸靜脈怒張** → 張力性氣胸直到排除為止，**不要為了 CXR 而延遲**。
> 2. **治療先於影像**：立即**針刺減壓**——傳統第 2 肋間鎖骨中線，但成人胸壁較厚，**現多建議第 4/5 肋間腋前線（ATLS 10th）**，14–16G 針；聽到「噗」氣體噴出即暫時緩解。
> 3. 針刺**只是暫時**→ 必接**胸管（chest tube / thoracostomy）**做確定性引流。
> 4. **正壓通氣（插管／NIV／麻醉）病人氣體會急劇惡化** → 上呼吸器後突發低血壓＋SpO₂ 掉＋peak pressure 飆高，先想張力性氣胸。
> 5. 針刺減壓後仍休克 → 排除誤診（心包填塞、massive PE、血胸），或針太短沒進胸膜腔（超音波確認 lung sliding 消失、barcode/stratosphere sign）。

## 📖 定義 / 病生理（這是什麼）
- **機轉**：胸膜破口形成**單向活瓣（one-way valve）**，吸氣時氣體進入胸膜腔、吐氣時無法排出 → 胸內正壓進行性上升 → ①壓迫同側肺完全塌陷 ②縱膈往對側移位、壓迫對側肺 ③壓迫上下腔靜脈與心臟→**靜脈回流↓→心輸出↓→阻塞性休克 (obstructive shock)**。
- **與單純氣胸的差別**：張力性氣胸有**血流動力學不穩定**（休克），是外科／急救急症；單純氣胸多可觀察或引流。
- **危險因子 / 成因**：
  | 類別 | 例子 |
  | --- | --- |
  | 創傷 | 穿刺傷、肋骨骨折、鈍傷 |
  | 醫源性 | 中央靜脈導管、鎖骨下穿刺、胸腔穿刺、切片、**正壓通氣 barotrauma** |
  | 自發性 | 原發（高瘦年輕男、抽菸）、次發（COPD bullae、氣喘、肺囊腫、PJP） |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Shock(休克)]]、[[Chest pain(胸痛)]]
- **症狀**：突發劇烈**胸痛 + 進行性呼吸困難**、焦躁不安。
- **理學徵象（患側）**：呼吸音消失／減弱、**叩診過度共鳴（hyperresonance）**、患側胸廓膨脹不動。
- **全身**：**低血壓、心搏過速、頸靜脈怒張（JVD）、氣管往健側偏移、發紺**。
- 正壓通氣病人：peak airway pressure 突升、SpO₂ 驟降、突發低血壓。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Physical examination(理學檢查)]]　影像 [[Chest X-ray(胸部X光)]]、[[Lung ultrasound(肺部超音波)]]
- **臨床診斷**：血流動力學不穩定 + 上述徵象即可，**不應等待 CXR**（會延遲致命）。
- **床邊超音波（eFAST/BLUE）**：**lung sliding 消失**、無 comet-tail、M-mode 呈 **barcode / stratosphere sign**、找 lung point（特異）。比 CXR 快、敏感。
- **CXR（僅在穩定或減壓後補做）**：患側透亮度↑無肺紋、**縱膈往健側偏移、患側橫膈壓低、氣管偏移**。
- ⚠️ 血流動力學不穩定者**禁止**為了排隊 CT 而延誤減壓。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **張力性氣胸是臨床＋治療診斷，不是放射診斷**——「看到 CXR 才處理」是致命錯誤。
- 針刺減壓的成人**針長不足**是常見失敗原因；肥胖／肌肉厚者第 2 肋間鎖骨中線可能刺不進胸膜腔 → 改**第 4/5 肋間腋前線**。
- 針刺位置：**肋骨上緣**進針（血管神經束走在肋骨下緣，避開）。
- 上呼吸器後「三聯」——低血壓＋SpO₂ 掉＋peak pressure 飆 → 立刻鑑別張力性氣胸。
- 針刺後把單純氣胸「製造成」開放性氣胸 → 一定要接胸管確定性處理。

## 💊 治療（Acute / Chronic）
> 用藥 [[Oxygen therapy(氧氣治療)]]
> ⚠️ 劑量／器械依最新指引，操作前查 ATLS / 院內流程
### 急性（救命順序）
1. **高流量氧氣**、監測、建立 IV。
2. **立即針刺減壓（needle decompression）**：14–16G 靜脈留置針，第 4/5 肋間腋前線（或第 2 肋間鎖骨中線），沿**肋骨上緣**進針，聽到氣噴出即暫時解除張力。
   - 部分新流程（尤其醫院內、有經驗者）直接做 **finger thoracostomy** 更可靠。
3. **確定性處置：胸管（tube thoracostomy）** 置於**第 4/5 肋間腋前～腋中線（safe triangle）**，接水下引流。
4. 減壓後仍不穩 → 重新評估針位置、超音波確認、鑑別其他阻塞性休克。
### 慢性 / 後續
- 找並處理病因（bullae → 考慮 VATS bullectomy + pleurodesis 防復發）；戒菸；氣胸復發或雙側／職業（潛水、飛行員）評估手術。

## ⚠️ 併發症
- **阻塞性休克、心跳停止（PEA）**（最致命）
- 針刺／胸管併發：出血、肺實質傷、感染、放錯位置
- 復發性氣胸、支氣管胸膜瘻管、再擴張性肺水腫（re-expansion pulmonary edema，快速大量引流塌陷已久的肺時）

## 📊 嚴重度評分 / 風險分層（scoring）
> 張力性氣胸無正式評分——「有無血流動力學不穩定」即分層關鍵
| 項目 | 意義 |
| --- | --- |
| 血流動力學穩定 vs 不穩定 | 不穩定（休克/JVD/氣管偏移）＝張力性→**立即減壓** |
| 氣胸大小 (BTS: >2cm at hilum) | 用於**單純**氣胸決定觀察 vs 引流，非張力性判準 |
