# Respiratory Failure（呼吸衰竭）

> [!abstract] 一句話：氣體交換系統無法維持正常氧合或換氣——**Type I（PaO₂ <60）= 氧合衰竭**、**Type II（PaCO₂ >45 + 酸血）= 換氣衰竭**。值班最關鍵：**先判「保護氣道 / 撐通氣」的急迫性，再慢慢查病因**。

> [!danger] 🚨 值班紅旗
> 1. **意識改變（嗜睡、CO₂ narcosis）、無法維持氣道、silent chest、SpO₂ 給高流量仍 <90%** → 準備插管，不要死撐 NIV。
> 2. **COPD/慢性 CO₂ 儲留者給高濃度氧的目標 SpO₂ 88–92%**（過高 O₂ 會惡化 V/Q、抑制呼吸驅動、Haldane 效應 → CO₂ 飆更高），但**缺氧會致命——絕不因怕儲留而讓病人缺氧**。
> 3. **急性 Type II（pH <7.35 + PaCO₂↑）且清醒可配合 → 先試 NIV（BiPAP）**，1–2 小時 ABG 沒改善／惡化就插管。
> 4. **P/F ratio (PaO₂/FiO₂) ≤ 300 想 ARDS**；≤100 為重度，肺保護通氣（低潮氣量 6 mL/kg PBW、plateau <30）。
> 5. 找可逆病因：肺水腫（利尿）、氣胸（引流）、PE（抗凝）、鴉片過量（naloxone）、黏液栓（抽痰）、氣道異物。

## 📖 定義 / 病生理（這是什麼）
- **定義（ABG）**：室氣下 **PaO₂ < 60 mmHg** 和／或 **PaCO₂ > 45–50 mmHg 併酸血症**。
- **分型**：
  | 型 | 定義 | 機轉 | 代表病 |
  | --- | --- | --- | --- |
  | **Type I（低氧）** | PaO₂↓、PaCO₂ 正常或↓ | **V/Q mismatch、shunt、擴散障礙** | 肺炎、肺水腫、[[Pulmonary embolism(肺栓塞)]]、ARDS、[[Pulmonary Fibrosis(肺纖維化)]] |
  | **Type II（高碳酸）** | PaCO₂↑ 併酸血 | **肺泡換氣不足（pump failure）** | [[Chronic Obstructive Pulmonary Disease(慢性阻塞性肺病)]]、氣喘重積、神經肌肉病、鴉片過量、胸壁病 |
  | **Type III（術後）** | atelectasis | FRC↓ | 手術後 |
  | **Type IV（休克）** | 灌流不足 | 呼吸肌氧供不足 | 敗血症/心因性休克 |
- **急 vs 慢**：急性 pH 明顯異常（代償不及）；慢性 CO₂ 儲留者 HCO₃⁻ 代償、pH 接近正常（看 base excess／HCO₃ 判斷慢性）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Cyanosis(發紺)]]、[[Altered mental status(意識改變)]]
- **低氧**：呼吸急促、發紺、心搏過速、躁動、意識混亂。
- **高碳酸**：頭痛、嗜睡、asterixis（撲翼樣震顫）、warm peripheries、bounding pulse、CO₂ narcosis。
- **呼吸窘迫徵象**：使用輔助肌、三凹、paradoxical breathing（橫膈疲乏）、無法講完整句子、**silent chest（極重度氣喘/COPD 危險徵）**。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Arterial blood gas(動脈血液氣體分析)]]、[[Pulse oximetry(血氧監測)]]　影像 [[Chest X-ray(胸部X光)]]
- **ABG（核心）**：判 Type I/II、pH（急/慢）、A-a gradient、乳酸。
  - **A-a gradient**：升高 → shunt / V/Q mismatch / 擴散障礙；正常 + 高 CO₂ → 肺泡換氣不足（神經肌肉、藥物、中樞）。
- **CXR**：找肺炎、肺水腫、氣胸、積液、過度充氣。
- **其他**：CBC、電解質、心電圖、BNP、D-dimer/CTA（疑 PE）、必要時床邊超音波（B-lines、心功能）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **判斷 A-a gradient**：`A-a = [FiO₂×(760−47) − PaCO₂/0.8] − PaO₂`；室氣約 `150 − PaCO₂×1.25 − PaO₂`；正常 ≈ (年齡/4)+4。
- 高 CO₂ + **正常 A-a gradient** → 肺本身沒問題，是「幫浦」問題（中樞抑制、神經肌肉、藥物）。
- COPD 給氧惡化 CO₂ 的三機轉：解除低氧性血管收縮惡化 V/Q、Haldane 效應、降低呼吸驅動。
- NIV **禁忌**：意識喪失無法保護氣道、嘔吐/大量分泌物、臉部創傷、血流動力學極不穩、需立即插管者。
- 慢性儲留者 pH 接近正常但 CO₂ 很高 → 不要積極把 CO₂ 拉到正常（會鹼血、抽搐）。

## 💊 治療（Acute / Chronic）
> 用藥 [[Oxygen therapy(氧氣治療)]]、[[Bronchodilator(支氣管擴張劑)]]、[[Naloxone]]
> ⚠️ 劑量／FiO₂ 依 ABG 與最新指引調整
### 急性
- **氧療階梯**：nasal cannula → face mask → **HFNC（高流量鼻導管，Type I 首選之一）** → NIV → 插管。目標 SpO₂ 一般 ≥94%，**CO₂ 儲留族群 88–92%**。
- **Type II 急性酸血（pH <7.35）**：**NIV（BiPAP）** 為第一線（COPD 惡化、心因性肺水腫、免疫低下者證據最強）；1–2h 追 ABG，無改善即插管。
- **插管 + 機械通氣**指徵：氣道無法保護、NIV 失敗、心跳呼吸驟停、極度疲乏。
- **ARDS**：肺保護通氣（Vt 6 mL/kg PBW、plateau <30、允許性高碳酸、PEEP 依 FiO₂ 表、P/F<150 考慮俯臥）。
- **治病因**：抗生素、利尿、支氣管擴張劑＋類固醇、抽痰、抗凝、拮抗劑。
### 慢性
- 居家氧療（慢性缺氧 PaO₂ ≤55 或 ≤59+cor pulmonale/紅血球增多）、居家 NIV（慢性高碳酸 COPD / 神經肌肉病）、肺復健、戒菸、疫苗、治療原病。

## ⚠️ 併發症
- 心律不整、缺氧腦病變、多重器官衰竭
- 機械通氣：VAP（[[Ventilator associated pneumonia(呼吸器相關肺炎)]]）、barotrauma/氣胸、VILI、譫妄、ICU 衰弱
- 慢性：肺高壓、cor pulmonale、續發性紅血球增多

## 📊 嚴重度評分 / 風險分層（scoring）
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **PaO₂/FiO₂ (P/F ratio)** | ARDS 嚴重度（Berlin） | ≤300 輕、≤200 中、≤100 重 |
| **ROX index** (SpO₂/FiO₂ ÷ RR) | HFNC 是否會失敗 | <3.85（尤其 2、6、12h）→ 高失敗風險，考慮插管 |
| **pH / PaCO₂ 趨勢** | NIV 反應 | NIV 1–2h 後 pH↑、CO₂↓＝有效；反之插管 |
