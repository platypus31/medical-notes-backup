# Acute Bronchitis（急性支氣管炎）

> [!abstract] 一句話：大氣道（氣管支氣管）自限性發炎，**> 90% 是病毒**，核心症狀是**咳嗽（常 > 5 天、可拖 2–3 週）**。值班最關鍵一點：**絕大多數不需要抗生素**，主要工作是「排除肺炎」而非治療細菌。

## 📖 定義 / 病生理（這是什麼）
- **定義**：無慢性肺病基礎的病人，下呼吸道（氣管支氣管樹）急性發炎，以**咳嗽為主**、病程 < 3 週，無肺炎影像證據。
- **機轉**：病毒感染 → 支氣管上皮損傷、發炎 → 氣道過度反應（bronchial hyperresponsiveness）→ 咳嗽可持續數週（即使感染已清除）。
- **病原**：
  | 類別 | 常見病原 |
  | --- | --- |
  | **病毒（>90%）** | Influenza A/B、Rhinovirus、Coronavirus、RSV、Parainfluenza、Adenovirus、hMPV |
  | **非典型細菌（<10%）** | *Mycoplasma pneumoniae*、*Chlamydophila pneumoniae*、*Bordetella pertussis*（百日咳，成人慢性咳要想到） |
- **危險因子**：抽菸、空氣污染、上呼吸道感染流行季。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Cough(咳嗽)]]、[[Dyspnea(呼吸困難)]]
- **咳嗽為主**：初期乾咳，後轉有痰（黃綠痰**不代表**細菌感染，是發炎細胞與脫落上皮，勿據此開抗生素）。
- 常有前驅上呼吸道症狀（喉嚨痛、鼻塞、低燒）。
- 可有胸骨後灼熱、輕度喘鳴（wheezing）、聽診 rhonchi（會隨咳嗽變化）。
- **無 consolidation 徵象**：無高燒、無 tachypnea、無局部 crackles、無低血氧 → 傾向支氣管炎而非肺炎。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Sputum culture(痰液培養)]]　影像 [[Chest X-ray(胸部X光)]]
- **臨床診斷**：咳嗽 + 無肺炎證據。多數**不需影像**。
- **CXR 適應症（排除肺炎）**：出現以下任一 → 照 CXR
  - 心跳 > 100、呼吸 > 24、體溫 > 38°C
  - 局部聽診 consolidation（crackles、egophony、fremitus↑）
  - 老年、免疫低下、意識改變
- **百日咳（pertussis）**：陣發性咳嗽 + 吸氣哮聲（whoop）+ 咳後嘔吐、病程 > 2 週 → PCR / 培養。
- 流感季可考慮流感快篩（影響抗病毒決策）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **膿痰 ≠ 細菌感染**：這是最常見的過度開抗生素理由，勿中招。
- 咳嗽可**持續 2–3 週甚至更久**（post-bronchitic cough / airway hyperreactivity），不代表治療失敗。
- **鑑別重點**：肺炎（局部 crackles + 影像 consolidation）、氣喘/COPD 急性惡化、鬱血性心衰、GERD、後鼻滴漏、ACEI 引起的咳嗽、肺栓塞。
- 成人慢性咳（> 3 週）+ 陣發 → 想**百日咳**（台灣成人抗體下降，會傳染嬰兒）。

## 💊 治療（Acute / Chronic）
> 用藥 [[Dextromethorphan(右美沙芬)]]、[[Oseltamivir(克流感)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 急性（以症狀治療為主）
- **不常規給抗生素**（CHEST / ACP 明確建議）：無合併症的健康成人急性支氣管炎不需抗生素。
- **症狀緩解**：止咳（dextromethorphan）、NSAID/acetaminophen 退燒止痛、多喝水。
- **喘鳴/氣道過度反應明顯**：可短期吸入 β-agonist（SABA，如 salbutamol），但無喘鳴者不建議常規使用。
- **流感** 且 < 48 小時或高危：oseltamivir 75 mg BID × 5 天。
- **百日咳確診**：macrolide（azithromycin），主要目的是**減少傳染**（發病 3 週後給藥對病程已無益）。
### 慢性 / 預防
- 戒菸、流感與百日咳（Tdap）疫苗。

## ⚠️ 併發症
- 續發細菌性肺炎（少見）。
- 氣喘/COPD 病人可能誘發急性惡化。
- 長期咳嗽導致肋骨疲勞骨折、咳嗽性昏厥（少見）。

## 📊 嚴重度評分 / 風險分層（scoring）
> 急性支氣管炎本身不需 severity score；重點是「有沒有肺炎」。若疑肺炎 → 用 [[CURB-65]] / [[Pneumonia Severity Index(PSI)]]。
