# Interstitial Lung Disease（間質性肺疾病）

> [!abstract] 一句話：一大群侵犯**肺間質（肺泡壁／間隔）**的瀰漫性肺實質疾病，共同表現為**漸進性喘、乾咳、雙下肺 velcro rales、restrictive PFT + DLCO↓**。值班關鍵：先分「有無可逆病因（藥物／過敏原／結締組織病／感染）」再談抗纖維化，**急性惡化（AE-ILD）是致命的**。

## 📖 定義 / 病生理（這是什麼）
- [[Usual interstitial pneumonia(尋常性間質性肺炎)]] ：發炎末期，纖維化
  - CT下可見honeycombing, traction bronchiectasis
- [[Nonspecific interstitial pneumonia(非特異性間質性肺炎)]]：發炎期， CT下可見Ground glass opacity, consolidation，類固醇效果好
- 間質性肺疾病：[[Restrictive lung disease(限制性肺疾病)]]，FVC, FEV1, TLC, DLCO,PaO2皆減少，DLCO/VA(alveolar ventilation)不一定下降

- **本質**：反覆或持續的肺泡上皮／內皮損傷 → 異常修復 → 纖維母細胞增生、膠原沉積 → 肺間質增厚、氣體交換面積↓、肺順應性↓ → **restrictive + 擴散障礙**。
- **依病因分類（值班先想「可不可逆」）**：

  | 大類 | 代表 | 可逆性 |
  | --- | --- | --- |
  | **已知原因** | 藥物（amiodarone、bleomycin、MTX、nitrofurantoin）、放射、職業塵肺（silicosis、asbestosis） | 移除病因可能穩定 |
  | **過敏性肺炎（HP）** | 鳥/黴菌抗原（農夫肺、養鳥人肺） | 避開抗原 + 類固醇可逆 |
  | **結締組織病相關（CTD-ILD）** | RA、SSc（硬皮症）、皮肌炎/多發性肌炎（**anti-MDA5、anti-Jo-1**）、Sjögren | 治原病 + 免疫抑制 |
  | **肉芽腫** | [[Sarcoidosis(類肉瘤病)]] | 常自癒/類固醇反應好 |
  | **特發性間質性肺炎（IIP）** | IPF（=UIP pattern）、NSIP、COP、AIP | IPF 不可逆、抗纖維化 |
- **IPF vs NSIP 關鍵鑑別**：IPF = 老年、男、**UIP pattern（honeycombing 為主、subpleural/basal）、類固醇無效、抗纖維化**；NSIP = 較年輕、女、常合併 CTD、**GGO 為主、類固醇/免疫抑制反應好**。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Chronic cough(慢性咳嗽)]]
- **漸進性勞動性呼吸困難**（數月～數年）+ **持續乾咳**。
- 徵象：**雙下肺後基底 fine end-inspiratory velcro crackles（乾囉音，如撕開魔鬼氈）**、**杵狀指（clubbing，IPF 常見）**、晚期發紺 + 肺心症徵象（P2↑、下肢水腫）。
- 找系統性線索：關節痛/晨僵、皮疹、Raynaud、肌無力、乾眼乾口 → 提示 CTD-ILD。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Pulmonary function test(肺功能檢查)]]、[[Autoantibody panel(自體抗體)]]　影像 [[HRCT(高解析度電腦斷層)]]
- **HRCT（核心）**：判 pattern — UIP（honeycombing + traction bronchiectasis，subpleural/basal 分布）vs NSIP（GGO、相對 subpleural sparing）vs HP（mosaic、air-trapping、centrilobular nodule）。
- **PFT**：restrictive（TLC、FVC、FEV1↓，**FEV1/FVC 正常或↑**）+ **DLCO↓（最早、最敏感）**；6MWT desaturation 評估嚴重度與預後。
- **血清學（找 CTD）**：ANA、RF、anti-CCP、anti-Scl-70、anti-Jo-1、**anti-MDA5**、肌肉酵素（CK）；HP 抗原沉澱抗體。
- **BAL / 經支氣管或外科肺切片**：pattern 不典型、疑感染或需組織確診時。
- **MDD（multidisciplinary discussion）**：胸腔科＋放射科＋病理科共同會診是現行 ILD 診斷金標準。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **DLCO 是 ILD 最早、最敏感的異常**，可早於 CXR 變化。
- **UIP pattern ≠ 一定是 IPF** — 先排除 CTD、藥物、慢性 HP 才能診斷 IPF。
- **杵狀指 + velcro rales + 老年男性 → 高度懷疑 IPF**。
- **急性惡化（AE-ILD）**：一個月內新發低氧 + HRCT 新 GGO/consolidation 且非感染/心衰/PE 可解釋 → 死亡率極高（>50%）。
- 別把 ILD 的乾咳誤當「久咳未癒」拖延；amiodarone 用藥史 + 新肺浸潤要想到藥物性肺炎。

## 💊 治療（Acute / Chronic）
> 用藥 [[Nintedanib(尼達尼布)]]、[[Pirfenidone(吡非尼酮)]]、[[Prednisolone(潑尼松龍)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 依 pattern 分流
- **IPF（UIP）**：**抗纖維化** — Nintedanib 150 mg BID 或 Pirfenidone（逐步加至 801 mg TID）延緩 FVC 下降；**類固醇/免疫抑制對純 IPF 無效甚至有害**（PANTHER-IPF：pred+azathioprine+NAC 增死亡）。
- **NSIP / CTD-ILD / COP / HP**：**類固醇 ± 免疫抑制**（prednisolone 0.5–1 mg/kg/day 起，依反應減量；mycophenolate、azathioprine、rituximab）。SSc-ILD 可用 Nintedanib（SENSCIS）或 Mycophenolate（SLS-II）。
- **進行性纖維化表型（PF-ILD）**：不論病因，若持續進展 → Nintedanib（INBUILD）。
### 支持性 / 急性
- 長期氧療（靜息 SpO₂ <88%）、肺復健、疫苗（流感/肺炎鏈球菌）、**戒菸**、胃食道逆流治療。
- **AE-ILD**：高劑量類固醇（如 methylprednisolone pulse）、找並排除感染、支持性治療；預後差。
- **末期 → 肺移植評估**（IPF 是肺移植主要適應症之一）。

## ⚠️ 併發症
- **急性惡化（AE-ILD）**（致命）
- **肺高壓 + 肺心症**（[[Cor pulmonale(肺心症)]]）
- 慢性呼吸衰竭、續發感染
- **肺癌風險↑**（尤其 IPF）
- 抗纖維化藥副作用：Nintedanib 腹瀉/肝功能↑、Pirfenidone 光敏感/腸胃不適

## 📊 嚴重度評分 / 風險分層（scoring）
> IPF 預後用 GAP；追蹤看 FVC 年降幅
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **GAP index**（Gender, Age, 2 Physiology：FVC%、DLCO%） | IPF 死亡率分層 | Stage I–III，分數越高 1 年/3 年死亡率越高 |
| **FVC 下降** | 追蹤惡化 | 年降 ≥10%（或 DLCO 年降 ≥15%）= 顯著惡化、預後差 |
| **6MWT desaturation** | 運動去飽和 | SpO₂ <88% 預後差 |
