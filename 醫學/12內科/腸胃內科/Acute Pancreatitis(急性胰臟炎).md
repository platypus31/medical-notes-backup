# Acute Pancreatitis（急性胰臟炎）

> [!abstract] 一句話：

## 📖 定義 / 病生理（這是什麼）

- 台灣多因膽石症,外國多因酗酒

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[]]、[[]]

- 上腹痛,輻射到背後,向前屈坐可緩解

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[]]　影像 [[]]

- Amylase約2-12小時開始上升,2-5天回復正常值
- lipase(較具專一性)3-5天回復正常值(此兩數值具診斷價值但不具預後意義)
- 上腹痛加上胰臟酵素上升3倍有70%診斷率,若上升5倍以80%診斷率

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）


## 💊 治療（Acute / Chronic）
> 用藥 [[]]

- NPO
- 避免使用morphine
- Foy可減少病發生但不會改善死亡率
- necrotizing pancreatis建議使用抗生素(Imipenem)
- TPN對胰臟刺激較小但併發症較多(與給予腸道營養相比)

## ⚠️ 併發症

- 併發症：[[Pancreatic Pseudocyst(胰臟假性囊腫病)]]可先觀察大部分會自行緩解,若>6cm或持續>6個月則需引流

## 📊 嚴重度評分 / 風險分層（scoring）
> 用哪個 score 判斷 severity / 預後 / treatment 強度

- 住院先檢查:
  - 急性胰臟炎嚴重度評估：Ranson criteria


|               | 酒精性胰臟炎  | 非酒精性胰臟炎 |
| ------------- | ------- | ------- |
| Age           | >55     | >70     |
| Sugar         | >200    | >220    |
| AST           | >250    | >440    |
| LDH           | >350    | >400    |
| WBC           | >16000  | >18000  |
| 住院48小時        |         |         |
| Base deficit  | >4      | >5      |
| BUN上升         | >5      | >2      |
| Sequestration | >6L     |         |
| Ca            | <8mg/dL |         |
| Hct下降         | >10%    |         |
| PaO2          | <60mmHg |         |
 Ranson’s score之判定：3項以下為輕度急性胰臟炎(mild acute pancreatitis, MAP)，3項以上為重度急性胰臟炎(severe acute pancreatitis, SAP)，大於5項以上表示預後愈差
