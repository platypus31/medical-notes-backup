|                    | [[Mechanical obstruction(機械性腸阻塞)]]                 | [[Adynamic ileus(無動性腸阻塞)]]                                                 |
| ------------------ | -------------------------------------------------- | -------------------------------------------------------------------------- |
| KUB                | 小腸(近端)擴張<br><br>大腸(遠端)口徑變小                         | 廣泛性：大小腸皆擴張<br><br>局部性：局部擴張                                                 |
| 危險因子               | 術後沾黏/疝氣                                            | 任何腹膜傷害/低血鉀                                                                 |

| 理學檢查               | 絞痛時聽到大聲高頻的Borborygm                                | Hypoactive bowel sound                                                     |
| 治療 and<br><br>注意事項 | NG decompression/NPO<br><br>禁用Bucopan,Primperan等藥物 | 廣泛性：矯正鉀鎂離子<br><br>局部性：[[Pancreatitis(胰臟炎)]],[[Acute cholecystitis(急性膽囊炎)]] |

- [[Ischemic Colitis(缺血性大腸炎)]]好發：Giffith's point(脾彎曲)/Sudeck’s point(乙狀結腸)
- [[Acute Mesenteric Ischemia(急性腸繫膜缺血)]]：SMA(上腸系膜)emboli(血栓)造成
