# Peptic ulcer disease（消化性潰瘍）

> [!abstract] 一句話：

## 📖 定義 / 病生理（這是什麼）
- H. pylori：G(-),會產生Urease,糞口傳染,侵犯Antrum為主

|           | [[Corpus predominant gastritis(胃體炎)]]                                   | [[Antral predominant gastritis(賁門部胃炎)]]                                   |
| --------- | ----------------------------------------------------------------------- | ------------------------------------------------------------------------- |
| 致病機轉      | HP inf.(70%) or [[NSAID]]<br>=>胃酸下降/胃黏膜保護被破壞<br>=>胃體炎(Corpus gastritis) | HP inf.(90%)<br>=>賁門部胃炎(Antral gastritis)<br>=>酸分泌上升再流到十二指腸<br>=>破壞十二指腸黏膜 |
| 基礎酸分(BAS) | BAS<10=>與GERD較無關                                                        | BAS>20=>與[[GastroEsophageal Reflux Disease(胃食道逆流)]]較有關                    |
| 症狀        | 胃潰瘍(吃食物惡化)                                                              | 十二指腸潰瘍(吃食物緩解,凌晨被痛醒)                                                       |

| 位置            | cell                                  | 激素           |
| ------------- | ------------------------------------- | ------------ |
| Antrum(胃竇)    | G cell                                | Gastrin(胃泌素) |
| Body(胃體)      | Parietal cell                         | HCl          |
| Chief cell    | H+刺激使Pepsinogen=>Pepsin(胃蛋白酶)         |              |
| Parietal cell | Intrinsic factor(內在因子)=>幫助迴腸吸收Vit B12 |              |

## 💊 治療（Acute / Chronic）
> 用藥 [[]]
- 治療：triple therapy
  - PCM or PCA，HP根除率可達80~90%
   - a.PPI+Clarithromycin+Metronidazole for 14d
   - b.PPI+Clarithromycin+Amoxicillin for 14d
   - c.if有Clarithromycin抗藥性,考慮PPI+Metronidazole+Amoxicillin
  - Bismuth based treatment
   - a.Bismuth+PPI+Tetracycline+Metronidazole
