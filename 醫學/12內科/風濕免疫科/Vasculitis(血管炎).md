# Vasculitis（血管炎）

> [!abstract] 一句話：

## 📖 定義 / 病生理（這是什麼）
- 分類
  - T細胞及肉芽腫形成：delayed hypersensitivity
   - cell mediated immune injury=>granuloma
   - 與侵犯大血管的Takayasu arteritis/[[Giant cell arteritis(巨細胞血管炎)]]有關
  - Antineutrophil cytoplasmic Ab (ANCA)

|        | c-ANCA                                     | p-ANCA                                                                                      |
| ------ | ------------------------------------------ | ------------------------------------------------------------------------------------------- |
| 攻擊對象   | Anti-PR3(Proteinase 3)                     | Anti-MPO(myeloperoxidase)                                                                   |
| 中小型血管炎 | [[Wegener’s granulomatosis(衛格式肉芽腫)]] (90%) | [[Churg-Strauss syndrome(嗜伊紅性肉芽腫多發性血管炎)]]<br><br>[[Wegener’s granulomatosis(衛格式肉芽腫)]] (10%) |

  - 免疫複合物(IC)沉積：[[Systemic Lupus Erythema(紅斑性狼瘡)]]
   - [[Hepatitis B(B型肝炎)]]=>[[Polyarteritis Nodosa(結節狀多發性動脈炎)]]
   - [[Hepatitis C(C型肝炎)]]=>[[Cryoglobulinemia(冷凝球蛋白血症)]]，Henoch-Schonlein Purpura([[Anaphylactoid purpura(過敏性紫斑症)]])
