# Endocarditis（心內膜炎）

> [!abstract] 一句話：**心內膜／瓣膜的感染**（最常 [[Infective Endocarditis(感染性心內膜炎)]]），微生物黏附形成**贅生物（vegetation）**→ 可致命的敗血症、瓣膜破壞、心衰、與**栓塞（腦／脾／腎／肺）**。核心診斷靠 **3 套血培養 ＋ 心臟超音波（TTE→TEE）＋ modified Duke / 2023 Duke-ISCVID criteria**。「不明熱 ＋ 新雜音」直到排除前都當它。

> [!danger] 🚨 值班紅旗
> 1. **不明原因發燒 ＋ 新出現／變化的心雜音** → 感染性心內膜炎直到排除為止；抽血培養前**先抽足 3 套（不同部位、間隔）再給抗生素**
> 2. **人工瓣膜／IVDU／已知瓣膜病 ＋ 發燒** → 高危險群，門檻放低、盡早 TEE
> 3. **新發心衰 ／ 急性重度瓣膜逆流 ／ 血行動力不穩** → 瓣膜穿孔或急性 [[Aortic Regurgitation(主動脈瓣逆流)]]／[[Mitral Regurgitation(二尖瓣逆流)]]，**手術急會診**，別只靠抗生素
> 4. **新神經缺損／中風徵象 ＋ 發燒** → 贅生物**栓塞**或黴菌性動脈瘤（mycotic aneurysm），影像評估、影響手術時機
> 5. 血培養抽好前別亂給抗生素蓋掉病原（除非血行不穩須經驗性治療）——**培養陰性會毀掉後續選藥**

## 📖 定義 / 病生理（這是什麼）
- **定義**：心內膜（尤其**瓣膜**）的感染或發炎。臨床上「心內膜炎」多指 **Infective Endocarditis (IE)**；非感染性者另有 marantic（NBTE，惡性腫瘤／高凝）與自體免疫（[[Löffler’s endocarditis(勒夫勒心內膜炎)]]、Libman-Sacks/SLE）。
- **病生理三部曲**：① 內皮受損（湍流、瓣膜病、人工物）→ ② 血小板-纖維蛋白栓子（sterile vegetation）→ ③ 菌血症時微生物黏附增殖 → **感染性贅生物**（含菌、纖維蛋白、血小板；血流沖刷 → 栓塞、瓣膜破壞、膿瘍）。
- **分類**：
  | 分類 | 內容 |
  | --- | --- |
  | 病程 | **急性**（毒力強，如 *S. aureus*，數天惡化）／**亞急性**（毒力弱，如 viridans strep，數週） |
  | 瓣膜 | **自體瓣膜 NVE** ／ **人工瓣膜 PVE**（早發 < 1 年多院內菌、晚發 > 1 年似 NVE） |
  | 族群 | **IVDU 相關**（好發**右側三尖瓣**、*S. aureus*） |
  | 培養 | 培養陽性 ／ **培養陰性**（先前抗生素、HACEK、Coxiella/Bartonella 等挑剔菌） |
- **常見致病菌**：*Staphylococcus aureus*（現代最常見、急性、毒力強）、viridans group streptococci（口腔源、亞急性）、*Enterococcus*（泌尿／腸胃道術後）、coagulase-negative staph（人工瓣膜）、**HACEK** 群、培養陰性（Coxiella burnetii、Bartonella）。
- **危險因子**：**人工瓣膜**、既往 IE、先天性心臟病、後天瓣膜病（風濕性、退化鈣化、MVP 伴逆流）、**IVDU**、血液透析、免疫低下、留置導管／心內裝置（CIED）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Fever(發燒)]]、[[Syncope(昏厥)]]
- **全身感染**：發燒（最常見）、畏寒、盜汗、倦怠、體重下降、食慾差（亞急性可只是慢性消耗）。
- **心臟**：**新出現／變化的心雜音**（逆流為主）、心衰徵象（瓣膜破壞）。
- **周邊／免疫血管徵象（經典但現代少見，考試常考）**：
  | 徵象 | 機轉 | 特徵 |
  | --- | --- | --- |
  | **Osler node** | 免疫複合物（痛） | 指／趾墊**痛性**結節（**O**sler = **O**uch，痛） |
  | **Janeway lesion** | 敗血性栓塞（不痛） | 手掌／腳掌**無痛**紅斑 |
  | **Roth spot** | 視網膜出血伴白心 | 眼底檢查 |
  | **Splinter hemorrhage** | 甲床線狀出血 | 指甲下線狀暗紅 |
  | **瘀點 petechiae** | 微栓塞／血管炎 | 結膜、口腔黏膜、四肢 |
- **栓塞表現**：中風／TIA（左側贅生物）、脾梗塞（左上腹痛）、腎梗塞（血尿）、肺栓塞（右側 IE，IVDU）、急性肢體缺血。
- **脾腫大**（亞急性）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 血培養 ×3、CBC/CRP/ESR、尿液　影像 [[Echocardiography(心臟超音波)]]（TTE→TEE）
- **血培養（診斷基石）**：**至少 3 套**，不同穿刺部位、間隔取（理想跨 ≥ 1 小時，亞急性可分散取）；**抽好再給抗生素**。持續性菌血症是 IE 特徵。
- **心臟超音波**：
  - **TTE** 先做（快、非侵入），敏感度較低。
  - **TEE**：懷疑度高／TTE 陰性但臨床強烈懷疑／**人工瓣膜**／找瓣周膿瘍時；敏感度高，為金標準級。找 **vegetation、膿瘍、瓣膜穿孔、瓣膜裂開、逆流**。
- **輔助**：CBC（貧血、白血球↑）、CRP/ESR↑、rheumatoid factor、鏡下血尿、免疫複合物；不明熱＋高危可加 Q fever/Bartonella 血清學。
- **進階影像（2023 Duke-ISCVID 納入）**：**心臟 CT**（瓣周膿瘍、假性動脈瘤）、**¹⁸F-FDG PET/CT**（人工瓣膜／CIED IE 診斷，術後 3 個月內偽陽性須留意）、腦／全身影像找栓塞與 mycotic aneurysm。
- **診斷準則**：**Modified Duke criteria**，2023 已更新為 **2023 Duke-ISCVID criteria**（納入新影像與微生物證據）：
  - **Major**：① 典型 IE 病原之陽性血培養（符合持續性菌血症條件）／血清學／分子證據；② **影像證據**（echo 顯示 vegetation／膿瘍／瓣膜穿孔／人工瓣膜裂開，或 PET/CT、心臟 CT 陽性）。
  - **Minor**：易感心臟條件或 IVDU、發燒 ≥ 38°C、**血管現象**（栓塞、Janeway、mycotic aneurysm、顱內出血、結膜出血）、**免疫現象**（Osler node、Roth spot、GN、RF）、不符 major 的微生物證據。
  - **判定**：**Definite** = 2 major／1 major＋3 minor／5 minor（或病理確診）；**Possible** = 1 major＋1 minor／3 minor。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **Osler = 痛（Ouch）；Janeway = 不痛**——最愛考的一組對比。
- **右側 IE（三尖瓣）想到 IVDU**，表現常是**發燒＋肺部多發性栓塞／膿瘍**而非左心雜音。
- **血培養在給抗生素前抽足 3 套**是最重要的可修正錯誤；已給藥後培養陰性 = 選藥地獄。
- **TTE 陰性不能排除 IE**：高懷疑或人工瓣膜一定升級 **TEE**。
- **持續性 S. aureus 菌血症**要主動找 IE（門檻放低做 echo）。
- **培養陰性 IE** 想到：先前抗生素、HACEK、Coxiella（Q fever）、Bartonella、黴菌。
- **手術時機取決於**心衰、無法控制的感染（膿瘍、持續菌血症）、與**大贅生物／反覆栓塞**——別一味等「抗生素打滿療程」。
- **NBTE／Libman-Sacks（非感染性）**：血培養陰性、找惡性腫瘤或 SLE。

## 💊 治療（Acute / Chronic）
> ⚠️ 抗生素方案複雜、依**病原／瓣膜（自體 vs 人工）／MIC** 決定，**劑量查藥典**，此處只給原則不給硬數字
### 抗生素原則
- **經驗性治療（血培養抽好後、危重不能等時）**：涵蓋最可能病原——需**覆蓋 staph（含 MRSA）＋ strep ＋ entero**；具體用藥與劑量**依 2023 ESC／AHA IE guideline，依自體或人工瓣膜選擇，劑量查藥典**（⚠️需查證，勿憑記憶下劑量）。
- **針對性治療**：血培養／MIC 出來後 **de-escalate 到窄效**；strep、staph、entero、HACEK 各有標準方案（如 β-lactam ± aminoglycoside、MRSA 用 vancomycin/daptomycin 等），**依病原選藥、依 IDSA/AHA/ESC 表格與藥典定劑量與療程**（⚠️需查證）。
- **療程**：多為 **4–6 週**靜脈治療（人工瓣膜、entero、某些 staph 偏長）；**確切療程依病原與瓣膜類型查指引**（⚠️需查證）。
### 手術（antibiotic 之外的關鍵，勿延誤會診）
- **適應症（記憶三大類）**：
  1. **心衰**（重度急性瓣膜逆流／瘻管造成血行不穩）— 最主要、最緊急。
  2. **無法控制的感染**（瓣周膿瘍、瘻管、傳導阻斷擴大、持續菌血症 > 5–7 天正確抗生素仍陽性、黴菌／高抗藥病原）。
  3. **預防栓塞**（大贅生物 **> 10 mm** 且已有栓塞或位於前二尖瓣葉；反覆栓塞）。
- **人工瓣膜 IE、CIED 相關感染**多需移除／置換裝置。
### 支持與監測
- 監測心衰、栓塞、腎功能（抗生素毒性）、抗生素血中濃度（vancomycin/aminoglycoside 需 TDM，⚠️劑量查證）；追蹤血培養轉陰、發燒趨勢、重複 echo。
### 預防（prophylaxis，2007 AHA 後大幅限縮）
- **僅對「最高風險」病人**（人工瓣膜／瓣膜修補材料、既往 IE、未修補的紫紺型先天性心臟病或修補後殘餘缺損、心臟移植後瓣膜病變）於**高風險牙科操作**（牽涉牙齦／根尖周或口腔黏膜穿破）前給預防性抗生素。
- **一般瓣膜病、MVP、二尖瓣狹窄等不再常規預防**；具體藥物與劑量**依 AHA/ESC 預防指引與藥典**（⚠️需查證）。

## ⚠️ 併發症
- **心臟**：急性重度瓣膜逆流 → 心衰、心因性休克；瓣周**膿瘍**、瘻管、**傳導阻斷**（主動脈瓣環膿瘍侵犯 → 新 AV block）
- **栓塞**：腦（中風／TIA）、脾梗塞、腎梗塞、肢體缺血、腸系膜；右側 IE → 肺栓塞／膿瘍
- **黴菌性動脈瘤（mycotic aneurysm）** → 顱內出血
- **免疫相關**：免疫複合物性腎絲球腎炎
- 持續／復發菌血症、轉移性感染灶（脊椎骨髓炎、關節）
- 未治療致死率極高

## 📊 嚴重度評分 / 風險分層（scoring）
> 診斷用 Duke；預後／手術風險看併發症與病原
| 工具 | 用途 | 意義 |
| --- | --- | --- |
| **2023 Duke-ISCVID（原 Modified Duke）** | 診斷分層 | Definite／Possible／Rejected（見上診斷段） |
| 併發症盤點 | 手術時機 | 心衰／未控感染／栓塞風險三大類任一 → 評估手術 |
| 病原毒力 | 預後 | *S. aureus*、人工瓣膜、培養陰性、真菌 → 預後較差 |
| 贅生物大小／活動度 | 栓塞風險 | **> 10 mm**、活動度高 → 栓塞風險高 |
