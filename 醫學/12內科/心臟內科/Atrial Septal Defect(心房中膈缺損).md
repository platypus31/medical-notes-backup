# Atrial Septal Defect（心房中膈缺損）

> [!abstract] 一句話：**心房中膈的缺損造成左心房→右心房分流**，是**成人最常見、最晚才被診斷**的先天性心臟病之一；多數兒童期無症狀，長期右心容量負荷可致右心擴大、房性心律不整、肺高壓，末期甚至逆轉為 **Eisenmenger 症候群**。聽診金標記＝**固定分裂第二心音（fixed split S2）**。

> [!danger] 🚨 值班紅旗
> 1. **固定分裂 S2 ＋ 肺動脈瓣區收縮期血流雜音** 的成年人 → 想 ASD，即使無症狀
> 2. **不明原因栓塞性中風（尤其年輕人）＋ ASD/PFO** → **矛盾栓塞（paradoxical embolism）**，靜脈血栓經缺損跨到體循環
> 3. **已發紺（右→左分流）＝ Eisenmenger** → **禁止封閉缺損**（此時缺損是右心的減壓閥，關掉會急性右心衰死亡）
> 4. **新發房顫／房撲＋右心擴大** 可能是被忽略多年的 ASD 首次表現

## 📖 定義 / 病生理（這是什麼）
- **定義**：心房中膈上有交通，造成心房層次的分流。因左房壓 > 右房壓、且右心室較順應（compliant），血流方向為 **左心房 → 右心房（左→右分流）**。
- **分型（依解剖位置）**：
  - **Ostium secundum（續發孔型）**：最常見（約 70%），位於卵圓窩處 → **經導管封堵器**的主要適應症。
  - **Ostium primum（原發孔型）**：位於房室瓣附近，屬**房室中膈缺損（AVSD／endocardial cushion defect）譜系**，常合併二尖瓣裂（cleft mitral valve）與二尖瓣逆流；與 **Down syndrome（唐氏症）** 相關。
  - **Sinus venosus（靜脈竇型）**：位於上腔靜脈或下腔靜脈入口附近，**常合併肺靜脈異常回流（PAPVR，尤右上肺靜脈）**。
  - **Coronary sinus（冠狀竇型）**：最少見，冠狀竇頂缺損（unroofed），可合併左上腔靜脈殘存。
- **病生理連鎖**：左→右分流 → **右心房、右心室容量負荷 ↑** → 右心擴大 → 肺血流增加（Qp:Qs ↑）→ 長期肺血管重塑 → **肺動脈高壓** → 右心壓超過左心 → **分流逆轉（右→左）→ 發紺 ＝ Eisenmenger 症候群**（多為晚期、進展較 VSD/PDA 慢）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Palpitation(心悸)]]、[[Fatigue(疲倦)]]
- **多數兒童期無症狀**，常於成年（30–40 歲後）才出現：**運動耐受下降、勞力性呼吸困難、疲倦**。
- **房性心律不整**：心房擴大 → **心房顫動／心房撲動**、心悸。
- **矛盾栓塞**：靜脈系統血栓經 ASD 跨入體循環 → 中風、TIA、系統性栓塞。
- **反覆呼吸道感染**（肺血流增加）；晚期 → 右心衰、發紺（若 Eisenmenger）。
- **聽診經典組合（血流量增加造成的功能性雜音，非缺損本身發聲）**：
  - **固定分裂第二心音（fixed split S2）**：ASD 的招牌 —— 右心恆定容量負荷使肺動脈瓣關閉恆定延遲，分裂不隨呼吸變化。
  - **肺動脈瓣區收縮期噴射雜音**：流經肺動脈瓣的血流增加（相對性肺動脈瓣狹窄）。
  - **三尖瓣區舒張中期隆隆雜音（tricuspid rumble）**：大分流時流經三尖瓣血流增加（相對性三尖瓣狹窄），代表 Qp:Qs 大。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[ECG(心電圖)]]　影像 [[Echocardiography(心臟超音波)]]、[[Transesophageal Echocardiography(經食道心臟超音波)]]、[[Cardiac MRI(心臟磁振造影)]]
- **心臟超音波（首選）**：直接顯示缺損、分流方向（彩色都卜勒）、右心擴大、估算 Qp:Qs 與肺動脈壓。
  - **Bubble study（agitated saline contrast）**：注入攪拌生理食鹽水微氣泡，觀察氣泡經缺損進入左心 → 證實右→左分流成分／矛盾栓塞路徑。
  - **經食道超音波（TEE）**：secundum 封堵器術前評估邊緣（rim）、sinus venosus / PAPVR 顯示更佳。
- **ECG**：
  - **Secundum ASD** → **右軸偏移（RAD）＋ 不完全性右束支傳導阻滯（incomplete RBBB / rSR′ in V1）**。
  - **Primum ASD** → **左軸偏移（LAD）／電軸左上偏**（房室中膈受累特徵）＋ 一度房室阻滯。
  - 晚期：右心房擴大、房顫。
- **CXR**：心臟擴大（右房右室）、**肺動脈主幹突出、肺血管紋增加（plethora）**。
- **心導管**：測 Qp:Qs、肺血管阻力（PVR）—— 評估是否仍可閉合（Eisenmenger 排除）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **fixed split S2 是考試與臨床的招牌**——正常 S2 分裂隨吸氣增寬、吐氣變窄；ASD 因右心恆定容量負荷而「固定不變」。
- **ASD 的雜音不是缺損本身**（心房壓差太小不發聲），而是**血流量增加流經肺動脈瓣／三尖瓣**造成的相對性狹窄雜音。
- **Secundum → RAD/RBBB；Primum → LAD**（電軸方向是快速分型線索）。
- **矛盾栓塞**：年輕人不明原因中風要找 ASD/PFO ＋ 下肢靜脈血栓。
- **Eisenmenger（已發紺）是封閉的絕對禁忌**——先評估 PVR 可逆性。
- Sinus venosus 型**常合併 PAPVR**，單看 secundum 思路會漏診 → 分流量與缺損大小不符時想到。
- 小型 secundum ASD（<5 mm）可自然閉合；primum/sinus venosus 不會自癒。

## 💊 治療（Acute / Chronic）
> 用藥 [[Warfarin(可邁丁)]]、[[Diuretics(利尿劑)]]（心律不整／心衰對症）
- **閉合適應症**：**顯著左→右分流（Qp:Qs > 1.5）** 或 **右心房/右心室擴大**，且肺血管阻力尚可逆 → 建議關閉（改善運動耐受、預防房性心律不整與右心衰）。
- **Secundum ASD**：**經導管封堵器（transcatheter device closure）為首選**（邊緣足夠時），微創、成效佳。
- **Primum / Sinus venosus / Coronary sinus 型**：解剖不適合封堵器 → **外科修補**（primum 常需同時修補二尖瓣裂；sinus venosus 需重導 PAPVR）。
- **心律不整**：房顫／房撲 → 心率控制 ＋ 依風險抗凝；必要時電燒。
- **禁忌**：**Eisenmenger（右→左分流、發紺）不可閉合**，改走肺高壓標靶治療／心肺移植評估。
- 現行指引已不常規建議 ASD 病人的感染性心內膜炎抗生素預防（除非合併其他適應症）。

## ⚠️ 併發症
- **心房顫動／心房撲動**（心房擴大）
- **肺動脈高壓 → Eisenmenger 症候群**（右→左分流、發紺、不可逆）
- **矛盾栓塞 → 缺血性中風／系統性栓塞**
- **右心衰竭**
- 妊娠期血流動力學惡化；封堵器相關併發症（少見：移位、侵蝕）

## 📊 嚴重度評分 / 風險分層（scoring）
> ASD 無專用命名 score，以 **Qp:Qs 分流量 + 右心大小 + 肺血管阻力（PVR）** 決定是否閉合
| 指標 | 用途 | 數值 → 意義 |
| --- | --- | --- |
| **Qp:Qs（肺/體血流比）** | 分流顯著度 | **> 1.5 ＝ 顯著分流**，建議閉合 |
| **右心室/右心房擴大** | 容量負荷證據 | 有 RV 容量負荷 ＝ 閉合適應症（即使 Qp:Qs 邊緣） |
| **肺血管阻力（PVR, Wood units）** | 可逆性判斷 | PVR 明顯上升／固定 ＝ 接近 Eisenmenger，閉合風險高／禁忌 |
| **分流方向（bubble study）** | Eisenmenger 判定 | 右→左分流、發紺 ＝ 禁止閉合 |
