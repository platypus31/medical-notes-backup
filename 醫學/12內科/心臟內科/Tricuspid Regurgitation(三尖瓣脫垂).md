# Tricuspid Regurgitation（三尖瓣逆流）

> [!abstract] 一句話：三尖瓣關閉不全 → 收縮期血液由右室逆流回右房，造成**體循環靜脈鬱血（JVD、肝腫、腹水、下肢水腫）**。**絕大多數是「功能性/續發性」**——因左心病或肺高壓致右室／瓣環擴張，而非瓣膜本身壞掉。（註：本卡即三尖瓣閉鎖不全 TR；純三尖瓣脫垂較少見）

> [!danger] 🚨 值班紅旗
> 1. **右心衰徵象（頸靜脈怒張、肝腫大、腹水、下肢水腫）＋收縮期雜音吸氣加強（Carvallo sign）** → 想 TR
> 2. **先找根因**：TR 多續發於左心衰／肺高壓／二尖瓣病／AF → 治根因比治瓣膜先
> 3. **感染性心內膜炎的 TR**：IVDU（靜脈藥癮）＋發燒＋敗血性肺栓塞 → 右側心內膜炎，查血培＋echo
> 4. **重度 TR＋肝鬱血／腎鬱血（cardiorenal）** → 利尿要兼顧前負荷，過度利尿反致低輸出

## 📖 定義 / 病生理（這是什麼）
- **定義**：三尖瓣收縮期關閉不全，右室血液逆流入右房 → 右房壓↑、體循環靜脈鬱血、右室容積負荷。
- **分類（機轉決定治療）**：
  - **功能性／續發性（最常見，~80–90%）**：瓣葉正常，因**右室擴大、三尖瓣環擴張、右室重塑**——多**續發於左心疾病、肺高壓、二尖瓣病、[[Atrial Fibrillation(心房顫動)]]（房性功能性 TR）**。
  - **原發性（瓣膜本身）**：感染性心內膜炎（尤 IVDU）、風濕性、類癌（carcinoid）、Ebstein anomaly（[[Ebstein's anomaly(埃勃斯坦畸形)]]）、瓣膜脫垂、外傷、**導線相關（pacemaker/ICD lead）**、鈍傷。
- **病生理循環**：TR → 右室容量負荷 → 右室／瓣環進一步擴大 → TR 更重（惡性循環）；長期 → 右心衰、肝鬱血（cardiac cirrhosis）、低心輸出。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Edema(水腫)]]、[[Ascites(腹水)]]
- 常被**左心病症狀掩蓋**；獨立症狀多為右心衰：**腹脹、下肢水腫、體重增加、疲倦、運動耐受↓、右上腹脹痛（肝鬱血）**。
- **理學徵象**：**頸靜脈怒張伴明顯 CV wave**、**吸氣時增強的胸骨左緣全收縮期雜音（Carvallo sign）**、**搏動性肝腫大**、腹水、下肢壓凹水腫、右室抬舉。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[ECG(心電圖)]]、[[BNP(利鈉肽)]]、[[Liver function test(肝功能)]]　影像 [[Echocardiography(心臟超音波)]]
- **心臟超音波（首選）**：確認 TR、分級嚴重度（逆流 jet 寬度／vena contracta／PISA/EROA、肝靜脈收縮期逆流）、評估右室大小與功能、瓣環直徑、估**肺動脈收縮壓（由 TR jet 速度推算）**、找根因（左心病／肺高壓／導線）。
- **ECG**：右房擴大、AF、右室肥厚（肺高壓）。
- **實驗室**：BNP、肝功能（鬱血性肝病）、腎功能（cardiorenal）。
- 進階：右心導管（量肺壓與阻力）、心臟 MRI（右室量化）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **Carvallo sign（雜音吸氣增強）**是區分右側（TR）與左側雜音的 pearl（吸氣增回心血量 → 右心雜音↑）。
- **頸靜脈大 CV wave＋搏動性肝**是重度 TR 的床邊線索。
- **絕大多數 TR 是功能性的**——別急著換瓣，先問「右室／肺壓／左心／AF 出了什麼事」。
- **導線相關 TR（pacemaker/ICD）** 是常被忽略的原發原因。
- 重度 TR 常合併肝、腎鬱血 → 利尿與後負荷處理要小心低輸出。
- **孤立性重度 TR 若拖到右室衰竭才處理，外科死亡率高**——近年**經導管介入（T-TEER/瓣膜置換）**興起。

## 💊 治療（Acute / Chronic）
> 用藥 [[Diuretics(利尿劑)]]、[[Aldosterone antagonist(醛固酮拮抗劑)]]
### 內科（多數功能性 TR 的主力）
- **治根因**：左心衰 GDMT、降肺高壓病因、控制 AF、處理二尖瓣病。
- **利尿劑（loop ± MRA）**減體循環鬱血；限鈉。
- 感染性心內膜炎 → 抗生素（依培養）。
### 介入 / 手術
- **原發重度 TR 有症狀**，或**做左心瓣膜手術時併同修補**（有瓣環擴大或既有 TR）→ **三尖瓣修補（瓣環成形 annuloplasty）優於置換**。
- 孤立性重度症狀性 TR、手術高風險者 → **經導管修補（tricuspid TEER）或經導管瓣膜置換**（新興治療，選擇性病人）。
### 慢性
- 長期利尿、根因追蹤、右室功能監測；導線相關者評估導線處置。

## ⚠️ 併發症
- **右心衰竭、鬱血性肝病（cardiac cirrhosis）、腎鬱血（cardiorenal）**
- 心房顫動、低心輸出、腹水／全身水腫（anasarca）
- 感染性心內膜炎（原發者）；術後殘餘／復發 TR

## 📊 嚴重度評分 / 風險分層（scoring）
> Echo 分級為主；新增「重度以上」分層（massive/torrential）
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **Echo TR 分級** | 嚴重度 | 由 vena contracta、EROA、jet、肝靜脈逆流等綜合判 mild/moderate/severe |
| **擴充分級（近年）** | 重度以上細分 | severe → massive → **torrential**（EROA 更大），影響介入決策 |
| **右室功能／肺壓** | 預後與手術風險 | 右室衰竭、肺高壓越重預後越差、手術風險越高 |
