# Right Bundle Branch Block（右束支傳導阻滯）

> [!abstract] 一句話：右束支傳導受阻 → 右心室延遲除極，QRS 增寬(≥120 ms)、**V1–V2 呈 rsR′（兔耳 M 型）、I 與 V6 有寬鈍 S 波**。常為良性/正常變異，但**新發 RBBB + 胸痛可能是近端 LAD 阻塞**、與 PE/右心負荷相關。

> [!danger] 🚨 值班紅旗
> 1. **新發 RBBB + 缺血性胸痛** → 可能**近端 LAD 阻塞**（預後差），積極評估 ACS
> 2. **新發 RBBB + 喘/低血氧/右心負荷徵象** → 想大面積 [[Pulmonary embolism(肺栓塞)]]
> 3. **RBBB + 左前分支阻斷 (LAFB)（雙束支阻斷）+ 昏厥** → 高風險進展為完全房室阻斷，備節律器
> 4. **RBBB 不像 LBBB 會遮蔽 MI**：仍可從 V1–V3 以外導程判讀 ST 變化

## 📖 定義 / 病生理（這是什麼）
- **右束支阻斷** → 右心室由左室慢慢傳導延遲除極。
- **診斷準則（完全性 RBBB）**：
  - **QRS ≥120 ms**
  - **V1–V2**：**rsR′ / rSR′（M 型「兔耳」）**，R′ 高
  - **I、aVL、V5–V6**：寬而鈍的 **S 波**（除極晚期向右）
  - 繼發性 ST-T：V1–V3 ST 輕度下移、T 波倒置（discordant）
- **病因**：**可為正常變異**（尤其年輕人、不完全性）；RV 壓力/容積負荷（[[Pulmonary embolism(肺栓塞)]]、肺高壓、[[Atrial Septal Defect(心房中膈缺損)]]）、缺血、退化性傳導疾病、心肌病

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Palpitation(心悸)]]
- 多**無症狀**，ECG 偶然發現
- 症狀依背後病因（PE、ASD、缺血、心衰）
- 可有寬分裂第二心音(wide S2 split)

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 12-lead ECG（比對舊片是否新發）　影像 Echo（RV/ASD/肺高壓）；疑 PE → CTPA
- **12-lead ECG** 依上述準則（完全性 ≥120 ms、不完全性 100–120 ms）
- **與舊 ECG 比對**：新發者臨床意義較大
- **Echo**：評估 RV、ASD、肺高壓、LVEF
- 依情境：疑 PE → D-dimer/CTPA；疑缺血 → troponin/導管

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
![[Pocket Medicine The Massachusetts General Hospital Handbook of Internal Medicine, 8th Edition.pdf#page=20&rect=77,654,536,706&color=note|Pocket Medicine The Massachusetts General Hospital Handbook of Internal Medicine, 8th Edition, p.20]]
- **RBBB 不遮蔽 MI 判讀**（相對 LBBB）：仍可看 ST 抬高/壓低
- **新發 RBBB + 前壁缺血症狀 → 想近端 LAD occlusion**（間隔支缺血），預後不良
- **孤立、長期存在、不完全性 RBBB** 在無心臟病者多為良性
- **雙束支阻斷（RBBB + LAFB 或 LPFB）** 或 **三束支（+ 一度 AV block）+ 昏厥** → 節律器適應症評估
- 判讀 V1：真 RBBB 的 R′ > r；與 Brugada Type 1（coved）要區分（見 [[Brugada syndrome(布魯蓋達氏症候群)]]）

## 💊 治療（Acute / Chronic）
> 用藥 —（RBBB 本身無專屬藥物，治病因）
- **無症狀孤立 RBBB**：不需治療，找並追蹤病因
- **合併 ACS**：依 ACS 流程（新發 RBBB + 缺血症狀積極再灌流評估）
- **合併雙/三束支阻斷 + 昏厥或高位 AV block**：節律器評估
- **PE/ASD/肺高壓相關**：治療原發病

## ⚠️ 併發症
- 進展為完全性房室阻斷（合併其他束支/分支阻斷時風險↑）
- 掩蓋部分診斷、易與 Brugada 混淆
- 反映背後右心疾病之預後

## 📊 嚴重度評分 / 風險分層（scoring）
> RBBB 無分數；靠「新發與否 + 合併束支阻斷 + 背後病因」
| 工具 | 用途 | 意義 |
| --- | --- | --- |
| **新發 vs 陳舊** | 臨床意義 | 新發（尤其伴胸痛/喘）需積極查因 |
| **束支阻斷數目** | 房室阻斷風險 | 雙/三束支 + 昏厥 → 節律器評估 |
| **合併病因** | 預後 | PE/缺血/心肌病者依原病分層 |
