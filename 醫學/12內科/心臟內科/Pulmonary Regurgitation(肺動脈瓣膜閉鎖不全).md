# Pulmonary Regurgitation（肺動脈瓣膜閉鎖不全）

> [!abstract] 一句話：**舒張期血流由肺動脈逆流回右心室**，造成**右心室容量負荷 → 進行性擴大 → 右心衰／心律不整**；最常見繼發於**肺高壓致瓣環擴張**，臨床最重要情境是**法洛氏四重症（TOF）修補術後的殘存 PR**；肺高壓相關者聽診＝**Graham Steell 雜音**。

> [!danger] 🚨 值班紅旗
> 1. **TOF 修補術後多年、右心室進行擴大** → PR 是長期主因，需監測換瓣時機（免右心室不可逆惡化）
> 2. **PR ＋ 肺高壓徵象**（響亮 P2、右心衰）→ 治本要處理肺高壓，不只看瓣膜
> 3. **右心衰徵象**（頸靜脈怒張、肝腫大、下肢水腫）＋ 已知 PR → 評估右心室功能與逆流量
> 4. **感染性心內膜炎徵象＋ 三尖瓣/肺動脈瓣受累**（尤 IVDU）→ 可致急性 PR

## 📖 定義 / 病生理（這是什麼）
- **定義**：肺動脈瓣關閉不全，**舒張期血液由肺動脈逆流回右心室**。
- **病因**：
  - **最常見：肺動脈高壓 → 肺動脈與瓣環擴張 → 瓣葉對合不良（functional / 續發性 PR）**。
  - **先天性**：瓣膜發育異常、瓣膜缺如。
  - **TOF 修補術後**：右心室流出道重建／跨瓣補片或球囊/外科解除 PS 後，常留下**殘存 PR**——**是 TOF 長期追蹤最重要的問題**（慢性 PR → 右心室進行擴大）。
  - **感染性心內膜炎**（尤靜脈藥癮者，右心瓣膜）。
  - **肺動脈瓣球囊成形術（如治療 PS）後**、類癌心臟病（carcinoid）、罕見風濕性。
- **病生理**：慢性逆流 → **右心室容量負荷 ↑ → 右心室擴大 → 收縮功能漸退** → 右心衰、三尖瓣環擴張致次發性 TR、**心律不整（房性/室性）與猝死風險**（TOF 術後尤然）。多數**耐受良好多年**，症狀晚出現。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Fatigue(疲倦)]]、[[Palpitation(心悸)]]、[[Edema(水腫)]]
- **多年無症狀**（右心室代償）；後期出現**運動耐受下降、勞力性呼吸困難、疲倦、右心衰徵象（水腫、腹脹、肝腫大、頸靜脈怒張）、心悸／心律不整**。
- **聽診**：
  - **Graham Steell 雜音**：**肺動脈高壓相關**的 PR ——左上胸骨緣的**高頻、遞減（decrescendo）舒張早期雜音**，隨吸氣增強；伴響亮 P2。
  - **低壓型 PR（如 TOF 術後、瓣環擴張而肺壓不高）**：多為**低頻、短促遞減的舒張期雜音**（右心室與肺動脈壓差小、逆流速度低），臨床上雜音可不明顯，**逆流嚴重卻雜音輕**是陷阱。
  - S2 分裂變化、右心室 S3、右心室 heave。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[ECG(心電圖)]]　影像 [[Echocardiography(心臟超音波)]]、[[Cardiac MRI(心臟磁振造影)]]
- **心臟超音波（首選）**：彩色都卜勒顯示逆流噴流寬度／範圍、逆流減速時間、右心室大小與功能、估算肺動脈壓、合併 TR。
- **Cardiac MRI（金標準量化）**：**量測逆流分率（regurgitant fraction）與右心室容積（RVEDV/RVESV）與 EF**——**特別用於 TOF 術後決定肺動脈瓣置換（PVR）的時機**（依右心室容積/功能閾值）。
- **ECG**：右心室肥厚／擴大、RBBB（TOF 術後常見）、**QRS 增寬（≥180 ms 為 TOF 術後室性心律不整/猝死危險指標）**。
- **CXR**：右心室／肺動脈擴大。
- **心導管／運動測試**：血流動力學與功能評估輔助。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **PR 最常見成因是肺高壓**（功能性），治本要處理肺高壓。
- **TOF 術後殘存 PR 是主角**：慢性右心室容量負荷 → 右心室擴大／功能退化／室性心律不整；**MRI 右心室容積是決定換瓣時機的關鍵**。
- **Graham Steell** ＝ 肺高壓相關 PR 的**高頻遞減舒張雜音**；要與**主動脈瓣逆流（AR）**鑑別（AR 也是舒張遞減雜音，但沿胸骨左緣中下、與外周脈壓大/水沖脈相關；PR 隨吸氣增強、伴右心徵象與肺高壓 P2）。
- **逆流重、雜音卻不明顯**（低壓型 PR）是陷阱——別以雜音大小判嚴重度，靠影像。
- QRS 過寬（TOF 術後）＝心律不整/猝死警訊。

## 💊 治療（Acute / Chronic）
> 用藥 [[Diuretics(利尿劑)]]（右心衰對症）、肺高壓標靶藥（依病因，劑量查藥典）
- **治本**：**處理肺動脈高壓**（若為續發性 PR，最重要）；感染性心內膜炎所致者治療感染。
- **輕–中度、右心室功能與大小正常**：**追蹤觀察**（超音波／MRI 定期），多可長期穩定。
- **肺動脈瓣置換（PVR, pulmonary valve replacement）適應症**（尤 TOF 術後重度 PR）：
  - **有症狀**（運動耐受下降、右心衰、心律不整），或
  - **無症狀但右心室進行性擴大／功能下降**（依 MRI 右心室容積與 EF 閾值判斷），或伴顯著 TR、殘存 RVOT 阻塞、持續性室性心律不整等。
  - 手術（外科換瓣）或**經導管肺動脈瓣植入（PPVI / TPVI，如 Melody/Sapien 系統）**——選項依解剖與流出道狀況。
- **心律不整**：依情況藥物／電燒；高危者評估 ICD。

## ⚠️ 併發症
- **右心室擴大 → 右心衰竭**
- **次發性三尖瓣逆流（TR）**（右心室/瓣環擴大）
- **心律不整（房性/室性）與猝死**（TOF 術後 QRS 寬者風險高）
- 運動耐受進行性下降
- 換瓣後人工瓣退化／再介入需求

## 📊 嚴重度評分 / 風險分層（scoring）
> 無命名式 score；以 **MRI 右心室容積/功能 + 逆流分率 + 症狀** 決定換瓣時機
| 指標 | 用途 | 意義 |
| --- | --- | --- |
| **逆流分率（MRI, regurgitant fraction）** | PR 嚴重度量化 | 分率越高＝逆流越重（通常 > 40% 視為顯著） |
| **右心室舒張末容積指數（RVEDVi, MRI）** | 換瓣時機 | 右心室進行擴大到閾值（各中心採 RVEDVi 上限值）＝考慮 PVR，過晚換瓣右心室難回復 |
| **右心室 EF** | 功能評估 | EF 下降＝右心室失代償 |
| **QRS 寬度（ECG）** | 心律不整/猝死風險 | TOF 術後 QRS ≥180 ms＝室性心律不整/猝死高危 |
| **症狀（運動耐受/心衰）** | 介入決策 | 有症狀＝換瓣適應症 |
