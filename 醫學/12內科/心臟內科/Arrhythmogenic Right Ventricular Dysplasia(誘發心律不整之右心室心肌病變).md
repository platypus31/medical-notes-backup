# Arrhythmogenic Right Ventricular Dysplasia（誘發心律不整之右心室心肌病變）

> [!abstract] 一句話：**ARVC/ARVD** 是右心室心肌被**纖維脂肪組織取代**的遺傳性心肌病（多為 desmosome 基因突變、體染色體顯性），是**年輕人與運動員運動性室速/猝死（SCD）的重要病因**。標誌 ECG 為**右胸導 T 波倒置 + epsilon wave**，cardiac MRI 是關鍵影像，診斷靠 **2010 Task Force Criteria**；**運動會加速惡化**，處置核心是**限制劇烈運動 + ICD（高風險者）**。

> [!danger] 🚨 值班紅旗
> 1. **年輕人/運動員運動時暈厥或猝死倒下** → 想 ARVC（與 HCM、CPVT、Brugada、冠脈異常同列 SCD 鑑別）。
> 2. **LBBB 型態、電軸上偏的室速**（源自 RV）＋右胸導 T 倒置 → 高度懷疑 ARVC。
> 3. 一等親有 ARVC 或年輕不明原因猝死家族史 → 該病人與家屬都要**篩檢**。
> 4. 確診/高風險者**禁止競技與高強度耐力運動**——運動是明確的疾病進展與致命心律不整誘因。

## 📖 定義 / 病生理（這是什麼）
- **定義**：以**右心室游離壁進行性纖維脂肪化（fibrofatty replacement）** 為特徵的遺傳性心肌病；晚期可累及左心室（故現多稱 **ARVC / arrhythmogenic cardiomyopathy**）。
- **遺傳**：多為**體染色體顯性、外顯率不全**。核心是**橋粒（desmosome）蛋白基因**突變——最常見 **PKP2（plakophilin-2）**，其他 DSP、DSG2、DSC2、JUP。少數隱性型（Naxos 病 = JUP + 掌蹠角化 + 羊毛狀髮）。
- **病生理**：desmosome 缺陷 → 機械負荷下心肌細胞連接分離、細胞死亡 → **纖維脂肪替代**，好發「三角區（triangle of dysplasia）」：RV 流入道、流出道、心尖。病灶區成為**折返（reentry）基質 → 折返性室速**；心肌喪失 → RV 收縮功能↓、擴大 → 右心衰。**機械應力（運動）加速去黏附與進展** → 「運動員心臟」高危、且運動誘發致命心律不整。

## 🩺 臨床表現（症狀 / 徵象）
> 四階段：① 隱匿期（結構正常仍可猝死）② 明顯電學異常期（室性心律不整）③ RV 衰竭 ④ 雙心室衰竭。
- **心悸、運動性暈厥、運動誘發室速（VT）**，可為**首發即猝死（SCD）**。
- 室速多為 **LBBB 型態**（源自 RV）；電軸方向可提示起源（RVOT 起源者下方電軸）。
- 晚期：右心衰徵象（水腫、肝腫大、JVP↑），或雙心室衰竭。
- 誘因：**劇烈/耐力運動**明確加速惡化與觸發事件。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[ECG(心電圖)]]、[[Signal-Averaged ECG(訊號平均心電圖)]]、[[Holter(24小時心電圖)]]　影像 [[Cardiac MRI(心臟磁振造影)]]、[[Echocardiography(心臟超音波)]]
- **診斷架構：2010 Revised Task Force Criteria（Marcus et al., Circulation 2010）**——六大類（每類分 major/minor），組合達 **definite / borderline / possible**：
  1. **整體/局部功能與結構異常**（影像）：echo/MRI/血管攝影顯示 **RV 局部無運動/運動異常/膨出（akinesia/dyskinesia/aneurysm）** 加上 RV 擴大或 RVEF 下降。
  2. **心肌組織特徵**：心內膜心肌切片顯示殘存心肌細胞比例下降 + 纖維（±脂肪）替代。
  3. **復極化異常**：**右胸導（V1–V3）T 波倒置**（>14 歲、無 RBBB）為 major。
  4. **去極化/傳導異常**：**epsilon wave（V1–V3 QRS 末小棘波）** 為 major；**訊號平均心電圖 late potentials（晚電位）**；右胸導 QRS 增寬/終末啟動延遲。
  5. **心律不整**：**LBBB 型態室速合併上方/不確定電軸**（major）、RVOT 型態（下方電軸，minor）、頻發 PVC（>500/24h）。
  6. **家族史 / 基因**：一等親確診或屍檢確診、或病人帶**致病性 desmosome 突變**（major）。
- **Cardiac MRI（關鍵影像）**：評估 RV 容積/RVEF、局部壁運動異常與膨出、脂肪浸潤與 **LGE 纖維化**（含左室受累）。
- **心臟超音波**：初步評估 RV 大小/功能與局部異常。
- **基因檢測**：對先證者與家族篩檢有價值。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **epsilon wave** 是特徵性但不敏感；**右胸導 T 倒置（V1–V3）** 較常見。留意年輕人 V1–V3 T 倒置需與正常「juvenile pattern」（通常 <14 歲/隨年齡消退）區分。
- **早期結構可正常仍會猝死**（電學不穩定先於結構異常）——正常 echo/MRI 不能完全排除高危個案，需綜合 ECG、Holter、家族史、基因。
- **與 RVOT 特發性室速鑑別**：後者心臟結構正常、預後良性；ARVC 有結構/電學/家族線索。
- **運動是明確促進因子**：耐力運動員外顯率與事件率更高；生活型態介入（限制運動）本身就是治療。
- 現代觀點：常累及左室，「純右室」是過時印象；家族篩檢應涵蓋左室型表現。

## 💊 治療（Acute / Chronic）
> 目標：**預防猝死** + 控制心律不整 + 處理心衰。
- **生活型態**：**限制競技與高強度耐力運動**（一級介入，降低進展與事件）。
- **藥物**：**β-blocker** 為基礎；症狀性室性心律不整加 **抗心律不整藥（常用 sotalol 或 amiodarone）**——劑量依指引/藥典，⚠️需查證。
- **ICD**：
  - **二級預防**：曾發生心跳驟停或血流動力學不穩定 VT → 建議植入。
  - **一級預防**：依高風險特徵（不明原因暈厥、非持續性 VT、顯著 RV/LV 功能不良、廣泛結構受累等）個別化評估。
- **導管電燒（catheter ablation）**：對反覆室速者減少發作（因病灶進展常需心外膜途徑，非治癒）。
- **心衰治療**：右心/雙心室衰竭者依心衰指引；末期考慮心臟移植。
- **家族篩檢**：一等親做 ECG、影像、（可行時）基因檢測，並定期追蹤。

## ⚠️ 併發症
- **心因性猝死（SCD）**、持續性/反覆室速、右心衰、晚期雙心室衰竭；ICD 相關併發（不當放電、導線問題）。
