# Infiltrative Cardiomyopathy（浸潤性心肌病）

> [!abstract] 一句話：**異常物質浸潤心肌**造成**心室壁增厚但收縮功能相對保留**、以**限制性充填障礙（restrictive physiology）＋舒張性心衰（HFpEF）**表現的一大類心肌病。**最重要病因是類澱粉沉積（amyloidosis）**；臨床常誤當高血壓性肥厚或縮窄性心包炎，關鍵是「**心電圖低電壓 vs 超音波厚壁**」的不對稱要想到浸潤。

> [!danger] 🚨 值班紅旗
> 1. **超音波心室壁很厚，但 ECG 電壓反而低（甚至低電壓）** → 高度懷疑浸潤（尤其 amyloid），不要只當高血壓心臟病
> 2. **HFpEF ＋ 對利尿劑很敏感、血壓偏低、不耐受 ACEI/β-blocker** → 想 amyloid（尤其對 digoxin／CCB 敏感易毒性）
> 3. **新發傳導阻滯（AV block）／持續性心律不整＋心肌增厚** → 想 sarcoidosis（可致猝死、需及早診斷）
> 4. **「厚壁＋限制性生理」易與縮窄性心包炎混淆** → 兩者治療完全不同（心包切除 vs 內科／病因治療），務必鑑別

## 📖 定義 / 病生理（這是什麼）
- **定義**：異常物質（蛋白、脂質、鐵、肉芽腫、糖原等）**沉積／浸潤於心肌間質或心肌細胞內**，使心肌**變硬、順應性下降**。
- **血流動力學**：心室壁增厚但**心腔不擴大、收縮分率（EF）早期相對保留**，主要問題是**舒張期充填受限（restrictive physiology）** → 心室充填壓上升 → **舒張性心衰（HFpEF）**、雙心房壓力後負荷 → **雙心房擴大**。
- **晚期**：浸潤加重可累及傳導系統與收縮功能，出現傳導阻滯、心律不整、收縮功能下降。
- **與肥厚性心肌病（HCM）區別**：HCM 是**肌節基因病造成的真性心肌細胞肥厚**；浸潤性是**細胞外／異物堆積**造成「假性肥厚（pseudohypertrophy）」，故電壓不升反降。

### 主要病因（記憶：蛋白／肉芽腫／金屬／酵素／糖原）
| 病因 | 沉積物 | 關鍵線索 |
| --- | --- | --- |
| **Amyloidosis（最重要）** | 錯誤摺疊澱粉蛋白（AL／ATTR） | 厚壁＋低電壓、apical sparing strain、雙側腕隧道症候群、PYP 骨掃描；見 [[Transthyretin Amyloid Cardiomyopathy(類澱粉蛋白沉積症)]] |
| **Sarcoidosis** | 非乾酪性肉芽腫 | 年輕人、AV block／室性心律不整／猝死、斑塊狀 LGE、FDG-PET 陽性 |
| **Hemochromatosis** | 鐵 | 鐵過載（原發 HFE 或反覆輸血）、可致擴張＋限制混合型、肝／胰／皮膚、鐵蛋白↑、MRI T2* 短 |
| **Fabry disease** | 醣鞘脂（α-galactosidase A 缺乏） | X 染色體、少年男性、向心性肥厚、角膜渦狀混濁、腎病變、神經痛、可酵素替代 |
| **糖原儲積（如 Danon、PRKAG2、Pompe）** | 肝醣 | 兒童／青少年、極厚壁＋預激（WPW）／傳導病、骨骼肌無力 |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Edema(水腫)]]、[[Palpitation(心悸)]]、[[Syncope(暈厥)]]
- **以右心衰為主的體循環鬱血**：頸靜脈怒張、肝腫大、腹水、下肢水腫（限制性生理心室充填受限，靜脈壓上升明顯）。
- **舒張性心衰（HFpEF）**：勞力性呼吸困難、端坐呼吸、運動耐受下降。
- **傳導阻滯／心律不整**：AV block、AF、室性心律不整；**sarcoidosis 特別容易猝死**。
- **低心輸出徵象**：疲倦、低血壓、暈厥（尤 amyloid 自律神經受累合併姿勢性低血壓）。
- 依病因附帶**心外表現**（腕隧道、蛋白尿、肝脾腫大、角膜病變、皮膚色素沉著等）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[ECG(心電圖)]]、[[BNP(腦鈉肽)]]、[[Troponin(肌鈣蛋白)]]、[[SPEP(血清蛋白電泳)]]、[[Ferritin(鐵蛋白)]]　影像 [[Echocardiography(心臟超音波)]]、[[Cardiac MRI(心臟磁振造影)]]、[[Endomyocardial Biopsy(心內膜心肌切片)]]
- **ECG**：**低電壓或電壓與壁厚不成比例**（amyloid 典型）、假性梗塞 Q 波、AV block／束支阻滯（sarcoid）、預激（糖原病）。
- **心臟超音波**：**心室壁增厚、心腔小、EF 早期保留、雙心房擴大**；amyloid 可見**心肌顆粒狀閃爍（granular sparkling）**、瓣膜／房間隔增厚、心包積液；**縱向 strain 呈 apical sparing（心尖保留）**。組織都卜勒示 e' 下降的限制性充填。
- **Cardiac MRI**：**延遲增強（LGE）型態辨病因** — amyloid 常為**瀰漫性內膜下／transmural LGE、T1 mapping 與細胞外容積（ECV）明顯升高**；sarcoid 呈**斑塊狀中壁 LGE**；並可 T2* 定量鐵過載。
- **核醫**：**⁹⁹ᵐTc-PYP／DPD 骨掃描**（ATTR 心臟類澱粉高攝取，grade 2–3 且排除 AL 可免切片）；**FDG-PET**（sarcoid 活動性發炎）。
- **Endomyocardial biopsy（金標準）**：組織染色定病因（Congo red 剛果紅 → amyloid apple-green birefringence；肉芽腫 → sarcoid；鐵染色 → hemochromatosis）。
- **依病因特殊檢查**：血／尿免疫固定電泳＋血清游離輕鏈（排 AL）、TTR 基因、α-galactosidase A 活性（Fabry）、鐵蛋白／轉鐵蛋白飽和度／HFE 基因、ACE／全身影像（sarcoid）。

### 與縮窄性心包炎（constrictive pericarditis）鑑別（考點）
| 特徵 | 限制性／浸潤性心肌病 | 縮窄性心包炎 |
| --- | --- | --- |
| 心肌 | 增厚、心肌本身病變 | 心肌正常，**心包增厚鈣化** |
| 呼吸相互依存（interdependence） | 不明顯 | **明顯**（吸氣 septal bounce、心室互依） |
| 組織都卜勒 e' | **下降**（心肌病） | **正常或增高（annulus paradoxus）** |
| BNP | 明顯升高 | 相對較低 |
| 心導管 | 左右室舒張壓可分離 | **左右室舒張壓相等、dip-and-plateau** |
| 治療 | 病因／內科 | **心包切除術可治癒** |

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **「厚壁 ＋ 低電壓」= 浸潤（尤 amyloid）的招牌組合**；真性 HCM／高血壓心臟病電壓通常增高。
- **Strain apical sparing（心尖保留、基部受損的櫻桃派圖）** 對心臟類澱粉高度提示。
- **Amyloid 對 digoxin／CCB／β-blocker 敏感易中毒或低血壓**，用藥保守。
- **Sarcoid 心臟病要積極找**（可猝死），懷疑時做 FDG-PET／MRI，早期免疫抑制 + 視情況 ICD。
- **EF 保留 ≠ 心臟沒事**：浸潤性心肌病的核心問題在舒張，EF 正常仍可嚴重心衰。
- 別把限制性生理一律歸咎心包，**務必用 e'／interdependence／導管鑑別縮窄性心包炎**（可手術治癒的別漏掉）。

## 💊 治療（Acute / Chronic）
> 用藥 [[Diuretics(利尿劑)]]、[[Tafamidis(氯苯唑酸)]]、[[Anticoagulants(抗凝血劑)]]
- **對症（共通）**：**利尿劑**謹慎減輕鬱血（限制性生理前負荷依賴，過度利尿易低血壓、低心輸出）；控制心律（AF 常需抗凝，浸潤心肌栓塞風險高）。
- **病因治療（關鍵，決定預後）**：
  - **ATTR-CM**：**tafamidis（TTR 穩定劑）**改善存活與住院（ATTR-ACT）；見 [[Transthyretin Amyloid Cardiomyopathy(類澱粉蛋白沉積症)]]。
  - **AL amyloid**：血液科主導的**抗漿細胞化療／自體移植**（預後最差，需儘速轉診）。
  - **Sarcoidosis**：**corticosteroid ± 免疫抑制**；高危傳導病／室速考慮 **ICD**、節律器。
  - **Hemochromatosis**：**放血（原發）或鐵螯合劑 deferasirox／deferoxamine（輸血相關）**。
  - **Fabry disease**：**酵素替代療法（agalsidase）／chaperone（migalastat）**。
- **進展性心衰**：一般 HFrEF 藥物（ACEI/β-blocker）在 amyloid 常不耐受、須小心；末期選擇性個案考慮心臟移植（AL 需血液病控制）。
- 劑量一律查藥典／對應指引。

## ⚠️ 併發症
- **難治性心衰（右心為主）、反覆鬱血住院**
- **傳導阻滯、AF、致命性室性心律不整、猝死**（sarcoid 尤甚）
- **心房內血栓 → 全身性栓塞／中風**（amyloid 即使竇性也可栓塞）
- **姿勢性低血壓、腎受累、末期多器官衰竭**（依病因）

## 📊 嚴重度評分 / 風險分層（scoring）
> 無單一通用評分；依病因分期。心臟類澱粉常用 biomarker 分期（Mayo）
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **ATTR 分期（NAC/UK：NT-proBNP + eGFR）** | ATTR-CM 預後分層 | 依 NT-proBNP＞3000 ng/L 與 eGFR＜45 分 I–III 期，期別越高存活越差 |
| **AL amyloid Mayo 2012 分期** | AL 心臟預後 | 依 troponin、NT-proBNP、游離輕鏈差值計分，分 I–IV 期 |
| **Perugini grade（PYP 骨掃描）** | ATTR 心臟攝取 | grade 0–3；**grade 2–3 且排除 AL** 支持 ATTR-CM 診斷 |
