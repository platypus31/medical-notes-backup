# Hypertensive emergency（高血壓急症）

> [!abstract] 一句話：**血壓嚴重升高（常 >180/120）+ 進行中的急性標的器官損害**（腦/心/主動脈/腎/眼）。定調靠「有沒有器官受損」而**非血壓數字**——有就是 emergency，要**靜脈藥物 + ICU**；降壓速度依受損器官不同（多數第一小時降 MAP ≤25%），唯剝離與某些中風例外。

> [!danger] 🚨 值班紅旗
> 1. **判斷關鍵是「器官有沒有受損」不是血壓多高**——血壓 200/120 但無症狀無器官損害 = urgency（口服慢降）；血壓 170/110 但正在中風/剝離/肺水腫 = emergency（靜脈快降）。
> 2. **降壓速度：一般第 1 小時 MAP 降 ≤25%**，接下來 2–6 小時降到 ~160/100；**降太快、太猛會造成腦/心/腎低灌流缺血**。
> 3. **器官別的例外一定要記**：主動脈剝離 → SBP <120 且 HR <60（最積極、最快）；急性缺血性中風 → 一般 <220/120 才降（要溶栓則 <185/110）；出血性中風 → <180/130（積極）；SAH → MAP <130。
> 4. **禁忌**：疑主動脈剝離**先給 β-blocker 再降壓**（純血管擴張劑先給會 reflex 心搏過速增剪力）；懷孕子癲前症用 labetalol/hydralazine/nicardipine + 硫酸鎂，**禁 ACEI/ARB/nitroprusside**。
> 5. 找誘因：漏藥、古柯鹼/安非他命、嗜鉻細胞瘤、可樂定驟停反彈。

## 📖 定義 / 病生理（這是什麼）
- **定義**：嚴重高血壓（通常 SBP >180 或 DBP >120）**合併急性、進行性標的器官損害**。無器官損害則為 [[Hypertensive urgency(高血壓緊急狀況)]]。
- **標的器官損害光譜**：高血壓腦病變、缺血/出血性中風、SAH、[[Acute Coronary Syndrome(急性冠心症)]]、急性肺水腫/心衰、[[Aortic dissection(主動脈剝離)]]、急性腎損傷、子癲前症/子癇症、微血管溶血（MAHA）、視乳頭水腫。
- **病生理**：血壓超過腦血流自動調節上限→血腦障壁破壞、纖維素樣壞死、內皮受損→器官缺血/水腫；急性劇升時自動調節曲線右移，故**降太快反致缺血**。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Headache(頭痛)]]、[[Altered mental status(意識改變)]]、[[Dyspnea(呼吸困難)]]
> [!PDF|yellow] [[長庚銀蛋口袋書(印).pdf#page=32&selection=197,0,223,4&color=yellow|長庚銀蛋口袋書(印), p.32]]
> > 定調：有目標器官受損（不管血壓多高） ->很急  -> 靜脈藥物治療 
> > 目標器官受損：腦中風、心肌梗塞、主動脈剝離、急性肺水腫、急性腎衰竭 
> > 臨床表現：頭痛、意識不清、噁心嘔吐、視力模糊、癲癇發作、喘、寡尿，血壓常飆高到 220/140 mmHg

- 神經：劇烈頭痛、意識改變、癲癇、局部神經缺損、視力模糊。
- 心肺：胸痛、呼吸困難、端坐呼吸、肺水腫囉音。
- 腎：寡尿、血尿；眼底：出血、滲出液、視乳頭水腫（第 III–IV 級）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[ECG(心電圖)]]、[[Troponin(肌鈣蛋白)]]、[[Creatinine(肌酸酐)]]、[[Urinalysis(尿液分析)]]　影像 [[Brain CT(腦部電腦斷層)]]、[[Chest X-ray(胸部X光)]]、[[CTA(電腦斷層血管攝影)]]
- **重點是找標的器官損害**：ECG（缺血/LVH）、troponin、腎功能 + 尿液（血尿/蛋白/圓柱）、眼底。
- **依症狀導向影像**：頭痛/神經症狀→腦 CT；胸背痛→CXR + 主動脈 CTA；喘/肺水腫→CXR、BNP。
- 溶血指標（LDH、破裂紅血球、haptoglobin）評估 MAHA；孕婦查子癲前症（尿蛋白、肝功能、血小板）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **emergency vs urgency 只看器官損害**——這是國考與臨床反覆考點。
- 別追求「立刻降到正常」；**過度快速降壓本身會造成腦/心/腎梗塞**。
- 每個器官有各自降壓目標與禁忌藥（見紅旗），開藥前先問「是哪個器官受損」。
- 胸背撕裂痛 + 高血壓 → 先排除主動脈剝離再決定降壓策略。

## 💊 治療（Acute / Chronic）
> 用藥 [[Labetalol]]、[[Nicardipine(尼卡地平)]]、[[Nitroglycerin(硝化甘油)]]、[[Nitroprusside(硝普鈉)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
> [!PDF|yellow] [[長庚銀蛋口袋書(印).pdf#page=32&selection=225,0,287,2&color=yellow|長庚銀蛋口袋書(印), p.32]]
> > 治療：給予靜脈注射 NTG 或 Labetalol ，使快速降壓至 160/100 mmHg 以下 # 特殊情況： 
> > 1）[[Ischemic Stroke(缺血性中風)]]：僅在 220/130 mmHg 以上才要降壓 
> > 2）[[Hemorrhagic Stroke(出血性中風)]]：需降至 180/130 mmHg 以下 
> > 3）缺血性腦中風欲給予血栓溶解劑治療者：需降至 185/110 mmHg 以下 
> > 4）[[SubArachnoid Hemorrhage(蜘蛛膜下腔出血)]]（SAH）：MAP 應降至 130 mmHg 以下 
> > 5）[[Aortic dissection(主動脈剝離)]]：SBP 需降至 120 mmHg 以下

### 一般原則
- **靜脈藥物 + 動脈/密切血壓監測（ICU）**。
- **降壓幅度**：第 1 小時 MAP ↓ ≤25% → 2–6 小時降到約 160/100 → 24–48 小時漸回正常。

### 器官別首選藥
| 情境 | 首選藥 | 目標 |
| --- | --- | --- |
| 主動脈剝離 | **esmolol/labetalol**（先） + nicardipine/nitroprusside | SBP <120、HR <60 |
| 急性肺水腫/心衰 | **nitroglycerin / nitroprusside** + loop 利尿劑 | 快速卸前後負荷 |
| 急性冠心症 | **nitroglycerin + β-blocker** | 緩解缺血 |
| 高血壓腦病變 | **nicardipine / labetalol** | 第 1 小時 MAP ↓ ~25% |
| 急性缺血性中風 | 一般 <220/120 才降；溶栓則 <185/110（labetalol/nicardipine） | 保護半暗帶灌流 |
| 出血性中風 | nicardipine/labetalol | SBP ~130–150（依指引） |
| 子癲前症/子癇症 | **labetalol / hydralazine / nicardipine + MgSO₄** | 禁 ACEI/ARB/nitroprusside |
| 嗜鉻細胞瘤 | **phentolamine（α 先）** 再 β | 先 α 後 β，勿單用 β |

### 慢性
- 器官損害穩定後轉口服長效降壓藥，找並治療誘因（漏藥、次發性病因），強化居家血壓監測與服藥依從。

## ⚠️ 併發症
- 中風（缺血/出血）、腦水腫/腦病變、癲癇。
- 心肌梗塞、急性心衰/肺水腫、主動脈剝離。
- 急性腎損傷、視力喪失、MAHA。
- 醫源性：降壓過快 → 分水嶺腦梗塞、心肌缺血、失明。
