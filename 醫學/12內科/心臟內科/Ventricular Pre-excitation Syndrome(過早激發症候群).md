# Ventricular Pre-excitation Syndrome（過早激發症候群）

> [!abstract] 一句話：**心房電訊號透過「旁路（accessory pathway）」繞過房室結，早一步激發部分心室**，ECG 出現**短 PR + delta wave + 寬 QRS** 三特徵。旁路本身是良性，但一旦合上心搏過速（尤其 **WPW 合併心房顫動經旁路快速下傳**）可惡化成 VF 而**猝死**——這是它的危險所在。

> [!danger] 🚨 值班紅旗
> 1. **WPW + 心房顫動（不規則、寬 QRS、心室率極快、QRS 形態多變）→ 絕對禁用 AV 結阻斷劑**（adenosine / verapamil / diltiazem / digoxin / β-blocker）——阻斷房室結會把所有心房衝動逼向旁路更快下傳，可誘發 **VF 猝死**。用 **procainamide / ibutilide**，血流動力學不穩直接**同步電擊**。
> 2. delta wave + 短 PR 的年輕病人主訴陣發性心悸 → 想 WPW 相關 AVRT。
> 3. 已知 WPW 出現寬 QRS 心搏過速，先當成「經旁路傳導」處理，別當一般 SVT 亂推 AV 結阻斷劑。

## 📖 定義 / 病生理（這是什麼）
> [!PDF|yellow] [[長庚銀蛋口袋書(印).pdf#page=100&selection=15,0,21,7&color=yellow|長庚銀蛋口袋書(印), p.100]]
> > 定義：電傳導繞過房室結，循著「多餘通路（Accessory pathway）」早一步到達心室

正常心臟電訊號只能經**房室結（AV node）**下傳，房室結刻意「延遲」讓心房先收縮完再傳到心室。**預激（pre-excitation）**是指心房與心室之間存在一條**先天性的旁路（accessory pathway）**——最典型是 **Kent bundle（Kent 束）**，直接連接心房與心室肌，繞過房室結的生理延遲，使部分心室肌被「提早（pre-）」激發。

**ECG 三特徵（機轉對應）**：
- **短 PR（< 120 ms）**：旁路不經房室結延遲，訊號提早到心室。
- **Delta wave（δ 波）**：QRS 起始有一段緩升的鈍化波，是心室經旁路被緩慢逐一激發（非經希浦系統快速傳導）的表現。
- **寬 QRS（> 110–120 ms）**：整個 QRS 是「旁路提早激發」與「房室結正常傳導」兩股波前的**融合波（fusion）**，故變寬。

旁路可**順向（antegrade）**、**逆向（retrograde）**或雙向傳導；若只逆向傳導，靜息 ECG 無 delta wave，稱**隱匿性旁路（concealed pathway）**，仍可引發 AVRT。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Palpitation(心悸)]]、[[Syncope(暈厥)]]

- 多數**無症狀**，僅 ECG 意外發現預激圖形（此時稱 pre-excitation *pattern*）。
- 出現**陣發性心搏過速症狀**（心悸、頭暈、胸悶、呼吸困難、暈厥）時，才稱 **WPW *syndrome***（預激圖形 + 相關心律不整症狀）。
- 最危險表現：WPW + AF 快速心室反應造成血流動力學崩潰 / 暈厥 / 猝死。

- **12 導程 ECG**：短 PR + delta wave + 寬 QRS 即可診斷預激；delta 波方向與 QRS 型態可粗略定位旁路。
- **電生理檢查（EPS）**：確定旁路存在、**定位**、測**旁路不反應期（判斷經 AF 快速下傳猝死風險）**，並可同步進行導管電燒。
- **運動測試**：運動時預激圖形突然消失（旁路不反應期長）提示低猝死風險。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **pattern vs syndrome**：只有 ECG 圖形 = pre-excitation *pattern*；圖形 + 心搏過速症狀 = **WPW *syndrome***。
- **WPW vs LGL（Lown-Ganong-Levine）**：

| | **WPW** | **LGL** |
| --- | --- | --- |
| 旁路 | **Kent bundle**（房室肌直接相連） | 概念上為繞過房室結的房室結旁 / James 纖維（機轉有爭議） |
| PR | 短（< 120 ms） | 短（< 120 ms） |
| Delta wave | **有** | **無** |
| QRS | 寬 | 正常 |

- **AVRT 的兩型**：
  - **Orthodromic（順向房室折返，最常見 ~90–95%）**：衝動經房室結**下傳**、經旁路**逆傳** → **窄 QRS** 規則心搏過速。
  - **Antidromic（逆向）**：衝動經旁路**下傳**、房室結逆傳 → **寬 QRS** 規則心搏過速，易與室速混淆。
- **最危險組合：WPW + AF**——心房亂顫的多重衝動可經旁路無延遲快速下傳 → 極快、不規則、寬且多變 QRS → 可退化成 **VF 猝死**。

## 💊 治療（Acute / Chronic）
> 用藥 [[Procainamide(普卡因醯胺)]]、[[Adenosine(腺苷)]]（劑量請查藥典 / 依指引）

**急性 — 依機轉區分（關鍵決策）**：
- **Orthodromic AVRT（窄 QRS 規則）**：走 SVT 流程——**迷走神經動作 → adenosine**，或 AV 結阻斷（謹慎）；不穩定則電擊。
- **🚫 WPW + AF / 任何寬 QRS 不規則 / 疑經旁路傳導**：**禁用 AV 結阻斷劑（adenosine、verapamil、diltiazem、digoxin、β-blocker）**——會逼衝動全走旁路加速下傳，誘發 VF。
  - 血流動力學**不穩** → **立即同步電擊（synchronized cardioversion）**。
  - **穩定** → 用**不阻斷旁路的抗心律不整藥**：**procainamide** 或 **ibutilide**。

**根治（Chronic）**：
- **導管電燒（catheter ablation）為首選根治法**，成功率高、可消除旁路一勞永逸，尤其有症狀 WPW syndrome 或高風險旁路（AF 經旁路快速下傳）者。
- 無症狀預激圖形者依 EPS 風險分層決定是否預防性電燒（尤其高風險職業 / 運動員）。

## ⚠️ 併發症
- **心室顫動 / 猝死**（WPW + AF 經旁路快速下傳退化為 VF，最致命）。
- 反覆 AVRT 造成心悸 / 暈厥 / 生活品質下降；長期高頻率者可致心搏過速誘發心肌病。
- 藥物誤用併發症（對 WPW+AF 用 AV 結阻斷劑加速惡化）。
- 電燒相關風險（房室傳導阻滯、心包填塞等，少見）。

## 📊 嚴重度評分 / 風險分層（scoring）
- **無專用評分**；猝死風險分層依**旁路電生理特性**。

| 指標 | 用途 | 意義 |
| --- | --- | --- |
| 旁路前傳不反應期（EPS 測） | 猝死風險 | 不反應期短 → AF 可經旁路快速下傳 → **高風險** |
| 預激間歇性消失 / 運動時消失 | 低風險指標 | 旁路傳導能力弱，猝死風險低 |
| AF 時最短 pre-excited RR 間期 | 猝死風險 | RR 越短（心室率越快）風險越高 |
