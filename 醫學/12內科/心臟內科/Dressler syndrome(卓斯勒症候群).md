# Dressler syndrome（卓斯勒症候群）

> [!abstract] 一句話：心肌梗塞（或心臟手術／創傷）後**數週**發生的**免疫介導性心包炎**——發燒＋胸膜性胸痛＋心包摩擦音三聯。屬「延遲型」post-cardiac injury syndrome，與 MI 後**數天內**的早期心包炎（PIP）機轉不同、要分清楚。

> [!danger] 🚨 值班紅旗
> 1. **MI 或心臟術後 1–8 週新發燒＋胸痛＋摩擦音** → 想 Dressler，別一律當作再梗塞或肺炎
> 2. **心包積液→心包填塞**：低血壓、頸靜脈怒張、奇異脈、心音遙遠 → 急做 echo，警覺 [[Cardiac tamponade(心包膜填塞)]]
> 3. **MI 後早期慎用某些 NSAID／類固醇**：可能干擾梗塞癒合、增加心室破裂風險——首選**高劑量 aspirin**
> 4. 抗凝病人合併心包積液 → 警覺**出血性心包積液**，評估是否調整抗凝

## 📖 定義 / 病生理（這是什麼）
  - 又稱後心肌梗塞症候群（postmyocardial infarction syndrome）
- **機轉**：心肌壞死後暴露的心肌抗原引發**自體免疫反應**（抗心肌抗體），造成延遲型、免疫介導的**心包炎 ± 肋膜炎 ± 肺炎**。屬 **post-cardiac injury syndrome（PCIS）** 的一種（同類還有心臟手術後、外傷後）。
- **時間**：典型在 MI／心臟損傷後**約 1–8 週（經典 2–6 週）**出現——這是與早期心包炎的關鍵區別。
- **現況**：再灌流治療普及後**已很罕見**（早期再通減少大範圍壞死與抗原暴露）。
- **與早期 post-MI 心包炎（PIP，peri-infarction pericarditis）的區別**：
  | 特徵 | 早期 PIP | Dressler（延遲） |
  | --- | --- | --- |
  | 發生時間 | MI 後**數天內（2–4 天）** | MI 後**數週（1–8 週）** |
  | 機轉 | 透壁壞死**直接**發炎波及心包 | **免疫介導**（自體抗體） |
  | 全身症狀 | 較少（無明顯發燒） | **發燒、全身不適、ESR/CRP↑ 明顯** |
  | 積液／復發 | 少 | 較常有心包／肋膜積液、可復發 |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Chest pain(胸痛)]]、[[Fever(發燒)]]
- **發燒**（低到中度）、全身不適、倦怠、肌肉關節痛。
- **胸膜性胸痛（pleuritic chest pain）**：吸氣／平躺加重，**前傾坐姿緩解**（典型心包炎特徵）。
- **心包摩擦音（pericardial friction rub）**：床邊聽診關鍵徵象。
- 可伴**肋膜積液**（肋膜性胸痛、呼吸音減弱）、少量肺浸潤。
- 心包積液量大時 → 心包填塞徵象。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[ECG(心電圖)]]、ESR/CRP　影像 [[Echocardiography(心臟超音波)]]、[[CXR(胸部X光)]]
- **臨床診斷**：MI／心臟損傷病史 ＋ 延遲發作的心包炎特徵（發燒、胸膜性胸痛、摩擦音）。
- **發炎指標**：**ESR、CRP 顯著上升**、白血球增多（幫助與再梗塞區分——但需結合心肌酵素）。
- **ECG**：心包炎變化（**廣泛 ST 上升、PR 段下移**），需與再梗塞的局部變化區分。
- **Echo**：偵測心包／肋膜積液、評估填塞。
- **CXR**：心影擴大（積液）、肋膜積液、肺浸潤。
- 心肌酵素（troponin）幫助排除再梗塞（Dressler 本身可輕微升高）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **時間點是靈魂**：MI 後**數天**內 = 早期 PIP；**數週後** = Dressler。考試與臨床都愛考。
- **再灌流時代已罕見**：現代 STEMI 快速 PCI 後很少見到典型 Dressler。
- **勿當再梗塞**：發燒 + ESR/CRP 顯著升高 + 摩擦音 + 廣泛（非局部）ST 變化，偏向心包炎而非新梗塞。
- **NSAID／類固醇在 MI 後早期的陷阱**：某些 NSAID 與類固醇可能**干擾梗塞癒合、增加心室破裂**風險，故 post-MI 情境**首選 aspirin**。
- 可**復發**；反覆者考慮 colchicine 延長療程。

## 💊 治療（Acute / Chronic）
> 用藥 高劑量 [[Aspirin(阿斯匹靈)]]、[[Colchicine(秋水仙素)]]、NSAID
### 急性
- **高劑量 aspirin（首選，尤其 post-MI 情境）**：劑量⚠️需查證（一般為高劑量分次，逐步減量）。
- **Colchicine**：作為輔助可降低復發率（劑量⚠️需查證）。
- **NSAID**（如 ibuprofen／indomethacin）：可用於非 post-MI 早期情境；**MI 後早期避免某些 NSAID**（干擾癒合、心室破裂風險）。
- **類固醇**：保留給對 aspirin/NSAID 反應不佳或禁忌者；**post-MI 早期慎用**（影響梗塞癒合、增加復發）。
- 抗凝評估：合併心包積液時警覺出血性積液，評估是否停／調抗凝。
### 併發症處置
- **心包填塞** → 心包穿刺引流。
### 慢性 / 復發
- 反覆發作 → 延長 colchicine 療程；極少數頑固 → 免疫治療評估。

## ⚠️ 併發症
- **心包積液 → 心包填塞**（最需警覺的急性併發症）
- **肋膜積液**
- **縮窄性心包炎**（[[Constrictive pericarditis(縮窄性心包膜炎)]]，罕見的慢性後果）
- **復發性心包炎**

## 📊 嚴重度評分 / 風險分層（scoring）
> 無專屬 score；風險分層靠是否**心包填塞**與**復發**判斷
| 項目 | 用途 | 意義 |
| --- | --- | --- |
| 有無心包填塞徵象 | 判斷急性危險度 | 有 → 需緊急引流 |
| 復發次數 / 療程反應 | 判斷慢性治療強度 | 反覆 → 延長 colchicine、考慮進階治療 |
