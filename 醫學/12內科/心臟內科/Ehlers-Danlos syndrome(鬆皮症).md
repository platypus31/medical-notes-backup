# Ehlers-Danlos syndrome（埃勒斯-當洛斯症候群 / 鬆皮症）

> [!abstract] 一句話：**一群膠原蛋白合成缺陷的遺傳性結締組織病**，共通表現是「皮膚過度延展 + 關節過動 + 組織脆弱易瘀」。心臟內科最怕的是 **vascular type（第 IV 型，COL3A1 突變）**——動脈、腸、子宮可在毫無外傷下自發破裂，是死亡主因。

> [!danger] 🚨 值班紅旗
> 1. **年輕人非外傷性動脈破裂 / 主動脈或內臟動脈剝離 / 自發性氣胸 + 皮膚薄透見血管 → 想 vascular EDS**，這是會在急診猝死的表現。
> 2. **vascular EDS 禁做侵入性血管檢查（動脈導管、血管攝影 DSA）**——血管壁極脆，穿刺處可撕裂、術中出血難止；影像優先選非侵入性（CTA/MRA/超音波）。
> 3. 孕婦 vascular EDS 有子宮破裂與動脈剝離風險，屬高危妊娠，需多科團隊。
> 4. 這類病人「傷口難癒合、縫線易撕脫、組織一夾就破」，任何手術都要當高風險看待。

## 📖 定義 / 病生理（這是什麼）
EDS 是一組**遺傳性結締組織病（heritable connective tissue disorder）**，核心缺陷在**膠原蛋白（collagen）的合成、修飾或組裝異常**，導致結締組織抗張強度不足。膠原是皮膚、血管壁、肌腱韌帶、腸壁、子宮的主要支架，缺陷因此表現為：

- **皮膚過度延展（hyperextensibility）**、質地柔軟如天鵝絨、薄而透可見皮下血管
- **關節過動（joint hypermobility）**，易脫臼、慢性疼痛
- **組織脆弱**：易瘀青、傷口難癒、疤痕呈「香菸紙（cigarette-paper）」樣萎縮性瘢痕、血管與內臟脆弱

依 2017 國際分類分為 **13 型**，各由不同基因與膠原路徑缺陷造成。臨床最重要的三型：

| 型別 | 舊名 | 基因 / 缺陷 | 重點 |
| --- | --- | --- | --- |
| **Vascular type** | Type IV | **COL3A1**（第 III 型膠原，體染色體顯性） | **最致命**；動脈 / 腸 / 子宮自發破裂；薄透皮膚、特殊面容、易瘀 |
| **Classical type** | Type I/II | COL5A1 / COL5A2 | 皮膚顯著過度延展 + 萎縮性瘢痕 + 關節過動 |
| **Hypermobile type (hEDS)** | Type III | 多數基因未明（臨床診斷） | 最常見；以關節過動與慢性疼痛為主，血管風險低 |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Easy Bruising(易瘀青)]]、[[Joint Hypermobility(關節過動)]]

- **皮膚**：過度延展、柔軟、薄而透（vascular type 尤其皮膚薄可見胸腹靜脈網）、萎縮性瘢痕、易瘀青
- **關節**：全身性關節過動、反覆脫臼 / 半脫位、慢性關節痛、扁平足
- **出血傾向**：微血管與血管壁脆弱造成自發瘀斑、術後 / 產後出血、牙齦出血（凝血功能多正常，是「血管性」出血）
- **心血管（見特徵段）**：vascular type 的動脈瘤 / 剝離 / 破裂；其他型的二尖瓣脫垂、主動脈根部擴張
- **vascular type 特殊面容**：大眼、薄唇、窄鼻、耳垂缺乏；肢端早老（acrogeria）樣外觀
- **其他**：疝氣、直腸脫垂、自發性氣胸、靜脈曲張、動脈破裂造成的急性劇痛

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Genetic Testing(基因檢測)]]　影像 [[CT Angiography(電腦斷層血管攝影)]]、[[Echocardiography(心臟超音波)]]

- **2017 國際 EDS 分類（2017 International Classification of EDS）**：各型有明確的臨床主要 / 次要準則 + 基因確認。
  - **hEDS**：目前仍以**臨床準則**診斷（無單一確定基因），需符合全身性關節過動 + 系統性結締組織受累特徵 + 排除其他結締組織病。
  - **vascular type 等其餘型別**：臨床懷疑 → **分子基因檢測確認**（vascular type 找 COL3A1 致病變異）。
- **Beighton score**：量化關節過動度（0–9 分），輔助 hEDS 診斷。
- **心血管監測影像**：
  - **Echocardiography**：評估主動脈根部直徑、二尖瓣脫垂（基線 + 定期追蹤）。
  - **vascular type 全身動脈監測**：以**非侵入性 CTA / MRA / 超音波**追蹤動脈瘤與剝離，**避免 DSA 等侵入性血管攝影**。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **vascular EDS（COL3A1）是唯一會讓病人「血管自發破裂猝死」的型**——考題與臨床的核心區辨點；其餘型別血管風險低。
- **禁忌操作**：vascular type 避免動脈穿刺 / 導管 / 血管攝影、慎行任何手術、避免劇烈等長運動與外傷。
- **EDS vs Marfan 鑑別**（皆為遺傳性結締組織病、皆可有主動脈根部擴張與二尖瓣脫垂）：

| 特徵 | **Vascular EDS** | **Marfan syndrome** |
| --- | --- | --- |
| 基因 | **COL3A1**（第 III 型膠原） | **FBN1**（fibrillin-1） |
| 主要血管風險 | **中小動脈自發破裂 / 剝離**（可無擴張先破） | **升主動脈根部瘤 → 剝離**（Type A） |
| 破裂前是否先擴張 | 常**不先擴張**就破裂 | 多先進行性擴張再剝離 |
| 皮膚 | 薄、透、易瘀、萎縮性瘢痕 | 皮膚多正常，有妊娠紋 |
| 骨骼 | 無高身材 / 蜘蛛指特徵 | 高瘦、蜘蛛指、關節過動 |
| 眼 | 無晶狀體脫位 | 晶狀體向上脫位（ectopia lentis） |

- 兩者都要監測主動脈；但 Marfan 用直徑閾值決定手術，vascular EDS 因「可不擴張就破」而更難以直徑預測，處置更保守。

## 💊 治療（Acute / Chronic）
> 用藥 [[Celiprolol(西利洛爾)]]、[[Beta Blocker(乙型阻斷劑)]]（劑量請查藥典 / 依指引）

- **無根治法**，以**預防血管事件、降低破裂風險、支持與症狀處理**為主。
- **Celiprolol（β1 拮抗 + β2 促效之乙型阻斷劑）**：法國隨機試驗（BBEST trial, Ong et al., *Lancet* 2010）顯示可**降低 vascular EDS 動脈破裂 / 剝離事件**，為目前少數有證據的藥物介入；血壓正常者亦可用。⚠️ 實際劑量與可近性依地區，請查藥典 / 指引。
- **嚴格控制血壓、避免血壓驟升**（避免重負荷等長運動、Valsalva、劇烈碰撞運動）。
- **急性破裂 / 剝離**：緊急處置以**支持性、盡量非侵入 / 保守**為原則；手術止血極困難（組織一夾就破、縫線易撕脫），需經驗豐富團隊，且開放手術本身高風險。
- **classical / hypermobile type**：關節保護、物理治療強化肌力穩定關節、慢性疼痛管理、皮膚傷口小心縫合減張。
- **手術 / 麻醉全期**：視為高出血 / 組織脆弱風險，減少侵入操作、輕柔止血、加強傷口支持。

## ⚠️ 併發症
- **vascular type**：動脈瘤、動脈剝離、動脈自發破裂（致命大出血）、腸道（乙狀結腸）自發穿孔、妊娠子宮破裂、自發性氣胸、頸動脈海綿竇瘻。
- **其他型**：二尖瓣脫垂相關逆流、主動脈根部擴張、慢性關節脫臼與退化、慢性疼痛與失能、傷口癒合不良與疝氣復發、直腸脫垂。

## 📊 嚴重度評分 / 風險分層（scoring）
- **無 EDS 專用嚴重度評分**；風險分層主要看**型別**（vascular type = 高死亡風險）與**器官受累**。
- 輔助工具：**Beighton score（0–9）** 量化關節過動；主動脈追蹤沿用心臟影像直徑監測邏輯。

| 評分 / 分型 | 用途 | 意義 |
| --- | --- | --- |
| Beighton score | 量化關節過動 | ≥5/9（成人）支持過動；輔助 hEDS 診斷 |
| 型別（2017 分類） | 預後分層 | Vascular type → 血管破裂高死亡風險；hEDS → 以疼痛 / 失能為主 |
