# Pulmonary Stenosis（肺動脈瓣狹窄）

> [!abstract] 一句話：**右心室流出道／肺動脈瓣狹窄造成右心室後負荷增加 → 右心室肥厚（RVH）**；**絕大多數為先天性**，多數輕度無症狀終生良好，中重度可致運動性呼吸困難、暈厥；治療首選**經皮球囊瓣膜成形術（balloon valvuloplasty）**。聽診招牌＝**左上胸骨緣收縮噴射性雜音 ＋ 隨吸氣減弱的噴射喀音（ejection click）**。

> [!danger] 🚨 值班紅旗
> 1. **重度 PS（peak gradient > 64 mmHg）＋ 症狀（運動呼吸困難／暈厥）** → 需介入（球囊成形術）
> 2. **PS ＋ 發紺** → 想合併右→左分流（如 ASD/PFO 或 TOF），單純 PS 不發紺
> 3. **關鍵性新生兒 PS（critical PS）** → PDA 依賴供肺、嚴重發紺 → **PGE1 維持導管 ＋ 緊急球囊成形**
> 4. **PS ＋ 特殊面容/發育** → 想 **Noonan 症候群**（發育不良的肺動脈瓣）

## 📖 定義 / 病生理（這是什麼）
- **定義**：右心室到肺動脈路徑的阻塞，依部位分層：
  - **瓣膜型（valvular，最常見）**：肺動脈瓣葉融合／增厚呈圓頂狀（doming）；**Noonan 症候群**常為瓣葉發育不良（dysplastic）型。
  - **瓣膜下型（subvalvular / infundibular）**：右心室流出道肌肉肥厚，常見於**法洛氏四重症（TOF）**。
  - **瓣膜上型（supravalvular）**：肺動脈主幹或分支狹窄，見於 **Williams 症候群、先天性德國麻疹（congenital rubella）**。
- **成因**：**先天為主**；先天關聯——**TOF、Noonan 症候群、congenital rubella、Williams 症候群**；後天罕見（類癌心臟病 carcinoid → PS/PR、風濕性罕）。
- **病生理**：肺動脈瓣狹窄 → **右心室後負荷（afterload）↑ → 代償性右心室肥厚（RVH）→ 右心室壓上升** → 長期失代償 → **右心衰**；嚴重時右房壓升高，若有心房交通則右→左分流 → 發紺。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Syncope(暈厥)]]、[[Chest Pain(胸痛)]]、[[Fatigue(疲倦)]]
- **多數無症狀**（輕中度可終生良好）。
- **中重度**：勞力性呼吸困難、疲倦、運動性暈厥、少數運動性胸痛（RV 缺血）。
- **發紺**：僅在合併心房層次右→左分流時出現。
- **聽診（右心來源特徵）**：
  - **左上胸骨緣（肺動脈瓣區）收縮期噴射性雜音（crescendo-decrescendo）**；**越嚴重 → 雜音越長、峰值越晚（peak 往收縮末移）**。
  - **噴射喀音（ejection click）**：瓣膜型特徵，**隨吸氣減弱／消失**（這是**右心瓣膜喀音的特色**，與左心喀音相反——吸氣增加右心回流反而使瓣膜在收縮前已被推開）。
  - **P2 減弱且延遲 → S2 寬分裂**（右室射血時間延長、肺動脈瓣關閉延遲、關閉力量弱）。
  - 重度可觸及左上胸骨緣 thrill、右心室隆起（RV heave）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[ECG(心電圖)]]　影像 [[Echocardiography(心臟超音波)]]、[[Cardiac MRI(心臟磁振造影)]]
- **心臟超音波（首選）**：
  - 顯示瓣膜形態（doming / dysplastic）、狹窄部位、**都卜勒量測跨瓣峰值壓差（peak gradient）分輕中重**、RVH、右心室功能、合併病灶（ASD/VSD）。
- **ECG**：**右心室肥厚（RVH）＋ 右軸偏移（RAD）**，重度可見右房擴大（高聳 P 波）。
- **CXR**：**主肺動脈狹窄後擴張（post-stenotic dilatation）**（瓣膜融合噴流衝擊）、心影多正常大小、肺血管紋正常或減少。
- **Cardiac MRI**：評估瓣膜上型（分支狹窄）與右心室功能。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **右心喀音（PS）隨吸氣減弱**——與大多數右心雜音「吸氣增強」相反，是必考鑑別點（左心 AS 喀音不隨呼吸變）。
- **越嚴重雜音越長、峰值越晚、P2 越弱越延遲**——用雜音時序判斷嚴重度。
- **主肺動脈狹窄後擴張** 是 CXR 招牌。
- **PS ＋ 發育遲緩/特殊面容 → Noonan**；**PS ＋ VSD/騎跨/RVH → TOF**（PS 是 TOF 的組成）；**supravalvular ＋ Williams / rubella**。
- **單純 PS 不發紺**；有發紺要找分流。
- **無症狀輕度 PS 預後極佳**，通常只需追蹤。

## 💊 治療（Acute / Chronic）
> 用藥 [[Prostaglandin E1(前列腺素E1)]]（新生兒 critical PS 維持 PDA，劑量查藥典）
- **輕度、無症狀**：**追蹤即可**（定期超音波），不需介入。
- **介入適應症**：**症狀性** 或 **重度（peak gradient > 64 mmHg）**，或中度伴右心室功能受損 →
  - **經皮球囊瓣膜成形術（balloon pulmonary valvuloplasty）為首選**（尤其典型融合圓頂型瓣膜，成效佳、微創）。
- **外科**（少用）：球囊無效、瓣膜發育不良型（dysplastic，常見於 Noonan，球囊效果較差）、合併瓣下/瓣上或需同時修補其他病灶時。
- **新生兒 critical PS**：**PGE1 維持 PDA 供肺 ＋ 緊急球囊成形**。
- **併發 PR**：球囊或外科後常留下**肺動脈瓣逆流（PR）**，需長期追蹤（見 [[Pulmonary Regurgitation(肺動脈瓣膜閉鎖不全)]]）。

## ⚠️ 併發症
- **右心室肥厚 → 右心衰竭**
- **介入後肺動脈瓣逆流（PR）**（長期需追蹤是否需換瓣）
- **心律不整**（房性、右心來源）
- 合併分流時 → 發紺、矛盾栓塞
- 感染性心內膜炎（風險相對低）

## 📊 嚴重度評分 / 風險分層（scoring）
> 以 **超音波都卜勒 peak gradient** 分級（無命名式 score）
| 分級 | Peak gradient（echo, 都卜勒） | 意義 / 處置 |
| --- | --- | --- |
| **輕度** | **< 36 mmHg**（或峰值流速 < 3 m/s） | 追蹤，預後佳 |
| **中度** | **36–64 mmHg** | 追蹤；有症狀/RV 功能受損時考慮介入 |
| **重度** | **> 64 mmHg**（或峰值流速 > 4 m/s） | 介入（球囊成形術），尤其有症狀 |

> 註：不同指引亦以尖峰即時壓差 vs 平均壓差、右心室壓表述，數值採自 ACC/AHA、ESC 成人先天心指引；臨床決策合併症狀與右心室功能，非單看單一數字。
