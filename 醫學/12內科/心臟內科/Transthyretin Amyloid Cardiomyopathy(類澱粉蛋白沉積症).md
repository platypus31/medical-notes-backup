# Transthyretin Amyloid Cardiomyopathy（類澱粉蛋白沉積症 / ATTR-CM）

> [!abstract] 一句話：**TTR（transthyretin）蛋白錯誤摺疊沉積於心肌**造成的浸潤性、限制性心肌病，表現為 **HFpEF＋厚壁＋低電壓**。**是近年「可治療」的心臟類澱粉**（tafamidis 穩定劑改善存活），關鍵是**用非侵入的 PYP 骨掃描診斷**、並**先排除 AL 型**（AL 需血液科化療、預後更差）。**雙側腕隧道症候群常早於心臟症狀數年**是重要伏筆。

> [!danger] 🚨 值班紅旗
> 1. **老年 HFpEF＋超音波厚壁＋ECG 低電壓** → 想 ATTR-CM，別只當高血壓心臟病
> 2. **對 digoxin / 鈣阻斷劑（CCB）敏感，易中毒／低血壓** → amyloid 心肌會與這些藥結合，**盡量避免使用**
> 3. **主動脈瓣狹窄（尤 low-flow low-gradient）合併不成比例厚壁的老人** → 常合併 ATTR，TAVR 前要想到
> 4. **診斷前務必排除 AL**（血/尿免疫固定電泳＋血清游離輕鏈）；PYP 陽性但輕鏈異常 → 不能直接當 ATTR，須切片

## 📖 定義 / 病生理（這是什麼）
- **TTR（transthyretin，甲狀腺素運載蛋白）**由肝臟合成，運送甲狀腺素與視黃醇。當四聚體（tetramer）**解離、單體錯誤摺疊 → 形成澱粉纖維沉積**於心肌（及神經、韌帶、腸道）。
- **心肌沉積 → 心室壁增厚、僵硬 → 限制性生理 → HFpEF**；為 [[Infiltrative Cardiomyopathy(浸潤性心肌病)]] 的代表病因。
- **兩型**：
  - **Wild-type ATTR（ATTRwt，舊稱老年性系統性類澱粉）**：**基因正常、隨年齡 TTR 自發不穩定**；**老年男性為主（常 >70 歲）**，以心臟為主，常合併雙側腕隧道、脊椎狹窄、肱二頭肌肌腱斷裂。
  - **Hereditary／variant ATTR（ATTRv）**：**TTR 基因突變（體染色體顯性）**，依突變型別可偏心臟型、神經型或混合型（如 Val122Ile 偏心臟、Val30Met 偏神經）。發病較早、有家族史。

### 與 AL amyloid 的區別（最重要鑑別）
| 特徵 | **ATTR** | **AL（免疫球蛋白輕鏈型）** |
| --- | --- | --- |
| 前驅蛋白 | 錯誤摺疊 TTR | **漿細胞病產生的單株輕鏈** |
| 本質 | 蛋白穩定性問題 | **血液惡性／漿細胞病** |
| 好發 | 老年男性（wt）／家族史（v） | 各年齡、可合併多發性骨髓瘤 |
| 進展 | 較慢 | **快、預後更差** |
| 治療主體 | 心臟科 + TTR 穩定劑 | **血液科化療／自體幹細胞移植** |
| 檢驗 | 免疫固定電泳／游離輕鏈**正常** | **免疫固定電泳／游離輕鏈異常** |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Edema(水腫)]]、[[Syncope(暈厥)]]、[[Palpitation(心悸)]]
- **心臟**：HFpEF（勞力性呼吸困難、水腫、鬱血），限制性生理以右心衰／體循環鬱血為主；AF 常見、傳導阻滯、姿勢性低血壓（自律神經受累）。
- **心外「紅旗伏筆」（常早於心臟數年）**：
  - **雙側腕隧道症候群（carpal tunnel syndrome）**——TTR 沉積於韌帶，常在心臟症狀前 5–10 年出現。
  - **腰椎脊椎狹窄（spinal stenosis）**、**肱二頭肌遠端肌腱自發斷裂（Popeye sign）**、**扳機指**。
  - ATTRv 可有**周邊多發性神經病變、自律神經失調（腹瀉/便秘、勃起障礙、姿勢性低血壓）、玻璃體混濁**。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Free Light Chain(血清游離輕鏈)]]、[[Immunofixation(免疫固定電泳)]]、[[TTR Gene Testing(TTR基因檢測)]]、[[NT-proBNP(N端腦鈉肽前體)]]、[[Troponin(肌鈣蛋白)]]　影像 [[Echocardiography(心臟超音波)]]、[[Cardiac MRI(心臟磁振造影)]]、[[Bone Scintigraphy PYP(焦磷酸鹽骨掃描)]]
- **心臟超音波**：**向心性心室壁增厚、EF 早期保留、雙心房大、瓣膜／房間隔增厚、心包積液**；**縱向 strain 呈 apical sparing（心尖保留，櫻桃派/bull's-eye 圖）**是高度提示徵象。
- **Cardiac MRI**：**瀰漫性內膜下或 transmural LGE、T1 native 與細胞外容積（ECV）明顯升高**，難以將血池訊號歸零。
- **⁹⁹ᵐTc-PYP / DPD / HMDP 骨掃描（核心非侵入診斷）**：心臟攝取 **Perugini grade 2–3**（心臟 ≥ 肋骨，或 H/CL ratio ≥1.5）**且免疫固定電泳與游離輕鏈皆正常（排除 AL）** → **可不需心內膜切片即診斷 ATTR-CM**（Gillmore 非侵入診斷演算法）。
- **排除 AL（必做）**：**血清＋尿免疫固定電泳 ＋ 血清游離輕鏈比值**；任一異常 → **不能只憑 PYP 判 ATTR，須組織切片＋質譜分型**。
- **基因檢測（TTR gene）**：確診後分 ATTRwt vs ATTRv，指導家族篩檢與治療（部分突變適用基因沉默療法）。
- **切片**：Congo red 剛果紅 apple-green 雙折射；免疫組化／質譜確定澱粉型別。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **「厚壁＋低電壓」＋老年男性 HFpEF ＝ 想 ATTR**；strain apical sparing 招牌。
- **雙側腕隧道症候群病史是重要早期線索**——遇 HFpEF 一定要問。
- **PYP 陽性 ≠ ATTR，除非同時排除 AL**（單株蛋白陰性）；此為診斷最常見陷阱。
- **用藥地雷**：amyloid 心肌對 **digoxin、鈣阻斷劑（尤 verapamil/diltiazem）** 敏感易毒／低血壓 → **避免**；β-blocker、ACEI/ARB 也常不耐受（低血壓、心搏過緩），須小心低劑量。
- **AF 抗凝門檻低**（即使竇性也可能有心房血栓），栓塞風險高。
- ATTRwt 常被長年當「高血壓心臟病／舒張功能不良」漏診，可治療的病別錯過。

## 💊 治療（Acute / Chronic）
> 用藥 [[Tafamidis(氯苯唑酸)]]、[[Diuretics(利尿劑)]]、[[Anticoagulants(抗凝血劑)]]
### 疾病修飾治療（ATTR 專屬）
- **TTR 穩定劑 tafamidis**：穩定 TTR 四聚體、抑制纖維形成；**ATTR-ACT 試驗證實降低全因死亡與心血管住院**（NNT 顯著），為 ATTR-CM 一線疾病修飾藥。（劑量依仿單／查藥典）
- **TTR 基因沉默／降低生成（主要用於 ATTRv 神經病變、心臟角色演進中）**：patisiran（siRNA）、inotersen（ASO）、以及新一代 vutrisiran；**acoramidis** 為新型口服 TTR 穩定劑（近年試驗陽性）。
- 早期診斷、早期起始穩定劑效益最大（NYHA I–II 期最受惠）。
### 支持性心衰治療
- **利尿劑**謹慎控制鬱血（限制性生理前負荷依賴，避免過度利尿低血壓）。
- **避免/慎用會惡化的藥**：digoxin、CCB（見上）；β-blocker/ACEI 依血壓耐受低劑量。
- **AF**：常需**抗凝**（栓塞高風險）；節律控制優先於激進 rate control（避免低心輸出）。
- 進階傳導病／心搏過緩考慮節律器；末期選擇性個案評估心臟移植。
### AL 型（若鑑別為 AL）
- **轉血液科**：抗漿細胞化療（如 daratumumab-based、bortezomib）± 自體幹細胞移植；治療方向與 ATTR 完全不同。

## ⚠️ 併發症
- **難治性 HFpEF、反覆鬱血住院**
- **AF、傳導阻滯（需節律器）、室性心律不整**
- **心房內血栓 → 全身性栓塞／中風**
- **姿勢性低血壓、暈厥**（自律神經受累）
- ATTRv：進行性周邊／自律神經病變、腸胃／眼受累

## 📊 嚴重度評分 / 風險分層（scoring）
> 以 biomarker 分期預測預後與治療時機
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **NAC/UK ATTR 分期** | ATTR-CM 預後 | 依 **NT-proBNP >3000 ng/L** 與 **eGFR <45 mL/min** 分 I–III 期，期別越高存活越差 |
| **Mayo ATTR 分期（2016）** | ATTR-CM 預後 | 依 troponin T 與 NT-proBNP 閾值分 I–III 期 |
| **Perugini grade（PYP 掃描）** | 心臟攝取程度 | grade 0–3；**grade 2–3 且無單株蛋白** 支持 ATTR-CM 診斷 |
| **NYHA 功能分級** | 治療時機 | tafamidis 於 NYHA I–II 效益最大 |
