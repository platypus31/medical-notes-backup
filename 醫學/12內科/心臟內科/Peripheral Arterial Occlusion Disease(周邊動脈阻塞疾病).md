# Peripheral Arterial Occlusion Disease（周邊動脈阻塞疾病）

> [!abstract] 一句話：周邊動脈粥狀硬化狹窄/阻塞，光譜從**慢性間歇性跛行**到**急性肢體缺血 (ALI)**。慢性 PAD 是全身動脈硬化的窗口（等於心血管高風險）；**急性肢體缺血是外科急症（6 Ps）**，時間就是肢體。

> [!danger] 🚨 值班紅旗
> 1. **急性肢體缺血 (ALI) 6 Ps**：Pain、Pallor、Pulselessness、Paresthesia、Paralysis、Poikilothermia（冰冷）→ **立即全身抗凝 (IV heparin) + 血管外科會診**，可挽救肢體的窗口約 6 小時
> 2. **運動/感覺喪失 (Rutherford IIb) = 立即威脅肢體** → 緊急再灌流（取栓/溶栓/繞道），別等到 fixed mottling(III 期，多不可逆)
> 3. **突發、單側、無跛行病史 + 有 AF/心臟血栓來源 → 想動脈栓塞 (embolic)**；有慢性跛行史則多為原位血栓 (thrombotic)
> 4. **休息痛/潰瘍/壞疽 = 嚴重肢體缺血 (CLTI)** → 血管重建評估，避免截肢

## 📖 定義 / 病生理（這是什麼）
- **慢性 PAD**：動脈粥狀硬化 → 下肢供血不足；與冠心病/腦血管病共病（全身性動脈硬化）。
- **急性肢體缺血 (ALI)**：突然血流中斷
  - **栓塞 (embolic)**：心源性（AF、心室血栓、心內膜炎）→ 突發、無側支、界線分明
  - **原位血栓 (thrombotic)**：既有 PAD 斑塊破裂/狹窄 → 有跛行史、側支較多、症狀較緩
- **危險因子**：吸菸（最強）、糖尿病、高血壓、高血脂、年齡、慢性腎病

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Leg pain(下肢疼痛)]]、[[Claudication(間歇性跛行)]]
- **慢性**：
  - **間歇性跛行**（走一段固定距離小腿/臀部痛，休息緩解）
  - **休息痛**（平躺加劇、垂足緩解）、**潰瘍/壞疽**（嚴重肢體缺血 CLTI）
  - 患肢冰冷、脈搏減弱、毛髮脫落、指甲營養不良、傷口難癒
- **急性 (ALI)**：**6 Ps**（見紅旗）— 突發劇痛、蒼白、無脈、麻木、無力、冰冷

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 **ABI（踝肱指數）**、脈搏理學、Doppler　影像 動脈 Duplex US、CTA / MRA / DSA
- **ABI（踝肱指數）**：
  - **正常 1.0–1.4；≤0.90 診斷 PAD**；0.4–0.9 輕中度、<0.4 嚴重
  - **>1.4 = 血管鈣化不可壓縮**（糖尿病/CKD，需改用趾肱指數 TBI）
- **運動後 ABI / 跑步機測試**：靜態 ABI 正常但有症狀時
- **影像（規劃血管重建時）**：Duplex US、CTA、MRA、**數位減影血管攝影 (DSA，金標準兼可介入)**
- **ALI**：臨床 6 Ps 為主，手持 Doppler 聽動/靜脈訊號分 Rutherford 分級

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **PAD = 心血管高風險等同**：診斷 PAD 就要全面二級預防（statin、抗血小板、控糖控壓、戒菸）
- **糖尿病/CKD 的 ABI 可假性正常/偏高（鈣化）** → 用 TBI
- **ALI embolic vs thrombotic** 影響治療策略（栓塞→取栓；血栓→溶栓/繞道）
- **Rutherford ALI 分級**決定急迫性：I 可存活、**IIa 邊緣**、**IIb 立即威脅（感覺+運動喪失）**、**III 不可逆（大理石紋、僵硬）**
- **別對缺血肢體亂用溫敷/抬高**；ALI 疑診先抗凝、勿延誤會診

## 💊 治療（Acute / Chronic）
> 用藥 [[Heparin(肝素)]]、[[Cilostazol]]、[[Statin(他汀類)]]、[[Aspirin(阿斯匹靈)]]
### 急性肢體缺血 (ALI)
- **立即全身抗凝：IV unfractionated heparin**（防血栓延伸，劑量依體重/protocol，⚠️需查證 bolus 與滴定）
- **緊急血管外科/介入會診**：導管導向溶栓、經皮/手術取栓 (embolectomy)、繞道；不可逆(III) → 截肢
- 止痛、保暖軀幹（患肢勿直接加熱）、監測再灌流傷害（高鉀、腔室症候群、橫紋肌溶解）
### 慢性 PAD
- **二級預防（核心）**：**戒菸**、**high-intensity statin**、**抗血小板（aspirin 或 clopidogrel）**、控糖控壓；低劑量 rivaroxaban + aspirin 於高風險者（⚠️需查證適應症/劑量）
- **症狀（跛行）**：**監督式運動治療（第一線）**、**cilostazol**（PDE3i，改善跛行距離；心衰禁用）
- **嚴重肢體缺血 (CLTI) / 生活受限跛行**：血管重建（血管成形 ± 支架、繞道手術）+ 傷口照護

## ⚠️ 併發症
- **肢體喪失/截肢**（尤其 CLTI、延誤的 ALI）
- **再灌流傷害**：高血鉀、腔室症候群、橫紋肌溶解、急性腎損傷
- 全身動脈硬化 → **心肌梗塞、中風**（PAD 病人主要死因是心血管事件）
- 慢性潰瘍感染、壞疽

## 📊 嚴重度評分 / 風險分層（scoring）
> 慢性用 Fontaine/Rutherford；急性用 Rutherford ALI 分級
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **ABI** | 診斷/嚴重度 | ≤0.9 診斷；<0.4 嚴重；>1.4 鈣化(改 TBI) |
| **Fontaine I–IV** | 慢性分期 | I 無症狀 / IIa-b 跛行 / III 休息痛 / IV 潰瘍壞疽 |
| **Rutherford (慢性 0–6)** | 慢性分級 | 對應 Fontaine，更細 |
| **Rutherford ALI (I / IIa / IIb / III)** | 急性急迫性 | IIb 立即威脅→急救；III 不可逆→截肢 |
