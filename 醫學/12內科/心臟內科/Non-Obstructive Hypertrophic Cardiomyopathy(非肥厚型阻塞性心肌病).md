# Non-Obstructive Hypertrophic Cardiomyopathy（非阻塞型肥厚性心肌病）

> [!abstract] 一句話：**有心肌肥厚但無左心室流出道（LVOT）阻塞的 HCM**——靜態與激發（Valsalva/運動）下 **LVOT gradient < 30 mmHg**、**無收縮期二尖瓣前葉前移（SAM）**。雖無阻塞，仍有**舒張功能障礙、心肌缺血、心律不整與猝死（SCD）風險**，治療重點在**症狀控制＋SCD 風險分層**，**不做針對阻塞的中膈減容（septal reduction）**。（註：此檔中文標題「非肥厚型阻塞性心肌病」用詞有誤，正確概念即 non-obstructive HCM。）

> [!danger] 🚨 值班紅旗
> 1. **年輕人／運動員暈厥、猝死家族史＋不明左室肥厚** → 想 HCM，需 SCD 風險分層，別當單純高血壓肥厚
> 2. **非阻塞不代表低風險**：SCD 由心肌病本身（纖維化／心律不整）驅動，仍可能需要 ICD 一級預防
> 3. **新發 AF** → HCM 病人栓塞風險高，多需抗凝（不用 CHA₂DS₂-VASc 門檻，AF 即抗凝）
> 4. **胸痛／呼吸困難＋左室肥厚但冠脈正常** → 微血管缺血／舒張障礙，避免過度利尿與降前負荷

## 📖 定義 / 病生理（這是什麼）
- **HCM 本質**：多為**肌節（sarcomere）基因突變**造成的**不明原因左心室肥厚（通常最大壁厚 ≥15 mm，或家族史者 ≥13 mm）**，非由後負荷（高血壓、主動脈瓣狹窄）可解釋。
- **Non-obstructive 定義**：**靜態且激發下 LVOT 峰值壓差 < 30 mmHg**，**無 SAM 造成的動態阻塞**。相對於 **obstructive HCM（HOCM，gradient ≥30 mmHg）**；30–50 mmHg 為潛在／激發性阻塞的灰帶。
- **仍存在的病理生理（即使不阻塞）**：
  - **舒張功能障礙**：心肌僵硬、鬆弛異常 → 充填壓上升 → **舒張性心衰（HFpEF）**。
  - **心肌缺血**：小血管病變、肥厚需氧增加、心肌纖維化 → 胸痛（冠脈正常）。
  - **心肌纖維化（MRI 上 LGE）**：致心律不整基質 → 室性心律不整、SCD。
  - **心律不整**：AF（左房大）、NSVT、室速/室顫。
- **與 obstructive HCM（HOCM）區別**：見下表。

### Obstructive vs Non-Obstructive HCM
| 特徵 | **Obstructive（HOCM）** | **Non-Obstructive** |
| --- | --- | --- |
| LVOT gradient | **≥30 mmHg**（靜態或激發） | **<30 mmHg** |
| SAM（二尖瓣前移） | **有**、常伴二尖瓣逆流 | 無 |
| 典型雜音 | 隨 Valsalva/站立**增強**的收縮期雜音 | 無阻塞性動態雜音 |
| 症狀機轉 | 阻塞＋舒張障礙＋缺血 | 舒張障礙＋缺血為主 |
| 中膈減容（myectomy/酒精消融） | **適用（症狀性難治阻塞）** | **不適用**（無阻塞可解） |
| mavacamten | **主要適應症** | 目前非主要適應（用於阻塞型） |
| SCD 風險基礎 | 心肌病本身 | 同樣由心肌病本身驅動 |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Chest Pain(胸痛)]]、[[Syncope(暈厥)]]、[[Palpitation(心悸)]]
- **勞力性呼吸困難**（舒張性心衰、充填壓升高）——常為主要症狀。
- **胸痛（angina）**：冠脈可正常，源自微血管缺血／需氧增加。
- **心悸／暈厥**：心律不整（AF、NSVT、室速）或運動時。
- **AF 相關**：心悸、心衰惡化、栓塞。
- 可完全無症狀，因家族篩檢或心電圖異常被發現；部分以 SCD 為首發（尤年輕運動員）。
- 理學：**無阻塞型動態收縮期雜音**（與 HOCM 區別）；可有 S4、心尖搏動加強。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[ECG(心電圖)]]、[[Genetic Testing(基因檢測)]]、[[Holter(24小時心電圖)]]、[[BNP(腦鈉肽)]]　影像 [[Echocardiography(心臟超音波)]]、[[Cardiac MRI(心臟磁振造影)]]
- **心臟超音波（核心）**：**不明原因左室肥厚（最大壁厚 ≥15 mm，家族史 ≥13 mm）**，評估型態（中膈／心尖／向心）、**LVOT gradient（靜態＋Valsalva/運動激發）確認 <30 mmHg 且無 SAM**、舒張功能、左房大小、EF。
- **激發（provocation）**：**必做 Valsalva／站立／運動負荷**排除潛在（latent）阻塞——靜態無阻塞≠激發無阻塞。
- **ECG**：左室肥厚電壓、異常 Q 波、T 波倒置（心尖型可見巨大負向 T）、傳導異常；篩檢常見異常。
- **Cardiac MRI**：精準測壁厚（超音波易低估心尖／前側壁）、**LGE 定量心肌纖維化（重要 SCD 修飾因子）**、辨鑑別診斷（amyloid、Fabry、運動員心臟）。
- **Holter／事件記錄**：偵測 **NSVT**（SCD 風險因子）與 AF。
- **基因檢測＋家族篩檢**：肌節基因 panel；一等親臨床＋（必要時）基因篩檢。
- **鑑別**：高血壓性肥厚、主動脈瓣狹窄、運動員心臟、浸潤性（[[Infiltrative Cardiomyopathy(浸潤性心肌病)]]／amyloid）、Fabry、糖原病。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **「非阻塞」不是「低風險」**——SCD 由心肌病本質（纖維化、心律不整）決定，**仍要做完整 SCD 風險分層**。
- **診斷必做激發**：靜態 gradient 正常也可能是激發性（latent）阻塞，漏做會誤分類。
- **AF ＝ 抗凝**：HCM 合併 AF 栓塞風險高，**不套用 CHA₂DS₂-VASc 門檻，有 AF 即建議抗凝**（優先 DOAC，除非合併中重度 MS/機械瓣）。
- **不做中膈減容**（myectomy／酒精中膈消融）——那是治「阻塞」的手段，非阻塞病人無阻塞可解。
- **mavacamten（心肌肌凝蛋白抑制劑）主要適應為 obstructive HCM**，非阻塞型目前非主要適應。
- **避免劇烈競技運動**（依現行指引個別化評估），並衛教避免脫水、突然降前/後負荷。
- MRI **LGE 範圍大（≥15% 心肌）** 是新興 SCD 風險修飾因子，納入灰帶決策。

## 💊 治療（Acute / Chronic）
> 用藥 [[Beta Blockers(乙型阻斷劑)]]、[[Verapamil(維拉帕米)]]、[[Diltiazem(地爾硫卓)]]、[[Anticoagulants(抗凝血劑)]]、[[Amiodarone(胺碘酮)]]
### 症狀治療（舒張障礙／缺血）
- **一線 β-blocker**：減慢心率、延長舒張充填、緩解胸痛與呼吸困難。
- **不耐受 β-blocker → 非二氫吡啶類 CCB（verapamil / diltiazem）**（改善鬆弛與缺血）。
- **謹慎利尿**改善鬱血症狀（避免過度降前負荷）；一般避免純血管擴張劑／高劑量利尿（可惡化）。
- （劑量一律查藥典）
### 心律不整／栓塞
- **AF**：**抗凝**（栓塞高風險）＋節律/心率控制（amiodarone 常用維持竇性；rate control 用 β-blocker/CCB）。
- **NSVT／症狀性室性心律不整**：依風險評估，必要時 ICD ± 抗心律不整藥。
### SCD 一級預防 — ICD
- **依 SCD 風險分層決定 ICD**（見下）；已發生心跳停止／持續性室速 → **ICD 二級預防**。
### 不建議
- **中膈減容（surgical myectomy／alcohol septal ablation）**：僅用於藥物難治的**阻塞型**，非阻塞型不適用。
- 末期收縮功能惡化（burnt-out HCM，EF↓）→ 依 HFrEF 治療、評估移植。

## 📊 嚴重度評分 / 風險分層（scoring）
> SCD 風險分層是 non-obstructive HCM 治療核心
| 評分 / 因子 | 用途 | 內容 → 意義 |
| --- | --- | --- |
| **ESC HCM Risk-SCD（2014/2023）** | 5 年 SCD 風險 → ICD 決策 | 納入年齡、**最大壁厚、左房內徑、LVOT gradient、SCD 家族史、非持續性室速（NSVT）、不明原因暈厥**；風險 ≥6% 應考慮 ICD，4–<6% 可考慮 |
| **AHA/ACC 主要危險因子（2020 指引）** | 個別化 ICD 評估 | **SCD 家族史、不明原因暈厥、極厚壁（≥30 mm）、NSVT、LV 心尖動脈瘤、EF<50%**；再加 **MRI 廣泛 LGE** 作修飾因子 |
| **NYHA 功能分級** | 症狀治療強度 | 分級越高越積極症狀治療 |
| **AF 併發** | 抗凝決策 | HCM＋AF 即建議抗凝（不套 CHA₂DS₂-VASc） |
