# Löffler's endocarditis（勒夫勒心內膜炎）

> [!abstract] 一句話：**嗜酸性球增多（hypereosinophilia）造成的嗜酸性心肌炎 → 心內膜心肌纖維化**，屬**限制性心肌病**。嗜酸性球去顆粒毒害心內膜，經**急性壞死期→血栓期→纖維化期**三期演變，特徵是**心尖閉塞＋附壁血栓＋二尖瓣受累**。與熱帶的 **endomyocardial fibrosis（EMF）** 同屬「嗜酸性心內膜心肌病」，差別在 Löffler 有明顯周邊嗜酸性球增多、常見於溫帶。

> [!danger] 🚨 值班紅旗
> 1. **不明原因心衰 ＋ 周邊血液嗜酸性球顯著增多（常 >1.5×10⁹/L）** → 想 Löffler，查心臟超音波看心尖血栓
> 2. **心尖附壁血栓 → 全身性栓塞（中風／肢體／內臟）** 是主要併發，需及早抗凝
> 3. **嗜酸性球增多找病因**：原發 HES、寄生蟲、藥物過敏、惡性、EGPA（Churg-Strauss）——病因決定治療
> 4. **快速進展的嗜酸性心肌炎可猝死／心因性休克** → 高嗜酸血 + troponin 升高要積極處置（類固醇）

## 📖 定義 / 病生理（這是什麼）
- **定義**：**持續性嗜酸性球增多**下，活化的嗜酸性球浸潤心肌並**釋出毒性顆粒蛋白（major basic protein、eosinophil cationic protein）**，損傷心內膜與心肌 → **嗜酸性心肌炎（eosinophilic myocarditis）→ 心內膜心肌纖維化** → 心室僵硬、順應性下降 → **限制性心肌病（restrictive physiology）**。
- **好發部位**：**左右心室心尖與流入道／二尖瓣、三尖瓣瓣下裝置**，纖維化與附壁血栓使**心尖閉塞（apical obliteration）**、瓣膜牽扯造成逆流。
- **三期病程（Löffler 經典分期）**：
  1. **急性壞死期（acute necrotic stage，約前 5 週）**：嗜酸性球浸潤、心肌壞死與心內膜炎，可有 troponin 升高、心肌炎表現。
  2. **血栓期（thrombotic stage，數月）**：受損心內膜上**形成附壁血栓**，栓塞風險高峰。
  3. **纖維化期（fibrotic stage，晚期）**：心內膜心肌**纖維化增厚、心尖閉塞、瓣膜牽扯逆流** → 定型的限制性心肌病。
- **與熱帶 endomyocardial fibrosis（EMF）關係**：兩者同屬 **eosinophilic endomyocardial disease** 光譜；**EMF** 多見於熱帶（非洲/南美/亞洲），周邊嗜酸性球不一定顯著、常已是纖維化末期；**Löffler** 多見於溫帶、伴明顯 hypereosinophilia，被視為同一疾病的早期／溫帶版。

### 病因（嗜酸性球增多的來源）
| 分類 | 例子 |
| --- | --- |
| **原發／腫瘤性** | **Hypereosinophilic syndrome（HES）**（含 FIP1L1-PDGFRA 融合的 myeloid 型）、嗜酸性白血病 |
| **反應性** | **寄生蟲感染**（尤組織侵襲型）、**藥物過敏（DRESS）**、過敏／氣喘 |
| **自體免疫／血管炎** | **嗜酸性肉芽腫性多血管炎（EGPA / Churg-Strauss）** |
| **腫瘤旁** | 淋巴瘤（T 細胞）、實體腫瘤旁嗜酸血 |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Edema(水腫)]]、[[Palpitation(心悸)]]、[[Chest Pain(胸痛)]]
- **限制性生理／心衰**：勞力性呼吸困難、右心衰體循環鬱血（頸靜脈怒張、肝腫大、腹水、水腫）。
- **栓塞事件**：心尖血栓脫落 → **中風、腸系膜／脾／腎梗塞、肢體動脈栓塞**。
- **二尖瓣／三尖瓣受累**：瓣下纖維牽扯 → 逆流雜音、心衰惡化。
- **心律不整**：AF、傳導異常；急性期可胸痛、類心肌炎、甚至心因性休克。
- **全身性嗜酸血表現**：發燒、體重減輕、皮疹、氣喘（EGPA）、依病因的器官侵犯（肺、皮膚、神經）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[CBC with Differential(全血球計數含分類)]]、[[Troponin(肌鈣蛋白)]]、[[Eosinophil Count(嗜酸性球計數)]]、[[Tryptase(類胰蛋白酶)]]、[[ANCA(抗嗜中性球細胞質抗體)]]　影像 [[Echocardiography(心臟超音波)]]、[[Cardiac MRI(心臟磁振造影)]]、[[Endomyocardial Biopsy(心內膜心肌切片)]]
- **血液**：**周邊嗜酸性球顯著增多**（HES 定義 ≥1.5×10⁹/L 持續）；查血抹片、tryptase、維生素 B12、FIP1L1-PDGFRA（骨髓型 HES）。心肌損傷時 **troponin↑**。
- **心臟超音波**：**心尖閉塞／心尖附壁血栓、心內膜增厚回音增強、二/三尖瓣後葉受牽扯逆流、心房擴大、限制性充填**；EF 早期可保留。
- **Cardiac MRI（首選組織定性）**：急性期心肌水腫（T2↑）、**心內膜下延遲增強（subendocardial LGE）沿心尖／流入道分布**、可見附壁血栓（低訊號充填缺損）。
- **Endomyocardial biopsy**：嗜酸性球浸潤、心肌壞死／纖維化（陽性可確診，但斑塊分布可能偽陰）。
- **病因檢查**：糞便寄生蟲／血清學、ANCA 與氣喘/鼻竇病史（EGPA）、藥物史（DRESS）、骨髓／基因（HES/嗜酸性白血病）、腫瘤篩檢。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **「心衰／限制性生理 ＋ 心尖血栓 ＋ 顯著嗜酸血」＝ Löffler**，別把心尖充填缺損當單純心尖肥厚或腫瘤。
- **心尖閉塞（apical obliteration）** 是超音波與 MRI 招牌；MRI 心內膜下 LGE 有助鑑別於缺血（缺血循冠脈分布）。
- **治病因＋控嗜酸血是根本**：嗜酸性球持續高就會繼續傷心，只治心衰不夠。
- **EGPA（Churg-Strauss）** 是需記得的血管炎病因（氣喘＋嗜酸血＋ANCA/系統受累）。
- **急性嗜酸性心肌炎可猛爆致命**——高嗜酸血＋troponin 升高＋心功能惡化要積極（類固醇不要等）。
- 抗凝重要（血栓期栓塞風險高），但纖維化期效益／出血權衡需個別評估。

## 💊 治療（Acute / Chronic）
> 用藥 [[Corticosteroids(皮質類固醇)]]、[[Hydroxyurea(羥基脲)]]、[[Imatinib(伊馬替尼)]]、[[Anticoagulants(抗凝血劑)]]、[[Diuretics(利尿劑)]]
### 針對嗜酸性球（根本治療，依病因）
- **皮質類固醇（corticosteroid）**：多數反應性／特發 HES、EGPA、DRESS 相關者的一線，快速降嗜酸血、抑制心肌發炎。
- **FIP1L1-PDGFRA 陽性（骨髓型 HES）→ imatinib**（酪胺酸激酶抑制劑，反應佳）。
- **類固醇不足／依賴** → **hydroxyurea、interferon-α、mepolizumab（抗 IL-5）** 等第二線降嗜酸血。
- **寄生蟲**先驅蟲（但侵襲型驅蟲前後注意類固醇時機）、**藥物過敏**停藥。
### 心臟支持治療
- **抗凝（血栓期／有附壁血栓）**降低栓塞（劑量查藥典，權衡出血）。
- **心衰處理**：利尿控鬱血（限制性生理謹慎前負荷）、AF 控制；瓣膜嚴重逆流或纖維化定型 → **外科心內膜剝除（endocardectomy）／瓣膜修補或置換**。
- 末期難治限制性心肌病選擇性個案評估心臟移植。

## ⚠️ 併發症
- **附壁血栓 → 全身性栓塞（中風／內臟／肢體梗塞）**
- **二/三尖瓣逆流、進行性限制性心衰**
- **心律不整（AF、傳導病）、急性期猝死／心因性休克**
- 依病因的心外器官損害（肺、神經、皮膚、腎）

## 📊 嚴重度評分 / 風險分層（scoring）
> 無專用心臟評分；以病程分期＋嗜酸血程度＋心臟損傷指標評估
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **Löffler 三期分期** | 病程／治療策略 | 壞死期（抗發炎為主）→ 血栓期（抗凝關鍵）→ 纖維化期（心衰／手術） |
| **HES 定義（嗜酸性球）** | 診斷／嚴重度 | 持續 ≥1.5×10⁹/L ＋器官損傷（含心臟）＝需治療 |
| **心肌損傷指標** | 急性風險 | troponin↑、EF 下降、MRI 心肌水腫 → 積極免疫抑制 |
