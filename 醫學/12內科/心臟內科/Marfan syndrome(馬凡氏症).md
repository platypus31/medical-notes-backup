# Marfan syndrome（馬凡氏症）

> [!abstract] 一句話：**FBN1（fibrillin-1）基因缺陷**造成的**體染色體顯性**結締組織病，侵犯**骨骼、眼、心血管**三大系統。**主動脈根部擴張／剝離是最致命的併發症**，追蹤與預防性手術是保命關鍵。

> [!danger] 🚨 值班紅旗
> 1. **高瘦馬凡體型 + 突發撕裂樣胸背痛** → **主動脈剝離**（[[Aortic dissection(主動脈剝離)]]）直到排除為止，即使很年輕
> 2. **主動脈根部擴張是頭號死因**：新發 AR 雜音、胸痛、暈厥 → 急查主動脈根部
> 3. **突發胸痛＋呼吸困難** → 也要想**自發性氣胸**（Marfan 高風險）
> 4. 已知 Marfan 孕婦：**懷孕大幅升高主動脈剝離風險**，屬高危妊娠

## 📖 定義 / 病生理（這是什麼）
- **基因**：**FBN1** 基因突變 → **fibrillin-1** 蛋白缺陷 → 微纖維（microfibril）與**彈性纖維**結構異常，同時**TGF-β 訊號過度活化** → 結締組織強度下降、主動脈中膜退化。
- **遺傳**：**體染色體顯性**，表現度差異大；約 1/4 為新生突變（無家族史）。
- **本質**：全身結締組織病，最危險處在**主動脈中膜囊狀退化（cystic medial degeneration）** → 主動脈根部（Valsalva 竇）擴張 → 動脈瘤 → 剝離／破裂。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Chest pain(胸痛)]]、[[Aortic Regurgitation(主動脈瓣逆流)]]
- **骨骼**：**高瘦身材**、四肢細長（臂展 > 身高）、**蜘蛛指（arachnodactyly）**、**手腕徵／拇指徵（wrist/thumb sign）**、漏斗胸／雞胸（pectus）、**脊柱側彎**、關節過度鬆弛、高拱顎、扁平足。
- **眼**：**晶狀體脫位（ectopia lentis，典型向上／顳側移位）**、高度近視、視網膜剝離、青光眼、白內障早發。
- **心血管（決定預後）**：
  - **主動脈根部擴張／動脈瘤 → 剝離／破裂**（最致命）
  - **二尖瓣脫垂（MVP）** → 二尖瓣逆流（[[Mitral valve prolapse(二尖瓣脫垂)]]、[[Mitral Regurgitation(二尖瓣逆流)]]）
  - 主動脈瓣逆流（根部擴張引起）
- **其他**：硬脊膜膨出（dural ectasia）、自發性氣胸、皮膚萎縮紋。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 FBN1 基因檢測、裂隙燈　影像 [[Echocardiography(心臟超音波)]]、[[CTA(電腦斷層血管攝影)]]
- **修訂版 Ghent criteria（2010 revised）**：以兩大**主徵**為核心——**主動脈根部動脈瘤／剝離**與**晶狀體脫位**——搭配 FBN1 突變、家族史、systemic score（骨骼／皮膚等系統性特徵計分）綜合判定。
- **Echocardiography**：測**主動脈根部直徑（Z-score）**、評估 MVP／AR——診斷與**終身追蹤**主力。
- **CT／MR 血管攝影**：全主動脈評估、追蹤遠端擴張、疑剝離時。
- **裂隙燈（slit-lamp）**：找晶狀體脫位。
- **基因檢測**：FBN1 突變（協助不典型或家族篩檢）。
- **家族史**：一等親罹病是重要線索。
- 鑑別：Loeys-Dietz、血管型 [[Ehlers-Danlos syndrome(鬆皮症)]]、同型胱胺酸尿症（homocystinuria，晶狀體向**下**移位、血栓傾向）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **晶狀體移位方向**：Marfan **向上／顳側**；同型胱胺酸尿症**向下／鼻側**——鑑別經典考點。
- **年輕人的主動脈剝離**：Marfan 是重要背景病因，年輕撕裂痛別掉以輕心。
- **主動脈根部（Valsalva 竇）** 是最典型的擴張位置，測量與手術決策都盯這裡。
- **運動限制**：避免劇烈接觸性／等長（重量）運動，降低主動脈剪力。
- **懷孕高危**：血流動力負荷 + 荷爾蒙使主動脈更脆弱，剝離風險大增，需孕前評估根部直徑。
- 手腕徵（拇指與小指繞手腕重疊）、拇指徵（拇指過掌緣）是快速床邊篩檢。

## 💊 治療（Acute / Chronic）
> 用藥 [[Beta blocker(乙型阻斷劑)]]、ARB（losartan）
### 慢性（核心：減緩主動脈擴張 + 追蹤）
- **β-blocker**：降低主動脈剪力（dP/dt），減緩根部擴張——長期基石。
- **ARB（losartan）**：拮抗 TGF-β 訊號，減緩擴張（可與 β-blocker 併用；相對效益仍有討論）。
- **定期 echo 追蹤**主動脈根部直徑（穩定者約每年一次，快速擴張者更密）。
- **生活型態**：避免劇烈／接觸性／等長運動；戒菸（降氣胸／血管風險）。
- **遺傳諮詢**、一等親篩檢、孕前評估。
### 預防性手術（主動脈根部置換）
- 達**手術直徑閾值**時做**預防性主動脈根部置換**（避免剝離）。
- 閾值約**≥ 5.0 cm**（⚠️需查證確切數字），若合併**家族剝離史、快速擴張（>0.5 cm/年）、將懷孕、重度 AR/MR** 則**閾值下修**。
### 急性
- **急性主動脈剝離** → 依 [[Aortic dissection(主動脈剝離)]] 原則：先 β-blocker 控 HR/BP，**Stanford A 緊急外科**。

## ⚠️ 併發症
- **主動脈剝離／破裂**（[[Aortic dissection(主動脈剝離)]]，最致命）
- **主動脈瓣逆流**（根部擴張）、二尖瓣逆流／脫垂
- **自發性氣胸**
- 視網膜剝離、青光眼、白內障（眼部）
- 硬脊膜膨出（下背痛、頭痛）
- 脊柱側彎、關節問題
- **妊娠期主動脈剝離**（高危）

## 📊 嚴重度評分 / 風險分層（scoring）
> Ghent 定「是不是 Marfan」；主動脈根部直徑（Z-score）定「何時手術」
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **修訂版 Ghent criteria（2010）** | 診斷 | 主動脈根部瘤／剝離 + 晶狀體脫位為兩大主徵；配 FBN1、家族史、systemic score 綜合判定 |
| **主動脈根部直徑 / Z-score** | 追蹤 + 手術決策 | 直徑越大／擴張越快 → 剝離風險↑；達閾值（約 ≥5.0 cm⚠️需查證，有危險因子則下修）→ 預防性手術 |
