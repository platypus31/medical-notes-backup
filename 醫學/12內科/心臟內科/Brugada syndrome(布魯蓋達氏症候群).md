# Brugada syndrome（布魯蓋達氏症候群）

> [!abstract] 一句話：遺傳性鈉離子通道病變（多為 SCN5A），結構正常的心臟卻在 **V1–V2 出現 coved 型 ST 抬高**，可引發 VF、**休息/睡眠中猝死**。**發燒會誘發**，唯一被證實降猝死的治療是 **ICD**。

> [!danger] 🚨 值班紅旗
> 1. **年輕人（尤其東南亞男性）夜間猝死/被救回 + V1–V2 coved ST↑ → 想 Brugada**
> 2. **發燒是強誘發因子** → Brugada 病人發燒要**積極退燒 (acetaminophen)** 並監測心電圖
> 3. **避免誘發藥物**：鈉通道阻斷劑（Ic 類 flecainide/propafenone）、某些抗憂鬱/抗精神病、古柯鹼、過量酒精（查 brugadadrugs.org）
> 4. **電風暴 (electrical storm) → isoproterenol 輸注**；反覆 VF 且有 ICD → 加 quinidine

## 📖 定義 / 病生理（這是什麼）
- **離子通道病**：最常見 **SCN5A（心臟鈉通道 INa↓）**，體染色體顯性、不完全外顯；右心室流出道(RVOT)復極異常。
- **結構性心臟正常**（與心肌病、缺血不同）。
- **ECG 型態**：
  | 型 | 型態 | 意義 |
  | --- | --- | --- |
  | **Type 1（診斷性）** | **coved（穹窿型）ST 抬高 ≥2 mm + T 波倒置**，V1–V2 | 自發或藥物誘發即可診斷 |
  | **Type 2** | saddleback（馬鞍型）ST ≥0.5 mm | 非診斷，需誘發試驗轉 Type 1 |
- **誘發/加重因子**：**發燒**、鈉通道阻斷劑、迷走張力↑（睡眠/餐後）、電解質異常、酒精/古柯鹼

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Syncope(昏厥)]]、[[Palpitation(心悸)]]
- 多數**無症狀**（健檢/家族篩檢發現）
- **昏厥、夜間瀕死呼吸(agonal breathing)**、心悸
- **被救回的心跳停止 / 猝死**可為首發，常發生於**休息或睡眠中**
- 家族史：年輕不明猝死

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 12-lead ECG（含高位右胸導程）、電解質、發燒排查　影像 Echo（排除結構病）／基因檢測
- **Type 1 ECG 型態**（自發或藥物誘發）為診斷核心
- **高位右胸導程**：V1–V2 上移到第 2、3 肋間可提高偵測率
- **藥物誘發試驗**：ajmaline / flecainide / procainamide（監測下）把 Type 2 轉成 Type 1（⚠️需查證試驗劑量與流程）
- **Echo/MRI 排除結構性疾病**（如 ARVC）
- **基因檢測 + 家族篩檢**

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **Brugada phenocopy**：發燒、高血鉀、缺血、藥物可造成類似 ECG → 要在**無誘因狀態**下重評
- **發燒誘發**：發燒時做 ECG 有機會現形；病人一發燒就要退燒 + 監測
- **危險分層以「症狀 + 自發 Type 1」為主**（曾心跳停止/昏厥 + 自發 Type 1 風險最高）
- **EP study 的角色有爭議**，不作為唯一決策依據
- 男性、東南亞（含台灣）盛行率較高，猝死多在夜間

## 💊 治療（Acute / Chronic）
> 用藥 [[Isoproterenol]]、[[Quinidine(奎尼丁)]]、[[Acetaminophen(乙醯胺酚)]]
### 急性
- **VF/心跳停止 → 去顫 + CPR (ACLS)**
- **電風暴 → isoproterenol 輸注**（增加 L-type Ca 電流、拉平 ST）
- **積極退燒**（acetaminophen），矯正電解質，停一切誘發藥
### 慢性（唯一降猝死＝ICD）
- **ICD**：適用於曾心跳停止、記錄到 VT/VF、或自發 Type 1 + 不明原因昏厥的高風險者
- **Quinidine**：減少 VF/ICD 放電（ICD 不適用或電風暴頻繁者）
- **生活型態**：避免誘發藥、發燒立即處理、避免過量酒精
- 家族基因諮詢與篩檢

## ⚠️ 併發症
- **多形性 VT / VF → 猝死**（尤其夜間）
- 電風暴、反覆 ICD 放電
- ICD 相關併發症（不當放電、感染、導線）

## 📊 嚴重度評分 / 風險分層（scoring）
> 以「症狀 + 自發 Type 1」為核心；Sudden Death Risk 模型輔助
| 因子 | 風險方向 |
| --- | --- |
| **曾心跳停止 / 記錄到 VF** | 最高風險 → ICD |
| **自發性 Type 1 + 不明昏厥** | 高風險 |
| **僅藥物誘發 Type 1、無症狀** | 較低風險，多觀察 |
| **Shanghai score / 相關風險模型** | 綜合 ECG＋病史＋家族史＋基因 |
