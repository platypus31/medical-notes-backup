# Hypertrophic Obstructive CardioMyopathy（肥厚型阻塞性心肌病）

> [!abstract] 一句話：基因性（sarcomere 突變）**非對稱性中膈肥厚**造成左心室出口動態阻塞（LVOT obstruction）+ 舒張功能不良；是**年輕人／運動員猝死**的頭號心因。值班最關鍵：**任何降前負荷／降後負荷的動作（利尿、硝酸鹽、脫水、Valsalva）都會加重阻塞使病人惡化**。

> [!danger] 🚨 值班紅旗
> 1. **年輕人運動中暈厥／猝死** → 想 HOCM，直到排除為止；有家族猝死史更要警覺
> 2. **絕對禁忌：低血壓／胸痛時反射性給 NTG、利尿劑、脫水** → 降前負荷 = LVOT 阻塞更嚴重 → 血壓更低（惡性循環）。HOCM 低血壓要**補液 + phenylephrine（純 α，升後負荷）**，不要用 dobutamine／dopamine 等強心升壓（增加收縮力反而加重阻塞）
> 3. **雜音在 Valsalva／站立時變大聲**（前負荷↓ → 阻塞↑）＝ HOCM；相對地 AS 在 Valsalva 變小聲——床邊快速鑑別
> 4. 出現新發**心房顫動（AF）→ 血流動力可急速崩解**（失去心房收縮，仰賴心房充盈的僵硬心室代償不足）；需積極恢復竇律／控速
> 5. 高危猝死特徵 → 評估植入 **ICD**：家族猝死史、不明原因暈厥、非持續性 VT、中膈厚度 ≥30 mm、運動後血壓反應異常

## 📖 定義 / 病生理（這是什麼）
- **本質**：體染色體顯性遺傳的 **sarcomere 蛋白基因突變**（最常見 β-myosin heavy chain *MYH7*、myosin-binding protein C *MYBPC3*），造成心肌不成比例肥厚（多為非對稱性中膈肥厚 ASH）。
- **兩大血流動力問題**：
  1. **動態 LVOT 阻塞**：肥厚中膈 + 二尖瓣前葉在收縮期被高速血流吸向中膈（**SAM, systolic anterior motion**）→ 出口變窄 + 常合併二尖瓣逆流。阻塞程度**隨負荷狀態變動**（動態）。
  2. **舒張功能不良**：僵硬肥厚心室充盈受限 → 高左心室舒張末壓、肺鬱血。
- **阻塞的動態性（值班核心）**：
  | 讓阻塞**變嚴重**（雜音↑） | 讓阻塞**變輕**（雜音↓） |
  | --- | --- |
  | ↓前負荷：Valsalva、站立、脫水、利尿、硝酸鹽 | ↑前負荷：蹲下、平躺抬腿、補液 |
  | ↓後負荷：血管擴張劑、運動 | ↑後負荷：handgrip、phenylephrine |
  | ↑收縮力：digoxin、dobutamine、運動 | ↓收縮力：β-blocker、CCB |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Chest pain(胸痛)]]、[[Syncope(昏厥)]]、[[Dyspnea(呼吸困難)]]
- 多數**無症狀**，常於健檢雜音或家族篩檢發現。
- 症狀：**運動性呼吸困難**（最常見）、胸痛（供需失衡型心絞痛，冠脈可正常）、**運動性暈厥／近暈厥**、心悸（AF/VT）、猝死可為初發表現。
- 理學徵象：
  - **收縮中晚期漸強漸弱雜音**（胸骨左緣，不傳向頸部，與 AS 相反）；**Valsalva／站立時變大聲**、蹲下／handgrip 變小聲。
  - **雙峰快速上升的頸動脈脈（bisferiens / spike-and-dome）**、apical thrill、S4。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[ECG(心電圖)]]　影像 [[Echocardiography(心臟超音波)]]、[[Cardiac MRI(心臟磁振造影)]]
- **心臟超音波（首選）**：非對稱性中膈肥厚（中膈/後壁 ≥1.3）、**LVOT 壓力差（≥30 mmHg 有意義，≥50 mmHg 需積極處理）**、SAM、二尖瓣逆流、舒張功能。誘發性（Valsalva／運動）測壓差抓隱性阻塞。
- **Cardiac MRI**：精確測壁厚、心尖型／局部肥厚、**延遲增強（LGE）纖維化 → 猝死風險評估**。
- **ECG**：LVH、深而窄的**中膈型 Q 波**（下壁 II/III/aVF + 側壁 I/aVL/V5/V6）、廣泛 T 波倒置（心尖型 HOCM 可見巨大 T 倒置）。
- **基因檢測 + 一等親家族篩檢**（超音波 ± 基因）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）

> > 心電圖變化： 左側導極（I、AVL、V5、V6）、下方導極（II、III、AVF）Q 波明顯

> [!PDF|red] [[長庚銀蛋口袋書(印).pdf#page=150&selection=162,0,202,1&color=red|長庚銀蛋口袋書(印), p.150]]
> > 【補充】AS & HOCM 在 Valsalva maneuver 下雜音的變化： 在 Valsalva maneuver 下，venous return 及回 LV 的血流減少，因此： 
> > 1）[[Aortic stenosis(主動脈瓣狹窄)]]：回左心的血量減少，產生擾流的血就比較少，murmur 變小聲。 
> > 2）HOCM：回左心的血量減少，更容易使原本已經狹窄的出口更阻塞，murmur 變大聲。

- **與 AS 的床邊鑑別**：HOCM 雜音在 Valsalva／站立**變大**、頸動脈脈**快速雙峰**；AS 雜音 Valsalva 變小、頸動脈脈 **parvus et tardus（細遲）**、傳向頸部。
- **HOCM 的心絞痛冠脈可完全正常**（供需失衡 + 心肌橋），別因冠脈乾淨就排除心因胸痛。
- 心尖型 HOCM（Yamaguchi）→ 巨大對稱 T 波倒置，易被誤判缺血。
- **競技運動禁忌**（誘發猝死）；避免脫水、桑拿、突然用力屏氣。

## 💊 治療（Acute / Chronic）
> 用藥 [[Beta Blocker]]、[[Calcium Channel Blocker(CCB)]]、[[Disopyramide]]、[[Phenylephrine]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 減輕症狀 / 降阻塞（慢性）
- **一線：β-blocker**（如 metoprolol、bisoprolol）—— 降心率、延長舒張充盈、降收縮力 → 降阻塞。
- β-blocker 不耐受 → **non-DHP CCB（verapamil / diltiazem）**（⚠️ 嚴重阻塞 + 低血壓／肺高壓時 verapamil 慎用，血管擴張可惡化）。
- 難治性阻塞加 **disopyramide**（負性收縮力）。
- **新一代（2020 後）：mavacamten**（心肌肌凝蛋白抑制劑）—— 症狀性阻塞型 HOCM 可顯著降 LVOT 壓差（EXPLORER-HCM）。
- **避免**：純血管擴張劑（NTG、DHP CCB、ACEI 高劑量）、利尿劑（除非明顯充血、小心用）、digoxin、強心劑。
### 侵入性（藥物難治 + LVOT ≥50 mmHg 有症狀）
- **中膈肌肉切除術（surgical septal myectomy）**：首選、長期效果最佳（有經驗中心）。
- **酒精中膈消融（alcohol septal ablation）**：手術高風險者替代（併發完全房室阻斷需節律器風險）。
### 急性血流動力惡化 / 低血壓（值班）
- **補液擴容 + phenylephrine（純 α 升後負荷）**；β-blocker（esmolol）降心率。
- **禁**：NTG、利尿、正性肌力藥、IABP（降後負荷）。
### 猝死預防
- 高危特徵 → **ICD**（見紅旗第 5 條）。

## ⚠️ 併發症
- **心因性猝死（SCD）**（VT/VF）—— 最嚴重
- **心房顫動**（血栓栓塞風險高，需抗凝）
- 進行性心衰（舒張性 → 末期可轉擴張／收縮衰竭「burnt-out」）
- 感染性心內膜炎（SAM 造成二尖瓣受損處）
- 二尖瓣逆流

## 📊 嚴重度評分 / 風險分層（scoring）
> LVOT 壓差定「阻塞嚴重度」；HCM Risk-SCD 定「要不要植 ICD」
| 評分 / 指標 | 用途 | 意義 |
| --- | --- | --- |
| **LVOT 壓力差** | 阻塞嚴重度 | ≥30 mmHg 有意義；**≥50 mmHg** = 症狀性需積極治療門檻 |
| **ESC HCM Risk-SCD** | 5 年猝死風險 | 納入年齡、中膈厚度、LA 大小、LVOT 壓差、家族猝死史、暈厥、NSVT → 估 5 年 SCD% 決定 ICD |
| 中膈厚度 | 猝死風險 | **≥30 mm** 為主要高危因子 |
