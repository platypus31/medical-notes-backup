# Massive Hemorrhage（大量失血）

> [!abstract] 一句話：短時間內大量流血 → **出血性休克＋創傷性凝血病（trauma-induced coagulopathy）**。核心是**及早啟動大量輸血流程（MTP）、平衡輸血（1:1:1）、止血優先於補液、permissive hypotension**，避免「致命三角」（低體溫／酸中毒／凝血病）惡性循環。

> [!danger] 🚨 值班紅旗
> 1. **心跳快＋脈壓窄＋意識改變 = 休克代償期**，血壓掉下來已是晚期（年輕人可掉到 Class III 才低血壓）→ 別被「血壓還好」騙
> 2. **早期啟動 MTP + tranexamic acid（TXA）**：外傷出血 **3 小時內**給 TXA（越早越好），逾時反可能有害
> 3. **血品 1:1:1（RBC:FFP:PLT）平衡輸**，別只灌 RBC 與大量晶體液（稀釋性凝血病＋沖走血塊）
> 4. **止血是根本**：找出血源（外壓／內視鏡／血管栓塞／手術），並打斷**致命三角**（保暖／糾正酸中毒／補凝血）

## 📖 定義 / 病生理（這是什麼）
- **大量失血定義（常用）**：24 小時內失血 ≥ 一個全身血量（成人約 70 mL/kg）／3 小時內失血 ≥ 半個血量／出血速率 ≥ 150 mL/min／需輸 ≥ 10 U RBC/24h（或成人 ≥ 4 U/1h 且持續）。
- **出血性休克病生理**：血量↓ → 前負荷↓ → 心輸出↓ → 組織灌流不足 → 無氧代謝、**乳酸酸中毒**；交感代償（心跳↑、血管收縮）維持血壓直到失代償。
- **致命三角（lethal triad）**：**低體溫 → 凝血酶活性↓；酸中毒 → 凝血因子失能；凝血病 → 更多出血**，三者互相加乘惡化。
- **創傷性凝血病（TIC）**：組織損傷＋休克即早期出現（非單純稀釋），機轉含 protein C 活化、纖溶亢進、內皮 glycocalyx 破壞。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Shock(休克)]]、[[Tachycardia(心搏過速)]]
- **休克徵象**：心跳快、脈搏弱、四肢冰冷濕黏、微血管回填延遲、脈壓窄、尿量↓、焦躁→嗜睡→意識喪失。
- **失血分級（ATLS，約略）**：
  | Class | 失血量（%血量） | HR | 血壓 | 意識 |
  | --- | --- | --- | --- | --- |
  | I | <15%（<750 mL） | 正常 | 正常 | 略焦慮 |
  | II | 15–30% | ↑（>100） | 收縮壓多正常、**脈壓窄** | 焦慮 |
  | III | 30–40% | ↑↑（>120） | **收縮壓開始下降** | 混亂 |
  | IV | >40%（>2000 mL） | ↑↑↑ | 明顯低血壓 | 嗜睡／昏迷 |
- 出血源線索：外傷部位、黑便／吐血（GI）、腹脹（腹內）、骨盆／長骨骨折、術後引流量暴增、產後出血。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[CBC(全血球計數)]]、[[ABG(動脈血液氣體分析)]]、[[Coagulation profile(凝血功能)]]、[[Lactate(乳酸)]]　影像 [[FAST(創傷超音波)]]、[[CTA(電腦斷層血管攝影)]]
- **床邊即時**：ABC 評估、生命徵象、乳酸／base deficit（休克嚴重度與復甦指標）、**FAST／eFAST**（找腹腔／心包／胸腔積血）。
- **實驗室**：CBC（Hb 早期可正常，別依賴單次值）、PT/aPTT/INR/fibrinogen、**TEG/ROTEM（黏彈性檢測，指導目標性輸血）**、電解質（尤其 K⁺、鈣，大量輸血致低鈣）、血型與交叉配對。
- **影像**：血流動力穩定才去 CT（含 CTA 找出血點）；不穩定 → 直接手術／介入。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **Hb 初期正常不代表沒大出血**（急性失血血液未稀釋前 Hb 可正常）→ 看趨勢＋臨床。
- **大量輸血代謝陷阱**：檸檬酸鈣結合 → **低血鈣（ionized Ca↓）**（影響凝血與心肌）；儲存血 K⁺↑ → **高血鉀**；輸太多冷血 → **低體溫**。要主動監測補鈣、保溫。
- **permissive hypotension**（未止血前目標 SBP 約 80–90 mmHg／可觸及橈動脈脈搏）避免沖走血塊、稀釋凝血——**但頭部外傷例外**（需維持腦灌流，目標較高）[⚠️需查證：目標值依院規／病情]。
- **TXA 時間窗**：外傷出血 **≤3 小時**給效益最大，逾 3 小時給可能增加死亡（CRASH-2）。
- **止血 > 補液**：大量晶體液復甦會加重稀釋性凝血病與致命三角。

## 💊 治療（Acute / Chronic）
> 用藥 [[Tranexamic acid(傳明酸)]]、[[Calcium(鈣)]]、[[Fibrinogen(纖維蛋白原)]]
### 急性（同步進行）
1. **止血優先**：直接壓迫／止血帶／填塞、骨盆固定帶、內視鏡止血、血管栓塞、緊急手術「damage control」。
2. **啟動 MTP**：**RBC:FFP:PLT ≈ 1:1:1** 平衡輸血；有 TEG/ROTEM 則走目標導向（低 fibrinogen → cryoprecipitate／fibrinogen concentrate）。
3. **TXA**：外傷 3 小時內給（loading + maintenance，**確切劑量查 protocol／CRASH-2 regimen**）[⚠️需查證：實際 1 g IV over 10 min 再 1 g over 8h 為 CRASH-2 用法，用前確認院規]。
4. **打斷致命三角**：主動保溫（暖血、暖液、升溫毯）、糾正酸中毒（改善灌流為主）、補凝血因子。
5. **監測補鈣**：大量輸血監測 ionized Ca，低鈣則補鈣（劑量查藥典）。
6. **permissive hypotension**（未止血前，頭傷除外）＋限制晶體液。
### 後續
- 找並根治出血源、加護監測、預防／處理 ARDS、AKI、DIC；復甦終點看乳酸清除、base deficit、尿量、末梢灌流。

## ⚠️ 併發症
- **出血性休克 → 多重器官衰竭**、心跳停止
- **DIC**、稀釋性／消耗性凝血病
- 大量輸血：**低血鈣、高血鉀、低體溫、代謝性鹼中毒（檸檬酸代謝後）、TRALI／TACO、輸血反應**
- 急性腎損傷、ARDS、腸道缺血、腔室症候群（腹腔）

## 📊 嚴重度評分 / 風險分層（scoring）
> ABC / TASH 早期預測「要不要啟動 MTP」；ATLS class 估失血量
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **ATLS 失血 Class I–IV** | 估失血量與嚴重度 | Class III（30–40%）開始血壓下降；IV（>40%）危及生命 |
| **ABC score** | 外傷是否啟動 MTP（0–4） | penetrating機轉／SBP≤90／HR≥120／FAST陽性各1分，**≥2 分**建議啟動 MTP |
| **Shock Index（HR/SBP）** | 休克嚴重度 | **>0.9–1.0** 提示明顯休克／需大量輸血 |
| **TASH score** | 創傷大量輸血機率 | 分數越高需 MTP 機率越高 |
