# Ischemic CardioMyopathy（缺血性心肌病）

> [!abstract] 一句話：**冠狀動脈疾病（CAD）造成心肌缺血/梗塞 → 左心室收縮功能低下（LV dysfunction）的心肌病**。是**擴張型心肌病（DCM）最常見的病因**（約佔 2/3），與 non-ischemic DCM 的關鍵分野在於「有沒有明顯冠脈病變」——這決定了要不要 revascularization。

> [!danger] 🚨 值班紅旗
> 1. **已知 ICM/低 EF 病人突發胸痛 + 新 ST 變化 / troponin↑** → 疑急性缺血事件（ACS），走 [[Acute Coronary Syndrome(急性冠心症)]] 流程，別當單純慢性心衰惡化
> 2. **EF 極低 + 心悸/暈厥/持續性 VT** → 惡性心律不整/猝死高風險，接監視器、備 defibrillator，查電解質（K⁺/Mg²⁺）
> 3. **急性喘惡化 + 肺水腫** → 缺血可能是誘因，先排除新發缺血再處理鬱血（見 [[Congestive Heart Failure(鬱血性心臟病)]]）
> 4. **EF ≤ 35%（優化 GDMT ≥3 月後）** → 記得評估 **ICD 一級預防**，這是值班/查房容易漏掉但攸關存活的動作

## 📖 定義 / 病生理（這是什麼）
- **定義**：因 **CAD（冠狀動脈狹窄/阻塞）**導致心肌缺血、梗塞或反覆頓抑（stunning）/冬眠（hibernation），造成左心室收縮功能顯著低下（常定義為 **LVEF ≤ 40%**）的心肌病變。
- **常見的操作型定義（研究常用）**：LVEF ≤ 35–40% 且合併以下任一：① 左主幹或近端 LAD ≥ 75% 狹窄；② ≥ 2 條主要冠脈 ≥ 75% 狹窄；③ 過去心肌梗塞/血運重建史。（⚠️確切 cut-off 各文獻略有出入，需查證）
- **病生理**：
  - **梗塞區**：不可逆壞死 → 疤痕/纖維化 → 局部無收縮 + adverse remodeling（心室擴大、球形化）。
  - **冬眠心肌（hibernating myocardium）**：慢性低灌流下**存活但不收縮**，血運重建後**功能可回復**——這是評估 viability 的意義所在。
  - **頓抑心肌（stunned）**：短暫缺血後暫時失能，灌流恢復後自行改善。
  - remodeling 啟動 RAAS/交感代償 → 惡化為臨床心衰（與 [[Heart Failure(心臟衰竭)]] 的神經內分泌軸相同）。
- **危險因子** = CAD 危險因子：[[Hypertension(高血壓)]]、糖尿病、[[Hyperlipidemia(高血脂)]]、吸菸、家族史、[[Atherosclerosis(動脈粥狀硬化)]]。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Chest pain(胸痛)]]
- **心衰症狀為主**：勞動性呼吸困難、端坐呼吸、PND、疲倦、運動耐受↓、下肢水腫。
- **心絞痛可有可無**：部分病人有典型/非典型胸痛（提示存活但缺血心肌）；也可**完全無胸痛**（尤其糖尿病），只表現心衰。
- **既往史線索**：心肌梗塞、PCI/CABG、明確 CAD 危險因子群。
- **理學**：S3、心尖搏動外移、二尖瓣功能性逆流雜音、鬱血徵象（JVD、rales、水腫）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[ECG(心電圖)]]、[[Troponin(肌鈣蛋白)]]　影像 [[Echocardiography(心臟超音波)]]、[[Coronary angiography(冠狀動脈攝影)]]
- **關鍵鑑別任務：ischemic vs non-ischemic**——決定是否需血運重建，故**新診斷的收縮性心衰都應評估冠脈**。
- **Echocardiography**：測 LVEF、**局部區域性室壁運動異常（RWMA，沿冠脈分布）**強烈提示缺血成因；看瓣膜、附壁血栓。
- **冠脈評估**：
  - **Coronary angiography（冠脈攝影，診斷金標準）**：確認冠脈解剖與病變程度、可同時血運重建。
  - **Coronary CTA**：低-中度可能性者可非侵襲性排除顯著 CAD。
- **心肌存活性/缺血評估**（決定 revascularization 效益）：cardiac MRI（LGE 看疤痕/透壁程度）、dobutamine stress echo、PET/SPECT viability study。
- **ECG**：病理性 Q wave（舊梗塞）、ST-T 變化、LBBB、心律不整。
- **心臟 MRI**：**LGE 呈冠脈分布的心內膜下/透壁延遲增強** → 支持缺血性；non-ischemic 常為中層/心外膜下型態（有助鑑別）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **ICM 是 DCM 最常見病因**——看到擴大低 EF 的心臟，第一件事是問「冠脈評估做了沒」。
- **RWMA 沿冠脈分布 = ischemic 傾向**；全心瀰漫性運動低下較偏 non-ischemic（但非絕對）。
- **Viability 是關鍵字**：有存活（冬眠）心肌者，血運重建才可能回復功能；全疤痕區重建無益。
- **ICD 一級預防別漏**：缺血性、優化 GDMT ≥3 個月後 LVEF **≤ 35%**（NYHA II–III）即符合適應症（缺血性心肌梗塞後至少 40 天）。（⚠️天數/EF 門檻依指引，需查證個別條件）
- 別把「急性缺血事件」誤當「慢性心衰惡化」——troponin/ECG 動態變化要追。

## 💊 治療（Acute / Chronic）
> 用藥 [[Aspirin(阿斯匹靈)]]、[[Statin(施德丁)]]、[[Furosemide(呋洛西邁)]]
### 急性（合併急性缺血/失代償時）
- 若為 **ACS** → 走 ACS/再灌流流程（見 [[Acute Coronary Syndrome(急性冠心症)]]、[[ST elevation Myocardial infarction(ST段上升心肌梗塞)]]）。
- 急性心衰鬱血 → LMNOP 處置（利尿、硝酸鹽、O₂、體位），見 [[Congestive Heart Failure(鬱血性心臟病)]]。
- 惡性心律不整 → 依 ACLS 處置。

### 慢性（雙軌：治 CAD ＋ 治心衰）
1. **心衰 GDMT 四支柱**（HFrEF）：**ARNI + β-blocker + MRA + SGLT2i**，儘早疊加滴定（同 [[Heart Failure(心臟衰竭)]]）。
2. **抗缺血 / 二級預防**：抗血小板（aspirin ± P2Y12）、**高強度 statin**、控制血壓/血糖、戒菸。
3. **血運重建（revascularization）**：對有**存活心肌 + 顯著 CAD**者評估 **PCI 或 CABG**（多血管/左主幹病變 + 低 EF 常偏 CABG；STICH 研究支持 CABG + 藥物於特定族群長期存活獲益）。
4. **裝置治療**：
   - **ICD 一級預防**：優化治療後 LVEF **≤ 35%**（見上）。
   - **CRT**：LVEF 低 + 寬 QRS（尤其 LBBB）。
5. 末期：LVAD／心臟移植。

## ⚠️ 併發症
- **惡性心律不整（VT/VF）與猝死**（疤痕基質致命性 re-entry）
- 反覆急性缺血事件 / 再梗塞
- 進行性心衰、心源性休克
- **左心室附壁血栓**（EF 極低、心尖無收縮）→ 全身性栓塞/中風
- 功能性二尖瓣逆流、心室動脈瘤

## 📊 嚴重度評分 / 風險分層
> 無單一專屬 score；用 LVEF + NYHA + 冠脈負擔 + viability 綜合分層
| 評分/指標 | 用途 | 意義 |
| --- | --- | --- |
| **LVEF** | 收縮功能 + 裝置適應症 | ≤ 35% → 評估 ICD；越低預後越差 |
| **NYHA class** | 症狀嚴重度 | I–IV（見 HF 卡） |
| **冠脈病變範圍** | 血運重建決策 | 左主幹/多血管 → 傾向 CABG |
| **Viability（存活心肌）** | revascularization 效益 | 有存活心肌 → 重建可回復功能 |
| 預後模型 | 風險量化 | Seattle HF Model、MAGGIC |
