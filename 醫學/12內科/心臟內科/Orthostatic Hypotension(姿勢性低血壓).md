# Orthostatic Hypotension（姿勢性低血壓）

> [!abstract] 一句話：站立（或直立傾斜）**3 分鐘內** SBP 下降 **≥20 mmHg** 或 DBP 下降 **≥10 mmHg**，是值班病房**暈厥、跌倒、頭暈**最常見且最容易被忽略的可逆原因。先分「神經性 vs 非神經性」——量對「躺→站」血壓與**心率代償**是關鍵。

> [!danger] 🚨 值班紅旗
> 1. **老人跌倒 / 姿勢性頭暈**先量「躺 5 分 → 站 1 分、3 分」的血壓+心率，別只量一次坐姿血壓就放過
> 2. **站起來 BP 掉但 HR 幾乎沒升（代償性心搏過速鈍化）→ 神經性 OH**（autonomic failure）；**HR 明顯上升 → 非神經性**（脫水／失血），要找容積流失來源（消化道出血、嘔吐腹瀉、利尿過度）
> 3. **新開的藥常是元兇**：利尿劑、α-blocker（如攝護腺藥 tamsulosin）、降壓藥、硝酸鹽、TCA、抗精神病藥——查藥歷
> 4. **治療 OH 的藥會造成臥位高血壓（supine hypertension）**：midodrine／fludrocortisone 睡前避免給藥，最後一劑距躺下 ≥4 小時 ⚠️需查證（依院內規範）

## 📖 定義 / 病生理（這是什麼）
- **定義（共識標準）**：由平躺變為站立（或 head-up tilt ≥60°）**3 分鐘內**，SBP 下降 **≥20 mmHg** 或 DBP 下降 **≥10 mmHg**。若基礎為仰臥高血壓者，SBP 下降 ≥30 mmHg 亦可採用 ⚠️需查證。
- **正常生理代償**：站立時約 500–800 mL 血液滯留於下肢／內臟 → 靜脈回流↓、心輸出↓ → 頸動脈竇／主動脈弓壓力感受器偵測 → 交感興奮（血管收縮 + 心率上升）維持腦灌流。此反射任一環節失效即 OH。
- **亞型（時間軸）**：
  | 亞型 | 定義 | 意義 |
  | --- | --- | --- |
  | **Initial OH** | 站立 **15 秒內**短暫 SBP↓>40／DBP↓>20，隨即回復 | 需 beat-to-beat 監測；年輕人常見 |
  | **Classic OH** | **3 分鐘內**達診斷閾值 | 最常用臨床定義 |
  | **Delayed OH** | **>3 分鐘**才出現 | 早期／輕度 autonomic failure，tilt table 才抓得到 |
- **分兩大類（決定處置方向）**：
  | 類別 | 機轉 | 心率反應 | 常見病因 |
  | --- | --- | --- | --- |
  | **神經性（neurogenic）** | 交感無法適當釋放 norepinephrine，血管收縮失效 | **代償性上升鈍化**（ΔHR 相對小） | Parkinson's disease、多系統萎縮 MSA、純自主神經失調、[[Diabetes Mellitus(糖尿病)]] autonomic neuropathy、amyloidosis |
  | **非神經性（non-neurogenic）** | 容積不足或藥物／機械因素 | **代償性上升明顯**（心率↑） | 脫水、出血、利尿劑、α-blocker、降壓藥、腎上腺功能不全、長期臥床 deconditioning、老化 |
- **神經性 vs 非神經性快速判別**：站立時 **ΔHR / ΔSBP 比值**低（心率沒跟著補）強烈指向神經性；ΔHR 大幅上升指向容積／藥物性 ⚠️需查證（比值切點各文獻不一）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Syncope(昏厥)]]、[[Hypotension(低血壓)]]
- **腦灌流不足**：站立時**頭暈、眼前發黑、視力模糊、暈厥（syncope）／近暈厥（presyncope）**、跌倒。
- **姿勢相關性**是最重要線索：症狀在**站起、久站、餐後（postprandial）、清晨、天熱、運動後**加重；躺下緩解。
- **coat-hanger pain**：頸肩（斜方肌）缺血性痠痛，神經性 OH 特徵表現之一。
- **合併自主神經失調線索**（指向神經性）：便秘／腹瀉、排尿障礙、勃起障礙、無汗／異常出汗、餐後低血壓。
- 老人可能**無典型頭暈**，只表現為**反覆跌倒、意識混亂、全身無力**——高度警覺。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 姿勢性生命徵象（orthostatic vitals）、tilt table test、autonomic function test　影像 —
- **床邊 orthostatic vitals（第一線、值班必做）**：平躺休息 **5 分鐘** → 測 BP/HR → 站立（或坐起）後 **1 分鐘、3 分鐘** 各測一次。同時記錄**心率變化**（區分神經性）。
- **Tilt table test（傾斜床）**：懷疑 delayed OH／反射性暈厥／床邊測不到時使用；可同步做 autonomic testing。
- **自主神經功能檢查**：Valsalva ratio、deep breathing HR variability、QSART（定量發汗）——確認並定位 autonomic failure。
- **找病因的抽血／檢查**：血容量狀態（BUN/Cr、血比容）、血糖／HbA1c、B12、晨間 cortisol（疑腎上腺不全）、蛋白電泳（疑 amyloid）、藥歷回顧。
- **合併檢查**：疑心因性暈厥時做 [[ECG(心電圖)]]；排除貧血、出血。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **量測時機錯過會漏診**：只量坐姿或只量站立 1 分鐘可能錯過 delayed OH——**務必測到站立 3 分鐘**。
- **心率是分型金鑰**：BP 掉但 HR 沒代償 = 神經性；很多人只記血壓忘了看心率。
- **餐後低血壓（postprandial hypotension）**是老人與自主神經失調者的獨立現象，機轉為內臟血流轉移，常與 OH 併存。
- **仰臥高血壓（supine hypertension）與 OH 常在同一病人身上並存**（尤其神經性），治療時是兩難：升壓治日間 OH 又惡化夜間臥位高血壓。
- **藥物是最可逆的元兇**——先減／停可疑藥物，往往比加藥更有效。

## 💊 治療（Acute / Chronic）
> 用藥 [[Midodrine]]、[[Fludrocortisone]]
### 急性 / 立即處置
- 症狀發作時**立即平躺並抬高下肢**恢復腦灌流。
- 非神經性者：**矯正病因**——補液（疑脫水／出血先評估容積並找出血源）、停可疑藥物。
- **物理反壓動作（physical counter-maneuvers）**：交叉雙腿、蹲踞、握拳、繃緊臀腿肌肉，快速升高回心血量。
### 非藥物（第一線、長期基礎）
- **衛教漸進起身**：由臥→坐→站分段，避免猛然站起。
- **增加水分與鹽分**：每日飲水約 2–2.5 L、鈉攝取增加 ⚠️需查證（心衰／腎病／高血壓者須個別化）；症狀急性發作可**快速喝 500 mL 冷開水（water bolus）**升壓。
- **物理輔具**：**腹帶（abdominal binder）**、**下肢壓力襪（compression stockings）**減少下肢滯留。
- **抬高床頭 head-up tilt sleeping**（減少夜間臥位高血壓與晨起 OH）。
- 避免誘因：大餐、酒精、高溫、久站、脫水。
### 藥物（非藥物無效時）
- **Midodrine**（α1 agonist）：升周邊血管阻力；常見 2.5–10 mg 每日三次（日間給，避免睡前）⚠️需查證（起始劑量與上限依處方）；主要副作用**臥位高血壓、起雞皮疙瘩、排尿障礙**。
- **Fludrocortisone**（mineralocorticoid）：擴充血容量；約 0.1–0.2 mg/day ⚠️需查證；注意**低血鉀、水腫、臥位高血壓、心衰惡化**。
- **Droxidopa**（norepinephrine 前驅物）：用於神經性 OH ⚠️需查證（可用性依地區）。
- **輔助**：pyridostigmine（輕度、較不惡化臥位高血壓）；餐後低血壓可考慮 caffeine／少量多餐。

## ⚠️ 併發症
- **跌倒、骨折、頭部外傷**（老人最嚴重後果）
- **暈厥反覆發作**、生活品質下降、失能與活動受限
- **治療引起的臥位高血壓（supine hypertension）** → 長期標的器官傷害風險
- 潛在基礎疾病（Parkinson、amyloidosis、DM autonomic neuropathy）之進展

## 📊 嚴重度評分 / 風險分層（scoring）
> OH 本身無單一通用評分；分層著重「症狀負擔 + 是否神經性 + 跌倒風險」
| 分層維度 | 用途 | 意義 |
| --- | --- | --- |
| **ΔBP 幅度 + 症狀有無** | 症狀性 vs 無症狀 OH | 有症狀者才需積極升壓治療 |
| **心率代償反應** | 神經性 vs 非神經性 | 神經性 → 找 autonomic failure；非神經性 → 找容積／藥物 |
| **跌倒風險評估**（如 Morse Fall Scale） | 住院安全 | 高風險 → 跌倒防護、床邊照護 |
| **合併臥位高血壓** | 治療兩難分層 | 影響升壓藥選擇與給藥時間 |
