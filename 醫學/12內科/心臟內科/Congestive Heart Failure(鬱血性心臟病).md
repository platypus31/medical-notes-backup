# Congestive Heart Failure（鬱血性心臟病）

> [!abstract] 一句話：**「Congestive」強調的是鬱血這個臨床表現**——心臟無法在正常充填壓下維持足夠心輸出，導致血液在肺循環（左心衰）與體循環（右心衰）**鬱積**。本質與 [[Heart Failure(心臟衰竭)]] 同一疾病譜，只是從「積水/鬱血」角度切入；值班最常見的就是這個「濕（wet）」的病人。

> [!danger] 🚨 值班紅旗
> 1. **端坐呼吸 + 冒冷汗 + SpO₂ 掉 + 雙側濕囉音** → 急性肺水腫，坐起來、給 O₂、IV Lasix、硝酸鹽，別讓他平躺
> 2. **收縮壓偏低 / 四肢冰冷 / 尿少（cold & wet）** → 疑心源性休克傾向，此時**猛利尿反而更惡化灌流**，要想 inotrope／找 cardiology（見 [[Cardiogenic shock(心源性休克)]]）
> 3. **體重數天內快速↑ + 下肢壓凹水腫 + JVD** → 容積過載惡化，先問「是不是自己停利尿劑 / 吃太鹹」（~40% 急性惡化誘因）
> 4. **新發或惡化的喘 + 心律不整（AF RVR）/ 胸痛** → 先排除 ACS 與心律不整這兩個可逆誘因，別只當單純 CHF

## 📖 定義 / 病生理（這是什麼）
- **定義**：心臟結構或功能異常 → 無法在正常充填壓下打出足夠心輸出，或只能靠**異常升高的充填壓**維持 → 血液在心臟上游**鬱積（congestion）**。
- **「Congestive」的臨床意義**：CHF 這個詞歷史上泛指伴隨鬱血/水腫表現的心衰，現行指引多統一用 **Heart Failure**（依 LVEF 分 HFrEF/HFmrEF/HFpEF）。臨床把它當成「有明顯體液鬱積表現的心衰」來理解即可。
- **病生理核心（惡性循環）**：心肌受傷 → 心輸出↓ → **神經內分泌代償**（RAAS 活化 → 鈉水滯留↑preload；交感活化 → 血管收縮↑afterload、心率↑）→ 短期撐住血壓，長期加速 **remodeling／纖維化** → 心衰更惡化。GDMT 的邏輯就是**阻斷這條代償軸**。
- **鬱血的兩側（要分左右）**：

| | 左心衰（肺鬱血） | 右心衰（體循環鬱血） |
| --- | --- | --- |
| 上游鬱積處 | 肺靜脈／肺 | 體靜脈／內臟 |
| 典型表現 | 喘、端坐呼吸、PND、[[Pulmonary Edema(肺水腫)]] | JVD、下肢水腫、肝腫大、腹水 |
| 最常見成因 | 缺血/心肌病/高血壓/瓣膜 | **最常見是左心衰本身**；其次肺高壓、右室梗塞 |

- **常見誘因（記憶）**：不遵醫囑/停利尿劑（~40%）、高鹽飲食、ACS/缺血、[[Arrythmia(心律不整)]]（AF RVR）、感染、貧血、腎衰、甲狀腺、藥物（NSAID／CCB／TZD）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Edema(水腫)]]
- **左心鬱血**：勞動性呼吸困難、**端坐呼吸 orthopnea**、**陣發性夜間呼吸困難 PND**、夜咳、粉紅色泡沫痰、肺底 **crackles/rales**。
- **右心鬱血**：**頸靜脈怒張 JVD**、肝頸迴流陽性、下肢**壓凹性水腫**、肝腫大/觸痛、腹水、體重快速增加。
- **低灌流（forward failure）**：疲倦、運動耐受↓、四肢冰冷、脈壓窄、尿少、意識改變。
- **理學重點**：**S3 gallop**（容積過載，較特異）、心尖搏動外移、二尖瓣功能性逆流雜音。
- **Stevenson 床邊分型**（決定處置方向）：以「有無鬱血 wet/dry」×「有無低灌流 warm/cold」分四象限，**warm & wet 最常見**。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[BNP]]、[[ECG(心電圖)]]　影像 [[Chest X-ray(胸部X光)]]、[[Echocardiography(心臟超音波)]]
- **臨床診斷 + 客觀佐證**，可參考 **Framingham criteria**（2 major，或 1 major + 2 minor）。
- **利鈉肽（BNP / NT-proBNP）**：喘的病人**排除/支持心衰**的關鍵；陰性有高陰性預測值。注意肥胖偏低、腎衰/AF/高齡偏高；ARNI 會**升 BNP、降 NT-proBNP**。
- **CXR**：心臟擴大、肺血管重新分布（cephalization）、Kerley B lines、間質/肺泡水腫、肋膜積液。
- **Echocardiography（關鍵）**：測 **LVEF**（決定 HFrEF/mrEF/pEF）、看局部收縮異常（提示缺血）、瓣膜、舒張功能、肺高壓。
- **ECG**：找誘因/病因——缺血、心律不整、LVH、Q wave（舊梗塞）。
- **抽血**：electrolytes、腎功能、troponin（排除 ACS）、TSH、CBC（貧血）、鐵代謝。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **右心衰最常見的原因是左心衰**——別漏了左心這個上游。
- **cold & wet 不要盲目狂利尿**：灌流已差，過度利尿→腎前氮質血症/低血壓惡化，考慮 inotrope。
- **HFpEF 是陷阱**：EF 正常 ≠ 沒心衰；靠症狀＋利鈉肽＋舒張功能。
- **BNP 判讀陷阱**：ARNI 用藥者追 **NT-proBNP** 較準；肥胖者 BNP 假性偏低。
- **S3** 對容積過載較特異；**利尿有效的證據**是體重下降 + 症狀改善，而非單看數字。
- 別忘了找**可逆誘因**（ACS、AF、感染、貧血、不遵醫囑），治誘因常比加藥更關鍵。

## 💊 治療（Acute / Chronic）
> 用藥 [[Furosemide(呋洛西邁)]]、[[Nitroglycerin(硝化甘油)]]
### 急性鬱血處置——「LMNOP」助記
- **L**asix（IV furosemide）：利尿降 preload。IV 劑量常為平日口服總量的 **1–2.5 倍**（⚠️需查證個別化劑量）。
- **M**orphine：降焦慮、venodilator、降 afterload（現行趨勢保守使用，非首選）。
- **N**itrates（硝酸鹽）：venodilator 降 preload；血壓夠時可用，是急性肺水腫快速緩解主力之一。
- **O**xygen ± 非侵襲性正壓通氣（NIPPV/CPAP）：改善氧合、降 preload/afterload。
- **P**osition：坐起、雙腿下垂 → 降 preload。
- 同時**找並治誘因**（ACS→再灌流、AF RVR→控率、感染→抗生素）。低灌流/休克 → 見 [[Cardiogenic shock(心源性休克)]]。

### 慢性——HFrEF 現代 GDMT「四支柱」（越早疊加、儘量滴定到目標劑量）
| 支柱 | 代表藥 | 重點 |
| --- | --- | --- |
| **ARNI**（優於 ACEI/ARB） | sacubitril/valsartan | 需與 ACEI 間隔 36h 避免血管性水腫 |
| **β-blocker** | carvedilol、bisoprolol、metoprolol succinate | **急性肺水腫惡化期不啟動/不加量** |
| **MRA** | spironolactone、eplerenone | 監測 K⁺ 與腎功能 |
| **SGLT2i** | dapagliflozin、empagliflozin | **不論有無糖尿病**都降住院/死亡 |
- 症狀控制：**利尿劑**（緩解鬱血、不改善存活）；容積管理 + 限鈉限水 + 每日量體重。
- 其他：缺鐵者補鐵（IV ferric carboxymaltose）；適應症者裝置治療 **CRT / ICD**；末期考慮 LVAD／移植。
- **HFpEF**：SGLT2i 有實證；其餘以控制血壓、容積、共病為主。

## ⚠️ 併發症
- 惡性 [[Arrythmia(心律不整)]]（AF、VT/VF）與**猝死**
- **Cardiorenal syndrome**（心腎交互惡化）
- 急性肺水腫、[[Cardiogenic shock(心源性休克)]]
- 附壁血栓／全身性栓塞（EF 極低 ± AF）
- 肝鬱血（cardiac cirrhosis）、cardiac cachexia、反覆住院

## 📊 嚴重度評分 / 風險分層
> NYHA 看**當下症狀**（可回復）；ACC/AHA stage 看**疾病進程**（不可回退）；Stevenson 定**急性處置方向**
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **NYHA class** | 症狀嚴重度 | I 無症狀／II 日常活動有症狀／III 輕度活動即症狀／IV 休息即症狀 |
| **ACC/AHA stage** | 疾病階段 | A 高風險無結構病／B 有結構病無症狀／C 有症狀／D 末期難治 |
| **Stevenson 分型** | 急性床邊處置 | warm-dry／warm-wet（利尿）／cold-wet（利尿±inotrope）／cold-dry（補液/inotrope） |
| 預後模型 | 風險量化 | Seattle HF Model、MAGGIC score |
