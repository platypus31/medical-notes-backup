# Single Ventricle（單一心室症）

> [!abstract] 一句話：**只有一個具功能的心室同時承擔體循環與肺循環**的一大類複雜先天心臟病總稱（如 HLHS、三尖瓣閉鎖、DILV、不平衡型 AVSD）；生理核心是**體肺血流在單一心室混合**，平衡取決於 **Qp:Qs**，新生兒常**依賴 PDA 或分流存活**；外科走**分期姑息（Norwood → Glenn → Fontan）**達成單心室循環。

> [!danger] 🚨 值班紅旗
> 1. **PDA 依賴型循環的新生兒**（如 HLHS）→ **PDA 一關就崩潰（休克/發紺）** → **前列腺素 E1（PGE1）維持開放**是救命第一步
> 2. **過度肺血流（Qp:Qs 太高）** → 體循環灌流不足 → 代謝性酸中毒、休克（給高濃度氧會擴張肺血管、惡化竊流，要小心）
> 3. **Fontan 病人低血壓** → Fontan 循環是被動肺灌流，對前負荷/心律/肺阻力極敏感，處置邏輯與正常兩心室不同
> 4. **Fontan 病人新發水腫/腹瀉/低白蛋白** → 想 **蛋白流失性腸病（PLE）**

## 📖 定義 / 病生理（這是什麼）
- **定義**：**一個功能性心室**（另一心室缺如或發育不全到無法獨立支撐一個循環）同時供應體循環與肺循環，屬「功能性單心室生理」的統稱，而非單一疾病。
- **代表病灶**：
  - **左心發育不全症候群（HLHS，hypoplastic left heart syndrome）**——右心室當單心室。
  - **三尖瓣閉鎖（tricuspid atresia）**——左心室當單心室。
  - **雙入口左心室（DILV, double-inlet left ventricle）**。
  - **不平衡型房室中膈缺損（unbalanced AVSD）**、部分內臟異位（heterotaxy）等。
- **病生理核心**：
  - **體、肺循環在單心室內混合** → 動脈血氧飽和度取決於肺血流與體血流的比例 **Qp:Qs**。
  - **平衡（balanced circulation）**：Qp:Qs ≈ 1，SpO₂ 約 75–85%。
  - **Qp 過高**（肺阻力低、肺血流搶走體循環）→ 體灌流不足、心衰、酸中毒；**Qp 過低**（肺流阻塞）→ 嚴重發紺。
  - **常依賴交通存活**：PDA（供體或供肺）、ASD（心房層次混合必需）、VSD 等；PDA 依賴型在動脈導管自然閉合時崩潰 → 需 **PGE1**。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Cyanosis(發紺)]]、[[Dyspnea(呼吸困難)]]、[[Failure to thrive(生長遲滯)]]
- **新生兒期**（依解剖）：
  - **發紺型**（肺血流少）：明顯低血氧、發紺。
  - **心衰型**（肺血流多）：呼吸急促、餵食困難、多汗、生長遲滯、肝腫大。
  - **PDA 依賴型**：導管關閉時急性休克或深度發紺。
- **姑息術後**：術式不同表現不同；Glenn/Fontan 後為慢性輕度發紺（Glenn 期）到接近正常飽和（Fontan 完成）。
- 聽診：依合併病灶有分流雜音、單一 S2（大動脈異常時）等。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[ECG(心電圖)]]、[[Pulse Oximetry(血氧飽和度)]]　影像 [[Fetal Echocardiography(胎兒心臟超音波)]]、[[Echocardiography(心臟超音波)]]、[[Cardiac MRI(心臟磁振造影)]]、[[Cardiac Catheterization(心導管)]]
- **胎兒超音波**：多數可產前診斷，利於分娩規劃（生於有小兒心臟外科的中心）。
- **心臟超音波**：確認單心室解剖、心室形態（左型/右型）、房室瓣、大動脈關係、分流、PDA 依賴與否。
- **心導管**：Glenn/Fontan 前必做——測**肺血管阻力（PVR）、肺動脈壓、心室舒張末壓、解剖**，判斷能否進入下一階段（PVR 過高＝Fontan 禁忌）。
- **Cardiac MRI**：量化心室功能、體肺血流、側枝。
- **新生兒 pulse oximetry 篩檢** 可及早發現關鍵性先天心臟病。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **單心室不是根治而是「改道姑息」**：最終目標是把**體、肺循環分開串聯**（Fontan），讓單心室只推體循環，肺循環變成**被動（無心室泵）灌流**。
- **平衡循環管理靠 Qp:Qs**——盲目給高濃度氧會降肺阻力、增肺血流、偷走體循環（"pulmonary steal"）。
- **PGE1 是 PDA 依賴型的救命藥**（維持導管開放）。
- **Fontan 生理反直覺**：肺灌流靠中心靜脈壓被動驅動 → **極度依賴足夠前負荷、竇性節律、低肺血管阻力**；正壓通氣、脫水、心律不整都會使 Fontan 崩。
- **低肺血管阻力是所有分期能成功的前提**——PVR 高則走不到 Fontan。

## 💊 治療（Acute / Chronic）— 分期外科姑息路徑
> 用藥 [[Prostaglandin E1(前列腺素E1)]]、[[Diuretics(利尿劑)]]、[[Anticoagulation(抗凝治療)]]（劑量查藥典）
- **新生兒穩定**：PDA 依賴型 → **PGE1** 維持導管；平衡 Qp:Qs（避免過度給氧）；必要時控制肺血流。
- **三階段單心室姑息**：
  1. **第一期（新生兒）**：依解剖 —— HLHS 做 **Norwood 手術**（重建主動脈弓＋建立體循環出口＋Blalock-Taussig/Sano 分流供肺）；肺血流過多者做**肺動脈環縮（PA banding）**；肺血流過少者做**體肺分流（BT shunt）**。
  2. **第二期（約 4–6 個月）**：**雙向 Glenn（BDG, bidirectional Glenn / 雙向腔肺吻合）**——上腔靜脈直接接肺動脈，上半身靜脈血被動入肺，減輕單心室容量負荷。
  3. **第三期（約 2–4 歲）**：**Fontan 手術**——將下腔靜脈血也導入肺動脈，**完成單心室循環**：全身靜脈血被動流入肺，單心室專責體循環。
- **Fontan 完成後長期照護**：抗凝/抗血小板（栓塞預防）、心律不整處理、心衰藥物、監測併發症。

## ⚠️ 併發症（尤其 Fontan 循環長期）
- **蛋白流失性腸病（PLE, protein-losing enteropathy）**——低白蛋白、水腫、腹瀉、免疫低下
- **Fontan 相關肝病（FALD, Fontan-associated liver disease）**——鬱血性肝纖維化／肝硬化，可致肝癌
- **心律不整**（房性為主）、**血栓栓塞**
- **塑型性支氣管炎（plastic bronchitis）**、乳糜胸
- 心室功能衰退、發紺（分流殘存）、運動耐受受限；最終部分需心臟移植

## 📊 嚴重度評分 / 風險分層（scoring）
> 無單一命名 score；以 **能否進入下一姑息階段的血流動力學門檻** 為核心風險分層
| 指標 | 用途 | 意義 |
| --- | --- | --- |
| **肺血管阻力（PVR）** | Glenn/Fontan 可行性 | PVR 高＝Fontan 禁忌（被動肺灌流無法克服高阻力） |
| **心室舒張末壓 / 房室瓣逆流** | 手術風險 | 心室功能差、瓣膜逆流重＝高風險 |
| **Qp:Qs** | 循環平衡 | ≈1 為平衡；過高偷體循環、過低則發紺 |
| **SpO₂** | 混合/分流評估 | 平衡期約 75–85%；Fontan 完成後接近正常 |
