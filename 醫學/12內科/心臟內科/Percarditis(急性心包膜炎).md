# Percarditis（急性心包膜炎）

> [!abstract] 一句話：心包膜發炎，經典為**與呼吸/平躺相關的胸痛（前傾坐姿減輕）**＋ 摩擦音 ＋ 廣泛 ST 抬高/PR 下降。多為病毒/特發性、預後好；**NSAID + colchicine** 是主力，值班要盯的是**心包填塞**這個致命併發症。

> [!danger] 🚨 值班紅旗
> 1. **胸痛別只想 ACS**：pericarditis 是**廣泛(diffuse) concave ST↑ + PR 下降**、無 reciprocal change、troponin 可微升(心肌心包炎) — 誤溶栓/誤抗凝有害
> 2. **出現低血壓 + 頸靜脈怒張 + 心音遠 (Beck triad) / 奇異脈 → 心包填塞**，緊急 echo + 心包穿刺
> 3. **高風險特徵**（發燒>38、亞急性、大量積液/填塞、免疫抑制、口服抗凝、外傷、NSAID 治療無效、troponin↑）→ 住院查因
> 4. **急性心包膜炎的類固醇是第二線**（第一線會增加復發）；懷疑細菌性/結核性/尿毒性走各自路線

## 📖 定義 / 病生理（這是什麼）
- **心包膜（臟層+壁層）發炎**，可乾性(dry) 或伴心包積液。
- **診斷需 ≥2/4（歐洲 ESC 標準）**：
  1. **典型胸痛**（尖銳、胸膜性、平躺加劇、前傾坐姿緩解）
  2. **心包摩擦音 (friction rub)**（三相、坐前傾更清楚）
  3. **ECG 變化**（廣泛 ST↑、PR 下降）
  4. **新發/惡化的心包積液**
- **病因**：
  - **病毒/特發性（最常見）**、細菌（含化膿性/結核）、尿毒症、自體免疫（SLE/RA）、腫瘤、放射
  - **心肌梗塞後**：早期(peri-infarction) vs 晚期免疫性 [[Dressler syndrome(卓斯勒症候群)]]

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Chest pain(胸痛)]]、[[Dyspnea(呼吸困難)]]
- **胸痛**：尖銳/胸膜性，**平躺與吸氣加劇、坐起前傾緩解**，可放射至斜方肌脊(trapezius ridge，較特異)
- **心包摩擦音**：三相、抓刮樣，坐前傾呼氣末最清楚（時有時無）
- 可有低度發燒、前驅上呼吸道/腸胃道感染史
- 填塞徵象：頸靜脈怒張、奇異脈、低血壓、心音遠

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 12-lead ECG、CRP/ESR、Troponin、CBC、腎功能　影像 Echocardiography、CXR
- **ECG 四階段**：
  - Stage I：**廣泛 concave ST↑ + PR 下降**（aVR 相反：ST↓、PR↑）
  - Stage II：ST 回到基線、T 波平
  - Stage III：廣泛 T 波倒置
  - Stage IV：回復正常
  - **Spodick sign**：TP 段下斜
- **Echo（必做）**：評估積液量、有無填塞生理（RV 舒張塌陷、IVC 擴張、二尖瓣血流呼吸變異）
- **Lab**：CRP/ESR（發炎與追蹤療效）、troponin（升→myopericarditis）、腎功能（尿毒）、依情況查 TB/自體免疫/培養
- **CXR**：心影多正常，積液 >200 mL 才擴大

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **pericarditis vs STEMI**：pericarditis 為**廣泛**ST↑（非單一冠脈分布）、**PR 下降**、**無 reciprocal ST↓**、無動態演變成 Q 波
- **PR depression 是早期且相對特異的線索**
- **NSAID 無效/復發**要想非病毒病因（TB、自體免疫、腫瘤）
- **類固醇是把雙面刃**：緩解快但**增加復發**，只留給有特定適應症者（自體免疫、尿毒、NSAID/colchicine 禁忌）
- **myopericarditis**（troponin↑）多預後良好但建議**限制運動**並住院觀察

## 💊 治療（Acute / Chronic）
> 用藥 [[NSAID]]、[[Colchicine(秋水仙素)]]、[[Aspirin(阿斯匹靈)]]
### 一線（病毒/特發性）— 抗發炎 + colchicine
- **NSAID**：ibuprofen 600 mg PO q8h，或 aspirin 750–1000 mg PO q8h（**心肌梗塞後 pericarditis 用 aspirin，不用其他 NSAID**）；症狀緩解後逐步減量，療程通常至 CRP 正常（⚠️需查證確切減量方案）
- **Colchicine（加上去降復發）**：0.5 mg PO **BID（<70 kg 或不耐受可 QD）× 3 個月**（復發者 6 個月）
- **限制運動**至症狀消失且 CRP 正常（運動員需更久）
- 加 PPI 護胃
### 二線 / 特殊
- **NSAID+colchicine 無效/禁忌/自體免疫/尿毒 → 低劑量 corticosteroid**（緩慢減量，⚠️需查證起始劑量）
- **化膿性心包炎 → 引流 + 靜脈抗生素**；**結核性 → 抗結核 ± 類固醇**；**尿毒 → 加強透析**
- **復發性 → colchicine 延長 ± anakinra (IL-1 阻斷) / 心包切除（難治）**
### 併填塞
- **心包穿刺 (pericardiocentesis)** 引流減壓（見 [[Cardiac tamponade(心包膜填塞)]]）

## ⚠️ 併發症
- **心包填塞 [[Cardiac tamponade(心包膜填塞)]]**（致命，值班重點）
- **復發性心包炎**（15–30%）
- **縮窄性心包炎 [[Constrictive pericarditis(縮窄性心包膜炎)]]**（尤其結核/化膿/放射後）
- Myopericarditis（心肌受累）

## 📊 嚴重度評分 / 風險分層（scoring）
> 靠「高風險特徵」決定住院 vs 門診
| 分層 | 內容 |
| --- | --- |
| **高風險（住院查因）** | 發燒 >38℃、亞急性起病、大量積液/填塞、免疫抑制、口服抗凝、外傷、troponin↑(myopericarditis)、NSAID 7 天無反應 |
| **低風險** | 典型病毒/特發性、無上述 → 門診經驗性治療 + 追蹤 CRP |
