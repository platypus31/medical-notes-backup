# Pericardium Disease（心包疾病）

> [!abstract] 一句話：心包疾病譜系（發炎→積液→填塞→縮窄）的**總覽卡**。值班最關鍵：分清三個致命鑑別——**急性心包炎（痛但通常良性）、心包填塞（急性血流動力學崩潰）、縮窄性心包炎（慢性右心衰樣）**，三者處置南轅北轍。

> [!danger] 🚨 值班紅旗
> 1. **心包填塞是唯一「分秒必爭」的心包急症**：Beck's triad（低血壓＋頸靜脈怒張＋心音遙遠）＋奇異脈 → 床邊 echo → 緊急心包穿刺。**填塞時禁利尿劑／血管擴張劑，先給輸液**。
> 2. **急性心包炎誤當 STEMI**：心包炎 ECG 是**廣泛（diffuse）concave-up STE ＋ PR 段下降（aVR 相反）**，不侷限單一冠脈分佈；別急著溶栓／導管。
> 3. **縮窄性 vs 限制型心肌病** 難分：縮窄有 **Kussmaul's sign、pericardial knock、CVP 深 x 深 y**，影像見心包鈣化／增厚；治療是**外科心包切除**。
> 4. 尋找可逆病因：尿毒（加強透析）、甲低、結核、化膿性（引流＋抗生素）。

## 📖 定義 / 病生理（這是什麼）
- 心包由臟層＋壁層＋兩層間漿液組成，正常 15–50 mL。疾病譜系：
  1. **[[Acute pericarditis(急性心包膜炎)]]**：心包發炎，胸痛為主。
  2. **[[Pericardial Effusion(心包膜積液)]]**：液體堆積。
  3. **[[Cardiac tamponade(心包膜填塞)]]**：積液壓迫致舒張充盈受限、心輸出下降。
  4. **[[Constrictive pericarditis(縮窄性心包膜炎)]]**：心包纖維化／鈣化僵硬，慢性充盈受限。
- **血流動力學核心差異**：填塞是「**外壓、全期充盈受限、奇異脈明顯**」；縮窄是「**僵硬殼、早期充盈正常後期驟停（dip-and-plateau / square root sign）、Kussmaul 陽性**」。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[ECG(心電圖)]]、[[Cardiac catheterization(心導管)]]　影像 [[Echocardiography(心臟超音波)]]、[[CT(電腦斷層)]]、[[Cardiac MRI(心臟磁振造影)]]
- **急性心包炎（≥2 項）**：典型胸痛、心包摩擦音、廣泛 STE／PR 下降、新發或惡化積液。輔助：CRP/ESR ↑、影像發炎。
- **填塞**：echo 見 RA/RV 舒張塌陷、IVC 擴張不隨呼吸變化、瓣膜血流呼吸變異；床邊即可診斷。
- **縮窄**：CT/MRI 見**心包增厚（>3–4 mm）／鈣化**；心導管見 LV/RV 舒張壓均等化、**dip-and-plateau**、心室互依（ventricular interdependence）。
- **鑑別縮窄 vs 限制型心肌病**：縮窄 → 心包厚、二尖瓣 e' 正常或升高（annulus reversus）、呼吸性血流變異大；限制型 → 心肌浸潤、e' 降低、BNP 顯著升高。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **急性心包炎 STE 是「廣泛且 concave-up ＋ PR 段下降」**，與 STEMI 的「侷限凸面向上 ＋ 對應導程互逆 ST 下降」不同。
- **填塞 vs 縮窄的 JVP 波形**：填塞 = 深 x 無 y；縮窄 = 深 x 深 y（M 或 W 形）。
- **Kussmaul's sign**（吸氣 JVP 上升）→ 縮窄、右心梗塞、限制型；**典型填塞反而不常見 Kussmaul**。
- **奇異脈**在填塞明顯，在縮窄較少見。

## 💊 治療（Acute / Chronic）
> 用藥 [[NSAID]]、[[Colchicine(秋水仙素)]]、[[Aspirin(阿斯匹靈)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
- **急性心包炎**：高劑量 NSAID/ASA（ibuprofen 600–800 mg tid 或 ASA 650–1000 mg tid）＋ **colchicine 0.5 mg bid ×3 個月**；避免類固醇（增復發，除非自體免疫／尿毒／禁 NSAID）。
- **心包填塞**：**緊急心包穿刺**；穩定前給輸液維持前負荷，**避免利尿劑／血管擴張劑**。
- **縮窄性心包炎**：症狀性者行**外科心包切除術（pericardiectomy）**；早期發炎性縮窄可先試抗發炎。
- 皆須**治療潛在病因**（結核抗結核、化膿引流＋抗生素、尿毒透析）。

## ⚠️ 併發症
- 急性心包炎 → 復發性心包炎、積液、填塞、進展為縮窄。
- 填塞 → 休克、心跳驟停。
- 縮窄 → 慢性右心衰、肝鬱血／心因性肝硬化、惡病質。
