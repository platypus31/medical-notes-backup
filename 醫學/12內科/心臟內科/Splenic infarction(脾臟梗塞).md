# Splenic infarction（脾臟梗塞）

> [!abstract] 一句話：脾動脈或其分支阻塞 → 脾實質缺血壞死。表現為**急性左上腹痛**，最常見來自**心因性栓塞（AF）**與**血液疾病**；診斷靠 **contrast CT**，多數**保守治療**，但要警覺膿瘍與破裂。

> [!danger] 🚨 值班紅旗
> 1. **左上腹痛＋發燒＋AF／已知栓子來源** → 想到脾梗塞，別只當「腸胃炎／肌肉痛」
> 2. **找源頭**：脾梗塞常是**全身栓塞的哨兵**（AF、感染性心內膜炎、心內血栓）→ 一定查心臟來源，否則會再栓到腦／腸／腎
> 3. **警覺併發症**：持續高燒／敗血症樣 → 脾**膿瘍**；血壓掉／腹膜刺激 → 脾**破裂／出血**（少見但致命）
> 4. 年輕、無心臟病的脾梗塞 → 篩**血液惡性（骨髓增生）／高凝狀態／鐮刀型**

## 📖 定義 / 病生理（這是什麼）
- **機轉**：脾動脈或分支被栓子或血栓阻塞 → 局部或整脾缺血壞死（脾為**末端動脈**供應，側支少，易梗塞）。
- **常見病因分類**：
  - **栓塞性（心因，最常見）**：[[Atrial Fibrillation(心房顫動)]]、[[Infective Endocarditis(感染性心內膜炎)]] 贅生物、心內附壁血栓、心房黏液瘤、機械瓣、[[Arterial embolism(動脈栓塞)]] 的一部分
  - **血液疾病**：骨髓增生性腫瘤（PV／ET／CML）、鐮刀型細胞病（sickle）、淋巴瘤／白血病脾腫、瘧疾、傳染性單核球增多症（脾腫易梗塞／破裂）
  - **高凝狀態**：抗磷脂症候群、遺傳性血栓體質、惡性腫瘤
  - **局部／血管**：脾動脈瘤／剝離、[[Aortic dissection(主動脈剝離)]] 波及、胰臟炎、外傷、脾扭轉（wandering spleen）、醫源性栓塞治療

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Abdominal pain(腹痛)]]、[[Fever(發燒)]]
- **典型**：急性**左上腹（LUQ）痛**，可轉移至左肩（Kehr sign，橫膈刺激）、隨呼吸／體位加劇。
- 伴隨：發燒、噁心嘔吐、左側肋膜性胸痛；理學可有 LUQ 壓痛、脾腫、偶聞脾摩擦音。
- **約 1/3 可無症狀**（影像偶然發現）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[CBC(全血球計數)]]、[[LDH(乳酸脫氫酶)]]、[[Blood culture(血液培養)]]　影像 [[CT(電腦斷層)]]、[[Echocardiography(心臟超音波)]]
- **顯影劑 CT（首選）**：脾內**楔形（wedge-shaped）低密度區、尖端朝脾門**，是特徵表現；可評估範圍、膿瘍、出血、破裂。
- 超音波／MRI 可輔助（孕婦／腎功能差）。
- **實驗室**：白血球↑、LDH↑（組織壞死）、可有輕度肝酵素／膽紅素上升；找病因 → 血液抹片、凝血／血栓體質、**血液培養**（懷疑心內膜炎）。
- **源頭檢查**：**ECG（找 AF）＋ echocardiography（TTE/TEE 找贅生物／血栓／黏液瘤）** 是必做，找栓子來源。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- 脾梗塞常是「**冰山一角**」——查到就要問「栓子哪來？還會栓去哪？」（腦／腎／腸／肢體）。
- **CT 楔形低密度、尖端指向脾門** = 經典影像 pearl。
- 年輕無心臟病者 → 別漏**骨髓增生性腫瘤／JAK2、高凝狀態、感染（EBV／瘧疾）**。
- **感染性心內膜炎**的脾梗塞可進一步變**脾膿瘍**（持續菌血症）——反覆發燒別當單純梗塞。

## 💊 治療（Acute / Chronic）
> 用藥 [[Analgesics(止痛藥)]]、[[Heparin(肝素)]]、[[Anticoagulant(抗凝血劑)]]
### 急性
- **多數保守**：止痛（含麻醉性止痛依需要）、輸液、監測。
- **治病因**：栓子來源（AF → 抗凝，依 CHA₂DS₂-VASc）、感染性心內膜炎 → 抗生素、血液病 → 血液科處理。
- **抗凝**：由病因決定（心因栓塞需抗凝），非梗塞本身常規；出血風險者權衡。
### 併發症導向
- **脾膿瘍** → 抗生素 ± 引流／脾切除。
- **脾破裂／大出血 / 血流動力不穩** → 血管栓塞或**脾切除**。
### 慢性 / 預防
- 針對來源長期抗凝或抗血小板；脾切除者接種莢膜菌疫苗（肺炎鏈球菌／Hib／腦膜炎球菌）並衛教敗血症風險。

## ⚠️ 併發症
- **脾膿瘍**（尤其感染性心內膜炎源）
- **脾破裂／脾內出血、脾包膜下血腫**
- **脾假性囊腫**、脾功能低下（大範圍梗塞）
- 全身**再栓塞**（腦中風／腸系膜／腎／肢體）——源頭未除

## 📊 嚴重度評分 / 風險分層（scoring）
> 脾梗塞無專屬評分；重點在「找源頭」與「有無併發症」
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **CHA₂DS₂-VASc** | AF 源者抗凝決策 | 分數越高栓塞風險越高，決定長期抗凝 |
| **Duke criteria** | 疑感染性心內膜炎源 | 符合則走 IE 診療（血培＋echo） |
| （臨床分層） | 有無膿瘍／破裂 | 保守 vs 需引流／脾切除 |
