# Aortic dissection（主動脈剝離）

> [!abstract] 一句話：主動脈**內膜撕裂**→血液灌入中膜形成 **false lumen** 並沿血管延伸。是「不能漏掉」的胸痛急症——**Stanford A（侵犯升主動脈）是外科急症**，每延遲 1 小時死亡率上升約 1–2%。

> [!danger] 🚨 值班紅旗
> 1. **突發「撕裂樣／刀割樣」胸背痛 + 兩臂血壓差 >20 mmHg** → 主動脈剝離直到排除為止
> 2. **絕對禁忌：把它當 STEML 溶栓／全抗凝** → dissection 誤溶栓＝災難性出血。給溶栓／抗凝前先想「這會不會是剝離」
> 3. **下壁 STEMI 樣變化 + 撕裂痛** → 疑 type A 波及右冠 (RCA)，別急著單純當心梗處理
> 4. 血壓控制目標 **SBP 100–120 / HR <60**，**先 β-blocker 降心率再降壓**（先降壓會 reflex tachycardia↑ 剪力）

## 📖 定義 / 病生理（這是什麼）
- **機轉**：主動脈**內膜（intima）撕裂** → 血流在脈搏剪力（dP/dt）驅動下進入**中膜（media）**，形成真腔／假腔 → 剝離向遠或近端延伸，可波及分支血管造成 malperfusion，或向外破裂。
- **危險因子**：
  - **高血壓（最重要、最常見）**、既有主動脈瘤／粥狀硬化
  - **結締組織病**：[[Marfan syndrome(馬凡氏症)]]、[[Ehlers-Danlos syndrome(鬆皮症)]]、Loeys-Dietz
  - **二葉主動脈瓣**、主動脈狹窄／[[Coarctation of Aorta(主動脈狹窄)]]
  - 懷孕（第三孕期）、古柯鹼／安非他命、創傷（減速傷）、發炎性大動脈炎、醫源性（導管）
- **分類（決定治療路線）**：
  | 系統 | 分型 | 範圍 | 處置傾向 |
  | --- | --- | --- | --- |
  | **Stanford A** | 升主動脈受累（不論起源） | → **緊急外科** |
  | **Stanford B** | 僅降主動脈（左鎖骨下動脈遠端） | → 多內科；有併發症才介入 |
  | DeBakey I | 升＋降 | 同 Stanford A |
  | DeBakey II | 僅升 | 同 Stanford A |
  | DeBakey III | 僅降 | 同 Stanford B |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Chest pain(胸痛)]]、[[Syncope(昏厥)]]
- **疼痛**：突發、劇烈、**撕裂／刀割樣**，達到高峰快；升主動脈→前胸痛，降主動脈→**肩胛間／背痛**；痛可**遷移**（隨剝離延伸）。
- **理學徵象**：
  - **兩臂血壓不對稱**（差 >20 mmHg）、脈搏短絀／不對稱
  - 新發**主動脈瓣逆流雜音**（[[Aortic Regurgitation(主動脈瓣逆流)]]，type A 波及瓣膜）
  - **心包填塞**徵象（[[Cardiac tamponade(心包膜填塞)]]：頸靜脈怒張、奇異脈、低血壓）
  - 分支缺血：局部神經缺損／中風（頸動脈）、下肢缺血、腸系膜缺血、少尿（腎動脈）、下肢輕癱（脊髓）
- 可表現為**昏厥、休克、猝死**（破裂）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[D-dimer]]、[[ECG(心電圖)]]　影像 [[CTA(電腦斷層血管攝影)]]、[[Echocardiography(心臟超音波)]]
- **CTA（首選，穩定病人）**：快速、看整段主動脈與分支、辨真假腔、破裂徵象。
- **TEE**：血流動力**不穩定**或無法搬去 CT 時床邊做；對升主動脈敏感。
- **MRA**：穩定／慢性追蹤（急性少用）。
- **CXR**：**縱膈腔增寬（widened mediastinum）**、主動脈輪廓異常（非特異，正常也不能排除）。
- **D-dimer**：高敏感，**陰性可協助排除低風險者**（配合 ADD-RS），但不可單獨作決策。
- **ECG**：主要用來**排除／辨識合併 STEMI**（RCA 受累）——避免誤診純心梗而溶栓。
- **ADD-RS（Aortic Dissection Detection Risk Score）**：高風險狀況／疼痛特徵／理學徵象三大類各記分做初篩分層。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **經典三聯**：撕裂痛 ＋ 兩臂血壓差 ＋ 縱膈腔增寬（三者齊全少見，但出現任一都要警覺）。
- **最致命誤診**：當成單純 STEMI 給溶栓／抗凝。撕裂痛 + 疑心梗 → 先想剝離。
- 疼痛極劇但初期 ECG／心肌酵素不成比例 → 提高剝離懷疑。
- Type A 可併發：AR、心包填塞、冠脈開口受累（多為 RCA→下壁）、中風。
- D-dimer 陰性不能取代影像於高風險病人。

## 💊 治療（Acute / Chronic）
> 用藥 [[Labetalol]]、[[Esmolol]]、[[Nitroprusside(硝普鈉)]]
### 急性內科穩定（所有型別都要先做）
- **目標**：**SBP 100–120 mmHg、HR <60**，降低 dP/dt（脈搏剪力）。
- **順序**：**先靜脈 β-blocker**（esmolol／labetalol）降心率 → 若血壓仍高再加血管擴張劑（nitroprusside／nicardipine）。**切勿先給血管擴張劑**（reflex tachycardia 反增剪力）。
- 充分**鎮痛**（morphine，降交感刺激）。
### 依分型決定性治療
- **Stanford A → 緊急外科**（升主動脈置換 ± 瓣膜／弓部重建）；延遲即高死亡。
- **Stanford B（無併發症）→ 內科**（嚴格血壓控制 + 影像追蹤）。
- **Stanford B（有併發症：malperfusion／持續痛／破裂徵象）→ TEVAR**（胸主動脈血管內支架）。
### 慢性期
- 終生**嚴格血壓控制**（β-blocker 為主）、戒菸、影像監測假腔與瘤化、結締組織病者遺傳諮詢。

## ⚠️ 併發症
- **破裂**（→ 心包填塞／血胸／猝死，最致命）
- 急性**主動脈瓣逆流**、心衰
- **心肌梗塞**（冠脈開口受累，多 RCA）
- **中風／脊髓缺血**（截癱）
- **Malperfusion**：腸系膜／腎／肢體缺血
- 術後：假腔持續灌流、再剝離、瘤化

## 📊 嚴重度評分 / 風險分層（scoring）
> 分型定治療；ADD-RS 定「要不要積極影像」
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **Stanford A/B** | 決定外科 vs 內科 | A→外科急症；B→內科為主 |
| **DeBakey I/II/III** | 描述解剖範圍 | I、II（含升）比照 A；III 比照 B |
| **ADD-RS** | 急診前測概率分層 | 高風險特徵越多、越該直接 CTA（0 分低風險可配 D-dimer 排除） |
