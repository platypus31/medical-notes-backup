# Atherosclerosis（動脈粥狀硬化）

> [!abstract] 一句話：動脈壁**慢性發炎 + 脂質沉積**的漸進過程——內皮受傷→LDL 進入並氧化→巨噬細胞吞噬成**泡沫細胞(foam cell)→脂肪條痕(fatty streak，可逆)→纖維斑塊→複雜斑塊**；斑塊**破裂＋血栓**才是急性事件（[[Acute Coronary Syndrome(急性冠心症)]]、中風、急性肢體缺血）的共同機轉。控制核心＝**LDL↓＋戒菸＋危險因子管理**。

> [!danger] 🚨 值班紅旗
> 1. **急性事件多來自「不穩定斑塊破裂」而非狹窄程度**——輕度狹窄的斑塊照樣可破裂致 MI；別以「造影只有 30%」就掉以輕心
> 2. **同一病人多血管受累**：胸痛病人問中風／間歇性跛行史，反之亦然（CAD＝[[Peripheral Vascular Disease(周邊血管疾病)]]＝carotid 是同一疾病不同床）
> 3. **statin 是二級預防基石**，急性冠心症／中風後應**高強度 statin 儘早啟動**，不要等門診
> 4. 家族性高膽固醇（FH）、極高 LDL、年輕早發 ASCVD → 積極追查與治療

## 📖 定義 / 病生理（這是什麼）
> 既有重點（保留）：
> - LDL 高，有 **foam cell(泡沫細胞)/fatty streak(脂肪條痕)** 仍可逆，atheroma(動脈粥狀硬化斑)
> - **動情激素(雌激素)會抑制 atheroma 形成**
> - **飲酒 / DM / 使用雌激素**：加速動脈粥狀硬化形成
> （註：雌激素在細胞層次抑制斑塊，但**外源性雌激素治療**臨床上反增血栓/心血管事件，兩者不矛盾——前者是機轉、後者是整體臨床效應。⚠️雌激素臨床角色需依情境查證。）

**斑塊形成分期（response-to-injury 假說）**：
1. **內皮損傷／功能失調**：高血壓剪力、吸菸、高血糖、氧化壓力 → 內皮通透性↑、黏附分子表現。
2. **LDL 滲入內膜並被氧化(ox-LDL)** → 促發炎、吸引單核球。
3. **單核球→巨噬細胞吞 ox-LDL → 泡沫細胞(foam cell)** → 聚集成**脂肪條痕(fatty streak，此階段可逆)**。
4. 平滑肌細胞由中膜遷移增生、分泌膠原 → **纖維帽(fibrous cap)** 覆蓋脂質核心 → **纖維斑塊**。
5. **複雜／不穩定斑塊**：大脂質核心 + 薄纖維帽 + 發炎細胞多 → **破裂／糜爛** → 暴露組織因子 → **血栓形成** → 急性血管閉塞事件。

**危險因子（ASCVD）**：
| 不可控 | 可控（主力介入標的） |
| --- | --- |
| 年齡、男性/停經後女性、早發心血管家族史、遺傳(FH) | **吸菸、LDL/血脂異常、[[Hypertension(高血壓)]]、DM、肥胖、久坐、代謝症候群** |
| — | 增修因子：Lp(a)↑、CKD、慢性發炎(RA/SLE)、hs-CRP↑ |

## 🩺 臨床表現（症狀 / 徵象）
> 依受累血管床決定表現（連症狀卡 [[Chest pain(胸痛)]]、[[Claudication(間歇性跛行)]]）
- **冠狀動脈**：[[Chronic Stable Angina(慢性穩定型心絞痛)]]、[[Acute Coronary Syndrome(急性冠心症)]]、[[Acute Myocardial infarction(急性心肌梗塞)]]
- **腦動脈／頸動脈**：TIA、缺血性中風、頸動脈雜音
- **周邊動脈**：[[Peripheral Vascular Disease(周邊血管疾病)]]（間歇性跛行、休息痛、缺血潰瘍）
- **主動脈**：主動脈瘤、[[Atherosclerosis(動脈粥狀硬化)]] 相關 [[Aorta disease(大動脈疾病)]]
- **腎動脈**：腎血管性高血壓、缺血性腎病
- **早期無症狀**——常由 ASCVD 事件或篩檢（ABI/頸動脈超音波/冠脈鈣化分數）發現。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 既有重點（保留）：
> - **ABI(Ankle/brachial index)**：正常值 0.9–1.2，**<0.9** 懷疑周邊血管阻塞，**>1.4（血管鈣化不可壓）** 懷疑血管變硬（原記 >1.2，惟臨床「不可壓縮」門檻常用 >1.4）。

- **血脂檢查**：total cholesterol、LDL-C、HDL-C、TG；必要時 Lp(a)、apoB。
- **風險評估工具**：**ASCVD Pooled Cohort Equations（10 年風險）**——分低/邊際/中/高風險，決定 statin 強度。
- **影像／亞臨床偵測**：
  - **冠脈鈣化分數 CAC（coronary artery calcium score）**：中度風險者再分層（CAC=0 通常可暫緩 statin；CAC 高 → 積極治療）。
  - 頸動脈超音波（IMT/斑塊）、CTA、血管攝影。
- **糖化血色素/血糖、腎功能、血壓**：找共病危險因子。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **狹窄程度 ≠ 破裂風險**：不穩定斑塊（薄帽、大脂核）破裂才致命，輕度狹窄也可致 MI（斑塊生物學 > 幾何狹窄）。
- **fatty streak 可逆**、纖維化後不可逆——早期積極控 LDL 有意義。
- **全身性疾病**：一處有病灶＝其他血管床也要評估。
- **雌激素矛盾**：內源性雌激素機轉上抑制斑塊，但停經後荷爾蒙替代療法（HRT）不建議用於心血管一級/二級預防。
- **statin 多效性(pleiotropy)**：除降 LDL 外有穩定斑塊、抗發炎效果——事件後即用不看當下 LDL。

## 💊 治療（Acute / Chronic）
> 用藥 [[Statin(史他汀)]]、[[Aspirin(阿斯匹靈)]]、[[Ezetimibe]]
### 生活型態（一切基礎）
- **戒菸（效益最大）**、地中海飲食/減飽和脂肪與反式脂肪、規律有氧運動、體重與血糖控制、限酒。
### 藥物（降脂為核心）
- **Statin（第一線）**：
  - **高強度**（atorvastatin 40–80 mg／rosuvastatin 20–40 mg）：臨床 ASCVD、LDL ≥190、糖尿病高風險、高 10 年風險。
  - 中強度：中度風險者。⚠️確切劑量依耐受性/交互作用/肝腎查證。
- **LDL 目標概念**：確診 ASCVD → LDL **降幅 ≥50% 且 <70 mg/dL**；極高風險（多次事件）更嚴格目標 **<55 mg/dL**（ESC）。
- **加乘（未達標時）**：**ezetimibe** → **PCSK9 抑制劑**（evolocumab/alirocumab）→ bempedoic acid、inclisiran。
- **抗血小板**：**低劑量 aspirin** 用於**次級預防**（已有 ASCVD）；一級預防常規使用已趨保守（出血 vs 效益權衡）。
- 控制共病：血壓（[[Hypertension(高血壓)]]）、血糖（SGLT2i/GLP-1RA 具心血管保護）、[[Hyperlipidemia(高血脂)]]。

## ⚠️ 併發症
- **急性冠心症 / 心肌梗塞**、缺血性心肌病變
- **缺血性中風 / TIA**、頸動脈狹窄
- **周邊動脈疾病**（跛行→critical limb ischemia→截肢）
- **主動脈瘤 / 動脈剝離**、腎動脈狹窄致腎性高血壓
- **腸系膜缺血**、動脈栓塞來源（[[Arterial embolism(動脈栓塞)]]）

## 📊 嚴重度評分 / 風險分層（scoring）
> 用「10 年 ASCVD 風險」決定 statin 強度；CAC 再分層
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **ASCVD Pooled Cohort Eq.** | 40–75 歲一級預防風險分層 | <5% 低／5–7.5% 邊際／7.5–20% 中／≥20% 高 → 決定 statin 強度 |
| **CAC score（Agatston）** | 中度風險再分層 | 0＝很低風險可延緩用藥；1–99 輕；100–299 中；≥300 高 |
| **LDL / 風險增修因子** | 治療目標與強度 | LDL≥190、DM、Lp(a)↑、家族史、hs-CRP↑ 上調積極度 |
