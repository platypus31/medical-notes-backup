# Cardiogenic shock（心源性休克）

> [!abstract] 一句話：心臟「幫浦」衰竭 → 心輸出不足以維持組織灌流，即使**充填壓正常或偏高**仍低血壓＋末梢器官低灌流。最常見病因是**大面積急性心肌梗塞（LV >40%）**，院內死亡率仍高達 40–50%。

> [!danger] 🚨 值班紅旗
> 1. **低血壓 + 四肢濕冷 + 尿少/意識改變 → cold & wet，先想 cardiogenic shock**（別當單純脫水猛沖水，肺會更水）
> 2. **AMI 併休克 = 緊急再灌流**（PCI/溶栓不能等），time is muscle；下壁 MI 併低血壓要想 **RV infarct**（此時**要給水、忌 nitrate/morphine**）
> 3. **突發雜音 + 血行動力崩壞** → 機械性併發症（VSR、乳突肌斷裂急性 MR、游離壁破裂）→ echo + 外科會診
> 4. **升壓劑首選 norepinephrine**（vs dopamine 心律不整少、死亡率低）；純降血壓不找病因＝致命

## 📖 定義 / 病生理（這是什麼）
- **血行動力定義（典型）**：SBP <90 mmHg 持續 >30 min（或需升壓劑維持）＋ 末梢低灌流證據（尿少 <30 mL/h、四肢冰冷、意識改變、乳酸↑），且**充填壓足夠/偏高**（PCWP >15 mmHg、CI <2.2 L/min/m²）。
- **核心機轉**：心肌收縮力↓ → CO↓ → 代償性血管收縮＋RAAS/交感活化 → afterload↑、體液滯留 → 心臟做工更重 → 惡性循環（downward spiral）。
- **病因分類**：
  | 類別 | 代表 |
  | --- | --- |
  | **泵衰竭（最常見）** | 大面積 [[Acute Myocardial infarction(急性心肌梗塞)]]（LV >40%）、[[Dilated CardioMyopathy(擴張型心肌病)]]、[[Myocarditis(心肌炎)]] 急性惡化 |
  | **機械性併發症** | 心室中膈破裂(VSR)、乳突肌斷裂→急性 [[Mitral Regurgitation(二尖瓣逆流)]]、游離壁破裂→[[Cardiac tamponade(心包膜填塞)]] |
  | **右心休克** | RV infarct（下壁 MI 延伸）、大面積 [[Pulmonary embolism(肺栓塞)]] |
  | **其他** | 嚴重 [[Aortic stenosis(主動脈瓣狹窄)]]/瓣膜急症、惡性心律不整、心臟術後 |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Hypotension(低血壓)]]、[[Dyspnea(呼吸困難)]]
- **低灌流（forward failure）**：四肢冰冷、皮膚花斑(mottling)、脈壓窄、尿少、意識混亂、乳酸↑
- **鬱血（backward failure）**：肺囉音、端坐呼吸、頸靜脈怒張、S3 gallop
- **Forrester / Stevenson 分型**：cardiogenic shock 多為 **cold & wet**（低灌流＋鬱血）
- 依病因可有新雜音（VSR/MR）、Kussmaul sign（RV infarct）

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[ECG(心電圖)]]、Troponin、Lactate　影像 [[Echocardiography(心臟超音波)]]、CXR
- **ECG**：找 STEMI/缺血、RV infarct（V4R ST↑）、心律不整
- **床邊 echo（關鍵）**：LV function、找機械性併發症（VSR/MR/tamponade）、RV size、瓣膜
- **Lab**：troponin、lactate（灌流/預後指標）、BNP、腎功能、ABG
- **PA catheter（Swan-Ganz，選擇性）**：CI <2.2、PCWP >15 支持診斷，鑑別分佈型/低容積型休克
- **冠脈攝影**：AMI 相關者直接送 cath 兼診斷兼治療

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **SCAI SHOCK 分期 A→E**（At risk / Beginning / Classic / Deteriorating / Extremis）— 越後期死亡率越高，用來分層與升級治療
- **RV infarct 陷阱**：下壁 MI + 低血壓 + 清肺野 + JVD → **給輸液**，**禁 nitrate/morphine/利尿劑**（會讓 preload-dependent 的 RV 崩掉）
- **CULPRIT-SHOCK trial**：AMI 併休克多血管病變 → **只打罪犯血管**優於一次全打（30 天死亡/洗腎較低）
- **IABP-SHOCK II**：IABP 對 AMI 心因休克**無存活率好處** → 不再常規使用
- 別把 cardiogenic 當 septic 猛沖水；也別把 RV infarct 當 LV 衰竭猛利尿

## 💊 治療（Acute / Chronic）
> 用藥 [[Norepinephrine(正腎上腺素)]]、[[Dobutamine]]、[[Antiarrhythmics(抗心律不整藥物)]]
### 急性穩定（同時找並治病因）
- **氧合/通氣**：必要時 NIV 或插管；治療肺水腫
- **升壓劑**：**norepinephrine 為第一線**（優於 dopamine，心律不整與死亡率較低）
- **正性肌力藥**：dobutamine（β1，↑收縮但↑心肌耗氧、可能↓BP）；milrinone（PDE3i，腎功能好者）— 劑量範圍見藥典，⚠️需查證確切 mcg/kg/min
- **謹慎輸液**：cold & wet 者少沖；RV infarct 者反而要補
### 決定性治療（治病因）
- **AMI → 緊急再灌流（primary PCI，優先罪犯血管）**
- **機械性併發症 → 外科修補**（VSR/乳突肌斷裂）
- **機械循環支持 (MCS)**：IABP（證據弱）、Impella、VA-ECMO（難治性休克橋接）
- 難治 → 心室輔助器 (VAD) / 心臟移植評估
### 慢性期
- 依 underlying（HFrEF → GDMT 四支柱）、二級預防、心臟復健

## ⚠️ 併發症
- 多重器官衰竭（腎前性腎衰、肝鬱血/缺血性肝炎）、乳酸中毒
- 惡性心律不整、心臟停止
- MCS 相關：溶血、肢體缺血、出血、中風
- 高住院死亡率（~40–50%）

## 📊 嚴重度評分 / 風險分層（scoring）
> SCAI 分期定「休克多深、要不要升級 MCS」
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **SCAI SHOCK A–E** | 休克嚴重度分期 | A 有風險→E 極危(refractory/CPR)，越後死亡率越高 |
| **Forrester / Stevenson** | 血行動力象限 | cardiogenic 多 cold & wet |
| **CardShock / IABP-SHOCK II score** | 院內死亡預測 | 年齡/乳酸/eGFR/意識等加權 |
