# ArterioVenous Malformation（動靜脈血管畸形 / AVM）

> [!abstract] 一句話：**動脈與靜脈之間缺少正常微血管床、直接相連的一團異常血管（nidus）**。核心生理是**高流量的左→右分流**——腦部會出血 / 癲癇，肺部造成低血氧與矛盾栓塞，大量分流則可拖出**高輸出心衰**。心臟內科看 AVM 的兩個重點就是**高輸出心衰**與**矛盾栓塞**。

> [!danger] 🚨 值班紅旗
> 1. **肺 AVM 病人的中風 / 腦膿瘍 = 矛盾栓塞（paradoxical embolism）**——血栓 / 細菌繞過肺微血管濾網直入體循環。這類病人打點滴 / 中心導管要**嚴防氣泡與血栓**（air filter）。
> 2. **腦 AVM 破裂 = 自發性顱內 / 蛛網膜下腔出血**，年輕人突發劇烈頭痛 + 局部神經缺損 + 癲癇要想到。
> 3. **無法解釋的低血氧、杵狀指、平躺改善坐起惡化（platypnea-orthodeoxia）→ 想肺 AVM 右→左分流**。
> 4. **反覆鼻血 + 皮膚黏膜微血管擴張 + 家族史 → HHT（Osler-Weber-Rendu）**，要主動篩肺 / 腦 AVM。

## 📖 定義 / 病生理（這是什麼）
AVM 是**動脈與靜脈間缺乏正常微血管床、經異常血管團（nidus）直接交通**的先天或後天血管畸形。正常微血管床負責降壓、氣體交換與過濾；缺少它會造成三大後果：

- **高流量左→右分流**：動脈血未經微血管阻力直接灌入靜脈，回心血量增加。
- **盜血（steal phenomenon）**：血液走低阻力的畸形通路，鄰近正常組織灌流不足而缺血。
- **破裂出血**：畸形血管與引流靜脈承受動脈壓，管壁脆弱易破（尤其腦 AVM）。
- **右→左分流（肺 AVM）**：肺 AVM 讓靜脈血繞過肺泡氣體交換與微血管濾網，造成**低血氧**與**矛盾栓塞**（血栓 / 空氣 / 細菌直入體循環 → 中風、腦膿瘍）。
- **高輸出心衰（high-output heart failure）**：分流量夠大時，心臟須持續高心輸出維持體循環灌流，長期導致容量負荷過重與心衰（肝 AVM、大型或多發 AVM 尤其相關）。

**好發部位**：腦（cerebral AVM）、肺（pulmonary AVM）、肝、脊髓、周邊軟組織。

**與 HHT / Osler-Weber-Rendu 的關聯**：遺傳性出血性微血管擴張症（Hereditary Hemorrhagic Telangiectasia, HHT）是體染色體顯性血管病（ENG / ACVRL1 突變），特徵為皮膚黏膜微血管擴張 + 反覆鼻血 + 內臟 AVM。**肺 AVM 常與 HHT 並存**，是其矛盾栓塞（缺血性中風、腦膿瘍）與咳血 / 血胸的來源；診斷 HHT 應主動篩查肺與腦 AVM。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Hemoptysis(咳血)]]、[[Cyanosis(發紺)]]、[[Seizure(癲癇)]]

依部位不同：

- **腦 AVM**：顱內 / 蛛網膜下腔**出血**（突發劇烈頭痛、意識改變、局部神經缺損）、**癲癇**、慢性頭痛、盜血造成的局部缺血症狀。
- **肺 AVM**：**低血氧 / 發紺、運動耐受下降、杵狀指**、platypnea-orthodeoxia、咳血 / 血胸；**矛盾栓塞**造成的缺血性中風 / 短暫性腦缺血 / 腦膿瘍；聽診可有連續性血管雜音。
- **肝 AVM**：多與 HHT 相關，可致**高輸出心衰**、門脈高壓、膽道缺血。
- **高輸出心衰共通表現**：脈壓大、心搏過速、頸靜脈怒張、周邊水腫、活動性呼吸困難；病灶處可觸及震顫（thrill）／聽到雜音（bruit）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Contrast Echocardiography(對比劑心臟超音波)]]　影像 [[CT Angiography(電腦斷層血管攝影)]]、[[MR Angiography(磁振血管攝影)]]、[[Digital Subtraction Angiography(數位減影血管攝影)]]

- **CT / MR angiography（CTA / MRA）**：非侵入性定位 nidus、供養動脈與引流靜脈，評估大小與周邊結構。
- **Digital Subtraction Angiography（DSA，數位減影血管攝影）**：**血管解剖金標準**，可精確描繪供養動脈 / nidus / 引流靜脈，且能同步做血管內栓塞治療。
- **對比劑（bubble / agitated saline）echocardiography**：**篩檢肺 AVM 右→左分流**——注入攪動生理食鹽水微氣泡，若**經 3–5 個心動週期後**於左心出現氣泡，提示肺內分流（與心內分流如 PFO 的即時出現區分）；陽性再以胸部 CTA 確認。
- 血氧 / shunt fraction 評估、對比 CT 定量分流；HHT 病人依 Curaçao 準則診斷後系統性篩 AVM。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **心臟內科最相關的兩件事：高輸出心衰 與 矛盾栓塞**——AVM 不是單純的局部血管病，大分流會拖垮心臟、右→左分流會送栓子進腦。
- **氣泡對比 echo 分流出現時機**：心內分流（PFO/ASD）氣泡**立即**出現在左心；肺 AVM（肺內分流）氣泡**延遲 3–5 個心跳**才出現——是床邊鑑別關鍵。
- **肺 AVM = 矛盾栓塞源**：不明原因年輕人中風 / 腦膿瘍 + 低血氧 → 找肺 AVM + HHT。
- **AVM vs 血管瘤（hemangioma）**：AVM 是高流量的動靜脈直接分流（有供養動脈與引流靜脈）；hemangioma 為血管內皮增生的低流量腫瘤，兩者處置不同。
- 肺 AVM 病人靜脈輸液 / 導管務必用**空氣濾器**，避免氣泡矛盾栓塞。

## 💊 治療（Acute / Chronic）
> 用藥 [[Embolization(血管內栓塞)]]（處置，非藥物）

依部位、大小與破裂風險決定：

- **血管內栓塞（endovascular embolization）**：以線圈 / 塞劑 / 膠封閉 nidus 或供養動脈，是**肺 AVM 首選**（供養動脈直徑常以 ≥2–3 mm 為介入門檻，⚠️ 確切標準依指引 / 團隊），也用於腦 AVM 減積或術前輔助。
- **立體定位放射手術（stereotactic radiosurgery, e.g. Gamma Knife）**：適合位置深、體積較小、手術風險高的腦 AVM，經數月至數年使 nidus 逐漸閉塞。
- **外科切除（microsurgical resection）**：適合可近、破裂或高再出血風險的腦 AVM（依 Spetzler-Martin 分級評估手術風險）。
- **高輸出心衰處置**：先治療根本分流（栓塞 / 切除減少分流量），輔以利尿、容量與心衰藥物支持。
- **HHT 相關**：系統性篩查與處理肺 / 腦 AVM 以預防矛盾栓塞與出血；鼻血與貧血另行處理。
- **腦 AVM 治療策略取捨**（含 ARUBA 試驗爭議）：對「未破裂」腦 AVM，介入 vs 保守觀察需個別化權衡，並非一律介入。

## ⚠️ 併發症
- **腦 AVM**：顱內出血（可致命 / 神經後遺症）、頑固癲癇、盜血缺血、治療相關神經缺損。
- **肺 AVM**：矛盾栓塞（缺血性中風、腦膿瘍）、咳血 / 血胸、進行性低血氧、栓塞後再通 / 復發。
- **心臟 / 全身**：高輸出心衰、肺高壓、容量過負荷。
- **肝 AVM（HHT）**：高輸出心衰、門脈高壓、膽道缺血。

## 📊 嚴重度評分 / 風險分層（scoring）
- **無跨部位統一評分**；分部位使用專屬分級。

| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **Spetzler-Martin grade（腦 AVM）** | 手術風險分層 | 綜合 nidus 大小 + 是否位於功能區 + 引流靜脈型態（1–5 級，級數越高手術風險越大） |
| **Curaçao criteria（HHT）** | HHT 臨床診斷 | 鼻血 / 微血管擴張 / 內臟 AVM / 家族史，符合 ≥3 項確診 |
