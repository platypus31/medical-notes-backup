# Infective Endocarditis（感染性心內膜炎）

> [!abstract] 一句話：心內膜／瓣膜上的**贅生物感染**，血培養 + 心臟超音波是診斷兩大主軸。值班最關鍵：**發燒 + 新雜音 = 抽足夠血培養（3 套、不同部位、抗生素前）再打抗生素**；別因急著退燒糊掉培養。

> [!danger] 🚨 值班紅旗
> 1. **不明原因發燒 + 新發或變化的心雜音 → IE 直到排除為止**；靜脈藥癮者發燒優先想右心 IE（三尖瓣，肺栓塞影）
> 2. **抗生素「之前」抽 3 套血培養**（不同部位、間隔），否則培養陰性 IE 極難處理；血流動力穩定者可等培養再打
> 3. **栓塞警訊**：新發神經缺損／中風（贅生物脫落）、脾／腎梗塞、Janeway/Osler/Roth → 及早影像 + 感染科／心外科會診
> 4. **開刀不能拖的情況**：瓣膜破壞致**急性心衰**、抗生素無法控制的感染（膿瘍、真菌、抗藥菌）、**反覆栓塞 + 大贅生物（>10 mm）** → 早期手術可救命
> 5. 人工瓣膜、心律器導線、洗腎、免疫低下者發燒 → IE 前測概率高，門檻放低

## 📖 定義 / 病生理（這是什麼）
- **機轉**：受損／異常內皮 → 血小板-纖維蛋白血栓（NBTE）→ 菌血症時細菌黏附 → 贅生物（vegetation）。贅生物 = 感染源 + 栓子來源 + 瓣膜破壞。
- **常見菌與情境**：
  | 情境 | 常見致病菌 |
  | --- | --- |
  | 急性、原生瓣、IVDU、健康人 | **Staphylococcus aureus**（最常見、猛爆） |
  | 亞急性、原本瓣膜病、口腔源 | **Viridans streptococci** |
  | 大腸腺癌／腸道 → 找癌 | **Streptococcus gallolyticus（bovis）** |
  | 早期人工瓣（<1 年） | coagulase-negative staph（*S. epidermidis*） |
  | 培養陰性 | HACEK、Coxiella burnetii、Bartonella、Brucella |
  | 腸球菌（泌尿道／老年男） | Enterococci |
- **左心 vs 右心**：左心（二尖瓣／主動脈瓣）多見；**右心（三尖瓣）常見於 IVDU**，表現肺栓塞／膿瘍、雜音可不明顯。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Fever(發燒)]]、[[Heart murmur(心雜音)]]
- **全身**：發燒（最常見）、畏寒、盜汗、體重減輕、倦怠（亞急性像消耗病）。
- **心臟**：新發／變化的**逆流雜音**、心衰徵象（瓣膜破壞）。
- **免疫／血管周邊表現**（見下方 FROM JANE 口訣）：Osler 結節（痛，指腹）、Janeway 病灶（不痛，掌蹠）、Roth 斑（視網膜）、瘀點、指甲下線狀出血、脾腫大。
- **栓塞**：中風、脾／腎梗塞、肺栓塞（右心 IE）、黴菌性動脈瘤。

- **檢查順序**：**血培養 3 套（抗生素前、不同部位）** → **TTE（首選起手）**，敏感度不足或人工瓣／高度懷疑 → **TEE**（敏感度高，看膿瘍、人工瓣裂開）。
- 2023 ESC / 2023 Duke-ISCVID 新增影像主要標準：**PET-CT / 心臟 CT** 對人工瓣、心內裝置 IE 的角色提升。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
**口訣** ( FROM JANE ) ： Fever 、 Roths spots 、 Oslers nodes 、 Murmur 、 Janeway lesions 、 Anemia 、 Nail hemorrhage 、 Emboli 。

- **Osler 痛（Ouch, Osler）；Janeway 不痛**——高頻考點。
- *S. gallolyticus (bovis)* 菌血症／IE → **一定要找大腸腫瘤**（做大腸鏡）。
- **TTE 陰性不能排除 IE**；臨床高度懷疑要做 TEE。
- 培養陰性別急著停：問有無先用過抗生素，加驗 Coxiella / Bartonella 血清、延長培養。
- **抗生素預防**只給**最高風險**（人工瓣、曾患 IE、特定先天心）做**牙科侵入性處置**——單純 MVP／ASD 不需預防。

## 💊 治療（Acute / Chronic）
> 用藥 [[Vancomycin]]、[[Ceftriaxone]]、[[Gentamicin]]、[[Penicillin]]
> ⚠️ 劑量依最新指引 + 培養藥敏調整，用藥前查藥典
### 經驗性（培養未回，血流動力允許者盡量等培養）
- **原生瓣、急性**：涵蓋 MSSA/MRSA + 鏈球菌/腸球菌 → **Vancomycin ± ceftriaxone**（依當地流病）。
- **人工瓣**：**Vancomycin + Gentamicin + Rifampin**（涵蓋 CoNS/MRSA、生物膜）。
### 標的治療（依培養藥敏，療程通常 4–6 週 IV）
- Viridans strep（penicillin 敏感）：penicillin G 或 ceftriaxone ± gentamicin。
- **MSSA**：nafcillin / cefazolin；**MRSA**：vancomycin 或 daptomycin。
- Enterococcus：ampicillin + ceftriaxone（或 + gentamicin）。
### 手術（早期手術可救命，見紅旗第 4 條）
- **適應症**：瓣膜破壞致心衰、抗生素無法控制感染（膿瘍、真菌、抗藥菌、人工瓣）、**反覆栓塞或大贅生物（>10 mm）**。
- 人工瓣 IE、瓣周膿瘍、傳導阻斷（膿瘍侵中膈）→ 積極心外科評估。

## ⚠️ 併發症
- **心衰**（瓣膜破壞逆流）—— 最常見死因、手術首要適應症
- **栓塞**：中風、黴菌性動脈瘤、脾／腎／肺（右心）梗塞
- **瓣周膿瘍 + 傳導阻斷**（PR 延長 → 完全房室阻斷）
- 免疫複合體腎絲球腎炎
- 敗血症、轉移性感染灶

## 📊 嚴重度評分 / 風險分層（scoring）
> Duke 定「是不是 IE」；下列指標定「要不要早開刀」
| 評分 / 指標 | 用途 | 意義 |
| --- | --- | --- |
| **Modified Duke / 2023 ISCVID** | 診斷分類 | 確定：2主 / 1主3次 / 5次；可能：1主1次 / 3次 |
| 贅生物大小 | 栓塞 + 手術 | **>10 mm** 栓塞風險高、反覆栓塞 → 早期手術考量 |
| 併發心衰 / 抗藥菌 / 膿瘍 | 手術指征 | 越多 → 越早手術，死亡率越高 |
