# Peripheral Vascular Disease（周邊血管疾病）

> [!abstract] 一句話：多指下肢**周邊動脈疾病(PAD)**——[[Atherosclerosis(動脈粥狀硬化)]] 造成肢體動脈狹窄／阻塞，典型症狀是**間歇性跛行(claudication)**（走一段固定距離即腿痛、休息緩解）；診斷靠 **ABI <0.9**；病程進展至**休息痛／潰瘍／壞疽＝critical limb ischemia (CLI)** 有截肢風險。**PAD ＝全身動脈硬化的窗口，心血管死亡風險高**。

> [!danger] 🚨 值班紅旗
> 1. **急性肢體缺血(acute limb ischemia) — 6P**：Pain、Pallor、Pulselessness、Paresthesia、Paralysis、Poikilothermia(冰冷)。**時間就是肢體**（黃金 ~6 小時），立即血管外科會診 + 抗凝(heparin)
> 2. **Paralysis／Paresthesia 出現＝晚期缺血、肢體受威脅** → 不可觀察，緊急介入
> 3. **休息痛（rest pain，尤其平躺夜間、垂足緩解）＋ 潰瘍／壞疽 ＝ CLI**：需血管重建評估，勿當單純傷口
> 4. **PAD 病人 = 高心血管風險**：抗血小板 + 高強度 statin 是必備二級預防，別只顧腿

## 📖 定義 / 病生理（這是什麼）
- **定義**：周邊（非冠狀、非顱內）動脈因**動脈粥狀硬化**（最常見）致狹窄/阻塞 → 肢體血流供需失衡。廣義 PVD 亦含靜脈疾病，但臨床「PAD」多指**下肢動脈阻塞**。
- **機轉**：慢性 [[Atherosclerosis(動脈粥狀硬化)]] → 運動時肌肉需氧↑ 但供血受限 → 缺血代謝物堆積 → **勞力性疼痛（跛行）**；病程進展至靜息血流不足 → **rest pain／組織壞死**。
- **好發部位**：股淺動脈(SFA)最常見，其次主髂(aortoiliac)、膕動脈及以下(infrapopliteal，糖尿病/CKD 多見)。
- **其他病因**（非動脈硬化，鑑別用）：血栓栓塞（心源性 → 見 [[Arterial embolism(動脈栓塞)]]）、血管炎（Buerger's/TAO、Takayasu）、纖維肌發育不良、外傷、膕動脈陷夾。
- **危險因子**：**吸菸（最強）**、**糖尿病**、[[Hypertension(高血壓)]]、[[Hyperlipidemia(高血脂)]]、年齡、CKD、高同半胱胺酸。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Claudication(間歇性跛行)]]、[[Chest pain(胸痛)]]（共病 CAD）
- **間歇性跛行(claudication)**：走**固定距離**後小腿（或臀/大腿，依阻塞位置）**痙攣痛**，**休息數分鐘緩解**、再走再痛（可重現）。阻塞位置決定痛的位置：主髂→臀/大腿（Leriche 症候群：跛行＋陽痿＋股動脈搏動消失）；股膕→小腿。
- **理學徵象**：脈搏減弱/消失、下肢**毛髮脫落、皮膚光亮萎縮、指甲增厚、傷口難癒**、抬高蒼白/下垂潮紅(dependent rubor)、皮溫低、微血管回填延遲。
- **critical/chronic limb-threatening ischemia (CLI/CLTI)**：**休息痛**（前足/趾，平躺加劇、垂足緩解）、**不癒潰瘍、壞疽**。
- **急性肢體缺血**：突發 **6P**（見紅旗），常為栓塞或原位血栓。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Ankle-Brachial Index(踝肱指數)]]　影像 [[Duplex Ultrasound(血管超音波)]]、[[CTA(電腦斷層血管攝影)]]
- **ABI（Ankle-Brachial Index，首選篩檢/診斷）**：踝收縮壓 ÷ 肱收縮壓（取高側）。
  | ABI | 判讀 |
  | --- | --- |
  | 0.90–1.40 | 正常 |
  | **≤0.90** | **診斷 PAD**（≤0.5 常見於 CLI/嚴重缺血） |
  | >1.40 | 動脈**鈣化不可壓縮**（糖尿病/CKD）→ ABI 失真，改測 **toe-brachial index(TBI, <0.7 異常)** |
- **運動後 ABI**：靜息 ABI 正常但症狀典型時，運動後下降 ≥20% 支持診斷。
- **影像（規劃血管重建前）**：**動脈 duplex 超音波** → **CTA / MRA** → **DSA（導管血管攝影，介入同時可治療）**。
- **合併評估**：血糖、血脂、腎功能；因高共病，須評估 [[Ischemic Heart Disease(缺血性心臟病)]]、頸動脈。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **糖尿病/CKD 的 ABI 假性正常/偏高**（血管鈣化）——症狀典型但 ABI >1.4 別放心，改用 **TBI**。
- **跛行 vs 神經性跛行（脊椎狹窄）**：血管性＝走固定距離即痛、站著休息即緩解；神經性＝需**坐下/前彎**才緩解、距離不固定、姿勢相關。
- **糖尿病足潰瘍**常合併神經病變（感覺缺失）→ 痛不明顯反而更晚就醫、更嚴重。
- **PAD 是 CAD/中風的等危症**：多數 PAD 病人死於心血管事件而非肢體。
- **急性 vs 慢性缺血**：慢性有側枝循環較耐缺血；急性栓塞無側枝、6P 進展快、更緊急。

## 💊 治療（Acute / Chronic）
> 用藥 [[Cilostazol]]、[[Statin(史他汀)]]、[[Aspirin(阿斯匹靈)]]、[[Clopidogrel]]
### 全體必備：心血管風險降低（二級預防，最影響存活）
- **戒菸（最關鍵）**、運動治療、控糖控壓控脂。
- **高強度 statin**（LDL 目標 <70 mg/dL）。
- **抗血小板**：aspirin 或 clopidogrel（單方）；症狀性/高風險者可考慮 **低劑量 rivaroxaban 2.5 mg BID ＋ aspirin**（COMPASS/VOYAGER 策略，需權衡出血）。⚠️劑量與適應症依個案查證。
### 間歇性跛行（改善症狀/行走距離）
- **監督式運動治療（第一線非藥物，證據最強）**。
- **Cilostazol**（phosphodiesterase-3 抑制劑，100 mg BID）：改善跛行距離；🔴 **任何程度心衰為禁忌**。
- （pentoxifylline 效益有限。）
### CLI / 急性肢體缺血
- **血管重建**：血管內介入（PTA/stent）或外科繞道(bypass)。
- **急性肢體缺血**：立即 **IV heparin 抗凝** + 血管外科 → 導管取栓/溶栓或手術栓塞切除；不可逆壞死則截肢。
- 傷口照護、感染控制、疼痛控制。

## ⚠️ 併發症
- **肢體喪失**（CLI/急性缺血 → 截肢）
- **不癒潰瘍、壞疽、感染／敗血症**
- **急性肢體缺血**（原位血栓或栓塞）
- **再灌流傷害／compartment syndrome**（血管重建後）
- 高**心肌梗塞、中風、心血管死亡**風險（全身動脈硬化）

## 📊 嚴重度評分 / 風險分層（scoring）
> Fontaine / Rutherford 分期定嚴重度與介入時機
| Fontaine | Rutherford | 臨床 |
| --- | --- | --- |
| **I** | 0 | 無症狀（ABI 已異常） |
| **IIa** | 1 | 輕度跛行（走 >200 m 才痛） |
| **IIb** | 2–3 | 中/重度跛行（<200 m 即痛） |
| **III** | 4 | **靜息痛(rest pain)**（CLI 起點） |
| **IV** | 5–6 | **潰瘍 / 壞疽**（組織缺損） |

> 補充分類：**WIfI**（Wound/Ischemia/foot Infection）評估 CLTI 截肢風險；**Leriche 症候群**＝主髂阻塞（跛行＋陽痿＋股脈消失）。
