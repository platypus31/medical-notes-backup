# Bifascicular Block（雙小束阻斷）

> [!abstract] 一句話：**RBBB ＋ 一條左束分支阻斷（LAFB 或 LPFB）**＝三條束支（RBBB／LAFB／LPFB）中壞了兩條，只剩一條在傳導。多數**無症狀不需 pacemaker**，但一旦剩下那條也壞掉就變 **complete heart block**——所以「暈厥 or 高度 AV block」是升格評估節律器的紅旗。

> [!danger] 🚨 值班紅旗
> 1. **Bifascicular block ＋ 暈厥／幾乎昏倒（presyncope）** → 疑間歇性完全房室阻斷（Stokes-Adams），排除其他病因後評估**永久節律器**；別當單純迷走性暈厥打發
> 2. **Alternating BBB（同一份或不同份 ECG 上 RBBB 與 LBBB 交替出現）** → 傳導系統瀕臨崩潰，**Class I 節律器適應症**，直接會診電生理
> 3. **Bifascicular block ＋ 新發高度（Mobitz II／2:1／3度）AV block** → His-Purkinje 系統廣泛病變，需節律器；等待時備 transcutaneous pacing
> 4. **急性前壁 [[Acute Myocardial infarction(急性心肌梗塞)]] 新出現 bifascicular block** → 大面積梗塞、易惡化成完全阻斷，備臨時節律器並密切監測

## 📖 定義 / 病生理（這是什麼）
- **傳導解剖**：房室結下方 His bundle 分成三條「fascicle」：
  1. **右束支 Right bundle（RBBB 相關）**
  2. **左前分支 Left anterior fascicle（LAFB 相關）**
  3. **左後分支 Left posterior fascicle（LPFB 相關）**
- **Bifascicular block 定義**：三條中**兩條**傳導受阻，衝動只能靠剩下一條下傳。組合：
  | 組合 | 頻率 | ECG 電軸 |
  | --- | --- | --- |
  | **RBBB ＋ LAFB** | **最常見** | RBBB 型 ＋ 明顯**電軸左偏（LAD）** |
  | **RBBB ＋ LPFB** | 較少見（LPF 較粗、雙重血供、不易壞） | RBBB 型 ＋ **電軸右偏（RAD）**，需先排除其他 RAD 原因 |
  | LBBB（LAFB ＋ LPFB 同時） | 有時歸類為 bifascicular（左側兩分支皆壞） | LBBB 型 |
- **病生理**：慢性退化（**Lenègre-Lev 傳導系統纖維化**，最常見）、缺血性心臟病、鈣化（主動脈瓣環／二尖瓣環延伸）、心肌病、高血壓性心臟病。剩餘單一 fascicle 若再受損 → **完全性心臟阻斷（complete heart block）** → 心搏過緩／停搏／暈厥。
- **「Trifascicular block」名詞陷阱**：舊稱 bifascicular block ＋ 第一度 AV block（PR 延長），但 PR 延長多來自房室結而非第三條 fascicle，**不等於三條都壞**；真正三分支受累的可靠指標是 **alternating BBB**。此詞現多避免使用。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Syncope(昏厥)]]、[[Arrythmia(心律不整)]]
- **多數無症狀**：常在例行 ECG 偶然發現，慢性穩定者進展到完全阻斷的年風險低（約 1–4%/年，依基礎心臟病而異）。
- **有症狀時**：暈厥、近乎昏倒、頭暈、運動耐受下降、心悸——多因間歇性高度 AV block／心搏過緩。
- **理學**：無特異徵象；聽診可有 wide 或 paradoxical S2 split（因 RBBB）。合併心衰／缺血時見對應徵象。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[ECG(心電圖)]]、[[簡易心電圖判讀（EKG）]]　影像 [[Echocardiography(心臟超音波)]]
- **12 導程 ECG（診斷核心）**：
  - **RBBB 判準**：QRS ≥ 120 ms；V1 呈 **rSR′（M 型／rabbit-ear）**；I、V5–V6 有**寬鈍的 S 波**；V1 T 波倒置。
  - **加 LAFB**：電軸左偏（約 −45° ~ −90°）；I／aVL 呈 **qR**；II／III／aVF 呈 **rS**。→ 合起來 = **RBBB ＋ LAFB**。
  - **加 LPFB**：電軸右偏（+90° ~ +180°）；I／aVL 呈 **rS**；II／III／aVF 呈 **qR**；**須先排除**右心室肥厚、側壁梗塞、垂直心、肺病等其他 RAD 原因。→ = **RBBB ＋ LPFB**。
- **危險徵象**：PR 延長（合併房室結病變）、QRS 進一步增寬、alternating BBB。
- **Holter／event monitor**：暈厥但靜態 ECG 未捕捉到高度阻斷時，記錄陣發性 AV block。
- **Echo**：找結構性病因（缺血、心肌病、瓣膜鈣化）。
- **電生理檢查（EPS）＋ HV interval**：暈厥原因不明時測 His-ventricle 傳導時間；**HV > 70 ms（尤其 > 100 ms）或 pacing 誘發 infra-His block** → 支持植入節律器。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **無症狀 bifascicular block 不需 pacemaker**——是常見考點與臨床誤區；預防性植入未證實有益。
- **RBBB ＋ LPFB 比 RBBB ＋ LAFB 更不祥**：LPF 粗、雙重血供不易壞，一旦壞代表病變廣泛。
- **暈厥 ＋ bifascicular block ≠ 自動植入**：先排除其他暈厥原因（姿勢性、迷走、[[Aortic stenosis(主動脈瓣狹窄)]]、心律不整），再看 EPS／monitor。
- **Alternating BBB 是唯一可靠的「三分支瀕危」ECG 指標**，見到就是強適應症。
- **圍術期（perioperative）**：單純慢性無症狀 bifascicular block **不需**術前預防性臨時節律器；除非合併暈厥史或高度 AV block。全身麻醉／手術本身極少誘發完全阻斷。
- 與新發前壁心梗一起出現時意義完全不同（急性大面積缺血）——不要跟慢性退化性混為一談。

## 💊 治療（Acute / Chronic）
> 用藥 —（本病無專屬藥物，處置核心是節律器與病因治療）
### Acute（急性 / 值班）
- **無症狀且穩定** → 監測、找病因，**不需**急性介入。
- **暈厥／血流動力不穩／高度 AV block** → 接上 monitor、備 **transcutaneous pacing pad**；心搏過緩有症狀可暫用 atropine（對 infra-His block 常無效）、必要時 transvenous 臨時節律器；避免使用會抑制傳導的藥物（β-blocker、CCB、digoxin、抗心律不整藥）。
- 急性心梗合併 → 依 STEMI/NSTEMI 流程再灌流，備臨時節律器。
### Chronic（長期 / 永久節律器適應症，依 2018 ACC/AHA/HRS 心搏過緩指引概念）
- **永久節律器 Class I**：① bifascicular block ＋ 高度或間歇性第三度 AV block；② **alternating BBB**。
- **合理考慮（Class IIa）**：暈厥且未找到其他病因、EPS 顯示 **HV ≥ 70 ms** 或 pacing 誘發 His 以下阻斷。
- **不建議**：完全無症狀、無高度阻斷證據者植入。
- 病因治療：控制缺血、心衰 GDMT、避免加重傳導藥物。

## ⚠️ 併發症
- **完全性心臟阻斷（complete/3rd-degree AV block）** → 心搏停頓、Stokes-Adams 暈厥、猝死（最重要）
- 心搏過緩相關低心輸出、跌倒外傷
- 合併結構性心臟病時的心衰、缺血惡化
- 節律器植入後併發症（導線、感染、氣胸等）

## 📊 嚴重度評分 / 風險分層（scoring）
> 無正式評分；用「症狀 × 阻斷程度 × HV interval」決定是否植入節律器
| 分層依據 | 用途 | 意義 |
| --- | --- | --- |
| **有無暈厥／高度 AV block** | 決定 pacemaker | 無症狀→觀察；有症狀或高度阻斷→植入 |
| **Alternating BBB** | 強適應症標記 | 出現＝Class I 節律器 |
| **HV interval（EPS）** | 暈厥病因不明時 | ≥ 70 ms（尤 >100 ms）／誘發 infra-His block → 傾向植入 |
| 組合類型 | 風險提示 | RBBB＋LPFB 較 RBBB＋LAFB 病變更廣 |
