# Hashimoto’s thyroiditis（橋本氏甲狀腺炎）

> [!abstract] 一句話：**自體免疫**（抗 TPO / 抗 Tg 抗體）造成的**慢性淋巴球性甲狀腺炎**，是**碘充足地區成人甲狀腺低下最常見原因**。典型是「無痛瀰漫性甲狀腺腫 + 逐漸低下」，值班最需記得的是它可誘發**黏液水腫昏迷**且會併發甲狀腺淋巴瘤。

## 📖 定義 / 病生理（這是什麼）
- **機轉**：T 細胞主導的自體免疫破壞甲狀腺濾泡 → 淋巴球浸潤、生髮中心形成、濾泡萎縮纖維化 → 甲狀腺荷爾蒙產生下降。抗體以 **anti-TPO（>90% 陽性）**、anti-thyroglobulin 為代表（多為破壞的標記而非直接致病）。
- **自然病程**：常先經一段暫時性甲狀腺毒（Hashitoxicosis，濾泡破壞釋出既存荷爾蒙）→ 多數最終走向**永久性甲狀腺低下**。
- **危險因子**：女性（女:男 ≈ 7–10:1）、家族史、其他自體免疫病、Turner／Down 症、產後、碘攝取過量、鋰鹽、干擾素、免疫檢查點抑制劑。
- **關聯**：常與其他自體免疫病共病（第 1 型糖尿病、[[Addison’s disease(愛迪生氏病)]]、白斑、乳糜瀉、惡性貧血）→ **自體免疫多腺體症候群（APS type 2）**。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Fatigue(疲倦)]]、[[Goiter(甲狀腺腫)]]
- **甲狀腺**：**無痛、瀰漫、質地偏硬（rubbery）**的甲狀腺腫；病程久者反而萎縮。
- **甲狀腺低下症狀**：怕冷、疲倦、體重增加、便秘、皮膚乾、掉髮、月經不規則、反射鬆弛期延長、[[Hypothyroidism(甲狀腺機能低下)]] 表現。
- **早期 Hashitoxicosis**：短暫心悸、怕熱、體重下降（與 Graves 鑑別，見下）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Thyroid function(甲狀腺功能)]]、[[Anti-TPO antibody(抗甲狀腺過氧化酶抗體)]]　影像 [[Thyroid ultrasound(甲狀腺超音波)]]
- **抗體**：**anti-TPO 陽性**是核心（診斷 + 預測進展為低下）；anti-Tg 可輔助。
- **甲狀腺功能**：依病程可為正常 / 亞臨床低下（TSH↑、free T4 正常）/ 顯性低下（TSH↑、free T4↓）；早期偶見毒性期。
- **超音波**：瀰漫低回音、不均質、假結節（pseudonodules）；有助評估結節與腫大。
- **不需常規做**：放射性碘攝取（與 Graves 鑑別才需）；細針抽吸（FNA）僅在可疑結節或快速增大疑淋巴瘤時做。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **碘充足地區成人原發性甲狀腺低下最常見原因**（國考高頻）。
- **與 Graves 鑑別**：Hashitoxicosis 攝碘率**低**、無眼病變 / 脛前黏液水腫、TRAb 陰性；Graves 攝碘率**高**、TRAb 陽性。
- **甲狀腺淋巴瘤風險**：橋本是原發性甲狀腺淋巴瘤最強危險因子（相對風險 60–80 倍）→ 甲狀腺**快速無痛增大**要警覺，做 FNA / 切片。
- **產後甲狀腺炎**與橋本同屬自體免疫譜系，anti-TPO 陽性者產後易發。
- 亞臨床低下者 anti-TPO 陽性→進展為顯性低下風險高，追蹤要更密。

## 💊 治療（Acute / Chronic）
> 用藥 [[Levothyroxine(左旋甲狀腺素)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
- **顯性甲狀腺低下**：**Levothyroxine（L-T4）** 全量替代約 **1.6 μg/kg/day**；老人 / 冠心病者由低劑量（12.5–25 μg/day）起始緩慢加量，避免誘發心絞痛 / 心律不整。
- **亞臨床低下**：TSH >10 mIU/L、有症狀、抗體陽性、孕婦 / 備孕者建議治療；輕微者可追蹤。
- **監測**：起始 / 調量後 **6–8 週**驗 TSH 調整；目標 TSH 落在正常範圍（孕婦依孕期特定範圍）。
- **服藥衛教**：空腹（早餐前 30–60 分鐘）、與鈣 / 鐵 / 制酸劑 / PPI 間隔服用（影響吸收）。
- 甲狀腺腫本身不需藥物縮小；壓迫症狀或疑惡性才手術。

## ⚠️ 併發症
- **黏液水腫昏迷（[[Myxedema Coma(黏液水腫性昏迷)]]）**：長期未治療 + 感染 / 寒冷 / 鎮靜劑誘發，**急症、高死亡率**。
- **原發性甲狀腺淋巴瘤（MALT / DLBCL）**。
- 心包 / 胸膜積液、高膽固醇血症、不孕 / 流產、憂鬱。
- 合併其他自體免疫腺體病（APS）。

## 📊 嚴重度評分 / 風險分層（scoring）
> 無專屬評分；以甲狀腺功能分層決定治療強度
| 分層 | 定義 | 處置傾向 |
| --- | --- | --- |
| Euthyroid（抗體+，功能正常） | TSH / free T4 正常 | 追蹤，暫不治療 |
| Subclinical hypothyroid | TSH↑、free T4 正常 | TSH>10 或有症狀 / 孕婦 → 治療 |
| Overt hypothyroid | TSH↑、free T4↓ | L-T4 全量替代 |
