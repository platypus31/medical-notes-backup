# Hypoglycemia（低血糖）

> [!abstract] 一句話：血糖過低導致交感興奮 + 神經缺糖症狀的臨床急症。**值班最關鍵一點：測到 <70 mg/dL 就立刻給糖，別等確認病因**——腦子缺糖每一分鐘都在死細胞，**「先治療、後追因」**。糖尿病人的低血糖比高血糖更快致命。

> [!danger] 🚨 值班紅旗
> 1. **意識改變 / 抽搐 / 局部神經學缺損的病人，第一件事是驗血糖（fingerstick）** — 低血糖可完美模仿中風、癲癇、精神病，漏測就漏診。
> 2. **清醒能吞 → 口服 15–20 g 快速糖**（15 g = 3–4 顆方糖 / 半杯果汁 / 1 湯匙蜂蜜）；**意識不清 / 不能安全吞 → D50W 20–50 mL IV push**（兒童 D25W 2–4 mL/kg，新生兒 D10W）。
> 3. **無 IV 管路 → Glucagon 1 mg IM/SC**（家中/院前首選）；但**肝醣耗竭者（酗酒、飢餓、肝衰）glucagon 無效**，必須想辦法上 IV 給糖。
> 4. **懷疑酗酒 / 營養不良 → 給糖同時或之前補 Thiamine 100 mg IV**，避免誘發 Wernicke 腦病變。
> 5. **Sulfonylurea（如 glimepiride、glibenclamide）或長效胰島素造成的低血糖會「反覆復發」** — 一次 D50 拉起來不代表沒事，**必須留觀 + 持續 D10W infusion**，SU 中毒可考慮 Octreotide 50–100 µg SC q6–8h 抑制內源胰島素。
> 6. **Whipple 三徵**：低血糖症狀 + 當下血糖低 + 給糖後症狀緩解 — 三者齊全才算真低血糖，避免對「數字低但無症狀」過度治療（但糖尿病人放寬）。

## 📖 定義 / 病生理（這是什麼）
- **定義（ADA 分級，2023 Standards of Care）**：
  | Level | 血糖 | 意義 |
  | --- | --- | --- |
  | **Level 1（警戒）** | 54–70 mg/dL（3.0–3.9 mmol/L） | 需處置的警戒值 |
  | **Level 2（臨床顯著）** | <54 mg/dL（<3.0 mmol/L） | 明確且臨床顯著低血糖 |
  | **Level 3（嚴重）** | 無特定閾值，**需他人協助** | 意識/身體功能受損的嚴重事件 |
- **機轉**：血糖↓ → ① 胰島素分泌↓ → ② glucagon↑（第一道防線）→ ③ epinephrine↑（第二道，產生交感症狀）→ 皮質醇/GH（慢）。糖尿病久病者這套**反調節反應鈍化 + hypoglycemia unawareness（低血糖不自覺）**，是反覆嚴重低血糖的關鍵。
- **危險因子 / 病因分類**：
  | 類別 | 病因 |
  | --- | --- |
  | **藥物（最常見）** | Insulin、Sulfonylurea、Meglitinide；藥量錯、進食少、腎功能惡化致藥物蓄積 |
  | **酒精** | 抑制糖質新生（gluconeogenesis），空腹飲酒尤危 |
  | **重症/器官衰竭** | 敗血症、肝衰竭、腎衰竭、心衰 |
  | **荷爾蒙缺乏** | 腎上腺功能不全（[[Adrenal insufficiency(腎上腺功能不全)]]）、垂體功能低下 |
  | **胰島素瘤（Insulinoma）** | 內源性高胰島素；空腹低血糖、發作性 |
  | **非胰島細胞腫瘤** | 大型間葉腫瘤分泌 IGF-2 |
  | **餐後（reactive）** | 胃切除後傾食症候群、早期糖尿病 |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Altered mental status(意識改變)]]、[[Seizure(癲癇發作)]]、[[Palpitation(心悸)]]
- **交感/腎上腺素症狀（血糖 55–70，早期警訊）**：冒冷汗、心悸、發抖、焦慮、飢餓感、臉色蒼白。
- **神經缺糖症狀（neuroglycopenia，血糖 <50，較危險）**：意識混亂、行為異常、視力模糊、講話含糊、局部神經缺損（偽中風）、抽搐、昏迷。
- **夜間低血糖**：惡夢、盜汗醒來、晨起頭痛。
- ⚠️ 服 β-blocker 者交感症狀被遮蔽，可能直接以神經症狀表現（無預警昏迷）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Fingerstick glucose(指尖血糖)]]、[[C-peptide(C胜肽)]]、[[Insulin level(胰島素濃度)]]　影像 [[Abdominal CT(腹部電腦斷層)]]
- **床邊指尖血糖**：第一線，但危急時勿因等實驗室血糖延誤治療。
- **診斷不明原因低血糖（非糖尿病人）→ 發作當下抽「critical sample」**：血糖、insulin、C-peptide、proinsulin、β-hydroxybutyrate、sulfonylurea/meglitinide screen、皮質醇。
  | 結果模式 | 判讀 |
  | --- | --- |
  | Insulin↑ + C-peptide↑ | **內源性高胰島素** → insulinoma / SU 藥物（需驗藥物 screen 區分） |
  | Insulin↑ + C-peptide↓ | **外源性胰島素**（注射） |
  | Insulin↓ + 酮體↑ | 飢餓、酒精、荷爾蒙缺乏 |
  | Insulin↓ + 酮體↓ | IGF-2 腫瘤、脂肪酸氧化缺陷 |
- **72 小時飢餓試驗**：懷疑 insulinoma 的黃金診斷；誘發後測上述 critical sample。
- **影像**：確認內源高胰島素後，腹部 CT/MRI ± EUS 定位胰島素瘤。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **任何意識改變先驗血糖** — 最便宜、最快、最不該漏的檢查。
- **Sulfonylurea 低血糖會反覆數十小時**（尤其腎功能差、長效 SU），一次 D50 不算結案，需留觀 + D10W 持續輸注。
- **Glucagon 對肝醣耗竭者（酗酒、久餓、肝衰、嬰兒）無效** — 別把它當萬用解藥。
- **C-peptide 是區分內源 vs 外源胰島素的關鍵**（外源胰島素不含 C-peptide）— 這是「詐病/謀殺低血糖」的法醫重點。
- **糖尿病人 hypoglycemia unawareness** → 建議放寬血糖目標數週讓警覺閾值回復。
- 給糖後意識若不回復，重新想：是否 SU 持續作用、是否合併腦傷/中風、是否血糖其實已上來但腦損傷已成。

## 💊 治療（Acute / Chronic）
> 用藥 [[Dextrose(葡萄糖)]]、[[Glucagon]]、[[Octreotide]]、[[Thiamine(維生素B1)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 急性（Rule of 15）
- **清醒能吞**：口服 **15–20 g 快速碳水**（葡萄糖錠/果汁/含糖飲）→ **15 分鐘後複測**，仍 <70 再給一次 → 血糖回升後補一份含蛋白的正餐/點心防復發。
- **意識不清 / 不能吞**：
  - **D50W 20–50 mL IV push**（= 10–25 g 葡萄糖）；兒童 **D25W 2–4 mL/kg**；新生兒 **D10W 2 mL/kg**。
  - 無 IV → **Glucagon 1 mg IM/SC**（成人）；恢復意識後盡快補口服糖。
- **後續維持**：嚴重/藥物性低血糖給 **D10W 持續 IV infusion**，滴定維持血糖 >100 mg/dL。
- **酗酒/營養不良**：**Thiamine 100 mg IV** 先於或併同給糖。
### SU 中毒特異治療
- **Octreotide 50–100 µg SC q6–8h**（抑制胰島素分泌，減少 D50 反覆需求）。
### 慢性 / 病因治療
- 糖尿病人：檢討藥物、劑量、進食與腎功能；CGM + 衛教；放寬目標矯正 unawareness。
- Insulinoma：手術切除（治癒性）；不可切除者 diazoxide / octreotide。

## ⚠️ 併發症
- **中樞神經損傷**：長時間嚴重低血糖 → 不可逆昏迷、認知退化、死亡。
- **抽搐、跌倒外傷**。
- **心律不整、心肌缺血**（交感激增 + QT 延長，「dead-in-bed」夜間猝死）。
- 反覆低血糖 → hypoglycemia unawareness 惡性循環。

## 📊 嚴重度評分 / 風險分層（scoring）
> 用分級與能否自理判斷嚴重度
| 分級 | 判準 | 意義 / 處置 |
| --- | --- | --- |
| **Level 1** | 54–70 mg/dL | 警戒，口服糖自理 |
| **Level 2** | <54 mg/dL | 臨床顯著，積極矯正 |
