# Hyperglycemia（高血糖）

> [!abstract] 一句話：血糖高於正常範圍的**狀態**（非單一疾病），值班最重要是**先分辨「單純高血糖」還是已進展到急症（[[Diabetic Ketoacidosis(糖尿病酮酸血症)]] / [[Hyperosmolar hyperglycemic syndrome(高滲透壓高血糖症)]]）**，並找出**誘因（感染、漏打胰島素、類固醇、壓力）**。

> [!danger] 🚨 值班紅旗
> 1. **高血糖 + 意識改變 / 深快呼吸（Kussmaul）/ 脫水** → 立刻查 **ketone + venous blood gas + 滲透壓**，別當「只是血糖高」放著 → 可能是 DKA / HHS。
> 2. **糖尿病人喊「不舒服」但血糖只是輕中度高** → 高血糖常是**感染 / AMI / 敗血症**的表現，找誘因（發燒、尿路、肺、足部傷口、心電圖）。
> 3. **住院病人壓力性高血糖 / 類固醇引起** → 不要用「一針到位」的滑動式胰島素當唯一策略；重症血糖目標 **140–180 mg/dL**，過度嚴格（<110）反增低血糖與死亡（NICE-SUGAR）。
> 4. **打胰島素校正高血糖時警覺低血鉀**：胰島素把鉀打進細胞，K⁺ <3.3 先補鉀再給胰島素（同 DKA 原則）。

## 📖 定義 / 病生理（這是什麼）
- **定義（診斷糖尿病閾值供參）**：空腹血糖 ≥126 mg/dL、隨機 ≥200 mg/dL 併症狀、OGTT 2 小時 ≥200 mg/dL、HbA1c ≥6.5%。
- **機轉**：胰島素**絕對不足**（第 1 型）或**相對不足 + 阻抗**（第 2 型）→ 肝糖新生↑、周邊攝取↓ → 血糖上升；超過腎閾（~180 mg/dL）→ 糖尿 → 滲透性利尿 → 脫水。
- **常見誘因（住院 / 急性升高）**：感染 / 敗血症、漏打或減量胰島素、**類固醇**、TPN / 葡萄糖輸液、急性壓力（AMI、中風、術後）、胰臟疾病、升糖藥物（thiazide、非典型抗精神病藥）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Polyuria(多尿)]]、[[Polydipsia(多渴)]]
- **三多一少**：多尿、多渴、多食、體重下降。
- 視力模糊、疲倦、傷口難癒、反覆感染（黴菌、泌尿道）。
- 嚴重 / 急症化：脫水、姿勢性低血壓、意識改變、Kussmaul 呼吸（DKA）、酮味口氣。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Blood glucose(血糖)]]、[[Ketone(酮體)]]、[[HbA1c(糖化血色素)]]、[[Venous blood gas(靜脈血氣)]]
- **床邊 / 靜脈血糖**確認數值。
- **分辨急症**：血 / 尿 **ketone**、**血氣（pH、HCO₃⁻、anion gap）**、**血漿滲透壓**、電解質（尤其 K⁺、Na⁺ 校正）、腎功能。
- **找誘因**：CBC/DC、尿液分析、胸片、血 / 尿培養、心電圖、乳酸（疑敗血症）。
- **慢性評估**：HbA1c（分辨急性壓力性 vs 長期控制不良）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **「壓力性高血糖」是預後不良指標**：非糖尿病住院病人出現高血糖，死亡率高於已知糖尿病者 → 別忽略。
- **Na⁺ 校正**：高血糖會稀釋鈉，血糖每↑100 mg/dL（>100 部分），校正 Na⁺ 加 ~1.6–2.4 mEq/L。
- HbA1c 在溶血、輸血、缺鐵性貧血、慢性腎病等情況會失真。
- 類固醇引起的高血糖**以午後 / 餐後為主**（早上給 prednisolone 者），監測時機要對。

## 💊 治療（Acute / Chronic）
> 用藥 [[Insulin(胰島素)]]、[[Metformin(二甲雙胍)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 急性 / 住院（非急症的高血糖）
- **補液**矯正脫水（同時降糖）；找並治療**誘因**。
- **胰島素為住院高血糖主力**：basal-bolus（基礎 + 餐前 + 校正）優於單純滑動式（sliding scale alone）。重症用 IV 胰島素輸注滴定。
- **血糖目標**：一般住院 **140–180 mg/dL**；重症同（避免 <110 的過嚴控制）。
- 若已達 DKA / HHS → 走各自 protocol（IV 胰島素 + 大量補液 + 補鉀），見 [[Diabetic Ketoacidosis(糖尿病酮酸血症)]]、[[Hyperosmolar hyperglycemic syndrome(高滲透壓高血糖症)]]。
### 慢性（門診糖尿病管理）
- 生活型態 + **Metformin** 為第 2 型第一線；依心 / 腎共病加 **SGLT2i / GLP-1 RA**；第 1 型須胰島素。
- 個別化 HbA1c 目標（多數 <7%，年長 / 共病放寬）。

## ⚠️ 併發症
- **急性**：DKA、HHS、[[Hypoglycemia(低血糖)]]（治療過頭）、電解質異常。
- **慢性（微血管）**：[[Diabetes Mellitus Nephropathy(糖尿病腎臟病變)]]、視網膜病變、神經病變。
- **慢性（大血管）**：冠心病、中風、周邊動脈疾病、感染 / 傷口不癒。

## 📊 嚴重度評分 / 風險分層（scoring）
> 無單一 score；以「是否進展為急症」分層
| 分層 | 特徵 | 處置 |
| --- | --- | --- |
| 單純高血糖 | 血糖高、ketone−、gap 正常、意識清 | 補液 + 校正胰島素 + 找誘因 |
| DKA | 高血糖 + 酮體 + anion gap 酸中毒 | DKA protocol |
| HHS | 血糖極高(>600) + 高滲透壓 + 意識改變、酮少 | HHS protocol |
