# Mass Effect（腫塊效應）

> [!abstract] 一句話：內分泌科的「mass effect」多指**蝶鞍/下視丘區腫瘤（最常見 [[Pituitary tumor(腦下垂體腫瘤)]]）壓迫周邊構造**——最經典是壓到**視交叉（optic chiasm）造成雙顳側偏盲**。值班最不能漏的是**pituitary apoplexy（腦下垂體卒中）**：突發劇烈頭痛 + 視力驟降 + 眼肌麻痺，可併發**急性腎上腺功能不全**而致命。

> [!danger] 🚨 值班紅旗（Pituitary apoplexy 腦下垂體卒中）
> 1. **突發雷擊樣頭痛 + 急速視力惡化 / 雙顳側偏盲 + 眼外肌麻痺（複視）** → 疑腦下垂體卒中，緊急 MRI（或 CT）。
> 2. **致命關鍵：合併急性續發性腎上腺功能不全** → **先給壓力劑量 hydrocortisone（100 mg IV bolus，之後 50 mg q6h 或 200 mg/day 連續輸注）**，不要等抽血結果。
> 3. 抽血前**先留 baseline cortisol / ACTH / 全套腦垂體荷爾蒙 + 電解質（找低血鈉）**，但**給類固醇不能因等結果而延誤**。
> 4. 視力進行性惡化 / 意識改變 → 神經外科**緊急減壓評估**（經蝶竇手術）。
> 5. 常見誘因：抗凝、產後（Sheehan）、大手術、DA agonist 起始/停用、頭部外傷。

## 📖 定義 / 病生理（這是什麼）
- **Mass effect** = 局部佔位病灶對相鄰神經血管構造的**壓迫/移位效應**，症狀取決於**位置**而非荷爾蒙分泌。
- 內分泌情境最常見來源：**腦下垂體巨腺瘤（macroadenoma >1 cm）**、顱咽管瘤、Rathke's cleft cyst、下視丘腫瘤、轉移。
- 解剖對應：
  | 受壓構造 | 臨床後果 |
  | --- | --- |
  | **視交叉（上方）** | **雙顳側偏盲**（bitemporal hemianopia）、紅色辨識↓ |
  | **正常腦垂體實質** | 全腦垂體功能低下（hypopituitarism） |
  | **海綿竇（外側）** | CN III/IV/VI（眼肌麻痺、複視、眼瞼下垂）、V1/V2（臉麻） |
  | **下視丘（上/後）** | 體溫、食慾/肥胖、睡眠、自主神經、行為/人格改變 |
  | **垂體柄** | 泌乳素↑（stalk effect，去抑制）、[[Central Diabetes Insipidus(中樞性尿崩症)]] |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Bitemporal hemianopia(雙顳側偏盲)]]、[[Headache(頭痛)]]
- loss of red perception
- bitemporal hemianopia
- 體溫失調
- 食慾改變
- 肥胖
- [[Central Diabetes Insipidus(中樞性尿崩症)]]
- 睡眠,行為,自主神經失調
- Personality disorder(人格異常)
- Anosmia(嗅覺喪失症)
- Opthalmoplegoa(眼肌麻痺)
- Ptosis(眼瞼下垂)
- Facial numbness(臉麻)
- CSF rhinorrhea

- **補充**：頭痛（硬腦膜牽張）、hypopituitarism 相關（疲倦、性慾↓、閉經、怕冷）；**DI 較常見於下視丘/漏斗病灶或術後，單純腦垂體腺瘤反而少見 DI**（是易考陷阱）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Prolactin(泌乳素)]]、[[Pituitary hormone panel(腦垂體荷爾蒙套組)]]、[[Cortisol(皮質醇)]]　影像 [[Pituitary MRI(腦下垂體核磁共振)]]、[[Visual field test(視野檢查)]]
- **首選影像：蝶鞍薄切 MRI（含/不含顯影）**——評估大小、視交叉受壓、海綿竇侵犯（Knosp grade）。
- **正式視野檢查（Humphrey / Goldmann）**：客觀量化雙顳側偏盲、追蹤治療反應。
- **全套腦垂體軸線評估**：prolactin、IGF-1/GH、ACTH-cortisol、TSH-free T4、LH/FSH、testosterone/estradiol；找**功能性 vs 無功能** → 見 [[Nonfunctioning adenoma(無功能腫瘤)]]。
- **Stalk effect 判讀**：巨腺瘤壓垂體柄使 prolactin **輕中度↑（通常 <100–150 ng/mL）**；若 prolactin **>200** 多為真正 prolactinoma。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **雙顳側偏盲 = 視交叉受壓的招牌**，先想蝶鞍腫瘤。
- **Stalk effect 陷阱**：大腫瘤 + 中度高泌乳素 ≠ prolactinoma。誤把無功能巨腺瘤當 prolactinoma 給 DA agonist 會延誤手術；用 prolactin 高低與腫瘤大小「不成比例」來分辨。
- **Hook effect（假性低泌乳素）**：極高 prolactin 時免疫分析飽和反而報偏低，臨床高度懷疑時要求**稀釋（1:100）重測**。
- 巨腺瘤即使無症狀，若**逼近視交叉**通常仍建議手術。

## 💊 治療（Acute / Chronic）
> 用藥 [[Cabergoline]]、[[Hydrocortisone(氫化可體松)]]、[[Levothyroxine(左旋甲狀腺素)]]
### 急性（apoplexy）
- **壓力劑量 hydrocortisone**（見紅旗）、矯正低血鈉/血容、緊急影像、神外評估經蝶竇減壓。
### 慢性 / 依病因
- **Prolactinoma** → DA agonist（cabergoline 首選）多可縮小腫瘤、免手術。
- **無功能巨腺瘤壓視交叉** → **經蝶竇手術（TSS）** 為主，殘留/復發者放療。
- **Hypopituitarism** → 荷爾蒙補充（**先補 glucocorticoid 再補 thyroid hormone**，避免誘發腎上腺危象）。
- **DI** → desmopressin（DDAVP）。
> ⚠️ 劑量依最新指引，用藥前查藥典。

## ⚠️ 併發症
- 永久性視野缺損 / 失明（延遲減壓）
- 全腦垂體功能低下、續發性腎上腺不全（可致命）
- 尿崩症、SIADH（術後雙相/三相反應）
- CSF 鼻漏、腦膜炎（術後）
- 腦下垂體卒中復發

## 📊 嚴重度評分 / 風險分層（scoring）
| 評分 | 用途 | 意義 |
| --- | --- | --- |
| **Knosp grade (0–4)** | 海綿竇侵犯程度 | grade 越高，全切率越低、殘留風險越高 |
| **Hardy classification** | 蝶鞍/鞍上侵犯 | 描述腫瘤延伸範圍 |
| **PAS（Pituitary Apoplexy Score）** | apoplexy 保守 vs 手術 | 依意識、視力、視野、眼動決定介入 |
