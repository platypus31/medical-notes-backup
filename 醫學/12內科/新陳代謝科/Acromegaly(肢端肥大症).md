# Acromegaly（肢端肥大症）

> [!abstract] 一句話：成人（生長板已閉合）長期 **GH／IGF-1 過量**（99% 來自腦下垂體 GH 分泌腺瘤）→ 手腳粗大、面貌變粗、內臟肥大。病程隱匿常延遲診斷 7–10 年，**死因主要是心血管**，值班端最該記得的是它會共病 **OSA、糖尿病、高血壓、心肌病變**。

## 📖 定義 / 病生理（這是什麼）
- **機轉**：GH 過量 → 肝臟分泌 **IGF-1** 上升 → 全身軟組織／骨骼／內臟增生。若發生在生長板閉合**前**（兒童）則為**巨人症（gigantism）**。
- **病因**：
  | 來源 | 佔比 | 備註 |
  | --- | --- | --- |
  | 腦下垂體 GH 分泌腺瘤 | ~95% | 多為 **macroadenoma（>1cm）**，可壓迫視交叉 |
  | 異位 GHRH 分泌（肺／胰神經內分泌瘤） | <1% | 罕見 |
  | 異位 GH 分泌 | 極罕 | |
- **共病基因症候群**：MEN1、McCune-Albright、Carney complex、家族性孤立性腦垂體腺瘤（FIPA / AIP 突變）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Headache(頭痛)]]、[[Visual field defect(視野缺損)]]
- **肢端／軟組織**：手腳變大（戒指／鞋變緊）、面容粗獷（額突、下顎前突 prognathism、齒縫變寬、舌大 macroglossia）、皮膚增厚多汗、皮膚息肉（skin tag）。
- **壓迫效應（macroadenoma）**：頭痛、**雙顳側偏盲（bitemporal hemianopia）**、垂體其他軸線功能低下。
- **代謝／全身**：糖尿病／糖耐受不良、高血壓、[[Obstructive sleep apnea(阻塞型睡眠呼吸中止)]]、腕隧道症候群、關節炎、心肌肥厚。
- **易漏診線索**：新發 OSA + 糖尿病 + 高血壓 + 腕隧道，同時出現時要想到。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[IGF-1]]、[[Growth hormone(生長激素)]]、[[OGTT(口服葡萄糖耐受試驗)]]　影像 [[Pituitary MRI(腦下垂體核磁共振)]]
- **第一線篩檢**：**血清 IGF-1**（年齡／性別校正）升高 → 高度懷疑（GH 為脈衝式分泌，單次隨機 GH 無診斷價值）。
- **確診**：**OGTT 抑制試驗** — 口服 75g 葡萄糖後 GH **無法被抑制到 <1 ng/mL**（新式 assay <0.4 ng/mL）即確診。
- **定位**：確診後做**腦下垂體 MRI**（含／不含顯影）找腺瘤大小與壓迫。
- **其他**：視野檢查、評估其他垂體軸線（PRL、TSH/free T4、ACTH/cortisol、性腺軸）。

## ⭐ 特徵 / 注意（pearls & pitfalls）
- **單次隨機 GH 不能診斷也不能排除** — 一律先驗 IGF-1，再用 OGTT 確診。
- **懷孕、青春期、肝病、營養不良** 會影響 IGF-1 解讀。
- Macroglossia + 齒縫變寬 + 手套變緊，回顧舊照片對照面貌變化是實用線索。
- 診斷時常已合併心血管重塑，處置要同時管理共病（血糖、血壓、OSA）。

## 💊 治療（Acute / Chronic）
> 用藥 [[Octreotide]]、[[Lanreotide]]、[[Pegvisomant]]、[[Cabergoline]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 首選：手術
- **經蝶竇腺瘤切除（transsphenoidal surgery）** 為首選根治手段，尤其 microadenoma 或可切除的 macroadenoma。
### 藥物（術後未緩解、無法手術）
- **體抑素類似物 SSA（第一線藥物）**：octreotide LAR、lanreotide → 抑制 GH 分泌並縮小腫瘤。
- **GH 受體拮抗劑**：**pegvisomant** → 阻斷周邊 GH 作用、正常化 IGF-1（不縮腫瘤）。
- **多巴胺促效劑**：cabergoline（對輕度升高或合併 PRL 分泌者）。
- **新一代 SSA**：pasireotide（注意惡化高血糖）。
### 放射治療
- 藥物／手術皆未達標時輔助，效果延遲數年。
### 治療目標
- IGF-1 正常化（年齡校正範圍內）、隨機 GH <1.0 ng/mL。

## ⚠️ 併發症
- **心血管**：高血壓、心肌病變／心衰、心律不整（主要死因）
- **代謝**：糖尿病、血脂異常
- **呼吸**：OSA
- **腫瘤壓迫**：視野缺損、垂體功能低下、頭痛
- **骨關節**：退化性關節炎、腕隧道症候群
- **腫瘤風險**：大腸息肉／大腸癌風險上升（需定期大腸鏡）
- **甲狀腺結節**

## 📊 嚴重度評分 / 風險分層
| 指標 | 用途 | 意義 |
| --- | --- | --- |
| IGF-1（年齡校正） | 疾病活性 / 治療反應 | 正常化 = 生化緩解 |
| 隨機 GH | 治療達標 | <1.0 ng/mL 為控制良好 |
| 腺瘤大小 | 手術難度 / 壓迫風險 | macroadenoma(>1cm) 壓迫視交叉風險高 |
