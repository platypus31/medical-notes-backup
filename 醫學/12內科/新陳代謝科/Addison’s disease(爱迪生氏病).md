# Addison’s disease（愛迪生氏病）

> [!abstract] 一句話：**原發性腎上腺皮質功能不全** — 腎上腺本身被破壞 → 同時缺 **cortisol（糖皮質）＋ aldosterone（鹽皮質）**。慢性表現隱匿（疲倦、色素沉著、低血鈉），但遇壓力可急性惡化成致命的 **[[Adrenal Crisis(腎上腺危象)]]**。

> [!danger] 🚨 值班紅旗
> 1. **已知 Addison／長期類固醇病人 + 感染／手術／嘔吐 → 隨時可能腎上腺危象** → 低血壓＋低血鈉＋高血鉀＋低血糖，先給 **Hydrocortisone 100 mg IV** 別等抽血結果
> 2. **色素沉著（掌紋、口腔黏膜、疤痕）+ 姿勢性低血壓 + 莫名低血鈉高血鉀** → 想 primary AI，是 secondary 沒有的線索（ACTH↑ 才有色素沉著、缺 aldosterone 才有高血鉀）
> 3. **懷疑 crisis 時抽 cortisol/ACTH/renin/aldosterone 後「立即」給類固醇**，不要為了做 ACTH 刺激試驗延誤治療（可先給 dexamethasone 不影響 cortisol assay）
> 4. **不要只補水不補鹽皮質／糖皮質** — 純輸生理食鹽水無法糾正 cortisol 缺乏造成的血管張力低下

## 📖 定義 / 病生理（這是什麼）
- **定義**：腎上腺**皮質本身**遭破壞導致糖皮質＋鹽皮質激素分泌不足；因回饋失去抑制，**ACTH 代償性升高**（→ 色素沉著）。
- **病因**：
  | 類別 | 舉例 |
  | --- | --- |
  | **自體免疫腎上腺炎（已開發國家最常見，~80%）** | 可為 **APS-1／APS-2** 一部分（合併甲狀腺炎、T1DM、白斑、惡性貧血） |
  | 感染 | **結核（全球最常見之一）**、HIV、黴菌、CMV |
  | 出血／梗塞 | **Waterhouse-Friderichsen（腦膜炎雙球菌敗血）**、抗凝血、抗磷脂症候群 |
  | 浸潤 | 轉移癌、澱粉樣、血鐵沉積 |
  | 藥物 | ketoconazole、etomidate、abiraterone、mitotane（抑制皮質醇合成） |
  | 遺傳 | 先天性腎上腺增生（CAH）、腎上腺腦白質失養症（ALD） |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Fatigue(疲倦)]]、[[Hypotension(低血壓)]]
- Weakness、疲倦、Anorexia(厭食)、weight loss、噁心嘔吐、腹痛
- [[Hypotension(低血壓)]]（尤其**姿勢性**）、鹽分渴求（craving salt）
- **[[Hyperpigmentation(色素沉著病灶)]]**（掌紋、指節、口腔黏膜、疤痕、乳暈 — primary 才有，因 ACTH/MSH↑）
- 女性腋毛／陰毛稀疏、性慾下降（腎上腺雄激素缺乏）
- 實驗室：[[Hyponatremia(低血鈉)]]、高血鉀、低血糖、可見 eosinophilia、輕度高血鈣

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Cortisol(皮質醇)]]、[[ACTH(促腎上腺皮質激素)]]、[[ACTH stimulation test(促腎上腺皮質激素刺激試驗)]]　影像 [[Adrenal CT(腎上腺電腦斷層)]]
- **篩檢**：晨間（08:00）血清 cortisol；<3 μg/dL 高度支持、>15–18 μg/dL 可排除。
- **確診 — ACTH（Cosyntropin）刺激試驗**：注射 **250 μg** cosyntropin，30–60 分後 cortisol **無法上升到 ≥18 μg/dL** → 診斷腎上腺功能不全。
- **鑑別 primary vs secondary — 驗 ACTH**：primary → **ACTH 明顯升高**、renin↑ / aldosterone↓；secondary → ACTH 低或不當正常、鹽皮質軸多正常。
- **病因學**：**21-羥化酶自體抗體（anti-21-hydroxylase）**（自體免疫）、腎上腺 CT（感染／出血／轉移／鈣化 → 結核）、極長鏈脂肪酸（ALD，年輕男性）。

## ⭐ 特徵 / 注意（pearls & pitfalls）
- **色素沉著 = primary 專屬線索**，secondary（垂體問題）不會有。
- **高血鉀 = 缺 aldosterone**，也是 primary 才明顯（secondary 鹽皮質軸靠 RAAS 保留，多無高血鉀）。
- 自體免疫者要篩 **APS**（甲狀腺、T1DM、B12、性腺）。
- 懷疑急症時**先治療再確診**：先抽血 → 立即 dexamethasone（不干擾 cortisol assay）→ 之後補做刺激試驗。

## 💊 治療（Acute / Chronic）
> 用藥 [[Hydrocortisone]]、[[Fludrocortisone]]、[[Dexamethasone]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 急性（腎上腺危象）→ 見 [[Adrenal Crisis(腎上腺危象)]]
- **Hydrocortisone 100 mg IV** stat，接 **50 mg IV q6h**（或 200 mg/24h 連續輸注）＋ 大量生理食鹽水 ± 葡萄糖矯正低血糖。
### 慢性替代
- **糖皮質**：**Hydrocortisone 15–25 mg/day 分次**（早上給 2/3、下午給 1/3，模擬日夜節律）；或 prednisolone。
- **鹽皮質**：**Fludrocortisone 50–200 μg/day**（primary 才需要；監測血壓、血鉀、renin）。
- **雄激素**：部分女性可考慮 DHEA。
### 壓力劑量（sick-day rules，衛教重點）
- 發燒／感染 → 口服劑量 **加倍 2–3 倍**；嘔吐無法口服 / 手術 → 改 IM/IV hydrocortisone。
- 病人**隨身攜帶類固醇警示卡 + 緊急 hydrocortisone 注射劑**。

## ⚠️ 併發症
- **[[Adrenal Crisis(腎上腺危象)]]**：嗜睡／昏迷、體液不足導致低血壓／休克、低血糖、致命
- 過度替代 → 醫源性 Cushing、骨質疏鬆
- 合併其他自體免疫疾病（APS）

## 📊 嚴重度評分 / 風險分層
| 指標 | 用途 | 意義 |
| --- | --- | --- |
| 晨間 cortisol | 篩檢 | <3 支持、>15–18 排除 |
| ACTH 刺激試驗 60min cortisol | 確診 | <18 μg/dL → AI |
| ACTH | primary vs secondary | 高 → primary |
| Na⁺／K⁺／血糖 | 危象嚴重度 | 低鈉高鉀低糖 = 需緊急處置 |
