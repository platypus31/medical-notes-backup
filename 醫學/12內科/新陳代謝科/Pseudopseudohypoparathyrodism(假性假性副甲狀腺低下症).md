# Pseudopseudohypoparathyrodism（假性假性副甲狀腺低下症）

> [!abstract] 一句話：**有 AHO 骨骼表型（矮胖、圓臉、短掌骨），但血鈣、血磷、PTH 全部正常**——「長得像 PHP，代謝卻沒事」。與 PHP 1a 的唯一差別是 **GNAS 突變來自父系（父源印記）**，所以沒有腎小管 PTH 阻抗。

## 📖 定義 / 病生理（這是什麼）
- 與 [[Pseudohypoparathyrodism(假性副甲狀腺低下症)]] 1a **同一個 GNAS(Gsα) 失能突變**，差別只在**父系遺傳**。
- **印記邏輯（本病的全部重點）**：GNAS 的 Gsα 在**腎近端小管以母系對偶基因為主表現**。
  - 突變來自**母方** → 腎小管失去主要 Gsα → **PTH 阻抗（= PHP 1a，低血鈣）**
  - 突變來自**父方** → 腎小管仍有母系正常 Gsα → **無 PTH 阻抗、血鈣正常（= PPHP）**；但骨骼等非印記組織仍受影響 → **只留下 AHO 表型**
- 因此 PPHP＝**AHO without hormone resistance（有骨表型、無荷爾蒙阻抗）**。同一家族內母傳孩子得 PHP 1a、父傳孩子得 PPHP。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Short stature(身材矮小)]]、[[Brachydactyly(短指症)]]
- **AHO 表型**：**矮胖身材、圓臉、短頸、短掌骨／蹠骨（第 4、5 掌骨短，握拳呈「凹陷指節」）、皮下骨化／鈣化、輕度智能障礙**。
- **無低血鈣症狀**：沒有手足抽搐、Chvostek/Trousseau 陰性（因血鈣正常）。
- 部分報告合併子宮內生長遲緩、出生體重偏低（Gsα 父源印記效應）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Calcium(血鈣)]]、[[Phosphorus(血磷)]]、[[PTH(副甲狀腺素)]]　影像 [[Hand X-ray(手部X光)]]
- **生化全正常**：血鈣正常、血磷正常、**PTH 正常**（這是與 PHP 的分水嶺）。
- **臨床 AHO 表型** + 家族史（父系傳遞）。
- **手部 X 光**：短掌骨、皮下鈣化（progressive osseous heteroplasia 需鑑別）。
- **基因檢測**：GNAS 失能突變 + 確認為**父系來源**。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **一句話記法**：**PPHP = AHO 外型 + 正常生化**；**PHP 1a = AHO 外型 + 低血鈣/高磷/高 PTH**。
- **關鍵鑑別點**：兩者長得一樣，抽血才能分——**PPHP 抽血全正常**。
- 遺傳諮詢重點：**同一突變、父母來源決定表型**（imprinting 經典教材例子，與 Prader-Willi/Angelman 並列印記疾病考點）。
- PPHP 患者的女性後代若把突變傳給下一代，孩子可能表現為 **PHP 1a**（因換成母系傳遞）。
- 需與 **progressive osseous heteroplasia（POH）**（廣泛皮下→深部異位骨化，多父系 GNAS 突變）鑑別。

## 💊 治療（Acute / Chronic）
> 用藥（本病無荷爾蒙阻抗，通常不需鈣/維生素D）
- **血鈣正常者不需鈣或維生素 D 治療**——這是與 PHP 的重要處置差異，避免不必要補鈣造成高鈣尿。
- 以**支持性照護**為主：矮身材／骨骼異常之骨科與復健評估、皮下骨化處理、遺傳諮詢。
- 定期追蹤生化（確認未來不轉為需治療的狀態，理論上不會，但確保診斷正確）。

## ⚠️ 併發症
- 皮下／異位骨化造成局部不適、關節活動受限
- 身材矮小、外觀相關心理社會影響
- 誤診為 PHP 而**不必要補鈣 → 高鈣尿／腎結石**（治療陷阱）

## 📊 嚴重度評分 / 風險分層（scoring）
> 無評分系統；診斷確立後多為良性、以支持治療為主
| 分層 | 意義 |
| --- | --- |
| 生化（鈣/磷/PTH 全正常） | 與 PHP 分界，決定「不需補鈣」 |
| 骨化範圍 | 決定骨科/復健介入 |
