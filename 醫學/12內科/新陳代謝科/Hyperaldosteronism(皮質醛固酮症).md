# Hyperaldosteronism（皮質醛固酮症）

> [!abstract] 一句話：醛固酮（aldosterone）分泌過多 → **鈉滯留 + 排鉀排氫** → **高血壓 + 低血鉀 + 代謝性鹼中毒**。是**可矯正的續發性高血壓**中最常見者，靠 **aldosterone/renin ratio（ARR）** 篩檢，別漏掉。

## 📖 定義 / 病生理（這是什麼）
- **原發性（Primary，Conn’s）**：腎上腺本身自主分泌大量 aldosterone → 血容量充足 → **回饋抑制 renin（低 renin）**。→ **高 aldosterone / 低 renin（ARR 高）**。
- **續發性（Secondary）**：有效循環血量不足 / 腎灌流下降 → **RAAS 活化、renin 升高** → 續發 aldosterone 升高（腎動脈狹窄、心衰、肝硬化、利尿劑、腎素瘤）。→ **高 aldosterone / 高 renin**。
- **病生理連鎖**：aldosterone 作用於遠端腎小管 → 保鈉排鉀排氫 → 高血壓、[[Hypokalemia(低血鉀)]]、代謝性鹼中毒；長期還造成心 / 腎 / 血管纖維化（獨立於血壓的靶器官傷害）。
- **原發性病因**：雙側腎上腺增生（idiopathic hyperaldosteronism，最常見 ~60%）> 單側醛固酮腺瘤（[[Conn’s syndrome(康氏症)]] ~30%）> 罕見（單側增生、腎上腺癌、家族型 GRA/FH-II）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Hypertension(高血壓)]]、[[Muscle weakness(肌肉無力)]]
- **高血壓**（常為中重度 / 頑固型，對多種藥反應差）。
- **低血鉀相關**：肌肉無力、抽筋、多尿多渴、心悸 / 心律不整；**但約一半原發性患者血鉀正常**（血鉀正常不能排除！）。
- 通常**無水腫**（aldosterone escape 現象）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Aldosterone-renin ratio(醛固酮腎素比值)]]、[[Serum potassium(血鉀)]]　影像 [[Adrenal CT(腎上腺電腦斷層)]]
- **篩檢**：**Aldosterone/Renin Ratio（ARR）** — 高 aldosterone + 被壓抑的 renin → ARR 上升。抽血前盡量停用干擾藥（MRA 停 4–6 週；β-blocker、ACEI/ARB 也會干擾），矯正低血鉀。
- **確診（負荷試驗）**：口服鈉負荷 24 小時尿 aldosterone、鹽水灌注試驗、fludrocortisone 抑制、captopril challenge — 顯示 aldosterone**不被抑制**。
- **分型定位**：**腎上腺 CT**（腺瘤 vs 增生）；**腎上腺靜脈取樣（AVS）** 是區分單側 vs 雙側的**金標準**（尤其 >35 歲、CT 結節易誤導者）。
- **續發性**：測 renin 若也升高 → 找腎動脈狹窄 / 心衰 / 肝硬化等原因。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **原發性 = 低 renin；續發性 = 高 renin**（一線之隔，考試核心）。
- **血鉀正常 ≠ 排除**：約 50% 原發性患者血鉀正常，僅靠低血鉀篩檢會漏診。
- **頑固型高血壓 + 自發或利尿劑誘發的低血鉀**是最典型的篩檢時機；建議所有中重度 / 頑固高血壓皆篩 ARR。
- **CT 看到腺瘤不代表就是元凶**：可能是無功能偶發瘤 + 對側增生才是分泌源 → AVS 定位避免切錯邊。
- MRA（spironolactone）會嚴重干擾 ARR，篩檢前須停藥或換用不影響 RAAS 的藥（如 verapamil、doxazosin、hydralazine）。

## 💊 治療（Acute / Chronic）
> 用藥 [[Spironolactone(螺內酯)]]、[[Eplerenone(依普利酮)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
- **單側（腺瘤 / 單側增生 / 癌）→ 手術**：腹腔鏡腎上腺切除，多數可改善血壓 / 治癒低血鉀。
- **雙側增生 / 不宜手術 → 藥物**：**礦皮質激素受體拮抗劑（MRA）**
  - **Spironolactone** 起始 12.5–25 mg/day，依血壓 / 血鉀漸增（副作用：男性女乳症、性功能障礙）。
  - **Eplerenone**（選擇性，副作用少，需每日兩次）為替代。
- 補鉀 + 監測腎功能 / 血鉀；家族型 GRA 用低劑量 glucocorticoid 抑制 ACTH。
- **續發性**：治療根本病因（腎動脈狹窄 revascularization、心衰 / 肝硬化本身管理）。

## ⚠️ 併發症
- 頑固高血壓相關：中風、[[Left ventricular hypertrophy(左心室肥大)]]、心衰、心房顫動（獨立於血壓的心血管風險增高）。
- 慢性低血鉀：心律不整、腎小管損傷（低鉀腎病）、肌肉病變。
- 未治療靶器官纖維化（心 / 腎 / 血管）。

## 📊 嚴重度評分 / 風險分層（scoring）
> 無專屬 score；以病因分型決定手術 vs 藥物
| 分型 | Renin | 影像 / AVS | 處置 |
| --- | --- | --- | --- |
| 原發 - 單側腺瘤 / 增生 | 低 | CT 結節 + AVS 側化 | 腎上腺切除 |
| 原發 - 雙側增生 | 低 | AVS 無側化 | MRA 長期 |
| 續發性 | 高 | 找腎 / 心 / 肝病因 | 治本 |
