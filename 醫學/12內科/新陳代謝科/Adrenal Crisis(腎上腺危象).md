# Adrenal Crisis（腎上腺危象）

> [!abstract] 一句話：腎上腺功能不全病人在**壓力（感染／手術／脫水／漏藥）**下急性失代償 → **糖皮質激素絕對不足** → 頑固性低血壓／休克。**懷疑就治療、別等確診** — 延遲給 hydrocortisone 會致死。

> [!danger] 🚨 值班紅旗
> 1. **對輸液／升壓劑無反應的低血壓（distributive shock）+ 已知或疑似 AI** → 立刻 **Hydrocortisone 100 mg IV push**，不要等 cortisol 結果
> 2. 典型組合：**低血壓 + 低血鈉 + 高血鉀 + 低血糖**（primary）；發燒、意識改變、腹痛（可像急腹症）
> 3. **常見誘因是「該加壓劑量沒加」**：長期口服類固醇 / Addison 病人遇感染、嘔吐、手術卻沒 stress dose
> 4. **急救順序：類固醇 → 大量生理食鹽水 → 葡萄糖（低血糖）→ 找誘因（感染 → 給抗生素）**
> 5. 高血鉀通常隨 cortisol + 輸液矯正，**不要急著用 insulin 打鉀導致醫源性低血糖**；先給類固醇
> 6. 若診斷未定，可先用 **Dexamethasone 4 mg IV**（不干擾後續 cortisol / ACTH 刺激試驗），穩定後再確診

## 📖 定義 / 病生理（這是什麼）
- **定義**：急性、危及生命的糖皮質激素（± 鹽皮質）缺乏，導致血管張力喪失、脫水與循環衰竭。
- **機轉**：cortisol 維持血管對兒茶酚胺的反應性與肝醣生成；缺乏 → 血管麻痺性低血壓 + 低血糖；缺 aldosterone → 失鈉保鉀 → 低血鈉高血鉀低血容。
- **常見誘因**：
  | 類別 | 舉例 |
  | --- | --- |
  | 感染 / 敗血症 | 最常見誘因（腸胃炎、肺炎、泌尿道） |
  | 壓力未加量 | 手術、外傷、分娩、精神壓力 |
  | 藥物 | **突然停用長期類固醇**（醫源性最常見）、漏服、嘔吐無法吸收 |
  | 急性破壞 | 腎上腺出血（抗凝、抗磷脂、腦膜炎雙球菌 Waterhouse-Friderichsen）、垂體卒中 |
  | 甲狀腺素啟動 | 未補類固醇即開始 thyroxine（加速 cortisol 代謝） |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Hypotension(低血壓)]]、[[Shock(休克)]]、[[Altered mental status(意識改變)]]
- **循環**：頑固性低血壓、休克、心搏過速
- **腸胃**：噁心、嘔吐、**腹痛（可誤為急腹症）**、腹瀉
- **神經**：無力、倦怠、嗜睡、意識混亂、昏迷
- **其他**：發燒（可為誘因感染或危象本身）、脫水
- **實驗室**：**低血鈉、高血鉀、低血糖**、azotemia、可見高血鈣、eosinophilia

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Cortisol(皮質醇)]]、[[ACTH(促腎上腺皮質激素)]]、[[Electrolytes(電解質)]]、[[Glucose(血糖)]]　影像 [[Adrenal CT(腎上腺電腦斷層)]]
- **臨床診斷為主，治療不可延誤**：抽 **cortisol + ACTH + renin + aldosterone + 電解質 + 血糖**後**立即**治療。
- 危象時**隨機 cortisol < 18 μg/dL 屬不當偏低**（重壓下應 >18–20）。
- 確診（穩定後補做）：**ACTH（Cosyntropin）250 μg 刺激試驗**，60min cortisol <18 μg/dL。
- **找誘因**：血/尿培養、CXR、感染源評估；懷疑出血做腎上腺 CT。

## ⭐ 特徵 / 注意（pearls & pitfalls）
- **「治療不能等診斷」是本病最重要原則** — 死亡多來自延誤給類固醇。
- 對升壓劑反應差的休克，尤其病史有類固醇使用，一律想到本病。
- **急性期 hydrocortisone 高劑量本身有足夠鹽皮質作用**，通常不需另加 fludrocortisone（劑量降到 <50 mg/day 才需補）。
- 甲狀腺功能不全 + 腎上腺不全並存時，**先補類固醇再補甲狀腺素**（反之會誘發危象）。
- 別把腹痛誤當外科急腹症而延誤治療。

## 💊 治療（Acute / Chronic）
> 用藥 [[Hydrocortisone]]、[[Dexamethasone]]、[[Normal saline(生理食鹽水)]]、[[Dextrose(葡萄糖)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 急救（黃金第一小時）
1. **Hydrocortisone 100 mg IV bolus** stat → 續 **50 mg IV q6h** 或 **200 mg/24h 連續輸注**。
   - 診斷未明可先 **Dexamethasone 4 mg IV**（不干擾 cortisol assay）。
2. **大量輸液**：**0.9% Normal Saline 1L 於第 1 小時**快速灌注，依血壓／CVP 續補（常需 2–3 L/首日）。
3. **葡萄糖**：低血糖 → D50W；輸液可用 D5NS。
4. **矯正誘因**：疑感染即經驗性抗生素、控制出血等。
5. 監測：血壓、電解質（Na/K）、血糖、尿量。
### 恢復與維持
- 血流動力穩定後 hydrocortisone **1–3 天內逐步減量**至口服維持劑量（15–25 mg/day 分次）。
- primary AI 病人減量後恢復 **fludrocortisone 50–200 μg/day**。
- **衛教 sick-day rules + 隨身緊急注射 hydrocortisone**，避免復發。

## ⚠️ 併發症
- 循環休克 → 多重器官衰竭、死亡
- 嚴重低血糖 → 腦損傷
- 心律不整（高血鉀）
- 治療延誤 = 高死亡率
- 過度矯正低血鈉 → 滲透性脫髓鞘（矯正速率 <8–10 mmol/L/24h）

## 📊 嚴重度評分 / 風險分層
| 指標 | 用途 | 意義 |
| --- | --- | --- |
| 收縮壓 / 對升壓劑反應 | 休克嚴重度 | 頑固低血壓 = 立即類固醇 |
| Na⁺ / K⁺ / 血糖 | 代謝失衡 | 低鈉高鉀低糖 = 典型危象 |
| 隨機 cortisol | 支持診斷 | 壓力下 <18 μg/dL 不當偏低 |
| 誘因（感染/出血） | 預後 / 處置 | 敗血症合併預後差 |
