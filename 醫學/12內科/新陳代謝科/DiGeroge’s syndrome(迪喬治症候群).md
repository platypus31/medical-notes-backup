# DiGeorge’s syndrome（迪喬治症候群）

> [!abstract] 一句話：**22q11.2 微缺失** → 第三、四咽囊發育缺陷 → **副甲狀腺 + 胸腺不發育**，造成**低血鈣（hypoparathyroidism）+ T 細胞免疫缺損 + 圓錐動脈幹心臟畸形**。值班最怕的是**低血鈣抽搐 / 心律不整**。

> [!danger] 🚨 值班紅旗
> 1. **低血鈣抽搐 / 手足搐搦（tetany）／喉頭痙攣／癲癇** → 查 **ionized Ca、Mg、PTH、心電圖（QT 延長）**；急性有症狀性低血鈣給 **IV calcium gluconate 10% 10–20 mL 稀釋於 D5W 緩推 10–20 分鐘**，續接靜脈點滴，**同時補鎂**（低鎂會使補鈣無效）。
> 2. **嚴重低血鈣可致 QT 延長 → torsades / 心律不整 / 心衰**——低血鈣別當單純電解質小事。
> 3. **完全型（complete DiGeorge）＝ T 細胞近乎缺如** → **只能輸照射過（irradiated）、去白血球的血品**，避免輸血相關 GVHD；**活疫苗禁忌**。
> 4. 新生兒/嬰兒不明原因低血鈣抽搐 + 先天性心臟病 + 特殊面容 → 想 22q11 缺失，送 FISH / CMA。

## 📖 定義 / 病生理（這是什麼）
- **遺傳**：**22q11.2 半合子微缺失**（含 *TBX1* 基因），多為 de novo；體染色體顯性遺傳（若親代帶因）。是最常見的染色體微缺失症候群（~1/4000 活產）。
- **胚胎學**：**第三、第四咽囊（pharyngeal pouch）**發育失敗 →
  - **副甲狀腺不發育** → 原發性副甲狀腺功能低下 → **低血鈣、高血磷、PTH 低/不足**。
  - **胸腺不發育／發育不良** → **T 細胞成熟障礙**（B 細胞相對正常）→ 細胞免疫缺損。
  - **圓錐動脈幹（conotruncal）心臟畸形**：法洛四重症、truncus arteriosus、主動脈弓中斷、VSD。
- **光譜疾病**：與 velocardiofacial syndrome（Shprintzen）同屬 22q11.2 deletion spectrum；表現由**部分型（partial）到完全型（complete）**不等。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Hypocalcemia(低血鈣)]]、[[Tetany(手足搐搦)]]
- 記憶法 **CATCH-22**：**C**ardiac defects、**A**bnormal facies、**T**hymic hypoplasia、**C**left palate（顎裂 / 顎咽功能不全）、**H**ypocalcemia／**H**ypoparathyroidism，缺失位於 **22**q11。
- **低血鈣**：新生兒抽搐、手足搐搦、Chvostek／Trousseau 徵象、感覺異常。
- **免疫**：反覆病毒／黴菌／伺機性感染（T 細胞缺損），完全型近似 SCID。
- **心臟**：conotruncal 畸形（見上）。
- **特殊面容**：眼距寬、低位耳、小下巴、人中短、鼻樑寬。
- 其他：學習障礙、精神疾病（成年思覺失調風險↑）、餵食困難、免疫失調（自體免疫）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Ionized calcium(游離鈣)]]、[[PTH]]、[[T cell subset(T細胞亞群)]]、[[FISH]]　影像 [[Echocardiography(心臟超音波)]]、[[Chest X-ray(胸部X光)]]
- **確診 = 分子診斷**：**FISH（22q11.2 探針）** 或 **染色體微陣列（CMA / array-CGH）**（首選、涵蓋範圍廣）、MLPA。
- **內分泌**：低血鈣、高血磷、PTH 低或不當偏低（原發性副甲狀腺低下）、25-OH vitD。
- **免疫**：淋巴球計數、**T 細胞亞群（CD3/CD4/CD8）**、疫苗抗體反應；胸部 X 光看胸腺影（新生兒胸腺缺如）。
- **心臟**：心臟超音波評估 conotruncal 畸形。
- **鈣化 / 抽搐時**：心電圖看 QT 延長。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **副甲狀腺低下 → 低鈣 + 高磷 + PTH 低**（對比腎衰的低鈣是 PTH 高）。
- **不是所有病人 T 細胞都嚴重缺損**：多為部分型，隨年齡 T 細胞可改善；只有**完全型**需胸腺移植 / 造血幹細胞移植。
- **輸血一律用照射血品**（防輸血相關 GVHD）；**活疫苗**（MMR、水痘、BCG、口服輪狀）在確認 T 細胞功能足夠前**禁忌**。
- 成年後精神科共病（思覺失調、焦慮）風險顯著升高，長期追蹤要含精神評估。
- 顎咽功能不全→鼻音、餵食嗆咳，別誤為單純發展遲緩。

## 💊 治療（Acute / Chronic）
> 用藥 [[Calcium gluconate(葡萄糖酸鈣)]]、[[Calcitriol(骨化三醇)]]、[[Magnesium sulfate(硫酸鎂)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 急性（症狀性低血鈣）
- **IV calcium gluconate 10%**：有症狀（抽搐、tetany、QT 延長）時 10–20 mL 稀釋緩推 10–20 分鐘，續接 calcium gluconate 持續點滴滴定至症狀緩解。
- **同時補鎂**（低鎂使 PTH 分泌與作用受損，不補鎂補鈣無效）。
### 慢性（副甲狀腺低下）
- 口服**鈣鹽 + 活性維生素 D（calcitriol）**維持血鈣於正常低值（避免高尿鈣、腎結石）。
- 難治者可考慮 **recombinant PTH(1-84)**。
### 免疫
- 部分型：抗生素預防、避免活疫苗直到 T 細胞功能達標。
- **完全型**：**胸腺移植**（thymus transplantation）或造血幹細胞移植重建免疫。
### 心臟 / 其他
- 心臟畸形依病灶外科矯正；多科團隊（心臟、免疫、內分泌、遺傳、精神、語言）長期追蹤。

## ⚠️ 併發症
- 低血鈣抽搐、心律不整（QT 延長 → torsades）。
- 反覆 / 伺機性感染、自體免疫疾病。
- 先天性心臟病相關心衰、猝死。
- 發展遲緩、學習障礙、成年精神疾病。
- 顎裂 → 餵食 / 語言 / 中耳炎問題。

## 📊 嚴重度評分 / 風險分層（scoring）
> 無正式 score；以「免疫缺損程度」＋「心臟畸形嚴重度」＋「低血鈣控制」分層
| 面向 | 分層依據 | 意義 |
| --- | --- | --- |
| 免疫 | T 細胞數 / 功能 | 部分型可保守；完全型（近 SCID）需移植 |
| 低血鈣 | 血鈣、抽搐、QT | 有症狀 → 靜脈急救；長期口服鈣+calcitriol |
| 心臟 | conotruncal 畸形型別 | 決定手術時機與預後 |
