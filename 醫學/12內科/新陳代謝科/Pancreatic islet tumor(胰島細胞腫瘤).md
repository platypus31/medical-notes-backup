# Pancreatic islet tumor（胰島細胞腫瘤 / 胰臟神經內分泌腫瘤 pNET）

> [!abstract] 一句話：源自胰島細胞的**神經內分泌腫瘤（pNET）**，依分泌激素分成 insulinoma、gastrinoma 等**功能性**與**非功能性**。值班最關鍵：**Insulinoma → 反覆空腹低血糖（Whipple 三聯）**，是可治癒卻常被當成癲癇/精神病拖延的漏診。

> [!danger] 🚨 值班紅旗（Insulinoma 低血糖）
> 1. **反覆空腹/運動後意識改變、冒冷汗、抽搐 → 先驗血糖，不要先當癲癇/中風**
> 2. **Whipple 三聯**：① 低血糖症狀 ② 當下血糖 < 55 mg/dL ③ 給糖後症狀緩解 → 高度懷疑 insulinoma
> 3. 急性處理：意識清醒給口服糖；意識不清 **IV D50W 50 mL（1 amp）** 或 glucagon 1 mg IM/SC，之後接 **D10W 持續輸注**維持血糖
> 4. **Gastrinoma（Zollinger-Ellison）**：頑固/多發/非典型部位消化性潰瘍 + 腹瀉 → 別只當一般 PU，可致大出血/穿孔
> 5. 疑診時**驗血一定要「症狀發作當下同時抽」**（血糖 + insulin + C-peptide + proinsulin），事後補抽無意義

## 📖 定義 / 病生理（這是什麼）
- 起源於胰島（islet of Langerhans）細胞的神經內分泌腫瘤，分**功能性**（分泌激素有臨床症候群）與**非功能性**（以腫塊/轉移表現）。
- 約 10–20% 與 **[[MEN1(第一型多發性內分泌腫瘤)]]** 相關（尤其 gastrinoma、多發性 pNET）。

| 腫瘤 | 分泌 | 核心症候群 |
| --- | --- | --- |
| **Insulinoma** | Insulin | 空腹低血糖、Whipple 三聯；90% 良性、多為單發 |
| **Gastrinoma** | Gastrin | **Zollinger-Ellison**：頑固/多發消化性潰瘍、腹瀉、GERD（常惡性、位「gastrinoma triangle」） |
| **Glucagonoma** | Glucagon | 高血糖、**壞死性游走性紅斑（NME）**、體重↓、口炎、VTE、貧血 |
| **VIPoma** | VIP | **WDHA**：Watery Diarrhea + Hypokalemia + Achlorhydria（Verner-Morrison） |
| **Somatostatinoma** | Somatostatin | 糖尿病 + 膽石 + 脂肪瀉 三聯 |
| 非功能性 | — | 腹痛/腫塊/黃疸/轉移（診斷時常已大） |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Hypoglycemia(低血糖)]]、[[Diarrhea(腹瀉)]]
- **Insulinoma**：神經低血糖症狀（意識混亂、視力模糊、行為異常、抽搐、昏迷）+ 自主神經症狀（冒汗、心悸、飢餓、顫抖），空腹/運動誘發，進食緩解 → 常伴體重增加。
- **Gastrinoma**：上腹痛、多發或術後復發潰瘍、腹瀉（酸破壞胰脂酶）。
- **Glucagonoma**：特徵性壞死性游走性紅斑（NME）、糖尿病、深部靜脈血栓。
- **VIPoma**：大量水瀉（禁食也不停）、脫水、低血鉀、代謝性酸中毒。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Insulin(胰島素)]]、[[C-peptide(C胜肽)]]、[[Gastrin(胃泌素)]]　影像 [[CT(電腦斷層)]]、[[EUS(內視鏡超音波)]]
- **Insulinoma → 72 小時飢餓試驗（gold standard）**：低血糖時（< 55 mg/dL）同步 insulin ≥3 μU/mL、**C-peptide ≥0.6 ng/mL、proinsulin ≥5 pmol/L**、β-hydroxybutyrate 低、sulfonylurea 篩檢陰性。
  - **C-peptide 高 → 內源性（腫瘤/磺醯脲類）；C-peptide 低 → 外源性打胰島素**（factitious）。
- **Gastrinoma**：空腹 gastrin ↑（>1000 pg/mL 幾近確診）+ **胃 pH < 2**；secretin 刺激試驗（gastrin 反常上升 >120）。**測 gastrin 前須停 PPI**（PPI 會假性升高）。
- **定位**：三相 pancreatic protocol CT/MRI、**EUS（小 insulinoma 最敏感）**、**DOTATATE PET（somatostatin receptor imaging）** 取代舊 OctreoScan；insulinoma SSTR 表現低，選擇性動脈鈣刺激（ASVS）可定位。
- 標記：**Chromogranin A（CgA）** 用於非功能性/追蹤。

## ⭐ 特徵 / 注意（pearls & pitfalls）
- 低血糖 workup 一定**症狀當下同抽**血糖 + insulin + C-peptide + proinsulin + sulfonylurea screen；C-peptide 是區分內源 vs 外源的關鍵。
- 年輕、多發性 pNET 或合併副甲亢/腦垂體瘤 → 篩 **MEN1**。
- Insulinoma 90% 為良性、單發、< 2 cm → 手術（摘除/局部切除）常可治癒。
- Gastrinoma 檢查前務必停 PPI，否則假陽性。

## 💊 治療（Acute / Chronic）
> 用藥 [[Diazoxide]]、[[Octreotide(善得定)]]、[[Proton pump inhibitor(質子幫浦抑制劑)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 急性
- **低血糖**：意識清醒口服糖；不清 IV D50W 1 amp，接 D10W 輸注維持血糖，反覆監測。
- **Insulinoma 術前控糖**：**Diazoxide**（抑制胰島素分泌）± 少量多餐；**Octreotide** 部分有效。
### 根治
- **手術切除**為功能性 pNET 首選（insulinoma enucleation、gastrinoma 切除）。
### 藥物（無法手術/轉移）
- **Somatostatin analog（Octreotide / Lanreotide）** 控症狀 + 抗增殖。
- **Gastrinoma → 高劑量 PPI** 控酸為核心。
- 進展期：**PRRT（Lu-177 DOTATATE）**、標靶 **Everolimus / Sunitinib**、化療（Streptozocin、Temozolomide-based）。

## ⚠️ 併發症
- 反覆嚴重低血糖 → 神經損傷、外傷、猝死。
- Gastrinoma → 潰瘍出血/穿孔。
- VIPoma → 嚴重脫水/低鉀致心律不整。
- 惡性 pNET → 肝轉移、類癌症候群樣。
