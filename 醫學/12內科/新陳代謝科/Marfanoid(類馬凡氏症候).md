# Marfanoid（類馬凡氏症候）

> [!abstract] 一句話：Marfanoid 是一種**體型表現型（habitus）而非單一疾病**——高瘦、四肢/手指細長（arachnodactyly）、關節過度活動。內分泌科看到它最重要的反射動作是：**想到 [[Multiple endocrine neoplasia(多發性內分泌腫瘤)]] type 2B**（marfanoid + 黏膜神經瘤 + 甲狀腺髓質癌 + 嗜鉻細胞瘤），別只當成單純的 [[Marfan syndrome(馬凡氏症)]]。

## 📖 定義 / 病生理（這是什麼）
- **Marfanoid habitus**：一組骨骼型態學特徵的統稱，不是診斷。核心是**四肢相對軀幹過長（dolichostenomelia）**、蜘蛛指（arachnodactyly）、身高偏高、體型瘦削、關節鬆弛。
- 機轉多與**結締組織（fibrillin / collagen）異常**或**RET 相關神經內分泌發育**有關，依病因不同：
  | 病因 | 基因 / 機轉 | 關鍵區辨點 |
  | --- | --- | --- |
  | [[Marfan syndrome(馬凡氏症)]] | **FBN1**（fibrillin-1，AD） | 晶狀體**向上**脫位、主動脈根部擴張/剝離、無智能障礙 |
  | **MEN 2B** | **RET** proto-oncogene（AD） | 黏膜神經瘤、甲狀腺髓質癌、嗜鉻細胞瘤、有 marfanoid 但**無晶狀體脫位/主動脈根擴張** |
  | Homocystinuria（同型胱胺酸尿症） | **CBS**（cystathionine β-synthase，**AR**） | 晶狀體**向下**脫位、**血栓傾向**、智能障礙、可 methionine 誘發 |
  | Loeys-Dietz syndrome | TGFBR1/2 | 動脈迂曲、眼距過寬、懸雍垂裂 |
  | 同型胱胺酸尿以外的 MASS/MFS 譜系 | FBN1 變異度 | 心臟二尖瓣脫垂為主 |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Arachnodactyly(蜘蛛指)]]、[[Joint hypermobility(關節過動)]]
- **骨骼**：身高高、臂展 > 身高、上下半身比例下降、蜘蛛指（**Steinberg thumb sign** / **Walker-Murdoch wrist sign** 陽性）、漏斗胸/雞胸、脊柱側彎、扁平足、高腭弓。
- **依病因附加特徵**：
  - MEN 2B → 嘴唇/舌尖**黏膜神經瘤**（bumpy lips）、腸道神經節瘤致便秘、無淚。
  - Marfan → 晶狀體上移、主動脈根擴張、二尖瓣脫垂。
  - Homocystinuria → 反覆動靜脈血栓、malar flush、智能障礙。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Plasma homocysteine(血漿同型胱胺酸)]]、[[RET gene test(RET基因檢測)]]、[[Calcitonin(降鈣素)]]　影像 [[Echocardiography(心臟超音波)]]、[[Slit lamp exam(裂隙燈檢查)]]
- **第一步是「分流」**：看到 marfanoid habitus，靠附加線索決定往哪個病因查。
- **Marfan**：Ghent nosology（主動脈根 Z-score + 晶狀體脫位 + FBN1 + 系統評分）。心臟超音波看主動脈根、裂隙燈看晶狀體。
- **MEN 2B**：**RET 基因檢測**（codon M918T 最典型）、血清 **calcitonin / CEA**（篩甲狀腺髓質癌）、24 小時尿/血漿 metanephrines（篩嗜鉻細胞瘤）。
- **Homocystinuria**：血漿 homocysteine ↑↑、尿液篩檢、CBS 活性。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **考點鑑別（晶狀體脫位方向）**：Marfan **向上（superotemporal）**、Homocystinuria **向下（inferonasal）**——經典國考題。
- 看到 marfanoid + **黏膜神經瘤 / 便秘 / 甲狀腺結節** → **一定要排 MEN 2B**，因為甲狀腺髓質癌（MTC）在 2B 侵襲性最強、發病最早，可能需**嬰兒期預防性甲狀腺全切**。
- Marfanoid 本身**不是遺傳診斷**——不能因為「長得像 Marfan」就貼標籤，需基因/系統評分佐證。
- Homocystinuria 是**唯一可治療（維生素反應型）** 的——B6(pyridoxine)/B12/folate/betaine，早期偵測可防血栓與智能退化。

## 💊 治療（Acute / Chronic）
> 用藥 [[Losartan]]、[[Beta blocker(乙型阻斷劑)]]、[[Pyridoxine(維生素B6)]]
- **針對病因治療，不治「marfanoid」本身**：
  - Marfan → β-blocker / ARB（losartan）降主動脈剪力、定期心超監測、主動脈根 > 5 cm 考慮預防性手術。
  - MEN 2B → 見 [[Multiple endocrine neoplasia(多發性內分泌腫瘤)]]：**先處理嗜鉻細胞瘤再開甲狀腺/副甲狀腺**，預防性甲狀腺切除。
  - Homocystinuria → pyridoxine ± betaine、低 methionine 飲食、抗血栓風險管理。
> ⚠️ 劑量依最新指引，用藥前查藥典。

## ⚠️ 併發症
- Marfan：主動脈剝離/破裂（致命）、視網膜剝離、氣胸。
- MEN 2B：轉移性 MTC、嗜鉻細胞瘤高血壓危象。
- Homocystinuria：動靜脈血栓栓塞、早發冠心病、骨質疏鬆。

## 📊 嚴重度評分 / 風險分層（scoring）
> Marfan 用 Ghent；MEN 2B 依 RET codon 分風險層決定切除時機
| 評分 / 系統 | 用途 | 意義 |
| --- | --- | --- |
| **Ghent nosology (2010 revised)** | 診斷 Marfan syndrome | 主動脈根 Z-score + ectopia lentis + FBN1 + 系統分 ≥7 |
| **ATA MTC risk（RET codon）** | MEN 2B 甲狀腺切除時機 | M918T = **highest risk**，建議 ≤1 歲切除 |
