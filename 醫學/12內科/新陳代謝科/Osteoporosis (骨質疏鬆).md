# Osteoporosis（骨質疏鬆）

> [!abstract] 一句話：骨量減少 + 骨微結構破壞 → 骨強度下降、**脆弱性骨折（fragility fracture）** 風險上升的全身性骨骼疾病。值班最關鍵：**任何低衝擊（站高跌倒）造成的髖部／脊椎骨折就等於骨鬆，不必等 DEXA**；長期臥床、類固醇使用者是隱形高危群。

## 📖 定義 / 病生理（這是什麼）
- **本質**：骨吸收（osteoclast）> 骨生成（osteoblast） → 骨密度（BMD）↓ + 骨小樑破壞 → 骨脆化。
- **分類**：
  | 類型 | 機轉 / 族群 |
  | --- | --- |
  | **原發性 Type I（停經後）** | Estrogen 缺乏 → osteoclast 活性↑，以**小樑骨（脊椎、遠端橈骨）**流失為主 |
  | **原發性 Type II（老年性）** | 年齡相關、鈣/維生素 D 不足、成骨↓，皮質骨＋小樑骨皆流失（髖部骨折） |
  | **次發性** | **類固醇（最常見藥物性）**、甲狀腺亢進、副甲亢、性腺低下、多發性骨髓瘤、腎病、酗酒、抗癲癇藥、PPI、SGLT2i／thiazolidinedione |
- **主要危險因子**：高齡、女性／停經早、低 BMI、家族史、抽菸、酗酒、久坐、鈣/維生素 D 缺乏、長期糖皮質激素（**≥5 mg prednisolone × ≥3 個月**）。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Back pain(背痛)]]、[[Pathologic fracture(病理性骨折)]]
- 早期**無症狀**（silent disease）——常在第一次骨折才發現。
- **脆弱性骨折**：脊椎壓迫性骨折（身高變矮、駝背 kyphosis / dowager's hump、慢性背痛）、髖部骨折、遠端橈骨（Colles）骨折。
- 脊椎壓迫性骨折**約 2/3 無急性症狀**，靠影像才發現。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Calcium(鈣)]]、[[Vitamin D]]　影像 [[DEXA(雙能量X光吸收儀)]]、[[Spine X-ray(脊椎X光)]]
- **DEXA（Dual energy X-ray Absorptiometry）** 測腰椎（L1–L4）與髖部（total hip / femoral neck）BMD，為金標準。
- **T score**（與同種族年輕人比）：WHO 診斷標準
- **Z score**（同種族同年齡比）：< −2.0 提示「不符年齡的骨量流失」→ 找**次發性病因**

| T score（與同種族年輕人比） | 意義 |
| ----------------- | --------------------- |
| −1 < T | 正常 |
| −2.5 < T < −1 | [[Osteopenia(骨質減少症)]]（骨質減少症） |
| T < −2.5 | Osteoporosis（骨質疏鬆症） |
| T < −2.5 + 脆弱性骨折 | **嚴重骨質疏鬆症（Severe osteoporosis）** |

- **次發性 workup**：Ca、P、ALP、Cr、25-OH vitamin D、TSH、PTH、testosterone（男）、SPEP/尿本瓊氏蛋白（疑骨髓瘤）、24hr 尿鈣。
- **FRAX®**：10 年主要骨鬆性骨折 + 髖部骨折風險，輔助 osteopenia 病人是否需治療。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **DEXA 診斷用 T score；Z score 用來抓次發性病因**（尤其停經前女性/50 歲以下男性用 Z score，不用 T score）。
- 脊椎有壓迫性骨折 → 即使 T score 未達 −2.5 也可診斷**臨床骨質疏鬆**並治療。
- **類固醇性骨鬆**：骨流失最快在頭 3–6 個月，且在**尚未達到 −2.5 前就骨折**（骨質量以外的骨品質下降）→ 開始長期類固醇即應同時補鈣/維生素 D，高風險加雙磷酸鹽。
- 退化性關節炎、脊椎壓迫性骨折、主動脈鈣化會**假性升高**腰椎 BMD → 髖部數值更可靠。

## 💊 治療（Acute / Chronic）
> 用藥 [[Alendronate(福善美)]]、[[Denosumab(保骼麗)]]、[[Teriparatide(骨穩)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 基礎（所有病人）
- **鈣 1000–1200 mg/day + 維生素 D 800–1000 IU/day**、負重運動、戒菸限酒、防跌。
### 抗骨吸收（antiresorptive，第一線）
- **雙磷酸鹽（Bisphosphonate）**：
  - Alendronate（Fosamax）70 mg PO **每週一次**；Risedronate（Actonel）35 mg 每週；Zoledronic acid 5 mg IV **每年一次**。
  - 服用衛教：晨起空腹、配 200–250 mL 白開水、**服後直立 ≥30 分鐘、勿平躺**（防食道炎/潰瘍）。
  - 禁忌：CrCl < 30–35、低血鈣、食道弛緩不能／狹窄。
- **Denosumab（Prolia）** 60 mg SC **每 6 個月**：RANKL 單株抗體；腎功能不佳可用；**停藥反彈** → 不可自行中斷，須接續其他藥。
- **SERM（Raloxifene）**：兼降乳癌風險，但**增加 VTE 與熱潮紅**。
- **Calcitonin**：僅短期用於急性脊椎壓迫性骨折止痛，非長期首選。
### 造骨（anabolic，嚴重/多發骨折/極高風險）
- **Teriparatide / Abaloparatide（PTH 類似物）** SC 每日：促成骨；限用 2 年、有骨肉瘤動物疑慮者禁用；停藥後須接抗骨吸收藥固效。
- **Romosozumab**（抗 sclerostin）每月 SC × 12 個月：一年內心血管事件病史者禁用。
### 其他
- 停經後可考慮 **Estrogen（+ Progesterone）荷爾蒙補充**（權衡乳癌/VTE 風險）。
- **Percutaneous transpedicular vertebroplasty / kyphoplasty（經皮下脊椎修補術）**：疼痛難控的急性脊椎壓迫性骨折。

## ⚠️ 併發症
- **髖部骨折**：一年死亡率約 20–30%，重大失能來源。
- 脊椎壓迫性骨折反覆 → 慢性背痛、駝背、限制性肺功能。
- 治療相關罕見併發：雙磷酸鹽/denosumab → **顎骨壞死（ONJ）**、**非典型股骨骨折（AFF）**（拔牙前評估、長期用藥定期再評估藥物假期）。

## 📊 嚴重度評分 / 風險分層（scoring）
> T score 定診斷；FRAX 定「osteopenia 要不要治療」
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **T score** | 診斷分級 | > −1 正常；−2.5~−1 osteopenia；< −2.5 osteoporosis |
| **FRAX®** | 10 年骨折風險 | 主要骨鬆性骨折 ≥20% 或髖部 ≥3% → 建議治療 |
| **Z score** | 抓次發性 | < −2.0 → 找次發病因 |
