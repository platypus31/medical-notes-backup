# Respiratory Acidosis（呼吸性酸中毒）

> [!abstract] 一句話：肺泡通氣不足 → **CO₂ 蓄積（PaCO₂ ↑）→ pH ↓**。值班最關鍵：**急性 CO₂ retention + 意識改變/CO₂ narcosis 是插管訊號**，而慢性代償者「pH 近正常但 HCO₃⁻ 很高」不要盲目給高濃度 O₂ 把呼吸驅動關掉。

## 📖 定義 / 病生理（這是什麼）
- **核心**：原發性 PaCO₂ 上升（> 45 mmHg）導致 pH < 7.35。CO₂ + H₂O ⇌ H₂CO₃ ⇌ H⁺ + HCO₃⁻，CO₂ 蓄積直接把反應往右推、產生 H⁺。
- **腎臟代償**（保留/新生 HCO₃⁻）：
  | 型別 | 時程 | 代償公式（每 ↑10 mmHg PaCO₂） |
  | --- | --- | --- |
  | **急性** | 分鐘–小時（組織緩衝為主） | HCO₃⁻ ↑ **1** mEq/L；pH ↓ **0.08** |
  | **慢性** | 3–5 天（腎臟代償完成） | HCO₃⁻ ↑ **3.5–4** mEq/L；pH ↓ **0.03** |
- **病生理分類（依 CO₂ 為何排不掉）**：
  | 機轉 | 常見病因 |
  | --- | --- |
  | 中樞呼吸驅動 ↓ | 鴉片/BZD 過量、腦幹病變、Ondine curse、麻醉 |
  | 神經肌肉 | 重症肌無力、GBS、ALS、低血鉀/低血磷、肌肉鬆弛劑殘留 |
  | 胸廓/肋膜 | Kyphoscoliosis、flail chest、大量胸水、肥胖低通氣（OHS） |
  | 氣道/肺 | **COPD 急性惡化（最常見）**、氣喘 status、嚴重肺炎/ARDS 晚期、上呼吸道阻塞 |
  | 呼吸器設定不當 | dead space ↑、通氣量設太低 |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Altered mental status(意識改變)]]
- **急性**：頭痛、焦躁、嗜睡 →（PaCO₂ 極高）**CO₂ narcosis**（昏迷）、asterixis（撲翼樣震顫）、視乳突水腫（腦血管擴張、ICP↑）。
- **交感興奮**：心搏過速、多汗、高血壓；嚴重時心律不整、周邊血管擴張、warm periphery。
- **慢性（如 COPD）**：常無明顯急性症狀，靠代償 pH 接近正常；急性再惡化才出現上述表現。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Arterial Blood Gas(動脈血液氣體分析)]]、[[Electrolytes(電解質)]]　影像 [[Chest X-ray(胸部X光)]]
- **ABG**：pH < 7.35、PaCO₂ > 45 mmHg，是診斷核心。
- **判讀步驟**：① 看 pH → 酸中毒 ② PaCO₂ 高 → 呼吸性 ③ 用上表代償公式判斷急/慢性或混合 ④ **算 A-a gradient** 分辨「純通氣不足（A-a 正常，如過量/中樞）」vs「合併肺實質病（A-a 增大）」。
- **找病因**：胸部 X 光/CT、毒物篩檢、神經肌肉評估、必要時肺功能。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **急慢性鑑別是重點**：相同 PaCO₂ 60，急性 pH≈7.2（危險），慢性 pH≈7.34（代償良好）。看 HCO₃⁻ 升多少判斷。
- **COPD 給氧陷阱**：目標 **SpO₂ 88–92%**，不是越高越好。過度給氧會經 V/Q 失衡（Haldane effect + 低氧性血管收縮解除）加重 CO₂ retention，非單純「關掉缺氧驅動」。
- **HCO₃⁻ > 45 mEq/L** 幾乎一定合併代謝性鹼中毒（如利尿劑、嘔吐），不可能只靠慢性呼酸代償到這麼高。
- Asterixis 不只肝腦病變，**CO₂ narcosis 也會**。

## 💊 治療（Acute / Chronic）
> 用藥 [[Naloxone(納洛酮)]]、[[Bronchodilator(支氣管擴張劑)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 急性
- **治本 = 改善通氣**：找可逆病因（鴉片 → naloxone 0.04–0.4 mg IV 滴定；支氣管痙攣 → 短效 β2 + ipratropium + 類固醇）。
- **通氣支持**：COPD/心衰急性 CO₂ retention 且清醒可配合 → **NIV（BiPAP）為首選**（大幅降低插管率與死亡率）。
- **插管指徵**：意識惡化無法保護呼吸道、NIV 失敗、pH 持續 < 7.25 且惡化、呼吸疲勞。
- **NaHCO₃ 一般不用**（治標且可能加重高碳酸血症、細胞內反常酸化），重點是排 CO₂。
### 慢性
- 治療原發病（COPD 最佳化、OHS 減重 + 夜間 NIV/CPAP、神經肌肉病照護）、長期居家 NIV 於選定病人。

## ⚠️ 併發症
- CO₂ narcosis / 昏迷、呼吸衰竭需插管
- 顱內壓上升、癲癇
- 心律不整（合併低血鉀/缺氧時）
- 肺心症（chronic）、與缺氧並存造成多器官影響

## 📊 嚴重度評分 / 風險分層（scoring）
> 主要看 pH 惡化速度與意識，無單一 score；ABG 趨勢 + 臨床即決策
| 指標 | 意義 |
| --- | --- |
| pH < 7.25 且下降 | 高危、考慮通氣升級/插管 |
| 急性 vs 慢性（HCO₃⁻ 代償幅度） | 決定緊急度 |
| A-a gradient | 分辨通氣 vs 肺實質問題 |
