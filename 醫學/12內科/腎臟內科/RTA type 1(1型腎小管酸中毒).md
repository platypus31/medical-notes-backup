# RTA type 1（1型腎小管酸中毒）

> [!abstract] 一句話：**遠端腎小管（α-intercalated cell）無法泌 H⁺** → normal anion gap（高血氯）代謝性酸中毒，尿液**無法酸化（urine pH >5.5）**，典型伴**低血鉀 + 腎結石／腎鈣化**。值班關鍵：normal AG 代酸 + 尿 pH 頑固偏鹼 + 低 K，就是它。

## 📖 定義 / 病生理（這是什麼）
- **缺陷位置**：遠端腎小管（collecting duct）**α-intercalated cell 的 H⁺-ATPase / H⁺-K⁺-ATPase 或 basolateral Cl⁻/HCO₃⁻ exchanger (AE1)** 功能異常 → **無法分泌 H⁺**、無法產生足夠 titratable acid 與 NH₄⁺ → 全身酸滯留。
- **後果連鎖**：泌 H⁺ 失敗 → 為維持電中性改增泌 **K⁺** → **低血鉀**（與 type 4 相反，是重要鑑別點）；慢性酸血 → 骨鈣動員、hypocitraturia + 鹼性尿 → **calcium phosphate 結石 / nephrocalcinosis**。
- **可低至危險程度**：血 HCO₃⁻ 可 <10 mEq/L（比 type 2 更嚴重的酸中毒）。

| 特徵 | Type 1 (Distal) | Type 2 (Proximal) | Type 4 (Hypoaldo) |
| --- | --- | --- | --- |
| 缺陷 | 遠端泌 H⁺ 失敗 | 近端 HCO₃⁻ 再吸收失敗 | 醛固酮缺乏/阻抗 |
| 血 K⁺ | **↓ 低** | ↓ 低 | **↑ 高** |
| 尿 pH | **>5.5（無法酸化）** | 酸負荷後可 <5.5 | 通常 <5.5 |
| 血 HCO₃⁻ | 可極低（<10） | 12–18（有底限） | 15–20（輕） |
| 結石/鈣化 | **常見（nephrocalcinosis）** | 罕見（反有骨病） | 無 |

- **常見病因**：原發／遺傳（AE1、ATP6V1B1）、**自體免疫（Sjögren 症候群、RA、SLE）**、高鈣尿症、amphotericin B、鋰、腎移植排斥、阻塞性腎病。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Metabolic acidosis(代謝性酸中毒)]]、[[Hypokalemia(低血鉀)]]
- 疲倦、肌無力（低 K）、嚴重者**低鉀癱瘓 / 心律不整**。
- **反覆腎結石 / nephrocalcinosis**、多尿、夜尿。
- 兒童：生長遲滯、佝僂病；成人：骨軟化（osteomalacia）。
- 找病因線索：乾眼乾口（Sjögren）、關節炎（RA/SLE）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Anion Gap(陰離子間隙)]]、[[Urine anion gap(尿液陰離子間隙)]]　影像 [[KUB]]、[[Renal ultrasound(腎臟超音波)]]
- **血氣 + 電解質**：normal AG（高血氯）代謝性酸中毒 + **低血鉀**。
- **尿 pH >5.5 且無法酸化**（即使全身嚴重酸血）→ 診斷核心。
- **Urine anion gap (UAG = U_Na + U_K − U_Cl)：正值（positive）** → 尿 NH₄⁺ 排除不足（腎因性），支持 RTA；相對地腹瀉是負值。
- **氯化銨負荷試驗（NH₄Cl loading）**：酸負荷後尿 pH 仍 >5.5 → 確診 dRTA（現多用替代法）。
- 影像：KUB / 超音波看 **nephrocalcinosis / 結石**。
- 找續發：ANA、anti-Ro/La（Sjögren）、免疫球蛋白。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **記憶法**：type **1** = **1** 的尿 pH「太高」（>5.5 酸化失敗，distal）；type 1、2 都低 K，type 4 高 K。
- Sjögren 是成人 dRTA 最重要續發原因，遇到低 K + normal AG 代酸 + 乾燥症狀要想到。
- 結石／鈣化幾乎只見於 type 1（type 2 因鹼性尿在近端但整體反有骨病、少結石）。
- 補鹼前務必先評估並補 K：**先補鹼會讓 K⁺ 更往細胞內移 → 加重低鉀甚至致命**。

## 💊 治療（Acute / Chronic）
> 用藥 [[Potassium citrate(檸檬酸鉀)]]、[[Sodium bicarbonate(碳酸氫鈉)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
- **口服鹼劑（首選 potassium citrate）**：同時補鹼 + 補 K + citrate 抑制結石，一舉三得。成人約 1–2 mEq/kg/day 分次，依血氣調整。
- 若用 sodium bicarbonate，須**另外並行補鉀**（避免加重低 K）。
- 治療病因（停 amphotericin/鋰、控制自體免疫）。
- 目標：矯正酸血、正常化血 K、預防結石與骨病。

## ⚠️ 併發症
- 低鉀性肌無力 / 週期性癱瘓 / 心律不整
- Nephrocalcinosis、反覆腎結石 → 慢性腎病
- 骨軟化 / 佝僂病、兒童生長遲滯

## 📊 嚴重度評分 / 風險分層（scoring）
> RTA 無正式評分；以血 HCO₃⁻、K⁺ 程度與結石／腎功能損害評估嚴重度
| 指標 | 意義 |
| --- | --- |
| 血 HCO₃⁻ | type 1 可 <10，愈低愈重 |
| 血 K⁺ | 低 K 程度 → 癱瘓/心律不整風險 |
| Nephrocalcinosis / eGFR | 慢性損害與預後 |
