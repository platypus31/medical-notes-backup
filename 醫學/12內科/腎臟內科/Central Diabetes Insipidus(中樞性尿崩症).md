# Central Diabetes Insipidus（中樞性尿崩症）

> [!abstract] 一句話：下視丘／腦垂體後葉分泌 **ADH（vasopressin）不足** → 腎臟無法濃縮尿液 → 大量稀釋尿（多尿）+ 口渴多飲。值班最關鍵：**多尿 + 稀釋尿（Uosm 低）+ 高血鈉傾向**，給 desmopressin（dDAVP）能濃縮尿 = 中樞型（vs 腎源型無反應）。

> [!danger] 🚨 值班紅旗
> 1. **神經外科術後 / 頭部外傷 / 腦死病人突發大量清澈尿（>3–4 mL/kg/hr、Uosm <300）+ 血鈉快速上升** → 疑 CDI，趕快追 Na、換算尿量、給 dDAVP
> 2. **意識清醒且口渴中樞完整者靠猛喝水可維持血鈉正常**；一旦**禁食、插管、意識障礙、口渴中樞受損**，喝不到水 → **血鈉飆升致命**（癲癇、腦病變）
> 3. 高血鈉矯正**不能太快**：慢性高血鈉每 24h 降 **≤ 10–12 mmol/L**，過快 → 腦水腫、癲癇
> 4. **三相反應（triphasic response）**：垂體手術後 DI（1–3 天）→ 短暫 SIADH（低血鈉，dDAVP 別給過頭）→ 永久 DI；別在第二相繼續猛給 dDAVP 造成水中毒
> 5. dDAVP 過量 → 醫源性低血鈉；用藥期間監測血鈉

## 📖 定義 / 病生理（這是什麼）
- **機轉**：ADH（抗利尿激素）由下視丘 supraoptic/paraventricular nucleus 合成、儲存於垂體後葉。CDI = ADH **分泌不足** → 集尿管 aquaporin-2 水通道無法插入頂膜 → 水無法回收 → 排出大量低張尿。
- **對比核心**：CDI 是「ADH 缺」，[[Nephrogenic Diabetes Insipidus(腎源性尿崩症)]] 是「腎臟對 ADH 無反應」。
- **常見病因**：
  | 分類 | 病因 |
  | --- | --- |
  | **腫瘤/浸潤** | craniopharyngioma、germinoma、metastasis、Langerhans cell histiocytosis、sarcoidosis |
  | **手術/外傷** | 經蝶竇垂體手術、頭部外傷、腦死 |
  | **血管** | Sheehan syndrome、動脈瘤、下視丘出血 |
  | **感染/發炎** | 腦膜炎、腦炎、autoimmune hypophysitis |
  | **先天/遺傳** | AVP 基因突變、Wolfram syndrome（DIDMOAD） |
  | **特發性** | 約 1/3（部分為自體免疫） |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Polyuria(多尿)]]、[[Polydipsia(多飲)]]、[[Hypernatremia(高血鈉)]]
- **多尿**（通常 >3 L/day，可達 >10 L）、**夜尿**、強烈**口渴**、偏好冰水。
- 若補水足夠 → 血鈉可正常；**補水不足**（意識障礙、禁食、嬰幼兒、老人）→ 脫水、高血鈉、意識改變、癲癇。
- 合併垂體病灶時可有頭痛、視野缺損、其他垂體前葉功能低下表現。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Serum Osmolality(血漿滲透壓)]]、[[Urine Osmolality(尿滲透壓)]]、[[Copeptin(和肽素)]]　影像 [[Brain MRI(腦部核磁共振)]]
- **初步**：血鈉、血漿滲透壓、尿滲透壓、尿比重、24h 尿量。典型：**尿量大、Uosm 低（<300 mOsm/kg，常 <100）、血漿滲透壓正常偏高、Posm > Uosm**。
- **禁水試驗（water deprivation test）**：禁水後 Uosm 仍無法上升 → DI；接著給 **desmopressin（dDAVP）**：
  - **CDI → Uosm 上升 >50%**（腎臟本來能反應，只是缺 ADH）
  - **腎源型 → 幾乎無反應**
- **Copeptin（和肽素，ADH 前驅片段代理指標）**：現代取代傳統禁水試驗；刺激後 copeptin 低 → CDI。
- **腦部 MRI**：正常後葉的「T1 bright spot」在 CDI 常**消失**；找腫瘤/浸潤病灶。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **與 primary polydipsia（心因性多飲）鑑別**：後者是喝太多把 ADH 壓下去，禁水後尿能濃縮、血鈉偏低；DI 血鈉偏高。
- **與滲透性利尿（高血糖、mannitol）鑑別**：DI 是**低張**尿；滲透性利尿是等張或高張、尿糖/尿素高。
- 垂體術後 **三相反應**別漏：第二相 SIADH 若還照給 dDAVP → 嚴重低血鈉。
- 腦死病人大量多尿常是 DI，會加速血鈉紊亂與器官保存困難。
- 部分型（partial CDI）dDAVP 反應介於中間，需綜合判讀。

## 💊 治療（Acute / Chronic）
> 用藥 [[Desmopressin(dDAVP)]]、[[Vasopressin(血管加壓素)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 急性 / 住院
- **矯正水分缺失**：能喝水者鼓勵飲水；不能喝者靜脈補**低張液（D5W 或 half saline）**，依血鈉缺水量計算，**慢速矯正（每 24h 血鈉降 ≤10–12 mmol/L）**。
- **Desmopressin（dDAVP）**：可鼻噴、口服或靜脈/皮下；急性可 IV/SC 1–2 µg，依尿量與血鈉滴定；用藥後監測血鈉避免醫源性低血鈉。
- 神經外科/腦死情境的大量多尿可用 **vasopressin** 連續輸注控制尿量。
### 慢性
- 長期 **desmopressin**（口服/鼻噴/舌下），劑量個別化，通常睡前一劑控夜尿。衛教「口渴時喝、避免過量飲水」。
- 治療根本病因（腫瘤、浸潤性疾病）。

## ⚠️ 併發症
- **高血鈉性脫水**（尤其口渴中樞受損或無法自主飲水者）→ 意識障礙、癲癇、死亡
- **醫源性低血鈉**（dDAVP 過量 / 補水過多）
- 高血鈉矯正過快 → 腦水腫
- 慢性夜尿影響睡眠與生活品質

## 📊 嚴重度評分 / 風險分層（scoring）
> 無 score；以尿量、血鈉偏移速率、口渴中樞是否完整、能否自主飲水判風險
| 指標 | 意義 |
| --- | --- |
| 血鈉上升速率 | 越快越危險，反映無法代償 |
| 口渴中樞 / 意識 | 受損者無法自保，急症風險高 |
| 尿量 (mL/kg/hr) | 監測 dDAVP 反應與液體管理 |
