# Nephrogenic diabetes insipidus（腎源性尿崩症）

> [!abstract] 一句話：集尿管**對 ADH（vasopressin）無反應** → 無法濃縮尿液的 **water diuresis**（純排水、跟排鹽無關）→ 大量低張多尿 + **高血鈉**。值班最關鍵：跟 central DI 的鑑別靠 **desmopressin 挑戰無反應**，急性期先補水降鈉、慢性期靠 thiazide + 低鹽。

## 📖 定義 / 病生理（這是什麼）
- **機轉**：下視丘/腦下垂體 ADH 分泌正常，但**腎臟集尿管（collecting duct）對 ADH 阻抗** → V2 receptor 或 aquaporin-2（AQP2）水通道失能 → 自由水無法再吸收 → 排出大量稀釋尿。
- 因為丟的是**純水**（water diuresis，非 solute diuresis），血漿滲透壓與血鈉**上升** → 若口渴機制或飲水受限（老人、插管、意識不清）就會急速 **hypernatremia**。
- **危險因子 / 病因分類**：

| 類別 | 常見病因 |
| --- | --- |
| **藥物（最常見後天）** | **Lithium（鋰鹽，經典）**、Demeclocycline、Foscarnet、Cidofovir、Amphotericin B |
| **電解質** | **高血鈣（Hypercalcemia）**、**低血鉀（Hypokalemia）** — 兩者干擾 AQP2 |
| **腎臟病** | 阻塞後利尿、鐮刀型貧血腎病、amyloidosis、Sjögren、多囊腎 |
| **先天（罕見）** | X-linked **AVPR2（V2 receptor）突變**（男嬰居多）、AR/AD **AQP2 突變** |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Polyuria(多尿)]]、[[Hypernatremia(高血鈉)]]
- **多尿（>3 L/day）、夜尿、劇渴（polydipsia）、嗜飲冰水**。
- 若補水不及 → 高血鈉症狀：倦怠、易怒、肌抽動、癲癇、意識改變（嚴重腦部脫水）。
- 先天型嬰兒：反覆發燒、生長遲滯、脫水、嘔吐。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Serum Osmolality(血漿滲透壓)]]、[[Urine Osmolality(尿液滲透壓)]]　影像 [[Brain MRI(腦部磁振造影)]]
- **核心指標**：多尿 + **尿滲透壓低（<300 mOsm/kg，常 <200）+ 血滲透壓高/血鈉高**。
- **禁水試驗（Water deprivation test）**：NDI 禁水後尿仍無法濃縮。
- **Desmopressin（dDAVP）挑戰（鑑別關鍵）**：
  - **NDI → 尿滲透壓上升 <50%（幾乎不動）**；
  - Central DI → 尿滲透壓明顯上升 >50%（因外源 ADH 有效）。
- **Copeptin**（ADH 替代標記）：NDI 基礎值高。
- 找病因：血鈣、血鉀、lithium level、腎功能、腎超音波。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **口訣**：Central DI 給 ADH「有救」；Nephrogenic DI 給 ADH「沒救」（腎臟聾了）。
- **Lithium** 是最常考、最常見的後天 NDI 病因 → 用鋰鹽病人多尿先想 NDI。
- 先矯正**高血鈣、低血鉀**，NDI 可能可逆。
- 跟 [[Primary Polydipsia(原發性多飲症)]] 鑑別：primary polydipsia 血鈉偏**低**、禁水後尿可濃縮；NDI 血鈉偏**高**、禁水尿仍稀。
- **Thiazide 治 NDI 的機轉**（見下方筆者標註）：藉排鈉造成輕度容積收縮 → 促近曲小管等滲吸收 → 到達集尿管的水量減少。

> [!PDF|red] [[長庚銀蛋口袋書(印).pdf#page=28&selection=3,0,85,33&color=red|長庚銀蛋口袋書(印), p.28]]
> > 【Q1 】為何 Thiazide 可以治療 Nephrogenic DI （腎源性尿崩症，NDI ）?
> > 【A1 】首先要區分「利尿劑」與「抗利尿激素」兩者「利尿」的不同。利尿劑是排鹽+排水，代表真正體液的流失（只是每種利尿劑排的鹽跟水的比例不同，ex：Lasix 排水大於排鹽，Thiazide 排鹽大於排水）；而抗利尿激素主要影響水的再吸收，跟鹽無關。 NDI 代表集尿管對於 ADH 沒有反應，因此無法留住水分（Water diuresis，跟鹽無關），而且會造成高血鈉。Thiazide 的作用是減少 Distal tubule 對於鈉的再吸收（藉由抑制 NCC），造成鈉的流失，這邊鈉的流失其實就是真正體液（鹽+水）的流失，流失後會促使 Proximal tubule 再吸收，此處的再吸收是用 Iso-osmotic 的方式吸收，也就是「鹽+水」一起吸收。當「鹽+水」同時吸收時，往下游流的部分自然減少，進而減少因 ADH 沒有作用而造成的水分流失的量。（想像一下，水龍頭沒有水下來，就算下面水管破了，也沒有水可以流失了。）

## 💊 治療（Acute / Chronic）
> 用藥 [[Hydrochlorothiazide]]、[[Amiloride]]、[[Indomethacin]]、[[Desmopressin(dDAVP)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 急性（高血鈉矯正）
- **停掉致病藥物**（lithium 等）、矯正高血鈣/低血鉀。
- **補自由水**：能喝就口服白開水；不能喝則靜脈 D5W 或低張。**降鈉速度 ≤ 10–12 mEq/L/24h**（過快 → 腦水腫；慢性高血鈉更要慢）。
- 先算 free water deficit 再補。
### 慢性
- **低鈉低蛋白飲食**（減少滲透負荷 → 減尿量）。
- **Thiazide（HCTZ 25 mg PO qd–bid）**：矛盾性減少尿量（見上）。
- **Amiloride**：尤其 **lithium 誘發 NDI 首選**（阻 ENaC → 減少 lithium 進入主細胞）。
- **NSAID（Indomethacin）**：抑制前列腺素 → 增加集尿管對水再吸收（注意腎毒/胃腸副作用）。
- 先天 NDI：thiazide + amiloride + 低鹽 ± NSAID 合併。

## ⚠️ 併發症
- **高血鈉腦病變**（脫水、癲癇、意識改變、嬰兒腦損傷）
- 反覆脫水、生長遲滯（先天型）
- 長期大量尿 → **非阻塞性腎積水/巨大膀胱**（hydronephrosis）
- 治療相關：thiazide 低血鉀/低血鈉、NSAID 腎損傷與消化道出血

## 📊 嚴重度評分 / 風險分層（scoring）
> NDI 無正式評分系統；嚴重度由血鈉與症狀決定
| 指標 | 用途 | 意義 |
| --- | --- | --- |
| 血鈉 / 血滲透壓 | 判急性嚴重度 | Na >160 mEq/L 或神經症狀 → 急重症 |
| dDAVP 反應 | 鑑別 central vs nephrogenic | 尿滲透壓升 <50% → NDI |
