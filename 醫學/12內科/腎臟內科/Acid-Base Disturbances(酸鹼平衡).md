# Acid-Base Disturbances（酸鹼平衡）

> [!abstract] 一句話：用 ABG（pH／PaCO₂／HCO₃⁻）三步驟系統化判讀「原發障礙 → 代償是否恰當 → anion gap」，**值班最關鍵**是別漏掉「pH 正常卻是雙重障礙互相抵消」與「代償不足／過度＝合併第二種障礙」。

> [!danger] 🚨 值班紅旗
> 1. **pH < 7.20 的嚴重酸血症** → 心肌收縮力↓、對 catecholamine 反應↓、致命性心律不整；查 K⁺（酸中毒把 K⁺ 趕出細胞，**血鉀假性偏高、總體鉀其實缺**，補鹼會使 K⁺ 驟降）
> 2. **High AG metabolic acidosis 的 MUDPILES**（尤其 Methanol／Ethylene glycol／DKA／Lactate／Salicylate）→ 找得到可救的病因；lactate >4 是敗血症紅旗
> 3. **代償永遠不會過度矯正 pH**：若「代謝酸的呼吸代償」讓 pH 反而 >7.45，代表合併呼吸鹼（如 salicylate 中毒的經典雙重障礙）
> 4. **嚴重代謝鹼 pH > 7.55** → 抽搐、心律不整、組織缺氧（氧解離曲線左移）；多為 volume/Cl⁻/K⁺ 缺失，補 NS + KCl 即改善
> 5. **急性呼吸酸 PaCO₂ 急升 + 意識變化** → CO₂ narcosis，準備氣道／NIV，別只給高濃度 O₂（COPD 病人可能抑制呼吸驅動）

## 📖 定義 / 病生理（這是什麼）
- **四大原發障礙**：代謝酸（HCO₃⁻↓）、代謝鹼（HCO₃⁻↑）、呼吸酸（PaCO₂↑）、呼吸鹼（PaCO₂↓）。身體以另一系統代償（腎↔肺），但**代償不會把 pH 拉回完全正常**。
- **代謝酸分兩型**：
  - **High Anion Gap（AG = Na⁻ − (Cl⁻ + HCO₃⁻)，正常 8–12）**：加了未測定酸 → **MUDPILES**（Methanol、Uremia、DKA/ketoacidosis、Paraldehyde/Propylene glycol、Isoniazid/Iron、Lactic acidosis、Ethylene glycol、Salicylate）
  - **Normal AG（hyperchloremic）**：丟 HCO₃⁻ 或加 Cl⁻ → 腹瀉、RTA、carbonic anhydrase inhibitor、生理食鹽水過量
- **代謝鹼分兩型**（看尿 Cl⁻）：**Cl⁻-responsive（尿 Cl⁻ <20）**：嘔吐/NG suction、利尿劑（後期）、volume depletion；**Cl⁻-resistant（尿 Cl⁻ >20）**：原發性醛固酮增多、Cushing、嚴重低血鉀、Bartter/Gitelman。

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Dyspnea(呼吸困難)]]、[[Altered Mental Status(意識改變)]]
- **代謝酸**：Kussmaul 呼吸（深快代償）、噁心、意識改變、低血壓；水果味口氣（DKA ketone）。
- **代謝鹼**：低鉀相關無力、抽搐、感覺異常（游離鈣↓）、代償性換氣不足。
- **呼吸酸**：嗜睡、頭痛、asterixis、CO₂ narcosis。
- **呼吸鹼**：焦慮、口周／指端麻木、頭暈、tetany（急性游離鈣↓）。

### 五步驟系統判讀（值班 SOP）
1. **看 pH**：<7.35 酸血症 / >7.45 鹼血症（即使代償後 pH 正常，仍有原發障礙）。
2. **找原發障礙**：pH 偏酸 → 看是 HCO₃⁻↓（代謝酸）還是 PaCO₂↑（呼吸酸）；pH 偏鹼反之。
3. **算代償是否恰當**（見下表）：代償不足或過度＝合併第二種障礙。
4. **算 Anion Gap**：AG >12 → high AG metabolic acidosis（即使 pH/HCO₃⁻ 正常也可能藏著）。
5. **High AG 時算 Delta-Delta（ΔAG/ΔHCO₃⁻）**：比值 <1 合併 normal AG 代謝酸；>2 合併代謝鹼。

### 預期代償公式
| 原發障礙 | 預期代償 |
| --- | --- |
| 代謝酸 | **Winter's：PaCO₂ = 1.5×HCO₃⁻ + 8 ± 2**（PaCO₂ 應≈末兩位小數 pH） |
| 代謝鹼 | PaCO₂ ↑ 0.7 mmHg／HCO₃⁻ ↑1 mmol/L |
| 急性呼吸酸 | HCO₃⁻ ↑1／PaCO₂ ↑10 |
| 慢性呼吸酸 | HCO₃⁻ ↑3.5／PaCO₂ ↑10 |
| 急性呼吸鹼 | HCO₃⁻ ↓2／PaCO₂ ↓10 |
| 慢性呼吸鹼 | HCO₃⁻ ↓4–5／PaCO₂ ↓10 |

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **代償永遠不會把 pH 過度矯正**：若代謝酸卻 pH 偏鹼，一定合併呼吸鹼（salicylate 中毒經典）。
- **Osmolar gap** 升高 + high AG → 疑 methanol/ethylene glycol 中毒（給 fomepizole）。
- **低白蛋白會低估 AG**：每降 1 g/dL albumin，AG 校正 +2.5。
- **DKA 治療中**：high AG 逐漸轉成 normal AG（酮酸排掉、給 NS 補 Cl⁻），此時 HCO₃⁻ 回升慢是正常。
- **混合障礙提示**：AG 明顯升高但 HCO₃⁻ 只輕微下降 → 疑合併代謝鹼掩蓋。

## 💊 治療（Acute / Chronic）
> 用藥 [[Sodium Bicarbonate(碳酸氫鈉)]]、[[Potassium Chloride(氯化鉀)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
- **治本 > 治標**：多數酸鹼障礙的核心是**矯正病因**（給 volume/insulin、停毒物、改善通氣）。
- **代謝酸給 HCO₃⁻ 的時機有爭議**：一般 pH <7.1（或嚴重高鉀合併酸）才考慮；**lactic acidosis 常規補鹼無明確 benefit**（BICAR-ICU trial 僅在 AKI 亞群見益處）。DKA 除非 pH <6.9 否則不建議補鹼。
- **代謝鹼**：Cl⁻-responsive 給 **NS + KCl**（補 volume/Cl⁻/K⁺）；Cl⁻-resistant 治病因（醛固酮拮抗劑如 spironolactone）；嚴重時 acetazolamide。
- **呼吸酸**：改善通氣（NIV / 插管），治潛在原因（COPD、鴉片過量給 naloxone）。
- **呼吸鹼**：治焦慮／疼痛／缺氧／敗血症等原發原因。

## ⚠️ 併發症
- 嚴重酸血症：心肌抑制、血管擴張性低血壓、致命性心律不整、高血鉀。
- 過快矯正低血鈉相關代謝障礙 → 滲透性脫髓鞘。
- 補鹼過度 → 反彈性代謝鹼、游離鈣↓抽搐、CSF 反常酸化。

## 📊 嚴重度評分 / 風險分層（scoring）
| 評分 | 用途 | 分數 → 意義 |
| --- | --- | --- |
| **pH 絕對值** | 立即危險度 | <7.20 或 >7.55 為危急，須積極處理 |
| **Anion Gap** | 分類代謝酸 | >12 → high AG，觸發 MUDPILES 鑑別 |
| **Delta-Delta** | 偵測混合障礙 | <1 合併 normal AG 酸；>2 合併代謝鹼 |
| **Osmolar gap** | 毒物性酸中毒 | >10 → 疑甲醇/乙二醇 |
