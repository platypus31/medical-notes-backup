- 計算 AG（anion gap）= (Na⁺) − ((Cl⁻) + (HCO₃⁻)) = 10–12mmol/L
> [!PDF|yellow] [[長庚銀蛋口袋書(印).pdf#page=56&selection=41,0,355,3&color=yellow|長庚銀蛋口袋書(印), p.56]]
> > 公式由來： 測到的「陽」離子 + 未測到的「陽」離子 = 測到的「陰」離子 + 未測到的「陰」離子 -> 「測到的」陽離子 - 「測到的」陰離子 = 「未測到的」陰離子 - 「未測到的」陽離子 -> (Na+) - ( (Cl-) + (HCO3-) ) = ( (Albumin) + (酸根陰離子) ) - ( (K+) + (Ca2+) + (Mg2+) ) 
> > ![[長庚銀蛋口袋書(印).pdf#page=56&rect=37,518,547,643&color=yellow|長庚銀蛋口袋書(印), p.56]]
> > 正常值：AG = 10 ± 2
> > 若 AG 大於正常值，需再多計算 Δ AG/Δ HCO3-（看 Δ HCO3-是否正確代償 Δ AG） 
> > - 若 Δ AG/Δ HCO3- = 1~2（AG 上升 = HCO3-下降）（多出來的酸根陰離子恰好由 HCO3-滴定中和） ->  單純的 High AG 代謝酸 
> > - 若 Δ AG/Δ HCO3- < 1（AG 上升 < HCO3-下降）（有多的 HCO3- loss） ->  High AG 代謝酸 + Normal AG 代謝酸 
> > - 若 Δ AG/Δ HCO3- > 2（AG 上升 > HCO3-下降）（有多的 HCO3- gain） ->  High AG 代謝酸 + 代謝鹼

> [!PDF|note] [[Pocket Medicine The Massachusetts General Hospital Handbook of Internal Medicine, 8th Edition.pdf#page=251&selection=42,0,78,23&color=note|Pocket Medicine The Massachusetts General Hospital Handbook of Internal Medicine, 8th Edition, p.251]]
> > If ↑ glc, use measured not corrected Na 
> > Expected AG is (albumin) × 2.5 (ie, 10 if albumin is 4 g/dL, 7.5 if albumin is 3 g/dL) 
> > ↑ AG → ↑ unmeasured anions such as organic acids, phosphates, sulfates 
> > ↓ AG → ↓ alb or ↑ unmeasured cations (Ca, Mg, K, Li, Ig), bromide/iodine toxicity

> > 1）Ketoacidosis （酮酸中毒）：[[Diabetic Ketoacidosis(糖尿病酮酸血症)]]、酗酒、過度飢餓 
> > 檢驗：Ketone Body (U) 、Blood Ketone Quantitative Test (B) 
> > 2）Lactic Acidosis （乳酸中毒）：[[Sepsis(敗血症)]]、缺血、[[Hypoxmia(低血氧)]]、[[Respiratory Failure(呼吸衰竭)]] 
> > 檢驗：Lactate 
> > 3）[[Uremia(尿毒症)]] （[[Renal Failure(腎衰竭)]]）：磷酸根、硫酸根堆積
> > 檢驗：Creatinine 
> > 4） 甲醇（Methanol ）、乙醇（Ethanol ）、乙二醇（Ethylene glycol ）、異丙醇（Isopropanol ）中毒 
> > 問病史、算 Osmolality gap （OG > 10 ） 
> > 5）水楊酸（Salicylate ）、乙醛（Paraldehyde ）中毒
> > 問病史、算 Osmolality gap （OG < 10 ） 
> > 《公式》 OG （Osmolality Gap ）= Measured Osmoles - Calculated Osmoles Calculated Osmole = （2[Na-] + [Glucose]/18 + [BUN]/2.8 ）

  - Normal AG 代謝酸
  > [!PDF|yellow] [[長庚銀蛋口袋書(印).pdf#page=57&selection=109,0,262,11&color=yellow|長庚銀蛋口袋書(印), p.57]]
> > - 計算 UAG（Urine Anion Gap）《公式》UAG = (UNa + UK) - UCl
> > - 公式由來： 測到的「陽」離子 + 未測到的「陽」離子 = 測到的「陰」離子 + 未測到的「陰」離子 -> 「測到的」陽離子 - 「測到的」陰離子 = 「未測到的」陰離子 - 「未測到的」陽離子 -> (UNa + UK) - UCl = 未測到的陰離子 - 未測到的陽離子（NH4+）
> > - 若 UAG < 0 -> 代表尿中未測到的陽離子（NH4+）增加 -> 身體酸中毒時，腎臟卻可正常排 H+ -> 考慮 GI loss HCO3-或 Type II RTA 無法再吸收 HCO3-造成酸中毒
> > - 若 UAG > 0 -> 代表尿中未測到的陽離子（NH4+）減少 -> 腎臟排 H+減少 -> 考慮 Type I RTA、Type IV RTA

- GI loss
- RTA
> [!PDF|yellow] [[長庚銀蛋口袋書(印).pdf#page=57&selection=264,0,405,16&color=yellow|長庚銀蛋口袋書(印), p.57]]
> > - GI Loss HCO3- ：Diarrhea、Fistula、External pancreatic or small-bowel drainage、Ureterosigmoidostomy 
> > UAG < 0 
> > - Renal Tubular Acidosis （RTA ） 
> > a. [[RTA type 1(1型腎小管酸中毒)]] （Distal ）
> > 定調：自體免疫（[[Sjogren’s syndrome(修格蘭氏症候群)]]）破壞遠端腎小管 -> 無法排 H+，造成酸中毒、低血鉀 
> > UAG > 0（尿中 H+減少 -> NH4+減少）
> > Urine pH > 5.3 
> > b. [[RTA type 2(2型腎小管酸中毒)]] （Proximal ）
> > 定調：蛋白（[[Multiple myeloma(多發性骨髓瘤)]]）破壞近端腎小管或服用 Acetazolamide -> 無法再吸收 HCO3-，造成酸中毒、低血鉀 
> > UAG < 0 
> > FEHCO3- > 15%（尿中 HCO3-增加） 
> > c. [[RTA Type 4(4型腎小管酸中毒)]] - 定調：Hypo-RAS -> Aldosterone 作用減少 -> 腎臟留鉀留氫，高血鉀造成近端腎小管製造 NH3 減少 
> > UAG > 0（尿中 NH3 減少 ->NH4+減少）
> > Urine pH < 5.3
