# Capillary Leakage Syndrome（毛細血管滲漏症候群）

> [!abstract] 一句話：微血管內皮通透性突然暴增 → 血漿與蛋白漏到組織間隙，造成「**低血壓休克＋全身水腫＋血液濃縮（Hct↑）＋低白蛋白**」的矛盾組合。值班最關鍵：這是**分佈性＋低容積**休克，急性期猛灌液會在滲漏停止後回吸階段變成致命肺水腫。

> [!danger] 🚨 值班紅旗
> 1. **休克＋水腫＋血比容不降反升（血液濃縮）＋低白蛋白** 這個「矛盾三聯」→ 想到 SCLS / capillary leak；一般敗血症或出血性休克 Hct 不會升
> 2. **三相病程**：前驅（虛弱/肌痛/腹痛）→ **滲漏期（休克、少尿、全身水腫、可 rhabdomyolysis + compartment syndrome）** → **回吸期（recruitment，液體回血管內 → 急性肺水腫、心衰）**
> 3. **液體復甦要克制**：滲漏期補到維持器官灌流即可，過度輸液會在回吸期爆發肺水腫；隨時準備利尿與呼吸支持
> 4. 找誘因：**IL-2 / gemcitabine 等化療、G-CSF、敗血症、蛇毒、造影劑、engraftment syndrome、Clarkson disease（特發性，常伴 MGUS monoclonal gammopathy）**
> 5. 特發型（Clarkson）急性發作死亡率高，**預防性 IVIG** 是目前最有實證的長期預防

## 📖 定義 / 病生理（這是什麼）
- **核心機轉**：微血管內皮細胞連接（endothelial junction）功能失常 → 蛋白質可自由通過的孔徑增大 → 血漿與白蛋白（分子量 ~69 kDa）漏入組織間隙。血管內容積驟減 → **低血容性、分佈性混合休克**；同時因失去的是「無細胞的血漿」，紅血球留在血管內 → **血液濃縮（hemoconcentration，Hct 反升）**。
- **回吸期（recruitment）**：滲漏停止後，漏到第三腔室的液體回流血管內 → 若前期補太多液 → 血管內容積過載 → **急性肺水腫、心衰**。
- **分類**：
  | 類型 | 誘因 | 特徵 |
  | --- | --- | --- |
  | **特發性（Clarkson disease）** | 不明，>80% 伴 MGUS（monoclonal gammopathy） | 反覆發作、可致命、需長期預防 |
  | **次發性（更常見於病房）** | IL-2、gemcitabine、G-CSF、敗血症、燒傷、胰臟炎、engraftment syndrome、蛇毒、造影劑 | 隨誘因移除多可恢復 |

## 🩺 臨床表現（症狀 / 徵象）
> 連症狀卡 [[Hypotension(低血壓)]]、[[Edema(水腫)]]、[[Shock(休克)]]
- **前驅期**：非特異性虛弱、疲倦、肌痛、腹痛、噁心、口渴（1–2 天）。
- **滲漏期**：突發低血壓／休克、全身性水腫（含四肢、顏面、漿膜腔積液）、少尿、體重快速上升。
- 嚴重者：肢體腫脹造成 **compartment syndrome**、橫紋肌溶解（rhabdomyolysis）、缺血性器官損傷。
- **回吸期**：多尿、水腫消退，但可出現急性肺水腫、心衰竭表現（呼吸困難、端坐呼吸）。

## 🔬 診斷（診斷標準 / 要開的檢查影像）
> 檢查 [[Hematocrit(血比容)]]、[[Albumin(白蛋白)]]、[[Serum Protein Electrophoresis(血清蛋白電泳)]]　影像 [[Chest X-ray(胸部X光)]]
- **診斷是臨床＋排除**：低血壓 + 血液濃縮（**Hct↑、Hb↑**）+ 低白蛋白血症（albumin↓），且**排除**敗血症、心因性、失血、過敏性休克等其他休克。
- **矛盾點是關鍵**：一般休克大量輸液後 Hct 會下降，SCLS 卻反升。
- **MGUS 篩檢**：疑特發型時查 SPEP / 免疫固定電泳 / 血清游離輕鏈（>80% Clarkson 有 monoclonal protein，多為 IgG-κ）。
- 排除鑑別：敗血症（血培養、procalcitonin）、過敏性休克（tryptase）、心衰（BNP、echo）、腎病症候群/肝硬化的低白蛋白性水腫（尿蛋白、肝功能）。

## ⭐ 特徵 / 注意（pearls & pitfalls，易漏易考）
- **「休克但 Hct 升高」是最經典的診斷線索**——多數休克 Hct 是降的。
- 特發型（Clarkson）**幾乎都伴 monoclonal gammopathy（MGUS）**，但很少進展成 multiple myeloma。
- 容易被誤當敗血症猛灌液 → 回吸期肺水腫致命。**輸液要保守、滾動評估**。
- 抽血時因血液濃縮，Hb/Hct/白血球/血小板都可能假性偏高。
- 化療（尤其 **IL-2、gemcitabine**）與 **G-CSF** 是病房常見的次發誘因，用藥史一定要問。

## 💊 治療（Acute / Chronic）
> 用藥 [[Intravenous Immunoglobulin(靜脈免疫球蛋白)]]、[[Terbutaline]]、[[Theophylline(茶鹼)]]
> ⚠️ 劑量依最新指引，用藥前查藥典
### 急性期（滲漏期）
- **支持為主**：維持組織灌流的**最低必要輸液**（晶體液），避免過度輸液；必要時加**升壓劑**（norepinephrine）維持 MAP。
- 監測 compartment syndrome、rhabdomyolysis、腎功能；嚴重肢體腫脹考慮筋膜切開。
- 移除／治療誘因（停化療、G-CSF；治療敗血症）。
- 嚴重急性特發性發作可考慮 **IVIG**、高劑量類固醇（實證有限，個案報告）。
### 回吸期
- 警覺容積過載 → **利尿劑**、必要時呼吸器支持治療肺水腫；停止積極輸液。
### 慢性預防（特發型 Clarkson）
- **月投 IVIG（約 1–2 g/kg/month）** 是目前最有實證的長期預防，能顯著降低發作頻率與死亡率。
- 次選：**terbutaline + theophylline**（提升 cAMP、穩定內皮）曾用於預防，實證較弱。

## ⚠️ 併發症
- **回吸期急性肺水腫、心衰竭**（過度輸液的醫源性後果）
- **Compartment syndrome**（肢體嚴重腫脹）→ 需筋膜切開
- **橫紋肌溶解、急性腎損傷（AKI）**
- 缺血性器官損傷、休克相關多重器官衰竭
- 深部靜脈栓塞（血液濃縮 + 制動）

## 📊 嚴重度評分 / 風險分層（scoring）
> 無標準化 score；以血流動力學不穩定程度、Hct 升幅、器官損傷（AKI/rhabdo/compartment）判嚴重度
| 指標 | 意義 |
| --- | --- |
| Hct 升幅 + 白蛋白降幅 | 反映滲漏嚴重度 |
| 需升壓劑 / 少尿 | 器官灌流不足、預後差 |
| Compartment / rhabdomyolysis | 重症、需外科與腎臟監測 |
