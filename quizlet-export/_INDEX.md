## 怎麼匯入 Quizlet
1. Quizlet 建新 set → 點「Import」（匯入）
2. 貼上對應分區 `.txt` 全文
3. Term/Definition 之間選 **Tab**，卡片之間選 **New line（換行）**
4. Import → 完成

每分區各建一個 set 較好管理；想要一次全部就用 `all.txt`（2389 張）。
